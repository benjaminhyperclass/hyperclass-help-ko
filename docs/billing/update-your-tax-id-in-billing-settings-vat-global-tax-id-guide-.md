# 결제 설정에서 세금 ID 업데이트하기 - VAT/국제 세금 ID 가이드

세금 ID 정보는 정확한 세금 계산과 관련 규정 준수를 위해 항상 최신 상태로 유지해야 합니다. 이 가이드에서는 세금 ID를 업데이트하는 방법, VAT 적용 시점, 그리고 지역에 맞는 올바른 세금 ID 유형을 선택하는 방법을 설명합니다.

조직이 세금 면제 자격을 갖춘 경우, 유효한 세금 ID를 입력하면 향후 인보이스에 적절한 세금 처리가 반영됩니다.

## **세금 요구사항 이해하기**

지역에 따라 구독이나 갱신과 같은 특정 구매에 세금이 부과될 수 있습니다. 예를 들어, 조직이 유럽연합, 노르웨이, 터키, 또는 영국에 위치한 경우 부가가치세(VAT)가 적용되어 인보이스에 별도 항목으로 표시될 수 있습니다.

등록된 사업체이고 유효한 VAT ID를 제공하는 경우(해당하는 경우), 향후 구매가 역과세 규칙에 따라 VAT 면제 자격을 얻을 수 있습니다.

---

## **세금 ID 업데이트하기**

결제 설정(Billing Settings)에서 직접 세금 ID를 업데이트할 수 있습니다.

### **1단계: 결제 설정 접근하기**

다음 경로로 이동하세요:

**Settings(설정) → Billing(결제) → Payments(결제)**

**Tax Information(세금 정보)** 섹션을 찾으세요.

![세금 정보 섹션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155066046736/original/iFya0ce2pE1WknIKd98NMIGB5mPcF6uXug.png?1772494499)

### **2단계: 세금 정보 편집하기**

Tax Information 섹션에서 **Edit(편집)** 아이콘을 선택하세요.

![편집 아이콘](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155066046757/original/Ce49fNeUorSvyvpOQUzlVZiIpCvDL22Ttw.png?1772494531)

### **3단계: 세금 세부정보 입력하기**

Tax ID 팝업에서:

- 적절한 **Tax ID Type(세금 ID 유형)**을 선택하세요
- **Tax ID number(세금 ID 번호)**를 입력하세요
- **Save(저장)**을 클릭하세요

![세금 ID 입력 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155066046775/original/1NwNRvE3R7ASBVM_hYedcQ8_Hynfw5ibRQ.jpeg?1772494580)

### **선택사항: 세금 ID가 없는 경우**

다음과 같은 경우 **"I do not have a Tax ID(세금 ID가 없습니다)"**를 선택하세요:

- 세금 ID가 없는 경우
- 드롭다운에 세금 ID가 목록에 없는 경우
- 세금 ID 공개를 원하지 않는 경우

이렇게 하면 세금 정보 업데이트에 대한 추가 안내가 표시되지 않습니다.

## **복수 세금 ID**

조직에 여러 세금 ID가 있고 어떤 것을 입력해야 할지 확실하지 않은 경우, 세부정보를 제출하기 전에 지원팀에 문의하여 명확하게 확인하세요.

---

## **올바른 세금 ID 유형 선택하기**

Tax ID Type 드롭다운은 지역별로 구성되어 있습니다. 올바른 지역을 선택하면 국가별 세금 ID의 적절한 검증이 보장됩니다.

올바른 지역을 선택한 후 국가별 세금 ID 번호를 입력하세요. 시스템이 자동으로 해당하는 세금 형식을 식별합니다.

아래는 지원되는 모든 국가와 세금 ID 유형의 전체 목록입니다.

## **자주 묻는 질문**

**Q: 세금 ID를 업데이트해야 하는 이유가 무엇인가요?**

세금 ID는 정확한 세금 계산과 사업체 위치에 기반한 적절한 인보이스 발행을 위해 필요합니다.

**Q: 세금 ID를 입력하면 VAT가 자동으로 제거되나요?**

VAT 면제 자격이 있고 유효한 VAT ID를 제공하면, 향후 인보이스에 해당하는 역과세 규칙이 반영될 수 있습니다.

**Q: 잘못된 세금 ID 유형을 선택했다면 어떻게 하나요?**

검증에 실패하면 세금 ID 모달을 다시 열어서 올바른 지역을 선택한 후 저장하세요.

**Q: 나중에 세금 ID를 업데이트할 수 있나요?**

네. 결제 설정에서 언제든지 세금 ID를 업데이트할 수 있습니다.

---

## **도움이 필요하신가요?**

어떤 세금 ID 유형을 선택해야 할지 확실하지 않다면:

- 사업체가 등록된 지역을 확인하세요
- 공식 세금 ID 형식을 확인하세요
- 최신 세금 인증서를 검토하세요

세금 ID 저장 시 지속적으로 문제가 발생하면, 지원팀에 문의하기 전에 세금 ID 유형과 국가 세부정보를 준비하세요.
