# 화상회의 계정 연결하기

지원되는 화상회의 계정을 연결하면 예약이 만들어질 때마다 자동으로 미팅이 생성됩니다. 한 번 활성화하면 모든 신규 예약에 고유한 화상 미팅 링크가 자동으로 포함되어, 별도의 수동 설정이 필요하지 않습니다. 이를 통해 불필요한 커뮤니케이션을 줄이고 시간을 절약하며, 모든 예약이 일정 잡히자마자 바로 참여할 수 있는 상태가 됩니다.

캘린더에서 화상 미팅 옵션을 미팅 장소로 선택하면, 시스템이 링크 생성과 전달을 자동으로 처리합니다. 생성된 링크는 캘린더 초대장에 포함되어 양쪽 모두 즉시 접근할 수 있습니다.

# **화상회의 계정 연결하기**

자동 미팅 링크 사용을 시작하려면 캘린더 연결 설정에서 화상회의 계정을 연결해야 합니다. 이 연결을 통해 안전한 인증이 가능해지고 시스템이 대신 미팅 링크를 생성할 수 있습니다.

**화상회의 연결 접근**

이 이미지는 캘린더 설정 내에서 화상회의 연결을 관리하는 위치와 새 연결을 추가하는 옵션을 보여줍니다.

이 영역에서 화상회의 옵션을 선택하면 연결 과정이 시작됩니다. 기존 계정 자격 증명으로 로그인하고 미팅 링크 생성에 대한 권한을 부여하는 과정을 안내받게 됩니다.

**보안 계정 로그인 화면**

성공적인 인증 후 계정이 연결된 상태로 표시되며, 해당 캘린더와 함께 사용할 준비가 완료됩니다.

## **캘린더에서 자동 미팅 링크 사용하기**

연결이 완료되면 캘린더의 미팅 장소를 설정할 때 화상회의 옵션을 사용할 수 있게 됩니다. 이 옵션을 선택하면 각 예약이 자동으로 새로운 고유 미팅 링크를 생성합니다.

![확인된 화상회의 연결](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063305049/original/yXAj-EUfnZKww93zk0uACKIAMWYj0-TViw.jpeg?1769152321)

**확인된 화상회의 연결**

이 이미지는 화상회의 계정이 연결되어 사용할 준비가 된 상태를 보여줍니다.

예약이 생성되면 시스템이 실시간으로 미팅 링크를 생성하고 캘린더 이벤트에 첨부합니다. 예약하는 사람이나 사용자 모두 추가 작업이 필요하지 않습니다.

## **화상회의 연결 관련 중요 사항**

각 사용자는 하나의 워크스페이스 내에서 한 번에 하나의 화상회의 계정만 연결할 수 있습니다. 필요한 경우 같은 계정을 여러 워크스페이스에서 재사용할 수 있습니다. 또한 각 사용자는 다른 사용자에게 영향을 주지 않고 독립적으로 자신만의 계정을 연결할 수 있습니다.

미팅 링크는 연결이 설정된 후 생성되는 신규 예약에만 생성됩니다. 기존 예약은 소급 적용되지 않습니다.

## **자주 묻는 질문**

**Q: 신규 예약에 대해 미팅 링크가 자동으로 생성되나요?**

네. 화상회의 계정이 연결되고 미팅 장소로 선택되면, 각 신규 예약에 대해 고유한 미팅 링크가 자동으로 생성됩니다.

**Q: 참가자들에게 수동으로 미팅 링크를 보내야 하나요?**

아니요. 미팅 링크는 캘린더 초대장에 포함되어 자동으로 공유됩니다.

**Q: 여러 사용자가 각자의 화상회의 계정을 연결할 수 있나요?**

네. 각 사용자는 다른 사용자에게 영향을 주지 않고 독립적으로 자신의 계정을 연결할 수 있습니다.

**Q: 화상회의 계정 연결을 해제하면 어떻게 되나요?**

화상회의 계정이 다시 연결될 때까지 신규 예약에서 미팅 링크가 더 이상 생성되지 않습니다.
