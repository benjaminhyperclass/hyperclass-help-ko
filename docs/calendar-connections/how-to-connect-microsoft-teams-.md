# Microsoft Teams 연결하는 방법

Microsoft Teams를 연결하면 수동으로 설정하지 않아도 미팅에 보안 화상 회의 링크가 자동으로 포함됩니다. 연결 후 Microsoft Teams를 미팅 장소로 선택하면 새로운 예약마다 고유한 미팅 링크가 생성되어 캘린더 이벤트에 추가되고 모든 참여자와 공유됩니다. 이를 통해 예약 과정이 더욱 원활해지고 미팅 링크를 수동으로 생성할 필요가 없어집니다.

# 연결하기 전에 필요한 것

연결을 완료하려면 미팅에 사용하고자 하는 Microsoft Teams 계정에 접근할 수 있어야 합니다. 연결한 계정이 Microsoft Teams가 미팅 장소로 선택된 예약의 미팅 링크를 생성하는 역할을 담당하게 됩니다.

---

## Microsoft Teams 연결하기

캘린더 연결 설정에서 직접 Microsoft Teams를 연결할 수 있습니다. 캘린더(Calendars) 영역에서 연결(Connections) 섹션을 열고 화상 회의 보기로 전환하세요. 아직 연결된 화상 회의 도구가 없다면 새 연결을 추가하는 옵션이 표시됩니다.

![Microsoft Teams 연결 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063305788/original/nQ_aPzFFGSndwSs0o7IBg-B5G9Du8aoLDQ.jpeg?1769153062)

화상 회의 연결 영역

이 이미지는 캘린더 연결 내 화상 회의 섹션을 보여주며, Microsoft Teams를 미팅 옵션으로 추가할 수 있는 곳입니다.

이 화면에서 새 화상 회의 연결 추가 옵션을 선택하면 로그인 프로세스가 시작됩니다. Microsoft Teams 자격 증명으로 로그인하고 접근 권한을 부여한 후에는 연결이 활성화되어 사용할 준비가 됩니다.

---

## 미팅 링크 생성 방식

Microsoft Teams가 연결되면 지원되는 캘린더의 선택 가능한 미팅 장소로 표시됩니다. Microsoft Teams를 미팅 장소로 선택하여 예약을 생성하면 고유한 미팅 링크가 자동으로 생성됩니다. 이 링크는 캘린더 초대에 포함되어 모든 참석자와 공유되므로, 모든 사람이 미팅에 즉시 접근할 수 있습니다.

초기 연결 후에는 추가 설정이 필요하지 않습니다.

---

## 중요 사용 참고사항

각 사용자는 한 번에 하나의 Microsoft Teams 계정만 연결할 수 있습니다. 필요한 경우 동일한 Microsoft Teams 계정을 여러 캘린더 환경에서 재사용할 수 있습니다. 하나의 캘린더 설정 내에서 서로 다른 사용자들은 선호도와 접근 권한에 따라 각자의 Microsoft Teams 계정을 연결하거나 동일한 계정을 사용할 수 있습니다.

나중에 Microsoft Teams 연결이 끊어지면 연결이 복원될 때까지 향후 예약에서 미팅 링크가 생성되지 않습니다.

## 자주 묻는 질문

**Q: Microsoft Teams를 연결한 후에도 미팅 링크를 수동으로 만들어야 하나요?**

아니요. Microsoft Teams가 미팅 장소로 선택되면 미팅 링크가 자동으로 생성됩니다.

**Q: 여러 사람이 Microsoft Teams를 연결할 수 있나요?**

네. 여러 사용자가 동일한 계정을 사용하거나 각자 다른 계정을 사용해서 Microsoft Teams를 연결할 수 있습니다.

**Q: 연결 후 기존 예약에도 미팅 링크가 추가되나요?**

연결 후 생성된 새로운 예약에만 자동으로 미팅 링크가 포함됩니다.

**Q: Microsoft Teams 연결이 끊어지면 어떻게 되나요?**

연결이 다시 설정될 때까지 새로운 예약에서 미팅 링크가 생성되지 않습니다.
