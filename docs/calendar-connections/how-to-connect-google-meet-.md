# 구글 미트(Google Meet) 연결 방법

이 문서에서는 구글 미트가 캘린더 예약과 연동하여 안전한 화상 회의 링크를 자동으로 생성하는 방법을 설명합니다. 활성화하면 모든 예약에 고유한 구글 미트 링크가 포함되어 수동 설정 없이도 화상 회의를 쉽게 진행하고 참여할 수 있습니다.

## 구글 미트란?

구글 미트(Google Meet)는 화상 회의를 진행하는 도구입니다. 캘린더와 연결하면 각 예약에 대해 고유한 회의 링크를 자동으로 생성합니다.

이를 통해 사용자가 수동으로 회의를 생성할 필요 없이 모든 예약에 안전하고 바로 사용 가능한 화상 링크가 포함됩니다.

## 구글 미트와 캘린더 예약의 연동 방식

구글 캘린더 계정이 연결되면 구글 미트가 자동으로 활성화됩니다.

새로운 예약마다 고유한 회의 링크가 생성되고, 이는 캘린더 이벤트에 직접 포함됩니다. 참가자들은 한 번의 클릭으로 회의에 참여할 수 있어 전반적인 예약 및 회의 경험이 향상됩니다.

구글 미트를 위한 별도 설정은 필요하지 않습니다.

## 구글 미트 연결하기

구글 미트는 구글 캘린더 연결을 추가하면 자동으로 사용 가능해집니다.

1. 설정(Settings)의 캘린더 연결 영역으로 이동합니다.
2. **Video Conferencing(화상 회의)** 섹션을 엽니다.
3. **Google Meet**를 선택하고 구글 캘린더 연결을 진행합니다.
4. 구글 계정을 선택하고 필요한 권한을 허용하여 인증을 완료합니다.

인증이 완료되면 구글 미트가 성공적으로 연결되어 사용할 준비가 됩니다.

![구글 미트 연결 설정 이미지](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063208367/original/xV3Wg4xmn8Grq-ce27TIv6NTsG8q3F6qeQ.png?1769061625)

## 회의 링크 생성 확인하기

회의 링크가 생성되려면 구글 캘린더가 **연결된 캘린더(linked calendar)**로 선택되어야 합니다.

구글 캘린더를 연결된 캘린더로 설정하면:

- 시스템에서 생성된 예약이 캘린더와 동기화됩니다.
- 캘린더에서 생성한 이벤트가 시스템으로 다시 동기화됩니다.
- 각 예약에 대해 구글 미트 링크가 자동으로 생성됩니다.

연결된 캘린더가 선택되지 않으면 예약에 회의 링크가 생성되지 않습니다.

올바르게 연결되면 구글 미트가 활성화된 화상 회의 옵션으로 표시됩니다.

![회의 링크 생성 확인 이미지](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063208375/original/Q83P2Cq4J055IHqrX48Ehwh0kf92Rtbmzw.png?1769061645)

## 구글 미트 제거하기

구글 미트와 구글 캘린더 연결은 직접 연결되어 있습니다.

- 구글 캘린더 연결을 제거하면 구글 미트도 함께 제거됩니다.
- 구글 미트 연결을 해제하면 관련 캘린더 연결도 제거됩니다.

제거 후에는 새로운 예약에 화상 회의 링크가 더 이상 포함되지 않습니다.

![구글 미트 제거 이미지](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063208377/original/cigDcPtF_rjOHnzom-txe4Gdj3i1fKmuVA.png?1769061662)

## 자주 묻는 질문

**Q: 별도의 구글 미트 설정이 필요한가요?**

아니요. 구글 캘린더를 연결하면 구글 미트가 자동으로 활성화됩니다.

**Q: 회의 링크가 생성되지 않는 이유는 무엇인가요?**

회의 링크가 생성되려면 구글 캘린더를 연결된 캘린더로 선택해야 합니다.

**Q: 여러 개의 캘린더를 연결할 수 있나요?**

예. 여러 캘린더를 연결할 수 있지만, 연결된 캘린더로는 하나만 선택할 수 있습니다.

**Q: 캘린더 연결을 제거하면 어떻게 되나요?**

구글 미트가 비활성화되고, 향후 예약에는 회의 링크가 포함되지 않습니다.
