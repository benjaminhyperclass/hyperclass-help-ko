# 마켓플레이스 앱 등록 유형 설정

하이퍼클래스 계정에서 앱이 노출되도록 등록 유형을 설정하세요.

처음 앱을 생성할 때 선택할 수 있습니다.

목차
- [유형: 스탠다드](#유형-스탠다드)
- [유형: 화이트라벨](#유형-화이트라벨)

## 유형: 스탠다드

스탠다드 앱은 배포 유형에 따라 Highlevel 도메인 하에서 에이전시와 하위 계정 레벨 모두에서 표시됩니다. 이러한 앱은 화이트라벨 도메인 하에서는 에이전시와 하위 계정 레벨에서 표시되지 않습니다.

## 유형: 화이트라벨

화이트라벨 앱은 배포 유형에 따라 화이트라벨 및 비화이트라벨 도메인 모두에서 에이전시와 하위 계정 레벨에서 표시됩니다. 이러한 앱은 앱 속성, 세부 정보 또는 회사 웹사이트에서 Highlevel 또는 Gohighlevel에 대한 참조를 포함해서는 안 됩니다. 이러한 참조가 있으면 마켓플레이스 팀의 앱 검토 과정에서 승인이 거부됩니다.

화이트라벨 도메인 하에 있는 에이전시와 하위 계정은 이러한 앱만 설치할 수 있습니다. 유형의 특성상 화이트라벨 인증 앱은 화이트라벨 및 비화이트라벨 에이전시와 하위 계정 모두에 대해 나열되므로 더 넓은 범위에 도달하는 경향이 있습니다. 따라서 최대한의 가시성을 위해 앱을 완전히 화이트라벨로 만드는 것을 권장합니다.
