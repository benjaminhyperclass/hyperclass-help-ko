## Hyperclass의 SEO 기능이란?

SEO는 Search Atlas로 구동되는 통합 검색 엔진 최적화 솔루션입니다. 이 도구를 통해 비즈니스와 에이전시는 별도의 서드파티 도구 없이도 웹사이트를 최적화할 수 있습니다. 키워드 리서치, 웹사이트 감사, 백링크 분석, 콘텐츠 최적화, SEO 보고서 등 강력한 기능을 제공합니다.

자동화 및 화이트라벨 기능을 통해 에이전시는 브랜딩을 일관성 있게 유지하면서 고객에게 SEO 서비스를 원활하게 제공할 수 있습니다.

## SEO 주요 기능

SEO는 웹사이트 최적화, 순위 추적, 감사 실행, SEO 캠페인 효율적 관리를 위한 포괄적인 도구 모음을 제공합니다. 모든 핵심 기능에 대한 자세한 분석은 다음과 같습니다:

#### 여러 사이트 관리

- 하나의 대시보드에서 여러 웹사이트의 SEO 성능을 최적화하고 추적합니다.
- 중앙 집중식 관리를 통해 에이전시가 여러 고객 웹사이트를 효율적으로 처리할 수 있습니다.
- 데이터를 체계적으로 유지하면서 다양한 사이트에 맞춤형 SEO 전략을 할당합니다.

#### AI를 활용한 자동 문제 해결

- 일반적인 SEO 오류를 즉시 해결하는 AI 기반 권장사항을 제공합니다.
- 다음 항목의 자동 수정:
  - 메타 설명 문제
  - 제목 태그 문제
  - 누락된 이미지 alt 태그
  - 키워드 문제

#### 온페이지 감사

- 상세한 온페이지 SEO 감사를 수행하여 순위에 영향을 미치는 오류를 식별합니다.
- 다음과 같은 핵심 온페이지 요소를 분석하고 최적화합니다:
  - 제목 태그 및 메타 설명
  - 헤더 구조 (H1, H2 등)
  - 이미지 최적화 및 alt 태그
  - 내부 링크 구조
- AI 기반 수정 권장사항과 함께 우선순위가 매겨진 문제 목록을 생성합니다.

#### 키워드 리서치 및 추적

- 고급 리서치 도구로 고성능 키워드를 발견합니다.
- 키워드 난이도, 검색량, 순위 잠재력을 분석합니다.
- 실시간 업데이트로 시간 경과에 따른 키워드 순위를 추적합니다.
- 히트맵을 사용하여 키워드 성능 트렌드를 시각화합니다.

#### 웹사이트 감사 및 최적화

- 검색 가시성에 영향을 미치는 문제를 식별하는 전체 사이트 감사를 수행합니다.
- 다음 항목의 자동 감지 및 해결:
  - 중복 콘텐츠
  - 깨진 링크 및 404 오류
  - 스키마 마크업 오류
  - 페이지 로드 속도 문제
- 지역 타겟팅 검색 결과를 최적화하는 로컬 SEO 도구
- 심층 도메인 및 경쟁사 분석을 위한 사이트 탐색기

#### 키워드 순위 추적기

- 최대 1,000개의 키워드 순위를 추적합니다.
- 검색 엔진에서의 순위 변동을 모니터링합니다.
- 성장 기회와 개선이 필요한 영역을 식별합니다.
- 여러 고객을 관리하는 에이전시를 위한 다중 사이트 키워드 추적

#### 백링크 분석 및 모니터링

- 웹사이트의 백링크를 분석하여 품질과 순위에 미치는 영향을 평가합니다.
- 독성 백링크를 식별하고 거부하여 검색 페널티를 방지합니다.
- 자동 추적으로 새로운 백링크와 손실된 백링크를 모니터링합니다.
- 링크 구축 전략을 개선하기 위한 경쟁 백링크 분석

#### Content Genius - AI 기반 콘텐츠 최적화

- NLP 콘텐츠 분석: AI가 키워드 사용, 의미적 관련성, 헤딩, 메타 데이터, 내부 링크, 사용자 의도에 대한 콘텐츠를 검토합니다.
- 즉시 수정 제안: 누락된 헤딩, 약한 내부 링크, 불완전한 메타데이터를 강조 표시합니다. 수정 적용 시 크레딧을 사용하며, 단순 조회는 무료입니다.
- 내부 링크 가이드: 권위와 참여도를 높이는 전략적 내부 링크를 권장합니다.
- 무한 스크롤 인터페이스: 필터링과 정렬로 빠른 분류가 가능한 연속 보기에서 모든 AI 권장사항을 탐색합니다.
- 정확한 문제 추적: 플래그된 페이지와 수정 사항의 카운트가 모든 보기에서 일관되게 유지됩니다.
- 크레딧 관리: 수정 적용 시 AI 할당량을 사용합니다(예: 프리미엄 플랜에서 기사당 3크레딧).
- 대규모 최적화: 연속 스크롤링으로 대량 검토를 지원하며, 세부 창을 닫고 필터를 사용하면 더 부드러운 성능을 제공합니다.

#### 콘텐츠 플래너

- 콘텐츠 전략을 효율적으로 구성하고 구조화합니다.
- 타겟 콘텐츠 제작을 위해 최대 15개의 주제 클러스터를 계획합니다.
- 기둥 페이지와 지원 콘텐츠에 대한 AI 기반 권장사항을 제공합니다.
- 검색 의도와 SEO 모범 사례에 맞는 콘텐츠 정렬을 보장합니다.

#### SEO 리포팅 및 분석

- 실행 가능한 인사이트가 포함된 심층적이고 데이터 중심의 SEO 보고서를 생성합니다.
- 다음과 같은 핵심 SEO 성능 지표를 추적합니다:
  - 키워드 순위
  - 오가닉 트래픽 트렌드
  - 참여 지표
- 여러 고객을 관리하는 에이전시를 위한 다중 사이트 보고
- 고객에게 브랜드별 전문적인 SEO 인사이트를 제공하는 화이트라벨 보고

#### 퍼널 및 웹사이트와의 긴밀한 연동

- 퍼널 페이지를 최적화하여 오가닉 트래픽과 전환율을 향상시킵니다.
- 고전환 세일즈 페이지를 위한 AI 기반 SEO 권장사항을 제공합니다.
- SEO 전략과 리드 생성 퍼널 간의 원활한 통합을 보장합니다.

## 요금제 구조 이해하기

SEO는 유연한 청구 모델을 따릅니다:

에이전시 요금:
- 활성화된 하위 계정당 월 $79

재판매 모델(에이전시용)
- 고객이 에이전시에 직접 결제합니다.
- Stripe 또는 다른 게이트웨이를 통해 결제를 관리합니다.
- SEO 기능을 사용하는 하위 계정당 요금을 청구합니다.

- **Show Sub-Accounts(하위 계정 표시)**를 클릭하여 각 고객별 요금을 확인합니다.

## 할당량 가시성 및 소진 표시기

- 기사 생성을 위한 프리미엄 AI 할당량은 할당량 정보/할당량 사용량 영역에 표시되어 사용 가능한 사용량을 추적할 수 있습니다.
- 할당량이 소진되면 소진된 할당량만 빨간색으로 강조 표시됩니다. 같은 행의 다른 할당량은 정상으로 유지되어 여전히 사용 가능한 것을 명확히 볼 수 있습니다.

## 시작하기 단계별 가이드

참고: SEO 모듈은 **에이전시가 해당 하위 계정에 대해 SEO 재판매를 활성화한 후에만** 하위 계정에 표시됩니다.

- 시작하려면 하위 계정에 로그인하여 **Sites(사이트)** > **SEO**로 이동합니다.

![SEO 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155058753768/original/hO9rFp7foN54lbfFDSiGihJ5B2HUzCxfcw.png?1763558735)

- 메인 SEO 대시보드 탭에서 SEO 섹션에는 세 개의 주요 탭이 있습니다: Site Projects, Local SEO, Settings.

Site Projects: 웹사이트(도메인) 모니터링용 - 전체 사이트 지표, 상태, 백링크, 트래픽 등을 표시합니다.

- Local SEO: 지역 비즈니스의 Google Maps 순위 추적용 - 지역 키워드 위치에 대한 인터랙티브 히트맵을 사용합니다.

- Settings: 할당량 사용량을 확인합니다.

![SEO 대시보드 탭](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155058753871/original/GvV513PquEABRYmNmvK3woK_1Yh62s9BXA.jpeg?1763558804)

- Launch Advanced SEO: SEO 대시보드 내에서 "Launch Advanced SEO"라고 표시된 버튼을 볼 수 있습니다. 이를 클릭하면 새 창에서 심층 분석, 콘텐츠 생성, 보고서를 위한 확장된 툴킷과 함께 전체 SEO 도구 모음이 열립니다.

![Launch Advanced SEO 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155058754007/original/pIKCMk8NSex53Sk1r6lp_wJmW4KW7sNbrw.jpeg?1763558857)

## 자주 묻는 질문

**Q: Sites 아래에 SEO 탭이 보이지 않는 이유는?**
SEO 탭은 에이전시가 해당 특정 하위 계정에 대해 SEO 재판매를 활성화한 후에만 나타납니다.

**Q: 여러 사이트를 관리할 수 있나요?**
예, 사이트별 전략으로 하나의 대시보드에서 여러 웹사이트를 추적하고 최적화할 수 있습니다.

**Q: AI가 자동으로 수정할 수 있는 것은?**
메타 설명, 제목 태그, 누락된 이미지 alt 텍스트, 키워드 문제(AI 권장사항과 즉시 수정 제공)를 자동으로 수정할 수 있습니다.

**Q: 순위 추적기의 제한은?**
최대 1,000개의 키워드를 추적할 수 있으며, 검색 엔진과 여러 사이트에서 추적 가능합니다.
