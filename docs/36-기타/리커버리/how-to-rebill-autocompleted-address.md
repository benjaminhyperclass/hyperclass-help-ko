# Hyperclass 폼과 설문에서 주소 자동완성 사용하기

주소 자동완성 기능이 모든 Hyperclass 계정에서 **무료이자 무제한**으로 제공됩니다. 이 기능을 통해 리드와 고객이 폼과 설문에서 정확한 주소 정보를 쉽게 입력할 수 있습니다. 사용자가 주소 필드에 타이핑하면 실시간 추천 주소가 나타나고 거리, 도시, 주/도, 국가, 우편번호 필드를 자동으로 채워줍니다. 빠르고 정확하며 일관된 데이터 수집을 보장합니다.

---

## 주소 자동완성이란?

주소 자동완성은 Hyperclass 폼과 설문의 표준 주소 필드를 스마트한 예측 검색 바로 변환합니다. 통합된 글로벌 데이터베이스를 사용하여 사용자가 타이핑할 때 관련 주소 제안을 표시해서 폼 제출을 더 빠르고 오류 없게 만듭니다.

정확한 주소 데이터는 워크플로우, 세분화, 리포팅, 배송 기반 액션에 필수적입니다. 주소 자동완성은 타이핑 오류를 제거하고 일관된 형식을 보장하며 폼 작성 속도를 높여서 사용자 경험과 데이터 신뢰성을 모두 개선합니다.

### **스마트 검색 제안**

사용자가 주소 필드에 타이핑하면 Hyperclass이 실제 주소 데이터를 기반으로 실시간 예측 제안을 제공합니다.

### **주소 필드 자동 입력**

사용자가 드롭다운에서 주소를 선택하면 모든 관련 필드(거리, 도시, 주/도, 국가, 우편번호)가 자동으로 채워집니다. 이를 통해 일관된 형식을 보장하고 수동 데이터 입력 오류를 제거합니다.

---

## 주소 자동완성의 주요 장점

주소 자동완성 기능은 연락처 데이터가 검증되고 일관되며 올바른 형식으로 되도록 보장하여 모든 폼 제출을 향상시킵니다. 이를 통해 후속 수정 작업을 줄이고 사용자 만족도를 높입니다.

- **무료 & 무제한:** 모든 Hyperclass 계정에 사용량 제한 없이 포함
- **데이터 정확도 개선:** 일관되고 검증된 주소 입력 보장
- **빠른 제출:** 리드와 고객의 타이핑 부담 감소
- **높은 전환율:** 마찰 없는 사용자 경험으로 더 많은 폼 완성 유도
- **리포팅을 위한 더 나은 데이터:** 표준화된 주소 형식으로 자동화 정확도 향상

## 주소 자동완성 활성화 방법

### **주소 필드 추가**

`Sites(사이트) → Forms(폼)` (또는 `Surveys(설문)`)로 이동하여 주소 자동완성을 사용할 폼 빌더를 엽니다.

`Form Elements(폼 요소)` 패널에서 `Address(주소)` 필드를 폼 레이아웃으로 드래그합니다. 주소 요소는 `Updated(업데이트됨)`로 표시되어 자동완성 기능을 지원함을 나타냅니다.

![주소 필드 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055758329/original/1keSyOM9IjxVc9cggUh6vpY5sReynZIdUQ.png?1760147263)

### 주소 필드 설정에 접근하기

주소 필드를 추가한 후 클릭하여 화면 오른쪽의 설정 패널을 엽니다.

이 패널에서 주소 동작을 커스터마이즈하고 자동완성을 활성화하며 사용자가 검증된 주소를 선택해야 하는지 제어할 수 있습니다.

![주소 필드 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055758336/original/pFSPtlYwSv71eVBrmuOAC8FmwNJFsykfug.png?1760147347)

### 주소 자동완성 활성화

설정 패널에서 `Auto-Complete Address(주소 자동완성)`라는 토글을 찾아 **ON**으로 전환합니다.

활성화하면 이 설정이 표준 주소 입력을 사용자가 타이핑할 때 정확한 주소를 제안하는 스마트 검색 바로 변환합니다.

![자동완성 활성화](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055758423/original/LW7dk4ZxDjLJEfzjjsy-vKEb3AyAzSizFQ.png?1760147656)

### 주소 선택을 필수로 만들기

사용자가 검증된 주소만 제출하도록 하려면 `Mandatory to Select Address in Search Bar(검색 바에서 주소 선택 필수)`를 **ON**으로 토글합니다.

활성화하면 사용자가 폼을 제출하기 전에 제안된 주소 중 하나를 선택해야 합니다. OFF로 두면 수동으로 주소를 입력할 수 있습니다.

![필수 선택 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055758426/original/RxZTM2OVNrnwIzCdd8ip3sbi6f63t_is_w.jpeg?1760147670)

### 폼 미리보기

폼 빌더 오른쪽 상단의 `Preview(미리보기)`를 클릭하여 실제 폼에서 주소 필드가 어떻게 나타나는지 확인합니다.

주소 필드가 이제 **검색 바**로 표시되어 사용자가 타이핑을 시작하고 즉시 주소 제안을 볼 수 있습니다.

![폼 미리보기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055758436/original/zGD2LeIC6RMs_jDGsds9JgI1fDnGEbLNOQ.jpeg?1760147782)

### 실시간 주소 제안 테스트

미리보기에서 주소 필드에 몇 글자(예: 거리명의 일부)를 입력해보세요.

검색 바 아래에 예측 제안이 나타나는 것을 볼 수 있으며, 하나를 선택하면 거리, 도시, 주/도, 국가, 우편번호 필드가 자동으로 채워집니다.

![주소 제안 테스트](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055758434/original/_pK5PJkq8QF9qLUAgJBfLtgDmsd7GQeIrA.jpeg?1760147771)

### 저장 및 발행

자동완성이 예상대로 작동하는지 확인한 후 `Save(저장)`를 클릭하여 변경사항을 적용합니다.

발행하면 폼을 작성하는 모든 방문자가 내장된 자동완성 기능으로 빠르고 정확한 주소 입력을 경험할 수 있습니다.

## 자주 묻는 질문

**Q: 주소 자동완성이 아직 유료 기능인가요?**

A: 아니요. 이제 모든 Hyperclass 계정에서 완전히 무료이자 무제한으로 제공됩니다.

**Q: 사용자가 수동으로 주소를 입력하도록 허용할 수 있나요?**

A: 네. 수동 입력을 선호한다면 자동완성 토글을 비활성화할 수 있습니다.

**Q: 주소 자동완성이 전 세계적으로 작동하나요?**

A: 네. 전 세계 대부분의 국가와 우편번호 형식을 지원합니다.

**Q: 기존 폼에 자동완성이 자동으로 활성화되나요?**

A: 아니요. 각 폼이나 설문을 열어서 수동으로 토글을 활성화해야 합니다.

**Q: 사용자가 제안된 주소를 선택하지 않으면 어떻게 되나요?**

A: "검색 바에서 주소 선택 필수" 토글이 ON이면 사용자가 폼을 제출하려면 제안 중 하나를 선택해야 합니다. OFF라면 자유롭게 타이핑할 수 있습니다.

**Q: 자동완성이 기존 연락처 데이터를 덮어쓰나요?**

A: 아니요. 제출은 평소의 연락처 업데이트 규칙과 매핑 설정을 따릅니다.
