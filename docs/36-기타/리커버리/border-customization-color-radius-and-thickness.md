# 테두리 커스터마이징: 색상, 라운드, 두께

테두리 커스터마이징 기능을 사용하면 레이아웃의 테두리 모양을 완전히 제어할 수 있습니다. 이 기능으로 테두리의 색상, 라운드(둥근 정도), 두께를 조정하여 디자인을 개선하고 사용자 경험을 향상시킬 수 있어요.

## 주요 기능

- **테두리 색상**: 색상 선택기나 hex 코드를 사용하여 테두리에 사용자 정의 색상을 선택하고 적용할 수 있어요.
- **테두리 라운드**: 모서리의 둥근 정도를 조정하여 날카로운 모서리나 곡선 모서리를 만들 수 있어요.
- **테두리 두께**: 테두리의 굵기를 설정하여 미묘하거나 굵은 효과를 연출할 수 있어요.

## 테두리 커스터마이징 사용 방법

### 1단계: 테두리 커스터마이징 설정 접근

- GoHighLevel 이메일 빌더를 열어요.
- 커스터마이징하려는 레이아웃/섹션을 선택해요.
- 왼쪽 패널에서 편집 섹션이 열려요.
- "Border Settings(테두리 설정)" 섹션으로 이동해요.

![테두리 설정 섹션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042281814/original/FeLjqo7iJUcpq1T8a1QeBSXnLze7VjpEVA.png?1740574725)

### 2단계: 테두리 속성 커스터마이징

- **테두리 색상**: 색상 박스를 클릭하여 색상을 선택해요.

![테두리 색상 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042281837/original/PA09tpInfbbCTA4qluDHgUKgvg41kh6PIA.png?1740574738)

- **테두리 라운드**: 슬라이더를 사용하거나 값을 입력하여 모서리를 둥글게 만들어요.
- **테두리 두께**: 슬라이더를 사용하거나 숫자 값을 입력하여 굵기를 조정해요.

![테두리 라운드와 두께 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042281895/original/fYmuvrPvsNFgANFL76G-wKO0Fh4SOTzpSw.png?1740574764)

- **특정 모서리 커스터마이징**: 특정 모서리에 대해서만 별도로 커스터마이징할 수 있어요.

![특정 모서리 커스터마이징](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042281913/original/LO3pd1COb9-Eg6LK2tbFsspisd-yP851rw.png?1740574772)

### 3단계: 미리보기 및 저장

- 편집기에서 변경 사항을 즉시 미리보거나 테스트 이메일을 발송해보세요.

![미리보기 1](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042281928/original/-hyz67B98dZMe-wBEVO_72Lliz7V0DJJjw.png?1740574782)

![미리보기 2](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042281994/original/DsUYpmiWq9uE6oU1897c7amNk7gnS5957g.png?1740574824)

- "Save(저장)"를 클릭하여 변경 사항을 적용해요.

## 자주 묻는 질문

#### 1. 서로 다른 면에 다른 테두리 스타일을 설정할 수 있나요?
네, 올바른 구현에서는 가능해요.

#### 2. 모든 요소(엘리먼트)에서 사용할 수 있나요?
아니요, 레이아웃에서만 사용 가능해요.

#### 3. 테두리 설정이 반응형에 영향을 주나요?
아니요, 테두리 설정은 다양한 화면 크기에 자동으로 적응해요.
