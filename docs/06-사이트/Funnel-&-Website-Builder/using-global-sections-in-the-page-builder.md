# 페이지 빌더에서 글로벌 섹션 사용하기

글로벌 섹션(Global Sections)을 사용하면 페이지 빌더(Page Builder)에서 여러 페이지에 걸쳐 공유 섹션을 재사용할 수 있습니다. 최신 개선 사항으로 글로벌 섹션에 검색 필드가 추가되어 저장된 글로벌 섹션을 이름으로 빠르게 찾을 수 있습니다. 특히 재사용 가능한 섹션이 많은 작업 공간에서 속도와 사용성이 크게 향상됩니다.

## 개요

글로벌 섹션은 여러 페이지에 추가할 수 있고 사이트나 퍼널 전체에서 동기화를 유지할 수 있는 재사용 가능한 섹션입니다. 페이지 빌더 내의 저장된 섹션(Saved Sections) → 글로벌 섹션(Global Sections) 영역에서 관리됩니다.

긴 목록을 스크롤하며 탐색하는 불편함을 줄이고자 글로벌 섹션에 새로운 검색 기능이 추가되었습니다. 이제 이름으로 실시간으로 섹션을 필터링할 수 있습니다.

## 글로벌 섹션이란?

글로벌 섹션은 여러 페이지에서 재사용할 수 있는 저장된 섹션입니다. 글로벌 섹션이 업데이트되면 해당 섹션을 사용하는 모든 곳에 변경 사항이 반영될 수 있어, 헤더, 푸터, 갤러리, 리뷰 구성 요소와 같은 공유 요소에 이상적입니다.

페이지에 섹션을 추가할 때 페이지 빌더에서 접근할 수 있습니다.

## 페이지 빌더에서 글로벌 섹션 접근하기

글로벌 섹션에 접근하는 방법:

- 페이지나 퍼널 단계(Funnel Step)의 페이지 빌더를 엽니다.
- 왼쪽 사이드바에서 요소 추가(Add Elements)를 클릭합니다.
- 저장된 섹션(Saved Sections)을 선택합니다.
- 글로벌 섹션(Global Sections)을 엽니다.

## 글로벌 섹션 검색하기 (신규 기능)

이미 생성한 섹션을 빠르게 찾을 수 있도록 글로벌 섹션 패널 상단에 검색 필드가 추가되었습니다.

### 검색 기능 작동 방식

- 전체 또는 부분 섹션 이름으로 검색
- 타이핑하는 동안 실시간으로 결과 업데이트
- 글로벌 섹션만 검색 (로컬 섹션(Local Sections)은 포함되지 않음)

이를 통해 긴 목록을 스크롤하지 않고도 콘텐츠를 더 쉽게 재사용할 수 있습니다.

## 예시: 이름으로 글로벌 섹션 필터링하기

검색 필드에 텍스트를 입력하면 입력한 텍스트와 일치하는 글로벌 섹션만 표시됩니다. 이는 대규모 작업 공간에서 헤더, 푸터, 갤러리와 같은 공유 구성 요소를 식별하는 데 특히 유용합니다.

![이름으로 필터링](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155064216270/original/1u_mpdlQu1w520s-yQNXKLkoFkUgQIBTEw.jpeg?1770239970)

## 왜 중요한가요?

새로운 글로벌 섹션 검색 기능의 장점:

- 대규모 라이브러리에서 저장된 섹션을 빠르게 찾기
- 섹션을 스크롤하는 시간 단축
- 공유 구성 요소를 더 효율적으로 재사용
- 올바른 글로벌 섹션을 더 빠르게 찾아 일관된 디자인 유지

## 변경되지 않은 사항

이번 업데이트로 변경되지 않은 사항:

- 글로벌 섹션 생성 및 편집 방법
- 페이지 간 글로벌 섹션 동기화 방식
- 섹션에 대한 권한(Permission) 또는 접근
- 로컬 섹션이나 요소 템플릿(Element Templates)

이는 사용성 개선에만 집중한 업데이트입니다.

## 자주 묻는 질문

**Q: 부분 이름으로 검색할 수 있나요?**
네. 검색은 전체 및 부분 섹션 이름을 모두 지원합니다.

**Q: 검색이 자동으로 업데이트되나요?**
네. 타이핑하는 동안 실시간으로 결과가 업데이트됩니다.

**Q: 이 검색에는 로컬 섹션이 포함되나요?**
아니요. 검색은 글로벌 섹션에만 적용됩니다.

**Q: 기존 글로벌 섹션에 영향을 주나요?**
아니요. 기존의 모든 글로벌 섹션은 변경되지 않습니다.
