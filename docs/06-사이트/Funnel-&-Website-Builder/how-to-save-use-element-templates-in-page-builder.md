# 페이지 빌더에서 요소 템플릿 저장 및 사용하는 방법

자주 사용하는 버튼, 폼, 이미지 및 기타 요소(Elements)를 재사용 가능한 요소 템플릿(Element Templates)으로 만들어 클릭 한 번으로 어떤 페이지에든 추가할 수 있습니다. 시간을 절약하고 모든 퍼널이나 웹사이트의 브랜드 일관성을 유지하세요.

---

## 페이지 빌더의 요소 템플릿이란?

요소 템플릿(Element Templates)을 사용하면 버튼, 제목, 폼 같은 페이지 빌더의 개별 요소를 스타일과 함께 캡처해서 로케이션 전체 라이브러리에 저장할 수 있습니다. 매 페이지마다 같은 요소를 다시 만드는 대신, 필요할 때마다 저장된 자료(Saved Assets) 패널에서 템플릿을 삽입하면 됩니다.

---

## 페이지 빌더 요소 템플릿의 주요 장점

- **일관된 브랜딩**: 승인된 동일한 디자인 요소를 재사용하여 모든 퍼널과 웹사이트가 브랜드 일관성을 유지하도록 보장합니다.

- **더 빠른 페이지 제작**: 처음부터 다시 만드는 대신 검증된 디자인 블록을 삽입하여 제작 속도를 높입니다.

- **중앙 집중식 라이브러리**: 저장된 모든 템플릿에 한 곳에서 액세스하며, 로케이션의 모든 사용자가 이용할 수 있습니다.

- **스타일 보존**: 글꼴, 색상, 간격을 유지하여 페이지 간 픽셀 단위의 완벽한 일관성을 제공합니다.

- **간편한 관리**: 업데이트가 필요할 때 템플릿을 쉽게 이름 변경하거나 삭제할 수 있습니다.

## 페이지 빌더에서 요소 템플릿 저장 및 사용 방법

- 하위 계정에서 **사이트(Sites)**를 클릭하세요.

![페이지 빌더에서 요소 템플릿 저장하기](https://jumpshare.com/share/G65n4X1DyWaOnmwmvjQZ+/Screen+Shot+2025-09-29+at+8.22.17+PM.png)

- 새 **퍼널/웹사이트를 만들거나** 기존 **퍼널/웹사이트를 편집**하세요.

![퍼널/웹사이트 편집하기](https://jumpshare.com/share/hhkDdIIQG8twWZ56NAoA+/GIF+Recording+2025-09-29+at+8.28.42+PM.gif)

- 버튼, 텍스트, 이미지 등의 **요소를 추가하거나 편집**하세요.

- 다음 방법 중 하나로 **저장**하세요:

  요소의 외곽선 툴바에서 저장 아이콘을 클릭하거나

- 레이어(Layers) → ⋯ → 요소 저장(Save Element)을 열어 저장하세요.

- 저장 모달에서 템플릿에 명확하고 설명적인 이름을 지정하세요(예: Primary CTA Button).

![요소 템플릿 저장하기](https://jumpshare.com/share/qvjPsWfFGbarIHqlWG8a+/GIF+Recording+2025-10-03+at+12.35.56+PM.gif)

- 요소와 스타일이 이제 요소 템플릿 라이브러리에 저장됩니다.

- 재사용하려면 요소 추가(Add Elements) (+) → 저장된 자료(Saved Assets) → 요소 템플릿(Element Templates)을 클릭하여 어떤 페이지에든 삽입하세요.

![요소 템플릿 사용하기](https://jumpshare.com/share/LGcbAeEbYpeSg2qTDUfr+/GIF+Recording+2025-10-03+at+12.52.04+PM.gif)

## 자주 묻는 질문

**Q: 요소 템플릿이 글로벌이나 유니버설 섹션처럼 동기화되나요?**

아니요. 요소 템플릿은 정적입니다. 한 인스턴스를 편집해도 다른 인스턴스에는 영향을 주지 않습니다.

**Q: 저장된 요소에 누가 접근할 수 있나요?**

같은 하위 계정(로케이션) 내에서 편집 권한이 있는 모든 사람이 접근할 수 있습니다.

**Q: 라이브 페이지의 기존 요소를 템플릿으로 변환할 수 있나요?**

네, 빌더에서 페이지를 열고 요소를 선택한 후 저장을 클릭하면 됩니다.

**Q: 글꼴과 색상이 보존되나요?**

네, 스타일이 템플릿과 함께 저장되어 일관된 재사용이 가능합니다.

**Q: 요소 템플릿을 몇 개까지 만들 수 있나요?**

현재 하드 제한은 없습니다. 쉬운 검색을 위해 이름을 체계적으로 정리하세요.

**Q: 템플릿을 폴더로 정리할 수 있나요?**

아직은 안 됩니다. 라이브러리를 깔끔하게 유지하려면 명확한 명명 규칙을 사용하세요.
