# Hyperclass 페이지 SEO 메타데이터 문제 해결

메타태그 테스트 도구:

[https://metatags.io/](https://metatags.io/)


Facebook, Pinterest, LinkedIn, Twitter에 저장된 링크 미리보기 캐시 지우기:

[https://support.shareaholic.com/hc/en-us/articles/360000263426-How-to-Clear-Link-Preview-Cache-Stored-by-Facebook-Pinterest-LinkedIn-Twitter](https://support.shareaholic.com/hc/en-us/articles/360000263426-How-to-Clear-Link-Preview-Cache-Stored-by-Facebook-Pinterest-LinkedIn-Twitter)


URL 검사 도구를 사용하여 Google에서 SEO 설정 업데이트: [https://support.google.com/webmasters/answer/9012289](https://support.google.com/webmasters/answer/9012289)

이 방법을 사용하려면 먼저 Google에서 도메인을 인증해야 합니다. Google 도메인 인증에는 메타태그 인증 방법을 사용할 수 있어요. Facebook 도메인 인증 관련 가이드를 참고하시면 됩니다: [Facebook 메타태그를 통한 도메인 인증]([facebook-domain-verification-by-meta-tag](facebook-domain-verification-by-meta-tag.md)) - Google 도메인 인증 방법도 이와 유사합니다.
