# 블로그 게시글 미리보기 방법

블로그 게시글이 완성되어 세상에 공개할 준비가 되었나요? 그 전에 데스크톱과 모바일에서 어떻게 보이는지 미리보기를 확인하는 것이 좋습니다.

이 가이드에서는 빌더 내에서 블로그 게시글을 미리보는 방법을 설명해드리겠습니다.

**중요 사항:**

카테고리(Category), 키워드(Keywords), 작성자(Author), 슬러그(Slug)는 블로그가 발행된 후에는 수정할 수 없습니다. 개발팀이 이 문제에 대한 해결책을 준비 중이며 곧 출시될 예정입니다.

## 미리보기 모드에서 표시되는 내용

미리보기 모드에서는 모든 하이퍼링크, 동영상, 작성자 정보를 클릭할 수 있습니다.

- SEO 블로그 게시글의 대표 이미지
- 블로그 게시글 제목  
- 카테고리 (발행 화면에서 추가한 경우)
- 발행 날짜
- 읽기 시간
- 블로그 게시글 내용
- 태그 (발행 화면에서 추가한 경우)
- 작성자 (발행 화면에서 추가한 경우)
- 블로그로 돌아가기 버튼 (미리보기에서는 작동하지 않음)

## 빌더에서 블로그 게시글 미리보기

상단 우측 메뉴에 있는 점 3개 버튼을 클릭하세요 > 메뉴가 열립니다 > "Preview Post(게시글 미리보기)"를 선택하세요

## 미리보기 모드에서 모바일/데스크톱 버전 확인
