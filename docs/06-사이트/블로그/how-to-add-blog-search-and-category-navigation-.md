# 블로그 검색 및 카테고리 내비게이션 추가하기

검색 기능과 카테고리 내비게이션을 추가하면 콘텐츠의 SEO 계층 구조를 개선하고, 사용자가 블로그 콘텐츠를 더 쉽게 탐색하고 이해할 수 있도록 도와줍니다.

## 블로그 검색 및 카테고리 내비게이션 추가 방법

1. `Sites(사이트) → Blogs(블로그)`로 이동합니다
2. 블로그 목록에서 원하는 블로그명을 클릭하고, 블로그 목록 뷰에서 "Edit the Site(사이트 편집)" 옵션을 선택합니다
3. 특정 블로그 사이트 편집기로 이동하면, 요소(Elements) 목록에서 **Category Navigation(카테고리 내비게이션)** 옵션을 찾을 수 있습니다
4. **Category Navigation** 요소를 드래그해서 원하는 위치에 드롭합니다
5. 편집 시 내비게이션에 더 많은 카테고리를 추가할 수 있습니다. 예를 들어, Category 1을 클릭해서 카테고리 이름을 수정하고 블로그 포스트와 연결된 카테고리로 링크를 설정합니다. 다른 카테고리 옵션들도 동일하게 설정합니다
6. 블로그 검색 기능의 경우, 활성화 또는 비활성화할 수 있는 옵션이 제공됩니다
