# 블로그에 목차(Table of Contents)를 추가하는 방법

이제 블로그 제목을 기반으로 **목차**를 자동 생성할 수 있어요. 블로그 구조를 체계화하고 클릭 가능한 링크를 제공하여 독자가 쉽게 탐색할 수 있도록 도와줍니다.

- 클릭 가능한 링크로 독자가 콘텐츠를 쉽게 탐색할 수 있어요.
- 블로그 테마에 맞게 목차를 커스터마이징하고 스타일링할 수 있어요.

**? 어떻게 작동하나요?**

- 블로그(Blog) → '블로그 사이트 편집(Edit blog site)'으로 이동하세요.
- 블로그 포스트 편집기에서 새로운 목차(Table of Contents) 아이콘을 찾아보세요. 세 가지 레이아웃 옵션이 제공됩니다.

![목차 아이콘 및 레이아웃 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155039083312/original/HXFy12Tp_0qaLZbJjpncKnYKp8ftjjhRHw.png?1735631501)

- 선택한 들여쓰기를 기반으로 해당 헤더에 하이퍼링크가 연결된 목차가 생성됩니다.

![생성된 목차 예시](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155039083313/original/vWt9WIbLyYlFtjMeTWSah4LLnOqNrBjkug.png?1735631501)

- 툴바 설정을 사용해서 굵게, 기울임, 밑줄, 취소선, 글꼴 크기 등의 스타일을 조정하세요.

![스타일 설정 툴바](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155039083314/original/ANtFdIZsXwPTpWGbiVzDqA40e4e0Y1-IdA.png?1735631501)

- 완성이에요! 이제 사용할 준비가 되었습니다!

❣️ 왜 이 기능을 좋아하게 될까요:

이 기능은 장문 콘텐츠의 가독성을 높이고, 게시물을 구조화하여 SEO를 개선하며, 독자에게 원활한 읽기 경험을 제공합니다.
