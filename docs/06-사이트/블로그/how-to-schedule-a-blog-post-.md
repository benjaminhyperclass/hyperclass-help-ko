# 블로그 게시물을 예약하는 방법

사용자는 블로그 게시물을 작성하고 특정 시간에 블로그에 표시되도록 예약할 수 있습니다. 시간대에 맞춰 날짜와 시간을 선택할 수 있어요.

## 블로그 게시물 예약 단계

1. **Create New Blog Post(새 블로그 게시물 작성)** 을 클릭하세요

![새 블로그 게시물 작성 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48268833762/original/SkX95wq_6ndkvGLl_MbWD0Ti4Ff-7JkQig.jpeg?1670848592)

2. 블로그 게시물 콘텐츠를 작성합니다

3. **"Continue(계속)"** 을 클릭하세요

![계속 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48268833760/original/04uAqINnleQZKKc6mmdgSIGKgO1N7LTBig.jpeg?1670848592)

4. **Schedule and Publish(예약 및 발행)** 세부사항을 추가하세요 - 로케이션 시간대에 맞춰 날짜와 시간으로 블로그 게시물을 예약합니다

![예약 설정 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48268833761/original/Wz2bU5GACKJFdS_MobsB-R7m2MLv60P6sg.jpeg?1670848592)
