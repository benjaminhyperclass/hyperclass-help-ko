# SEO: AI 추천 - 무한 스크롤

SEO 수정사항 검토 시간을 절약하세요—AI 추천(AI Recommendations)이 이제 무한 스크롤을 지원합니다. 테이블을 스크롤하면서 행이 계속 로드되며, 활성화된 필터나 정렬 기준이 그대로 유지됩니다. 이 업그레이드로 대형 사이트의 문제 분류가 더욱 효율적이 되고, 테이블, 필터 칩, 세부정보 창 간의 데이터 일관성이 향상됩니다.

## AI 추천 - 무한 스크롤이란?

AI 추천(AI Recommendations)은 페이지 내 문제점(제목 태그, 메타 설명, 이미지 alt 텍스트 등)을 찾아내고 원클릭 수정안을 제안하는 기능입니다. 새로운 무한 스크롤 기능은 기존의 페이지별 탐색을 지연 로딩 방식으로 대체하여, 집중력을 흩뜨리지 않고 단일한 연속 보기에서 더 많은 제안사항을 검토할 수 있게 합니다.

## AI 추천 - 무한 스크롤의 주요 장점

실질적인 이점을 이해하면 일상적인 SEO 분류 작업에 이 워크플로우를 즉시 도입할지, 아니면 팀에 점진적으로 도입할지 결정하는 데 도움이 됩니다.

- **손쉬운 탐색**: 다음/이전 버튼 없이 연속 스크롤로 수정사항을 확인할 수 있습니다.
- **빠른 분류**: 더 적은 클릭으로 많은 추천사항을 스캔하고 실행할 수 있습니다.
- **정확한 카운트**: 테이블, 필터 칩, 세부정보 창에서 문제 개수가 더욱 일관되게 표시됩니다.
- **필터 인식 로딩**: 새로 로드되는 행들이 현재의 필터와 정렬 기준을 자동으로 반영합니다.
- **매끄러운 워크플로우**: 세부정보 창을 열고 닫을 때도 컨텍스트를 유지합니다.

## 문제 개수 정확성 개선

정확한 총계로 UI의 다른 부분에서 진행 상황을 추적할 때 혼란을 방지합니다.

- 테이블 헤더, 필터 칩, 세부정보 창에 표시되는 개수가 이제 통합된 단일 소스에서 가져옵니다.
- 필터링, 정렬, 스크롤할 때 총계가 일관되게 유지되어 현재 보기를 더 정확하게 반영합니다.

## 대형 사이트를 위한 성능 팁

대규모 감사에서 수천 개의 잠재적 수정사항을 검토하면서 인터페이스 반응성을 유지하는 방법입니다.

- 콘텐츠를 적극적으로 검토하지 않을 때는 우측 미리보기/세부정보 창을 닫으세요.
- 긴 목록을 작업할 때는 다른 무거운 브라우저 탭을 제한하세요.
- 끊김 현상이 발생하면 필터를 사용해 표시되는 결과 세트를 줄이세요.

## 사용 가능 조건 및 필수 요구사항

팀이 예상치 못한 문제 없이 이 기능에 접근할 수 있도록 자격과 설정 요구사항을 확인하세요.

- 하위 계정에서 **Hyperclass SEO가 활성화**되어 있어야 합니다.
- **사이트 프로젝트가 추가**되어 있고 추천사항을 표시할 수 있는 감사가 available해야 합니다.
- **호환성**: AI 추천은 **워드프레스 사이트**나 **외부 웹사이트**에서는 사용할 수 없습니다.

## AI 추천 - 무한 스크롤 사용 방법

다음 단계를 따르면 올바른 위치에서 최대 효율을 위해 필터와 정렬을 적용하여 추천사항을 검토할 수 있습니다.

- 하위 계정에서 사이트(Sites) → SEO → 사이트 프로젝트(Site projects)로 이동하고 원하는 **사이트 프로젝트**를 선택하세요.

![사이트 프로젝트 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055733741/original/F0iCFXopwmgyDrwTDCnedmjMx-TO-vj1AA.png?1760108260)

- AI 추천(AI Recommendations)을 여세요. 스크롤을 시작하면 행들이 계속 로드됩니다. 필터와 정렬을 사용해 작업 대기열에 집중하세요.

![AI 추천 무한 스크롤](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055734702/original/l4vNn2Dv5k1GP6f5EntVUDU4eCrrOOsOcA.gif?1760108829)

- 아무 행이나 클릭하면 **문제 세부정보(Issue Details)**가 열립니다. 제안된 수정사항을 검토하고 **적용(Apply)** 또는 **무시(Dismiss)**를 선택하세요.

## 자주 묻는 질문

**Q: 무한 스크롤을 사용하려면 별도로 활성화해야 하나요?**
아니오. 무한 스크롤은 AI 추천 경험의 일부로, 하위 계정에 SEO가 활성화되면 자동으로 사용할 수 있습니다.

**Q: AI 추천에서 크레딧은 어떻게 작동하나요?**
각 수정 **제안**마다 **1크레딧**이 소모됩니다(월 500개까지 사용 가능). 기존 행을 보거나 스크롤하는 것은 추가 크레딧을 소모하지 않습니다.

**Q: 특정 페이지 번호로 바로 이동할 수 있나요?**
무한 스크롤에서는 페이지 번호를 사용하지 않습니다. 특정 URL이나 문제 유형을 찾으려면 필터나 검색창을 사용하세요.

**Q: 무한 스크롤이 매우 큰 감사에서 성능에 영향을 주나요?**
행들이 작은 배치 단위로 로드되어 반응성을 유지하는 데 도움이 됩니다. 느려짐이 발생하면 필터로 표시되는 목록을 줄이고 필요하지 않을 때는 세부정보 창을 닫으세요.

**Q: 워드프레스 사이트나 외부 웹사이트에서도 사용할 수 있나요?**
아니오. AI 추천은 워드프레스 사이트나 외부 웹사이트에서는 사용할 수 없습니다.
