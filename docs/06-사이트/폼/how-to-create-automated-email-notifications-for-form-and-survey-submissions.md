# 폼(Form)과 설문(Survey) 제출 시 자동 이메일 알림 설정하기

이 기능을 활성화하면 폼을 제출한 사람에게 자동 응답 이메일이 전송되며, 내부 팀원들에게도 알림을 보낼 수 있습니다.

이 가이드에서 다룰 내용:

- [이메일 알림 활성화하는 방법](#이메일-알림-활성화하는-방법)

참고사항:

이 옵션과 설정들은 설문(Survey)에서도 동일하게 사용할 수 있습니다.

## 이메일 알림 활성화하는 방법

하위 계정을 선택한 후 `Sites(사이트)` > `Forms(폼)` > `Form Builder(폼 빌더)`로 이동해서 새 폼을 추가하거나, 자동 응답을 설정하고 싶은 기존 폼을 목록에서 선택하세요.

1. 좌측 상단에 알림을 위한 "벨" 아이콘이 보입니다. 이 아이콘을 클릭하세요.

![알림 설정 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155020555297/original/KACnIMc4vTpL9bfpfoX_fkCehWQKcs6icg.png?1707925669)

2. 알림 아이콘을 클릭하면 두 가지 옵션이 나타납니다: 첫 번째는 누군가 폼을 제출했을 때 관리자에게 이메일을 보내는 것이고, 두 번째는 ***자동 응답*** 기능으로 폼을 제출한 사람에게 자동 메시지를 보내는 것입니다. 버튼을 토글하여 활성화하세요.

![자동 응답 설정 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155020556370/original/ceE2uO_z4WMbqmMxGAaFGoXsmLZWlDjTiA.png?1707926154)

3. 자동으로 전송될 이메일의 "제목"을 입력하세요. "회신 이메일"은 수신자가 이 자동 이메일에 답장할 경우 받게 될 이메일 주소입니다. 세 번째 필드인 "발신자 이름"은 공란으로 두면 로케이션 이름이 기본값으로 설정되며, 원하는 이름으로 커스터마이징할 수도 있습니다. 이 이름이 발송되는 이메일에 표시됩니다. 설정이 완료되면 "저장"을 클릭한 후, 우측 상단의 "저장" 버튼을 다시 클릭하여 폼의 변경사항을 저장하세요.

![이메일 내용 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155020557134/original/fBT19UzAYz-_WhsO3XvBOJmlV0d4HvjiXQ.png?1707926428)

**권장사항:**

새로운 설정과 폼이 저장되면 우측 상단의 "미리보기(Preview)" 버튼을 클릭해서 직접 폼을 작성해보며 기능이 제대로 작동하는지 테스트해보세요.
