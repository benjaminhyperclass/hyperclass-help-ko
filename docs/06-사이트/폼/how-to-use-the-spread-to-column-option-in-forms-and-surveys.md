# 폼과 설문에서 컬럼 분산 옵션 사용 방법

폼과 설문의 라디오 버튼, 라디오 이미지, 체크박스에서 사용할 수 있는 "컬럼 분산(Spread to Column)" 기능을 출시하게 되어 기쁩니다. 이 기능을 통해 라디오 버튼과 체크박스 옵션을 여러 컬럼으로 구성할 수 있어, 폼의 스타일링과 구성을 한층 더 향상시킬 수 있습니다.

## **사용 방법**

- 폼(Form) 또는 설문(Survey) 빌더를 열어주세요.
- 라디오 버튼, 체크박스, 또는 라디오 이미지 필드를 클릭하세요.
- 우측 설정 패널에서 "컬럼 분산(Spread to Column)" 옵션을 활성화하세요.
- 원하는 컬럼 수를 지정하세요.

## **주요 기능 및 장점**

**향상된 스타일링:**
라디오 옵션과 체크박스를 컬럼으로 분산시켜 시각적으로 매력적이고 잘 정리된 폼을 만들 수 있습니다.

## **더 나은 사용자 경험**

폼의 명확성과 사용성을 향상시켜 응답자가 더 쉽게 탐색하고 옵션을 선택할 수 있도록 도와줍니다.

## **참고사항**

- 단일 컬럼(Single Column)에서는 최대 5개 컬럼까지 설정 가능합니다.
- 두 개 컬럼(Two Column)에서는 최대 2개 컬럼까지 설정 가능합니다.
- 단일 라인 컬럼(Single Line Column)에서는 최대 5개 컬럼까지 설정 가능합니다.
- 이 기능은 기존에 만들어진 폼에는 영향을 주지 않습니다.
