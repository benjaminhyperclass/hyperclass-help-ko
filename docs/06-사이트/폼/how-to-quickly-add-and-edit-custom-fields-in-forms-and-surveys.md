# 폼(Form)과 설문(Survey)에서 커스텀 필드를 빠르게 추가하고 편집하는 방법

**목차**

- [새로운 기능](#새로운-기능)
- [작동 방식](#작동-방식)
- [이 기능을 만든 이유](#이-기능을-만든-이유)
- [오래되거나 불필요한 커스텀 필드 제거하기](#오래되거나-불필요한-커스텀-필드-제거하기)

폼(Form)/설문(Survey) 빌더에 **빠른 필드 추가 & 커스텀 필드 편집** 기능을 새롭게 추가했습니다. 이 개선사항으로 커스텀 필드를 더 쉽고 효율적으로 추가하고 관리할 수 있게 되었습니다.

**참고사항:** 커스텀 필드 이름에 "score"라는 단어가 포함되어 있으면, 시스템이 자동으로 점수를 매기는 숫자 요소로 인식합니다. 이는 폼, 설문, 퀴즈 전반에 걸쳐 적용되며, 현재 점수 요소를 식별하는 방식과 연결된 하드코딩된 동작입니다.

## 새로운 기능

### 빠른 커스텀 필드 추가
이전에는 커스텀 필드를 추가하려면 '커스텀 필드 추가' 모달을 통해 여러 단계를 거쳐야 했습니다. 이제 빠른 추가 기능으로 드래그 앤 드롭만으로 폼에 커스텀 필드를 쉽게 만들 수 있습니다.

### 드래그할 수 있는 커스텀 필드 종류

- 라디오 선택
- 단일 및 다중 드롭다운
- 체크박스
- 파일 업로드
- 한 줄 텍스트
- 여러 줄 텍스트
- 텍스트박스 목록
- 날짜 선택기
- 숫자
- 금액

### 사용자 정의 옵션
드롭다운, 라디오 버튼, 체크박스의 옵션을 커스터마이징할 수 있고, 파일 업로드 필드의 파일 입력 옵션도 설정할 수 있습니다. 커스텀 필드명과 고유 키도 설정 가능합니다.

### 간소화된 작업 흐름
폼/설문을 저장하면 커스텀 필드도 함께 저장되어 모든 곳에서 사용할 수 있게 됩니다.

### 편집 기능
새로 만든 커스텀 필드나 기존 커스텀 필드의 속성을 편집할 수 있습니다. 옵션 커스터마이징(편집, 순서 변경, 삭제)과 파일 입력 옵션 등을 수정할 수 있습니다.

폼이나 설문에 추가된 커스텀 필드를 편집하려면:

- 폼 또는 설문 빌더에서 커스텀 필드를 클릭하세요.
- 오른쪽에 네비게이션 바가 나타납니다.
- 일반 설정(General Settings)과 고급 설정(Advanced Settings) 드롭다운의 옵션을 사용해 필요한 변경을 하세요.

### 필드 잠금
폼/설문을 저장한 후에는 커스텀 필드명과 고유 키가 잠깁니다. 커스텀 필드 섹션에서만 편집할 수 있습니다.

## 작동 방식

### **새 폼 만들기 또는 기존 폼 열기:**

**사이트(Sites) > 폼(Forms) > 빌더(Builder)**로 이동하세요.

![폼 빌더 페이지](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155066200103/original/JtQyI-S7B8Zsb1UMb-LbQvGvy9ngxji7yQ.png?1772636976)

### **드래그 앤 드롭:**
원하는 필드를 폼으로 드래그해서 놓으세요.

### 커스터마이징
드롭다운, 라디오 버튼, 체크박스의 옵션을 커스터마이징하거나, 파일 업로드 필드의 파일 입력 옵션을 설정하세요. 필요하다면 커스텀 필드명과 고유 키를 설정하세요.

### 저장
폼/설문을 저장하여 커스텀 필드를 저장하면 모든 곳에서 사용할 수 있게 됩니다.

### 편집
커스텀 필드 섹션에서 새로 만든 커스텀 필드나 기존 커스텀 필드의 속성을 편집하고 폼/설문을 저장하세요.

## 이 기능을 만든 이유

빠른 커스텀 필드 추가 기능은 시간을 절약하고 커스텀 필드 관리의 복잡성을 줄이도록 설계되었습니다. 사용자가 즉석에서 필드를 추가할 수 있게 함으로써 전반적인 사용자 경험을 개선하고 작업 흐름의 효율성을 높이는 것이 목표입니다.

## 오래되거나 불필요한 커스텀 필드 제거하기

폼에서 삭제했는데도 폼 빌더에서 오래된 커스텀 필드가 계속 보인다면, CRM의 커스텀 필드 목록에 여전히 존재할 수 있습니다.

완전히 제거하려면:

- 설정(Settings) > 커스텀 필드(Custom Fields)로 이동하세요.
- 목록에서 오래된 필드를 찾으세요.
- 휴지통 아이콘을 클릭해서 영구적으로 삭제하세요.

**주의:** 이 섹션에서 커스텀 필드를 삭제하면 해당 필드가 사용된 모든 폼, 설문, 연락처에서 제거됩니다.

커스텀 필드 이름에 "score"라는 단어가 포함되어 있으면, 시스템이 자동으로 점수를 매기는 숫자 요소로 인식합니다. 이는 다음에 적용됩니다:

- 폼
- 설문
- 퀴즈

이 동작은 하드코딩되어 있으며 현재 점수 요소를 식별하는 방식과 연결되어 있습니다.
