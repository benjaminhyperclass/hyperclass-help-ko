# 폼(Forms)과 설문(Surveys)의 이메일 검증 기능

폼과 설문 모듈에 이메일 검증 기능이 출시되었습니다. 이 새로운 기능으로 폼이나 설문 참여자가 양식 안에서 직접 이메일 주소를 검증할 수 있어, 수집되는 데이터의 무결성을 높이고 더욱 매끄러운 사용자 경험을 제공합니다.

## 이메일 검증 버튼

![이메일 검증 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155027788086/original/dXp-CiWScwGbgG-pSkv_exCpXDB_S2vF9Q.png?1718692402)

폼과 설문의 이메일 입력 필드 옆에 "이메일 인증(Verify Email)" 버튼이 표시됩니다. 이 버튼은 사용자가 올바른 형식의 이메일 주소를 입력한 후에만 보입니다. 이 버튼을 클릭하면 입력한 이메일 주소로 검증 코드가 전송되며, 검증 과정이 시작됩니다.

## 검증 코드

![검증 코드](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155027788179/original/G5Et7r-kk6CJ-mHOOLDbdMgyP3B5sVZ1IQ.png?1718692530)

"이메일 인증(Verify Email)" 버튼을 클릭하면 사용자는 이메일로 고유한 검증 코드를 받게 됩니다. 이메일 주소를 확인하려면 이 코드를 폼이나 설문에 다시 입력해야 합니다. 이 단계를 통해 제공된 이메일 주소가 유효하고 사용자가 실제로 접근할 수 있는 주소임을 확인하여 데이터 정확도를 높입니다.

## 작동 방식

- **이메일 입력**: 사용자가 폼이나 설문의 지정된 필드에 이메일 주소를 입력합니다.
- **검증 알림**: 유효한 이메일을 입력하면 이메일 입력 필드 옆에 "이메일 인증(Verify Email)" 버튼이 나타납니다.
- **검증 코드 발송**: 이 버튼을 클릭하면 사용자의 이메일 주소로 고유한 검증 코드가 전송됩니다.
- **검증 코드 입력**: 사용자는 이메일 주소를 인증하기 위해 이 코드를 폼이나 설문에 다시 입력해야 합니다.
- **검증 확인**: 코드가 올바르게 입력되면 이메일 주소가 인증되고 제출 데이터에 그에 따라 표시됩니다.

## 자주 묻는 질문

#### Q1: 폼과 설문에서 이메일 검증 기능을 사용해야 하는 이유는 무엇인가요?

**A**: 이메일 검증 기능을 구현하면 수집되는 이메일 주소가 유효함을 보장하여 가짜이거나 잘못된 제출 가능성을 줄입니다. 이를 통해 리드 품질이 향상되고 마케팅 캠페인을 위한 더 신뢰할 수 있는 데이터를 확보할 수 있습니다.

#### Q2: 이메일 검증 과정은 어떻게 데이터 정확도를 개선하나요?

**A**: 사용자가 받은 편지함으로 전송된 고유 코드를 통해 이메일 주소를 검증하도록 요구함으로써, 제공된 이메일 주소가 실제이며 접근 가능함을 확인할 수 있습니다. 이를 통해 부정확하거나 가짜 이메일 주소를 수집할 가능성을 줄여 더 정확하고 효과적인 마케팅 활동이 가능합니다.

#### Q3: 사용자에게 발송되는 검증 이메일을 커스텀할 수 있나요?

**A**: 현재 검증 이메일은 일관성과 신뢰성을 보장하기 위해 표준화되어 있습니다. 하지만 향후 업데이트에서 커스텀 옵션이 포함될 수 있습니다. 이 기능에 대한 업데이트를 계속 확인해 주세요.

#### Q4: 폼과 설문에서 이메일 검증 기능을 어떻게 활성화하나요?

**A**: 이메일 검증을 활성화하려면 모듈 내의 폼이나 설문 설정으로 이동하여 이메일 필드에서 이메일 검증 옵션을 토글하세요. 그러면 "이메일 인증(Verify Email)" 버튼이 폼과 설문의 이메일 입력 필드 옆에 자동으로 나타납니다.

#### Q5: 사용자가 이메일 주소 검증에 실패하면 어떻게 되나요?

**A**: 사용자가 이메일 검증 과정을 완료하지 않으면 제출이 수락되지 않을 수 있습니다. 이를 통해 검증된 이메일 주소가 있는 제출만 수집되어 데이터의 무결성을 유지합니다.

#### Q6: 어떤 이메일 주소가 검증되었는지 어떻게 추적할 수 있나요?

**A**: 시스템은 폼과 설문 제출에 "검증된 이메일(Verified Email)" 필드를 포함합니다. 이메일이 성공적으로 검증되면 이 필드에서 "예(Yes)"로 표시되어 검증된 이메일 주소를 쉽게 식별할 수 있습니다.
