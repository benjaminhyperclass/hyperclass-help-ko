# SEO 구독 해지 플로우 및 프로젝트 상태

SEO 구독이 해지되면 시스템이 자동으로 오프보딩 플로우를 시작합니다. 이 과정에서 기존 최적화 작업은 유지하면서 활성 자동화 기능은 일시정지됩니다.

---

## SEO 구독 해지 플로우 개요

SEO 구독 해지 플로우는 SEO 구독이 해지된 후 SEO 프로젝트에 어떤 일이 일어나는지 설명합니다. 시스템은 기존 프로젝트 데이터와 적용된 SEO Pixel 수정 사항은 보존하면서, 크롤링이나 새로운 추천 등의 활성 자동화 기능은 일시정지합니다.

구독 해지 후, SEO Pixel을 사용하는 프로젝트는 동결 상태로 전환됩니다. 이 상태에서는 SEO Pixel 스크립트가 웹사이트에 설치되어 있는 한 이전에 적용된 수정 사항이 웹사이트에서 계속 작동하지만, 활성 자동화 기능은 중단됩니다. 기존 프로젝트 설정과 적용된 수정 사항은 계정에 그대로 유지되어 나중에 재활성화하기 쉽습니다.

---

## SEO 구독 해지 플로우의 주요 장점

- **수정 사항 보존**: SEO Pixel을 통해 이미 적용된 SEO 추천 사항과 수정 사항은 구독 해지 후에도 웹사이트에서 계속 작동하므로, 이전 구현 진행 상황이 손실되지 않습니다.

- **자동 오프보딩**: 프로젝트가 자동으로 동결 상태로 전환되며, 별도의 수동 오프보딩이 필요하지 않습니다.

- **프로젝트 유지**: 기존 프로젝트 설정과 적용된 수정 사항이 계정과 연결된 상태로 유지되어 향후 재활성화가 더욱 쉬워집니다.

---

## SEO 구독을 해지하는 방법

- HighLevel의 하위 계정에서 `Sites(사이트) > SEO`로 이동합니다.

- Site Projects(사이트 프로젝트) 탭에서 SEO 페이지 우측 상단의 점 3개 메뉴를 클릭하고 Cancel Subscription(구독 취소)을 선택합니다.

## "동결" 상태

SEO Pixel을 사용하는 활성 프로젝트가 있다면, 해당 프로젝트는 구독 해지 시 자동으로 동결 상태로 전환됩니다. 이 과정은 완전히 백그라운드에서 진행되며, 수동 조치가 필요하지 않습니다.

#### 동결된 프로젝트의 주요 특징:

- **수정 사항 보존**: 픽셀 스크립트를 통해 이전에 적용된 모든 SEO 추천 사항과 수정 사항은 웹사이트에서 계속 작동합니다. 구독이 종료되어도 구현 진행 상황이 손실되지 않습니다.

- **자동화 일시정지**: 시스템이 사이트 크롤링을 중단하고 새로운 SEO 감사 추천 사항 생성을 중단합니다.

- **데이터 유지**: 기존 프로젝트 설정과 적용된 수정 사항이 계정에 그대로 유지됩니다.

---

## 라이브 최적화 관리

구독 해지 후에도 SEO 변경 사항이 사이트에 반영되는 방식을 완전히 제어할 수 있습니다:

- **변경 사항 유지**: 웹사이트에 SEO Pixel 스크립트를 그대로 두세요. 스크립트가 유지되는 한 적용된 최적화가 방문자에게 계속 제공됩니다.

- **변경 사항 제거**: 사이트에서 최적화가 더 이상 나타나지 않게 하려면, 웹사이트 코드에서 SEO Pixel 스크립트를 수동으로 제거하세요. 이렇게 하면 즉시 사이트가 원래 상태로 복원됩니다.

---

## 재구독 및 재활성화

재구독을 선택하면 활성 상태로 다시 전환하는 것이 매우 간단합니다:

- 유효한 플랜으로 재구독합니다.

- 대시보드에서 프로젝트를 다시 연결합니다.

- 프로젝트가 해제 상태로 전환되어 재크롤링, 새 추천 생성, 새로운 SEO 수정 사항 적용 기능이 즉시 복원됩니다.

## 프로젝트 상태 비교

#### 활성 구독:

- 기존 Pixel 수정 사항: 라이브 및 편집 가능

- 사이트 크롤링: 활성화

- 새로운 추천: 활성화

- 시스템 작업: 완전 자동화

#### 동결됨(구독 해지):

- 기존 Pixel 수정 사항: 라이브(읽기 전용)

- 사이트 크롤링: 일시정지

- 새로운 추천: 일시정지

- 시스템 작업: 자동 중단
