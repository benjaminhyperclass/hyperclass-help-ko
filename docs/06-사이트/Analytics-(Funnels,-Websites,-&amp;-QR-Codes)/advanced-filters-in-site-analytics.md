# 사이트 분석의 고급 필터

사이트 분석의 고급 필터는 분석 데이터를 세분화하여 가장 중요한 트래픽 세그먼트에 집중할 수 있도록 도와줍니다. 도시, 주/도, 국가, 페이지 제목, 브라우저, 기기 유형, 소스, 매체 등의 필터를 사용해 결과를 세밀하게 조정할 수 있습니다. 이를 통해 트렌드를 발견하고, 고객 세그먼트를 비교하며, 분석 대시보드에서 더 정확한 의사결정을 내릴 수 있습니다.

# 사이트 분석의 고급 필터란?

사이트 분석의 고급 필터는 퍼널, 웹사이트, 웨비나의 분석 결과에 하나 이상의 조건을 적용하여 세밀하게 조정하는 기능입니다. 특정 트래픽 세그먼트를 분리하고, 다양한 고객층의 행동을 이해하며, 데이터를 내보내지 않고도 더 깊은 분석을 통해 성과를 검토할 수 있게 해줍니다.

고급 필터를 사용하면 필터 차원을 선택하고 AND, OR 같은 논리 조건을 결합하여 타겟팅된 분석 뷰를 구축할 수 있습니다. 또한 더 정확한 세분화가 필요할 때 관련 조건들을 그룹화하는 중첩 필터도 만들 수 있습니다.

사이트 분석 대시보드에 대한 자세한 정보는 Hyperclass에서 사이트 및 퍼널 분석 사용하기를 참고하세요.

---

## 사이트 분석 고급 필터의 주요 장점

고급 필터를 통해 성과 데이터 분석을 더 세밀하게 제어할 수 있어, 단순한 전체 합계를 넘어 비즈니스에 더 실용적인 인사이트를 발견할 수 있습니다.

- **데이터를 더 정확하게 세분화**: 지역, 트래픽 소스, 페이지 제목, 브라우저 등의 속성으로 분석을 세분화할 수 있습니다.

- **여러 조건을 동시에 분석**: 필터를 결합하여 매우 구체적인 고객 그룹과 트래픽 패턴을 검토할 수 있습니다.

- **유연한 필터 논리 사용**: AND 및 OR 조건을 적용하여 더 정확한 분석 뷰를 구축할 수 있습니다.

- **더 깊은 필터 그룹핑 생성**: 관련 조건들을 함께 구성해야 할 때 중첩 필터를 추가할 수 있습니다.

- **필터를 쉽게 재설정하거나 조정**: 개별 필터를 제거하거나 처음부터 다시 시작하고 싶을 때 모든 필터를 지울 수 있습니다.

## 지원되는 자산 유형

다음 자산 유형에서 고급 필터를 사용할 수 있습니다:

- 퍼널
- 웹사이트
- 웨비나

---

## 고급 필터 접근 방법

메인 사이트 분석 대시보드에서 시작하여 먼저 지원되는 자산 유형을 선택한 후 필터 빌더를 여는 방식으로 접근합니다. 고급 필터 접근 방법:

- 좌측 네비게이션 메뉴에서 **Sites(사이트)**를 클릭합니다.

- 상단 네비게이션 바에서 **Analytics(분석)**를 클릭합니다.

- 지원되는 자산 유형(퍼널, 웹사이트 또는 웨비나)을 선택합니다.

- 분석 대시보드 우상단에서 **Advanced Filter(고급 필터)**를 클릭합니다.

## 사이트 분석에서 고급 필터 사용하기

필터를 올바르게 생성하면 검토하고자 하는 정확한 고객 세그먼트로 분석을 좁힐 수 있습니다. 고급 필터를 생성하고 적용하려면 아래 단계를 따르세요.

- **+ Add Filter(필터 추가)**를 클릭합니다.

![필터 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155068638774/original/T29dMjCisB-1upKS8br3LzhfGu3HuDZVwg.png?1775584948)

- 사용하고자 하는 필터 필드를 선택하고, 조건(Is 또는 Is not)을 선택한 후, 포함하거나 제외할 하나 이상의 값을 선택합니다.

![필터 조건 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155068638966/original/beFDOg7mwiqzNFrhViGaShqbsQ5J0oMuvQ.png?1775585107)

- **Add Filter(필터 추가)**를 클릭하여 다른 필터를 추가합니다.

![추가 필터](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155068639132/original/PaWSgbwn83BMyAbkKYQ9AednVy7BDU5OYA.png?1775585195)

- **Add Nested Filter(중첩 필터 추가)**를 클릭하여 더 큰 필터 논리 안에 그룹화된 조건 세트를 생성합니다.

![중첩 필터 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155068639193/original/qsMOLwhhJ-R1XN_FNLCRzEOJEJgPAN6eNA.png?1775585250)

- **AND** 또는 **OR**를 선택하여 필터들이 함께 작동하는 방식을 정의합니다.

![논리 조건 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155068639229/original/SCXbB0-XdF3LH_kJ-4BNiQzmd49-ETUamA.png?1775585297)

- **Apply(적용)**를 클릭하여 분석 대시보드를 업데이트합니다.

![필터 적용](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155068639241/original/ieXe_J0h5VH9Nq-fyqC72UBA9y2pWkV-GQ.png?1775585329)

## 자주 묻는 질문

**Q: 고급 필터가 적용될 때 기본 필터도 여전히 적용되나요?**

네. 선택한 자산이나 기간 같은 기본 필터는 여전히 보고 있는 데이터 세트를 형성합니다. 고급 필터는 이러한 결과를 더욱 세분화합니다.

**Q: 필터 추가와 중첩 필터 추가의 차이점은 무엇인가요?**

필터 추가는 현재 필터 설정 레벨에 다른 조건을 추가합니다. 중첩 필터 추가는 그룹화된 필터 안에 더 많은 조건을 추가하여 나머지 논리와 결합하기 전에 해당 조건들이 함께 작동하는 방식을 제어할 수 있게 해줍니다.

**Q: 페이지를 새로고침하거나 사이트 분석을 떠나면 고급 필터는 어떻게 되나요?**

고급 필터는 페이지 새로고침 후나 분석 페이지를 떠난 후에는 유지되지 않습니다. 나중에 다시 돌아오면 고급 필터를 다시 만들어야 합니다.
