# 커스텀 메타 태그

# 커스텀 메타 태그란?

커스텀 메타 태그를 사용하면 검색 엔진, 브라우저, 소셜 미디어 플랫폼에 페이지가 어떻게 설명될지 직접 정의할 수 있습니다. 이러한 숨겨진 HTML 코드 조각은 사이트의 가시성과 검색 엔진 순위를 제어하는 데 도움이 됩니다.

참고로, 구글에서 지원하는 메타 태그 목록은 다음 링크에서 확인하실 수 있습니다: [https://developers.google.com/search/docs/crawling-indexing/special-tags](https://developers.google.com/search/docs/crawling-indexing/special-tags)

## **커스텀 메타 태그의 주요 이점**

커스텀 메타 태그는 웹사이트의 SEO 성능과 웹상에서의 표시 방식을 제어하는 데 필수적입니다.

- 검색 엔진에서 페이지가 표시되는 방식을 개선합니다.
- 페이지가 공유될 때 제목, 설명, 이미지를 제어합니다.
- 검색 엔진이 초안이나 비공개 페이지를 색인하지 않도록 방지합니다.
- 캐싱이나 리다이렉트 같은 브라우저 동작을 사용자 정의합니다.
- 코딩 없이도 SEO를 더 잘 제어할 수 있습니다.

## **커스텀 메타 태그 추가 방법**

페이지 빌더 상단의 SEO Meta Data 아이콘을 클릭하여 퍼널과 웹사이트 페이지에 커스텀 메타 태그를 추가할 수 있습니다.

- `Sites(사이트) → Funnels(퍼널)` 또는 `Sites(사이트) → Websites(웹사이트)`로 이동합니다.
- 편집하려는 퍼널 또는 웹사이트를 선택합니다.
- 특정 페이지나 단계를 클릭합니다.
- SEO Meta Data 탭으로 이동합니다.
- 아래로 스크롤하여 Links and Tags 드롭다운을 찾습니다.
- Custom Meta Tags 옆의 `+ Add`를 클릭합니다.
- 태그 이름과 내용을 추가합니다.
- `Add`를 클릭합니다.
- `Update SEO Meta`를 클릭하여 변경 사항을 저장합니다.

**예시**: 다음 메타 태그를 사용하여 검색 엔진이 특정 페이지를 색인하지 않도록 지시할 수 있습니다. 새로운 커스텀 메타 태그의 이름을 "robots"로, 내용을 "noindex"로 설정하면 됩니다.

## 자주 묻는 질문

**Q: 구글이 업데이트된 메타 태그를 인식하는 데 얼마나 걸리나요?**
A: 구글의 크롤링 일정에 따라 몇 시간에서 며칠까지 걸릴 수 있습니다.

**Q: 한 페이지에 여러 개의 커스텀 메타 태그를 추가할 수 있나요?**
A: 네, 페이지 최적화를 위해 필요한 만큼 추가하실 수 있습니다.

**Q: 메타 태그가 누락되면 SEO에 악영향이 있나요?**
A: description이나 Open Graph 데이터 같은 주요 메타 태그가 누락되면 검색 엔진 순위와 소셜 미디어 미리보기에 부정적인 영향을 줄 수 있습니다.

**Q: "noindex" 태그를 추가하면 어떻게 되나요?**
A: 구글 같은 검색 엔진이 해당 페이지를 완전히 색인에서 제외합니다.
