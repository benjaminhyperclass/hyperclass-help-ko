# 버튼 모서리를 둥글게 만드는 방법

퍼널 빌더(Funnel Builder)에서 버튼의 모서리를 둥글게 만드는 방법:

1. 버튼 요소(Button Element)를 선택하세요

2. 고급 설정(Advanced Settings)으로 이동하세요

3. 모든 면에 테두리를 추가하고, 필요시 테두리 색상을 조정하세요

4. 테두리 반지름(Border Radius)을 100px로 변경하세요
