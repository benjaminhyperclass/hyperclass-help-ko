# ClickFunnels에서 퍼널을 가져오는 방법

ClickFunnels에서 Hyperclass로 퍼널을 가져오는 기능은 ClickFunnels에서 Hyperclass로 플랫폼을 변경하려는 개인이나 기업에게 유용합니다. 기존 퍼널과 페이지를 처음부터 다시 만들지 않고 이전할 수 있어 상당한 시간과 노력을 절약할 수 있습니다. 특히 많은 퍼널이나 페이지를 이전해야 하는 경우 더욱 유용합니다.

참고 사항:

- 이 가져오기 기능은 ClickFunnels 1.0과 ClickFunnels 2.0 퍼널 모두에서 작동합니다.
- 공유 가능한 클릭 퍼널 링크가 아닌 발행 가능한(라이브) 링크를 사용해야 합니다. 원활한 경험을 위해 ClickFunnels에서 한 번에 하나의 퍼널 단계(Funnel Step)씩 가져오는 것이 중요합니다.

## 클릭 퍼널 페이지를 Hyperclass 퍼널 빌더로 가져오는 방법

- [1단계:](#1단계-템플릿-목록-뷰에-접근하기) Sites(사이트) 클릭
- [2단계:](#2단계-퍼널-클릭하기) Funnels(퍼널) 클릭
- [3단계:](#3단계-새-퍼널-만들기) "New Funnel(새 퍼널)" 클릭
- [4단계:](#4단계-새-단계-추가하기) "Add a new step(새 단계 추가)" 클릭

**중요 참고사항:** ClickFunnels 멤버십 영역은 퍼널 공유 URL을 사용해서 가져올 수 없습니다. Hyperclass 가져오기 도구는 표준 퍼널 페이지(옵트인, 세일즈, 감사 페이지 등)만 전송합니다. ClickFunnels 퍼널에 멤버십 영역이 포함되어 있다면, 좌측 네비게이션 바에 있는 Memberships(멤버십) 도구를 사용해서 수동으로 다시 만들어야 합니다.
