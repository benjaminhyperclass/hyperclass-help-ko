# WordPress: 도메인 커넥트 연동

도메인 관리를 더욱 쉽게 만들어주는 도메인 커넥트 업데이트를 발표하게 되어 기쁩니다! 새 버전에서는 시간을 절약하고 복잡한 단계를 건너뛸 수 있습니다.

## 새로운 기능

**주요 도메인 제공업체 지원**: 이제 도메인 커넥트가 GoDaddy, Cloudflare, IONOS와 연동됩니다. 이 업체들 중 어디서든 도메인을 구매하셨다면 새 기능을 쉽게 활용할 수 있습니다.

**빠른 도메인 설정**: GoDaddy, Cloudflare, IONOS에서 구매한 도메인을 연결하는 것이 매우 간단해졌습니다. 몇 번의 클릭만으로 완료되며, 기술적 지식이 필요하지 않습니다!

**수동 설정 불필요**: 더 이상 DNS 설정을 직접 조정할 필요가 없습니다. 도메인 커넥트가 모든 것을 자동으로 설정해주므로 복잡한 세부 사항을 걱정하지 않아도 됩니다.

**시간 절약**: 도메인 커넥트가 모든 작업을 처리하여 도메인 설정을 빠르고 쉽게 만들어줍니다. 덕분에 설정 작업 대신 웹사이트 관리에 집중할 수 있습니다.

## 사용 방법

1. **대시보드로 이동**: Hyperclass에 로그인하고 WordPress 호스팅 섹션으로 이동합니다.
![대시보드 WordPress 호스팅 섹션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036356159/original/CvLhPrzWicA_UQd6eW5rpyd1yjDLUW5tag.png?1731337464)

2. **사이트 선택**: 대시보드에서 새 도메인을 연결할 사이트를 선택합니다.

3. **도메인 추가**: "도메인 관리" 섹션에서 "도메인 추가" 버튼을 클릭합니다.
![도메인 추가 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036356743/original/jeL7iX4ebXfYdti8P6lv3_LoAuiVpsT02Q.png?1731337897)

4. **도메인 이름 입력**: 웹사이트에 연결할 도메인 이름을 입력합니다.
![도메인 이름 입력 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036356768/original/3OHQFsam4beeNPZ7jLwRbd-cpAEnjwUpgA.png?1731337916)

5. **자동 구성**: "자동 구성" 버튼을 클릭하여 도메인 연결을 시작합니다.
![자동 구성 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036356898/original/sZccfm6BexsrDa5o2UFpI7HKkrrhcXCiOQ.png?1731338025)

6. **프로세스 완료**: 도메인 연결을 완료하기 위한 나머지 단계를 따라주세요.

이번 업데이트로 도메인 커넥트는 도메인 설정을 더욱 쉽고 빠르게 만들어주므로, 바로 사이트 관리에 집중하실 수 있습니다!
