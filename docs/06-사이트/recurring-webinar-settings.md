# 반복 웨비나 설정

반복 웨비나 설정을 사용하면 몇 분 만에 일련의 라이브 세션을 자동화할 수 있습니다. 더 이상 이벤트를 복사하거나 날짜를 관리하거나 참석자를 혼란스럽게 할 필요가 없습니다. 일회성 웨비나를 안정적인 주간 수업, 월간 제품 데모 또는 "언제든지 시청 가능한" 상시 방송으로 바꿀 수 있습니다.

# **반복 웨비나 설정 개요**

반복 웨비나 설정은 하이레벨 웨비나에 선택적인 "반복" 레이어를 추가하여 단일 이벤트가 정의한 스케줄에 따라 자동으로 반복될 수 있도록 합니다. 내부적으로 하이레벨은 미래의 발생 일정을 생성하고, 방송 페이지를 다음 예정된 세션으로 업데이트하며, 등록, 리마인더, 분석이 전체 시리즈에 연결된 상태를 유지합니다. 이는 한 번만 주기를 설정하면 하이레벨이 나머지를 처리한다는 의미입니다. 일일 스탠드업이든, 격주 마스터마인드든, "셋째 주 금요일" 워크숍이든 상관없습니다.

---

## **반복 웨비나 설정의 주요 이점**

- 수동 복사 작업을 절약하고 대신 하나의 시리즈를 구성
- 단일 등록과 방송 링크로 일관된 참석자 경험 제공
- 예측 가능하고 "습관을 형성하는" 일정을 통한 높은 참여도와 유지율
- 유연한 패턴(일간, 주간, 월간, 사용자 정의)이 거의 모든 사용 사례에 적합
- "고정 시간 없음" 상시 옵션으로 재방송을 24/7 이용 가능하게 유지

---

## **반복 웨비나 토글**

모든 웨비나의 Schedule(일정) 탭에서 "Recurring(반복)" 스위치를 활성화하세요. 켜지면 빈도 선택기와 고급 옵션이 표시됩니다. 참석자는 여전히 한 번만 등록하면 되며, 하이레벨이 자동으로 모든 예정된 발생에 배정합니다.

## **유연한 반복 패턴**

프로그램에 맞는 주기를 선택하세요. 이러한 옵션은 친숙한 캘린더 규칙을 반영하여 신규 사용자도 직관적으로 설정할 수 있습니다. 각 반복 일정에 대해 사용 사례에 맞게 설정을 사용자 정의할 수 있습니다.

- 일간: X일마다 반복 (예: 2일마다)
- 주간: 특정 요일에 X주마다 반복 (예: 월요일과 수요일)
- 월간: 달력 날짜(15일) 또는 서수 요일(셋째 주 금요일)에 반복
- 사용자 정의 종료: "N회 세션 후" 또는 "특정 날짜에" 중지
- 고정 시간 없음: 참석자가 언제든지 시청할 수 있는 상시 재방송 생성

이러한 옵션은 친숙한 캘린더 규칙을 반영하여 신규 사용자도 직관적으로 설정할 수 있습니다.

## **방송 페이지 로직**

하나의 방송 URL이 전체 시리즈를 처리합니다. 각 세션 전에 페이지는 카운트다운을 표시하고, 라이브를 시작하면 웨비나를 스트리밍하며, 세션 사이에는 "다음 세션이 ... 후에 시작됩니다" 메시지를 표시합니다. 이를 통해 마케팅 자산을 상시 사용 가능하게 유지하고 깨진 링크를 제거합니다.

## **자주 묻는 질문**

**Q: 등록자는 모든 세션에 대해 가입해야 하나요?**

아닙니다. 한 번의 등록으로 전체 시리즈에 등록됩니다. 리마인더 이메일은 각 개인의 다음 세션에 맞춰 조정됩니다.

**Q: 시리즈가 시작된 후 일정을 편집할 수 있나요?**

네. 반복 패턴을 변경하면 모든 미래(과거 아님) 발생이 업데이트됩니다. 기존 등록자는 자동으로 새 날짜에 매핑됩니다.

**Q: 반복 웨비나의 리마인더는 어떻게 작동하나요?**

표준 웨비나 리마인더 워크플로우가 다음 발생으로 자동 재스케줄되므로 자동화를 다시 구축할 필요가 없습니다. Workflows(워크플로우)에서 타이밍을 추가로 사용자 정의할 수 있습니다.

**Q: 세션 사이에 방송 페이지는 어떻게 되나요?**

호스트가 라이브를 시작할 때까지 카운트다운 또는 "다음 세션이 ... 후에 시작됩니다" 배너를 표시한 다음 세션을 스트리밍합니다. 이후에는 다음 카운트다운으로 되돌아갑니다.

**Q: 하나의 시리즈에서 라이브와 온디맨드 콘텐츠를 혼합할 수 있나요?**

같은 웨비나 내에서는 불가능합니다. 별도의 온디맨드 웨비나를 생성하거나 웨비나 후 자동화를 통해 재방송 링크를 공유하세요.

**Q: "고정 시간 없음" 옵션은 어떻게 작동하나요?**

지속적인 상시 웨비나를 생성합니다. 등록자는 언제든지 참여할 수 있으며, 방송 페이지는 즉시 사전 녹화된 비디오를 재생합니다.
