# 설문 - 오더 범프 설정하고 사용하기

이 가이드에서는 하이퍼클래스 설문(Surveys)에서 오더 범프를 사용하는 방법을 알아보겠습니다. 매력적인 범프 오퍼를 추가하면 사용자 경험을 방해하지 않으면서도 설문에서 직접 업셀 기회를 제공하여 매출을 늘릴 수 있습니다. 이 업데이트를 통해 현대적인 비주얼, 다양한 템플릿, 유연한 활성화 기능을 갖춘 맞춤형 애드온으로 매출을 높일 수 있습니다.

## 설문의 오더 범프란?

설문의 모던 오더 범프는 설문 안에 업셀 옵션을 직접 삽입하여 매출을 높이는 기능입니다. 메인 상품과 함께 범프 상품을 통합함으로써, 하이퍼클래스는 매력적이고 맞춤화 가능한 경험을 통해 추가 가치를 창출할 수 있게 해줍니다.

## 오더 범프의 주요 장점

설문의 확장된 오더 범프 기능을 통해 외관과 성능을 모두 향상시키는 여러 사용자 중심의 장점을 얻을 수 있습니다:

- 클래식과 모던 범프 스타일 옵션으로 향상된 가시성
- 주목도를 높이고 더 높은 전환율을 이끄는 다양한 템플릿 선택
- 색상, 아이콘, 강조 텍스트, 가격 표시 등 비주얼의 완전한 커스터마이징
- 업셀 전략을 확장하는 다중 범프 상품 지원
- 설문당 하나의 구독 상품만 설정되도록 하는 스마트 구독 규칙
- 범프 상품을 쉽게 토글하고 재배치할 수 있는 유연한 활성화

## 설문에서 오더 범프 설정하고 사용하기

하이퍼클래스의 직관적인 설문 빌더를 사용하면 모던 오더 범프 설정이 간단합니다. 다음 단계별 안내에 따라 설문에 범프 상품을 통합해보세요:

### 1단계: 설문 만들기

- 하위 계정에 로그인합니다.
- 좌측 사이드바에서 사이트(Sites)를 클릭 → 설문(Surveys) 선택 → 빌더(Builder)를 클릭합니다.
- + 설문 추가(Add Survey)를 클릭합니다.
- 처음부터 시작(Start from Scratch)을 선택하거나 템플릿을 선택 → 만들기(Create)를 클릭합니다.
- 요소(Element) 바에서 상품 판매(Sell Products) 요소를 설문 캔버스로 드래그합니다.

![오더 범프 설문 만들기](https://jumpshare.com/v/Y5dypmurPaobZF4yKrQQ+/GIF+Recording+2025-07-24+at+8.40.36+PM.gif)

### 2단계: 메인 상품 추가하기

- 상품 판매(Sell Products) 요소를 클릭합니다.
- 우측 패널에서 상품 추가(Add Product)를 클릭합니다.
- 아직 상품을 만들지 않았다면: 여기를 클릭(Click here) 링크를 클릭하여 상품 생성 페이지를 열고 설정합니다.
- 이미 상품이 있다면: 상품을 선택해주세요(Please select the product) 드롭다운을 사용합니다.
- 상품 유형(Product Type) 선택: 메인 상품(Main Product)을 선택합니다.
- 표시 옵션(Show Options): 설명(Description), 이미지(Image), 수량(Quantity)을 토글하여 표시/숨김을 설정합니다.
- 상품 추가(Add Product)를 클릭하여 삽입합니다.

![메인 상품 추가](https://jumpshare.com/v/rDUo2mzTWTYwPhaFcBNv+/GIF+Recording+2025-07-24+at+8.48.46+PM.gif)

### 3단계: 범프 상품 추가하기

- 상품 추가(Add Product)를 다시 클릭합니다.
- 새로 만들거나 드롭다운에서 기존 상품을 선택합니다.
- 상품 유형(Product Type) 선택: 범프 상품(Bump Product)을 선택합니다.
- 매력적인 헤드라인을 추가합니다.
- 이미지를 활성화/비활성화하고, 새 이미지를 업로드하거나 시스템이 상품 레코드에서 가져오도록 합니다.

![범프 상품 추가](https://jumpshare.com/v/S9lGMnoD4uYUQAbT5JwR+/GIF+Recording+2025-07-24+at+9.44.04+PM.gif)

### 4단계: 범프 상품 콘텐츠와 비주얼 커스터마이징

- 오더 범프 스타일(Order Bump Style)에서 모던(Modern) 또는 클래식(Classic)을 선택합니다.
- 템플릿 선택(Select Template) 드롭다운을 사용하여 템플릿 1, 2 또는 3을 선택합니다.
- 강조 텍스트(Highlight Text)에 주의를 끄는 문구를 입력합니다(예: "클릭 한 번으로 특가 혜택을 받으세요").
- 아이콘(Icon): 강조 텍스트 옆에 표시할 아이콘을 선택합니다.
- 범프 상품의 가격 표시(Show Price) / 가격 숨김(Hide Price)을 토글합니다.
- 배경색, 강조 텍스트 색상, 헤드라인 색상, OTO 헤드라인 및 OTO 텍스트 색상을 커스터마이징합니다.
- 레이아웃 설정(Layout Settings)에서 1열, 2열 또는 3열 레이아웃을 선택합니다.
- 상품 선택(Product Selection): 단일 선택 또는 다중 선택을 허용합니다.
- 하단 정보(Footer Info)를 토글합니다(예: "100% 안전한 결제 시스템").
- 플레이스홀더 텍스트를 숨기거나 유지할지 선택합니다.

![범프 상품 커스터마이징](https://jumpshare.com/v/cQiqgSyDmTJrOFPQyazT+/GIF+Recording+2025-07-24+at+9.47.48+PM.gif)

### 5단계: 설문 저장하기

- 우측 상단의 저장(Save)을 클릭하여 변경사항을 저장합니다.

![설문 저장](https://jumpshare.com/v/nsXVDx4jYmDGelQt4pHQ+/Screen+Shot+2025-07-24+at+9.53.00+PM.png)

## 자주 묻는 질문

**Q: 하나의 설문에 여러 개의 범프 상품을 추가할 수 있나요?**
네, 같은 설문에 여러 개의 범프 상품을 포함할 수 있습니다. 이를 통해 메인 상품 플로우를 방해하지 않으면서 다중 업셀 기회를 제공할 수 있습니다.

**Q: 메인 상품과 범프 상품 모두 구독형으로 설정할 수 있나요?**
아니요. 설문당 하나의 상품만 구독형으로 설정할 수 있습니다. 메인 상품 또는 하나의 범프 상품 중 하나만 가능하며, 둘 다는 안 됩니다. 이는 결제 로직의 일관성을 보장하기 위함입니다.

**Q: 메인 상품을 포함하지 않으면 어떻게 되나요?**
메인 상품이 없으면 범프 상품이 표시되지 않습니다. 범프 상품 가시성을 활성화하려면 항상 메인 상품을 먼저 추가해야 합니다.

**Q: 범프 상품은 모든 결제 처리 업체와 호환되나요?**
네. 오더 범프는 Stripe, PayPal 및 하이퍼클래스에서 지원하는 기타 모든 결제 처리 업체와 원활하게 작동합니다.

**Q: 설문의 나머지 부분에 영향을 주지 않고 범프 상품을 재배치하거나 일시적으로 비활성화할 수 있나요?**
물론입니다. 메인 상품 설정이나 설문 로직을 방해하지 않고 언제든지 범프를 켜거나 끄거나 순서를 바꿀 수 있습니다.

**Q: 어떤 유형의 설문에서 오더 범프를 지원하나요?**
상품 판매(Sell Product) 요소가 포함된 모든 설문에서 오더 범프 기능을 활용할 수 있습니다.

**Q: 각 범프 상품의 디자인을 개인화할 수 있나요?**
네, 이 기능은 완전한 커스터마이징을 제공하여 각 범프 상품마다 색상, 아이콘, 강조 텍스트, 가격을 개별적으로 조정할 수 있습니다.

**Q: 하나의 설문에 몇 개의 범프 상품을 추가할 수 있나요?**
여러 개의 범프 상품을 추가할 수 있어 필요에 따라 업셀 전략을 확장할 수 있는 유연성을 제공합니다.

**Q: 클래식과 모던 범프 스타일을 어떻게 전환하나요?**
설문에서 범프 기능을 활성화할 때 범프 상품 설정 내에서 스타일을 토글할 수 있습니다.
