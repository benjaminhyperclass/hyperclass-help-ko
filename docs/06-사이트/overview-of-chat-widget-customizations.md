# 채팅 위젯(Chat Widget) 커스터마이징 개요

채팅 위젯에 완전히 새로운 기능이 추가되었습니다! 이제 배치, 디자인, 느낌을 원하는 대로 커스터마이징하여 더욱 다양한 맞춤 설정이 가능해요.

## 어떻게 사용하나요?

- Sites(사이트) → Chat Widget(채팅 위젯)으로 이동하세요
- New(새로 만들기)를 클릭해서 채팅 위젯을 생성하세요
- SMS/이메일 위젯, 라이브 채팅, 왓츠앱 채팅 중 하나를 선택하세요
- 선택이 완료되면 Widget(위젯) 탭에서 다양한 스타일과 테마 옵션으로 디자인을 커스터마이징할 수 있어요. 아바타 이미지 선택, 위젯 배치 및 크기 설정이 가능합니다.
- 프롬프트 포함 여부를 선택하여 채팅 런처를 추가하세요
- 채팅 아이콘을 선택하세요
- 테마를 선택하거나 브랜드에 맞게 커스터마이징하세요
- 첫 방문자와 재방문자를 위한 메시지를 설정하세요
- 아바타 이미지를 추가하세요
- 위젯을 배치할 위치를 선택하세요
- 위젯 크기 - 자동으로 설정하거나 너비와 높이를 직접 커스터마이징하세요

- 다음은 Chat Window(채팅 창) 설정이에요. 방문자에게 고정으로 보여지는 모든 메시지를 여기서 설정합니다. 각 유형별로 채팅 위젯에는 추가 필드가 있어요. 채팅 창에서는 브랜딩과 컴플라이언스 부분을 설정할 수 있습니다.
- 채팅 창 제목, 인트로 메시지를 추가하고, 이메일 활성화 및 CTA 버튼을 변경할 수 있어요
- 자체 브랜딩과 HIPAA 법적 동의를 추가할 수 있습니다

![채팅 창 브랜딩 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155030318955/original/PQYUAC5f8whizlJHA6l-5tn6yHkqvL0Qfw.jpeg?1722542586)

- 마지막 옵션은 Messaging(메시징) 설정으로, 확인 메시지와 언어 설정을 커스터마이징할 수 있어요.
- 고객 지원 연락처 정보, 확인 메시지, 인사말을 추가하세요
- 채팅 위젯의 언어를 선택할 수 있습니다

![메시징 및 언어 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155030318953/original/a9jaYcCuHDyhEBNWzsmCO-RRf5p-h06bww.jpeg?1722542585)

## 참고사항

- 이미 채팅 위젯을 사용하고 있었다면, Widget(위젯) 옵션의 테마에서 연한 파란색으로 표시됩니다
- 워드프레스 플러그인은 기본 위젯에서 작동하며, 위젯 선택 기능은 향후 업데이트에서 추가될 예정입니다.
