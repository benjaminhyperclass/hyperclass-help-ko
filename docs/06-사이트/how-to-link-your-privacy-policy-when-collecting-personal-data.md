# 개인정보를 수집할 때 개인정보처리방침 연결하는 방법

개인정보처리방침은 비즈니스가 이름, 전화번호, 이메일 등의 개인정보를 어떻게 수집, 사용, 보호하는지 설명하는 문서입니다. 이는 단순한 모범 사례가 아니라 GDPR, CCPA, A2P 10DLC 통신사 규정과 같은 법률에서 종종 법적으로 요구되는 사항입니다. Hyperclass는 사용자로부터 데이터를 수집하는 모든 곳에서 개인정보처리방침을 연결할 수 있는 다양한 방법을 제공합니다.

**면책조항**: 이는 법률 자문이 아닙니다. 이 도움말은 정보 제공 목적으로만 작성되었습니다. 항상 귀하의 법률 고문과 상담하세요.

## 개인정보처리방침 링크를 추가할 곳

Hyperclass의 각 데이터 수집 영역에서는 개인정보처리방침을 연결할 수 있는 곳을 제공합니다. 각 기능별로 아래 단계를 따라하세요.

### 퍼널과 웹사이트

- **텍스트 요소(Text Element)**: 텍스트 필드를 드래그하여 메시지를 입력하고 (예: "제출함으로써 개인정보처리방침에 동의합니다"), "개인정보처리방침"을 선택한 다음 링크를 클릭하여 URL을 붙여넣으세요

### 폼

- **텍스트 요소(Text Element)**: 텍스트 필드를 드래그하여 메시지를 입력하고 (예: "제출함으로써 개인정보처리방침에 동의합니다"), "개인정보처리방침"을 선택한 다음 링크를 클릭하여 URL을 붙여넣으세요

- **약관 동의 요소(Terms & Conditions Element)**: 약관 동의 요소를 드래그하고, 링크 편집을 선택한 다음 정책 URL을 붙여넣고 텍스트를 사용자 정의하세요

### 설문

- **텍스트 요소(Text Element)**: 텍스트 필드를 드래그하여 메시지를 입력하고 (예: "제출함으로써 개인정보처리방침에 동의합니다"), "개인정보처리방침"을 선택한 다음 링크를 클릭하여 URL을 붙여넣으세요

### 캘린더

- **텍스트 요소(Text Element)**: 캘린더가 임베드된 페이지의 텍스트/링크

- **약관 동의 요소(Terms & Conditions Element)**: 링크가 포함된 약관 동의 요소가 있는 커스텀 폼 사용 (참고: 약관이 포함된 웹사이트 페이지에 연결해야 합니다)

### 주문 폼

- **텍스트 요소(Text Element)**: 주문 페이지의 텍스트/링크

- **고급 설정(Advanced Settings)**: 주문 폼의 고급 설정에서 약관 동의를 활성화하고 거기에 링크를 만드세요

### 웹 채팅

- **링크**: 위젯 설명의 텍스트에 링크 URL

## 개인정보처리방침 페이지 만드는 방법

정책에 연결하기 전에 독립적인 페이지가 필요합니다 (Hyperclass 내부 또는 외부 호스팅). 다음과 같이 진행하세요:

- **Sites(사이트) > Funnels(퍼널)** 또는 **Websites(웹사이트)**로 이동하세요

- 새 퍼널/웹사이트를 만들거나 기존 것을 여세요

- 새 페이지를 추가하고 "개인정보처리방침"이라고 제목을 지으세요

- 전체 개인정보처리방침 내용을 붙여넣으세요

- 발행하고 페이지 URL을 복사하세요
