# 워드프레스 사이트 템플릿이란?

워드프레스 사이트(WordPress Sites)는 Hyperclass 웹사이트/퍼널의 대안입니다. Hyperclass의 내장 사이트 빌더와 호스팅 대신, 완전한 워드프레스 설치를 호스팅할 수 있어요. 이를 위해서는 Hyperclass의 워드프레스 호스팅 요금제를 구독해야 하며, 그러면 다른 호스팅된 워드프레스 사이트처럼 사용할 수 있습니다.

새 워드프레스 사이트를 만들 때 템플릿 라이브러리를 둘러보고 미리 만들어진 워드프레스 사이트 중 하나를 선택할 수 있어요. 선택한 템플릿으로 워드프레스 사이트가 설정되는 동안 잠시 대기 시간이 있습니다.

---

## 워드프레스 사이트 템플릿을 사용해야 하는 이유

워드프레스 사이트 템플릿은 거의 제로에 가까운 노력으로 아름답고 기능적인 워드프레스 사이트를 거의 즉시 런칭할 수 있도록 도와줍니다. 여러분이 할 일은 단지 커스터마이징하는 것뿐이에요.

---

## 템플릿으로 워드프레스 사이트 설정하는 방법:

템플릿으로 워드프레스 사이트를 설정하는 단계:

하위 계정 > Sites(사이트) > WordPress로 이동하여 "+ Create Site"를 선택하세요
