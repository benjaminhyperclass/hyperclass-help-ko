# 사이트 생성 중 인기 워드프레스 플러그인 설치하는 방법

이 가이드에서는 사이트 생성 과정에서 엄선된 인기 플러그인 목록을 선택하여 워드프레스 사이트 설정을 간소화하는 방법을 알려드립니다. 시간을 절약하고 처음부터 필요한 도구로 맞춤 설정해보세요!

## **플러그인 선택 기능이란?**

플러그인 선택 기능을 사용하면 사이트 생성 초기 단계에서 인기 있고 권장되는 워드프레스 플러그인 목록 중에서 선택할 수 있습니다. 이 옵션들은 하이레벨의 호스팅 워드프레스 환경에서 실제 사용 패턴을 바탕으로 엄선되어, 대부분의 사이트 구축에 즉시 도움이 되고 가치를 제공합니다.

이 단계는 설정 위저드에 직접 통합되어 있어 사용자가 신뢰할 수 있는 도구로 사이트를 빠르게 시작할 수 있습니다. 설정 후 플러그인을 찾는 번거로움이 없어요.

---

## **플러그인 선택의 주요 장점**

유용하고 신뢰할 수 있는 플러그인으로 워드프레스 사이트 커스터마이징과 최적화를 미리 시작하세요. 이 기능은 다음과 같은 장점을 제공합니다:

- 🎯 **실제 사이트 사용 트렌드를 기반으로 엄선된 플러그인 목록**

- ⚙️ **자동 설치 및 활성화**—수동 단계가 필요 없습니다.

- ✋ **강제 설치 없음**—필요한 것만 선택하면 됩니다.

- 🎛️ **설정 후 플러그인 대시보드를 통한 완전한 플러그인 제어**

- ⚡ **출시 후 플러그인 찾기 단계를 건너뛰어 더 빠른 사이트 설정**

- 🔄 **선택 사항인 선택 단계**—고급 사용자와 초보자 모두에게 이상적

## **워드프레스 사이트 설정 중 플러그인 선택 사용 방법**

새 워드프레스 사이트를 만들 때 플러그인 선택 기능을 활용하려면 다음 단계별 가이드를 사용하세요.

- 하이레벨 계정에서 **사이트(Sites) → 워드프레스 대시보드(WordPress Dashboard)**로 이동합니다.

- **사이트 생성(Create Site)** 또는 작업 중인 사이트 옆의 **설정 완료(Complete Setup)**를 클릭합니다.

- **인기 플러그인(Popular Plugins)** 단계에 도달할 때까지 설정 위저드를 진행합니다.

- 사용 가능한 플러그인 목록을 둘러보고 포함하고 싶은 플러그인의 체크박스를 선택합니다.

- **계속(Continue)** 또는 **설정 완료(Finish Setup)**를 클릭하여 사이트 생성을 완료합니다.

과정이 완료되면 선택한 플러그인이 자동으로 설치되고 활성화됩니다. 시간을 절약하고 선호하는 도구가 처음부터 준비되도록 해줍니다.

## **설정 후 플러그인 관리**

사이트가 라이브된 후에도 플러그인 환경에 대한 완전한 제어권을 유지합니다. 워드프레스 대시보드를 통해 필요에 따라 플러그인을 관리, 업데이트 또는 제거할 수 있습니다.

플러그인을 관리하려면:

- 하이레벨 **사이트(Sites)** 페이지에서 **"워드프레스 대시보드 열기(Open WordPress Dashboard)"**를 클릭합니다.

- 워드프레스 관리자 패널에서 **플러그인(Plugins) → 설치된 플러그인(Installed Plugins)**으로 이동합니다.

- 여기서 다음 작업을 할 수 있습니다:
  - 현재 사용하지 않는 플러그인 **비활성화(Deactivate)**
  - 불필요하거나 실험적인 플러그인 **삭제(Delete)**
  - 엄선된 목록에 포함되지 않은 추가 플러그인 **설치(Install)**

설정 중 플러그인 선택을 건너뛰거나 변경했더라도 유연성과 제어권을 보장합니다.

## **자주 묻는 질문**

**Q: 기본으로 설치되는 플러그인이 있나요?**

네. **하이레벨 호스팅을 통해 프로비저닝된 새 워드프레스 사이트**에서는 하이레벨이 설정 중에 **LeadConnector(LC) 플러그인**을 자동으로 설치합니다. **인기 플러그인(Popular Plugins)** 단계의 다른 모든 플러그인은 선택 사항이며 선택한 경우에만 설치됩니다.

**Q: 사이트 생성 후 플러그인 선택을 변경할 수 있나요?**

물론입니다. 워드프레스 플러그인 대시보드를 사용하여 언제든지 플러그인을 설치하거나 제거할 수 있습니다.

**Q: 플러그인 옵션은 어떻게 선택되나요?**

플러그인 목록은 하이레벨을 통해 호스팅되는 워드프레스 사이트에서의 실제 사용 트렌드와 인기 요청을 바탕으로 엄선됩니다.

**Q: 원하는 플러그인이 목록에 없으면 어떻게 하나요?**

설정 후 표준 워드프레스 플러그인 검색 및 업로드 기능을 사용하여 언제든지 추가 플러그인을 수동으로 설치할 수 있습니다.

**Q: 플러그인 선택 단계가 필수인가요?**

아니요. 이 단계는 완전히 선택 사항입니다. 건너뛰고도 깨끗한 설치로 사이트 설정을 완료할 수 있습니다.
