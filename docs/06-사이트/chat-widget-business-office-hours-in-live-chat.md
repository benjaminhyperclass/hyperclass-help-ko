# 채팅 위젯 - 실시간 채팅의 업무 시간 설정

채팅 위젯(Chat Widget)이 실시간 채팅에서 업무 시간 설정 기능으로 사용자 경험을 획기적으로 개선했습니다.

## **어떻게 작동하나요?**

- Sites(사이트) > Chat Widget(채팅 위젯)으로 이동하세요

- 채팅 위젯 유형으로 Live chat(실시간 채팅) 옵션을 선택하세요.

![실시간 채팅 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155035806380/original/wVv7589BVJuYDlX8tYZwRzqs1jfWEF2Izg.png?1730390876)

- 채팅 위젯 설정에서 **Chat Window(채팅 창)** 탭으로 이동하면 Business Office Hour Setup(업무 시간 설정) 옵션이 있습니다.

![업무 시간 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155035806423/original/Q92B35L1eHyIX8gGWe7bFzxTk5C7c8d-pw.png?1730390915)

- Outside Office Hours Welcome Message(업무 시간 외 환영 메시지) 옵션을 사용하여 업무 시간 외 환영 메시지를 사용자 정의할 수 있습니다. 기본 메시지는 "We are currently offline(현재 오프라인입니다)"입니다.

![업무 시간 외 환영 메시지](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155035806451/original/H4laTEHRPKXBWm00fYjwmwPI43B0Ih5RiQ.png?1730390947)

- 일요일부터 토요일까지 겹치지 않도록 여러 시간대를 설정할 수 있습니다.

- 시간 설정을 원하는 요일에 복사할 수 있는 유연성을 제공하며, 여러 개를 선택할 수도 있습니다. [캘린더 예약 가능 시간 기능과 동일한 기능을 복사하여 일관성을 구축했습니다.]

- 시간대는 비즈니스(로케이션) 시간대를 기준으로 합니다.

- 최종 사용자가 연락처 폼을 제출하면 채팅이 자동으로 닫힙니다.

업무 시간이 활성화되고 최종 사용자의 시간대가 설정된 업무 시간과 일치하면, Live Chat Welcome Message(실시간 채팅 환영 메시지) 필드의 환영 메시지가 표시됩니다. 업무 시간 외에는 연락처 정보 폼이 최종 사용자에게 전송됩니다.

참고: 위젯에 Notification Sound(알림 소리)가 활성화되어 있으면, 업무 시간 외 환영 메시지가 나타날 때 방문자에게 소리가 들릴 수 있습니다.
