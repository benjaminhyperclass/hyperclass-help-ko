# 설문에 슬라이드 버튼이 없어요

설문(Survey)에서 슬라이드 버튼이 보이지 않는 문제가 발생할 수 있습니다. 이는 일반적으로 설문 설정이나 브라우저 관련 문제입니다.

## 해결 방법

1. **설문 설정 확인**: Sites(사이트) 메뉴에서 해당 설문의 설정을 다시 확인해보세요.

2. **브라우저 새로고침**: 페이지를 새로고침하거나 다른 브라우저에서 테스트해보세요.

3. **캐시 삭제**: 브라우저 캐시와 쿠키를 삭제한 후 다시 시도해보세요.

4. **요소 설정 점검**: 페이지 빌더(Page Builder)에서 해당 요소(Element)의 설정이 올바른지 확인하세요.

문제가 계속 발생한다면 고객지원팀에 문의하시기 바랍니다.
