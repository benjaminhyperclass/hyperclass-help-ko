# 설문에서 점수 추가하기

## 1단계: 설문 생성 또는 편집

- 계정에 로그인합니다.
- Sites(사이트) > Quizzes(설문)로 이동합니다.
- 기존 설문을 선택하거나 + New Quiz(새 설문) 버튼을 클릭하여 새로 만듭니다.

![설문 생성 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155049005153/original/Glec9ZtQbZzu4KrQgDaNSJ0flyWbUSLmsQ.png?1751042063)

## 2단계: 설문에 질문 추가

- Add Question(질문 추가)을 클릭하고 질문 유형을 선택합니다(단일 선택, 다중 선택, 드롭다운 등).
- 질문과 가능한 답변 선택지를 입력합니다.
- 각 답변 선택지에 대해 가중치에 따른 점수 값을 배정합니다.

![질문 추가 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155049005260/original/OSmBr7KC7eIzXEetxYJmy7f_9KbDU1L_5w.png?1751042278)

## 3단계: 카테고리에 점수 배정

- 점수 카테고리를 생성합니다(예: 카테고리 A, 카테고리 B 등).

**참고:** 카테고리를 생성하거나 편집할 때, 제품 내 카테고리 팝오버에서 카테고리 작동 방식을 설명합니다. 설정부터 옵션 배정, 결과 페이지에서 카테고리 점수 표시까지 설명하므로, 설정하면서 빠른 가이드로 참조하세요.

![카테고리 생성 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155060370015/original/efDqMjNxfbOriZl-RfJqjlvKt4i8DDxz4A.png?1765438208)

- 각 답변에 점수를 배정합니다:
  - 단일 선택/드롭다운 질문: 답변당 고정 점수를 배정합니다.
  - 다중 선택/다중 드롭다운: 여러 선택이 가능할 때 합산되는 옵션별 점수를 배정합니다.
  - 각 답변의 점수를 관련 카테고리에 배정합니다.

![점수 배정 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155049005294/original/kec9fRftzKKlqDz9xYjvv0RghxFC_Fs-RA.png?1751042314)

## 4단계: 저장 및 미리보기

- 모든 질문과 점수 설정을 완료한 후 Save Quiz(설문 저장)를 클릭합니다.
- Preview(미리보기) 옵션을 사용하여 설문을 테스트하고 점수가 올바르게 계산되는지 확인합니다.

## 설문 점수 체계 작동 방식

## 1단계: 각 질문의 최대 점수 설정

모든 질문에는 유형에 따른 최대 점수가 있습니다:

- 단일 선택/드롭다운: 하나의 답변만 선택 가능하므로 최대 점수가 고정됩니다.
- 다중 선택/다중 드롭다운: 여러 답변 선택이 가능하므로 최대 점수가 더 높습니다. 최대 점수는 고정되며 드롭다운/다중 선택의 모든 옵션 점수의 합입니다.

#### 예시:
- 단일 선택 질문의 최대 점수는 10점일 수 있습니다.
- 다중 선택 질문은 최대 50점까지 가능할 수 있습니다.

## 2단계: 답변에 점수 배정

설문 응답자가 답변을 선택하면 다음에 기여합니다:

- 전체 점수만: 총 설문 점수에는 반영되지만 특정 카테고리에는 반영되지 않습니다.
- 카테고리별 점수: 특정 카테고리와 전체 점수 모두에 반영됩니다.
- 다중 카테고리: 여러 카테고리에 동시에 영향을 줍니다.

예시:
- 설문에 카테고리 A, B, C가 있고 답변이 카테고리 A에 배정되면, 점수는 카테고리 A와 전체 점수에 반영됩니다.

## 3단계: 최종 점수 계산

모든 답변이 선택되면 다음 공식으로 최종 점수가 계산됩니다:

최종 점수 = (선택된 점수 / 최대 점수) × 100

이는 전체 점수와 각 카테고리에 대해 별도로 계산됩니다.

#### 계산 예시

| 카테고리 | 선택된 점수 | 최대 점수 | 최종 % | 최종 점수 공식 |
|---------|------------|----------|--------|---------------|
| 전체 | 90 | 130 | 69.23% | 전체 선택 점수 / 전체 가능 최대 점수 |
| 카테고리 A | 20 | 50 | 40.00% | 카테고리 A 선택 점수 / 카테고리 A 가능 최대 점수 |
| 카테고리 B | 20 | 50 | 40.00% | 카테고리 B 선택 점수 / 카테고리 B 가능 최대 점수 |
| 카테고리 C | 10 | 50 | 20.00% | 카테고리 C 선택 점수 / 카테고리 C 가능 최대 점수 |

이 예시에서:
- 설문 응답자의 전체 점수는 69.23%입니다.
- 카테고리 A에서는 50점 중 20점으로 40%를 받았습니다.

## "전체만" 옵션의 핵심 포인트

- "전체만"은 통합 점수입니다
  답변이 "전체만"에만 배정되면 어떤 카테고리에도 반영되지 않습니다.
- 답변이 카테고리와 "전체만" 모두에 배정되면 둘 다에 반영됩니다.

- 카테고리는 독립적이지만 전체 점수에 기여합니다
  각 카테고리 점수는 별도로 계산되지만, 전체 점수는 모든 카테고리 점수의 합입니다.

- 공식 분석
  최종 점수 공식은 다음과 같이 계산됩니다: X에 대한 최대 선택 점수 / 각 카테고리의 최대 가능 점수
- 이는 전체, 카테고리 A, 카테고리 B 등에 적용되어 백분율 점수를 도출합니다.

# 점수 범위별 동적 결과 메시지 추가

설문 응답자의 점수 백분율에 따라 다른 메시지를 표시할 수 있습니다.

**단계**

- 설문 빌더에서 Result Page(결과 페이지)로 이동합니다.
- 점수 범위를 정의합니다(예: 낮음: 0-40%, 보통: 41-70%, 높음: 71-100%).
- 각 범위에 대한 맞춤 메시지를 추가합니다.
- 선택적으로 CTA(구독, 상담 예약, 퍼널 이동 등)를 추가합니다.

![동적 메시지 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054285236/original/jBzb00fwb08lNz70Un1eoxwLzKLWuOCSag.png?1758543591)

![점수 범위 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054285203/original/LBhUnbIT5-PUJ7Uw9HsSEmCDyIjzKaFnIQ.png?1758543574)

**예시**

- 낮음: "개선의 여지가 있습니다. 저희가 어떻게 도울 수 있는지 알아보세요."
- 보통: "올바른 방향으로 가고 있습니다. 더 최적화하는 방법을 확인하세요."
- 높음: "대단합니다! 완벽하게 해내셨네요. 다음 단계를 논의해보겠습니다."

**모범 사례**

- 합계가 맞지 않으면 카테고리 배정을 다시 확인하세요.
- 기억하세요: "전체만" 답변은 카테고리에 영향을 주지 않습니다.
- 점수 정확도를 확인하려면 항상 미리보기를 하세요.

## 결과 페이지를 처음부터 구성하기

섹션을 추가하거나 제거하고 완전히 맞춤화할 수 있습니다.

**사용 가능한 섹션**

- Header(헤더)
- Overall Score(전체 점수)
- Category Score(카테고리 점수)
- Individual Category(개별 카테고리)
- Call to Action(행동 유도)
- Footer(푸터)

### Header(헤더)

결과 페이지의 상단 섹션입니다.

**가능한 작업:**

- 비즈니스 이름 표시/숨기기
- 기본 레이아웃과 역순 레이아웃 전환
- 로고 업로드 및 크기 설정
- 설문 제목 편집(글꼴, 크기, 색상, 정렬)
- 간격 조정(여백/패딩)
- 배경색 설정

예시: 로고와 맞춤 설문 제목 "리더십 점수표"를 추가합니다.

### Overall Score(전체 점수)

총 설문 성과를 요약합니다.

**가능한 작업:**

- 낮음/보통/높음에 대해 다른 메시지를 보여주는 Dynamic Content(동적 콘텐츠) 활성화
- 섹션 제목 변경(예: "최종 점수")
- 숫자 점수와 단계 범례 표시/숨기기
- 형식 선택(백분율, 실제 점수 또는 10점 만점)
- 점수 위치 선택(왼쪽, 오른쪽)
- 각 단계에 맞춤 텍스트, 이미지 또는 링크 추가
- 간격 및 배경색 조정

예시: "68% 득점 — 올바른 방향입니다! 다음 단계로 나아가겠습니다."

### Category Score(카테고리 점수)

여러 카테고리의 성과를 분석합니다.

**가능한 작업:**

- 섹션 헤더 변경(예: "기술 분석")
- 선택적으로 카테고리별 메시징을 위한 Dynamic Content(동적 콘텐츠) 활성화
- 백분율 표시/숨기기
- 각 카테고리 아래에 설명 텍스트 추가
- 간격 및 배경색으로 스타일링
- 각 카테고리 점수 표시 방식 선택: 백분율, 실제 점수, 또는 10점 만점 — 전체 결과 스타일과 일치시킬 수 있습니다.

예시: "마케팅 80%, 영업 60%, 리더십 40%"와 각각에 대한 피드백.

### Individual Category(개별 카테고리)

단일 카테고리를 강조합니다 — 최고 점수 또는 최저 점수 중 하나.

**사용 가능한 설정:**

- Display Category(카테고리 표시) → Highest Score(최고 점수) 또는 Lowest Score(최저 점수) 선택
- Dynamic Tags(동적 태그) → 다음과 같은 변수 삽입:
  - {{quiz_tags.highest_category_name}}
  - {{quiz_tags.highest_category_score}}
- Content Editor(콘텐츠 편집기) → 설명 텍스트 추가, 글꼴 서식 지정, 인라인 링크 포함
- Media Options(미디어 옵션) → 이미지 또는 비디오 선택
  비디오인 경우: 비디오 링크 붙여넣기 및 배치 선택(상단, 하단, 왼쪽, 오른쪽, 고정)
- 참고: 템플릿의 이미지와 비디오는 교체할 수 없고 켜기/끄기 및 위치 지정만 가능합니다.
- CTA Button(CTA 버튼) → 버튼 텍스트, 링크, 스타일(색상, 정렬) 추가
- Spacing & Background(간격 및 배경) → 여백, 패딩, 배경색 조정

예시: "점수 결과 마케팅 전략이 80점으로 나타났습니다"와 오른쪽 정렬된 비디오 및 "구독" 라벨의 CTA 버튼.

### Call to Action(행동 유도)

설문 응답자가 다음 단계를 취하도록 유도합니다.

**가능한 작업:**

- 버튼 위에 맥락 텍스트 추가
- 버튼 라벨 맞춤화(예: "상담 예약" 또는 "리포트 다운로드")
- 버튼을 퍼널, 예약 캘린더 또는 외부 리소스에 연결
- 버튼 스타일링(색상, 텍스트 색상, 정렬)
- 간격 및 배경색 조정
- 특정 점수 단계에 대해 다른 CTA 또는 지원 텍스트를 보여주는 동적 콘텐츠 옵션 사용(예: 높은 점수에는 더 강력한 제안)

![CTA 설정 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155060371706/original/R-uERL5KuDW7a12do65zyC5jLeLbWc_W9Q.png?1765439304)

예시: "결과에 대해 이야기해보시겠어요? 15분 상담 예약" 브랜드 버튼과 함께.

### Footer(푸터)

결과 페이지를 마무리합니다.

**가능한 작업:**

- 푸터 텍스트 추가(예: 면책 조항 또는 저작권)
- 작은 로고 업로드
- 링크 삽입(개인정보 처리방침, 이용약관, 소셜)
- 간격 및 배경색으로 스타일링

예시: "© 2025 XYZ Co. | 개인정보처리방침" 연한 회색 배경과 함께.

## 모범 사례

- 필요하지 않은 섹션을 숨겨서 결과를 단순하게 유지하세요.
- 사용자가 올바른 다음 단계를 취할 수 있도록 CTA를 결과와 일치시키세요.
- 데스크톱과 모바일 모두에서 레이아웃과 콘텐츠를 확인하기 위해 자주 미리보기하세요.
- 합계가 맞지 않으면 점수와 카테고리 배정을 다시 확인하세요.
- 경계 점수(예: 40%, 70%)를 테스트하여 올바른 단계에 매핑되는지 확인하세요.
