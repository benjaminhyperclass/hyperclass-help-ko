# 채팅 위젯 연락처 폼에서 커스텀 필드 추가하기

채팅 위젯(Chat Widget)이 SMS/이메일 및 왓츠앱 채팅에서 사용하는 연락처 폼에 커스텀 필드(Custom Fields) 지원을 추가하여 사용자 경험을 획기적으로 확장했습니다. 이를 통해 연락처 폼을 놀랍도록 다양하게 커스터마이징할 수 있습니다.

**새로운 기능 포함 사항:**

- **커스텀 필드 추가**: 이제 채팅 위젯 빌더에서 기본 필드 외에도 연락처에 커스텀 필드를 추가할 수 있습니다. 이러한 커스텀 필드를 통해 SMS/이메일 및 왓츠앱 채팅 위젯 유형을 사용하여 고객과의 상호작용 중에 추가 정보를 수집할 수 있습니다.

- **기본 필드**: 채팅 위젯은 계속해서 다음과 같은 기본 필드를 지원합니다:
  - 이름 (필수)
  - 전화번호 또는 이메일 (연락처 생성을 위해 하나는 필수) 
    참고 - (제거하려고 할 경우 기본값은 전화번호입니다)
  - 메시지

- **지원되는 커스텀 필드 유형**: 위젯은 다음과 같은 커스텀 필드 데이터 유형을 지원합니다:
  - 한 줄 텍스트 (SINGLE LINE)
  - 여러 줄 텍스트 (MULTI LINE)
  - 금액 (MONETARY)
  - 숫자 (NUMERIC)
  - 전화번호 (PHONE)
  - 날짜 선택 (DATE PICKER)
  - 드롭다운 (단일 선택) (DROPDOWN SINGLE)

- **필드 제한사항**: 기본 필드와 커스텀 필드를 합쳐서 총 최대 5개 필드까지 추가할 수 있습니다. [전화번호 또는 이메일 중 하나는 필수입니다]

- **필드 검증 및 필수 필드**:
  - 기본 필드(이름, 전화번호, 이메일, 메시지)는 추가 시 필수입니다.
  - 연락처 폼에서 커스텀 연락처 필드를 필수로 표시할 수 있습니다.
  - 필수 필드는 위젯에서 별표(*)로 표시됩니다.
  - 방문자는 모든 필수 필드를 완성하기 전까지는 폼을 제출할 수 없습니다.
  - 필수 필드 강제 적용은 다음 경우에 작동합니다:
    - 방문자가 위젯 연락처 폼을 통해 대화를 시작할 때
    - 상담원이 활성 대화 중에 연락처 세부정보를 요청할 때
  - 커스텀 필드에 대한 입력 검증이 적용됩니다:
    - 전화번호 필드는 국가 코드를 기준으로 검증됩니다.
    - 이메일 필드는 올바른 이메일 형식인지 검증됩니다.
  - 기존 위젯은 필드를 필수로 표시하지 않는 한 변경되지 않습니다.

- **전화번호 필드 국제화**: 두 개 이상의 전화번호 필드가 추가된 경우, 채팅 위젯에서 국제 전화번호 형식을 위한 국가 코드를 선택할 수 있습니다. 하지만 모든 전화번호 필드에 대해 하나의 국가 플래그만 선택할 수 있습니다. 사용자가 한 필드에서 국가 플래그를 변경하면, 다른 전화번호 필드의 플래그도 자동으로 업데이트됩니다.
