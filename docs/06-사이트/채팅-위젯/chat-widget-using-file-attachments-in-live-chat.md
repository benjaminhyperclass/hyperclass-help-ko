# 채팅 위젯: 실시간 채팅에서 파일 첨부 기능 사용하기

고객 문제를 더 빠르게 해결하는 방법이 이제 클릭 한 번으로 가능합니다. 실시간 채팅 위젯의 첨부 파일 기능을 통해 웹사이트 방문자가 이미지나 PDF를 채팅창에 직접 드롭할 수 있어서, 팀이 필요한 맥락을 바로 파악할 수 있습니다. 더 이상 이메일로 스크린샷을 주고받을 필요가 없어요.

# **실시간 채팅 위젯 첨부 파일 기능이란?**

실시간 채팅 위젯 첨부 파일 기능은 Hyperclass 웹사이트 채팅 위젯의 Live Chat 유형에 파일 업로드 버튼(및 드래그 앤 드롭 지원)을 추가합니다. 방문자는 메시지당 최대 5개의 이미지(.jpg, .jpeg, .png, .heic) 또는 PDF 문서를 첨부할 수 있습니다(각각 최대 20MB). 상담원은 대화(Conversations) 인박스에서 다른 첨부파일과 마찬가지로 이러한 파일을 미리 보거나 다운로드할 수 있습니다.

---

## **실시간 채팅 위젯 첨부 파일 기능의 주요 장점**

- 문제 해결 속도 향상 — 고객이 긴 설명을 타이핑하는 대신 문제를 "보여줄" 수 있습니다.
- 모든 맥락을 한곳에서 수집하여 첫 번째 접촉에서 문제 해결률을 향상시킵니다.
- 스크린샷이나 문서를 요청하는 이메일 주고받기를 줄입니다.
- 모바일 친화적 워크플로우: 방문자가 휴대폰에서 카메라나 갤러리 사진을 보낼 수 있습니다.
- 안전한 공유 — 지원되지 않는 파일, 용량 초과 업로드, 중복 선택이 자동으로 차단됩니다.

---

## 실시간 채팅 위젯 첨부 파일 기능 설정 방법

방문자가 몇 번의 클릭만으로 파일을 전송할 수 있는 기능을 제공해보세요.

- Sites(사이트) ➞ Chat Widget(채팅 위젯)으로 이동하여 Live Chat 위젯을 만들거나 기존 위젯을 여세요. (추가 빌더 설정은 필요하지 않습니다.)
- 기존 임베드 코드를 사용하여 위젯을 웹사이트나 퍼널에 배포하세요.

---

### 파일 업로드 플로우 테스트하기

- 브라우저에서 페이지를 여세요.
- 클립 아이콘을 클릭하고 지원되는 파일을 선택한 다음, 메시지를 입력하고 전송하세요.
- 대화(Conversations) 인박스에 첨부파일이 나타나는지 확인하세요.

### **지원되는 파일 형식 및 제한사항**

방문자는 용량 초과 전송의 위험 없이 일반적인 비주얼과 문서를 안전하게 업로드할 수 있습니다.

- 허용되는 확장자: .jpg, .jpeg, .png, .heic, .pdf
- 메시지당 최대 파일 수: 5개
- 파일당 최대 크기: 20MB
- 반복되는 공백이나 +, <, > 같은 문자가 포함된 파일명은 보안상 거부될 수 있습니다.

---

### 데스크톱에서 업로드하기

컴퓨터의 파일 탐색기에서 위젯으로 파일을 직접 드래그하거나, 클립 아이콘을 클릭하여 파일을 찾아보세요.

### 모바일에서 업로드하기

휴대폰과 태블릿에서는 클립 아이콘을 탭하면 기기의 카메라 또는 갤러리 선택기가 나타나서 방문자가 실시간으로 사진을 찍거나 기존 이미지를 선택할 수 있습니다. 채팅창은 너무 큰 파일이나 지원되지 않는 파일을 자동으로 제거하고, 방문자에게 허용되는 파일로 다시 시도하라고 안내합니다.

모든 크기, 유형, 수량 규칙이 업로드를 깔끔하게 유지하기 위해 계속 적용됩니다.

---

### **대화에서 첨부파일 확인하기**

대화(Conversations) 내에서 상담원은 일관된 경험을 얻습니다:

- JPG/PNG 미리보기가 인라인으로 나타나며, 클릭하면 확대됩니다.
- HEIC/PDF는 파일명과 다운로드 버튼으로 표시됩니다.
- 원한다면 파일을 미디어 라이브러리에 저장하거나 연락처의 Documents(문서) 탭에 추가할 수 있습니다.

---

### 대화 AI 이미지 응답 활성화

AI Agent를 사용하는 경우, 봇이 고객 스크린샷을 분석하고 자동으로 응답하도록 설정할 수 있습니다:

- Agency view(에이전시 뷰) ➞ Settings(설정) ➞ Labs로 이동하세요.
- Sub-Account(하위 계정) 탭을 열고 "Respond to images(이미지에 응답)"를 토글하세요.

![이미지 응답 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055840146/original/u31p1epftgGY5_x15Q8jRIhYRESQfsWdZA.png?1760349806)

AI Agent에서 "Allow this bot to respond to images(이 봇이 이미지에 응답하도록 허용)"을 활성화하세요.

**참고:** AI는 현재 이미지에만 응답하며, PDF에는 응답하지 않습니다.

![봇 이미지 응답 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055837662/original/ToTFhZf6zUknBonejl25IXgEJyRsjTNVCg.png?1760348736)

## 자주 묻는 질문

**Q: 상담원이 방문자에게 파일을 보낼 수 있나요?**
아직은 불가능합니다. 대화(Conversations) 인박스에서의 파일 업로드는 향후 릴리스에서 계획되어 있습니다.

**Q: 방문자가 25MB PDF를 첨부하면 어떻게 되나요?**
위젯이 해당 파일을 제거하고 방문자에게 20MB 미만의 파일을 선택하라고 요청합니다.

**Q: 드래그 앤 드롭이 작동하지 않는 이유는 무엇인가요?**
다른 브라우저 탭이나 앱 창이 아닌 파일 탐색기/파인더 창에서 드래그하고 있는지 확인하세요.

**Q: 동영상이 지원되나요?**
아니요. 첫 번째 릴리스는 JPG, JPEG, PNG, HEIC, PDF만 지원합니다.

**Q: 파일 업로드가 위젯 운영시간을 준수하나요?**
네. 채팅이 오프라인 상태여도 방문자는 첨부파일과 함께 메시지를 남길 수 있으며, 이는 후속 조치를 위해 대화(Conversations)에 나타납니다.

**Q: 이러한 첨부파일이 연락처의 Documents(문서) 탭과 동기화될 수 있나요?**
네 — 상담원은 수신 메시지의 점 3개 메뉴를 사용하여 "Add to Documents(문서에 추가)"할 수 있습니다.

**Q: AI가 PDF 내부의 텍스트를 읽나요?**
현재로서는 불가능합니다. AI 이미지 응답은 이미지만 분석합니다.
