# 하나의 하위 계정에서 여러 채팅 위젯 만들기

Hyperclass에서 여러 채팅 위젯(Chat Widget)을 사용하여 대화를 맞춤화하세요. 이 가이드에서는 하나의 하위 계정에서 여러 위젯을 만들고 사이트에 배치하는 방법을 설명합니다.

---

## 여러 채팅 위젯 개요

여러 채팅 위젯 기능을 사용하면 동일한 Hyperclass 계정에서 하나 이상의 채팅 위젯을 운영할 수 있습니다. 각 위젯은 고유한 브랜딩, 채널, 동작 방식을 가질 수 있어서 다양한 퍼널/사이트에서 가장 적합한 경험을 제공할 수 있습니다. 목적에 맞게 구축된 위젯을 사용하면 전환율이 향상되면서도 모든 대화(Conversations)를 한 곳에서 관리할 수 있습니다!

---

## **여러 채팅 위젯 사용의 주요 장점**

- **타겟팅된 경험**: 서로 다른 디자인과 커스터마이징이 적용된 여러 채팅 위젯을 다양한 사이트에 추가할 수 있습니다.

- **번거로움 없는 업데이트**: 변경사항이나 위젯 설정을 업데이트할 때 코드를 다시 설치할 필요가 없습니다.

- **유지보수 효율성**: 다른 페이지나 퍼널에 영향을 주지 않으면서 각 위젯의 설정을 업데이트할 수 있습니다.

- **유연성**: 전체 사이트에 기본 위젯을 설정하거나 페이지별로 개별 설정할 수 있습니다.

---

## **여러 채팅 위젯 만들기 (전제 조건)**

사이트에 채팅 위젯을 할당하기 전에, 먼저 사용할 위젯을 만들어야 합니다. 시작하기 전에 아래 단계를 따라 여러 채팅 위젯을 만드세요. 이미 여러 채팅 위젯이 있다면 이 섹션을 건너뛰어도 됩니다.

채팅 위젯을 만드는 방법에 대한 자세한 내용은 [여기를 클릭하세요!](../getting-started-with-chat-widget.md)

- **Sites(사이트)** > **Chat Widget(채팅 위젯)**으로 이동합니다

![사이트 메뉴에서 채팅 위젯 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155062268806/original/SvbwjAsxfv3ohjid0ZUvbW_5-VO2nzOL_g.png?1767911948)

- 파란색 **+ New(새로 만들기)** 버튼을 선택합니다

![새 채팅 위젯 만들기 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155062268830/original/VciyapzBMSehabO3TVWU6x-v2LjGXyPCGQ.png?1767912041)

- 만들려는 채팅 위젯 유형을 선택합니다. 각 채팅 위젯 유형에 대한 자세한 정보는 [여기를 클릭하세요.](../getting-started-with-chat-widget.md)

![채팅 위젯 유형 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155062268855/original/URwAt2nYePnu9Cu2l9ekbLNI23fTMic38g.png?1767912177)

## **사이트 전체 채팅 위젯 설정**

**사이트 수준**에서 채팅 위젯을 선택하면 전체 웹사이트나 퍼널에 하나의 위젯이 적용됩니다. 이를 통해 만든 모든 채팅 위젯 중에서 선택하여 각 사이트에 다른 위젯을 할당할 수 있습니다.

사이트에 채팅 위젯을 선택하려면 다음 단계를 따르세요:

- **Sites(사이트)** → **Websites(웹사이트)** 또는 **Funnels(퍼널)**로 이동한 후, 편집하려는 웹사이트를 선택합니다.

![사이트 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155062269027/original/aFqNqaBGVsysAMzvEpl83OtLMZa6FoXleQ.png?1767912731)

- **Settings(설정)** 탭으로 이동합니다. **Chat Widget(채팅 위젯)** 드롭다운을 사용하여 만든 모든 채팅 위젯을 스크롤하고 이 사이트에 적용할 위젯을 선택합니다.

![채팅 위젯 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155062269058/original/Hi5LPtSeut-2d1VCF_e6sYUVl8fbQjT1Bg.png?1767912781)

## 페이지별 채팅 위젯 할당

페이지별 코드 삽입을 사용하여 웹사이트나 퍼널 내의 특정 페이지나 퍼널 단계에서만 채팅 위젯을 표시하세요! 이 방법을 사용하면 사용자 경험을 세밀하게 제어할 수 있습니다.

단일 페이지에 채팅 위젯을 할당하려면 다음 단계를 따르세요:

- **Sites(사이트)** → **Websites(웹사이트)** 또는 **Funnels(퍼널)**로 이동한 후, 편집하려는 웹사이트를 선택합니다.

![사이트 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155062269027/original/aFqNqaBGVsysAMzvEpl83OtLMZa6FoXleQ.png?1767912731)

- **Settings(설정)** 탭으로 이동합니다. **Chat Widget(채팅 위젯)** 드롭다운을 "None(없음)"으로 설정한 후 **Save(저장)**를 클릭합니다.

![채팅 위젯을 없음으로 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155062269058/original/Hi5LPtSeut-2d1VCF_e6sYUVl8fbQjT1Bg.png?1767912781)

- **Sites(사이트)** → **Chat Widgets(채팅 위젯)**으로 이동하여 원하는 채팅 위젯을 열고, **Get Code(코드 받기)**를 클릭하여 코드 스니펫을 복사합니다.

![채팅 위젯 코드 복사](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155062269283/original/j-XR95jST0HPVTb5oxI53bz9dABsS-Dl0A.png?1767913407)

- 대상 페이지/단계의 페이지 편집기를 엽니다. **Add Element (+)(요소 추가)**를 클릭하고 **Code(코드)**를 찾아 캔버스에 드래그합니다.

![코드 요소 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155062269306/original/swHQJbQxniDAXbK-Us90xEpEsyCYAWDTwg.png?1767913506)

- 코드 요소(Code Element)를 선택합니다. 우측 설정 패널에서 **Open Code Editor(코드 편집기 열기)**를 클릭하고, 코드 스니펫을 붙여넣은 후 **Save(저장)**를 클릭합니다.

![코드 편집기에 코드 붙여넣기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155062269325/original/wKE7dpOwG8iEVe4QTi2-sztg959o2P9Ydg.png?1767913605)

## **자주 묻는 질문**

**Q: 위젯 스타일이나 설정을 변경한 후 코드를 다시 설치해야 하나요?**
아니요, 채팅 위젯은 변경사항에 맞춰 자동으로 업데이트됩니다.

**Q: Settings(설정)에서 채팅 위젯을 구성하고 코드 요소도 사용한다면 어떤 위젯이 표시되나요?**
둘 다 구성되어 있다면 Settings(설정) 선택이 우선됩니다. Settings(설정)에서 선택한 위젯만 표시됩니다.

**Q: 올인원 채팅 위젯이 여러 위젯을 대체하나요?**
올인원 옵션은 채널을 하나의 위젯으로 중앙화합니다. 페이지나 사이트별로 다른 디자인이나 CTA를 원할 때는 여전히 여러 위젯을 실행할 수 있습니다.

**Q: 페이지에서 올바른 위젯이 활성화되었는지 어떻게 확인하나요?**
편집기 내 미리보기(Preview) 기능을 사용하여 페이지에서 활성화된 위젯을 확인할 수 있습니다.
