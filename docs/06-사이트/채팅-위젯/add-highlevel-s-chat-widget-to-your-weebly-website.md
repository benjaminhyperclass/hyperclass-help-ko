# **웹사이트용 웹채팅 임베드 코드 얻기**

`Sites(사이트) → Chat Widget(채팅 위젯) → Get Code`에서 임베드 코드를 복사하세요.

## **Weebly 웹사이트에 커스텀 채팅 위젯을 추가하는 다양한 방법들의 상세 단계**

### **방법 1: **Weebly의 헤더 코드 주입 기능으로 커스텀 채팅 위젯 추가하기

- Weebly 계정에 로그인하고 웹사이트 편집기로 이동하세요.
- "Settings"를 클릭하세요.

![Weebly 설정 메뉴 클릭](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48297280726/original/_MAQaHtpb9-qXJ4aoqdEUsIuPy78wgX_jQ.jpeg?1683906475)

- 왼쪽 메뉴에서 "SEO" 옵션을 선택하세요.

![Weebly SEO 메뉴 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48297280723/original/vgjwqVj1oZOeepZ0pIjTe4994G_T6EHb9g.jpeg?1683906474)

- 아래로 스크롤하여 "Header Code Injection" 섹션을 찾고 "Edit"을 클릭하세요.
- Weebly의 "Header Code Injection" 박스에 코드를 붙여넣으세요.

![Weebly 헤더 코드 주입 영역에 코드 붙여넣기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48297280724/original/Dt2bkzxdUvhDnfiVfX39l2TcIUbKHsZq_Q.jpeg?1683906474)

- "Save"를 클릭하여 변경사항을 저장하세요.

---

### **방법 2: **Weebly의 커스텀 HTML/임베드 코드 요소로 커스텀 채팅 위젯 추가하기

- Weebly 계정에 로그인하고 웹사이트 편집기로 이동하세요.
- 왼쪽 메뉴에서 "Elements"를 클릭하고 옵션에서 "Custom HTML/Embed Code"를 선택하세요.
- 채팅 위젯을 표시하고 싶은 페이지 위치로 커스텀 HTML 요소를 드래그 앤 드롭하세요.

![Weebly 커스텀 HTML 요소 드래그 앤 드롭](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48297280725/original/P8tKnHQ9CnrLaJr_ZYiWHjVZWplVlSVrtQ.jpeg?1683906474)

- Weebly의 커스텀 HTML 요소에 채팅 코드를 붙여넣으세요.
- "Publish"를 클릭하여 변경사항을 저장하세요.
