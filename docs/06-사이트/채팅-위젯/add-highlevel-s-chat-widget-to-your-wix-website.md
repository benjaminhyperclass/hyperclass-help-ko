## 여러 방법으로 Wix 웹사이트에 커스텀 채팅 위젯 추가하는 자세한 단계:

### 

### **방법 1:** 

- Wix 계정에 로그인하고 웹사이트 대시보드로 이동합니다.
- 좌측 메뉴에서 "Settings(설정)"을 클릭한 후, "Advanced(고급)" 탭 아래의 "Custom Code(커스텀 코드)"를 선택합니다.

![Wix 설정에서 커스텀 코드 메뉴 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48297317762/original/afvuactXKgGh8MrZioPfgp44yPZvZ9dVMQ.jpeg?1683921093)

- 해당 섹션에서 "Add Custom Code(커스텀 코드 추가)"를 클릭합니다.
- 팝업 창의 "Code Settings(코드 설정)"에 코드 조각을 붙여넣습니다.
- 옵션을 설정합니다:
  - Name(이름): 코드에 대한 이름을 입력합니다
  - Add Code to Pages(페이지에 코드 추가): "All Pages(모든 페이지)"를 선택하고 "Load code once(코드 한 번 로드)"를 선택합니다
  - Place Code In(코드 배치 위치): "Head(헤드)"를 선택합니다
- "Apply(적용)"를 클릭하여 변경사항을 저장합니다.

![Wix 커스텀 코드 설정 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48297317756/original/w4V_CFuVKnEFUt7QitNud1aLOySbUz5emQ.png?1683921092)

- 이제 위젯이 Wix 사이트의 모든 페이지에 나타납니다.

---

### **방법 2:**

- Wix 계정에 로그인하고 Wix 편집기를 실행합니다. 대시보드에서 Edit Site(사이트 편집)를 클릭합니다.

![Wix 대시보드에서 사이트 편집 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48297317759/original/9B-EBPNVDe9gBv25cEYvyoVTLUMwuVXzNQ.jpeg?1683921093)

- 좌측 메뉴의 플러스(+) 아이콘을 클릭하여 "Add Elements(요소 추가)"에 접근합니다.
- 좌측 사이드바 메뉴에서 "Embed(임베드)"를 선택합니다.
- "Popular Embeds(인기 임베드)" 아래의 "Embed HTML(HTML 임베드)"를 선택합니다.

![Wix 편집기에서 HTML 임베드 요소 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48297317761/original/If6mts5CUAfyzOaTOstmGhmjsYsATMm4MA.jpeg?1683921093)

- 나타나는 새 HTML 박스에서 "Code(코드)"를 클릭합니다.
- 위젯 코드를 텍스트 필드에 붙여넣고 "Update(업데이트)"를 누릅니다.

![Wix HTML 박스에 채팅 위젯 코드 붙여넣기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48297317760/original/KLsgOAnNolAWPwLltKgtg67R88vUyXUHug.jpeg?1683921093)

- "HTML Settings(HTML 설정)" 박스를 닫습니다.
- 이제 페이지의 HTML 박스에서 위젯이 보일 것입니다.
