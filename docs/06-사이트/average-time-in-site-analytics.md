# 사이트 평균 체류 시간 분석

사이트 평균 체류 시간 분석으로 방문자가 사이트에서 얼마나 시간을 보내는지 알아보세요! 이 기능은 선택한 기간 동안 사용자가 사이트와 어떻게 상호작용하는지 더 잘 이해할 수 있도록 도와주는 참여도 지표를 제공합니다.

---

## 사이트 평균 체류 시간 분석이란?

사이트 평균 체류 시간은 **사이트(Sites) → 분석(Analytics)**에 있는 전용 뷰로, 방문자가 퍼널, 웹사이트, 웨비나의 페이지에서 실제로 보내는 시간을 보여줍니다.

한눈에 **평균 시간**과 **30초 이내 이탈** KPI 카드를 볼 수 있으며, 시간별 참여도를 비교할 수 있는 인터랙티브 트렌드 차트도 제공됩니다.

## 평균 시간 지표의 주요 장점

- **트렌드 가시성**: 퍼널, 웹사이트, 웨비나 전반에서 참여도가 높은 지점과 낮은 지점을 식별합니다.

- **실행 가능한 우선순위 설정**: 콘텐츠, UX, 디자인 최적화를 위한 명확한 신호를 제공합니다.

- **추측 작업 감소**: 유지율과 전환율 개선 시 추측을 줄여줍니다.

- **데이터 기반 행동**: 실제 사용자 참여 행동을 바탕으로 데이터에 근거한 결정을 내릴 수 있게 합니다.

---

## 주요 지표

#### 평균 시간

선택한 날짜 범위와 사이트에서 방문당 평균 활성 시간입니다. 사이트 전반의 일반적인 사용자 참여도를 측정하는 데 도움이 됩니다.

#### 30초 이내 이탈

해당 페이지에 착륙한 후 첫 30초 이내에 종료된 페이지 방문의 비율(이탈률)입니다. 기대에 부응하지 못한 콘텐츠, 느린 로딩 시간, 관련성 없는 콘텐츠로 인한 짧은 체류나 낮은 참여도 세션을 식별하는 데 도움이 됩니다.

**참고**: 각 방문/세션은 독립적으로 계산됩니다. 노이즈를 줄이기 위해 매우 짧은 방문(<2초)은 제외됩니다.

---

## 평균 시간에 접근하는 방법

사이트 분석 내에서 평균 시간 기능을 찾고 활용하는 단계를 따라하세요.

#### **1단계**: 분석으로 이동

하위 계정 대시보드에서 왼쪽 네비게이션 바를 사용해 **사이트(Sites)**로 이동한 후, 상단 네비게이션 리본에서 **분석(Analytics)** 탭을 선택하세요.

![사이트 분석 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063888457/original/R4gVKNNpOhHUVpIEUAsVENx_yc6ub-QN7A.png?1769800955)

#### **2단계**: 퍼널 또는 웹사이트 선택

대시보드 왼쪽 상단의 드롭다운을 사용해 퍼널, 웹사이트, 웨비나 중 어떤 데이터를 볼지 선택하세요. 두 번째 드롭다운을 사용해 특정 사이트로 결과를 필터링할 수도 있습니다.

![사이트 유형 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063889272/original/_HPOKnGVNwIptt2cqCzcg9CLhFUdY0mxrw.png?1769802569)

#### **3단계**: 날짜 범위 설정

오른쪽 상단의 날짜 선택기를 사용해 데이터를 필터링할 기간을 선택하세요.

![날짜 범위 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063889278/original/tK4CGtOddbMIulXVOw9mHCuVCUgXoO5dbg.png?1769802590)

#### **4단계**: "평균 시간" 카드 보기

"평균 시간" 카드까지 스크롤하세요. 여기서 다음을 할 수 있습니다:

- 30초 이내에 이탈한 비율을 봅니다.

- 선택한 기간 동안 세션당 평균 체류 시간을 봅니다.

- 어떤 컬럼에든 호버하면 해당 기간 동안의 세션당 평균 체류 시간을 볼 수 있습니다.

![평균 시간 카드](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063889282/original/PN32_d73Wv4iqkENcdsBIgG4bLBMjd-axA.jpeg?1769802613)

## 자주 묻는 질문

**Q: 어떤 유형의 사이트가 평균 시간 지표를 지원하나요?**
퍼널, 웹사이트, 웨비나를 지원합니다.

**Q: 방문자가 탭을 열어둔 채로 떠났을 때 유휴 시간은 어떻게 처리되나요?**
비활성 시간은 제한되어 유휴 탭이 평균 세션 시간을 부풀리지 않습니다.

**Q: 매우 짧은 방문은 어떻게 처리되나요?**
실수로 클릭하거나 즉시 이탈하는 노이즈를 줄이기 위해 2초 미만의 세션은 제외됩니다.
