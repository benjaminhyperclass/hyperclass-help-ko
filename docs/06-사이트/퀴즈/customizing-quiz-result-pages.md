# 퀴즈 결과 페이지 커스터마이징

Hyperclass의 향상된 **퀴즈 빌더(Quiz Builder)**에 강력한 **결과 페이지(Result Page)** 커스터마이징 도구가 추가되었습니다. 드래그 앤 드롭 섹션 관리, 섹션 템플릿, 표시 제어, 그리고 통합된 0~100점 글로벌 점수 척도를 제공합니다.

이러한 업그레이드로 퀴즈 제작, 분석, 관리가 더욱 빠르고 유연해지며, 계정 전체에서 완전히 표준화됩니다.

---

# 퀴즈 섹션의 글로벌 점수 등급 관리란?

점수 등급 관리 기능으로 결과 페이지를 직관적으로 구성하고 퀴즈 점수를 일관성 있게 매길 수 있습니다. 섹션 제어 패널에서 결과 페이지의 모든 요소를 보고, 재배열하고, 추가하거나 숨길 수 있으며, 점수 등급 시스템은 모든 퀴즈가 표준화된 0~100점 범위를 사용하도록 하여 신뢰할 수 있는 비교와 자동화를 보장합니다.

각 퀴즈에는 다음이 포함됩니다:

- 모든 결과 페이지 요소를 보여주는 **좌측 섹션 표시 패널**
- 섹션의 **드래그 앤 드롭 재배열**
- 빠른 구축을 위한 **섹션 템플릿**
- 깔끔한 레이아웃을 위한 **숨기기/삭제 도구**
- 글로벌 점수 로직을 위한 점수 등급 구성

## 섹션 제어 및 점수 등급의 주요 이점

이러한 기능들은 결과 페이지 커스터마이징을 간소화하고 퀴즈 점수 표시 방식을 표준화하여 분석, 보고, 자동화를 훨씬 더 정확하게 만듭니다.

- **중앙집중식 섹션 관리:** 하나의 패널에서 모든 섹션을 빠르게 탐색할 수 있습니다
- **드래그 앤 드롭 편집:** 재구축 없이 퀴즈 요소를 즉시 재배열할 수 있습니다
- **미리 만들어진 섹션 템플릿:** "전체 점수"나 "행동 유도" 같은 형식화된 블록을 클릭 한 번으로 추가할 수 있습니다
- **표시 토글:** 콘텐츠를 유지하면서 퀴즈 응시자에게는 섹션을 숨길 수 있습니다
- **표준화된 점수:** 모든 결과가 동일한 0~100 백분율 척도를 따라 모든 퀴즈에서 일관성을 보장합니다

## 섹션 제어 및 점수 등급으로 퀴즈 설정하는 방법

적절한 설정으로 퀴즈 섹션과 점수가 올바르게 표시되고 글로벌 점수 일관성을 유지할 수 있습니다. 다음 단계에 따라 퀴즈를 생성하고 커스터마이징하세요.

### **퀴즈 메뉴 접근**

사이트(Sites) → 퀴즈(Quizzes) → + 퀴즈 추가(Add Quiz)로 이동합니다.

![퀴즈 메뉴 접근](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055679014/original/jppCPx1LFgCITxJ05VJuO_5LhnPmG3K8nQ.png?1760084003)

### 템플릿 사용하기

템플릿을 사용하면 전체 점수, 카테고리 점수, 행동 유도 섹션과 같은 미리 디자인된 블록을 삽입하여 생성 속도를 높일 수 있으며, 이는 자동으로 퀴즈 디자인과 일치합니다.

빌더에서 퀴즈 콘텐츠를 편집하고, 질문을 추가하고, 결과 템플릿을 검토할 수 있습니다.

![퀴즈 빌더 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055679051/original/pQwMph45D0cMFPAhO1h9DlNLSw3WJrEG-Q.png?1760084027)

### **템플릿 라이브러리**

다양한 카테고리의 템플릿 중에서 선택할 수 있습니다. 선호하는 템플릿을 선택하려면 해당 템플릿 위에 마우스를 올리고 템플릿 선택(Choose Template)을 클릭하여 퀴즈에 적용하세요.

![템플릿 라이브러리](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055679864/original/nyLlW6_C-X4c2nq7jpFtl6uEb3tisb_zZg.png?1760084472)

### 결과 템플릿 접근

퀴즈 빌더 하단에서 **"결과 템플릿(Result Template)"** 카드를 찾으세요. **템플릿 편집(Edit Template)**을 클릭하면 섹션 관리, 재배열, 점수 등급을 관리할 수 있는 결과 페이지 커스터마이징 화면이 열립니다.

![결과 템플릿 접근](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055684315/original/GyyZgQzZFY5p6PbhkrINd29crXiqGKMLQQ.png?1760086874)

### 표시 패널 및 드래그 앤 드롭 사용

섹션 표시 패널은 퀴즈 결과 페이지의 모든 섹션에 대한 구조화된 뷰를 제공합니다. 이 패널에서 어떤 섹션으로든 즉시 이동하고, 레이아웃 순서를 관리하거나, 섹션을 일시적으로 숨길 수 있습니다.

왼쪽 패널에서 섹션을 드래그하기만 하면 결과 페이지 섹션의 순서를 변경할 수 있습니다. 이로써 수동 재배열이나 페이지 재구축이 필요하지 않습니다.

![섹션 드래그 앤 드롭](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055683151/original/o5Jxyp1g3Fr7DuxAl54J_7Bq8UbxudRk7w.png?1760086336)

### 섹션 재배열

- 섹션 이름에 마우스를 올려 드래그 핸들을 표시합니다
- 핸들을 클릭하고 누릅니다
- 섹션을 원하는 위치로 이동합니다
- 놓으면 실시간 미리보기가 즉시 업데이트됩니다

![섹션 재배열 GIF](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055685747/original/B5_RzW9qYeAbG6t3m0AXHWLMA4JUim6I1g.gif?1760087260)

### 섹션 숨기기 또는 삭제

섹션 패널을 사용하여 결과 페이지를 커스터마이징하세요 — 필요에 따라 섹션을 추가하고, 재배열하고, 숨길 수 있습니다. 표시 제어 기능으로 퀴즈 결과 페이지를 깔끔하고 집중된 상태로 유지할 수 있습니다.

섹션을 숨기면 구성은 그대로 유지되면서 공개 표시에서만 제거되지만, 삭제하면 완전히 제거됩니다.

**섹션을 숨기거나 삭제하려면:**

- **눈 아이콘**을 클릭하여 표시를 켜거나 끕니다
- **휴지통 아이콘**을 클릭하여 영구 삭제합니다
- 숨겨진 섹션은 섹션 패널에서 **회색으로** 표시됩니다

![섹션 숨기기/삭제](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055684599/original/43NJUyDIZS0rePuev_iLpY7jt9IaFCQLYQ.jpeg?1760086937)

### 글로벌 점수 등급 구성

글로벌 점수 등급은 백분율을 기반으로 점수 범위(낮음, 보통, 높음)를 정의하여 모든 퀴즈 결과가 일관되게 측정되도록 합니다. 이러한 등급은 워크플로우 트리거와 조건부 자동화도 가능하게 합니다.

**점수 등급을 구성하려면:**

- 결과 페이지 편집기 우상단의 **점수 등급 구성(Configure Score Tiers)**을 클릭합니다
- 점수 라벨과 백분율 범위를 설정합니다
- 각 범위는 **0~100** 사이여야 합니다
- **저장(Save)**을 클릭하여 전역적으로 적용합니다

**예시:**
- 낮음: 0~39%
- 보통: 40~79%
- 높음: 80~100%

낮음/보통/높음 점수 로직을 정의하기 위해 점수 등급을 구성하세요. 저장을 클릭한 후 퀴즈를 미리보기하여 흐름과 점수를 확인하세요.

![점수 등급 구성](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055683016/original/TzAgzJj4gB29dtX8GDnVmLZMjyzRVYEFIw.jpeg?1760086301)

## 자주 묻는 질문

**Q: 숨겨진 섹션이 퀴즈 점수에 영향을 주나요?**

아니요. 숨겨진 섹션은 표시와 점수 모두에서 제외됩니다.

**Q: 점수 등급 라벨을 변경할 수 있나요?**

네. 점수 등급 모달에서 라벨을 편집할 수 있습니다. 단, 모든 값은 0~100 사이여야 합니다.

**Q: 새 시스템에서 기존 퀴즈는 어떻게 처리되나요?**

기존 퀴즈는 자동으로 표준화된 점수를 사용하도록 마이그레이션됩니다. 마이그레이션 후 등급 범위를 확인해주세요.

**Q: 워크플로우에서 특정 점수 등급을 트리거할 수 있나요?**

네. **"퀴즈 제출(Quiz Submitted)"** 트리거를 사용하고 **전체 점수 등급 = 낮음, 보통, 또는 높음**으로 필터링하세요.

**Q: 섹션을 삭제하면 제출 내역은 어떻게 되나요?**

삭제는 향후 퀴즈 응시자에게만 영향을 줍니다. 과거 데이터는 그대로 유지됩니다.

**Q: 모바일에서도 드래그 앤 드롭이 가능한가요?**

섹션 재배열은 현재 데스크톱 브라우저에서만 작동합니다. 모바일 지원은 곧 추가될 예정입니다.

**Q: 여러 섹션을 한 번에 숨길 수 있나요?**

네. **Shift(또는 Cmd/Ctrl)**를 누른 상태에서 다중 선택하고 여러 섹션의 표시를 토글할 수 있습니다.
