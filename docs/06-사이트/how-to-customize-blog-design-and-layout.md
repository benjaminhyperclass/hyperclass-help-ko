# 블로그 디자인 및 레이아웃 커스터마이징 방법

블로그는 블로그 이름을 클릭한 후 점 세 개 편집 버튼을 눌러서 블로그 사이트를 편집하여 커스터마이징할 수 있습니다.

- Sites(사이트) → Blogs(블로그)로 이동하세요
- 블로그 목록 보기에서 블로그 이름을 클릭하세요
- 블로그를 열면 "Edit the Blog Site(블로그 사이트 편집)" 버튼(새 블로그 포스트 옵션 옆)이 표시되고 블로그 포스트 목록이 나타납니다
- "블로그 사이트 편집"을 클릭하면 모든 요소(Element)와 글로벌 섹션이 포함된 블로그 편집기가 열립니다
- 레이아웃, 블로그 목록 카드에 표시할 내용, 블로그 포스트 스타일을 편집할 수 있습니다

## 자주 묻는 질문

**Q: 블로그 홈 페이지에 표시되는 "다음"과 "이전" 버튼을 커스터마이징하거나 편집할 수 있나요?**

"다음"과 "이전" 버튼은 블로그 페이지네이션을 위해 자동 생성되며 블로그 편집기에서 직접 편집할 수 없습니다. Advanced(고급) 탭에서 커스텀 CSS를 사용하여 스타일을 변경하거나 숨길 수는 있지만, 편집기 내에서 텍스트와 구조를 변경할 수는 없습니다.
