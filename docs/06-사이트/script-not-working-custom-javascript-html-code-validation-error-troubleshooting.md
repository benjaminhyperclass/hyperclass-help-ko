# 스크립트가 작동하지 않나요? - 커스텀 자바스크립트/HTML 코드 유효성 오류 해결

커스텀 코드에서 `<script>` 태그가 `<div>` 태그 안에 포함되어 있다면, 스크립트 태그를 잘라내어 `<div>` 태그 아래로 이동시켜 두 태그가 분리되도록 해주세요. 퍼널/웹사이트 빌더 설정에 따라 `<script>` 태그는 `<div>` 태그 안에서 작동하지 않습니다.
