# 즉시 리다이렉트 페이지 - 퍼널 문제 해결

이 가이드는 퍼널에서 즉시 리다이렉트 페이지(Instant Redirect Page)가 제대로 작동하지 않을 때 확인해야 할 사항들을 설명합니다.

## 일반적인 문제와 해결 방법

### 1. 리다이렉트가 작동하지 않는 경우

**확인할 점:**
- 대상 URL이 올바르게 입력되었는지 확인하세요
- URL에 `http://` 또는 `https://`가 포함되어 있는지 확인하세요
- 대상 페이지가 실제로 존재하고 접근 가능한지 확인하세요

### 2. 리다이렉트 지연이 발생하는 경우

**원인과 해결책:**
- 브라우저 캐시 때문일 수 있습니다. 브라우저 캐시를 삭제해보세요
- 대상 서버의 응답 속도가 느릴 수 있습니다
- 네트워크 연결 상태를 확인해보세요

### 3. 특정 브라우저에서만 문제가 발생하는 경우

**해결 방법:**
- 다른 브라우저에서 테스트해보세요
- 브라우저의 JavaScript 설정이 활성화되어 있는지 확인하세요
- 브라우저의 팝업 차단 기능이 리다이렉트를 방해하지 않는지 확인하세요

### 4. 모바일에서 리다이렉트 문제

**확인 사항:**
- 모바일 브라우저에서 JavaScript가 활성화되어 있는지 확인
- 모바일 데이터 연결이 안정적인지 확인
- 대상 페이지가 모바일 친화적인지 확인

## 문제 해결 단계

1. **퍼널 설정 확인**
   - Sites(사이트) → Funnels(퍼널) 메뉴로 이동
   - 해당 퍼널을 선택하고 편집
   - 리다이렉트 페이지 설정을 다시 확인

2. **URL 테스트**
   - 리다이렉트 대상 URL을 직접 브라우저에 입력해서 접근 가능한지 확인
   - URL 형식이 올바른지 검증

3. **캐시 초기화**
   - 브라우저 캐시 삭제
   - 시크릿 모드에서 테스트

4. **다른 환경에서 테스트**
   - 다른 브라우저, 다른 기기에서 테스트
   - 모바일과 데스크톱에서 모두 확인

## 추가 도움이 필요한 경우

위 방법들로도 문제가 해결되지 않으면:
- 정확한 오류 메시지나 현상을 기록해두세요
- 사용 중인 브라우저와 버전 정보를 확인해두세요
- 고객 지원팀에 문의할 때 이 정보들을 함께 전달하세요
