# 블로그를 만드는 방법

하위 계정에서 Sites(사이트) 메뉴로 이동하여 블로그를 만들 수 있습니다.

블로그 생성 흐름:

[https://www.canva.com/design/DAF_ma7Cu8o/TGEAd7rp9YsKq8fpv6u4cA/edit?utm_content=DAF_ma7Cu8o&utm_campaign=designshare&utm_medium=link2&utm_source=sharebutton](https://www.canva.com/design/DAF_ma7Cu8o/TGEAd7rp9YsKq8fpv6u4cA/edit?utm_content=DAF_ma7Cu8o&utm_campaign=designshare&utm_medium=link2&utm_source=sharebutton)

1. Sites(사이트) > Blog(블로그)로 이동합니다

2. 다음 설정으로 새 블로그를 만듭니다:

- 블로그 이름 추가
- 도메인 선택 및 슬러그 추가 (선택사항)
- 설명 추가 - SEO를 위해 약 250자 정도 권장

참고 - 도메인을 반드시 추가해주세요

**a. 도메인이 abc.com인 경우:**

- 슬러그로 /blog를 지정하여 abc.com/blog로 설정할 수 있습니다. (해당 도메인 페이지가 이미 존재하는 경우, 퍼널/웹사이트의 모든 URL과 도메인을 스캔한 후 고유한 매개변수를 추가합니다)

- 블로그 글 이름은 abc.com/post/post-name 형식으로 정의됩니다

- 카테고리 URL은 abc.com/blog/category/category-name이 됩니다

- 태그 URL은 abc.com/blog/tag/tag-name이 됩니다

- 작성자 URL은 abc.com/blog/author/author-name이 됩니다

**b. 서브도메인이 blog.abc.com인 경우:**

- blog.abc.com으로 지정할 수 있으며, 슬러그를 추가할 필요가 없습니다
- 블로그 글 이름은 blog.abc.com/post-name 형식으로 정의됩니다
- 카테고리 URL은 blog.abc.com/category/category-name이 됩니다
- 태그 URL은 blog.abc.com/tag/tag-name이 됩니다
- 작성자 URL은 blog.abc.com/author/author-name이 됩니다

4. 세부 정보를 추가한 후, 새 블로그 글을 작성하고 블로그 사이트를 편집할 수 있는 목록 보기 화면이 표시됩니다.

5. 블로그 사이트 레이아웃, 목록 보기, 카드 세부 정보, 블로그 글 페이지를 편집하려면 다음을 수행할 수 있습니다:

- 레이아웃 디자인을 선택할 수 있습니다
![레이아웃 디자인 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155026313028/original/EJAJavtLHnY1f6VeCuiaMkOkfNscm0ahjw.jpeg?1716233088)

- 카드가 어떻게 표시될지 관리할 수 있습니다
![카드 모양 관리](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155026313029/original/EjZF83MMoZQacKjYBLOoytbrokh1f3r9XA.jpeg?1716233088)

- 스타일을 관리할 수 있습니다 - 이미지, 글, 태그, 제목, 타이틀, 설명, 작성자, 발행일, 작성자
