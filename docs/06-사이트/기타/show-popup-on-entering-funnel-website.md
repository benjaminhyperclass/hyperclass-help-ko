# 퍼널/웹사이트 진입 시 팝업 표시하기

이 가이드에서는 방문자가 퍼널이나 웹사이트 페이지에 도착한 직후 자동으로 나타나는 팝업을 만드는 방법을 알려드립니다. 이 기능은 방문자가 클릭하거나 스크롤하기를 기다리지 않고 즉시 주목을 끌어 전환율을 높이는 데 도움이 됩니다.

## 페이지 로드 시 표시되는 팝업이란?

페이지 로드 팝업은 방문자가 웹사이트나 퍼널 페이지에 도착한 직후, 클릭이나 스크롤 없이 자동으로 나타나는 메시지나 폼입니다. 이런 유형의 팝업은 타이밍 설정(예: 2초 지연)에 의해 트리거되며, 방문자의 주의를 즉시 끄는 데 사용됩니다. 일반적으로 시간 제한 오퍼를 홍보하거나, 리드를 수집하거나, 방문 초기에 행동을 유도하는 환영 메시지를 표시하는 데 활용됩니다.

## 페이지 로드 팝업 사용의 주요 장점

- **자동화**: 사용자가 아무것도 클릭하지 않아도 자동으로 팝업이 실행됩니다.
- **리드 생성**: 사용자가 도착하는 순간 이메일 리스트를 늘리거나 한정 시간 딜을 제공하기에 완벽합니다.
- **전환율 최적화**: 즉시 나타나는 팝업으로 시간 제한 CTA를 통해 퍼널 전환율을 높입니다.
- **행동 타겟팅**: 맞춤형 지연 타이머로 팝업이 언제 나타날지 정확히 선택할 수 있습니다.
- **광범위한 호환성**: Hyperclass 플랫폼 내 퍼널과 웹사이트 모두에서 작동합니다.

## 페이지 로드 시 팝업 표시 설정 방법

사용자가 페이지에 도착한 후 자동으로 나타나는 팝업을 설정하려면 다음 단계를 따르세요.

### 1단계: 팝업 설정으로 이동

- 하위 계정 대시보드의 왼쪽 메뉴에서 **Sites(사이트)**를 클릭합니다. 그런 다음 상단 네비게이션 바에서 퍼널(Funnels) 또는 웹사이트(Websites) 탭 중 하나를 선택합니다.

- 원하는 퍼널이나 웹사이트를 열고 팝업을 추가할 특정 단계나 페이지를 클릭합니다.

- **Edit(편집)** 파란 버튼을 클릭하여 페이지 편집기를 실행합니다.

- 웹사이트 빌더의 상단 메뉴 바에서 **Popup Settings(팝업 설정)** 아이콘을 클릭하여 팝업 설정을 엽니다.

![팝업 설정 아이콘](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046955271/original/a195Uqak0HIUOVjM8Ts8xT_xMOhgN_OkgQ.png?1747767733)

### 2단계: 팝업 설정 구성

- 오른쪽의 팝업 설정 패널에서 아래로 스크롤하여 Show Popup on 드롭다운을 찾습니다. 목록에서 **Delay(지연)**를 선택합니다.

- 드롭다운 바로 아래의 슬라이더를 사용하여 페이지 로드 후 팝업이 나타날 시간(초)을 선택합니다.

- 편집기 우상단의 **Save(저장)** 또는 **Publish(발행)** 버튼을 클릭하여 팝업 설정을 적용합니다.

## 자주 묻는 질문

**Q: 팝업을 방문당 또는 세션당 한 번만 표시하도록 설정할 수 있나요?**
아니요, 방문자가 같은 페이지에 머물러 있지 않는 한 페이지가 로드될 때마다 팝업이 나타납니다.

**Q: 누군가 페이지를 나가려 할 때 팝업을 표시하고 싶다면 어떻게 하나요?**
"Show popup on" 드롭다운에서 "Exit" 옵션을 사용하여 사용자가 커서를 브라우저 툴바 쪽으로 움직일 때 트리거되는 팝업을 만드세요.

**Q: 퍼널과 웹사이트 모두에서 이 기능을 사용할 수 있나요?**
네! 이 기능은 두 곳에서 동일하게 작동하며, 설정 과정도 정확히 같습니다.

**Q: 지연 시간이 끝나기 전에 팝업을 수동으로 열면 어떻게 되나요?**
지연 타이머가 끝나기 전에 팝업이 수동으로(예: 버튼 클릭) 트리거되면, 자동 팝업은 취소됩니다. 이렇게 하면 같은 페이지 뷰에서 동일한 팝업이 두 번 나타나는 것을 방지할 수 있습니다.
