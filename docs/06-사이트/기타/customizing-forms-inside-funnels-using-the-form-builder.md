# 퍼널 내에서 폼 빌더를 사용해 폼 커스터마이징하기

퍼널 작업 흐름을 벗어나지 않고도 폼을 커스터마이징하고 관리할 수 있습니다. 내장된 팝업 폼 빌더(Form Builder)를 사용하면 퍼널 빌더(Funnel Builder) 내에서 직접 폼을 편집할 수 있어 더욱 간소화된 경험을 제공합니다.

# **퍼널 빌더의 폼 빌더 팝업이란?**

폼 빌더(Form Builder) 팝업을 사용하면 퍼널 빌더 위에 오버레이되는 큰 모달 창을 통해 퍼널 페이지 내에서 직접 폼을 만들거나 편집할 수 있습니다. 이 설정을 통해 다른 곳으로 이동하거나 탭을 전환할 필요가 없어져 퍼널을 구축하는 동안 집중력을 유지할 수 있습니다. 폼 요소, 커스텀 필드, 스타일링 옵션, 테마까지 모든 기능을 같은 팝업 창에서 이용할 수 있습니다.

---

## **폼 빌더 팝업의 주요 장점**

내장된 에디터를 사용해 퍼널 구축 흐름을 유지하고 작업 중단을 줄일 수 있습니다:

- **퍼널 내 편집**: 퍼널 빌더 내에서 직접 폼을 수정할 수 있습니다.

- **탭 전환 없음**: 편집, 스타일링, 저장을 하나의 뷰에서 처리합니다.

- **컨텍스트 유지**: 퍼널에서의 위치가 변경되지 않습니다.

- **올인원 접근**: 필드, 스타일링, 테마를 같은 팝업에서 관리합니다.

- **빠른 전환**: 편집과 퍼널 구축 간의 로딩 시간이 단축됩니다.

---

## **팝업 폼 빌더를 사용해 폼 편집하는 방법**

폼 빌더 팝업에 접근하고 사용하는 방법을 단계별로 안내합니다:

### **퍼널로 이동하기**

왼쪽 메뉴에서 **Sites(사이트)**를 클릭한 다음 상단의 **Funnels(퍼널)** 탭을 선택합니다. 이렇게 하면 폼 요소가 포함된 퍼널 페이지를 관리하거나 만들 수 있는 퍼널 목록이 열립니다.

![퍼널 메뉴로 이동](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048935733/original/aCWv-d_I7PuDucHFcwB7hLXoogwh5ZMpaw.png?1750948570)

### 퍼널 열기 또는 만들기

**Search for Funnels(퍼널 검색)** 바를 사용해 기존 퍼널을 찾거나 **+ New Funnel(+ 새 퍼널)**을 클릭해 새로 만듭니다. 이렇게 하면 페이지를 편집하고 폼 요소를 추가할 수 있는 퍼널 작업 공간으로 이동합니다.

![퍼널 검색 및 생성](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048935756/original/qNDP0mdVai3YEpmvzT7GFlys4Mjb8LKH4w.png?1750948591)

### **퍼널 단계 선택하기**

**Steps(단계)** 탭을 클릭해 선택한 퍼널 내의 모든 퍼널 단계를 확인합니다. 여기서 폼을 추가하거나 편집할 특정 단계를 선택합니다.

![퍼널 단계 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048935775/original/7NsC41QzCA9JT3NfneeVWkVQg7dSDve2dw.png?1750948620)

### **페이지 에디터 열기**

퍼널 단계 미리보기 아래의 **Edit(편집)** 버튼을 클릭해 페이지 편집을 시작합니다. 기본 설정을 업데이트하려면 **Edit page details(페이지 세부사항 편집)**을, 전체 페이지 편집을 위해 별도 탭에서 퍼널 페이지 빌더를 열려면 **Edit in a New Tab(새 탭에서 편집)**을 선택할 수 있습니다.

![페이지 에디터 열기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048935818/original/53D4_SCRwk5exbKD0OsxudpPLtjRQdh2Gg.png?1750948647)

### 폼 요소 추가하기

상단 툴바의 **+** 아이콘을 클릭해 사이드바를 연 다음, 메뉴에서 **Elements(요소)**를 선택합니다. **Form(폼)** 섹션으로 스크롤해서 **Form(폼)** 요소를 페이지로 드래그하여 설정을 시작합니다.

![폼 요소 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048935841/original/H2kN5ygX6ZYaqOIyRUyvbo-HJYJqwH1TzA.png?1750948678)

### 기존 폼 선택하거나 새 폼 만들기

**Form Picker(폼 선택기)**가 나타나면 드롭다운을 사용해 기존 폼 중에서 선택합니다.

새 폼을 만들고 싶다면 **"Click Here(여기를 클릭)"** 링크를 클릭하세요. 이렇게 하면 **Sites(사이트)** 메뉴 내의 **Forms(폼)** 하위 섹션이 열리고, 여기서 새 폼을 만들고 저장할 수 있습니다.

![폼 선택하거나 새로 만들기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048935876/original/R8QYx9u0QiPzhOmq7qWG39-dG7IojhNBpg.png?1750948709)

### 폼 편집 옵션

폼을 선택한 후 오른쪽 패널에 표시된 **Edit form(폼 편집)** 버튼을 사용해 폼 빌더를 엽니다.

이렇게 하면 퍼널 빌더 바로 안에 큰 팝업 창이 열려서 퍼널을 벗어나지 않고도 폼 필드, 스타일, 설정을 커스터마이징할 수 있습니다.

![폼 편집 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048935890/original/5ILY-yEugwqvNoYkDQROx8OgLGh1aIycqQ.png?1750948732)

### **퍼널 페이지 내에서 폼 커스터마이징**

팝업에서 폼 빌더가 열리면 **Quick Add(빠른 추가)**와 **Custom Fields(커스텀 필드)** 패널을 사용해 폼을 설정합니다:

- **Quick Add(빠른 추가):** 성명, 이메일, 전화번호 같은 일반적인 필드를 폼에 즉시 드래그할 수 있습니다.

- **Custom Fields(커스텀 필드):** 리드 수집 전략에 맞는 커스텀 입력 유형을 추가합니다. 개인화되거나 캠페인별 데이터 수집에 이상적입니다.

이 편집 경험은 퍼널 페이지를 백그라운드에 유지해서 정확한 폼 업데이트를 하면서도 디자인 컨텍스트를 유지할 수 있도록 도와줍니다.

![퍼널 페이지 내에서 폼 커스터마이징](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048935936/original/WX3Nh1EHzcocgumGM_NwlrW8XB5V2DwrqQ.png?1750948771)

### 변경사항 저장하기

폼 편집이 완료되면 팝업 오른쪽 상단의 **Save(저장)** 버튼을 클릭합니다. 이렇게 하면 업데이트가 즉시 적용되고 모달이 닫히면서 페이지 새로고침이나 탭 전환 없이 퍼널 빌더의 정확한 위치로 돌아갑니다. 이를 통해 퍼널 구축 흐름이 중단되지 않는 매끄러운 편집 경험을 보장합니다.

![변경사항 저장](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048935962/original/-v3fFeR8m0BUpOHQHdxTvEsvW9_gWUs3fw.png?1750948800)

## **자주 묻는 질문**

**Q: 기존 폼이나 퍼널에 영향을 주나요?**

아니요, 폼과 퍼널은 변경되지 않습니다. 팝업은 단지 접근하고 편집하는 방식만 바꿉니다.

**Q: 팝업에서 모든 폼 설정에 접근할 수 있나요?**

네, 모달 내에서 직접 필드, 스타일, 테마를 커스터마이징할 수 있습니다.

**Q: 저장하지 않고 팝업을 닫으면 어떻게 되나요?**

저장하지 않은 변경사항은 모두 사라집니다. 닫기 전에 반드시 저장하세요.

**Q: 모든 사용자가 사용할 수 있나요?**

네, 퍼널 빌더와 폼 빌더에 접근할 수 있는 사용자는 내장된 팝업 인터페이스를 볼 수 있습니다.

**Q: 이 인터페이스에서 새 폼을 만들 수 있나요?**

네, 팝업 내에서 새 폼 생성과 편집 모두 지원됩니다.
