## **블로그 RSS 피드란?**

RSS는 Really Simple Syndication의 줄임말입니다. 중요한 뉴스와 정보를 쉽게 받아볼 수 있는 방법으로, 웹사이트를 직접 방문하거나 검색하지 않고도 새로운 정보를 확인할 수 있게 해줍니다. 블로그 RSS 링크는 이메일 캠페인(Email Campaign)으로 뉴스레터를 발송하거나, 소셜 플래너(Social Planner)의 RSS 기능을 활용한 게시물 자동화에 사용할 수 있습니다.

---

## 블로그 RSS 피드 사용의 주요 장점

RSS 피드를 워크플로우에 통합했을 때의 핵심 장점들을 확인해보세요. 효율성과 콘텐츠 도달 범위를 크게 향상시킬 수 있습니다.

- **콘텐츠 배포 자동화**: 이메일과 소셜 미디어 채널로 자동 전송
- **시간 절약**: 수동 게시 작업 불필요
- **콘텐츠 신선도 유지**: 바쁠 때도 일관된 콘텐츠 제공
- **다중 채널 지원**: 이메일 캠페인과 소셜 플래너 게시 모두 활용 가능

## 블로그 RSS 피드 설정 방법

Hyperclass 블로그에서 직접 RSS 피드 링크를 생성하는 방법을 알아보세요. 수동 업데이트 없이도 이메일 캠페인이나 소셜 미디어 자동화를 위해 콘텐츠를 원활하게 배포할 수 있습니다.

**중요한 사전 준비사항:** 도메인이 Hyperclass 계정에 연결되어 있어야 합니다.

## **자주 묻는 질문**

**Q: RSS 생성 후 드롭다운에서 블로그를 변경하면 어떻게 되나요?**
A: 원본 URL은 기능을 유지하며 드롭다운 변경 후에도 계속해서 피드를 생성합니다.

**Q: 게시글 수에 제한이 있나요?**
A: 네, 현재 인터페이스에서 최대 개수가 설정되어 있습니다.

**Q: RSS에서 초안(Draft) 글도 가져올 수 있나요?**
A: 아니요, 발행된(Published) 게시글만 RSS 피드에 나타납니다.

**Q: 피드 길이를 변경해야 할 경우 기존 RSS 설정을 편집할 수 있나요?**
A: 블로그 설정(Blog Settings)의 업데이트된 UI를 사용하여 피드 설정을 쉽게 관리하고 편집할 수 있습니다.
