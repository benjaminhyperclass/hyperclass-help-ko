# 클릭 투 콜, 클릭 투 텍스트, 클릭 투 이메일 링크와 버튼이 뭔가요?

전화 걸기, 문자 메시지, 이메일을 즉시 시작할 수 있는 클릭 가능한 링크는 사용자가 바로 행동을 취하기 쉽게 만들어줍니다. 올바르게 설정하면, 이런 링크를 클릭할 때 사용자의 전화 다이얼러, SMS 앱, 또는 기본 이메일 클라이언트가 미리 입력된 정보와 함께 열립니다. 이런 링크는 퍼널과 웹사이트에서 사용자 경험을 개선하고 전환율을 높이는 데 자주 사용됩니다.

---

## 클릭 투 액션 사용의 주요 장점

클릭 투 링크는 리드가 한 번의 탭으로 쉽게 연락할 수 있도록 하여 커뮤니케이션을 단순화합니다.

- **원탭 커뮤니케이션**: 사용자가 복사하거나 입력할 필요 없이 즉시 전화, 문자, 이메일을 보낼 수 있습니다
- **모바일 참여도 향상**: 모바일 사용자는 클릭 가능한 액션과 상호작용할 가능성이 더 높습니다
- **리드 시간 절약**: 전화번호, 메시지 내용, 이메일 내용 등을 자동으로 채워줍니다
- **자동화 트리거 준비**: 후속 워크플로우(Workflow)를 위해 트리거 링크(Trigger Links)와 결합할 수 있습니다
- **유연한 CTA 디자인**: 클릭 투 링크는 텍스트와 버튼 요소(Elements) 모두에 추가할 수 있습니다

## 클릭 투 콜 링크 추가하는 방법

사람들이 클릭/탭하여 전화를 걸 수 있는 링크를 만들려면:

- 퍼널이나 웹사이트 빌더(Page Builder)에서 텍스트를 하이라이트하세요
- 링크 아이콘을 클릭하세요
- 다음 형식으로 링크를 입력하세요: tel:+1234567890
- 저장(Save)을 클릭하고 모바일 기기에서 미리보기/테스트하세요

## 자주 묻는 질문

**Q: 이런 링크의 클릭을 어떻게 추적할 수 있나요?**
트리거 링크를 사용하여 클릭 기반 추적 및 자동화 액션을 설정하세요.

**Q: 사용자의 기기가 클릭 투 링크를 지원하지 않으면 어떻게 되나요?**
기기가 해당 프로토콜(예: 데스크톱에서 sms:)을 지원하지 않으면 클릭해도 아무 일이 일어나지 않습니다. 다양한 기기에서 링크를 테스트하고 대체 연락 정보 제공을 고려하는 것이 좋습니다.

**Q: 하나의 링크에서 여러 액션(전화와 문자 등)을 추가할 수 있나요?**
아니요. 각 링크는 한 번에 하나의 액션(전화, 문자, 이메일 중 하나)만 수행할 수 있습니다. 여러 옵션을 제공하려면 각 액션별로 별도의 링크나 버튼을 사용하세요.

**Q: 이미지나 아이콘 안에서 클릭 투 링크를 사용할 수 있나요?**
네. 퍼널이나 웹사이트 빌더에서 같은 형식을 사용해 이미지나 아이콘에 하이퍼링크를 걸 수 있습니다. 전화나 메시지 아이콘과 같은 시각적 CTA에 유용합니다.
