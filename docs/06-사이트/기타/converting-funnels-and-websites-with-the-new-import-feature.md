# 새로운 가져오기 기능으로 퍼널과 웹사이트 변환하기

## **개요**

기존 웹사이트나 퍼널 페이지를 새로 만드는 퍼널/웹사이트로 변환하거나 가져오고 싶으셨나요? 새로운 가져오기 기능을 사용하면 이제 즉석에서 이 작업을 수행할 수 있습니다.

![가져오기 기능 데모](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48189609541/original/9FHdr4r9qAuPXa6Ez2WI1JEPi5IrWLWIyA.gif?1644648564)

## **어떻게 작동하나요?**

**1단계**: "**+ 새 단계 추가(+ Add New Step)**" 버튼을 클릭하여 퍼널이나 웹사이트에 새 페이지를 추가합니다.

**2단계**: 팝업이 나타나면 페이지 이름과 경로를 입력합니다.

**3단계**: "퍼널 단계 생성(Create Funnel Step)"을 클릭하여 팝업을 닫습니다.

**4단계**: 퍼널 미리보기 영역에 새 모달 창이 나타나면 "가져오기(Import)" 버튼을 클릭합니다.

**5단계**: 가져올 페이지를 선택하고 "**가져오기(import)**"를 클릭합니다.

![가져오기 페이지 선택 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48189609496/original/EgUlJmYTcXPipvScV9pAuhcBPh2t1LqdzA.png?1644648509)

### 주의사항:

- **퍼널/웹사이트를 가져오거나 변환하려면 먼저 새 단계를 추가해야 합니다**

- **퍼널/웹사이트 가져오기 기능은 한 번에 하나의 퍼널 단계 또는 웹페이지만 가져옵니다**

- **Hyperclass 내의 기존 퍼널 단계나 웹사이트 페이지만 가져올 수 있습니다**

- **다른 웹사이트나 퍼널 제공업체에서는 가져올 수 없습니다**

- **가져온 단계/페이지를 변경해도 원본 단계/페이지에는 영향을 주지 않습니다**

- **가져오기 기능은 원본 퍼널 단계에 설정된 상품이나 누적된 통계를 가져오지 않습니다. 가져온 후 상품을 다시 설정해야 합니다**

- **퍼널의 컨트롤 또는 변형 단계도 가져올 수 있습니다**

- **웹사이트에서 분할 테스트를 실행하려면 해당 페이지를 퍼널로 가져와서 거기서 분할 테스트를 실행하세요**

- 분할 테스트는 모든 플랜에서 퍼널 기능으로 제공됩니다
