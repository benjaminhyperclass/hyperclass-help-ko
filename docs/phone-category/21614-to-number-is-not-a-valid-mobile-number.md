# LeadConnector

**오류 코드: 21614**

---

**오류 설명**

'받는 사람' 번호가 유효하지 않은 휴대폰 번호로 SMS를 전송하려고 시도했습니다. 지정한 번호가 유선전화번호이거나 유효하지 않은 번호일 가능성이 높습니다.

**가능한 원인**

- 제공한 번호가 유선전화번호일 수 있습니다.
- 제공한 번호가 유효하지 않거나 형식이 올바르지 않을 수 있습니다.
- 사물인터넷(IoT) 또는 M2M(기기 간 통신) 번호로 SMS를 전송하려는 경우, 해당 번호들이 Lead Connector의 번호 검증 시스템에 아직 추가되지 않은 비표준 형식을 사용할 수 있습니다.

**해결 방법**

- Lookup API를 사용하여 전송하려는 번호가 유선전화가 아닌지 확인하세요.
- 올바른 E.164 형식의 유효한 휴대폰 번호를 제공했는지 확인하세요.
- IoT 또는 M2M 번호로 SMS를 전송하려는 경우, 해당 번호 형식이 해당 국가나 지역의 표준 휴대폰 번호와 다른지 확인하세요. 이러한 번호들은 종종 추가 자릿수나 특이한 형식을 가지고 있어 Lead Connector의 번호 검증을 통과하지 못할 수 있습니다. 이것이 문제라고 생각되면 고객지원팀에 도움을 요청하세요.
