# 오류 코드: 21211 - 잘못된 수신 전화번호

[](https://www.twilio.com/docs/api/errors/21211#error-21211)

---

**설명**

발신 전화를 걸거나 메시지를 보내려고 했지만, 입력한 수신 전화번호가 유효하지 않거나 형식이 잘못되었습니다. 플랫폼에서는 [E164](https://en.wikipedia.org/wiki/E.164) 형식의 전화번호만 허용합니다: [+] [국가 코드] [지역번호를 포함한 가입자 번호].

**가능한 원인**

입력한 수신 번호의 형식이 잘못되었습니다.

- LeadConnector 번호에서 같은 번호로 메시지를 보내거나 전화를 걸려고 했습니다 (**To**와 **From** 파라미터에 동일한 LeadConnector 번호를 입력).
- 단축 번호(Short Code)로 메시지를 보내려고 했습니다.
- 영숫자 발신자 ID로 메시지를 보내려고 했습니다.
- 메시징 서비스를 사용하여 잘못된 접두사가 포함된 **To** 번호로 메시지를 보내려고 했습니다.

**해결 방법**

- 전화번호를 E164 형식으로 올바르게 입력했는지 확인하세요: [+] [국가 코드] [지역번호를 포함한 가입자 번호].
- 전화를 걸려는 번호에 올바른 [국가 통화 코드](https://en.wikipedia.org/wiki/List_of_telephone_country_codes#Alphabetical_listing_by_country_or_region)를 사용했는지 확인하세요.
- LeadConnector 번호에서 동일한 번호로 전화를 걸거나 메시지를 보내려고 하지 않는지 확인하세요.
- 단축 번호나 영숫자 발신자 ID로 메시지를 보내려고 하지 않는지 확인하세요. 이러한 발신자 유형은 LeadConnector 번호로부터 메시지를 받을 수 없습니다.
- 'To' 번호에 오타나 잘못된 입력이 있는지 확인하세요.
- 메시징 서비스를 사용하여 메시지를 보낼 때, 잘못된 접두사가 포함된 To 번호 대신 접두사를 제거하고 E164 형식을 사용하여 직접 보내세요: [+] [국가 코드] [지역번호를 포함한 가입자 번호]. To 번호가 RCS 메시지를 받을 수 있고 메시징 서비스 발신자 풀에 RCS 에이전트가 있는 경우, 해당 To 번호로 보내기 위해 선택됩니다.
