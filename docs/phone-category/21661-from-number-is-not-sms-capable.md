# 오류 21661: 발신 번호가 SMS 발송 불가

**오류 코드: 21661**

## 오류 설명

전화번호는 SMS 발송이 가능해야 합니다. 계정의 전화번호 페이지를 방문하여 보유 중인 전화번호 목록을 확인하고 SMS 발송 가능 여부를 확인하세요.

![SMS 발송 불가 오류 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155045240572/original/oOtOWoEkOWqRQBqpDa5SEkFiDq3fjzRXMw.png?1744870299)

## 발생 원인

사용 중인 전화번호가 메시지 발송 기능을 지원하지 않습니다.

## 해결 방법

- SMS 기능이 지원되는 Lead Connector 전화번호를 사용하고 있는지 확인하세요. `Settings(설정) → Phone Numbers(전화번호) → Manage Number(번호 관리) → +Add Number(번호 추가)`에서 구매한 전화번호와 지원 기능을 확인할 수 있습니다.

- `Settings(설정) → Phone Numbers(전화번호) → Manage Number(번호 관리) → +Add Number(번호 추가)`에서 SMS 발송이 가능한 번호를 구매하세요.
