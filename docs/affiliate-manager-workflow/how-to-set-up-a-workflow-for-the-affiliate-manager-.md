# 제휴 관리를 위한 워크플로우 설정 방법

워크플로우는 제휴 관리를 효율적으로 수행하기 위한 자동화 액션을 제공합니다. 연락처를 제휴사 목록에 추가하고, 상태를 업데이트하며, 캠페인에 추가하거나 제거하는 기능이 포함됩니다. 새로운 Affiliate Created 트리거도 제공되어 제휴 관리를 더욱 체계적이고 효율적으로 만들어줍니다.

## 제휴 관리 워크플로우 기능이란?

제휴 관리(Affiliate Manager)에서 이제 제휴사를 효율적으로 관리하는 데 도움이 되는 여러 자동화 액션과 트리거를 지원합니다. 워크플로우에는 네 가지 단계가 포함됩니다: 연락처를 제휴사 목록에 활성 상태로 추가하기, 제휴사 상태를 활성 또는 비활성으로 업데이트하기, 제휴사를 캠페인에 추가하기, 제휴사를 캠페인에서 제거하기입니다.

또한 제휴 관리에는 이제 'Affiliate Created'라는 트리거가 추가되어, 제휴사가 생성될 때 자동으로 활성화됩니다. 이 새로운 자동화 기능은 제휴 관리를 간소화하고 체계적으로 만들어 더욱 효율적인 제휴 관리를 가능하게 합니다.

### 새로운 워크플로우 액션과 트리거는 누구에게 유용한가요?

제휴 관리의 새로운 자동화 기능은 제휴 네트워크를 관리하는 모든 사람에게 유용합니다. 예를 들어 제휴 관리자, 마케팅 팀, 사업 소유자들이 해당됩니다. 이 기능들은 제휴 관리를 간소화하여 다양한 작업을 자동화하고, 제휴사가 적절히 분류되어 올바른 캠페인에 배정되도록 도와줍니다.

### 이 기능들의 장점은 무엇인가요?

제휴 관리의 새로운 자동화 기능은 다음과 같은 여러 장점을 제공합니다:

**효율성 증대**: 자동화를 통해 제휴 관리를 간소화하여 시간과 리소스를 절약할 수 있습니다.

**체계적 관리**: 새로운 기능으로 제휴사를 더욱 체계적이고 구조적으로 관리할 수 있어, 각 제휴사가 적절히 분류되고 올바른 캠페인에 배정되도록 합니다.

**정확성 향상**: 자동화를 통해 오류를 줄이고 제휴 관리의 정확성을 높여, 적시에 올바른 작업이 수행되도록 합니다.

**매출 증대**: 다양한 작업을 자동화함으로써 비즈니스의 더 중요한 측면에 집중할 수 있어 잠재적으로 매출과 수익성을 증대시킬 수 있습니다.

## 제휴 관리 워크플로우 액션과 트리거 설정 방법

Automation(자동화) > Workflows(워크플로우) > Create Workflow(워크플로우 생성)으로 이동하세요.

![제휴 관리 워크플로우 생성](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48290942176/original/gNfhyS5hJSk6-c-pTG_OapbvQutIn7cLGA.png?1680598073)

Start from Scratch(처음부터 시작) > Create New Workflow(새 워크플로우 생성)을 클릭하세요. (워크플로우가 아직 없는 경우)

![새 워크플로우 생성](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48290942177/original/PciQXJ-M4TDrioRRmgYxBiYn4RjE8PtO2Q.png?1680598073)

### Affiliate Created 워크플로우 트리거

Add new Workflow Trigger(새 워크플로우 트리거 추가)를 클릭하고 > Affiliate Created를 검색 > 선택하세요.

![Affiliate Created 트리거 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48290942174/original/tvurjdZCORKlRGFFgucD_VvpsA3UeRB9rw.png?1680598073)

Save Trigger(트리거 저장)를 클릭하세요.

![트리거 저장](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48290942175/original/tauJYfNADRGRCZ9HwPw3oSvDqhXbTIBbXQ.png?1680598073)

이 워크플로우 트리거는 해당 하위 계정에서 제휴사가 생성될 때 실행됩니다.

### 제휴 관리 워크플로우 액션

워크플로우 액션의 제휴 관리 섹션에는 네 가지 액션이 제공됩니다.

![제휴 관리 워크플로우 액션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48290942172/original/nDkPdxo1axAp6iMAS9iddghH7InwpPl9Ow.png?1680598072)

#### 제휴 관리자에 추가 (Add to Affiliate Manager)

연락처를 제휴 관리자의 제휴사 목록에 기본적으로 활성 상태로 추가합니다.

![제휴 관리자에 연락처 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48290942169/original/NY0fdWk7R1kG3j2c65pDQBcFRM3sCothog.png?1680598071)

#### 제휴사 업데이트 (Update Affiliate)

제휴사를 활성 또는 비활성 상태로 업데이트합니다.

지원되는 옵션:
- Mark as Active (활성으로 표시)
- Mark as Inactive (비활성으로 표시)

![제휴사 상태 업데이트](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48290942173/original/eI_gtrNaOWX1uaZ5qpyB7635VvQbsUjWSw.png?1680598073)

#### 제휴 캠페인에 추가 (Add to Affiliate Campaign)

제휴 관리자 > 캠페인 탭의 캠페인에 제휴사를 추가합니다. 드롭다운에서 원하는 제휴 캠페인을 선택할 수 있습니다.

![제휴 캠페인에 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48290942166/original/os-Smcjml_BrAJWzT4qw95ZJ1nsJHpV90g.png?1680598071)

#### 제휴 캠페인에서 제거 (Remove from Affiliate Campaign)

캠페인에서 제휴사를 제거합니다. 드롭다운을 사용하여 제거할 캠페인을 선택할 수 있습니다.

![제휴 캠페인에서 제거](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48290942171/original/OijsACD3u_CDUuGlMPqOJjgweru6WZn21g.png?1680598072)
