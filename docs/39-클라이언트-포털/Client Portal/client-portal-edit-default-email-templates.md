# 클라이언트 포털의 기본 이메일 템플릿 편집이란?

기본 이메일 템플릿 편집 기능을 사용하면 클라이언트 포털 소유자가 포털 앱(예: 문서 및 계약서(Documents & Contracts), 강의(Courses), 수료증(Certificates))에서 사용하는 시스템 기본 이메일의 편집 가능한 사본을 생성할 수 있습니다. 처음부터 새로 만드는 대신, 기본값을 변환하고 콘텐츠를 맞춤화한 다음 해당 알림의 활성 템플릿으로 설정할 수 있습니다.

## **기본 이메일 템플릿 편집의 주요 이점**

- 속도: 기본값을 편집 가능한 템플릿으로 즉시 변환—처음부터 다시 만들 필요가 없습니다.

- 일관성: 원래 레이아웃을 유지하면서 대상에 맞게 톤과 콘텐츠를 다듬을 수 있습니다.

- 개인화: 병합 필드/커스텀 값을 사용하여 동적 세부 정보를 추가할 수 있습니다.

- 정확성: 커스텀 필드로 미리보기하여 연락처가 정확히 무엇을 보게 될지 알 수 있습니다.

- 제어: 지원되는 클라이언트 포털 앱 전체에서 알림별 편집을 적용할 수 있습니다.

## **기본 이메일 템플릿 편집 설정 방법**

명확한 설정 순서를 따르면 오류를 줄이고 맞춤화된 템플릿이 의도한 알림에 대해 활성화되도록 할 수 있습니다.

- Sites(사이트) → Client Portal(클라이언트 포털) → Settings(설정) → Email Settings(이메일 설정)으로 이동하세요.

![클라이언트 포털 이메일 설정으로 이동](https://jumpshare.com/share/IaruuKABWh6R9nVuUjnX+/Screenshot+2025-12-05+at+6.22.02%E2%80%AFPM.png)

- 또는 Memberships(멤버십) → Client Portal(클라이언트 포털) → Settings(설정) → Email Settings(이메일 설정)으로 이동하세요.

![멤버십을 통한 이메일 설정 접근](https://jumpshare.com/share/fQMnDfMSVjHNux1iX93F+/Screenshot+2025-12-05+at+6.29.05%E2%80%AFPM.png)

- Main Notification Features(주요 알림 기능)(Client Portal(클라이언트 포털), Communities(커뮤니티), Certificates(수료증), Courses(강의))를 찾으세요.

![주요 알림 기능 찾기](https://jumpshare.com/share/pRUHX38LQI3CGS1kS4KC+/Screen+Shot+2025-12-05+at+6.40.49+PM.png)

- 주요 기능 아래의 각 하위 카테고리 알림에서 눈 아이콘을 클릭하여 템플릿을 미리보고 Default Template(기본 템플릿) 버튼을 클릭하여 맞춤화를 시작하세요.

![템플릿 미리보기 및 편집 시작](https://jumpshare.com/share/kDAC8pMKRH6UG177SFP2+/GIF+Recording+2025-12-05+at+6.42.53+PM.gif)

- **Select Template(템플릿 선택)** 드롭다운을 클릭하여 이전에 저장한 이메일 템플릿을 선택할 수도 있습니다.

![이전에 저장한 템플릿 선택](https://jumpshare.com/share/LA7CTwCSjbYJnHXU5eQ8+/Screenshot+2025-12-05+at+7.22.29%E2%80%AFPM.png)

- 기본 템플릿 또는 기존 템플릿을 선택한 후 **Edit(편집)** 버튼을 클릭하여 편집기에서 이메일 콘텐츠를 맞춤화하세요.

![이메일 템플릿 편집](https://jumpshare.com/share/vg1vsjMeiQOCIjDWusUA+/Screen+Shot+2025-12-05+at+7.50.42+PM.png)

- 주의사항을 읽고 **Proceed(진행)**을 클릭하세요.

![진행하기 전 주의사항 확인](https://jumpshare.com/share/IoMsgYXd0b1p9Ljitlve+/Screen+Shot+2025-12-05+at+7.51.18+PM.png)

- **Email Builder(이메일 빌더)**를 사용하여 원하는 대로 이메일 템플릿을 편집하세요. (이메일 생성 및 편집 방법을 자세히 알아보려면 [이메일 빌더 가이드]([how-to-use-the-email-builder-and-its-in-line-editor](how-to-use-the-email-builder-and-its-in-line-editor.md))를 확인하세요)

![이메일 빌더로 템플릿 편집](https://jumpshare.com/share/hqvnlGAChztL82T65RiE+/Screen+Shot+2025-12-05+at+7.53.12+PM.png)

- 템플릿을 저장하세요.

- 저장된 템플릿이 자동으로 알림에 선택됩니다. **Save(저장)**를 클릭하여 선택을 확인하세요.

![템플릿 저장 및 적용 확인](https://jumpshare.com/share/a7RyAFDQ6X42jPCRXZwF+/GIF+Recording+2025-12-05+at+7.57.07+PM.gif)

## 자주 묻는 질문

**Q. 클라이언트 포털에서 기본 이메일 템플릿을 편집할 수 있는 사람은 누구인가요?**
**Email Settings(이메일 설정)**에 접근할 수 있는 클라이언트 포털 소유자입니다. 역할 기반 가시성은 계정 구성에 따라 달라질 수 있습니다.

**Q. 맞춤 사본을 저장하면 즉시 활성화되나요?**
네. 저장한 후에는 해당 특정 알림에 맞춤화된 버전이 사용됩니다.

**Q. 기능별로(예: 강의, 수료증, 문서 및 계약서) 템플릿을 관리할 수 있나요?**
네. 각각 지원되는 기능은 템플릿 컨트롤과 함께 **Email Settings(이메일 설정)**에 알림을 나열합니다.

**Q. 이전에 만든 템플릿을 선택할 수 있나요?**
계속 사용할 수 있습니다. 이전에 저장한 이메일 템플릿으로 알림의 기본값을 대체할 수 있습니다.

**Q. 기본 템플릿으로 다시 전환할 수 있나요?**
네.
