## 강의(Courses) - PWA에서 과제 기능 지원

**모바일 과제 기능:**

더 이상 책상에만 묶여있을 필요 없어요! 최신 업데이트로 이제 모바일 기기에서도 과제 관리를 할 수 있습니다. 공원에서 여유롭게 있을 때든, 교통 체증에 갇혔을 때든(물론 승객으로서요), 과제 관리가 그 어느 때보다 쉬워졌습니다.

- 모바일에서 과제 기능을 이용할 수 있어요. 모바일 브라우저와 PWA를 통해 접근 가능합니다.
- 최근 과제 제출 내역을 빠르게 확인하세요!
- 강사 피드백도 간편하게 살펴볼 수 있어요!

**모든 테마의 데스크톱 과제 UI 최적화:**

- 새롭게 개선된 확인 메시지 토글과 모바일 기기용 세련된 디자인을 경험해보세요!
- 과제 지시사항과 제출한 답안 및 파일 사이에 깔끔한 간격을 만들었습니다.
- 이전/다음 버튼의 성능을 향상시켜 원활한 게시물 탐색이 가능해졌습니다!
