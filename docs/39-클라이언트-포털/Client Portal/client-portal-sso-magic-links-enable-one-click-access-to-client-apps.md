# 클라이언트 포털 SSO 매직 링크: 클라이언트 앱에 원클릭 접근 제공

이 가이드에서는 SSO(Single Sign-On) 매직 링크를 생성하고 사용하여 로그인 정보 없이도 고객이 강의, 커뮤니티, 제휴 등 개별 클라이언트 포털 앱에 즉시 안전하게 접근할 수 있는 방법을 설명합니다. 또한 이러한 링크를 클라이언트 포털의 커스텀 메뉴 항목으로 삽입하고, 접근 권한을 제어하며, 보다 전문적인 고객 경험을 위해 네비게이션을 최적화하는 방법도 알아보겠습니다.

---

# **SSO 매직 링크란 무엇인가요?**

SSO 매직 링크는 사용자가 로그인 정보를 입력하지 않고도 클라이언트 포털 앱에 직접 접근할 수 있게 해주는 고유한 인증 URL입니다. 고객이 개인화된 링크를 클릭하면 즉시 로그인되어 강의 대시보드, 커뮤니티 영역, 제휴 포털 등 정확한 목적지로 이동합니다.

이 링크는 앱별, 연락처별로 생성되므로 각 사용자는 자신에게 부여된 정확한 포털 경험에 안전하게 접근할 수 있습니다.

---

## **SSO 매직 링크의 주요 장점**

SSO 매직 링크는 마찰을 줄이고, 시간을 절약하며, 고객 만족도를 높이도록 설계되었습니다. 주요 장점은 다음과 같습니다:

- **원클릭 로그인:** 사용자명이나 비밀번호 입력 없이 즉시 접근
- **빠른 온보딩:** 고객이 앱 경험을 바로 시작할 수 있음
- **지원 요청 감소:** 비밀번호 분실 등 로그인 관련 문제 해결
- **향상된 네비게이션:** 더 부드럽고 직접적인 로그인 경험 제공
- **멀티 앱 편의성:** 강의 + 커뮤니티 + 제휴 등 여러 앱을 포함한 설정에 이상적
- **높은 참여도:** 쉬운 접근으로 재방문 및 지속적인 상호작용 촉진

## 매직 링크 생성 및 사용 방법

매직 링크는 클라이언트 포털 대시보드에서 생성할 수 있으며, 특정 앱에 직접 접근할 수 있도록 연락처와 공유할 수 있습니다.

### **사이트(Sites) 대시보드 접근**

왼쪽 메뉴에서 **Sites(사이트)** 탭으로 이동하여 클라이언트 포털 대시보드에 접근하세요. 이 영역에서는 고객 대상 앱을 관리하고 안전한 로그인 경험을 생성할 수 있습니다.

![사이트 메뉴 접근](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050728644/original/QwcVGFD8E9HS47bLH5NfFM_Q8M_2iarrwg.png?1753961509)

### **클라이언트 포털(Client Portal) 섹션 열기**

Sites 메뉴에서 상단의 **Client Portal(클라이언트 포털)** 드롭다운을 클릭하고 **Dashboard(대시보드)**를 선택하세요. 이를 통해 매직 링크 생성 및 앱 설정과 같은 클라이언트 포털 관리 도구에 접근할 수 있습니다.

![클라이언트 포털 대시보드](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050728791/original/Cn3Kztkto9dJKAMAwRmhihuvouQy-fg6eA.png?1753961576)

### **매직 링크 "Generate(생성)" 클릭**

**Actions(액션)** 패널에서 "Generate Magic Link" 옆의 **Generate(생성)** 버튼을 클릭하세요. 특정 연락처와 앱에 대한 안전한 원클릭 로그인을 생성하는 과정이 시작됩니다.

![매직 링크 생성 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050729026/original/7RdLEyS0IepSQCqkcIAesQXxtNWZbxNEDQ.png?1753961741)

### **매직 링크 유형 선택**

클라이언트 포털 네비게이션을 위한 재사용 가능한 매직 링크를 생성하려면 **Custom Menu(커스텀 메뉴)**를 선택하세요. 이 유형의 링크는 포털의 사이드바에 직접 삽입하기에 이상적입니다.

![매직 링크 유형 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050729114/original/RGZhCO315eahK9CuvUxUxDtq_a4R-kzBug.png?1753961860)

### **앱 선택**

**Select App(앱 선택)** 드롭다운을 사용하여 매직 링크가 열어야 할 클라이언트 포털 앱을 선택하세요—**Courses(강의)**, **Affiliates(제휴)**, 또는 **Communities(커뮤니티)**. 이것이 링크 사용 시 도착할 목적지를 결정합니다.

![앱 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050729176/original/aiLwGjt_kEl898qQef_RT2IWNTZPIEWngQ.png?1753961925)

### **매직 링크 생성**

앱을 선택한 후 **Generate(생성)** 버튼을 클릭하세요. 삽입하거나 공유할 수 있는 안전한 원클릭 로그인 URL이 즉시 생성됩니다.

![매직 링크 생성 실행](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050729267/original/au6OxawdM0mQP0A4JztT_lqP5rWhywZ8jA.png?1753961997)

### **매직 링크 복사**

링크가 생성되면 **복사 아이콘**을 클릭하여 클립보드에 저장하세요. 이제 이메일이나 커스텀 메뉴 항목 등 필요한 곳에 이 안전한 SSO 링크를 붙여넣어 원활한 고객 접근을 제공할 수 있습니다.

![매직 링크 복사](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050729400/original/wtBufc4AaGmWKi4-RSAZuiwv5rXWxrPUYA.png?1753962096)

## **매직 링크를 커스텀 메뉴 링크로 사용하는 방법**

더 빠른 접근을 위해 매직 링크를 클라이언트 포털 내부의 **커스텀 메뉴 항목**으로 삽입할 수 있습니다.

매직 링크를 메뉴 항목으로 추가하려면:

### 에이전시 뷰로 전환

하위 계정 대시보드에서 계정명 옆의 드롭다운을 클릭하고 **Switch to Agency View(에이전시 뷰로 전환)**를 선택하세요. 이를 통해 모든 하위 계정에 대한 커스텀 메뉴 링크 같은 전역 설정을 관리할 수 있습니다.

![에이전시 뷰 전환](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050729573/original/I53tUsI43Rfc2aAhDAV7E0xPTKCJjsGNTA.png?1753962224)

### **에이전시 설정(Agency Settings) 접근**

에이전시 뷰에서 왼쪽 사이드바 하단으로 스크롤하여 **Settings(설정)**를 클릭하세요. 여기에서 커스텀 메뉴 링크 추가를 포함한 플랫폼 전체 환경설정을 구성할 수 있습니다.

![에이전시 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050729712/original/Ib3m497jMk0ErzzdxpRTbkeuQL7uFAVOWQ.png?1753962303)

### **커스텀 메뉴 링크(Custom Menu Links)로 이동**

**Settings(설정)** 패널에서 스크롤을 내려 **Custom Menu Links(커스텀 메뉴 링크)**를 클릭하세요. 이 섹션에서는 기존 메뉴 링크를 모두 표시하며, 에이전시와 하위 계정을 위한 새로운 커스텀 링크를 관리하거나 생성할 수 있습니다.

![커스텀 메뉴 링크](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050729770/original/ELclgWfbIwfzJREkp0sn_PEX1wPD45eQAA.png?1753962348)

### **새 커스텀 링크 생성**

커스텀 메뉴 링크 섹션의 오른쪽 상단에 있는 **+ Create New(+ 새로 생성)** 버튼을 클릭하여 새 메뉴 링크 생성을 시작하세요.

![새 커스텀 링크 생성](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050729905/original/8DQLlsuy97Bq7M99CoswTFmnP2JE1xa67g.png?1753962464)

### **링크 세부 정보 입력**

필수 필드를 채우세요:

- **Link Icon(링크 아이콘)**: 아이콘 식별자를 선택하세요 (예: vr-cardboard)
- **Link Title(링크 제목)**: 메뉴 항목에 이름을 지정하세요 (예: "테스트 링크")
- **URL**: 이전에 생성한 전체 매직 링크를 붙여넣으세요.

![링크 세부 정보 입력](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050730145/original/piM2e1feZmZTFZEULHCiFh41q6bfKfAMjg.png?1753962675)

### 링크 동작 선택

링크를 클릭했을 때 어떻게 열릴지 선택하세요. 현재 창에 임베드된 아이프레임으로 표시하거나, 새 탭에서 열거나, 같은 창에서 로드할 수 있습니다.

![링크 동작 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050730280/original/_LgCu09DXrA4iiUykNgX9S05HBEMGKdZrQ.png?1753962777)

### 사이드바 표시 설정

커스텀 메뉴 링크가 어디에 표시될지 선택하세요. 에이전시 사이드바, 하위 계정 사이드바, 또는 서로 다른 사용자 레벨에서 쉬운 네비게이션을 위해 두 곳 모두에 표시할 수 있습니다.

![사이드바 표시 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050730378/original/Ma9e_IcpAQK7K0zWOtuPS73yt9JGlSsYjQ.png?1753962848)

### **하위 계정 선택**

커스텀 메뉴 링크를 하나 이상의 하위 계정에 할당하세요. 선택된 계정에만 네비게이션 사이드바에 이 링크가 표시됩니다.

![하위 계정 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050730522/original/ZTeeKMtDLvg0tkkjFuVFqHLPFgGdWyQ72w.png?1753962931)

### 역할 기반 표시 설정

사용자 역할에 따라 이 커스텀 메뉴 링크에 접근할 수 있는 사람을 선택하세요. Admin(관리자), User(사용자)로 표시를 제한하거나, 할당된 하위 계정 내의 모든 사용자가 사용할 수 있도록 설정할 수 있습니다.

![역할 기반 표시 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050730664/original/QOrfQw-iCiso42qnsJOC6pFc4K53DbNM_w.png?1753963033)

### 권한 활성화 (선택사항)

임베드된 콘텐츠가 이러한 기능을 필요로 하는 경우—화상 회의나 녹화 도구에 이상적—커스텀 메뉴 링크에 마이크와 카메라 접근 권한을 부여할 수 있습니다.

![권한 활성화](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050730751/original/OAb5ZI1qj9mWMSnECaG1Q4p-IBdSUuUHiQ.png?1753963110)

### 메뉴 링크 저장

URL, 사이드바 표시, 선택된 계정, 역할 기반 표시, 필요한 권한 등 모든 옵션을 구성한 후 **Save(저장)**를 클릭하여 커스텀 메뉴 링크를 완료하세요. 이렇게 하면 정의한 규칙에 따라 사용자가 쉽게 접근할 수 있도록 지정된 사이드바에 매직 링크가 추가됩니다.

![메뉴 링크 저장](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050730957/original/dJn2dDIJ31gpRR3O_Yci8ueDjFykuxWJFg.png?1753963266)

## **자주 묻는 질문**

**Q: 매직 링크는 모든 클라이언트 포털 앱에서 작동하나요?**
**Courses(강의)**, **Communities(커뮤니티)**, **Affiliates(제휴)**에서 작동합니다.

**Q: 매직 링크는 안전한가요?**
네. 각 링크는 연락처와 앱에 고유하며, 필요한 접근 권한만 제공합니다.

**Q: 자동화나 이메일에서 매직 링크를 사용할 수 있나요?**
네. 자동화 메시지, 온보딩 이메일, 커스텀 버튼에 링크를 삽입할 수 있습니다.

**Q: 사용자가 먼저 포털에 로그인해야 하나요?**
아니요. 매직 링크를 클릭하면 자동으로 로그인되고 리디렉션됩니다.

**Q: 매직 링크를 다시 생성할 수 있나요?**
네. 생성 과정을 다시 반복하여 새 링크를 만들 수 있습니다.
