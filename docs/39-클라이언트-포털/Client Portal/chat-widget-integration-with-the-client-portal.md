# 클라이언트 포털과 채팅 위젯 연동

**채팅 위젯(Chat Widget)** 연동으로 클라이언트 포털에 실시간 지원 기능을 추가하세요. 이제 개별 클라이언트 포털 하위 앱에서 채팅 위젯을 활성화하고 사용할 수 있어, 실시간 소통을 통해 더 나은 지원 경험을 제공합니다. 더욱 좋은 점은 위젯 표시를 **하위 앱별로 독립적으로 설정**할 수 있어 완벽한 유연성과 제어가 가능하다는 것입니다.

# 클라이언트 포털과 채팅 위젯 연동이란?

채팅 위젯 연동은 하나의 Hyperclass 채팅 위젯을 클라이언트 포털에 연결하고, **어떤 하위 앱**에서 표시할지 정확히 결정할 수 있게 해줍니다. 코드를 삽입하거나 여러 위젯을 관리하는 대신, 기존 위젯 하나를 선택하고 **커뮤니티, 강의, 제휴 프로그램**에 대한 표시 여부를 설정할 수 있습니다. 대화는 다른 웹 채팅과 마찬가지로 Hyperclass 인박스로 라우팅됩니다.

---

## 클라이언트 포털 채팅 위젯 연동의 주요 장점

- **실시간 지원**: 고객이 커뮤니티, 강의, 제휴 프로그램에서 작업하는 동안 탭을 전환하지 않고도 도움을 받을 수 있습니다.

- **세밀한 제어**: 비즈니스 목표에 맞춰 정확히 어떤 하위 앱에서 위젯을 표시할지 선택할 수 있습니다.

- **중앙 집중식 대화 관리**: 모든 채팅이 Hyperclass로 들어와 기록이 통합되어 더 빠른 후속 조치와 보고가 가능합니다.

- **낮은 진입 장벽**: 포털에 코드 삽입이 필요 없고, 클라이언트 포털 설정에서 선택만 하면 됩니다.

- **높은 참여도**: 적시 지원으로 강의 완수율, 커뮤니티 참여도, 제휴 프로그램 성과가 향상됩니다.

---

## 클라이언트 포털에서 채팅 위젯 연동 설정 방법

다음 단계에 따라 기존 채팅 위젯을 클라이언트 포털에 연결하고 표시할 위치를 선택하세요.

- 하위 계정에서 **멤버십(Memberships)** → 클라이언트 포털(Client Portal) → 설정(Settings) → 채팅 위젯(Chat Widget)으로 이동합니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051812720/original/7N256pP56TDZeXDqnNwXVmHUVPksxVk-8Q.gif?1755520196)

- **채팅 선택(Select Chat)** 위젯 드롭다운을 열고 클라이언트 포털에서 사용할 채팅 위젯 하나를 선택합니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051812809/original/WEIcAZG_9ViPF6DNzRkIz-mO-CXJhrkOuw.gif?1755520233)

- 페이지 선택(Select Pages) 아래에서 커뮤니티(Communities), 강의(Courses), 제휴 프로그램(Affiliates), **홈 및 계정 설정(Home & Account settings)**의 표시 여부를 필요에 따라 설정하고 설정 저장(Save Settings)을 클릭합니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051812893/original/JXb6KVrhrtGhbD20mf9CYbyYe_-TMOuI8A.gif?1755520261)

- 포털에서 확인하기:

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051507159/original/XtEPYFRrnD7N2bRkyo774KcM85Pojs4-vw.png?1755077959)

팁: 위젯의 모양이나 채널(실시간, 이메일/SMS, 왓츠앱)을 변경하려면 사이트(Sites) → 채팅 위젯(Chat Widget)에서 위젯 자체를 편집하고 클라이언트 포털 페이지를 새로고침하세요.

## 하위 앱별 위젯 표시 설정

표시 제어를 통해 대상과 상황에 맞는 지원 전략을 수립할 수 있습니다. 실시간 도움이 성과를 높이는 앱에서는 위젯을 켜고, 필요하지 않은 곳에서는 끄세요.

- **커뮤니티, 강의, 제휴 프로그램**에 대해 독립적으로 표시 여부를 설정할 수 있습니다.

- 다양한 조합을 사용하세요(예: 강의 + 제휴 프로그램은 켜고, 커뮤니티는 끄기).

- 변경 사항은 저장 후 적용되며, 포털 페이지를 새로고침하여 확인하세요.

---

## 사전 요구사항 및 권한

나중에 "위젯이 표시되지 않음" 상황을 피하기 위해 설정 전 다음 기본 사항을 확인하세요.

- 하위 계정에 최소 하나의 채팅 위젯을 이미 생성해둬야 합니다([채팅 위젯 시작하기]([getting-started-with-chat-widget](getting-started-with-chat-widget.md)) 및 [올인원 채팅 위젯 사용법]([how-to-use-the-all-in-one-chat-widget](how-to-use-the-all-in-one-chat-widget.md)) 참고).

- 하위 계정의 클라이언트 포털 설정에 접근할 수 있어야 합니다.

- (선택 사항) 채널 요구 사항을 충족해야 합니다(예: 위젯에서 왓츠앱을 사용할 계획이라면 왓츠앱 전화번호 필요).

---

## 제한 사항 및 요구 사항

- **클라이언트 포털의 단일 위젯**: 클라이언트 포털에는 하나의 채팅 위젯만 선택할 수 있습니다.

- **하위 앱별 표시만 가능**: 하위 앱별로 위젯을 표시하거나 숨길 수 있지만, 다른 하위 앱에 서로 다른 여러 위젯을 할당할 수는 없습니다.

- **위젯 콘텐츠는 전역적**: 선택한 위젯의 구성(브랜딩, 인사말, 채널)이 포털 내 모든 표시 위치에 적용됩니다.

- **웹사이트 vs 클라이언트 포털**: 웹사이트/퍼널용 멀티 위젯 설정은 클라이언트 포털 내부에는 적용되지 않습니다.

## 자주 묻는 질문

**Q: 다른 하위 앱에 서로 다른 위젯을 할당할 수 있나요?**
현재로서는 불가능합니다. 클라이언트 포털은 하위 앱별 표시 설정과 함께 단일 위젯 선택을 지원합니다.

**Q: 클라이언트 포털에서 위젯을 사용하려면 코드를 붙여넣어야 하나요?**
아니요. 클라이언트 포털(Client Portal) → 설정(Settings) → 채팅 위젯(Chat Widget)에서 선택하면 됩니다. 포털에는 임베드 코드가 필요하지 않습니다.

**Q: 위젯이 하위 앱 내 모든 페이지에 나타나나요?**
해당 하위 앱의 지원되는 뷰에 나타납니다. 보이지 않으면 새로고침하고 하위 앱 토글이 활성화되어 있는지 확인하세요.

**Q: 공개 웹사이트나 퍼널에서는 여전히 여러 위젯을 사용할 수 있나요?**
네, 멀티 위젯 설정은 웹사이트/퍼널에 적용됩니다. 클라이언트 포털 내부에는 적용되지 않습니다.

**Q: 위젯의 인사말, 색상, 채널을 변경하려면 어떻게 하나요?**
사이트(Sites) → 채팅 위젯(Chat Widget)에서 선택한 위젯을 편집하세요. 저장 후 새로고침하면 업데이트가 클라이언트 포털에 반영됩니다.

**Q: 왓츠앱을 사용하려면 추가 설정이 필요한가요?**
네, 위젯에서 해당 채널을 활성화하기 전에 제대로 구성된 왓츠앱 전화번호가 필요합니다.
