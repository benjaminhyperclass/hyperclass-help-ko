# 연락처를 통한 커뮤니티 그룹 액세스 권한 부여

## 개요

연락처(Contacts) 섹션에서 직접 **커뮤니티 그룹**과 강의 오퍼를 보고 관리할 수 있는 새로운 기능을 소개합니다. 이 연동 기능으로 연락처 정리, 오퍼 관리, 비밀번호 재설정 과정이 간소화되어 사용자 편의성과 시스템 기능이 크게 향상됩니다.

## 주요 기능

### 1. 연락처에 오퍼를 쉽게 추가

- 이제 연락처(Contacts) 페이지에서 모든 연락처에 강의나 커뮤니티 그룹 같은 오퍼를 쉽게 할당할 수 있습니다. 페이지를 떠나지 않고도 거래, 프로모션, 강의 등록을 관리할 수 있어 시간이 절약되고 작업 흐름이 개선됩니다.

### 2. 간편한 그룹 관리

- 연락처 페이지에서 직접 관련 커뮤니티 그룹에 연락처를 추가하여 연락처를 체계적으로 정리하세요. 이 기능으로 타겟 캠페인을 위한 오디언스 세분화나 커뮤니티 그룹을 효율적으로 관리할 수 있습니다. 모든 작업이 한 곳에서 가능합니다.

### 3. 사용자 비밀번호 관리

- 관리자는 이제 연락처 페이지에서 직접 비밀번호를 관리할 수 있습니다:
  - 연락처의 새 비밀번호 설정
  - 사용자에게 비밀번호 재설정 링크 전송으로 계정 복구 및 보안 관리를 더 빠르고 쉽게 처리

## 단계별 사용 가이드

### 1. 연락처로 이동

- 클라이언트 포털의 **연락처(Contacts)** 섹션으로 이동하세요.
- 관리하고자 하는 연락처를 선택하세요.

### 2. 클라이언트 포털 또는 강의 보기

- 특정 로케이션의 설정에 따라:
  - 강의가 활성화된 경우: 강의(Courses) 섹션이 클라이언트 포털 섹션을 대신하여 표시됩니다.
  - 클라이언트 포털이 활성화된 경우: 커뮤니티 그룹을 포함한 클라이언트 포털 섹션이 표시됩니다.

### 3. 오퍼 할당

- 연락처를 선택하면 오퍼(강의 또는 커뮤니티 그룹)를 할당하는 옵션이 표시됩니다.
- 사용 가능한 오퍼 목록에서 할당하고자 하는 항목을 선택하고 변경사항을 확인하세요.

### 4. 그룹 관리

- 커뮤니티 그룹 옵션을 사용하여 선택한 연락처를 적절한 그룹에 추가하세요.
- 타겟 캠페인이나 그룹별 알림 등 다양한 목적을 위해 서로 다른 커뮤니티 카테고리 내에서 연락처를 쉽게 관리할 수 있습니다.

### 5. 비밀번호 관리

- 연락처 페이지에서 다음 중 선택할 수 있습니다:
  - 새 비밀번호 설정: 연락처의 비밀번호를 직접 업데이트합니다.
  - 재설정 비밀번호 링크 전송: 사용자에게 비밀번호 재설정 링크를 이메일로 보내어 안전하게 비밀번호를 재설정할 수 있도록 합니다.

**참고사항:**

- 특정 로케이션에서 강의가 활성화된 경우, 연락처 페이지에 강의 섹션이 표시되어 클라이언트 포털을 대체합니다.
- 클라이언트 포털이 활성화된 경우, 연락처 페이지에는 더 쉬운 관리를 위한 커뮤니티 그룹이 포함된 클라이언트 포털 섹션이 표시됩니다.

연락처를 통해 부여되는 그룹 액세스는 가입을 위한 추가 질문이나 그룹 결제 요구 사항을 우회합니다. 즉, 회원들이 그룹에 직접 액세스할 수 있습니다.
