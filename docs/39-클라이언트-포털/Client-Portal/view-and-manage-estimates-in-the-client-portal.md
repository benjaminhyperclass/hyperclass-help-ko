# 견적서 기능이란?

클라이언트 포털의 견적서 기능은 고객이 하나의 인터페이스에서 모든 견적서에 접근하고 관리할 수 있게 해줍니다. 고객은 견적서 상세 정보를 확인하고, 서명된 견적서를 다운로드하며, 포털에서 직접 제안을 승인하거나 거절할 수 있습니다. 이 기능은 웹 버전과 모바일 앱 모두에서 사용할 수 있으며, 관리자는 앱 권한을 통해 이 기능의 사용 여부를 제어할 수 있습니다.

## 클라이언트 포털 견적서 기능의 주요 장점

견적서 기능은 고객이 정보를 파악하고 빠르게 결정을 내릴 수 있는 효율적인 방법을 제공합니다. 이 기능에는 여러 실용적인 장점이 있습니다:

- 고객과 관련된 모든 견적서를 한 곳에서 조회할 수 있는 중앙 집중식 위치
- 보기 옵션을 통한 견적서 상세 정보 쉽게 접근
- 서명된 견적서를 PDF 형식으로 다운로드하는 기능
- 승인 또는 거절에 따른 견적서 상태 업데이트
- 클라이언트 포털의 웹과 모바일 버전 모두와 호환
- 앱 권한을 사용하여 견적서 기능을 활성화하거나 비활성화하는 관리자 제어

## 웹에서 견적서 접근 및 사용하기

클라이언트 포털 웹 버전을 사용하는 고객은 대시보드의 전용 위젯을 통해 견적서를 찾고 상호작용할 수 있습니다.

### 단계:

- 웹 브라우저를 사용하여 클라이언트 포털에 로그인하세요.
- 대시보드에서 견적서 위젯을 찾으세요.
- 눈 모양 아이콘을 클릭하여 견적서의 상세 정보를 확인하세요.
- 서명된 견적서를 다운로드하려면 다운로드 아이콘을 클릭하세요.
- 5개 이상의 견적서가 있는 경우, "전체 보기"를 클릭하여 전체 목록을 확인하세요.

## 모바일 앱에서 견적서 접근 및 사용하기

클라이언트 포털 모바일 버전을 사용하는 고객은 앱 인터페이스의 전용 탭을 통해 견적서를 관리할 수 있습니다.

### 단계:

- 클라이언트 포털 모바일 애플리케이션을 여세요.
- 메인 대시보드에서 견적서 탭으로 이동하세요.
- 눈 모양 아이콘을 탭하여 견적서 상세 정보를 확인하세요.
- 서명된 견적서를 다운로드하려면 다운로드 아이콘을 탭하세요.

참고: 다운로드는 서명된 견적서에만 가능합니다.

- 목록을 스크롤하여 사용 가능한 모든 견적서를 둘러보세요.

## 관리자 권한을 통한 견적서 기능 관리

관리자는 앱 권한 설정을 통해 클라이언트 포털 내의 견적서 기능을 활성화하거나 비활성화할 수 있습니다.

### 단계:

- 관리자 대시보드에서 앱 설정(App Settings)으로 이동하세요.
- 권한(Permissions) 섹션으로 이동하세요.
- 견적서(Estimates) 토글을 찾아 필요에 따라 활성화하거나 비활성화하세요.
- 이 권한은 웹과 모바일 클라이언트 인터페이스 모두에 적용됩니다.

## 자주 묻는 질문

**Q: 일부 견적서에서 다운로드 옵션을 사용할 수 없는 이유는 무엇인가요?**
A: 다운로드는 서명된 견적서에만 가능합니다. 서명되지 않은 견적서는 다운로드할 수 없습니다.

**Q: 고객이 이전에 승인하거나 거절한 견적서를 볼 수 있나요?**
A: 네, 고객은 승인하거나 거절한 견적서를 포함하여 견적서의 전체 기록을 볼 수 있습니다.

**Q: 모바일 앱에서 견적서 탭은 어디에 있나요?**
A: 견적서 탭은 클라이언트 포털 모바일 애플리케이션의 대시보드에서 직접 접근할 수 있습니다.

**Q: 관리자가 고객의 견적서 기능을 비활성화할 수 있나요?**
A: 네, 관리자는 앱 권한(App Permissions) 설정을 통해 견적서 기능에 대한 접근을 제어할 수 있습니다.

**Q: 클라이언트 포털에서 견적서 위젯이 보이지 않는 경우 어떻게 해야 하나요?**
A: 관리자에게 해당 클라이언트의 권한 설정에서 견적서 기능이 활성화되어 있는지 확인해 달라고 요청하세요.
