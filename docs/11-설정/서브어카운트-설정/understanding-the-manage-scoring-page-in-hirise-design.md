# **점수 관리 페이지란? (HiRise 디자인 업그레이드)**

점수 관리(Manage Scoring)는 연락처가 특정 행동(이메일 열기, 클릭, 결제 등)을 할 때 점수를 더하거나 빼는 기능입니다. 이 점수들이 모여서 연락처 참여도 점수를 만들어내며, 이를 통해 고객을 세분화하고 자동 후속 조치를 설정할 수 있습니다.

새로운 HiRise 인터페이스는 기존의 모든 점수 매기기 로직을 그대로 유지하면서, 페이지 레이아웃, 성능, 협업 옵션을 대폭 개선했습니다.

---

## HiRise 업그레이드의 주요 장점

- 현대적이고 깔끔한 HiRise 스타일링으로 규칙을 훨씬 쉽게 스캔하고 편집할 수 있습니다.

- 대용량 규칙 세트에서 최대 2배 빨라진 로딩 속도로 더 부드러운 스크롤링을 경험하실 수 있습니다.

- 공유 가능한 URL이 검색 텍스트, 페이지 번호, 행 제한 설정을 모두 기억하므로, 팀원이 정확히 같은 화면에 도달할 수 있어 재필터링이 필요 없습니다.

- 다른 HiRise 기반 페이지들과 일관된 경험을 제공하여 새로운 사용자의 학습 곡선을 줄여줍니다.

- 향상된 접근성과 키보드 네비게이션으로 더 빠른 규칙 관리가 가능합니다.

---

## 더 깔끔해진 HiRise 레이아웃

이제 페이지는 단일 컬럼의 넓은 테이블 디자인을 사용하며, 개선된 간격, 색상으로 구분된 점수 값, 고정된 액션 헤더를 제공합니다. 이러한 시각적 명확함 덕분에 중복되거나 오래된 규칙을 한눈에 쉽게 찾아낼 수 있습니다.

## 지속적인 화면 공유 링크

검색창, 페이지네이션, "페이지당 행 수" 컨트롤에서 변경한 모든 내용이 URL에 저장됩니다. 링크를 붙여넣거나 북마크해서 나중에 다시 열거나 팀원과 공유할 때, 정확히 같은 필터 세트로 작업했던 화면을 볼 수 있습니다.

---

## **성능 향상 및 확장성**

HiRise의 더 가벼워진 프론트엔드 자산과 새로운 캐싱 전략으로 로딩 시간이 대폭 단축되었습니다. 특히 50개 이상의 점수 규칙이 있는 하위 계정에서 그 효과가 두드러집니다. 기다리는 시간이 줄어든 만큼 참여도 로직 최적화에 더 많은 시간을 투자할 수 있습니다.

---

## 점수 관리 설정 방법

참여도 점수를 세밀하게 조정하는 방법은 그대로 유지되며, UI만 새로워졌습니다.

### **점수 관리 접근하기**

하위 계정에서 Settings(설정) → Manage Scoring(점수 관리)를 클릭하세요.

![점수 관리 메뉴 접근](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155056056490/original/sAK8CgThZ0A5dFexBbGSy5ukjc4k3mWMDQ.png?1760533290)

### 초안/발행

실제 점수에 영향을 주지 않고 변경 사항을 준비하고 싶다면 Draft Mode(초안 모드) 토글(우측 상단)을 활용하세요.

![초안 모드 토글](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155056056556/original/AJb0SdADJ1q5sPLnPdFvK4001Rg9c4EeOQ.png?1760533311)

### **새 규칙 추가하기**

+ Add New Rule(새 규칙 추가)을 클릭하세요. Action Type(액션 타입, 예: Email Opened(이메일 열림))을 선택하고 긍정적이거나 부정적인 점수를 할당하세요.

![새 규칙 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155056056574/original/BMfMbRB0oRQVaCYobmF1DtCTO9hIqAmJnA.png?1760533329)

### 편집/삭제 메뉴

규칙 옆의 점 세 개 메뉴를 사용하여 편집하거나 삭제할 수 있습니다.

![편집/삭제 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155056056733/original/In35bg2s-C3VzP_kIetBvBPD9S3erUfkxg.png?1760533407)

### **규칙 설정/수정하기**

Scoring Rules Table(점수 규칙 테이블)에서 모든 변경 사항을 검토하세요. 실제로 적용할 준비가 되면 Draft/Publish(초안/발행) 토글을 Publish(발행)로 전환하세요. 팀원과 필터링된 화면을 공유하려면 페이지 URL을 복사하세요.

![규칙 설정 완료](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155056056936/original/fU8IEHjTrWAkkFW-p45nlU_vQVYtj-bJhg.png?1760533500)

## **자주 묻는 질문**

**Q: 기존 점수 규칙이 변경되나요?**

아니요. 이번 업그레이드는 외관과 성능 위주의 변경이므로, 기존 규칙과 점수 총합은 그대로 유지됩니다.

**Q: 공유 링크도 초안 vs. 발행 모드를 구분하나요?**

네. 페이지는 저장했던 모드 그대로 열리므로, 협업자들이 안전하게 초안을 검토할 수 있습니다.

**Q: 새 인터페이스에서 여러 규칙을 한 번에 편집할 수 있나요?**

일괄 편집 기능은 아직 제공되지 않습니다. 점 세 개 메뉴를 통해 한 번에 한 규칙씩 편집해야 합니다.

**Q: 빨라진 로딩 속도는 모든 브라우저에 적용되나요?**

네. HiRise는 더 가벼운 자산과 브라우저 레벨 캐싱을 사용하여 Chrome, Firefox, Edge, Safari 모두에서 혜택을 받을 수 있습니다.

**Q: 두 명의 사용자가 동시에 규칙을 편집하면 어떻게 되나요?**

마지막에 저장된 버전이 적용됩니다. 공유 URL과 초안 모드를 통해 조율하시길 권장합니다.

**Q: 규칙 변경에 대한 감사 로그가 있나요?**

점수 관리 페이지 자체에는 없지만, 워크스페이스 활동 로그에서 누가 변경 사항을 발행했는지 확인할 수 있습니다.

**Q: 점수 규칙을 내보낼 수 있나요?**

기본 제공되는 내보내기 기능은 아직 없습니다. API를 사용하거나 테이블을 CSV로 복사하는 우회 방법을 활용하세요.
