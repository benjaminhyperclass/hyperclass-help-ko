# 커스텀 값(Custom Values) 설정

커스텀 값은 한 번 생성하면 플랫폼 전체에서 재사용할 수 있는 키:값 쌍입니다.

## **커스텀 값이란?**

커스텀 값 기능은 플랫폼 어디서나 접근할 수 있는 키(Key)와 값(Value)입니다. 여러 곳에서 같은 값을 반복해서 작성하는 대신, 어디서든 키만 입력하면 됩니다. 커스텀 값 목록에서 하나의 값을 변경하면, 해당 키가 사용된 모든 곳에서 자동으로 업데이트됩니다.

---

## **커스텀 값 생성 방법**

**1단계: 커스텀 값으로 이동**

하위 계정에서 **Settings(설정) > Custom Values(커스텀 값)**으로 이동하세요.

![커스텀 값 메뉴 위치](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050457210/original/GKsAQ5KWxCNwdvGJ5CJ6o4sKrzP5-p7JUw.png?1753463469)

![커스텀 값 페이지](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050457249/original/hfKjk08r-zZpk_8AfL768DREg-q8hwQECQ.png?1753463520)

**2단계: 커스텀 값 생성**

우측 상단의 New Custom Value(새 커스텀 값) 버튼을 클릭하세요.

![새 커스텀 값 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050457285/original/t55SiIF--MpH5SyvCwniLAdfkrVOs2vQUg.png?1753463602)

**3단계: 필드 작성**

관련 필드를 모두 작성하세요:

- Name(이름) 필드. 커스텀 값의 키 부분은 이름에서 생성됩니다 (예: My Custom Value는 custom_values.my_custom_value가 됩니다).
- Value(값) 필드.
- Folder(폴더) 필드 (선택사항)

![커스텀 값 생성 폼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050457352/original/uWnKj8S5dCfrlcYtDd0xHZUvt3zo7EgKTQ.png?1753463661)

![커스텀 값 생성 완료](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050457367/original/SRmpCDLX7HwI3b_NHa4zYIcvQHOVoAoQYA.png?1753463709)

## **커스텀 값 관리 방법**

커스텀 값을 생성한 후에는 여러 관리 작업을 수행할 수 있습니다.

### **폴더**

Folders(폴더) 탭을 클릭한 다음 Add New Custom Value Folder(새 커스텀 값 폴더 추가) 버튼을 클릭하세요. 커스텀 값이 있는 폴더는 All Values(전체 값) 목록 > Folder(폴더) 열에 표시됩니다. 모든 폴더는 Folders(폴더) 목록에 나열됩니다.

커스텀 값은 한 번에 하나의 폴더에만 속할 수 있습니다. 폴더는 다른 폴더 안에 중첩될 수 없습니다.

![폴더 탭](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050457394/original/OwVtZYxqzYIOZHEdEoIepyS9r9seHZnoFw.png?1753463772)

![새 폴더 생성](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050457410/original/509r9j_xLfLIR-DqZZHzw2VN_r3wqB3rsA.png?1753463808)

![폴더 목록](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050457182/original/W8ctMTXRHWML1s9MYdNY4JaZRMHxlUTKag.png?1753463424)

### **폴더 메뉴**

폴더 목록 항목 우측의 점 3개 메뉴를 클릭하세요.

![폴더 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050457426/original/Pg7sksW8BSGbnNe4J5iQgOfi-sEokQ-rxw.png?1753463852)

### **폴더 이름 변경**

Rename Folder(폴더 이름 변경)를 클릭하세요. 새 폴더 이름을 입력하세요.

![폴더 이름 변경](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050457437/original/xpbGiHkEJ_pAVR4oBWzqOV7-xI9GVa_YcA.png?1753463884)

### **폴더 삭제**

Delete Folder(폴더 삭제)를 클릭하세요. 확인을 요청하는 모달이 표시됩니다.

![폴더 삭제 확인](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050457456/original/m54IEnw2lDzPuPAdMEOscrG2TdHBG4B4Fg.png?1753463959)

### **검색**

All Values(전체 값) 목록 우측 상단에 검색 필드가 있습니다. 검색어를 입력하면 커스텀 값 목록이 키워드와 일치하는 항목만으로 필터링됩니다.

![검색 기능](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050457469/original/K6TzrQexRko7lskG4Y0VxJY3rt1xo5vklA.png?1753463994)

### **항목 메뉴**

커스텀 값 좌측의 점 3개 메뉴를 클릭하여 메뉴를 여세요.

![항목 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050457489/original/dnDkO6KYic_mLEy43IaI8NR6uPdxPRZypw.png?1753464045)

### **항목을 폴더로 이동**

Move To Folder(폴더로 이동)를 클릭하세요. 모달에서 대상 폴더를 선택하고 Move(이동)를 클릭하세요.

![폴더로 이동](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050457506/original/PZjAqO-GAB1lVZeUakHmTpDdZdS36Vwd9g.png?1753464086)

### **커스텀 값 편집**

Edit Custom Value(커스텀 값 편집)를 클릭하세요. 모달에서 새 Name(이름), 새 Value(값) 및/또는 새 폴더로 이동할 수 있습니다. Name을 변경해도 Key는 변경되지 않는다는 점을 기억하세요.

![커스텀 값 편집](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050457522/original/gWCNfUfgF4LI03jFcjLDxss3AhtTNnFgRg.png?1753464132)

### **커스텀 값 삭제**

Delete Custom Value(커스텀 값 삭제)를 클릭하세요. 확인을 요청하는 모달이 표시됩니다.

![커스텀 값 삭제](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050457544/original/UiGnKDsv4m7k_mxGhCu8WxTvF2RF3GukuA.png?1753464180)

### **일괄 작업**

하나 이상의 체크박스를 클릭하면 Bulk Actions(일괄 작업) 버튼이 나타납니다. 버튼을 클릭하면 메뉴가 표시되고, Unselect All(전체 선택 해제)를 클릭할 수도 있습니다.

![일괄 작업](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050457629/original/KsUClz-I9MBxWbu5Dffe-b_7HV2NbEwtdg.jpeg?1753464223)

### **일괄 폴더 이동**

Move To Folder(폴더로 이동)를 클릭한 다음 모달에서 대상 폴더를 선택하세요. 선택된 커스텀 값들을 이동한 후에는 Folder(폴더) 열에 대상 폴더가 표시됩니다.

![일괄 폴더 이동](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050457654/original/FivusXF53NG11T_sUxQPEZ7Iq8TlhOuXgg.png?1753464273)

![폴더 이동 완료](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050457689/original/PbGqMxQ-PBafLsC-KQmK5_1QCqWLm0JcZQ.png?1753464326)

### **일괄 커스텀 값 삭제**

Delete Custom Value(커스텀 값 삭제)를 클릭하세요. 선택된 커스텀 값들을 삭제할지 확인하는 모달이 표시됩니다.

![일괄 삭제 확인](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050457712/original/5PzM1j5yA1FE_KOdkw8Vjz1pRt_yTXXD_g.png?1753464367)
