# 미디어 저장소 다중 선택 기능

Hyperclass의 미디어 저장소(Media Storage)에 다중 선택(Multi-Select) 기능이 추가되어, 파일 이동, 다운로드, 삭제 등의 일괄 작업을 수행할 수 있습니다. 이 개선사항으로 파일 관리가 간소화되고 작업 효율성이 크게 향상됩니다.

## 미디어 저장소 다중 선택 기능이란?

Hyperclass 미디어 저장소의 다중 선택 기능은 여러 파일이나 폴더를 동시에 선택할 수 있게 해줍니다. 이 기능을 통해 일괄 작업이 가능해져 파일 관리 효율성이 크게 향상됩니다.

![미디어 저장소 다중 선택 개요](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036267746/original/NX3e9TZYTp7wD72Bz6lOrv6fpVOQPkp4aA.png?1731093113)

## 다중 선택 사용의 장점

- **효율적인 파일 관리**: 체크마크를 사용하거나 마우스를 드래그해서 여러 파일을 한 번에 선택할 수 있어, 대량의 파일 처리가 간편해집니다.

- **일괄 작업**: 선택한 파일들을 한 번의 작업으로 이동, 다운로드, 삭제할 수 있어 반복 작업에 소요되는 시간을 줄일 수 있습니다.

- **사용자 친화적 인터페이스**: 드래그로 선택하거나 점 3개 메뉴의 "선택" 옵션을 사용해서 다중 선택 모드에 진입할 수 있습니다. 선택된 파일은 왼쪽 상단에 체크마크가 표시됩니다.

## 그리드 뷰에서 미디어 다중 선택

선택하고 싶은 파일들에 마우스를 클릭하고 드래그하거나, 점 3개 메뉴에서 "선택" 옵션을 사용하세요. 선택된 파일과 폴더에는 체크마크가 표시됩니다.

![그리드 뷰 다중 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036267856/original/rTHYSN7slGc1QsR4Bhbcx2hKqoVWGKjbYw.gif?1731093366)

## 목록 뷰에서 미디어 다중 선택

각 파일이나 폴더 옆의 체크박스를 사용해서 여러 항목을 선택하세요.

![목록 뷰 다중 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036267862/original/s3fnMKdvkKSEddCkUViwjhOVgXUbY46bGg.gif?1731093389)

## 다중 선택 일괄 작업 실행

파일을 선택한 후, 해당 버튼을 사용해서 이동, 다운로드, 삭제 등의 일괄 작업을 수행할 수 있습니다.

![일괄 작업 실행](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036271616/original/g0Ii1Km_oycg71fndZ5wRMkzoi81d0dqEw.gif?1731105664)

- 한 번에 최대 100개 파일까지 선택해서 다운로드할 수 있습니다.
- 파일 크기가 2GB를 초과하는 경우, 다중 다운로드에서 제외되며 개별 다운로드해야 합니다.
- 다중 선택 모드에서는 일괄 작업의 단순성을 위해 선택된 파일의 "이름 바꾸기"와 "링크 가져오기" 옵션이 비활성화됩니다.

## 자주 묻는 질문

**Q: 다중 선택으로 서로 다른 폴더의 파일들을 선택할 수 있나요?**

현재 다중 선택은 단일 폴더 내에서만 가능합니다. 다른 폴더의 파일들을 관리하려면 각 폴더에서 별도로 작업을 수행해주세요.

**Q: 다중 선택으로 선택할 수 없는 파일 형식이 있나요?**

다중 선택은 미디어 저장소에서 사용 가능한 모든 파일 형식을 지원합니다. 다만, 파일 형식에 따라 특정 작업이 제한될 수 있습니다. 예를 들어, 일부 파일 형식은 미리보기가 지원되지 않을 수 있습니다.

**Q: 일괄 작업 시 같은 이름의 파일들은 어떻게 처리되나요?**

같은 이름의 여러 파일을 이동하거나 다운로드할 때, 시스템에서 덮어쓰기를 방지하기 위해 파일 이름을 바꾸거나 건너뛸 것인지 묻는 메시지가 나타납니다.

**Q: 한 번에 선택할 수 있는 파일 수에 제한이 있나요?**

일괄 작업을 위해 한 번에 최대 100개의 파일을 선택할 수 있습니다. 이 제한을 초과해서 선택하려고 하면 알림이 표시됩니다.
