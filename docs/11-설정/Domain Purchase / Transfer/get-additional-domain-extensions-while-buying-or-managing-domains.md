## 추가 도메인 확장자 제안이란?

추가 도메인 확장자 제안은 도메인을 구매하거나 관리할 때 인기 있는 대체 도메인 확장자를 추천하여 브랜드를 보호하는 데 도움이 되는 Hyperclass 기능입니다. 추가 TLD를 자동으로 제안함으로써 이 기능은 사칭 위험을 최소화할 뿐만 아니라 관련된 모든 도메인 확장자를 투명한 한 번의 거래로 확보하는 과정을 간소화합니다.

### 추가 도메인 확장자 제안의 주요 장점

이 기능은 보안과 편의성을 모두 향상시키는 여러 가지 장점을 제공합니다:

- **브랜드 보호**: 여러 도메인 확장자를 확보하여 경쟁자나 사기꾼이 유사한 이름을 악용할 가능성을 줄입니다.
- **사용자 편의성**: 스마트하고 자동화된 추천으로 직관적이고 간소화된 도메인 구매 경험을 제공합니다.
- **비용 투명성**: 추가 도메인 확장자에 대한 총 추가 비용을 미리 확인하여 구매 과정에서 완전한 투명성을 보장합니다.
- **경쟁 우위**: 온라인 존재감을 강화하고 사칭을 방지하는 핵심 확장자를 확보합니다.

## 추가 도메인 확장자 제안 사용 방법

Hyperclass에서는 이제 새 도메인을 구매할 때나 기존 도메인을 관리할 때 추가 도메인 확장자를 구매할 수 있습니다. 이 기능은 도메인 이름의 여러 확장자를 확보하여 브랜드를 보호하는 데 도움이 됩니다. 아래 단계를 따라하세요:

### 1. 도메인 구매 시

- 하위 계정에 로그인합니다.

- 왼쪽 메뉴에서 Settings(설정)을 클릭합니다.
![설정 메뉴 선택](https://jumpshare.com/share/YDsWLpuEsdBw8PKu0Wxs+/Screen+Shot+2025-08-18+at+10.59.17+PM.png)

- 아래로 스크롤하여 Domains(도메인) 탭을 선택합니다.
![도메인 탭 선택](https://jumpshare.com/share/hqWEypTz2rRS221Njfzu+/Screen+Shot+2025-08-18+at+11.01.50+PM.png)

- Purchase / Transfer Domain(도메인 구매/이관) 버튼을 클릭합니다.

- 검색창에 구매하려는 도메인 이름을 입력합니다.

- 결과에서 원하는 도메인 옆의 Buy(구매)를 클릭합니다.

- "도메인 구매 확인" 창이 나타납니다.

- "경쟁자로부터 브랜드를 보호하세요 — 이런 확장자도 함께 구매하세요"라는 섹션이 표시됩니다.

- 최대 3개의 대체 확장자(예: .net, .uk, .org)가 제안됩니다.

- 추가하려는 확장자를 클릭한 다음 Add for $X.XX(가격은 도메인에 따라 달라짐) 버튼을 클릭합니다.

- 원하는 각 추가 확장자에 대해 반복합니다.

- 모달창에 다음의 총 비용이 표시됩니다:
  - 기본 도메인
  - 선택한 추가 확장자들

- "이용약관에 동의합니다" 체크박스를 선택합니다.

- Pay $X.XX(결제)를 클릭하여 도메인과 선택한 확장자의 구매를 완료합니다.

![도메인 확장자 추가 과정](https://jumpshare.com/share/Dh1i0jqF4dQINib23FPI+/GIF+Recording+2025-08-18+at+11.02.56+PM.gif)

### 2. 기존 도메인 관리 시

- 하위 계정에 로그인합니다.

- Settings(설정)을 클릭합니다.
![설정 메뉴 선택](https://jumpshare.com/share/YDsWLpuEsdBw8PKu0Wxs+/Screen+Shot+2025-08-18+at+10.59.17+PM.png)

- Domains(도메인) 탭을 클릭합니다.
![도메인 탭 선택](https://jumpshare.com/share/hqWEypTz2rRS221Njfzu+/Screen+Shot+2025-08-18+at+11.01.50+PM.png)

- 등록되거나 이관된 도메인 목록이 표시됩니다.

- 새 확장자를 추가하려는 도메인을 찾습니다.

- 해당 도메인 옆의 Manage(관리)를 클릭합니다.
![도메인 관리 버튼](https://jumpshare.com/share/UbWETuWCAfrnF2SDR6mh+/Screen+Shot+2025-08-18+at+11.11.20+PM.png)

- 도메인 관리 화면 상단에 다음 메시지가 표시됩니다: "제안된 확장자로 도메인 이름을 구매하여 경쟁자로부터 브랜드를 보호하세요 $X"

- Buy Now(지금 구매)를 클릭합니다.
![브랜드 보호 배너](https://jumpshare.com/share/uWl1fyZGk3F5SQxsGAqQ+/Screen+Shot+2025-08-18+at+11.12.52+PM.png)

- 도메인 구매 확인 모달창이 나타납니다.

- 추가 확장자(예: .net, .uk)를 선택하고 Add for $X.XX(추가) 버튼을 클릭합니다.
![확장자 선택](https://jumpshare.com/share/x6RtF3pD91MAKMDtJvb1+/Screen+Shot+2025-08-18+at+11.19.51+PM.png)

- 이용약관 체크박스를 선택합니다.

- Pay $X.XX(결제)를 클릭하여 추가 확장자 구매를 완료합니다.

## 자주 묻는 질문

**Q: 한 번에 몇 개의 추가 도메인 확장자를 구매할 수 있나요?**

주 도메인과 함께 최대 3개의 추가 도메인 제안을 선택할 수 있습니다.

**Q: 추천된 추가 확장자를 반드시 구매해야 하나요?**

아니요, 추가 확장자를 선택하지 않고 단독 도메인만 구매하도록 선택할 수 있습니다.

**Q: 이러한 추가 도메인 확장자에 추가 비용이 발생하나요?**

네, 추가 비용은 구매 과정에서 명확히 표시되어 완전한 가격 투명성을 제공합니다.

**Q: 이미 소유한 도메인에 추가 도메인 확장자를 추가할 수 있나요?**

물론입니다. 도메인을 소유하고 있다면 도메인 세부 페이지의 "브랜드 보호" 배너를 통해 추가 확장자를 구매할 수 있습니다.

**Q: 확장자를 추가하면 현재 주 도메인의 DNS에 영향을 미치나요?**

아니요. 추가 확장자를 구매하면 자체 DNS 존을 가진 별도의 도메인이 생성됩니다. 그 레코드는 Hyperclass 내에서 독립적으로 관리되며, 명시적으로 레코드를 편집하거나 리다이렉트를 설정하지 않는 한 기존 주 도메인의 DNS를 변경하지 않습니다.
