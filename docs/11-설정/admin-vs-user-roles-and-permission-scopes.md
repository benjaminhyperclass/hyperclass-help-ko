# 관리자 vs 일반 사용자 역할과 권한 범위

아래는 하위 계정(Sub-account) 레벨 역할의 권한을 정리한 표입니다.

## 빠른 참조 권한 매트릭스

| 에이전시 전용 | 하위 계정 전용 | 둘 다 사용 가능 |
|---|---|---|
| 하위 계정 생성/편집/삭제 | 파이프라인 & 기회 관리 | 사용자 관리(관리자는 해당 레벨 내 사용자 추가/편집 가능) |
| SaaS 모드 & 재판매 관리 | 워크플로우 / 캠페인 | 대시보드 리포팅 |
| 글로벌 연동(Mailgun, Twilio 리빌링, Google API 등) | 캘린더 & 예약 설정 | 미디어 라이브러리 |
| 스냅샷 관리 | 대화(Conversations) (SMS, 이메일, 채팅) | 감사 로그 |
| 에이전시 레벨 결제 & 브랜딩 | 연락처 관리 & 스마트 리스트 | |
| 에이전시 설정(회사 정보, 리빌링 설정) | 평판 관리 | |
| 화이트라벨 설정(커스텀 도메인, 로고, 색상) | 퍼널 & 웹사이트 빌더 | |

**참고**: 목록에 없는 권한은 해당 레벨에서 역할의 기본 권한을 상속받는다고 보시면 됩니다.

관리자는 Settings(설정) > My Staff(내 직원) > Edit(편집, 연필 아이콘) > "User Roles(사용자 역할)" 섹션을 펼쳐서 특정 사용자의 역할을 변경할 수 있습니다.

![사용자 역할 설정 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155018238329/original/SO6U66a5eqB-L00Pot42Kknck7OO6w0K8Q.png?1705709967)

## 자주 묻는 질문

**Q. 에이전시 관리자가 하위 계정 관리자의 특정 기능 접근을 제한할 수 있나요?**

아니요. 에이전시 관리자는 하위 계정 관리자 역할 권한을 직접 커스터마이징할 수 없습니다. 하위 계정 관리자는 항상 배정된 하위 계정 내에서 모든 권한을 상속받습니다. 범위를 제한하려면 해당 하위 계정에서 관리자 역할 대신 *일반 사용자* 역할을 생성하세요.

**Q. 권한이 에이전시와 하위 계정 양쪽에서 모두 사용 가능하다면, 에이전시 관리자가 하위 계정 관리자보다 우선권을 가지나요?**

네. 에이전시 관리자가 글로벌 권한을 갖습니다. 사용자 관리, 리포팅, 미디어 라이브러리 같은 권한의 경우, 에이전시 관리자 액션은 모든 하위 계정에 적용되는 반면, 하위 계정 관리자는 자신에게 배정된 하위 계정 내에서만 관리할 수 있습니다.

**Q. 사용자가 에이전시 레벨에 추가된 후 하위 계정에도 접근 권한을 받으면 어떻게 되나요?**

해당 사용자는 **두 개의 역할 범위**를 갖게 됩니다: 에이전시 역할로부터의 에이전시 전체 접근권과 하위 계정 역할로부터의 하위 계정별 접근권을 동시에 갖습니다. 권한은 서로 상쇄되지 않고 누적되며, 더 넓은 범위인 에이전시 권한이 항상 우선합니다.

**Q. 모든 에이전시 관리자가 "대신 로그인(Login As)" 기능을 사용할 수 있나요?**

**아니요**. "대신 로그인"은 **에이전시 레벨의 Enable Login As 권한**으로 제어됩니다. 관리자에게 이 기능이 비활성화되어 있다면, 해당 사용자에게는 Login As 옵션이 숨겨집니다.
