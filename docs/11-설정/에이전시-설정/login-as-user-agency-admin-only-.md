# 사용자로 로그인 (에이전시 관리자 전용)

참고: 이 기능은 에이전시 관리자만 사용할 수 있습니다. 다른 역할의 사용자에게는 이 옵션이 표시되지 않습니다.

사용자는 에이전시 권한에서 "로그인 허용" 권한이 켜져 있어야 합니다. 그렇지 않으면 "사용자로 로그인" 버튼이 나타나지 않습니다. 이 기능을 통해 에이전시 관리자는 사용자 계정으로 대신 로그인하여 문제 해결, 설정 변경, 설정 업데이트 또는 로그인한 사용자로서 고객에게 응답할 수 있습니다. 이전에는 에이전시 관리자가 사용자 계정을 변경하거나 고객을 대신해 응답하려면 여러 번 로그아웃과 로그인을 반복해야 했습니다. 이 가이드에서는 사용자 간 전환 방법을 안내드립니다.

# 사용자로 로그인 기능이란?

사용자로 로그인 기능을 통해 에이전시 관리자는 사용자의 로그인 정보 없이도 하위 계정에 사용자 권한으로 접근할 수 있습니다.

---

## 누가 사용할 수 있나요?

이 기능은 에이전시 관리자만 사용할 수 있습니다. 다른 역할의 사용자에게는 이 옵션이 표시되지 않습니다.

사용자로 로그인 기능 접근은 사용자별 권한으로 제어됩니다. 기본적으로 에이전시 소유자/관리자는 "로그인 허용" 권한이 켜져 있습니다. 관리자의 이 권한이 비활성화되면 해당 사용자에게는 "사용자로 로그인" 옵션이 숨겨집니다.

## 어떻게 사용하나요?

에이전시 관리자가 사용자로 로그인하는 단계별 과정은 다음과 같습니다:

**1단계:** 우측 상단의 프로필 아이콘을 클릭하세요.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155034618433/original/px4vo_6QMMjQ_ZYpHxTVqpwU-7sKjkZuwQ.png?1728889199)

**2단계:** "Login as(사용자로 로그인)" 버튼을 클릭하여 접근하려는 사용자 계정을 선택하세요.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155034618460/original/IJ4niJOLeeZr5yXX5wFVv0xL2-TVzn9iTQ.png?1728889222)

**3단계:** 사용자 선택: 검색 바에 사용자 이름이나 이메일 주소를 입력하거나, 목록을 스크롤하여 로그인하려는 사용자를 찾을 수 있습니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155034618833/original/nt6472FJ6iC-2PItze1sonlfhqQgAFdmxA.png?1728889567)

## 이 기능을 어떻게 활용하나요?

다른 사용자로 로그인하면, 에이전시 관리자는 이전에 사용자가 직접 로그인해야만 할 수 있었던 모든 계정 변경을 수행할 수 있습니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48255768432/original/eeqNbXO851vH6i5A9-02i9A_b6Sko0nb6g.gif?1665410588)

계정 전환 옵션을 통해 에이전시 관리자는 사용자 계정에서 에이전시 관리자 계정으로 다시 이동할 수도 있습니다. 이 기능에는 사용자의 이메일 주소나 이름을 검색하여 해당 사용자로 로그인할 수 있는 검색 기능도 포함되어 있습니다.

## 자주 묻는 질문

**Q: 에이전시 관리자가 사용자로 로그인한 상태에서 변경한 내용이 감사 로그에 어떻게 기록되나요?**

이 기능의 첫 번째 단계에서는 감사 로그(Audit Log) 페이지의 변경 기록에 계정 사용자의 세부 정보(에이전시 관리자 정보가 아닌)가 기록됩니다. 향후 개발될 두 번째 단계에서는 감사 로그에 에이전시 관리자와 계정 사용자의 세부 정보가 모두 기록될 예정입니다.

**Q: "사용자로 로그인" 메뉴가 보이지 않아요. 왜 그런가요?**

사용자의 "로그인 허용" 권한이 꺼져 있을 수 있습니다. 에이전시 소유자/관리자에게 사용자 관리 권한에서 이 기능을 활성화해달라고 요청하세요. 이 권한이 비활성화되면 해당 옵션이 숨겨집니다.
