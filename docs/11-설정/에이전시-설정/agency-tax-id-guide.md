# 에이전시 세금 ID 가이드

Hyperclass는 세금 의무를 준수하기 위해 전 세계 고객으로부터 세금 ID 번호를 수집하기 시작했습니다. 에이전시의 세금 정보를 업데이트하는 빠른 가이드를 제공합니다.

**부가가치세(VAT)**
조직이 유럽연합, 노르웨이, 터키 또는 영국에 위치한 경우, 구매(예: 구독, 갱신)에 부가가치세(VAT)가 추가되며 인보이스에서 세금을 별도 항목으로 확인할 수 있습니다.

**VAT 면세**
등록된 사업체인 경우, Agency Settings(에이전시 설정) > Billing(청구) > Payments(결제) 페이지를 통해 유효한 VAT ID를 입력하면 향후 구매에서 VAT가 면제될 수 있습니다(즉, 역과금이 적용될 수 있음).

## 세금 ID 업데이트

- 세금 ID 정보는 Agency Settings(에이전시 설정) > Billing(청구) > Payments(결제)에서 접근할 수 있습니다.

![세금 ID 접근 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155025689072/original/woWrc1qkJs_DoQ2NbBSgeFm2SkCxV1K19A.png?1715157380)

- Edit(편집) 버튼을 선택하여 세금 ID 정보 업데이트를 시작합니다.

![편집 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155025689611/original/7N1-JO9HRHOIDAeLxTFmlnPgICQBNuYY0Q.png?1715157704)

- 아래와 같이 표시되는 Tax ID 팝업에서:

해당 지역에 따라 세금 ID 유형을 선택합니다. (국가별 올바른 지역 선택에 대한 자세한 가이드는 아래에서 확인할 수 있습니다.)

- 세금 ID 번호를 입력합니다.

![세금 ID 팝업](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155025689848/original/TqXTvTFBDEZmPbwQZgvi5cL8xvZpSfuMiw.png?1715157900)

- Save(저장)를 선택합니다.

다음 경우에는 'I do not have a Tax ID' 체크박스를 선택하세요:
- 세금 ID를 보유하고 있지만 아래 목록에 없는 경우
- 세금 ID 번호가 없는 경우  
- 세금 ID를 공개하고 싶지 않은 경우

여러 세금 ID를 보유한 경우, 어떤 것을 사용해야 하는지 지원팀에 문의하세요. 예를 들어, "ca_bn", "ca_gst_hst", "ca_pst_bc"를 모두 보유한 경우, 지원팀은 HST와 PST 모두에서 면제받을 수 있는 유일한 ID인 "ca_gst_hst" ID를 사용하라고 안내할 것입니다.
