# 이메일 서명

Hyperclass의 이메일 서명 기능을 통해 팀에서 발송하는 모든 이메일을 일관성 있고 전문적으로 관리할 수 있습니다. 각 팀원이 개별 서명을 설정할 수 있어, 1:1 이메일과 자동 발송 이메일 모두에서 브랜딩과 연락처 정보를 일관되게 유지할 수 있습니다.

# **이메일 서명이란**

이메일 서명은 하위 계정의 각 팀원별로 저장된 서명을 만드는 기능입니다. 서명에는 이름, 직책, 전화번호, 링크 등의 서식이 적용된 텍스트를 포함할 수 있으며, 브랜딩에 맞게 완전히 커스터마이징할 수 있습니다.

이메일 서명은 발신 이메일에 자동으로 적용되거나(활성화 시), 플랫폼 전반에서 커스텀 값을 사용하여 수동으로 삽입할 수도 있습니다. 서명을 올바르게 설정하면 Hyperclass에서 발송하는 모든 이메일에 적절한 발신자 정보가 항상 포함되도록 할 수 있습니다.

---

## 이메일 서명의 주요 장점

- **전문적인 브랜딩:** 모든 이메일에 비즈니스 정체성, 역할 세부사항, 링크를 일관되게 유지합니다.

- **사용자별 개인화:** 각 팀원이 자신의 이름, 직책, 연락처 정보를 사용할 수 있습니다.

- **유연한 사용:** 발신 이메일에 서명을 자동 적용하거나, 커스텀 값을 사용하여 필요한 곳에 삽입할 수 있습니다.

- **깔끔한 답장:** 이메일 스레드에 답장할 때 인용 텍스트 앞에 서명이 나타나는지 제어할 수 있습니다.

---

## 이메일 서명 설정 및 편집하기

다음 단계를 따라 팀원의 서명을 업데이트하세요.

#### **1단계:** My Staff(내 팀) 이동

하위 계정에서 Settings(설정)으로 이동한 후 **My Staff(내 팀)**를 선택하세요.

![이메일 서명 설정 - 내 팀 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155065531605/original/8nf18wMx7bI-oBTsXuov-QmUMwq8KXiGgQ.png?1771873155)

#### **2단계:** 팀원 선택

업데이트할 팀원을 찾아서 **Edit(편집)** 버튼(연필 아이콘)을 클릭하세요.

![이메일 서명 편집 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155065531665/original/kvMynXvy1jSblJMSikV0N4DnHo9_eM_S6w.png?1771873238)

#### **3단계:** 서명 업데이트

**Signature(서명)** 섹션까지 스크롤하세요. 편집기에서 서명 텍스트와 서식을 업데이트할 수 있습니다. 더 많은 커스터마이징 옵션을 위해 코드를 삽입할 수도 있습니다.

![이메일 서명 편집기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155065531867/original/85Wgx9wooRoaTLnwYi802TPYCoARZgnIyQ.png?1771873633)

#### **4단계:** 선택적 설정 구성

- 발신 이메일에 서명을 자동으로 적용하려면 **Enable Signature in All Outgoing Messages(모든 발신 메시지에 서명 활성화)**를 켜세요.

- 답장에서 인용된 이메일 스레드 위에 서명이 나타나게 하려면 **Include This Signature Before Quoted Text in Replies(답장 시 인용 텍스트 앞에 이 서명 포함)**에 체크하세요.

![이메일 서명 옵션 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155065531938/original/fWyolA4-iqjPgEjn2I0KmSixI-Ca4Nvogg.png?1771873758)

#### **5단계:** 변경사항 저장

파란색 Next 버튼을 클릭한 후 파란색 **Save(저장)** 버튼을 클릭하여 이메일 서명 변경사항을 적용하세요.

![이메일 서명 저장](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155065532039/original/-mveQJq9T-nOOmoMzI4wmfuxupDpLhu-aQ.png?1771873853)

## 커스텀 값을 사용하여 사용자 서명 삽입하기

커스텀 값으로 서명을 배치하면 이메일에서 서명이 나타나는 위치를 더 세밀하게 제어할 수 있습니다. 이는 캠페인 이메일 같은 템플릿이나 1:1 이메일에서 특정 위치에 서명을 표시하고 싶을 때 유용합니다.

- 서명을 추가하고 싶은 이메일 편집기를 여세요(예: 캠페인 또는 대화 이메일 작성기).

- **Custom Values(커스텀 값)**를 클릭하세요.

- **User(사용자)**로 이동한 후 **Signature(서명)**를 선택하세요.

이메일 서명에 사용되는 커스텀 값은 다음과 같습니다: {{user.email_signature}}

![커스텀 값으로 서명 삽입](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155065532251/original/m1xtGeQ9pLGalUVYZ0T0UYZt3_Ihu38SDg.png?1771874099)

## **자주 묻는 질문**

**Q: 이메일 서명은 사용자별로 설정되나요, 아니면 하위 계정별로 설정되나요?**

이메일 서명은 하위 계정 내 팀원별로 저장됩니다. 각 팀원이 서로 다른 서명을 가질 수 있습니다.

**Q: 서명을 업데이트했는데 이메일 발송 시 보이지 않습니다. 무엇을 확인해야 하나요?**

올바른 팀원을 업데이트했는지 확인하고, *모든 발신 메시지에 서명 활성화*가 켜져 있는지 확인하세요.

**Q: 서명에 서식(굵게, 링크 등)을 적용할 수 있나요?**

네. 서명 편집기는 기본 서식을 지원하므로 텍스트 스타일을 적용하고 링크를 포함할 수 있습니다.
