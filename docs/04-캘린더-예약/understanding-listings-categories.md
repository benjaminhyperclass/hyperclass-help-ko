# 리스팅과 카테고리란?

**리스팅**은 여러분이 제공하는 개별 렌털 항목을 의미합니다. 예를 들어 부동산, 차량, 객실, 장비 등이 될 수 있습니다. 각 리스팅에는 고유한 이미지, 설명, 가격, 재고(선택사항), 상태가 포함됩니다. 리스팅은 렌털 설정의 기본 구조입니다.

**카테고리**는 비슷한 리스팅들을 그룹화하여 내부 사용자와 고객 모두에게 체계적인 정리를 제공합니다.

**예시:**
- 부동산 관리업체는 해변가 주택, 도심 로프트, 산악 오두막 같은 카테고리를 사용할 수 있습니다.
- 장비 렌털 회사는 카메라나 조명 장비 같은 카테고리로 특정 장비 리스팅을 정리할 수 있습니다.

---

## 리스팅과 카테고리의 주요 장점

- **체계적 정리**: 카테고리를 통해 대규모 렌털 인벤토리를 쉽게 탐색하고 관리할 수 있습니다.
- **공개 레이아웃 제어**: 카테고리 순서를 변경하면 별도 설정 없이도 예약 페이지의 그룹화와 순서가 바뀝니다.
- **간편한 공유**: 리스팅이나 카테고리 링크를 공유하여 고객이 원하는 것을 정확히 찾을 수 있게 하여 의사결정을 빠르게 하고 문의를 줄입니다.
- **맞춤 설정**: 가격, 예약 설정, 캘린더 동기화에 대한 구성 세부사항을 설정할 수 있습니다.

## 리스팅 패널 이해하기

리스팅 패널은 모든 렌털 리스팅과 세부사항을 한눈에 볼 수 있는 화면을 제공합니다.

#### 리스팅 패널 구성:

- **Image(이미지)** – 리스팅의 썸네일을 표시합니다.
- **Name(이름)** – 리스팅 제목입니다.
- **Status(상태)** – 활성 또는 비활성 상태입니다.
- **Base Price(기본 가격)** – 시간/일/주/월당 렌털 가격을 보여줍니다.
- **Stock(재고)** – 사용 가능한 유닛 수입니다.
- **Actions(작업)** – 리스팅 편집, 복제, 공유, 삭제 기능입니다.

![리스팅 패널 구성](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155056487135/original/DxaSA9sKi0wSLYBlTPK8h2_45ui-aoGx7A.png?1761055859)

#### 도구 모음 컨트롤:

- **Search Bar(검색창)** – 이름이나 키워드로 리스팅을 빠르게 찾습니다.
- **Sort by Dropdown(정렬 드롭다운)** – 이름 오름차순 또는 내림차순으로 정렬합니다.

![도구 모음](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155056487165/original/r633n8KLcFwQr5qJD2tnpv3llKvpFR-7MQ.png?1761055896)

#### 카테고리 패널:

- 왼쪽에 표시되며, 모든 기존 카테고리를 나열합니다.
- 카테고리를 선택하면 해당 그룹의 항목만 필터링하여 표시됩니다.
- 각 카테고리 옆의 **⋮ 메뉴**를 통해 카테고리를 **편집**, **공유**, **삭제**할 수 있습니다.

**팁**: 왼쪽에서 카테고리 순서를 변경하면 고객 예약 페이지에서의 순서도 업데이트됩니다.

![카테고리 패널](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155056487240/original/Mhov4eOv9CvshH-B0RZJIwGiC0Dbxtf0nw.png?1761055928)

#### 행 작업:

- **✏️ Edit(편집)** – 리스팅 편집기를 열어 설정을 수정합니다.
- **📋 Duplicate(복제)** – 리스팅을 복제하여 구성을 재사용합니다.
- **🔗 Share(공유)** – 해당 리스팅의 공개 예약 URL을 복사합니다.
- **🗑️ Delete(삭제)** – 리스팅을 영구적으로 제거합니다.

![행 작업 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155056487372/original/FlT-xRB29EzvGZfA2KZxDnI5_zgB1xFiOg.png?1761056003)

## 새 카테고리 만들기

생성한 각 카테고리는 사이드바에 표시되어 빠른 접근이 가능하며, 여러 리스팅을 포함할 수 있습니다.

**팁**: 깔끔한 UI를 위해 카테고리 이름은 짧고 명확하게 유지하세요(예: *럭셔리 카*, *이코노미 차량*).

- **Rentals(렌털) → Listings(리스팅)**로 이동하세요.

![렌털 리스팅 이동](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155056487452/original/YHPTOA9pKNXeVtkA4TqF4CSyXopk2bHsog.jpeg?1761056062)

- 카테고리 패널 하단의 **+ New Category(+ 새 카테고리)**를 클릭하세요.

![새 카테고리 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155056487518/original/Z7K5zP8lvbPuNmKSgeZoRUNg7gvX1nHJnw.png?1761056088)

- **Add New Category(새 카테고리 추가)** 모달에서 다음을 입력하세요:
  - Category Name(카테고리 이름) (필수)
  - Custom URL Slug(커스텀 URL 슬러그) – 공유 가능한 링크를 형성합니다.

![새 카테고리 모달](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155056487614/original/JV8KLvcCXSwqTL3vpBWapRlrThoBs39bBQ.png?1761056147)

- **Create Category(카테고리 생성)**를 클릭하세요.

![카테고리 생성 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155056487673/original/s2Mc3vnTNSZvNSGz1o7HFj6pZDchaNzMnQ.png?1761056170)

## 카테고리 관리 및 공유

#### 카테고리 편집

- 카테고리 이름 옆의 **⋮ 메뉴**를 클릭한 후 **Edit(편집)**을 선택하세요.

![카테고리 편집 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155056487908/original/CXjnUrumMvwhjItBj2RjorBVjMaIKbL6yg.png?1761056315)

- **이름**이나 **슬러그**를 업데이트하고 변경사항을 저장하세요.

![카테고리 편집 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155056487985/original/tyLR4jFlNQUFyz0fIs5pp8L28mimWxOmhA.png?1761056347)

#### 카테고리 공유

- **Share Category(카테고리 공유)** 옵션을 사용하여 해당 카테고리로 필터링된 공개 예약 링크를 공유하세요.

![카테고리 공유](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155056488859/original/IhpHMRdkbvPW7GCy_a5qhIbyLGjNAe_kQg.png?1761056731)

#### 카테고리 삭제

- **⋮**를 클릭한 후 **Delete(삭제)**를 선택하세요.
- 카테고리가 삭제되면 해당 카테고리에 그룹화된 리스팅들은 기본 카테고리에 연결됩니다.

![카테고리 삭제](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155056488929/original/ohpTqp667rObEtSTNgksOt-bJGn6dGhs2A.png?1761056792)

#### 카테고리 순서 변경

- 왼쪽 패널에서 카테고리를 드래그 앤 드롭하면 고객이 예약 페이지에서 보는 순서가 동일하게 변경됩니다.

![카테고리 순서 변경](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155056489089/original/dtnXS4Qrmco6nBqunk_SDxjm9V6Ir_gP8A.png?1761056872)

## 카테고리가 리스팅에 미치는 영향

- 각 리스팅은 반드시 하나의 카테고리에 속해야 합니다. 이를 생략하면 **Default Category(기본 카테고리)**로 자동 할당됩니다.
- 카테고리의 순서가 공개적으로 표시되는 리스팅의 순서를 결정합니다.

## 자주 묻는 질문

**Q: 리스팅이 여러 카테고리에 속할 수 있나요?**
A: 아니요. 각 리스팅은 하나의 카테고리에만 할당됩니다.

**Q: 카테고리 페이지를 공개적으로 공유할 수 있나요?**
A: 네. **Share Category(카테고리 공유)**를 사용하여 해당 그룹의 직접 예약 링크를 복사할 수 있습니다.

**Q: 카테고리를 삭제하면 어떻게 되나요?**
A: 카테고리가 삭제되면 해당 카테고리에 그룹화된 리스팅들은 기본 카테고리에 연결됩니다.

**Q: 몇 개의 카테고리를 만들 수 있나요?**
A: 계정당 최대 50개의 카테고리를 생성할 수 있습니다.

**Q: 카테고리가 가격이나 캘린더 가용성에 영향을 주나요?**
A: 아니요. 가격과 가용성은 리스팅별 또는 변형별로 설정됩니다.
