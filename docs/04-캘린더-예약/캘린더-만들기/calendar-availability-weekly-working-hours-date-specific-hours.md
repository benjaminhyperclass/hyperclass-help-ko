## **캘린더 예약 가능 시간이란?**

캘린더 예약 가능 시간은 캘린더가 예약을 받을 수 있는 시간대를 정의합니다. 주간 근무 시간은 반복되는 주간 패턴을 만들고, 날짜별 시간은 개별 날짜(예: 휴일, 특별 이벤트)에 대해 해당 패턴을 재정의합니다. 이 두 설정이 함께 작동하여 캘린더 예약 페이지에서 어떤 시간대가 예약 가능한지를 제어합니다.

이 두 개념의 차이점을 이해하면 캘린더를 효과적으로 관리하고 실시간 가용성과 일치시킬 수 있습니다.

**중요:** 날짜별 시간은 항상 주간 근무 시간보다 우선됩니다. 즉, 특정 날짜에 대해 날짜별 시간을 설정하면 해당 날의 기본 주간 근무 시간이 무시됩니다.

## **주간 근무 시간**

주간 근무 시간은 반복적인 기준으로 정기적인 예약 가능 시간을 나타냅니다. 매주 반복되는 기본 일정을 설정하는 데 유용합니다.

예를 들어, 매주 월요일, 화요일, 목요일 오전 8시부터 오후 10시까지 일관되게 예약을 받을 수 있다면, 이 시간을 주간 근무 시간으로 설정할 수 있습니다. 시스템은 이 일정을 해당 요일과 시간에 매주 자동으로 적용합니다.

### 주간 근무 시간 설정하는 방법

- 캘린더(Calendars) → 캘린더 설정(Calendar Settings)으로 이동하여 캘린더 편집을 클릭합니다.

![주간 근무 시간 설정하는 방법](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063381466/original/1IsYwx3nVmjdY4ZC7zhtyjMPtKK_U4qhCw.png?1769199652)

- 왼쪽 메뉴에서 **예약 가능 시간(Availability)**을 선택합니다. 예약 가능 시간 섹션에서 사용자 이름을 클릭하여 사이드 패널을 엽니다.

![예약 가능 시간 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063381580/original/a3brAjglqejecXiTd_b2mxxSA8RwLfTs0Q.png?1769199836)

- **주간 근무 시간(Weekly Working Hours)**에서 **편집(Edit)**을 클릭하고 각 요일별로 예약 가능 시간을 설정합니다.

![주간 근무 시간 편집](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063381637/original/q0gm9Vjn8ffVqJAL-0U-h5DxTv8Fk4lFlA.png?1769199937)

- (선택사항) 복사를 클릭하여 설정을 다른 요일에도 적용할 수 있습니다.

![설정 복사](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063381663/original/EDq6a6JQR0SZTlef5vw-mzCTebnJbMR-jA.png?1769200013)

- **파란색 저장** 버튼을 클릭하여 변경사항을 적용합니다.

## 날짜별 시간

날짜별 시간은 특정 날짜에 대해 예약 가능 시간이나 불가능 시간을 사용자 정의할 수 있게 해줍니다. 특정 날짜를 추가하고 해당 날짜에만 적용되는 시간을 정의할 수 있습니다.

이 기능은 특별한 상황, 휴일 또는 개인적인 약속을 처리할 때 특히 유용합니다. 몇 가지 사용 예시를 들어보겠습니다:

- **휴일**: 부활절, 크리스마스, 공휴일과 같은 다가오는 휴일을 예약 불가로 표시할 수 있습니다. 이러한 특정 날짜에는 캘린더에 예약 불가 상태가 반영되어 정확한 일정 관리를 보장합니다.

- **개인 약속**: 정규 일정과 다른 사전 개인 약속이 있는 경우, 해당 특정 날짜의 예약 가능 시간을 쉽게 조정할 수 있습니다. 예를 들어, 2024년 2월 10일 오후 12시부터 3시까지 병원 예약이 있다면, 별도의 블록 슬롯을 만들지 않고도 해당 시간을 예약 불가로 표시할 수 있습니다.

- **연장 가능 시간**: 휴일 주말과 같은 특별한 경우에 제한된 기간 동안 근무 시간을 연장하고 싶다면, 날짜별 시간을 사용하여 해당 특정 날짜에만 예약 가능 시간을 수정할 수 있습니다.

날짜별 시간은 해당 날짜의 주간 근무 시간을 완전히 재정의합니다. 즉, 날짜별 시간이 존재하는 날짜에는 주간 근무 시간이 전혀 적용되지 않습니다.

**예시 (하루 연장하기)**: 주간 근무 시간이 오전 9시~오후 5시인데, 화요일(3월 10일)에 오전 9시~오후 8시까지 예약을 받고 싶다면, 해당 날짜에 오전 9시~오후 8시로 날짜별 시간을 설정해야 합니다. 오후 5시~8시만 날짜별 시간으로 설정하면, 시스템은 화요일을 오후 5~8시에만 예약 가능한 것으로 처리하고 오전 9시~오후 5시는 예약 불가로 표시됩니다.

### 날짜별 시간 설정하는 방법

- 캘린더 예약 가능 시간 설정으로 이동합니다: 설정(Settings) → 캘린더(Calendars) → [캘린더 선택] → 예약 가능 시간(Availability)

- 날짜별 시간 섹션에서 + 날짜별 시간 추가를 클릭합니다.

![날짜별 시간 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155067030845/original/WOAl5x3tXwtd2eKZKYYWCk0jZDI-zMee4A.png?1773669406)

- 날짜 선택기에서 재정의하려는 하나 이상의 날짜를 선택합니다.

![날짜 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155067030908/original/5-e72I4fUtIURqa9DY_CPcYFhCCNnlkDiw.png?1773669449)

- 시작 시간과 종료 시간을 설정합니다.

- 저장을 클릭하여 선택한 모든 날짜에 동일한 시간을 적용합니다.

참고: 이 기능은 출시 과정에서 특정 계정에만 제공될 수 있습니다.

### **날짜 그룹핑 작동 방식**

여러 날짜에 대해 날짜별 시간을 저장하면, Hyperclass는 목록을 체계적으로 유지하기 위해 선택사항을 그룹화합니다:

- 연속된 날짜는 하나의 그룹화된 범위로 표시됩니다.

- 연속되지 않은 날짜는 별도의 그룹화된 범위로 표시됩니다.

**예시:** 1월 2, 3, 4, 5, 8, 9일을 선택한 경우:

- 1월 2~5일은 하나의 그룹화된 항목으로 표시됩니다.

- 1월 8~9일은 또 다른 그룹화된 항목으로 표시됩니다.

나중에 각 그룹화된 항목을 독립적으로 편집하거나 삭제할 수 있습니다.

## 자주 묻는 질문

**Q: 격주(2주마다) 반복 예약 가능 일정을 설정할 수 있나요?**

현재 캘린더 설정은 주간 반복 일정만 지원합니다. 격주 패턴을 따르는 예약 가능 시간이 있다면, **날짜별 시간**을 사용하여 해당하는 각 날짜에 대해 사용자 정의 예약 가능 시간을 정의하는 것이 가장 좋은 방법입니다. 수동 설정이 필요하지만, 캘린더가 정확한 근무 패턴을 반영하도록 보장합니다.

2주마다 캘린더를 업데이트하는 일정 알림을 만들거나, 앞으로 몇 달에 대해 미리 날짜별 시간을 대량으로 설정하는 것을 권장합니다.

**Q: 날짜별 시간을 추가할 때 주간 근무 시간이 사라지는 이유는 무엇인가요?**

날짜별 시간이 우선순위를 가지며 해당 날짜의 주간 근무 시간을 완전히 대체합니다. 하루를 "연장"하려면 추가 시간(예: 5~8시)만이 아니라 전체 예약 가능 시간 범위(예: 9~8시)를 입력해야 합니다.
