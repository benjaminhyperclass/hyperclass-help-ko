# 모바일 앱에서 서비스 캘린더 사용하기

모바일 앱의 최신 업데이트 소식을 알려드립니다 - 서비스 캘린더(Service Calendars) 기능이 추가되었습니다. 이제 하위 계정 관리자와 사용자들이 캘린더 섹션에서 모든 서비스 예약을 확인할 수 있고, 연락처와 새로운 서비스 예약을 만들 수 있습니다. 이 업데이트는 HighLevel, LeadConnector, 그리고 화이트라벨 모바일 앱(업데이트 요청 필요)에서 사용할 수 있습니다.

## **새로운 기능**

- 서비스 캘린더 예약 확인
- 새로운 서비스 캘린더 예약 생성

## **사용 방법**

- 원하는 하위 계정을 선택한 상태로 모바일 앱을 열어주세요.
- 캘린더 섹션으로 이동하세요. 해당 날짜별로 모든 서비스 예약을 확인할 수 있습니다.
- 새로운 서비스 예약을 만들려면 "+" 버튼을 누르고 'Schedule appointment'를 선택하세요.
- 캘린더 드롭다운에서 웹에서 생성한 서비스 캘린더를 선택하세요.
- 해당 서비스의 관련 룸을 선택해야 합니다. 룸을 선택하면 가용성에 따라 장비가 자동으로 배정됩니다.
- 기타 관련 정보를 모두 입력하고 저장을 누르면 예약이 성공적으로 등록됩니다!

**왜 중요한가요**
이 업데이트를 통해 마침내 사용자들이 모바일 앱에서도 서비스 예약을 확인하고 생성할 수 있게 되었습니다.

## **지원 버전**

- HighLevel (iOS, Android) - v3.74 이상
- LeadConnector (iOS, Android) - v3.74 이상
- WhiteLabel (iOS, Android) - v3.74 이상
