# 캘린더 위젯 커스터마이징이란?

캘린더 위젯 커스터마이징(Calendar Widget Customization)은 위젯의 외관과 기능을 직접 제어할 수 있는 기능입니다. 이 강력한 기능으로 위젯을 개인 취향에 맞게 개인화하여 나만의 고유한 위젯을 만들 수 있어요.

---

## 위젯을 커스터마이징하는 방법

- Calendars(캘린더) → Calendar Settings(캘린더 설정)으로 이동하세요.

- 커스터마이징하고 싶은 캘린더의 Edit(편집)을 클릭하세요.

- Advanced settings(고급 설정) → Widget appearance(위젯 외관)를 열어 위젯 커스터마이징 옵션을 업데이트하세요.

**참고**: 위젯 커스터마이징은 **NEO 위젯에서만** 작동합니다.

---

## 커스터마이징 옵션 이해하기

캘린더 위젯을 취향에 맞게 맞춤 설정할 수 있는 다음 옵션들을 살펴보세요:

- **Primary Colour(기본 색상)**: 버튼, 날짜, 시간 슬롯, Staff Selection(직원 선택), Add Guests(게스트 추가) 등 위젯 내 다양한 액션에 영향을 줍니다.

- **Background Colour(배경 색상)**: 날짜-시간 선택기와 폼 모두의 배경 색상을 설정합니다. **커스텀 폼**을 사용하는 경우, 폼의 배경은 선택한 배경 색상과 일치하지만 폼 필드는 특정 커스텀 폼 설정에 따라 결정됩니다.

## 자주 묻는 질문(FAQ)

**Q: 어떤 캘린더가 커스터마이징을 지원하나요?**

A: Event(이벤트), Round Robin(라운드 로빈), Class Booking(수업 예약), Collective(공동 예약), Service Calendars(서비스 캘린더) 등 모든 캘린더 유형에서 커스터마이징 기능을 사용할 수 있습니다.

**참고**: **Service Menu(서비스 메뉴)**에서는 커스터마이징이 작동하지 **않습니다**.

**Q: 커스텀 폼을 추가할 수 있나요?**

A: 네! 커스텀 폼을 추가할 수 있습니다. 다만 커스텀 폼 테마가 전체 위젯 커스터마이징과 잘 어우러지도록 하세요.

**Q: CSS 커스터마이징을 추가한 경우는 어떻게 되나요?**

A: CSS 커스터마이징이 위젯 설정보다 우선적으로 적용됩니다. 커스텀 스타일이 우선순위를 갖기 때문에 캘린더 위젯에서 개인화된 디자인 선택이 그대로 반영됩니다.

**Q: 좌측 패널 전체를 비활성화하고 싶은데 설정을 꺼도 여전히 보여요. 어떻게 해야 하나요?**

A: 좌측 패널을 완전히 비활성화하려면 다음 설정을 확인하세요:

캘린더 세부사항 설정:
- Calendar Name(캘린더 이름) 비활성화
- Calendar Description(캘린더 설명) 비활성화  
- Calendar Details(캘린더 세부사항) 비활성화

로고 제거:
- 캘린더와 연결된 로고 제거

Customizations(커스터마이징) 탭:
- "Allow Select Staff(직원 선택 허용)" 옵션 비활성화
