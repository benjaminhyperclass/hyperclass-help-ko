# 미할당 캘린더를 그룹에 추가하기

이제 **미할당 캘린더(Unassigned Calendar)** 유형뿐만 아니라 다른 **라운드 로빈(Round-robin)** 캘린더에도 그룹을 할당할 수 있습니다.

`Settings(설정) > Calendars(캘린더)`로 이동하세요:

![설정 메뉴에서 캘린더 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48275919878/original/gUC1FoTf1TkFGD4EjVwNtoSiNONwucJNGw.png?1673969124)

원하는 캘린더 오른쪽의 **점 3개** 아이콘을 클릭한 후 Move to Group(그룹으로 이동)을 클릭하세요:

![캘린더 옵션 메뉴에서 그룹으로 이동 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48275920143/original/IaGc2orq1dFPZYTiaRHLr7vXXpmjMra8hA.png?1673969175)

팝업 창이 열리면서 캘린더에 할당할 그룹을 선택하라고 나타납니다. 원하는 그룹을 선택한 후 **Select(선택)** 버튼을 클릭하세요:

![그룹 선택 팝업 창](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48275920666/original/1F0J1bCyUySo4ajbnqGnMjKzfasxNqZjOw.png?1673969263)
