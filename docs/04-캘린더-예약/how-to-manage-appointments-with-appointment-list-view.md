# 예약 목록 보기로 예약 관리하는 방법

Hyperclass의 예약 목록 보기(Appointment List View)는 기존 캘린더 레이아웃 대신 목록 기반 작업 공간을 사용하여 예약을 관리하는 강력한 방법을 제공합니다. 스마트 리스트를 활용하여 예약을 대량으로 필터링하고, 정렬하고, 커스터마이징하고, 작업을 수행할 수 있습니다. 이 보기는 여러 캘린더와 사용자에 걸쳐 예약을 효율적으로 관리해야 하는 관리자, 영업팀, 지원팀에게 이상적입니다.

## 예약 목록 보기(스마트 리스트)란 무엇인가요?

예약 목록 보기는 날짜 기반 캘린더 그리드가 아닌 구조화된 목록으로 예약을 표시하는 Hyperclass의 전용 캘린더 뷰입니다. 대량의 예약 관리를 위해 설계되었으며, 사용자는 필터, 정렬, 컬럼 구성이 결합된 저장된 스마트 리스트를 활용할 수 있습니다.

스마트 리스트는 예약 목록 보기를 수동적인 뷰가 아닌 운영 작업 공간으로 만들어줍니다. 사용자는 목록에서 직접 예약을 생성하고, 업데이트하고, 관리할 수 있어 우선순위를 정하고, 결과를 추적하고, 팀 간 일관성을 유지하기가 더 쉬워집니다.

---

## 예약 목록 보기의 주요 장점

예약 목록 보기는 팀이 예약 스케줄링에서 유연하고 재사용 가능한 워크플로우를 통한 적극적인 관리로 전환할 수 있도록 도와줍니다.

- **중앙 집중식 관리:** 일별 뷰를 탐색하지 않고도 여러 캘린더의 예약을 보고 관리할 수 있습니다.

- **스마트 리스트:** 필터, 정렬, 컬럼을 함께 재사용 가능한 예약 워크플로우로 저장합니다.

- **고급 필터링:** 상태, 담당자, 캘린더, 날짜 범위 등으로 예약 목록을 세밀하게 필터링합니다.

- **커스터마이징 가능한 컬럼:** 역할이나 작업에 따라 어떤 예약 세부 정보가 보이는지 제어합니다.

- **액션 중심 워크플로우:** 목록에서 직접 상태 업데이트, 편집, 일정 변경, 예약 생성을 수행합니다.

## 예약 목록 보기로 예약 관리하는 방법

### 예약 목록 보기 접근하기

예약 목록 보기는 캘린더(Calendars) 영역에서 직접 접근할 수 있으며, 캘린더 뷰와 함께 주요 작업 공간으로 기능합니다.

- 왼쪽 메뉴에서 **캘린더(Calendars)**로 이동합니다

- 상단의 **Appointment List View** 탭을 선택합니다

![예약 목록 보기 접근](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063057237/original/dmswuRIbFPYkVfbnQqeT0gayY0E1Jsgmpg.png?1768911244)

### 예약 유형

예약 유형은 예약에 사용된 캘린더에 따라 자동으로 결정됩니다. 각 캘린더는 미팅, 서비스 또는 렌탈로 구성되며, 예약은 예약 시 해당 유형을 상속받습니다.

예약 목록 보기는 선택한 유형에 따라 예약을 표시하므로, 올바른 예약 유형을 선택하면 의도한 예약을 확실히 볼 수 있습니다.

![예약 유형 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063058293/original/rtkXUL9kLwwAXfCzaV9R7IwRwnzsV-kD6w.png?1768911686)

### 새 예약 만들기

**새 예약(New Appointment)** 버튼을 사용하면 예약 목록 보기에서 직접 예약을 수동으로 생성할 수 있습니다. 이는 통화 후나 내부 조율 중에 연락처를 대신해서 예약을 잡을 때 유용합니다.

**새 예약**을 클릭하면 수동 예약 플로우가 열리며, 적절한 캘린더를 선택하고, 날짜와 시간을 선택하고, 예약을 기존 CRM 연락처와 연결하거나 필요시 새 연락처를 만들 수 있습니다. 그러면 예약이 선택한 예약 유형과 스마트 리스트 조건에 따라 목록에 추가됩니다.

![새 예약 만들기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063172569/original/LbV-tEXJnJPQknsUYa3GCE3Sey8RtPdStQ.png?1769007722)

### 예약 목록 및 스마트 리스트 만들기

기본 목록인 **예정(Upcoming)**, **취소됨(Cancelled)**, **전체(All)**는 자주 사용하는 예약 뷰에 빠르게 접근할 수 있게 해줍니다.

더 구체적인 것이 필요할 때는 **+ 스마트 리스트**를 클릭하여 커스텀 뷰를 만들 수 있습니다. 이렇게 하면 자신만의 필터, 정렬, 컬럼 설정을 저장할 수 있습니다. 스마트 리스트를 사용하면 매번 목록을 다시 구성할 필요 없이 다양한 예약 관리 방식을 쉽게 전환할 수 있습니다.

![스마트 리스트 만들기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063133539/original/IzFgl4V1gz7ogAFZrG1XzIY_MpvDAj3knA.png?1768987933)

### 목록 커스터마이징

커스터마이징을 통해 어떤 예약이 나타나는지, 어떻게 우선순위가 매겨지는지, 어떤 정보가 보이는지를 제어할 수 있습니다.

#### 목록 커스터마이징 열기

- **목록 커스터마이징(Customize List)**을 클릭하여 설정 패널을 엽니다

이 패널은 스마트 리스트의 제어 센터 역할을 합니다.

![목록 커스터마이징](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063134494/original/8mt4CRcdBR-eteAWy-Y6H4llLpqaADi-dQ.png?1768988523)

### 목록 커스터마이징 패널

**목록 커스터마이징** 패널은 예약 목록이 어떻게 구성되고 저장되는지를 제어하는 곳입니다. 여기에서 현재 목록의 이름을 변경하고, **고급 필터(Advanced Filters)**를 적용하여 나타날 예약을 좁히고, 예약이 어떻게 **정렬**되는지 선택하고, 목록에 표시될 **컬럼**을 관리할 수 있습니다.

이 패널에서 변경한 사항은 현재 목록 뷰에 영향을 미치며 스마트 리스트로 저장할 수 있어, 예약 목록 보기로 돌아올 때마다 동일한 설정을 재사용할 수 있습니다.

![목록 커스터마이징 패널](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063134521/original/02Hz6Czn7NRNKNGSBoBdavBmUhkMzf2d-w.png?1768988550)

### 고급 필터 적용

고급 필터는 예약 목록 보기에 어떤 예약이 나타나는지를 제어하며, 워크플로우에 중요한 예약만으로 목록을 좁힐 수 있게 해줍니다. 소유권, 상태, 시간, 캘린더 소스 등 특정 조건에 집중하기 위해 하나 또는 여러 조건을 적용할 수 있습니다.

일반적인 필터 옵션은 다음과 같습니다:

- **예약 상태** (예: 확정됨, 취소됨, 노쇼)

- **배정된 사용자** - 특정 팀원의 예약 보기

- **캘린더** - 결과를 하나 이상의 캘린더로 제한

- **예약 날짜 또는 시간** - 특정 범위에 집중

- **연락처 세부 정보** - 연결된 연락처로 필터링

필터는 **AND 논리**를 사용하여 결합할 수 있습니다. 즉, 예약이 선택된 모든 조건을 만족해야 나타납니다. 이렇게 하면 특정 사용자의 정의된 날짜 범위 내 예정된 예약과 같은 정확한 뷰를 쉽게 만들고, 지속적인 관리를 위해 스마트 리스트로 재사용할 수 있습니다.

![고급 필터 적용](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063162282/original/n5yfK0FWjh7V_nzweKjbV-n0BnZmzlvIDg.png?1769003306)

### 필터 추가

**필터 추가(Add Filter)**를 사용하여 추가 조건을 포함하고 목록을 더 세밀하게 조정하세요. 필터는 **AND 논리**를 사용하여 결합되므로, 선택된 모든 조건을 만족하는 예약만 나타납니다. 이렇게 하면 목록을 매우 구체적인 예약 세트로 좁힌 후 구성을 스마트 리스트로 저장하기가 쉬워집니다.

![필터 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063161743/original/3ZAopkZU4qNn4w9dNQtHYjIxucH56tYSTg.png?1769003041)

### 정렬 적용

정렬은 예약이 목록에 나타나는 순서를 제어하며 스마트 리스트의 일부로 저장됩니다.

일반적인 정렬 옵션은 다음과 같습니다:

- 예약 시간 (오름차순 또는 내림차순)

- 추가 날짜 (오름차순 또는 내림차순)

정렬은 긴급성이나 워크플로우 필요에 따라 예약의 우선순위를 정하는 데 도움이 됩니다.

### 컬럼 관리

컬럼은 목록에서 어떤 예약 세부 정보가 보이는지를 결정합니다.

다음과 같은 작업이 가능합니다:

- 컬럼 표시 또는 숨김

- 드래그 앤 드롭을 사용하여 컬럼 순서 변경

- 제거할 수 없는 잠긴 시스템 컬럼 보기

컬럼 구성은 스마트 리스트의 일부로 저장됩니다.

![컬럼 관리](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063170813/original/jYXjCD5QH6yIjo_P12lUoJdrL4PartU6pg.png?1769006923)

### 예약 상태 업데이트

예약 목록 보기에서는 예약 기록을 열지 않고도 목록에서 직접 예약 상태를 업데이트할 수 있습니다. 예약 행의 **상태(Status)** 드롭다운을 클릭하여 **확정됨(Confirmed)**, **참석함(Showed)**, **노쇼(No Show)**, **취소됨(Cancelled)**, **유효하지 않음(Invalid)** 등 적절한 상태를 선택하세요.

여기서 변경한 내용은 즉시 저장되며 Hyperclass 전체에 반영되므로, 대량의 예약을 관리하면서도 예약 결과를 정확하게 유지하기가 쉬워집니다.

![예약 상태 업데이트](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063174660/original/Dn-Lq5L7zdqI7cukzbuuEwrGotppKOycpw.png?1769008626)

### 행 액션을 사용한 예약 관리

예약 목록 보기의 각 예약에는 **점 세 개(⋮) 메뉴**가 포함되어 있어 추가 액션에 빠르게 접근할 수 있습니다. 이 메뉴를 클릭하면 **예약 세부 정보 보기**, **예약 편집**, **일정 변경**, 필요시 **삭제** 옵션이 열립니다. 동의 추적이 활성화되어 있다면 **동의 보기(View Consent)** 옵션도 볼 수 있습니다.

- **세부 정보 보기:** 모든 예약 정보와 활동을 검토할 수 있는 전체 예약 기록을 엽니다.

- **동의 보기:** 동의 추적이 활성화되어 있을 때 예약과 관련된 동의 세부 정보를 표시합니다.

- **편집:** 연락처, 메모, 배정된 캘린더 등 예약 세부 정보를 업데이트할 수 있습니다.

- **일정 변경:** 동일한 예약 기록을 유지하면서 예약 날짜와 시간을 변경합니다.

- **삭제:** 캘린더와 예약 목록 보기에서 예약을 영구적으로 제거합니다.

![행 액션 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063175156/original/gV-oOP9hVqXaNLm00B5IN56yRc9zDY3w5A.png?1769008745)

## 자주 묻는 질문

**Q: 예약 목록 보기와 캘린더 뷰의 차이점은 무엇인가요?**

캘린더 뷰는 날짜와 가용성을 기반으로 한 스케줄링을 위해 설계되었고, 예약 목록 보기는 목록, 필터, 액션을 사용하여 대량의 예약을 관리하도록 설계되었습니다.

**Q: 예약 목록 보기의 스마트 리스트란 무엇인가요?**

스마트 리스트는 필터, 정렬, 컬럼 구성을 포함하는 저장된 예약 뷰로, 사용자가 동일한 설정을 반복해서 재사용할 수 있게 해줍니다.

**Q: 고급 필터와 스마트 리스트의 차이점은 무엇인가요?**

고급 필터는 일시적으로 나타날 예약을 세밀하게 조정합니다. 스마트 리스트는 해당 필터를 정렬 및 컬럼과 함께 향후 사용을 위해 저장합니다.

**Q: 모든 예약이 보이지 않는 이유는 무엇인가요?**

선택한 예약 유형, 활성 필터, 현재 선택한 스마트 리스트로 인해 예약이 숨겨져 있을 수 있습니다.

**Q: 스마트 리스트는 다른 사용자와 공유되나요?**

스마트 리스트 가시성은 사용자 권한과 계정 구성에 따라 달라집니다.
