## 로케이션이란?

로케이션은 서비스가 진행되는 장소, 즉 고객과 만나는 지점입니다. 사무실, 매장 또는 실제 주소가 될 수 있어요.

계정을 처음 설정하면 `Settings(설정) → Business Profile(비즈니스 프로필)`에 입력한 사업장 주소를 사용해서 기본 로케이션이 자동으로 생성됩니다.

- 한 곳에서만 운영하는 사업체에게 이상적이에요.
- `Calendar Settings(캘린더 설정) → Services(서비스) → Locations(로케이션)` 탭에서 Edit(편집)을 클릭해서 언제든지 세부 정보를 업데이트할 수 있습니다.

![기본 로케이션 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054380527/original/aNY37RfgPKImI9iTUSwxkHuRiGHyVdJAWg.png?1758628384)

## 다중 로케이션 관리

사업체가 여러 물리적 위치에서 운영된다면 다중 로케이션 옵션을 활성화할 수 있습니다. 이 기능은 여러 지점이나 사무실을 운영하는 사업체를 위해 설계되었어요.

활성화 방법:

- `Calendar Settings(캘린더 설정) → Services(서비스)`로 이동하세요.
- Global Settings(전역 설정) 아래에서 Enable Multiple Locations(다중 로케이션 활성화)를 토글하세요.
- 활성화되면 Locations(로케이션) 탭에서 더 많은 로케이션을 추가할 수 있습니다.

![다중 로케이션 활성화](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054380709/original/D81sA1XuUhH-V0APmqA3W3e-23N1Rz935Q.png?1758628479)

## 새 로케이션 추가 방법

다중 로케이션을 활성화한 후 다음 단계에 따라 새 로케이션을 추가할 수 있습니다:

- Locations(로케이션) 탭으로 이동하세요.
- Add New(새로 추가)를 클릭하세요.
- 로케이션 이미지를 업로드하세요 (예약 페이지에 표시됩니다).
- 로케이션 이름, 전화번호, 주소를 입력하세요.
- Save Location(로케이션 저장)을 클릭하세요.

참고: 온라인 로케이션(예: Zoom, Google Meet)은 아직 직접 지원되지 않습니다. 임시방편으로 고정 화상회의 링크를 생성하여 주소 필드에 입력할 수 있어요.

## 로케이션별 직원 가용성 설정

로케이션을 생성한 후에는 서비스가 올바른 로케이션에 적절하게 배정될 수 있도록 직원 가용성을 구성해야 합니다.

설정 방법:

- `Calendar Settings(캘린더 설정) → Services(서비스) → Staff(직원) → 직원 편집`으로 이동하세요.
- Assign Services(서비스 배정) 탭으로 이동해서 필요한 서비스를 배정하세요.
- 그다음 Weekly Working Hours(주간 근무 시간) 탭으로 이동하세요.
- 특정 로케이션에서 특정 서비스에 대한 직원의 가용성을 기준으로 근무 시간을 추가하세요.

직원은 서로 다른 일정으로 여러 로케이션에 배정될 수 있어, 서비스 제공에 유연성을 제공합니다.

⚠️ 중요: 예약 과정에서 로케이션이 나타나려면 Staff Settings(직원 설정)에서 직원 가용성과 연결되어 있어야 합니다.

## 자주 묻는 질문

1. **한 곳에서만 운영한다면 로케이션을 만들어야 하나요?**
   아니요. 비즈니스 프로필의 주소를 사용해서 기본 로케이션이 자동으로 생성됩니다. 여러 곳에서 운영하는 경우에만 새 로케이션을 만들면 돼요.

2. **기본 로케이션을 편집할 수 있나요?**
   네. `Calendar Settings(캘린더 설정) → Services(서비스) → Locations(로케이션)` 탭에서 Edit(편집)을 클릭해서 세부 정보를 업데이트하세요.

3. **Zoom이나 Google Meet 같은 온라인 로케이션을 사용할 수 있나요?**
   온라인 로케이션은 아직 직접 지원되지 않습니다. 하지만 임시방편으로 고정 화상회의 링크(예: Zoom, Google Meet)를 주소 필드에 붙여넣을 수 있어요.

4. **로케이션에 직원 가용성을 배정하지 않으면 어떻게 되나요?**
   직원 가용성이 로케이션과 연결되지 않으면 예약 과정에서 해당 로케이션이 옵션으로 나타나지 않습니다.

5. **직원이 여러 로케이션에서 일할 수 있나요?**
   네. 직원은 서로 다른 일정으로 여러 로케이션에 배정될 수 있어, 지점 간 유연성을 제공합니다.

6. **고객은 로케이션 정보를 어디서 보나요?**
   고객은 예약 페이지와 예약 확인서에서 선택된 로케이션 세부 정보(이미지, 이름, 주소 포함)를 볼 수 있습니다.

7. **로케이션을 삭제할 수 있나요?**
   네, 계정에서 생성한 추가 로케이션은 삭제할 수 있습니다. 하지만 기본 로케이션은 삭제할 수 없고, 필요한 경우 세부 정보만 편집할 수 있어요.
