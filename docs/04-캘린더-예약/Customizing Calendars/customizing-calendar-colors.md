# 왜 캘린더를 커스터마이징해야 하나요?

**브랜드 일관성을 유지**하고 싶다면, 모든 마케팅 자산이 동일한 색상 체계를 사용하는 것이 매우 중요합니다!

예약 캘린더와 같은 자산을 커스터마이징하면 **고객에게 돋보일 수 있고**, 고객과 잠재 고객에게 **깔끔하고 전문적인 인상**을 유지하는 데 도움이 됩니다.

아래 이미지에서 보시는 것처럼, 회사 로고는 분명히 빨간색이지만 캘린더의 모든 요소는 파란색입니다. 브랜드 일관성을 유지하려면 **캘린더의 색상도 로고에 맞춰 빨간색이어야 합니다.**

![캘린더 색상 일관성 예시](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155030366782/original/zfs8Tv7KlDog4CVixn6B3pKcRfID4rMr9g.jpg?1722612913)

# 캘린더 색상 커스터마이징 방법

다음은 캘린더 색상을 커스터마이징하는 방법에 대한 인터랙티브 가이드입니다. 추가 도움이 필요하시면 인터랙티브 가이드 아래의 도움말 문서를 확인해주세요.

중요: NEO 캘린더 스타일을 사용하는 경우에만 캘린더 색상을 커스터마이징할 수 있습니다.

![NEO 캘린더 스타일](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155030367091/original/FO4uAkBUun_dQk5VOTVHWUorGLwy9O8tHg.png?1722613320)

### 1단계: 캘린더 설정(Calendar Settings)으로 이동

캘린더 색상을 커스터마이징하려면 캘린더 설정 섹션으로 이동해야 합니다. 메인 네비게이션에서 캘린더(CALENDARS) 탭을 클릭한 후, 화면 상단의 보조 네비게이션에서 캘린더 설정(CALENDAR SETTINGS) 버튼을 클릭하세요.

![캘린더 설정 이동](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155030254627/original/3PH8h4gMtf1YqWNdk6Hvs6BFBQEhotNoHQ.gif?1722449472)

### 2단계: 캘린더 편집

캘린더 설정 섹션에 진입했으면, 색상을 편집하기 위해 새 캘린더를 만들거나 기존 캘린더를 열어야 합니다.

캘린더 표의 "작업 드롭다운(ACTION DROPDOWN)" 열 맨 오른쪽에 있는 점 3개 아이콘을 클릭하세요. 아이콘을 클릭하면 작은 팝업이 나타나고, 팝업에서 "편집(EDIT)" 버튼을 볼 수 있습니다.

해당 캘린더 편집을 시작하려면 "편집(EDIT)" 버튼을 클릭하세요.

![캘린더 편집 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155030256790/original/VlHMl5JGZSD6qclnqKkzbpOMUZYqqE3hxQ.gif?1722453959)

![캘린더 편집 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155030254272/original/-cAdV9q31OkJgkeCBUd26VIINr-_wvxZCA.jpg?1722448977)

### 3단계: "커스터마이제이션(Customization)" 탭으로 이동

캘린더 편집 섹션에 진입하면, 화면 왼쪽에 작은 네비게이션 바가 보입니다. 이 네비게이션의 맨 아래 옵션이 "커스터마이제이션(CUSTOMIZATIONS)"입니다.

캘린더 색상을 조정하려면 "커스터마이제이션(CUSTOMIZATIONS)" 탭을 클릭하세요.

![커스터마이제이션 탭](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155030254082/original/bTve_bpyk7hnSoOIhCmZhh7FrOWqk1Kwpw.jpg?1722448622)

### 4단계: 색상 커스터마이징

"커스터마이제이션(CUSTOMIZATIONS)" 탭에 들어가면, 색상 선택기가 보일 때까지 아래로 스크롤하세요. 캘린더에서 색상을 변경할 수 있는 2가지 옵션이 있습니다: 기본 색상(Primary Color)과 배경 색상(Background Color). 이 2가지 옵션으로 HEXA 코드를 사용하거나 색상 선택기에서 슬라이더를 움직여 캘린더 색상을 조정할 수 있습니다.

![색상 커스터마이징 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155030254115/original/64mGCoCHWcwVzjkkpGvjKCjlzSRS5Eknpg.jpg?1722448670)

**HEXA 코드** - #으로 시작하는 이 코드들은 많은 기업에서 여러 비즈니스 자산에 걸쳐 정확한 브랜드 색상을 유지하기 위해 사용합니다.

**색상 슬라이더** - 색상 선택기의 작은 원을 클릭하면, 슬라이더를 움직여 브랜드나 스타일에 맞는 정확한 색상을 선택할 수 있습니다.

![HEXA 코드와 색상 슬라이더](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155030367415/original/YEjSjZRqDoKXqgYMZAyyhURnNwk1WYs9gQ.jpg?1722613740)
