# 리치 텍스트 에디터로 캘린더 설명 꾸미기

캘린더 설명에 리치 텍스트 에디터가 추가되었습니다! 이제 더욱 매력적이고 읽기 쉬우며 맞춤형으로 캘린더 설명을 작성할 수 있어요.

**목차**

- [새로운 기능](#새로운-기능)
- [사용 방법](#사용-방법)

## **새로운 기능**

리치 텍스트 에디터를 통해 다양한 서식 옵션으로 캘린더 설명을 꾸밀 수 있습니다. 사용할 수 있는 기능은 다음과 같아요:

- **텍스트 서식**: 굵게, 기울임꼴, 밑줄, 취소선

- **목록**: 글머리 기호 및 번호 목록

- **하이퍼링크**: 예약 위젯에서 다른 페이지로 이동하는 클릭 가능한 링크 추가

- **텍스트 및 배경 색상**: 브랜드 테마에 맞게 텍스트와 배경 색상 맞춤 설정

- **단락과 제목**: 단락과 다양한 제목으로 콘텐츠 구성

- **텍스트 정렬**: 원하는 대로 텍스트를 왼쪽, 가운데, 오른쪽 정렬

이러한 기능으로 중요한 정보를 강조하고, 가독성을 높이며, 브랜드에 맞는 시각적으로 매력적인 캘린더 설명을 만들 수 있어요.

## **사용 방법**

- 캘린더 설정으로 이동해서 편집할 캘린더를 선택하세요.
- 'Meeting Details(미팅 세부사항)' 섹션으로 이동합니다.
- 캘린더 설명 아래에서 리치 텍스트 에디터 옵션을 찾을 수 있어요.
- 사용 가능한 서식 옵션을 활용해 원하는 설명을 작성하세요.
- "Save(저장)"을 클릭해서 변경사항을 적용합니다.
