# 라운드 로빈 캘린더의 예약 배정 로직

이 가이드는 Hyperclass의 라운드 로빈 캘린더(Round Robin Calendars) 예약 배정 로직에 대한 포괄적인 개요를 제공합니다.

예약이 팀원 간에 자동으로 어떻게 배정되는지 설명하여 공정한 스케줄링과 최적화된 예약 가능 시간을 보장합니다.

## 개요

**라운드 로빈 캘린더**는 팀원들 간에 예약을 자동으로 배정하는 기능입니다. 이 기능은 예약 가능 시간을 최적화하면서 공정한 스케줄링을 보장하므로, 영업팀, 고객 지원, 의료 서비스 등 팀 기반 예약에 의존하는 비즈니스에 이상적입니다.

**라운드 로빈 캘린더** 유형을 사용하면 고객에게 더 많은 예약 가능 시간을 제공하는 동시에 팀의 업무 부하를 효율적으로 처리할 수 있습니다. 새로운 미팅을 팀원에게 자동으로 배정하여 수동 배정의 필요성을 없애고 시간을 절약해줍니다.

자세한 내용은 [라운드 로빈 캘린더: 설정 가이드](../기타/how-to-create-round-robin-calendars.md)를 참조하세요.

## 예약 배정 방식

라운드 로빈 캘린더는 **예약 가능 시간 최적화**와 **균등 배분 최적화** 두 가지 주요 배정 방식을 사용합니다. 올바른 방식을 선택하는 것은 비즈니스 목표에 따라 달라집니다.

### 1. 예약 가능 시간 최적화

이 방식은 팀원의 예약 가능 시간을 기준으로 예약을 우선 배정합니다. 선택된 예약 시간에 팀원이 예약 가능하면 해당 팀원에게 예약이 배정됩니다.

![예약 가능 시간 최적화 방식](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042622108/original/-up1q4x6Nm4PtLUshPrU_oAmBaUY93q6cw.gif?1741094283)

- **적합한 비즈니스**: 즉시 예약 가능 시간을 우선시하는 비즈니스 (예: 고객 지원, 응급 의료 서비스)

- **주요 특징**:
  - 예약 가능한 팀원에게 우선적으로 예약을 배정
  - 예약 가능 시간의 효율적 활용 보장
  - 고객 대기 시간 단축

**중요 참고사항**: 예약 배정 로직(두 방식 모두)은 24시간마다 한 번씩 계산됩니다. "균등 배분"을 사용하는 경우, 예약이 들어올 때마다 실시간으로 재계산하거나 순환 배정하지 않습니다.

### 2. 균등 배분 최적화

이 방식은 즉각적인 예약 가능 시간과 관계없이 예약이 팀원 간에 균등하게 배분되도록 보장합니다. 공정함을 유지하기 위해 순환식으로 배정합니다.

![균등 배분 최적화 방식](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042622325/original/DMYkCkFxLlpzVb0klXDXWAnwTmcT3txZEg.png?1741094449)

- **적합한 비즈니스**: 영업팀, 컨설팅 서비스, 또는 업무 부하를 균등하게 배분하려는 비즈니스

- **주요 특징**:
  - 특정 팀원이 너무 많은 예약을 받지 않도록 보장
  - 팀 전체의 공정함 유지
  - 모든 팀원이 비슷한 근무 시간을 가질 때 가장 효과적

## 팀원 우선순위 설정

팀원에게 우선순위 레벨을 지정하여 예약 배정에 영향을 줄 수 있습니다. 여러 팀원이 동시에 예약 가능한 경우, 높은 우선순위를 가진 팀원이 먼저 예약을 받게 됩니다.

![팀원 우선순위 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042622842/original/VZ8QsNLxxVoogNnOMCJ9ccxOKcNO228F_A.jpeg?1741094722)

사용 가능한 우선순위 레벨은 다음과 같습니다:

| 우선순위 레벨 | 설명 |
|---------------|------|
| 높은 우선순위 | 예약 가능할 때 가장 먼저 선택됩니다. |
| 중간 우선순위 | 높은 우선순위 팀원이 예약 불가능할 때 선택됩니다. |
| 낮은 우선순위 | 높은 우선순위나 중간 우선순위 팀원이 모두 예약 불가능할 때만 선택됩니다. |

## 미팅 장소 선택

예약을 설정할 때 각 팀원마다 선호하는 미팅 장소를 정의할 수 있습니다. 이를 통해 미팅이 어떻게, 어디서 진행될지에 대한 유연성을 제공합니다.

![미팅 장소 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042623125/original/pLuJn5bVs_3zZoEExFdf0zXfSdoLLgW1ng.png?1741094898)

| 미팅 장소 옵션 | 설명 |
|----------------|------|
| **Zoom** | 줌을 통해 미팅을 진행할 때 선택합니다. |
| **Google Meet** | Google Meet 예약을 스케줄링할 때 사용합니다. |
| **전화** | 전화 통화로 미팅을 진행할 때 선택합니다. |
| **전체 주소** | 물리적인 장소가 필요한 대면 미팅에 유용합니다. |
| **예약자에게 문의** | 예약하는 사람이 미팅 장소를 지정할 수 있도록 허용합니다. |
| **Microsoft Teams** | Microsoft Teams를 통해 미팅을 진행할 때 선택합니다. |

## 자주 묻는 질문

**Q: 설정 후 배정 방식을 변경할 수 있나요?**
네, 캘린더 설정 패널에서 언제든지 설정을 업데이트할 수 있습니다.

**Q: 팀원이 예약 불가능한 상태라면 어떻게 되나요?**

- **예약 가능 시간 최적화**의 경우: 시스템이 다음으로 예약 가능한 팀원에게 예약을 배정합니다.
- **균등 배분 최적화**의 경우: 시스템이 예약 불가능한 팀원을 건너뛰고 순환 배정을 계속합니다.

**Q: 라운드 로빈 캘린더에서 특정 팀원을 제외할 수 있나요?**
네, 캘린더 설정에서 순환 배정에서 팀원을 제거할 수 있습니다.
