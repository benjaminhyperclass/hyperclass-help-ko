## 하이퍼클래스 캘린더의 줌 연동이란?

하이퍼클래스 캘린더와 줌 연동 기능을 사용하면 예약된 약속에 대해 줌 미팅 링크를 자동으로 생성할 수 있습니다. 이를 통해 캘린더 이벤트와 화상 미팅 간의 매끄러운 연결을 보장하며, 수동으로 미팅 링크를 생성할 필요가 없어 인적 오류를 줄일 수 있습니다.

참고: 동적 줌 링크는 개인 예약(Personal), 라운드 로빈(Round Robin), 서비스 캘린더(Service), 공동 예약(Collective) 캘린더에서만 사용 가능합니다.

## 줌 연동의 주요 장점

하이퍼클래스 캘린더와 줌을 연동하면 생산성과 사용자 경험 측면에서 상당한 이점을 얻을 수 있습니다. 미팅 링크 생성을 자동화하여 관리 업무를 줄이고 실시간 화상 상호작용을 통해 고객 참여를 향상시킵니다.

- **자동 줌 링크 생성**: 모든 캘린더 예약에 자동으로 줌 미팅 링크가 포함되어 수동 생성이 필요 없습니다.
- **수동 오류 감소**: 미팅 세부사항이나 링크 포함을 잊는 실수를 최소화합니다.
- **시간 절약**: 이전에 수동으로 하던 작업을 자동화하여 사용자가 고부가가치 업무에 집중할 시간을 확보합니다.
- **일정 관리 간소화**: 예약 관리를 더욱 효율적이고 안정적으로 만듭니다.
- **고객을 위한 전문적인 경험**: 모든 고객이 일관되고 완전히 준비된 미팅 초대를 받을 수 있도록 보장합니다.

- **다중 로케이션 및 다중 사용자 공유**: 동일한 줌 계정을 여러 로케이션에서 연결하고, 여러 사용자가 예약을 위해 같은 줌 계정을 사용할 수 있습니다.

최신 연결(Connections) 플로우를 사용하여 줌을 연결하려면:
캘린더(Calendars) → 설정(Settings) → 연결(Connections) → 화상 회의(Video Conferencing) → 새로 추가(Add New) → 줌(Zoom)

단계별 설정 및 관리자 시나리오는 "[사용자 캘린더 예약을 위한 줌 연동](zoom-integration-for-users-calendar-bookings.md)"을 참조하세요.

## 하이퍼클래스 캘린더와 줌 연동 설정 방법

하이퍼클래스는 줌 계정을 연결하는 두 가지 내장 방법을 제공하며, 두 방법 모두 캘린더 예약 시 자동 미팅 링크 생성을 지원합니다. 개인 프로필을 통해 줌을 연결하거나 캘린더 설정에서 직접 연결하는 것 중 어떤 방법을 선택하든, 각 방법은 수동 입력 없이 고객이 줌 링크를 받을 수 있도록 보장합니다.

### 방법 1: "내 프로필"에서 줌 연결하기

- 설정(Settings) 내비게이션 바에서 내 프로필(My Profile)로 이동합니다

- 캘린더(Calendar) 섹션으로 이동하여 보조 내비게이션 리본에서 화상 회의(Video Conferencing) 탭을 클릭합니다.

- + 새로 추가(Add New) 라고 표시된 파란색 버튼을 클릭합니다.

- 줌(Zoom) 옆에 있는 연결(Connect) 라고 표시된 파란색 버튼을 클릭합니다. 줌 로그인 창이 나타납니다. 줌 계정에 로그인하고 액세스를 승인합니다.

![줌 연동 설정 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155045060111/original/zud4q5MiZ0k7LHLU6A5Mkq11EpuUdzMltA.png?1744640417)

### 방법 2: 캘린더 설정에서 줌 연결하기

- 설정(Settings) 내비게이션 바에서 캘린더 설정(Calendar Settings)으로 이동합니다

- 보조 내비게이션 리본에서 연결(Connections) 탭을 클릭합니다.

- 연결(Connections) 하위 내비게이션 바에서 화상 회의(Video Conferencing) 탭을 선택합니다

- 화상 회의(Video Conferencing) 섹션에서 새로 추가(Add New) 라고 표시된 파란색 버튼을 클릭합니다.

- 그런 다음 줌(Zoom) 옆에 있는 연결(Connect)을 클릭합니다. 줌 계정에 로그인하고 액세스를 승인합니다

![캘린더 설정에서 줌 연동하기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044906616/original/eeNzLyA9tdU0lhQ98K5JxtFohCRxY9_8_w.png?1744312261)

## 연결 상태 확인하기

줌 계정이 연결되었는지 확인하려면 내 프로필(My Profile) > 캘린더 설정(Calendar Settings) > 화상 회의(Video Conferencing)로 이동하세요. 계정이 연결되어 있다면 여기에 표시됩니다.

줌이 연결 해제된 후 다시 연결되면, 하이퍼클래스는 연결 중단 기간 동안 생성되어 링크가 누락된 향후 예약에 대해 자동으로 줌 링크를 생성하므로 각 예약을 수동으로 업데이트할 필요가 없습니다.

## 자주 묻는 질문(FAQ)

**Q: 하이퍼클래스와 연동하려면 유료 줌 계정이 필요한가요?**
아니요, 무료 줌 계정으로도 연동이 가능합니다. 다만 유료 플랜은 미팅 시간 연장과 추가 기능을 제공할 수 있습니다.

**Q: 서로 다른 캘린더에 각각 다른 줌 계정을 배정할 수 있나요?**
줌 연동은 사용자 프로필과 연결되므로 사용자당 하나의 줌 계정만 연결할 수 있습니다.

**Q: 줌 계정을 연결 해제하거나 변경하려면 어떻게 하나요?**
캘린더 설정(Calendar Settings) > 연결(Connections) > 화상 회의(Video Conferencing) 또는 내 프로필(My Profile) > 캘린더 설정(Calendar Settings) > 화상 회의(Video Conferencing)로 이동하세요. 줌(Zoom) 옆의 연결 해제(Disconnect)를 클릭한 후 다른 계정으로 다시 연결하면 됩니다.

**Q: 줌 링크가 예약 이메일과 SMS에 자동으로 포함되나요?**
네, 미팅 위치가 줌으로 선택되고 알림 템플릿에서 {{appointment.location}} 병합 필드를 사용하면 링크가 포함됩니다.

**Q: 확인 이메일에서 줌 미팅 링크가 누락되는 이유는 무엇인가요?**
미팅 위치가 줌으로 설정되어 있고 메시지 템플릿에 {{appointment.location}} 병합 필드가 있는지 확인하세요.

**Q: 줌과 구글 미트를 동시에 사용할 수 있나요?**
네, 둘 다 연동에서 연결되어 있다면 줌과 구글 미트를 동시에 사용할 수 있습니다. 다만 사용자 프로필이나 캘린더당 한 번에 하나의 화상 회의 플랫폼만 활성화할 수 있습니다.

**Q: 줌 연동은 모든 하이퍼클래스 요금제에서 사용할 수 있나요?**
네, 줌 연동은 모든 요금제 등급에서 사용 가능합니다.

**Q: 어떤 캘린더가 동적 줌 링크를 지원하나요?**
개인 예약(Personal Booking), 라운드 로빈(Round Robin), 서비스 캘린더(Service), 공동 예약(Collective) 캘린더가 줌 연동을 지원하며 예약된 각 약속마다 새로운 줌 링크를 생성합니다.
