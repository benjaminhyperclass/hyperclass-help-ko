# **Calendly 연동이란?**

Calendly 연동을 통해 하이퍼클래스는 선택한 Calendly 캘린더의 예약을 자동으로 가져올 수 있습니다. 이를 통해 실시간 예약을 기반으로 자동화를 실행하고, 연락처를 생성하며, 워크플로우를 예약할 수 있습니다. 연동 설정은 Calendly 스케줄링 활동을 CRM과 동기화하는 첫 번째 단계입니다.

## **Calendly 연동의 주요 장점**

Calendly 계정을 연결하면 강력한 자동화 및 연락처 관리 기능을 활용할 수 있습니다. 동기화가 완료되면 예약이 하이퍼클래스로 원활하게 전송되어 다양한 용도로 활용됩니다.

- 새 예약을 자동으로 캘린더에 가져오기
- Calendly 예약에서 연락처 자동 생성
- 캘린더 소유권에 따라 예약을 사용자에게 자동 배정
- 공유 또는 개인 Calendly 캘린더 동기화
- 새 예약을 기반으로 자동화 트리거 활성화

## **Calendly 계정 연결하기**

연동 설정은 빠르고 간단하며 몇 번의 클릭만 필요합니다. 다음 단계를 따라 Calendly 계정을 안전하게 연결하고 동기화할 캘린더를 선택하세요.

### **캘린더 설정 접속**

좌측 메뉴에서 **캘린더(Calendars)** 섹션으로 이동한 후, 우상단의 **캘린더 설정(Calendar Settings)**을 클릭하세요. Calendly를 포함한 모든 캘린더 연결을 관리하는 곳입니다.

캘린더(Calendars) > 캘린더 설정(Calendar Settings)으로 이동

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155045071601/original/vZZKvoSawr1RWtOWQj1kP-CDjnbRO_CvGg.png?1744648219)

### **연결 탭 열기**

상단의 **연결(Connections)** 탭을 클릭하여 연동된 모든 캘린더 계정을 확인하고 관리하세요. Calendly와 같은 외부 캘린더를 추가, 편집, 제거하는 곳입니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155045072687/original/5MKziRpBSJ71mdpIOzryZhQaoTnx_i2OvA.png?1744649594)

### **새 연동 추가**

**+ 새로 추가(Add New)** 버튼을 클릭하여 새 캘린더 연결을 시작하세요. Calendly 계정을 하이퍼클래스와 연결하는 과정을 시작하는 곳입니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155045085523/original/azEr6YdXCbtVcFZvrW5okbNILcMSCyEI7Q.png?1744684783)

### **Calendly 옵션 선택**

캘린더 제공업체 목록에서 **Calendly**를 찾아 **연결(Connect)**을 클릭하세요. Calendly 예약을 하이퍼클래스와 동기화하는 인증 과정이 시작됩니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155045085588/original/XZ1xeFIB8pzm2LXCeDcv8rIKJgUUmrDWdQ.png?1744685049)

### **Calendly 로그인**

Calendly 계정에 로그인하고 접근 권한을 부여하여 인증 과정을 완료하세요. Calendly 계정과 연결된 이메일 주소를 입력하고 **계속(Continue)**을 클릭하세요. Google 또는 Microsoft 계정과 연결되어 있다면 해당 자격 증명으로도 로그인할 수 있습니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155045085817/original/NW3v4HAVhVc1P-sk5Di-OtQcUJZiZJlO8w.png?1744685803)

### **Calendly 캘린더 선택**

하이퍼클래스와 동기화할 Calendly 캘린더를 선택하세요. 선택된 캘린더의 예약만 시스템으로 가져와집니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155045085848/original/U2vaqQjMGOFL3avKm1IMMeB70oUHy4e_DQ.png?1744685820)

### **동기화 접근 승인**

하이퍼클래스가 Calendly 예약과 관련 세부 정보를 가져올 수 있도록 동의 체크박스를 선택하세요. 그런 다음 **저장(Save)**을 클릭하여 동기화 설정을 완료하세요.

Calendly 계정을 연결할 때 향후 예약과 지난 180일 이내에 생성된 과거 예약을 모두 자동으로 가져옵니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155045085950/original/Utshyqs6u4d3HlZC1zbxHYnmebShkoSiaQ.png?1744686152)

## **전화번호 동기화 (위치 vs 폼 필드)**

하이퍼클래스는 Calendly에서 게스트의 전화번호를 여러 방법으로 가져올 수 있습니다. 여러 소스가 가능한 경우, 연락처 프로필에 동기화할 항목을 선택할 수 있습니다.

### **미팅 위치 전화번호에서 자동 동기화**

게스트의 전화번호가 Calendly 미팅 위치에 입력되고 Calendly 폼에 전화번호 필드가 없는 경우, 하이퍼클래스는 자동으로 해당 번호를 연락처에 동기화합니다.

### **Calendly 폼 필드에서 전화번호를 가져올 때 매핑 모달**

Calendly 폼(Q&A)에 전화번호 필드가 있는 경우, 하이퍼클래스는 매핑 화면을 표시하여 동기화할 항목을 선택할 수 있습니다:

- 폼 필드에서 전화번호 값 선택
- 미팅 위치 전화번호 필드 선택(있는 경우)
- 동기화하지 않음 선택(전화번호 매핑 방지)

참고: 선택된 Calendly 캘린더/예약 유형에 전화번호 필드가 없는 경우(위치 전화번호도, 폼 전화번호도 없음), 전화번호 동기화 단계는 프롬프트나 오류 없이 건너뛰어집니다.

공유 캘린더: 사용자가 전화번호 동기화 옵션을 업데이트하면, 해당 캘린더에 연결된 모든 사용자에게 변경 사항이 적용됩니다.

## **동기화된 캘린더 편집 및 업데이트**

### **캘린더 선택 편집**

연결이 완료되면 Calendly 연동이 연결된 캘린더 아래에 나타납니다. **연필 아이콘**을 클릭하여 하이퍼클래스와 동기화되는 Calendly 캘린더를 편집하세요.

설정이 변경되거나 새 팀 캘린더가 추가되는 경우 캘린더 선택을 쉽게 업데이트할 수 있습니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155045088297/original/ujUVrZ68WR2OPJ_jzSfp6OKcPYgNgMi9Wg.png?1744693740)

### **동기화된 캘린더 수정**

동기화하려는 예약을 체크하거나 체크 해제하여 캘린더 선택을 업데이트하세요. 여기서 선택된 캘린더만 하이퍼클래스와 계속 동기화됩니다.

새로 선택된 캘린더의 예약은 변경 사항을 저장한 후부터 동기화가 시작됩니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155045088388/original/c4hvqQvJ4P_uMmHSAthFqkuggh0O8b6MqQ.png?1744693886)

## **자주 묻는 질문**

**Q: 한 사용자가 여러 Calendly 계정을 연결할 수 있나요?**

아니요. 각 사용자는 하위 계정당 하나의 Calendly 계정만 연결할 수 있습니다.

**Q: 동일한 Calendly 계정을 여러 사용자가 사용할 수 있나요?**

예. 하나의 Calendly 계정을 여러 하위 계정의 다양한 사용자가 연결할 수 있습니다.

**Q: 이 연동은 어떤 플랜을 지원하나요?**

Standard, Teams, Enterprise Calendly 플랜만 지원됩니다. Free와 Essentials 플랜은 호환되지 않습니다.

**Q: 체험 기간 중에는 어떻게 되나요?**

체험 기간 중에도 Calendly를 연결할 수 있습니다. 체험 기간이 끝난 후 지원되는 플랜으로 업그레이드하지 않으면 연동이 작동하지 않습니다.

**Q: 하이퍼클래스에서 Calendly 예약을 편집할 수 있나요?**

아니요. 이것은 단방향 동기화입니다. 모든 예약 변경은 Calendly 내에서만 가능합니다.
