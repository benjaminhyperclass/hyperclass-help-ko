# **예약 모달의 폼 탭이란?**

폼(Forms) 탭은 예약 모달 안에 새롭게 추가된 기능으로, 캘린더 예약과 연결된 커스텀 폼 제출 내용을 표시합니다. 예약 과정에서 연락처가 작성한 폼이 포함된 예약의 경우, 해당 데이터가 깔끔하고 구조화된 레이아웃으로 모달에서 바로 표시됩니다. 이를 통해 폼(Forms) 모듈이나 연락처 레코드로 이동할 필요 없이 더 빠르게 검토하고 맥락을 파악할 수 있습니다.

---

## **예약에서 폼 제출 내용을 확인하는 주요 이점**

커스텀 폼 제출 내용에는 미팅, 상담, 영업 통화 준비에 도움이 되는 중요한 맥락 정보가 담겨 있습니다. 이 데이터를 예약 안에서 바로 확인할 수 있게 되면서, 플랫폼의 여러 섹션 간 이동 필요성이 줄어들고 팀의 효율성이 높아집니다.

• 예약 모달 안에서 폼 응답을 즉시 확인
• 다른 모듈에서 폼 항목을 찾을 필요가 없음
• 예약 전 준비 시간 단축
• 서비스 기반 업종과 접수가 많은 업종의 운영 흐름 개선
• 쉬운 협업을 위한 고객 데이터 중앙화

---

## **예약 모달에서 폼 제출 내용에 접근하는 방법**

이제 플랫폼에서 예약 모달을 여는 모든 영역에서 폼 제출 내용을 직접 확인할 수 있습니다. 캘린더(Calendars) 탭, 연락처(Contacts) 탭, 또는 대화(Conversations) 보기에서 작업하든 상관없이, 예약과 연결된 폼 데이터에 일관되게 접근할 수 있습니다.

폼 제출 내용을 확인하려면:

1. **캘린더(Calendars)**, **연락처(Contacts)**, 또는 **대화(Conversations)** 영역으로 이동합니다.

2. 예약을 클릭하여 **예약 모달(Appointment Modal)**을 엽니다.

3. 예약에 커스텀 폼이 포함되어 있으면 모달에 새로운 **폼(Forms)** 탭이 나타납니다.

4. **폼(Forms)** 탭을 선택하면 폼(Forms) 모듈과 동일한 구조화된 레이아웃으로 전체 폼 제출 내용을 확인할 수 있습니다.

---

## **지원되는 폼 유형**

이 기능은 현재 예약 과정의 일부로 캘린더에 추가된 **커스텀 폼**을 지원합니다. 이는 Hyperclass 폼 빌더(Forms Builder)를 통해 생성하고 관리하는 폼과 동일합니다.

**지원되는 폼**

• **폼 빌더(Forms Builder)**에서 생성된 폼
• 캘린더의 **폼 설정(Form settings)**을 사용하여 캘린더에 삽입된 폼

**아직 지원되지 않는 폼**

• 이름, 이메일, 전화번호만 수집하는 **기본 예약 폼**

기본 폼이 현재 제외된 이유는 해당 폼에서 수집하는 정보가 이미 예약 모달의 **연락처 세부정보(Contact Details)** 섹션에서 볼 수 있기 때문입니다. 하지만 폼 탭에서 이런 정보를 표시하는 지원은 향후 릴리스에서 계획되어 있습니다.

---

## **일반적인 사용 사례와 효율성 향상**

폼 제출 내용에 즉시 접근할 수 있는 기능은 예약 과정에서 상세한 고객 정보 수집에 의존하는 비즈니스에게 특히 유용합니다. 캘린더와 폼 간의 이러한 통합은 운영 효율성을 개선하고 더욱 원활한 사용자 경험을 만들어 줍니다.

일반적인 사용 사례 예시:

• **에이전시**: 전략 미팅 전에 클라이언트 발견 폼에 접근
• **건강 및 웰니스 제공업체**: 환자 접수 세부정보 확인
• **영업 전문가**: 커스텀 접수 필드를 기반으로 리드 자격 평가
• **컨설턴트와 코치**: 클라이언트가 제출한 응답을 활용한 세션 맞춤화

이를 통해 불필요한 클릭이 줄어들고 고객과 더욱 정보에 기반한 생산적인 대화를 할 수 있습니다.

## **자주 묻는 질문**

**Q: 예약 모달에서 폼 탭이 보이지 않는 이유는?**

폼 탭은 예약에 **커스텀 폼 제출** 내용이 포함된 경우에만 나타납니다. 기본 폼으로 생성된 예약에서는 탭이 표시되지 않습니다.

**Q: 예약 모달에서 폼 제출 내용을 수정할 수 있나요?**

아니오, 폼 제출 내용은 **읽기 전용**으로 표시됩니다. 변경하려면 **폼(Forms) 모듈**로 이동하거나 연락처 레코드에서 폼을 편집해야 합니다.

**Q: 폼 데이터는 어디에서 나오나요?**

캘린더 예약에 첨부된 폼이나 예약 워크플로우 중에 추가된 폼에서 나옵니다.

**Q: 기본 폼 제출 지원이 추가될 예정인가요?**

네, 폼 탭에서 기본 예약 폼 제출 내용을 확인할 수 있는 기능은 현재 개발 중이며 향후 릴리스에서 추가될 예정입니다.

**Q: 대화 탭이나 연락처 탭에서도 폼 탭에 접근할 수 있나요?**

네, 캘린더, 연락처, 대화를 포함하여 예약을 여는 위치에 관계없이 예약 모달에서 폼 탭이 나타납니다.
