# 수업 예약 캘린더 개요

수업 캘린더(Class Calendars)는 여러 명의 참가자가 하나의 예약에 참석할 수 있는 일대다 이벤트를 관리하도록 설계되었습니다. 이러한 캘린더는 웨비나, 워크숍, 교육 세션, 스포츠 이벤트에 적합합니다. 수업 캘린더에서는 일반적으로 강사나 주최자인 한 명의 예약 소유자만 있을 수 있습니다. 슬롯당 인원수를 설정할 수 있으며, 최대 9,999명의 참가자까지 가능하지만 미팅 장소의 제약 사항이 적용될 수 있습니다.

## 수업 예약이란 무엇인가요?

수업 예약(Class Booking)은 하나의 예약에 여러 명의 참가자를 수용하는 일대다 방식의 캘린더입니다. 수업 예약은 여러 참가자가 한 번에 동일한 이벤트에 참석하기를 원할 때 유용하며, 몇 가지 예시는 다음과 같습니다:

- 웨비나/워크숍
- 교육 세션
- 스포츠 이벤트

## **수업 예약 캘린더를 사용하는 이유는?**

수업 캘린더를 사용하면 여러 참가자가 있는 이벤트를 관리할 때 다음과 같은 여러 이점이 있습니다:

- **효율적인 조직화:** 수업 캘린더는 여러 참가자를 수용하는 단일 예약을 생성할 수 있게 하여 일대다 이벤트의 조직화와 관리를 간소화하고 일정 관리 프로세스를 효율화합니다.

- **맞춤 가능한 수용 인원:** 슬롯당 인원수를 설정하여 수업 규모를 제어하고 최적의 학습 또는 참여 환경을 보장할 수 있습니다.

- **중앙집중식 관리:** 수업 캘린더는 예약, 취소, 일정 변경을 관리할 수 있는 중앙집중식 플랫폼을 제공하여 예약 소유자가 이벤트의 모든 측면을 더 쉽게 관리할 수 있게 합니다.

- **향상된 의사소통:** 수업 캘린더는 날짜, 시간, 장소와 같은 이벤트 세부 사항의 명확한 개요를 제공하여 참가자와의 의사소통을 개선합니다. 이는 혼란을 줄이고 노쇼 가능성을 낮춥니다.

- **시간 절약:** 예약 프로세스를 자동화하고 참가자가 예약을 취소하거나 일정을 변경할 수 있는 옵션을 제공함으로써 수업 캘린더는 주최자와 참가자 모두의 시간을 절약합니다.

참고: 수업 예약 캘린더 링크를 공유할 때, 항상 이벤트 월이 아닌 현재 월로 열립니다. 수업이 다른 월에 예정되어 있다면, 캘린더에서 기본적으로 "사용 가능한 시간대 없음"이 표시될 수 있습니다. 사용자는 화살표를 클릭하여 이벤트 월로 이동해야 합니다.

## 자주 묻는 질문

**Q: 수업에 등록한 모든 참가자에게 즉석 이메일을 어떻게 보낼 수 있나요?**

수업 세션에 등록한 모든 참가자에게 일회성 이메일을 보내려면 **연락처(Contacts)**로 이동하여 특정 수업 예약별로 연락처를 필터링하세요(고급 필터 또는 예약 기반 필터 사용). 등록된 참가자 목록이 표시되면 연락처를 선택하고 일괄 작업을 사용하여 이메일을 발송하세요. 해당 그룹과 두 번 이상 소통할 계획이라면 그 수업 예약으로 필터링된 스마트 리스트를 생성할 수도 있습니다.

**Q: 특정 참가자에 대해서만 수업 예약을 취소하거나 일정을 변경할 수 있나요?**

네, 수업 예약의 경우 연락처 옆의 옵션 메뉴를 클릭하면 특정 연락처의 예약을 취소하거나 일정을 변경할 수 있는 옵션을 볼 수 있습니다.

**Q: 연락처와 직접 수업 예약을 할 수 있나요?**

네, 가능합니다. "예약하기" 버튼을 클릭하고 예약하고자 하는 연락처를 선택하세요. 그 다음 캘린더를 선택하고(여기서 수업 예약 캘린더 선택), 슬롯을 선택한 후 예약을 진행하세요.

**Q: 참가자가 수업 예약을 취소/일정 변경할 수 있나요?**

참가자가 수업 예약을 취소/일정 변경할 수 있도록 하려면, 캘린더 설정(Calendar Settings)의 확인(Confirmation) 탭에서 "취소 허용(Allow Cancellation)" 및 "일정 변경 허용(Allow Reschedule)" 옵션을 활성화하세요.

**Q: 모든 참가자가 포함된 전체 수업 세션을 어떻게 취소하나요?**

전체 수업 세션(등록된 모든 참가자 포함)을 취소하려면 **캘린더(Calendars)**로 이동하여 **캘린더 뷰(Calendar View)** 또는 **예약 목록 뷰(Appointment List View)**에서 수업 예약을 열고 **삭제(휴지통 아이콘)** 옵션을 클릭하세요. 메시지가 나타나면 삭제를 확인하세요. 이 작업은 해당 시간대의 수업 세션과 관련된 모든 참가자를 제거합니다.
