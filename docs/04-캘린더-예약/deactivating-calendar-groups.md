# 캘린더 그룹 비활성화하기

비활성화하려는 그룹에서 **점 3개(ellipses)** 버튼을 클릭하여 개별 그룹을 비활성화할 수 있어요.

1. 점 3개 버튼을 클릭하면 **옵션** 서랍이 열려요

2. "그룹 내 모든 캘린더 비활성화" 새로운 옵션을 확인할 수 있어요

3. 이 옵션을 클릭하면 확인을 요청하는 팝업이 표시돼요
