# 장비 생성하는 방법

장비 설정을 통해 예약 서비스 중에 사용되는 공유 물리적 자원(마사지 테이블, 진료용 의자, 프로젝터 등)을 관리할 수 있습니다. 장비를 캘린더에 연결하면 중복 예약을 방지하고 운영 효율성을 유지할 수 있습니다.

# 장비란 무엇인가요?

장비는 비즈니스에서 제공하는 서비스에 필요한 공유 도구, 기기 또는 물리적 자원을 의미합니다. 예를 들어 프로젝터, 치료용 테이블, 치료용 의자 등이 있습니다. 시스템에 추가된 이러한 자원은 캘린더와 예약 가능 시간 설정에 연결하여 예약 충돌을 방지할 수 있습니다. 이를 통해 사용 가능한 장비만 예약되도록 하여 원활한 운영과 개선된 고객 경험을 제공합니다.

## 장비의 주요 이점

예약 설정에서 장비를 사용하면 가시성이 향상되고 사용 제한이 있는 자원의 중복을 방지할 수 있습니다.

- 공유 자원의 중복 예약 방지
- 특정 캘린더나 서비스에 장비 배정
- 예약 일정 중 자원 가용성 자동화
- 수동 조정 감소 및 신뢰성 증대
- 직원 효율성 및 서비스 품질 향상
- 여러 지점에서 자원 관리

## 장비 설정 방법

장비를 생성하면 모든 예약된 약속에서 가용성 로직이 적용됩니다.

### 1단계: 장비 기능 활성화

- Settings(설정) → Calendar Settings(캘린더 설정)으로 이동합니다
- 보조 네비게이션 리본에서 Preferences(환경설정) 탭을 선택합니다
- 환경설정 하위에 있는 Account Preference(계정 환경설정) 탭을 선택합니다
- Services(서비스) 섹션에서 Equipments(장비)라고 표시된 파란색 스위치를 토글하여 활성화합니다

![장비 기능 활성화](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044905852/original/D0dlWKZE_KWAwmTBb6LEJObe1ArHlvaCRw.png?1744310540)

### 2단계: 장비 생성

- 보조 네비게이션 리본에 있는 Calendar Settings(캘린더 설정)의 Calendars(캘린더) 탭으로 돌아갑니다
- Calendar Settings(캘린더 설정) 하위에 있는 Equipment(장비) 탭으로 이동합니다
- 우측 상단의 Create Equipment(장비 생성)라고 표시된 파란색 버튼을 선택합니다

![장비 생성 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044905893/original/pTptD2wTKHNfAqseU_EmfxQUfvYH_J2j2A.png?1744310613)

### 3단계: 장비 세부 정보 입력

장비를 설정하려면 다음 세부 정보를 입력하세요:

- Equipment Name(장비명): 장비에 고유한 이름을 지정합니다.
- Description(설명): 장비의 용도를 명확히 하는 간단한 설명을 제공합니다.
- Total Quantity(전체 수량): 사용 가능한 장비 단위의 총 개수를 지정합니다.
- Out of Service Quantity(서비스 불가 수량): 현재 작동하지 않거나 수리가 필요하거나 서비스가 불가능한 장비 단위의 개수를 입력합니다.
- Select Calendar(캘린더 선택): 이 장비와 연결하려는 캘린더를 선택합니다.

모든 세부 정보를 입력한 후, 우측 하단의 Create(생성)라고 표시된 파란색 버튼을 클릭합니다.

![장비 세부 정보 입력](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044761121/original/kCCYA-PqDKeUr8Hs2Hu1zyVCw-u6SmI4cw.png?1744148565)

## 장비 편집 또는 삭제

시간이 지남에 따라 장비를 관리하면 시스템을 최신 상태로 유지할 수 있습니다. 자원의 이름을 바꾸거나 더 이상 사용하지 않는 자원을 제거하는 경우, 목록을 최신 상태로 유지하는 것이 중요합니다.

- Settings(설정) → Calendar Settings(캘린더 설정)으로 이동합니다
- Calendar Settings(캘린더 설정) 하위 리본에서 Equipment(장비) 탭을 선택합니다
- 점 3개 아이콘을 클릭합니다. Edit(편집)을 선택하여 장비를 편집하거나 Delete(삭제)를 선택하여 장비를 삭제합니다
- 변경사항을 적용하고 Save(저장)을 클릭합니다

![장비 편집 또는 삭제](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044905927/original/ED7U8sxGK0FFTyb1EzTIuXtka4X8MIvjOQ.png?1744310702)

## 자주 묻는 질문

**Q: 룸과 장비의 차이점은 무엇인가요?**
룸은 사무실이나 스튜디오 같은 물리적 공간입니다. 장비는 의자나 기계 같은 도구나 자원을 의미합니다. 둘 다 자원으로 예약할 수 있습니다.

**Q: 장비를 여러 캘린더에 배정할 수 있나요?**
네. 장비는 여러 캘린더에 연결할 수 있지만, 한 번에 하나의 예약에서만 사용할 수 있습니다.

**Q: 동일한 장비의 여러 버전을 어떻게 추가하나요?**
생성 시 각 장비 항목의 수량을 설정할 수 있습니다. 예를 들어, 동일한 마사지 테이블이 3개 있는 경우, 하나의 항목("마사지 테이블")만 생성하고 수량을 3으로 설정하면 됩니다.

**Q: 사용 중인 장비를 삭제하면 어떻게 되나요?**
삭제하면 예약에 영향을 미치고 연결된 가용성 제한이 제거될 수 있습니다. 장비를 제거하기 전에 연결된 캘린더를 검토하세요.

**Q: 장비 사용량에 대한 보고서를 생성할 수 있나요?**
장비 사용량에 대한 기본 보고서는 없지만, 특정 자원과 연결된 예약은 캘린더를 통해 수동으로 검토할 수 있습니다.
