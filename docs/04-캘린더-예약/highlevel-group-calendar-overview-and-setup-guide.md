# 하이퍼클래스 그룹 캘린더 개요 및 설정 가이드

이 글에서는 하이퍼클래스의 그룹 캘린더(Group Calendars)를 사용하여 여러 캘린더를 하나의 링크로 통합하고 유연한 예약 옵션을 제공하는 방법을 설명합니다.

주의: 그룹 캘린더는 개별 캘린더를 자동으로 생성하지 않습니다. 그룹에 배정하기 전에 먼저 개별 캘린더를 생성해야 합니다.

## 하이퍼클래스 그룹 캘린더란?

하이퍼클래스의 그룹 캘린더는 여러 개별 캘린더를 하나의 링크 아래 묶어주는 컨테이너입니다. 이를 통해 여러 서비스 제공자 중에서 선택하거나, 예약을 분산시키거나, 특정 팀원과 연결된 서비스 기반 선택권을 제공하는 등 유연한 예약 경험이 가능합니다.

## 그룹 캘린더의 주요 장점

그룹 캘린더는 다음과 같은 방식으로 예약 워크플로우를 간소화합니다:

- 여러 캘린더를 하나의 예약 링크로 통합
- 고객이 팀원이나 서비스 중에서 선택 가능
- 부서별, 가용성별, 서비스별로 캘린더 링크 정리
- 여러 개별 캘린더 링크를 대체하여 복잡성 감소

## 그룹 캘린더 생성하기

### 1단계: 캘린더 설정으로 이동

- 하위 계정 설정(Settings)으로 이동합니다.

![그룹 캘린더 설정 화면](https://jumpshare.com/v/9drFWZMZUKzK9Yz5Q6QP+/Screen+Shot+2025-05-30+at+6.54.15+PM.png)

- 비즈니스 서비스(Business Services) 하위의 캘린더(Calendars)를 선택합니다.

- 그룹 생성(Create Group) 버튼을 클릭합니다.

![그룹 생성 버튼](https://jumpshare.com/v/nOoenAOqpMgwEMImXQtH+/Screen+Shot+2025-05-30+at+7.06.34+PM.png)

### 2단계: 새 캘린더 그룹 추가

- 그룹명(Group Name)과 간단한 설명을 입력합니다(선택사항).
- 그룹 보기용으로 Classic 또는 Neo 템플릿을 선택합니다.
- 적절한 슬러그(Slug)로 그룹 URL을 생성합니다.
- 생성(Create) 버튼을 클릭합니다.

![그룹 정보 입력](https://jumpshare.com/v/eq4sOBnJdw1wU4ihydX7+/Screen+Shot+2025-05-30+at+7.11.38+PM.png)

### 3단계: 그룹에 캘린더 추가

그룹 캘린더 생성 후 두 가지 옵션이 있는 팝업이 나타납니다:

- **캘린더 선택(Choose Calendars)**: 팝업에서 직접 그룹용 캘린더를 선택할 수 있습니다.

- **나중에 하기(I'll Do It Later)**: 건너뛰고 나중에 그룹에 캘린더를 추가할 수 있습니다.

![캘린더 선택 팝업](https://jumpshare.com/v/ALemq7ZutFibgGANpTs9+/Screen+Shot+2025-05-30+at+7.15.17+PM.png)

- 캘린더 선택(Choose Calendars)을 클릭한 후, 그룹에 추가할 **캘린더를 선택**합니다.

- **확인(Confirm)**을 클릭합니다.

![캘린더 선택 및 확인](https://jumpshare.com/v/umTpXTkUYetmOiZg88Ug+/Screen+Shot+2025-05-30+at+7.19.12+PM.png)

참고: 캘린더가 나타나지 않으면 먼저 개별 캘린더를 생성했는지 확인하세요.

## 외부 캘린더와 동기화

각 팀원의 캘린더 설정에서 기본 캘린더(Primary Calendar)가 구성되어 있는지 확인하세요. 이는 Google, Outlook 등 타사 캘린더와 예약을 동기화하는 데 필요합니다.

자세한 정보: [기본 캘린더 지정](assigning-a-primary-calendar.md)

## 자주 묻는 질문

**Q: 그룹 캘린더 설정 화면에서 직접 캘린더를 생성할 수 있나요?**

아니요. 먼저 캘린더(Calendars) 탭에서 개별 캘린더를 생성해야 합니다. 생성 후에 그룹 캘린더에 배정할 수 있습니다. 그룹 캘린더는 컨테이너 역할을 하며, 개별 캘린더를 자체적으로 생성하거나 구성하지 않습니다.

**Q: 수업 예약(Class)이나 공동 예약(Collective) 캘린더도 그룹으로 묶을 수 있나요?**

아니요. 그룹 캘린더는 수업 예약이나 공동 예약 유형과는 별개입니다. 각각 고유한 구조와 동작을 가집니다. 사용 목적에 따라 해당 캘린더 유형들에는 별도의 예약 플로우를 사용할 수 있습니다.

**Q: 그룹 내 캘린더를 삭제하면 어떻게 되나요?**

그룹 캘린더는 그대로 존재하지만 삭제된 캘린더는 더 이상 예약 옵션으로 표시되지 않습니다. 캘린더를 편집하거나 삭제한 후에는 항상 그룹 구성을 확인하세요.

**Q: 그룹 캘린더는 어떻게 공유하거나 임베드하나요?**

하이퍼클래스 캘린더 [임베드 가이드](embedding-highlevel-calendars-using-html-code.md)를 확인하세요.

**Q: 캘린더 그룹을 사용하지 않고도 여러 캘린더를 표시할 수 있나요?**

네, 자세한 정보는 [여기](can-i-show-multiple-calendars-without-using-a-calendar-group-.md) 가이드를 참고하세요.
