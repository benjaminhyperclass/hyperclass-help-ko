# 수업 예약 위젯에서 슬롯별 좌석 수 표시하기

"슬롯별 좌석 수 표시" 설정을 사용하면 수업 예약(Class Booking) 캘린더의 각 시간대별 남은 좌석 수를 예약 위젯에 표시할 수 있습니다. 업데이트된 슬롯 수용량 로직으로 이제 예약이 여러 슬롯에 걸쳐 있어도 실제 남은 수용량이 정확히 반영됩니다.

## 개요

수업 예약(Class Booking) 캘린더는 여러 수강생이 같은 시간대에 예약할 수 있게 해줍니다. **슬롯별 좌석 수 표시** 옵션을 사용하면 각 시간대의 남은 좌석 수를 예약 위젯에 직접 표시할 수 있습니다.

이 기능의 장점:

- 남은 수용량을 명확하게 확인 가능
- 고객에게 예약 투명성 제공
- 대용량 이벤트의 일정 관리 효율성 향상

---

## **슬롯별 좌석 수 표시 활성화 방법**

예약 위젯에서 좌석 표시를 활성화하려면:

- 수업 예약 캘린더를 엽니다.
- **편집(Edit)**을 클릭합니다.
- **Customizations(커스터마이제이션)**으로 이동합니다.
- **Calendar Widget Style(캘린더 위젯 스타일)** 아래에서 **Show seats per slot(슬롯별 좌석 수 표시)**을 토글합니다.
- **저장(Save)**을 클릭합니다.

![슬롯별 좌석 수 표시 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155064723882/original/6EbYHBwQ9dB0zVnk3t7ABMt4XwSTaUJBGA.png?1770838784)

이 토글은 공개 예약 위젯에서 좌석 표시를 활성화합니다.

---

## **좌석 가용성 계산 방법 (업데이트된 로직)**

좌석 가용성은 캘린더 설정에서 구성한 **Seats per class(수업당 좌석 수)** 값에 따라 결정됩니다.

업데이트된 일정 관리 로직:

- 확정된 예약 하나당 좌석 가용성이 1 감소합니다.
- 예약이 여러 슬롯에 걸쳐 있어도 각 해당 슬롯에서 1개 좌석만 차감됩니다.
- 총 좌석 수용량이 완전히 찼을 때만 슬롯이 차단됩니다.
- 겹치는 예약이 더 이상 영향을 받는 모든 슬롯을 조기에 완전히 차단하지 않습니다.

이를 통해 더 정확한 가용성과 대용량 캘린더의 활용도가 향상됩니다.

---

## **예약 위젯에서 좌석 표시 방법**

활성화되면 각 시간대별 총 남은 좌석 수가 예약 위젯에 직접 표시됩니다.

![예약 위젯 좌석 표시](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155064723901/original/qzsAhmvJsnn32x8qma5F8_AyCB1Vc8h5Ww.png?1770838825)

고객은 시간대를 선택하기 전에 남은 좌석 수를 확인할 수 있습니다.

---

## **중요 정보**

- 이 기능은 **수업 예약(Class Booking) 캘린더**에서만 사용할 수 있습니다.
- **Seats per class(수업당 좌석 수)** 설정이 1보다 커야 합니다.
- Neo 위젯과 호환됩니다.
- 업데이트된 수용량 로직은 다음 모든 곳에 적용됩니다:

  - 공개 예약 위젯
  - 인앱 예약 모달
  - 개인 캘린더
  - 라운드 로빈 캘린더
  - 이벤트 캘린더

슬롯은 정의된 수용량 한계에 도달할 때까지 예약 가능 상태를 유지합니다.

## **자주 묻는 질문**

**Q: "슬롯별 좌석 수 표시"는 무엇을 하나요?**
A: 수업 예약 캘린더에서 각 시간대별 남은 좌석 가용성을 표시합니다.

**Q: 좌석 가용성은 어떻게 계산되나요?**
A: 확정된 예약 하나당 슬롯 수용량이 1씩 감소합니다. 정의된 수용량 한계에 도달했을 때만 슬롯이 차단됩니다.

**Q: 겹치는 예약이 전체 슬롯을 차단하나요?**
A: 아닙니다. 겹치는 예약은 슬롯을 완전히 차단하는 대신 해당 슬롯마다 수용량을 1씩 감소시킵니다.

**Q: 반복 수업 예약에서도 작동하나요?**
A: 네, 작동합니다.
