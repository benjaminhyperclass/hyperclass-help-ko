# 캘린더 뷰 개선사항

Hyperclass의 개선된 **캘린더 뷰**는 더 깔끔하고 넓으며 직관적인 방식으로 일정을 관리할 수 있게 해줍니다. 전용 예약 유형, 상황별 필터, 최적화된 레이아웃을 통해 사용자는 가장 중요한 것에 집중할 수 있습니다. 바로 적절한 예약, 직원, 서비스를 한눈에 보는 것입니다. 이러한 개선사항으로 미팅, 서비스, 향후 렌탈 예약 관리가 그 어느 때보다 간단하고 효율적이 되었습니다.

# **캘린더 뷰 개선사항이란?**

캘린더 뷰 개선사항은 복잡함을 줄이고 워크플로우에 동적으로 적응하도록 설계된 현대적인 예약 경험을 제공합니다. 예약 유형에 따라 **미팅**, **서비스(v2)**, 또는 향후 **렌탈**에 맞춰 뷰가 필터, 표시, 네비게이션 도구를 조정하므로 손쉽게 예약을 정리할 수 있습니다.

*이 업데이트는 유형별 캘린더 뷰, 간소화된 필터, 더 나은 가시성을 위해 화면 공간을 확대하는 유연한 보기 옵션을 도입하여 사용성을 개선하는 데 중점을 둡니다.*

## **캘린더 뷰 개선사항의 주요 장점**

주요 장점을 이해하면 생산성을 극대화하고 일일 예약을 간소화할 수 있습니다.

- **집중된 뷰:** 활성 예약 유형과 관련된 예약과 필터만 표시되어 더 이상 혼재된 결과나 시각적 혼란이 없습니다.

- **스마트 네비게이션:** 새 드롭다운으로 *미팅*과 *서비스* 간에 즉시 전환할 수 있습니다.

- **최적화된 작업 공간:** **뷰 관리(Manage View)** 토글을 사용해 필터를 숨기고 보이는 캘린더 영역을 확장할 수 있습니다.

- **빠른 월별 네비게이션:** 미니 월 뷰가 메인 날짜 아래로 이동하여 쉽게 접근할 수 있습니다.

- **더 깔끔한 인터페이스:** 에이전시에서 하나의 예약 유형만 활성화한 경우 드롭다운이 자동으로 숨겨져 방해받지 않는 작업 공간을 제공합니다.

- **통합 서비스 뷰:** 멀티 직원 서비스 예약을 하나의 뷰에서 명확하게 표시합니다.

## **전용 예약 유형**

Hyperclass의 캘린더 뷰는 이제 비즈니스 모델과 예약 요구사항에 더 잘 맞도록 별도의 예약 유형을 인식합니다.

*이 기능은 각 유형(미팅, 서비스, 렌탈)이 고유한 집중 작업 공간, 맞춤형 필터, 전용 관리 도구를 갖도록 보장합니다.*

- **미팅:** 모든 기존 캘린더 유형을 포함합니다. 개인 예약(Personal), 라운드 로빈(Round Robin), 공동 예약(Collective), 수업 예약(Class), 서비스 메뉴, 서비스 캘린더(v1).

- **서비스(v2):** 서비스 기반 비즈니스를 위해 구축되었습니다. **변형**, **애드온**, **직원 가격** 등의 고급 기능을 지원합니다.

- **렌탈(출시 예정):** 렌탈 기반 비즈니스(예: 스튜디오, 장소, 장비 대여)를 위해 설계된 향후 릴리스입니다.

## **스마트 드롭다운 네비게이션**

새로운 **스마트 드롭다운**으로 예약 유형 간 전환이 손쉬워집니다. **에이전시 설정(Agency Settings)**에서 여러 예약 유형이 활성화된 경우 캘린더 상단에 나타납니다. 하나의 유형만 활성 상태인 경우 드롭다운은 숨겨져 더 깔끔한 레이아웃을 보장합니다.

이러한 상황별 네비게이션은 워크플로우 연속성을 개선합니다. 더 이상 다른 캘린더 유형을 보기 위해 탭이나 하위 계정을 전환할 필요가 없습니다.

## **상황별 필터**

우측 필터 패널은 선택한 예약 유형에 따라 동적으로 업데이트됩니다. 각 뷰는 관련 필터만 제공하여 워크플로우를 간소화합니다.

*필터를 동적으로 조정함으로써 시각적 과부하 없이 가장 중요한 특정 예약, 서비스 또는 직원에 빠르게 집중할 수 있습니다.*

**미팅 뷰 필터:**
- 사용자(Users)
- 캘린더(Calendars)
- 그룹(Groups)

**서비스 뷰 필터:**
- 직원(Staff)
- 서비스(Services)
- 카테고리(Categories)
- 로케이션(Location, 통합 서비스 뷰 활성화 시)

## **통합 서비스 뷰**

활성화하면 **통합 서비스 뷰**는 모든 서비스 기반 예약을(직원 전체에 걸쳐) 하나의 통합 타임라인에 그룹화하여 서비스 일정의 완전한 가시성을 한눈에 제공합니다.

이 뷰는 서비스 기반 팀이 여러 로케이션에 걸친 중복, 직원 배정, 이용 가능한 시간대를 빠르게 식별하는 데 도움이 됩니다.

## **이벤트 보기 공간 확대**

가시성을 극대화하기 위해 개선된 캘린더 뷰는 패딩을 줄이고 필터 패널을 숨길 수 있게 해줍니다.

*이는 많은 예약을 처리하고 스크롤 없이 가능한 한 많은 이벤트를 봐야 하는 사용자에게 이상적입니다.*

**뷰 공간을 확장하는 방법:**
- **뷰 관리(Manage View)** 클릭 (우측 상단 모서리)
- 필요에 따라 우측 패널을 숨기거나 표시
- 줌이나 화면 크기를 조정하여 뷰에서 더 많은 예약을 표시

*이 위치 변경으로 흐름이 개선되고 필수 네비게이션 도구를 쉽게 접근할 수 있는 범위 내에 유지합니다.*

## **캘린더 뷰 개선사항 이용 가능 여부**

개선된 **캘린더 뷰** 기능은 에이전시 설정에서 **미팅**, **서비스(v2)** 또는 두 예약 유형이 모두 활성화되면 자동으로 이용할 수 있습니다. 활성화되면 캘린더 인터페이스가 자동으로 업데이트되며 추가 설정이 필요하지 않습니다.

여러 예약 유형이 활성화되면 캘린더 뷰 상단에 **유형 선택기**가 나타나 사용자가 **미팅**과 **서비스** 간에 전환할 수 있습니다. 하나의 예약 유형만 활성 상태인 경우 선택기는 숨겨져 간소화된 레이아웃을 유지합니다.

우측의 **뷰 관리** 패널에는 선택한 예약 유형에 따라 조정되는 동적 필터가 포함됩니다. 이를 통해 사용자는 미팅의 경우 **사용자**, **캘린더** 또는 **그룹**과 같은 관련 필터만 보고, 서비스의 경우 **직원**, **서비스** 또는 **카테고리**만 봅니다. 서비스 기반 예약을 제공하는 에이전시는 **통합 서비스 뷰**를 사용하여 모든 팀원의 서비스 예약을 하나의 통합된 타임라인에서 볼 수 있습니다.

업데이트된 레이아웃은 더 나은 가시성과 유연성을 위해 설계되었습니다. 필터 패널을 접어서 화면에 더 많은 예약을 표시할 수 있고, 메인 날짜 헤더 아래에 위치한 **미니 월 캘린더**로 현재 뷰를 벗어나지 않고도 빠른 날짜 네비게이션이 가능합니다.

## **자주 묻는 질문**

**Q: 새 뷰를 사용하기 위해 기존 캘린더를 마이그레이션해야 하나요?**

A: 아니요. 모든 기존 캘린더는 자동으로 **미팅** 뷰에 나타납니다. **서비스(v2)** 캘린더는 해당 기능이 활성화되면 **서비스** 뷰에 표시됩니다.

**Q: 미팅/서비스 드롭다운이 보이지 않는 이유는 무엇인가요?**

A: 드롭다운은 **에이전시 설정**에서 여러 예약 유형이 활성 상태일 때만 나타납니다.

**Q: 예약을 여전히 드래그 앤 드롭할 수 있나요?**

A: 네, 이전과 같이 예약을 드래그, 드롭, 일정 변경, 취소할 수 있습니다.

**Q: 필터 패널을 숨기는 것이 다른 사용자에게 영향을 미치나요?**

A: 아니요. **뷰 관리** 설정은 사용자별로 적용되며 다른 사용자에게 영향을 주지 않습니다.

**Q: Google 캘린더가 서비스(v2)와 동기화되나요?**

A: 현재는 되지 않습니다. Google 캘린더 동기화는 미팅에만 적용되며, 서비스는 Hyperclass 내에서 직접 관리해야 합니다.

**Q: 렌탈을 이미 이용할 수 있나요?**

A: 렌탈은 개발 중이며 향후 업데이트에서 추가될 예정입니다.

**Q: 여러 하위 계정에 대해 서비스(v2)를 한 번에 활성화할 수 있나요?**

A: 네. `에이전시 설정(Agency Settings) → 하위계정(Subaccount) → 일괄 작업(Bulk Actions) → 캘린더 설정(Setup Calendars)`으로 이동하여 계정 전체에서 서비스를 활성화할 수 있습니다.
