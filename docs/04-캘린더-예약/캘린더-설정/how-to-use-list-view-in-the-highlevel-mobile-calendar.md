# 하이퍼클래스 모바일 캘린더의 목록 보기 사용법

이 문서에서는 하이퍼클래스 모바일 캘린더(Calendar) 모듈의 새로운 **목록 보기(List View)** 사용법을 알려드려요. 빠른 스캔과 깔끔한 정리를 위해 설계된 이 뷰는 다가오는 예약(Appointment)들을 스크롤 가능한 목록으로 보여줍니다—복잡함 없이 명확함을 원하는 바쁜 전문가들에게 완벽해요.

## **모바일 캘린더의 목록 보기란 무엇인가요?**

목록 보기는 하이퍼클래스와 LeadConnector **모바일 앱**에서 사용할 수 있는 새로운 캘린더 표시 형식이에요. 기존의 그리드 기반 레이아웃(월간 또는 주간)과 달리, 목록 보기는 캘린더 이벤트를 **시간순으로 정렬된 스크롤 가능한 피드**로 표시합니다. 이 설정은 시각적 복잡함을 제거하고 사용자가 다가올 일정에 집중할 수 있게 해줍니다—특히 작은 화면에서 매우 유용해요.

## **목록 보기 사용의 주요 장점**

**목록 보기는 명확성, 속도, 사용 편의성을 위해 최적화되었으며, 특히 작은 화면에서 뛰어나요. 여러분이 좋아할 이유는 다음과 같아요:**

- **선형 계획**: 캘린더 그리드 없이 위에서 아래로 간단한 목록으로 이벤트를 확인하세요.
- **모바일 친화적 표시**: 시각적 방해 요소를 최소화하여 제한된 화면 공간을 최대한 활용해요.
- **스크롤 기반 네비게이션**: 간단한 스크롤만으로 전체 캘린더를 탐색하세요—날짜 간 탭 이동이 필요 없어요.
- **집중 친화적**: 오늘의 일정에 쉽게 집중하거나 앞으로 스크롤하여 향후 이벤트를 준비하세요.
- **부드러운 토글 접근**: 한 번의 탭으로 목록 보기와 다른 뷰 사이를 빠르게 전환할 수 있어요.

## **모바일 캘린더에서 목록 보기로 전환하는 방법**

**목록 보기로 전환하는 것은 간단하며 몇 번의 탭만으로 완료됩니다. 읽기 쉬운 형식으로 전체 캘린더에 즉시 접근할 수 있어요.**

1. **하이퍼클래스 모바일 앱을 열고** **캘린더 모듈**로 이동하세요.

*스크린샷: 캘린더 아이콘이 선택된 메인 네비게이션.*

2. **캘린더 화면 오른쪽 상단의 뷰 토글(그리드 아이콘)을 탭**하세요.

*스크린샷: 그리드 아이콘 주변에 원이 그려진 캘린더 화면.*

3. **드롭다운 메뉴에서 "목록 보기(List View)"를 선택**하세요.

*스크린샷: "Monthly View", "Weekly View", "List View"가 표시된 드롭다운 메뉴에서 List View가 선택된 모습.*

4. 이제 캘린더가 날짜와 시간별로 정리된 **수직 스택 목록**으로 이벤트를 표시합니다.

*스크린샷: 날짜와 시간순으로 정렬된 이벤트들의 스크롤 목록, 각각 고유한 카드나 행에 표시.*

5. **아래로 스크롤**하여 다가오는 이벤트를 탐색하거나 **위로 스크롤**하여 최근 활동을 확인하세요.

6. 그리드 기반 뷰로 돌아가려면 이 단계를 반복하고 "월간 보기(Monthly View)" 또는 "주간 보기(Weekly View)"를 선택하면 됩니다.

## **기능 하이라이트: 모바일 계획에 완벽**

**목록 보기는 하나의 깔끔한 레이아웃에서 효율성, 가시성, 제어를 원하는 모바일 사용자를 위해 맞춤 제작되었어요.**

작은 화면에서는 기존의 그리드 뷰가 복잡하고 해석하기 어려울 수 있어요. 목록 보기를 사용하면 모든 이벤트가 고유한 공간을 갖게 되고, 사용자는 일정을 자연스럽게 탐색할 수 있습니다. 오늘의 다음 일정을 확인하든 다음 주를 계획하든, 목록 보기는 프로세스를 단순화하고 워크플로우에서 마찰을 제거해줘요.

## **자주 묻는 질문**

**Q: 목록 보기는 데스크톱에서도 사용할 수 있나요, 아니면 모바일에서만 가능한가요?**

A: 목록 보기는 현재 **모바일 버전에서만** 사용 가능합니다(하이퍼클래스 및 LeadConnector 앱). 데스크톱 지원은 향후 업데이트에서 검토될 수 있어요.

**Q: 월간 캘린더 뷰로 다시 전환할 수 있나요?**

A: 네! 동일한 뷰 토글을 탭하고 "월간(Monthly)" 또는 "주간(Weekly)"을 선택하면 그리드 레이아웃으로 돌아갈 수 있어요.

**Q: 목록 보기에서는 어떤 유형의 이벤트가 표시되나요?**

A: 할 일(Task), 예약, 약속 등 모든 유형의 이벤트가 목록 보기 피드에 포함됩니다.

**Q: 목록 보기에서 필터링하거나 검색할 수 있나요?**

A: 목록 보기에서는 아직 필터(Filter)가 지원되지 않지만, 스크롤해서 이벤트를 찾거나 캘린더 모듈 내 검색(Search) 기능을 사용할 수 있어요.
