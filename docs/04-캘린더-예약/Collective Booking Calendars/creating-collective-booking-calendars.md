# 공동 예약 캘린더 만들기

**목차**

- [캘린더 기본 소유자 설정](#캘린더-기본-소유자-설정)
- [알림은 어떻게 작동하나요?](#알림은-어떻게-작동하나요)
- [결제](#결제)

- Calendars(캘린더) ➝ Calendar Settings(캘린더 설정)으로 이동하세요

- "Create Calendar(캘린더 만들기)"를 클릭하세요

- "Collective Booking(공동 예약)"을 선택하세요

![공동 예약 선택 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155004350274/original/RpvzHmj3XUSnFky8A8G_PFRP4ppgcsBrAA.png?1691038898)

- 캘린더의 필수 정보를 입력하세요
  - Calendar Name(캘린더 이름)
  - Select Team members(팀원 선택) - 공동 예약 캘린더는 최소 2명의 팀원이 필요합니다
  - Custom URL(사용자 정의 URL) - 캘린더 이름을 기반으로 자동 생성됩니다. 필요시 수동으로 수정할 수 있습니다.
  - Meeting Location(미팅 장소) - 필수는 아니지만 권장합니다. 미팅 장소가 설정되지 않으면 기본 소유자의 사용자 정의 미팅 장소가 기본값으로 사용됩니다.

- "Confirm(확인)"을 클릭하면 공동 예약 캘린더가 생성됩니다.

![공동 예약 캘린더 생성 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155004350272/original/ubYHojSvUY0ZC2qiofg-Hf6ANvpL1DG0oQ.png?1691038897)

3. 생성 후 캘린더를 맞춤 설정하려면 Edit calendar(캘린더 편집)를 클릭하세요. 캘린더 설정은 쉬운 탐색을 위해 섹션별로 그룹화되어 있습니다. 결제, 알림 및 정책, 위젯 모양, 외부 동기화, 예약 채널과 같은 항목의 경우 Advanced settings(고급 설정)을 여세요.

**다양한 서비스를 제공하는 비즈니스를 위한 팁:**

냉장고 수리, 세탁기 수리, 건조기 수리 등 여러 서비스를 제공하는 경우, 예약을 효과적으로 정리할 수 있는 두 가지 방법이 있습니다:

**옵션 1:** 서비스별로 별도의 캘린더를 만드세요. 이렇게 하면 특정 서비스에 따라 다른 팀원, 예약 가능 시간, 미팅 선호도를 할당할 수 있습니다.

**옵션 2:** 단일 공동 예약 캘린더를 사용하고 서비스 선택 드롭다운이 있는 커스텀 폼을 추가하세요. 이를 통해 고객이 예약 시 서비스 유형을 선택할 수 있고, 연락처 할당, 태그 또는 자동화를 사용하여 내부적으로 해당 예약을 라우팅할 수 있습니다.

## 캘린더 기본 소유자 설정

공동 예약에서 기본 소유자는 예약을 주도하는 팀원입니다. 기본 소유자는 예약의 소유자가 되며, 연락처 할당이 활성화되어 있다면 연락처가 기본 소유자에게 할당됩니다.

기본적으로 공동 예약에 추가된 첫 번째 팀원이 기본 소유자가 되며, 공동 예약에서 기본 소유자는 삭제할 수 없습니다.

**기본 소유자 변경**

기본 소유자를 변경하려면:

- 기본 소유자를 변경하려면 Edit calendar(캘린더 편집)을 열고 Basic details(기본 세부 정보) 또는 미팅/소유자 세부 정보가 포함된 섹션으로 이동하세요.

- 현재 기본 소유자가 아닌 모든 팀원 옆에 "Make owner(소유자로 만들기)" 옵션이 표시됩니다

- 기본 소유자로 만들고자 하는 팀원 옆의 "Make owner(소유자로 만들기)"를 클릭하세요

- 아래와 같이 소유자가 변경되며 그들 옆에 "Owner(소유자)" 라벨로 표시됩니다.

![기본 소유자 변경 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155066595536/original/qZOHlub9CbviIBOEjCjlLv1LOLKyl3a2Zw.png?1773146140)

**주의사항:**

기본 소유자는 공동 예약 캘린더에서 삭제할 수 없습니다. 팀원을 삭제하려면 해당 팀원이 기본 소유자가 아닌지 확인하세요

![주의사항 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155004350273/original/_hR4OVjyuR5g_jgkE9AzpGFhfWWbItXI8w.png?1691038897)

## 알림은 어떻게 작동하나요?

할당된 사용자만 이메일 알림을 받습니다. 하지만 모든 사용자는 연동된 서드파티 캘린더(Google, Outlook, iCloud)로부터 캘린더 초대 알림을 받게 됩니다.

## 결제

캘린더에서 결제가 활성화되어 있다면 공동 예약에서도 결제가 계속 작동합니다. 사용자는 먼저 결제를 완료해야 하며, 결제가 확인되어야만 예약이 완료됩니다.
