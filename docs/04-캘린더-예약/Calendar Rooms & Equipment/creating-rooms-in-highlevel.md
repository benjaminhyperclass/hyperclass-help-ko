# 룸이란 무엇인가요?

룸은 서비스 캘린더에서 예약, 가용성 및 리소스 관리를 제어하기 위해 할당할 수 있는 자원입니다. 상담실, 서비스 베이, 치료실, 또는 이벤트 공간을 관리하든, 룸은 중복 예약을 방지하고 비즈니스 전반에 걸쳐 자원이 적절히 배정되도록 돕습니다. 룸은 장비, 사용자, 가용성 설정과도 연결할 수 있어 예약 기반 워크플로우에서 필수적인 요소입니다.

## **룸의 주요 장점**

룸 사용의 장점을 이해하면 예약 효율성을 극대화하고 리소스 충돌을 방지할 수 있습니다. 룸은 공간, 도구, 사람이 캘린더 로직과 일치하도록 하여 원활한 예약 경험을 지원합니다.

- **룸을 캘린더와 연결하여 중복 예약 방지**
- **룸을 장비와 연결하여 리소스 관리 가능**
- **사용자나 팀 배정을 통한 팀 기반 접근 제어 지원**
- **가용성 설정과 통합하여 예약 시간대 제어**
- **가용성을 준수하면서 여러 서비스에 걸쳐 유연성 제공**
- **예약 기반 비즈니스를 위한 워크플로우 최적화**

## **룸 설정 방법**

룸을 만들면 캘린더 가용성이 물리적 공간이나 가상 리소스와 일치하도록 하여 겹치는 예약을 방지하고 예약 정확도를 향상시킬 수 있습니다.

### **1단계:** 룸 기능 활성화

- 기본 네비게이션 바에서 Settings(설정) > Calendar Settings(캘린더 설정)로 이동하세요
- 보조 네비게이션 리본에서 Preferences(환경설정) 탭을 선택하세요
- Preferences 하위의 Account Preference(계정 환경설정) 탭을 선택하세요
- Services 섹션에 있는 파란색 Rooms 토글을 켜세요
- 우측 하단의 Save Preferences(환경설정 저장) 파란색 버튼을 클릭하세요

![룸 기능 활성화](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044906951/original/32eZsrTzhZLooDpzZQ01UBOBYVJANRl-mw.png?1744312903)

### **2단계:** 룸 만들기

- Calendar Settings(캘린더 설정)의 보조 네비게이션 리본에서 Calendars(캘린더) 탭으로 돌아가세요
- Calendar Settings 하위의 Rooms(룸) 탭을 선택하세요
- 우측 상단의 + Create Room(룸 만들기) 파란색 버튼을 클릭하세요

![룸 만들기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044907019/original/38Jt5GHOrUJQTKNSH5W4o7nqlh-uwVvE0A.png?1744313048)

### **3단계:** 룸 세부사항 입력

룸을 설정하려면 다음 필드를 완성하세요:

- **Name**: 공간을 식별할 수 있는 명확하고 구체적인 이름을 지정하세요 (예: "상담실 2").
- **Description**: 룸의 용도에 대한 추가 메모를 작성하세요.
- **Total Capacity**: 룸에서 동시에 예약할 수 있는 최대 예약 건수를 지정하세요.
- **Select Calendar**: 이 장비와 연결하고자 하는 캘린더를 선택하세요.

모든 세부사항을 입력한 후 Create(만들기)를 클릭하여 룸을 생성하세요.

![룸 세부사항 입력](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044824333/original/rA5bnZ_Z_1kFGC7xCNztKhDNuhKbIDwM-w.png?1744215319)

## **룸 편집 또는 삭제하기**

룸 목록을 정확하게 유지하면 예약 문제와 사용자 혼란을 방지할 수 있습니다. 룸 이름을 바꾸거나, 연결된 캘린더를 업데이트하거나, 사용하지 않는 룸을 완전히 제거해야 하는 경우, 이러한 간단한 작업으로 작업 공간을 체계적이고 기능적으로 유지할 수 있습니다.

- **Settings(설정) > Calendar Settings(캘린더 설정)**로 이동하세요
- Rooms(룸) 탭을 선택하세요
- **점 3개** 아이콘을 클릭하고 Edit(편집)을 선택하여 룸을 편집하거나 Delete(삭제)를 선택하여 룸을 삭제하세요
- 변경사항을 적용한 후 **Save(저장)** 파란색 버튼을 클릭하세요

![룸 편집 또는 삭제](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044907138/original/lRuwhEsB_Ox2R5Obi2TCF2NHkr5itjWTdA.png?1744313221)

**자주 묻는 질문**

**Q: 같은 룸을 여러 캘린더에 배정할 수 있나요?**
A: 네, 가능합니다. 하지만 가용성이 제대로 관리되지 않으면 겹치는 예약이 발생할 수 있습니다.

**Q: 사용자가 룸에 추가되면 알림을 받나요?**
A: 네, 사용자가 룸에 배정되면 알림이 전송됩니다.

**Q: 고객이 룸을 보거나 참여할 수 있나요?**
A: 아니요. 룸은 내부적으로만 볼 수 있으며 리소스 관리용으로만 사용됩니다.

**Q: 룸이 캘린더에 연결되지 않으면 어떻게 되나요?**
A: 예약 로직에 반영되지 않아 중복 예약이 발생할 수 있습니다.

**Q: 룸을 만든 후 이름을 바꿀 수 있나요?**
A: 네, 편집 메뉴를 통해 언제든지 룸 이름을 바꿀 수 있습니다.

**Q: 룸을 몇 개까지 만들 수 있나요?**
A: 공식적인 제한은 없지만, 이름과 배정을 체계적으로 유지하는 것을 강력히 권장합니다.
