# **카테고리란?**

카테고리는 비슷하거나 관련된 서비스를 그룹화하여 고객이 서비스를 더 쉽게 탐색할 수 있게 해줍니다. 카테고리는 예약 페이지에서 정리된 섹션으로 표시되며, 각 서비스는 할당된 카테고리 아래에 나열됩니다.

예를 들어, 스파의 경우 마사지, 헤어, 피부, 살롱과 같은 카테고리를 만들고, 각 카테고리 아래에 여러 서비스를 나열할 수 있습니다.

---

## **카테고리의 주요 장점**

이러한 장점을 이해하면 예약 페이지를 깔끔하게 유지하고 고객이 적절한 위치에서 올바른 서비스를 볼 수 있습니다.

- **조직화:** 서비스를 명명된 섹션으로 그룹화하여 예약 페이지에 표시합니다.

- **빠른 발견:** 고객이 관심 있는 서비스를 빠르게 찾을 수 있도록 도와줍니다.

- **직접 공유:** 카테고리 URL(슬러그)을 사용하여 해당 카테고리의 서비스만 표시하는 링크를 공유할 수 있습니다.

- **표시 순서 제어:** 고객용 예약 페이지에서 카테고리가 나타나는 순서를 제어할 수 있습니다.

- **내장 백업:** 기본 카테고리가 존재하여 삭제된 카테고리의 서비스(및 할당되지 않은 새 서비스)가 예약 가능 상태로 유지됩니다.

---

## **기본 카테고리**

모든 계정은 기본 카테고리로 시작하며, 이는:

- 삭제할 수 없습니다(하지만 이름을 변경할 수는 있습니다).

- 카테고리가 할당되지 않은 모든 새 서비스는 시스템에 의해 기본 카테고리에 배치됩니다.

- 카테고리가 삭제되면 해당 카테고리의 모든 서비스가 기본 카테고리로 이동하여 예약 가능 상태를 유지합니다.

이렇게 해서 모든 서비스는 항상 카테고리에 속하게 되어 예약 페이지가 정리되고 기능적으로 유지됩니다.

![기본 카테고리](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155053324896/original/8xI4ErI9elk7axqmBaVszFT-yCpzORRuDA.png?1757355794)

## **새 카테고리 만들기**

관련 서비스를 그룹화하고 해당 카테고리에 대한 공유 가능한 링크를 생성하기 위해 카테고리를 설정하세요.

- `Calendar(캘린더) > Calendar Settings(캘린더 설정) > Services(서비스)`로 이동합니다.

- 왼쪽 사이드바에서 `+ New Category` 버튼을 클릭합니다.

![새 카테고리 만들기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155053325082/original/-gl_csE2AD2dJDkv_dDJ3-96SRxfII-3Zw.png?1757355992)

- 카테고리 이름을 입력합니다.

- 카테고리 URL(슬러그)을 정의합니다 — 이것으로 카테고리의 직접 공유 링크가 생성됩니다.

- 파란색 `Create Category` 버튼을 클릭해서 변경사항을 저장합니다.

![카테고리 생성](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155053325107/original/S9jria6y7lKUg_eJNkwOabAMw6gTydvmrQ.png?1757356059)

이미 생성된 서비스가 있다면, 카테고리 생성 후에 해당 새 카테고리에 할당할 수 있습니다.

---

## **서비스를 카테고리에 할당하기**

각 서비스를 적절한 카테고리에 배치하여 공개 예약 페이지의 올바른 섹션에 나타나도록 하세요. 할당되지 않은 서비스는 기본 카테고리에 위치하며, 삭제된 카테고리의 서비스는 자동으로 기본 카테고리로 이동됩니다. 필요할 때마다 재할당할 수 있습니다.

- 서비스 옆의 **점 3개 메뉴**를 클릭합니다.

- `Edit`을 선택합니다.

![서비스 편집](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155053328077/original/eh8A9xHd6y9EYazpkdhW89tZZl8pcWhqag.png?1757361153)

- 카테고리 드롭다운을 사용하여 서비스를 할당할 카테고리를 선택합니다.

- 우측 상단의 파란색 `Save Changes` 버튼을 클릭해서 변경사항을 저장합니다.

![서비스 카테고리 할당](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155053328113/original/ZFYLNKFLoqgTGd5rX-skWOjSEyKhFmQxRw.png?1757361256)

## **카테고리 편집하기**

서비스 제공이 변화함에 따라 카테고리 이름을 변경하거나 공유 가능한 URL을 조정하세요.

- 카테고리 옆의 **점 3개 메뉴**를 클릭합니다.

- `Edit`을 선택합니다.

![카테고리 편집](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155053325251/original/WE9VHvPXpjzKUqDVuQJ2deHCZ-Ly00t2Bg.png?1757356197)

- 카테고리 이름 또는 카테고리 URL(슬러그)을 업데이트합니다.

- 변경사항을 저장합니다.

![카테고리 정보 수정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155053325303/original/MPykI0FLoJP78tJj6HsVysV899ZVTLX2Ng.png?1757356265)

## **카테고리 공유하기**

카테고리에는 고유한 공유 가능한 예약 링크가 있습니다. 특정 카테고리를 홍보하고 싶을 때 유용합니다(예: "모든 헤어 서비스").

- **점 3개 메뉴** → `Share Booking`을 클릭합니다.

![카테고리 공유](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155053325380/original/j8w3FyeXYARmJhibB3qtMURCCRITtCYv3w.png?1757356414)

- 해당 카테고리 내의 서비스만 고객이 볼 수 있도록 링크를 공유하세요.

![카테고리 링크 공유](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155053325400/original/IRcprN1SXB6NsPd48fp20Kd3PRovpwZV6w.png?1757356450)

---

## **카테고리 삭제하기**

서비스는 고객이 접근 가능한 상태로 유지하면서 더 이상 필요하지 않은 카테고리를 제거하세요.

- **기본 카테고리**를 제외한 카테고리는 삭제할 수 있습니다.

- 카테고리를 삭제할 때:
  - 해당 카테고리의 **모든 서비스**는 자동으로 **기본 카테고리로 이동**합니다.

- 필요하다면 해당 서비스들을 **다른 카테고리로 재할당**할 수 있습니다.

카테고리를 삭제하려면 다음 단계를 따르세요:

- **점 3개 메뉴** → `Delete`를 클릭합니다.

![카테고리 삭제](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155053325567/original/4V9fPB-JSBIiiOoUECO07dW7NlA1O04Taw.png?1757356647)

- 빨간색 `Delete` 버튼을 클릭해서 삭제를 확인합니다.

![삭제 확인](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155053325601/original/pYH5OzOJzVCani4cMvaTPLM5IThrnjg0Vw.png?1757356683)

## **카테고리 순서 변경**

더 나은 브라우징 경험을 위해 고객이 예약 페이지에서 보는 카테고리 순서를 제어하세요.

- 왼쪽 사이드바에서 카테고리를 드래그 앤 드롭하여 순서를 변경합니다.

- 새로운 순서가 고객용 예약 페이지에 자동으로 반영됩니다.

![카테고리 순서 변경](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155053325835/original/8XzjOgpHoudiSGzFqq2RkLvpE2fm1fJbfw.png?1757357006)

## **모범 사례**

- 의미 있는 카테고리 이름을 사용하세요(예: "웰니스" 대신 "마사지").

- 기본 카테고리를 비즈니스에 맞는 구체적인 이름으로 변경하는 것을 고려해보세요.

- 프로모션이나 타겟 캠페인을 위해 카테고리별 예약 링크를 공유하세요.

---

## **자주 묻는 질문**

**Q: 나중에 카테고리 URL(슬러그)을 변경할 수 있나요?**

네. 점 3개(⋮) 메뉴에서 편집을 사용하여 카테고리 URL(슬러그)을 수정할 수 있습니다.

**Q: 카테고리를 사용하기 전에 서비스를 활성화해야 하나요?**

네. 하위 계정에서 서비스를 활성화해야 서비스 패널에 접근하고 카테고리를 생성하며 예약 링크를 공유할 수 있습니다. 에이전시 뷰에서 하나 또는 여러 하위 계정에 대해 서비스를 활성화할 수 있습니다.

**Q: 카테고리 링크를 공유할 때 비공개 서비스가 나타나나요?**

아니요. 비공개 서비스는 공개 카테고리 페이지에서 숨겨집니다. 비공개 서비스를 공유해야 한다면 직접 링크를 복사해서 공유하세요.
