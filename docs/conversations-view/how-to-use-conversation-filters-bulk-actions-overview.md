# 대화 필터와 일괄 작업 사용법 - 개요

대화 좌측 패널은 대화(Conversations) UI에 새롭게 추가된 기능으로, 사용자가 대화를 필터링하고 다양한 작업을 수행할 수 있도록 도와줍니다.

## 기능:

### 일괄 작업 (한 번에 최대 100개 대화)

대화 관련 일괄 작업(Bulk Actions)을 사용해 여러 대화를 한 번에 처리할 수 있습니다. 다음과 같은 대화 일괄 작업을 수행할 수 있어요:

- 대화를 읽음으로 표시: 모든 대화가 읽음으로 표시됩니다
- 대화를 읽지 않음으로 표시: 모든 대화가 읽지 않음으로 표시됩니다
- 대화를 즐겨찾기로 표시: 모든 대화가 즐겨찾기로 표시됩니다
- 대화 즐겨찾기 해제: 모든 대화의 즐겨찾기가 해제됩니다
- 대화 삭제: 대화가 삭제됩니다

![일괄 작업 사용 예시](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48252140099/original/ZFVOXmJkjeFg4vlOU2A2W0qudclJZGsG0Q.gif?1663699138)

### 대화 필터 (빠른 필터)

몇 가지 필터를 사용해 대화를 필터링할 수 있어요. 현재 필요한 대화만 좁혀서 볼 수 있는 기능입니다. 다음과 같은 필터를 사용할 수 있어요:

- 대화 유형별
- 담당자별
- 마지막 메시지 채널별: 선택한 채널에서 마지막 메시지가 온 대화를 필터링합니다
- 마지막 메시지 유형별: 선택한 유형으로 마지막 메시지(발신)를 보낸 대화를 필터링합니다
- 마지막 메시지 방향별: 선택한 방향으로 마지막 메시지가 온 대화를 필터링합니다
- 마지막 발신 메시지 유형별
- 마지막 발신 메시지 액션별: 수동 또는 자동 (이 필터는 2023년 5월 9일 이후 메시지에 적용됩니다)

![필터 사용 예시](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48299575358/original/zoKVYLITFWkYro7ETUOb50p3cXlGoJ-OGA.gif?1685125690)

### 대화 정렬

시간을 기준으로 대화를 정렬할 수 있어요. 대화를 더 쉽게 확인할 수 있도록 도와줍니다. 다음과 같이 정렬할 수 있어요:

- 최신순
- 오래된 순

![정렬 사용 예시](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48252137561/original/xr6GrnN_JuvC9gqJdg0-Gbqlfi_ZLOuKlQ.gif?1663698260)

**참고사항**

이 기능은 위에서 언급한 빠른 필터와 함께 사용하여 대화를 더 체계적으로 관리할 수 있습니다.

![필터와 정렬 조합 예시](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48252138548/original/oa-EtJAzAhcAABHrfHcymxV7yq4XsNSRaA.png?1663698642)

즉, 필터를 적용한 상태에서 최신순이나 오래된 순으로 정렬하면 선택한 필터 조건도 함께 고려됩니다.

### 즐겨찾기 기능:

대화를 즐겨찾기로 표시할 수 있어요. 즐겨찾기로 표시하면 대화에 식별용 아이콘이 붙습니다.

대화를 즐겨찾기로 표시하는 방법:

대화 미리보기 창의 우측 상단에 있는 별표 아이콘을 클릭하세요.

대화의 즐겨찾기를 해제하려면 별표 아이콘을 다시 클릭하세요.

![즐겨찾기 사용 예시](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48252134426/original/bqutf0YAwKHGdUS4uEsQXbPtm3hAp0EznQ.gif?1663697310)
