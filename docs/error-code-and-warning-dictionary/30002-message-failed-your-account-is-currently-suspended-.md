# 오류 30002: 메시지 전송 실패 - 계정이 현재 정지됨

## **오류: 30002**

**오류 메시지:** "Message failed: Your account is currently suspended"

**의미:** 메시지 전송이 실패했습니다. 현재 계정이 정지된 상태입니다.

**해결 방법:**

1. **계정 상태 확인**: Settings(설정) → Billing(결제/청구)에서 계정 상태를 확인해주세요.

2. **결제 정보 업데이트**: 결제 실패로 인한 정지인 경우, 올바른 결제 수단을 등록해주세요.

3. **고객 지원팀 문의**: 정지 사유를 정확히 파악하기 위해 Hyperclass 고객 지원팀에 문의하시기 바랍니다.

4. **정책 위반 검토**: 메시징 정책이나 이용약관 위반 여부를 검토해주세요.

계정이 정지된 동안에는 SMS(문자), 이메일, 전화 등 모든 외부 통신 기능이 제한됩니다. 정지가 해제되면 정상적으로 메시지를 보낼 수 있습니다.
