# 오류: 30006 - 유선전화 또는 접근 불가능한 통신사

## **오류 코드: 30006**

---

**설명**

**메시지 전달 실패 - 유선전화 또는 접근 불가능한 통신사**

수신 번호가 이 메시지를 받을 수 없습니다.

**가능한 원인**

- 수신 번호가 이 메시지를 받을 수 없습니다. 유선전화로 발송을 시도했거나, 단축 번호(short code)의 경우 접근할 수 없는 통신사일 가능성이 있습니다.
- 메시지가 유선전화로 발송되었거나, 해당 전화번호 유형에 대해 접근할 수 없는 통신사로 전송되었습니다.
