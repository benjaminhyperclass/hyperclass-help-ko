# 오류: 30005 - 수신자 번호가 비활성 상태이거나 존재하지 않습니다

## 오류 코드: 30005

---

**발생 가능한 원인**

- 연결하려는 대상 번호가 알 수 없는 번호이며 더 이상 존재하지 않을 수 있습니다.
- 연결하려는 기기가 켜져 있지 않거나 신호가 충분하지 않습니다.
- 해당 기기가 SMS(문자)를 수신할 수 없습니다 (예: 전화번호가 유선전화에 속한 경우)
- 이동통신사에 문제가 있습니다.

**해결 방법**

- 요청 내용을 주의 깊게 확인하고, 올바른 E.164 형식으로 정확한 전화번호에 메시지를 보내려고 하는지 다시 한 번 검증하세요: [+] [국가번호] [지역번호를 포함한 가입자번호]
- Lead Connector 지원팀이 메시지 전송 중 발생한 문제를 조사하는 데 도움을 드릴 수 있습니다. 지난 24시간 동안 SMS(문자) 로그에서 오류 30005로 표시된 3개 이상의 사례를 수집하여 지원 요청을 열어주세요.
