# 오류 코드 30004: 수신자가 수신 거부했거나 DND가 활성화됨

**오류: 30004**

---

## 발생 가능한 원인

- 메시지를 보내려는 대상 번호가 이 메시지 수신을 차단했습니다.
- 연결하려는 기기의 신호가 불충분합니다.
- 해당 기기가 SMS를 수신할 수 없습니다(예: 전화번호가 유선전화에 속함).
- 대상 번호가 인도의 국가 수신 거부 등록부(Do Not Call registry)에 등록되어 있습니다.
- 이동통신사에 문제가 있습니다.
- 미국/캐나다 무료 번호에서 이전에 "STOP" 또는 다른 수신 거부 키워드로 응답한 최종 사용자 기기로 메시지를 보냈습니다.

## 해결 방법

**인도로 보낸 "미전달" 메시지**

인도로 보낸 메시지가 전달되지 않아 오류 30004가 발생한 경우, 대상 번호가 국가 수신 거부(DNC) 등록부에 등록되어 있을 수 있습니다.

**Lead Connector 지원팀**이 메시지 전달 실패 원인을 조사하는 데 도움을 드릴 수 있습니다. 지난 24시간 동안 SMS 로그에서 오류 30004로 표시된 메시지 예시 3개 이상을 수집하여 지원 요청을 해주세요.
