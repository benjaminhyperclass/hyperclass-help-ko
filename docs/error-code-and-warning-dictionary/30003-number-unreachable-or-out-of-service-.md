# 오류 30003: 연결할 수 없거나 서비스 중단된 번호

## **오류: 30003**

### 설명

연결하려는 대상 전화기가 꺼져 있거나 사용할 수 없는 상태입니다.

### 가능한 원인

- 연결하려는 대상 전화기가 꺼져 있거나 사용할 수 없음
- 연결하려는 기기의 신호가 불충분함
- 기기가 SMS를 받을 수 없음 (예: 유선전화 번호)
- 통신사에 문제가 있음

### 가능한 해결 방법

일시적인 다운스트림 문제로 인해 배달 오류가 한 번 발생할 수 있습니다. 문제가 다시 발생하는지 테스트해보세요.

비슷한 결과가 나온다면 다음 체크리스트로 문제를 해결해보세요:

- 대상 기기의 전원이 켜져 있나요?
- 기기에 충분한 신호가 있나요? 신호가 없다면 기기를 끄고 30초 기다린 후 다시 켜보세요.
- 기기가 본국 통신사 네트워크에 연결되어 있나요? 네트워크 로밍 중인 기기에는 메시지 배달을 보장할 수 없습니다.
- 기기가 Hyperclass가 아닌 다른 SMS를 받을 수 있나요?
- 기기가 다른 Hyperclass 번호(영숫자 발신자 ID가 아닌)나 더 짧은 단일 세그먼트(연결되지 않은) 메시지를 받을 수 있나요?
- 같은 통신사를 사용하는 다른 기기들이 내 메시지를 받을 수 있나요?
- 위의 모든 문제를 배제할 수 있다면 아래의 문제 해결 과정을 계속 진행하세요.

### 롱 코드에서 보낸 메시지

롱 코드(일반 전화번호)와 무료 번호에서 보낸 메시지가 배달되지 않고 반복적으로 오류 30003이 발생한다면 통신사 필터링 때문일 수 있습니다.

통신사 필터링 규칙은 업계마다 다르기 때문에 메시지가 왜 필터링되는지 정확히 파악하기 어려울 때가 있습니다. 잠재적 해결 방법을 포함한 자세한 내용은 "통신사 필터링은 어떻게 작동하나요?" 도움말 센터 문서를 참조하세요.

### 지속적인 오류 30003 문제

Hyperclass 지원팀이 메시지 배달에 무엇이 잘못되었는지 조사하는 데 도움을 드릴 수 있습니다. 최근 24시간 내에 SMS(문자) 로그에서 오류 30003으로 표시된 메시지 예시를 3개 이상 수집하여 지원 요청을 열어주세요.
