# 텍스트 및 날짜 필드에 커스텀 값 연결: 문서 및 계약

## 새로운 기능

- **텍스트 및 날짜 필드 연결:** 문서 및 계약에서 텍스트와 날짜 필드를 연락처의 커스텀 값(Custom Values)에 연결하는 기능을 추가했어요.
- **유연한 연결 방식:** 사용자는 이제 문서의 필드를 연락처 또는 문서 자체의 커스텀 값에 연결할 수 있어요.
- **자동 업데이트:** 문서가 서명되면 연결된 필드가 연락처나 문서 레코드에 자동으로 업데이트돼요.
- **CSV 다운로드:** 대시보드에서 모든 입력된 필드의 CSV 파일을 다운로드하여 더 쉽게 추적하고 데이터를 관리할 수 있어요.

## 사용 방법

- **필드를 커스텀 값에 연결하기:**
Documents and Contracts(문서 및 계약) 섹션에서 편집기를 열고 새 문서를 만든 후, 필요한 곳에 텍스트 또는 날짜 필드를 끌어다 놓으세요.

- **수신자 추가 및 필드 연결:**
수신자를 추가하고, 텍스트 및 날짜 필드 상단에 있는 연결 버튼을 선택하세요.
- 필요한 경우 하나의 필드를 여러 개의 커스텀 값에 연결할 수 있어요.

- **서명 시 자동 업데이트:**
수신자가 문서에 서명하면 연결된 필드가 해당 레코드(연락처 또는 문서)에 자동으로 업데이트돼요.

- **입력된 필드 데이터 다운로드:**
문서에서 모든 입력된 필드를 확인하려면 대시보드로 이동해서 관련 데이터를 CSV 파일로 다운로드하세요.

## 왜 만들었을까요?

- **수동 업데이트 감소:** 텍스트와 날짜 필드를 커스텀 값에 연결하면 문서 서명 후 수동으로 업데이트할 필요가 줄어들고, 연락처에 데이터가 자동으로 동기화돼요.
- **편리한 데이터 관리:** CSV 다운로드 옵션으로 입력된 필드 데이터를 편리하게 관리하고 분석할 수 있어서 문서 프로세스가 간소화돼요.
