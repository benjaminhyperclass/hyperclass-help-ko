# 분할 컬럼 지원: 문서 및 계약

## 새로운 기능

- 분할 컬럼 기능이 이제 문서(Documents)와 계약(Contracts)에서 사용할 수 있습니다.
- 사용자는 요소를 나란히 드래그 앤 드롭하여 자동으로 컬럼을 생성할 수 있어, 더 나은 구성과 레이아웃이 가능합니다.
- 이 기능은 정보 표시를 개선하여 명확하고 전문적인 문서를 더 쉽게 만들 수 있게 해줍니다.

## 사용 방법

- 문서 편집기에서 요소(텍스트, 이미지, 표 등)를 다른 요소 옆으로 드래그하면 분할 컬럼 레이아웃이 생성됩니다.

## 개발 배경

- 문서 디자인에서 사용자에게 더 큰 유연성을 제공하기 위해 이 기능을 개발했습니다.
- 향상된 서식 옵션에 대한 요구를 해결하여, 사용자가 더 명확하고 체계적인 제안서와 계약서를 작성할 수 있도록 도와줍니다.
