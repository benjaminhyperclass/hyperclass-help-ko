# 문서 및 계약서 활동 추적 방법

## **새로운 기능**

- 문서를 전송한 후 발송부터 편집, 열람, 서명까지 전체 이력을 확인할 수 있습니다. 옵션 메뉴에서 '이력 보기'를 클릭하기만 하면 됩니다.
- 연락처 아이콘에 마우스를 올리면 최종 열람 시간과 사용자가 해야 할 작업을 확인할 수 있습니다.

## **사용 방법**

- 문서 및 계약서(Documents and Contracts) 메뉴를 여세요
- 대시보드에서 상태를 확인하고 싶은 문서의 항목으로 스크롤하세요
- 고객 아이콘에 마우스를 올려보세요
- 문서에 관련된 모든 사용자의 최종 활동 시간, 태그, 상태를 확인할 수 있습니다
- 상세한 로그를 보려면 점 3개 아이콘을 클릭하고 '이력 보기'를 선택하세요
- 우측 패널에서 해당 문서에서 발생한 모든 활동을 확인할 수 있습니다

## **개발 배경**

- 사용자가 매번 문서를 열어보지 않고도 이동 중에 문서의 실제 상태를 파악할 수 있도록 도와줍니다.
- 즉시 확인 가능한 정보를 통해 문서를 완료 단계로 이끌 수 있도록 지원합니다.

## **참고사항**
호버 창의 최종 활동 시간은 지금부터 전송되는 문서에만 표시됩니다. 이전 문서들의 경우 시간 정보를 확인할 수 없습니다.
