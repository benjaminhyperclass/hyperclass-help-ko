# 지갑 자동 충전

## 자동 충전이란 무엇이며 어떻게 작동하나요?

![자동 충전 설정 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155045443507/original/tb8s53SRWznc5Y-FT4QpSquowdP981WGRQ.jpeg?1745319934)

- 첫 번째 드롭다운에 표시된 금액은 두 번째 드롭다운에 설정된 금액 이하로 잔액이 떨어질 때 결제 수단에서 차감될 금액입니다.
- 두 번째 드롭다운은 임계값 드롭다운입니다. 여기에 설정한 금액이 임계값으로 작동합니다. 지갑 잔액이 이 금액 아래로 떨어지는 즉시 첫 번째 드롭다운에 설정된 금액으로 자동 충전이 실행됩니다.

## 자주 묻는 질문

### 자동 충전이란 무엇이며 어떻게 작동하나요?

자동 충전은 지갑 잔액이 설정된 임계값 이하로 떨어지면 자동으로 잔액을 보충하는 기능입니다. 지갑 잔액이 설정한 임계값(두 번째 드롭다운) 이하로 떨어지면, 선택한 충전 금액(첫 번째 드롭다운)이 결제 수단에서 차감되어 지갑을 다시 채웁니다.
