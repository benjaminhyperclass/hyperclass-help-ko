## 자동 충전이란 무엇이며 어떻게 작동하나요?

![자동 충전 설정 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155045443507/original/tb8s53SRWznc5Y-FT4QpSquowdP981WGRQ.jpeg?1745319934)

- 드롭다운 1에서 보이는 금액은 잔액이 드롭다운 2에서 설정한 금액 아래로 떨어질 때 결제 수단에서 차감될 금액입니다.
- 드롭다운 2는 임계값 드롭다운입니다. 여기서 설정한 금액이 임계값으로 작동합니다. 지갑 잔액이 이 임계값 아래로 떨어지면 드롭다운 1에서 설정한 금액으로 자동 충전이 실행됩니다.
