# 음성 AI 대시보드에서 테스트 통화 로그 확인

## 개요

새로운 통화 유형 필터를 통해 음성 AI 대시보드 로그에서 테스트 통화와 실제 고객 응대를 함께 확인할 수 있습니다.

## **새로운 기능**

- **통화 유형 필터**: 단일 선택 드롭다운으로 **테스트 통화**, **실제 통화**, 또는 **모두**를 선택할 수 있습니다.

- **전체 적용**: **모든 상담원** 및 **개별 상담원** 로그 모두에서 필터 사용 가능합니다.

- **완전한 상세 정보**: 테스트 통화 항목에는 녹음과 대화 내용이 포함되어 있습니다(분석 지표에는 제외됨).

## **사용 방법**

- **음성 AI(Voice AI) > 대시보드 로그(Dashboard Logs)**로 이동합니다.

- 로그 위의 **통화 유형(Call Type)** 드롭다운을 클릭합니다.

- **테스트 통화**, **실제 통화**, 또는 **모두** 중 선택합니다.

- 필터를 해제하면 기본 통합 보기로 돌아갑니다.

## **중요 사항**

- 테스트 통화는 대시보드 분석 데이터나 성과 지표에 **포함되지 않습니다**.
