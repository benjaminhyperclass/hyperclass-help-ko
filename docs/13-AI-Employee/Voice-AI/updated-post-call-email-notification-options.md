# 통화 후 이메일 알림 옵션 업데이트

## 개요

통화 후 이메일 알림 기능이 개선되어 누가 알림을 받을지 완전히 제어할 수 있게 되었습니다. 이전에는 알림이 관리자로 제한되었고 추가 수신자를 한 명만 지정할 수 있었습니다. 이제 더 이상 관리자나 단일 수신자로 제한되지 않습니다.

## 중요한 이유

- **개인화**: 꼭 알아야 할 사람에게만 알림을 보냅니다.
- **노이즈 감소**: 원치 않는 이메일로 모든 관리자를 귀찮게 하지 않습니다.
- **사용성 향상**: 여러 수신자를 쉽게 설정할 수 있습니다.

## 새로운 기능

- **유연한 선택**: 한 가지 대신 하나 이상의 수신자 유형을 선택할 수 있습니다.
- **관리자 선택 해제**: 알림이 더 이상 모든 관리자에게 기본적으로 전송되지 않습니다.
- **수신자 카테고리**:
  - 모든 관리자
  - 모든 사용자
  - 연락처 담당 사용자
  - 특정 사용자
  - 커스텀 이메일
- **UI 업데이트**: Agent Goals(에이전트 목표) 탭에 새로운 다중 선택 컨트롤이 추가되었습니다.

## 작동 방식

- 음성 AI(Voice AI)에서 **Create or Update Agent(에이전트 생성 또는 업데이트)** 플로우로 이동합니다.
- **Agent Goals(에이전트 목표)** 탭으로 이동합니다.
- **Receive Email Notification Post Call Completion(통화 완료 후 이메일 알림 받기)**를 활성화합니다.
- 하나 이상의 수신자 유형을 선택합니다(모든 관리자, 모든 사용자, 연락처 담당 사용자, 특정 사용자 또는 커스텀 이메일).
- **Specific Users(특정 사용자)**를 선택한 경우 드롭다운 목록에서 사용자를 선택합니다.
- **Custom Email(커스텀 이메일)**을 선택한 경우 원하는 이메일 주소를 입력합니다.
- **Save(저장)**를 클릭하여 알림 설정을 적용합니다.
