# Ask AI 어시스턴트 소개

Hyperclass 내에 새로운 AI 기반 어시스턴트인 Ask AI를 출시하게 되어 기쁩니다. 브랜드 톤에 맞는 콘텐츠 작성이 필요하거나, 새로운 커뮤니티를 만들고 싶거나, 빠른 답변이 필요할 때 Ask AI가 도와드립니다. 더 이상 여러 도구나 탭을 오갈 필요가 없어요.

Ask AI는 Hyperclass 내장 어시스턴트 역할을 합니다. 몇 가지 핵심 기능부터 시작해서 시간이 지나면서 계속해서 더 많은 기능을 추가할 예정입니다.

**중요**: 완전히 새로운 Ask AI 경험이 현재 Labs(연구실)에서 제공됩니다. 보이지 않는다면 관리자에게 Agency Settings(에이전시 설정) > Labs(연구실)에서 활성화를 요청하세요. 자세한 내용은 [Labs 기능 - 전체 개요]([labs-features-complete-overview](labs-features-complete-overview.md))를 참고하세요.

---

## Ask AI란 무엇인가요?

Ask AI는 Hyperclass 사용자를 위한 내장 가상 어시스턴트입니다. 스마트 AI 기능과 브랜드 컨텍스트를 결합하여 단일 인터페이스에서 직접 Hyperclass 내 작업을 수행할 수 있도록 도와줍니다. 상단 네비게이션 바에서 Ask AI 아이콘을 찾을 수 있어요. Ask AI는 방해받지 않는 전체 화면 작업공간도 지원해서 화면 전환 없이 계획하고, 생성하고, 실행할 수 있습니다.

참고: 개인화된/브랜드화된 콘텐츠를 생성하려면 Marketing(마케팅) > Branded Content(브랜드 콘텐츠) > Brand Voice(브랜드 보이스)에서 브랜드 보이스를 설정했는지 확인하세요.

---

## 누가 Ask AI를 사용할 수 있나요?

현재 Ask AI는 모든 에이전시 관리자와 에이전시 사용자가 사용할 수 있으며, 하위 계정에서는 97달러 Unlimited AI Employee Plan(무제한 AI 직원 플랜)을 사용하는 하위 계정 관리자와 하위 계정 사용자만 이용할 수 있습니다. 상단 네비게이션 바에서 찾아 사용할 수 있으며, 아이콘을 클릭하고 프롬프트를 입력하면 됩니다.

---

## 봇이 할 수 있는 일은 무엇인가요?

Ask AI가 현재 지원하는 기능들을 자세히 살펴보겠습니다:

- **브랜드 콘텐츠 생성**: 더 이상 일반적인 AI 글쓰기는 그만. 봇이 브랜드 보이스 설정을 불러와서 여러분의 목소리처럼 들리는 콘텐츠를 생성합니다. 이메일 뉴스레터부터 소셜 미디어 캡션까지, 여러분의 톤과 브랜드 보이스 설정에 맞춘 고품질 콘텐츠를 만들어줍니다.

- **커뮤니티 생성**: 새 커뮤니티 설정에 더 이상 메뉴를 뒤질 필요가 없어요. 어떤 종류의 커뮤니티를 만들고 싶은지 봇에게 말하기만 하면 채팅에서 단계별로 안내해줍니다.

- **질문 답변 또는 아이디어 제안**: 캠페인 영감이 필요하시나요? 블로그 CTA에 대한 아이디어를 원하시나요? Ask AI에게 직접 물어보세요.

- **Hyperclass 내 액션 실행 (MCP 기반)**: Ask AI는 권한에 따라 채팅에서 레코드 생성 또는 업데이트, 예약 일정 변경, 데이터 가져오기, 기회 일괄 업데이트 등 앱 내 지원 액션을 실행할 수 있습니다. 예시와 가이드라인은 [Ask AI 내 MCP 서버 사용 방법]([how-to-use-the-mcp-server-within-ask-ai](how-to-use-the-mcp-server-within-ask-ai.md))을 참고하세요.

## 개선된 목록 기반 요소 업데이트

Ask AI가 이제 웹사이트 및 퍼널 빌더 내 목록 기반 요소에 대한 항목별 업데이트를 지원합니다.

이전에는 목록 기반 구성 요소(FAQ나 가격표 등)에 대한 작은 AI 편집이 단 하나의 항목만 변경하면 되는 경우에도 전체 목록이 다시 작성되는 경우가 있었습니다.

| 카테고리 | 항목 |
|---------|------|
| **새로 지원** | Testimonials(추천사); Pricing Tables(가격표); Logo Showcase(로고 쇼케이스); Number Counters(숫자 카운터) |
| **기존 요소 개선** | FAQs; Image Sliders(이미지 슬라이더); Photo & Video Gallery(사진 및 비디오 갤러리); Navigation Menus(네비게이션 메뉴) |
| **주요 장점** | 선택된 목록 항목만 업데이트되어 나머지 콘텐츠가 보존되고 덮어쓰기 위험이 줄어듭니다. |

- 봇이 안내하거나 요청한 내용에 즉시 응답해줍니다.

![AI 응답 예시](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155067499887/original/W7zikhPngC6zamaSUMS77JpKXzzT8wzvaw.gif?1774271863)

## 좌측 네비게이션에서 Ask AI 열기

Ask AI는 전체 화면 작업공간에서 열리므로 구석에 숨어있는 챗봇이 아닙니다. 실제 작업에 집중하고 Hyperclass를 떠나지 않고 대용량 작업을 완료하는 데 사용하세요. **에이전시** 레벨과 **하위 계정** 레벨 모두에서 좌측 네비게이션을 통해 Ask AI에 접근할 수 있습니다.

1. 계정에서 **좌측 네비게이션 패널**을 확인하세요.

2. **Ask AI**를 클릭하세요 (전용 메뉴 항목으로 나타납니다).

3. Ask AI 작업공간에서 할 수 있는 일:
   - New chat을 클릭해서 새로운 대화 시작
   - Search를 사용해서 이전 대화 찾기 (사용 가능한 경우)

## 예시 1: 브랜드 콘텐츠 생성

참고: 개인화된 콘텐츠를 위해서는 반드시 브랜드 보이스를 설정해야 합니다.

## 예시 2: 음성 AI 에이전트 생성

Ask AI를 사용해서 음성 AI 에이전트를 설정하고 Call Transfer Action(통화 전환 액션)을 사용해 긴급 통화를 라우팅하세요.

- Ask AI를 열고 다음과 같은 프롬프트를 입력하세요: "인바운드 통화에 응답하고, 발신자의 이름과 통화 목적을 수집하며, 긴급 요청을 라우팅하는 음성 AI 에이전트를 만들어주세요."

![음성 AI 에이전트 요청](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155067494576/original/vc2T3l1OlLUusWmt_2PiE8KcBHZN9pXvcg.png?1774270008)

- 어시스턴트가 묻는 설정 질문에 답하세요. 어시스턴트는 무언가를 생성하기 전에 회사명, 긴급 통화를 전환할 전화번호, "긴급"의 기준(라우팅 시점을 알기 위해), 에이전트명 등의 간단한 세부사항을 수집합니다. 프롬프트에 따라 세부사항을 입력하세요.

![설정 질문](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155067494629/original/cl8C5mKxNjEZPLEAPRvUEzP88J0V1m-WtQ.png?1774270062)

- 제안서를 검토한 후 생성을 확인하세요. 세부사항을 제공하면 어시스턴트가 검토용 제안서를 준비합니다. 확인하면 음성 AI 에이전트를 생성합니다.

![제안서 검토](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155067494775/original/niyQhZlIF-j0-XaRpNIclyZ842_uM7C_lA.png?1774270103)

## 자주 묻는 질문

**Q: Ask AI 목록 기반 업데이트에서 무엇이 변했나요?**

Ask AI가 이제 전체 목록을 다시 작성하는 대신 항목 레벨에서 목록 기반 요소를 업데이트합니다. 수정되는 특정 항목만 업데이트됩니다.

**Q: 어떤 요소가 항목별 업데이트를 지원하나요?**

새로 지원되는 요소는 Testimonials(추천사), Pricing Tables(가격표), Logo Showcase(로고 쇼케이스), Number Counters(숫자 카운터)입니다. FAQ, Image Sliders(이미지 슬라이더), Photo & Video Gallery(사진 및 비디오 갤러리), Navigation Menus(네비게이션 메뉴) 같은 기존 요소들도 개선된 업데이트 동작의 혜택을 받습니다.

**Q: 작은 변경을 할 때 Ask AI가 전체 FAQ나 가격표를 덮어쓸까요?**

아니요. 이번 업데이트로 Ask AI는 요청된 특정 목록 항목만 수정하여 의도하지 않은 덮어쓰기 위험을 줄입니다.

**Q: 수동 편집 방식이 바뀌나요?**

아니요. 수동 편집 동작은 동일하게 유지됩니다. 이 업데이트는 Ask AI가 업데이트를 처리하는 방식에만 영향을 줍니다.
