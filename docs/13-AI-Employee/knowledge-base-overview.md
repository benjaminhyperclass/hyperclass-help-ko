# 지식 베이스 개요

지식 베이스(Knowledge Base) UI가 새롭게 개선되어 하이레벨 전반에서 더 명확하고 일관된 탐색 경험을 제공합니다. 이번 업데이트에서는 **요약 보기**, **탭형 데이터 소스**, 그리고 **제품 전반에 걸쳐 사용되는 통합된 연합형 지식 베이스 컴포넌트**를 도입했습니다.

## **개요**

새롭게 개선된 지식 베이스 UI는 관리자와 편집자가 어떤 콘텐츠가 있는지, 어디에서 온 것인지, 어떻게 관리되는지 빠르게 파악할 수 있도록 도와줍니다. 새로운 레이아웃은 더 나은 가시성, 간편한 탐색, 그리고 지식 베이스가 사용되는 모든 곳에서 일관된 경험을 제공하는 데 중점을 두었습니다.

**메뉴 경로:**

**AI 직원(AI Agents) → 지식 베이스(Knowledge Base)**

---

## **지식 베이스 메인 페이지**

지식 베이스 메인 페이지에서는 기존의 모든 지식 베이스를 확인하고, 새로운 지식 베이스를 생성하며, 지식 베이스를 열어서 소스를 관리할 수 있습니다.

## **지식 베이스 UI의 새로운 기능**

### **요약 보기**

이제 각 지식 베이스는 요약 보기로 열립니다. 이 보기는 연결된 소스와 전체 구조를 포함하여 지식 베이스의 전반적인 현황을 한눈에 보여주며, 현재 상태를 빠르게 파악할 수 있도록 도와줍니다.

---

### **탭형 데이터 소스**

이제 모든 데이터 소스가 개별 탭으로 구성됩니다. 각 탭은 특정 소스 유형을 나타내므로 페이지를 전환하지 않고도 콘텐츠를 쉽게 관리하고 검토할 수 있습니다. 추가 데이터 소스가 출시되면 새로운 탭으로 표시됩니다.

---

### **통합된 연합형 컴포넌트**

지식 베이스는 이제 여러 제품에서 공유되는 연합형 컴포넌트를 사용합니다. 이를 통해 하이레벨 내에서 지식 베이스가 나타나는 모든 곳에서 일관된 레이아웃, 탐색, 관리 경험을 보장합니다.

---

## **지식 베이스 소스 (탭 레이아웃)**

지식 베이스를 열면 다음과 같은 지원되는 소스 유형이 탭으로 나타납니다:

- 웹 크롤러(Web Crawler)
- FAQ
- 웹 검색(Web Search)
- 테이블(Tables)
- 리치 텍스트(Rich Text)
- 파일 업로드(File Upload)

각 탭에는 연결된 항목의 개수가 표시되며, 소스를 추가하거나 관리하는 작업을 수행할 수 있습니다.

## 이 인터페이스 대상 사용자

새롭게 개선된 지식 베이스 UI는 다음 사용자를 위해 설계되었습니다:

- 지식 베이스와 데이터 소스를 관리하는 **관리자**
- 지식 콘텐츠를 수정하고 유지 관리하는 **편집자**

보기 전용 사용자는 콘텐츠를 확인할 수 있지만 관리 컨트롤은 볼 수 없습니다.

---

## **변경되지 않은 부분**

이번 업데이트는 다음 사항에 영향을 주지 않습니다:

- 콘텐츠 작성 또는 편집 방법
- 권한 또는 접근 수준
- 인덱싱 로직 또는 AI 동작
- 기존 지식 베이스 또는 소스

---

## **출시 및 이용 가능성**

새롭게 개선된 지식 베이스 UI는 단계적으로 출시되고 있습니다. 아직 요약 보기나 탭형 데이터 소스가 보이지 않는다면, 업데이트가 귀하의 계정에 적용되는 대로 자동으로 표시됩니다. 별도의 조치는 필요하지 않습니다.

---

## **자주 묻는 질문**

**Q: 요약 보기가 개별 소스 세부 정보를 대체하나요?**

아니요. 요약 보기는 전반적인 개요를 제공하며, 각 데이터 소스 탭에는 상세 정보와 관리 옵션이 포함되어 있습니다.

**Q: 같은 유형의 소스를 여러 개 추가할 수 있나요?**

네. 같은 데이터 소스 탭 아래에 여러 소스를 추가할 수 있습니다.

**Q: 새로운 데이터 소스가 자동으로 나타나나요?**

네. 새로운 데이터 소스가 출시되면 전체 UI를 변경하지 않고도 추가 탭으로 자동 표시됩니다.

**Q: 연합형 지식 베이스 컴포넌트를 모든 곳에서 사용할 수 있나요?**

연합형 컴포넌트는 제품 전반에 걸쳐 단계적으로 출시되고 있습니다. 사용 가능한 곳에서는 UI와 컨트롤이 동일합니다.
