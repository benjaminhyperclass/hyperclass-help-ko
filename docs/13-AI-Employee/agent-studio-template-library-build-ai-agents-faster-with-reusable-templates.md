# 에이전트 스튜디오 템플릿 라이브러리: 재사용 가능한 템플릿으로 AI 에이전트를 더 빠르게 구축하세요

Hyperclass의 에이전트 스튜디오 템플릿 라이브러리를 사용하여 몇 분 만에 프로덕션급 AI 에이전트를 배포하세요. 에이전시가 만든 템플릿이나 공식 스타터 템플릿 중에서 선택한 후, 각 클라이언트에 맞게 변수를 커스터마이징하면 됩니다. 템플릿을 사용하면 온보딩 속도를 높이고, 모범 사례를 표준화하며, 모든 하위 계정에서 일관된 결과를 확장할 수 있습니다.

# **에이전트 스튜디오 템플릿 라이브러리란?**

템플릿 라이브러리는 에이전트 스튜디오 내에서 재사용 가능한 **에이전트 템플릿**을 저장하는 작업 공간입니다. 각 템플릿은 에이전트의 로직(프롬프트, 도구, 변수, 설정)을 스냅샷으로 저장한 것으로, 한 번의 클릭으로 설치할 수 있습니다. 일반적인 흐름을 처음부터 다시 구축하는 대신, 팀은 템플릿을 선택하여 개인화한 후 배포할 수 있습니다.

AI 에이전트 스튜디오 사용법에 대해 더 자세히 알아보려면 [이 글]([how-to-use-the-ai-agent-studio-in-highlevel](how-to-use-the-ai-agent-studio-in-highlevel.md))을 참고하세요.

---

## **에이전트 스튜디오 템플릿 라이브러리의 주요 장점**

템플릿은 에이전시와 하위 계정에 일관성을 만들고 배포 속도를 가속화합니다.

- **빠른 가치 실현:** 한 번의 클릭 설치로 초기 구축 시간을 몇 시간 또는 며칠 단축할 수 있습니다.

- **모범 사례 디자인:** 검증된 에이전트 패턴과 가이드라인에서 시작하세요.

- **일관된 품질:** 명명 규칙, 프롬프트 위생, 라이프사이클 제어를 강제할 수 있습니다.

- **확장 가능한 배포:** 템플릿을 업데이트하고 여러 계정에 개선사항을 배포하세요.

- **온보딩 지원:** 새 팀원들이 작동하는 에이전트를 분석하며 학습할 수 있습니다.

## 사전 준비: 템플릿 라이브러리 활성화

에이전시 레벨 토글을 활성화하고 권한을 확인하면 팀이 템플릿 라이브러리에 접근할 수 있고, 자격이 되는 경우 배포용 새 템플릿을 게시할 수 있습니다.

**요구사항**

- **에이전시 관리자** 역할

- 템플릿을 **업로드/게시**하려면 **에이전시 프로 플랜($497/월)**이 필요합니다. (**$97/$297** 플랜의 관리자는 일반적으로 공유 템플릿을 **탐색/설치**할 수 있지만 게시는 불가능합니다.)

**단계**

**에이전시 보기로 전환**

좌측 메뉴에서 하위 계정 선택 드롭다운을 클릭한 후, **"Switch to agency View(에이전시 보기로 전환)"**를 클릭하세요.

![에이전시 보기 전환](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063766239/original/B0gACQpP6dBy_8FcKYh6nSIZT7q2DS0V5w.png?1769688285)

**설정(Settings) 열기**

회사 설정에 접근하기 위해 에이전시 설정으로 이동하세요.

![설정 열기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063766404/original/EGUdr8WGgRDl2m6gh-cToAXDdq0ezIFYqw.png?1769688344)

**Company(회사) 탭 클릭**

**Company(회사)** 메뉴를 클릭하세요. 스크롤을 내려서 **Template Library for Clients(클라이언트용 템플릿 라이브러리)**를 **켜기**로 전환하세요.

![템플릿 라이브러리 활성화](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063766531/original/l4bzP3kDoMKDB9sHvEos8QcpIdZhHG2mDQ.png?1769688405)

## 템플릿 라이브러리 접근하기

템플릿을 빠르게 찾으면 처음부터 올바른 위치에서 구축할 수 있습니다.

- AI 에이전트(AI Agents) → 에이전트 스튜디오(Agent Studio)로 이동하세요.

- 에이전트 생성(Create Agent) → 템플릿 탐색(Browse templates)을 클릭하여 템플릿 라이브러리를 여세요.

## **템플릿 라이브러리 보기 옵션 살펴보기 (정렬, 필터, 카테고리, 태그)**

강력한 탐색 도구를 통해 각 시나리오에 적합한 템플릿을 쉽게 찾을 수 있습니다.

- **필터:** **필터(Filters)**를 열어 소유권별로 정제할 수 있습니다 (전체 템플릿, 내 템플릿, 공유된 템플릿, 즐겨찾기).

- **카테고리 탐색:** 카테고리별로 결과를 좁힐 수 있습니다 (예: Learning, Test, Industry). 카테고리는 에이전시 관리자가 관리하는 **숨김** 옵션도 지원합니다.

- **태그:** 태그를 사용하여 틈새 사용 사례를 빠르게 찾을 수 있습니다.

- **검색:** 이름이나 키워드로 템플릿을 찾을 수 있습니다.

- **정렬 및 레이아웃:** **최신순** 또는 유사한 옵션으로 정렬하고, 격자/목록 보기를 전환할 수 있습니다.

![템플릿 라이브러리 보기 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063768708/original/f_-4dv2pJn80c1bYojCpnNIe751M7gtkMw.png?1769689333)

## **전체 템플릿 보기: 미리보기 vs. 템플릿 선택**

템플릿에 마우스를 올리면 다음 옵션들이 표시됩니다:

- **미리보기(Preview):** 설치하기 전에 설명, 이미지, 메모를 검토할 수 있는 상세 정보를 엽니다. 템플릿이 확실하다면 우측 상단의 **"Choose Template(템플릿 선택)"** 버튼을 클릭하여 템플릿을 사용할 수 있습니다.

- **템플릿 선택(Choose Template):** 이 버튼은 즉시 편집할 수 있는 **새 에이전트 초안**으로 에이전트 스튜디오에 사본을 설치하는 프로세스를 시작합니다.

![템플릿 미리보기 및 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063768810/original/blEO1cD7FpZnLZMxB4UH_tNGlkdaU7tNRA.png?1769689400)

## 설치한 에이전트 편집하기 (에이전트 스튜디오 빌더)

템플릿을 선택한 후에는 자신만의 에이전트 사본으로 작업하게 됩니다. 수정하는 내용은 원본 템플릿을 편집하지 **않으며**, 작업 공간의 **에이전트**를 버전 관리하는 것입니다.

- 에이전트 캔버스를 열어 노드, 프롬프트, 도구, 변수, 연결을 조정하세요.

- 자주 저장하고, 테스트가 완료되면 **프로덕션**으로 승격하세요.

## **기존 에이전트를 템플릿 라이브러리에 업로드하기 (에이전시)**

**프로 플랜($497/월)**의 에이전시 관리자는 성공적인 에이전트를 모든 하위 계정과 공유할 수 있습니다.

- **에이전트 스튜디오**에서 소스 에이전트 행을 찾으세요.

- 점 세 개 메뉴를 클릭하고 템플릿 라이브러리에 업로드(Upload to Template Library) 옵션을 선택하세요.

- 다음 단계에서 템플릿 세부사항(이름, 카테고리, 커버/미리보기 이미지, 사전 요구사항, 태그, 소개, 설명)을 완성하라는 메시지가 표시되며, 그런 다음 다른 사용자가 설치할 수 있도록 게시할 수 있습니다.

![템플릿 업로드](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063770744/original/fDttkSuBgbMsymH_HumPs5qlWeyYY5i-jg.png?1769690422)

## **자주 묻는 질문**

**Q: 미리보기와 템플릿 선택의 차이점은 무엇인가요?**

**미리보기**는 설치하기 전에 세부사항을 읽을 수 있게 해줍니다. **템플릿 선택**은 에이전트 스튜디오에 초안 에이전트로 작업 사본을 설치합니다.

**Q: 내 에이전트의 변경사항이 원본 템플릿에 영향을 주나요?**

아니요. 설치 후 편집은 에이전트 사본에만 적용됩니다. 원본 템플릿은 변경되지 않습니다.

**Q: 템플릿 선택 후 에이전트는 어디서 찾을 수 있나요?**

**AI 에이전트(AI Agents) → 에이전트 스튜디오(Agent Studio)**를 여세요. 새 에이전트가 목록 보기에 나타납니다.

**Q: 하위 계정이 보는 템플릿을 제어할 수 있나요?**

예. 에이전시 관리자는 카테고리(**숨김** 포함)와 라이브러리 공개 설정을 사용하여 가시성을 관리할 수 있습니다.

**Q: 내 고유 템플릿을 기여하려면 어떻게 하나요?**

**프로 플랜($497/월)**의 에이전시 관리자는 템플릿을 게시할 수 있습니다. **에이전트 스튜디오**에서 검증된 에이전트의 **⋯ → 템플릿 라이브러리에 업로드** 옵션을 사용하여 세부사항을 완성하고 게시하세요.

**Q: 템플릿 설치 시 AI 크레딧이 사용되나요?**

아니요. 크레딧은 에이전트가 실행될 때 사용되며, 템플릿 설치 시에는 사용되지 않습니다.
