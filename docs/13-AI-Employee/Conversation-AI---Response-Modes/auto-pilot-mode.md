# 자동 파일럿 모드

대화 AI(Conversation AI) 봇이 기본 설정(Preference)에서 선택한 "지원 채널"의 모든 수신 메시지에 자동으로 응답합니다.

---

자동 파일럿 모드에서 봇의 응답과 동작을 더 세밀하게 제어하려면 아래 설정을 조정하세요.

## 1. 응답 전 대기 시간

필드 유형 - 최소 1초, 최대 5분까지 설정 가능한 숫자 입력 필드

봇이 연락처에게 응답하기 전에 이 시간만큼 대기합니다. 이 대기 시간을 통해 연락처가 짧은 시간 내에 여러 메시지를 보낼 경우, 봇이 모든 메시지를 수집하여 한 번에 종합적으로 응답할 수 있습니다.

## 2. 봇이 연락처에게 보낼 수 있는 최대 메시지 수

필드 유형 - 최대 25까지 설정 가능한 숫자 입력 필드

특정 채널에서 한 대화 내에서 이 한도에 도달하면 봇이 해당 연락처에 대해 휴면 상태로 전환됩니다. 봇을 다시 활성화하려면 대화를 "읽음"으로 표시하세요.

이는 대화 AI에 추가된 안전장치로, 연락처별 대화 채널별 응답 수에 상한선을 설정하여 봇을 휴면 상태로 보내는 기능입니다.

## 3. 수동 또는 워크플로우로 메시지를 보낼 때 봇을 휴면 상태로 전환

필드 유형 - 토글 스위치이며, 하위 계정에서 연락처에게 수동/워크플로우 메시지를 보낼 때 봇이 휴면 상태로 전환되는 시간을 제어

Hyperclass 사용자나 워크플로우가 연락처에게 메시지를 보내면 봇이 설정한 시간만큼 휴면 상태가 됩니다. 휴면 시간이 지나면 봇이 해당 연락처에 대해 자동으로 다시 활성화됩니다.
