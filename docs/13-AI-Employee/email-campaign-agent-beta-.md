# 이메일 캠페인 에이전트 (베타)

Hyperclass의 이메일 캠페인 에이전트(Email Campaign Agent, 베타)로 몇 분 만에 세련되고 데이터 기반의 이메일을 발송해보세요. 이 AI 가이드 빌더는 카피라이팅, 제목 A/B 테스트, 오디언스 타겟팅, 발송 도메인 설정을 간소화하여 더 스마트한 캠페인을 더 빠르게 보낼 수 있게 해줍니다. Ask AI로 브랜드에 맞는 콘텐츠를 생성하고, A/B 테스트를 실행하며, 세그먼트나 스마트 리스트를 타겟팅하고, 전용 발송 도메인을 인증하여 더 나은 전달률을 확보하세요.

## 이메일 캠페인 에이전트란?

이메일 캠페인 에이전트는 **Ask AI 기반의 워크플로우**로, Ask AI에 "이메일 캠페인 만들어줘" 또는 "내 상품을 홍보하는 이메일 보내줘" 같은 문구로 요청하면 실행됩니다. Ask AI가 단계별 가이드 빌더를 시작하여 캠페인 빌더(Campaign Builder) 안에서 브랜드에 맞는 카피, 제목 테스트, 오디언스 선택, 발송 도메인 확인을 모두 조합해줍니다.

---

## 이메일 캠페인 에이전트의 주요 장점

팀들이 이 도구를 사용하는 이유와 결과를 가속화하는 방법에 대한 핵심 개요입니다. 이러한 장점을 통해 가치를 일치시키고, 기대치를 설정하며, 관계자들이 전달률과 참여도 개선 효과를 이해할 수 있도록 도와줍니다.

- **브랜드 일관성**: Ask AI와 브랜드 보이스로 완성된 이메일을 작성하여 빈 페이지 시간을 줄이면서 일관성을 유지합니다.

- **빠른 실험**: 여러 제목을 생성하고 내장된 A/B 테스트를 실행하여 열람률과 클릭률을 개선합니다.

- **정확한 타겟팅**: 동적 **세그먼트**, **스마트 리스트**, 태그 또는 미리 만들어진 세그먼트에서 선택하여 적합한 오디언스에게 도달합니다.

- **전달률 보호**: **전용 발송 도메인**에 대한 가이드 설정과 자동 구성 또는 수동 DNS, 그리고 DMARC 정렬 대체를 위한 **기본 헤더** 제공

- **명확한 성과 추적**: **이메일 마케팅 통합 대시보드**와 관련 통계 페이지에서 결과를 확인합니다.

## Ask AI로 브랜드에 맞는 이메일 카피 작성

- 이 기능은 콘텐츠 AI(Content AI)와 브랜드 보이스를 사용하여 짧은 브리핑을 완성된 이메일 카피로 변환합니다. 톤과 의도를 세밀하게 조정하거나, 재생성하거나, 직접 편집할 수 있어서 AI가 속도를 높여주지만 자유도를 제한하지는 않습니다.

- 캠페인 초안에서 Ask AI를 실행하여 프롬프트와 톤에 기반한 본문 카피를 생성합니다.

- 브랜드 보드(Brand Boards)에서 **브랜드 보이스**를 연결하여 이메일, 퍼널, 소셜 전반에 걸쳐 출력의 일관성을 유지합니다.

## 제목 A/B 테스트

스플릿 테스트는 제목 성과에서 추측을 제거합니다. 변형 수, 테스트 기간, 승리 지표를 정의하면 자동으로 승리한 제목이 나머지에게 발송됩니다.

- **최대 6개 변형** 테스트 가능; **변형당 최소 10명의 연락처**와 **총 40명의 연락처** 필요

- **고유 열람률** 또는 고유 클릭률로 승자를 선택합니다.

- A/B 테스트는 **지금 발송**과 **예약** 발송만 지원하며, **일괄 발송**이나 **RSS 예약**은 지원하지 않습니다.

## 오디언스 타겟팅 옵션

적절한 오디언스를 선택하는 것은 메시지만큼 중요합니다. 이메일 캠페인 에이전트는 여러 타겟팅 방법을 지원하여 속도와 정확성의 균형을 맞출 수 있습니다.

• **세그먼트(캠페인 빌더)** – 참여도, 태그, 필드에서 구축된 실시간 로직 기반 그룹으로, 동적으로 평가되며 캠페인 간에 재사용 가능합니다.

• **미리 만들어진 세그먼트** – "최근 7/30/60일 동안 참여함" 또는 "최근 5개 캠페인과 상호작용함" 같은 원클릭 선택. 생성/예약 시점에 스냅샷을 만듭니다.

• **스마트 리스트(연락처 → 스마트 리스트)** – **발송 시점**에 업데이트되는 동적 CRM 뷰로, 새로운 일치 항목이 포함되고 일치하지 않는 항목은 제외됩니다.

• **태그** – 기존 태그를 사용하거나 발송/예약 페이지에서 추가하여 오디언스를 정리하고 다운스트림을 자동화합니다.

## 이메일 캠페인 에이전트 설정 방법

Ask AI를 사용하여 캠페인을 시작하는 간결한 단계별 경로입니다. Ask AI에 요청하는 것으로 시작한 다음(메뉴를 찾아다니지 않고), 도메인 설정, 카피 생성, A/B 테스트, 오디언스 선택, 예약을 처리할 수 있는 가이드 플로우를 따라가면 모든 것이 캠페인 빌더에서 발송 준비 상태로 완료됩니다.

1. **Ask AI를 열고 캠페인 생성을 요청하세요**

   예시 프롬프트: "이메일 캠페인 만들어줘", "내 상품을 홍보하는 이메일을 만들어줘"

   Ask AI가 작업을 인식하고 빌더에서 이메일 캠페인 에이전트 가이드 플로우를 시작합니다.

   ![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055865684/original/GhABF39GlQZSJz2ohpnv5RxXcnK_F_-EZg.gif?1760361925)

2. **전용 발송 도메인 확인 검토 (건너뛰기 가능)**

   플로우가 먼저 전용 발송 도메인이 설정되어 있는지 확인합니다.

   지금 가이드 도메인 설정을 완료하거나(전달률을 위해 권장) 건너뛰고 나중에 완료할 수 있습니다.

   설정이 필요한 경우, 필요한 DNS 레코드가 표시됩니다(지원되는 경우 자동 구성, 그렇지 않으면 수동 복사/붙여넣기).

3. **Ask AI의 간단한 질문에 답하세요**

   캠페인 이름, 오퍼/주제, 오디언스 의도, 톤을 제공합니다.

   Ask AI가 정제된 프롬프트를 구축하고, 정보를 사용하여 이메일 본문을 생성하며, 예약 세부사항을 미리 채웁니다.

   ![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055865975/original/OhqzyckIxgWjmfobdPB0K8WlNMn95HAxBA.png?1760362117)

4. **일관된 스타일링을 위해 브랜드 키트 연결 (선택사항)**

   Ask AI가 브랜드 보드/브랜드 보이스에 연결하여 카피와 형식이 브랜드 가이드라인과 일치하도록 할 수 있습니다.

   필요에 따라 톤을 조정하거나 콘텐츠를 재생성할 수 있습니다.

   ![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055866337/original/44n13CfpO1m7t0Z195G8cFS8pbROwBd6Dg.png?1760362267)

5. **제목 A/B 테스트 활성화 (권장)**

   제목 테스트(Subject Line Test)를 켜서 여러 제목을 생성합니다.

   변형 수(최대 6개), 샘플 크기, 테스트 기간, 승리 지표(고유 열람수 또는 고유 클릭수)를 선택합니다.

   "기본 설정" 옵션을 사용할 수 있습니다.

   ![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055866707/original/bipPguyNfZs41pG0KLnTOH7BWTFBov-zQg.png?1760362433)

6. **오디언스를 선택하세요**

   미리 만들어진 세그먼트, 태그 또는 스마트 리스트를 사용하여 지금 수신자를 타겟팅합니다.

   ![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055867139/original/UuzCsDVgtEjpT0IFSNFRvaJKn6kybmSZkQ.png?1760362503)

7. **Ask AI 요약 검토 및 확인**

   Ask AI가 캠페인 이름, **설명**, **목표**, **오디언스**, **발송** 상태가 포함된 빠른 요약 카드를 표시합니다.

   **"이게 맞나요? (예/아니오/편집)"**이라는 메시지가 나타납니다.

   **예**를 선택하면 에이전트가 캠페인을 생성하고 **"모두 완료! 캠페인을 생성했습니다."**라고 게시하며 **캠페인 빌더**에서 캠페인을 여는 **여기서 검토 및 예약** 링크가 나타납니다.

   ![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055868234/original/d33bu5tRZFwCP9pafmqnSfeOoZm5ou35KA.png?1760362839)

8. **실행 및 결과 모니터링**

   준비되면 캠페인 실행을 클릭합니다.

   발송 후, 이메일 대시보드에서 참여도를 추적하고 필요에 따라 후속 조치를 취합니다(예: 해당하는 경우 열람하지 않은 사람에게 재발송).

   ![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055868095/original/IPls_sed467cBFOzgXXdPYGcz8n5N_OmoQ.png?1760362744)

## 자주 묻는 질문

**Q: AI가 작성한 콘텐츠를 발송 전에 편집할 수 있나요?**

A: 네. 실행 전에 Ask AI의 출력을 자유롭게 편집, 재생성, 형식 지정할 수 있습니다.

**Q: 몇 개의 제목 변형을 테스트할 수 있고, 규칙은 무엇인가요?**

A: 최대 **6개** 변형; 최소 **변형당 10명의 연락처**와 **총 40명**의 연락처가 필요합니다. A/B 테스트는 **지금 발송**과 예약만 지원합니다(일괄 발송/RSS는 지원하지 않음).

**Q: 예약 후 스마트 리스트나 세그먼트가 업데이트되나요?**

A: **스마트 리스트**는 **발송 시점**에 평가되어 동적으로 업데이트됩니다. **미리 만들어진 세그먼트**는 생성/예약 시점에 스냅샷을 만들고, 빌더의 **세그먼트**는 설계상 동적입니다.

**Q: 발송 후 결과는 어디서 볼 수 있나요?**

A: **마케팅 → 이메일 → 통계** (이메일 통합 대시보드)를 사용하여 캠페인, 워크플로우, 일괄 작업 전반의 통합 지표를 확인하세요.

**Q: LC Email/Mailgun 대신 SMTP를 사용 중인데 일부 통계가 누락되는 이유는?**

A: SMTP 연동은 일반적으로 **열람**과 **클릭**만 표시하며, **전달됨/반송됨**은 Mailgun/LC Email이 필요합니다.

**Q: 이메일을 열람하지 않은 연락처에게 재발송할 수 있나요?**

A: 네, **열람하지 않은 사람에게 재발송**은 **RSS**와 **일괄** 예약 캠페인(지금 발송 포함)에서 지원됩니다.

**Q: 캠페인에서 어떤 발송 방법을 사용할 수 있나요?**

A: 목표와 리스트 크기에 따라 **지금 발송**, **예약**, **일괄 예약**, 또는 **RSS 예약** 중에서 선택할 수 있습니다.
