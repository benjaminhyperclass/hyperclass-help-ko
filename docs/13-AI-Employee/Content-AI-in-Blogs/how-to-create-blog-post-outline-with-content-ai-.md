# 콘텐츠 AI로 블로그 포스트 개요 만드는 방법

콘텐츠 AI(Content AI)는 최첨단 AI 글쓰기 어시스턴트로, 콘텐츠 제작을 10배 빠르고 쉽게 만들어 더 높은 품질의 결과를 만들어냅니다. 고급 머신러닝 알고리즘과 자연어 처리 기술을 활용하여 블로그 개요, 글 서론, 특정 섹션을 몇 분 만에 생성할 수 있어, 작성자와 콘텐츠 크리에이터가 연구와 초안 작성에 수많은 시간을 쓰는 대신 내용을 다듬고 완성하는 데 집중할 수 있습니다.

## 콘텐츠 AI 사용의 장점

- 더 짧은 시간에 몇 달치 콘텐츠 생성
- 비즈니스 요구에 맞춰 콘텐츠 맞춤화

## 콘텐츠 AI를 활성화하는 방법

에이전시 Settings(설정) >> Company(회사) 탭으로 이동해서 기능에 접근할 수 있습니다. 그다음 콘텐츠 AI를 활성화할 수 있습니다. 모든 요금제에서 사용할 수 있습니다. 비즈니스는 SaaS 구성 도구로 이동하거나, SaaS가 포함된 $497/월 또는 $4970/년 요금제를 사용하는 경우 SubAccounts(하위 계정) > manage clients(클라이언트 관리)로 이동하여 재청구를 설정할 수 있습니다.

자세한 내용은 콘텐츠 AI 활성화 및 재청구 방법 문서를 참조하세요.

## 블로그에서 콘텐츠 AI 시작하기

- Sites(사이트) > Blogs(블로그) > Create New Post(새 포스트 작성)로 이동합니다.

![새 블로그 포스트 작성](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48289092707/original/u-xPHdj-h9xErqJl9CTa1_WTJ-NaAxZe-Q.png?1679652801)

**참고사항**
하위 계정에서 콘텐츠 AI를 사용할 수 있도록 사용자 권한을 활성화해야 합니다.

- 새 블로그 포스트를 작성한 후 메타 설명을 추가합니다.

![메타 설명 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48289093439/original/r5c95Qv__ZHOgbxfPZ513v30bdPa2JuSAg.png?1679652968)

- 그다음 콘텐츠 AI를 사용할 수 있는 블로그 포스트 편집기로 이동할 수 있습니다.
- Content AI(콘텐츠 AI)를 클릭하면 개요(Outline), 서론(Introduction), 특정 섹션(Specific Section) 옵션 중 선택하고 포스트 제목, 포스트 설명, 키워드, 콘텐츠 톤, 여러 변형을 작성하라고 요청합니다.

**참고사항:**
- 개요(Outline) - 블로그 구조를 잡고 포스트에 대한 전반적인 아이디어를 제공하는 불릿 포인트입니다.
- 서론(Introduction) - 블로그 포스트 작성의 주요 개념을 공유하는 블로그의 첫 번째 문단입니다.
- 특정 섹션(Specific Section) - 블로그에 5개 섹션이 있다면, 이 부분으로 한 블로그 섹션의 콘텐츠를 만들 수 있습니다.

![콘텐츠 AI 옵션 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48289100842/original/jiB9VS9Z4WMQ2nCdpEe11hRfZN9wA7Xiqw.png?1679654737)

**포스트 제목** - "콘텐츠 마케팅"과 같은 주제나 카테고리 이름을 입력합니다.
**포스트에 대한 간단한 설명 제공** - 포스트 설명은 "상위 5가지 콘텐츠 마케팅 채널"과 같아야 합니다.
**키워드 추가** - "콘텐츠, 마케팅, 전략, 콘텐츠 생성"과 같은 키워드를 입력합니다(단어를 입력하고 엔터를 누르세요).
**글쓰기 톤** - 주어진 옵션에서 톤을 선택합니다.

- **예를 들어** 콘텐츠 생성을 위해:
- 콘텐츠 유형 - 블로그의 "개요(Outline)"를 선택합니다.
- 1부터 5까지 변형 수를 선택합니다.

- 양식을 작성하면 제안된 프롬프트의 변형이 생성됩니다. 변형 콘텐츠를 복사하면 편집기 공간 오른쪽에 나타납니다. 포스트 세부 정보가 좋으면 계속을 클릭하여 포스트를 만듭니다. 콘텐츠는 캡션 영역에 추가됩니다.

## 자주 묻는 질문

### 콘텐츠에 몇 가지 톤을 생성할 수 있나요?

콘텐츠 AI는 재미있는(funny), 캐주얼(casual), 흥미진진한(excited), 전문적인(professional), 재치 있는(witty), 비꼬는(sarcastic), 여성적인(feminine), 남성적인(masculine), 대담한(bold), 드라마틱한(dramatic), 투덜거리는(grumpy), 비밀스러운(secretive), 정중한(respectful), 설득력 있는(persuasive) 톤을 제공합니다.

### 이전에 생성한 콘텐츠를 확인하고 싶은데 어디서 볼 수 있나요?

콘텐츠 AI로 생성한 콘텐츠를 확인하려면 **Automation(자동화) > Content AI(콘텐츠 AI)**로 이동하여 자세한 내용을 확인하고 이미 생성한 콘텐츠를 볼 수 있습니다. 자세한 정보는 [이 문서](../how-to-find-past-records-of-content-generated-by-content-ai.md)를 참조하세요.

### 콘텐츠 AI에서 언어 옵션이 제공되나요?

콘텐츠 변형을 생성할 수 있는 언어 선택 기능을 제공하도록 범위를 확장할 예정입니다.

### 콘텐츠 AI로 블로그에서 장문 콘텐츠가 제공되나요?

블로그에서 장문 콘텐츠를 제공하도록 범위를 확장할 예정입니다.
