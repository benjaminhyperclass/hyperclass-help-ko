# **Agent Studio에서 브랜드 보이스와 브랜드 킷이란?**

브랜드 보이스와 브랜드 킷을 사용하면 Agent Studio 내에서 AI가 생성하는 콘텐츠에 브랜드 아이덴티티를 적용할 수 있습니다.

**브랜드 보이스**는 콘텐츠의 사운드를 제어합니다:

- 톤
- 메시징 스타일
- 성격
- CTA 스타일

**브랜드 킷**은 콘텐츠의 외관을 제어합니다:

- 색상
- 폰트
- 로고
- 시각적 아이덴티티

노드 내에서 선택되면, 이러한 설정이 AI 출력을 자동으로 안내합니다.

---

## **Agent Studio 브랜드 통합의 주요 이점**

Agent Studio 내에서 브랜드 보이스와 브랜드 킷을 사용하면 일관되고 확장 가능한 브랜딩이 보장됩니다.

- 일관된 출력: 텍스트, 이미지, 미디어가 자동으로 브랜드와 정렬됩니다.
- 멀티 브랜드 워크플로우: 동일한 에이전트 내의 다른 노드에 다른 브랜드를 할당합니다.
- 수동 편집 감소: 생성 후 재작성 및 재디자인을 최소화합니다.
- 자동 업데이트: 브랜드 보드 변경 사항이 향후 노드 실행에 적용됩니다.
- 품질 관리 향상: 내장된 브랜드 규칙으로 브랜드에 맞지 않는 메시징을 줄입니다.

---

## **미디어 유형별 브랜딩 작동 원리**

선택되면 브랜딩이 다음에 영향을 미칩니다:

### **텍스트 생성**

- 글쓰기 스타일
- 톤
- 단어 선택
- CTA 형식

### **이미지 및 동영상 생성**

- 브랜드 색상
- 로고 배치
- 폰트 스타일링(해당하는 경우)

### **오디오 생성**

- 스크립트 톤
- 음성 모델 매핑(구성된 경우)

브랜드 규칙은 노드가 실행될 때 AI에 자동으로 전달됩니다.

---

## **브랜드 보드 동기화 작동 방식**

브랜드 설정은 브랜드 보드에서 가져옵니다.

다음을 업데이트하면:

- 색상
- 로고
- 메시징 가이드라인

향후 노드 실행에 이러한 변경 사항이 자동으로 반영됩니다.

기존에 생성된 콘텐츠는 소급적으로 변경되지 않습니다.

---

## **Agent Studio에서 브랜드 보이스와 브랜드 킷 설정하기**

다음 단계를 따르세요:

### **1단계: 브랜드 보드 생성(미완료 시)**

- `Marketing(마케팅) → Brand Boards(브랜드 보드)`로 이동
- 최소한 다음 중 하나를 생성:
  - 브랜드 보이스
  - 브랜드 킷(디자인 킷)

![브랜드 보드 생성](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155066858028/original/mGV9vJ-hLs1eLorZRioP-J8vhr7z_jiIYw.png?1773392875)

### **2단계: Agent Studio 열기**

- `AI Agents(AI 에이전트) → Agent Studio`로 이동
- 에이전트를 열거나 생성
- 생성 노드(텍스트, 이미지, 동영상, 오디오) 추가

![Agent Studio 열기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155066858286/original/ABUGMKqk7maVgzRfoXOL0BZpghLbIa6POQ.png?1773393025)

### **3단계: 브랜드 설정 선택**

- 노드를 클릭하여 설정 열기
- **Brand Voice(브랜드 보이스)** 드롭다운 찾기
- 원하는 브랜드 보이스 선택
- **Brand Kit(브랜드 킷)** 선택
- **Save(저장)** 클릭

노드를 떠나지 않고 새 브랜드를 생성하려면 **+ Add New(+ 새로 추가)**를 클릭할 수도 있습니다.

![브랜드 설정 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155066858703/original/erMkIHFUte2dwQIzW7xTVsns0T21zGIpSQ.png?1773393192)

### **4단계: 출력 테스트**

- **Test(테스트)** 클릭
- 샘플 콘텐츠 생성
- 톤과 비주얼이 브랜드와 일치하는지 확인

### **5단계: 발행**

만족스러우면 에이전트를 스테이징 또는 프로덕션으로 이동합니다.

---

## **멀티 브랜드 워크플로우**

다른 노드에 다른 브랜드를 할당할 수 있습니다.

다음과 같은 경우에 유용합니다:

- 에이전시가 여러 클라이언트 브랜드를 관리하는 경우
- 프랜차이즈가 워크플로우를 공유하지만 현지화된 톤이 필요한 경우
- 메시징 스타일을 A/B 테스트하는 경우

각 노드는 할당된 브랜드를 독립적으로 따릅니다.

## **자주 묻는 질문**

**Q: 이 기능을 사용하기 전에 브랜드 보드가 필요한가요?**
네. `Marketing(마케팅) → Brand Boards(브랜드 보드)`에 최소한 하나의 브랜드 보이스와 하나의 브랜드 킷이 존재해야 합니다.

**Q: 브랜드 킷을 업데이트하면 기존 콘텐츠가 변경되나요?**
아니요. 향후 노드 실행에서만 업데이트된 설정을 사용합니다.

**Q: 브랜드 보이스만 적용하거나 브랜드 킷만 적용할 수 있나요?**
네. 필요시 하나의 드롭다운을 None으로 설정하면 됩니다.

**Q: 몇 개의 브랜드를 생성할 수 있나요?**
로케이션당 여러 브랜드 보이스와 브랜드 킷을 저장할 수 있습니다. 관리의 편의를 위해 총 개수를 합리적으로 유지하세요.

**Q: Ask AI와 함께 작동하나요?**
Ask AI가 브랜드가 적용된 노드가 포함된 워크플로우를 트리거하면, 출력에 해당 브랜드 설정이 상속됩니다.

**Q: 프롬프트 내에서 브랜드 톤을 재정의할 수 있나요?**
네. 노드의 수동 지시사항은 해당 실행에 적용되지만 브랜드 보드를 수정하지는 않습니다.

**Q: 오디오 AI가 커스텀 음성 모델을 사용하나요?**
브랜드 보이스 설정에서 매핑된 경우, 지원되는 음성 모델이 자동으로 적용됩니다.

**Q: 모든 플랜에서 사용할 수 있나요?**
브랜드 통합은 Agent Studio와 동일한 플랜 자격을 따릅니다.
