# Ask AI의 퍼널 생성 에이전트

Hyperclass Ask AI의 퍼널 생성 에이전트가 어떻게 지능형 자동화, 브랜드 개인화, 원활한 연동을 통해 퍼널 제작을 간소화하고, 그 어느 때보다 빠르게 높은 전환율의 퍼널을 런칭할 수 있게 도와주는지 알아보세요.

## Ask AI의 퍼널 생성 에이전트란?

퍼널 생성 에이전트는 Hyperclass Ask AI 플랫폼 내의 AI 기반 어시스턴트로, 최소한의 노력으로 전문적이고 높은 전환율의 퍼널을 만들 수 있도록 도와줍니다. 스마트 자동화, 사전 제작된 템플릿, 지능형 Q&A를 활용하여 퍼널 생성 과정을 안내하므로, 기술적인 설정보다는 비즈니스 목표에 집중할 수 있습니다.

## 퍼널 생성 에이전트의 주요 장점

퍼널 생성 에이전트를 사용하여 마케팅 및 영업 워크플로우를 가속화하는 이점들을 확인해보세요:

- **빠른 퍼널 생성**: AI에게 "퍼널을 만들어 줘"라고 간단히 요청하면 즉시 퍼널이 생성됩니다.
- **사전 제작된 템플릿**: 일반적인 사용 사례에 맞게 제작된 바로 사용 가능한 템플릿을 선택할 수 있습니다.
- **AI 가이드 설정**: 상황에 맞는 안내 질문과 지능형 제안을 통해 퍼널을 맞춤 설정할 수 있습니다.
- **손쉬운 이미지 처리**: 직접 이미지를 업로드하거나 연결된 구글 비즈니스 프로필에서 자동으로 이미지를 가져올 수 있습니다.
- **브랜드 개인화**: 퍼널 색상을 브랜드에 즉시 맞추고 브랜드 보이스를 사용하여 브랜드에 맞는 콘텐츠를 생성할 수 있습니다.
- **네이티브 연동**: 캘린더, 정적 폼, 리뷰 API를 원활하게 연결하여 통합된 경험을 제공합니다.
- **즉시 빌더 및 미리보기 링크**: 퍼널 빌더와 페이지 미리보기에 대한 직접 링크를 받아 즉시 검토하고 런칭할 수 있습니다.

## Ask AI에서 퍼널 생성 에이전트 사용하는 방법

퍼널 생성 에이전트 설정은 간단하며, 브랜드와 비즈니스 요구사항에 맞게 퍼널이 맞춤 설정됩니다. 적절한 설정을 통해 AI 기반 퍼널 생성과 개인화의 효과를 극대화할 수 있습니다.

### 1단계: Ask AI에서 퍼널 생성기 실행

- Hyperclass 계정에 로그인합니다.
- 상단 우측 네비게이션 바의 Ask AI 버튼을 클릭합니다.
- 다음 중 하나를 선택합니다:
  - Ask AI 인터페이스에서 직접 퍼널 생성(Generate Funnel)을 클릭합니다.
  - 또는 Ask Anything 입력창에 "퍼널을 만들어 줘"를 입력합니다.

![퍼널 생성기 실행](https://jumpshare.com/v/J9rdWVHp3unLY9efM0En+/GIF+Recording+2025-06-19+at+7.09.52+PM.gif)

### 2단계: 퍼널 유형 선택

- Ask AI가 현재 사용 가능한 퍼널 유형을 제시합니다:
  - 리드 마그넷 퍼널
  - 예약 퍼널
- 필요에 맞는 것을 선택합니다.

![퍼널 유형 선택](https://jumpshare.com/v/TaC1KrRWYmvSaQMGckUA+/Screen+Shot+2025-06-19+at+8.57.36+PM.png)

### 3단계: 스마트 Q&A에 답변

- 퍼널을 개인화하는 데 도움이 되도록 AI의 안내형, 상황 인식 질문에 답변합니다.

![스마트 Q&A](https://jumpshare.com/v/90ksIWNfXcTLOV3ZxOht+/GIF+Recording+2025-06-19+at+9.06.29+PM.gif)

### 4단계: 이미지와 로고 업로드

- 제품 이미지, 브랜드 자산, 로고를 업로드합니다.
- 구글 마이 비즈니스(GMB) 프로필이 연결되어 있으면 이미지가 자동으로 가져와질 수 있습니다.
- 최상의 결과를 위해 진정한 브랜드 비주얼을 사용하세요.

![이미지와 로고 업로드](https://jumpshare.com/v/k9OjIvremi7TINK3UeaK+/Screen+Shot+2025-06-19+at+9.12.00+PM.png)

### 5단계: 퍼널 개인화

- 브랜드에 맞는 색상 테마를 선택합니다.
- 브랜드 보이스가 설정되어 있다면, AI가 이를 사용하여 일관되고 브랜드에 맞는 콘텐츠를 생성합니다.

![퍼널 개인화](https://jumpshare.com/v/PiJ1m4EXWRH17jOjxiQO+/GIF+Recording+2025-06-19+at+9.13.49+PM.gif)

### 6단계: 퍼널 검토 및 접근

입력이 완료되면:

- Ask AI가 사용 사례와 대화에 따라 퍼널을 생성합니다.
- 다음을 받게 됩니다:
  - 빌더 링크(직접 편집용)
  - 각 퍼널 페이지의 미리보기 링크

## 템플릿 선택 및 확장

적절한 템플릿 선택은 퍼널을 캠페인 목표에 맞게 정렬하는 데 중요합니다. 퍼널 에이전트는 현재 두 가지 핵심 템플릿을 제공하며, 더 많은 템플릿이 준비되고 있습니다.

- 사용 가능한 템플릿을 검색하고 레이아웃을 미리 봅니다.
- 템플릿은 높은 전환율과 쉬운 맞춤 설정을 위해 설계되었습니다.
- 선택 옵션을 확장할 향후 템플릿 출시를 기대해 주세요.

## 구글 비즈니스 프로필(GBP) 이미지 연동

구글 비즈니스 프로필을 연결하면 AI가 자동으로 비즈니스 이미지를 가져와 시간을 절약하고 브랜드 일관성을 보장합니다.

- Hyperclass 설정에서 GBP를 연결합니다.
- 퍼널 에이전트가 퍼널에서 사용할 수 있는 이미지를 가져옵니다.
- 원한다면 수동 업로드로 쉽게 재정의할 수 있습니다.

## 브랜드 보이스 개인화

일관된 메시지는 효과적인 퍼널의 핵심입니다. 퍼널 에이전트는 브랜드 보이스(설정된 경우)를 활용하여 생성된 모든 콘텐츠가 브랜드의 톤과 스타일에 맞도록 합니다.

- Hyperclass에서 브랜드 보이스를 설정합니다.
- AI가 이 프로필을 사용하여 헤드라인, 설명, 행동 유도 문구를 자동 생성합니다.

## 네이티브 연동: 캘린더, 폼, 리뷰

원활한 고객 여정을 위해 필수 도구를 퍼널에 직접 연동합니다.

- 예약을 받기 위해 캘린더 예약 위젯을 추가합니다.
- 리드 수집이나 피드백을 위해 정적 폼을 삽입합니다.
- 신뢰를 구축하기 위해 리뷰 API를 사용하여 리뷰를 표시합니다.

## 자주 묻는 질문

**Q: 퍼널 에이전트가 생성한 퍼널을 나중에 맞춤 설정할 수 있나요?**
A: 네, 필요에 따라 퍼널을 추가로 편집하고 세밀하게 조정할 수 있는 빌더 링크를 받습니다.

**Q: 구글 비즈니스 프로필이 연결되어 있지 않으면 어떻게 하나요?**
A: 퍼널 설정 과정에서 직접 이미지를 수동으로 업로드할 수 있습니다.

**Q: 현재 사용 가능한 템플릿은 몇 개인가요?**
A: 퍼널 에이전트는 현재 두 가지 핵심 템플릿을 제공합니다.

**Q: AI가 내 브랜드 보이스를 어떻게 사용하나요?**
A: Hyperclass에서 브랜드 보이스 프로필을 설정했다면, AI가 브랜드의 톤과 메시징에 맞는 퍼널 콘텐츠를 생성합니다.

**Q: 여러 비즈니스나 브랜드에 퍼널 에이전트를 사용할 수 있나요?**
A: 네, Hyperclass 계정 내의 각 비즈니스나 브랜드 프로필에 대해 퍼널 생성 프로세스를 시작하기만 하면 됩니다.
