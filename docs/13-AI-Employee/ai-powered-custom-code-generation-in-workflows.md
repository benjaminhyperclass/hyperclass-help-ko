# **워크플로우에서 AI 기반 코드 생성이란?**

**Code with AI (Beta)** 기능으로 워크플로우에서 커스텀 코드 작성이 더 스마트하고, 빠르고, 쉬워졌습니다. 이 새로운 기능을 통해 사용자는 원하는 기능을 일반 언어로 설명만 하면 JavaScript 코드 스니펫을 생성할 수 있어, 수동 코딩과 디버깅의 필요성을 크게 줄였습니다.

## **AI 기반 코드 생성의 주요 장점**

AI를 활용한 코드 생성은 생산성을 높이고 사용자의 기술적 진입 장벽을 낮춥니다. 주요 장점은 다음과 같습니다:

- **AI 기반 코드 생성** – 필요한 기능을 설명하기만 하면 AI가 해당하는 JavaScript 코드를 생성해줍니다.

- **자동 속성 연동** – AI가 이전 워크플로우 단계의 입력 속성을 자동으로 매핑해줍니다.

- **원클릭 구현** – AI가 생성한 코드를 빠르게 검토하고 워크플로우에 삽입할 수 있습니다.

- **간편한 재생성** – 다른 버전이 필요한가요? 생성 버튼을 다시 클릭하면 업데이트된 스니펫을 받을 수 있습니다.

- **효율성 향상** – 커스텀 코딩에 필요한 시간과 기술적 전문성을 줄입니다.

## **AI와 함께 코드 사용하는 방법**

다음 단계를 따라 워크플로우 내에서 커스텀 코드를 원활하게 생성하고 구현할 수 있습니다:

- **커스텀 코드 액션(Custom Code Action) 선택** – 워크플로우 내에서 커스텀 코드 액션을 선택합니다.

![커스텀 코드 액션 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042231921/original/6pr37vsp1ZczytBU3QnMBwy77wPNQMCHSQ.png?1740505496)

- **AI 지원 활성화** – **"Code with AI"** 버튼을 클릭하여 AI 기반 코딩 도우미를 엽니다.

![AI 지원 활성화](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042231927/original/pArmHNNNJau5suK0wFDZSZ8M9rGevpintQ.png?1740505511)

- **기능 설명** – 필요한 기능에 대한 명확한 설명을 제공합니다 (예: *"MM-DD-YYYY 형식으로 날짜를 반환하는 API가 있습니다. 이를 YYYY-MM-DD 형식으로 변환해주세요."*).

![기능 설명 입력](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042231930/original/32_Ea_x5X--l0IQ6uPylQONQJLv7u8qtEg.png?1740505526)

- **코드 생성** – **Generate** 버튼을 클릭하면 AI가 필요한 JavaScript 스니펫을 생성합니다.

![코드 생성 결과](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042231945/original/Eioo-OL4gtNyTpnpNoCunTlpEYAm57EN8A.png?1740505567)

- **검토 및 구현** – 결과물을 확인하고 **Use Code** 버튼을 클릭하여 워크플로우에 삽입합니다.

**Pro Tip:** 프롬프트가 구체적일수록 AI가 생성하는 코드의 품질이 좋아집니다!

## **활용 사례 예시**

워크플로우에서 AI 기반 코드 생성을 활용할 수 있는 실용적인 방법들입니다:

- **데이터 형식 변환** – 전화번호나 날짜를 표준화된 형식으로 변환합니다.

- **API 연동** – HTTP 요청을 통해 외부 서비스에서 데이터를 가져오는 코드를 생성합니다.

- **수학적 계산** – 이자율이나 할인율 같은 실시간 계산을 수행합니다.

- **문자열 조작** – 제목에서 이메일 주소를 추출하는 것처럼 텍스트에서 핵심 정보를 추출합니다.

- **조건부 로직** – 특정 기준에 따라 리드를 라우팅하는 것과 같은 커스텀 의사결정 프로세스를 구현합니다.

## **자주 묻는 질문**

**Q: Code with AI를 사용하려면 JavaScript를 알아야 하나요?**
A: 아니요, Code with AI는 프로그래밍 경험이 없는 사용자도 도울 수 있도록 설계되었습니다. 필요한 기능만 설명하면 AI가 필요한 JavaScript 코드를 생성해드립니다.

**Q: AI가 생성한 코드를 편집할 수 있나요?**
A: 네! 생성된 코드를 워크플로우에 구현하기 전에 검토하고 수정할 수 있습니다.

**Q: AI가 생성한 코드가 제가 원하는 것과 정확히 맞지 않으면 어떻게 하나요?**
A: 코드가 기대에 부합하지 않으면 **Generate** 버튼을 다시 클릭해서 새 버전을 받거나, 설명을 더 구체적으로 다시 작성해서 더 나은 결과를 얻을 수 있습니다.

**Q: 이 기능은 모든 워크플로우 액션에서 사용할 수 있나요?**
A: Code with AI는 현재 워크플로우 내의 **커스텀 코드** 액션에서만 사용할 수 있습니다.

**Q: AI가 생성한 코드로 API 호출을 할 수 있나요?**
A: 네, Code with AI는 HTTP 요청 생성과 응답 처리를 포함한 API 연동 코드를 생성할 수 있습니다.

**Q: AI가 생성한 코드가 매번 완벽하게 작동하나요?**
A: AI는 작동하는 코드를 생성하도록 설계되어 있지만, 실제 워크플로우에 배포하기 전에 항상 결과물을 검토하고 테스트하는 것을 권장합니다.
