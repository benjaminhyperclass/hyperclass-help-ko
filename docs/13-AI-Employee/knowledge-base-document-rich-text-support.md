# 지식 베이스 - 문서 및 서식 있는 텍스트 지원

기존 Word 문서, PDF, 서식 있는 텍스트 노트를 Hyperclass AI 직원이 활용할 수 있는 지식으로 빠르게 변환하세요. 문서를 업로드하거나 서식 있는 텍스트 편집기에서 직접 작성하여 정보를 더 빠르게 중앙화하고 수동 복사-붙여넣기를 줄일 수 있습니다. 자동 파싱이 콘텐츠를 검색 가능한 청크로 구조화하여 AI 직원이 정확한 답변을 검색할 수 있게 해줍니다. SOP, 제품 문서, 고객지원 플레이북을 통합하는 팀에게 이상적입니다.

# 지식 베이스 문서 지원이란?

문서 지원은 Hyperclass 지식 베이스를 채우는 두 가지 방법을 추가합니다: 문서 파일(DOC, DOCX, PDF) 업로드와 내장 서식 있는 텍스트 편집기에서 직접 콘텐츠 생성/편집. 수집 과정에서 Hyperclass는 제목과 텍스트를 추출하여 AI 직원이 정확히 맞는 구절을 찾고 인용할 수 있도록 합니다.

Hyperclass의 지식 베이스는 이제 일반적인 문서 형식과 서식 있는 텍스트에서 콘텐츠를 수집할 수 있습니다. 시스템이 문서를 구조화된 섹션으로 자동 파싱하여 답변 품질을 향상시키고 수동 서식 작업을 줄여줍니다.

## 지식 베이스 문서 지원의 주요 이점

이점을 이해하면 언제 파일을 업로드할지 편집기에서 작성할지 결정하는 데 도움이 됩니다. 이러한 이점은 속도, 정확성, 유지 관리에 중점을 두어 팀이 자신감을 가지고 업데이트를 배포할 수 있도록 합니다.

- **다양한 형식 지원:** 업계 표준 파일 유형(DOC, DOCX, PDF)을 허용하여 이미 유지 관리하고 있는 콘텐츠를 재사용할 수 있습니다.

- **시간 절약:** 섹션을 복사하여 붙여넣는 대신 한 번에 전체 문서를 수집할 수 있습니다.

- **인라인 편집:** 서식 있는 텍스트 편집기를 사용하여 Hyperclass를 떠나지 않고 빠른 콘텐츠 업데이트를 하거나 새로운 항목을 생성할 수 있습니다.

- **높은 답변 품질:** 자동 파싱이 콘텐츠를 제목과 섹션에 맞춰진 검색 가능한 청크로 구조화합니다.

- **기존 소스와 연동:** 문서와 서식 있는 텍스트를 URL 및 FAQ와 결합하여 훈련 데이터를 강화할 수 있습니다.

## 지원되는 파일 형식 및 수집 방식

어떤 콘텐츠가 추출되고 어떻게 처리되는지 알면 깔끔하게 수집되고 고품질 답변을 제공하는 문서를 준비할 수 있습니다.

**참고:** 업로드하기 전에 민감한 콘텐츠를 제거하고 제목이 구조를 정확히 반영하는지 확인하세요. 명확한 제목은 더 나은 청크와 더 나은 답변을 만들어냅니다.

- 지원 형식: DOC, DOCX, PDF.

- **텍스트 전용 수집:** 텍스트만 수집됩니다. 임베드된 이미지는 건너뛰어 지식을 가볍고 검색 가능한 콘텐츠에 집중하도록 합니다.

- **제목 인식:** 감지된 제목 계층 구조(예: H1-H4)가 청킹과 검색 관련성을 향상시킵니다.

- **즉시 인덱싱:** 새 콘텐츠는 파일 크기와 인덱싱에 따라 즉시 또는 약 1분 내에 사용할 수 있게 됩니다.

## 자동 파싱 및 구조화

파싱은 긴 문서를 섹션과 제목에 맞춰진 관리 가능한 청크로 변환하여 AI 직원 응답에서 정확한 검색과 인용을 가능하게 합니다.

- 텍스트를 추출하고 제목을 감지하여 논리적 계층 구조를 형성합니다.

- 대용량 파일을 더 빠르고 관련성 있는 검색을 위해 최적화된 청크로 분할합니다.

- 완료 시 콘텐츠를 인덱싱하여 봇이 최신 정보를 사용할 수 있도록 합니다.

- 문서 수준 컨텍스트를 보존하여 검색 중에 관련 섹션이 연결된 상태를 유지합니다.

# 지식 베이스에서 문서 업로드 및 사용 방법

적절한 설정은 문서 항목이 깔끔하게 수집되고 AI 직원을 위해 빠르게 검색 가능해지도록 보장합니다.

- 하위 계정에서 AI Agents(AI 직원)로 이동합니다.

- Knowledge Base(지식 베이스) 탭을 클릭합니다.

![지식 베이스 탭](https://jumpshare.com/share/s08lZnvGFCuaWND1fh5S+/Screen+Shot+2025-10-13+at+7.34.33+PM.png)

- 기존 지식 베이스를 편집합니다.
 
- 또는 **Create(생성)** New를 클릭하고 이름과 설명을 입력합니다.

![새 지식 베이스 생성](https://jumpshare.com/share/rl6Si4GgSsXRSp3mIfcB+/GIF+Recording+2025-10-13+at+7.36.42+PM.gif)

- **Add Source(소스 추가)** → File Upload(파일 업로드)를 클릭합니다.

![파일 업로드 선택](https://jumpshare.com/share/xqx6xqMDPjU6dndjlp9A+/Screen+Shot+2025-10-13+at+7.39.50+PM.png)

- DOC, DOCX, 또는 PDF 파일을 **드래그 앤 드롭** 하거나 **찾아보기**로 선택합니다.

- **Upload Files(파일 업로드)**를 클릭합니다.

![파일 업로드](https://jumpshare.com/share/LZvgpjZTWaNHkVa0ax1N+/Screen+Shot+2025-10-13+at+7.41.20+PM.png)

- 파싱이 완료되면 상태가 **Processed(처리됨)**로 바뀝니다.

- **화살표 아이콘**을 사용하여 업로드된 **파일**을 **새 탭**에서 **열거나** **휴지통 아이콘**을 클릭하여 **삭제**할 수 있습니다.

![파일 관리](https://jumpshare.com/share/O8im9ddFWHXC5kZnm2I6+/GIF+Recording+2025-10-13+at+7.50.37+PM.gif)

- **서식 있는 텍스트**를 추가하여 편집기에서 직접 콘텐츠를 **작성**하거나 **편집**할 수도 있습니다.

- **Save(저장)**를 클릭하여 서식 있는 텍스트를 지식 베이스에 추가합니다.

![서식 있는 텍스트 추가](https://jumpshare.com/share/zdeoHcHdw2HmvP32EUvJ+/GIF+Recording+2025-10-13+at+7.53.59+PM.gif)

## 자주 묻는 질문

**Q: Word나 PDF 안의 이미지도 가져와지나요?**

텍스트만 수집됩니다. 이미지는 지식을 가볍고 검색 가능하게 유지하기 위해 건너뜁니다.

**Q: 새 업로드가 AI 직원에게 얼마나 빨리 제공되나요?**

파일 크기와 인덱싱에 따라 즉시 또는 약 1분 내에 콘텐츠를 사용할 수 있습니다.

**Q: 구글 문서를 직접 업로드할 수 있나요?**

구글 문서를 .DOCX 또는 .PDF로 내보낸 다음 내보낸 파일을 업로드하세요.

**Q: 서식 있는 텍스트 편집이 업로드한 파일을 덮어쓰나요?**

아니요. 파일 기반 소스와 서식 있는 텍스트 항목은 별도로 저장되어 두 버전 모두 보존됩니다.

**Q: 문서 지원이 URL이나 FAQ 소스를 대체하나요?**

아니요. 문서와 서식 있는 텍스트는 URL과 FAQ를 보완하여 봇에게 더 깊고 신뢰할 수 있는 훈련 데이터를 제공합니다.

**Q: 문서 업로드가 기존 지식 베이스 항목을 덮어쓰나요?**

아니요. 각 업로드는 새로운 소스로 추가됩니다. 파일 기반 소스와 서식 있는 텍스트 항목을 URL 및 FAQ와 함께 유지할 수 있습니다.
