# Ask AI의 연락처 요약 에이전트

연락처 요약 에이전트는 Hyperclass Copilot의 Ask AI 도구에 새로 추가된 AI 기반 기능으로, 연락처 레코드를 즉시 요약하고 모든 연락처에 대한 세부 질문에 답변해드립니다. 이 기능은 연락처 데이터에 접근하고 해석하는 방식을 간소화하여 시간을 절약하고, 간단한 대화형 프롬프트로 실행 가능한 인사이트를 제공합니다.

## 연락처 요약 에이전트란?

연락처 요약 에이전트는 Hyperclass Copilot 내의 지능형 기능으로, 커뮤니케이션, 결제, 예약, 할 일 등 흩어져 있는 연락처 정보를 하나의 이해하기 쉬운 요약으로 통합합니다. 자연어 처리 기술을 활용하여 사용자가 모든 연락처에 대해 질문하거나 요약을 요청할 수 있으며, 여러 도구나 화면을 탐색할 필요 없이 즉시 사람과 같은 응답을 제공합니다.

이 도구는 대량의 고객 데이터를 관리하는 에이전시와 팀에 특히 유용하며, 분산된 레코드를 명확하고 실행 가능한 개요로 변환하여 더 나은 고객 참여와 빠른 의사결정을 지원합니다.

## 연락처 요약 에이전트의 주요 장점

연락처 요약 에이전트는 사용자가 더 효율적으로 작업하고 정보에 입각한 결정을 내릴 수 있도록 도와주는 여러 가지 장점을 제공합니다.

### 즉시 연락처 요약

- 최근 커뮤니케이션과 활동 로그를 한눈에 확인
- 총 결제금액과 미결제 잔액을 포함한 재정적 인사이트 제공
- 예정된 예약과 예약된 세션 확인
- 미완료 할 일과 영업 기회 파악
- 다양한 시스템 전반의 참여 트렌드 분석

### 대화형 Q&A

- "이 연락처와 마지막으로 상호작용한 것은 언제인가요?" 또는 "결제 이력이 어떻게 되나요?"와 같은 자연스러운 질문
- 설정이나 교육 없이 빠르고 사람과 같은 답변 수신
- 여러 탭이나 레코드를 검색할 필요 없음

### 시간 절약과 정확도 향상

- 수동 데이터 수집 및 해석 감소
- 항상 최신의 가장 관련성 높은 정보를 손쉽게 확인

## 연락처 요약 에이전트 사용 방법

연락처 요약 에이전트를 설정하고 사용하는 것은 간단하며, 최소한의 노력으로 인사이트에 액세스할 수 있습니다. 적절한 설정을 통해 연락처에 대한 가장 정확하고 포괄적인 요약을 얻을 수 있습니다.

### 1단계: 기능 접근

- Hyperclass 계정에 로그인합니다.
- 메인 메뉴에서 Ask AI 도구를 클릭합니다.

### 2단계: 에이전트와 상호작용

- Ask AI 채팅 창에서 "연락처 요약" 제안을 클릭합니다.

![연락처 요약 제안 클릭](https://jumpshare.com/v/d8jIeJ7bO3fWQrccw7H4+/Screen+Shot+2025-06-10+at+7.49.25+PM.png)

- "연락처 요약"과 같은 프롬프트를 직접 입력하고 제출 버튼을 클릭할 수도 있습니다.

![프롬프트 입력](https://jumpshare.com/v/Cc734mSCIpEXjOksFUAA+/Screen+Shot+2025-06-10+at+7.51.49+PM.png)

- 사용자는 연락처의 이력, 결제, 할 일 또는 예약에 대한 구체적인 질문도 할 수 있습니다(예: "결제 이력이 어떻게 되나요?").

### 3단계: 연락처 선택

- 기존 연락처 목록에서 원하는 연락처를 선택합니다.
- 제출 버튼을 클릭합니다.

### 4단계: 요약 검토

- 연락처 요약 에이전트가 Hyperclass 계정 전반의 최신 데이터를 사용하여 즉시 요약을 생성하거나 질문에 답변합니다.
- 제공된 인사이트를 활용하여 후속 조치, 미팅 일정 잡기 또는 레코드 업데이트를 진행하세요.

## 대화형 프롬프트 사용

연락처 요약 에이전트는 자연어를 이해하도록 설계되어 명령어를 외우지 않고도 필요한 정보를 쉽게 얻을 수 있습니다.

다음과 같은 개방형 질문을 해보세요:

- "김철수님의 대기 중인 할 일은 무엇인가요?"
- "이지영님의 다음 예약은 언제인가요?"
- "박민수님이 지금까지 얼마나 결제했나요?"
- 요약과 구체적인 질문을 조합하여 더 깊은 인사이트를 얻어보세요.

## 지원 데이터 유형

에이전트는 포괄적인 요약을 제공하기 위해 광범위한 연락처 관련 데이터 소스에서 정보를 가져옵니다.

- 커뮤니케이션 로그(이메일, 전화, SMS)
- 노트 및 활동 이력
- 결제 기록 및 미결제 잔액
- 예약 및 세션 예약
- 할 일 및 영업 파이프라인
- 참여 분석

## 자주 묻는 질문

**Q: 연락처 요약 에이전트를 사용하기 전에 설정해야 할 것이 있나요?**
설정이 필요 없습니다. Ask AI에 접근하여 바로 질문하거나 요약을 요청하면 됩니다.

**Q: 연락처 요약 에이전트에게 어떤 유형의 질문을 할 수 있나요?**
연락처의 커뮤니케이션, 결제, 할 일, 예약 등에 대한 요약이나 구체적인 세부사항에 대해 질문할 수 있습니다.

**Q: 연락처 요약 에이전트가 모든 연락처와 작동하나요?**
에이전트는 데이터가 있는 경우 Hyperclass 하위 계정의 모든 연락처 레코드와 작동합니다.

**Q: 에이전트는 어떻게 데이터 정확성을 보장하나요?**
연락처 요약 에이전트는 Hyperclass 시스템에서 실시간 데이터를 가져와 요약이 항상 최신 상태를 유지하도록 합니다.

**Q: 연락처 요약 에이전트를 사용할 때 내 데이터가 안전한가요?**
네, 모든 데이터는 표준 개인정보보호 및 보안 프로토콜에 따라 Hyperclass 플랫폼 내에서 안전하게 처리됩니다.
