## 모바일 대화 AI란?

모바일 대화 AI는 AI 기반 고객 상호작용의 강력함을 손끝에서 활용할 수 있게 해줍니다. 자동 파일럿(Autopilot)과 제안 모드(Suggestive) 두 가지 지능형 모드를 통해 더 빠르게 응답하고, 일관성을 유지하며, 책상에서 벗어나 있을 때도 고객 참여를 지속할 수 있어요.

모바일 연동으로 어디에 있든 효율적으로 리드와 소통할 수 있고, 대화 중에서도 AI 동작을 완전히 제어할 수 있습니다.

## 모바일 대화 AI의 주요 장점

모바일 대화 AI는 응답성과 제어력을 향상시키면서 수동 작업을 줄여줍니다.

- 자동 파일럿 모드로 부재 중에도 자동 응답
- 제안 모드에서 스마트 답변을 즉시 확인하고 커스터마이징
- 스마트폰에서 AI 사용을 빠르게 일시정지, 재시작, 예약
- 왓츠앱, 페이스북 메신저, 인스타그램 DM에서 일관성 유지
- 모바일에서 AI 활성화 모니터링으로 사용량 기반 요금제 관리
- 응답 품질을 타협하지 않으면서 고객 경험 개선

## 모바일에서 대화 AI 활성화 및 사용법

모바일에서 AI를 사용하려면 데스크톱에서 한 번만 설정하면 됩니다. 구성 완료 후에는 모바일 앱 대화에서 직접 기능을 관리하고 활용할 수 있어요.

### 1단계: 데스크톱에서 설정 (1회만)

- Hyperclass 데스크톱 대시보드에 로그인
- Settings(설정) → Conversation AI로 이동
- 원하는 모드 선택: 자동 파일럿 또는 제안 모드
- 자동 파일럿 사용 시 지연 시간 설정
- FAQ 및/또는 웹 크롤러를 사용하여 AI 학습
- 설정 저장

### 2단계: 모바일 대화에서 AI 사용

- Hyperclass 또는 LeadConnector 모바일 앱 열기
- 왓츠앱, 페이스북 메신저, 인스타그램 대화로 이동
- 상호작용을 시작하여 AI 지원 확인

### 3단계: AI 동작 관찰

- 자동 파일럿 모드에서는 설정된 지연 시간 후 AI가 자동으로 답변 전송
- 제안 모드에서는 메시지 입력창을 터치하면 제안 답변이 나타남

## 자동 파일럿 모드

자동 파일럿 모드는 대화 AI가 미리 설정된 지연 시간 후에 자동으로 대신 응답하게 해줍니다. 업무 시간 외, 휴식 시간, 또는 대량 문의 시기에 대화를 관리하는 데 이상적이에요.

작동 방식:

- 사용자 입력 없이 AI가 답변 전송
- 지연 설정은 데스크톱 설정에서 정의
- 모든 응답이 대화 기록에 저장

모바일이나 데스크톱에서 벗어나 있을 때도 응답성을 유지하고 싶다면 이 모드를 사용하세요.

## 제안 모드

제안 모드는 대화를 열고 답장 입력창을 터치하는 순간 실시간 답변 제안을 제공합니다.

작동 방식:

- AI가 마지막 메시지를 기반으로 관련성 높은 제안 생성
- 응답을 선택하고, 편집하고, 전송 가능
- 사용자가 완전히 제어 — 자동으로 전송되지 않음

AI 지원을 받으면서도 완전한 통제권을 유지하고 싶은 사용자에게 이상적인 모드입니다.

## 모바일에서 봇 설정 관리

모바일 앱의 모든 대화 화면에서 직접 대화 AI를 켜거나 끌 수 있습니다. 이를 통해 컴퓨터에서 벗어나 있을 때도 AI 동작을 반응적으로 관리할 수 있어요.

AI 설정 접근 방법:

- 대화 열기
- 상단 우측 모서리의 점 세 개 메뉴(⋯) 터치
- Edit AI Settings(AI 설정 편집) 선택
- 설정 화면에서 다음이 가능:
  - AI 켜기/끄기
  - AI 상태 확인 (활성, 일시정지, 재활성화 중)
  - 재활성화 예약 (예: "2시간 후 다시 켜기")

이를 통해 고객 상호작용을 더 잘 제어하고 AI 사용량을 효과적으로 관리할 수 있어요. 특히 사용량 기반 요금제를 사용하는 경우에 유용합니다.

## 자주 묻는 질문

**Q: 모바일에서만으로 대화 AI를 완전히 설정할 수 있나요?**
A: 아니요. AI 학습과 모드 선택을 포함한 핵심 설정은 데스크톱 대시보드에서 완료해야 합니다.

**Q: 모바일에서 지원되는 채널은 무엇인가요?**
A: 대화 AI는 왓츠앱, 페이스북 메신저, 인스타그램 DM을 지원합니다. 단, 이러한 채널이 계정에 연동되어 있어야 해요.

**Q: 모바일에서 자동 파일럿과 제안 모드 간 전환이 가능한가요?**
A: 아니요. 모드 전환은 데스크톱 설정 페이지에서 해야 합니다.

**Q: 모바일에서의 AI 사용이 요금에 영향을 주나요?**
A: 네. 데스크톱이든 모바일이든 모든 AI 상호작용이 사용량 기반 요금에 포함됩니다.

**Q: 이 기능을 모든 Hyperclass 사용자가 사용할 수 있나요?**
A: 이 기능은 현재 공개 베타 단계이며 Hyperclass 및 LeadConnector 모바일 앱의 모든 사용자가 이용할 수 있습니다.
