# 소셜 플래너의 콘텐츠 AI

Hyperclass의 새로운 소셜 플래너(Social Planner) 가이드형 콘텐츠 AI(Content AI) 질문 기능으로 플랫폼별 고품질 소셜 미디어 콘텐츠를 더 빠르게 만들어 보세요. 이 업데이트는 작업 흐름을 간소화하고, 브랜드 일관성을 보장하며, 각 플랫폼에 맞춤화된 더 매력적인 게시물을 생성할 수 있도록 도와줍니다.

## **콘텐츠 AI 가이드형 콘텐츠 질문 기능이란?**

소셜 플래너의 콘텐츠 AI 가이드형 콘텐츠 질문 기능은 적응형 콘텐츠 생성 경험을 제공합니다. 콘텐츠 AI로 새 게시물을 만들 때, 폼이 선택한 소셜 플랫폼(페이스북, 인스타그램, 링크드인 등)과 콘텐츠 유형(게시물, 릴 등)에 따라 질문을 동적으로 조정합니다. 이를 통해 가장 관련성 높은 질문에만 답하면 되므로, 각 채널에 맞는 타겟팅된 브랜드 콘텐츠를 더 쉽게 제작할 수 있습니다.

---

## 콘텐츠 AI 가이드형 콘텐츠 질문의 주요 장점

- **플랫폼별 맞춤 가이드:** 선택한 플랫폼과 콘텐츠 유형에 필요한 질문만 표시하여 혼란을 줄이고 명확성을 높입니다.

- **브랜드 보이스 일관성:** 브랜드 보이스 필드가 자동으로 채워져 메시징을 브랜드에 맞게 유지하면서도 필요에 따라 커스터마이징할 수 있습니다.

- **간소화된 워크플로우:** 필수 필드는 항상 표시되고, 선택 질문은 접을 수 있는 메뉴에 정리하여 더 깔끔한 인터페이스를 제공합니다.

- **명확한 콘텐츠 목표:** 가이드 질문이 각 게시물의 메시지, 콘텐츠 목표, CTA(행동 유도)를 명확히 정의할 수 있도록 도와줍니다.

- **향상된 참여도:** 더 관련성 높고 맞춤화된 게시물로 오디언스 상호작용이 개선되고 수동 편집이 줄어듭니다.

- **시간 절약:** AI가 생성한 콘텐츠를 수정하는 시간을 줄이고 오디언스와 소통하는 시간을 늘릴 수 있습니다.

## **콘텐츠 AI 가이드형 콘텐츠 질문 설정 방법**

소셜 플래너의 새로운 가이드형 콘텐츠 AI 질문을 설정하고 사용하는 것은 간단하며, 각 플랫폼에 최적화된 게시물을 만들 수 있습니다.

### 1단계: 소셜 플래너로 이동

Hyperclass 대시보드의 **Marketing(마케팅)** 섹션으로 이동하여 **Social Planner(소셜 플래너)**를 선택합니다.

![소셜 플래너 메뉴 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155049682816/original/lXCSjOT-9AXKgJdgY9t9en7s7EnyFYQ0WQ.png?1752175542)

### 2단계: 콘텐츠 AI로 새 게시물 만들기

소셜 계정 연결 후, **+ New Post(새 게시물)** 버튼을 클릭합니다. 드롭다운 메뉴에서 **Content AI(콘텐츠 AI)**를 선택하여 가이드형 콘텐츠 생성 환경을 시작합니다.

![콘텐츠 AI 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155049683051/original/x3sUYal14ox-6_jmQnRkVOzFArN63ugzUw.png?1752175882)

### 3단계: 플랫폼 및 콘텐츠 유형 선택

원하는 소셜 미디어 플랫폼(예: 페이스북, 인스타그램, 링크드인)과 콘텐츠 유형(게시물, 릴 등)을 선택합니다.

![플랫폼 및 콘텐츠 유형 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155049683096/original/xFvjPMBVZsUJYDuRR4AzD_nujZwNIxSBWg.png?1752175977)

### 4단계: 브랜드 보이스 필드 검토

가능한 경우 브랜드 보이스 필드가 자동으로 채워집니다. 이를 수정하거나 콘텐츠를 더욱 커스터마이징하기 위해 추가 세부사항을 입력할 수 있습니다.

![브랜드 보이스 필드 검토](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155049683153/original/DVw0G3BnKmNZFj1zdv2IGExcyxQTlOwNmw.png?1752176124)

### 5단계: 가이드 질문에 답하기

항상 표시되는 필수 필드를 작성합니다. 게시물 세부사항을 더 세밀하게 제어하고 싶다면 접을 수 있는 메뉴를 열어서 선택 질문에 답할 수 있습니다.

![가이드 질문에 답하기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155049683238/original/ZAVjpoNEENDzhD5vKtZevQ6RqLaWjFZh-A.png?1752176354)

### 6단계: 콘텐츠 생성 및 검토

게시물 생성 버튼을 클릭합니다. AI가 생성한 콘텐츠를 검토하고, 필요한 최종 수정을 한 후 예약하거나 발행합니다.

![콘텐츠 생성 및 검토](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155049683225/original/PVeJBmpCNUrBHejeDu8DyolteaGJvzv_dA.png?1752176303)

## **자주 묻는 질문**

**Q: 소셜 플래너의 모든 게시물 유형에서 가이드형 콘텐츠 AI 환경을 사용할 수 있나요?**

현재 가이드형 콘텐츠 AI 환경은 새 게시물 드롭다운에서 "Content AI" 옵션을 선택했을 때만 사용할 수 있습니다.

**Q: 각 게시물에 대해 브랜드 보이스 필드를 커스터마이징할 수 있나요?**

네, 브랜드 보이스 필드는 자동으로 채워지지만 각 개별 게시물에 대해 수정하거나 편집할 수 있습니다.

**Q: 선택 질문을 건너뛰면 어떻게 되나요?**

선택 질문은 필수가 아니지만, 답변하면 더 맞춤화되고 효과적인 콘텐츠를 생성하는 데 도움이 됩니다.

**Q: AI가 생성한 콘텐츠가 각 플랫폼마다 다르게 나오나요?**

네, 가이드 질문과 AI 출력은 선택한 플랫폼과 콘텐츠 유형에 맞춤화되어 최대한 관련성 높은 콘텐츠를 제공합니다.

**Q: 여러 플랫폼에 동시에 게시물을 예약하는 데 이 기능을 사용할 수 있나요?**

콘텐츠 AI를 사용하여 플랫폼별 콘텐츠를 만들 수 있지만, 각 게시물은 해당 플랫폼에 대해 개별적으로 생성됩니다.

**Q: 가이드형 콘텐츠 AI 기능을 사용하는 데 추가 비용이 있나요?**

가이드형 콘텐츠 AI 환경은 기존 콘텐츠 AI 제공의 일부입니다. 표준 콘텐츠 AI 사용 한도와 가격 정책이 적용됩니다.

**Q: 생성된 콘텐츠에 대한 피드백을 어떻게 제공할 수 있나요?**

콘텐츠 생성 후 소셜 플래너 인터페이스 내에서 직접 피드백을 공유하거나 Hyperclass 지원 채널을 통해 제공할 수 있습니다.

**Q: 이전 콘텐츠 생성 환경으로 되돌릴 수 있나요?**

새로운 가이드형 환경은 현재 콘텐츠 AI 옵션을 통해서만 사용할 수 있습니다. 다른 게시물 생성 방법들은 그대로 유지됩니다.
