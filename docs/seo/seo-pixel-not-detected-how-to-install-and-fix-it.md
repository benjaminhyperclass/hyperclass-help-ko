# SEO 픽셀이 감지되지 않음 - 설치 및 수정 방법

SEO 픽셀을 사용하면 플랫폼 외부에 호스팅된 웹사이트에 AI가 생성한 SEO 수정 사항(제목 태그, 메타 설명, 이미지 alt 텍스트 등)을 자동으로 적용할 수 있습니다. 이를 통해 외부 사이트 코드를 매번 수동으로 편집하지 않고도 검색 가시성을 개선할 수 있습니다.

플랫폼 내부에 호스팅된 웹사이트의 경우 SEO 픽셀을 설치하지 않고도 SEO 수정 사항을 직접 적용할 수 있습니다.

---

## SEO 픽셀이란?

SEO 픽셀은 SEO 도구 내에서 생성되는 작은 JavaScript 스니펫입니다. 외부 웹사이트에 설치하면 시스템이 해당 사이트에 AI 생성 SEO 업데이트를 직접 적용할 수 있게 됩니다.

이러한 업데이트에는 다음이 포함될 수 있습니다:

- 제목 태그
- 메타 설명
- 이미지 alt 텍스트

픽셀은 외부 웹사이트와 SEO 도구 간의 안전한 브리지 역할을 하여 최적화가 자동으로 적용되도록 합니다.

---

## 누가 SEO 픽셀을 사용해야 하나요?

다음의 경우 SEO 픽셀을 사용하세요:

- 웹사이트가 외부에 호스팅된 경우(예: WordPress, Wix, Shopify 또는 커스텀 구축 사이트)
- SEO 도구 내에서 사이트가 연결되어 있지만 **"SEO 픽셀이 감지되지 않음"**이 표시되는 경우
- AI 생성 SEO 업데이트를 외부 사이트에 자동으로 적용하고 싶은 경우

플랫폼 내부에 웹사이트가 호스팅된 경우에는 SEO 픽셀이 필요하지 않습니다.

---

## SEO 픽셀 작동 방식

1. 웹사이트에 대한 SEO 도구를 엽니다.
2. 웹사이트가 외부에 있거나 픽셀이 누락된 경우 **SEO 픽셀이 감지되지 않음**이 표시됩니다.
3. **설치 단계 보기(View Installation Steps)**를 클릭합니다.
4. 생성된 코드 스니펫을 복사합니다.
5. 웹사이트의 `<body>` 섹션에 스니펫을 붙여넣습니다.
6. SEO 도구로 돌아가서 **설치 확인(Verify Installation)**을 클릭합니다.
7. 확인이 완료되면 **문제 수정(Fix Issue)**을 사용하여 제안된 SEO 업데이트를 자동으로 적용합니다.

---

## SEO 픽셀 설치

픽셀은 `</body>` 태그가 닫히기 전에 웹사이트의 `<body>` 섹션 내부에 배치되어야 합니다.

### WordPress 설치

1. WordPress 관리자 대시보드에 로그인합니다.
2. Body에 코드를 삽입할 수 있는 플러그인을 설치합니다(예: header/body/footer 삽입 플러그인).
3. SEO 도구에서 **설치 단계 보기**를 클릭합니다.
4. SEO 픽셀 스니펫을 복사합니다.
5. 플러그인의 **Body** 섹션에 스니펫을 붙여넣습니다.
6. 변경 사항을 저장합니다.

![WordPress SEO 픽셀 설치](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155066321075/original/hIRLjDcGq4j5QnicSOl9eI24pSIkLU2ryg.png?1772743540)

### Wix 설치

1. Wix 대시보드에 로그인합니다.
2. **설정(Settings) → 커스텀 코드(Custom Code)**로 이동합니다.
3. **코드 추가(Add Code)**를 클릭합니다.
4. **Body**에 배치를 선택합니다.
5. SEO 픽셀 스니펫을 붙여넣습니다.
6. 웹사이트를 저장하고 게시합니다.

---

## 커스텀 또는 기타 웹사이트

1. 웹사이트의 HTML 템플릿 파일이나 커스텀 코드 설정을 엽니다.
2. `</body>` 태그가 닫히기 바로 전에 SEO 픽셀 스니펫을 붙여넣습니다.
3. 변경 사항을 저장합니다.
4. 웹사이트를 게시합니다.

---

## 설치 확인

스니펫을 설치한 후:

1. SEO 도구로 돌아갑니다.
2. 웹사이트를 다시 클릭합니다(새로운 감지 확인이 실행됩니다).
3. **설치 확인**을 클릭합니다.

확인이 실패하는 경우 다음을 확인하세요:

- 스니펫이 `<body>` 섹션 내부에 배치되었는지
- 사이트가 저장되고 게시되었는지
- 캐싱이나 CDN 시스템이 클리어되었는지
- 스니펫이 수정되지 않았는지

---

## SEO 수정 사항 적용

설치가 확인되면:

1. 수정 카테고리를 엽니다(예: 제목 태그 또는 메타 설명).
2. **제안된 수정(Suggested Fix)** 열을 검토합니다.
3. **문제 수정**을 클릭하여 업데이트를 적용합니다.
4. **수정되지 않음(Not Fixed)** 및 **수정됨(Fixed)**과 같은 필터를 사용하여 업데이트를 관리합니다.

## 자주 묻는 질문

#### 웹사이트가 플랫폼 내부에 호스팅되어 있다면 SEO 픽셀이 필요한가요?

아니요. 플랫폼 내부에 호스팅된 웹사이트는 픽셀을 설치하지 않고도 SEO 업데이트를 직접 받을 수 있습니다.

#### SEO 픽셀이 웹사이트를 느리게 만드나요?

스니펫은 가볍고 성능에 미치는 영향을 최소화하도록 설계되었습니다.

#### 웹사이트를 다른 계정이나 워크스페이스로 이동하면 어떻게 되나요?

새 위치에 대해 새로운 SEO 픽셀 스니펫을 생성하고 설치해야 합니다.

#### 수정 사항이 적용된 후 되돌릴 수 있나요?

예. 웹사이트 편집기 내에서 수동으로 업데이트하거나 변경 사항을 재정의할 수 있습니다.

#### SEO 픽셀을 설치했는데 여전히 "SEO 픽셀이 감지되지 않음"이 표시되는 이유는?

다음과 같은 경우에 발생할 수 있습니다:

- 스니펫이 `<body>` 대신 `<head>`에 배치된 경우
- 사이트가 다시 게시되지 않은 경우
- 캐싱/CDN이 이전 버전을 제공하는 경우
- 스니펫이 변경되거나 일부만 복사된 경우

---

## 주요 참고 사항

- SEO 픽셀 스니펫은 웹사이트별로 고유합니다.
- SEO 도구 내에서 생성됩니다.
- 픽셀은 SEO 관련 변경 사항(제목, 메타 설명, alt 텍스트)만 적용합니다.
- 레이아웃, 구조 또는 디자인은 변경하지 않습니다.
- 스니펫은 수정해서는 안 됩니다.

---

## 다음 단계

1. SEO 도구를 엽니다.
2. **SEO 픽셀이 감지되지 않음**이 표시되면 **설치 단계 보기**를 클릭합니다.
3. 웹사이트에 스니펫을 설치합니다.
4. SEO 도구로 돌아가서 **설치 확인**을 클릭합니다.
5. **문제 수정**을 사용하여 제안된 SEO 업데이트를 적용합니다.

---

## 도움이 필요하신가요?

SEO 픽셀이 확인되지 않거나 업데이트가 제대로 적용되지 않는 경우:

- 스니펫이 `<body>` 섹션 내부에 있는지 확인하세요.
- 웹사이트가 게시되었는지 확인하세요.
- 캐시나 CDN 레이어를 클리어하세요.
- 포맷 문제를 방지하기 위해 SEO 도구에서 스니펫을 다시 복사하세요.
- 차단된 스크립트에 대한 브라우저 콘솔 오류를 확인하세요.

문제가 지속되면 다음 정보를 수집하여 지원팀에 문의하세요:

- 웹사이트 URL
- 스니펫 배치 스크린샷
- SEO 도구 상태 스크린샷
