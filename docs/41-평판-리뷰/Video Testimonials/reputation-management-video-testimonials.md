# 평판 관리 - 비디오 추천

비디오 추천(Video Testimonials) 기능을 사용하면 고객의 진짜 비디오 리뷰를 수집하고 웹사이트에 직접 게시할 수 있습니다. 브랜딩된 비디오 수집기를 만들고, 이메일, SMS, 왓츠앱으로 요청을 보내며, 응답을 관리하고, 리뷰 위젯으로 표시할 수 있습니다. 이 가이드는 설정, 발송, 관리, 표시 모범 사례를 다룹니다.

## 평판 관리의 비디오 추천이란?

비디오 추천은 공유 가능한 브랜딩된 "**비디오 수집기**"를 사용해 고객 비디오 리뷰를 수집할 수 있는 평판 관리 기능입니다. 연락처는 데스크톱이나 모바일에서 앱 설치 없이 비디오를 녹화하거나 업로드할 수 있습니다. 수집된 비디오는 내부에서 검토한 후 리뷰 위젯으로 사이트에 노출할 수 있습니다.

## 비디오 추천의 주요 장점

- **진정성:** 실제 얼굴과 목소리가 텍스트 리뷰보다 훨씬 빠르게 신뢰를 쌓아줍니다.

- **간편한 수집:** 연락처가 최신 기기에서 앱 설치 없이 녹화하거나 업로드할 수 있습니다.

- **브랜드 제어:** 수집기 브랜딩(로고, 배경, 텍스트, 포인트 색상, 버튼 색상)을 브랜드 아이덴티티에 맞게 커스터마이징할 수 있습니다.

- **유연한 배포:** 직접 링크를 공유하거나 Hyperclass에서 이메일, SMS, 왓츠앱으로 발송할 수 있습니다.

- **중앙 집중식 관리:** 응답 탭에서 모든 제출 내용을 확인하고 다운로드할 수 있습니다.

- **간편한 노출:** 기존 리뷰 위젯에 비디오를 추가할 수 있으며, 레이아웃 옵션과 수집기별 필터링을 지원합니다.

- **성능 제한:** 각 비디오는 2분 30초로 제한되어 파일이 가볍고 시청하기 좋습니다.

## 비디오 수집기 만드는 방법

다음 단계에 따라 비디오 수집기를 만들어보세요.

### 1단계: 비디오 수집기 만들기

- 하위 계정에 로그인합니다.

- **평판 관리(Reputation)** → **비디오 추천(Video Testimonials)**으로 이동합니다.

![비디오 추천 메뉴](https://jumpshare.com/share/VqgfyloSbiBdNL11RcZz+/Screen+Shot+2025-11-27+at+7.32.45+PM.png)

- **+ 수집기 만들기** 또는 **수집기 만들기** 버튼을 클릭합니다.

![수집기 만들기 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155059385390/original/a5Bwg2mIkgGRZqYNdJCy7NyAU9PY0NlKMQ.jpeg?1764252759)

### 2단계: 환영 페이지 설정

- **공개 수집기 제목**을 추가합니다.

![공개 수집기 제목](https://jumpshare.com/share/fwLsUTx7DFKNM8vu61q5+/Screen+Shot+2025-11-27+at+7.50.19+PM.png)

- **대변인(Spokesperson)** 이미지 활성화 또는 **비활성화**: 회사의 친근한 얼굴로 수집기를 개인적으로 만들어보세요. 본인이나 동료의 사진을 사용할 수 있습니다.

- **대변인** 정보를 추가합니다.

![대변인 설정](https://jumpshare.com/share/hX0EPT7tDzFcttU03uqA+/Screen+Shot+2025-11-27+at+7.51.26+PM.png)

- 작동 방식 섹션 추가: 모든 것이 어떻게 작동하는지에 대한 간단한 설명입니다.

![작동 방식 섹션](https://jumpshare.com/share/1Ch6ZE8LQPzaONtWHEbk+/Screen+Shot+2025-11-27+at+7.52.46+PM.png)

- 환영 메시지 추가: 수집기 대변인의 짧은 메시지입니다.

![환영 메시지](https://jumpshare.com/share/fLTUjohlT2TIdYQTrNBG+/Screen+Shot+2025-11-27+at+7.53.47+PM.png)

### 3단계: 설문 페이지 설정

- 최대 **3개**의 질문을 추가합니다.

- **가이드 플로우**를 위해 순서를 정합니다.

![설문 질문 설정](https://jumpshare.com/share/hnMRa0ybKAaOGKTR3hkM+/Screen+Shot+2025-11-27+at+8.05.48+PM.png)

### 4단계: 귀속 페이지 설정

질문 후에는 응답자에게 자신을 확인하도록 요청하거나, 개인화된 링크로 미리 로드된 정보를 확인하도록 합니다.

- **제목**을 추가합니다.

- **메시지**를 추가합니다.

![귀속 페이지 설정](https://jumpshare.com/share/8W7duvySYxPLj8eVrKm8+/Screen+Shot+2025-11-27+at+8.09.19+PM.png)

### 5단계: 감사 페이지

- **제목**을 추가합니다.

- 메시지를 추가합니다.

- **대변인의 이메일**을 추가합니다.

![감사 페이지 설정](https://jumpshare.com/share/z9pJtDx1qCqfEebwiBkR+/Screen+Shot+2025-11-27+at+8.14.31+PM.png)

### 6단계: 로고 및 색상 설정

- 로고를 활성화하고 추가합니다.

- **로고 뒤 흰색 배경** 활성화 또는 **비활성화**를 선택합니다.

- **배경**, **텍스트**, **보조 텍스트**, **포인트**, **버튼** **색상**을 설정합니다.

- **저장**을 클릭합니다.

![로고 및 색상 설정](https://jumpshare.com/share/CXsWO5GWj4Qt9oEbecNf+/Screen+Shot+2025-11-27+at+8.25.04+PM.png)

- **미리보기**를 클릭하면 고객이 수집기를 열어 비디오 리뷰를 녹화할 때 보게 될 화면을 정확히 확인할 수 있습니다.

![미리보기 화면](https://jumpshare.com/share/cSGrkW7hCcD6hnMiwCoT+/Screen+Shot+2025-11-27+at+9.00.42+PM.png)

## 비디오 추천 공유 및 관리하는 방법

### 1. 링크로 공유

- **비디오 추천(Video Testimonials)** → **수집기(Collectors)**에서 **수집기를 찾습니다**.

- **링크 복사**를 클릭하고 **이메일, 채팅, 소셜, QR**로 공유합니다.

![링크 복사 및 공유](https://jumpshare.com/share/0XyIYWbhUeZvpNnjiIR4+/GIF+Recording+2025-11-27+at+9.41.51+PM.gif)

### 2. 비디오 리뷰 요청 발송(이메일/SMS/왓츠앱)

- **리뷰 요청 발송** 또는 수집기의 **공유 아이콘**을 클릭합니다.

![리뷰 요청 발송](https://jumpshare.com/share/T9DURcaVubhhe1bwin4T+/Screen+Shot+2025-11-27+at+9.43.15+PM.png)

- **연락처**를 선택합니다.

- **이메일**, SMS, 또는 **왓츠앱**을 선택합니다.

- **비디오 링크 재정의**를 사용해 포함할 수집기를 선택합니다.

- 기본 이메일/SMS/왓츠앱 템플릿 또는 미리 만든 커스텀 템플릿을 선택합니다.

- **리뷰 요청 발송**을 클릭합니다.

![리뷰 요청 발송 설정](https://jumpshare.com/share/lWRSNbGGsMwhqaikRmEv+/Screen+Shot+2025-11-27+at+9.51.38+PM.png)

## 응답 관리 및 다운로드하는 방법

- **평판 관리(Reputation)** → **비디오 추천(Video Testimonials)** → **응답(Responses)**을 엽니다.

- **임의의 제출물**을 클릭해서 비디오를 확인합니다.

- **다운로드**를 클릭해서 원본 파일을 로컬에 저장합니다.

![응답 관리 및 다운로드](https://jumpshare.com/share/5MaLuBcvjHZ4kGMdJ2n5+/GIF+Recording+2025-11-27+at+9.56.04+PM.gif)

## 리뷰 위젯에서 비디오 표시하는 방법

- 평판 관리(Reputation) > **위젯(Widgets)**으로 이동합니다.

- 리뷰 위젯을 만들거나 **편집**합니다.

- **리뷰 유형(Review Type)**을 **비디오 추천(Video Testimonials)**으로 선택합니다.

- **비디오 추천** 소스로 표시하려는 수집기를 선택합니다.

- **위젯 유형**, **리뷰 개수**, **비디오 링크 재정의**를 선택하고 **powered by** 표시 **활성화** 또는 비활성화를 설정합니다.

- 저장을 클릭합니다.

![리뷰 위젯에서 비디오 표시](https://jumpshare.com/share/MajrGgax40XeaxAopVog+/GIF+Recording+2025-11-27+at+9.58.58+PM.gif)

## 자주 묻는 질문

**Q: 고객이 비디오를 녹화하려면 앱을 설치해야 하나요?**

아니요. 수집기 링크를 사용해서 데스크톱이나 모바일에서 직접 녹화하거나 업로드할 수 있습니다.

**Q: 최대 비디오 길이는 얼마나 되나요?**

제출당 최대 **2분 30초**입니다.

**Q: 수집기에 몇 개의 질문을 추가할 수 있나요?**

설문 단계에서 최대 3개의 질문을 추가할 수 있습니다.

**Q: 수집기를 여러 개 만들 수 있나요?**

네. 로케이션당 무제한으로 수집기를 만들 수 있고, 기존 수집기를 복사할 수도 있습니다.

**Q: 제출된 비디오는 어디에서 확인할 수 있나요?**

평판 관리 → 비디오 추천 → 응답으로 이동해서 확인하고 다운로드할 수 있습니다.

**Q: 사이트에 특정 비디오만 표시할 수 있나요?**

네. 리뷰 위젯의 수집기별 필터를 사용해서 특정 수집기의 비디오만 표시할 수 있습니다.

**Q: 수집기 링크 발송을 지원하는 채널은 무엇인가요?**

리뷰 요청 발송을 통해 이메일, SMS, 왓츠앱을 지원합니다.
