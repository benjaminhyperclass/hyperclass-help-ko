# 여러 리뷰 플랫폼 연동으로 리뷰 관리 및 모니터링하기

다양한 플랫폼의 리뷰를 관리하고 응답하는 것이 비즈니스에 얼마나 중요한지 잘 알고 있습니다. 이를 더 쉽게 만들기 위해, 여러 리뷰 플랫폼을 계정에 간단하게 연동할 수 있도록 했습니다. 이 가이드에서는 Yelp, TrustPilot, Google 등 다양한 리뷰 플랫폼을 연결하여 리뷰를 중앙에서 관리하는 방법을 안내해드립니다.

## 리뷰 플랫폼 연동의 주요 장점

- **중앙 관리**: 여러 플랫폼의 모든 리뷰를 한 곳에서 확인할 수 있습니다.
- **효율적인 응답**: 각 플랫폼에 개별적으로 로그인할 필요 없이 여러 플랫폼의 리뷰에 빠르게 응답할 수 있습니다.
- **향상된 평판 모니터링**: 다양한 사이트에서 브랜드 평판을 주시하고 문제를 신속하게 해결할 수 있습니다.
- **간소화된 작업 흐름**: 리뷰 수집 과정을 자동화하여 시간을 절약하고, 각 사이트를 수동으로 확인할 필요가 없습니다.

## 리뷰 플랫폼 연동 방법

### 1단계: 연동(Integrations) 메뉴로 이동하기

- 좌측 사이드바에서 **Reputation(평판 관리)** 메뉴로 이동합니다.

- 평판 관리 페이지 상단 메뉴에서 **Settings(설정)** 옵션을 클릭합니다. (일반 설정이 아닌 평판 설정 페이지로 이동됩니다)

- 설정 메뉴에서 **Integrations(연동)** 탭을 찾아 클릭합니다.

![연동 설정 페이지](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155045977731/original/OG3iL2sVuaDkt_LV30SRg5MNP00EpauYZw.png?1746116258)

### 2단계: 연동 설정하기

- 연동 페이지에서 각 플랫폼별로 페이지 URL을 입력하는 옵션을 볼 수 있습니다. 연동하려는 리뷰 플랫폼의 구체적인 URL을 입력하세요. 이 URL이 귀하의 비즈니스 리뷰 페이지(예: Yelp 페이지나 Google 비즈니스 프로필)를 정확히 가리키는지 확인하세요.

- URL을 입력한 후 **Integrate(연동)** 버튼을 클릭합니다. 플랫폼에서 자동으로 해당 플랫폼의 최신 리뷰를 가져와서 대시보드에 표시합니다.

- 다른 리뷰 플랫폼도 동일한 단계를 반복하여 연동할 수 있습니다. 완료하면 모든 리뷰가 한 곳에 표시되어 다양한 소스의 리뷰를 더 쉽게 모니터링하고 관리하며 응답할 수 있습니다.

## 문제해결

리뷰 플랫폼 연동 중 문제가 발생하면 다음을 시도해보세요:

- **URL 확인**: 페이지 URL이 올바르고 해당 플랫폼의 비즈니스 리뷰 페이지로 직접 연결되는지 확인하세요.
- **액세스 권한 확인**: 선택한 플랫폼에서 리뷰를 가져올 수 있는 필요한 권한이 있는지 확인하세요.
- **캐시 삭제**: 때로는 브라우저 캐시를 삭제하면 연동 문제가 해결될 수 있습니다.
- **고객지원**: 문제가 지속되면 고객지원팀에 문의하시면 모든 것이 제대로 작동하도록 도와드리겠습니다.

여러 리뷰 플랫폼을 연동하면 온라인 평판 관리를 간소화하고, 시간과 노력을 절약하면서 고객 피드백에 신속하게 응답할 수 있는 능력을 향상시킬 수 있습니다.
