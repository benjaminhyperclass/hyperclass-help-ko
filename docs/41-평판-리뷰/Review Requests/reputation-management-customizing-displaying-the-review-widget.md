### 리뷰 위젯 커스터마이징 전 알아두어야 할 사항

- 제공되는 코드를 삽입하여 외부 플랫폼에서도 리뷰 위젯을 사용할 수 있습니다
- 퍼널과 웹사이트에서는 'Review Widget Element(리뷰 위젯 요소)'를 사용하여 Facebook과 Google 리뷰를 표시할 수 있습니다 (아래 동영상 1분 30초 부분 참조)
- 리뷰는 Reputation(평판) 탭에서 확인할 수 있습니다 - [평판 - Reviews 탭 알아보기](reputation-breaking-down-the-reviews-tab.md)

## 외부 플랫폼에 리뷰 위젯 삽입하는 방법

하위 계정(Sub-Account) 설정으로 이동 > Reputation Management(평판 관리) > 아래로 스크롤하여 'Copy Code(코드 복사)' 버튼 클릭 > 외부 플랫폼에 코드 붙여넣기
