# 평판 관리 - 리뷰 탭 살펴보기

CRM 내에서 빠르게 리뷰를 필터링하고 바로 응답하는 방법을 찾고 계시다면 여기가 바로 맞는 곳입니다. Reputation(평판) > Reviews(리뷰) 탭에서 리뷰를 필터링하고 응답할 수 있는 여러 옵션들을 확인하실 수 있어요.

이 가이드에서는 Google과 Facebook 리뷰를 필터링하고 응답하는 방법을 알아보겠습니다.

## 리뷰 탭 살펴보기

왼쪽 메뉴에서 Reputation(평판) > Reviews(리뷰)를 클릭해주세요.
