# 평판 관리: Hyperclass의 경쟁사 분석

Hyperclass의 평판 관리(Reputation Management)에 새로 추가된 경쟁사 분석(Competitor Analysis) 기능으로 경쟁 우위를 확보하세요. Google 및 주요 플랫폼의 실시간 리뷰 데이터를 활용하여 고객의 온라인 평판을 최대 3개 경쟁사와 손쉽게 비교할 수 있습니다.

## 평판 관리의 경쟁사 분석이란?

경쟁사 분석은 Hyperclass 평판 관리 도구의 강력한 기능으로, 에이전시와 비즈니스가 온라인 평판 지표를 최대 3개 경쟁사와 비교할 수 있게 해줍니다. 여러 플랫폼의 리뷰 데이터를 자동으로 수집하고 분석하여, 시장에서 내 비즈니스의 위치를 종합적으로 파악하고 강점, 약점, 개선 기회를 식별할 수 있도록 도와줍니다.

## 경쟁사 분석의 주요 이점

경쟁사 분석의 장점을 이해하면 이 기능을 최대한 활용할 수 있습니다:

- **실시간 벤치마킹**: 최대 3개 경쟁사와 Google 별점 및 리뷰 수량을 즉시 비교
- **다중 플랫폼 인사이트**: Google 및 주요 리뷰 플랫폼의 평판 데이터에 접근하여 전체적인 시각 확보
- **실행 가능한 시각 자료**: 직관적인 차트로 성과 격차와 기회를 한눈에 파악
- **간편한 설정**: 경쟁사를 빠르게 검색하고 선택하면 모든 데이터가 자동으로 가져와져 정기 업데이트
- **향상된 고객 보고**: 인사이트를 내보내고 공유하여 고객 프레젠테이션 강화
- **시간 절약**: 수동 조사 불필요 - 필요한 모든 인사이트를 한 곳에서 확인
- **전략적 의사 결정**: 데이터 기반 인사이트로 평판 관리 전략 개선

## 경쟁사 분석 설정 방법

경쟁사 분석 설정은 간단하며, 최소한의 노력으로 정확하고 최신의 인사이트를 얻을 수 있습니다.

### 1단계: 경쟁사 분석으로 이동

- 계정 대시보드에서 **Reputation(평판 관리)** 옵션을 클릭합니다.
- **Competitor Analysis(경쟁사 분석)** 탭을 클릭하여 설정 프로세스를 시작합니다.

![경쟁사 분석 탭](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155047141343/original/n9vsnI1mdNxdgR35Dp8ffB4-cSnf1dnJTw.png?1748011230)

### 2단계: 내 비즈니스 추가

- 경쟁사 분석 탭 하단의 **+ Add Your Business(+ 내 비즈니스 추가)** 버튼을 클릭합니다.
- 위치 권한을 허용합니다.
- 내 비즈니스를 검색하고 올바른 항목을 선택합니다.
- 검색창 옆의 **+ Add Your Business(+ 내 비즈니스 추가)** 버튼을 클릭합니다.

![비즈니스 추가 과정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155047141618/original/AN4VrpqWCePTth1K02vX3kqUFn62TMclWg.gif?1748011619)

### 3단계: 경쟁사 추가

- **+ Add Competitor(+ 경쟁사 추가)** 버튼을 클릭합니다. (**최대 3개** 경쟁사 추가 가능)
- 경쟁사 비즈니스를 검색하고 올바른 항목을 선택합니다.
- 검색창 옆의 **+ Add Competitor(+ 경쟁사 추가)** 버튼을 클릭합니다.

![경쟁사 추가 과정](https://jumpshare.com/v/HaiSrHqfVnfrqcR7nlcx+/GIF+Recording+2025-05-23+at+8.27.34+PM.gif)

### 4단계: 소스별 평점 확인

- 아래로 스크롤하여 Google, Facebook, TripAdvisor 등의 "소스별 평점(Rating by Source)"을 확인합니다.
- 시스템이 자동으로 실시간 데이터를 가져와 별점, 리뷰 수량, 응답 시간 등을 표시합니다.

## 경쟁사 분석 추가 기능

더 깊은 인사이트와 전략적 활동을 위해 경쟁사 분석을 활용하는 다양한 방법을 살펴보세요:

### 실시간 벤치마킹

실시간 벤치마킹을 통해 언제든지 내 비즈니스의 평판 지표와 경쟁사를 비교할 수 있어, 시장 변화에 빠르게 대응할 수 있습니다.

- Google 별점을 나란히 비교
- 전체 리뷰 수량 트렌드 분석
- 고객 리뷰에 대한 평균 응답 시간 모니터링

### 다중 플랫폼 인사이트

다중 플랫폼 인사이트는 Google을 넘어서 경쟁 분석을 확장하여 평판 환경에 대한 더 넓은 관점을 제공합니다.

- 주요 리뷰 플랫폼(Facebook, Yelp 등)의 추가 데이터에 접근
- 경쟁사가 우수한 성과를 보이는 플랫폼 식별

### 간편한 설정 및 자동화

자동화된 설정 프로세스로 데이터 수집에 시간을 덜 쓰고 인사이트 활용에 더 많은 시간을 할애할 수 있습니다.

- 경쟁사 데이터가 자동으로 가져와져 정기적으로 새로고침됨
- 수동 데이터 입력 불필요 - 한 번 설정하면 자동 운영

## 자주 묻는 질문

**Q: 한 번에 몇 개의 경쟁사를 비교할 수 있나요?**
A: 최대 3개의 경쟁사와 내 비즈니스를 벤치마킹할 수 있습니다.

**Q: 어떤 리뷰 플랫폼을 지원하나요?**
A: Google이 완전히 지원되며, 더 넓은 인사이트를 위해 주요 리뷰 플랫폼들이 추가로 포함됩니다.

**Q: 경쟁사 데이터는 얼마나 자주 업데이트되나요?**
A: 항상 최신 인사이트를 제공하기 위해 데이터가 자동으로 가져와져 정기적으로 새로고침됩니다.

**Q: 수동으로 데이터를 입력해야 하나요?**
A: 아니요, 모든 경쟁사 데이터는 Hyperclass에서 자동으로 수집하고 업데이트합니다.

**Q: 설정 후 선택한 경쟁사를 변경할 수 있나요?**
A: 네, 언제든지 경쟁사 선택을 업데이트할 수 있습니다.
