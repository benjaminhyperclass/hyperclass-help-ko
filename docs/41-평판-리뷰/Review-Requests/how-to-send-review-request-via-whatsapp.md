# 왓츠앱으로 리뷰 요청 보내는 방법

왓츠앱을 통해 리뷰 요청을 보내는 것은 고객과 소통하고 소중한 피드백을 수집하는 강력한 방법입니다. 이 가이드는 계정에서 직접 리뷰 요청을 설정하고 전송하는 과정을 안내해드립니다.

## 1단계: 왓츠앱 연동 설정하기

리뷰 요청을 보내기 전에 먼저 왓츠앱이 계정에 연동되어 있는지 확인하세요:

- Settings(설정) > WhatsApp(왓츠앱)으로 이동하세요.
- 아직 구독하지 않았다면 **왓츠앱 구독**을 진행하세요.
- 안내에 따라 왓츠앱 비즈니스 번호를 인증하고 연결하세요.
- 연결이 완료되면 연동을 테스트하여 메시지가 성공적으로 전송되는지 확인하세요.

---

## 2단계: 리뷰 요청 메시지 커스터마이징

- 대시보드의 **WhatsApp Review Request(왓츠앱 리뷰 요청)** 섹션으로 이동하세요.
- 메시지를 커스터마이징하세요:
  - 비즈니스 이름을 포함하세요.
  - 친근한 행동 유도 문구를 추가하세요(예: "고객님의 소중한 피드백을 듣고 싶어요!").
  - 원하는 플랫폼(예: 구글 또는 페이스북)의 리뷰 링크를 첨부하세요.
- 향후 사용을 위해 템플릿을 저장하세요.

---

## 3단계: 리뷰 요청 보내기

### 개별 리뷰 요청

- 대시보드의 **Reviews(리뷰)** 섹션으로 이동하세요.
- **Send Review Request(리뷰 요청 보내기)**를 클릭하고 채널로 **WhatsApp(왓츠앱)**을 선택하세요.
- 요청을 보낼 고객을 선택하세요.
- 커스터마이징한 템플릿을 선택하고 검토한 후 **Send(보내기)**를 클릭하세요.

### 일괄 리뷰 요청

- **Reviews(리뷰)** 섹션에서 고객 정보(이름 및 왓츠앱 번호)가 포함된 CSV 파일을 업로드하세요.
- 필드를 올바르게 매핑하세요(예: Name → Customer Name, Phone → WhatsApp Number).
- **Send Bulk Review Requests(일괄 리뷰 요청 보내기)**를 선택하고 채널로 **WhatsApp(왓츠앱)**을 선택하세요.
- 미리보기를 검토하고 메시지를 확인하세요.

### 워크플로우 리뷰 요청

개별 또는 일괄 리뷰 요청 외에도 워크플로우를 사용하여 프로세스를 자동화할 수 있습니다. 이 방법은 특정 이벤트나 액션을 기반으로 리뷰 요청을 트리거하는 데 이상적입니다.

- **워크플로우 설정** - 대시보드의 **Workflows(워크플로우)** 섹션으로 이동하여 새 워크플로우를 생성하거나 기존 워크플로우를 편집하세요.
- **트리거 이벤트 추가** - 리뷰 요청을 트리거해야 하는 이벤트를 정의하세요. 예를 들어:
  - 구매 완료 후
- **"리뷰 요청 보내기" 액션 추가**
  워크플로우의 액션으로 **Send Review Request via WhatsApp(왓츠앱을 통한 리뷰 요청 보내기)**를 선택하세요.
- **템플릿 선택 또는 생성**
  사전 승인된 왓츠앱 템플릿을 선택하고 {고객명} 또는 {주문 ID} 같은 동적 플레이스홀더로 개인화하세요.
- **워크플로우 활성화**
  워크플로우를 저장하고 활성화하세요. 트리거 조건이 충족될 때마다 리뷰 요청이 자동으로 전송됩니다.

## 왓츠앱 리뷰 요청 전송 모범 사례

- **메시지 개인화**: 응답률을 높이기 위해 고객의 이름을 포함하여 메시지를 개인화하세요.
- **타이밍이 핵심**: 긍정적인 상호작용 직후에 요청을 보내세요.
- **규정 준수**: 왓츠앱 메시지를 보내기 전에 고객의 동의를 받았는지 확인하세요.
