# 플랫폼 간 리뷰 요청 균형 배분

리뷰 요청의 배분과 효과를 개선하기 위해 여러 리뷰 플랫폼에서 리뷰 요청 균형 배분 기능을 구현했습니다. 이를 통해 리뷰가 더욱 고르게 배분되어 비즈니스의 전반적인 온라인 평판과 가시성을 향상시킬 수 있습니다.

# 리뷰 요청 균형 배분이란?

리뷰 요청 균형 배분을 통해 구글과 페이스북 같은 여러 리뷰 플랫폼에 리뷰 요청을 분산시킬 수 있습니다. 사용자는 리다이렉션 비율을 설정하여 플랫폼 간 리뷰가 더욱 균등하게 분할되도록 할 수 있습니다.

**예시:**

- 리뷰 균형 배분을 처음 활성화하면 기본 비율은 **구글 50%**와 페이스북 50%입니다 *(두 페이지가 모두 연동된 경우)*.

- 비즈니스 요구사항에 맞게 리다이렉션 비율을 조정할 수 있습니다.

---

# 리뷰 요청 균형 배분이 중요한 이유는?

- **온라인 리뷰 다각화:** 여러 플랫폼에 리뷰를 분산시켜 다양한 온라인 공간에서 가시성과 신뢰성을 향상시킵니다.

- 평판 강화: 리뷰 균형 배분을 통해 한 플랫폼에만 과도하게 의존하지 않고 더욱 포괄적인 온라인 평판을 구축합니다.

- 커스터마이제이션 증가: 필요에 따라 리다이렉션 비율을 조정하여 리뷰 요청을 비즈니스 목표에 맞출 수 있습니다.

# 리뷰 요청 균형 배분 설정 방법

### **1단계:** 리뷰 균형 배분 활성화

- `Reputation(평판 관리) → Settings(설정) → Review Link(리뷰 링크)`로 이동하세요.

- **Enable Review Balancing(리뷰 균형 배분 활성화)** 옵션을 토글하세요.

![리뷰 균형 배분 활성화](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036266243/original/6nxoN7HJ26aDjgSqVOqGuapq-WgPgVbFRA.gif?1731089841)

### 2단계: GBP 페이지 설정(구글 비즈니스 프로필)

- `Reputation(평판 관리) → Settings(설정) → Review Link(리뷰 링크)` 하위에서 왼쪽 패널의 **Get Reviews on Google(구글에서 리뷰 받기)** 섹션을 찾으세요.

- 오른쪽 패널의 **Select Page(페이지 선택)** 드롭다운을 사용하여 리뷰 배분에 사용할 구글 비즈니스 프로필(GBP) 페이지를 선택하세요.

![GBP 페이지 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036266658/original/Hl0VzUJACsuZGbMtG1L5an5pyBwkyYNe9A.png?1731090471)

**중요:** 구글 비즈니스 프로필이나 페이스북 비즈니스 페이지를 아직 연동하지 않았다면 해당 채널에 대한 리뷰 배분을 설정할 수 없습니다.

GBP와 페이스북 연동을 설정하려면 `SETTINGS(설정) → INTEGRATIONS(연동)`으로 이동하세요.

### 3단계: 균형 배분 구성

- 리뷰 균형 배분을 활성화한 후 **Configure Balance(균형 배분 구성)**을 클릭하세요.

- 구글과 페이스북 간 리다이렉션 비율을 원하는 대로 조정하세요.

**예시:**

- 구글 70% & 페이스북 30%
- 또는 다른 맞춤 배분

![균형 배분 구성](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036266549/original/MhJlAnW-jOxKNXwDSMgLHdKajK13zPHo8w.png?1731090263)

![균형 배분 조정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036266551/original/BsBeq_Fi8QxePNZOVNdjpvxEHVVoOcyr-w.gif?1731090274)

### **4단계:** 설정 저장

- 원하는 설정을 구성한 후 변경 사항을 저장하세요.

# 자주 묻는 질문

**Q: 한 플랫폼만 연동되어 있으면 어떻게 되나요?**

한 플랫폼(예: 구글)만 연동되어 있으면 다른 플랫폼(예: 페이스북)이 연동될 때까지 모든 리뷰 요청이 자동으로 해당 플랫폼으로 리다이렉트됩니다.

**Q: 활성화한 후에도 리뷰 균형 배분을 비활성화할 수 있나요?**

네. 리뷰 균형 배분을 비활성화하려면:

- `Reputation(평판 관리) → Settings(설정) → Review Link(리뷰 링크)`로 이동하세요.

- **Enable Review Balancing(리뷰 균형 배분 활성화)** 옵션을 토글 해제하세요.

기능을 비활성화하면 모든 리뷰 요청이 기본 플랫폼(예: 구글)으로 리다이렉트됩니다(단일 연동 옵션인 경우).

**Q: 설정 후에 리다이렉션 비율을 편집할 수 있나요?**

네. 언제든지 다음 방법으로 비율을 조정할 수 있습니다:

- `Reputation(평판 관리) → Settings(설정) → Review Link(리뷰 링크)`로 이동하세요.

- **Configure Balance(균형 배분 구성)**을 클릭하고 비율을 수정하세요.

**Q: 기능을 활성화한 후 균형 배분을 구성하지 않으면 어떻게 되나요?**

균형 배분을 구성하지 않으면 시스템이 구글과 페이스북 간에 기본적으로 50%-50%로 분할합니다(두 플랫폼이 모두 연동된 경우).

**Q: 현재 리뷰 배분 상황을 어떻게 알 수 있나요?**

**Configure Balance(균형 배분 구성)** 모달에서 현재 배분 비율을 표시하므로 필요에 따라 모니터링하고 조정할 수 있습니다.

**Q: 플랫폼별 리뷰 성과에 대한 분석 기능이 있나요?**

리뷰 균형 배분은 리다이렉션에 중점을 두지만, 성과와 플랫폼별 생성된 리뷰 수에 대한 분석은 **Reputation(평판 관리)** 모듈에서 확인할 수 있습니다.
