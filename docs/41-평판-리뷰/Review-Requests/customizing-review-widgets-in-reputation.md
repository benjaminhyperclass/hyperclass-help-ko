# 리뷰 위젯이란?

리뷰 위젯(Review Widget)을 통해 실제 고객 피드백을 웹사이트에 직접 표시하여 즉각적인 신뢰와 사회적 증명을 구축할 수 있습니다. 구글과 페이스북 같은 플랫폼의 리뷰를 지원하며, 이들을 하나의 매끄럽고 브랜드화된 디스플레이로 결합할 수 있습니다.

위젯의 레이아웃, 스타일, 콘텐츠를 웹사이트 디자인에 맞게 쉽게 커스터마이징할 수 있습니다. 추가하면 새로운 리뷰가 자동으로 업데이트되어, 수동 작업 없이도 방문자에게 신선하고 신뢰할 수 있는 추천사를 제공합니다.

---

## 리뷰 위젯의 주요 장점

웹사이트에 리뷰 위젯을 추가하는 것은 단순한 디자인 선택이 아니라 전략적 결정입니다. 리뷰를 표시하면 사회적 신뢰성을 구축하고 잠재 고객이 확신을 가지고 구매 결정을 내리는 데 도움이 됩니다.

- **사회적 증명**: 실제 고객 리뷰를 보여주어 신뢰를 구축하고 전환율을 높입니다

- **리뷰 통합 및 필터링**: 구글과 페이스북 같은 플랫폼의 리뷰를 매끄럽게 결합합니다. 플랫폼이나 평점으로 필터를 적용하여 최고의 리뷰만 보여줄 수 있습니다

- **브랜드 일관성**: 위젯의 레이아웃, 폰트, 색상 체계를 웹사이트 브랜딩과 완벽하게 맞춥니다

- **자동 업데이트**: 새로운 리뷰가 자동으로 위젯에 추가되어 지속적인 관리 없이도 콘텐츠를 신선하게 유지합니다

- **템플릿**: 빠른 설정과 전문적이고 일관된 디자인을 위한 미리 만들어진 노코드 템플릿을 사용합니다

- **간단한 통합**: 코딩 지식 없이도 모든 웹사이트 페이지에 위젯을 쉽게 삽입할 수 있습니다

- **AI 요약**: 전체적인 리뷰 감정을 한눈에 보여주는 자동 생성 요약으로 즉시 신뢰를 구축합니다

## 리뷰 위젯 빌더 접근 및 사용법

리뷰 위젯을 생성하고 커스터마이징하려면 다음 단계를 따라하세요:

### **1단계:** Widgets(위젯) 탭으로 이동

- 하위 계정 대시보드에서 왼쪽 메뉴의 Reputation(평판 관리)으로 이동합니다.

- 상단 네비게이션에서 Widgets(위젯) 탭을 클릭합니다.

![위젯 탭 네비게이션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048742832/original/XKzNYgLBwwdl_3dbuSDyCyJBJqiDgn2gUA.png?1750707612)

### **2단계:** 새 위젯 선택 또는 생성

- 새 위젯을 생성하려면 + Create Widget(위젯 생성) 버튼을 클릭합니다

- 기존 위젯을 편집하려면 원하는 위젯을 클릭한 후 오른쪽 하단의 Edit Widget(위젯 편집) 파란색 버튼을 클릭합니다

- 처음부터 시작하는 경우 미리 만들어진 템플릿을 선택하여 전문적인 레이아웃과 스타일을 즉시 적용할 수도 있습니다. [템플릿 사용에 대한 자세한 정보는 여기를 클릭하세요.](#템플릿)

![위젯 생성 및 편집 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048742915/original/EMYU2h6RqZch8DvSRA6qPapyfDlE-PFAew.png?1750707936)

### **3단계:** 위젯 설정 구성

위젯 빌더에는 리뷰 위젯의 구조, 콘텐츠, 모양을 완전히 커스터마이징할 수 있는 여러 탭이 포함되어 있습니다. 레이아웃부터 소스와 테마까지 모든 것을 제어할 수 있습니다.

각 탭은 위젯 경험의 다른 부분에 초점을 맞춥니다:

- **Layout 탭**: 위젯 유형, 리뷰 소스, 요약 옵션, 평점 필터, "Powered by" 표시 여부를 설정합니다. [Layout 커스터마이징에 대한 자세한 정보는 여기를 클릭하세요.](#layout-탭-위젯-구조-설정)

- **Content 탭**: 위젯 제목, 설명, 위치 및 헤더 요소를 관리합니다. [Content 커스터마이징에 대한 자세한 정보는 여기를 클릭하세요.](#content-탭-위젯-헤더-및-설명-편집)

- **Appearance 탭**: 테마를 선택하고 브랜드 폰트를 적용하며 Custom 옵션 사용 시 개별 색상을 설정합니다. [Appearance 커스터마이징에 대한 자세한 정보는 여기를 클릭하세요.](#appearance-탭-테마-및-색상-선택)

- **Settings 탭**: 리뷰 표시 요소를 활성화/비활성화하고 사이트용 임베드 코드를 생성합니다. [Settings 커스터마이징에 대한 자세한 정보는 여기를 클릭하세요.](#settings-탭-위젯-동작-세부-조정)

참고: 각 탭의 자세한 설정 옵션은 아래의 전체 설명을 참조하세요.

### **4단계:** 위젯 저장 및 삽입

- 모든 필요한 변경사항을 완료한 후 오른쪽 하단의 파란색 Save(저장) 버튼을 클릭합니다.

- Hyperclass 사이트나 퍼널에 위젯을 추가하려면 Reviews(리뷰) 요소를 사이트 페이지에 드래그 앤 드롭합니다.

- 코드를 사용해서 리뷰 위젯을 삽입하려면 Code(코드) 아이콘을 클릭한 후 Copy Code(코드 복사)를 클릭합니다. 이 코드를 웹사이트나 퍼널의 커스텀 HTML 블록에 붙여넣습니다.

![위젯 임베드 코드 복사](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048743288/original/R85gbS6-qStRCI6W82PAHVy3A4Wjy8MUCQ.png?1750708645)

## 리뷰 위젯 커스터마이징

위젯 빌더의 각 탭은 구조와 레이아웃부터 콘텐츠, 디자인, 동작까지 리뷰 위젯의 다양한 측면을 제어할 수 있게 해줍니다.

참고: 선택한 위젯 유형에 따라 사용 가능한 커스터마이징 옵션이 달라집니다.

### **Layout 탭: 위젯 구조 설정**

리뷰 표시의 기본 구조와 소스를 정의합니다:

- **Summary Type(요약 유형)**: 리뷰를 기반으로 한 AI 생성 감정 개요입니다. 요약 유형(짧은 요약, 상세 요약, 행동 지향 요약)을 선택하고 필요에 따라 켜고 끌 수 있습니다.

- **Summary Position(요약 위치)**: AI 요약이 나타날 위치를 선택합니다. 위젯 상단 또는 리뷰와 함께 인라인으로 표시할 수 있습니다.

- **Review Source(리뷰 소스)**: 리뷰를 가져올 플랫폼을 선택합니다.

- **Widget Type(위젯 유형)**: 리뷰의 표시 형식을 선택합니다. List, Grid, Masonry, Carousel, Slider, Flash, Legacy 옵션이 있습니다.

- **Max Number of Reviews(최대 리뷰 수)**: 위젯에 표시할 총 리뷰 수를 설정합니다.

- **Minimum Ratings(최소 평점)**: 특정 별점 기준을 만족하는 리뷰만 표시하도록 선택합니다.

- **Powered by**: 위젯 하단의 "Powered by" 메시지를 표시하거나 숨깁니다.

![Layout 탭 설정 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048743501/original/VsRXMjHsQTnNBoXWiqY11wnS4TcAQhqG0Q.png?1750709276)

### **Content 탭: 위젯 헤더 및 설명 편집**

위젯이 리뷰를 소개하고 틀을 잡는 방식을 커스터마이징합니다:

- **Title (Toggle)(제목 토글)**: 위젯 상단에 커스텀 제목을 표시할지 여부를 토글합니다.

- **Heading(제목)**: 위젯 제목으로 나타날 텍스트를 정의합니다. 브랜드 톤이나 콘텐츠 초점을 반영할 수 있습니다(예: "고객들의 한마디").

- **Description(설명)**: 제목 아래에 컨텍스트나 격려를 제공하는 간단한 보조 텍스트를 추가합니다.

- **Header (Toggle)(헤더 토글)**: 평점 데이터, 리뷰 수, 리뷰 작성 버튼이 포함된 헤더 요약 섹션의 표시 여부를 제어합니다.

- **Rating(평점)**: 모든 리뷰의 평균 별점을 보여줍니다.

- **Total Review Count(전체 리뷰 수)**: 총 몇 개의 리뷰가 남겨졌는지 표시합니다.

- **Write a Review Button(리뷰 작성 버튼)**: 사용자가 리뷰를 제출할 수 있도록 안내하는 클릭 가능한 버튼을 추가합니다.

- **Widget Position(위젯 위치)**: 화면에서 위젯이 표시될 위치입니다.

- **Slider Interval(슬라이더 간격)**: 각 리뷰가 회전하기 전까지 표시되는 시간(초)을 설정합니다. 캐러셀이나 슬라이더 레이아웃 사용 시 사용할 수 있습니다.

![Content 탭 설정 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048743648/original/x2gWp6YEwpHiNRDZzW4AAbIgst118-J2WA.png?1750709673)

### **Appearance 탭: 테마 및 색상 선택**

웹사이트 브랜딩에 맞춰 위젯의 전반적인 시각적 스타일을 제어합니다:

- **Theme(테마)**: Light, Dark 또는 완전히 커스터마이징 가능한 옵션 중에서 선택합니다. 각 테마는 서로 다른 기본 시각적 경험을 제공합니다.

- **Font(폰트)**: 브랜드의 타이포그래피에 맞춰 위젯에서 사용되는 폰트 패밀리를 조정합니다.

- **Color(색상)**: Custom 테마 사용 시 배경, 테두리, 별점, 리뷰 텍스트, 버튼 등 다양한 색상 요소를 개별적으로 편집할 수 있습니다.

![Appearance 탭 설정 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048743710/original/Jo9j_aUcvymr-Emyx643P3BkMDpsrmwvTQ.png?1750709827)

### **Settings 탭: 위젯 동작 세부 조정**

위젯의 동작과 각 리뷰에 표시되는 정보를 더욱 세밀하게 조정합니다:

- **Display Date(날짜 표시)**: 각 리뷰가 게시된 날짜를 표시하려면 토글을 켭니다.

- **Display Reviewer Icon(리뷰 작성자 아이콘 표시)**: 각 리뷰 옆에 리뷰 작성자의 아바타 또는 이니셜을 표시하려면 활성화합니다.

- **Exclude Reviews Containing No Description(설명이 없는 리뷰 제외)**: 별점만 있고 서면 피드백이 없는 리뷰를 필터링합니다.

- **Embed Code to Website(웹사이트에 임베드 코드)**: Copy Code(코드 복사)를 클릭하여 자동 생성된 임베드 스크립트를 가져옵니다. 이 코드를 웹사이트나 퍼널의 HTML 블록에 붙여넣어 위젯을 표시합니다.

![Settings 탭 설정 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048743770/original/Vonk7q9JWxpuSOgsI3gPM59s5TcGIWgA4w.png?1750709979)

## 위젯 저장, 이름 지정 및 관리

쉬운 참조를 위해 위젯에 의미 있는 이름을 지정하세요. 저장된 위젯 탭에서 모든 저장된 위젯을 볼 수 있습니다. 나중에 사용하기 위해 위젯을 저장하거나 즉시 활성화할 수 있습니다.

![위젯 저장 및 이름 지정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155007981528/original/i5hDyYsrlVq8APk28m-HSSzxC0Wv5sFNqw.png?1695106902)

## 템플릿

템플릿을 사용하면 몇 초 안에 아름답게 스타일링된 리뷰 위젯을 런칭할 수 있습니다. 이러한 미리 만들어진 디자인은 모든 요소를 수동으로 커스터마이징하지 않고도 전문적인 레이아웃을 원하는 경우 특히 유용합니다. 템플릿 적용은 빠르고 쉽습니다!

- 위젯 패널 오른쪽 상단의 Templates(템플릿) 탭을 클릭합니다.

- 사용 가능한 디자인 목록을 둘러보고 아무 템플릿이나 클릭하여 해당 디자인을 리뷰 위젯에 즉시 적용합니다.

적용한 후에는 다른 위젯과 마찬가지로 계속 커스터마이징할 수 있습니다. 레이아웃, 색상, 폰트, 콘텐츠를 조정하여 자신만의 것으로 만드세요. 파란색 Edit Widget(위젯 편집) 버튼을 클릭하기만 하면 됩니다.

![템플릿 선택 및 적용](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048745241/original/MkvFXEafZrEa26Llwtbliuoy3iLRiXTU1A.png?1750714297)

## **자주 묻는 질문**

**Q: 하나의 웹사이트에 여러 개의 다른 리뷰 위젯을 삽입할 수 있나요?**
네, 가능합니다. 다만 빌더의 "Review" 블록을 사용하면 항상 기본 위젯이 표시됩니다. 같은 사이트에 여러 위젯을 삽입하려면 "Embed Code" 옵션을 사용해서 원하는 위젯의 코드를 복사하세요. 그런 다음 "Code" 블록을 사용하여 위젯 코드를 웹사이트에 추가합니다. 코드를 복사할 때 드롭다운에서 올바른 위젯을 선택하고 있는지 확인하세요. 사이트에 추가하고 싶은 각 위젯에 대해 이 과정을 반복합니다.

**Q: 새로운 리뷰를 받으면 위젯이 자동으로 업데이트되나요?**
네. 위젯은 구글과 페이스북 같은 연결된 플랫폼과 동기화됩니다. 설정한 필터와 표시 설정에 따라 새로운 리뷰가 자동으로 위젯에 나타납니다.

**Q: 리뷰 위젯이 Hyperclass이 아닌 웹사이트에서도 작동하나요?**
네. 리뷰 위젯은 거의 모든 웹사이트 플랫폼에 삽입할 수 있습니다. 위젯에서 임베드 코드를 복사하여 외부 사이트의 커스텀 코드 블록에 붙여넣으면 됩니다.

**Q: 사이트에 위젯을 추가한 후에도 계속 편집할 수 있나요?**
네. 위젯을 삽입한 후에도 언제든지 편집할 수 있습니다. 위젯 빌더에서 변경한 내용(레이아웃, 콘텐츠, 스타일링 등)은 위젯을 저장하면 사이트에 자동으로 업데이트됩니다.

**Q: 리뷰 위젯이 모바일 반응형인가요?**
물론입니다. 모든 리뷰 위젯 템플릿과 레이아웃은 완전히 반응형이며 모바일, 태블릿, 데스크톱 보기에 자동으로 최적화됩니다.
