# 리뷰 스팸 탐지

이 기능을 사용하면 스팸 리뷰를 쉽게 식별하고 관리할 수 있습니다. 이번 업데이트를 통해 하위 계정은 가짜이거나 부적절한 피드백을 필터링하여 온라인 평판의 무결성을 더욱 효과적으로 유지할 수 있으며, 진짜 고객 리뷰에만 집중할 수 있게 됩니다.

# 스팸 탐지 개요

- 리뷰 섹션에서 스팸으로 표시된 리뷰를 명확하게 구분할 수 있는 시각적 표시
- 각 리뷰의 액션 메뉴에 "스팸으로 표시" 옵션 추가
- 스팸으로 리뷰를 필터링하는 옵션 추가
- 잘못된 스팸 분류를 수정할 수 있는 "스팸 해제" 액션

---

## 리뷰 스팸 탐지를 사용해야 하는 이유

- 스팸을 쉽게 필터링하여 진짜 고객 피드백에만 집중할 수 있어 리뷰 품질이 향상됩니다.
- 자동 스팸 탐지로 수동 검토 시간이 줄어듭니다.
- AI가 스팸에 응답하는 것을 방지하여 잠재적인 평판 손상을 피할 수 있습니다.
- 사용자가 수동으로 리뷰를 스팸으로 표시하거나 해제할 수 있어 정확성을 보장합니다.

## 자주 묻는 질문

**1. 새로운 "스팸으로 표시" 기능은 어떤 역할을 하나요?**
"스팸으로 표시" 옵션을 사용하면 가짜이거나 부적절해 보이는 리뷰를 신고할 수 있습니다. 표시한 후에는 해당 리뷰가 스팸으로 시각적으로 표시되고 필터링할 수 있어, 진짜 고객 피드백에 집중할 수 있습니다.

**2. 스팸으로 표시된 리뷰는 어떻게 구분하나요?**
스팸으로 신고된 리뷰는 리뷰 섹션에서 명확한 시각적 표시가 나타나므로 한눈에 쉽게 알아볼 수 있습니다.

**3. 스팸 리뷰만 보도록 필터링할 수 있나요?**
네, 스팸으로 표시된 리뷰만 볼 수 있는 필터 옵션을 추가했습니다. 이를 통해 신고된 콘텐츠를 빠르게 관리하고 검토할 수 있습니다.

**4. 실수로 리뷰를 스팸으로 표시했다면 어떻게 하나요?**
실수로 리뷰를 스팸으로 신고했다면 "스팸 해제" 액션을 사용해서 스팸 라벨을 제거하고 리뷰를 원래 상태로 복원할 수 있습니다.

**5. AI가 스팸으로 표시된 리뷰에도 응답하나요?**
아니요, AI는 스팸으로 신고된 리뷰에는 응답하지 않습니다. 이를 통해 부적절하거나 가짜 리뷰가 자동 응답이나 평판 관리 활동에 영향을 주지 않도록 보장합니다.

**6. 이 기능이 우리 사업에 왜 중요한가요?**
이 기능은 스팸을 필터링하여 리뷰 품질을 향상시키고, 피드백 관리에 드는 시간과 노력을 절약해줍니다. 또한 AI가 부적절하거나 해로운 리뷰와 상호작용하는 것을 방지하여 평판을 온전히 유지할 수 있게 해줍니다.
