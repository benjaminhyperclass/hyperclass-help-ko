# SaaS 구성도구 온보딩

SaaS 구성도구를 처음 사용하여 SaaS 모드를 설정할 때 도움이 되는 온보딩 위젯이 제공됩니다.

하이레벨은 SaaS 모드를 설정하는 에이전시를 위한 가이드 온보딩 체크리스트를 제공합니다. 이 체크리스트는 올바른 순서로 필수 설정 단계를 완료하는 데 도움을 줍니다.

## 체크리스트 찾는 방법

에이전시 뷰에서 SaaS 구성도구를 여세요. 에이전시가 SaaS 설정을 완료하지 않은 경우, 체크리스트가 구성도구에 나타납니다.

## 설정 체크리스트 완료하기

**Stripe 연결 (결제 제공업체):** SaaS 청구를 활성화하려면 Stripe를 연결하세요.

**참고:** Stripe는 SaaS 상품의 기본 결제 처리업체입니다. 현재 SaaS 상품에는 다른 결제 게이트웨이가 지원되지 않습니다.

![Stripe 연결](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155067018290/original/elDPp1QngS7w_Ymr6rcqvHXaSNoOSFxrbw.jpeg?1773663357)

**통화 선택:** SaaS 요금제 가격에 사용할 기본 통화를 선택하세요.

"잘못된 가격 통화" 배너가 보이면, SaaS 구성도구 → 고급 설정에서 카테고리 통화를 확인하세요. 그다음 Stripe 가격 통화를 일치하도록 업데이트하세요.

**중요:** 가격에 대해 판매 링크가 이미 생성되었거나 활성 구독이 있는 경우, Stripe에서 기존 가격의 통화 변경을 허용하지 않을 수 있습니다. 새로운 가격을 생성하고 새 Stripe 가격에서 통화를 업데이트하세요.

**SaaS 요금제 생성:** 템플릿을 사용하거나 수동으로 요금제를 구축하여 SaaS 요금제를 생성하세요. 필수 필드를 작성하고 요금제를 저장하세요.

![SaaS 요금제 생성](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155067018462/original/nbi3VjGRx8Qq10RLeEcHBKc8yabvnYB2rg.jpeg?1773663497)

**판매 링크 생성:** 잠재고객과 체크아웃 링크를 공유하고 판매를 시작할 수 있도록 판매 링크를 생성하세요.

**팁:** 동일한 SaaS 상품에 대해 판매 링크가 이미 존재하는 경우, 하이레벨은 중복 링크를 생성하는 대신 기존 URL을 반환합니다.
