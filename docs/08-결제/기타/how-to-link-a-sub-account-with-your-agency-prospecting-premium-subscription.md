# 에이전시 프로스펙팅 프리미엄 구독을 하위 계정과 공유하는 방법

에이전시 레벨에서 이미 프로스펙팅 프리미엄 구독을 이용 중인 에이전시는 이제 하위 계정 하나를 기존 프로스펙팅 구독에 연결하여 청구를 간소화하고 비용을 절약할 수 있습니다. 이 업데이트를 통해 두 계정 모두에서 프리미엄 기능에 공유 접근이 가능해지며, 구독 관리를 간소화하면서도 유연성을 유지할 수 있습니다.

# 하위 계정과의 새로운 프로스펙팅 도구 공유 옵션이란?

새로운 공유 옵션은 통합 구독 모델을 도입하여, 활성 프리미엄 프로스펙팅 구독을 보유한 에이전시가 하위 계정 하나와 프리미엄 구독을 공유할 수 있게 해줍니다. 이를 통해 중복 구독의 필요성을 없애고 한 곳에서 청구 및 접근을 더 효과적으로 제어할 수 있습니다.

에이전시가 공유를 원하지 않는 경우, 에이전시 레벨 구독을 취소하고 아무 하위 계정에서나 새로운 구독을 시작하여 하위 계정에서 새로운 프로스펙팅을 시작할 수도 있습니다(에이전시 뷰의 재판매(Reselling) 탭에서 활성화 필요).

## 새로운 프로스펙팅 도구 공유 옵션의 주요 장점

에이전시는 이번 업데이트를 통해 향상된 비용 효율성과 더 간단한 워크플로우의 혜택을 얻을 수 있습니다:

- **비용 절약**: 에이전시와 하위 계정을 위한 두 개의 별도 구독을 유지할 필요가 없음
- **간소화된 구독 관리**: 한 곳에서 두 계정의 청구 및 사용자 접근을 관리
- **전체 기능 접근**: 에이전시와 연결된 하위 계정 모두 프로스펙팅 프리미엄 도구에 완전한 접근 권한 획득
- **운영 유연성**: 비즈니스 요구 변화에 따라 연결된 하위 계정을 조정하거나 변경 가능

## 에이전시 프로스펙팅 프리미엄 구독을 하위 계정과 공유하는 방법

구독을 올바르게 설정하고 하위 계정을 연결하면 단일 청구 구조 하에서 모든 기능에 접근할 수 있습니다.

**중요**: 이는 일회성 작업입니다. 나중에 하위 계정을 변경할 수 없습니다.

### 에이전시 레벨에서 이미 프로스펙팅 프리미엄을 구독 중인 경우

- 에이전시 계정에서 프로스펙팅(Prospecting)을 클릭합니다.
![](https://jumpshare.com/v/rvWEy11fSlpCCrhiL4MQ+/Screen+Shot+2025-06-20+at+7.39.38+PM.png)

- 설정(Settings) 탭을 클릭한 후 구독 관리(Manage Subscription)를 클릭합니다. 그 다음 하위 계정 연결(Link your Sub-Account) 버튼을 클릭합니다.
![](https://jumpshare.com/v/UMNh1LoVBsDQ9REOLsyq+/Screen+Shot+2025-06-20+at+7.43.48+PM.png)

- 목록에서 연결하고자 하는 하위 계정을 선택합니다.
![](https://jumpshare.com/v/n4PXVRu1NVk9Vj5kkupG+/Screen+Shot+2025-06-20+at+7.45.49+PM.png)

- 지금 연결(Link Now) 버튼을 클릭하여 선택을 확인합니다.
![](https://jumpshare.com/v/L51JDZNG7cNy2kNEggEU+/Screen+Shot+2025-06-20+at+8.04.09+PM.png)

# 에이전시 프로스펙팅 도구에서 연결된 하위 계정을 제거하거나 업데이트하는 방법:

각 에이전시 프로스펙팅 구독은 하나의 하위 계정에 연결할 수 있습니다. 연결된 하위 계정을 제거하거나 업데이트하고 싶다면 다음과 같이 하세요:

### 연결된 하위 계정을 제거하는 방법(다른 계정 추가 없이):

- 설정(Settings)으로 가서 에이전시 레벨 또는 연결된 하위 계정 레벨에서 구독을 취소합니다.
- 이렇게 하면 두 계정 모두에서 프리미엄 접근 권한이 제거됩니다.
- 하위 계정을 연결하지 않고 계속 사용하고 싶다면, 하위 계정 연결 없이 새 구독을 시작하세요.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155053442062/original/zfhsbed4B-7n7qVNBX_XEcgeZo1UhyAKYQ.png?1757491339)

### 연결된 하위 계정을 다른 계정으로 교체하는 방법:

두 가지 옵션이 있습니다:

- **옵션 1**:
  - 설정(Settings)으로 가서 에이전시 또는 현재 연결된 하위 계정에서 구독을 취소합니다.
  - 새 구독을 시작하고 연결할 새로운 하위 계정을 선택합니다.

- **옵션 2**:
  - 기존 구독을 취소합니다.
  - 원하는 하위 계정에서 직접 새 구독을 시작합니다 — 재판매(Reselling)를 통해서든 직접 회사 구매(Direct Company Purchase)를 통해서든(아래에서 설명).

## 프로스펙팅 프리미엄을 처음 구독하는 경우:

프로스펙팅 도구를 프리미엄으로 업그레이드하는 방법은 두 가지가 있습니다:

## 1. 에이전시 레벨 프로스펙팅

- 모든 프리미엄 혜택 포함:
  - 월 무제한 리드
  - 웹사이트 위젯
  - 리드 POC 인리치먼트
  - 리포트 빌더

**중요**: 에이전시 레벨 프로스펙팅은 CRM 기능(예: 연락처 추가, SMS 발송, 기회 관리 추가, 워크플로우 트리거)을 포함하지 않습니다.

- CRM 기능을 원한다면 나중에 하위 계정으로 이동해야 합니다.
- **권장사항**: 프로스펙팅 + CRM 기능에 바로 접근하려면 하위 계정 내에서 프로스펙팅을 시작하세요.

## 2. 하위 계정 레벨 프로스펙팅

- 모든 프리미엄 혜택과 CRM 기능 포함:
  - 프로스펙트를 연락처에 추가
  - SMS 발송
  - 기회 관리에 추가
  - 워크플로우 트리거

### **하위 계정에서 프로스펙팅을 활성화하는 방법**

세 가지 방법으로 활성화할 수 있습니다:

- **재판매 탭에서**:
  - 에이전시 뷰(Agency View) > 재판매 탭(Reselling Tab)으로 이동
  - 재판매 프로스펙팅(Re-sell Prospecting) 토글을 저장하고 활성화
![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155053443359/original/bIIB6n4lo_MN4Pvecai6IVaCsnoPJOEbWg.png?1757492086)

- **직접 회사 구매**:
  - 하위 계정 레벨에서 직접 프로스펙팅 프리미엄을 구매
![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155053443374/original/tc9eB_VStl7xlp-ZRUWAs1yq_j2qAdobVA.jpeg?1757492104)

- **에이전시 프로스펙팅 도구에서 업그레이드**:
  - 에이전시 계정에서 프로스펙팅으로 이동하여 업그레이드
  - 프리미엄 플랜 하에서 지금 업그레이드(Upgrade Now) 버튼 클릭
  - 다음 선택이 나타납니다: 지금 하위 계정 선택, 또는 연결 없이 계속: 이름을 입력하여 하위 계정 선택

## 자주 묻는 질문

**Q: 프리미엄 구독에 하위 계정을 연결하는 곳은 어디인가요?**
이 작업은 에이전시 프로스펙팅 섹션 내에서만 수행할 수 있습니다.

**Q: 프로스펙팅 프리미엄 구독에 하나 이상의 하위 계정을 연결할 수 있나요?**
아니요, 에이전시 구독당 하나의 하위 계정만 연결할 수 있습니다.

**Q: 나중에 연결된 하위 계정을 변경할 수 있나요?**
아니요, 하위 계정 연결은 취소할 수 없습니다. 한 번 연결되면 변경할 수 없습니다.

**Q: 에이전시와 하위 계정 모두 프로스펙팅 프리미엄에 완전한 접근 권한을 얻나요?**
네, 두 계정 모두 단일 구독 하에서 완전한 접근 권한을 받습니다.

**Q: 하위 계정을 연결하지 않고 구독하고 싶다면 어떻게 하나요?**
설정 중에 연결하지 않고 계속 진행하도록 선택하고, 나중에 구독 설정에서 연결할 수 있습니다.

**Q: 선택한 하위 계정이 이미 활성 프리미엄 구독을 가지고 있다면 어떻게 되나요?**
하위 계정이 이미 활성 프리미엄 구독을 가지고 있다면, 저희가 자동으로 청구를 조정하고 중복 요금을 방지하기 위해 중복 구독을 취소합니다.
