## 새로운 기능은?

- 새로운 견적서 기능 출시를 발표하게 되어 기쁩니다!
- 이 기능을 통해 최종 인보이스(Invoice)를 보내기 전에 견적서를 생성하고 고객과 공유할 수 있습니다.
- 고객은 간단히 이 견적서를 **승인/거절**할 수 있습니다.
- 견적서를 **인보이스로 바로 변환**하여 쉽게 발송할 수 있습니다.

## 사용 방법은?

### 견적서 접근하기

- 하위 계정(Sub-account)으로 이동 → 결제(Payments) 선택
- 인보이스 및 견적서(Invoices and Estimates) 클릭 → 견적서(Estimates) 선택

![견적서 메뉴 접근](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033906697/original/4iDRwJTvq3KG8ZYtl61ONnuZOXECOU24eQ.jpeg?1727834340)

### 견적서 생성하기

- 견적서 생성하기: '새로 만들기(New)' 버튼을 클릭하고 비즈니스 정보, 고객 정보, 견적서 번호, 만료일 등의 세부사항을 입력합니다.

![견적서 생성 폼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033906696/original/mzTO1YnsOF0euWT0oywMMcnZpt0Y0lReuw.jpeg?1727834340)

- 견적서에 상품(Product)을 추가하고 발송 버튼을 클릭합니다.
- 고객에게 발송하기: 견적서를 검토한 후 이메일 또는 SMS를 통해 고객에게 바로 발송합니다.

![견적서 상품 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033906695/original/HsB6j0eymE8lDG4fCzwSrC1l41G14O0Uag.jpeg?1727834340)

![견적서 발송 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033906688/original/NGuK4JCPQyeCn09P-s03sMBbEWXGHeG4-A.jpeg?1727834340)

### 고객이 견적서 승인/거절하기

- 고객은 승인 버튼을 클릭할 수 있으며, 견적서가 승인됨으로 표시됩니다.

![견적서 승인 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033906692/original/1mVFnaw_U8etfLfIINiyZL8J9hHf8pJEnQ.jpeg?1727834340)

- 고객은 거절 버튼을 클릭하여 메모를 추가할 수 있으며, 견적서가 거절됨으로 표시됩니다.

![견적서 거절 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033906693/original/2UIhjV4ysOlb7JG3ZjTtQEF7MNci63vmAA.jpeg?1727834340)

- 고객이 이미 전화로 확인한 경우, '승인/거절로 표시하기(Mark as accepted/rejected)' 버튼을 클릭할 수 있습니다.

![수동 상태 변경](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033906689/original/hC2x2Cp3oM8uF1cQwVEZe6ypD0bdz2tm5Q.jpeg?1727834340)

### 인보이스로 변환

- 발송된 인보이스 항목을 클릭합니다.

![인보이스 변환 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033906690/original/_CBQ8DKjHtzz-gLRrO5Wh_p3DA_eQQJj4A.jpeg?1727834340)

- 인보이스로 변환: 고객이 견적서를 승인하면, 대시보드에서 '인보이스 생성'을 선택하거나 승인된 견적서를 열어서 '인보이스 생성' 버튼을 클릭하여 쉽게 인보이스로 변환할 수 있습니다.

![인보이스 생성 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033906691/original/IkAojIbBURxokJT9IHRQx6iddxraBxcr6w.jpeg?1727834340)

### 추적

- 다양한 단계에 있는 모든 견적서 추적: 초안(Draft), 발송됨(Sent), 승인됨(Accepted), 거절됨(Declined), 인보이스 발행됨(Invoiced)
- 대시보드 상단의 카드에서 파이프라인에 있는 견적서의 금액 추적
- 대시보드와 견적서 빌더 내에서 고객이 추가한 메모 추적

![견적서 추적 대시보드](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033906694/original/TGP8lYPQQDZn4ep7PPIKKEelcRvhNHHyFA.jpeg?1727834340)

### 기타

- 커스터마이징: 설정에서 회사 브랜딩, 약관을 추가하여 견적서를 맞춤 설정할 수 있습니다.

## 왜 만들었나요?

견적서 기능은 모든 비즈니스 요구사항에 대한 완전한 결제 도구 모음을 완성하기 위해 만들어졌습니다!

- 견적서 생성
- 문서 서명 받기
- 인보이스 발송
