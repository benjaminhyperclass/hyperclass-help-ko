# 인보이스 - 새롭게 개선된 UI

## 새로운 기능

- 오른쪽에는 미리보기, 왼쪽에는 편집 옵션을 배치한 새로운 레이아웃으로 인보이스 디자인을 전면 개편했습니다.
- 상단 배너를 사용해서 새로운 디자인으로 전환할 수 있으며, Labs 섹션에서 활성화하면 됩니다.
- 이제 사용자가 첫 번째 인보이스(청구서)를 훨씬 쉽게 시작하고 전송할 수 있습니다.

## 사용 방법

**새 디자인 접근하기:**
인보이스 페이지 상단의 배너를 클릭해서 새 레이아웃으로 전환하세요. Labs 섹션에서 활성화하는 것을 잊지 마세요.

![인보이스 배너](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155035132548/original/_GwBk9q9dU08FOmWA9g68yCgmW1maXjEAA.jpeg?1729544351)

![Labs 섹션 활성화](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155035132547/original/XwLD87_lzi__QIdIqZIpPHXpPDeOdhYlUA.jpeg?1729544351)

**인보이스 편집하기:**
왼쪽에서 고객 정보, 항목 라인, 총액 등의 인보이스 세부사항을 입력하세요. 툴바를 사용해서 연체료, 팁, 결제 일정 등의 요소를 추가할 수 있으며, 이 기능들은 모두 정상적으로 작동합니다.

![인보이스 편집 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155035132549/original/dZlv5mKI34BgYSSiazQqyDqAIrmPZ9rOmw.jpeg?1729544351)

**인라인 상품 추가:**
이제 상품명을 입력하고 엔터만 누르면 바로 추가할 수 있어서 인보이스를 빠르게 전송하는 것이 매우 간단해졌습니다.

![인라인 상품 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155035132544/original/C--q45ah2XDwRQhUVMdQQAtBBhUx8VxSow.png?1729544350)

**변경사항 미리보기:**
편집하면서 오른쪽의 실시간 미리보기가 자동으로 업데이트되어, 고객에게 어떻게 보일지 확인할 수 있습니다.

![실시간 미리보기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155035132545/original/yjM4mo49ds1TJ0282VH2L9FSRbxC0H7jTA.jpeg?1729544350)

**정기 인보이스:**
첫 번째 정기 상품을 생성하면 시스템이 자동으로 해당 인보이스의 정기 속성을 설정합니다.

**최종 검토 및 저장:**
인보이스를 완성한 후 미리보기에서 마지막 수정사항을 확인하고 "저장"을 클릭해서 최종 완료하고 전송하세요.

## 개발 배경

- 이 디자인 개편은 더 직관적이고 매력적인 레이아웃으로 사용자 경험을 향상시키기 위해 진행되었습니다.
- 새로운 설정으로 사용자가 인보이스를 편집하고 미리보기하기 더 쉬워졌으며, 인보이스 작성 과정이 간소화되었습니다.
