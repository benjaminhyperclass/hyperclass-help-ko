# 상품 삭제 작동 방식

이 가이드에서는 계정에서 상품을 삭제하는 방법을 알려드립니다. 상품을 삭제하는 여러 방법, 일괄 삭제 작동 방식, 그리고 진행 중인 프로세스가 중단되지 않도록 삭제가 제한되는 상품 유형에 대해 배우게 됩니다.

## 이 기능은 무엇인가요?

상품 삭제 기능을 사용하면 더 이상 필요하지 않은 상품을 영구적으로 제거할 수 있습니다. 결제와 고객 액세스를 보호하기 위해, 시스템은 특정 상품 유형이나 활성/연체 구독이 존재할 때 삭제를 차단합니다. 일괄 작업에서는 삭제 가능한 상품만 제거되며, 보호된 항목은 명확한 메시지와 함께 건너뛰어집니다.

상품을 삭제한다고 해서 이미 판매된 상품(거래, 주문, 인보이스, 구독 등)이 반드시 삭제되는 것은 아닙니다. 해당 상품이 포함된 기존 거래는 계속 유지되며, 향후 구매나 해당 상품에 대한 작업만 차단됩니다.

## 이 기능을 사용하려면 무엇이 필요한가요?

- **권한**: 상품 → 삭제 권한(그리고 상품 → 보기 권한)이 있는 역할
- **네비게이션**: Payments(결제) → Products(상품) (목록 및 상세 페이지)
- **주의사항**: 삭제는 영구적이며 되돌릴 수 없습니다. 해당 상품이 활성 결제나 캘린더/멤버십, 또는 활성 SaaS 구독에서 필요하지 않은지 확인하세요.

## 상품을 삭제하는 방법

### 방법 1: 상품 목록에서 삭제

1. `Payments(결제) → Products(상품)`로 이동하세요.
2. 삭제할 상품을 찾으세요.
3. 삭제(휴지통) 아이콘을 클릭하세요.
4. 대화상자에서 확인하세요.

### 방법 2: 상품 상세 페이지에서 삭제 (가격/변형 삭제)

1. `Payments(결제) → Products(상품) → Edit(편집)`으로 이동하세요.
2. Delete Price(가격 삭제)를 클릭하세요. 주의 - 이는 상품에 2개 이상의 가격/변형이 있을 때만 작동하며, 마지막 가격/변형은 삭제할 수 없습니다.

참고: 삭제 컨트롤이 비활성화되어 있거나 자격 관련 메시지가 표시되면, 아래 "제한사항" 섹션을 확인하세요.

## 상품을 일괄 삭제하는 방법

1. `Payments(결제) → Products(상품)`로 이동하세요.
2. 체크박스로 여러 상품을 선택하세요(또는 전체 선택).
3. **Bulk Actions(일괄 작업)** 아래의 Delete(삭제)를 클릭하세요.
4. 확인 메시지를 검토하고 확인하세요.

확인 메시지(표시되는 내용):

- "정말로 이 상품들을 삭제하시겠습니까? 이 작업은 되돌릴 수 없습니다. 삭제 자격이 있는 상품만 삭제됩니다."
- 고정 개수가 표시되는 경우 (예: 10개 선택): "정말로 이 10개 상품을 삭제하시겠습니까? 이 작업은 되돌릴 수 없습니다. 삭제 자격이 있는 상품만 삭제됩니다."

삭제 자격이 있는 상품만 삭제되며, 보호된 상품은 남아있습니다.

## 제한사항: 삭제할 수 없는 것들 (그리고 그 이유)

결제와 기능 무결성을 보호하기 위해 다음 규칙이 적용됩니다:

### SaaS 상품 (SaaS Plans & Agency Plans)

- 해당 상품에 활성 또는 연체 구독이 존재하면 차단됩니다.
- 활성/연체 구독이 없을 때만 허용됩니다.
- 차단된 경우 삭제 컨트롤이 비활성화되고 상품이 유지됩니다.

### 캘린더 상품

- 캘린더에서 생성된 상품은 삭제할 수 없습니다.
- 삭제 CTA가 숨겨집니다.

### 멤버십/커뮤니티 상품

- 멤버십/커뮤니티에서 생성된 상품은 삭제할 수 없습니다.
- 삭제 CTA가 숨겨집니다.

### 가격 및 변형 (SaaS, 캘린더, 멤버십/커뮤니티)

- 저장된 가격/변형: 삭제할 수 없으며, 삭제 아이콘이 비활성화됩니다.
- 저장되지 않은 (방금 추가했지만 아직 저장하지 않은) 가격/변형: 제거할 수 있습니다.

## 자주 묻는 질문

**Q: 내 상품에 삭제 버튼이 보이지 않는 이유가 뭔가요?**
A: 캘린더 또는 멤버십/커뮤니티 상품이거나(삭제 불가), 활성/연체 구독이 있는 SaaS 상품입니다.

**Q: 차단된 SaaS 상품을 삭제하려면 어떻게 해야 하나요?**
A: 먼저 해당 상품과 연결된 모든 활성/연체 구독을 해결하거나 취소하세요. 구독이 더 이상 남아있지 않으면 상품을 삭제할 수 있습니다.

**Q: 일괄 삭제에서 선택한 항목에 보호된 상품이 포함되어 있으면 어떻게 되나요?**
A: 삭제 자격이 있는 상품만 삭제됩니다. 보호된 항목은 건너뛰어지며, 확인 대화상자에서 이런 동작을 명확히 설명합니다.

**Q: 보호된 상품의 가격/변형을 삭제할 수 있나요?**
A: 아직 저장되지 않은 경우에만 제거할 수 있습니다. SaaS, 캘린더, 또는 멤버십/커뮤니티 상품의 저장된 가격/변형은 삭제할 수 없습니다.

**Q: 삭제를 되돌릴 수 있나요?**
A: 아니요. 삭제는 영구적입니다. 진행하기 전에 종속성을 확인하세요.

**Q: CRM의 결제 탭에서 상품을 삭제했는데, 왜 내 계정에 여전히 나타나나요?**

A: 외부 결제 게이트웨이(예: Stripe)에 연결된 상품은 이중 삭제가 필요합니다. 플랫폼에서 상품을 삭제하면 로컬 기록은 제거되지만, 결제 게이트웨이에 저장된 해당 상품이나 플랜은 종종 제거되지 않습니다.

상품을 완전히 영구적으로 제거하려면:

- **플랫폼에서 삭제**: `Payments(결제) → Products(상품)`에서 항목을 삭제하세요.
- **게이트웨이에서 삭제**: 외부 결제 처리업체 대시보드(예: Stripe, PayPal)에 직접 로그인하여 해당 상품/플랜을 수동으로 삭제하거나 보관 처리하세요.
