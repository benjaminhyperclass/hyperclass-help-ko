# Stripe에서 상품/가격 가져오기

## 어떤 요금제를 가져올 수 있나요?

- 정기 결제/구독 요금제만 가능합니다. 일회성 결제는 가져올 수 없으며, 시스템에서 직접 생성해야 합니다.
- Stripe에서 요금제/가격을 가져오려면 라이브 모드(Live mode)여야 합니다(테스트 모드 불가).
- 정기 요금제/가격을 가져올 때 "계량형(Metered)" 가격은 가져올 수 없습니다.
- 이미 하이퍼클래스로 가져온 Stripe 요금제/가격은 "Stripe에서 가져오기" 기능 사용 시 선택 옵션에 나타나지 않습니다.

## Stripe에서 가격을 가져올 때 어떤 작업이 수행되나요?

- Stripe 상품(가격과 연결된)을 가져와서 하이퍼클래스에 없는 경우 생성합니다.
- 해당 상품 하위에 가격을 생성합니다.
- Stripe 테스트 모드와 PayPal(연동된 경우)에도 동일한 상품과 가격을 생성합니다.

## Stripe 계정에는 3가지 유형의 요금제가 있습니다

### 1. 에이전시 레벨에서 SaaS 설정 도구를 통해 생성된 가격:

- SaaS 설정 도구로 생성된 가격은 상품 목록과 상품 상세 페이지에서 상품명과 가격명에 "Agency Plan" 태그가 표시됩니다.
- SaaS 설정 도구로 생성된 가격은 하이퍼클래스 결제(Payments)에서 편집할 수 없습니다.
- Stripe에서 직접 가격을 수정하면 하이퍼클래스 상품/가격에 변경 사항이 반영되지 않습니다.
- SaaS 설정 도구로 생성된 상품의 가격을 변경하려면 Stripe에서 가격을 편집한 다음 "Stripe에서 가져오기" 버튼을 사용하여 새 가격을 가져와야 합니다. 이렇게 하면 "Agency Plan" 태그가 있는 새 하이퍼클래스 상품이 생성되며, 퍼널(Funnel)이나 웹사이트(Website) 페이지 설정에서 선택할 수 있습니다.

### 2. 하이퍼클래스에서 상품 생성으로 만들어진 가격:

- 하이퍼클래스에서 상품을 생성하면 Stripe에도 해당 요금제/가격이 생성됩니다.
- 하이퍼클래스 상품 가격을 편집하면 Stripe에 새 가격을 생성하여 자동으로 동기화됩니다. 이는 기존 가격을 편집하는 Stripe API의 제한 때문입니다.
- Stripe 가격 편집은 하이퍼클래스로 자동 동기화되지 않습니다. Stripe에서 변경된 가격은 새 상품으로 하이퍼클래스에 가져와야 합니다.

### 3. Stripe에서 직접 생성한 가격:

- Stripe에서 직접 요금제/가격을 생성한 경우 "Stripe에서 가져오기" 버튼을 사용하여 하이퍼클래스 상품으로 가져올 수 있습니다.
- 가져온 상품의 가격을 편집해야 하는 경우 하이퍼클래스에서 편집하면 Stripe에 새 가격을 생성하여 변경 사항이 동기화됩니다.
- Stripe에서 가져온 요금제/가격을 편집하면 하이퍼클래스 상품에 변경 사항이 반영되지 않으므로, 변경된 요금제/가격을 새 하이퍼클래스 상품으로 가져와야 합니다.
