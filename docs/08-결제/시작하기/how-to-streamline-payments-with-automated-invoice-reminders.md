# 자동 인보이스 알림으로 결제 프로세스 간소화하기

기존에는 워크플로우를 사용해서 고객에게 결제 알림을 보내셨나요? 이제 설정에서 직접 결제 알림을 설정하여 인보이스 결제 전환율을 높일 수 있습니다.

## 새로운 기능

- 새 알림 추가로 여러 개의 알림을 설정하고, 토글 버튼으로 간단히 활성화/비활성화 가능
- 필요시 알림 삭제 기능
- 알림 횟수 설정, 커스텀 이메일 작성, 인보이스 마감일 또는 발송일 기준으로 발송 시점 설정

## 사용 방법

- 인보이스(Invoices) 섹션에서 설정(Settings)으로 이동하세요

![인보이스 설정 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155030185444/original/VMJYJzneKUpXplnAnLu9aSoU4wECUJQMyg.jpeg?1722368251)

- 알림 설정(Reminder Settings)을 클릭하세요

![알림 설정 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155030185445/original/uPBhX7ZNehkXR3EOU4NkXDlCeIjQh6Ya-A.jpeg?1722368250)

- 각 알림에 대해 아래 세부사항을 설정하세요:
  - 알림 이름
  - 이메일 및 SMS용 알림 템플릿 - 기존 템플릿 사용 또는 기존 템플릿 편집하여 즉석에서 새 템플릿 생성
  - 이메일 제목 편집
  - 알림 주기 설정: [X]일/월/시간 [전/후] [인보이스 마감일/인보이스 발송일]마다
  - 최대 알림 횟수 설정으로 알림 중단 시점 제한

![알림 설정 상세 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155030185443/original/As47VMX6JTO773Yik8CN6JJ7lP-ObF2RCw.jpeg?1722368250)

- 새 알림 추가(Add New Reminder)를 클릭하여 추가 알림을 설정하세요
- 필요시 알림을 삭제하거나 토글 버튼으로 비활성화하세요

## 인보이스 알림 동작 및 수동 알림 처리

### 1. 인보이스 알림 템플릿 동작

- 인보이스 설정에서 미리보기로 표시되는 알림 템플릿이 자동 알림에 사용되는 정확한 템플릿입니다.
- 여러 개의 알림을 설정할 수 있으며(예: 알림 1, 알림 2), 각각 고유한 타이밍을 갖습니다.
- 알림 자동화는 활성화된 후에만 실행됩니다.

### 2. 기존 연체 인보이스 동작

- 자동 알림은 알림 기능이 활성화되기 전에 이미 연체된 인보이스에는 적용되지 않습니다.
- 시스템은 설정을 활성화한 후 생성된 인보이스에만 알림을 발송합니다.
- 이전에 연체된 인보이스는 수동으로 처리해야 합니다.

### 3. 수동 알림 발송

- 사용자는 연체된 인보이스가 있는 고객에게 수동으로 알림을 보낼 수 있습니다.
- 수동 발송 시:
  - 연락처를 선택하면 연락처 정보만 자동 입력됩니다.
  - 수동 메시지 작성기로 발송할 때는 동적 인보이스 필드(인보이스 금액, 마감일, 인보이스 번호)가 자동 입력되지 않습니다.
- 템플릿 불일치 문제를 피하기 위해 수동 알림 메시지는 꼭 필요한 경우에만 사용하는 것이 좋습니다.

### 4. 인보이스 마감일 설정

- 결제 설정(Payment Settings)에서 인보이스 마감일(Invoice Due After) 업데이트는 향후 인보이스에만 영향을 미칩니다.
- 이미 생성된 반복 인보이스를 포함한 기존 인보이스는 소급 업데이트되지 않습니다.
- 모든 변경사항은 업데이트 후 생성되는 인보이스에만 적용됩니다.

### 5. 주요 운영 참고사항

- 알림 자동화를 활성화하기 전에 생성된 인보이스와 관련된 알림은 수동으로 처리해야 합니다.
- 담당자는 수동 알림 이메일의 동적 필드 동작에 대해 사용자와 명확히 소통해야 합니다.
- 마감일 설정 변경은 소급 적용되지 않으므로, 정기 결제를 설정하는 사용자에게 명확히 안내해야 합니다.
