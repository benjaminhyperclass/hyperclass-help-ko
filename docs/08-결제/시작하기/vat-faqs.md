# VAT FAQ

#### 이 문서는 미국 외 고객에게만 적용됩니다.

에이전시의 세금 ID 정보 업데이트에 대한 자세한 안내는 [여기를 클릭하세요](../../11-설정/에이전시-설정/agency-tax-id-guide.md).

**Q: 하이레벨(Hyperclass)에서 VAT/GST/HST를 청구하는 이유가 무엇인가요?**

A: 하이레벨에서는 국제 세금 규정을 준수하기 위해 VAT/GST/HST를 청구하고 있습니다.

VAT/GST 적용 여부는 해당 국가의 관할권에 따라 달라질 수 있습니다. 대부분의 고객은 결제 프로필에 세금 ID 번호를 제공하지 않은 경우에만 VAT/GST가 청구됩니다. 일부 국가(멕시코, 스위스, 남아프리카공화국, 말레이시아, 인도네시아, 세르비아, 일부 캐나다 주 등)에서는 고객이 세금 ID를 제공했더라도 VAT/GST를 청구해야 합니다.

**Q: 세금 청구는 언제부터 시작되나요?**

A: 정확한 시기는 해당 국가의 관할권에 따라 다르지만, 2025년 초부터 인보이스에 VAT/GST 요금이 추가될 것으로 예상됩니다.

**Q: 모든 해외 고객에게 VAT/GST/HST가 청구되나요?**

A: 네, 하이레벨에서는 해당되는 모든 곳에서 VAT/GST를 청구할 예정입니다. 하지만 특정 예외가 적용될 수 있습니다. 예를 들어, 다음 상황에서는 VAT/GST가 징수되지 않습니다:

- 해당 국가가 등록된 사업체에 대한 판매세를 요구하지 않으며, 결제 프로필에 세금 ID 번호를 제공한 경우
- 하이레벨이 단독 재량으로 해당 거래에 VAT/GST가 적용되지 않는다고 판단한 경우. 세법은 수시로 변경될 수 있으며, 이러한 법률 변경이 청구되는 세금의 적용성 및/또는 금액에 영향을 미칠 수 있다는 점을 유의해 주세요.

**Q: 프로필에 세금 ID 번호를 제공했는데도 VAT/GST/HST가 청구되는 이유가 무엇인가요?**

A: 특정 국가에서는 세금 ID 번호를 제공한 B2B 고객을 포함하여 모든 고객에게 세금을 적용하도록 요구하고 있기 때문입니다. 이러한 국가에는 남아프리카공화국, 멕시코, 스위스, 세르비아, 말레이시아, 인도네시아, 일부 캐나다 주 등이 포함되지만 이에 국한되지 않습니다.

**Q: 프로필에 세금 ID 번호를 입력하려고 했는데 오류 메시지가 나타났습니다. 어떻게 해야 하나요?**

A: vat-tax@Hyperclass.com으로 이메일을 보내주세요.

**Q: "세금 ID 번호가 없습니다"에 체크했더니 VAT/GST/HST가 청구되고 있습니다. 하지만 최근에 세금 ID 번호를 받았습니다. 프로필에 세금 ID 번호를 어떻게 업데이트하나요?**

A: 하이레벨 플랫폼의 프로필 내 결제 세부사항으로 이동해 주세요: https://app.gohighlevel.com/settings/billing?tab=payments

**Q: "세금 ID 번호가 없습니다"에 체크했더니 VAT/GST/HST가 청구되고 있습니다. 하지만 최근에 세금 ID 번호를 찾았습니다. 이미 납부한 세금을 환불받을 수 있나요?**

A: 각 사례별로 검토하여 판단하겠습니다. vat-tax@Hyperclass.com으로 이메일을 보내주세요.

**Q: 추가 문의사항이 있으면 누구에게 연락해야 하나요?**

A: 세금의 성격이나 적용성에 대한 문의가 있으시면 회사의 세무 고문에게 문의하거나 vat-tax@Hyperclass.com으로 이메일을 보내주세요.
