# Xero 연동에서 "계정과목표" 전체 보기하는 방법

Xero와 연동한 후 CRM에서 계정과목표(Chart of Accounts)가 보이지 않는다면, 아래 단계를 따라 활성화하세요.

## 1단계: Xero에서 설정으로 이동

- Xero 대시보드에서 조직을 선택하세요.

- 드롭다운 메뉴에서 **Settings(설정)**을 클릭하세요.
![Xero 설정 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155052650330/original/5_XZCY86_ZHc2dPXrhldibblRqN87dogZA.png?1756455000)

## 2단계: 고급 설정 열기

- 조직 설정에서 아래로 스크롤하여 **Looking for advanced settings?(고급 설정을 찾고 계세요?)**을 클릭하세요.
![고급 설정 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155052650367/original/jBCzMxCrQomq1eUS8pNZtOUgBhzh22z9DA.png?1756455020)

## 3단계: 계정과목표 접근

- Advanced Accounting(고급 회계) 하에서 **Chart of accounts(계정과목표)**를 클릭하세요.
![계정과목표 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155052650378/original/xKkZ8mHLKOSt3o5JuFCY_pOU-V_6JpdnPg.png?1756455037)

## 4단계: 관련 계정 선택

- 계정 목록에서 연결하려는 계정을 선택하세요(예: Prepayments).

- **Edit(편집)**을 클릭하여 해당 계정의 세부 정보를 수정하세요.
![계정 편집](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155052650391/original/jbHFs4sPiZLTfRSD3pueoBxuMUYgOID8dg.png?1756455051)

## 5단계: 계정에 대한 결제 활성화

- 편집 팝업에서:

**Enable payments to this account(이 계정에 대한 결제 활성화)**에 체크 ✅ 하세요.

- **Save(저장)**를 클릭하세요.
![결제 활성화](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155052650409/original/hGFDs2N5-dDy7YwODn13cQrfz_fUPSys4g.png?1756455070)

## 6단계: CRM에서 연결

- CRM의 Xero 연동으로 돌아가세요.

- Connected Account(연결된 계정) 하에서 방금 활성화한 계정을 선택하세요(예: Prepayments).

![CRM 연동 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155052650454/original/gnKyh9uVAVrjp1XMuFF9QxOSivfo0RJgaw.png?1756455111)

✅ 완료되면 계정과목표가 CRM에서 보이게 되어 사용할 수 있게 됩니다.
