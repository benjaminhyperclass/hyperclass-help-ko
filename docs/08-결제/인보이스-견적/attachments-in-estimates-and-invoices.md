# 견적서와 인보이스의 첨부파일 기능

이제 **견적서, 인보이스, 정기 인보이스, 견적서 템플릿**에 직접 **최대 10개의 파일**을 첨부할 수 있습니다(**총 20MB 크기 제한**). 이 새로운 기능을 통해 사양서, 약관, 이미지 자료 등의 지원 문서를 포함시켜 처음부터 더욱 전문적이고 유익한 패키지를 고객에게 전달할 수 있습니다. 첨부파일 기능을 사용하여 고객 소통을 더욱 매끄럽게 만드는 방법을 알아보세요.

## 새로운 기능

- 이제 견적서에 최대 10개의 파일을 첨부할 수 있으며, 이는 이메일과 함께 전송됩니다.
- 견적서당 총 파일 크기는 20MB로 제한됩니다.
- 첨부파일 기능으로 사양서, 약관, 시각적 자료와 같은 추가 문서를 포함할 수 있는 유연성을 제공합니다.

## 지원 기능

- 인보이스
- 정기 인보이스
- 견적서
- 견적서 템플릿

## 사용법

- Payments(결제) 탭 아래의 Estimates(견적) 섹션으로 이동하세요.
- 견적서를 생성하거나 편집할 때 첨부파일 아이콘을 찾아보세요.

![견적서 첨부파일 아이콘](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155047518869/original/F0DzFLiBSv9-9JWfmR_JM4rUKWySC8FIWQ.jpeg?1748637862)

- 아이콘을 클릭하여 기기에서 파일을 업로드하세요.

![첨부파일 업로드 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155047518867/original/adNlY50FyXe_SG4UhF9C-Xipi4BCTBOELw.jpeg?1748637862)

- 모든 첨부파일의 총 크기가 20MB를 초과하지 않도록 주의하세요.
- 사용자는 이메일을 통해 첨부파일을 받게 됩니다.

## 개발 배경

- 상세한 지원 문서를 고객과 공유하는 과정을 간소화하기 위해서입니다.
- 견적서 내에서 전문적이고 포괄적인 패키지를 제공합니다.
- 추가 세부사항을 위한 반복적인 연락을 피하고, 사용자가 명확하게 소통할 수 있도록 돕습니다.

참고: 첨부파일은 인보이스를 처음 전송할 때만 함께 전송되며, 이후 예약된 발송에서는 포함되지 않습니다.

- 결제 일정이 있는 인보이스에서의 편집 기능
