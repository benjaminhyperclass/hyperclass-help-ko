# Hyperclass에 상품 대량 가져오기하는 방법

대량의 상품 재고를 관리하는 것은 시간이 많이 걸리는 작업이지만, Hyperclass에서는 간단한 CSV 파일을 사용하여 여러 상품을 한 번에 가져올 수 있어 매우 편리합니다. Shopify에서 상품을 옮기든 다른 출처에서 상품을 추가하든, 이 가이드에서는 Hyperclass로 상품을 대량 가져오기하는 전체 과정을 안내해드릴게요.

## 상품 대량 가져오기 사전 준비사항

시작하기 전에 다음 사항을 준비해주세요:

- Hyperclass의 Payments(결제) → Products(상품) 페이지에 접근 권한
- 상품 정보가 포함된 올바른 형식의 CSV 파일
- 참고용으로 Hyperclass에서 다운로드한 샘플 CSV 파일

## 상품 대량 가져오기 방법

### 1단계: 상품 페이지 접속

- Hyperclass 계정에 로그인하세요.
- 메인 대시보드에서 Payments(결제) → Products(상품)로 이동하세요.

![상품 페이지 접속](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155034842506/original/3umRXzbM9xZMwYElr-IR0-e7Hbp6ZE32EA.png?1729108671)

### 2단계: 샘플 CSV 다운로드

- 상품 페이지에서 'Import as CSV(CSV로 가져오기)' 버튼을 클릭하세요.
- 팝업이 나타나면 하단에 샘플 CSV 파일 다운로드 옵션을 찾을 수 있습니다.
- 필수 형식과 필드를 이해하기 위해 이 파일을 다운로드하세요.

![CSV 가져오기 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155034842586/original/q4bkzD20ec-QWmtOl0_3KdGjkQ_ztWP1EA.png?1729108801)

![샘플 CSV 다운로드](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155034842591/original/rLs6KJ1OgtwUTcZquwkTUYth8P4YeVaDUg.png?1729108811)

### 3단계: CSV 파일 준비

- Excel, Google 스프레드시트 또는 선호하는 스프레드시트 도구로 샘플 CSV 파일을 열어주세요.
- 각 상품에 대한 필수 필드를 작성하세요. 필수 필드는 다음과 같습니다:
  - 상품명(Product Name)
  - 가격(Price)
  - SKU
  - 설명(Description)
  - 카테고리(Category)
- 가져오기 오류를 방지하기 위해 각 필드가 샘플 파일의 형식과 일치하는지 확인하세요.
- 작성 완료된 파일을 .csv 형식으로 저장하세요.

**빠른 팁:** 중복된 SKU나 문제를 일으킬 수 있는 특수 문자가 있는지 데이터를 다시 한 번 확인해보세요.

샘플 파일의 일부 필드(무게, 재고, 이미지 소스 등)는 서비스를 가져오는 경우 해당되지 않을 수 있습니다. 괜찮습니다. 해당 필드는 공백으로 두되, 컬럼 헤더는 삭제하지 마세요. 형식 오류를 방지하기 위해 모든 컬럼 헤더는 그대로 유지해야 합니다.

![CSV 파일 준비](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155034842604/original/TUKWiKIpnKGr-TaN6BcEMpeeOjaYyUNiyA.png?1729108834)

### 4단계: CSV 파일 업로드

- Payments(결제) → Products(상품) 페이지로 돌아가세요.
- 'Import as CSV(CSV로 가져오기)' 버튼을 클릭하세요.
- 기기에서 준비된 CSV 파일을 선택하세요.
- 파일 내용의 미리보기가 표시됩니다.

### 5단계: 미리보기 및 확인

- 모든 상품 세부 정보가 정확한지 미리보기를 검토하세요.
- 문제가 있는 경우 CSV 파일에서 직접 수정한 후 다시 업로드할 수 있습니다.
- 모든 내용이 올바른지 확인하세요.

![미리보기 확인](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155034842623/original/bB7cV2piSf2tAfbJEcBqUXtr68Qw7TYcfQ.png?1729108852)

### 6단계: 상품 가져오기

- 'Import Products(상품 가져오기)'를 클릭하여 과정을 완료하세요.
- 가져오기가 성공하면 Hyperclass에서 상품을 사용할 준비가 완료됩니다.

## 주요 참고사항 및 팁

- **오류:** CSV 파일에 문제가 있는 경우, 수정해야 할 내용에 대한 세부사항이 포함된 오류 메시지가 표시됩니다. 파일의 오류를 수정한 후 다시 업로드하세요.

- **기존 상품 업데이트:** 같은 과정을 사용하여 상품 세부 정보를 업데이트할 수 있습니다. 원활한 업데이트를 위해 SKU가 일치하는지 확인하세요.

- **제한사항:** Hyperclass는 CSV 업로드에 최대 파일 크기 제한이 있을 수 있습니다. 업로드 전에 파일 크기를 확인하세요.

## 일반적인 문제 해결

대량 가져오기 중에 사용자가 겪을 수 있는 일반적인 문제들을 살펴보겠습니다:

- **파일 형식 오류:** 파일이 .csv로 저장되었고 각 필드가 샘플 형식과 일치하는지 확인하세요.

- **필수 필드 누락:** 모든 필수 필드(예: 상품명, SKU, 가격)가 완성되었는지 다시 확인하세요.

- **특수 문자:** 가져오기 중에 문제를 일으킬 수 있는 특수 문자를 제거하세요.

**참고:** 스스로 해결할 수 없는 문제가 발생하면 Hyperclass 지원팀에 추가 도움을 요청하세요!

## 자주 묻는 질문

**Q: CSV 가져오기를 사용하여 기존 상품을 업데이트할 수 있나요?**

아니요, CSV 가져오기로는 기존 상품을 업데이트할 수 없습니다.

**Q: 가져오기 과정에서 CSV 파일에 오류가 있으면 어떻게 되나요?**

화면에 오류 메시지가 표시되며, 이를 해결한 후 파일을 다시 업로드해야 합니다.

**Q: 하나의 CSV 파일에서 가져올 수 있는 상품 수에 제한이 있나요?**

10MB까지 가능합니다.

**Q: 물리적 상품이 아닌 서비스를 가져오는 경우, 무게나 재고와 같이 해당되지 않는 필드는 어떻게 해야 하나요?**

물리적 상품이 아닌 제품(서비스 등)을 가져오는 경우, 해당되지 않는 필드(무게, 치수, 이미지 소스, 재고 등)는 공백으로 둘 수 있지만, CSV 파일에 해당 컬럼 헤더는 반드시 포함해야 합니다.

컬럼을 완전히 제거하거나 헤더 이름을 변경하면 "Column 1 is not correct"와 같은 오류가 발생합니다. 일부 값이 비어 있더라도 항상 샘플 CSV의 정확한 컬럼 구조를 따라주세요.

**Q: CSV 파일을 가져오면 Hyperclass의 기존 상품 데이터를 덮어쓰나요?**

아니요, 이 기능으로는 상품을 업데이트할 수 없으므로 덮어쓰기가 불가능합니다.
