# 쿠폰 시작하기

## 개요

쿠폰(Coupons)을 사용하면 상품에 할인을 제공하고 결제 전환율을 높일 수 있습니다. 일회성 상품과 정기 구독 상품에 적용되는 쿠폰을 만들고, 언제, 어디서, 얼마나 자주 사용할 수 있는지 제어할 수 있습니다.

쿠폰은 다음 곳에서 사용 가능합니다:

- 퍼널(주문 양식)
- 스토어
- 웹사이트
- 결제 링크(Payment Links)
- 캘린더
- 서비스(Services)
- 기프트 카드 판매 페이지
- 비즈니스용 구독 생성

⚠️ 쿠폰은 견적서와 인보이스에서는 지원되지 않습니다. PayPal로 결제하는 구독 상품의 경우 일부 쿠폰 설정이 지원되지 않습니다.

## 이 기능은 무엇을 하나요?

쿠폰을 사용하면 다음이 가능합니다:

- 퍼센트 또는 고정 금액 할인 제공
- 전체 장바구니 또는 특정 상품에 할인 적용
- 사용 횟수 및 제한 조건으로 사용 통제
- 구독 상품에 할인 적용 (적용 기간 제어 가능)
- 사용량 및 사용 내역 추적

## 쿠폰 사용에 필요한 것은?

- Payments(결제) → Coupons(쿠폰) 메뉴 접근 권한
- 쿠폰이 지원되는 결제 페이지 (상품 판매 중)

## 쿠폰 만드는 방법

### 1단계: 쿠폰 페이지로 이동

다음 경로로 이동하세요:
Payments(결제) → Coupons(쿠폰) → Create Coupon(쿠폰 생성)

![쿠폰 메뉴 이동](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155067095692/original/bXahrvIuWIAvgNTrKzos4JrrvBDe-9cLFw.png?1773739904)

![쿠폰 생성 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155067274628/original/JkrjdFQtc4tIxL3Rnnt7W-X79xocw9fYFA.png?1773901987)

### 2단계: 기본 정보 입력

#### 쿠폰 이름

- 쿠폰을 구분하기 위한 내부 참조 이름
- 예시: 여름 할인 2025

#### 쿠폰 코드

- 고객이 결제 시 입력할 코드
- 조건:
  - 영문과 숫자만 가능
  - 공백이나 특수문자 불가
  - 대소문자 구분 없음 (자동으로 대문자로 변환)
- Generate(생성) 버튼을 클릭하면 자동으로 코드가 생성됩니다

![쿠폰 기본 정보](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155067298017/original/jqwGXNit07EQDY8NMffBuG4sCCftLNnu9Q.png?1773919127)

### 3단계: 쿠폰 종류 선택

두 가지 유형 중 선택할 수 있습니다:

#### 1. 퍼센트 할인 쿠폰

- 전체 금액에 퍼센트 할인 적용
- 예시:
  - 장바구니 = $100
  - 쿠폰 = 25%
  - 최종 금액 = $75

![퍼센트 할인 쿠폰](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155067298055/original/kYtJhdCVYauIBfmA1zyuzVSrxUYWbC9ELw.png?1773919160)

#### 2. 고정 금액 할인 쿠폰

- 주문 금액에 관계없이 고정 금액 할인
- 예시:
  - 장바구니 = $100
  - 쿠폰 = $25
  - 최종 금액 = $75

![고정 금액 할인 쿠폰](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155067298093/original/36kr_pbEV_6P6fcQaSZQLfPsh137mTXx1g.png?1773919196)

## 쿠폰 통화

- 쿠폰 통화는 하위 계정의 기본 통화로 자동 설정됩니다
- 수동으로 변경할 수 없습니다

### 4단계: 유효 기간 설정 (선택사항)

쿠폰을 언제 사용할 수 있는지 정의할 수 있습니다:

- 시작 날짜 및 시간 → 쿠폰이 활성화되는 시점
- 종료 날짜 및 시간 → 쿠폰이 만료되는 시점

종료 날짜를 설정하지 않으면 → 쿠폰이 무제한으로 활성화됩니다

![쿠폰 유효 기간](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155067298143/original/-nlSVBYAQMFWsJRrktsurBBaUzE9ICniew.png?1773919223)

### 5단계: 사용 제한 설정

다음 제한 조건을 적용할 수 있습니다:

#### 1. 총 사용 횟수 제한

- 쿠폰을 사용할 수 있는 총 횟수 제한
- 예시: 100회 사용으로 제한

#### 2. 특정 상품으로 제한

- 특정 상품에만 쿠폰 사용 제한
- 예시:
  - "프리미엄 플랜"에만 쿠폰 적용
  - 다른 상품에는 사용 불가
- 상품 전체가 아닌 특정 가격이나 옵션으로만 제한할 수도 있습니다

![상품별 쿠폰 제한](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155067297860/original/rpJemNEmBQm5_GgtsWaZrEn5Ls87WvxjFg.png?1773918991)

![상품 선택 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155067297906/original/-SC0iex1rPRmyAEGlC-mW7aHHqXxtprvpg.png?1773919028)

#### 3. 고객당 1회 사용 제한

- 한 고객이 같은 쿠폰을 반복 사용하지 못하도록 제한

### 6단계: 구독 적용 방식 설정 (해당하는 경우)

정기 구독 상품의 경우, 할인이 얼마나 오래 적용될지 제어할 수 있습니다:

#### 미래 결제에 적용

다음 중 선택할 수 있습니다:

- Forever(무제한) → 모든 결제 주기에 할인 적용
- Multiple Months(여러 달) → 정해진 기간 동안만 할인 적용

예시:
- 구독료 = 월 $100
- 쿠폰 = 20% 할인, 3개월간
- 고객은 첫 3개월 동안 월 $80 결제
- 그 후에는 월 $100으로 복원

참고: 결제 주기와 관계없이 적용됩니다

![구독 할인 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155067298181/original/cucHH0eRoLNkVRF2swJ0Z7MZAXMhjwVhOQ.png?1773919266)

### 7단계: 쿠폰 생성

Create(생성) 버튼을 클릭하세요.

설정에 따라 쿠폰이 바로 활성화되거나 예약됩니다.

## 결제 시 쿠폰 작동 방식

- 유효한 쿠폰이 있을 때만 쿠폰 입력 필드가 나타납니다
- 적용 가능한 쿠폰이 없으면 → 필드가 자동으로 숨겨집니다
- 할인은:
  - 결제 요약에 표시
  - 해당 상품에 비례적으로 적용
  - 주문 상세 내역에서 확인 가능
- 많은 결제 페이지(퍼널, 결제 링크 등)에서는 빌더에서 결제 시 쿠폰 표시 및 적용 여부를 설정할 수 있습니다

## 중요한 동작 방식 및 제한사항

- 쿠폰이 적용되는 곳:
  - 일회성 상품
  - 정기 구독 상품

- 쿠폰이 적용되지 않는 곳:
  - 원클릭 업셀(퍼널의 경우)

- 쿠폰으로 장바구니 금액을 $0으로 만들 수 있습니다
  - 빌더 레벨에서 이 설정이 체크되어 있으면, 고객은 여전히 카드 정보를 입력해야 합니다 (확인용)
  - 빌더에 $0 거래 관리 설정이 없으면 항상 카드 정보가 필요합니다

- PayPal을 제외한 대부분의 결제 수단에서 100% 할인을 지원합니다

- 쿠폰 코드는:
  - 대소문자 구분하지 않음
  - 고유해야 함

- 할인은 상품 간에 비례적으로 분배됩니다

## 자주 묻는 질문

### 쿠폰으로 무료 상품을 만들 수 있나요?

네, 100% 할인 쿠폰을 사용하면 됩니다.

### 쿠폰을 나중에 수정하거나 삭제할 수 있나요?

네, 언제든지 쿠폰을 수정, 삭제하거나 사용 내역을 확인할 수 있습니다.

### 하나의 주문에서 일회성 상품과 구독 상품 모두에 쿠폰을 적용할 수 있나요?

네. 쿠폰은 같은 주문 내의 일회성 상품과 정기 구독 상품 모두에 적용됩니다. 구독의 경우, 설정된 기간(1회, 여러 달, 또는 무제한)에 따라 쿠폰이 적용됩니다.

### 수량이 여러 개인 상품에 쿠폰을 적용하면 어떻게 되나요?

할인은 해당 상품의 총 금액(수량 × 가격)을 기준으로 계산됩니다. 예를 들어, 상품 가격이 $20이고 고객이 5개를 구매하면서 10% 쿠폰을 사용하면, $10(총 $100의 10%)가 할인됩니다.

### 이미 사용된 쿠폰을 변경할 수 있나요?

일부 쿠폰 설정(이름, 제한, 유효 기간)은 수정할 수 있지만, 변경사항은 향후 사용에만 적용됩니다. 쿠폰이 이미 적용된 과거 주문은 변경되지 않습니다.

### 쿠폰이 설정 수수료에도 적용되나요?

아니요. 구독 상품의 경우 쿠폰은 설정 수수료에는 적용되지 않고, 실제 구독 결제 금액에만 적용됩니다.

### PayPal에서 지원되지 않는 쿠폰 설정은 무엇인가요?

다음 설정은 PayPal에서 지원되지 않습니다:

1. '무제한(Forever)' 대신 특정 개월 수로 설정한 구독 할인은 PayPal에서 작동하지 않습니다. 정해진 개월 수가 아닌 전체 구독 기간에 대해 요금이 청구됩니다.

2. 주문 금액을 $0으로 만드는 쿠폰(100% 할인 또는 최종 주문 금액이 $0이 되는 고정 금액 쿠폰)은 구독 상품이 포함된 결제에서 PayPal과 작동하지 않습니다.
