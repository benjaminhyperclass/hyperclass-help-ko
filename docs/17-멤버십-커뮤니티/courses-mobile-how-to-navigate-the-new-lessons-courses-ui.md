# 모바일 강의 - 새로운 레슨 및 강의 UI 사용법

## **1. 매끄러운 카테고리 및 레슨 탐색**

같은 화면에서 카테고리와 레슨 간에 앞뒤로 이동할 필요 없이 매끄럽게 전환할 수 있습니다. 작동 방식은 다음과 같습니다:

- 확장: 카테고리나 레슨을 더 자세히 탐색하여 콘텐츠를 상세히 살펴보세요.

- 축소: 핵심에 집중하고 싶을 때 빠른 개요를 확인하세요.

이를 통해 레슨과 카테고리 간 탐색이 더욱 쉽고 효율적으로 이루어집니다.

## **2. 모든 레슨 유형에서 손쉬운 학습**

텍스트, 오디오, 비디오 등 어떤 유형의 레슨을 수강하든, 새로운 인터페이스가 학습 경험을 단순화합니다:

- 텍스트 레슨: 모든 화면 크기에서 쉽게 읽을 수 있습니다.

- 오디오 레슨: 매끄러운 재생과 직관적인 컨트롤을 제공합니다.

- 비디오 레슨: 전체 화면 보기와 반응형 디자인으로 끊김 없는 학습을 제공합니다.

## **3. 완전히 새로운 댓글 섹션**

새롭게 디자인된 댓글 섹션에서 동료들과 소통하세요:
- 직접 답글: 개별 댓글에 쉽게 응답하여 대화의 흐름을 유지할 수 있습니다.
- 향상된 상호작용: 개선된 레이아웃으로 가독성과 상호작용이 향상되어 더 매끄러운 대화를 나눌 수 있습니다.

## **4. 세련된 퀴즈 및 과제 인터페이스**

잘린 콘텐츠와 혼란스러운 전환은 이제 안녕입니다. 이제 다음이 가능합니다:

- 집중력을 잃지 않고 질문 간에 매끄럽게 전환할 수 있습니다.

- 과제와 퀴즈가 더욱 명확하고 탐색하기 쉬워져, 더 간소화된 경험을 제공합니다.

## **5. 빠른 상태 개요**

새로운 상태 추적기로 진도를 한눈에 파악하세요:

- 한 눈에 보기: 퀴즈와 과제가 검토 중인지, 통과했는지, 개선이 필요한지 빠르게 확인할 수 있습니다.

- 명확한 색상 코드: 각 과제의 상태를 쉽게 확인하여 더 나은 관리가 가능합니다.

## **6. 로더 대신 시머 효과**

기존의 로딩 스피너를 세련된 시머링 플레이스홀더로 교체했습니다:

- 모던한 외관: 지루한 스피너는 이제 그만—이제 콘텐츠가 스타일리시하게 로드됩니다.

- 매끄러운 로딩: 레슨, 퀴즈, 과제가 로드되는 동안 시각적으로 매끄러운 경험을 즐기세요.

## **7. 코드 구조 및 성능 향상**

성능 개선과 유지보수성을 위해 백엔드 업데이트를 진행했습니다:

- 개선된 코드 구조: 더 나은 코드 가독성과 구조를 위해 Vue 3 composition API로 업데이트했습니다.

- 번들 크기 감소: 큰 페이지를 작은 컴포넌트로 분할하고 지연 로딩을 적용하여 앱의 초기 로딩 시간을 단축했습니다.

## **8. 통합 로그인 및 네비게이션 업데이트**

- **통합 로그인 흐름:** 로그인 후, 하나의 매끄러운 화면에서 가입한 모든 커뮤니티와 강의에 접근할 수 있습니다.

- **하단 네비게이션 바:** 하단 네비게이션을 사용하여 한 번의 탭으로 강의와 커뮤니티 간에 전환할 수 있습니다.

- **간소화된 그룹 전환기:** 가입한 그룹 간 전환이 더욱 깔끔하고 빨라졌으며, 공간 간 빠른 탐색을 위해 설계되었습니다.

### 시작하는 방법

- 이러한 흥미진진한 새로운 기능을 경험하려면 앱을 최신 버전으로 업데이트하세요.

- 강의를 탐색하고 새로운 댓글 섹션을 시도해보며, 퀴즈와 과제의 상태를 확인해보세요.
