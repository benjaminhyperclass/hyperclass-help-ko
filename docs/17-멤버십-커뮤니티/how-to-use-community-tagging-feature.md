# 커뮤니티 태깅 기능 사용법

플랫폼에서 커뮤니티 회원들과 소통하는 것이 더욱 쉬워졌습니다! 새로운 태깅 기능을 통해 사용자는 게시글과 댓글 모두에서 특정 회원을 언급할 수 있습니다. 이는 더 나은 참여를 촉진할 뿐만 아니라 대화를 체계적으로 유지하는 데 도움이 됩니다.

이 기능을 효과적으로 활용하는 방법을 알아보세요:

## 1. 게시글에서 태깅하기

- **태그 시작하기:** 게시글을 작성하는 동안 "@" 기호를 입력한 후 회원 이름의 첫 글자들을 입력하세요.
- **회원 선택하기:** 입력하면 입력한 글자를 기반으로 관련 회원 목록이 표시됩니다. 원하는 이름으로 이동해서 클릭하거나 화살표 키를 사용해 강조 표시하세요. 이름에 마우스를 올리면 파란색으로 강조 표시됩니다.
- **태그 삽입하기:** 회원의 이름을 선택하면 게시글 내용에 강조 표시된 태그로 나타납니다.
- **게시하기:** 게시글을 완성한 후 게시하세요. 태그된 회원들은 게시글에서 언급되었다는 알림을 받게 됩니다.

![게시글에서 태깅하는 방법](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155009090586/original/Zeq2Kp4i7qCWhmM2yZ2F4pZnfERGSgFFLg.png?1696328466)

## 2. 댓글에서 태깅하기

- **태그 시작하기:** 댓글을 작성할 때 "@" 기호를 입력한 후 회원 이름의 일부를 입력하세요.
- **회원 선택하기:** 입력 내용을 바탕으로 관련 회원들이 드롭다운으로 표시됩니다. 회원 이름에 마우스를 올리면 파란색으로 강조 표시됩니다. 클릭해서 원하는 회원을 선택하세요.
- **태그 삽입하기:** 선택한 회원의 이름이 댓글 내용에 강조 표시된 태그로 삽입됩니다.
- **게시하기:** 댓글 작성을 완료했다면 게시하세요. 태그된 회원들은 언급되었다는 알림을 받게 됩니다.
- **댓글 답글에서 태깅하기:** 태깅 기능은 새 댓글에만 제한되지 않습니다. 기존 댓글의 답글에서도 회원을 태그할 수 있습니다.

## 3. 태그 이메일 알림

- **알림 받기:** 누군가 게시글이나 댓글에서 당신을 태그하면, Hyperclass가 이메일 알림을 보내서 알려줍니다.
- **직접 이동:** 이 알림에는 당신이 태그된 게시글이나 댓글로 바로 이동할 수 있는 직접 링크가 포함되어 있어, 대화에 원활하게 참여할 수 있습니다.

## 자주 묻는 질문

### 커뮤니티(Communities)의 태깅 기능이 뭔가요?

태깅 기능은 사용자가 게시글과 댓글에서 특정 회원을 언급할 수 있게 해주는 기능으로, 참여도와 대화의 체계성을 향상시킵니다.

### 게시글에서 누군가를 태그하려면 어떻게 해야 하나요?

게시글을 작성할 때 "@" 기호를 입력한 후 회원 이름의 첫 글자들을 입력하세요. 관련 회원 목록이 나타나면 원하는 이름을 클릭해서 태그하세요.

### 댓글에서도 회원을 태그할 수 있나요?

네, 태깅 기능은 새 댓글과 기존 댓글의 답글 모두에서 작동합니다. "@" 기호를 입력한 후 회원 이름을 입력하기만 하면 됩니다.

### 누군가 저를 태그했을 때 어떻게 알 수 있나요?

게시글이나 댓글에서 태그되면 이메일 알림을 받게 됩니다. 이 이메일에는 언급된 콘텐츠로 바로 이동할 수 있는 직접 링크도 포함되어 있습니다.

### 태깅할 때 회원 이름에 마우스를 올리면 어떻게 되나요?

제안 목록에서 회원 이름에 마우스를 올리면 이름이 파란색으로 강조 표시되어 선택 가능하다는 것을 나타냅니다.

### 하나의 댓글이나 게시글에서 여러 회원을 태그할 수 있나요?

물론입니다! 하나의 댓글이나 게시글에서 원하는 만큼 많은 회원을 태그할 수 있어, 토론에 여러 커뮤니티 회원들을 쉽게 참여시킬 수 있습니다.

### 태깅 기능이 커뮤니티 참여에 어떤 도움이 되나요?

태깅은 커뮤니티 내에서 더 직접적이고 개인화된 상호작용을 가능하게 합니다. 회원들을 태그함으로써 관련 콘텐츠를 확실히 보게 하고, 토론에 참여하게 하며, 커뮤니티에 계속 참여하도록 할 수 있습니다.

### 태그된 회원의 이름이 콘텐츠에서 다르게 표시되나요?

네, 회원의 이름을 선택하면 콘텐츠에서 태그로 강조 표시되어 나머지 텍스트와 구별됩니다.

### 태깅 기능에 문제가 있을 때 누구에게 문의하나요?

태깅 기능에 문제가 있거나 추가 질문이 있으시면 지원팀에 도움을 요청해 주세요.
