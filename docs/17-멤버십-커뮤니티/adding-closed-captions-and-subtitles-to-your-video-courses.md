## 자막과 캡션 지원이란?

자막과 캡션 지원은 Hyperclass 강의 빌더(Course Builder)의 기능으로, 강의 비디오에 자막 파일을 업로드할 수 있게 해줍니다. 여러 언어로 자막을 제공할 수 있으며, 플랫폼 내에서 직접 편집도 가능합니다. 시각적 편집기와 Raw 편집기 모두 제공하여 접근성을 높이고, 법적 요구사항을 충족하며, 텍스트 보완이 필요한 더 많은 학습자에게 도움을 줄 수 있습니다.

## 자막과 캡션 지원의 주요 장점

자막 지원은 다양한 언어의 캡션을 포함하여 모든 사용자의 이해도를 높이고 강의 접근성을 향상시킵니다.

- **여러 자막 언어를 지원하여 글로벌 사용자에게 다가가세요.**

- **청각 장애인 사용자의 콘텐츠 접근성을 개선하세요.**

- **시각적 보강을 통해 기억력과 이해력을 높이세요.**

- **실시간 자막 문제 감지로 오류를 찾아 수정하세요.**

- **수동 또는 시각적 편집으로 명확성과 정확성을 보장하세요.**

- **비디오 콘텐츠와 자막이 정확히 동기화되도록 하세요.**

## Hyperclass에서 자막 업로드하기

표준 .srt 파일을 사용해 비디오 레슨에 자막을 업로드하여 여러 언어의 캡션을 빠르게 활성화할 수 있습니다.

**강의 비디오에 자막을 업로드하려면:**

- **Sites(사이트) > Courses(강의)로 이동해 편집할 강의를 열어주세요.**

- **비디오가 포함된 레슨을 클릭하세요.**

- **비디오 플레이어 옆의 Upload Subtitles(자막 업로드) 옵션을 찾으세요.**

- **Upload Subtitle(자막 업로드)을 클릭하고 기기에서 .srt 파일을 선택하세요.**

- **드롭다운 목록에서 적절한 언어를 선택하세요.**

- **Save(저장)를 클릭해 자막 파일을 비디오에 연결하세요.**

## 고급 자막 편집기 사용하기

실시간 오류 감지와 시각적 타임스탬프 탐색 기능이 있는 Hyperclass의 고급 편집기로 자막을 편집하고 개선하세요.

**자막 파일을 업로드한 후 내장 편집기로 개선할 수 있습니다:**

- **업로드된 파일 옆의 Edit Subtitles(자막 편집) 버튼을 클릭하세요.**

- **타임스탬프와 텍스트 내용이 포함된 구조화된 목록으로 자막을 확인하세요.**

- **타임스탬프를 클릭해 특정 비디오 순간으로 쉽게 이동하세요.**

- **문제가 있는 경우(겹치는 줄이나 공백 등) 경고 아이콘이 나타납니다.**

- **경고 아이콘에 마우스를 올려 문제 설명을 확인하세요.**

- **텍스트를 직접 클릭해 수정한 후 Save Changes(변경사항 저장)를 선택하세요.**

## Raw 모드 편집

Raw 모드는 정확성이 필수적일 때 텍스트와 타이밍을 완전히 제어할 수 있도록 자막을 수동으로 조정할 수 있게 해줍니다.

**Raw 모드는 자막 파일 구조에 직접 접근하고 싶을 때 유용합니다:**

- **자막 편집기에서 Switch to Raw Mode(Raw 모드로 전환) 버튼을 클릭하세요.**

- **표준 .srt 형식(타임스탬프 다음에 자막 텍스트)을 사용해 파일을 편집하세요.**

- **형식 오류를 방지하기 위해 구문을 주의 깊게 검토하세요.**

- **완료되면 Save(저장)를 클릭해 변경사항으로 비디오를 업데이트하세요.**

## 자주 묻는 질문

**Q: 어떤 자막 형식을 지원하나요?
현재 Hyperclass는 .srt 파일만 지원합니다.**

**Q: 한 개 이상의 언어로 자막을 업로드할 수 있나요?
네, 가능합니다. 여러 .srt 파일을 업로드하되 각각 다른 언어로 할당하면, 시청자가 선호하는 옵션을 선택할 수 있습니다.**

**Q: 자막이 비디오와 자동으로 동기화되나요?
자막은 .srt 파일에 제공된 타임스탬프를 기준으로 동기화됩니다. 필요시 자막 편집기에서 타이밍을 조정할 수 있습니다.**

**Q: 자막 편집에 Raw 모드가 필수인가요?
아닙니다. Raw 모드는 선택사항으로 수동 편집을 선호하는 사용자를 위한 것입니다. 시각적 편집기가 사용자 친화적인 대안을 제공합니다.**

**Q: 학습자가 비디오를 보면서 자막 언어를 선택할 수 있나요?
네, 가능합니다. 시청자는 플레이어 컨트롤을 사용해 사용 가능한 자막 언어 간에 전환할 수 있습니다.**

**Q: 자막 파일에 오류가 있으면 어떻게 되나요?
시스템이 겹치는 줄이나 누락된 타임스탬프에 대해 경고 아이콘을 표시합니다. 저장하기 전에 편집기에서 이를 수정할 수 있습니다.**
