# 커뮤니티 고급 강의 잠금 해제 옵션

## 개요

유연한 강의 잠금 해제 시스템을 도입하여 커뮤니티 관리자가 회원들의 강의 접근 방식을 맞춤 설정할 수 있게 되었습니다. 이제 커뮤니티의 참여 모델과 수익화 전략에 맞춰 접근 규칙을 설정할 수 있어요.

## 강의 잠금 해제 옵션

다음 강의 접근 옵션 중에서 선택할 수 있어요:

### 1. 모든 회원

- 모든 커뮤니티 회원에게 공개 접근
- 제한 없음: 커뮤니티 내 누구나 수강 신청 가능

### 2. 레벨 잠금 해제

- 회원이 특정 리더보드 레벨에 도달해야 강의 접근 가능
- 커뮤니티 내 참여와 활동을 장려하는 옵션
- 관리자가 필요한 리더보드 레벨을 정의할 수 있음

### 3. 바로 구매

- **일회성 구매** 또는 **구독(Subscription)** 방식으로 강의 제공
- 회원이 구매 완료 후 즉시 강의 접근 가능
- 관리자가 가격, 결제 옵션, 구독 세부사항 설정 가능

### 4. 시간 잠금 해제

- 회원 가입일로부터 설정된 기간 후 강의 이용 가능
- 체계적인 학습 경로나 드립 콘텐츠(Drip Content) 전략에 이상적
- 관리자가 접근 권한이 부여되기까지의 일수를 정의 가능

## 강의 접근 설정 방법

강의 잠금 해제 옵션을 설정하려면 다음 단계를 따라하세요:

1. **강의 설정으로 이동**
   - 커뮤니티(Community) 관리자 패널로 이동
   - 설정하려는 강의를 선택하세요

![커뮤니티 강의 설정 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155041845121/original/nu6Pt958RQP4eMyp3W_v3NL2neXYdcZ9Kw.png?1739942888)

2. **잠금 해제 옵션 선택**
   
   Access Settings(접근 설정) 하에서 4가지 잠금 해제 옵션 중 하나를 선택하세요:

   - **모든 회원**: 추가 설정 불필요
   - **레벨 잠금 해제**: 필요한 리더보드 레벨 입력
   - **바로 구매**: 가격 설정 및 결제 옵션 선택
   - **시간 잠금 해제**: 가입 후 며칠 뒤 접근 허용할지 정의

3. **변경사항 저장**
   
   Save(저장) 버튼을 클릭하여 설정을 적용하세요. 이제 회원들은 선택한 옵션에 따라 접근 제한을 보게 됩니다.

## 자주 묻는 질문(FAQ)

### Q: 하나의 강의에 여러 잠금 해제 옵션을 적용할 수 있나요?

A: 아니요, 강의마다 하나의 잠금 해제 방법만 선택해야 해요.

### Q: 강의를 시작한 후에도 잠금 해제 설정을 변경할 수 있나요?

A: 네, 가능해요. 하지만 변경사항은 새로운 수강 신청에만 적용됩니다. 기존 회원들은 현재 접근 권한을 유지하게 돼요.
