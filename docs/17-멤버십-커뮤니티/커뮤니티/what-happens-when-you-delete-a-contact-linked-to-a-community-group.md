# 커뮤니티 그룹에 연결된 연락처를 삭제하면 어떻게 되나요?

## 연락처가 그룹 멤버인 경우

- 커뮤니티에서 연락처가 삭제되면, 해당 개인은 커뮤니티에서 완전히 제거되며 모든 콘텐츠와 기능에 대한 접근 권한을 잃게 됩니다.
- 이는 해당 사용자가 더 이상 커뮤니티 내의 게시물, 업데이트, 토론을 볼 수 없고, 게시물을 작성하거나 어떤 방식으로든 상호작용할 수 없음을 의미합니다.
- **삭제는 되돌릴 수 없으며**, 해당 개인을 모든 커뮤니티 활동과 소통에서 완전히 차단하는 효과를 가집니다.

## 연락처가 그룹 소유자인 경우

- 만약 해당 사용자가 커뮤니티 내 어떤 그룹이나 로케이션의 소유자라면, 삭제할 수 없다는 점을 주의해야 합니다.
- 그룹이나 로케이션의 소유자 상태는 해당 사용자에게 특정 수준의 권한과 통제권을 부여하므로, 이들의 제거를 방지하여 커뮤니티 내 특정 영역의 연속성과 관리를 보장합니다.
- 따라서 이러한 사용자를 삭제하려는 시도는 차단되며, 그들이 관리하는 그룹이나 로케이션과 관련된 접근 권한과 책임이 보존됩니다.
