# 학습 탭 (강의)

학습 탭의 무료 강의 소개: 관리자와 소유자는 이제 무료 강의를 추가하여 커뮤니티 회원들이 가치 있는 콘텐츠에 쉽게 접근할 수 있도록 제공할 수 있습니다.

이 문서에서는

- [강의 추가하기](#강의-추가하기)
- [강의 삭제하기](#강의-삭제하기)
- [강의 접근 권한](#강의-접근-권한)
- [알림](#알림)

## 강의 추가하기

- 관리자와 소유자는 "강의 추가(Add Courses)"를 클릭하여 학습 탭에 무료 강의를 손쉽게 추가할 수 있습니다
- **[여기를 클릭하여](https://community.hyperclass.ai/courses/offers/a79d58bf-4d8d-4842-9efd-2194329404f5) 하이레벨 멤버십/강의 웹사이트 구축 방법을 학습하세요. 강의 신청 페이지로 이동합니다.**
- 추가하고 싶은 강의를 선택합니다
- 강의가 추가되면 페이지에서 강의를 이용할 수 있습니다

## 강의 삭제하기

강의를 제거하려면 삭제 버튼을 클릭하세요.

![강의 삭제 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155010513815/original/pRvq1bPTr4bPtP80StBq5nAEKIGouiQuZQ.png?1697702467)

## 강의 접근 권한

관리자나 소유자가 그룹의 학습 탭에 강의를 추가하면, 해당 그룹의 모든 회원이 즉시 강의에 접근할 수 있습니다.

## 알림

관리자나 소유자가 그룹에 새 강의를 추가할 때마다 모든 사용자에게 이메일로 알림이 전송됩니다.
