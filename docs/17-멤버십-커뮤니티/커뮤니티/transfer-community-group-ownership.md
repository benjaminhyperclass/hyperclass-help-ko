# 커뮤니티 그룹 소유권 이전

더 나은 커뮤니티 경험을 제공하기 위한 셀프 서비스 옵션을 새롭게 출시했습니다.

## **새로운 기능:**

- **소유권 이전(Transfer Ownership):** 이제 그룹 소유자가 해당 로케이션에 접근 권한이 있는 다른 에이전시 멤버에게 소유권을 이전할 수 있습니다.
활용 예시: 저희 비즈니스는 고객을 대신해서 커뮤니티 그룹을 만든 후 소유권을 고객에게 이전하는 것이 주요 업무입니다. 이렇게 하면 고객이 그룹의 소유자가 되고, 제 이름이 생성자로 표시되지 않습니다.

- **다음으로 로그인(Login As):** 에이전시나 계정 사용자/소유자로서, 이제 에이전시 사용자와 연결된 연락처 목록에서 특정 연락처로 그룹에 로그인할 수 있습니다.
연락처 목록은 에이전시/로케이션 사용자와 연결된 연락처가 2개 이상 있을 때 표시됩니다.
- 목록에서 연락처를 클릭하면 해당 연락처로 그룹에 로그인됩니다.

- **연락처 링크:** 그룹 카드에서 소유자 이름을 클릭하면 해당 소유자의 연락처 스마트 리스트 페이지가 열립니다.
