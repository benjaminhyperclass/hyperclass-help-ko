## 비공개 채널과 채널 이모지란?

비공개 채널은 승인된 멤버만 보고 참여할 수 있도록 접근을 제한하여, 유료 그룹이나 프리미엄 코호트를 운영할 수 있게 해줍니다. 채널 이모지는 채널 이름 옆에 나타나는 작은 시각적 식별자로, 목록과 헤더에서 채널을 쉽게 구분할 수 있도록 도와줍니다. 이러한 기능들은 모바일과 웹 버전 모두에서 사용할 수 있으며, 커뮤니티의 개인정보 보호와 커스터마이징 옵션을 향상시킵니다.

## 비공개 채널과 채널 이모지의 주요 장점

- **멤버 통제**: 채널을 보고 게시할 수 있는 사용자를 정확히 승인할 수 있습니다.
- **집중된 대화**: 민감한 콘텐츠나 유료 콘텐츠를 공개 채팅과 분리하여 관리할 수 있습니다.
- **유연한 개인정보 보호**: 필요에 따라 채널을 공개와 비공개 사이에서 전환할 수 있습니다.
- **빠른 스캔**: 이모지가 목록과 알림에서 채널 인식을 향상시킵니다.

## 비공개 채널 만들기

처음부터 제한된 접근이 필요한 경우 새 채널을 비공개로 시작하세요.

**중요 참고사항:**
1. 그룹 소유자와 채널 생성자는 자동으로 채널 관리자로 추가됩니다.
2. 채널 관리자는 해당 특정 채널의 관리자 역할을 합니다. 소유자나 그룹 관리자와 같은 그룹 역할은 비공개 채널 내에서는 의미가 없습니다.
3. 그룹 소유자는 비공개 채널을 떠날 수 없습니다.
4. 비공개 채널은 해당 채널에 속하지 않은 사용자에게는 채널 목록에 표시되지 않습니다.

- 하위 계정에서 **Memberships(멤버십)**을 클릭하세요.
- Communities(커뮤니티)로 이동하세요.
- 드롭다운에서 Groups(그룹)을 클릭하세요.

![](https://jumpshare.com/share/F7VQklFSnqf9q7s8Sp8l+/Screenshot+2026-02-10+at+12.45.46%E2%80%AFAM.png)

- 그룹을 열거나 로그인하세요.

![](https://jumpshare.com/share/E57Y9g9gMoqiDrCtMJPT+/Screen+Shot+2026-02-10+at+12.49.05+AM.png)

- **+ Add Channel(+ 채널 추가)** 버튼을 클릭하세요.

![](https://jumpshare.com/share/RIZ3PJO6Iq7pfIxh0C76+/Screen+Shot+2026-02-10+at+12.50.33+AM.png)

- **Channel Name(채널 이름)**을 입력하세요. 선택적으로 채널 설명을 추가하고 채널 아이콘을 선택할 수 있습니다.
- **Make this Channel Private(이 채널을 비공개로 만들기)** 옵션을 **ON**으로 토글하세요.
- Create Channel(채널 만들기)을 클릭하세요.

## 공개 채널을 비공개로 전환하기

기존의 공개 채널을 비공개로 전환하여 향후 가시성과 참여를 제한하세요.

**참고**: 전환 후 멤버가 아닌 사용자는 더 이상 목록에서 채널을 볼 수 없습니다.

- 채널을 클릭하여 여세요.
- **Settings(설정)** (톱니바퀴) 아이콘을 클릭하세요.
- **Settings(설정)** 탭을 여세요.
- **Make Channel Private(채널 비공개로 만들기)**를 클릭하세요.
- **확인** 팝업을 읽고 **Confirm(확인)**을 클릭하세요.

![](https://jumpshare.com/share/h4pdDItaYmSjrMlkLRnE+/GIF+Recording+2026-02-10+at+1.00.09+AM.gif)

## 비공개 채널을 공개로 다시 전환하기

그룹의 누구나 채널을 발견하고 참여할 수 있도록 하려면 접근을 다시 개방하세요.

**중요**: 전환 후 채널은 모든 그룹 멤버에게 보입니다. 기존 게시글 기록은 채널에 그대로 남아있습니다.

- 채널을 여세요.
- **Settings(설정)** (톱니바퀴) 아이콘을 클릭하세요.
- **Settings(설정)** 탭을 여세요.
- **Make Channel Public(채널 공개로 만들기)**을 클릭하세요.
- **확인** 팝업을 읽고 **Confirm(확인)**을 클릭하세요.

![](https://jumpshare.com/share/vAh4ZgRnGFIqzcihVJqg+/GIF+Recording+2026-02-10+at+12.56.14+AM.gif)

## 비공개 채널에서 멤버 추가/제거하기

언제든지 멤버를 추가하거나 제거하여 비공개 채널에 접근할 수 있는 사용자를 제어하세요.

- 채널을 여세요.
- **Settings(설정)** (톱니바퀴) 아이콘을 클릭하세요.
- **Members(멤버)** 탭을 여세요.
- **멤버 추가**: 사용자를 검색하고 추가하세요.
- **멤버 제거**: 사용자를 찾아서 이름 옆의 **세 개 점**을 클릭한 후 Remove from Channel(채널에서 제거)을 클릭하세요.

![](https://jumpshare.com/share/y6BNiMxVpDZtAhfkrbFg+/GIF+Recording+2026-02-10+at+1.04.42+AM.gif)

## 채널 이모지나 아이콘 설정/변경하기

멤버들이 목록과 알림에서 채널을 빠르게 식별할 수 있도록 이모지를 할당하세요.

- 채널을 여세요.
- **Settings(설정)** (톱니바퀴) 아이콘을 클릭하세요.
- Channel Name(채널 이름) 옆의 **Edit(편집)** 버튼을 클릭하세요.
- **Channel Icon option(채널 아이콘 옵션)**을 클릭하고 원하는 **Emoji(이모지)** 또는 **Icon(아이콘)**을 **선택**하세요.
- **Save(저장)**을 클릭하세요.

![](https://jumpshare.com/share/DUt50fbROAdD8x0OTBkW+/GIF+Recording+2026-02-10+at+1.11.40+AM.gif)

## 자주 묻는 질문

**Q: 비공개 채널의 존재를 누가 볼 수 있나요?**
승인된 멤버와 해당 채널에 대한 충분한 권한을 가진 사용자만 볼 수 있습니다.

**Q: 비공개 채널을 공개로 만들면 과거 게시글은 어떻게 되나요?**
채널이 발견 가능해지고 기존 콘텐츠가 모든 그룹 멤버에게 보이게 됩니다.

**Q: 워크플로우를 통해 접근을 취소해야 하나요, 아니면 채널을 공개로 만들어야 하나요?**
워크플로우는 멤버 수준 접근을 관리하는 데 사용하세요. 전체 채널이 그룹에 개방되어야 하는 경우 채널을 공개로 만드세요.

**Q: 채널의 이모지를 누가 변경할 수 있나요?**
채널 소유자나 채널 수준 관리 권한을 가진 사용자가 변경할 수 있습니다.
