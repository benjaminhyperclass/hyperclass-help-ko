# 커뮤니티에서 멤버 차단 및 콘텐츠 신고하기

## **새로운 기능:**

- **멤버 차단**
이제 그룹 관리자와 소유자가 그룹에서 멤버를 차단할 수 있습니다. 차단된 멤버는 그룹에서 제거되며, 해당 그룹에 다시 가입할 수 없게 되고, 일반 그룹 멤버가 할 수 있는 모든 활동이 금지됩니다.

- **콘텐츠 신고**
이제 모든 게시물과 댓글을 사용자들이 신고할 수 있습니다. 그룹 관리자는 신고된 콘텐츠 목록에 접근하여 신고된 게시물/댓글을 유지하거나 삭제할지 결정하고 조치를 취할 수 있습니다.

그룹 관리자는 콘텐츠가 신고될 때마다 이메일, 앱 내 알림, 푸시 알림을 받게 됩니다.

## **이 기능들을 사용하는 방법:**

### **멤버 차단 (웹에서)**

- Members(멤버) 탭으로 이동합니다
- 차단하고자 하는 사용자의 "More options icon(더보기 옵션 아이콘)" (또는 점 3개 아이콘)을 클릭한 다음 "Ban from group(그룹에서 차단)"을 클릭합니다

### **멤버 차단 (모바일에서)**

- 커뮤니티 홈페이지에서 관리자라면 "Manage(관리)" 버튼이 보일 것입니다
- "Manage(관리)" 버튼을 클릭합니다
- 커뮤니티 설정 페이지에서 Members(멤버)를 클릭합니다
- 차단하고자 하는 사용자의 "More options icon(더보기 옵션 아이콘)" (또는 점 3개 아이콘)을 클릭한 다음 "Ban from group(그룹에서 차단)"을 클릭합니다

### **신고된 콘텐츠 접근 (웹에서)**

- 커뮤니티 홈페이지에서 Settings(설정)을 클릭한 다음 "Reported content(신고된 콘텐츠)" 탭을 선택합니다
- 신고된 콘텐츠가 있다면 목록이 표시되며, 적절한 조치를 취할 수 있도록 "Keep(유지)"과 "Remove(삭제)" 옵션이 제공됩니다

### **신고된 콘텐츠 접근 (모바일에서)**

- 커뮤니티 홈페이지에서 관리자라면 "Manage(관리)" 버튼이 보일 것입니다
- "Manage(관리)" 버튼을 클릭합니다
- 커뮤니티 설정 페이지에서 "To Review(검토 대기)"를 클릭합니다
- 신고된 콘텐츠가 있다면 목록이 표시되며, 적절한 조치를 취할 수 있도록 "Keep(유지)"과 "Remove(삭제)" 옵션이 제공됩니다

### **콘텐츠 신고**

- 게시물이나 댓글을 신고하려면 해당 게시물이나 댓글의 "More options icon(더보기 옵션 아이콘)" (또는 점 3개 아이콘)을 클릭한 다음 "Report(신고)"를 클릭합니다

멤버들은 콘텐츠가 신고되었다는 토스트 알림을 받게 됩니다.
