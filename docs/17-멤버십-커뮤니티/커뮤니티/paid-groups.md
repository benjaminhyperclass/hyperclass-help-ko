# 유료 그룹

커뮤니티(Communities)의 새로운 기능인 유료 그룹(Paid Groups)을 통해 플랫폼 내 공개 및 비공개 커뮤니티 모두에서 수익을 창출할 수 있습니다. 관리자와 소유자는 이제 사용자가 커뮤니티 그룹의 콘텐츠와 기능에 액세스하는 대가로 요금을 부과할 수 있습니다.

## 비공개 및 공개 그룹

커뮤니티 내 "설정(Settings)" 버튼에서 관리자와 소유자는 그룹을 공개/비공개로 설정할지 선택할 수 있습니다.

## 일회성 결제

일회성 결제 타입은 결제 완료 후 그룹 액세스 권한을 얻은 회원에게 무제한 그룹 액세스를 제공합니다. 일회성 결제를 설정하려면 다음 단계를 따르세요:

- 커뮤니티에 로그인
- "설정(Settings)" > "구독(Subscriptions)"으로 이동
- "새 가격 추가(Add new price)" 클릭: 금액, 유형을 일회성으로 지정
- "추가(Add)" 클릭 후 "저장(Save)"을 클릭하여 일회성 결제 유료 그룹 생성

참고: 결제를 테스트하려면 테스트 모드(Test Mode)를 활성화하세요

- 저장하면 가입을 원하는 회원에게 그룹 설명에 동일한 정보가 표시됩니다

## 정기 구독

정기 구독 타입은 결제 완료 후 그룹 액세스 권한을 얻은 회원에게 제한된 기간 동안 그룹 액세스를 제공합니다. 정기 구독을 설정하려면 다음 단계를 따르세요:

- 커뮤니티에 로그인
- "설정(Settings)" > "구독(Subscriptions)"으로 이동
- "새 가격 추가(Add new price)" 클릭: 금액, 유형을 정기로 지정, 체험 기간(Trial Days, 있는 경우), 청구 주기를 월간 또는 연간으로 설정
- "추가(Add)" 클릭 후 "저장(Save)"을 클릭하여 정기 구독 유료 그룹 생성
- 추가로 결제를 테스트하려면 테스트 모드(Test Mode)를 활성화하세요
- 저장하면 가입을 원하는 회원에게 그룹 설명에 동일한 정보가 표시됩니다

## 그룹 액세스

결제 완료 후 액세스 권한을 부여하려면:

- 결제가 완료되면 관리자와 소유자는 비공개 그룹(Private Groups)에서 회원에게 액세스를 제공하는 동일한 과정을 따라야 합니다 (사람 탭(People Tab) > 필터(Filter) 클릭 > 요청됨(Requested) > 멤버십 승인 또는 거부)
- 공개 그룹의 경우, 결제 완료 후 사용자가 가입하면 바로 회원이 됩니다

![멤버십 승인 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155010488248/original/vxnbdvM81OlRIfLr0YbVvXCf2k_9J9D_KA.png?1697655047)

## 알림

- 모든 관리자와 소유자는 사용자가 그룹 가입을 위해 결제한 내용과 멤버십 요청(그룹 유형에 따라)에 대한 모든 세부 정보가 포함된 이메일 알림을 받습니다
- 마찬가지로 사용자도 결제에 대한 모든 세부 정보가 포함된 결제 확인 이메일을 받습니다
- 관리자 또는 소유자가 멤버십을 승인하면 멤버십과 그룹 액세스에 대한 추가 알림을 받습니다

## 취소 및 환불

**회원이 GHL 결제(Payments) 화면에서 구독을 취소하면 어떻게 되나요?**
사용자가 GHL 결제 화면에서 구독을 취소하면 다음과 같은 작업이 시작됩니다:
- 사용자에게 취소 확인 이메일이 발송됩니다.
- 해당 구독과 연결된 그룹의 관리자와 소유자에게 취소 접수 이메일이 발송됩니다.
- 사용자는 자동으로 그룹에서 제거됩니다.

**회원이 Stripe 화면에서 구독을 취소하면 어떻게 되나요?**
사용자가 Stripe 화면에서 구독을 취소하면 다음과 같은 작업이 시작됩니다:
- 사용자에게 취소 확인 이메일이 발송됩니다.
- 해당 구독과 연결된 그룹의 관리자와 소유자에게 취소 접수 이메일이 발송됩니다.
- 사용자는 자동으로 그룹에서 제거됩니다.

**회원이 결제 환불을 받으려면 어떻게 해야 하나요?**
회원은 Stripe 또는 GHL 결제 두 곳 모두에서 환불을 받을 수 있습니다.

**GHL에서:**
환불은 "결제(Payments)" > "거래 내역(Transactions)"으로 이동 > 거래 항목 오른쪽의 "점 세 개" 아이콘 클릭 > "환불(Refund)" 클릭하여 처리할 수 있습니다.

**Stripe에서:**
환불은 연락처 상세 화면을 통해 Stripe에서도 처리할 수 있습니다.

**사용자의 연락처 카드가 삭제된 후 복원되면 사용자가 다시 결제해야 하나요?**
현재 시스템 기능상, 사용자의 연락처 카드가 실수로 삭제된 후 복원되면 사용자는 다시 결제할 필요 없이 그룹에 복원되어야 합니다.

**다른 통화의 최소 및 최대 청구 금액은 얼마인가요?**
최소 및 최대 청구 금액은 [https://stripe.com/docs/currencies](https://stripe.com/docs/currencies)를 참조하세요.
