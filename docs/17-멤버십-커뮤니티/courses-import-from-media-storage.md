# 강의 - 미디어 저장소에서 가져오기

강의 레슨에 비디오와 오디오를 추가할 때 이미 업로드한 미디어를 재사용하면 훨씬 빠릅니다. 미디어 저장소에서 가져오기 기능을 사용하면, 강의 제작자는 기존 오디오 및 비디오 파일을 재업로드하거나 중복 파일을 관리할 필요 없이 레슨에 직접 가져올 수 있습니다. 이 글에서는 하이퍼클래스 강의(Courses)에서 미디어 저장소 가져오기 기능을 사용하는 방법을 설명합니다.

# 강의에서 미디어 저장소 가져오기 기능이란?

미디어 저장소에서 가져오기 기능은 하이퍼클래스 미디어 저장소(Media Storage)에 저장된 기존 오디오 또는 비디오 파일을 강의 레슨에 첨부할 수 있게 해줍니다. 같은 파일을 다시 업로드하는 대신, 미디어 저장소에서 선택하여 레슨으로 가져올 수 있습니다.

이 방식은 콘텐츠 라이브러리를 체계적으로 유지하고 반복 작업을 줄여줍니다. 특히 여러 레슨이나 강의에서 같은 비디오나 오디오를 재사용할 때 유용합니다.

## 강의에서 미디어 저장소 가져오기 기능의 주요 장점

- **중복 업로드 방지**: 같은 파일을 여러 번 업로드하는 대신 기존 오디오/비디오를 가져옵니다.

- **빠른 강의 제작**: 미디어 저장소에 이미 있는 콘텐츠를 몇 번의 클릭으로 레슨 미디어에 추가할 수 있습니다.

- **중앙화된 콘텐츠 관리**: 강의 미디어를 미디어 저장소에서 체계적으로 관리하여 재사용이 쉽습니다.

- **깔끔한 워크플로우**: 중복 파일 버전을 줄이고 강의 전반의 미디어 업데이트를 단순화합니다.

## **미디어 저장소에서 강의 레슨으로 미디어를 가져오는 방법**

미디어 저장소에서 직접 미디어를 가져오면 미디어 라이브러리를 중앙화하고 재사용 가능하게 유지하면서 레슨을 더 빠르게 만들 수 있습니다.

- **멤버십(Memberships)** > 강의(Courses) > 상품(Products)으로 이동합니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155065595345/original/8GurbWjMl4MAEI78EI3yKRscaYG1lvIM-Q.png?1771941100)

- 편집하려는 **강의**를 클릭하여 엽니다.

- 업데이트하려는 레슨으로 이동합니다.

- 새 레슨을 **생성**하거나 기존 레슨을 **편집**하기 위해 클릭합니다.

![](https://jumpshare.com/share/6Bg2KiruBEo2dVgCeyjd+/GIF+Recording+2026-02-24+at+19.24.24.gif)

- 미디어 저장소에서 가져오기(Import from Media Storage) 버튼을 클릭합니다.

- **미디어 저장소에서 선택하려면 클릭(Click to select from media storage)** 버튼을 클릭하여 미디어 저장소 라이브러리를 탐색하고 사용하려는 오디오/비디오 파일을 선택합니다.

![](https://jumpshare.com/share/3eYw0OSVaABgmNZBnSmA+/Screen+Shot+2026-02-24+at+19.26.00.png)

- **오디오/비디오 파일을 선택**하고 미디어 삽입(Insert Media)을 클릭합니다.

![](https://jumpshare.com/share/4D9sfLPklAD0JIMjXMYH+/Screen+Shot+2026-02-24+at+19.29.19.png)

## **자주 묻는 질문**

**Q: 이미지도 지원하나요?**

아니요, 오디오/비디오만 가져올 수 있습니다.

**Q: 미디어 저장소에서 가져올 때 파일의 새 사본이 업로드되나요?**

아니요. 가져오기는 미디어 저장소에 저장된 기존 오디오/비디오를 사용하므로 재업로드할 필요가 없습니다.

**Q: 레슨에 "미디어 라이브러리를 통해 가져오기"가 보이지 않습니다. 무엇을 확인해야 하나요?**

레슨 편집기(생성/편집)에 있는지 확인하고, 사용자 역할(User Role)에 **미디어(Medias)**와 **멤버십/강의(Memberships/Courses)** 접근 권한이 포함되어 있는지 확인하세요.

**Q: 여러 레슨이나 강의에서 같은 비디오를 재사용할 수 있나요?**

네. 가져오기는 재사용을 위해 설계되었습니다. 필요한 만큼 많은 레슨에서 같은 미디어 저장소 파일을 선택할 수 있습니다.

**Q: 레슨이 발행되지 않은 경우에도 수강생이 가져온 미디어를 볼 수 있나요?**

수강생은 발행된 콘텐츠만 볼 수 있습니다. 저장(Save)은 변경사항을 저장하고, 발행(Publishing)은 강의 공개 설정에 따라 콘텐츠를 이용 가능하게 만듭니다.
