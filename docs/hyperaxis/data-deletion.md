## 삭제 처리 절차

삭제 요청은 다음 절차로 처리됩니다.

1. **요청 접수**: 이메일([geunha2025@naver.com](mailto:geunha2025@naver.com))로 삭제 요청을 접수합니다. 제목에 **"개인정보 삭제 요청"**이라고 적어 주시면 처리가 빠릅니다.

2. **본인 확인**: 요청자 본인 확인 절차를 진행합니다. (가입 이메일 또는 연동 계정 정보 기준)

3. **삭제 실행**: 확인 완료 후 지체 없이 처리에 착수하여, 회사 및 위탁·재위탁 처리자가 보관한 회원님의 개인정보를 복구 불가능한 방식으로 삭제합니다. 삭제 범위는 다음을 포함합니다.
   - Meta 광고 인사이트 데이터 (캠페인·광고세트·광고 ID, 노출·클릭·전환·광고비 내역)
   - Meta 시스템 사용자 토큰 (BYOT 연동 토큰)
   - 플랫폼 매출·주문 데이터 집계
   - 서비스 계정 정보 및 설정값
   - **Facebook·Instagram 등 Meta 연동을 통해 수집된 모든 데이터 포함**

4. **완료 통보**: **접수일로부터 영업일 기준 10일 이내**에 삭제를 완료하며, 완료 시 요청하신 이메일로 결과를 통지합니다. 부득이하게 기간을 연장해야 하는 경우 사유와 예상 완료일을 함께 안내합니다.

## Facebook 앱 연결 해제

Facebook 계정 설정에서 본 서비스(Hyperaxis)의 앱 연결을 직접 해제하시는 경우에도 동일하게 데이터 삭제 요청으로 처리됩니다.

**Facebook 앱 연결 해제 방법**:

1. Facebook 계정에 로그인합니다.
2. 오른쪽 상단 **▼ 메뉴 > 설정 및 개인정보 > 설정**을 클릭합니다.
3. 왼쪽 메뉴에서 **앱 및 웹사이트**를 선택합니다.
4. 목록에서 **Hyperaxis**를 찾아 **제거**를 클릭합니다.

앱 연결 해제 이후에도 이메일([geunha2025@naver.com](mailto:geunha2025@naver.com))로 삭제 확인을 요청하시면 처리 결과를 통보해 드립니다.
