# GBP 페이지 관리: 페이지 추가, 삭제 및 설정 조정

하위 계정 내에서 구글 비즈니스 프로필(GBP) 페이지를 쉽게 추가하거나 삭제하고, GBP 메시징 및 통화 추적 설정을 조정하는 방법을 알아보세요. 이 가이드는 GBP 페이지를 관리하는 간단한 단계별 과정을 제공합니다.

"페이지 관리" 옵션은 GBP 페이지를 하위 계정에 성공적으로 연동한 후에만 사용할 수 있습니다. 여러 GBP 페이지를 하위 계정에 연결하는 방법을 알아보려면 [여기를 클릭하세요](https://help.leadconnectorhq.com/en/support/solutions/articles/155000002138).

목차

- [페이지 관리](#페이지-관리)
- [설정 업데이트](#설정-업데이트)
- [업데이트 확인](#업데이트-확인)

### 페이지 관리:

페이지를 추가하거나 삭제하거나 GBP 메시징 또는 통화 추적 옵션을 변경하려면 '페이지 관리'를 클릭하세요.

![페이지 관리 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155023403542/original/8q70GEgTiStrKEg-I7LTPHrWt9WjrpIirA.png?1711216244)

### 설정 업데이트: 

필요한 조정을 수행한 후 '업데이트'를 클릭하여 변경사항을 적용하세요.

![설정 업데이트](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155023403557/original/u-iunMUeQP7y5LvurCtf4pXvk_wAuQEb-Q.png?1711216322)

참고: 페이지 수준에서 변경한 사항은 이전 데이터나 기록에 영향을 주지 않으며, 변경사항을 기반으로 향후 데이터에만 영향을 미칩니다.

### 업데이트 확인:

연결 업데이트 상태를 확인하려면 '새로 고침' 아이콘을 클릭하세요.

![새로 고침 아이콘](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155023403562/original/Ys_XYZxUMRLWCnOLTJ89iZxKdTiYpHvObA.png?1711216377)

연결이 성공적으로 완료되면 업데이트된 페이지의 세부정보가 GBP 섹션에 표시됩니다.

![업데이트된 페이지 표시](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155023403575/original/hXTvEw34Z32pdSAIQyHRlHhvLHfHGKn5hw.png?1711216420)
