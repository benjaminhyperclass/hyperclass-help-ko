# LeadConnector

Zapier는 서로 다른 앱들이 즉시 연동되어 워크플로우를 자동화할 수 있게 해주는 플랫폼입니다. CRM을 Zapier와 연동하면 리드 수집, 연락처 업데이트 등 다양한 작업을 자동화할 수 있어요.

## 연동 방법을 단계별로 안내해드릴게요:

**1단계:** Zapier 계정에 로그인하세요.

**2단계:** 홈 화면에서 Create > Zap을 클릭하세요.

![Zapier 홈 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155060279737/original/MxwssDT5flWKeZdx8g_bBitBvb4nrp0Euw.jpeg?1765353496)

**3단계:** Zap 편집기에서 트리거 블록을 클릭하고, 앱 목록에서 WL 앱 이름을 검색하세요.

![앱 검색 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155060280551/original/NV9c6K4TUlZch0dpEx-6ZPqyKFg8KQUhAg.jpeg?1765354023)

**4단계:** 트리거 설정의 계정 섹션에서 "Sign In" 버튼을 클릭하세요.

![로그인 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155060281775/original/3SlhsB_g0nj0YgRkrf27O-uMaKDVGZAYPQ.png?1765354621)

**5단계:** 아래 화면으로 리다이렉트되어 API 키를 입력하라는 메시지가 나타납니다.

![API 키 입력 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155060282949/original/_thHEDpC_EYhCi9By5CkPPLrpW6wKerOHQ.png?1765355192)

**6단계:** Location >> Settings >> Private Integration에서 PIT 토큰을 생성하세요. 토큰을 식별할 수 있는 이름과 필요에 따라 설명을 추가하세요. Next를 클릭하고 아래 스크린샷과 같이 스코프를 추가하세요.
**정상적인 작동을 위해 아래 스크린샷에 강조 표시된 모든 스코프를 반드시 추가해주세요.**

![스코프 설정 1](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155060283219/original/Dcp5W5SU7WWl2tQ3Pg0rQhzC0dQfEKazlQ.png?1765355342)

![스코프 설정 2](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155065172798/original/TUJREPRrIsNkPXLkYiTkh6DHyNP3k3aKmw.png?1771416113)

![스코프 설정 3](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155060283812/original/z3LvOMgNsMpU6ux_DfD_sJLc0mJ24TKpZw.png?1765355508)

**7단계:** 5단계에서 생성한 PIT 토큰을 사용하여 Zapier를 하위 계정에 연결하세요.
