# 페이스북 연동 시 페이지 선택 드롭다운에서 페이스북 페이지가 보이지 않는 문제

CRM 계정과 페이스북을 연동할 때 페이스북 비즈니스 페이지가 표시되지 않는 문제는 비교적 흔하게 발생합니다. 이 가이드는 해당 문제를 해결하는 방법을 안내합니다.

## 페이지 선택(Page Selector) 드롭다운에서 페이스북 페이지가 보이지 않는 문제를 해결하는 방법

먼저 [페이스북 계정](http://www.Facebook.com)에 접속합니다. 그 다음 우측 상단의 프로필 사진이 있는 아바타 아이콘을 클릭하고 Settings and Privacy(설정 및 개인정보)로 이동합니다:

![페이스북 설정 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48285091296/original/jyxWYZ4ZcuGHSZ3Y-Takz5Y5Xjvxwexk2w.png?1677853417)

Settings and Privacy(설정 및 개인정보) 메뉴에서 Settings(설정)을 클릭합니다.

![설정 메뉴 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48285091548/original/MY4la9WnthkRntpMaAdBlu_f6dhJhKsRpA.png?1677853476)

왼쪽 메뉴에서 [Business Integrations(비즈니스 연동)](https://www.facebook.com/settings?tab=business_tools&ref=settings)으로 이동합니다:

![비즈니스 연동 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48285091853/original/s3zujjbzAF5uRlCs2fykCNKLtuEIPjggTA.png?1677853560)

LeadConnector 앱 앞의 view and edit(보기 및 편집) 버튼을 클릭합니다:

![LeadConnector 앱 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48285092344/original/jlqQEEAyTN6cIIlaiDZTsdKyR7qG1tUgoA.png?1677853642)

모든 토글이 켜져 있는지 확인하세요. 페이지 하단까지 스크롤하여 모든 항목을 활성화합니다:

![모든 권한 활성화](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48285092785/original/ZlVTDWKCD6_Afo0yub7nrhW66R5N-1waog.png?1677853718)

**중요 사항:**

페이스북 페이지를 연동한 사용자만이 페이지 드롭다운 목록에서 해당 페이지를 볼 수 있습니다. 드롭다운에서 페이지가 보이려면 해당 페이스북 페이지의 관리자여야 하며, 다른 계정의 페이스북 페이지는 목록에서 더 이상 볼 수 없습니다.

![페이지 선택 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48291400274/original/g41sAjRNyGtEtdYy-9Cv6ezS7sGvCqVccA.png?1680771876)

체크박스가 있는 페이지 목록이 나타나면, 에이전시의 하위 계정에 연결할 관련 페이지들을 모두 선택합니다:

![관련 페이지 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48285093690/original/GnmWUdLTUrDK53xlp16hxtC5P-W0QY5R_g.png?1677853807)

완료되면 Save(저장) 버튼을 클릭합니다.

![저장 버튼 클릭](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48285094456/original/VcAwN-8XK9TNOxA3JBA3wZUSYbzILsQNmw.png?1677853940)

**중요 사항:**

이미 선택되어 있는 페이지가 있다면 절대로 선택을 해제하지 마세요. 현재 이 하위 계정에 연동하려는 페이지가 아니더라도, 선택을 해제하면 에이전시의 다른 하위 계정과의 연동이 끊어질 수 있습니다.
