# 하위 계정을 Zapier와 연결하기

Zapier는 다양한 앱들이 즉시 소통할 수 있게 하여 워크플로우를 자동화해주는 플랫폼입니다. CRM을 Zapier와 연동하면 리드 수집, 연락처 업데이트 등의 작업을 자동화할 수 있습니다.

## 연결 단계를 따라해보세요:

1단계: Zapier 계정에 로그인합니다.

2단계: 홈 화면에서 Create(생성) > Zap을 클릭합니다.

![Zapier 홈 화면에서 Zap 생성하기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051136624/original/E18QlbidJLzvzSAKUCgdN40SBsTFKON3-w.png?1754577643)

3단계: Zap 편집기에서 Trigger(트리거) 블록을 클릭하고 앱 목록에서 LeadConnector를 검색합니다.

![Trigger 블록에서 앱 검색](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051136628/original/gCzh2JFrovwDcrEFESIg46Or5Ip3BJ3L3Q.jpeg?1754577646)

![LeadConnector 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051136626/original/DNovU37i0XmXF-mzLoTpnfdoksJWzjPFSQ.jpeg?1754577646)

4단계: 트리거 설정의 계정 섹션에서 "Sign In(로그인)" 버튼을 클릭합니다.

![Sign In 버튼 클릭](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051136636/original/M-BixejU609kwV3dJI5VbXGwL7UVDitFGg.jpeg?1754577647)

5단계: 로그인 화면으로 이동됩니다. CRM 이메일과 비밀번호를 입력하여 계속 진행합니다.

![로그인 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051136625/original/oq31dijswcQEhSgZ9tzYf6jGbTDovyKTtQ.jpeg?1754577644)

![비밀번호 입력](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051136632/original/FCwg3g8jFxGIjGafCf5Y1KkQnUzoTALmEQ.jpeg?1754577647)

6단계: 로그인 후 Zapier와 연결할 하위 계정을 선택하라는 메시지가 표시됩니다.

![하위 계정 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051136637/original/x6YhiG5r-DLBBpLh05VaJ-AIvjKkl_WXmA.jpeg?1754577647)

완료되면 Zapier 계정이 CRM에 성공적으로 연결됩니다. Zap 설정의 "Accounts(계정)" 하에 LeadConnector가 나열되어 연동이 활성화되었음을 확인할 수 있습니다.

![연결 완료 확인](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051136650/original/aM4xQFIDfpO1_wyZ5VaC1otJblm1JBbWVg.jpeg?1754577649)

## LeadConnector와 Zapier 연결 해제하기:

- Zapier 홈페이지에서 왼쪽 메뉴의 Apps(앱)를 클릭합니다.

- App Connections(앱 연결) 섹션으로 이동합니다.

- 연결된 앱 목록에서 LeadConnector를 찾습니다.

- LeadConnector 옆의 세 점 메뉴(•••)를 클릭합니다.

- Delete(삭제)를 선택하여 연결을 제거합니다.

이렇게 하면 Zapier 계정에서 LeadConnector 연결이 성공적으로 해제됩니다.

![연결 해제 과정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051136648/original/sG9Sh80rEqTA8lw2WD72sAo9qsWhl9_DPg.jpeg?1754577648)
