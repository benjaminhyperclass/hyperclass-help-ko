# 연결된 Google Business Profile 페이지에서 고객 상호작용 관리하기

이 문서에서는 하위 계정에 연결된 Google Business Profile 페이지에서 오는 메시지, 리드, 리뷰 등의 자산을 추적하는 방법을 설명합니다.

### 

**목차**

- [메시지](#Messages)
- [평점 및 리뷰](#Ratings-&-Reviews)
- [연락처](#Contacts)

### 메시지

고객이 GBP 페이지를 방문해서 채팅 섹션을 통해 메시지를 보내면, 연결된 하위 계정의 대화(Conversations) 페이지에서 해당 메시지를 받을 수 있습니다.

![GBP 메시지가 대화 페이지에 수신되는 모습](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155023403870/original/zFsH2wSftBAaniIaowoLmow1S-HZcLMnlg.png?1711218167)

수신된 메시지를 바탕으로 CRM(대화 페이지)에서 직접 답장 메시지를 보내서 고객과 더 소통할 수 있습니다.

메시지에는 메시지가 시작된 GBP 페이지 이름도 표시됩니다. 이 기능은 여러 비즈니스 페이지를 하위 계정에 연결했을 때 고객이 어느 특정 비즈니스 페이지를 통해 연락했는지 식별하는 데 특히 유용합니다.

![메시지에 표시된 GBP 페이지 이름](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155023403882/original/A9VI8SP0L2rsB9z8bl0HDGFSfN6tHqTnew.png?1711218272)

### 평점 및 리뷰

고객에게서 리뷰를 받으면, 평판 관리(Reputation) 모듈 안의 리뷰(Reviews) 페이지에서 리뷰 데이터에 접근할 수 있습니다. 여기서 하위 계정에 연결된 Google 비즈니스 페이지에서 받은 리뷰를 확인하고 추적하며 답변할 수 있습니다.

각 리뷰에는 리뷰가 접수된 비즈니스 페이지 이름도 표시되는데, 하위 계정에 여러 페이지를 연결해서 관리할 때 매우 중요한 정보입니다.

![리뷰 페이지에서 GBP 리뷰 확인](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155023403896/original/69t6FXddN1JcYcit576tD5UlhHDHoFmLXg.png?1711218355)

하위 계정에 여러 페이지가 연결되어 있다면, 각 페이지마다 리뷰 링크를 구성하고 기본 페이지를 선택할 수 있습니다. 이를 통해 연락처에게 리뷰 요청 링크를 보내서 비즈니스의 Google 리뷰를 더 많이 수집할 수 있습니다.

이 설정은 하위 계정 설정(Sub-Account Settings) 모듈 안의 평판 관리(Reputation Management) 탭에서 할 수 있습니다.

![평판 관리 설정에서 리뷰 링크 구성](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155023403898/original/WjtGcT76jEGl0_3k1uG2IxzHBn-zhHYVOw.png?1711218387)

### 연락처

연결된 GBP 페이지의 메시지나 리뷰를 통해 생성된 연락처 또는 리드 레코드를 연락처(Contacts) 모듈에서 확인할 수 있습니다.

![연락처 모듈에서 GBP 연락처 확인](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155023403906/original/gDdv6zSkSA2w3jOnK5Qf6C-JwVjRNMJ5Jg.png?1711218441)

연락처를 클릭하면 상세 페이지로 이동하며, 여기서 연락처가 생성된 소스와 페이지 정보를 확인할 수 있습니다.

![연락처 상세 페이지에서 소스 정보 확인](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155023403935/original/2afRKnOmX5yd1IJoyfPAHFz-mRNHSDd1tg.png?1711218625)
