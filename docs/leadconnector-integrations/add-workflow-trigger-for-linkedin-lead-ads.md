# 링크드인 리드 광고용 워크플로우 트리거 추가하기

이 가이드에서는 링크드인 리드 광고 연동을 위한 워크플로우 자동화를 설정하는 방법을 안내해드립니다.

## 사전 준비사항

먼저 링크드인 계정을 CRM과 연결해야 합니다. 링크드인을 CRM과 연결하는 방법에 대해 자세히 알아보려면 [여기를 클릭하세요](https://help.leadconnectorhq.com/en/support/solutions/articles/155000001151-integrating-linkedin-lead-ads-with-crm).

## 워크플로우 트리거 추가하기

- 좌측 메뉴의 "Automation(자동화)"으로 이동하여 "Create Workflow(워크플로우 만들기)"를 클릭하세요.

- "Add a new trigger(새 트리거 추가)"를 클릭하고 "LinkedIn lead form submitted(링크드인 리드 폼 제출됨)" 트리거를 선택하세요.

![링크드인 리드 폼 트리거 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155010670567/original/FjiBd5gOxs1U_01WZStGAPlYp0t3GLFJfg.png?1697806049)

- 필터를 사용하여 페이지/광고 계정을 선택하세요(하위 계정에 여러 광고 계정을 추가한 경우). 드롭다운에 광고 계정이 보이지 않는다면 링크드인 계정이 하위 계정과 연동되어 있는지 확인하세요.

![페이지/광고 계정 필터 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155010670653/original/x9ZzK-IuPt8-OdP_yYtAyw19BewK1z-LEw.png?1697806094)

- 필터를 사용하여 특정 폼을 선택하세요. 선택한 폼에 대해 필드 매핑이 완료되었는지 확인하세요. 그렇지 않으면 리드가 CRM으로 가져와지지 않습니다.

![특정 폼 필터 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155010670672/original/VBPUgcWHF1zBltucQz5iuOj_BhIXxsqSuw.png?1697806119)

- 트리거 설정을 완료한 후 자동화를 위한 액션 세트를 정의하고 저장하세요.

![워크플로우 액션 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155010670808/original/ul5g5Tu2Lv5gOwT6I6eYoCN_GyRz9Ohgxg.png?1697806186)

- 자동화 설정이 완료되면 워크플로우를 발행하여 활성화하세요.
