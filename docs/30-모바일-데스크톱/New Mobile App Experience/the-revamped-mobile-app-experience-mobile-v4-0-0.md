# 새로워진 모바일 앱 경험 - Mobile v4.0.0

### 개요

새로워진 모바일 앱 경험은 속도, 지능성, 사용성을 위해 완전히 새롭게 디자인된 HighLevel 모바일 인터페이스를 선보입니다. 이번 업데이트는 모바일 경험을 웹 앱의 디자인 언어와 일치시켜, 이동 중에도 비즈니스를 관리하는 것이 그 어느 때보다 쉬워졌습니다.

현대적인 UI와 다크 모드부터 Universal Search와 Ask AI 같은 지능형 도구까지, 이번 릴리스는 HighLevel의 모든 기능을 여러분의 손끝으로 가져다줍니다. 친숙하고, 더 빠르고, 더 스마트하게 느껴지도록 재설계되었습니다.

## 1. 새로워진 모바일 앱의 새로운 기능

업데이트된 앱은 **현대적인 디자인**, **역할 인식 홈페이지**, **스마트한 네비게이션**을 도입하여 모바일에서의 생산성을 높여줍니다. 주요 하이라이트는 다음과 같습니다:

- 위젯, 고정된 앱, 빠른 작업이 포함된 **동적 홈페이지**
- 대화, 연락처, 기회 관리 전반에서 무엇이든 찾을 수 있는 **Universal Search**
- 논리적인 비즈니스 카테고리로 구성된 새로운 **앱 드로어**
- 휴대폰에서 바로 초안 작성, 생성, 실행을 도와주는 개인 비서 **Ask AI**
- 사용성을 향상시키고 배터리 수명을 절약하는 **진정한 다크 모드**

## 2. 새로운 모바일 경험 활성화 방법

새로운 디자인을 살펴보기 전에, Labs에서 이 경험을 활성화해야 합니다.

- App Store 또는 Google Play에서 **앱을 업데이트**하세요
- Agency Settings(에이전시 설정) → Labs → Subaccounts(하위 계정)으로 이동하세요
- "A New Mobile App Experience"를 켜세요
- 활성화할 하위 계정을 선택하세요

**Pro Tip:**
클라이언트 계정에 배포하기 전에, 먼저 에이전시의 내부 하위 계정에서 활성화하여 새로운 레이아웃과 네비게이션을 살펴보는 것을 권장합니다.

활성화한 후 앱을 열면 재디자인된 홈페이지, 새로운 앱 드로어, Ask AI 등을 즉시 경험할 수 있습니다.

## 3. 홈페이지 개요

새로운 홈페이지는 여러분의 **명령 센터** 역할을 하며, 로그인하는 순간 가장 중요한 정보를 표시합니다.

### 고정 위젯 (상단 섹션)

4개의 핵심 데이터 위젯을 표시합니다:

- 마감된 할 일
- 기회 관리 파이프라인 값
- 읽지 않은 메시지
- 오늘의 예약

이 위젯들은 현재는 고정되어 있지만, 2025년 4분기에는 설정 가능하게 되어 사용자가 홈페이지에 표시할 위젯을 커스터마이즈할 수 있습니다.

### 고정된 앱 위젯

- 빠른 접근을 위해 **최대 4개 모듈**을 고정할 수 있습니다
- **Edit(편집)** 아이콘을 눌러 고정된 앱을 수정하세요
- **햄버거 아이콘**을 사용해 고정된 모듈을 드래그하여 순서를 바꾸세요

### 빠른 작업 위젯

- 모듈 간 이동 없이 자주 사용하는 작업을 즉시 수행할 수 있는 **8개의 고정된 빠른 작업**에 접근하세요

### 매출 차트

- **지난 6개월간 벌어들인 매출**을 보여주는 성과 그래프입니다

모든 홈페이지 요소는 **역할 인식**이 가능합니다. 사용자가 특정 모듈에 접근할 수 없다면, 해당 위젯은 표시되지 않습니다.

## 4. Universal Search

새로운 **Universal Search**는 필요한 것을 정확히 찾는 것을 그 어느 때보다 빠르게 만들어줍니다.

- 현재 **대화, 연락처, 기회 관리**에서 검색할 수 있습니다
- 다른 모든 모듈 지원은 **2025년 4분기**에 제공됩니다
- **모듈**이나 **빠른 작업**을 직접 검색할 수 있습니다
- 앱은 더 빠른 재접근을 위해 **최근 5개 검색**을 기억합니다
- 컨텍스트를 이해하도록 구축되어, 키워드만으로도 몇 달 전의 특정 메시지까지 찾을 수 있습니다

**예시:** "meeting notes"를 검색하면 3개월 전까지의 해당 문구가 포함된 대화를 즉시 보여줍니다.

## 5. 앱 드로어(App Drawer)

기존의 왼쪽 네비게이션이 현대적인 **앱 드로어**로 교체되어 탐색이 쉬워졌습니다.

- 모듈을 **4열 그리드 뷰**로 표시합니다
- 다음과 같이 카테고리별로 분류됩니다:
  - **인사이트 & 생산성**
  - **커뮤니케이션**
  - **영업 & 운영**
  - **마케팅 & 성장**
- 일관성을 위해 웹 앱의 구조를 반영합니다

**Settings(설정)**과 **Notifications(알림)** 같은 네비게이션 항목은 이제 전용 아이콘과 함께 앱의 **우측 상단**에 위치합니다.

## 6. Ask AI

Ask AI는 자연어를 사용해 일상적인 작업을 수행하도록 도와주는 내장 비즈니스 어시스턴트입니다.

Ask AI를 사용해서 할 수 있는 일:

- 소셜 미디어 콘텐츠 초안 작성
- 연락처 생성 또는 편집
- 이미지 생성 또는 편집
- 지원되는 MCP(Multi-Channel Platform) 액션 실행

Ask AI는 모바일에서 사용할 수 없는 모듈(예: 블로그, 퍼널)과 연결된 것을 제외한 **모든 MCP 액션**을 지원합니다.

**Chat History(채팅 기록)** 섹션을 통해 전체 대화 기록에 접근할 수 있어, 중단한 곳에서 다시 시작하거나 이전 결과물을 다시 확인할 수 있습니다.

## 7. 핵심 개선사항

### 다크 모드 & 현대적인 UI

더 나은 가독성과 에너지 효율성을 제공하는 진정한 블랙 다크 모드와 함께 세련되고 새로워진 인터페이스를 즐기세요.

### 성능 & 안정성

더 빠른 로딩 시간, 더 부드러운 전환, 최적화된 데이터 캐싱으로 이동 중에도 안정적인 경험을 보장합니다.

### 간소화된 네비게이션

- Settings(설정)과 **Notifications(알림)**이 이제 우측 상단 아이콘으로 위치합니다
- 알림에는 쉬운 추적을 위한 읽지 않음 표시가 포함됩니다

### SaaS Configurator 준수

새로워진 모바일 앱은 이제 **SaaS Configurator를 완전히 준수**하여, 모바일 접근을 웹 기반 구독 제어와 일치시킵니다.

이전에는 모든 사용자가 플랜에 관계없이 모바일에서 모든 기능에 접근할 수 있었습니다. 이번 업데이트로:

- 기능과 모듈 접근이 이제 **SaaS 플랜 제한을 준수**합니다
- 하위 등급 플랜의 사용자는 더 이상 자신의 플랜에 포함되지 않은 모듈을 보거나 접근할 수 없습니다

이 변경사항은 **모바일과 웹 간의 일치**를 보장하여, 리셀러와 고객 모두에게 일관성을 유지합니다.

## 8. FAQ

**Q. 기존 앱이 자동으로 새로운 경험으로 업데이트되나요?**
A. 아직은 아닙니다. 새로운 경험은 Labs에서 수동으로 활성화해야 합니다.

**Q. 홈페이지 위젯을 커스터마이즈할 수 있나요?**
A. 출시 시점에는 불가능하지만, 전체 커스터마이제이션은 2025년 4분기에 제공됩니다.

**Q. Ask AI가 모든 모듈을 지원하나요?**
A. Ask AI는 블로그와 퍼널 같이 아직 모바일에서 사용할 수 없는 모듈을 제외한 모든 MCP 기능을 지원합니다.

**Q. SaaS Configurator 업데이트가 제 클라이언트에게 어떤 영향을 미치나요?**
A. 클라이언트의 모듈 접근이 이제 자신의 SaaS 플랜에 포함된 기능과 일치하게 되어, 웹과 모바일 간의 일관성을 유지합니다.

**Q. Universal Search가 언제 모든 모듈을 지원하나요?**
A. 전체 멀티 모듈 지원은 2025년 4분기에 출시됩니다.

**Q. 화이트라벨 모바일 앱을 사용하고 있는데, 새로운 경험에 접근할 수 있나요?**
A. 물론입니다. 새로운 경험은 모든 모바일 앱에서 사용할 수 있습니다. 하지만 새로운 디자인의 구조적 변경으로 인해 기존 앱의 커스터마이제이션이 깨질 수 있어서, 에이전시의 내부 하위 계정에서 먼저 새로운 경험을 켜보는 것을 권장합니다. 이전에는 볼 수 없었던 방식으로 모바일 앱을 커스터마이즈할 수 있게 해주는 완전히 새로운 화이트라벨 모바일 앱 커스터마이저가 준비 중입니다.
