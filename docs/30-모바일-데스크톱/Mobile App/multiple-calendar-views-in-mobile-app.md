# 모바일 앱에서 여러 캘린더 보기 사용하기

여러 사용자, 캘린더, 또는 그룹의 일정을 하나의 모바일 캘린더 화면에서 함께 볼 수 있습니다.

- **여러 사용자 선택**
  - 여러 사용자의 일정을 동시에 볼 수 있습니다.
- **여러 캘린더 선택**
  - 여러 캘린더의 일정을 동시에 볼 수 있습니다.
- **여러 그룹 선택**
  - 여러 그룹의 일정을 동시에 볼 수 있습니다.

---

## 모바일에서 여러 캘린더를 사용하는 방법

선호하는 모바일 앱(HighLevel, LeadConnector 또는 화이트라벨 앱)이 기기에 설치되어 있는지 확인하세요.

앱 하단 바에서 캘린더 섹션으로 이동하세요. 캘린더 아래에 드롭다운이 보일 것입니다. 드롭다운을 탭하여 선택기를 여세요.

![모바일 캘린더 드롭다운 선택기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036376125/original/zT3BVngnobpkNqBjIWu207n73boQybNn5w.png?1731377858)

선택기에서 여러 사용자, 캘린더 또는 그룹을 추가할 수 있습니다.

![다중 선택 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036376137/original/VQY6QSWr8SFF4b6KwXtqWZjzX4W5DY_maw.png?1731377934)

선택을 완료하면 이벤트 영역에서 선택한 모든 항목의 일정을 볼 수 있습니다. 각 선택 항목마다 다른 색상이 지정되어 이벤트 전체에 반영됩니다.

![여러 캘린더 통합 보기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036376170/original/GY1_wjJ7DKG2jGNHJliCKu7hkevptKUAtw.png?1731377989)
