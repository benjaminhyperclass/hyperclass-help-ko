# 모바일 앱의 문서 및 파일 관리

이 가이드에서는 연락처 상세(Contact Record) 내 새로운 **문서 및 파일(Docs and Files)** 탭을 사용하여 문서 관리를 효율화하는 방법을 알아보세요! 계약서, 견적서, 인보이스 등을 한 곳에서 업로드하고, 전송하고, 추적할 수 있습니다. 자동 정렬 폴더로 문서를 체계적으로 정리하고 쉽게 접근할 수 있어요.

## **문서 및 파일 탭이란?**

문서 및 파일(Docs and Files) 탭은 특정 연락처와 관련된 모든 파일을 저장하는 연락처 수준의 문서 센터입니다. 내부 업로드, 공유 문서, 수신 첨부파일을 체계적인 폴더로 통합 관리하여 대화 내용이나 외부 도구를 뒤져가며 찾을 필요가 없어요.

계약서 첨부, 제안서 발송, 참고 자료 저장 등 어떤 작업이든 이 탭에서 모든 것을 손쉽게 관리할 수 있습니다.

## **문서 및 파일 탭의 주요 장점**

연락처의 파일을 더 스마트하고 빠르게 관리하세요. 문서 및 파일 탭은 파일 추적을 단순화하면서 모든 것을 해당 연락처와 연결해 둡니다.

- 모든 파일을 각 연락처 하위에 직접 정리
- 문서를 관련 폴더로 자동 분류: 내부, 발송, 수신, 커스텀 필드
- 한 번에 최대 10개 파일 업로드 (총 250MB까지)
- 다양한 파일 형식 지원 (PDF, DOCX, JPG, PNG 등)
- 다운로드 없이 문서 미리보기 가능
- 접근 가능한 중앙집중식 파일 저장으로 협업 향상

## **문서 및 파일 탭 사용법**

문서 및 파일 탭은 데스크톱과 모바일 모두에서 직관적이고 쉽게 접근할 수 있도록 설계되었습니다.

- CRM 대시보드에서 **연락처를 열기**
- **문서 및 파일(Docs and Files)** 탭 클릭 또는 스와이프
- **업로드(Upload)** 선택하여 컴퓨터나 기기에서 최대 10개 파일 선택
- 파일이 추가되는 방식에 따라 자동으로 폴더별로 분류됩니다:
  - 수동 업로드는 **내부(Internal)**로
  - 메시지를 통해 발송하거나 수신한 파일은 **발송(Sent)** 또는 **수신(Received)**으로
  - 커스텀 필드 파일은 **커스텀 필드(Custom Fields)** 폴더 하위에 표시

## **문서 및 파일 폴더 구조**

문서가 어떻게 정리되는지 이해하면 필요한 것을 빠르게 찾고 혼란을 줄일 수 있어요.

- **내부(Internal)**: 본인이나 팀에서 직접 업로드한 파일
- **발송(Sent)**: 발신 대화에서 자동 저장된 파일
- **수신(Received)**: 파일 첨부가 있는 수신 메시지에서 가져온 파일
- **커스텀 필드(Custom Fields)**: ID 이미지나 계약서 등 커스텀 필드 업로드와 연결된 파일들이 내부 폴더 하위에 중첩됨

## **지원 파일 형식 및 업로드 제한**

파일이 지원되고 안전하게 업로드될 것을 확신하고 업로드하세요.

**허용 파일 형식**:
DOC, DOCX, PDF, JPG, PNG, PPT, GIF

**업로드 용량**:
- 업로드당 최대 **10개 파일**
- 최대 총 업로드 크기: **250MB**

## **모바일 앱 지원**

HighLevel 또는 LeadConnector 모바일 앱을 사용하여 이동 중에도 문서에 접근하고 관리할 수 있습니다.

- **버전 3.97.3 이상**인지 확인하세요
- 연락처를 탭하여 프로필 보기
- **문서 및 파일(Docs and Files)**로 스와이프
- **업로드(Upload)**를 탭하여 갤러리, 카메라 또는 기기 저장소에서 파일 선택

## **자주 묻는 질문**

**Q: 파일이 나에게만 비공개인가요, 아니면 팀 전체와 공유되나요?**
파일은 해당 연락처 상세에 접근 권한이 있는 팀원들과 공유됩니다.

**Q: 업로드한 문서를 삭제하거나 이름을 바꿀 수 있나요?**
문서는 삭제할 수 있지만, 이름 변경은 현재 지원되지 않습니다.

**Q: 다운로드하기 전에 파일을 미리볼 수 있나요?**
네, PDF, 이미지, PowerPoint 등의 파일은 플랫폼 내에서 미리볼 수 있습니다.

**Q: 파일이 어떤 폴더에 들어갈지는 어떻게 결정되나요?**
수동 업로드는 **내부**로, 발송 메시지의 파일은 **발송**으로, 수신 메시지의 파일은 **수신**으로 갑니다. 커스텀 필드 연결 파일은 **커스텀 필드** 하위에 그룹화됩니다.

**Q: 스마트폰이나 태블릿에서 파일을 업로드할 수 있나요?**
네, 모바일 앱에서 갤러리, 파일, 카메라로부터 파일 업로드를 지원합니다.

**Q: 폴더 외에 파일을 태그하거나 분류할 방법이 있나요?**
아직은 없지만, 커스텀 필드를 활용해 특정 업로드를 체계적으로 라벨링할 수 있습니다.
