# **모바일 수신 전화 전환이란?**

수신 전화 전환(Inbound Call Transfer) 기능을 통해 모바일 앱 사용자는 받은 전화를 실시간으로 다른 직원에게 넘길 수 있습니다. 즉시 통화를 종료하고 넘기거나(블라인드 전환) 상황 설명을 위해 잠시 통화에 남아있을 수도 있어요(웜 전환). 모바일 기기에서 전화를 처리할 때 더 많은 유연성과 제어권을 제공합니다.

---

## **모바일 수신 전화 전환의 주요 장점**

모바일 전화 전환은 특히 사무실 밖에 있을 때 더 나은 팀워크와 원활한 고객 서비스를 가능하게 해줍니다.

- 모바일 앱에서 전화를 매끄럽게 전환
- 통화 유형에 따라 웜 전환 또는 블라인드 전환 선택
- 필요시 중요한 상황 설명을 위해 대화에 계속 참여
- 앱에서 직접 모든 통화 참여자 확인 및 관리
- 다른 사람들을 위해 통화를 종료하지 않고 본인만 통화 종료
- 필요한 경우 다른 참여자(팀원 등) 제거 또는 연결 해제

---

## **모바일 앱에서 수신 전화 전환 사용하기**

모바일 기기에서 전화 전환을 시작하는 방법을 이해하면 실시간 통화 중 빠르고 효과적인 인계가 가능합니다.

#### **1단계:** 수신 전화 받기

Hyperclass 또는 LeadConnector 모바일 앱에서 들어오는 전화를 받아주세요.

#### **2단계:** 전환 버튼 누르기

통화 화면 **우측 상단**에 있는 **Transfer(전환)** 버튼을 눌러주세요.

![모바일 앱 전화 전환 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155066132030/original/nAmB8AEr_pZRs-cTQdVBB0G1Pdlz2NWQoQ.png?1772571890)

#### **3단계:** 수신자 선택

다음 중에서 팀원을 선택해주세요:

- 직원 목록(Staff List)
- 연락처(Contacts)
- 다이얼 패드(Dial Pad)
- 최근 통화(Recent Calls)

#### **4단계:** 전환 유형 선택

다음 중 하나를 선택하라는 메시지가 나타납니다:

- **웜 전환(Warm Transfer)** – 상황 설명이나 발신자 소개를 위해 통화에 계속 참여
- **블라인드 전환(Blind Transfer)** – 즉시 전화를 넘기고 연결 해제

![전환 유형 선택 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155066132232/original/dce6z0ViD3wgBt6sSF13QkG1TuQ0vfG5Og.png?1772572201)

#### **5단계:** 통화 참여자 관리 (웜 전환의 경우)

- 화면 하단에서 참여자 패널을 위로 슬라이드해주세요
- 다음 중 선택하세요:
  - 통화를 종료하지 않고 본인만 통화 종료
  - 모든 사람을 위해 통화 종료
  - 개별 참여자 제거 (예: 다른 직원)

![통화 참여자 관리 패널](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155066131957/original/p8nulVcE04DAYIjP3yI3D8g8qJAf4KkHHw.png?1772571756)

## **웜 전환 vs 블라인드 전환**

웜 전환과 블라인드 전환의 차이점을 이해하면 상황에 따라 최적의 방법을 선택할 수 있습니다.

**웜 전환(Warm Transfer)**

본인이 통화에 계속 참여한 상태로 전화를 전환하는 방식으로, 다음 상황에 이상적입니다:

- 다른 팀원 소개
- 고객 상황 설명 제공
- 에스컬레이션 감독 또는 모니터링

**블라인드 전환(Blind Transfer)**

즉시 전화를 전환하고 본인은 연결을 해제하는 방식입니다. 다음 상황에 가장 적합해요:

- 본인이 계속 참여할 필요 없는 빠른 인계
- 외부 부서나 제3자로의 통화
- 음성메일이나 외부 번호로 에스컬레이션

---

## **스마트 참여자 관리 기능**

스마트 참여자 제어 기능을 통해 휴대폰에서 직접 누가 언제 통화에 참여하는지를 동적으로 관리할 수 있습니다.

다음 작업이 가능해요:

- 스크롤 가능한 하단 시트로 모든 참여자 확인
- 다른 사람들을 위해 통화를 종료하지 않고 본인만 통화 종료
- 다른 팀원을 통화에서 제거
- 필요한 경우 모든 참여자를 위해 전체 통화 종료

## **자주 묻는 질문**

**Q: 발신 전환과 수신 전환의 차이는 무엇인가요?**

수신 전환은 전화를 받았을 때 다른 사람에게 넘기려고 사용하는 기능이고, 발신 전환은 본인이 전화를 걸었을 때 다른 사람을 참여시키려고 할 때 발생합니다.

**Q: 외부 번호로도 전화를 전환할 수 있나요?**

네! 사용자 설정에서 허용된다면 다이얼 패드를 사용해서 외부 전화번호로 전환할 수 있어요.

**Q: 모든 모바일 버전에서 사용할 수 있나요?**

이 기능은 다음 버전에서 지원됩니다:

- **Hyperclass v3.97.3 이상**
- **LeadConnector v3.97.3 이상**

화이트라벨 앱의 경우, 제공업체에 업데이트를 요청해야 합니다.

**Q: 전환한 후에 통화로 다시 돌아갈 수 있나요?**

아니요, 블라인드 또는 웜 전환을 완료하고 나간 후에는 대화에 다시 참여할 수 없어요.

**Q: 발신자가 전환되고 있다는 것을 알 수 있나요?**

시스템에서 자동으로 알림을 주지는 않지만, 웜 전환 중에 본인이 말하면 발신자가 목소리를 들어서 상황을 이해할 수 있어요.
