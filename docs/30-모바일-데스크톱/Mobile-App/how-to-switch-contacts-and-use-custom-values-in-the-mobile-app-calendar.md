# 모바일 앱 캘린더에서 연락처 변경 및 커스텀 값 사용하는 방법

**목차**

- [새로운 기능](#새로운-기능)
- [왜 중요한가요](#왜-중요한가요)

모바일 앱 캘린더(Calendars)에 대한 최신 업데이트를 발표하게 되어 기쁩니다. 이제 커스텀 미팅 장소 필드에서 커스텀 값(Custom Values)을 지원합니다. 하위 계정 관리자와 사용자들은 새 예약(Appointment)을 진행하는 도중에도 연락처(Contacts)를 변경할 수 있습니다. 이 업데이트는 HighLevel, LeadConnector, 그리고 화이트라벨(White-labelled) 모바일 앱에서 이용 가능합니다(화이트라벨의 경우 업데이트 요청 필요).

## **새로운 기능**

- 예약 진행 중 연락처 변경 - 새 예약을 잡는 도중에 연락처를 업데이트할 수 있습니다.
- 커스텀 미팅 장소에 커스텀 값 지원: 이제 커스텀 미팅 장소 필드에서 커스텀 값을 사용할 수 있습니다.
- 캘린더 노트 개선: 캘린더 노트가 더 이상 잘리지 않습니다. "더 보기"를 탭하면 캘린더 노트에 추가된 전체 텍스트를 볼 수 있습니다.

## **왜 중요한가요**

이번 업데이트는 캘린더 모듈의 기능을 향상시켜 전반적인 사용자 경험을 개선합니다.
