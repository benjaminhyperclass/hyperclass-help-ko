# 모바일 앱 통합 연락처 워크스페이스

하이퍼클래스 모바일 앱의 통합 연락처 워크스페이스(Unified Contact Workspace)를 소개합니다. 연락처 워크스페이스는 연락처의 모든 세부 정보, 할 일, 노트, 활동 피드를 하나의 깔끔한 화면에 모아주고, 길게 누르기 빠른 작업을 추가하며, 몇 초 만에 태그를 달거나 일괄 업데이트를 할 수 있게 해줍니다. v3.9 (또는 그 이상)로 업데이트하고 여러 메뉴를 뒤지지 않고도 고객 관리를 더 빠르게 해보세요.

# 모바일 연락처 워크스페이스란?

워크스페이스는 흩어진 화면들을 하나의 탭 기반 허브로 교체합니다. 연락처의 핵심 정보는 상단에 위치하고, 전화 및 SMS 같은 작업은 언제나 한 번의 탭으로 가능하며, 탭을 통해 복잡한 데이터를 체계적으로 정리하면서도 쉽게 접근할 수 있습니다. 이제 필요한 모든 것이 한 곳에 있어 스크롤과 화면 전환을 줄입니다.

## 연락처 워크스페이스의 주요 장점

'왜' 바뀌었는지 이해하면 팀 전체에 변화를 더 잘 전파할 수 있습니다.

- **올인원 뷰 → 앱 전환 제로**: 일반 정보, 할 일, 노트, 전체 활동 피드가 이제 하나의 지붕 아래 쌓여 있습니다.

- **더 빠른 아웃리치**: 원탭 전화, SMS, 이메일 버튼으로 추가 탭을 제거합니다.

- **탭 내비게이션**: 복잡한 정보를 깔끔하게 유지—컨텍스트를 잃지 않고 세부 정보에서 할 일 & 노트로 전환할 수 있습니다.

- **일괄 태그 지정**: 몇 번의 탭으로 수백 개의 레코드를 정리합니다.

- **미래 대응 설계**: 동일한 패턴이 기회 관리, 예약 등으로 확장될 예정이므로 모든 모듈이 친숙하게 느껴집니다.

## 연락처 화면 탐색하기

이 빠른 투어로 1분 안에 생산성을 높일 수 있습니다.

- 연락처(Contacts) 탭을 열어보세요 (앱 v3.9 이상).

- 연락처 타일을 길게 눌러 빠른 작업 트레이를 팝업으로 띄워보세요.

- 세부 정보(Details)를 탭하여 접힌 섹션들을 탐색해보세요—일반 정보, 추가 정보, 작업, DND.

- 탭 바를 통해 할 일 & 노트(Tasks & Notes) 또는 활동(Activities)으로 스와이프하세요.

- 다중 선택 (••• 메뉴 → 선택)으로 연락처를 일괄 태그 지정하세요.

## 탭별 상세 안내

탭은 "지금 당장 필요한" 정보와 더 깊은 히스토리를 분리하여, 집중하면서도 정보를 놓치지 않게 합니다.

| 탭 | 목적 중심의 가치 |
|---|---|
| 할 일 & 노트 | 연락처와 연결된 할 일과 미팅 노트를 추적—아무것도 놓치지 않습니다. |
| 세부 정보 | 웹으로 이동하지 않고 일반 필드, 커스텀 값, 작업, DND 토글을 보고 편집합니다. |
| 활동 | 전화, 예약, 설문, 결제, 페이지 방문 등의 시간순 피드를 스크롤—다시 전화하기 전 빠른 컨텍스트 파악에 완벽합니다. |

## 태그 관리 및 일괄 작업

태그는 데이터베이스를 깔끔하게 유지하고 타겟 캠페인을 강화하며, 다중 선택은 유지보수를 쉽게 만듭니다.

- 체크박스를 체크하여 여러 연락처를 하이라이트하세요.

- 하단 액션 바를 사용하여 태그 추가(Add Tags) 또는 태그 제거(Remove Tags)—변경 사항이 웹에 즉시 동기화됩니다.

- 곧 출시 예정: 삭제, 배정, DND 토글 같은 추가 일괄 작업.

## 연락처 수준의 커스텀 값

커스텀 값은 대규모로 아웃리치를 개인화합니다—연락처에 바로 저장하여 모든 텍스트, 이메일, 워크플로우 활용이 정확해집니다.

- 세부 정보 → 추가 정보에서 추가하거나 편집하세요.

- 일반적인 {{custom_value}} 구문으로 모든 템플릿에서 값을 참조하세요.

- 커스텀 값은 자동화된 캠페인을 포함하여 연락처 레코드가 이동하는 모든 곳에 함께 이동합니다.

## 빠른 작업 타일

속도가 중요할 때, 길게 누르기는 깊은 메뉴보다 낫습니다.

- **전화**는 휴대폰 다이얼러를 실행합니다.

- **SMS**는 인앱 텍스트 작성기를 엽니다.

- **이메일**은 기본 메일 클라이언트를 트리거합니다.

- **예약하기**는 캘린더 플로우에 연결됩니다.

- **결제 수집**은 내장 결제 시트로 라우팅됩니다.

작업들은 이미 신뢰하는 기기 기본값을 상속합니다—추가 설정이 필요 없습니다.

## 자주 묻는 질문

**1. 어떤 앱 버전이 필요한가요?**
iOS 또는 Android에서 GoHighLevel v3.9+ (또는 LeadConnector v3.9+). 화이트라벨 앱: 지원팀에 3.9 빌드를 요청하세요.

**2. 이전 화면으로 되돌릴 수 있나요?**
아니요—레거시 레이아웃은 코드를 간소하게 유지하고 새 기능을 더 빠르게 제공하기 위해 종료되었습니다.

**3. 웹 태그가 모바일에 즉시 동기화되나요?**
예—태그는 실시간입니다. 웹에서 편집하면 몇 초 안에 모바일에서 변경 사항을 볼 수 있습니다 (그 반대도 마찬가지).

**4. 다중 선택이 오프라인에서 작동하나요?**
오프라인에서 연락처를 선택할 수 있지만, 태그 업데이트는 다시 온라인이 될 때까지 대기열에 들어갑니다.

**5. 한 번에 몇 개의 태그를 일괄 적용할 수 있나요?**
일괄 작업당 최대 500개의 연락처와 10개의 태그—제한은 실수로 인한 과부하를 방지합니다.

**6. 커스텀 값이 자동화에서 바로 사용 가능한가요?**
물론입니다—저장되면 워크플로우, 이메일 빌더, SMS 작성기에서 즉시 액세스할 수 있습니다.
