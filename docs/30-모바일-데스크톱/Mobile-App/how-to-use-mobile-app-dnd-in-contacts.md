# **연락처 관리에서 모바일 DND란?**

모바일 DND 향상 기능을 통해 Hyperclass 모바일 앱의 가장 중요한 상호작용 지점에서 통화, SMS 및 기타 연락 방법을 차단할 수 있습니다. 연락처 설정으로 들어갈 필요 없이, 이제 통화 화면, 최근 통화 기록 또는 대화 인박스에서 번호를 DND 목록에 즉시 추가할 수 있습니다.

## **모바일 DND의 주요 장점**

더 빠른 차단 및 규정 준수 도구 접근으로 커뮤니케이션과 연락처 전략을 제어하세요.

- 수신 통화, 통화 기록, 대화 스레드에서 원터치 DND 접근
- DND 표시된 연락처가 캠페인, 워크플로우, 자동화에 포함되지 않도록 방지
- 채널별(통화, SMS, 이메일, 소셜) 선택적 차단 지원
- 아웃리치 규정 준수 유지에 도움
- 다단계 네비게이션을 줄여 시간 절약
- 대용량, 이동 중 커뮤니케이션 환경을 지원하도록 설계

## **모바일 DND 사용 방법**

모바일 DND 도구는 필요한 곳에서 필요할 때 바로 제어할 수 있게 해줍니다. 최소한의 노력과 최대한의 효율성으로 원하지 않는 연락처를 차단하거나 방해 요소를 제거할 수 있습니다.

앱에서 현재 위치에 따라 DND를 적용하세요: 통화 중, 통화 기록 검토 중, 또는 진행 중인 대화 관리 중.

## **수신 통화 화면에서 DND**

통화 화면의 새로운 DND 단축키를 사용하여 실시간 통화 중에 원하지 않는 번호를 빠르게 차단하세요.

통화를 받을 때:

- 수신 통화 화면에서 "DND" 버튼을 탭하세요
- 번호가 즉시 DND 목록에 추가됩니다
- 현재 통화가 자동으로 종료됩니다
- 해당 번호의 모든 향후 통화와 메시지가 차단됩니다

## **통화 기록에서 DND**

연락처 프로필을 열지 않고도 최근 통화에서 직접 DND 목록에 연락처를 추가하세요.

방법:

- 통화 기록 섹션으로 이동하세요
- 최근 통화 항목을 탭하세요
- 새로운 DND 옵션을 선택하여 해당 번호의 향후 통화와 메시지를 차단하세요

이는 부재중 전화나 완료된 통화 후에 스팸이나 부적격 리드를 빠르게 차단하는 데 특히 유용합니다.

## **대화 스와이프 액션에서 DND**

간단한 스와이프로 전체 대화 스레드에 DND를 적용하여 채널별 제어 기능을 제공합니다.

단계:

- 대화 화면을 여세요
- 원하는 대화를 왼쪽으로 스와이프하세요
- DND 아이콘을 탭하세요
- 차단할 커뮤니케이션 채널을 선택하세요(예: 통화만, SMS, 또는 모든 채널)

이를 통해 아웃리치 노력을 규정에 맞게 유지하고 집중하면서도 깨끗한 인박스를 유지할 수 있습니다.

## **자주 묻는 질문**

**Q: DND 액션을 하면 CRM에서 연락처가 삭제되나요?**

아니요. 연락처는 CRM에 남아있지만, 선택된 채널에 따라 커뮤니케이션이 차단됩니다.

**Q: DND를 되돌리거나 제거할 수 있나요?**

네. 연락처 프로필을 방문하여 DND 구성을 업데이트하면 DND 설정을 변경할 수 있습니다.

**Q: 모바일에서 설정한 DND가 데스크톱 버전과 동기화되나요?**

네. 모바일 앱에서 변경한 DND 설정은 계정의 모든 플랫폼에 반영됩니다.

**Q: 소셜 미디어 메시지도 차단되나요?**

네. 대화에서 DND를 적용할 때 소셜 미디어 채널을 선택하면 차단됩니다.

**Q: SMS만 또는 통화만 차단할 수 있나요?**

네. DND 기능을 통해 차단을 적용할 특정 채널을 선택할 수 있습니다.
