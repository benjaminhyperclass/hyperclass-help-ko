# 동적 상품 - 모바일 거래에서 상세 정보 편집

**거래를 진행하면서 상품에 이미 구성된 기존 정보만 사용해야 하는 것은 아닙니다. 이름, 설명, 세금, 가격, 수량 등 모든 정보를 자유롭게 수정할 수 있습니다. 변경 사항을 카탈로그에 저장해서 나중에 다시 사용할 수도 있습니다.**

- 커스텀 항목 편집기
  - 키패드를 사용해서 POS/Invoice/Estimate에 커스텀 금액을 추가할 때, 이름을 '커스텀 항목'에서 원하는 이름으로 변경할 수 있습니다! 이 기능은 키패드 탭에서 금액 아래에 표시되는 강조된 항목 상세 버튼을 탭하면 사용할 수 있습니다. 최종 인보이스나 영수증에 포함시키고 싶다면 해당 커스텀 항목에 설명도 추가할 수 있습니다.

- 커스텀 항목에 세금 적용
  - 키패드로 추가한 커스텀 항목에 세금을 연결할 수 있습니다. 이 기능은 키패드 탭에서 금액 아래의 강조된 항목 상세 버튼을 탭하면 사용할 수 있습니다.

- 나중에 사용하기 위해 저장
  - 나중에 다시 사용하고 싶다면 커스텀 항목을 카탈로그에 저장할 수 있습니다. 키패드 탭에서 금액 아래의 강조된 항목 상세 버튼을 탭하고 '나중에 사용하기 위해 저장' 옵션을 켜면 됩니다.

- 검토 단계에서 항목 편집
  - 위에서 언급한 모든 기능은 키패드 페이지와 검토 페이지, 두 곳에서 모두 사용할 수 있습니다. 검토 페이지에서도 카탈로그 항목의 상세 정보를 편집할 수 있는데, 항목 타일을 탭하고 필요한 모든 정보를 수정하면 됩니다.

## 동적 상품 사용 방법

- 선호하는 모바일 앱(HighLevel, LeadConnector 또는 화이트라벨 앱)을 기기에 설치하세요.
- 앱 하단 바의 "$" 아이콘을 탭해서 POS 섹션으로 이동하세요.
- 키패드(커스텀 금액용) 또는 카탈로그를 사용해서 체크아웃에 항목을 추가하세요.
- 키패드로 추가한 금액의 경우, 강조된 "항목 상세" 버튼이 표시됩니다. 이 버튼을 탭하세요.
- 새로운 화면이 열리면서 항목 이름을 '커스텀 항목'에서 원하는 이름으로 변경할 수 있습니다. 설명 추가, 세금 연결, 수량 조정, 가격 변경도 가능합니다. 항목을 카탈로그에 저장해서 나중에 사용할 수도 있습니다.
- 카탈로그 항목을 추가하는 경우, 항목 타일을 탭해서 이름, 설명, 가격, 수량을 편집할 수 있습니다. 여러 변형이 있다면 올바른 변형 조합을 선택해서 체크아웃 단계에 추가할 수 있습니다.
- 같은 상품의 다른 변형 조합을 추가하고 싶다면, 카탈로그 페이지에서 수량 카운터의 "+" 버튼을 탭하고, 새 커스터마이징을 탭하세요.
- 체크아웃에 항목을 추가했다면, "결제" 버튼을 탭해서 검토 단계로 이동하세요.
- 상품 타일을 탭하면 이전과 같은 화면이 열리면서 상품 상세 정보를 추가로 편집할 수 있습니다.
- 거래에 대한 최종 검토를 완료한 후, "결제 방법 선택" 버튼을 탭하고 고객에게 결제할 결제 방법을 선택하거나, 거래를 인보이스나 견적서로 변환할 수 있습니다.
