## 데스크톱 앱 설치 (macOS, Windows & Linux)

아래에서 사용하는 운영체제를 선택하세요. 각 과정은 간단하고 명확하니, 순서대로 네 단계를 따라하세요.

### **macOS**

- **설치 파일 다운로드** - 위의 macOS **(.dmg)** 링크를 사용하세요.

![macOS 설치 파일 다운로드](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155061959620/original/aKSZgs_a0R1EUTGZjZInzLj9LIPAxajuMQ.gif?1767666117)

- **설치** - .dmg 파일을 열고 **HighLevel을 Applications 폴더로 드래그**하세요.

![macOS HighLevel 설치](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155061959840/original/kaDik86DRGPYgBxrulUxatN4JLTwI3L8Uw.gif?1767666746)

- **HighLevel 실행** - **Applications**, **Launchpad** 또는 **Spotlight**에서 실행하세요.

- **로그인** - 데스크톱 앱에서 **Sign In**을 클릭하세요. 기본 브라우저가 열립니다. 해당 브라우저에서 이미 HighLevel 웹 앱에 로그인되어 있다면 자동으로 데스크톱 앱으로 돌아옵니다. 그렇지 않다면 브라우저에서 로그인을 완료하면(화이트라벨 도메인이 활성화된 경우 해당 도메인 사용) 앱으로 돌아갑니다.

![macOS HighLevel 로그인](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155061960526/original/AC9XbfFHvob-n3S1MOHcZtu2HLqMMUREbQ.gif?1767669339)

### **Windows**

- **설치 파일 다운로드** — 위의 Windows **(.exe)** 링크를 사용하세요.

![Windows 설치 파일 다운로드](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155061959970/original/iuvZ8-UIbTH7DUtiONX6Lso-Gw73xBGAsw.gif?1767667253)

- **설치** - .exe 파일을 열고 설치 마법사를 완료하세요.

![Windows HighLevel 설치](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155061960682/original/NfEem5BRad__9Ro3V_FL_WVd8ojbjCvo7A.gif?1767670217)

- **HighLevel 실행** — **시작 메뉴**에서 실행하세요.

- **로그인** - 데스크톱 앱에서 **Sign In**을 클릭하세요. 기본 브라우저가 열립니다. 해당 브라우저에서 이미 HighLevel 웹 앱에 로그인되어 있다면 자동으로 데스크톱 앱으로 돌아옵니다. 그렇지 않다면 브라우저에서 인증 정보를 입력하면(화이트라벨 도메인이 활성화된 경우 해당 도메인 사용) 앱으로 돌아갑니다.

![Windows HighLevel 로그인](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155061960415/original/pBaH-trhaoOaSG4d4u-vPFN-6eRkaz57IA.gif?1767668890)

### **Linux**

- 위 링크에서 사용하는 배포판과 아키텍처에 맞는 Linux 설치 파일을 다운로드하세요.

- 시스템의 표준 설치 과정을 사용하여 설치하세요.

- 애플리케이션 메뉴에서 데스크톱 앱을 실행하세요.

![Linux 데스크톱 앱 실행](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155068381151/original/kc-_bCYFbw_cmfGo-mMBYBCz8TAdb-BZ7A.png?1775147156)

- **Sign In**을 클릭하세요. 기본 브라우저가 열려서 로그인을 완료하고, 다시 데스크톱 앱으로 돌아옵니다.

![Linux HighLevel 로그인](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155068381154/original/ihE3t3mCqZLagYJspRJa9WfQ8RqBwnVyYQ.png?1775147173)

## 자주 묻는 질문

**Q: 데스크톱 앱이 HighLevel 웹 앱과 다른가요?**
사용 경험과 데이터는 동일합니다. 데스크톱 앱은 HighLevel을 전용 최적화된 창에서 네이티브 알림과 함께 제공하는 것입니다.

**Q: 여러 계정이나 하위 계정을 사용할 수 있나요?**
네 - 일반적으로 로그인한 후, 여러 하위 계정에 액세스 권한이 있다면 계정 메뉴를 사용해서 전환할 수 있습니다.

**Q: 화이트라벨 도메인이 로그인에 영향을 주나요?**
네 - 활성화되어 있다면, 로그인 시 브랜드 로그인 페이지를 거쳐 앱으로 돌아오게 됩니다.

**Q: 알림이 보이지 않는다면 어떻게 하나요?**
HighLevel에 대한 OS 수준 알림을 활성화하고, 앱 내 환경설정을 검토하세요.
