# Stripe 모바일 POS 단말기로 결제 받기

Stripe에서 판매하는 특정 결제 단말기를 모바일 앱에 연결하여 고객이 카드를 탭하거나 삽입, 스와이프하여 결제할 수 있습니다! 이 기능은 Hyperclass, Leadconnector, 화이트라벨 앱 모두에서 사용 가능합니다.

지원되는 단말기는 Stripe M2 리더와 BBPOS WisePad 3입니다.

---

## 하드웨어 단말기로 결제 받는 방법

원하는 모바일 앱(Hyperclass, LeadConnector, 또는 화이트라벨)을 기기에 설치한 후 다음 단계를 진행하세요.

1. Navigation 메뉴(네비게이션 메뉴) > Settings(설정) (기어 아이콘) 열기
2. Payment Devices(결제 디바이스)를 탭하세요

이전에 이미 결제 기기를 연결한 적이 있다면, "My devices(내 디바이스)" 섹션에서 해당 기록을 볼 수 있습니다.

처음으로 기기를 연결하는 경우, 화면이 자동으로 페어링 모드로 열립니다. 결제 단말기가 켜져 있고 배터리가 최소 25% 이상인지 확인하세요.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036272657/original/AwsTkFf7tcdfGSZNMyGht5_GrDOrBcP6lA.png?1731113374)

페어링 화면에 단말기가 표시되면 연결(connect)을 탭하세요. 앱이 단말기의 업데이트와 상태를 확인합니다. 페어링 과정이 완료되고 모든 업데이트가 설치되면 단말기를 사용하여 결제를 받을 수 있습니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036272669/original/9O1DjTwCzyt0gn2pfF8krf6hz_QFWk3mYw.png?1731113477)

3. 선호하는 방식에 따라 POS 탭 또는 Invoices(인보이스) 탭으로 이동하세요.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036272675/original/Xim0--vS3VtbeORRbknK2XAym5bavuyoNQ.png?1731113549)

4. 결제 수단 화면에서 "Pay with Hardware(하드웨어로 결제)"라는 제목의 타일을 볼 수 있습니다. 이를 탭하면 단말기가 결제를 받을 준비가 됩니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036272684/original/plKFYBMyWxMz_b2B5v_spSidRIgENylQZw.png?1731113588)

이제 고객이 카드를 탭하거나 삽입, 스와이프하여 거래를 완료할 수 있습니다!

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036272698/original/3tQTbbfls8yF3Nv2rHy1yp_dlUZxXkuqng.png?1731113667)
