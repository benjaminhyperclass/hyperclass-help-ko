# 채팅 위젯 어트리뷰션

## 웹 채팅 대화의 트래픽 소스 확인하기

### 1. 연락처 상세 페이지에서 채팅 위젯 소스 보기

연락처가 라이브 채팅 위젯을 통해 대화를 시작하면, 연락처(Contacts) 메뉴의 연락처 상세 페이지에서 활동(Activity) 탭에 해당 이벤트가 표시됩니다.

![채팅 위젯으로 시작된 대화가 활동 탭에 표시되는 모습](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48087692052/original/cHxVNz-bkXEPjlbMI6bxwTHMO5PU1ni4Og.jpg?1613574964)

클립보드 아이콘을 클릭하면 폼 제출 내역(즉, 채팅 위젯에 처음 입력한 메시지)이 열리며, 여기서 소스(오가닉 검색, 직접 접속 등)를 포함한 더 자세한 정보를 확인할 수 있습니다.

![폼 제출 상세 정보에서 트래픽 소스를 확인하는 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48087692717/original/BoqPdR_08SjAQndwajoOr2I8Hm_aNy53Aw.jpg?1613575103)

### 2. 폼 제출에서 채팅 위젯 소스 보기

라이브 채팅 위젯을 통해 시작된 대화의 트래픽 소스는 `마케팅(Marketing) → 폼 빌더(Form Builder) → 제출(Submissions)` 탭으로 이동한 후, 폼 선택 드롭다운의 채팅 위젯 섹션에서 "위젯 폼(Widget Form)"을 선택해서도 확인할 수 있습니다.

![폼 제출 페이지에서 위젯 폼을 선택하는 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48087689254/original/Yjx-8g6ASgeLimbZr2sAbPRWLZWdxOJy_A.jpg?1613574447)

## 참고사항

- 하이퍼클래스가 아닌 외부 웹사이트에 추가한 라이브 채팅 위젯에서도 어트리뷰션이 작동합니다. 여기에는 LeadConnector 워드프레스 위젯을 사용하는 사이트도 포함됩니다.
- 일부 iOS 사용자나 프라이버시 모드를 켠 사용자의 경우 쿠키가 차단되어 어트리뷰션이 작동하지 않을 수 있습니다.
