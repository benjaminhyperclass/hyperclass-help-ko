# 워크플로우 트리거 - 카테고리 완료

이 가이드에서는 상품이나 강의 완료에 따라 워크플로우를 자동화하는 강력한 도구인 **"카테고리 완료(Category Completed)"** 트리거를 소개합니다. 이 트리거가 무엇인지, 주요 장점, 단계별 설정 방법, 다양한 업종에서의 활용 사례를 알아보겠습니다. 마지막으로 자주 묻는 질문을 통해 이 트리거의 잠재력을 최대한 활용할 수 있도록 도와드리겠습니다.

---

# **카테고리 완료 트리거란?**

카테고리 완료 트리거는 연락처가 지정된 상품이나 강의 카테고리를 완료했을 때 워크플로우를 시작하는 기능입니다. 사용자 진도를 추적하고 완료 이메일 발송, 수료증 발급, 다음 단계 추천과 같은 후속 작업을 자동화하는 데 특히 유용합니다. 선택 가능한 필터를 통해 관련 카테고리가 완료됐을 때만 워크플로우가 실행되도록 할 수 있습니다.

---

## **트리거 사용 시 주요 장점**

- **자동화된 후속 조치:** 이메일 발송, 수료증 생성, 새로운 워크플로우 배정 같은 완료 후 작업을 간소화합니다.

- **개인화된 참여:** 완료 활동을 기반으로 맞춤형 추천이나 축하 메시지를 전달합니다.

- **효율성 향상:** 카테고리 완료를 수동으로 모니터링할 필요가 없어 시간과 리소스를 절약합니다.

- **고객 유지율 증대:** 강의나 상품 카테고리 완료 후 관련 다음 단계를 제공하여 지속적인 참여를 유도합니다.

- **커스터마이징 가능한 필터:** 워크플로우 실행을 위한 특정 카테고리를 타겟팅하여 정확성과 관련성을 보장합니다.

### **트리거 저장**

설정을 확인하고 워크플로우가 예상대로 작동하는지 테스트합니다.

### **워크플로우 발행**

트리거 조건이 충족될 때 자동화가 시작되도록 워크플로우를 활성화합니다.

---

## **활용 사례**

### **교육 플랫폼**

**시나리오:** 온라인 학습 플랫폼에서 강의의 첫 3개 레슨을 완료한 수강생에게 자동으로 후속 조치를 취하고 싶은 경우

**트리거 설정:**

- **트리거:** 카테고리 완료
- **이름:** 3개 레슨 완료
- **필터:** 상품: "스프린트 코스", "마라톤 코스" 중 하나
- **워크플로우 액션**
  - 수강생에게 축하 이메일 발송
  - 강사에게 추가 후속 조치를 위한 알림

**결과:** 적시에 개인화된 후속 조치로 수강생의 학습 경험을 향상시킵니다.

---

### 이커머스

**시나리오:** 이커머스 스토어에서 고객이 특정 상품 카테고리에서 구매를 완료했을 때 관련 상품을 추천하고 싶은 경우

**트리거 설정:**

- **트리거:** 카테고리 완료
- **이름:** 교차 판매 트리거
- **필터:** 상품 카테고리: "전자기기"
- **워크플로우 액션:**
  - 구매 상품에 대한 액세서리나 업그레이드를 제안하는 이메일 발송

**결과:** 고객이 보완 상품을 탐색하도록 유도하여 매출을 증대시킵니다.

## **자주 묻는 질문**

**트리거가 연락처의 중복 카테고리를 처리할 수 있나요?**

네, 연락처가 동시에 여러 카테고리를 완료하는 경우, 필터가 올바르게 설정되었다면 각 카테고리에 대해 별도로 워크플로우를 활성화할 수 있습니다.

**이 트리거를 단일 워크플로우에서 다른 트리거와 함께 사용할 수 있나요?**

네, 카테고리 완료 트리거는 "할 일 완료(Task Completed)"나 "구매 완료(Purchase Made)"와 같은 다른 트리거와 결합하여 복잡한 워크플로우를 만들 수 있습니다.

**연락처가 카테고리를 여러 번 완료하면 어떻게 되나요?**

완료 조건이 충족될 때마다 트리거가 활성화되어 완료를 놓치지 않도록 보장합니다.

**카테고리 내의 특정 상품을 대상으로 할 수 있나요?**

네, 필터를 사용하여 특정 상품이나 하위 카테고리에 집중하도록 트리거를 좁혀서 정확성을 보장할 수 있습니다.
