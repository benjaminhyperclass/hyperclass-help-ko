# 트리거 복제하기

### 참고: 이제 모든 계정에서 워크플로우를 사용할 수 있습니다. 워크플로우에서는 트리거와 캠페인의 모든 기능(그리고 더 많은 기능!)을 하나의 빌더에서 사용할 수 있어요! 워크플로우에 대해 자세히 알아보기

트리거를 복제하는 기능은 같은 트리거를 여러 클라이언트 로케이션에서 실행하고 싶을 때 유용합니다. 처음부터 트리거를 만들고 설정하는 대신, 간단히 복제한 후 필요한 부분만 조정하면 됩니다.

트리거를 복사하고 싶다면 아래 단계를 따라하세요:

- 클라이언트 뷰에서 Triggers(트리거) 메뉴를 클릭하세요.
- 복제하고 싶은 트리거를 클릭하세요.
- 우측 상단의 Actions(작업) 드롭다운을 클릭하세요.
- Duplicate Trigger(트리거 복제)를 클릭하세요.
- 새 트리거 이름을 입력하고 트리거를 복제할 클라이언트 로케이션을 선택하세요.
- Save(저장)를 클릭하세요.
- 다른 클라이언트 로케이션의 설정과 구성에 따라 복제된 트리거를 조정해야 할 수 있습니다.
- 모든 복제된 트리거는 초안 모드로 시작되므로 반드시 활성화하세요.
