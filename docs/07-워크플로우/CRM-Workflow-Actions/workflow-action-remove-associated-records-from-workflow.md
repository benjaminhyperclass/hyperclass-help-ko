# 워크플로우 액션 – 연관된 레코드를 워크플로우에서 제거하기

## 개요

'연관된 레코드를 워크플로우에서 제거하기(Remove Associated Records from Workflow)' 액션은 모든 워크플로우 유형에서 사용할 수 있어요 — 연락처 기반, 회사 기반, 커스텀 오브젝트 기반 워크플로우 모두에서요. 이 액션은 **트리거 레코드와 연관된 레코드**를 (선택한 오브젝트 + 라벨을 통해) 다른 워크플로우에서 자동으로 제거합니다. 상황이 변할 때 관련된 연락처, 회사 또는 커스텀 오브젝트 레코드의 깔끔한 워크플로우 해제를 보장해요.

이 액션은 기존의 [워크플로우에서 제거하기 액션]([workflow-action-remove-from-workflow](workflow-action-remove-from-workflow.md))과는 다릅니다. 기존 액션은 현재 레코드(예: 연락처 하나)만 워크플로우에서 제거하는 반면, '연관된 레코드를 워크플로우에서 제거하기'는 연관 라벨을 기반으로 연관된 레코드들(연락처, 회사 또는 커스텀 오브젝트)을 워크플로우에서 해제할 수 있어요.

## 액션 이름

연관된 레코드를 워크플로우에서 제거하기(Remove Associated Records From Workflow)

## 액션 설명

이 액션을 통해 워크플로우 사용자는 다음을 할 수 있어요:

- 오브젝트 유형 선택
- 제거할 연관된 레코드를 지정하는 연관 라벨 선택
- 연관된 레코드들이 제거될 워크플로우 선택

이 액션은 하위 계정에서 커스텀 오브젝트를 생성한 후(또는 연락처/회사 사용) 트리거 레코드와 제거될 레코드 사이에 연관 라벨이 존재할 때만 사용 가능해요. 여기에는 연락처-연락처 및 회사-연락처 연관도 포함됩니다.

## 액션 세부사항

![연관된 레코드를 워크플로우에서 제거하기 액션 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054812396/original/5fxj8Jwe5eWzFilNjCjFCjgaPgYdH7KDig.png?1759145373)

- **오브젝트 선택**: 제거할 연관된 레코드의 유형을 선택하세요 (예: 연락처, 딜러, 부동산)
- **연관 라벨(필수)**: 제거 대상인 연관된 레코드를 결정합니다 (회사-연락처 연관에는 적용되지 않음)
- **워크플로우**: 연관된 레코드들이 제거될 대상 워크플로우를 선택하세요

**주의사항 및 특수 경우:**
- 최소 하나의 일치하는 연관(라벨별)이 존재해야 하며, 그렇지 않으면 제거되는 레코드가 없어요
- 제거는 즉시 실행되며, 나중에 다시 등록해야 한다면 '연관된 레코드를 워크플로우에 추가하기' 액션이나 별도의 자동화를 사용하세요

## 예시

#### 예시 1: 자동차 ↔ 딜러 (커스텀 오브젝트 워크플로우)

자동차 레코드의 상태가 '판매됨'으로 변경되면, 연관된 딜러들을 (라벨: 배정된 딜러) '딜러 알림 워크플로우'에서 제거하여 더 이상 업데이트를 받지 않도록 해요.

**설정**
- 오브젝트 → 딜러
- 연관 라벨 → 배정된 딜러
- 워크플로우 → 딜러 알림 워크플로우

**작동 방식**
- 자동차 상태가 '판매됨'이 됩니다 (워크플로우 트리거)
- 액션이 '배정된 딜러'를 통해 자동차와 연관된 딜러들을 찾습니다
- 해당 딜러들이 딜러 알림 워크플로우에서 제거됩니다

#### 예시 2: 연락처 ↔ 연락처 (학부모-자녀, 연락처 워크플로우)

학생 연락처가 졸업하면, 모든 학부모 연락처들을 (라벨: 학부모-자녀) '학기 시작 알림' 워크플로우에서 제거해요.

**설정**
- 오브젝트 → 연락처(학부모)
- 연관 라벨 → 학부모-자녀
- 워크플로우 → 학기 시작 알림

**작동 방식**
- 학생 레코드가 '졸업함'으로 워크플로우를 트리거합니다
- 액션이 '학부모-자녀'를 통해 연관된 학부모들을 찾습니다
- 해당 학부모들이 학기 시작 알림 워크플로우에서 제거됩니다

#### 예시 3: 회사 ↔ 연락처 (회사 워크플로우)

회사 레코드의 상태가 '비활성'으로 변경되면, 모든 연락처를 '활성 고객 커뮤니케이션 워크플로우'에서 제거해요.

**설정**
- 오브젝트 → 연락처
- 워크플로우 → 활성 고객 커뮤니케이션 워크플로우

**작동 방식**
- 회사 레코드가 '비활성'으로 업데이트됩니다
- 해당 연락처들이 활성 고객 커뮤니케이션 워크플로우에서 제거됩니다
