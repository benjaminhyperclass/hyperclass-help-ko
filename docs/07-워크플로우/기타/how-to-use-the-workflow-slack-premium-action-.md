# 슬랙 워크플로우 액션 사용 가이드

슬랙 프리미엄 워크플로우 액션(Slack Premium Workflow Action)을 사용하면 슬랙 채널과 사용자에게 자동 메시지를 보낼 수 있어, 커뮤니케이션을 간소화하고 팀 협업을 개선할 수 있습니다. 이 액션은 세 가지 주요 이벤트를 제공합니다: 사용자에게 메시지 보내기, 비공개 채널에 메시지 보내기, 공개 채널에 메시지 보내기.

모든 요금제($97, $970, $297, $2970, $497, $4970)의 에이전시는 LC 프리미엄 트리거 & 액션에 접근할 수 있습니다. - 에이전시 설정(Agency settings)에서 프리미엄 액션 & 트리거가 활성화되면, 기존 및 새로운 하위 계정(Sub-Accounts)은 100회 무료 실행을 받습니다. - 에이전시가 기존 하위 계정에 대한 실행 비용이 발생하지 않도록 하려면, 에이전시 보기에서 각 하위 계정에 대해 재청구를 수동으로 활성화해야 합니다(자세히 보기). - SaaS Configurator에서 프리미엄 액션이 활성화된 경우, 생성된 새로운 하위 계정은 자동으로 LC 프리미엄 액션 & 트리거에 등록되므로 에이전시의 추가 조치가 필요하지 않습니다.

## 슬랙 프리미엄 액션이란?

슬랙 프리미엄 워크플로우 액션은 널리 사용되는 팀 협업 도구인 슬랙 내에서 메시징을 자동화하고 간소화하도록 설계된 고급 커뮤니케이션 기능입니다. 이 기능을 사용하면 슬랙 워크스페이스 내의 특정 사용자, 비공개 또는 공개 채널에 타겟 메시지를 보낼 수 있습니다. 사용자 친화적인 인터페이스와 커스터마이징 가능한 옵션을 통해 사용자가 필요에 따라 알림을 설정하고 예약할 수 있어 효과적인 커뮤니케이션을 보장합니다.

이 액션은 세 가지 주요 이벤트를 제공합니다:

- **사용자**에게 메시지 보내기: 사용자는 적절한 사람에게 직접 메시지를 보내기 위해 네 가지 옵션 중에서 선택할 수 있습니다: **배정된 사용자(Assigned User), 커스텀 이메일(Custom Email), 내부 사용자(Internal User)**, 또는 **슬랙 사용자(Slack User)**. 각 옵션은 메시지 수신자를 결정하는 고유한 목적을 가지고 있어 정확하고 효율적인 커뮤니케이션을 보장합니다.

- **비공개 채널**에 메시지 보내기: 사용자는 워크스페이스 내의 비공개 채널을 선택하여 메시지를 보낼 수 있으며, 이 메시지는 **슬랙 연동을 생성한 사용자가 직접 보낸 것처럼 표시됩니다**. 이를 통해 선택된 팀 구성원 그룹 내에서 보안 커뮤니케이션이 가능합니다.

- **공개 채널**에 메시지 보내기: 사용자는 워크스페이스 내의 공개 채널을 선택하여 메시지를 브로드캐스트할 수 있으며, **모든 워크스페이스 구성원이 정보에 접근할 수 있게** 됩니다.

## 누구를 위한 것인가요?

슬랙 프리미엄 워크플로우 액션은 커뮤니케이션과 협업을 위해 슬랙에 의존하는 조직과 팀에 유용합니다. 메시지와 알림을 자동화하여 워크스페이스 내의 정보 흐름을 간소화합니다. 이 기능의 좋은 사용 사례로는 다음과 같습니다:

- 프로젝트 관리: 새 작업이 할당되거나, 프로젝트 마감일이 다가오거나, 작업이 완료로 표시될 때 팀원에게 자동으로 알림을 보냅니다. 이를 통해 모든 사람이 순조롭게 진행하고 자신의 책임을 알 수 있도록 도울 수 있습니다.

- 영업 및 마케팅: 새로운 리드, 기회 또는 성사된 거래를 영업팀에 알려 빠르고 효율적으로 대응할 수 있도록 합니다. 마케팅 팀도 캠페인 성과, 콘텐츠 발행 또는 소셜 미디어 참여에 대한 자동 알림으로 이익을 얻을 수 있습니다.

- 고객 지원: 새로운 지원 티켓이 생성되거나, 에스컬레이션되거나, 해결될 때 지원팀에 자동 알림을 보내 신속한 응답과 효율적인 문제 해결을 보장합니다.

- 인사: 직원 온보딩, 교육 세션, 성과 리뷰 또는 정책 업데이트에 대해 HR과 관련 팀원에게 알려 모든 사람이 정보를 얻고 참여할 수 있도록 합니다.

- IT 및 개발: 새로운 버그가 보고되거나, 시스템 중단이 발생하거나, 소프트웨어 업데이트가 릴리스될 때 IT 또는 개발팀에 알림을 보내 문제를 신속하게 해결하고 시스템 안정성을 유지할 수 있도록 합니다.

- 회사 전체 공지: 중요한 뉴스, 업데이트 또는 알림을 공개 채널을 통해 전체 조직과 공유하여 모든 사람이 정보를 얻고 참여할 수 있도록 합니다.

## 주요 이점

워크플로우 슬랙 프리미엄 액션은 커뮤니케이션과 협업을 위해 슬랙을 사용하는 조직과 팀에 여러 이점을 제공합니다:

- 단일 인박스: 다양한 시스템 이벤트와 알림을 슬랙으로 가져옴으로써 팀원들이 주의를 기울여야 할 사항을 추적할 수 있는 한 곳을 갖게 됩니다.

- 책임감 증대: 작업 할당, 업데이트 또는 마감일에 대한 자동 알림은 팀원들이 자신의 업무에 책임을 지도록 장려하여 전반적인 성과와 책임감 향상으로 이어집니다.

- 실시간 업데이트: 워크플로우 슬랙 프리미엄 액션을 통해 팀은 중요한 이벤트나 업데이트에 대한 즉시 알림을 받을 수 있어 빠르게 응답하고 데이터 기반 의사결정을 내릴 수 있습니다.

- 확장성: 워크플로우 슬랙 프리미엄 액션은 성장하는 조직의 변화하는 요구사항에 빠르게 적응할 수 있어 팀이 확장되면서도 커뮤니케이션이 효율적이고 간소화된 상태를 유지할 수 있습니다.

## 슬랙 액션 설정 방법

### 사용자에게 직접 메시지 보내기

이벤트가 '사용자에게 직접 메시지 보내기(Send Direct Message to User)'인 경우 네 가지 옵션이 있습니다.

- 배정된 사용자(Assigned User) - 연락처의 배정된 사용자
- 커스텀 이메일(Custom Email) - 슬랙의 사용자와 일치하는 특정 이메일 추가
- 내부 사용자(Internal User) - 하위 계정의 사용자
- 슬랙 사용자(Slack User) - 이름으로 슬랙 워크스페이스 사용자를 직접 검색

![슬랙 사용자 옵션 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155043882515/original/WY-jy_j3uLrthoHitqP6TfAY5bc2DsZ-Pg.png?1742863301)

#### 배정된 사용자 (연락처의)

이는 배정된 연락처 사용자에게 알림을 보내는 데 사용됩니다. 이 경우, 연락처에 배정된 사용자의 이메일(설정 > 내 직원 > 사용자 정보)을 사용하여 슬랙 사용자를 찾습니다.

![배정된 사용자 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155043882521/original/qDbHeOIN8eGtShrDbIKb-Qdkk274cdCbCg.png?1742863333)

#### 커스텀 이메일

이는 동적 사용자에게 알림을 보내야 하는 모든 경우에 사용됩니다. 입력된 커스텀 이메일을 사용하여 일치하는 슬랙 사용자를 찾습니다.

이는 미리 정의된 사용자가 없는 모든 경우를 위한 것입니다. 이메일을 직접 입력하거나 커스텀 변수 선택기를 사용하여 커스텀 변수 목록에서 선택할 수 있습니다.

![커스텀 이메일 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155043882536/original/bIKGMvkR18mPfIqXxh6k-hiCUwL2AkUAtg.png?1742863413)

#### 내부 사용자

이 옵션을 사용하여 계정 사용자 중 한 명에게 메시지를 보냅니다. 일반적으로 비즈니스의 특정 섹션을 담당하는 특정 사용자에게 알림을 보내는 데 사용됩니다. 예를 들어, 성공한 기회에 대한 정보를 재무 담당자에게 보내는 것입니다. 선택된 내부 사용자의 이메일을 사용하여 슬랙 사용자를 찾습니다.

![내부 사용자 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155043882541/original/IBK7EGuhOxyzZsR_5fFfFxBLGeuDqc-HpA.png?1742863436)

#### 슬랙 사용자

이름 그대로 간단합니다. 워크스페이스의 모든 슬랙 사용자에게 직접 메시지를 보냅니다.

슬랙은 동시에 가져올 수 있는 사용자 수가 제한되어 있습니다. 따라서 이름을 입력해도 사용자가 보이지 않으면 '더 보기(load more)' 버튼을 눌러주세요. 필요한 사용자를 찾을 때까지 계속 검색합니다. 사용자가 목록 하단에 있다면 '더 보기' 버튼을 여러 번 눌러야 할 수도 있습니다. 이는 1000명 이상의 사용자가 있는 워크스페이스에만 해당됩니다.

![슬랙 사용자 검색](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155043882553/original/OIeh4on5XsyhM70sh01x1o4nNI7FTwPaLw.png?1742863500)

#### 메시지 내용

사용자에게 직접 메시지를 보낼 메시지를 입력합니다. 더 많은 형식 옵션은 [여기에서 확인하세요](https://api.slack.com/reference/surfaces/formatting).

광범위한 커스텀 변수 선택기를 사용하여 연락처 또는 CRM 관련 정보에 대한 정보를 보낼 수 있습니다.

![메시지 내용 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155043882609/original/paYnKdzTJ0bTdHmalMcDfFUgiav2hFNstg.png?1742863676)

#### 비공개 채널 선택

메시지를 보낼 원하는 채널을 선택합니다.

슬랙은 한 번에 가져올 수 있는 비공개 채널의 수가 제한되어 있습니다. 따라서 이름을 입력해도 원하는 채널이 보이지 않으면 '더 보기(load more)' 버튼을 눌러주세요. 필요한 채널을 찾을 때까지 계속 검색합니다. 채널이 목록 하단에 있다면 '더 보기' 버튼을 몇 번 눌러야 할 수도 있습니다. 이는 1000개 이상의 채널이 있는 워크스페이스에만 해당됩니다.

광범위한 커스텀 변수 선택기를 사용하여 연락처 또는 CRM 관련 정보에 대한 정보를 보낼 수 있습니다.

## **자주 묻는 질문**

**Q: 이 액션을 사용하여 보낸 모든 슬랙 메시지에 대해 요금이 청구되나요?**

네, 슬랙 프리미엄 액션은 실행당 요금이 발생합니다. 하지만 활성화되면 계정에 100회 무료 실행이 제공됩니다.

**Q: 드롭다운에서 내 슬랙 사용자나 채널이 보이지 않으면 어떻게 하나요?**

"더 보기(Load More)" 버튼을 사용하여 슬랙 워크스페이스를 더 깊이 검색하세요. 1000명 이상의 사용자나 채널이 있는 워크스페이스에서는 추가로 더 보기를 해야 할 수 있습니다.

**Q: 재청구가 활성화되지 않은 상태에서 슬랙 프리미엄 액션을 사용할 수 있나요?**

무료 실행을 사용하는 경우에만 가능합니다. 그 이후에는 중단 없이 사용하려면 하위 계정에 재청구가 활성화되어야 합니다.

**Q: 슬랙 메시지에 사용자 정보를 동적으로 삽입할 수 있나요?**

네. 커스텀 변수 선택기를 사용하여 연락처 정보, 거래 데이터 또는 사용자 세부 정보를 슬랙 메시지에 직접 삽입할 수 있습니다.

**Q: 비공개 채널에 보낸 메시지가 봇에서 온 것처럼 표시되나요?**

아니요. 비공개 채널 메시지는 봇이 아닌 슬랙 연동을 설정한 사용자로부터 보낸 것으로 표시됩니다.
