# 워크플로우의 폼, 설문, 퀴즈용 약관 동의 필터

## 개요

워크플로우에 새로운 기능인 약관 동의 필터(T&C Filter)가 추가되었습니다. 이 기능을 통해 사용자가 이용약관에 동의했는지 여부에 따라 대상자를 필터링할 수 있습니다. 이 개선사항은 사용자 타겟팅을 간소화하고 워크플로우 단계가 규정 준수 요구사항에 맞춰지도록 도와줍니다.

---

## 주요 기능 혜택

- **향상된 사용자 세분화**: 약관 동의 상태에 따라 사용자를 필터링할 수 있습니다.

- **강화된 규정 준수**: 특정 이용약관에 동의한 사용자만 워크플로우에서 진행하도록 보장합니다.

- **맞춤형 워크플로우 경로**: 약관에 동의한 사용자와 동의하지 않은 사용자를 위한 조건을 설정하여 타겟팅된 사용자 여정을 만들 수 있습니다.

---

## 사용 방법

### 1. 워크플로우에 약관 동의 필터 추가하기

- Workflows(워크플로우) 섹션으로 이동합니다.
![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155041376626/original/LWXOQf3ddmc6VlM0bSl8bM1FruET4qQNmQ.png?1739273801)

- 수정할 워크플로우를 선택하거나 새 워크플로우를 만듭니다.

- 워크플로우 트리거를 선택합니다 - 'Form Submitted(폼 제출)', 'Quiz Submitted(퀴즈 제출)' 또는 'Survey Submitted(설문 제출)'

- Add Filters(필터 추가)를 클릭합니다.

- T&C Filter(약관 동의 필터) 옵션을 선택합니다.
![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155041376678/original/D075y86pZckMYaRMxqUfgyw0JfHytn3izw.png?1739273826)

### 2. 필터 설정하기

- **조건 옵션**:

  약관에 동의한 사용자

- 약관에 동의하지 않은 사용자

- 워크플로우의 해당 단계에 필터를 적용합니다.

### 3. 저장 및 테스트

- 워크플로우 변경사항을 저장합니다.

- 테스트를 실행하여 필터가 예상대로 작동하는지 확인합니다.

---

## 활용 사례

- **프로모션 제공**: 업데이트된 약관에 동의하지 않은 사용자를 특정 제안에서 제외시킵니다.

- **기능 접근**: 약관 동의를 기반으로 프리미엄 기능에 대한 접근을 제한합니다.

- **규정 준수 관리**: 규제된 워크플로우에 규정을 준수하는 사용자만 참여하도록 보장합니다.

---

## 자주 묻는 질문

**1. 약관 동의 필터를 다른 필터와 함께 사용할 수 있나요?**
네, 약관 동의 필터는 더 고급 세분화를 위해 다른 워크플로우 필터와 함께 사용할 수 있습니다.

**2. 약관과 상호작용하지 않은 사용자는 어떻게 처리되나요?**
이들은 "동의하지 않음"으로 분류되어 워크플로우 결정에서 명확성을 보장합니다.
