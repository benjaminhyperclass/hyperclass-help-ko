# 워크플로우에서 GPT-5 사용하기

자동화 워크플로우 안에서 최첨단 AI의 힘을 직접 활용해 보세요. 워크플로우의 GPT-5는 OpenAI의 최신 언어 모델인 GPT-5, GPT-5 Mini, GPT-5.1, GPT-5 Nano를 Hyperclass 워크플로우 단계에 직접 연동하여 더 스마트하고, 빠르고, 비용 효율적인 작업 실행을 가능하게 합니다. 이 가이드에서는 이 기능이 무엇인지, 왜 중요한지, 비용은 얼마나 들고, 설정하는 정확한 방법을 설명합니다.

## **워크플로우의 GPT-5란 무엇인가요?**

워크플로우의 GPT-5는 OpenAI의 GPT-5 제품군을 Hyperclass에서 최신 통합한 기능으로, 모든 워크플로우에서 전례 없는 정확도와 속도로 텍스트를 생성하거나 변환할 수 있게 해줍니다. GPT 액션을 선택하고 GPT-5 변형을 선택하면, 이전 모델보다 더 긴 컨텍스트 윈도우, 더 풍부한 추론 능력, 더 낮은 지연 시간을 활용할 수 있습니다.

---

## **워크플로우 GPT-5의 주요 장점**

지능형 자동화가 실시간 엔터프라이즈급 언어 이해 능력을 갖게 됩니다.

- **속도:** 거의 실시간에 가까운 처리 속도로 고객 대기 시간을 줄입니다.

- **안정성:** GPT-5의 정교한 추론 능력 덕분에 일관되고 고품질의 결과를 얻을 수 있습니다.

- **유연성:** 3가지 모델 티어로 워크플로우를 다시 작성하지 않고도 성능과 예산의 균형을 맞출 수 있습니다.

- **간편성:** 기본 제공되는 클릭 설정 액션으로 연동과 유지보수가 간편합니다.

## GPT-5 모델, 가격, 토큰 계산법

사용량과 성능 요구사항에 맞는 티어를 선택하고 사용한 토큰만큼만 비용을 지불하세요.

| 모델 | 최적 용도 | 최대 컨텍스트 | 상대적 지연시간* | 입력 토큰 100만개당 비용 | 출력 토큰 100만개당 비용 |
|------|----------|-------------|-----------------|-------------------|-------------------|
| GPT-5 | 장문 추론, 문서 분석, 중요한 카피라이팅 | 256k | ◼︎◼︎◼︎◼︎ (느림) | $1.25 | $10.00 |
| GPT-5 Mini (기본값) | 일반 이메일 작성, 채팅 답변, 마케팅 카피 | 128k | ◼︎◼︎◼︎ (균형) | $0.25 | $2.00 |
| GPT-5 Nano | 대량 분류, 짧은 답변, 간단한 요약 | 64k | ◼︎◼︎ (가장 빠름) | $0.05 | $0.40 |
| GPT-5.1 | 다른 OpenAI 모델보다 나은 지시 따르기와 더 따뜻하고 대화적인 성격 | 128k | ◼︎◼︎◼︎ (균형) | $1.25 | $10.00 |

**중요 사항**: 가격은 OpenAI의 공개 요금과 동일하며 Hyperclass AI 지갑을 통해 청구됩니다. 에이전시 마크업과 클라이언트 재청구가 자동으로 적용됩니다.

### **토큰 계산법**

토큰 계산법을 마스터하면 AI 비용을 정확하게 예측하고 청구 관련 문제를 피할 수 있습니다. Hyperclass는 OpenAI와 동일하게 입력 토큰(프롬프트 + 컨텍스트)과 출력 토큰(모델 응답)을 계산하여 GPT-5 사용량을 청구합니다. 토큰 하나는 대략 영어 텍스트 4글자에 해당합니다. 요금은 백만 토큰의 천분의 일(0.001 M) 단위로 비례 계산됩니다.

#### 예시:
- 400단어 이메일(~2,000글자 ≈ 500토큰)을 GPT-5 Mini에 보낼 때 비용:

- **입력**: 500토큰 × $0.25 / 1M = $0.000125
- **출력**: 500토큰 × $2.00 / 1M = $0.001
- **총계**: 해당 액션당 $0.001125

## 워크플로우에서 GPT-5 설정하는 방법

적절한 설정으로 올바른 GPT-5 모델이 적용되고, 속도와 비용의 균형을 맞추며, 워크플로우를 운영에 적합하게 유지할 수 있습니다. 몇 번의 클릭으로 설정하는 방법을 알아보세요.

### **1단계:** GPT 액션 선택하기

워크플로우를 열고 '액션 추가' 아래에서 AI 단계를 추가하고 싶은 곳의 "+" 아이콘을 클릭한 후 "GPT powered by OpenAI"를 검색하세요. GPT 액션을 선택합니다.

![워크플로우에서 GPT 액션 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051630525/original/xqhHze4dKq1K8OUUxD6KbnJU_g9-AKFQMw.png?1755178385)

### **2단계:** GPT-5 Mini, GPT-5, GPT-5 Nano 또는 GPT-5.1 선택하기

모델 드롭다운에서 기본값인 GPT-5 Mini를 그대로 두거나 사용 사례에 맞게 GPT-5 / GPT-5 Nano를 선택하세요.

![GPT 모델 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051630743/original/Y2gNqJmyLdVsbkiusI1G3MlhrYXJ6lwIpg.png?1755178517)

**참고사항:** 프롬프트를 작성하거나 입력 데이터 필드를 매핑하세요. (모범 사례는 [AI 프롬프트 작성 기초](ai-prompting-101.md)를 참고하세요.) 단계를 저장하고 워크플로우를 발행하세요. 이제 자동화가 GPT-5 인텔리전스와 함께 실행됩니다.

## GPT-5 모델을 위한 프롬프트 작성 모범 사례

- **역할과 형식 제공:** 시스템 프롬프트를 모델의 역할과 정확한 출력 스키마로 시작하세요.

- **긴 입력에는 구분자 사용**: 큰 문서를 삼중 백틱( ``` )으로 감싸서 형식을 유지하세요.

- **가능하면 청크 단위로 나누기:** Nano의 경우 여러 페이지 콘텐츠를 더 작은 호출로 나누어 비용을 극도로 낮게 유지하세요.

- **예시 포함:** Few-shot 예시는 특히 Mini나 Nano에서 여전히 일관성을 향상시킵니다.

## **자주 묻는 질문**

**Q: 대용량 작업에 가장 비용 효율적인 GPT-5 모델은 무엇인가요?**
GPT-5 Nano가 토큰당 요금이 가장 낮아 속도가 덜 중요한 대용량 또는 백그라운드 작업에 이상적입니다.

**Q: 나중에 워크플로우를 다시 구축하지 않고도 모델을 변경할 수 있나요?**
네, GPT 액션을 편집하고 다른 모델을 선택한 후 저장하면 됩니다. 다른 단계는 조정할 필요가 없습니다.

**Q: 입력 및 출력 토큰은 어떻게 계산되나요?**
입력 토큰에는 프롬프트와 컨텍스트가 포함되고, 출력 토큰은 모델의 응답입니다. 둘 다 별도로 측정되고 가격이 책정됩니다.

**Q: GPT-5 요청에 사용량 제한이 있나요?**
Hyperclass는 OpenAI와 동일한 요청 제한을 적용합니다. 피크 트래픽 시간에는 많은 사용량으로 인해 요청이 대기열에 들어갈 수 있습니다.

**Q: GPT-5가 GPT-4 Turbo보다 더 긴 컨텍스트 윈도우를 지원하나요?**
네, GPT-5는 최대 256k 토큰의 컨텍스트를 지원하여 더 풍부한 대화와 다단계 추론을 가능하게 합니다.
