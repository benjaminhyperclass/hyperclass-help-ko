# Google 리뷰 신고 방법 및 신고 상태 확인하기

리뷰를 부적절하다고 신고하면 Google에서 해당 리뷰어에 대해 조치를 취할 수 있어요. 리뷰를 삭제하거나 향후 리뷰 작성을 금지하는 등의 조치가 가능합니다. 이는 보통 리뷰가 스팸이나 욕설 등 Google 가이드라인을 위반했을 때만 시행되요. 만약 리뷰가 가짜이거나 편향적이라고 생각하신다면, Google에 신고하여 추가 조사를 요청할 수 있습니다.

**참고사항:**

시스템에서 생성된 리뷰만 대화(Conversations) 탭에서 신고할 수 있어요. 시스템에서 생성되지 않은 리뷰는 [**여기를 클릭해서 Google에 직접 신고해주세요**](https://support.google.com/business/workflow/9945796).

## 대화 탭에서 Google 리뷰 신고하기

CRM에서 Google 리뷰를 신고하려면 대화(Conversations) 탭으로 이동해서 신고하려는 리뷰의 연락처를 찾은 후 '리뷰 신고'를 클릭하세요.

### 해당 GMB 리뷰와 연결된 **Google 이메일**과 비즈니스 계정을 확인하라는 메시지가 나타납니다.
![Google 계정 확인](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48253494223/original/fU3-oTDFmGwtrG884xKaY-c7Y9uadG2omA.gif?1664305457)

### 다음으로, 리뷰를 신고할지 또는 이미 신고한 리뷰의 상태를 확인할지 선택하세요:

![신고 옵션 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48098341984/original/l6Sg0IGlM3RP_Yb9hubRkFeT-9LgK8-LNg.png?1618435743)

새 리뷰 삭제를 신고하면 Google에서 최근 리뷰 목록을 보여줘요. 각 리뷰를 클릭하면 Google Maps에서 확인하거나 삭제 요청을 제출할 수 있습니다:

![리뷰 목록](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48098342358/original/dIDfNOu3Bv4rgAPuOh-SK-l34QdrFQiCWA.jpeg?1618435912)

## 신고한 리뷰의 상태 확인하기

이미 신고한 리뷰의 상태를 확인하려면 [여기를 클릭하세요](https://support.google.com/business/workflow/9945796).

### 이전에 신고한 리뷰의 상태를 확인하면 현재 처리 상태와 함께 해당 리뷰들을 볼 수 있어요.

![신고 상태 확인](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48098342472/original/aQcP06r9IEUyhL93MSYr_lvuBTTWGx5w1A.png?1618435962)

### 해당 리뷰를 선택하면 더 자세한 요약을 볼 수 있고, 삭제 요청이 거부된 경우 이의를 제기할 수도 있어요.

![이의 제기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48098342573/original/klzJKb29MgiKe_VpSBoM5TuR_jezRmz5LA.png?1618436001)

**참고사항:**

Google에서 보여주는 리뷰 수에는 제한이 있어요. [Joy Hawkins의 보고](https://localu.org/google-my-business-launches-tool-to-manage-review-removal-requests/)에 따르면 "이 기능은 소수의 비즈니스 목록을 보유한 Google 마이 비즈니스 계정에서만 작동하는 것 같습니다. 33개 목록이 있는 계정에서 시도했을 때 '관리하시는 비즈니스 프로필 수로 인해 이 프로세스를 사용할 수 없습니다'라는 메시지가 나타났어요. 저희 에이전시 계정도 목록이 너무 많아서 이 기능을 사용할 자격이 되지 않았습니다."
