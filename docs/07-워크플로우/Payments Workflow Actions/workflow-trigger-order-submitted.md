# 워크플로우 트리거 - 주문 제출됨

## 결제 링크의 주문 제출됨 트리거란?

주문 제출됨(Order Submitted) 트리거는 결제 링크(Payment Links)에서 사용할 수 있으며, 고객이 구매를 완료했을 때 구매 후 워크플로우를 자동화할 수 있게 해줍니다.

## 주요 장점:

- **자동화**: 확인 이메일 및 알림과 같은 구매 후 작업을 간소화합니다.

- **고객 경험 개선**: 결제 완료 후 시의적절한 후속 조치를 보장합니다.

- **데이터 관리**: 주문이 제출되었을 때 자동으로 기록을 업데이트합니다.

주문 제출됨 트리거 사용 방법:

- 자동화 설정(Automation Settings)으로 이동합니다.

- 결제 링크 워크플로우(Payment Links Workflow)를 선택합니다.

- 주문 제출됨 트리거(Order Submitted Trigger)를 활성화합니다.

- 다음과 같은 액션을 구성합니다:

확인 이메일 발송

- 고객 데이터 업데이트

- 관련 팀에 알림

- 워크플로우를 저장하고 활성화합니다.

## 정기 인보이스의 PayPal 지원

비즈니스에서 자동 결제가 활성화되지 않은 경우, 정기 인보이스(Recurring Invoices)의 결제 방법으로 PayPal을 제공할 수 있습니다.

주요 장점:

- **더 많은 결제 옵션**: 고객이 정기 인보이스에 대해 PayPal을 선택할 수 있습니다.

- **사용성 개선**: 구독 기반 또는 반복 결제 모델에 의존하는 비즈니스를 지원합니다.

- **고객 만족도 향상**: 결제 방법에 유연성을 제공합니다.

정기 인보이스에 PayPal을 활성화하는 방법:

- 시스템에서 PayPal 연동이 활성화되어 있는지 확인합니다.

- PayPal은 현재 자동 결제와 함께 작동하지 않으므로 자동 결제가 활성화된 정기 인보이스를 설정하지 않도록 주의하세요. PayPal을 활성화하기 위해 추가 단계를 수행할 필요는 없습니다.

이제 고객이 인보이스를 결제할 때 PayPal을 결제 방법으로 선택할 수 있습니다.

## **자주 묻는 질문**

**Q: "주문 제출됨" 트리거는 어떤 용도로 사용하나요?**

"주문 제출됨" 트리거는 워크플로우에서 고객이 구매를 완료했을 때 자동화된 액션을 시작하는 데 사용됩니다. 이는 일회성 상품 주문과 정기 인보이스 구독을 모두 포함합니다. 확인 이메일, 내부 알림, 업셀 플로우 등 구매 후 프로세스를 간소화하는 데 도움이 됩니다.

**Q: "주문 제출됨" 트리거가 PayPal과 함께 작동하나요?**

네, 이제 이 트리거는 정기 인보이스와 관련된 거래를 포함하여 PayPal 거래를 완전히 지원합니다. 사용자가 PayPal 결제를 완료하면 Stripe 거래와 마찬가지로 트리거가 활성화됩니다.

**Q: 일회성 구매와 정기 구매 모두에 트리거를 사용할 수 있나요?**

물론입니다. "주문 제출됨" 트리거는 일회성 상품 구매와 정기 인보이스 설정 모두와 함께 작동합니다. 워크플로우 구성에서 필터를 사용하여 두 가지를 구분할 수 있습니다.

**Q: 주문이 PayPal 또는 Stripe를 통해 결제되었는지 어떻게 식별할 수 있나요?**

워크플로우 트리거 내에서 "결제 모드(Payment Mode)" 필터를 사용하여 PayPal과 Stripe 결제를 구분할 수 있습니다. 이를 통해 사용된 결제 방법에 따라 맞춤형 자동화를 만들 수 있습니다.

**Q: 실패한 PayPal 결제에도 트리거가 활성화되나요, 아니면 성공한 결제에만 활성화되나요?**

"주문 제출됨" 트리거는 Stripe 또는 PayPal을 통한 **성공적인** 주문 제출 및 결제 확인 후에만 실행됩니다. 실패하거나 불완전한 결제는 트리거를 활성화하지 않습니다.

**Q: PayPal 정기 결제가 이루어진 후 자동 이메일을 보낼 수 있나요?**

네. 워크플로우에서 "주문 제출됨" 트리거와 "이메일 발송(Send Email)" 액션을 결합하여 PayPal 정기 결제가 제출된 후 자동으로 확인 또는 온보딩 이메일을 발송할 수 있습니다. 또는 결제(Payments) → 설정(Settings) 하위의 판매 영수증을 간단히 사용할 수도 있습니다.
