## 개요

"**Stripe 일회성 결제(Stripe One Time Charge)**" 액션을 사용하면 Stripe 계정 내의 특정 고객에게 고정 금액으로 일회성 결제를 실행할 수 있습니다. 이 액션은 정기 구독이나 분할 결제 없이 단순히 결제를 받을 때 유용합니다.

## 액션명

Stripe 일회성 결제

## 액션 설명

이 액션은 지정된 고객에게 설정한 금액으로 Stripe에서 일회성 결제를 실행합니다. 결제를 처리하려면 유효한 Stripe 고객 ID, 결제 금액, 설명, 통화가 필요합니다.

### 설정 방법

- **액션명:** "Stripe 일회성 결제"와 같이 알아보기 쉬운 액션명을 설정하세요.
- **Stripe 고객 ID:** 결제할 고객의 Stripe 고객 ID를 입력하거나 선택하세요.
- **설명:** 결제에 대한 간단한 설명을 입력하세요(선택사항).
- **금액:** 고객에게 청구할 금액을 입력하세요.
- **통화:** 거래에 적절한 통화를 선택하세요.

| 필드명 | 설명 | 필수 여부 |
|--------|------|-----------|
| Stripe 고객 ID | 결제할 Stripe 고객의 고유 식별자 | 예 |
| 설명 | 결제에 대한 설명(예: 추적이나 메모용) | 아니오 |
| 금액 | 고객에게 청구할 금액 | 예 |
| 통화 | 금액이 청구될 통화 | 예 |

## 예시

비즈니스에서 이 액션을 사용하여 일회성 컨설팅 수수료 $150(USD)와 같은 단발성 서비스에 대해 고객에게 요금을 청구할 수 있습니다. 결제가 실행되면 해당 금액이 Stripe를 통해 처리되고 지정된 고객의 계정과 연결됩니다.

## 권장 트리거

"Stripe 일회성 결제" 액션과 함께 사용할 수 있는 권장 트리거는 다음과 같습니다:

- **폼 제출(Form Submission)**: 고객이 구매 준비나 서비스 요청을 표시하는 폼을 제출했을 때 자동으로 결제를 실행합니다.
- **태그 추가됨(Tag Added)**: 고객 프로필에 특정 태그(예: "구매 준비됨")가 추가됐을 때 일회성 결제를 실행합니다.
- **예약 확정됨(Appointment Confirmed)**: 서비스 예약이 확정된 후 고객에게 자동으로 요금을 청구합니다.
- **주문 완료됨(Order Completed)**: 시스템 내에서 주문이 완료됨으로 표시됐을 때 결제를 실행합니다.
- **이메일 링크 클릭됨(Email Link Clicked)**: 고객이 이메일의 결제 관련 링크를 클릭했을 때 자동으로 결제를 실행합니다.

## 추가 참고사항

- 이 액션을 사용하기 전에 시스템에서 Stripe 연동을 설정했는지 확인하세요.
- 결제 실패를 방지하려면 Stripe 고객 ID가 유효한지 확인하세요.
- 이 액션은 일회성 결제에만 사용하세요. 정기 결제의 경우 구독과 같은 다른 Stripe 관련 액션을 고려해 보세요.
- 이 Stripe 일회성 결제 액션은 하위 계정이 자신의 연락처에게 요금을 청구하도록 설계된 것이 아니라, 에이전시가 하위 계정 관리자에게 요금을 청구하기 위한 것입니다.
