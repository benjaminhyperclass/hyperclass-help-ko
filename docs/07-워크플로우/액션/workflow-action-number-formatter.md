## 개요

**숫자 포맷터(Number Formatter)** 워크플로우 액션은 숫자 필드를 포맷하고 생성하는 데 사용됩니다. 텍스트를 숫자로 변환하기, 숫자 포맷하기, 전화번호 포맷하기, 통화 포맷하기, 랜덤 숫자 생성하기 등 여러 기능을 포함합니다.

## 액션명

숫자 포맷터(Number Formatter)

## 액션 세부사항

### 텍스트를 숫자로 변환

텍스트로 표현된 숫자를 숫자 형식으로 변환합니다. 예를 들어 "$12,345.67"을 12345.67로 변환합니다.

![텍스트를 숫자로 변환 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155032444844/original/gQMRG6tJfZ6hZNRMhH63SfFLy1uiA5aOHw.png?1725710234)

### 액션 세부사항:
| 필드 | 설명 | 필수 여부 |
|------|------|----------|
| 액션명(Action Name) | 사용자가 지정할 수 있는 액션의 이름입니다. | 예 |
| 액션 유형(Action Type) | 액션을 정의하며, "텍스트를 숫자로(Text to Number)"로 설정됩니다. | 예 |
| 필드 선택(Select Field) | 숫자의 텍스트 표현이 포함된 필드를 선택합니다. | 예 |
| 입력 소수점 표시(Input Decimal Mark) | 입력값에서 소수점이 어떻게 표시되는지 지정합니다(예: 마침표). | 예 |

### 워크플로우 구성 예시:

- **트리거:** 폼 제출(Form Submitted)
- **액션:** 결제 데이터를 정확하게 처리하기 위해 텍스트를 숫자로 변환
