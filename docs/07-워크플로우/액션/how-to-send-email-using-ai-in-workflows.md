# **이메일 액션 AI란?**

**이메일 액션 AI**는 Hyperclass 워크플로우에서 상황에 맞는 고품질 이메일 콘텐츠를 즉시 생성해주는 내장 AI 도우미예요. 직접 이메일 내용을 작성하는 대신, 목표를 설명하기만 하면 AI가 전문적인 이메일을 만들어줍니다. 이를 통해 카피라이팅 경험이 없어도 누구나 세련되고 효과적인 이메일 콘텐츠를 쉽게 만들 수 있어요.

## **이메일 액션 AI의 주요 장점**

**이메일 액션 AI는 이메일 제작 과정을 간소화하고 개선해줍니다. 주요 장점은 다음과 같아요:**

- 워크플로우 필요에 맞는 전문적이고 매력적인 이메일을 즉시 생성
- 수동 카피라이팅에 비해 상당한 시간 절약
- 브랜드 보이스에 맞춰 톤과 스타일을 커스터마이즈
- 오픈율을 높이는 제목을 자동 생성
- AI 지원 편집으로 오류와 일관성 문제 감소
- 비전문가도 효과적인 이메일 콘텐츠를 쉽게 제작 가능

## **이메일 액션 AI 사용 방법**

**이메일 액션 AI를 사용하면 글쓰기에 익숙하지 않은 사용자도 최소한의 노력으로 훌륭한 이메일을 만들 수 있어요. 워크플로우에서 이메일을 생성하려면 다음 단계를 따라하세요:**

- **선택**
  원하는 워크플로우에서 **Send Email(이메일 발송)** 액션을 선택하세요.

*스크린샷: "Send Email" 액션이 선택된 워크플로우 빌더*

- **AI 활성화**
  **Write with AI(AI로 작성하기)** 버튼을 클릭해서 AI 이메일 생성기를 열어주세요.

*스크린샷: "Write with AI" 버튼 위치*

- **설명 입력**
  이메일의 목표를 간단히 설명해주세요 (예: *"여름 세일 20% 할인을 제안하는 프로모션 이메일을 작성해줘"*).

*스크린샷: 설명 입력 필드*

- **톤 선택**
  필요하면 이메일 톤을 선택하세요 (기본값: Neutral). **Friendly(친근함)**, **Professional(전문적)**, **Casual(캐주얼)** 등의 옵션이 있어요.

*스크린샷: 톤 선택 드롭다운*

- **생성**
  **Generate(생성)** 버튼을 클릭하고 AI가 이메일을 만들 때까지 기다려주세요.

- **제목 적용**
  **"Use Subject(제목 사용)"** 체크박스를 사용해서 AI가 생성한 제목을 적용할지 결정하세요.

- **재생성**
  이메일을 조정하고 싶으면 **Re-generate(재생성)** 버튼을 사용하고 세부 조정을 위한 추가 프롬프트를 제공하세요.

- **적용**
  만족스러우면 **Use Email(이메일 사용)** 버튼을 클릭해서 콘텐츠를 워크플로우에 직접 삽입하세요.

## **이메일 액션 AI의 핵심 기능**

**이메일 액션 AI는 더 스마트하고 유연한 경험을 제공하는 다양한 유용한 도구를 포함하고 있어요:**

### **AI 기반 작성**

설명을 바탕으로 전문적이고 설득력 있는 이메일 콘텐츠를 즉시 생성해서 수동 카피라이팅이 필요 없어져요.

### **톤 선택**

브랜드나 캠페인 목표에 맞게 이메일의 스타일과 보이스를 세밀하게 조정할 수 있어요. Friendly(친근함), Professional(전문적), Casual(캐주얼), Neutral(중립적) 등의 옵션이 있답니다.

### **제목 연동**

AI가 생성한 제목을 사용하거나 무시할 수 있는 옵션을 제공해서, 수신자의 받은편지함에서 이메일이 어떻게 보일지 완전히 제어할 수 있어요.

### **재생성 옵션**

후속 프롬프트를 통해 반복적인 편집이 가능해서 AI가 생성한 콘텐츠를 개선하거나 다시 표현할 수 있어요. 원하는 이메일을 정확히 만들 수 있도록 도와줍니다.

## **자주 묻는 질문**

**Q: 이메일 액션 AI는 모든 플랜에서 사용 가능한가요?**

이메일 액션 AI는 무료로 제공됩니다.

**Q: 거래용 이메일에도 이메일 액션 AI를 사용할 수 있나요?**

네! 워크플로우에 따라 마케팅 이메일과 거래용 이메일 모두에 사용할 수 있어요.

**Q: AI가 생성한 이메일 콘텐츠를 편집할 수 있나요?**

물론이죠! 생성된 이메일을 워크플로우에 삽입하기 전에 자유롭게 편집할 수 있어요.

**Q: AI가 자동으로 이메일을 개인화(이름, 날짜 등)해주나요?**

AI는 일반적인 콘텐츠를 생성해요. 필요에 따라 개인화 태그(예: {{first_name}})를 직접 추가해주셔야 합니다.

**Q: 어떤 톤을 선택할 수 있나요?**

**Neutral(중립적)**, **Friendly(친근함)**, **Professional(전문적)**, **Casual(캐주얼)** 등의 옵션이 있어요.

**Q: 글자 수 제한이 있나요?**

엄격한 글자 수 제한은 없지만, 간결한 설명이 보통 더 좋은 결과를 만들어내요.
