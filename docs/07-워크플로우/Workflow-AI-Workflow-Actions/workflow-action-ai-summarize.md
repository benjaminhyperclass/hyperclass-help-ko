# 워크플로우 액션 - AI 요약

AI 요약 액션은 AI를 사용하여 긴 콘텐츠를 자동으로 간결한 요약으로 압축합니다. 원하는 요약 길이와 입력 텍스트만 지정하면, 핵심 정보를 보존하면서 축약된 버전을 생성해드립니다.

## 작동 방식

이 액션은 AI를 통해 입력 텍스트를 처리하여:

- 핵심 포인트와 주요 아이디어를 식별합니다
- 중복 정보를 제거합니다
- 지정한 길이에 맞게 내용을 재구성합니다
- 맥락과 가독성을 유지합니다

## 설정 방법

### 1단계: 액션 추가

워크플로우 액션 메뉴에서 "AI Summarize"를 검색하여 워크플로우에 추가하세요.

![AI 요약 액션 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155062014376/original/dmV8LdoUk5g9t-uP0mar_rPnbWPwZcU4lg.png?1767706588)

### 2단계: 설정 구성

- **액션 이름**: 액션에 설명적인 이름을 지정하세요 (예: "고객 리뷰 요약")
- **최대 길이 입력**: 원하는 요약 길이를 지정하세요
  - 단어 수 (예: "50 words")
  - 문자 수 (예: "280 characters") 
  - 문장 수 (예: "3 sentences")
  - 문단 수 (예: "1 paragraph")
- **입력 텍스트**: 요약할 텍스트를 제공하세요
  - 정적 값: 텍스트를 직접 입력
  - 동적 변수: 커스텀 값 선택기로 변수 사용
- **저장**: 저장을 클릭하여 설정을 완료하세요

![AI 요약 설정 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155062014625/original/2AgugMjNLt4ZNFhA7Ttgybs37pDb8xUxhg.jpeg?1767706650)

설정 완료 후, 요약된 텍스트는 커스텀 변수로 사용할 수 있습니다:

![요약 결과 변수](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155062014690/original/iZTWZbHyc9A3MZV_lgtfJ6EqAutG6aZFfA.png?1767706692)

이 변수는 다음과 같은 후속 워크플로우 액션에서 사용할 수 있습니다:

- SMS나 이메일로 요약본 발송
- 커스텀 필드에 저장
- 알림에 표시

## 활용 사례

### SMS 친화적 예약 노트

- **트리거**: 예약 완료
- **입력**: 폼의 상세한 예약 노트
- **최대 길이**: 160자
- **액션**: 요약이 포함된 확인 SMS 발송

### 파이프라인 단계 변경 요약

- **트리거**: 기회 단계 변경
- **입력**: 모든 기회 노트와 활동
- **최대 길이**: 1문단
- **액션**: 계정 관리자에게 요약 이메일 발송

### 리뷰 응답 요약

- **트리거**: 리뷰 수신 (Google/Facebook)
- **입력**: 전체 고객 리뷰 텍스트
- **최대 길이**: 50단어
- **액션**: 팀 Slack 채널에 게시

### 회의록 압축

- **입력**: 전체 회의 녹취록
- **최대 길이**: 1문단
- **출력**: 이해관계자를 위한 요약 보고서

## 요금

- **비용**: 실행당 1센트 ($0.01)
- 프리미엄 액션 패키지에 포함

## 자주 묻는 질문

**Q: 여러 메시지나 연결된 텍스트를 요약할 수 있나요?**

A: 네, 요약하기 전에 커스텀 값을 사용하여 여러 텍스트 소스를 결합할 수 있습니다.

**Q: 여러 요약 액션을 연결할 수 있나요?**

A: 네, 콘텐츠를 점진적으로 요약할 수 있습니다. 먼저 500단어로, 그 다음 두 번째 액션에서 100단어로 요약하는 식으로 가능합니다.

**Q: 요약의 정확도는 어느 정도인가요?**

A: AI는 중복된 세부사항을 제거하면서 핵심 정보에 대해 높은 정확도를 유지합니다. 사용 전에 항상 샘플 데이터로 테스트해보시기 바랍니다.

**Q: 요약에서 특정 정보를 제외할 수 있나요?**

A: 현재 AI가 중요한 내용을 판단합니다. 특정 항목을 제외하려면 먼저 다른 워크플로우 액션으로 텍스트를 전처리하는 것을 고려해보세요.
