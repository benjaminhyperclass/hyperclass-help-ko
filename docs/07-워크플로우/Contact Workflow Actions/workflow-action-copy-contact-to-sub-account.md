## 개요

"**연락처 복사(Copy Contact)**" 액션을 사용하면 시스템 내의 한 계정에서 다른 계정으로 연락처를 복제할 수 있습니다. 이는 여러 하위 계정을 관리하면서 연락처 정보를 원활하게 공유해야 하는 조직에 유용합니다. 이 액션은 프리미엄 기능으로, 실행할 때마다 추가 요금이 발생한다는 점을 알아두세요.

## 액션 이름

**연락처 복사(Copy Contact)**

## 액션 설명

"연락처 복사" 액션은 선택된 연락처를 다른 지정된 계정으로 복사합니다. 연결된 태그와 커스텀 필드를 복사하는 옵션과, 대상 계정에 해당 연락처가 이미 존재하는 경우 업데이트하는 옵션이 포함되어 있습니다. 이 액션은 여러 계정 간에 일관된 데이터를 유지하고, 선택된 하위 계정에서 연락처가 정확하게 미러링되도록 도와줍니다.

## 액션 세부사항

![연락처 복사 액션 설정 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155032061758/original/z6mqxDsGGVuimRZ7U_xZ7bBlPJyryUQcnA.png?1725271781)

| 필드 | 설명 | 필수 여부 |
|------|------|----------|
| 계정 선택(Select an Account) | 연락처를 복사할 대상 계정을 선택합니다. | 예 |
| 하위 계정에 태그 추가(Add Tags to Sub-Account) | 선택적으로 하위 계정의 복사된 연락처에 추가할 태그를 선택합니다. | 아니오 |
| 태그 복사(Copy Tags) | 원본 연락처의 모든 태그를 복사된 연락처로 복사하려면 이 옵션을 켜세요. | 아니오 |
| 커스텀 필드 복사(Copy Custom Fields) | 원본 연락처의 커스텀 필드를 복사본에 포함하려면 이 옵션을 켜세요. | 아니오 |
| 이미 존재하는 연락처 업데이트(Update Contact if it Already Exists) | 활성화하면 대상 계정에 중복이 존재할 경우 연락처를 업데이트합니다. | 아니오 |

**설정 방법:**

- **계정 선택:** 연락처를 복사하고 싶은 대상 계정을 선택하세요. 이는 필수 단계입니다.
- **하위 계정에 태그 추가:** 필요한 경우, 하위 계정의 연락처에 추가해야 할 태그를 선택하세요.
- **태그 복사:** 원본 연락처의 기존 태그를 하위 계정의 새 연락처에 복사하려면 이 옵션을 켜세요.
- **커스텀 필드 복사:** 커스텀 필드 데이터를 새 연락처에 복사하려면 이 옵션을 켜세요.
- **이미 존재하는 연락처 업데이트:** 중복을 만드는 대신 대상 계정의 기존 연락처 정보를 업데이트하려면 이 옵션을 사용하세요.

**이 액션과 함께 사용할 수 있는 트리거 예시 (제한적이지 않음):**

- **연락처 추가됨(Contact Added):** 메인 계정에 새 연락처가 추가될 때마다 자동으로 다른 계정으로 연락처를 복사합니다.
- **태그 적용됨(Tag Applied):** 연락처에 특정 태그가 적용되면 복사 액션을 트리거하여 하위 계정으로 연락처를 복제합니다.
- **연락처 업데이트됨(Contact Updated):** 이 트리거를 사용하여 업데이트된 연락처 정보를 다른 계정으로 복사하여 모든 계정이 최신 연락처 데이터를 갖도록 합니다.

**파일 전송 제한사항**

연락처 복사 액션은 하위 계정 간 파일이나 문서 전송을 **지원하지 않습니다**. 연락처가 복사될 때 파일은 대상 하위 계정으로 **독립적인 자산으로 마이그레이션되지 않습니다**. 경우에 따라 복사된 연락처가 원본 연락처나 소스 레코드에 의존하는 파일 참조를 여전히 표시할 수 있습니다. 원본 계정에서 원본 연락처가 삭제되면 해당 관련 파일에 액세스할 수 없게 될 수 있습니다.

해당 파일에 중요한 정보가 포함되어 있다면 원본 계정에서 원본 연락처를 삭제하지 **마세요**.

## 예시

여러 지역 사무소를 운영하며 각각 자체 계정을 갖고 있다고 가정해보세요. 메인 계정에 새 리드가 추가되고 "VIP" 태그가 지정되면, 이 연락처를 자동으로 지역 계정들에 복사하고 싶습니다.

**워크플로우 설정 예시:**

- **트리거:** 연락처 추가됨(Contact Added)
  - **필터:** 태그 = VIP
- **액션:** 연락처 복사(Copy Contact)
  - **계정 선택:** 지역 사무소 계정
  - **하위 계정에 태그 추가:** VIP
  - **태그 복사:** 활성화
  - **커스텀 필드 복사:** 활성화
  - **이미 존재하는 연락처 업데이트:** 활성화
