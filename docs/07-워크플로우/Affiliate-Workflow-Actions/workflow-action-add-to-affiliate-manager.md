## 개요

제휴 관리에 추가(Add To Affiliate Manager) 액션은 연락처를 제휴 관리(Affiliate Manager)의 제휴 목록에 추가하며, 기본적으로 활성 상태로 설정됩니다. 이 액션은 기존 연락처나 고객을 제휴 파트너로 등록할 때 유용합니다.

## 액션 이름

Add To Affiliate Manager (제휴 관리에 추가)

## 액션 설명

제휴 관리에 추가 액션은 기존 연락처를 하위 계정의 제휴 관리자에서 제휴 파트너로 자동 전환하는 기능입니다. 새로 등록되는 제휴 파트너의 상태는 기본적으로 활성으로 설정됩니다.

## 액션 세부사항

**단계별 가이드**

- **액션 유형 선택:** 사용 가능한 액션 목록에서 "Add To Affiliate Manager(제휴 관리에 추가)"를 선택합니다.
- **액션 이름 입력:** "연락처를 제휴 파트너로 전환"과 같은 설명적인 액션 이름을 입력합니다.

![제휴 관리에 추가 액션 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155031799303/original/eupZ-f0OzJTEmPbY8YEgFDYLZzWNcZOh5w.png?1724837548)

## 예시

연락처에 태그가 추가될 때 기존 연락처를 제휴 파트너로 전환

**시나리오:** 비즈니스에서 게시물에 답변하거나 제휴 네트워크 참여에 관심을 보인 고객들을 제휴 파트너로 모집하려고 합니다. 사용자는 이런 연락처들을 분류하기 위해 태그를 추가하고, 이를 활성 제휴 파트너로 전환합니다.

**액션 설정:**

액션: Add To Affiliate Manager (제휴 관리에 추가)

이름: 연락처를 제휴 파트너로 전환

**워크플로우 트리거:**

- Contact Changed (연락처 변경): 선택된 필터가 변경되거나 추가될 때 연락처가 워크플로우에 진입합니다.
- 필터: "Affiliate" 태그 추가

**워크플로우 액션:**

- Add To Affiliate Manager (제휴 관리에 추가)

**결과:**
이렇게 설정하면 연락처에 태그가 추가될 때마다 해당 연락처가 제휴 파트너로 전환됩니다.
