# 액션 - 제휴 업데이트 → 워크플로우 액션 - 제휴 업데이트

## 개요

제휴 업데이트(Update Affiliate) 액션은 제휴 파트너의 상태를 활성 또는 비활성으로 업데이트하도록 설계되었습니다.

## 액션 이름

제휴 업데이트(Update Affiliate)

## 액션 설명

제휴 업데이트 액션은 제휴 파트너의 상태를 활성 또는 비활성으로 업데이트하도록 설계되었습니다.

## 액션 상세

단계별 가이드

- 액션 타입 선택: 사용 가능한 액션 목록에서 "Update Affiliate(제휴 업데이트)"를 선택하세요.
- 액션 이름 설정: "제휴를 비활성으로 업데이트"와 같이 액션을 설명하는 이름을 입력하세요.
- 상태 선택: 상태를 활성 또는 비활성 중 어느 것으로 변경할지 선택하세요.

## 예시

탈퇴 양식을 작성한 제휴 파트너를 비활성으로 표시

시나리오: 한 비즈니스가 활성 제휴 파트너 목록을 관리하고자 합니다. 제휴 캠페인과 네트워크에서 탈퇴하려는 현재 제휴 파트너들이 작성하는 양식이 있습니다.

액션 설정:

액션: 제휴 업데이트(Update Affiliate)

이름: 제휴를 비활성으로 업데이트

워크플로우 트리거:

- 폼 제출됨(Form Submitted): 기존 제휴 파트너가 선택한 양식을 제출했을 때가 워크플로우의 진입점이 됩니다.
- 필터: 폼이 "제휴 탈퇴 양식"인 경우

워크플로우 액션:

- 제휴 업데이트(Update Affiliate)
- 제휴를 비활성으로 표시 선택

결과: 이렇게 설정하면 기존 제휴 파트너가 "제휴 탈퇴 양식"을 작성할 때마다 해당 제휴 파트너의 프로필이 비활성으로 표시되어 더 이상 커미션과 매출을 창출할 수 없게 됩니다.
