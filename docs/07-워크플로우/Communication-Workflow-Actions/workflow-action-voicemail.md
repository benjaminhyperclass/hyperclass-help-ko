## 개요

음성메시지 액션을 사용하면 미리 녹음된 음성메시지를 연락처에게 직접 보낼 수 있어요. 실시간 통화 없이도 후속 조치, 알림 또는 특정 메시지를 전달하는 데 유용합니다.

## 액션 이름

**음성메시지 (Voicemail)**

## 액션 설명

미리 녹음된 음성메시지 파일을 선택한 연락처에게 보냅니다.

## 액션 세부사항

![음성메시지 액션 설정 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155032066704/original/S5yOIgMvdxybPNVPONEqVuXi7sUy1IQMiw.png?1725274416)

| 필드 | 설명 | 필수 |
|------|------|------|
| Action Name | 액션의 이름으로, "Voicemail"로 설정됩니다. | 예 |
| Drop Your File | 보내고자 하는 미리 녹음된 음성메시지 파일을 업로드합니다. 파일 형식은 음성메시지 재생과 호환되는 형식이어야 합니다. | 예 |

**설정 방법:**

- **액션 이름 설정:** "Action Name" 아래에 "Voicemail"이라고 표시되어 수행할 액션을 식별할 수 있도록 해요.
- **파일 업로드:** "Upload file" 버튼을 클릭하여 로컬 시스템에서 미리 녹음된 음성메시지를 선택하세요. 업로드가 완료되면 메시지를 재생하여 올바른 파일인지 확인할 수 있어요.
- **저장:** 설정을 완료한 후 워크플로우를 저장하여 변경사항을 적용하세요.

**활용 가능한 트리거 (일부 예시):**

음성메시지 액션을 효과적으로 사용하려면 다음과 같은 트리거와 결합하여 사용하는 것을 고려해보세요:

- **예약 노쇼:** 연락처가 예정된 예약에 나타나지 않았을 때 자동으로 음성메시지를 보내 재예약을 유도해요.
- **후속 조치 알림:** 연락처가 일정 시간 내에 이전 커뮤니케이션에 응답하지 않았을 때 음성메시지를 트리거해요.

## 예시

- **트리거:** "예약 상태 변경(Appointment Status Changes)"이 "No-Show"로 변경됨
- **액션:** "음성메시지(Voicemail)"
- **업로드된 파일:** "Reschedule.mp3" (놓친 예약을 재예약하도록 권하는 미리 녹음된 메시지)

이 예시에서 워크플로우는 연락처가 예약에 나타나지 않을 경우 이를 감지하고 자동으로 음성메시지를 보내 재예약을 권하게 됩니다. 수동 후속 조치 없이도 고객 참여도를 향상시킬 수 있어요.
