## 개요

**수동 SMS(Manual SMS)** 액션은 사용자가 연락처에게 SMS를 수동으로 전송할 수 있도록 할 일(태스크)을 생성해주는 기능입니다. 개인화된 메시지나 사람의 판단이 필요한 문자를 보낼 때 유용하죠. 생성된 할 일은 대화(Conversations) > 수동 작업(Manual Actions) 탭에서 확인할 수 있어요.

## 액션 이름

**수동 SMS(Manual SMS)**

## 액션 설명

이 액션은 특정 연락처에게 SMS 메시지를 보내는 수동 할 일을 생성합니다. 사용자가 직접 개입하여 메시지를 최종 확인하고 전송해야 해요. 사용자는 템플릿을 선택하고, 커스텀 값을 추가하며, 트리거 링크를 포함시키고, 전송 전에 메시지를 테스트할 수 있습니다. 또한 기본 연락처 담당자가 아닌 특정 사용자에게 할 일을 배정할 수도 있어요.

## 액션 세부사항

| 필드 | 설명 | 필수 여부 |
|------|------|----------|
| 액션 이름 | 워크플로우에서 액션을 식별하기 위한 이름 | 예 |
| 템플릿 | 미리 정의된 템플릿에서 SMS 템플릿 선택 | 아니오 |
| 메시지 | 전송할 메시지 내용. 커스텀 값과 트리거 링크를 지원합니다 | 예 |
| 커스텀 값 | 메시지 개인화에 사용할 수 있는 커스텀 값. 예: 연락처 이름, 예약 정보 등 | 아니오 |
| 첨부파일 추가 | URL을 통해 SMS 메시지에 첨부파일을 추가하는 옵션 | 아니오 |
| 테스트 전화번호 | 테스트 SMS를 보낼 전화번호. 국가 번호 포함 | 아니오 |
| 배정 대상 | 수동 SMS 할 일을 배정할 특정 사용자 선택. 고급 설정에서 찾을 수 있음. 비워두면 기본적으로 연락처 담당자에게 배정됨. 선택한 사용자가 연락처 접근 권한이 없으면 연락처 담당자에게 배정되거나, 담당자가 없으면 미배정 상태가 됨 | 아니오 |

### **설정 방법**

- **템플릿 선택**: 해당하는 경우 미리 정의된 SMS 템플릿을 선택하세요. 이렇게 하면 메시지 필드에 템플릿 내용이 자동으로 입력됩니다.
- **메시지 입력**: 보내고 싶은 메시지를 작성하세요. 커스텀 값 드롭다운을 사용해서 동적 데이터용 플레이스홀더를 삽입할 수 있습니다 (예: 연락처 이름, 예약 정보).
- **첨부파일 추가**: 필요한 경우 URL을 입력하여 파일을 추가하세요.
- **SMS 테스트**: 전화번호를 입력하여 SMS가 어떻게 표시되는지 테스트해보세요.
- **배정 대상 (고급 설정)**: 선택적으로 특정 사용자를 선택하여 할 일을 배정할 수 있습니다. 이렇게 하면 기본 연락처 담당자 배정을 무시합니다.

### **커스텀 값과 트리거 링크**

수동 SMS 액션은 메시지 본문에서 커스텀 값뿐만 아니라 트리거 링크도 지원합니다.

예를 들어 캘린더 커뮤니티 가입이나 커스텀 폼 작성 등의 기능이 있어요. 이러한 링크는 클릭할 수 있으며 연락처를 특정 작업으로 안내할 수 있습니다.

### **배정 대상 동작 방식**

고급 설정의 배정 대상 필드를 통해 기본 연락처 담당자가 아닌 특정 사용자에게 수동 SMS 할 일을 배정할 수 있습니다. 이는 연락처를 누가 담당하든 상관없이 특정 팀원이 특정 할 일을 처리하게 하고 싶을 때 유용해요.

**배정 로직:**

- 사용자가 선택되고 연락처 접근 권한이 있으면, 해당 사용자에게 할 일이 배정됩니다.
- 사용자가 선택되었지만 연락처 접근 권한이 없으면, 연락처 담당자에게 할 일이 배정됩니다.
- 사용자가 선택되지 않으면, 연락처 담당자에게 기본 배정됩니다.
- 선택된 사용자도 연락처 담당자도 사용할 수 없으면, 할 일은 미배정으로 표시됩니다.

## 예시

- **트리거**: 리드가 웹사이트의 폼을 작성했을 때
- **액션**: 배정된 사용자가 후속 SMS를 보낼 수 있도록 수동 SMS 할 일 생성
- **메시지**: "안녕하세요 {{Contact Name}}님, 폼을 작성해주셔서 감사합니다. 여기를 클릭하여 상담을 예약하세요: [캘린더 링크]"
- **테스트 전화번호**: +11234567890 (SMS 테스트용)

이 예시에서는 리드가 폼을 제출하면, 예약 링크가 포함된 개인화된 SMS를 보내는 수동 할 일이 생성됩니다. 메시지는 커스텀 값을 사용해서 텍스트를 개인화하고, 일정 예약용 트리거 링크를 포함하고 있어요.
