# 워크플로우 트리거 - 회사 생성됨

## 개요

회사 생성됨(Company Created) 트리거는 CRM에 새로운 회사(Company) 레코드가 추가될 때마다 워크플로우를 시작합니다. 수동으로 생성하거나, 가져오기(Import), 또는 폼, 웹훅, 연동을 통해 자동으로 추가되는 모든 경우에 동작합니다.

## 트리거 이름

Company Created (회사 생성됨)

## 트리거 설명

이 트리거는 다음과 같은 경우에 실행됩니다:

- 계정에 새로운 회사 레코드가 생성될 때
- 지원되는 모든 방법(수동 입력, 폼 제출, 가져오기, API, 웹훅)을 통해 생성될 때

이 트리거를 사용하면 새로운 회사를 자동으로 온보딩하거나, 연락처를 연결하거나, 다른 시스템으로 데이터를 동기화하는 워크플로우를 자동으로 시작할 수 있습니다.

## 설정 방법

![워크플로우 트리거 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055609900/original/Wbj5hmmod2uxr-2peomD6ly_OMWJym8AfQ.png?1760010716)

**1. 트리거 추가**
- 워크플로우 편집기를 열고 시작 트리거로 Company Created(회사 생성됨)를 선택합니다
- 이 트리거는 워크플로우 유형이 Company(회사) 기반일 때만 사용할 수 있습니다

**2. 필터 추가 (선택사항)**
특정 유형의 회사 생성만 대상으로 하려면 필터를 추가할 수 있습니다. 예시:
- Industry(업종)에 "부동산"이 포함됨
- Company Name(회사명)에 "주식회사"가 포함됨
- Owner(소유자)가 비어있지 않음

**3. 트리거 저장**
- 설정이 완료되면 Save Trigger(트리거 저장)를 클릭합니다
- 이제 필터 조건(있는 경우)을 만족하는 새로운 회사 레코드가 생성될 때마다 워크플로우가 자동으로 시작됩니다

## 예시

### **새로운 회사 자동 온보딩**

**목표**: 새로운 회사가 추가될 때마다 워크플로우를 시작하여 환영 이메일을 발송

**설정**
- 트리거: Company Created(회사 생성됨)
- 액션:
  - Send Email(이메일 발송) – 새로운 회사를 CRM에 환영하는 이메일

**실행 흐름**
1. CRM에 새로운 회사가 생성됩니다 (폼 또는 가져오기를 통해)
2. 워크플로우가 자동으로 시작됩니다
3. 시스템에서 환영 이메일을 발송합니다
