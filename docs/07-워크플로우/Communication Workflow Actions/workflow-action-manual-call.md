# 워크플로우 액션 - 수동 전화 (Manual Call)

## 개요

수동 전화(Manual Call) 액션을 사용하면 사용자가 연락처에게 직접 전화를 걸 수 있도록 전화 할 일(태스크)을 생성할 수 있습니다. 전화에 수동 개입이 필요할 때 유용한 기능입니다. 이 액션은 적시 후속 조치, 리드 연락, 예정된 전화를 수동으로 처리하는 데 도움이 됩니다. 생성된 할 일은 대화(Conversations) > 수동 작업(Manual Actions) 탭에 나타납니다.

## 액션 이름

Manual Call (수동 전화)

## 액션 설명

수동 전화 액션은 대화(Conversations) > 수동 작업(Manual Actions) 탭에 연락처에게 수동으로 전화를 걸기 위한 할 일을 생성합니다. 이 할 일은 사용자가 직접 연락처에게 전화를 걸어야 합니다. 수동 작업 탭에서 해당 할 일을 삭제하면, 연락처가 워크플로우의 다음 액션으로 이동합니다. 고급 설정(Advanced Settings)의 할당 대상(Assign To) 필드를 사용하여 기본 연락처 담당자 대신 특정 사용자에게 할 일을 할당할 수도 있습니다.

## 액션 세부사항

### **단계별 가이드**

- **액션 유형 선택:**
  워크플로우로 이동하여 사용 가능한 액션 목록에서 "Manual Call"을 선택하세요.

- **액션 이름 지정:**
  "후속 전화"와 같은 설명적인 액션 이름을 입력하세요.

- **할당 대상 (고급 설정):**
  선택적으로 할 일을 할당할 특정 사용자를 선택하세요. 이렇게 하면 기본 연락처 담당자 할당이 재정의됩니다.

- **대화(Conversations) > 수동 작업(Manual Actions) 탭으로 이동**
  할 일을 확인하세요.

- 완료 후 할 일을 삭제하세요.

### **할당 대상 동작 방식**

고급 설정의 할당 대상(Assign To) 필드를 사용하면 기본 연락처 담당자 대신 특정 사용자에게 수동 전화 할 일을 할당할 수 있습니다. 연락처 소유자와 관계없이 특정 팀원이 특정 전화를 처리하도록 하고 싶을 때 유용합니다.

**할당 로직:**

- 사용자가 선택되고 연락처 접근 권한이 있는 경우, 해당 사용자에게 할 일이 할당됩니다.
- 사용자가 선택되었지만 연락처 접근 권한이 없는 경우, 연락처 담당자로 할당이 넘어갑니다.
- 사용자가 선택되지 않은 경우, 기본적으로 연락처 담당자에게 할당됩니다.
- 선택된 사용자와 연락처 담당자 모두 사용할 수 없는 경우, 할 일은 미할당(Unassigned)으로 표시됩니다.

**실행 로그:**
- 액션 로그에는 "Manual Call added to the manual queue"라는 메시지와 함께 수동 전화 액션이 표시됩니다. 이는 전화 추적에 도움이 됩니다.

**중요:** 
수동 전화 액션은 수동 작업(Manual Actions) 탭에서 할 일을 명시적으로 삭제할 때만 워크플로우가 진행됩니다. 연락처(Contacts) 페이지나 다른 영역에서 단순히 연락처에게 전화를 거는 것만으로는 할 일이 완료되지 않고 워크플로우가 다음 단계로 진행되지 않습니다.

## 예시

**시나리오:** 한 비즈니스에서 모든 새로운 리드의 첫 문의 후 24시간 이내에 영업 담당자가 후속 조치를 취하도록 하고자 합니다. 전화가 완료되면 할 일을 삭제하여 워크플로우의 다음 단계로 이동합니다.

**워크플로우 설정:**

- **트리거:** 새 연락처 생성됨
- **액션:** 수동 전화
  - 이름: "리드 후속 전화"

**결과:** 영업팀 구성원이 새 리드에게 전화를 걸어야 한다는 할 일을 받게 되어 적시 후속 조치를 보장하며, 이는 전환율을 크게 높일 수 있습니다.

## 자주 묻는 질문

**Q: 수동 전화 할 일을 수동으로 삭제할 수 있나요?**

네! 전화 완료 후 **대화(Conversations) > 수동 작업(Manual Actions)** 탭에서 할 일을 수동으로 삭제해야 합니다. 이 단계를 통해 시스템이 연락처를 워크플로우의 다음 액션으로 이동시킵니다.

**Q: 수동 전화 할 일을 연락처 담당자가 아닌 다른 사람에게 할당할 수 있나요?**

네! 고급 설정의 할당 대상(Assign To) 필드를 사용하여 특정 사용자를 선택하세요. 해당 사용자가 연락처 접근 권한을 가지고 있다면 그 사용자에게 할 일이 할당됩니다. 그렇지 않으면 연락처 담당자로 넘어가거나 미할당으로 표시됩니다.

**Q: 할당된 사용자가 연락처에 대한 접근 권한이 없다면 어떻게 되나요?**

선택된 사용자가 연락처 접근 권한이 없는 경우, 할 일은 자동으로 연락처 담당자에게 넘어갑니다. 연락처 담당자도 사용할 수 없다면 할 일은 미할당(Unassigned)으로 표시됩니다.
