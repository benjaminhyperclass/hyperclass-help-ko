# 워크플로우 액션 - 인스타그램 인터랙티브 메신저

워크플로우의 **인스타그램 인터랙티브 메신저** 액션은 버튼과 빠른 답변 같은 인터랙티브 요소로 인스타그램 DM 대화를 자동화하는 데 도움을 줍니다.

이 가이드에서는 이 액션이 무엇을 하는지, 어디서 찾을 수 있는지, 그리고 구성 요소를 효과적으로 설정하는 방법을 알려드립니다.

실제 사용 예시와 모범 사례도 다루어 성과를 이끌어내는 매력적인 인스타그램 상호작용을 디자인할 수 있도록 해드리겠습니다.

# **인스타그램 인터랙티브 메신저란 무엇인가**

**인스타그램 인터랙티브 메신저** 워크플로우 액션을 사용하면 인스타그램의 댓글이나 다이렉트 메시지에 개인화된 DM을 보내어 자동으로 응답할 수 있습니다. 수동으로 답변하는 대신, 인터랙티브하고 매력적으로 느껴지는 자동화된 대화를 설정할 수 있습니다. 이 액션은 빠르게 리드를 확보하거나, 자료를 공유하거나, 공개 댓글 스레드를 개인 메시지로 이어가고 싶을 때 특히 유용합니다.

핵심적으로 이 기능은 자동화와 유연성을 결합합니다. **왜** 사용할지(잠재 고객을 육성하든, 오퍼를 보내든, 체크리스트를 전달하든), 그리고 메시지 템플릿, 첨부파일, 버튼 또는 빠른 답변을 통해 경험을 **어떻게** 구성할지 결정할 수 있습니다. **언제** 사용할지(예: 프로모션 게시물 댓글 후나 사용자가 DM을 시작할 때)를 이해하면, 인스타그램 참여를 더 넓은 마케팅 퍼널로 매끄럽게 안내하는 워크플로우를 디자인할 수 있습니다.

---

## 인스타그램 인터랙티브 메신저의 주요 이점

- **대규모 자동화된 참여**: 수동 작업 없이 댓글이나 DM에 즉시 답변하여, 팔로워가 시기적절하고 일관된 응답을 받을 수 있도록 보장합니다.

- **인스타그램에서 바로 전환 유도**: 인터랙티브 버튼이나 빠른 답변을 사용해 사용자를 웹사이트 방문, 자료 다운로드, 통화 예약 같은 행동으로 안내합니다.

- **개인화된 사용자 경험**: 템플릿, 커스텀 값, 트리거 링크를 활용해 각 연락처에게 관련성 있고 개인적으로 느껴지는 맞춤 메시지를 보냅니다.

- **효율적인 리드 확보**: 게시물의 공개 상호작용에서 세부 정보를 수집하고 추가 육성이 가능한 개인 대화로 잠재고객을 매끄럽게 이동시킵니다.

- **팀 시간 절약**: 워크플로우가 일반적인 상호작용을 처리하도록 하여 반복 작업을 줄이고, 팀이 더 가치 있는 활동에 집중할 수 있게 합니다.

## **인스타그램 인터랙티브 메신저 워크플로우 액션 설정하기**

### **워크플로우 빌더 열기**

**Automation(자동화) → Workflows(워크플로우) → Create Workflow(워크플로우 생성)** (또는 기존 워크플로우 열기)로 이동하여 **워크플로우 빌더** 캔버스에 접근합니다.

![워크플로우 빌더 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155052931027/original/HfHfuSACaZ_4rV-uFbbThDnnG7YdBSqZ8g.png?1756823930)

### **인스타그램 트리거 추가**

**+ Add Trigger(트리거 추가)**를 클릭하고 다음 중 하나를 선택합니다:

- **Instagram Comment on Post(인스타그램 게시물 댓글)** – 팔로워가 인스타그램 게시물에 댓글을 달 때
- **Customer Reply (Instagram)(고객 응답 (인스타그램))** – 연락처가 DM을 보내거나 답변할 때

**인스타그램 인터랙티브 메신저** 액션을 사용하려면 이것이 필수입니다. 다른 유형의 트리거를 사용하면 이 액션을 선택할 수 없습니다.

### 인스타그램 인터랙티브 메신저 액션 배치

워크플로우 캔버스에서 메시지를 보낼 위치의 **+**를 클릭한 후, **Actions(액션) → Communication(커뮤니케이션) → Instagram Interactive Messenger(인스타그램 인터랙티브 메신저)**를 선택합니다.

![액션 선택 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155052938808/original/FaXS0Zb9fCvBdDiWEmi-NtnbnUwtA1EIMg.png?1756828000)

### **액션 이름 명확하게 설정**

**Action Name(액션 이름)**에서 이 단계에 설명적인 라벨을 부여하세요(예: "IG DM – 리드 마그넷 프롬프트 발송"). 복잡한 워크플로우에서 쉽게 찾을 수 있습니다.

### **답변 유형 선택**

**Reply Type(답변 유형)**에서 다음을 선택합니다:

- 트리거가 새 댓글이고 대화를 비공개로 전환하려면 **Reply to Comment via DM(댓글에 DM으로 답변)**
- 들어오는 DM에 응답하는 경우 **Reply to DM(DM에 답변)**

![답변 유형 선택 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155052938859/original/bFKUSQGiVmA2giMHiXbnLDcb2OO3odyyiA.png?1756828036)

### (선택사항) 템플릿 가져오기

재사용 가능한 문구가 있다면 **Marketing(마케팅) → Snippets(스니펫)**에서 **Templates(템플릿)** (스니펫)를 선택하세요. 이는 캠페인 전반에서 톤과 규정 준수를 일관되게 유지하는 데 도움이 됩니다.

![템플릿 선택 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155052938872/original/ulaSb-dAJug9e-PwGa6QLhL5yqBwQ7I2ow.png?1756828060)

### **메시지 작성**

**Message(메시지) → Body(본문)**에서 DM을 작성합니다. 개인화를 위해 **Custom Values(커스텀 값)** (**Settings(설정) → Custom Values(커스텀 값)**에서)를 사용하고, 클릭 추적이나 특정 URL로 사용자를 안내하기 위해 **Trigger Links(트리거 링크)** (**Marketing(마케팅) → Trigger Links(트리거 링크)**에서)를 사용하세요.

**팁:** 짧은 훅으로 시작한 후, 단일하고 명확한 행동 유도 문구를 넣으세요. DM 형식에 맞게 메시지를 간결하게 유지하세요.

#### **첨부파일 추가 (필요시)**

**Add Attachment(첨부파일 추가)**를 사용해 파일을 업로드하거나 **Add file through URL(URL로 파일 추가)**로 호스팅된 자료를 첨부하세요. 첨부파일은 체크리스트나 가이드에 유용합니다.

**중요:** 같은 메시지에 첨부파일이 포함되면 빠른 답변을 추가할 수 없습니다. 빠른 답변이 필요하다면 여기서 첨부파일을 생략하거나 나중 액션에서 보내세요.

![첨부파일 추가 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155052939161/original/3yyju_-M717Dc6ppCD0Sbi89kauy2_5Aaw.png?1756828264)

### **인터랙티브 컨트롤 추가: 버튼 또는 빠른 답변**

이 메시지에 대해 다음 중 **하나**를 선택하세요:

- **버튼**: 버튼은 사용자가 상호작용한 후에도 대화에서 계속 보이는 지속적인 액션을 제공합니다. 웹사이트 열기, 전화번호로 전화걸기, 워크플로우의 다른 부분으로 분기하기 같은 특정 결과를 유도하는 데 이상적입니다. 사용자가 나중에 다시 돌아올 수 있는 지속적인 옵션을 제공하고 싶을 때 버튼을 사용하세요.

- **빠른 답변**: 빠른 답변은 사용자가 선택하면 사라지는 짧은 원탭 응답을 위해 설계되었습니다. 체크리스트, 가격 정보 또는 데모를 원하는지 묻는 것처럼 실시간으로 대화를 안내하는 데 최적입니다. 빠른 답변은 사용자의 인스타그램 DM 경험을 어수선하게 만들지 않으면서 의도를 파악하는 데 도움을 주어 흐름을 집중적이고 가볍게 유지합니다.

**팁**: 지속적이고 유용한 액션에는 버튼을 사용하세요(예: "통화 예약", "사이트 방문"). 타겟된 질문에는 빠른 답변을 사용하세요("예/아니오", "가격/데모/문서").

![인터랙티브 컨트롤 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155052939270/original/ZtoGtsiKRzIhUGNuKknnG3GUP18tZM9rcw.png?1756828355)

### 분기 생성 및 설정

**Button(버튼)** (Perform Action(액션 수행))이나 **Quick Reply(빠른 답변)**를 추가하면 시스템이 자동으로 **워크플로우 분기**를 생성합니다. 각 분기는 사용자가 특정 옵션을 클릭할 때 취하게 될 경로를 나타냅니다. 각 분기 안에서 후속 메시지 전송, 태그 적용, 파이프라인 단계 업데이트 또는 다른 워크플로우 트리거 같은 추가 액션을 추가할 수 있습니다. 이 분기 로직이 인스타그램 인터랙티브 메신저 액션을 역동적으로 만들어, 각 사용자 선택에 맞춤화된 경험을 구축할 수 있게 해줍니다.

#### **대기 시간 및 기본 경로 설정**

모든 메신저 액션에는 **Wait Step(대기 단계)**가 필요합니다. 이는 사용자가 버튼을 클릭하거나 빠른 답변을 선택하기를 기다리는 동안 시스템이 얼마나 일시정지해야 하는지 결정합니다. 기본적으로 **1분**으로 설정되어 있지만, 캠페인 필요에 맞게 편집할 수 있습니다. 예를 들어, 빠른 오퍼는 몇 분, 더 느린 응답이 예상되면 몇 시간으로 설정할 수 있습니다.

사용자가 설정된 시간 내에 상호작용하지 않으면 자동으로 **Default Timeout branch(기본 타임아웃 분기)**로 라우팅됩니다. 이 분기는 미리 알림 전송, "응답 없음" 같은 태그 적용 또는 대화를 우아하게 종료하는 데 유용합니다. 신중한 기본 경로를 디자인하면 인터랙티브 요소와 참여하지 않더라도 리드를 잃지 않게 됩니다.

![대기 시간 설정 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155052939318/original/_ZlqE4_ZvrfjYmscIBQ77-X5FqfM5oCixQ.png?1756828389)

### **저장, 발행 및 테스트**

**Save(저장)**과 **Publish(발행)**를 클릭하여 워크플로우를 활성화합니다. 정확한 시나리오(키워드로 댓글 달기 또는 테스트 DM 보내기)를 트리거해서 테스트하세요. 다음을 확인하세요:

- 올바른 **Reply Type(답변 유형)**으로 메시지가 전송되는지
- 버튼/빠른 답변이 예상대로 작동하는지
- 분기 액션이 적절하게 실행되는지

### **모니터링 및 개선**

응답, 클릭(**Trigger Links(트리거 링크)** 통해), 다운스트림 전환을 추적하세요. 성과를 바탕으로 메시지 문구, 버튼 라벨, 분기 로직을 개선하세요. 작은 조정(예: 선택지 줄이기, 더 명확한 라벨)이 종종 더 나은 결과를 가져옵니다.

## **자주 묻는 질문**

**Q: 워크플로우에서 인스타그램 인터랙티브 메신저 액션이 회색으로 나타나는 이유는 무엇인가요?**

이 액션은 워크플로우가 **Instagram Comment on Post(인스타그램 게시물 댓글)** 또는 **Customer Reply (Instagram)(고객 응답 (인스타그램))** 같은 인스타그램 전용 트리거를 사용할 때만 사용할 수 있습니다. 이러한 트리거 중 하나가 없으면 이 액션을 선택할 수 없습니다.

**Q: 버튼과 빠른 답변의 차이는 무엇인가요?**

버튼은 대화에서 계속 보이는 지속적인 액션으로, 웹사이트 열기, 전화번호로 전화걸기, 다른 워크플로우 경로로 분기하기 같은 작업을 수행할 수 있습니다. 빠른 답변은 선택되면 사라지는 임시 옵션으로, 대화를 집중적이고 깔끔하게 유지합니다.

**Q: 이 액션에서 대기 단계를 설정해야 하는 이유는 무엇인가요?**

대기 단계는 사용자가 버튼이나 빠른 답변과 상호작용할 시간을 주기 위해 시스템이 일시정지하도록 보장합니다. 설정된 시간 내에 응답이 없으면 연락처가 자동으로 **기본 타임아웃 분기**로 이동하여, 리드가 방치되지 않도록 합니다.

**Q: 알아야 할 제한사항이 있나요?**

네. 인스타그램은 Meta의 **24시간 메시징 규칙**을 따르므로, 마지막 사용자 상호작용으로부터 24시간 이내에만 자동화된 DM을 보낼 수 있습니다. 또한 첨부파일이 있는 메시지에는 빠른 답변을 포함할 수 없습니다.
