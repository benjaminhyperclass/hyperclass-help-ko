# 커스텀 값을 사용하여 MMS에 파일 첨부하는 방법

커스텀 값(Custom Values)은 매우 강력하고 다양하게 활용할 수 있는 기능입니다. 이 가이드에서는 워크플로우 내에서 커스텀 값을 사용하여 MMS 메시지를 보내는 방법을 알아보겠습니다.

커스텀 값은 스냅샷과 자주 사용하는 워크플로우를 효과적으로 활용하는 데 매우 유용합니다. 빠르고 간단한 설정으로 자동화를 더욱 확장 가능하고 흥미롭게 만들 수 있습니다.

# SMS 메시지에 커스텀 값 URL을 첨부하는 방법

## **1단계:** 커스텀 값 URL 설정하기

`Settings(설정) → Custom Values(커스텀 값) → + Add Custom Value(커스텀 값 추가) → MMS URL 입력`으로 이동합니다.

![커스텀 값 URL 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48240187218/original/Cq5T3LD3wiScR2JwiibCexevGo9I4SOlLQ.gif?1658417559)

## **2단계:** MMS를 전송할 워크플로우로 이동하기

워크플로우에 들어가서 커스텀 값을 첨부 파일로 추가하세요. 해당 값을 복사하여 붙여넣기 해야 합니다.

![워크플로우에서 MMS 첨부](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48240188567/original/PDE7_BYhWRlwXQ1KbFOTDcncrlAhAotBGQ.gif?1658417802)

**참고사항:**

통신사별로 MMS 크기에 대한 엄격한 제한이 있습니다. 이러한 제한은 통신사에서 설정한 것입니다.

통신사별 정확한 크기 요구사항에 대한 자세한 정보는 [SMS 전송 시 최대 MMS 크기는 얼마인가요?](#sms-전송-시-최대-mms-크기는-얼마인가요)를 확인해주세요.

# **문제 해결**

### SMS 전송 시 최대 MMS 크기는 얼마인가요?

| 통신사 | Long code MMS 첨부 크기 | Toll-Free MMS 첨부 크기 | Shortcode MMS 첨부 크기 |
|--------|-------------------------|-------------------------|-------------------------|
| AT&T | 0.6 MB | 0.6 MB | 0.6 MB |
| Sprint | 1.4 MB | 0.6 MB | 1.4 MB |
| T-mobile | 1.5 MB | 0.6 MB | 1.0 MB |
| Verizon | 0.675 MB | 0.6 MB | 1.2 MB |

### SMS를 통한 MMS 전송 시 지원되는 파일 형식은 무엇인가요?

- jpeg
- png  
- gif

### 메시지 전송 시 커스텀 값이 작동하지 않아요

다음 사항을 다시 확인해주세요:

- 커스텀 값 URL에 공백이 없는지 확인하세요. 테스트용 퍼널에 추가하고 페이지를 미리보기로 테스트해볼 수 있습니다. MMS가 나타나면 커스텀 값이 작동하는 것이니 다음 단계로 진행하세요.
- MMS URL이 해당 통신사의 크기 요구사항을 충족하는지 확인하세요(위 표 참조).
