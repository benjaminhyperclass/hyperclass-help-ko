# 워크플로우 트리거 - 연락처 참여 점수

## 개요

**연락처 참여 점수(Contact Engagement Score)** 트리거는 연락처의 참여 점수와 기타 조건을 기반으로 워크플로우에 연락처를 추가하는 기능입니다. 고객의 상호작용과 전체 참여도에 따라 자동화 작업을 실행할 수 있도록 도와줍니다.

## 트리거 이름

연락처 참여 점수(Contact Engagement Score)

## 트리거 설명

이 트리거는 연락처의 **참여 점수(Engagement Score)**와 기타 연락처 속성을 기반으로 워크플로우에 연락처를 필터링해서 추가하는 기능입니다. 후속 조치, 높은 참여도를 보이는 연락처 분류, 특정 조건에 따른 마케팅 활동 조정 등의 작업을 자동화하는 데 유용합니다.

## 트리거 세부사항

| 필드 이름 | 설명 | 필수 여부 |
|-----------|------|-----------|
| 점수(Score) | 연락처의 참여 점수입니다. 참여 조건을 기반으로 필터가 적용됩니다. | 예 |
| 비즈니스 니치(Business Niche) | 연락처와 연관된 비즈니스 카테고리입니다. 예: 교육, 의료 등 | 아니오 |

![참여 점수 트리거 설정 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033065696/original/RevefXKprkOBwX7qFdxChkT3dIs5xV6SFg.png?1726648831)

| 연산자 | 설명 |
|--------|------|
| ~와 같음(Equals to) | 점수나 필드 값이 입력된 값과 같을 때 트리거됩니다. |
| ~보다 큼(Greater than) | 점수나 필드 값이 입력된 값보다 클 때 트리거됩니다. |
| ~보다 크거나 같음(Greater than or Equal to) | 점수나 필드 값이 입력된 값보다 크거나 같을 때 트리거됩니다. |
| ~보다 작음(Less than) | 점수나 필드 값이 입력된 값보다 작을 때 트리거됩니다. |
| ~보다 작거나 같음(Less than or Equal to) | 점수나 필드 값이 입력된 값보다 작거나 같을 때 트리거됩니다. |
| 비어있음(Is empty) | 필드 값이 비어있을 때 트리거됩니다. |
| 비어있지 않음(Is not empty) | 필드 값이 비어있지 않을 때 트리거됩니다. |
| 구문 포함(Contains Phrase) | 필드에 입력된 구문이 포함되어 있을 때 트리거됩니다. |
| 구문 포함하지 않음(Does not contain Phrase) | 필드에 입력된 구문이 포함되어 있지 않을 때 트리거됩니다. |

## 설정 방법

- **1단계:** 워크플로우 빌더의 트리거 옵션에서 "연락처 참여 점수(Contact Engagement Score)" 트리거를 선택합니다.
- **2단계:** 트리거에 "높은 참여도 연락처"와 같은 이름을 지정합니다.
- **3단계:** 참여 점수나 기타 조건을 정의하는 필터를 설정합니다.
  - 예를 들어, 필터 드롭다운에서 **점수(Score)**를 선택하고 **~보다 큼(Greater than)**을 선택합니다. 그리고 **195**와 같은 값을 입력합니다.
  - **비즈니스 니치(Business Niche)**와 같은 추가 필터를 설정하고 **구문 포함(Contains Phrase)**을 선택한 후, 니치명(예: **교육(Education)**)을 입력할 수 있습니다.
- **4단계:** 트리거 설정을 저장하고 필요에 따라 워크플로우에 추가 액션이나 단계를 추가합니다.

## 예시

이 시나리오에서는 **연락처 참여 점수(Contact Engagement Score)** 트리거가 점수가 **195보다 크고** **교육(Education)** 니치에 속한 연락처를 포착하도록 설정되어 있습니다. 연락처가 두 조건을 모두 충족하면 후속 이메일 캠페인을 위한 워크플로우에 추가됩니다.
