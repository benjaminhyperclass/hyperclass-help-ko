# 워크플로우 트리거 - 트리거 링크 클릭됨

## 개요

**트리거 링크 클릭됨(Trigger Link Clicked)** 트리거는 연락처가 특정 트리거 링크를 클릭할 때마다 워크플로우를 시작합니다. 이 기능은 링크 참여도를 추적하고 미리 설정된 링크와의 사용자 상호작용을 기반으로 후속 작업을 자동화하는 데 이상적입니다.

## 트리거 이름

트리거 링크 클릭됨(Trigger Link Clicked)

## 트리거 설명

이 트리거는 연락처가 지정된 트리거 링크를 클릭할 때 워크플로우를 시작합니다. 트리거 링크는 이메일, SMS 또는 기타 커뮤니케이션 채널에서 사용할 수 있습니다. 워크플로우는 지정된 링크를 클릭한 연락처에 대해서만 활성화되므로, 타겟 참여를 위한 강력한 도구가 됩니다.

## 설정 방법

- **워크플로우 트리거 선택**: 사용 가능한 트리거 목록에서 **트리거 링크 클릭됨(Trigger Link Clicked)**을 선택하세요.

- **워크플로우 트리거 이름**: "프로모 링크 클릭됨" 또는 "설문 링크 참여"와 같은 설명적인 트리거 이름을 입력하세요.

- **필터**: 필터를 사용하여 워크플로우를 활성화할 트리거 링크를 지정하세요. 드롭다운 메뉴에서 정확한 트리거 링크를 선택하여 워크플로우 활성화 범위를 좁힐 수 있습니다.

| 값 | 설명 | 필수 |
|---|---|---|
| 트리거 링크 | 연락처가 클릭했을 때 워크플로우를 실행할 특정 링크를 선택합니다. | 예 |

## 예시

이메일을 통해 발송된 프로모션 링크를 클릭한 연락처에게 후속 이메일을 보내는 워크플로우를 만들고 싶다고 가정해보겠습니다.

- **워크플로우 트리거 이름**: "프로모 링크 참여 후속조치"
- **필터**:
  - **트리거 링크**: "봄 세일 프로모" (이 특정 링크 클릭만 워크플로우를 실행)

워크플로우 단계:

- **트리거**: 트리거 링크 클릭됨 (위와 같이 설정)
- **액션**: 이메일 발송 - 연락처의 관심에 감사하며 특별 할인을 제공하는 후속 이메일 발송
- **액션**: 연락처 업데이트 - 향후 프로모션 캠페인을 위해 해당 사용자를 "프로모 관심" 태그로 분류
