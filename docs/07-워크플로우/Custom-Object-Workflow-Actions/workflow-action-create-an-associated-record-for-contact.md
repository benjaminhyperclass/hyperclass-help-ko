# 워크플로우 액션 - 연락처의 연관 레코드 생성

## 개요

연락처 기반 워크플로우의 '연관 레코드 생성(Create an Associated Record)' 액션을 사용하면 새로운 커스텀 객체 레코드를 생성하고 연락처와 연결할 수 있습니다. 이 기능은 부동산, 계약서 또는 연락처와 관련된 기타 커스텀 객체 등을 관리할 때 유용합니다.

## 액션 이름

Create Associated Record for Contact (연락처의 연관 레코드 생성)

## 액션 설명

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155041052361/original/3dCs8bbLAWbHlPfWuwNbASA2jKu2OBKKfg.png?1738754630)

이 액션을 사용하면 워크플로우에서 직접 새로운 커스텀 객체 레코드를 생성하고 연락처와 연결할 수 있습니다. 생성 가능한 객체는 하위 계정에서 구성된 커스텀 객체에 따라 달라집니다.

## 액션 세부사항

- 이 액션은 하위 계정에서 커스텀 객체를 생성하고, 해당 객체 중 하나 이상과 연락처 사이에 연관 라벨을 생성한 후에만 사용할 수 있습니다.
- 새 객체 레코드에 대해 여러 필드를 정의할 수 있습니다.
- 사용자는 **연관 라벨**을 지정하여 연락처와 커스텀 객체 간의 관계를 분류할 수 있습니다.
- 새로 생성된 레코드는 선택한 라벨을 사용하여 자동으로 연락처에 연결됩니다.
- 커스텀 값과 커스텀 필드의 병합 필드를 사용하여 객체에 정보를 추가할 수도 있습니다.

## 예시

태양광 회사에서 다음과 같은 설치 관련 중요한 날짜를 추적해야 한다고 가정해보겠습니다:

- **현장 조사 날짜**
- **설치 날짜**

**영업 기회(Opportunity)**가 "성사(Won)"로 표시되면, 시스템이 이러한 날짜를 추적하기 위해 "설치(Install)" 커스텀 객체 레코드를 자동으로 생성해야 합니다.

**예시 설정:**

- **생성할 객체 레코드** → Install
- **필드:**
  - **현장 조사 날짜** → 오늘 날짜로 자동 입력
  - **설치 날짜** → 일정에 따라 자동 입력
- **연관 라벨** → Installation Scheduled

트리거가 실행되면 워크플로우가 **설치(Install)** 레코드를 생성하고 연락처에 연결하여 원활한 추적이 가능해집니다.
