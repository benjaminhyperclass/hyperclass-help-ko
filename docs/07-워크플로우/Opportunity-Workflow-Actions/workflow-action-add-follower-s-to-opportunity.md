## 개요

기회에 팔로워 추가하기(Add Follower(s) to Opportunity) 액션을 사용하면 기존 기회에 한 명 이상의 사용자를 팔로워로 지정할 수 있어요. 이렇게 하면 해당 사용자들이 기회의 변경사항에 대한 업데이트와 정보에 접근할 수 있게 됩니다. 일반적으로 이 액션은 기회 찾기(Find Opportunity) 단계 다음에 오거나, 기회 관련 트리거에 의해 실행되므로 특정 기회가 컨텍스트에 있어야 해요.

## 액션 이름

Add Follower(s) to Opportunity

## 액션 설명

트리거되면 기회에 팔로워 추가하기 액션은 선택한 각 사용자를 해당 기회의 팔로워 목록에 추가해요. 만약 사용자가 이미 그 기회를 팔로우하고 있다면, 시스템이 중복을 방지하여 같은 사용자가 두 번 추가되지 않도록 합니다. 기회를 찾을 수 없거나 트리거되지 않으면 이 액션은 건너뛰어져요.

### 예외 상황

- **컨텍스트에 기회가 없는 경우**: 워크플로우가 기회에 의해 트리거되지 않았고 앞서 기회 찾기 단계가 없다면, 이 액션은 아무것도 하지 않아요.
- **여러 기회**: 기회 찾기 단계가 여러 레코드와 일치할 수 있더라도, 오직 하나(기회 찾기 액션에서 정의된 가장 빠른 것 또는 가장 늦은 것)만 업데이트됩니다.
- **기존 팔로워**: 시스템이 이미 팔로우하고 있거나 기회에 배정된 사용자의 추가를 방지하므로 중복이 생성되지 않아요.

## 예시

시나리오: 고가치 기회가 생성되거나 업데이트되는 즉시 영업 매니저와 제품 전문가 모두에게 정보를 제공하고 싶어요.

- **트리거**: 기회가 생성되거나 업데이트되고 특정 조건을 만족함 (예: 가치 > 10,000)
- **액션**: 기회 찾기(Find Opportunity)
  - 필터:
    - 기회 금액이 10,000보다 큼
    - 상태가 "Open"
- **액션**: 기회에 팔로워 추가하기(Add Follower(s) to Opportunity)
  - 액션 이름: "주요 이해관계자 알림"
  - 팔로워: "영업 매니저," "제품 전문가"

기회 찾기 단계에서 조건에 맞는 기회를 찾으면, "영업 매니저"와 "제품 전문가"가 팔로워로 추가되어 (이미 팔로워가 아닌 경우) 기회의 세부사항과 업데이트에 접근할 수 있게 됩니다.
