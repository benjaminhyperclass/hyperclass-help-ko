# 커뮤니티 그룹의 게임화/리더보드 트리거 및 액션

**목표**: 이 가이드는 커뮤니티 그룹 멤버십을 기반으로 워크플로우 트리거와 액션을 설정하는 방법을 안내합니다. 게임화 포인트 지급과 리더보드 레벨 변경을 중심으로 다루며, 필터 설정과 사용자가 필수 커뮤니티 그룹에 속하지 않은 경우에도 워크플로우가 올바르게 작동하도록 하는 방법을 알아보세요.

### 커뮤니티 그룹 멤버 리더보드 레벨 변경 (트리거)

- **트리거**: 사용자의 리더보드 레벨이 변경될 때 활성화됩니다.
- **필터**: 트리거가 적용될 그룹과 레벨을 지정하기 위해 필터를 설정해야 합니다.
- **사용 사례**: 사용자가 새 레벨에 도달했을 때 축하 이메일을 자동으로 보내거나 포인트를 지급합니다.

사용 방법:

- 워크플로우 설정 페이지로 이동합니다.
- 트리거 선택: 사용 가능한 트리거 목록에서 "User Group Gamification Level Changed"를 선택합니다.
- 필터 설정: 특정 그룹이나 레벨 등의 조건을 지정하기 위해 필터를 사용합니다. 이렇게 하면 해당 조건이 충족될 때만 트리거가 활성화됩니다.

### 커뮤니티 그룹 리더보드 포인트 지급 - 액션

**1. 게임화 포인트 할당**

- **액션**: 특정 트리거에 대해 리더보드 포인트를 지급하는 기능입니다.
- **지급할 포인트**: 워크플로우 설정의 드롭다운을 통해 설정 가능합니다. 포인트 수와 적용할 그룹을 지정할 수 있습니다.
- **사용 사례**: 원하는 행동을 완료한 사용자에게 포인트를 지급하여 참여를 유도하고, 리더보드 순위를 올릴 수 있도록 도와줍니다.

**2. 사용 방법:**

- "Grant community group leaderboard points" 액션 선택: 그룹 내에서 특정 행동을 완료한 사용자에게 포인트를 지급하려면 이 액션을 선택합니다.
- 포인트 및 그룹 설정:
  - 포인트: 지급할 포인트 수를 설정합니다.
  - 그룹: 드롭다운을 사용하여 적용할 그룹을 선택합니다. 선택한 그룹의 사용자만 포인트를 받을 수 있습니다.
- **참고: 멤버가 해당 그룹에 속하지 않으면 워크플로우 실행이 실패합니다**
