# 워크플로우 액션 - AI 의도 감지

AI 의도 감지(AI Intent Detection) 액션은 텍스트를 분석하여 자동으로 감정을 판단하고, POSITIVE(긍정), NEGATIVE(부정), NONE(중립)으로 분류합니다. 이 프리미엄 액션은 기존 IF/Else 조건의 레거시 의도 감지를 대체하며, 고객 감정에 따른 자동화 응답의 정확도를 높이고 구현을 더욱 쉽게 해줍니다.

## AI 의도 감지란?

의도 감지 액션은 AI 기반 워크플로우 도구로, 텍스트 입력을 자동으로 분석하여 내재된 감정이나 의도를 판단합니다. 고객 메시지, 설문 응답, 폼 제출 내용 등 모든 텍스트를 처리하여 다음 세 가지 분기로 분류합니다:

- **POSITIVE(긍정)**: 긍정적 감정, 만족, 동의를 표현하는 텍스트
- **NEGATIVE(부정)**: 부정적 감정, 불만, 반대를 전달하는 텍스트  
- **NONE(중립)**: 중립적 감정이거나 시스템이 확신을 가지고 분류할 수 없는 텍스트

이 액션은 자동으로 세 개의 워크플로우 분기를 생성하여, 감지된 감정에 따라 서로 다른 자동화 경로를 구축할 수 있게 해줍니다.

## 주요 장점

- **자동 감정 분석**: 수동 검토 없이 즉시 고객 응답을 분류
- **향상된 정확도**: 기존 IF/Else 의도 감지 방법보다 더 안정적
- **간소화된 워크플로우 설계**: 명확한 세 분기로 복잡한 조건부 로직 제거
- **실시간 응답**: 고객 감정에 따라 즉시 적절한 액션 실행
- **향상된 고객 경험**: 고객 감정에 맞춤화된 개인화 응답 제공
- **확장 가능한 자동화**: 사람 개입 없이 무제한 메시지 처리

## AI 의도 감지 설정 방법

### 1단계: AI 의도 감지 액션 추가

워크플로우 빌더로 이동하여 액션 메뉴에서 "AI Intent Detection"을 검색합니다. 텍스트 감정 분석을 원하는 지점에 워크플로우에 추가하세요.

![AI 의도 감지 액션 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155062013506/original/o_RXz8dxn-FiBPvkv_xSI0KmQQXuHfr8Jw.png?1767706286)

### 2단계: 액션 설정

- **액션 이름**: 액션에 설명적인 이름을 부여하세요 (예: "피드백 감정 분석")
- **입력 텍스트**: 분석할 텍스트를 지정합니다. 두 가지 옵션이 있습니다:
  - **고정 값**: 직접 고정 텍스트 입력
  - **동적 변수**: 커스텀 값 선택기를 사용하여 변수 선택:
    - `{{message.body}}` (수신 메시지용)
    - 분류할 텍스트가 저장된 기타 변수

![액션 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155062013560/original/rW97oR8OAidk5wPExVEoPfk23qgeUbG1hA.png?1767706331)

### 3단계: 분기 액션 설정

저장 후 액션이 자동으로 세 개의 분기를 생성합니다:

![분기 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155062013821/original/hxCmz80mTWBT3drPXjJJXnQTHxtoiqMzBQ.png?1767706390)

**1. POSITIVE 분기**: 긍정적 응답을 위한 액션 설정

- 감사 메시지 발송
- 앱스토어 리뷰 요청
- 로열티 보상 제공
- "만족한 고객" 태그 추가

**2. NEGATIVE 분기**: 부정적 응답을 위한 액션 설정

- 지원팀 에스컬레이션 실행
- 할인 코드나 쿠폰 발송
- 후속 전화 예약
- "주의 필요" 태그 추가

**3. NONE 분기**: 중립/불분명한 응답을 위한 액션 설정

- 명확화 요청 발송
- 수동 검토로 라우팅
- 기본 후속 조치 시퀀스 적용

## 예시 활용 사례

### 1. 고객 피드백 관리

**시나리오**: 설문을 통해 피드백 수집 후

- **POSITIVE 응답**: "서비스가 정말 훌륭하네요!"
  - **액션**: 앱스토어 리뷰 요청 링크 발송
- **NEGATIVE 응답**: "경험이 정말 실망스러웠어요"
  - **액션**: 다음 구매 시 20% 할인 쿠폰 발송
- **NONE 응답**: "괜찮아요" 또는 "해당 없음"
  - **액션**: 더 자세한 내용을 위한 후속 질문 발송

### 2. 구매 후 후속 조치

**시나리오**: 배송 확인 후

- **POSITIVE**: 제품 리뷰 및 고객 후기 요청
- **NEGATIVE**: 반품/환불 워크플로우 실행
- **NONE**: 제품 사용 팁 발송

## 자주 묻는 질문

**Q: 기존 IF/Else 의도 감지와 어떻게 다른가요?**

A: 의도 감지 액션은 고급 AI를 사용하여 더 정확한 감정 분석을 제공하고, 자동 분기 기능과 복잡한 조건부 로직 설정의 필요성을 제거합니다.

**Q: 하나의 워크플로우에서 여러 의도 감지 액션을 사용할 수 있나요?**

A: 네, 다양한 텍스트 입력을 분석하거나 다단계 감정 분석 워크플로우를 만들기 위해 여러 의도 감지 액션을 연결할 수 있습니다.

**Q: 텍스트에 혼재된 감정이 있으면 어떻게 되나요?**

A: 시스템은 전체적으로 지배적인 감정을 분석합니다. 명확한 감정이 감지되지 않으면 NONE 분기로 라우팅됩니다.

**Q: 라이브 운영 전에 의도 감지를 테스트할 수 있나요?**

A: 네, 워크플로우 활성화 전에 샘플 텍스트 입력으로 워크플로우 테스트 모드를 사용하여 감정 감지 정확도를 확인할 수 있습니다.

**Q: 텍스트가 특정 방식으로 분류된 이유를 볼 수 있나요?**

A: 현재 액션은 분류 결과만 제공합니다. 자세한 추론이나 신뢰도 점수는 향후 업데이트에서 추가될 수 있습니다.
