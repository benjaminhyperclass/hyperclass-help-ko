## 개요

강의 오퍼 접근 권한 취소(Course Revoke Offer) 액션을 사용하면 연락처의 특정 강의 접근 권한을 자동으로 취소할 수 있습니다. 이 액션은 강의 구독 관리, 접근 권한 만료 처리, 또는 워크플로우의 특정 조건이나 트리거에 따라 교육 콘텐츠 접근을 제거할 때 특히 유용합니다.

## 액션명

Course Revoke Offer

## 액션 설명

강의 오퍼 접근 권한 취소 액션은 특정 조건이 충족되었을 때 연락처의 강의 접근 권한을 자동으로 제거하도록 설계되었습니다. 구독 만료, 결제 실패, 강의 완료 등의 시나리오에서 활용할 수 있습니다.

## 액션 세부사항

### 단계별 가이드

1. **액션 유형 선택:**
   워크플로우에서 사용 가능한 액션 목록에서 "Course Revoke Offer"를 선택하세요.

2. **액션 이름 설정:**
   "고급 마케팅 전략 접근 권한 취소"와 같이 설명적인 액션 이름을 입력하세요.

3. **오퍼 선택:**
   접근 권한을 취소할 특정 오퍼를 선택하세요. 예: "월 구독"

4. **트리거 및 조건 설정:**
   이 액션을 활성화할 트리거를 결정하세요. 예를 들어 결제 실패, 구독 취소, 또는 특정 날짜 도달 등이 있습니다.

5. **추가 옵션:**
   선택적으로 접근 권한 취소 시 연락처에게 알림 이메일을 발송하거나 향후 후속 작업을 위해 연락처에 태그를 추가하는 등의 추가 매개변수를 설정할 수 있습니다.

## 예시

**시나리오:** 사용자의 연락처 태그가 'expired(만료)'로 변경되었을 때 "고급 마케팅 전략" 강의 접근 권한을 자동으로 취소하고 싶습니다.

**워크플로우 설정:**

- **트리거:** 연락처 태그 변경(tag=expired 필터 적용)
- **액션:** 강의 오퍼 접근 권한 취소
  - **이름:** "고급 마케팅 전략 접근 권한 취소"
  - **강의:** "고급 마케팅 전략"
  - **취소 조건:** 구독 만료
- **액션:** 이메일 발송
  - **제목:** "고급 마케팅 전략 접근 권한 제거"
  - **템플릿:** "고급 마케팅 전략"
  - **발신 이메일 및 이름:** 로케이션의 기본 이메일 및 이름

**결과:** 연락처에 "expired" 태그가 추가되면 해당 사용자의 "고급 마케팅 전략" 강의 접근 권한이 자동으로 취소되어, 활성 구독자만 콘텐츠에 접근할 수 있도록 보장됩니다.
