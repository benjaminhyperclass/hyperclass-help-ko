## 개요

"노트 추가됨(Note Added)" 트리거는 연락처의 레코드에 노트가 추가될 때마다 워크플로우 내에서 특정 액션을 시작하는 데 사용됩니다.

## 트리거 이름

노트 추가됨(Note Added)

## 트리거 설명

이 트리거는 내부 알림 발송, 연락처 정보 업데이트, 또는 자동화 프로세스에서 후속 액션 실행 등 다양한 작업을 자동화하는 데 사용할 수 있습니다.

### 단계별 가이드

1. **워크플로우 트리거(Workflow Triggers)로 이동**
2. **워크플로우 트리거 선택**: 사용 가능한 트리거 목록에서 "노트 추가됨(Note Added)"을 선택합니다.
3. **트리거 이름 지정**: "노트 추가됨"과 같이 설명적인 트리거 이름을 입력합니다.
4. **필터 설정 (선택사항)**: 워크플로우를 실행할 조건을 지정하는 필터를 추가합니다. 예를 들어, 태그 있음/태그 없음과 같은 표준 필드나 커스텀 필드(있는 경우)로 필터링할 수 있습니다.

## 예시: 헤어컷 노트 알림

**시나리오:** 미용실에서 "헤어컷" 라벨이 붙은 노트가 추가되면 즉시 미용실 매니저에게 내부 알림을 보내고자 합니다.

### **트리거 설정:**

- **트리거:** 노트 추가됨(Note Added)
- **이름:** 헤어컷 노트 알림
- **필터:** 노트에 "헤어컷" 태그가 포함됨

### **워크플로우 액션:**

- **알림:** 노트 내용과 함께 미용실 매니저에게 내부 알림을 발송합니다.
- **할 일 생성:** 미용실 매니저가 노트를 검토할 수 있도록 자동으로 할 일을 생성합니다.

**결과:** 이 자동화를 통해 "헤어컷"으로 표시된 모든 노트가 즉시 미용실 매니저의 주의를 끌게 되어, 신속한 대응과 중요한 문제에 대한 응답 시간 개선을 가능하게 합니다.
