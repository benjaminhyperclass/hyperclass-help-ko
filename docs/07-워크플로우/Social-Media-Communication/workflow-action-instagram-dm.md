## 개요

**인스타그램 DM** 액션을 사용하면 연락처에게 인스타그램을 통해 다이렉트 메시지를 보낼 수 있어요. 성공적인 전달을 위해서는 해당 연락처가 이 이벤트에 도달하기 24시간 이내에 연결된 인스타그램 페이지에 메시지를 보낸 적이 있어야 합니다. 이는 인스타그램의 메시징 정책을 준수하고 규정을 따르기 위함이에요.

## 액션 이름

**인스타그램 DM**

## 액션 설명

**인스타그램 DM** 액션은 인스타그램을 통해 개인화된 다이렉트 메시지를 보낼 수 있게 해주어, 최근에 연결된 인스타그램 계정과 소통한 연락처들과의 상호작용과 참여를 향상시킵니다. 이 액션은 알림, 업데이트, 프로모션 메시지 또는 잠재 리드에 대한 후속 조치를 보내는 데 유용해요.

| Action Name(액션 이름) | 워크플로우 내에서 식별을 위한 액션의 이름을 지정합니다. | 예 |

| Message(메시지) | 전송할 메시지 내용을 입력하세요. 각 연락처에 맞게 메시지를 개인화하기 위해 커스텀 값을 사용할 수 있습니다. | 예 |

### **액션 설정 방법**

- **워크플로우에 액션 추가**: **인스타그램 DM** 액션을 원하는 워크플로우에 드래그 앤 드롭하세요.
- **템플릿 선택**: 미리 정의된 내용을 사용하고 싶다면 메시지 템플릿을 선택하세요. 이는 커뮤니케이션의 일관성 유지에 도움이 됩니다.
- **메시지 작성**: 텍스트 박스에 메시지를 입력하세요. 연락처의 이름이나 관련된 특정 세부 사항을 포함하여 메시지를 개인화하기 위해 커스텀 값을 사용하세요.
- **첨부 파일 추가** (선택사항): **첨부 파일 추가**를 클릭하여 프로모션 이미지나 제품 문서와 같은 파일이나 미디어를 메시지에 포함시키세요.
- **파일 URL 사용** (선택사항): 파일이 온라인에 호스팅되어 있다면, **+ 추가** 버튼을 클릭하여 URL을 통해 파일을 추가하는 옵션을 사용하세요.

## 예시

- **트리거**: 인스타그램 광고 리드 폼 제출
  **조건**: 타겟 인스타그램 광고를 통해 리드 폼이 제출됨.
- **액션**: 인스타그램 DM
  **메시지**: "저희 제품에 관심을 보여주셔서 감사합니다! 인스타그램을 통해 문의해 주신 내용을 잘 받았어요. 앞으로 더 많은 업데이트를 기대해 주세요."
- **템플릿**: 메시지 의도에 맞는 적절한 템플릿을 선택하세요.
- **첨부 파일 추가**: 제품 카탈로그나 프로모션 이미지를 첨부하세요.

워크플로우 내에서 **인스타그램 DM** 액션을 활용하면, 고객들이 적극적으로 사용하는 플랫폼에서 직접 연락할 수 있어 시기적절하고 관련성 있는 커뮤니케이션을 통해 참여를 유도하고 잠재 고객과 더 강한 관계를 구축할 수 있어요.
