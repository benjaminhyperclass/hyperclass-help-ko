## 개요

**사용자 로그인(User Login)** 트리거는 사용자가 레거시 포털에서 자신의 계정에 로그인할 때 워크플로우 내에서 특정 작업을 실행하도록 설계되었습니다. 이 트리거는 사용자 활동을 추적하거나, 로그인 알림을 보내거나, 로그인 이벤트를 기반으로 개인화된 작업을 실행하는 데 특히 유용합니다.

## 트리거 이름

**User Login(사용자 로그인)**

## 트리거 설명

**사용자 로그인(User Login)** 트리거는 사용자가 계정에 로그인할 때마다 워크플로우를 활성화합니다. 이를 통해 환영 이메일 발송, 로그인 빈도 추적, 로그인 시 리마인더나 오퍼 전송 등의 작업을 자동화할 수 있습니다.

## 설정 방법

**사용자 로그인(User Login)** 트리거를 설정하려면 워크플로우 트리거 이름만 설정하면 됩니다. 이 워크플로우가 트리거되는 조건을 더 세밀하게 조정하고 싶다면 추가 필터를 설정할 수 있습니다.

## 예시

이 예시에서는 사용자가 계정에 로그인할 때 워크플로우가 트리거되도록 설정되어 있습니다:

- **Workflow Trigger Name(워크플로우 트리거 이름):** Courses User Login

#### 예시 워크플로우

사용자가 계정에 로그인하면 다음과 같은 작업이 트리거될 수 있습니다:

- **환영 이메일 발송:** 사용자의 계정 재방문을 환영하는 개인화된 이메일을 자동으로 발송합니다.
- **개인화된 오퍼 실행:** 사용자가 일정 기간 동안 로그인하지 않았다면, 특별 오퍼나 리마인더를 자동으로 실행합니다.

이러한 설정을 통해 사용자가 로그인할 때마다 워크플로우가 활성화되어 실시간 고객 참여와 개인화된 커뮤니케이션이 가능해집니다.
