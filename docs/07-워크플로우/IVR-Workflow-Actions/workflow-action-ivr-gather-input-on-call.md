## 개요

통화 중 입력값 수집(Gather Input On Call) 액션을 사용하면 발신자로부터 키패드를 통해 입력값을 수집할 수 있습니다. 이는 발신자가 특정 키를 누르도록 안내하는 IVR 시스템에서 사용됩니다(예: "고객지원은 1번, 메시지 남기기는 2번을 눌러주세요"). 이 액션은 응답을 자동화하고 사용자 입력에 따라 다음 단계로 진행하는 데 도움이 됩니다.

## 액션 이름

**통화 중 입력값 수집(Gather Input On Call)**

## 액션 설명

**통화 중 입력값 수집(IVR)** 액션은 DTMF(Dual Tone Multi-Frequency) 입력을 통해 발신자로부터 정보를 수집합니다. 주로 IVR 플로우에서 사용되며, 사용자가 전화기의 키를 눌러 시스템과 상호작용합니다. 키 입력에 따라 다양한 분기를 설정할 수 있어, 입력값에 따른 맞춤형 플로우가 가능합니다.

### **설정 방법:**

- **액션 이름**: 이 IVR 입력값 수집 액션에 적절한 이름을 설정하세요.

- **메시지 말하기 또는 재생**: "메시지 말하기"(텍스트 음성 변환) 또는 미리 녹음된 오디오 파일 업로드 중 선택하세요.

- **읽을 텍스트**: "메시지 말하기"를 선택한 경우, 재생될 메시지를 입력하세요(예: "고객지원은 1번을 눌러주세요").

- **언어**: 텍스트 음성 변환에 사용할 언어를 선택하세요.

- **메시지 음성**: 텍스트 음성 변환 메시지에 "남성" 또는 "여성" 음성 중 선택하세요.

- **반복 횟수**: 다음 단계로 진행하기 전에 메시지를 몇 번 반복할지 설정하세요.

- **고급 설정**: 입력값 수집을 중단할 시간 초과(초 단위)를 설정하고, 선택적으로 사용자가 키를 누르면 입력값 수집을 중단하도록 설정할 수 있습니다.

- **매치 조건**: 이 기능을 사용하여 발신자의 입력에 따른 분기를 만들 수 있습니다. 예:
  - 분기 1: 고객지원 "1"번
  - 분기 2: 영업팀 "2"번

## 예시

**시나리오**: 은행 IVR 시스템에서 발신자로부터 입력값을 수집하는 경우.

**메시지**: "계좌 내역 조회는 1번, 상담원 연결은 2번, 종료는 3번을 눌러주세요."

키 입력:

- "1"번을 누르면 계좌 내역을 읽어주는 분기로 이동합니다.
- "2"번을 누르면 고객지원 상담원에게 연결됩니다.
- "3"번을 누르면 고객에게 감사 인사 후 통화를 종료합니다.
