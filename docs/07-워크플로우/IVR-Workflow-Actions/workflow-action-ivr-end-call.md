## 개요

**통화 종료(End Call) (IVR)** 액션을 사용하면 IVR 시스템이 즉시 통화를 끊을 수 있습니다. 또한 통화를 종료하기 전에 마지막 메시지나 커스텀 오디오를 재생하도록 선택할 수도 있습니다. 이는 IVR 상호작용을 마무리하거나 연결을 끊기 전에 안내 메시지를 제공하고 싶을 때 유용합니다.

## 액션 이름

**통화 종료(End Call)**

## 액션 설명

**통화 종료(End Call) (IVR)** 액션은 즉시 통화를 끊습니다. 선택적으로 통화를 종료하기 전에 커스텀 메시지나 오디오가 재생되도록 설정할 수 있습니다. 이 기능은 연결을 끊기 전에 발신자에게 마지막 메시지나 확인 안내를 전달할 때 유용합니다.

### 설정 방법

- **액션 이름(Action Name)**: 워크플로우에서 이 액션을 식별할 수 있도록 이름을 설정합니다.
- **음성 안내 추가(Add Voice Instructions)**: 통화가 끝나기 전에 마지막 메시지를 제공하려면 이 토글을 활성화하세요. 그렇지 않으면 통화가 즉시 끊어집니다.
- **메시지 음성 또는 재생(Say or Play Message)**: 텍스트 음성 변환 메시지("Say a message") 또는 업로드된 오디오 파일("Play audio file") 중 선택합니다.
- **음성으로 말할 텍스트(Text to Say)**: "Say a message"를 선택한 경우, 발신자에게 말할 텍스트를 입력합니다.
- **언어(Language)**: 텍스트 음성 변환 메시지의 언어를 선택합니다.
- **메시지 음성(Message Voice)**: 텍스트 음성 변환의 음성 유형(남성 또는 여성)을 선택합니다.
- **반복 횟수(Number of Loops)**: 메시지를 반복할 횟수를 지정합니다(기본값은 1회).

| 필드 이름 | 설명 | 필수 여부 |
|----------|------|----------|
| 액션 이름(Action Name) | 워크플로우에서 이 액션의 이름 | 예 |
| 음성 안내 추가(Add Voice Instructions) | 통화 종료 전 마지막 메시지 추가 활성화/비활성화 토글 | 아니오 |
| 메시지 음성 또는 재생(Say or Play Message) | 마지막 메시지 전달을 위해 "Say a message" 또는 "Play audio file" 중 선택 | 아니오 |
| 음성으로 말할 텍스트(Text to Say) | 음성으로 변환되어 발신자에게 재생될 텍스트 입력 | 아니오 |
| 언어(Language) | 텍스트 음성 변환 음성의 언어 선택 | 아니오 |
| 메시지 음성(Message Voice) | 텍스트 음성 변환을 위해 "남성" 또는 "여성" 중 선택 | 아니오 |
| 반복 횟수(Number of Loops) | 마지막 메시지를 반복할 횟수(기본값 1회) | 아니오 |

## 예시

시나리오: 고객 서비스 통화 마지막에 시스템이 연결을 끊을 것임을 확인하고 고객에게 통화해 주셔서 감사하다는 인사를 전하고 싶습니다.

- 액션 이름: IVR 통화 종료
- 음성 안내 추가: 활성화됨
- 메시지 음성 또는 재생: Say a message
- 음성으로 말할 텍스트: "ABC 서비스에 전화해 주셔서 감사합니다. 이제 통화가 종료됩니다."
- 언어: 한국어
- 메시지 음성: 여성
- 반복 횟수: 1

이렇게 설정하면 시스템이 메시지를 재생한 후 자동으로 통화를 끊습니다.

## 추가 참고사항:

- **음성 안내 추가(Add Voice Instructions)**가 비활성화되어 있으면 마지막 메시지 없이 통화가 즉시 종료됩니다.
- 텍스트 음성 변환을 사용하면 통화를 종료하기 전에 동적이고 실시간 정보를 제공하는 데 효과적입니다.
