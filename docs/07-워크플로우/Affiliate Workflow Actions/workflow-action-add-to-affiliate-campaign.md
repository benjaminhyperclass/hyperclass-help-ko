# 워크플로우 액션 - 제휴 캠페인에 추가

## 개요

제휴 캠페인에 추가(Add To Affiliate Campaign) 액션은 연락처를 선택한 제휴 캠페인에 활성 제휴사로 추가하는 기능입니다. 이 액션은 기존 연락처나 고객을 특정 제휴 캠페인의 제휴사로 등록할 때 유용합니다.

## 액션 이름

제휴 캠페인에 추가(Add To Affiliate Campaign)

## 액션 설명

제휴 캠페인에 추가 액션은 기존 연락처를 선택한 제휴 캠페인의 제휴사로 자동 전환하는 기능입니다.

## 액션 세부사항

### 단계별 가이드

- **액션 유형 선택**: 사용 가능한 액션 목록에서 "제휴 캠페인에 추가(Add To Affiliate Campaign)"를 선택합니다.
- **액션 이름 설정**: "XYZ 캠페인 제휴사 등록"과 같이 설명적인 이름을 입력합니다.
- **제휴 캠페인 선택**: 드롭다운에서 새로 모집한 제휴사를 추가할 제휴 캠페인을 선택합니다.

## 예시

연락처에 태그가 추가되면 기존 연락처를 제휴 테스트 캠페인의 제휴사로 전환

**시나리오**: 비즈니스에서 게시물에 댓글을 달거나 제휴 네트워크 참여에 관심을 보인 고객을 모집하려고 합니다. 사용자는 이러한 연락처를 분류하기 위해 태그를 추가한 후 활성 제휴사로 전환합니다. 사용자는 이 새로운 제휴사들을 제휴 테스트 캠페인에 자동으로 추가하고 싶습니다.

**액션 설정**:
- 액션: 제휴 캠페인에 추가
- 이름: 테스트 캠페인 제휴사 등록

**워크플로우 트리거**:
- 연락처 변경(Contact Changed): 선택한 필터가 변경되거나 추가될 때 연락처가 워크플로우에 진입합니다.
- 필터: "제휴사" 태그 추가

**워크플로우 액션**:
- 제휴 캠페인에 추가
- 제휴 테스트 캠페인 선택

**결과**: 연락처에 해당 태그가 추가될 때마다 연락처가 제휴사로 전환되어 제휴 테스트 캠페인에 추가됩니다. 제휴사는 이제 상품이나 서비스를 판매하여 수수료를 창출할 수 있습니다.
