# 워크플로우 액션 - 제휴 캠페인에서 제거

## 개요

제휴 캠페인에서 제거(Remove From Affiliate Campaign) 액션은 제휴 파트너를 특정 제휴 캠페인에서 제거하는 데 사용됩니다. 드롭다운 메뉴를 통해 어떤 캠페인에서 제거할지 선택할 수 있습니다.

## 액션명

제휴 캠페인에서 제거(Remove From Affiliate Campaign)

## 액션 설명

제휴 캠페인에서 제거 액션은 기존 제휴 파트너를 선택한 제휴 캠페인에서 제거하도록 설계되었습니다.

## 액션 세부사항

단계별 가이드

- 액션 유형 선택: 사용 가능한 액션 목록에서 "제휴 캠페인에서 제거(Remove From Affiliate Campaign)"를 선택하세요.
- 액션명 입력: "블랙 프라이데이 캠페인에서 제거"와 같이 액션을 설명하는 이름을 입력하세요.
- 캠페인 선택: 상태를 활성 또는 비활성 중 어느 것으로 변경할지 선택하세요.

## 예시

탈퇴하거나 탈퇴 폼을 작성할 때 블랙 프라이데이 캠페인에서 제휴 파트너 제거

시나리오: 비즈니스에서 활성 제휴 파트너 목록을 관리하려고 합니다. 현재 제휴 파트너들이 블랙 프라이데이 제휴 캠페인에서 탈퇴하고 싶을 때 작성하는 폼이 있습니다.

액션 설정:

액션: 제휴 캠페인에서 제거(Remove From Affiliate Campaign)

이름: 블랙 프라이데이 캠페인에서 제거

워크플로우 트리거:

- 폼 제출(Form Submitted): 기존 제휴 파트너가 선택한 폼을 제출했을 때가 워크플로우의 진입점이 됩니다
- 필터: 폼이 "제휴 탈퇴 폼"인 경우

워크플로우 액션:

- 제휴 캠페인에서 제거(Remove From Affiliate Campaign)
- 제휴 캠페인 필터에서 "블랙 프라이데이 캠페인" 선택

결과: 기존 제휴 파트너가 "제휴 탈퇴 폼"을 작성할 때마다 해당 제휴 파트너의 프로필이 블랙 프라이데이 제휴 캠페인에서 제거되어, 더 이상 추천 링크를 사용해 판매를 유도하고 수수료를 생성할 수 없게 됩니다.
