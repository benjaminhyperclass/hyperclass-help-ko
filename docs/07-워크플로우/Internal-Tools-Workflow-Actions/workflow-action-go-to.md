# 워크플로우 액션 - 이동(GoTo)

이동(Go-To) 이벤트를 사용하면 워크플로우에서 연락처를 한 이벤트에서 다른 이벤트로 바로 점프시킬 수 있습니다. 이동 단계를 사용하려면, 간단히 이동 이벤트를 추가하고 연락처를 이동시키고 싶은 곳으로 선을 그어주기만 하면 됩니다. 정말 간단해요!
