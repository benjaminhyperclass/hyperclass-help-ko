**중요 안내**: 제휴사 지급 방법 설정하기

이제 제휴사가 사용할 수 있는 지급 방법(PayPal, 은행 이체)을 직접 관리할 수 있습니다.

**설정 방법:**
1. 제휴 관리(Affiliate Manager) → 설정(Settings)으로 이동
2. 제휴사 포털 설정(Affiliate Portal Settings) → 지급 방법 설정(Configure Payout Methods) 클릭
3. 활성화하려는 지급 방법을 선택
4. 변경사항을 저장

제휴사는 관리자가 활성화한 옵션만 볼 수 있어서 지급 과정이 더욱 원활하고 체계적으로 진행됩니다.

## 관리자용

### 1단계: 제휴사 프로필 페이지로 이동

- 제휴 관리에서 해당 제휴사의 프로필 페이지에 접속합니다.

### 2단계: 지급 방법 탭으로 이동

- 아래로 스크롤하여 "지급 방법(Payout Methods)" 탭으로 전환합니다.

### 3단계: 지급 방법 추가

- "지급 방법 추가(Add Payout Method)"를 클릭합니다.

![지급 방법 추가 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155025232191/original/3RowXIU6AnU3FrHbZeCakFkUTLz3YQ70FA.png?1714391840)

### 4단계: 지급 방법 선택 및 필수 정보 입력

- **PayPal의 경우**: 제휴사의 유효한 PayPal 이메일 주소를 입력합니다.

![PayPal 이메일 입력 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155025232235/original/E_I9HKVWL-7IYo9_Gwo3zi2HW1Cn1CPIGw.png?1714391851)

- **은행 계좌의 경우**: 국가를 선택하고 필요한 세부 정보를 입력합니다.

### 5단계: 완료 클릭

- 정보 입력을 완료한 후 "완료(Done)"를 클릭하여 지급 방법을 저장합니다.

![지급 방법 저장 완료](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155025232272/original/sIM3G6KiYqr8k9gsPeSTaH4IafASAow9Cw.png?1714391880)

- 한 제휴사에 대해 여러 지급 방법을 추가할 수 있으며, 그 중 하나를 주 지급 방법으로 설정할 수 있습니다.

![여러 지급 방법 관리](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044056952/original/1M-5szwAbsiGGhN10kOjfbF3pruZQQZXeA.png?1743051536)

## 제휴사용

### 1단계: 제휴사 포털 접속

- 제공받은 매직 링크를 사용하여 제휴사 포털에 로그인합니다.

### 2단계: 수수료 탭으로 이동

- 포털 내에서 "지급 방법(Payout Methods)" 탭으로 전환합니다.

![제휴사 포털 지급 방법 탭](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044057041/original/NM0Ruzh1JcYkhfqgODg4ORyQBUnqHuYlGA.png?1743051690)

### 3단계: 지급 방법 추가

- "지급 방법 추가(Add Payout Method)"를 클릭합니다.

### 4단계: 지급 방법 선택 및 필수 정보 입력

- **PayPal의 경우**: 유효한 PayPal 이메일 주소를 입력합니다.

![제휴사용 PayPal 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155025232378/original/0KYFgdyfAWlaNP3f3iY03_ImJq56d34JSw.png?1714391935)

- **은행 계좌의 경우**: 국가를 선택하고 필요한 세부 정보를 입력합니다.

### 5단계: 완료 클릭

- 정보 입력을 완료한 후 "완료(Done)"를 클릭하여 지급 방법을 저장합니다.

**주의사항:** 현재 PayPal API는 계정 검증 기능을 지원하지 않으므로, 제휴사의 유효한 PayPal 이메일 주소가 올바르게 입력되었는지 주의깊게 확인해 주세요.
