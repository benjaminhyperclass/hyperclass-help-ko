# 제휴 수수료가 틀린 것 같아요

수수료 금액이 예상했던 것과 다르게 나타나나요?

다음과 같은 몇 가지 원인이 있을 수 있습니다:

- HighLevel에서 블랙 프라이데이나 사이버 먼데이 할인 같은 프로모션을 진행하는 경우, 평소 수수료보다 낮게 나올 수 있습니다.
- 추천한 고객이 HighLevel 플랜을 업그레이드했을 때 발생하는 수수료라면, 업그레이드 날짜에 따라 일할 계산된 금액으로 지급됩니다.
- 추천 고객이 HighLevel 팀으로부터 환불을 받은 경우, 포털에 마이너스 잔액으로 표시됩니다.

참고: 전체 잔액이 마이너스라고 해서 돈을 빚진 것은 **아닙니다**. 하지만 지급을 받으려면 마이너스 금액을 상쇄할 만큼의 수수료를 새로 발생시켜야 합니다.

수수료 금액은 맞는 것 같은데 지급을 받지 못하고 있다면, [이 아티클]([where-is-my-affiliate-payout-](where-is-my-affiliate-payout-.md))을 참고해 주세요.
