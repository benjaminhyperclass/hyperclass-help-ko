# 제휴 가입 페이지의 3DS 카드 지원

## **체험판 가입 시 3D Secure(3DS) 카드 인증 문제 해결하기**

일부 은행에서는 온라인 카드 결제를 확인하기 위해 **3D Secure(3DS)** 인증(보통 OTP)을 요구합니다. 인도에서 발급된 카드에 특히 흔한 현상입니다. 결제 과정에서 체험판 가입이 실패한다면, 아래의 3DS 지원 가입 절차를 이용하여 인증 단계를 완료하세요.

## **3DS 지원 가입 링크 사용하기**

1. 다음 링크를 여세요: [https://affiliates.gohighlevel.com](https://affiliates.gohighlevel.com)

2. 체험판 가입을 시작하고 카드 정보를 입력하세요.

3. 요청된다면 은행의 **3DS/OTP 인증**을 완료하세요.

![3DS 인증 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155064182183/original/RZk1EvbEtcU7zWE5Fsdi7AI2m8wP-nBb1A.png?1770211459)

이 가입 절차는 필요한 경우 3DS를 실행하고 카드 확인을 안전하게 검증하도록 설계되었습니다.

### 예상되는 화면

- 카드에 3DS가 필요한 경우 은행 인증 단계(OTP/승인)가 나타납니다.
- 인증을 승인한 후 가입이 완료됩니다.

### **$1 승인에 대한 안내**

3DS가 필요한 경우 이를 실행하고 카드를 확인하기 위해, 가입 절차에서 카드에 **$1 승인**이 발생할 수 있습니다.

- 이는 요금제나 구독에 대한 **실제 결제가 아닙니다**.
- **임시** 승인 대기로 나타날 수 있으며, 은행의 처리 시간에 따라 사라집니다.

### **인증이 여전히 실패하는 경우**

- 최신 브라우저에서 다시 시도해보세요.
- 가입 페이지에서 광고 차단 프로그램을 비활성화하세요.
- 브라우저가 은행 인증 페이지를 차단하는 경우 팝업/리다이렉트를 허용하세요.
- 은행에서 확인을 거부하는 경우 다른 카드를 시도해보세요.

참고: Hyperclass은 메인 가입 절차에도 동일한 3DS 지원 경험을 제공하기 위해 작업하고 있습니다. 그때까지는 3DS 관련 실패가 발생하면 위의 가입 링크를 사용하세요.
