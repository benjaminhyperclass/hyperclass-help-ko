# 하위 계정 목록 내보내기 개요

하위 계정 목록 내보내기 기능을 통해 에이전시 팀은 모든 하위 계정의 핵심 정보가 포함된 CSV 보고서를 빠르게 생성할 수 있습니다. 이 기능은 종합적인 하위 계정 데이터를 얻는 과정을 간소화하여, 수동으로 수집하기엔 시간이 많이 걸리는 방대한 정보에 쉽게 접근하고 분석할 수 있는 간단한 해결책을 제공합니다.

하위 계정 목록에 적용된 필터를 반영하고, 파일을 백그라운드에서 처리한 후, 완료되면 선택한 사용자들에게 이메일로 전송합니다. 브라우저를 계속 열어둘 필요가 없고 필터링된 보기의 완전한 스냅샷을 제공함으로써, 확장성, 정확성, 팀 전체 공유를 위해 설계되었습니다.

---

## **하위 계정 내보내기 기능의 주요 장점**

다음과 같은 장점들로 인해 하위 계정 목록 내보내기 기능은 여러 하위 계정에 대한 세부 분석이 필요한 팀에게 유용한 도구입니다:

- **필터된 내보내기**: CSV가 내보내기 전에 하위 계정 목록에서 필터링한 내용을 정확히 반영합니다
- **종합적인 내보내기 세부사항**: 한 개의 중앙 집중식 파일에서 광범위하고 유용한 계정 수준 데이터를 제공합니다
- **백그라운드 처리**: 파일이 준비되는 동안 안전하게 다른 페이지로 이동할 수 있도록 백그라운드에서 실행됩니다
- **타겟 이메일 전송**: 다운로드 링크를 선택한 팀원들에게 직접 전송하므로 수동 전달이 필요 없습니다
- **대용량 처리**: 복잡한 SaaS 설정에서도 수천 개의 레코드를 효율적으로 처리합니다

## 포함되는 데이터 필드

내보낸 하위 계정 목록은 각 하위 계정의 상태와 활동에 대한 전체적인 보기와 세부적인 보기를 모두 제공합니다. 이러한 필드들은 팀 간에 핵심 정보를 쉽게 분석하고, 조정하고, 공유할 수 있게 해줍니다. 파일에 포함되는 필드들을 그룹별로 나열하면 다음과 같습니다:

- **식별자 및 기본 정보**: ID, 이름, SaaS 활성화 여부, 추가된 날짜
- **사용자 및 연락처 데이터**: 활성 사용자, 연락처, 예약된 약속 수
- **이메일 활동**: 총 이메일, 수신 이메일, 발신 이메일
- **문자 메시지 활동**: 총 문자 메시지, 수신 문자 메시지, 발신 문자 메시지
- **통화 활동**: 총 통화, 수신 통화, 발신 통화, 수신 통화 지속 시간(시간), 발신 통화 지속 시간(시간)
- **폼 및 설문 참여**: 폼 제출, 설문 제출, 페이스북 리드 폼 제출
- **구글 비즈니스 프로필 메시지**: GBP 메시지, 수신 GBP 메시지, 발신 GBP 메시지
- **인스타그램 메시지**: 총 인스타그램 메시지, 수신 인스타그램 메시지, 발신 인스타그램 메시지
- **페이스북 메시지**: 총 페이스북 메시지, 수신 페이스북 메시지, 발신 페이스북 메시지
- **리뷰**: 리뷰, 구글 리뷰, 페이스북 리뷰, 긍정 리뷰, 부정 리뷰
- **생성된 사이트 및 퍼널**: 생성된 사이트 - 퍼널, 생성된 사이트 - 활성 퍼널, 생성된 사이트 - 웹사이트, 생성된 사이트 - 활성 웹사이트, 생성된 사이트 - 폼, 생성된 사이트 - 활성 폼, 생성된 사이트 - 설문, 생성된 사이트 - 활성 설문

## 내보내기에 포함되는 데이터 필터링

필터를 사용하면 내보내기 범위를 필요한 계정으로만 좁혀서 관련 없는 데이터를 제거하고 CSV가 현재 보기와 일치하도록 할 수 있습니다. 내보내기를 시작할 때, 적용한 모든 필터(SaaS 모드, 구독 상태, 할당된 스냅샷 등)가 최종 파일에 반영됩니다.

## **하위 계정 목록 내보내기 방법**

하위 계정 정보가 담긴 CSV 파일을 생성하려면 아래 단계를 따르세요:

#### **1단계: 하위 계정으로 이동**

- 에이전시 대시보드에서 왼쪽 메인 네비게이션 바의 Sub-Accounts(하위 계정)를 선택합니다
- 원하는 데이터셋에 맞게 필터를 적용합니다

![하위 계정 메뉴 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051477787/original/I9AL27weAnK04WMKT3rY5712jHcOmFk7DA.png?1755030339)

#### **2단계: Export Report 선택**

오른쪽 상단 모서리에 있는 Export Report(보고서 내보내기) 버튼을 클릭합니다

![보고서 내보내기 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051477810/original/meujoQ_0y7JElAZ8uyKXPZggM1fF4CvUCQ.png?1755030394)

#### **3단계: 수신자 선택**

- 팝업의 드롭다운을 사용하여 수신자(CSV를 받을 팀원들)를 선택합니다
- 수신자를 선택한 후, 파란색 Export(내보내기) 버튼을 클릭합니다

![수신자 선택 팝업](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051477865/original/BUGV_Izs41U321Um_XCQGz8Gdwp6CT3Z0w.png?1755030517)

#### **4단계: 확인**

- 확인 모달이 나타납니다: "내보내기가 시작되었습니다!"
- 탭을 닫거나 다른 작업을 계속할 수 있습니다. 내보내기는 백그라운드에서 처리되며, 처리가 완료되면 선택한 수신자들에게 파일이 이메일로 전송됩니다.

![내보내기 확인 모달](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051477901/original/EE1mVKjV358jPm-Gk-WmuUrJ6xudlgETsA.png?1755030627)

#### **5단계: 파일 다운로드**

- 이메일이 도착하면, Download Sub-Accounts List-YYYY-MM-DD.csv라는 이름의 파란색 버튼을 클릭하여 파일을 다운로드합니다
- 링크가 만료된 경우(7일 후), 동일한 필터로 내보내기를 다시 실행합니다
- 데이터에 액세스하려면 스프레드시트 소프트웨어를 사용하여 파일을 여세요

![다운로드 이메일](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051477950/original/6Lkw6znStPuZfSZCGI67CjPOva-STAZbOw.png?1755030740)

## 활용 사례

- **고객 보고서**: 내보낸 데이터를 사용하여 고객을 위한 종합적인 보고서를 작성하고 전략의 효과를 입증할 수 있습니다. 여기에는 예약된 약속, 제출된 폼, 인바운드 및 아웃바운드 커뮤니케이션 등의 지표가 포함됩니다.

- **고객 캠페인 전략 수립**: 소셜 미디어 참여도와 고객 상호작용에 대한 세부사항을 통해 고객의 마케팅 노력에 대한 대상의 반응을 더 잘 이해할 수 있습니다. 이는 캠페인 전략의 개선과 조정을 안내할 수 있습니다.

- **자원 관리**: 세부 지표를 통해 어떤 고객이나 캠페인이 가장 많은 자원(증가된 커뮤니케이션이나 약속 등)을 필요로 하는지 파악할 수 있습니다. 이는 팀의 시간과 노력을 더 잘 배분하는 데 도움이 될 수 있습니다.

- **성과 분석**: 설문 제출, 폼 제출, 활성 사용자 수와 같은 지표를 추적하여 각 고객에 대한 마케팅 전략의 효과를 모니터링하고 그에 따라 조정할 수 있습니다.

- **고객 기회 식별**: 데이터 내보내기 기능은 기존 고객에 대한 업셀링이나 서비스 확장 기회를 식별하는 데 도움이 될 수 있습니다. 예를 들어, 고객이 페이스북에서는 상당한 참여도를 보이지만 인스타그램에서는 적은 참여도를 보인다면, 추가적인 소셜 미디어 마케팅 서비스를 제공할 기회가 있을 수 있습니다.

- **온라인 평판 모니터링**: 구글과 페이스북의 리뷰를 통해 고객의 온라인 평판을 모니터링할 수 있습니다. 이는 리뷰 관리 및 고객의 온라인 존재감 개선을 위한 전략을 알려줄 수 있습니다.

- **리드 생성 및 전환**: 리드의 출처(페이스북, 메신저, 인스타그램)를 분석하여 최상의 결과를 산출하는 곳에 마케팅 노력을 집중할 수 있으며, 고객의 리드 생성과 전환을 최적화할 수 있습니다.

- **팀 회의 및 고객 검토**: 이 데이터는 팀 회의와 고객 검토에 유용할 수 있으며, 마케팅 노력의 현재 상태에 대한 명확하고 데이터 기반의 통찰을 제공합니다. 이는 현실적인 기대치 설정과 미래를 위한 정보에 기반한 전략 수립에 도움이 됩니다.

## **자주 묻는 질문**

**Q: 내보내기에는 필터링된 하위 계정만 포함되나요?**
네. CSV는 내보내기를 시작할 때 하위 계정 목록에 적용된 모든 필터를 반영합니다.

**Q: 내보내기는 얼마나 걸리나요?**
일반적인 처리 시간은 7-10분이지만, 대용량 내보내기(수천 개의 하위 계정)는 더 오래 걸릴 수 있습니다. 준비되면 이메일을 받게 됩니다.

**Q: 이메일 링크가 만료되면 어떻게 되나요?**
링크는 7일 후 만료됩니다. 새로운 링크를 생성하려면 내보내기를 다시 실행하세요.

**Q: 브라우저에서 파일을 직접 다운로드할 수 있나요?**
아니요. 내보내기는 비동기적으로 처리되어 선택한 수신자들에게 이메일로 전송됩니다.

**Q: 연락처 내보내기와 어떻게 다른가요?**
연락처 내보내기는 단일 하위 계정으로 범위가 제한된 앱 내 다운로드입니다. 하위 계정 목록 내보내기는 이메일로 전송되는 에이전시 수준의 CSV이며 여러 하위 계정을 다루는 정보가 포함되어 있습니다.
