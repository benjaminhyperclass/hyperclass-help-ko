# 게이미피케이션: 포인트, 배지, 리더보드

새로운 게이미피케이션과 리더보드 기능에 오신 것을 환영합니다! 커뮤니티(Community)의 게이미피케이션 기능은 그룹 참여를 더욱 재미있고 매력적으로 만들어줍니다. 포인트 획득은 사용자가 양질의 콘텐츠를 제작하고 다른 커뮤니티 멤버들과 상호작용하도록 격려하는 진행 시스템을 만듭니다.

목차

- 게이미피케이션 설정하기
- 레벨과 포인트는 어떻게 작동하나요?
- 리더보드
- 리워드

## 게이미피케이션 설정하기

게이미피케이션 레벨은 미리 정의되어 있습니다. 최대 레벨은 9단계로 설정되어 있습니다.

- 그룹 관리자는 레벨 이름을 편집할 수 있습니다. 이렇게 하려면 Settings(설정) > Gamification & Rewards(게이미피케이션 및 리워드) > Gamification(게이미피케이션)으로 이동한 후 편집하고자 하는 레벨 이름을 클릭하세요.
- 반드시 SAVE(저장) 버튼을 클릭하세요.

## 레벨과 포인트는 어떻게 작동하나요?

다른 사람들이 당신의 게시글, 댓글, 답글에 좋아요를 누르면 포인트를 획득합니다. 좋아요 1개 = 게시글 또는 댓글 작성자에게 포인트 1점입니다.
