# 멤버십 테마 사용법

## 개요

이제 직접 선별한 테마를 사용해서 상품 페이지의 느낌을 커스터마이즈할 수 있습니다. 테마를 통해 고유한 브랜드 아이덴티티와 경험을 만들 수 있어요.

커스터마이즈 기능을 사용하면 상품에 다양한 스타일을 추가할 수 있습니다.

## 1단계: 시스템 테마 선택

상품을 만들면 기본적으로 클래식 테마가 선택되지만, 디지털 상품과 개인 브랜드를 가장 잘 어울리게 할 수 있도록 다양한 시스템 테마를 제공해요. 각 테마마다 멤버들에게 제공되는 모양, 느낌, 전체적인 경험이 다르다는 점을 기억하세요.

### 테마 선택 방법

- 테마를 적용하고 싶은 상품을 선택하세요
- 커스터마이즈(Customize)를 클릭하세요

![멤버십 커스터마이즈 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48188471292/original/GMC5baHrqCQj9pqR_TqqD34yp1WB-mKVCQ.jpeg?1644413785)

![테마 갤러리 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48188471320/original/lPVFsn8Nl2PkRQ8KV6Ombx0SOojGK1QjUQ.jpeg?1644413789)

- 테마 갤러리(Theme Gallery)는 이번 릴리즈에 포함된 시스템 테마의 저장공간이며, 사용자가 만든 테마(내 테마)는 향후 릴리즈에서 제공될 예정입니다. 테마 갤러리에서 테마를 적용하거나 기존 테마를 커스터마이즈할 수 있어요.

## 2단계: 테마 커스터마이즈

테마의 상품 페이지를 커스터마이즈하세요

- 테마 갤러리에서 커스터마이즈를 클릭해서 커스터마이즈 빌더를 열어주세요
- 테마 설정을 커스터마이즈하세요
- 페이지의 섹션을 커스터마이즈하려면 왼쪽 사이드바 편집기에서 편집하고 싶은 섹션을 클릭하면 됩니다. 섹션 안을 클릭하면 다양한 커스터마이즈 옵션들을 확인할 수 있어요.

### 테마 설정에서 변경할 수 있는 속성들

1. **주요 색상** - 버튼, 상단 아이콘, 레슨 진행률, 강의 제목, 카테고리 제목, 레슨 제목, 강사 제목에 적용됩니다

2. **보조 색상** - 강사 이름, 강사 설명, 레슨 설명, 카테고리 설명, 버튼 텍스트에 적용됩니다

3. **주요 폰트** - 버튼, 상단 아이콘, 레슨 진행률, 강의 제목, 카테고리 제목, 레슨 제목, 강사 제목에 적용됩니다

4. **보조 폰트** - 강사 이름, 강사 소개, 레슨 설명, 카테고리 설명, 버튼 텍스트에 적용됩니다

5. **로고 위치** - 로고가 헤더에 위치할지 히어로 섹션에 위치할지 결정합니다

### 섹션에서 변경할 수 있는 속성들

**헤더(Header)**
- 경로: Sites(사이트) → Membership(멤버십) → Products(상품) → Customize(커스터마이즈) → Gallery(갤러리) → Customize(커스터마이즈) → Sections(섹션) → Header(헤더)
- 배경: 배경 색상 선택기
- 정렬: 왼쪽, 오른쪽, 또는 가운데

**히어로(Hero)**
- 경로: Sites(사이트) → Membership(멤버십) → Products(상품) → Customize(커스터마이즈) → Gallery(갤러리) → Customize(커스터마이즈) → Sections(섹션) → Hero(히어로)
- 크기: 매우 작음, 작음, 보통, 큼
- 강의 제목: 폰트, 정렬
- 강의 설명: 폰트, 정렬
- 배경: 이미지 크기, 투명도, 색상 선택기
- 버튼: 내용, 폰트, 배경 색상, 정렬

**레슨 진행률(Lesson Progress)**
- 경로: Sites(사이트) → Membership(멤버십) → Products(상품) → Customize(커스터마이즈) → Gallery(갤러리) → Customize(커스터마이즈) → Sections(섹션) → Lesson Progress(레슨 진행률)
- 배경: 색상
- 폰트
- 진행률 색상: 색상

**강의 본문(Course Body)**
- 경로: Sites(사이트) → Membership(멤버십) → Products(상품) → Customize(커스터마이즈) → Gallery(갤러리) → Customize(커스터마이즈) → Sections(섹션) → Course Body(강의 본문)
- 카테고리 폰트: 타입, 크기, 색상
- 하위 카테고리 폰트: 타입, 색상, 크기
- 카테고리 설명 폰트: 타입, 색상, 크기
- 레슨 폰트: 타입, 색상, 크기
- 레슨 설명 폰트: 타입, 색상, 크기
- 배경: 이미지, 색상, 투명도

**강사(Instructor)**
- 경로: Sites(사이트) → Membership(멤버십) → Products(상품) → Customize(커스터마이즈) → Gallery(갤러리) → Customize(커스터마이즈) → Sections(섹션) → Instructor(강사)
- 배경: 색상
- 제목 폰트: 색상, 타입, 크기
- 이름 폰트: 색상, 타입, 크기
- 직책 폰트: 색상, 타입, 크기
- 소개 폰트: 색상, 타입, 크기

### 레슨 페이지에서 섹션에서 변경할 수 있는 속성들

![레슨 페이지 섹션 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48201204369/original/nIDSt9MYWOdAYk9e2uEJCQa92NB763Hqpw.jpeg?1647262299)

**레슨 본문(Lesson Body)**
- 경로: Sites(사이트) → Membership(멤버십) → Products(상품) → Customize(커스터마이즈) → Gallery(갤러리) → Customize(커스터마이즈) → Lesson(레슨) → Sections(섹션) → Lesson Body(레슨 본문)
- 배경: 배경 색상
- 제목: 폰트 타입, 폰트 색상
- 설명: 폰트 타입, 폰트 색상
- 버튼: 내용, 색상, 폰트 타입, 폰트 색상
- 다음 레슨: 배경 색상, 제목 (폰트, 색상), 다음 레슨 버튼 (색상, 폰트 타입, 폰트 색상), 레슨 제목 (폰트 색상)

**강의 내비게이션(Course Navigation)**
- 배경 색상
- 카테고리: 폰트 타입, 폰트 색상
- 레슨: 폰트 타입, 폰트 색상
- 하이라이트: 색상, 폰트 색상
- 브레드크럼: 폰트 타입, 폰트 색상
- 다음 카테고리: 배경 색상, 폰트 타입, 폰트 색상

**강사(Instructor)**
- 경로: Sites(사이트) → Membership(멤버십) → Products(상품) → Customize(커스터마이즈) → Gallery(갤러리) → Customize(커스터마이즈) → Sections(섹션) → Instructor(강사)
- 배경: 색상
- 제목 폰트: 색상, 타입, 크기
- 이름 폰트: 색상, 타입, 크기
- 직책 폰트: 색상, 타입, 크기
- 소개 폰트: 색상, 타입, 크기

**참고사항:**
테마 설정에서 무언가를 커스터마이즈한 후 "변경사항 저장" 버튼을 클릭하면 미리보기에 반영됩니다. "기본값으로 재설정"은 모든 설정을 테마의 기본값으로 변경합니다. 테마 설정이 변경된 후 섹션을 커스터마이즈하면, 섹션 커스터마이즈가 테마 설정 커스터마이즈를 덮어씁니다.

## 3단계: 테마 적용 또는 저장

상품에 테마를 선택하고 상품 페이지를 커스터마이즈했다면, 마지막 단계는 테마를 적용하는 것입니다.

테마가 적용되면 이것이 멤버들이 보고 상호작용할 최종 미리보기가 됩니다!

- 상품을 선택하세요
- 커스터마이즈(Customize)를 클릭하세요
- 테마 갤러리를 둘러보고 적용하고 싶은 테마에서 적용을 클릭하세요
- 또는 테마를 커스터마이즈한 다음 우상단의 적용 버튼을 클릭해서 적용할 수도 있어요

### 테마 저장 버튼

"테마 저장" 버튼을 사용해서 로케이션에서 전역으로 사용할 수 있도록 테마를 저장하세요. "변경사항 저장"을 클릭하면 변경사항이 해당 테마에만 저장되고 전역 사용은 불가능합니다.

테마는 사용자가 "테마 저장" 버튼을 클릭하면 로케이션에서 전역 사용을 위해 저장될 수 있어요.

![테마 저장 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48188471381/original/Wsrrj5bvG2Rl4f63SmSlYNH-yE4T7_1AzQ.jpeg?1644413791)

**참고사항:**
"**테마 저장**"은 변경사항 저장이 클릭된 후 저장된 최신 변경사항만 저장합니다. "저장되지 않은 변경사항"은 반영되지 않아요. 템플릿이 커스터마이즈되어 전역 사용을 위해 저장될 때 이미 같은 테마를 사용하고 있는 다른 모든 상품에는 영향을 주지 않습니다. 사용자가 상품에 테마를 다시 적용하면 새로운 변경사항이 반영되기 시작해요.

### 테마 적용

특정 상품에 테마 적용하기

![테마 적용 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48188471323/original/YTNVI1Y1QdLojnbIwWVPiche0ZKg0z1BWA.jpeg?1644413789)

테마 갤러리에서 "**적용**"을 클릭하면 상품에 테마가 직접 적용됩니다.

![테마 적용 완료](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48188471383/original/bOeOyX1k0nvskjbvO47uY6TZBmAwQszKRg.jpeg?1644413792)

테마를 커스터마이즈한 다음 변경사항 저장을 클릭한 후 적용 버튼을 클릭해서 특정 상품에 테마를 적용할 수 있어요. 적용 버튼은 "적용됨"으로 변경됩니다.

**참고사항:**
적용 버튼을 사용해서 테마가 적용되면 선택된 상품에만 적용되고 커스터마이즈 설정은 로케이션에서 전역으로 사용할 수 없습니다.

### 변경사항 저장

상품 수준에서 테마의 초안을 커스터마이즈하고 저장하세요. 시스템에서 생성된 테마를 커스터마이즈해서 어떤 상품에도 적용하지 않고 초안으로 저장할 수 있어요.

"**변경사항 저장**"을 클릭하면 변경사항이 테마의 초안으로 저장됩니다.

초안 테마(변경사항 포함)는 나중에 상품에 적용할 수 있어요. 초안 테마는 전역이 아니며 모든 로케이션에서 사용할 수 없습니다.

![변경사항 저장](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48188471391/original/cD38dP29VnBWpw_gf-vO-wHCI2qFnylOvg.jpeg?1644413793)

**참고사항:**
상품에 적용되지 않은 테마에서 "**변경사항 저장**"을 클릭하면, 테마가 변경사항으로 업데이트되고 나중에 같은 상품에서 사용할 수 있어요. 테마 속성이 전역적으로 변경되지는 않습니다.

- 커스터마이즈를 클릭하세요
- 강사 세부사항, 파비콘, 로고를 업데이트하세요
