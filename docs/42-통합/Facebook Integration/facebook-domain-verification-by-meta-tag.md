# 메타 태그를 통한 Facebook 도메인 인증

Facebook 도메인 인증은 Facebook 광고와 비즈니스 도구를 사용할 때 필요한 중요한 단계입니다. 메타 태그를 사용하여 도메인 소유권을 확인하는 과정을 안내해드리겠습니다.

## 도메인 인증이 필요한 이유

- Facebook Pixel 데이터의 정확성 보장
- 광고 성과 측정 개선
- iOS 14.5+ 업데이트 대응
- 비즈니스 도메인 신뢰도 향상

## 메타 태그 인증 방법

### 1단계: Facebook Business Manager에서 메타 태그 확인

1. Facebook Business Manager에 로그인합니다
2. **Business Settings(비즈니스 설정)**으로 이동합니다
3. **Brand Safety(브랜드 안전) → Domains(도메인)**을 클릭합니다
4. 인증하려는 도메인을 선택합니다
5. **Verify Domain(도메인 인증)** 버튼을 클릭합니다
6. **Meta tag(메타 태그)** 옵션을 선택합니다
7. 제공된 메타 태그 코드를 복사합니다

### 2단계: Hyperclass에 메타 태그 추가

1. Hyperclass 대시보드에 로그인합니다
2. `Sites(사이트)` 메뉴로 이동합니다
3. 도메인 인증이 필요한 웹사이트 또는 퍼널을 선택합니다
4. **Settings(설정) → SEO Meta Data(SEO 메타 데이터)**를 클릭합니다
5. **Head Tracking Code(헤드 트래킹 코드)** 섹션에 복사한 메타 태그를 붙여넣습니다
6. **Save(저장)** 버튼을 클릭합니다

### 3단계: Facebook에서 인증 확인

1. Facebook Business Manager로 돌아갑니다
2. **Verify(인증)** 버튼을 클릭합니다
3. 인증이 성공적으로 완료되면 녹색 체크 표시가 나타납니다

## 주의사항

- 메타 태그 추가 후 인증까지 최대 24시간이 소요될 수 있습니다
- 도메인의 루트 페이지(홈페이지)에 메타 태그가 있어야 인증됩니다
- 하나의 도메인에 여러 메타 태그를 추가해도 문제없습니다

## 문제 해결

### 인증이 실패하는 경우
- 메타 태그가 정확히 복사되었는지 확인하세요
- 사이트가 발행(Published) 상태인지 확인하세요
- 브라우저 캐시를 지우고 다시 시도하세요
- 24시간 후에 다시 인증을 시도해보세요

### 메타 태그를 찾을 수 없다는 오류
- 사이트의 `<head>` 섹션에 메타 태그가 올바르게 추가되었는지 확인하세요
- 페이지 소스 보기(Ctrl+U)로 메타 태그가 실제로 표시되는지 확인하세요

도메인 인증 관련 추가 질문이 있으시면 Hyperclass 지원팀에 문의해주세요.
