# Facebook 페이지가 표시되지 않거나 연결되지 않는 경우

Facebook 페이지가 연동(Integration) 목록에 나타나지 않거나 연결할 수 없는 문제가 발생할 수 있습니다. 이런 경우 다음과 같은 원인들을 확인해보세요.

## 주요 원인과 해결 방법

### 1. 페이지 관리자 권한 부족
Facebook 페이지를 Hyperclass와 연결하려면 해당 페이지의 **관리자(Admin) 권한**이 필요합니다.

**해결 방법:**
- Facebook에서 페이지 설정 → 페이지 역할을 확인해주세요
- 본인이 관리자 권한을 가지고 있는지 확인하세요
- 권한이 없다면 페이지 소유자에게 관리자 권한을 요청하세요

### 2. Facebook 계정 연동 문제
연동 과정에서 잘못된 Facebook 계정으로 로그인했을 가능성이 있습니다.

**해결 방법:**
1. `Settings(설정) → Integration(연동) → Facebook` 페이지로 이동
2. 기존 Facebook 연결을 해제하세요
3. 올바른 Facebook 계정으로 다시 연결해보세요

### 3. Facebook 앱 권한 미승인
Facebook에서 Hyperclass 앱에 필요한 권한을 승인하지 않았을 수 있습니다.

**해결 방법:**
- Facebook 연동 시 나타나는 모든 권한 요청을 승인해주세요
- 특히 페이지 관리 관련 권한이 중요합니다

### 4. 브라우저 쿠키/캐시 문제
브라우저의 쿠키나 캐시로 인해 연동이 제대로 작동하지 않을 수 있습니다.

**해결 방법:**
- 브라우저 쿠키와 캐시를 삭제하세요
- 시크릿 모드(또는 개인 정보 보호 모드)에서 시도해보세요
- 다른 브라우저에서 시도해보세요

## 추가 확인사항

- Facebook 페이지가 비즈니스 페이지인지 확인하세요 (개인 프로필은 연동 불가)
- 페이지가 비활성화되거나 제재를 받은 상태가 아닌지 확인하세요
- 안정적인 인터넷 연결 상태에서 시도하세요

위 방법들을 시도해도 문제가 해결되지 않으면, Facebook 페이지 URL과 오류 메시지를 포함하여 고객지원팀에 문의해주세요.
