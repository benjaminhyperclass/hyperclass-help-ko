# 메시징 설정 및 문제 해결

이 가이드는 Facebook 페이지와 Instagram 비즈니스 계정을 연결하여 CRM에서 직접 대화를 관리할 수 있도록 도와드립니다.

## 목차

- [Facebook 페이지 메시징 설정](#facebook-페이지-메시징-설정)
- [Instagram 메시징 설정](#instagram-메시징-설정)
- [기타 Facebook/Instagram 관련 지원 가이드](#기타-facebookinstagram-관련-지원-가이드)

## Facebook 페이지 메시징 설정

1. Leadconnector가 해당 페이지에서 메시지 전송 및 메시지 접근 권한을 가져야 합니다 ([https://www.facebook.com/settings?tab=business_tools&ref=settings](https://www.facebook.com/settings?tab=business_tools&ref=settings))

![Facebook 페이지 권한 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051975375/original/YUqz2KdURT-9YHgb010cyxhuZSV0WJ7A0g.jpeg?1755682293)

2. 사용자 계정은 페이지에 대한 전체 액세스 권한과 메시지 접근 권한을 가져야 합니다. ([https://business.facebook.com/latest/settings](https://business.facebook.com/latest/settings))

![사용자 계정 권한 확인](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051975374/original/cp0IkVvwNzY-Ovfo4YuShCC4JyALOaiK9Q.jpeg?1755682293)

3. Settings(설정) >> Integration(연동) >> Facebook 카드 >> Settings(설정)에서 FB 메신저가 활성화되어 있어야 합니다.

![Facebook 메신저 활성화](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051975370/original/ubh0Lx2BwLcsbdL_e6QNzWcPkwidz_0A-Q.jpeg?1755682293)

![Facebook 메신저 설정 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051975368/original/a41m_xMR_kzMScBsSDM4Ro8SfMgMvXNxPw.jpeg?1755682292)

## Instagram 메시징 설정

1. Leadconnector가 해당 페이지에서 메시지 전송 및 메시지 접근 권한을 가져야 합니다 ([https://www.facebook.com/settings?tab=business_tools&ref=settings](https://www.facebook.com/settings?tab=business_tools&ref=settings))

![Instagram 권한 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051975373/original/3e4HfLHBuS2vcIzHqOPPlKkwyYesU5fDKg.jpeg?1755682293)

2. Instagram 계정에서 인스타그램 메시지 전송 및 수신 권한이 있는지 확인하세요 ([https://business.facebook.com/latest/settings](https://business.facebook.com/latest/settings))

![Instagram 메시지 권한 확인](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051975369/original/nhErqFoGV5OIegyHIHcNcbDrovfqJf9NCQ.jpeg?1755682293)

3. 연결된 페이지의 연결된 자산 목록에 인스타그램 계정이 표시되는지 확인하세요.

![Instagram 연결 확인](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051975371/original/rhD9Rep3yoErxPaN6MV6KiE2NpVJ1UIaRw.jpeg?1755682293)

4. Settings(설정) >> Integration(연동) >> Facebook 카드 >> Settings(설정)에서 Instagram 메시징이 활성화되어 있어야 합니다.

![Instagram 메시징 활성화](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051975372/original/DAyCnypW3ZQUyFnsnFIGJUt3SbX3qGNf5w.jpeg?1755682293)
