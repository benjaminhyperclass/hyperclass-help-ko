# 리드 광고 연동이 작동하지 않을 때 해결 방법

페이스북 리드 광고 연동이 갑자기 작동하지 않나요? 이런 문제는 여러 원인으로 발생할 수 있습니다. 다음 단계를 따라 해결해보세요.

## 1단계: 페이스북 연동 상태 확인

먼저 페이스북 연동이 여전히 활성화되어 있는지 확인하세요:

1. `Settings(설정) → Integrations(연동)` 메뉴로 이동합니다
2. Facebook 섹션을 확인합니다
3. 연결 상태가 "Connected"로 표시되는지 확인합니다

## 2단계: 페이스북 권한 재승인

페이스북에서 권한이 만료되거나 변경될 수 있습니다:

1. Facebook 연동 설정에서 **Disconnect** 버튼을 클릭합니다
2. 몇 분 후 다시 **Connect to Facebook** 버튼을 클릭합니다
3. 페이스북 로그인 후 모든 권한을 승인합니다
4. 특히 "페이지 관리" 및 "리드 광고 접근" 권한이 선택되어 있는지 확인하세요

## 3단계: 리드 폼 연결 재설정

리드 광고와 Hyperclass 간의 연결을 다시 설정해야 할 수 있습니다:

1. 페이스북 비즈니스 매니저에서 해당 리드 광고로 이동합니다
2. 리드 폼 설정을 확인합니다
3. CRM 연결 부분에서 Hyperclass 연결을 해제하고 다시 연결합니다

## 4단계: 웹훅 URL 재확인

리드 데이터를 받는 웹훅 URL이 올바른지 확인하세요:

1. `Settings(설정) → Integrations(연동) → Facebook`에서 웹훅 URL을 복사합니다
2. 페이스북 비즈니스 매니저의 리드 폼 설정에서 해당 URL이 정확히 입력되어 있는지 확인합니다

## 5단계: 테스트 리드 생성

연동이 제대로 작동하는지 테스트해보세요:

1. 페이스북 광고 관리자에서 "테스트 리드 생성" 기능을 사용합니다
2. Hyperclass의 `Contacts(연락처)` 메뉴에서 테스트 리드가 들어왔는지 확인합니다
3. 2-3분 내에 데이터가 나타나지 않으면 추가 점검이 필요합니다

## 6단계: 페이스북 페이지 권한 확인

리드 광고가 연결된 페이스북 페이지에 대한 관리자 권한이 있는지 확인하세요:

1. 페이스북 페이지 설정에서 "페이지 역할"을 확인합니다
2. 연동에 사용한 계정이 "관리자" 또는 "편집자" 권한을 가지고 있는지 확인합니다

## 그래도 문제가 해결되지 않는다면

위 단계를 모두 시도해도 리드 광고 연동이 작동하지 않는다면:

- 페이스북에서 최근 정책 변경이나 API 업데이트가 있었는지 확인해보세요
- 리드 광고 캠페인이 실제로 활성화되어 있고 리드를 생성하고 있는지 확인해보세요
- 페이스북 비즈니스 매니저에서 오류 알림이나 경고 메시지가 있는지 확인해보세요

대부분의 경우 위 단계 중 2단계(권한 재승인)에서 문제가 해결됩니다. 페이스북의 보안 정책으로 인해 주기적으로 권한을 다시 승인해야 하는 경우가 많습니다.
