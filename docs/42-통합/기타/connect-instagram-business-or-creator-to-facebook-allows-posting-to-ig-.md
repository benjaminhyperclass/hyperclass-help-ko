# 인스타그램 연결: 페이스북 연동 vs 직접 인스타그램 연동

페이스북 페이지 없이도 하이퍼클래스의 소셜 플래너(Social Planner)에서 인스타그램에 직접 연결하고 게시물을 예약하세요! 이번 업데이트로 인스타그램 관리가 더욱 간편해지고, 크리에이터 계정도 지원하며, SNS 작업 흐름이 단순해집니다.

## **새로운 기능**

- **직접 인스타그램 연동**: 페이스북 페이지 없이 인스타그램에 직접 연결하여 게시물을 예약할 수 있습니다.

- **크리에이터 계정 지원**: 크리에이터 계정에서 스토리를 자동 게시할 수 있습니다.

- **간편한 워크플로우**: 추가 페이스북 권한 없이 하이퍼클래스에서 게시물, 스토리, 릴을 관리할 수 있습니다.

---

## **소셜 플래너의 직접 인스타그램 연동이란?**

직접 인스타그램 연동은 하이퍼클래스 소셜 플래너의 새로운 기능으로, 페이스북 페이지 연결 없이도 인스타그램 비즈니스 또는 크리에이터 계정에 연결하여 콘텐츠를 예약할 수 있습니다. 이를 통해 주요 장벽을 제거하고, 프로세스를 간소화하며, 인스타그램 관리를 위한 통합 대시보드를 제공합니다.

이 개선 사항으로 다음이 가능해집니다:

- 스토리 자동 게시 (크리에이터 계정 포함)
- 게시물, 스토리, 릴 예약
- 소셜 플래너에서 직접 참여 지표 확인

---

## **직접 인스타그램 연동의 주요 장점**

- **페이스북 페이지 불필요**: 페이스북 연결 단계를 건너뛰고 인스타그램 권한만 부여하면 됩니다.

- **크리에이터 및 비즈니스 계정 지원**: 두 계정 유형 모두에서 전체 스토리 및 게시물 예약이 가능합니다.

- **간편한 워크플로우**: 모든 인스타그램 콘텐츠를 하나의 인터페이스에서 관리할 수 있습니다.

- **빠른 설정**: 몇 분 안에 가이드를 따라 연결할 수 있습니다.

- **통계 접근**: 참여 데이터(좋아요, 댓글) 확인 가능; 인스타그램 API 제한으로 이미지/비디오 조회수는 제외됩니다.

- **콘텐츠 유연성**: 인스타그램 가이드라인에 따라 게시물, 스토리, 릴을 예약할 수 있습니다.

- **시간 절약**: 더 적은 단계, 더 적은 권한으로 콘텐츠 제작에 집중할 수 있습니다.

## 시작하기 전에

옵션 간의 주요 차이점은 다음과 같습니다:

- **직접 인스타그램 연동 (신규)**: 소셜 플래너 설정 중 페이스북 페이지 연결이 필요하지 않습니다. 인스타그램으로만 인증합니다. 이를 통해 기존의 "페이스북 페이지 연결 후 인스타그램 비즈니스 페이지 연결" 과정을 건너뛸 수 있습니다. 단, 인스타그램/페이스북(하이퍼클래스가 아닌)에서는 여전히 기본 프로필이 다음 중 하나여야 합니다:
  - 페이스북 페이지에 연결된 비즈니스 프로필
  - 크리에이터 프로필 (페이스북 페이지 불필요)

- **페이스북 연동 연결**: 소셜 플래너 내에서 명시적으로 페이스북 페이지를 연결해야 합니다. 레거시 페이스북 권한이나 조회수 분석을 원하는 경우에만 필요합니다.

#### 1) 인스타그램 프로필이 비즈니스 또는 크리에이터 프로필인지 확인하세요.
- 인스타그램 비즈니스 또는 크리에이터 프로필이 없다면, 개인 프로필을 비즈니스 또는 크리에이터 프로필로 전환하는 방법을 참조하세요.

#### 2) 클래식 페이지 유형의 경우 페이스북 페이지에 대한 관리자 또는 편집자 페이지 역할이 있는지 확인하거나, 새 페이지 유형의 경우 전체 또는 부분 제어 권한이 있는 페이스북 액세스가 있는지 확인하세요. [페이스북 페이지에서 내 역할이 무엇인지 어떻게 확인하나요?](https://www.facebook.com/help/510247025775149) 페이스북 도움말 문서를 참조하세요.
- 페이지가 페이스북 비즈니스 매니저의 자산인 경우, 비즈니스 매니저 계정에서도 페이지에 대한 관리 권한이 필요합니다. 페이스북 도움말 문서 [비즈니스 매니저에 사람 추가하기](https://www.facebook.com/business/help/2169003770027706?id=2190812977867143)를 참조하세요.

## 방법 1: 페이스북 연동 연결 (기존 방식)

중요: 페이스북 권한에 여전히 의존하는 레거시 기능이 필요한 경우 이 방법을 사용해야 합니다.

이 방법은 페이스북을 통해 인스타그램 계정을 연결하며, 다음과 같은 기능을 제공합니다:

- 향상된 지표 (비디오/이미지 조회수 포함)
- 메타 비즈니스 기능 액세스

### **단계**:

1. 페이스북에 로그인한 후 우상단의 프로필 사진을 클릭합니다.

2. 모든 프로필 보기를 클릭한 후 전환하려는 페이지를 선택합니다.

3. 페이스북 우상단의 페이지 프로필 사진을 클릭합니다.

4. 설정 및 개인정보 보호(Settings & privacy)를 클릭합니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155047067929/original/ir8FH0rSyc8yfTcKAtWmLTD8YbHx_VxBEQ.png?1747915613)

5. 그 다음 설정(Settings)을 클릭합니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155047067910/original/Y7DU7H2rgpJPXpEGD_hDvQYdUVqM_Q0EHA.png?1747915597)

6. 권한 아래의 연결된 계정(Linked accounts)을 클릭합니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155047067896/original/LqYoIsViSLK4t_89n1K36IOsYG09ZepMOg.jpeg?1747915565)

7. 인스타그램 옆의 ![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155047066949/original/Gh1sa9kEcIcQk8FxU2Uwf-SQ_XoxjSWkJQ.png?1747914894) 보기(View)를 클릭합니다.

8. 계정 연결(Connect account)을 클릭한 후 안내에 따라 페이스북 페이지와 인스타그램 계정을 연결합니다.

9. 이제 인스타그램 비즈니스 프로필이 페이스북 계정에 연결되었습니다.

10. 마케팅(Marketing) > 소셜 플래너(Social Planner)로 이동합니다.

11. 새 게시물(New Post) 옆의 설정(설정 아이콘)을 클릭합니다.

12. 소셜 추가(Add Socials)를 선택합니다.

13. 인스타그램을 선택합니다.

14. 페이스북과 함께하는 인스타그램을 선택합니다.

15. 안내에 따라 페이스북 페이지와 인스타그램 계정을 연결합니다.

**참고**: 자세한 내용은 [메타 공식 설정 가이드](https://www.facebook.com/business/help/898752960195806)를 참조하세요.

## **방법 2: 직접 인스타그램 연동 (권장)**

빠른 설정과 스토리 예약에 이상적이며, 페이스북이 필요하지 않습니다.

### **단계**:

1. 마케팅(Marketing) > 소셜 플래너(Social Planner)로 이동합니다.

2. 새 게시물(New Post) 옆의 **설정**(설정 아이콘)을 클릭합니다.

3. **소셜 추가**(Add Socials)를 선택합니다.

4. **인스타그램**을 선택합니다.

5. **직접** 인스타그램 연동을 선택합니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048316055/original/RYGaSbqo4Vd4GHZngEsrLCaiBmPfAvcURA.gif?1750075121)

6. 팝업에서 직접 인스타그램 연동을 선택합니다.

7. 권한을 부여하고 인스타그램 비즈니스 또는 크리에이터 계정을 선택합니다.

## 기능 비교

| 기능 | 페이스북 연동 | 직접 연동 (신규) |
|------|------------|-----------------|
| **페이스북 페이지 필요** | 예 | 아니오 |
| **크리에이터 계정 지원** | 제한적 | 예 |
| **스토리 자동 게시** | 아니오 | 예 |
| **릴 예약** | 예 | 예 |
| **참여 지표 보기** | 예 | 예 (조회수 제외) |
| **설정 복잡성** | 높음 (다중 로그인) | 낮음 (단일 로그인) |

인스타그램 비즈니스 프로필에 직접 게시하기 위해서는 인스타그램 비즈니스 프로필이 페이스북 페이지나 크리에이터 프로필에 연결되어 있어야 합니다. 인스타그램을 사용할 때 페이스북 페이지에 게시할 필요는 없지만, 인스타그램 프로필을 관리할 수 있는 권한은 페이스북을 통해 LeadConnector에 부여됩니다. 이는 인스타그램을 소유한 페이스북의 요구사항입니다.

완료되면 소셜 플래너에 인스타그램 비즈니스 프로필을 연결할 수 있습니다.

## **자주 묻는 질문**

**Q: 개인 계정으로 직접 인스타그램 연동을 사용할 수 있나요?**
아니오, 비즈니스 및 크리에이터 계정만 지원됩니다.

**Q: 페이스북 페이지를 먼저 연결 해제해야 하나요?**
아니오, 설정 중에 연동 유형을 선택하세요. 직접 연동을 선택하기만 하면 됩니다.

**Q: 어떤 게시물 유형을 예약할 수 있나요?**
게시물, 스토리, 릴 (인스타그램 가이드라인에 따라)을 예약할 수 있습니다.

**Q: 모든 분석 기능을 사용할 수 있나요?**
참여 지표는 사용 가능하지만, 이미지/비디오 조회수는 제공되지 않습니다.

**Q: 크리에이터 계정에서 스토리를 자동 게시할 수 있나요?**
예, 크리에이터 계정에서 스토리 자동 게시가 지원됩니다.

**Q: 게시물 게시가 실패하면 어떻게 해야 하나요?**
콘텐츠가 인스타그램 사양을 충족하고 계정이 올바르게 연결되어 있는지 확인하세요.

**Q: 연결할 수 있는 계정 수에 제한이 있나요?**
하이퍼클래스 플랜 제한까지 여러 계정을 연결할 수 있습니다.

**Q: 최신 인스타그램 사양은 어디서 확인할 수 있나요?**
위의 [콘텐츠 가이드라인] 참고 섹션을 확인하세요.
