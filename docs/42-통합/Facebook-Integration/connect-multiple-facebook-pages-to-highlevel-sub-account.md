# 하이퍼클래스 하위 계정에 여러 페이스북 페이지 연결하기

가장 많은 요청을 받았던 연동 기능 중 하나인 여러 페이스북 페이지 연결 기능을 출시하게 되어 기쁩니다.

**목차**

- [새로운 기능](#새로운-기능)
- [이 기능이 중요한 이유](#이-기능이-중요한-이유)
- [곧 출시 예정](#곧-출시-예정)

## **새로운 기능**

- **리드 동기화**: 여러 페이스북 페이지의 Facebook Lead Ads에서 리드를 자동으로 동기화하며, 출처를 명확하게 추적할 수 있습니다

- **대화**: Facebook과 Instagram 채팅 연동을 통해 여러 페이스북 페이지의 고객들과 대화할 수 있습니다

- **자동화**: 여러 페이스북 페이지에서 고객 참여를 자동화하는 맞춤형 워크플로우를 구축할 수 있습니다

- **평판 관리**: 여러 페이스북 페이지의 리뷰와 리뷰 응답을 관리할 수 있습니다

## **이 기능이 중요한 이유**

Canny에서 800개 이상의 투표를 받으며 가장 많은 요청을 받았던 연동 기능입니다. 이 기능은 여러 매장을 관리하는 중소기업부터 여러 팬 페이지를 운영하는 인플루언서까지 다양한 사용 사례의 고객들이 활용할 수 있습니다.

## **곧 출시 예정:**

- **모바일 앱**: 하이퍼클래스 모바일 앱은 현재 여러 페이스북 페이지에서의 대화 기능을 지원하지 않습니다.

- **대화 AI**: 대화 AI는 현재 여러 페이스북 페이지의 메시지에 대한 자동 응답을 지원하지 않습니다.

### **문제 해결: 페이스북 페이지가 표시되지 않을 때**

설정 중에 연결하려는 페이스북 페이지가 목록에 나타나지 않는다면, 일반적으로 권한이나 접근 권한 문제입니다.

**중요:** 페이스북 페이지는 연결하는 페이스북 사용자 계정의 권한에 따라 표시됩니다. 하이퍼클래스는 Facebook Business Manager나 광고 계정에 직접 연결되지 않습니다.

**다음 사항을 확인하세요:**

- 페이스북 페이지에 대한 **관리자 권한**이 있는지 확인하세요 (편집자나 부분 권한이 아닌)

- 연결하는 **올바른 페이스북 프로필**에 로그인되어 있는지 확인하세요

- 연결 단계에서 **요청된 모든 권한을 허용**했는지 확인하세요

- 접근 권한이 최근에 업데이트된 경우, `Settings(설정) → Integrations(연동)`에서 **페이스북 연결을 해제하고 다시 연결**해 보세요

완전한 단계별 문제 해결 가이드(LeadConnector 권한, Business Manager 설정, 리드 접근 문제 포함)는 다음을 참조하세요:

[Facebook 다중 페이지 문제 해결](facebook-multi-page-troubleshoot.md)
