# 하이퍼클래스 계정에서 Facebook 토큰이 만료되었습니다

"A Facebook Token Has Expired In Your Hyperclass Account"라는 제목의 이메일을 받으셨다면, 계정 중 하나의 Facebook 연동(Integration)이 끊어진 것을 의미합니다.

### 다시 연결하기:

1. 받은 이메일에서 언급된 계정을 "계정 전환" 드롭다운에서 선택하세요

2. 좌측 사이드바에서 아래로 스크롤하여 "설정(Settings)"을 클릭하세요

3. 화면 상단의 메뉴에서 "연동(Integrations)"을 클릭하세요

4. Facebook 아이콘 아래의 "연결(Connect)" 버튼을 클릭하세요

5. 팝업 창에서 본인으로 계속 진행하고, 연결하려는 Facebook 페이지를 선택한 후 "페이지 연결(Connect Page)" 버튼을 클릭하세요

![Facebook 토큰 만료 시 재연결하는 방법](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48022122102/original/zoNotXcSOQ6ICBhoY7qOhe1nPhAw10mA0A.gif?1578582352)

### 연결이 끊어지는 이유는 무엇인가요?

연동이 끊어지는 원인은 여러 가지가 있습니다. 가장 일반적인 원인들은 다음과 같습니다:

- 사용자가 비밀번호를 변경한 경우
- Facebook 토큰이 일정 시간 후 자연스럽게 만료되는 경우
- 사용자가 앱 권한을 해제한 경우
- 사용자가 Facebook에서 로그아웃한 경우
- 사용자가 페이지 권한을 변경하거나 사용자를 추가/삭제한 경우
- 다른 나라의 가상 어시스턴트가 VPN 없이 로그인한 경우
