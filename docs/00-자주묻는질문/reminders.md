# 예약 확정·리마인드 알림이 자동으로 발송되나요?

네, 한 번 설정해 두면 자동으로 나갑니다. **현재 기본 발송 채널은 이메일**입니다.

**예약 확정 시 (즉시)**

- 예약이 확정되면 일정·장소(또는 화상회의 링크)가 담긴 **확정 안내가 이메일로 즉시 발송**되고, 캘린더 초대 파일도 함께 첨부됩니다.

**리마인드 (노쇼 방지)**

- 미팅 전 정해둔 시점에 자동으로 리마인드가 나갑니다. 시점은 **자유롭게 설정**할 수 있고, 흔히 쓰는 구성은 **24시간 전 · 2시간 전 · 15분 전** 3단계예요.
- 리마인드에 **"일정 변경(리스케줄)" 링크**를 함께 넣을 수 있어요. 이 자동 리마인드로 노쇼가 크게 줄어듭니다.

**발송 채널 안내 (중요)**

- **이메일** — 기본 제공. 바로 사용 가능합니다.
- **문자(SMS)** — 한국에서는 **SK브로드밴드 발신번호 신청 절차를 거친 뒤** 사용할 수 있습니다. (현재 준비 중인 기능으로, 준비되는 대로 안내드립니다.)
- **카카오 알림톡** — 기본 구독에 포함되지 않은 **별도 연동/추가 서비스**이며, 알림톡 템플릿은 카카오 검수를 거쳐야 합니다. (→ [카카오 채널 연동](kakao-channel.md), [알림톡 템플릿 등록](alimtalk.md))

**설정 방법**

- 캘린더의 리마인드 설정에서 시점·문구를 지정하거나, 자동화(워크플로우)에서 *"예약 확정 시 → 발송"*, *"미팅 N시간 전 → 발송"*으로 세밀하게 구성할 수 있어요.
- 한 번 만들어 두면 **이후 모든 예약에 자동 적용**됩니다.

> **따라 하기:** 캘린더(또는 워크플로우)에서 확정·리마인드 시점과 채널을 설정 → 문구 작성 → 저장. (현재는 이메일 기준, 문자는 발신번호 준비 후)
