# 상담·미팅 예약은 어떻게 받나요?

고객이 **빈 시간을 직접 골라 예약**하는 캘린더가 기본 내장돼 있어요. Calendly 같은 별도 예약 도구 없이, 수기로 시간 조율할 필요가 없습니다.

## 1) 예약 캘린더 만들기

- 받을 수 있는 **요일·시간대**와 1회 상담 길이(예: 45분), 예약 간 여유시간(버퍼)을 설정합니다.
- 예약 전 받고 싶은 정보를 **인테이크 폼**으로 붙이면, 미팅 전에 고객 정보를 미리 파악할 수 있어요.
- 용도에 따라 **1:1 예약 / 팀 라운드로빈 / 그룹·클래스(웨비나) / 서비스 메뉴형** 캘린더 중 선택할 수 있습니다.

## 2) 고객이 예약

- 캘린더는 **고유 예약 페이지 주소**가 생겨요. 이 링크를 랜딩페이지·이메일·SNS 프로필 어디에든 붙이면 됩니다.
- 고객은 빈 슬롯을 골라 예약하고, **예약 즉시 CRM에 고객으로 자동 등록**됩니다.

## 3) 확정 방식 — 자동 / 수동 중 선택

- **자동 확정(기본값)**: 고객이 예약하면 곧바로 확정 처리됩니다.
- **수동 확정**: 캘린더 설정에서 **"새 예약 자동 확정"을 끄면**, 모든 예약이 먼저 **'미확정' 상태**로 들어와 사장님이 검토 후 직접 확정합니다. (고가 상담처럼 한 번 거르고 싶을 때 적합)
- 확정 시 **확정 안내가 이메일로 자동 발송**되고, 노쇼를 줄이는 리마인드도 자동으로 이어집니다. ([예약 확정·리마인드 알림](reminders.md) 참고 — 문자·알림톡 발송 조건도 거기서 확인)

## 4) 화상 미팅 자동 링크

- **구글 미트 · Zoom · MS Teams**를 연동하면, 예약마다 **고유 화상회의 링크가 자동 발급**되어 확정 안내에 포함됩니다. 매번 회의방을 직접 만들 필요가 없어요.
- 대면 미팅이면 장소 주소를, 전화 상담이면 안내 문구를 넣을 수 있습니다.

## 5) 노쇼 방지·결제 옵션

- 예약 페이지에 **선결제/예약금(토스페이먼츠)**을 붙여, 결제가 완료돼야 예약이 확정되게 할 수 있어요. 노쇼를 크게 줄입니다.
- 내 **구글/아웃룩 캘린더와 동기화**하면 기존 일정과 겹치지 않게 빈 시간만 노출됩니다. ([구글·아웃룩 캘린더 연동](calendar-sync.md) 참고)

> **따라 하기:** 캘린더 만들기 → 가능 시간·상담 길이 설정 → (선택) 인테이크 폼·선결제 연결 → 화상회의 연동 → 자동/수동 확정 선택 → 예약 링크 복사해 배포.
