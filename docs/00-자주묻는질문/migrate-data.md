# 기존에 쓰던 고객 데이터/CRM을 옮겨올 수 있나요?

네, 기존 고객 명단을 가져오실 수 있습니다. 보통 10분 안에 끝나요.

- **CSV 파일로 한 번에 업로드**합니다. 엑셀(.xlsx)을 쓰셨다면 CSV로 먼저 내보낸 뒤 업로드하시면 돼요. (첫 줄은 반드시 제목 행(header), 시트는 하나만)
- **다른 CRM에서 옮겨오기**: "버튼 한 번에 자동 연동"되는 직접 연결은 없고, 기존 시스템에서 명단을 CSV로 내보낸 뒤 업로드하는 방식이 표준이에요. (대부분의 CRM·엑셀·구글시트가 CSV 내보내기를 지원합니다.)
- **중복은 알아서 정리됩니다.** 가져올 때 Contact ID → 이메일 → 전화번호 순서로 기존 기록과 대조해서, 같은 사람이면 새로 만들지 않고 기존 정보를 업데이트해요. "새로 추가만 / 기존 업데이트만 / 둘 다" 중에서 고르실 수 있습니다.
- **용량**: 파일은 CSV 형식, 30MB 이하. 명단이 많으면 500~1,000건씩 나눠서 올리는 걸 권장해요. (총 연락처 수에는 제한 없음)

**따라 하기:**

1. 기존 데이터를 `이름 · 휴대폰번호 · 이메일 · 메모` 열로 정리합니다. (이메일이나 전화 중 최소 하나는 꼭 채워야 중복 판별이 됩니다.)
2. 휴대폰번호는 하이픈·괄호·공백 없이 숫자만, 가능하면 국가번호 포함. 엑셀이면 CSV(UTF-8)로 저장.
3. 연락처(Contacts) 메뉴 → 가져오기(Import) 클릭 → 파일 업로드.
4. 열 매칭 화면에서 각 열을 시스템 항목에 연결. (시스템에 없는 항목은 맞춤 필드를 먼저 만들어 두면 매칭됩니다. 안 쓸 열은 비워두면 무시돼요.)
5. 중복 처리·태그(예: `2026-기존고객`)를 정하고 가져오기 실행.

> **팁:** 먼저 10~20건만 테스트로 올려서 제대로 들어가는지 확인한 뒤 전체를 올리세요. 원본 파일은 꼭 백업해두시고요.

> **개인정보 안내:** 고객 명단에는 개인정보가 포함되므로, 가져오기 전에 수집·이용 동의 여부를 꼭 확인해 주세요.
