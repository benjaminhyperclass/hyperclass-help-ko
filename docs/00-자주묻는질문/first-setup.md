# 가입하면 무엇부터 설정하나요? (처음 세팅 순서)

처음 가입하시면 아래 순서대로 따라오시면 됩니다. 한 번만 세팅해 두면 이후엔 시스템이 자동으로 돌아가요. 각 단계는 5~15분이면 충분합니다.

1. **기본 정보 등록** — 비즈니스 이름, 담당자 이름, 대표 연락처(전화·이메일)를 입력합니다. 여기 넣은 정보는 고객에게 나가는 안내·이메일·예약 확정 메시지에 자동으로 들어가니, 실제 고객에게 보여줄 이름·연락처로 정확히 적어주세요.
2. **비즈니스 페이지 주소 만들기** — 고객이 처음 만나는 웹페이지(랜딩페이지/사이트)를 만들고 주소(URL)를 연결합니다. 템플릿을 골라 내용을 채우면 코딩 없이 완성됩니다. 주소는 내가 가진 도메인 연결 또는 기본 제공 주소 중 선택. (새 도메인 *구매*는 사용량 항목이라 정식 구독 후 가능 → [사용량 비용](usage-costs.md))
3. **예약 캘린더 설정** — 상담·미팅을 받을 요일·시간대와 1회 길이(예: 45분)를 지정합니다. (→ [상담·미팅 예약](booking.md))
4. **설문조사·폼 등록** — 고객에게 받고 싶은 정보를 폼이나 설문으로 만듭니다. 2번 페이지나 예약 과정에 붙여두면 자동으로 CRM에 쌓여요.
5. **첫 자동화/퍼널 만들기** — "이런 일이 생기면 → 이렇게 한다"는 자동 흐름을 하나만 만들어 보세요. (→ [자동화](automation.md))
6. **CRM으로 고객 DB 통합** — 흩어져 있던 고객 정보를 마이그레이션 템플릿으로 한 곳에 모읍니다. (→ [기존 고객 데이터 옮겨오기](migrate-data.md))

> **팁:** 처음부터 모든 기능을 켜려 하지 마세요. **"문의 자동응답 → 예약 → 알림"** 이 한 줄기부터 완성하시는 걸 추천합니다.
