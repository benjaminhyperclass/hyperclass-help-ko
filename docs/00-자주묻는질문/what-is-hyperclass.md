# 하이퍼클래스는 어떤 서비스인가요?

하이퍼클래스는 **온라인으로 코칭·컨설팅·강의 비즈니스를 운영하는 분을 위한 '지식비즈니스 자동화 플랫폼'**이에요. 잠재고객이 처음 유입되는 순간부터 결제, 후기, 재구매까지 — 사업의 모든 단계를 **하나의 시스템(하나의 데이터베이스)** 안에서 돌립니다.

핵심은, 보통 따로 써야 하는 도구들이 여기서는 **처음부터 연결된 하나**라는 점이에요.

- 웹사이트·랜딩페이지·세일즈 퍼널 빌더
- 고객관리(CRM) + 모든 채널 대화를 모으는 **통합 인박스**
- **예약 캘린더** (결제 연동)
- 이메일·문자·자동화 워크플로우
- **토스페이먼츠 원화 결제**·영수증
- 후기 요청·평판 관리

원래라면 채널톡·Calendly·스티비·아임웹·Zapier·알리고처럼 **7~8개 도구를 매달 따로 구독**(월 19만~40만 원)하고, 데이터를 손으로 옮기고, 어디서 문제가 생기는지 못 보던 일을 — **한 곳에서, 자동으로** 돌아가게 만든 거예요.

## 무엇이 다른가요?

- **운영자를 위한 운영 시스템입니다.** 마케터용 도구 모음이 아니라, 사업의 다섯 단계가 같은 데이터 위에 올라가 있어 어느 단계에서 봐도 같은 고객의 같은 정보가 보여요.
- **AI가 기반입니다.** 음성 AI·대화 AI·콘텐츠 AI·후기 답글 AI가 시스템 바탕에 깔려 있어, 잘 세팅하면 직접 손대는 시간이 크게 줄어듭니다.
- **한국에서 바로 쓸 수 있습니다.** 전 세계 수십만 명이 검증한 글로벌 운영 엔진을 기반으로, 한국 시장에 맞춰 **한국어 UI·토스페이먼츠 원화 결제·한국 사용자 커뮤니티**까지 자체 구축했어요. 검증은 글로벌에서 끝났고, 사장님은 그것을 처음부터 한국어로 쓰시는 거예요.

## 사업의 5단계를 한 흐름으로

1. **잠재고객 들이기 (Capture)** — 광고·인스타 DM·카톡·전화·명함 등 모든 유입을 하나의 CRM으로 자동 수집
2. **관계 다지기 (Nurture)** — 채널별로 흩어진 대화를 한 화면에 모으고, 상담 리마인드·노쇼 자동 재예약
3. **결제로 이어가기 (Close)** — 견적·청구·결제·계약까지 한 흐름으로, 결제까지의 마찰을 최소화
4. **팬으로 만들기 (Evangelize)** — 결제 직후 자동 후기 요청 + AI 후기 답글로 만족 고객을 마케터로 전환
5. **다시 깨우기 (Reactivate)** — 잠든 고객 DB를 자동 캠페인으로 다시 매출로

한마디로, **"사업이 사장님을 운영하는" 상태에서 "사장님이 사업을 위에서 내려다보는" 상태로** 바꿔주는 운영 OS예요.
