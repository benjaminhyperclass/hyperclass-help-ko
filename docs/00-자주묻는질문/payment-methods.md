# 어떤 결제 수단을 지원하나요?

결제는 두 가지로 나눠 생각하시면 쉬워요: **① 하이퍼클래스 구독료 결제**, **② 내 고객에게 받는 결제.**

**① 하이퍼클래스 구독료 (사장님 → 하이퍼클래스)**

- **신용카드·체크카드**로 결제됩니다. 무료 체험 시작 시 카드를 등록하시면, 체험 종료 후 선택하신 요금제로 자동 청구돼요. (→ [무료 체험](free-trial.md))
- 월 구독료 + 사용량 기반 항목(AI·메시지 등)이 함께 청구됩니다. (→ [사용량 비용](usage-costs.md))

**② 내 고객에게 받는 결제 (고객 → 사장님)**

**사장님 본인 명의 토스페이먼츠**를 연동하기 때문에, **토스페이먼츠가 제공하는 모든 전자결제수단을 그대로** 쓰실 수 있어요. 토스 가맹점 계약에서 활성화하신 수단이 고객 결제창에 그대로 노출됩니다:

- **신용·체크카드** (국내/해외 카드)
- **간편결제** — 토스페이 · 네이버페이 · 카카오페이 · 페이코 등
- **계좌이체** (실시간 출금이체)
- **가상계좌** (무통장입금) — 현금영수증 발행 가능
- **휴대폰 결제**
- **상품권** (문화·도서문화·게임문화상품권)

대금은 중간 플랫폼을 거치지 않고 **사장님 계좌로 직접** 정산되고, 예약 페이지·퍼널·청구서·Text-2-Pay 등에 결제를 붙여 **상담 → 결제**를 한 흐름으로 만들 수 있습니다. (→ [상담·미팅 예약](booking.md))
