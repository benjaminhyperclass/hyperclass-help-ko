# 요금제는 어떻게 구성되나요?

하이퍼클래스는 **스타터**와 **그로스** 두 가지 플랜으로 구성됩니다. 두 플랜 모두 **연락처(고객) 수는 무제한**이고, 연간 결제 시 **약 2개월치를 아끼실 수 있어요.**

| 요금제 | 월 요금 (연간 결제 시) | 적합한 분 |
|---|---|---|
| **스타터** | ₩99,000/월 (연 ₩948,000) | 혼자, 또는 사장님 + 조수 1~2명이 함께 일하는 단계 |
| **그로스** | ₩159,000/월 (연 ₩1,528,000) | 운영·마케팅·세일즈가 분업되기 시작한 팀 단계 |

**스타터 — 포함 기능**: CRM & 통합 인박스 · 랜딩/퍼널/웹사이트 빌더 · 예약 캘린더(결제 통합) · 이메일 캠페인·워크플로우 자동화 · 토스페이먼츠 원화 결제 · 후기·평판 관리 · **사용자 3명 · 연락처 무제한**

**그로스 — 스타터의 모든 기능 + 추가**: 고급 자동화 워크플로우 · 전문 커뮤니티 플랫폼 · 퍼널·캘린더·웹사이트 **무제한** · A/B 테스트 · 고급 분석 대시보드 · **사용자 10명 · 연락처 무제한**

**한눈에 비교**

| 기능 | 스타터 | 그로스 |
|---|:---:|:---:|
| 핵심 도구 (CRM · 랜딩 · 캘린더 · 이메일 · 결제 · 웹사이트) | ✓ | ✓ |
| 토스페이먼츠 원화 결제 | ✓ | ✓ |
| 고급 자동화 워크플로우 | — | ✓ |
| 전문 커뮤니티 플랫폼 | — | ✓ |
| 퍼널 · 캘린더 · 웹사이트 무제한 | — | ✓ |
| A/B 테스트 | — | ✓ |
| 사용자 수 | 3명 | 10명 |

> **추가 비용:** 기본 구독에 핵심 기능은 다 포함되지만, AI 음성통화·대량 SMS·WhatsApp·도메인 구매·일부 고급 AI 같은 통신비 성격의 기능은 **사용한 만큼** 추가됩니다(휴대폰 통화료처럼). 자세한 단가는 [사용량 비용](usage-costs.md) 참고.

> **참고:** 도구를 따로 쓰면(채널톡·Calendly·스티비·아임웹·Zapier·알리고 등 8개) 보통 월 19만~40만 원이 들고 서로 연결도 안 됩니다.
