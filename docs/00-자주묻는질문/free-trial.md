# 무료로 먼저 써볼 수 있나요? 신용카드가 필요한가요?

네! **15일 무료 체험**을 제공합니다. 가입 시 원하시는 요금제(**스타터** 또는 **그로스**)를 고르시면, 체험 기간 동안 **그 플랜의 기능을 제한 없이** 사용해 보실 수 있어요. 실제 고객·예약·결제 데이터로 다섯 단계(유입 → 관계 → 결제 → 후기 → 재구매)를 직접 돌려보시면 됩니다.

**신용카드는 등록이 필요합니다.** 몰래 결제하려는 게 아니라, 체험이 끝났을 때 마음에 드시면 끊김 없이 바로 이어서 쓰실 수 있게 하기 위한 거예요. **등록 후 15일 동안은 한 푼도 청구되지 않습니다.**

## 진행 순서

1. **가입 (무료)** — 이메일·휴대폰으로 계정 생성. 한국어 화면에서 바로 시작.
2. **요금제 선택 + 카드 등록 (15일간 청구 없음)** — 스타터/그로스 중 선택하고 카드를 등록하되, 15일간은 청구되지 않습니다.
3. **15일 무료 체험** — 선택하신 플랜의 기능을 제한 없이 사용. 시스템을 실제로 구축하고 운영해 보세요.
4. **14일째 알림** — "내일부터 청구가 시작됩니다, 계속 쓰시겠어요?" 알림이 자동으로 발송됩니다. 모르고 결제되는 일이 없어요.
5. **계속 / 멈춤 선택** — 계속 쓰시면 자동으로 이어지고, 안 맞으시면 그냥 멈추시면 됩니다. 전화 상담이나 복잡한 해지 절차가 없어요.

## 체험 중 알아두실 점 (중요)

- **선택한 플랜 기준입니다.** 스타터로 체험하시면 스타터 기능을, 그로스로 체험하시면 그로스 기능을 쓰시게 돼요. (예: 고급 자동화·A/B 테스트·전문 커뮤니티·무제한 퍼널/캘린더/웹사이트는 **그로스 플랜** 기능입니다.)
- **사용량 기반 항목은 체험에서 사용할 수 없어요.** AI 음성통화, 대량 문자(SMS) 발송, WhatsApp 메시지, 도메인 구매, 일부 고급 AI처럼 '쓴 만큼 과금되는' 통신비 성격의 기능은 **정식 구독을 시작하신 뒤에 활성화**됩니다. 체험 기간에는 시스템을 구축·설정하는 데 집중하시고, 실제 발송·구매는 구독 후에 켜시면 돼요.

**체험이 끝나고 계속 쓰시면**, 선택하신 요금제로 자동 이어집니다 (스타터 ₩99,000/월 · 그로스 ₩159,000/월).

## 체험 기간(15일)을 이렇게 활용하세요

- **5일 챌린지로 시작하기** — 가입하시면 15일 무료 체험이 시작되고, 그 기간 동안 **유튜브 5일 챌린지**를 함께 따라오시면 됩니다. 매일 한 단계씩 직접 세팅하며 **내 비즈니스를 온라인에 정착**시켜요. 5일 차에는 **비즈니스 온라인 세팅 → 고객 예약 받기 → 결제(PG) 연결 준비**까지 갖춰집니다. 즉 **체험 15일 중 첫 5일이면** 고객이 들어와 예약·결제까지 이어지는 기본 흐름이 만들어져요. (챌린지는 서비스 출시와 동시에 오픈 예정)
- **마이그레이션 템플릿** — 기존 고객 DB·캘린더·결제 내역을 표준 포맷으로 가져오기
- **헬프센터 + 24시간 AI 챗봇** — 한밤중에 막혀도 자동 응답

> 한마디로, **결과를 먼저 보신 뒤에 결제**하시는 구조예요. (이미 청구가 시작된 뒤의 환불은 [환불정책](plan-change-refund.md)을 따릅니다.)
