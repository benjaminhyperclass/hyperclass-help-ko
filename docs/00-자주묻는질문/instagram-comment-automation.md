# 인스타그램 댓글 자동화·자동 DM, 여러 계정도 되나요?

핵심부터 말씀드리면, **인스타그램 계정 1개 기준으로는 완전히 지원됩니다.** 다만 여러 계정을 하나로 묶어 한 번에 자동화하는 것은 현재 제약이 있어, 인스타그램 1개당 서브계정 1개를 연동하는 방식으로 운영하셔야 합니다.

## 단일 계정 자동화 — 지원됩니다

특정 키워드가 담긴 댓글이 달리면 → 자동 답글 작성 → 해당 사용자에게 개인 DM 발송까지, 하이퍼클래스 워크플로우로 그대로 구현됩니다. 흔히 쓰는 "댓글에 OO 입력하면 DM 발송" 방식이 여기에 해당합니다.

## 여러 계정을 운영하는 경우

하나의 서브계정에 인스타그램 계정을 여러 개 연결하더라도, 자동화가 "어느 계정에서 들어온 댓글·DM인지"를 구분하지 못합니다. 그래서 계정마다 다른 답글·DM을 보내기 어렵습니다. 이는 특정 도구만의 한계가 아니라 현재 자동화 시스템 전반의 공통 제약입니다.

따라서 인스타그램 계정을 여러 개 운영하신다면, 아래 두 가지 방식 중 하나를 권장합니다.

### 방법 A — 인스타그램 1개당 서브계정 1개 연동 (간단)

운영 중인 인스타그램 계정마다 서브계정을 하나씩 만들어 각각 자동화를 설정합니다. 가장 빠르게 시작할 수 있으며, 계정별로 깔끔하게 분리되어 관리됩니다. 다만 계정 수만큼 설정·관리 지점이 늘어납니다.

### 방법 B — 자체 미들웨어 구축 (통합 관리)

여러 계정을 하나의 시스템에서 통합 관리하고 싶다면, Meta(인스타그램)의 웹훅을 직접 연동하는 방식을 권장합니다. Meta가 보내는 데이터에는 출처 계정 정보가 포함되어 있어, 들어온 댓글·DM이 어느 계정에서 왔는지 식별하고 계정별 맞춤 응답을 보낼 수 있습니다.

자체 구축 시 필요한 구성 요소는 다음과 같습니다.

- Meta 개발자 앱 및 인스타그램 권한 (메시지 관리·댓글 관리)
- 비즈니스 인증된 인스타그램 프로페셔널(비즈니스/크리에이터) 계정
- 웹훅 수신 서버 (댓글·멘션·메시지 이벤트 구독)
- 출처 계정 식별 → 응답 생성(AI 연동 가능) → 답글·DM 발송 로직
