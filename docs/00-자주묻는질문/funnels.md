# 웨비나/전략세션 퍼널(랜딩 → 신청 → 알림)을 만들 수 있나요?

네, 잠재고객을 모으는 퍼널 전체를 **코딩 없이** 만들 수 있어요. 드래그앤드롭 페이지 빌더와 템플릿이 기본 제공됩니다. 목적에 따라 두 가지 흐름이 있어요.

**A. 전략세션(1:1 상담) 퍼널**

1. **랜딩 페이지** — 무료 전략세션 신청 페이지를 템플릿에서 골라 만듭니다.
2. **신청·예약** — 신청 폼 작성 → 캘린더에서 시간 선택 → CRM에 고객으로 자동 등록 (→ [상담·미팅 예약](booking.md))
3. **자동 알림** — 신청 확인 + 리마인드가 **이메일로 자동 발송** (→ [예약 확정·리마인드 알림](reminders.md))
4. **사후 관리** — 참석/노쇼 여부에 따라 후속 메시지를 자동으로 분기 발송

**B. 웨비나(다수 대상) 퍼널**

1. **등록 페이지** — 웨비나 신청을 받는 랜딩을 만듭니다.
2. **등록** — 등록자 정보가 CRM에 자동 저장되고, 일정이 등록자 캘린더에 추가됩니다.
3. **시청** — 웨비나 방송은 **유튜브 라이브/Zoom을 하이퍼클래스 페이지에 임베드**해 진행합니다. (라이브 방송 자체는 자체 호스팅하지 않고, **등록·페이지·알림·참석 추적**을 담당해요.)
4. **자동 알림 + 사후** — 시작 전 리마인드, 종료 후 **리플레이 링크·후속 제안**을 자동 발송 (현재 이메일 기준 → [예약 확정·리마인드 알림](reminders.md))

**직접 만들기 vs 세팅 지원**

- 모든 페이지·폼·자동화는 **사장님이 직접** 템플릿으로 구성할 수 있어요. 가입하시면 15일 무료 체험이 시작되고, 그 기간에 **유튜브 5일 챌린지**를 따라오시면 비즈니스를 온라인에 정착시키는 한 줄기를 완성할 수 있습니다. (→ [무료 체험](free-trial.md))

> ⚠️ **카카오 알림톡 발송은 별도 연동 서비스입니다.** 위 자동 알림은 기본 채널(이메일) 기준이에요. (→ [카카오 채널 연동](kakao-channel.md), [알림톡 템플릿 등록](alimtalk.md))
