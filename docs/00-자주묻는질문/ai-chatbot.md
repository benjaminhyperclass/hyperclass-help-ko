# 챗봇으로 들어온 고객 문의에 자동 응답할 수 있나요?

네! 하이퍼클래스에는 **AI 챗봇이 기본 내장**돼 있어, 코딩 없이 챗봇을 만들고 학습시켜 문의에 자동 응답하게 할 수 있어요.

## 어디서 응답하나요?  (채널과 현재 상태)

- **웹사이트 채팅 위젯** — 바로 사용 가능 (홈페이지·랜딩에 삽입하는 채팅창)
- **페이스북 메신저 · 인스타그램 DM** — 해당 계정 연동 시 사용
- **문자(SMS)** — 한국은 SK브로드밴드 발신번호 신청·검증 후 사용 (현재 준비 중)
- **WhatsApp** — 해외 고객 대상, 별도 설정 필요

들어온 대화는 모두 **통합 인박스 한 곳**에 모여, 어느 채널이든 같은 화면에서 관리됩니다.

## 어떻게 똑똑해지나요? (학습 방식)

- 챗봇에게 **웹사이트 URL·FAQ·구글 문서·PDF·DOCX·CSV(가격표 등)**를 학습시키면, 그 내용을 바탕으로 답합니다.
- "영업시간 알려주세요" 같은 자주 묻는 질문은 **질문–답변을 직접 입력**해 정확도를 높일 수 있어요.
- 도움말·자료를 업데이트하면 **답변 품질도 함께 좋아집니다.** (팁: URL만 넣기보다 FAQ·정책을 정리한 '단일 기준 문서'를 학습시키면 엉뚱한 답변이 줄어요.)

## 응답 방식은 두 가지 중 선택

- **자동 응답(Auto-Pilot)** — 챗봇이 즉시 자동으로 답합니다. 24시간 1차 응대에 적합.
- **초안 제안(Suggestive)** — 챗봇이 답변 초안을 만들어 두면, 사장님이 검토·수정 후 발송합니다. 고가 상담처럼 사람이 한 번 더 확인하고 싶을 때 적합.

채널별로 **어디는 자동, 어디는 수동**으로 따로 켤 수도 있어요. 챗봇이 답하기 어려운 복잡한 문의는 **사람 상담으로 연결**할 수 있습니다. ([챗봇이 답을 못 할 때](chatbot-handover.md) 참고)

> **따라 하기:** 설정에서 챗봇을 켜고 → 웹사이트/FAQ/문서로 학습 → '봇 트라이얼'로 테스트 → 채널과 응답 모드를 지정해 활성화. (설정·학습·테스트는 무료이며, 외부 채널에 실제로 배포해 메시지를 주고받을 때부터 사용량 과금이 발생합니다. [사용량 비용](usage-costs.md) 참고)

> ⚠️ **카카오톡(카카오 채널) 연동은 별도 서비스입니다.** 위 자동 응답은 하이퍼클래스가 기본 지원하는 채널(웹챗·페이스북·인스타그램·문자·WhatsApp) 기준이에요. **카카오톡 채널 자동 응답은 기본 구독에 포함되지 않은 별도 연동/추가 서비스**로 제공됩니다. ([카카오톡 채널 연동](kakao-channel.md) 참고)
