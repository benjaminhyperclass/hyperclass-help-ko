# Hyperaxis 개인정보처리방침

> **시행일**: 2026년 6월 24일
> **운영사**: 하이퍼클래스 (이하 "회사")
> **서비스명**: Hyperaxis
> **문의**: geunha2025@naver.com

회사는 「개인정보 보호법」, 「정보통신망 이용촉진 및 정보보호 등에 관한 법률」(이하 "정보통신망법"), Meta Platform Terms 및 관계 법령을 준수하여 이용자의 개인정보를 보호합니다.

---

## 제01조 (개인정보의 처리 목적)

회사는 다음 목적으로 개인정보를 처리합니다. 처리한 개인정보는 아래 명시된 목적 이외의 용도로 사용하지 않으며, 목적이 변경될 경우 별도의 동의를 받겠습니다.

| 처리 목적 | 상세 내용 |
|---|---|
| 서비스 제공 | Meta 광고 인사이트 수집·분석, 단위경제 지표(CAC·LTV·AOV·ROAS) 자동 산출 |
| 계정 연동 및 인증 | GHL 서브계정 연결, Meta 시스템 사용자 토큰(BYOT) 암호화 저장 및 갱신 |
| 이용자 식별 및 접근 제어 | 멀티테넌트 환경에서의 테넌트별 데이터 격리 |
| 서비스 유지·개선 | 오류 탐지, 성능 모니터링, 이용 통계(집계·비식별) |
| 법적 의무 이행 | 세금계산서 발행, 분쟁 해결, 규제 기관 요청 대응 |

---

## 제02조 (처리하는 개인정보의 항목)

### 2-1. 이용자가 직접 제공하는 항목

| 연동 구분 | 수집 항목 |
|---|---|
| Meta 연동 (BYOT) | Meta 시스템 사용자 토큰, Meta 비즈니스 계정 ID, 광고 계정 ID(act\_xxxxxxx) |
| GHL 연동 | GHL Location ID, GHL Company ID |
| 서비스 가입 | 이름, 이메일 주소, 연락처(선택) |

### 2-2. API를 통해 자동 수집되는 항목

| 출처 | 수집 항목 |
|---|---|
| Meta Marketing API | 캠페인·광고세트·광고 ID 및 이름, 노출수, 클릭수, 일별 광고비, 전환수, 동영상 조회 지표(video\_continuous\_2\_sec\_watched\_actions 등), 광고 상태 |
| GHL API | 결제 주문 금액·건수·날짜(contactId 포함 가능, 집계 후 비식별 처리), Location 설정 정보 |
| 서비스 로그 | IP 주소, 접속 일시, 브라우저/기기 정보, 서비스 이용 기록 |
| 쿠키·세션 | 로그인 상태 유지용 세션 식별자, 환경 설정 쿠키 |

### 2-3. 수집하지 않는 항목

- 광고에 노출된 최종 소비자(광고 수신자)의 개인정보
- Meta 사용자 프로필·메시지·친구 목록·개인 식별 정보
- 결제 카드 번호·계좌번호 등 금융 정보 (결제는 GHL 마켓플레이스를 통해 처리)
- 만 14세 미만 아동의 개인정보

## 제03조 (개인정보의 보유·이용 기간)

| Meta 광고 인사이트 데이터 | 서비스 해지 후 30일 이내 파기 | 처리 목적 달성 |
| GHL 매출·주문 데이터 | 서비스 해지 후 30일 이내 파기 | 처리 목적 달성 |
| 서비스 접속 로그 | 6개월 | 통신비밀보호법 제41조 |
| 전자상거래 결제·공급 기록 | 5년 | 전자상거래법 제6조 |
| 소비자 불만·분쟁 처리 기록 | 3년 | 전자상거래법 제6조 |

법령에 따른 보관 의무가 있는 항목은 해당 기간 동안 다른 정보와 **분리하여 안전하게 보관**한 뒤 파기합니다.

## 제04조 (개인정보의 제3자 제공)

회사는 원칙적으로 이용자의 개인정보를 외부에 제공하지 않습니다. 다만, 다음의 경우에는 예외적으로 제공할 수 있습니다.

1. 이용자가 사전에 동의한 경우
2. 법령 규정에 따른 수사기관·감독기관의 요청이 있는 경우
3. 서비스 이용 계약 이행을 위해 불가피한 경우

회사는 원활한 서비스 제공을 위해 다음과 같이 개인정보 처리 업무를 위탁합니다.

## 제06조 (개인정보의 국외 이전)

제05조의 수탁사가 미국 소재 법인임에 따라 이용자의 정보 일부가 국외로 이전될 수 있습니다.

| 이전 대상국 | 이전 사업자 | 이전 항목 | 이전 목적 | 이전 방법 | 보유 기간 |
|---|---|---|---|---|---|
| 미국 | Meta Platforms, Inc. | 광고 계정 ID, 캠페인·광고 인사이트 | 광고 성과 조회 | HTTPS API 호출 | 서비스 이용 기간 |
| 미국 | HighLevel LLC | Location ID, 매출 데이터 | CRM 데이터 조회 | HTTPS API 호출 | 서비스 이용 기간 |
| 미국 | Railway | 서비스 운영 데이터 전반 | 서버 호스팅 | 암호화 전송 | 서비스 이용 기간 |

이용자는 국외 이전을 거부할 권리가 있으나, 거부 시 서비스 이용이 불가능할 수 있습니다. 거부 의사는 **geunha2025@naver.com**으로 알려 주시기 바랍니다.

## 제07조 (이용자의 권리·의무 및 행사 방법)

이용자는 회사에 대해 언제든지 다음 권리를 행사할 수 있습니다.

1. **열람권**: 보유 중인 개인정보 확인 요청
2. **정정·삭제권**: 부정확한 정보 수정 또는 삭제 요청
3. **처리 정지권**: 개인정보 처리 중단 요청
4. **이동권**: 개인정보를 기계가독형 형태로 이전 요청 (개인정보 보호법 제35조의2)

**행사 방법**: **geunha2025@naver.com**으로 이메일 발송 (제목: "개인정보 [열람/정정/삭제/처리정지] 요청")

회사는 요청일로부터 **영업일 기준 10일 이내** 처리 결과를 통보합니다.  
단, 법령상 보관 의무가 있는 정보(제03조 참고)는 해당 기간 만료 전에는 삭제되지 않을 수 있습니다.

법정 대리인이 미성년자를 대신하여 권리를 행사하는 경우, 법정 대리인임을 증명하는 서류를 함께 제출하여 주시기 바랍니다.

- **전자 파일**: 복구 불가능한 방식으로 영구 삭제 (논리적 삭제 후 스토리지 덮어쓰기)
- **Meta 연동 토큰**: 서비스 해지 즉시 암호화 저장소에서 삭제
- **출력물**: 분쇄기 파기 또는 소각

## 제10조 (쿠키 및 자동 수집 장치)

회사는 서비스 운영을 위해 쿠키(Cookie) 및 세션을 사용합니다.

- **사용 목적**: 로그인 상태 유지, 보안 세션 관리, 이용자 환경 설정 저장
- **저장 기간**: 세션 종료 시 또는 최대 24시간
- **거부 방법**: 브라우저 설정(도구 > 인터넷 옵션 > 개인정보 > 고급)에서 쿠키 저장 거부 가능. 단, 거부 시 로그인 등 서비스 이용에 제한이 생길 수 있습니다.

## 제11조 (개인정보의 안전성 확보 조치)

회사는 「개인정보 보호법」 제29조에 따라 다음 기술적·관리적 조치를 시행합니다.

| 구분 | 조치 내용 |
|---|---|
| 암호화 | Meta 토큰·시크릿은 Fernet 대칭 암호화로 저장, 전송 구간 HTTPS/TLS 적용 |
| 접근 제한 | DB 레벨 Row-Level Security(RLS)로 테넌트 간 데이터 완전 격리, 최소 권한 원칙 |
| 코드 보안 | CI 파이프라인 의존성 취약점 스캔, 비밀값 코드·로그 평문 저장 금지 |
| 감사 로그 | 데이터 접근·변경 이력 기록 및 이상 탐지 |
| 인적 관리 | 개인정보 취급자 최소화, 교육 및 서약 |

## 제12조 (만 14세 미만 아동의 개인정보)

본 서비스는 만 14세 미만 아동을 대상으로 하지 않습니다. 만 14세 미만 아동의 개인정보가 수집된 사실을 인지하는 즉시 해당 정보를 삭제합니다. 이러한 상황을 발견하신 경우 **geunha2025@naver.com**으로 알려 주시기 바랍니다.

---

## 제13조 (Meta 플랫폼 관련 특칙)

본 서비스는 Meta Platforms, Inc.의 API를 활용합니다.

- 이용자가 제공한 Meta 시스템 사용자 토큰은 광고 인사이트 조회 목적으로만 사용되며, Meta에 재공유되지 않습니다.
- 수집된 광고 데이터는 해당 이용자의 광고 계정 성과 분석 외의 목적(예: 타 이용자와의 벤치마크 공유, Meta 외 광고 플랫폼 최적화)에 사용하지 않습니다.
- Meta 계정 설정에서 본 서비스(Hyperaxis)의 앱 연결을 직접 해제하는 경우, 이는 데이터 삭제 요청으로 처리됩니다. 자세한 절차는 [개인정보(데이터) 삭제 요청](data-deletion.md) 페이지를 참고하세요.

이전 버전의 개인정보처리방침은 문의 이메일로 요청하시면 확인하실 수 있습니다.
