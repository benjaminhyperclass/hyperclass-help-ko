# 여러 개의 Facebook 페이지를 계정에 연결하기

이 기능은 현재 Labs에서 사용할 수 있습니다. Labs에서 이 기능을 활성화하여 여러 Facebook 페이지를 계정에 연결해보세요.

## 목차

- [이 기능으로 무엇을 할 수 있나요?](#이-기능으로-무엇을-할-수-있나요)
- [여러 페이지를 연결하는 방법](#여러-페이지를-연결하는-방법)
- [Facebook 연동 - 페이지 관리](#Facebook-연동---페이지-관리)

## 이 기능으로 무엇을 할 수 있나요?

- **리드 동기화**: 여러 페이지의 Facebook 리드 광고에서 리드를 자동으로 동기화하고 출처를 명확하게 구분할 수 있습니다
- **대화**: Facebook과 Instagram 채팅 연동을 사용하여 여러 Facebook 페이지의 고객과 대화할 수 있습니다
- **자동화**: 여러 Facebook 페이지에서 고객 참여를 자동화하는 맞춤형 워크플로우를 구축할 수 있습니다
- **평판 관리**: 여러 Facebook 페이지의 리뷰와 답변을 관리할 수 있습니다

## 여러 페이지를 연결하는 방법

**1단계**: Labs에서 이 기능을 활성화한 후, `Settings(설정) > Integrations(연동)`으로 이동합니다.

![여러 Facebook 페이지 연결 - 설정 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155035059804/original/e7FSaLLnj5xgfPe-kLFHCYP1HqO1yZpgNg.png?1729488858)

**2단계**: 'Connect(연결)' 버튼을 클릭하고, Facebook 계정에 로그인한 후 LeadConnector가 Facebook 계정과 연동하는 데 필요한 모든 권한을 승인합니다.

![Facebook 계정 연결 및 권한 승인](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155035059933/original/PFZn4-GyKGBJVee4HhFBvtyC8LgeW06_sQ.png?1729489096)

**3단계**: 연동하고 싶은 페이지를 선택합니다.

새 리드만 동기화할지, 아니면 지난 90일간의 리드도 함께 동기화할지 선택할 수 있습니다.

![페이지 선택 및 리드 동기화 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155035059978/original/ObtRkrwAWQ4QIwBbenlp5s7nZYLBYDNfvA.png?1729489220)

**4단계**: Facebook 리드 광고 폼 필드를 연락처의 커스텀 필드에 매핑합니다.

![리드 광고 폼 필드 매핑](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155035061463/original/9ikL0ynstvJ3pblYo6uqFXBU58cQtIbY1Q.png?1729491521)

![폼 필드 매핑 세부 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155035061456/original/Bld6_kC9DKsXWu4gjY-UEMIWNXw8tlsCWg.png?1729491507)

![매핑 완료 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155035061604/original/Bob8Y6YN09Zz5kOaOEOWIXQbvgWwzERZeg.png?1729491671)

## Facebook 연동 - 페이지 관리

'Manage pages(페이지 관리)'를 클릭하여 연동하고 싶은 페이지를 관리할 수 있습니다. 원하는 페이지를 선택하거나 선택 해제할 수 있습니다. 예를 들어, 특정 페이지를 연동하고 싶지 않다면 페이지 관리 메뉴에서 해당 페이지의 선택을 해제할 수 있습니다.

![페이지 관리 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155035061583/original/JJwmX6ofRx6Aa8-svzhs6h6YE0HxUM1cZA.png?1729491657)

![페이지 선택/해제 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155035061822/original/DCZoijxdN99Pur8rT5m3XwCyJhH2mS90pQ.png?1729491950)

## Facebook 연동 - 설정

Settings(설정) 버튼을 사용하여 연동된 페이지들의 메시징 기능을 관리할 수 있습니다. 예를 들어, 특정 페이지에서 메시징/채팅 기능을 사용하고 싶지 않다면 해당 페이지의 설정을 비활성화할 수 있습니다.
