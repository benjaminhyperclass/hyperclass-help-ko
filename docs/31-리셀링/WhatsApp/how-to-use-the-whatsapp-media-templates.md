# WhatsApp 미디어 템플릿 사용 방법

**모바일 앱에서 WhatsApp 미디어 템플릿 사용 가능**

**모바일 앱**에 새로운 강력한 기능인 **WhatsApp 미디어 템플릿 지원**이 추가되었습니다! 🎉

하위 계정(Sub-account) 관리자와 사용자는 이제 사전 승인된 템플릿을 사용하여 모바일 앱에서 직접 **미디어가 포함된 WhatsApp 메시지**를 보낼 수 있습니다. 이 기능은 **Hyperclass, LeadConnector** 및 **화이트라벨** 모바일 앱에서 사용할 수 있습니다(화이트라벨 앱 사용자는 업데이트를 요청해야 함).

---

## 새로운 기능

**다음과 같은 WhatsApp 미디어 템플릿을 지원합니다:**

• **텍스트**
• **이미지**  
• **동영상**
• **파일**
• **위치 공유**

---

## 사용 방법

1. 모바일 앱을 **열고** 원하는 하위 계정을 선택합니다.

2. **대화(Conversations)** 섹션으로 이동하여 새로운 WhatsApp 대화를 시작하거나 기존 대화를 계속합니다.

3. 적절한 **템플릿 유형**(텍스트, 이미지, 동영상, 파일 또는 위치)을 선택합니다.

4. 콘텐츠를 완성하고 **전송을 누르면** 끝입니다!

---

## 왜 중요한가요

• 이동 중에도 더욱 풍부하고 매력적인 고객 소통이 가능합니다.
• 데스크톱 환경 밖에서도 승인된 템플릿을 사용하여 시간을 절약할 수 있습니다.
• WhatsApp을 통한 모바일 지원, 영업, 후속 조치를 더욱 효율적으로 처리할 수 있습니다.

---

## 자주 묻는 질문(FAQ)

**Q1: 이 기능을 사용하려면 앱을 업데이트해야 하나요?**

**A:** 네. Hyperclass 또는 LeadConnector 앱이 최신 버전으로 업데이트되어 있는지 확인하세요. 화이트라벨 앱의 경우, 계정 담당자를 통해 업데이트를 요청해야 합니다.

**Q2: WhatsApp 템플릿에서 어떤 미디어 유형을 지원하나요?**

**A:** 모바일 앱에서 직접 미디어 템플릿에 **텍스트, 이미지, 동영상, 파일, 위치**를 사용할 수 있습니다.

**Q3: 앱에서 템플릿을 생성하거나 편집할 수 있나요?**

**A:** 아니요. 템플릿 생성과 승인은 여전히 웹 플랫폼을 통해 수행해야 합니다. 모바일 앱에서는 사전 승인된 템플릿을 **전송**할 수 있습니다.

**Q4: 기존 템플릿이 모바일 앱에서 작동하나요?**

**A:** 네. 모든 기존 **승인된 미디어 템플릿**이 모바일에서 사용 가능합니다.

**Q5: 이 기능은 하위 계정의 모든 사용자가 사용할 수 있나요?**

**A:** 네. 하위 계정에 접근 권한이 있는 **관리자와 일반 사용자** 모두 모바일을 통해 WhatsApp 미디어 템플릿을 전송할 수 있습니다.
