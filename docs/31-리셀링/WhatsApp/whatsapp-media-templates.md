# 왓츠앱 미디어 템플릿이란?

왓츠앱 미디어 템플릿을 사용하면 이미지, 비디오, 문서, 위치 데이터 등 풍부한 미디어 콘텐츠를 연락처나 고객에게 보낼 수 있습니다. 이 기능으로 왓츠앱에서 더욱 매력적이고 시각적으로 인상적인 메시지를 만들 수 있어 커뮤니케이션의 가능성이 크게 확장됩니다.

# 왓츠앱 미디어 템플릿 활용 사례

미디어 템플릿으로 왓츠앱 마케팅 캠페인을 한 단계 업그레이드하세요:

- **신제품 출시**: 이미지와 눈길을 끄는 비디오로 새로운 제품을 소개하세요.
- **프로모션과 할인 혜택**: 할인 코드나 특별 혜택을 시선을 끄는 비주얼과 함께 공유하세요.
- **이벤트 공지**: 위치 공유와 매력적인 이미지로 이벤트를 홍보하세요.
- **고객 서비스**: 도움이 되는 튜토리얼이나 가이드를 문서 형태로 제공하세요.

**1단계:** Settings(설정) > WhatsApp(왓츠앱) > Templates(템플릿) > Create Template(템플릿 만들기)로 이동하세요

**3단계:** **Create(만들기)** 버튼을 누르면 > 템플릿이 Meta로 승인 요청이 전송됩니다

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155025237581/original/viICB0j-AIQVIaVcuZCaqp3xS6lvkTFTKw.png?1714395178)

# 위치 왓츠앱 템플릿 만들기

**1단계:** Settings(설정) > WhatsApp(왓츠앱) > Templates(템플릿) > Create Template(템플릿 만들기)로 이동하세요

# 미디어 제한사항

| 미디어 | 지원 형식 | 크기 제한 |
|--------|-----------|----------|
| document(문서) | application/pdf | 100MB |
| image(이미지) | image/jpeg, image/png<br>이미지는 8비트, RGB 또는 RGBA여야 함 | 5MB |
| video(비디오) | video/mp4 | 16MB |

크기 제한을 초과하는 미디어 파일을 업로드하려고 하면 아래와 같은 오류가 표시됩니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155025293681/original/dneNh2U1Beor1QfgQUEKLroi708-uY4ZzA.png?1714473547)

# 자주 묻는 질문

## 왓츠앱 미디어 템플릿의 미디어 제한사항은 어떻게 되나요?

지원되는 형식과 크기 제한은 다음과 같습니다:

- 문서: application/pdf, 100MB
- 이미지: image/jpeg, image/png (이미지는 8비트, RGB 또는 RGBA여야 함), 5MB  
- 비디오: video/mp4, 16MB

크기 제한을 초과하는 미디어 파일을 업로드하려고 하면 오류 메시지가 표시됩니다. 예를 들어, 이미지 템플릿에 비디오 파일을 추가하려고 하면 오류 메시지를 받게 됩니다.

## 왓츠앱 미디어 템플릿이란 무엇인가요?

왓츠앱 메시지 내에 이미지, 비디오, 문서 또는 위치 정보를 포함할 수 있는 사전 승인된 메시지 형식입니다. 특히 마케팅, 고객 지원 또는 인증 목적으로 고객과의 커뮤니케이션을 효율화하는 데 도움이 됩니다.

## 가이드라인에 맞지 않는 미디어를 사용하면 어떻게 되나요?

오류 메시지가 표시되고 템플릿이 전송되지 않을 가능성이 높습니다. 미디어가 [왓츠앱 템플릿 가이드라인](https://developers.facebook.com/docs/whatsapp/message-templates/guidelines/)을 준수하는지 확인하세요.

## 왓츠앱 메시지 템플릿을 사용하기 전에 승인이 필요한가요?

네, 모든 템플릿은 특정 가이드라인을 충족하는지 확인하기 위해 왓츠앱(Meta)의 승인 과정을 거쳐야 합니다.

## 템플릿이 거부되는 이유는 무엇인가요?

일반적인 거부 사유는 다음과 같습니다:

- 변수 형식이 누락되거나 잘못됨
- [왓츠앱 비즈니스 또는 상거래 정책](https://developers.facebook.com/docs/whatsapp/overview/policy-enforcement/violations/)을 위반하는 콘텐츠
- 민감한 정보 요청 (예: 전체 결제 카드 번호)
- 남용적이거나 위협적인 언어
- 기존 템플릿과 중복

## 템플릿이 '일시정지'됐다는 것은 무엇을 의미하나요?

일시정지된 템플릿은 부정적인 고객 피드백이나 낮은 참여율을 받았다는 의미입니다. 상태가 다시 '활성'으로 변경될 때까지는 일시정지된 템플릿을 보낼 수 없습니다.
