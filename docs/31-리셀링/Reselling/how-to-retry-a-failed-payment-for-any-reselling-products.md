# 리셀링 상품 결제 실패 시 재시도하는 방법

상품 결제가 실패했나요? 걱정하지 마세요! 아래의 간단한 단계를 따라하면 쉽게 결제를 다시 시도할 수 있습니다.

### 결제가 실패하는 이유

결제 거래는 다양한 이유로 실패할 수 있습니다:

- **잔액 부족**
- **잘못된 카드 정보** (만료된 카드나 잘못된 CVV 등)
- **은행 제한** 또는 **보안 플래그**

이런 상황이 발생하면 결제를 다시 시도하라는 알림이 표시됩니다.

### **결제 재시도하기**

1. **상품 페이지로 이동**
   구독하고자 하는 상품 페이지를 방문하세요.

2. **결제를 위해 인증 정보 입력**
   
3. **"Retry Payment(결제 재시도)" 클릭**
   구매 페이지에서 "Retry Payment" 버튼을 찾으세요.
   버튼을 클릭하여 결제 과정을 다시 시작하세요.

![결제 재시도 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036139795/original/pW01VvpKfazWMFe763tjerTxGILxDDVFnA.jpeg?1730957431)

4. **필요시 결제 정보 업데이트**

- 잘못되거나 오래된 카드 정보로 인해 실패한 경우, 새로운 정보를 입력하라는 메시지가 표시됩니다.
- 결제가 성공적으로 진행되도록 모든 정보를 다시 확인한 후 진행하세요.

5. **결제 확인**

- 정보를 업데이트한 후 결제를 확인하세요.
- 결제가 승인되면 성공 알림을 받게 됩니다.

### **성공적인 결제를 위한 일반적인 해결책**

- **은행에 문의**하여 사용 중인 카드에 제한이 없는지 확인하세요.
- **계좌에 충분한 잔액이 있는지 확인**하세요.
- 재시도 옵션이 여전히 작동하지 않는다면 **다른 결제 방법을 고려**해보세요.

### **여전히 문제가 있나요?**

결제가 여전히 진행되지 않는다면, 언제든지 저희 고객지원팀에 문의하세요. 구매 완료를 도와드리겠습니다!
