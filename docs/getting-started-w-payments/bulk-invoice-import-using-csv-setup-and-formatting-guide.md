# CSV 파일을 이용한 인보이스 일괄 가져오기 설정 및 형식 가이드

CSV 파일로 인보이스를 가져오면 하나씩 수동으로 만들 필요 없이 여러 인보이스를 한 번에 빠르게 추가할 수 있습니다. 이 방법은 정확성을 유지하고 시간을 절약하며 청구 기록을 체계적으로 정리하는 데 도움이 됩니다. 올바른 형식을 사용하면 오류 없이 인보이스를 업로드할 수 있어요.

# **업로드 전 준비사항**

파일을 업로드하기 전에 CSV가 올바르게 형식화되어 있는지 확인하세요. 시스템은 정확한 열 이름과 유효한 데이터에 의존합니다. 잘못된 헤더나 누락된 값과 같은 작은 실수도 가져오기 실패의 원인이 될 수 있어요.

### **필수 필드 (정확히 일치해야 함)**

특정 필드는 필수이며 나열된 대로 정확히 포함되어야 합니다. 각 행은 이러한 필드에 대한 유효한 데이터를 포함해야 해요.

필수 필드는 다음과 같습니다:

- 인보이스 번호
- 인보이스 이름
- 고객 이름
- 고객 이메일
- 발행일
- 마감일
- 통화
- 항목 이름
- 항목 단가

이러한 필드는 인보이스의 핵심 구조를 정의하며 건너뛸 수 없습니다.

### **선택적 필드 (선택사항이지만 유용함)**

인보이스를 더욱 향상시키기 위해 추가 필드를 포함할 수도 있습니다:

- 결제 조건
- 항목 설명
- 수량
- 세금 세부사항
- 할인
- 결제 상태

이러한 필드는 선택사항이지만 더 나은 추적과 보고를 위해 유용해요.

## **CSV 형식 예시**

다음은 올바르게 형식화된 CSV의 간단한 예시입니다:

*InvoiceNo,*InvoiceName,*CustomerName,*CustomerEmail,*IssueDate,*DueDate,*Currency,*ItemName,*ItemRate INV001,Web Design,John Doe,john@email.com,03/01/2025,03/31/2025,USD,Homepage Design,300

## **인보이스 가져오는 방법**

![인보이스 가져오기 시작 위치](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155069414285/original/EyHlEOfww5HOt4S4uDlLkqYVabUpX826yQ.png?1776465298)

이 이미지는 인보이스 가져오기 과정을 시작하는 위치를 보여줍니다.

**Invoices(인보이스)** 섹션으로 이동하여 **New(새로 만들기)**를 클릭하고 **Import Invoices from CSV(CSV에서 인보이스 가져오기)**를 선택하세요.

**파일 업로드**

클릭하여 선택하거나 업로드 영역으로 끌어다 놓아 CSV 파일을 업로드하세요.

**필드 확인 및 매핑**

CSV 열이 시스템 필드와 어떻게 일치하는지 검토하세요. 모든 것이 올바르게 매핑되었는지 확인하세요.

**오류 수정 (있는 경우)**

오류가 나타나면 CSV 파일을 업데이트하고 모든 문제가 해결될 때까지 다시 업로드하세요.

**가져오기 완료**

모든 것이 올바르면 가져오기를 완료하세요. 인보이스가 생성되어 사용할 수 있게 됩니다.

---

## **피해야 할 일반적인 실수들**

대부분의 오류는 잘못된 열 이름, 누락된 필수 필드, 잘못된 날짜 형식 또는 중복된 인보이스 번호와 같은 간단한 문제에서 발생합니다. 이러한 문제를 방지하려면 업로드하기 전에 파일을 다시 한 번 확인하세요.

---

## **자주 묻는 질문**

**Q: 하나의 인보이스에 여러 항목을 추가할 수 있나요?**

네, 여러 항목을 포함하려면 여러 행에서 동일한 인보이스 번호를 사용하세요.

**Q: 필수 데이터가 누락되면 어떻게 되나요?**

가져오기가 실패하며 다시 시도하기 전에 파일을 수정해야 합니다.

**Q: CSV를 사용하여 기존 인보이스를 업데이트할 수 있나요?**

아니요, 이 과정은 새 인보이스만 생성합니다.

**Q: 선택적 필드를 반드시 포함해야 하나요?**

아니요, 선택적 필드는 비워둘 수 있습니다.

---

## **도움이 필요하신가요?**

가져오기가 작동하지 않으면 열 이름, 필수 필드, 날짜 형식을 확인해보세요. 그래도 문제가 해결되지 않으면 지원팀에 도움을 요청하세요.
