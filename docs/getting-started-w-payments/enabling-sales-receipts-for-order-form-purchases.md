# 주문 폼 구매에 대한 자동 판매 영수증 활성화하기

자동 판매 영수증은 수동 작업 없이 모든 고객이 구매 확인서를 받을 수 있도록 보장합니다. 이는 정확한 기록 관리, 고객 신뢰도 향상, 전문적인 구매 후 경험을 제공하는 데 도움이 됩니다. 간단한 설정과 유연한 커스터마이징으로 영수증을 비즈니스 요구사항에 완벽히 맞출 수 있어요.

## 자동 판매 영수증이란?

자동 판매 영수증은 거래가 완료된 직후 자동으로 구매 증명서를 생성하고 발송하는 기능입니다. 수동 작업을 없애면서도 모든 결제가 적절히 문서화되고 고객에게 전달되도록 보장해요.

## 주요 장점

- **자동화**: 구매 후 즉시 영수증 발송
- **일관성**: 모든 고객이 확인서를 받도록 보장
- **기록 관리**: 정확한 거래 내역 유지
- **유연한 지원**: 일반 결제, 업셀, 구독 결제 모두 지원
- **커스텀 브랜딩**: 영수증과 이메일 개인화

## 자동 영수증 활성화 방법

![자동 영수증 활성화 방법](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155069413806/original/67lx5g2ZgU5iYurc2eVGcximQ4e30lrRWA.gif?1776464028)

영수증 설정을 찾아서 토글을 켜서 모든 해당 거래에 대해 자동 영수증 전송을 활성화하세요.

## 영수증 커스터마이징

### 영수증 제목
브랜드에 맞는 커스텀 제목을 설정하세요.

### 영수증 번호 매기기
쉬운 추적을 위해 접두사를 추가하고 시작 번호를 정의하세요.

### 이메일 템플릿
일관된 브랜드 경험을 위해 영수증과 함께 발송되는 이메일 디자인과 메시지를 커스터마이징하세요.

## 영수증 전송 방식

- 결제 성공 후 자동 발송
- 고객 이메일로 전송
- 다운로드 가능한 PDF 영수증 포함

## 커스텀 템플릿 사용하기

커스텀 템플릿을 사용해서 브랜드화된 영수증 이메일을 만드세요. 레이아웃, 메시지, 전체적인 프레젠테이션을 제어할 수 있어요.

## 자주 묻는 질문

**Q: 자동 영수증은 어떻게 활성화하나요?**
결제 설정에서 영수증 토글을 켜세요.

**Q: 영수증 내용을 편집할 수 있나요?**
거래 세부사항은 자동 생성되지만, 제목, 번호 매기기, 이메일 디자인은 커스터마이징할 수 있어요.

**Q: 영수증 번호를 리셋할 수 있나요?**
번호를 증가시킬 수는 있지만, 영수증이 생성된 후에는 감소시킬 수 없어요.

**Q: 구독 결제도 포함되나요?**
네, 정기 결제에 대해서도 영수증이 생성됩니다.

## 도움이 필요하신가요?

- 자동 영수증이 활성화되어 있는지 확인하세요
- 고객 이메일이 유효한지 확인하세요
- 이메일 템플릿 설정을 확인하세요
- 결제가 성공했는지 확인하세요

문제가 계속되면 지원팀에 문의해서 도움을 받으세요.
