# 기회 커스텀 필드 사용하는 방법

기회 커스텀 필드는 정확한 영업 프로세스에 맞게 CRM을 맞춤 설정할 수 있는 유연성을 제공합니다. 커스텀 필드를 생성, 정리, 적용함으로써 영업 마감일과 고객 선호도부터 예산과 리드 소스까지 비즈니스에 가장 중요한 세부 사항을 추적할 수 있습니다. 이러한 필드는 필터링, 리포팅, 파이프라인 맞춤 설정과도 원활하게 연동되어 팀이 올바른 기회에 집중할 수 있도록 도와줍니다.

# 기회 커스텀 필드란 무엇인가요?

기회 커스텀 필드를 사용하면 HighLevel에서 제공하는 기본 필드 외에 영업 파이프라인에 추가 데이터 포인트를 추가, 관리, 정리할 수 있습니다. 이 기능은 파이프라인이 비즈니스 프로세스의 고유한 측면을 반영하도록 보장합니다.

텍스트, 드롭다운, 날짜, 금액 및 기타 필드 유형을 생성하고, 폴더로 분류하며, 더 깔끔한 보기를 위해 빈 필드를 숨길 수도 있습니다. 이러한 필드는 필터링, 리포팅, 자동화에서 사용하여 워크플로우를 간소화할 수 있습니다.

# 기회 커스텀 필드의 주요 장점

커스텀 필드를 통해 기회를 추적하고 정리하는 방법을 제어할 수 있어 CRM을 더욱 강력한 도구로 만들 수 있습니다.

- **개인화**: 예산, 선호도, 제품 세부 정보 등 비즈니스에 특화된 정보를 캡처합니다.

- **정리**: 필드를 폴더로 그룹화하고 순서를 변경하여 데이터 관리를 더욱 쉽게 합니다.

- **효율성**: "빈 필드 숨기기" 옵션을 사용하여 파이프라인 보기를 깔끔하고 집중적으로 유지합니다.

- **향상된 리포팅**: 대시보드, 위젯, 보고서에 커스텀 필드를 포함하여 더 깊은 인사이트를 얻습니다.

- **필터링 기능**: 커스텀 조건으로 기회를 필터링하고 정렬하여 리드의 우선순위를 빠르게 설정합니다.

- **자동화**: 워크플로우, 캠페인, 병합 필드에서 커스텀 필드를 사용하여 시간을 절약합니다.

# 커스텀 필드 생성 및 관리 방법

커스텀 필드를 생성하고 관리하면 팀이 모든 기회에 대해 올바른 세부 정보를 캡처할 수 있습니다.

### 커스텀 필드에 접근하기

HighLevel에서 커스텀 필드 대시보드에 도달하는 방법은 두 가지입니다. 두 방법 모두 같은 곳으로 이동하므로 더 편리한 방법을 선택할 수 있습니다.

**설정을 통한 접근**

하위 계정 대시보드에서 **Settings(설정)** 메뉴를 열고 아래로 스크롤하여 **Custom Fields(커스텀 필드)**를 선택합니다. 이렇게 하면 커스텀 필드 대시보드로 바로 이동합니다.

![커스텀 필드 설정 접근](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054373293/original/1GqOozmZr8R05rcBS02bYcwmABW5_Kvg-g.png?1758625175)

**기회를 통한 접근**

**Opportunities(기회)** 섹션 내에서 기존 기회를 선택하고 Add/Manage Fields를 클릭합니다. 이 바로가기는 같은 커스텀 필드 대시보드로 리디렉션되어 기회를 검토하면서 빠르게 편집할 수 있습니다.

![기회를 통한 커스텀 필드 접근](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054373315/original/GbQRrMA_9OgqCGYdKxEYDxnrdnGKI_MFTQ.png?1758625198)

## 새 커스텀 필드 생성하기

커스텀 필드를 생성하면 기회 내에서 직접 영업 프로세스에 고유한 세부 정보를 캡처할 수 있습니다. 고객 예산부터 후속 조치 날짜까지, 이러한 필드는 파이프라인을 맞춤 설정하고 중요한 정보가 항상 추적되도록 하는 유연성을 제공합니다.

### 필드 추가

커스텀 필드 대시보드의 오른쪽 상단에 있는 **+ Add Field(+ 필드 추가)** 버튼을 클릭합니다. 이는 기회나 기타 객체에 대한 새 커스텀 필드 생성 프로세스를 시작합니다.

![필드 추가 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054374744/original/bS4xfntd-FcxoDXa0dTIyuGnQaFO1W7cJg.png?1758625895)

### 필드 유형 선택

데이터 요구 사항에 맞는 다양한 필드 유형 중에서 선택합니다. 옵션에는 **텍스트 입력**(한 줄, 여러 줄, 목록), **값**(숫자, 전화번호, 금액), **선택**(드롭다운, 라디오, 체크박스), **날짜/시간**(날짜 선택기), **기타 입력**(파일 업로드 또는 서명 등)이 있습니다.

| **필드 유형** | **사용 예시** |
|---|---|
| **Single Line(한 줄)** | 프로젝트명이나 고객 회사명 입력 |
| **Multi Line(여러 줄)** | 고객 요구사항에 대한 상세 노트 추가 |
| **Text Box List(텍스트 박스 목록)** | 기회와 연관된 여러 제품 SKU 추적 |
| **Number(숫자)** | 전환 확률 점수 기록 (예: 80%) |
| **Phone(전화번호)** | 대체 의사결정권자 전화번호 저장 |
| **Monetary(금액)** | 고객 예산이나 예상 거래 가치 저장 |
| **Dropdown (Single)(드롭다운-단일)** | 영업 단계 선택 (예: 협상, 마감) |
| **Dropdown (Multiple)(드롭다운-복수)** | 고객이 관심 있는 모든 제품/서비스 표시 |
| **Radio Select(라디오 선택)** | "예산 승인됨?"과 같은 예/아니오 스타일 응답 캡처 |
| **Checkbox(체크박스)** | 계약이나 규정 준수 문서 필요 여부 플래그 |
| **Date Picker(날짜 선택기)** | 후속 조치나 예상 마감 날짜와 같은 주요 날짜 추적 |
| **File Upload(파일 업로드)** | 제안서 파일이나 고객 RFP 문서 첨부 |
| **Signature(서명)** | 기회 내에서 직접 서명된 승인 수집 |

적절한 필드 유형을 선택한 후 **Next(다음)**를 클릭하여 커스텀 필드 설정을 계속합니다.

![필드 유형 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054375370/original/ZNUH2N7b69MRRigSqT6DyeW4Th65JVT_6Q.png?1758626138)

### 기회에 할당

필드 이름을 지정한 후 **Object(객체)**를 **Opportunity(기회)**로 설정하여 필드가 기회 기록 내에서만 적용되도록 합니다. 모든 세부 정보를 입력한 후 **Save(저장)**를 클릭하여 새 커스텀 필드를 생성합니다.

**기타 필드**

- **Name(이름)**: 커스텀 필드의 표시 라벨 (예: "고객 예산")
- **Group(그룹)**: 필드를 정리할 폴더나 카테고리 (예: "예산 세부 정보")
- **Placeholder(플레이스홀더)**: 필드가 비어있을 때 표시되는 힌트 텍스트 (예: "예상 가치 입력")
- **Additional Preferences(추가 기본 설정)**: 특정 필드 유형에 대한 고급 옵션 (예: 기본 값, 필수 토글)

**Save(저장)** 버튼을 클릭하여 새 커스텀 기회 필드를 완료합니다. 저장되면 필드는 파이프라인의 모든 기회 기록에서 사용할 수 있게 됩니다.

**팁:** 실수로 필드를 기회 대신 연락처로 할당하면 연락처 기록에만 나타나고 파이프라인 기회에는 표시되지 않습니다.

![기회에 할당](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054376449/original/j9cabOyyj4J9ACZ1y84ZRvvPHq5ZpQhTFQ.png?1758626669)

## 폴더 생성하기

폴더를 사용하면 여러 커스텀 필드를 논리적 그룹으로 쉽게 정리할 수 있습니다. 폴더를 생성하여 관련 필드를 함께 유지하고, 혼란을 줄이며, 팀이 기회 내에서 올바른 정보를 찾고 업데이트하기 쉽게 만들 수 있습니다.

### 폴더 추가

커스텀 필드 대시보드의 오른쪽 상단에 있는 **Add Folder(폴더 추가)** 버튼을 클릭합니다. 폴더는 관련 필드를 정리하여 관리하고 찾기 더 쉽게 도와줍니다.

![폴더 추가 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054379864/original/AW0ZEWgSObj4Qqb_E_jpxhiK2fEfGyts1Q.png?1758628162)

### 폴더 이름 지정

폴더에 대한 설명적인 이름을 입력합니다(예: "예산 세부 정보" 또는 "제품 정보"). 명확한 이름은 팀이 어떤 유형의 필드가 내부에 속하는지 정확히 알 수 있게 해줍니다.

![폴더 이름 지정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054379921/original/LIt6RXEU7td8gNh9nD3aIBA72x15pLz5pw.png?1758628189)

### 객체 선택

**Opportunity(기회)**를 객체로 선택하여 폴더가 기회 기록에만 적용되도록 합니다. 이는 이 폴더 하위에 생성된 모든 필드가 연락처나 기타 객체가 아닌 기회에 나타나도록 보장합니다.

폴더 이름과 객체를 완료한 후 Save(저장)를 클릭합니다.

![객체 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054379944/original/WAQBiC9X50g1rf85uERvXNw4SQKH5ykh0A.png?1758628211)

## 커스텀 필드 관리하기

커스텀 필드와 폴더가 생성되면 순서 변경, 필터링, 그룹화 또는 삭제를 통해 효과적으로 관리할 수 있습니다. 이러한 도구는 기회를 정리된 상태로 유지하고 올바른 정보를 더 쉽게 찾을 수 있도록 도와줍니다.

### 객체별 필터링

오른쪽 상단의 드롭다운을 사용하여 **객체 유형**(예: Opportunity, Contact, Company)별로 커스텀 필드를 필터링합니다. **Opportunity(기회)**를 선택하면 기회에 연결된 필드만 표시되어 파이프라인 추적과 관련된 내용에 집중하기 더 쉽습니다.

![객체별 필터링](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054382831/original/mzo5MU_Qeyfhb1GhjAbeM-bpV-sZwRQl0w.png?1758629521)

### 폴더별 그룹화

**Group By(그룹화)** 드롭다운을 사용하여 커스텀 필드를 폴더나 카테고리별로 정리합니다. 이렇게 하면 컨텍스트를 기반으로 필드를 보기가 더 쉬워집니다. 예를 들어, 모든 예산 관련 필드를 "기회 세부 정보" 하위에 그룹화하거나 모든 고객 입력을 특정 양식 하위에 그룹화할 수 있습니다.

![폴더별 그룹화](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054383098/original/oJQwAGj8bvRdQfQ9kCSxC2ENRgcMt0GgCQ.png?1758629674)

### 일괄 작업

체크박스 메뉴를 사용하여 여러 커스텀 필드를 선택하고 **Bulk Actions(일괄 작업)** 메뉴를 사용하여 편집, 삭제 또는 다른 폴더로 이동시킵니다. 이는 개별적으로 조정하는 대신 여러 필드를 한 번에 관리할 때 시간을 절약해줍니다.

![일괄 작업](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054383273/original/-z9GcnpsgtJtCXrGEjrSEXLq2UuRXytvGQ.png?1758629760)

### 필드 순서 변경

드래그 앤 드롭 **hamburger 아이콘**을 사용하여 커스텀 필드를 목록에서 위아래로 이동시킵니다. 이를 통해 기회를 생성하거나 편집할 때 필드가 나타나는 순서를 제어할 수 있습니다.

![필드 순서 변경](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054379234/original/sU7SMNt7svHQQqdv0ccR0OSsu8UQBpz6Qg.gif?1758627952)

# 기회 대시보드에서 커스텀 필드 사용하기

커스텀 필드는 단순한 데이터가 아닙니다. 필터링, 리포팅, 파이프라인 보기와 통합되어 영업 관리를 더 쉽게 만들어줍니다.

### 고급 필터 열기

**Opportunities(기회)** 보기에서 **Advanced Filters(고급 필터)**를 클릭하여 파이프라인에 커스텀 필터 적용을 시작합니다. 이는 표준 필드와 커스텀 기회 필드를 모두 사용하여 결과를 세분화할 수 있는 필터링 패널을 엽니다.

![고급 필터 열기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054386664/original/m3QCoTo4lY8TTgO-6-4OyUOjZrGMXdKWKg.png?1758631008)

### 커스텀 필드로 필터링

필터링 패널 내에서 **Opportunities Custom Fields(기회 커스텀 필드)** 섹션으로 스크롤 다운합니다. 이 영역에는 생성한 모든 기회 커스텀 필드(기존 및 새로 추가된 것 모두)가 표시되어 파이프라인에 가장 중요한 데이터를 기반으로 기록을 필터링할 수 있습니다.

**팁:** 여러 커스텀 필드 필터를 조합하여 특정 자격 기준도 충족하는 고가치 리드 식별과 같은 정확한 보기를 구축할 수 있습니다.

## 기회 카드에 커스텀 필드 표시하기

커스텀 필드는 기회 기록 안에만 존재하는 것이 아니라 파이프라인 카드에도 표시할 수 있습니다. 이는 중요한 세부 정보를 한눈에 볼 수 있게 하여 팀이 각 기회를 열지 않고도 더 빠르게 작업할 수 있도록 도와줍니다.

### 필드 관리 열기

**Opportunities(기회)** 보기에서 오른쪽 상단의 **Manage Fields(필드 관리)** 버튼을 클릭합니다. 이는 기회 카드에 어떤 세부 정보가 나타날지 조정할 수 있는 맞춤 설정 패널을 엽니다.

커스텀 필드 추가의 경우, Add Fields 섹션으로 스크롤 다운하기만 하면 됩니다.

![필드 관리 열기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054389290/original/fwI06PtRXIIDKl8fHgZFCkUuHc5cAalezA.png?1758631844)

### 카드에 커스텀 필드 추가

**Add Fields(필드 추가)** 하위의 **Opportunity Details(기회 세부 정보)** 섹션을 확장하고 추가하려는 커스텀 필드를 체크합니다. 이제 이러한 필드가 카드에 직접 나타나 예산, 작업 유형 또는 고객 선호도와 같은 정보를 빠르게 볼 수 있습니다.

원하는 커스텀 필드를 선택한 후 패널 하단의 **Apply(적용)** 버튼을 클릭합니다. 기회 카드가 새 필드 선택을 반영하여 즉시 업데이트됩니다.

![카드에 커스텀 필드 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054393221/original/hbw0Ucm9KDvZwByBGaa-djBtGnYpkkJUfQ.png?1758633641)

### 빈 필드 숨기기

모든 기회 카드 내에서 상단에 **Hide Empty Fields(빈 필드 숨기기)** 옵션을 볼 수 있습니다. 이를 활성화하면 값이 있는 필드만 표시하여 보기를 깔끔하게 유지할 수 있어 실제로 중요한 정보에 집중할 수 있습니다.

이는 많은 커스텀 필드를 사용하지만 각 기회에 대해 실제로 채워진 필드만 검토하면 되는 경우에 특히 유용합니다.

![빈 필드 숨기기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054394452/original/PoC7GGDak7xaSdGLx5TE-tJjSV59aE1SDw.png?1758634022)

# 자주 묻는 질문

**Q: "필드 관리" 옵션을 사용하여 기회 카드에서 커스텀 필드를 추가하거나 제거하면 계정의 모든 사용자에게 이 변경 사항이 적용되나요?**

아니요. **Manage Fields(필드 관리)** 패널에서 변경한 사항은 기회 카드의 개인 보기에만 적용됩니다. 같은 하위 계정의 다른 사용자들은 직접 조정하지 않는 한 자신만의 맞춤 설정된 레이아웃을 계속 볼 수 있습니다. 이를 통해 각 사용자는 자신의 역할에 가장 관련된 필드를 표시할 수 있습니다.

**Q: 기회 커스텀 필드를 삭제하면 기존 데이터는 어떻게 되나요?**

커스텀 필드를 삭제하면 필드와 모든 기회에 저장된 데이터가 영구적으로 제거됩니다. 이 작업은 취소할 수 없습니다. 데이터를 잃지 않고 필드를 숨기기만 원한다면 삭제하는 대신 폴더로 이동하거나 "빈 필드 숨기기" 옵션을 사용하는 것을 고려해보세요.

**Q: 모든 커스텀 필드 유형을 기회 필터링에 사용할 수 있나요?**

아니요. 대부분의 기회 커스텀 필드(드롭다운, 숫자, 날짜, 텍스트 필드 등)는 필터링에 사용할 수 있지만 **File Upload(파일 업로드)** 및 **Signature(서명)**와 같은 특정 필드 유형은 필터링할 수 없습니다. 이러한 필드는 여전히 기회 기록에 나타나지만 고급 필터 패널에서는 사용할 수 없습니다.

**Q: 자동화나 워크플로우에서 기회 커스텀 필드를 어떻게 사용할 수 있나요?**

기회 커스텀 필드는 워크플로우에서 트리거, 조건 또는 액션으로 사용할 수 있습니다. 예를 들어:

- 커스텀 필드를 자동으로 업데이트 (예: 리드 활동을 기반으로 "전환 가능성" 증가)
- 기회 커스텀 필드 데이터를 가져오는 병합 필드를 사용하여 개인화된 메시지 전송
- 커스텀 필드가 특정 기준을 충족할 때 기회를 새 단계로 이동시키는 규칙 생성

이러한 통합은 커스텀 데이터가 단순히 저장되는 것이 아니라 파이프라인에서 자동화를 적극적으
