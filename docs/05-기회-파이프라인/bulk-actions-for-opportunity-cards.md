# 기회 카드 일괄 작업

하이레벨의 기회 관리 칸반/카드 보기에서 **일괄 작업(Bulk Actions)** 기능으로 영업 파이프라인을 효율적으로 관리하세요. 이 기능을 사용하면 시각적인 칸반 인터페이스에서 여러 기회 카드를 한 번에 선택하고 관리할 수 있습니다.

# **기회 카드 보기의 일괄 작업이란?**

**기회 카드 보기의 일괄 작업**은 사용자가 기회 관리 파이프라인의 칸반 보기에서 여러 기회 카드를 선택할 수 있게 해주는 기능입니다. 각 카드에 있는 체크박스를 통해 기회 이동, 편집, 삭제 등의 대량 업데이트나 작업을 시각적 인터페이스를 벗어나지 않고 수행할 수 있습니다. 이는 대용량 파이프라인을 관리하는 팀의 워크플로우를 간소화해줍니다.

---

# **기회 카드 보기 일괄 작업의 주요 장점**

강력한 파이프라인 관리 경험을 위한 핵심 장점들:

- **빠른 워크플로우:** 시각적인 칸반/카드 보기에서 여러 기회를 한 번에 업데이트
- **보기 전환 불필요:** 목록 보기로 이동하지 않고도 일괄 작업 수행
- **전체 선택 옵션:** 20개 이상의 기회를 볼 때 현재 페이지의 모든 카드를 한 번에 선택
- **오류 방지:** 선택 모드에서 카드 드래그 앤 드롭/열기 기능을 비활성화하여 실수 방지
- **더 나은 접근성:** 모든 사용자가 선호하는 보기에서 파이프라인을 쉽게 관리
- **간소화된 관리:** 대용량 업데이트를 빠르고 효율적으로 수행

# 기회 카드 보기에서 일괄 작업 사용하는 방법

다음 간단한 과정으로 일괄 작업을 시작하세요:

### **기회 관리 탭 접근**

일괄 작업을 사용하려면 왼쪽 메뉴에서 **기회 관리(Opportunities)** 탭으로 이동하세요. 파이프라인의 모든 기회 단계와 항목이 표시되는 카드 보기가 열립니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048890304/original/bzi32qGFBzcEO18XHeauDIaEyfB3oblS7A.png?1750905909)

### **파이프라인 선택**

상단의 드롭다운 메뉴를 사용하여 관리할 파이프라인을 선택하세요. 선택한 파이프라인의 항목만 표시되어 더 집중적이고 효율적인 일괄 업데이트가 가능합니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048890340/original/8PhClHpHMJ3mR-CkXFSQN4MiMRT_r5xqXA.png?1750906016)

### **보기 모드 선택**

우상단의 토글 아이콘을 사용하여 **카드 보기**와 **목록 보기** 중 선택하세요. 두 보기 모두 일괄 작업을 지원합니다.

- **카드 보기**는 파이프라인 단계별로 기회를 시각적으로 확인하고 선택한 후 일괄 변경을 적용하고 싶을 때 유용합니다.
- **목록 보기**는 대용량 데이터를 다룰 때 이상적입니다. 빠른 스크롤, 컬럼 정렬, 일괄 편집을 위한 대량 선택이 가능합니다.

시각적 파이프라인 맥락을 우선시하는지, 대용량 레코드 관리를 우선시하는지에 따라 보기를 선택하세요.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048891240/original/IfVoINQnX9Aq0wXyZ0HawK4tii6-Bw6mOw.gif?1750910051)

### 여러 기회 선택

기회를 선택하려면 해당 기회 위에 마우스를 올리고 우상단에 나타나는 체크박스를 클릭하세요.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048910865/original/OlcXQ14mgww3N3q5rC-9NJr8rqrmtOumpg.gif?1750930624)

### 기회 카드 편집

**카드** 또는 **목록** 보기에서 여러 기회를 선택한 후, 화면 상단에 **편집(Edit)** 버튼이 나타납니다. 이 버튼을 클릭하면 선택한 모든 레코드에 한 번에 변경 사항을 적용할 수 있습니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048891457/original/J4DhvHgyW9xm4D5NgA6ZTRhU-KGoVe4sQg.png?1750910955)

### 편집할 필드 선택

**편집**을 클릭한 후 일괄 편집 패널을 사용하여 여러 기회에서 업데이트할 필드를 선택하세요. 상태, 금액, 파이프라인, 담당자와 같은 필드는 재배정, 정리, 캠페인 기반 업데이트에 일반적으로 사용됩니다. 다음 부분에서는 담당자 지정과 같은 일부 필드를 설명하겠습니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048909905/original/L-xZtWRGFRwSNpRqrMXHno1aVLDgpTMWXQ.png?1750930005)

### 새 담당자 지정

**담당자(Owner)** 필드를 선택하여 기회를 다른 팀원에게 재배정하세요. 팀원 교대, 휴가, 업무 공백 기간 동안 리드를 재분배할 때 특히 유용합니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048910137/original/ufcirshS3bxcjRrQVBERj2DqNJGlmdRXPg.png?1750930143)

### 파이프라인 및 단계 변경

**파이프라인(Pipeline)**과 **단계(Stage)** 필드를 사용하여 여러 기회를 워크플로우 간에 이동시키세요. 리드를 새로운 영업 프로세스나 캠페인 후속 단계로 재그룹화해야 할 때 유용합니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048910204/original/3mYq9dZsizzr-oRhp1nb5AT5GkOSTVfMPw.png?1750930213)

### 기회 상태 업데이트

**상태(Status)**를 선택하여 선택한 기회를 진행 중(Open), 성사(Won), 실패(Lost), 포기됨(Abandoned)으로 표시하세요. 통화 검토, 아웃리치 시퀀스, 캠페인 후속 작업 후 업데이트를 간소화합니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048910116/original/7IRjHGxKea0uOp3Rn_eSaJo3mFmfy2OZKA.png?1750930125)

### 일괄 업데이트를 위한 "전체 선택" 사용

대용량 작업을 할 때는 **전체 선택 [숫자]**을 클릭하여 현재 파이프라인에 표시된 모든 기회를 포함할 수 있습니다. 전체 레코드 그룹에 동일한 변경 사항을 적용할 때 시간을 절약할 수 있습니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048910467/original/dOOpKx4yzUO1I1RPiipLQrRDnPzBMmpFyg.png?1750930360)

# **자주 묻는 질문**

**Q: "페이지 전체 선택"을 사용하여 여러 페이지의 카드를 선택할 수 있나요?**
A: 아니요, 이 옵션은 현재 페이지의 카드만 선택합니다.

**Q: 잘못된 카드를 선택했다면 어떻게 하나요?**
A: 체크박스를 다시 클릭하여 선택을 취소하거나, 모두 선택 취소를 눌러 선택 모드를 종료하세요.

**Q: 선택 모드가 카드 이동이나 열기에 영향을 주나요?**
A: 네, 오류를 방지하기 위해 선택 중에는 드래그 및 열기 기능이 일시정지됩니다.

**Q: 한 페이지에 20개 이상의 기회가 있는지 어떻게 알 수 있나요?**
A: 이 한도에 도달하면 "페이지 전체 선택" 버튼이 자동으로 나타납니다.

**Q: 일괄 작업을 실행 취소할 수 있나요?**
A: 일부 작업은 되돌릴 수 있지만(카드 이동 등), 선택을 신중하게 확인하는 것이 좋습니다.

**Q: 사용자 권한이 사용 가능한 작업에 영향을 주나요?**
A: 네, 사용자 역할과 권한에 따라 사용할 수 있는 작업이 달라집니다.
