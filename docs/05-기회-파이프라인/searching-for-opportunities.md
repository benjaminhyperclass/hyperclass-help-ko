# 기회 검색하기

Hyperclass는 사용자가 특정 기회를 쉽게 찾을 수 있는 강력한 검색 기능을 제공하여 시간을 절약하고 업무 흐름을 간소화합니다. 이 상세 가이드에서는 Hyperclass에서 기회를 검색하는 과정을 단계별로 안내해드립니다.

이 문서의 내용

### 1단계: 기회 관리 섹션 접근하기

- Hyperclass 계정에 로그인하고 하위 계정을 여세요.
- 사이드바 메뉴에서 "기회 관리(Opportunities)" 섹션으로 이동하세요.

![기회 관리 섹션 접근](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155021427877/original/pHaxiq16lJCxkIQKDac3zIwACAvtbBFZ6Q.png?1708707911)

### 2단계: 검색창 이해하기

- 기회 관리 섹션에 들어오면, 페이지 상단에 있는 검색창을 찾으세요.
- 검색창에서는 키워드, 구문, 또는 조건을 입력하여 검색 쿼리와 일치하는 특정 기회를 찾을 수 있습니다.
- 기회명, 전화번호, 이메일, 회사명, 태그, 연락처명으로 검색할 수 있습니다.

### 3단계: 기본 검색 수행하기

- 기본 검색을 수행하려면, 찾고자 하는 기회와 관련된 키워드나 구문을 검색창에 입력하세요. 입력하는 동안 Hyperclass가 기회 목록을 동적으로 필터링하여 검색 조건과 일치하는 결과를 실시간으로 표시합니다.
- 검색 기능은 검색창에 최소 3글자 이상 입력해야 작동합니다.
- 검색은 단어의 중간 부분은 찾지 못합니다. 예를 들어, "Opportunity details"라는 용어에서 'Oppor'로 검색하면 해당 용어가 검색되지만 'tunity'로는 검색되지 않습니다.
