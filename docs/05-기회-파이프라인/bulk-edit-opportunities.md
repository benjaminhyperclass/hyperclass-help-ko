# 기회 일괄 편집

기회 일괄 편집 기능은 여러 기회의 여러 필드를 동시에 업데이트할 수 있게 해서 워크플로우를 간소화하도록 설계되었습니다.

### 1단계: 기회 관리(Opportunities) > 목록 보기(List View)로 이동하세요.

![기회 관리 목록 보기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155031249618/original/5Ml_s2wzJ5YIVRkH39joEJTsWg43ApNitg.png?1724056144)

### 2단계: 기회 선택

- 편집하려는 여러 기회를 선택하고 상단의 편집(Edit) 버튼을 클릭하세요.

![기회 선택 및 편집 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155031249649/original/3TESo4DXSySiB0KeZnn2qDhCmUTP1tYZ4g.png?1724056177)

### 3단계: 필드 업데이트

업데이트하려는 필드를 선택하고 새 정보를 입력하세요.

![필드 업데이트](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155031249688/original/TDZqetWetKcGA0vOmRAlc-eTYHIjm6tx5w.png?1724056219)

### 4단계: 업데이트 적용

확인(Confirm) 버튼을 클릭해서 변경사항을 적용하세요.

![업데이트 확인](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155031249770/original/320issQpfB1u2bHZv3YphbaJX5z-fHk05g.png?1724056251)

### 5단계: 진행상황 모니터링

일괄 작업(Bulk Actions) 페이지로 이동해서 진행상황을 모니터링하고, 성공 결과를 검토하며, 오류가 있는지 확인하세요.

![일괄 작업 진행상황](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155031249827/original/vMp98rJSrAyxqBtcDLE04-zYqFcbiV7Tcw.png?1724056286)
