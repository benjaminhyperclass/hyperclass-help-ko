# 기회 파이프라인 색상 구분하기 (스마트 태그)

스마트 태그는 색상 칩을 이용해 **파이프라인 내 기회를 시각적으로 우선순위화**하는 데 도움을 줍니다. 필터나 텍스트 라벨에 의존하지 않고, 오래된 딜, 고가치 기회, 신규 리드와 같이 설정한 규칙에 따라 자동으로 거래를 강조 표시할 수 있습니다. 이 가이드에서는 하이퍼클래스에서 스마트 태그를 만들고 편집하며 관리하는 방법을 알려드립니다.

## 스마트 태그란?

스마트 태그는 파이프라인의 각 기회 카드에 직접 표시되는 동적 규칙 기반 색상 칩입니다. 쉬운 조건 빌더로 구동되며, 모든 기회를 실시간으로 평가하여 레코드가 설정한 조건(예: 거래 금액 > 10,000달러)을 만족할 때마다 자동으로 색상 라벨을 적용합니다. 그 결과 한눈에 알아볼 수 있는 시각적 레이어를 제공하여 수동 필터나 번거로운 스캔 작업이 필요 없어집니다.

## 스마트 태그의 주요 장점

- **즉각적인 우선순위 파악:** 중요한 거래가 화면에서 두드러져 보입니다.
- **컨텍스트 전환 감소:** 각 카드를 열거나 여러 필터를 적용할 필요가 없습니다.
- **일관된 팀 집중력:** 모든 구성원이 동일한 색상 신호를 보며 일관된 행동을 취할 수 있습니다.
- **모든 워크플로우에 맞춤화:** 표준 또는 커스텀 기회 필드에서 AND/OR 조건을 자유롭게 조합할 수 있습니다.

## 권한 및 접근

- **스마트 태그를 만들거나 편집할 수 있는 사용자:**
  - 에이전시 관리자
  - 로케이션 관리자
  - 파이프라인 편집 권한이 있는 사용자

- **스마트 태그를 볼 수 있는 사용자:**
  - 모든 사용자 (파이프라인 보기 권한이 있는 경우 - 편집 권한이 없어도 가능)

## 제한사항 및 검증

- 각 파이프라인당 최대 60개의 활성 스마트 태그를 지원합니다.
- 같은 파이프라인 내 중복 태그(같은 색상 + 규칙)는 자동으로 차단됩니다.
- 거래 데이터가 변경되면 태그가 자동으로 업데이트되므로 수동 새로고침이 필요 없습니다.

## 스마트 태그 설정 및 사용법

### 1단계: 스마트 태그 찾기

- 하위 계정에서 기회 관리(Opportunities)로 이동합니다.
- 파이프라인(Pipelines) 탭을 클릭합니다.

![파이프라인 탭](https://jumpshare.com/share/1lRYpa0DPkqgnn7vklNo+/Screen+Shot+2025-10-14+at+10.10.30+PM.png)

- 스마트 태그를 추가하고 싶은 파이프라인에서 편집(Edit) 버튼을 클릭합니다.

![편집 버튼](https://jumpshare.com/share/SiYTF4MUiKdXMRW6q8zM+/Screen+Shot+2025-10-14+at+10.12.33+PM.png)

- 스마트 태그(Smart Tags) 탭을 엽니다.

![스마트 태그 탭](https://jumpshare.com/share/QT0cNw8OUDULQHDTOQvA+/Screen+Shot+2025-10-14+at+10.18.05+PM.png)

### 2단계: 스마트 태그 만들기

- 스마트 태그 추가(Add Smart Tag)를 클릭합니다.

![스마트 태그 추가](https://jumpshare.com/share/xs6WW87OttyVfYr6Io9l+/Screen+Shot+2025-10-14+at+10.21.18+PM.png)

- 다음 중에서 선택합니다:
  - **새 태그 만들기:** 색상, 이름, 규칙 조건을 직접 정의합니다.
  - **기본 제공 태그:** 미리 만들어진 템플릿(예: 신규 리드, 고가치)을 사용합니다.

![태그 유형 선택](https://jumpshare.com/share/YTAhZp8CjAr7fQedHxkK+/Screen+Shot+2025-10-14+at+10.23.28+PM.png)

- 라벨 이름(예: "오래된 거래")을 입력합니다.
- 명확성을 위해 설명을 추가합니다. (선택사항)
- 태그 색상을 선택합니다.
- 이 규칙을 전체적으로 사용하려면 '모든 파이프라인에 적용'을 체크합니다.
- 다음: 규칙 추가(Next: Add Rule)를 클릭합니다.

![태그 설정](https://jumpshare.com/share/Y24qhRMvGpdtEt8pRgOr+/Screen+Shot+2025-10-14+at+10.28.48+PM.png)

### 3단계: 규칙 정의하기

- 규칙 빌더를 사용하여 태그가 언제 나타날지 지정합니다.
- 각 태그는 규칙 기반입니다. 예시:
  - 생성일 ≤ 3일 전 → "신규 리드"
  - 거래 가치 > $10,000 → "고가치"

- 다음 작업이 가능합니다:
  - AND/OR 논리로 여러 조건을 결합
  - 복잡한 논리를 위한 중첩 필터 추가
  - 태그 미리보기(Preview Tag)로 샘플 카드에서 어떻게 보일지 확인

- 완료되면 저장(Save)을 클릭합니다.

![규칙 정의](https://jumpshare.com/share/f1pRboRnSNkFOLOc17cm+/GIF+Recording+2025-10-14+at+10.31.43+PM.gif)

## 추천 기본 제공 태그

| 태그 이름 | 색상 | 규칙 예시 | 목적 |
|---------|------|----------|------|
| 오래된 거래 (출시 예정) | 회색 | 최종 활동이 14일 전 | 방치된 기회 강조 |
| 고가치 | 초록 | 기회 가치 > $10,000 | 큰 거래 표시 |
| 신규 리드 | 파랑 | 생성일 ≤ 3일 전 | 새로운 기회 추적 |
| 위험 (출시 예정) | 빨강 | 최종 단계 변경 > 30일 전이면서 상태 = "진행 중" | 차가워지는 거래 경고 |
| 핫딜 | 주황 | 생성일 < 7일이면서 기회 가치 > $5000 | 최근 생성된 고가 기회 강조 |
| 미배정 거래 | 보라 | 담당자가 없음 | 담당자가 없는 기회 파악 |

## 모범 사례

✅ 최우선순위에는 뚜렷한 색상을 사용하세요 (예: 긴급한 것은 빨강, 고가치는 초록).
✅ 태그 이름은 20자 이하로 간단하게 유지하세요.
✅ 규칙이 계속 관련성을 유지하도록 정기적으로 검토하세요.

## 문제 해결

- **카드에 스마트 태그가 보이지 않나요?**
  - 카드 사용자화(Customize Card)를 통해 스마트 태그가 카드 레이아웃에 추가되었는지 확인하세요.
  - 규칙 조건이 올바르고 보고 있는 카드에 적용되는지 확인하세요.
  - 파이프라인에서 변경사항을 동기화하려면 강제 새로고침을 하세요.

- **스마트 태그 탭이 보이지 않나요?**
  - 관리자만 태그 생성과 편집에 접근할 수 있습니다.

## 자주 묻는 질문

**Q: 스마트 태그는 모든 파이프라인에서 작동하나요?**
네 — "모든 파이프라인에 적용"을 선택하면 규칙 조건이 맞는 곳이라면 어디서든 태그가 나타납니다.

**Q: 파이프라인 간에 스마트 태그를 복제할 수 있나요?**
아직은 불가능하지만, 다른 파이프라인에서 같은 규칙을 수동으로 다시 적용할 수 있습니다.

**Q: 스마트 태그가 리포트에 포함되나요?**
현재 스마트 태그는 파이프라인 보기에서만 시각적으로 표시됩니다.

**Q: 누가 스마트 태그를 만들거나 편집할 수 있나요?**
파이프라인 설정을 편집할 권한이 있는 사용자라면 누구나 스마트 태그를 관리할 수 있습니다. 필요한 경우 파이프라인 권한을 사용하여 접근을 제한하세요.

**Q: 스마트 태그는 실시간으로 평가되나요?**
시스템은 기회가 생성, 업데이트되거나 단계/상태가 변경될 때마다 모든 태그 규칙을 다시 확인합니다.
