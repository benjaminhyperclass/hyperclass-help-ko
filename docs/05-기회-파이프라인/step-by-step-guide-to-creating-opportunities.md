## 기회란 무엇인가요?

기회(Opportunity)는 Hyperclass에서 추적하는 하나의 잠재적 영업 건을 의미합니다. 연락처, 파이프라인, 단계, 예상 금액, 담당자를 하나로 묶어서 첫 대화부터 최종 결과까지 영업 과정을 체계적으로 관리할 수 있습니다.

### 기회 생성의 주요 장점

- **중앙집중식 추적**: 기회 관리 대시보드에서 진행 중인 모든 영업 건을 한눈에 확인
- **매출 예측**: 잠재 매출액을 기록하여 영업 예측 정확도 향상
- **팀 협업**: 담당자를 지정하여 팀원들이 누가 책임지는지 명확하게 파악
- **상황별 메모**: 대화 내용을 기회 카드에 직접 기록하여 맥락 유지

## 기회 생성 방법

### 1단계: 기회 관리 모듈 접근

- Hyperclass 하위 계정에 로그인합니다
- `Opportunities(기회 관리)` 탭을 클릭합니다
- 우측 상단의 `+ Add Opportunity(기회 추가)` 버튼을 클릭합니다

![기회 추가 버튼](https://jumpshare.com/v/SDr5vZRXt5jg6EPOeNpw+/Screen+Shot+2025-07-17+at+7.09.44+PM.png)

### 2단계: 연락처 및 기회 정보 입력

- 저장된 연락처 목록에서 `Primary Contact(주 연락처)`를 선택합니다
- 이메일과 전화번호가 자동으로 채워지거나, 없는 경우 직접 입력합니다

![연락처 정보 입력](https://jumpshare.com/v/Dyzm3UJ6zfnTM3KUNOZQ+/Screen+Shot+2025-07-17+at+7.11.55+PM.png)

### 3단계: 파이프라인, 단계, 상태 설정

- 커스텀 `Opportunity Name(기회명)`을 추가하거나 기본 연락처 이름을 유지합니다
- 기회를 추가할 `Pipeline(파이프라인)`을 선택합니다
- 적절한 `Pipeline Stage(파이프라인 단계)`를 선택합니다
- `Opportunity Status(기회 상태)`를 설정합니다:
  - Open (진행 중)
  - Lost (실패)
  - Won (성사)
  - Abandon (포기)
- `Opportunity Value(기회 금액)`을 입력합니다

![파이프라인 및 상태 설정](https://jumpshare.com/v/35WyDdsrRPyqa1yjynmu+/Screen+Shot+2025-07-17+at+7.23.09+PM.png)

### 4단계: 담당자, 팀원 및 추가 세부사항 입력

- 기회에 `Owner(담당자)`를 할당합니다
- `Followers(팔로워)`를 추가합니다 (여러 명 추가 가능)
- 필요에 따라 `Additional Fields(추가 필드)`를 채웁니다:
  - Business Name (회사명)
  - Opportunity Source (기회 출처)
  - Tags (태그)
  - Task Type (작업 유형)
  - Favorite Pet (애완동물)
  - Efficiency Rating (효율성 평가)
  - Task Difficulty (작업 난이도)
  - Time Tracked (시간 추적)

![추가 정보 입력](https://jumpshare.com/v/TZi1I05DbkbE0nFVUpea+/Screen+Shot+2025-07-17+at+7.33.44+PM.png)

### 5단계: 기회 최종 확정

- `Add/Manage Fields(필드 추가/관리)`를 클릭하여 커스텀 필드를 관리합니다
- `Create(생성)` 버튼을 클릭하여 기회를 저장합니다
- 파이프라인에서 새로 생성된 기회를 확인합니다

![기회 생성 완료](https://jumpshare.com/v/qY55sCPKUHccRa77t2F1+/Screen+Shot+2025-07-17+at+7.36.10+PM.png)

## 자주 묻는 질문

**Q: 나중에 기회를 다른 사람에게 할당할 수 있나요?**
A: 네. 기회 카드를 열어서 새로운 담당자를 선택하고 저장하면 됩니다.

**Q: 기회 생성 후에 금액을 변경할 수 있나요?**
A: 물론입니다. 편집 창에서 기회 금액 필드를 수정하고 저장하면 됩니다.

**Q: 기존 기회를 복제할 수 있나요?**
A: 현재는 지원하지 않습니다. 새로운 기회를 생성하고 관련 정보를 수동으로 복사해야 합니다.

**Q: 기회에 연결된 연락처를 삭제하면 어떻게 되나요?**
A: 기회는 남아있지만 연락처 연결이 끊어집니다. 새로운 연락처를 다시 연결하거나 원래 연락처를 복원해야 맥락을 유지할 수 있습니다.

**Q: 기회 생성 후 다른 단계로 이동하려면 어떻게 하나요?**
A: 기회 관리 보드에서 기회 카드를 원하는 단계 열로 드래그하거나, 편집 창을 열어서 드롭다운에서 새 단계를 선택하면 됩니다.

**Q: 기존 기회의 주 연락처를 변경할 수 있나요?**
A: 네. 편집 창을 열어서 현재 주 연락처를 제거하고, 새 연락처를 추가한 후 주 연락처로 표시하고 저장하면 됩니다.
