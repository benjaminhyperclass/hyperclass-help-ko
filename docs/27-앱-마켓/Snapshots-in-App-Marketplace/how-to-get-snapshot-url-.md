# 스냅샷 URL을 가져오는 방법

앱에 스냅샷을 추가하려면 다음 단계를 따라하세요.

**1단계**: 에이전시 뷰에서 'Account Snapshots(계정 스냅샷)'으로 이동하여 앱 마켓플레이스를 통해 판매하려는 스냅샷을 선택하세요.

![계정 스냅샷 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155032499219/original/nczrNVQdHSDVNWj3uz18pCt-VvF1S2Td9g.png?1725876577)

**2단계**: 옵션에서 'Share Snapshot(스냅샷 공유)'을 선택하고, 사용 가능한 옵션 중에서 'Get Marketplace share link(마켓플레이스 공유 링크 가져오기)'를 선택하세요.

![마켓플레이스 공유 링크 가져오기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155032499357/original/fgI7LlW__l4LJ9lBjvoFLkyYzaGg7_Fabg.png?1725876673)

**3단계**: 앱에 스냅샷 링크를 붙여넣고 스냅샷과 함께 앱을 출시하세요!

![스냅샷 링크로 앱 출시](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155032499482/original/fdPQHlAlTrfCE9kVxGpx-BQrdg0mZVrkBw.png?1725876732)

이제 앱에 스냅샷 링크를 붙여넣고 스냅샷과 함께 앱을 출시할 수 있어요!
