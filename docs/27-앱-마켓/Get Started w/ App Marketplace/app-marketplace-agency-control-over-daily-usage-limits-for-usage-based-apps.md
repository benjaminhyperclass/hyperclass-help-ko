# 앱 마켓플레이스 - 사용량 기반 앱의 일일 사용량 제한에 대한 에이전시 제어

Hyperclass 마켓플레이스에서 에이전시 소유자와 관리자가 사용량 기반 요금제 앱에 대해 일일 사용량 제한을 설정할 수 있게 되었습니다. 이 가이드에서는 이러한 제어 기능을 사용하여 비용을 예측 가능하게 유지하고 간단한 설정 페이지 하나에서 모든 것을 관리하는 방법을 알려드립니다.

## 일일 사용량 제한에 대한 에이전시 제어란?

이 기능을 통해 에이전시 소유자와 관리자는 사용량 기반 요금제로 운영되는 앱에 대해 일일 사용량 임계값을 설정할 수 있습니다. 중앙 집중식 일일 제한을 도입함으로써 에이전시는 지출에 대한 더 나은 감독을 얻고, 외부 API 비용이 예측 가능하고 예산 내에서 유지되도록 보장할 수 있습니다. 이 기능은 에이전시 신뢰도와 운영 제어를 향상시키려는 Hyperclass의 지속적인 노력의 일환입니다.

## 일일 사용량 제한 에이전시 제어의 주요 이점

일일 사용량 제한은 에이전시가 비용을 제어하고 운영을 원활하게 유지하는 데 도움이 됩니다.

- **향상된 비용 제어**: 일일 지출을 제한하여 예상치 못한 초과 사용을 방지합니다.
- **향상된 에이전시 신뢰도**: 예측 가능한 청구를 제공하고 외부 비용으로 인한 놀라운 상황을 줄입니다.
- **간소화된 관리**: 모든 마켓플레이스 제어 기능을 에이전시 소유자와 관리자만 액세스할 수 있는 단일 설정 페이지로 통합합니다.
- **최적화된 사용자 경험**: 에이전시 요구사항에 맞춘 전용 제어 인터페이스로 일일 관리를 간소화합니다.

## 개선된 에이전시 제어 설정 페이지

업데이트된 설정 페이지는 에이전시 소유자와 관리자 전용 중앙 허브를 제공하며, 모든 마켓플레이스 제어 기능을 직관적이고 접근 가능한 인터페이스로 통합합니다. 이 재설계를 통해 일일 사용량 제한과 앱 설정 관리가 효율적이고 손쉬워집니다.

- 승인된 에이전시 사용자만을 위한 전용 제어 패널에 액세스합니다.
- 모든 설정을 편리한 한 곳에 그룹화한 깔끔하고 사용자 친화적인 레이아웃을 즐기세요.
- 사용량 제한을 쉽게 모니터링하고 조정하여 전반적인 재무 관리를 향상시킵니다.

## 앱의 일일 사용량 제한 설정 방법

이 기능 설정은 간단하여, 에이전시가 사용량 제한을 관리하고 비용 제어를 유지하기 시작할 수 있습니다:

- 에이전시 계정에 로그인합니다.
- 좌측 네비게이션에서 App Marketplace(앱 마켓플레이스)를 클릭합니다.
![앱 마켓플레이스 메뉴](https://jumpshare.com/share/GfcoMZUl7PnuWmdKCMzi+/Screen+Shot+2025-09-01+at+8.12.11+PM.png)

- Settings(설정) 버튼을 클릭합니다.
![설정 버튼](https://jumpshare.com/share/Akzp1baL0KFCQ4qelKKa+/Screen+Shot+2025-09-01+at+8.13.22+PM.png)

- Settings(설정)에서 App Usage Limits(앱 사용량 제한)을 클릭합니다.
![앱 사용량 제한](https://jumpshare.com/share/S4Dr8AR7Uj5iJXqq7v38+/Screen+Shot+2025-09-01+at+8.14.58+PM.png)

- 제어하려는 앱을 찾아 Set Usage Limit(사용량 제한 설정) 버튼을 클릭합니다.
![사용량 제한 설정 버튼](https://jumpshare.com/share/JKA39F5kCPQ9FKd8dF9g+/Screen+Shot+2025-09-01+at+8.16.25+PM.png)

- 숫자 필드에 하루에 허용할 실행 횟수를 입력합니다.

## 자주 묻는 질문

**Q: 누가 일일 사용량 제한 기능에 액세스할 수 있나요?**
이 기능은 에이전시 소유자와 관리자만 사용할 수 있습니다.

**Q: 일일 사용량 제한이 중요한 이유는 무엇인가요?**
사용량 기반 앱의 일일 지출을 제한하여 예상치 못한 비용을 방지하는 데 도움이 됩니다.

**Q: 사용량 제한을 설정한 후 조정할 수 있나요?**
네, 에이전시 관리자는 예산 요구사항에 더 잘 맞도록 필요에 따라 제한을 수정할 수 있습니다.

**Q: 설치한 앱만 제한할 수 있나요?**
네. 앱 사용량 제한에서 App Status(앱 상태) → Installed apps(설치된 앱)을 사용하여 이미 사용 중인 앱만 필터링하고 제한을 설정할 수 있습니다.
