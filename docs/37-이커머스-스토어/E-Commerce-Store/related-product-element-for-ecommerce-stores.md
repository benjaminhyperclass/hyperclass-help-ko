# 이커머스 스토어용 관련 상품 요소

### 개요

Hyperclass에서 이제 스토어 운영자가 상품 상세 페이지에서 관련 상품을 추천할 수 있습니다. 이 기능은 사용자에게 추천 상품이나 관련 상품을 보여줌으로써 전반적인 상품 발견 경험을 개선하고, 교차 판매 기회를 향상시킵니다.

### 사용 방법

- 스토어의 상품 상세 페이지를 페이지 빌더에서 열고, **관련 상품 표시(Show related products)** 토글을 활성화하세요. 다음 속성들을 커스터마이징할 수 있습니다:
  - 표시할 상품 개수 (*기본값: 6개*)
  - 데스크톱과 모바일에서 상품당 컬럼 수 (*기본값: 데스크톱 3개, 모바일 2개*)
  - 관련 상품 섹션 제목 (*기본값: "이런 상품도 좋아하실 거예요"*)

### 참고사항

- 동일한 컬렉션에 포함된 상품들이 해당 상품의 상세 페이지에서 관련 상품으로 표시됩니다. 예를 들어:
  - "iPhone 15"라는 상품이 "Apple"과 "Summer Sale" 컬렉션에 추가되어 있다면
  - "Apple"과 "Summer Sale" 컬렉션에 추가된 모든 상품이 표시됩니다
  - 먼저 "Apple" 컬렉션의 모든 상품이 표시되고, 그 다음 "Summer Sale" 상품이 표시되며, 이는 사용자가 설정한 관련 상품 제한 수 내에서 이루어집니다 (최대 10개까지 제한)
- 상품 목록 페이지의 이미지들은 이제 **1:1 비율**로 제한되며, 모든 상품과 이미지 간의 일관성과 더 나은 UI를 유지하기 위해 해당 크기로 맞춰집니다.
