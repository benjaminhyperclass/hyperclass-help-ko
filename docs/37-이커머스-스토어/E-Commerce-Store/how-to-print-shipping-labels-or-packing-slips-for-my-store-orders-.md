# 스토어 주문의 배송 라벨이나 포장 명세서를 인쇄하는 방법

## 개요:
이제 스토어 운영자는 이커머스 스토어의 모든 주문에 대해 배송 라벨과 포장 명세서를 생성하여 주문 처리 과정을 효율화할 수 있습니다. 이 새로운 기능을 통해 전체 주문이나 처리 중인 특정 상품에 대한 라벨을 인쇄할 수 있습니다.

생성되는 라벨과 명세서에는 다음 정보가 포함됩니다:

- 주문 ID
- 주문 날짜
- 스토어 이름
- 배송 주소 및 청구 주소
- 상품별 세부 목록:
  - 옵션명
  - 상품 이미지
  - 수량
- 발송인 주소 (스토어 주소)

## **사용 방법**

- 스토어 운영자는 다음 경로에서 배송 라벨 또는 포장 명세서 인쇄 기능에 접근할 수 있습니다: Payments(결제) > Orders(주문) > 스토어 주문 선택 > "Print Shipping Label(배송 라벨 인쇄)"
- 배송 라벨은 세 곳에서 생성할 수 있습니다:

주문 세부 페이지: "Print Shipping Label(배송 라벨 인쇄)" 버튼을 클릭하면 미처리 주문과 처리 완료된 주문이 각각 별도 페이지로 구분된 PDF를 다운로드할 수 있습니다.

![배송 라벨 인쇄 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155030203469/original/NCGmvEDm2w9a4npuDX5YxR6cjZWVxqjwlw.png?1722412012)

- 주문 처리 중: 처리 과정에서 선택한 상품과 수량이 나열되며, "Print Shipping Label(배송 라벨 인쇄)" 버튼을 클릭하면 해당 상품에 대한 배송 라벨이 생성됩니다.

![주문 처리 중 배송 라벨 생성](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155030203560/original/eWb54GdVuIO5k6vfXX4vjMubraVXBK2Byg.png?1722412127)

![배송 라벨 생성 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155030203496/original/nUmAXc2aHcvewMpXq_OuDPqtNx-F7b6wBg.png?1722412034)

- **부분 처리된 주문:** 부분 처리된 주문 옆의 점 3개 버튼을 클릭하면 해당 주문의 포장 명세서 또는 배송 라벨을 다운로드하는 옵션이 나타납니다.

![부분 처리된 주문 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155030203753/original/Zd-qJRIsdHpXDl0If7G8D4hmlyqHDVb4Pw.png?1722412283)

## 참고사항:

- 배송 라벨/포장 명세서는 이커머스 스토어 주문에서만 생성할 수 있습니다.
- 여러 번 부분 처리된 주문의 경우, 배송 라벨을 생성할 때 각 처리 건마다 별도 페이지에 상품이 나열됩니다.
- 현재 배송 라벨에서는 청구 주소와 배송 주소가 동일하게 표시됩니다. 체크아웃 페이지에 청구 주소 필드가 출시되면 수정 가능하도록 개선될 예정입니다.
