# Xero 연동에서 계정과목표(Chart of Accounts) 확인하는 방법

Xero와 연결한 후 CRM에서 계정과목표가 보이지 않는다면, 아래 단계를 따라 활성화하세요.

## 1단계: Xero에서 설정(Settings) 메뉴로 이동

- Xero 대시보드에서 조직을 선택합니다.

- 드롭다운 메뉴에서 Settings(설정)을 클릭합니다.
![Xero 설정 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155052657175/original/bPmYLbMmVN8r8q2T0SCWskIbj1jOWV0mMg.jpeg?1756458315)

## 2단계: 고급 설정(Advanced Settings) 열기

- 조직 설정에서 아래로 스크롤하여 "Looking for advanced settings?"를 클릭합니다.
![고급 설정 찾기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155052657176/original/npn3KpIABBrgAj5wZlzdyacFuuYC04W-Fg.jpeg?1756458315)

## 3단계: 계정과목표(Chart of Accounts) 접근

- Advanced Accounting 하위에서 "Chart of accounts"를 클릭합니다.
![계정과목표 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155052657174/original/4RS7ugmXw3Nn6k1XtkoQj6Ahp0oXHqyQPQ.jpeg?1756458314)

## 4단계: 해당 계정 선택

- 계정 목록에서 연결하려는 계정을 선택합니다(예: Prepayments).

- Edit을 클릭하여 해당 계정의 세부 정보를 편집합니다.
![계정 편집](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155052657177/original/p37sApxXTyqFr7FAnKfirLfjngedVQ1Eaw.png?1756458317)

## 5단계: 계정의 결제 기능 활성화

- 편집 팝업에서:

"Enable payments to this account" 옵션에 체크 ✅합니다.

- Save(저장)를 클릭합니다.
![결제 기능 활성화](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155052657178/original/53ywW6oFcgUI7Cnb_xdXEN_prPEj2sf4Hw.png?1756458318)

## 6단계: CRM에서 연결

- CRM의 Xero 연동 설정으로 돌아갑니다.

- Connected Account에서 방금 활성화한 계정을 선택합니다(예: Prepayments).
![CRM에서 계정 연결](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155052657173/original/pgQnADAFx8b5CM33vkKYz3BiKAq0pDWdQw.jpeg?1756458313)

✅ 완료되면 이제 CRM에서 계정과목표를 확인하고 사용할 수 있습니다.
