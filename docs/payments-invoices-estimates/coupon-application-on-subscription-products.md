# 구독 상품의 쿠폰 적용

**개요**

쿠폰 생성 과정에 구독 상품에 쿠폰이 얼마나 오래 적용될지 정의하는 기능이 추가되었습니다. 비즈니스에서 구독 할인의 고정 기간을 지정할 수 있어, 프로모션 제안에 더 큰 유연성을 제공합니다.

**프로세스**

1. 쿠폰 생성 또는 편집

- `Coupons(쿠폰) > Create Coupon(쿠폰 생성)` 또는 `Coupons(쿠폰) > Edit Coupon(쿠폰 편집)`으로 이동합니다.
- 할인 설정에서 "해당 시 기간 또는 영구적으로 반복/향후 결제에도 적용"이라는 체크박스를 찾습니다.
- 체크하면 다음 옵션 중에서 선택할 수 있습니다:
  - Forever(영구): 모든 반복 결제에 할인이 적용됩니다.
  - Multiple Months(여러 달): 지정된 개월 수 동안 할인이 적용됩니다. 선택 시 개월 수를 입력할 입력 필드가 나타납니다.

2. 구독 상품에 쿠폰 적용

- 일회성 상품에 쿠폰이 적용되면, 기간 설정은 할인 로직에 영향을 주지 않으며 한 번만 적용됩니다.
- 구독 상품에 적용되면, 선택된 기간에 따라 할인이 적용됩니다:
  - Once(한 번): 첫 번째 결제에만 할인이 적용됩니다.
  - Forever(영구): 모든 반복 결제에 할인이 적용됩니다.
  - Multiple Months(여러 달): 정의된 개월 수 동안 할인이 적용되고 그 후 중단됩니다.

3. 적용된 쿠폰 확인

- 선택된 기간은 쿠폰 상세 정보 및 관리 화면에 반영됩니다.
- 구독 상세 페이지와 주문 상세 페이지에서 할인이 결제 세부 정보 옆에 다음과 같은 문구로 표시됩니다: "처음 4개월간 10% 할인", "첫 달 10달러 할인", "모든 결제 10% 할인" 또는 "첫 결제 10달러 할인"

**예시**

- 10개월간 진행되는 월간 구독 상품 - "Multiple Months(여러 달)"가 선택되고 "4"가 입력된 쿠폰으로 구매한 경우, 구독이 바로 활성 상태로 시작하든 먼저 체험 후 활성 상태가 되든 상관없이 쿠폰 적용 날짜부터 4개월간 쿠폰이 적용됩니다.
- 3년간 진행되는 연간 구독 상품
  - "Forever(영구)"가 선택된 쿠폰으로 구매한 경우, 구독 전체 기간 동안 쿠폰이 적용됩니다.
  - "Multiple Months(여러 달)"가 선택되고 4가 입력된 쿠폰으로 구매한 경우, 향후 4개월 내 결제에만 적용될 수 있으므로 12개월 후 연간 갱신에는 쿠폰이 적용되지 않습니다.
- 6개월간 진행되는 주간 구독 상품 - "Multiple Months(여러 달)"가 선택되고 "2"가 입력된 쿠폰으로 구매한 경우, 구독이 바로 활성 상태로 시작하든 먼저 체험 후 활성 상태가 되든 상관없이 쿠폰 적용 날짜부터 2개월간 쿠폰이 적용됩니다. 따라서 구독 상품 구매 후 2개월 동안 매주 약 8회 쿠폰이 적용됩니다.

**추가 사항**

- 이 기능은 구독 빈도(일간, 주간, 월간, 분기별, 연간)가 몇 번의 결제에 할인이 적용되는지에 영향을 주지 않도록 보장합니다. 정의된 기간만 적용됩니다.
- 예약된 구독의 경우, 구독 시작일(활성 또는 체험)부터 쿠폰이 적용됩니다. 즉, 구독 생성 시 입력으로 사용된 예약 날짜입니다.
- 기존 쿠폰은 원래 동작을 유지합니다:
  - 체크박스가 이전에 체크되어 있었다면, 기본 기간은 Forever(영구)입니다.
  - 체크되지 않았다면, 기본 기간은 Once(한 번)입니다.
