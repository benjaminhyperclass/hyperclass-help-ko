# 앱 내 번역 피드백 제출 방법

이 기능을 사용하면 에이전시 관리자가 별도의 외부 티켓 없이 Hyperclass 인터페이스에서 직접 잘못되거나 불분명한 번역을 신고할 수 있습니다. 이 가이드는 기능의 작동 방식, 사용 대상, 표시 위치, 그리고 스크린샷과 함께 고품질 피드백을 제출하여 일상적인 워크플로우에서 현지화 정확성을 유지하는 방법을 설명합니다.

## 번역 피드백 기능이란?

번역 피드백 신고(Report Translation Feedback)는 적격 사용자가 현재 페이지를 떠나지 않고도 번역 문제를 제출할 수 있도록 상단 바에 신고 아이콘을 표시하는 내장 기능입니다. 제출된 피드백은 제품 팀이 검토가 필요한 문자열을 빠르게 식별하여 플랫폼 전반의 정확성과 일관성을 향상시키는 데 도움이 됩니다.

참고: 플랫폼 언어가 영어로 설정되어 있으면 이 기능이 표시되지 않습니다.

- 문제를 발견한 페이지에서 바로 피드백을 제출하세요.
- 정확한 문자열을 강조하기 위해 스크린샷을 첨부하세요.
- 신고가 검토를 위해 전달되며, 수정되면 알림을 받게 됩니다.

## 번역 피드백 기능의 주요 이점

이점을 이해하면 일반 지원 티켓 대신 앱 내 신고를 언제 사용할지 결정하는 데 도움이 됩니다.

- 실시간 캡처: 사용 중인 정확한 화면에서 즉시 문제를 신고하세요.
- 낮은 노력: 일상적인 번역 수정을 위한 외부 티켓을 피할 수 있습니다.
- 높은 품질: 검토자가 정확한 문자열을 찾을 수 있도록 스크린샷을 추가하세요.
- 빠른 반복: 중앙 집중식 신고로 현지화 개선을 간소화합니다.

## 번역 피드백 기능 사용법

UI에서 직접 실행 가능한 번역 피드백을 제출하려면 다음 단계를 따르세요.

1. 에이전시 소유자/관리자로 로그인했고 언어가 영어가 아닌지 확인하세요.
2. 번역 문제가 나타나는 페이지를 열어주세요.
3. 번역 문제 신고(Report Translation Issue) 아이콘을 클릭하세요 (상단 바).

![번역 문제 신고 아이콘](https://jumpshare.com/share/HnGm7ViM2dA1ylQO2LjV+/Screen+Shot+2025-09-15+at+8.11.17+PM.png)

4. 패널의 내용이 선택된 플랫폼 언어로 표시됩니다.
5. 문제가 있는 영역과 위치를 명확히 보여주는 스크린샷을 첨부하세요(필수).

![스크린샷 첨부](https://jumpshare.com/share/dz0ne7Of1YemNPH5Emvj+/Screen+Shot+2025-09-15+at+8.15.27+PM.png)

6. 문제 유형(예: 잘못된 번역, 문법, 잘못된 형식)을 선택하세요.

![문제 유형 선택](https://jumpshare.com/share/3B6FCm7pj5mk16opqtZu+/Screen+Shot+2025-09-15+at+8.18.33+PM.png)

7. 간단한 설명을 추가하세요.

![설명 추가](https://jumpshare.com/share/VEssGpFoVnqvkWLq2UXj+/Screen+Shot+2025-09-15+at+8.20.26+PM.png)

8. "어디서 보셨나요?" 섹션에 경로를 추가하세요 (예: 대시보드 > 퍼널 > 스텝 빌더).

![경로 추가](https://jumpshare.com/share/ko4QrRgXcpgczqq4QkGa+/Screen+Shot+2025-09-15+at+8.23.40+PM.png)

9. 문제 신고하기(Report Issue) 버튼을 클릭하세요.
10. 문제가 해결되면 알림을 받게 됩니다.

![신고 완료](https://jumpshare.com/share/vUne13Vnw47OexaRpfCg+/Screen+Shot+2025-09-15+at+8.24.54+PM.png)

## 자주 묻는 질문

**번역 피드백 신고 기능에 누가 액세스할 수 있나요?**
에이전시 소유자와 에이전시 관리자 역할만 가능합니다.

**아이콘이 보이지 않는 이유가 뭔가요?**
영어 언어 설정에서는 이 기능이 숨겨지며, 에이전시 소유자/관리자 역할에만 표시됩니다.

**신고에 무엇을 포함해야 하나요?**
간단한 설명, 올바른 표현, 그리고 문제를 맥락과 함께 보여주는 스크린샷을 포함하세요.

**언제 수정되었는지 어떻게 알 수 있나요?**
신고된 번역이 해결되면 알림을 받게 됩니다.
