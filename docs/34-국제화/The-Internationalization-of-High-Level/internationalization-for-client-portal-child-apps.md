# 클라이언트 포털 및 자식 앱 국제화

이 아티클에서는 클라이언트 포털(Client Portal)과 자식 앱(Child App)의 표시 언어를 변경하는 방법을 설명합니다. 웹과 모바일 플랫폼 모두에서 여러 언어를 지원하므로, 사용자가 선호하는 언어로 경험을 관리할 수 있습니다.

## 언어 국제화란?

언어 국제화를 통해 사용자는 클라이언트 포털과 자식 앱을 선호하는 언어로 이용할 수 있습니다. 이 기능은 영어가 아닌 언어를 사용하는 사용자의 편의성을 향상시키고 지역별로 현지화된 경험을 제공합니다. 클라이언트 포털 관리자는 모든 포털 사용자를 위한 기본 언어를 설정할 수도 있습니다. 사용자는 여전히 웹과 모바일 플랫폼 모두에서 자신의 계정 설정을 통해 기본값을 변경할 수 있습니다. 관리자나 사용자 수준에서 언어가 선택되지 않았다면 포털은 영어를 사용합니다.

## **언어 국제화의 주요 장점**

언어 커스터마이징은 사용성을 향상시키고 전 세계 사용자에게 더욱 접근하기 쉬운 경험을 보장합니다.

- 클라이언트 포털과 자식 앱의 웹 및 모바일 버전 모두에서 이용 가능
- 여러 전 세계 지역에서 12개 언어 지원
- 관리자가 모든 클라이언트 포털 사용자를 위한 기본 언어 설정 가능
- 영어가 아닌 언어 사용자의 접근성 향상
- 언어 변경이 빠르고 사용자별로 적용
- 고객이 모국어로 설정을 관리할 수 있음
- 언어가 선택되지 않은 경우 영어로 기본 설정

## 기본 언어 설정하기 (관리자)

모든 클라이언트 포털 사용자가 로그인 페이지와 포털 전체에서 동일한 언어를 보도록 하고 싶을 때 사용합니다.

- `Memberships(멤버십) → ClientPortal(클라이언트 포털) → Settings(설정) → Language Settings(언어 설정)`으로 이동합니다.

![클라이언트 포털 언어 설정 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063054806/original/Cjl9IIqShgK2vnsVVnQ78oBfkmTsBntyFw.png?1768910061)

- 언어를 선택합니다.

![언어 선택 드롭다운](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063054891/original/b-mfpa8o0_sv5y2frWoV--pI5_a9-p_Xew.png?1768910118)

- Save(저장)를 클릭합니다. 이제 사용자는 로그인 페이지뿐만 아니라 웹에서도 언어 변경을 확인할 수 있습니다.

## 웹에서 언어 변경하기 (사용자/고객)

클라이언트 포털에 로그인한 후 언어 기본 설정을 변경할 수 있습니다.
**Memberships(멤버십) → Client Portal(클라이언트 포털)**로 이동하여 포털에 로그인하거나 클라이언트 포털 URL에서 직접 로그인합니다.

![클라이언트 포털 접근 과정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046796710/original/kG2AXbYNNnknefEyR0HqZ4ELcIXgx1kvqg.gif?1747483477)

언어 변경 단계:

- 우측 상단의 로고를 클릭하고 Manage your Account(계정 관리)를 클릭합니다.

![계정 관리 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046796853/original/tfJGTmozjpQ4SGxhxOQfSfIbWfQTX_PuiQ.png?1747483819)

- Manage Account(계정 관리) 하에서 Language Settings(언어 설정)를 선택하고 드롭다운 메뉴에서 선호하는 언어를 선택합니다.

![언어 설정 변경 과정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046796976/original/gRSlkZmvSH6aD1Gnd8W8KvIufLTYJVFPIQ.gif?1747484034)

## **모바일 앱에서 언어 변경하기**

모바일 기기를 통해 앱에 접근하는 사용자도 몇 번의 탭만으로 언어 설정을 업데이트할 수 있습니다.

**모바일에서 언어 변경 단계:**

- 하단 네비게이션 바에서 **Profile(프로필)**을 탭합니다.
- **Settings(설정)**로 이동합니다.
- **Language Settings(언어 설정)**을 탭합니다.
- 목록에서 원하는 언어를 선택합니다.
- **Save(저장)**를 탭하여 업데이트를 적용합니다.

## 지원 언어

웹과 모바일 플랫폼 모두에서 현재 지원되는 언어는 다음과 같습니다:

| 언어 | 지역 |
|------|------|
| 독일어 | DE |
| 스페인어 | ES |
| 프랑스어(캐나다) | CA |
| 프랑스어(프랑스) | FR |
| 네덜란드어 | NL |
| 포르투갈어(포르투갈) | PT |
| 포르투갈어(브라질) | BR |
| 이탈리아어 | IT |
| 스웨덴어 | SE |
| 핀란드어 | FI |
| 노르웨이어 | NO |

**참고**: **영어**가 기본 언어이며, 다른 선택이 없을 경우 사용됩니다.

## **자주 묻는 질문**

**Q: 언어 변경이 즉시 적용되나요?**
네, 저장하면 새 언어가 즉시 적용됩니다.

**Q: 언어를 여러 번 바꿀 수 있나요?**
네, 언제든지 언어 설정을 변경할 수 있습니다.

**Q: 기능이 완전히 번역되지 않은 경우는 어떻게 하나요?**
번역이 제공되지 않는 경우 인터페이스는 영어로 기본 설정됩니다.

**Q: 모든 신규 사용자를 위한 기본 언어를 설정하는 방법이 있나요?**
`Memberships(멤버십) → ClientPortal(클라이언트 포털) → Settings(설정) → Language Settings(언어 설정)`으로 이동하여 언어를 선택하고 Save(저장)를 클릭합니다. 이 기본값은 모든 사용자의 /login 페이지와 클라이언트 포털 웹 경험에 적용됩니다.
