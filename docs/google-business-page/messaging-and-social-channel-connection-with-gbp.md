# 구글 비즈니스 프로필에 메시징 및 소셜 채널 연결하기

이제 구글 비즈니스 프로필 페이지에 메시징 채널(왓츠앱, 문자/SMS)과 소셜 채널(페이스북, 인스타그램, 링크드인, 유튜브, 틱톡, X, 핀터레스트)을 연결할 수 있습니다. 고객이 친숙한 방법으로 더 쉽게 연락할 수 있게 하고, 고객 참여와 리드 확보를 위한 추가 경로를 만들어보세요.

✅ 중요: 대화(Conversations) 모듈은 CRM을 통해 연동된 채널의 메시징만 지원합니다.

## 구글 비즈니스 프로필에 소셜 채널 연결하는 방법

**1단계:** 하위 계정 대시보드 페이지에서 설정(Settings)을 클릭하고 연동(Integrations)으로 이동하세요.

**2단계:** 구글 비즈니스 프로필 계정을 연결하고 페이지 선택 모달로 이동하세요.
![구글 비즈니스 프로필 계정 연결](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050129431/original/LUP0sRL2-E3CmPS-Uqbv8cJlun0MiKPj5Q.png?1753074330)

**3단계:** 메시징 채널을 연결하려면 왓츠앱 및 메시지 탭을 클릭하거나, 채널 연결(Connect Channels) 버튼을 클릭하세요.
![메시징 채널 연결](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050129495/original/YsK2Uq_wKJZOANXWPYHqg4p8bZj7gqwqAQ.png?1753074411)

**4단계:** 드롭다운에서 CRM에 연결된 Hyperclass 전화번호와 왓츠앱 번호를 선택하거나, 직접 입력하여 번호를 추가할 수 있습니다. 원하는 소셜 채널 링크도 추가할 수 있습니다.
![채널 정보 입력](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050129936/original/oVSNyVya9QVlTM8qePNYlOa3DLn9qdilHg.png?1753074846)

**5단계:** 연결(Connect) 버튼을 클릭하세요.

![연결 완료](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050129929/original/qz_mUO_Hw9iLiHn1p7xxozJ1AEkJf7EVeg.png?1753074828)

## 소셜 채널 지원 URL 형식

구글 비즈니스 프로필의 소셜 미디어 링크는 아래 형식을 준수해야 합니다.

| 소셜 플랫폼 | 형식 |
|------------|------|
| 페이스북 | https://www.facebook.com/{username} |
| 인스타그램 | https://www.instagram.com/{username} |
| 링크드인 | https://www.linkedin.com/in/{username}<br>https://www.linkedin.com/company/{username} |
| 핀터레스트 | https://www.pinterest.com/{username} |
| 틱톡 | https://www.tiktok.com/{username} |
| X (구 트위터) | https://www.twitter.com/{username} |
| 유튜브 | https://www.youtube.com/channel/{username}<br>https://www.youtube.com/user/{username}<br>https://www.youtube.com/@{username} |

자세한 URL 가이드라인은 각 플랫폼의 공식 문서를 참고하세요.

## 이 채널들이 어떻게 표시되나요?

메시징 채널은 두 개 모두 연결할 수 있지만, 구글 비즈니스 프로필 결과에는 한 번에 하나씩만 표시됩니다. 모든 소셜 채널은 구글 비즈니스 프로필 페이지 결과의 정보(About) 섹션에 표시됩니다.

✅ 중요: 왓츠앱과 문자 메시지를 모두 추가한 경우, 구글 비즈니스 프로필 결과에는 문자 메시지가 표시됩니다.

## 구글 비즈니스 프로필 결과에 문자/왓츠앱 메시지 옵션이 보이지 않아요

문자 및 왓츠앱 채널을 추가했는데 구글 비즈니스 프로필 결과에 보이지 않는다면, 주로 구글에서 추가된 각 번호에 대해 유효성 검사를 진행하기 때문입니다. 다음 단계를 따라 확인해보세요:

1. CRM의 페이지 선택 모듈에서 **구글 비즈니스 프로필 업데이트(Update GBP)**를 클릭하거나 구글 콘솔의 페이지 편집으로 이동하여 구글 비즈니스 프로필 페이지 설정에 접속하세요.

2. 편집이 승인되었는지 확인하세요. 승인은 구글 팀에서 처리하며 저희 쪽에서는 영향을 줄 수 없습니다. 편집이 승인되지 않은 경우 구글에 직접 문의하실 수 있습니다.

![구글 비즈니스 프로필 승인 상태 확인](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051425989/original/Tg29cEY35lAz0AiexBLit76LJlyh5lqR_A.png?1754995806)
