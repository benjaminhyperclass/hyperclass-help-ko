연락처 폼은 리드 수집, 문의 접수, 고객 피드백 수집에 필수적인 도구입니다. Hyperclass를 사용하면 연락처 폼을 쉽게 만들고 커스터마이징한 후 웹사이트나 퍼널에 임베드할 수 있어요. 이 단계별 가이드를 따라 Hyperclass에서 연락처 폼을 만들어보세요.

# 1단계: 폼 빌더로 이동하기

- Hyperclass 계정에 로그인하세요.
- 올바른 클라이언트 계정(하위 계정)으로 이동하세요.
- 왼쪽 메뉴에서 Sites(사이트) → Forms(폼)를 클릭하세요.
- + Create New Form(새 폼 만들기) 버튼을 클릭하여 새 폼을 시작하세요.

![폼 만들기 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155040687777/original/35tZOjm5Pp9UgivTfidlbyetiv22AjZ7pw.png?1738207766)

기존 "Contact Forms(연락처 폼)" 템플릿 중 하나를 선택하거나, 필요에 맞는 새로운 폼을 만들 수 있어요.

![템플릿 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155040687809/original/kOUtoeeh3xc0wzWtEuWfMdWKRrHKg45xVw.png?1738207981)

# 2단계: 폼 필드 추가하기 (선택사항)

Form Elements(폼 요소) 섹션에서 필요한 필드를 드래그 앤 드롭으로 추가하세요. 예를 들어 Full Name(전체 이름), Email(이메일), Phone Number(전화번호), Message/Custom Text Area(메시지/커스텀 텍스트 영역) 등을 추가할 수 있어요. 

추가 필드가 필요한 경우 다음을 활용하세요:
- Custom Fields(커스텀 필드): 고유한 데이터 수집
- Dropdowns(드롭다운), checkboxes(체크박스), radio buttons(라디오 버튼): 특정 옵션 제공

필드를 드래그하여 원하는 순서로 재배열할 수도 있어요.

이 예시에서는 연락처 폼에 "query(문의사항)"를 필드로 사용하겠습니다.

# 3단계: 폼 설정 커스터마이징 (선택사항)

## 알림 설정

본인과 폼을 작성하는 리드 모두에게 이메일 알림을 설정할 수 있어요.

![알림 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155040688055/original/I8uu5SgDXxTNgR34i9-rSxHI9ITWPZcl-w.png?1738209094)

## 폼 스타일 & 디자인

Styles(스타일) 탭을 클릭하여 폼의 배경, 텍스트 색상, 간격을 조정하세요.

반응형 디자인을 위해 Full Width(전체 너비) 옵션을 토글할 수 있어요.

![스타일 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155040688084/original/6LJu6G-RYXcivSnBXB7B0U_J3b8ISHDhUw.png?1738209199)

## 폼 액션 설정

Settings(설정)에서 폼 제출 후 어떤 일이 일어날지 선택하세요:
- Show a Thank You Message(감사 메시지 표시): 커스텀 메시지 입력
- Redirect to Another Page(다른 페이지로 리다이렉트): URL 입력
- Trigger a Workflow or Automation(워크플로우 또는 자동화 트리거): 예를 들어 후속 이메일 발송

![폼 액션 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155040688160/original/jrWU1ek7ZQazi79eLzTOX5PQ2HnkWvNBJg.png?1738209387)

# 4단계: 폼 저장 및 임베드하기

만족스러우면 Save Form(폼 저장)을 클릭하세요.

Integrate Form(폼 통합) 버튼을 클릭하여 임베드 옵션을 확인하세요.

![폼 통합](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155040688230/original/djlqTVB0SY2ZcsP_FGZicJJCCohw5vE0Mg.png?1738209572)

이 단계들을 따르면 Hyperclass에서 완전히 작동하는 연락처 폼을 만들어 리드를 수집하고 잠재 고객과 소통할 수 있어요. 🎯
