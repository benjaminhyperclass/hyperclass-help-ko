## 스마트 리스트를 사용하는 이유

- **시간 절약** – 수동으로 목록을 업데이트하는 대신 세분화를 자동화합니다.

- **타게팅 향상** – 행동, 리드 상태, 참여도를 기반으로 연락처를 그룹화합니다.

- **CRM 조직 개선** – 연락처 목록을 체계적이고 검색 가능하게 유지합니다.

- **자동화 강화** – 워크플로우에서 스마트 리스트를 사용하여 자동화된 액션을 수행합니다.

---

## 스마트 리스트란?

스마트 리스트는 미리 정의된 필터와 조건에 따라 자동으로 업데이트되는 동적 연락처 목록입니다. 수동으로 업데이트하지 않으면 변경되지 않는 정적 목록과 달리, 스마트 리스트는 새로운 연락처가 지정된 기준을 충족하거나 더 이상 충족하지 않을 때 지속적으로 조정됩니다. 사용자가 커스텀 필드와 같은 특정 조건을 기반으로 연락처를 사용자 지정하고 필터링할 수 있습니다. 표시할 컬럼을 선택할 수 있으며, 연락처 데이터가 변경되면 스마트 리스트가 실시간으로 업데이트됩니다.

이 기능은 **하위 계정(Subaccount) → 연락처(Contacts) 탭 → 스마트 리스트(Smartlists)**에서 찾을 수 있습니다.

## 스마트 리스트는 어떻게 작동하나요?

- **실시간 업데이트**: 태그, 커스텀 필드, 참여 활동 또는 기회 단계와 같은 필터를 기반으로 연락처가 자동으로 추가되거나 제거됩니다.

- **자동화된 세분화**: 수동 업데이트 없이 리드, 활성 고객 또는 잠재 고객 목록을 쉽게 만들 수 있습니다.

- **고급 필터링**: 스마트 리스트를 사용하면 여러 필터를 동시에 적용할 수 있습니다(예: "지난 30일 동안 이메일을 열었음" + "미결제 인보이스가 있음").

- **워크플로우 연동**: 스마트 리스트는 자동화된 워크플로우를 트리거하여 대규모로 개인화된 참여를 보장할 수 있습니다.

## 스마트 리스트 설정하기

Hyperclass에서 스마트 리스트를 만드는 것은 특정 기준에 따라 연락처를 자동으로 세분화할 수 있는 간단한 과정입니다. 목록을 수동으로 업데이트하는 대신, 스마트 리스트는 연락처 그룹이 체계적이고 실시간으로 업데이트되도록 보장합니다. 첫 번째 스마트 리스트를 만들기 위해 다음 단계를 따르세요:

- 연락처(Contacts) > 연락처/스마트 리스트(Contacts/Smart Lists)로 이동하여 "더 많은 필터(More Filters)"를 클릭합니다.

- 화면 오른쪽의 드롭다운 메뉴에서 하나 또는 여러 필터를 선택합니다.

- 필터를 선택한 후, 필요한 세부사항을 입력하고 "적용(Apply)"을 클릭하여 필터를 확정합니다.

- 더 많은 필터를 추가하려면 "AND" 옵션을 선택합니다.

- 필터를 목록으로 영구적으로 저장하려면 화면 하단의 "스마트 리스트로 저장(Save as smart list)"을 클릭합니다.

![스마트 리스트 만들기 과정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155041733215/original/JYZsQNRq6e2sAIa_XqmGiqLEDRR6dwv7BA.gif?1739800334)

## 스마트 리스트 관리하기

스마트 리스트는 비즈니스 요구사항에 더 잘 맞도록 수정, 필터링 및 관리할 수 있습니다. 다양한 필터를 적용하거나 조건을 조정하거나 기존 목록을 복제하여 더 정교한 세분화를 할 수 있습니다.

- 스마트 리스트에 접근하려면 연락처 섹션으로 이동하여 상단에서 스마트 리스트 이름을 선택하여 엽니다.

- 기존 스마트 리스트를 수정하려면 열면 오른쪽에서 해당 목록에 저장된 필터를 볼 수 있습니다.

- 필터를 제거하려면 휴지통 아이콘이나 X를 선택합니다. 필터를 수정하려면 연필 아이콘이나 드롭다운 화살표를 클릭합니다. 저장 아이콘을 사용하여 변경사항을 저장합니다.

- 연락처 페이지 오른쪽 상단의 "스마트 리스트 관리(Manage Smart Lists)"로 이동하여 스마트 리스트를 관리할 수 있습니다. 이 영역에서 오른쪽 아이콘을 사용하여 스마트 리스트를 **복제, 공유, 이름 변경 또는 삭제**할 수 있습니다.

## 스마트 리스트 만들기 예시

지난 90일 동안 연락한 적이 없거나 한 번도 연락하지 않은 모든 연락처의 목록을 만들어보세요.

1. 스마트 리스트 > 전체(All) 탭으로 이동
2. 더 많은 필터(More Filters) 클릭
3. '필터 추가(Add Filter)'가 표시된 텍스트 상자에 'Last Activity'를 입력하면 '마지막 활동(Last Activity)' 필터가 옵션 중 하나로 나타납니다
4. 마지막 활동 필터를 선택합니다. 추가되면 '다음보다 많음(More Than)' 라디오 버튼을 클릭하고 90일을 입력합니다
5. 컬럼 드롭다운을 클릭하여 마지막 활동(Last Activity) 컬럼을 선택합니다
6. + 아이콘을 눌러 이 목록에 이름을 지정하고 저장합니다

![스마트 리스트 만들기 예시](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155041734610/original/DYq6diiU1eO5FTlQUuwDWLaIjFopdJ1LaQ.gif?1739801031)

## 스마트 리스트에서 컬럼 순서 바꾸기

스마트 리스트는 드래그 앤 드롭 컬럼 관리를 지원하므로 목록 보기를 빠르게 재정렬하고 가장 중요한 필드를 앞쪽과 중앙에 유지할 수 있습니다.

- 연락처(Contacts) → 스마트 리스트(Smart Lists)로 이동한 다음 편집하려는 스마트 리스트를 엽니다.

- 목록 보기에서 컬럼/필드 관리 옵션을 엽니다(일반적으로 테이블 상단 근처에 있으며 표시할 필드를 관리할 수 있게 해줍니다).

- 컬럼 관리자에서 컬럼을 원하는 순서로 드래그 앤 드롭합니다.

- 컬럼 관리자를 닫으면 스마트 리스트로 돌아가며, 새로운 컬럼 순서가 즉시 반영됩니다.

![컬럼 순서 바꾸기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155061802130/original/TrhoI1Q3LVsXorsM-ctDqTz1HI7GvJb9kg.gif?1767376414)

**참고:** 스마트 리스트는 가장 최근에 저장된 컬럼 순서와 가시성을 저장합니다. 나중에 컬럼을 재정렬하거나 필드를 표시/숨김 처리하는 경우, 해당 변경사항을 유지하려면 스마트 리스트를 다시 저장해야 합니다. 그렇지 않으면 마지막으로 저장된 레이아웃을 계속 사용합니다.

## 자주 묻는 질문

**Q: 기존 연락처를 기존 스마트 리스트에 수동으로 추가할 수 있나요?**

아니요, 연락처는 스마트 리스트에 수동으로 추가되지 않습니다. 해당 스마트 리스트에 설정된 필터와 조건에 따라 자동으로 추가됩니다. 스마트 리스트의 필터 조건과 일치하는 태그를 연락처에 추가하면 자동으로 해당 스마트 리스트에 추가됩니다.

**Q: 사용자가 '설정(settings)' 접근 권한 없이 스마트 리스트를 관리할 수 있나요?**

사용자는 '설정' 접근 권한 없이는 스마트 리스트를 관리할 수 없습니다. 이 문제는 향후 업데이트를 위해 Hyperclass의 로드맵에 추가되었습니다.

**Q: 누가 스마트 리스트를 만들고 공유했는지 확인할 방법이 있나요?**

Hyperclass는 누가 스마트 리스트를 만들고 공유했는지에 대한 정보를 기록하지 않습니다.

**Q: 모든 사용자가 스마트 리스트를 볼 수 있나요, 아니면 만든 사용자만 볼 수 있나요?**

스마트 리스트는 로케이션 수준이 아닌 사용자 수준에서 만들어집니다. 하지만 "전역 대 로컬" 스마트 리스트를 통해 사용자는 이제 자신의 스마트 리스트를 전체 계정에 푸시하여 모든 사용자에게 보이게 할 수 있습니다.

**Q: 스마트 리스트를 계정 간에 이전하거나 스냅샷으로 이동할 수 있나요?**

이 글을 쓰는 시점에서는 스마트 리스트를 한 계정에서 다른 계정으로 이동하거나 스냅샷으로 가져오는 기능이 없습니다.

**Q: 스마트 리스트에 이메일 캠페인을 예약하면 발송 전에 목록이 업데이트되나요?**

네. 스마트 리스트는 동적이므로 수신자 목록은 캠페인이 예약될 때가 아닌 발송 시점에 생성됩니다. 즉, 예약된 발송 전에 필터 조건을 충족하는 새로운 연락처는 포함되고, 더 이상 조건을 충족하지 않는 연락처는 제외됩니다.

**Q: 스마트 리스트에서 페이지당 20개가 아닌 100개 연락처를 표시하려면 어떻게 해야 하나요?**

오른쪽 하단의 페이지당 연락처 수 선택기를 사용하여 100개를 선택하세요(옵션: 20, 50, 100).
