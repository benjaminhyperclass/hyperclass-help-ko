# 워크플로우에 연락처를 일괄 추가하는 방법

이 가이드에서는 연락처를 필터링(예: 태그별)하고 자동화 추가(Add to Automation) 기능을 사용하여 워크플로우에 일괄 추가하는 방법을 설명합니다. 단계별 안내, 사용 가능한 예약 모드, 자주 묻는 질문에 대한 답변을 제공합니다.

## 워크플로우에 연락처를 일괄 추가하는 것이란?

자동화 추가(Add to Automation) 기능을 사용하면 연락처(Contacts) 페이지에서 연락처를 선택하여 워크플로우에 일괄 등록할 수 있습니다. 연락처가 트리거를 통해 워크플로우에 진입하기를 기다리는 대신, 발행된 워크플로우에 연락처 그룹을 수동으로 추가할 수 있습니다 - 한 번에 모두, 예약된 시간에, 또는 드립 패턴으로 말이죠.

가장 일반적인 사용 사례는 특정 속성(예: 태그)으로 연락처 목록을 필터링하고, 필터링된 결과를 선택한 다음, 타겟 아웃리치, 재참여 유도, 일괄 처리를 위해 워크플로우에 추가하는 것입니다.

## 연락처를 일괄 추가하는 주요 이점

- **연락처를 더 빠르게 등록**: 개별적으로 등록하는 대신 여러 연락처를 한 번의 일괄 작업으로 워크플로우에 추가할 수 있습니다.

- **적절한 대상 타겟팅**: 고급 필터(태그, 필드, 기회 데이터 등)를 사용하여 필요한 연락처만 등록할 수 있습니다.

- **진입 시점 제어**: 연락처를 즉시, 예약된 날짜/시간에, 또는 드립 모드 배치로 등록할지 선택할 수 있습니다.

- **세그먼트 쉽게 재사용**: 필터링된 보기를 스마트 리스트(Smart Lists)로 저장하여 언제든지 동일한 일괄 등록을 반복할 수 있습니다.

- **진행 상황과 결과 추적**: 일괄 작업(Bulk Actions) 페이지에서 상태, 타임스탬프, 통계를 모니터링하여 처리된 내용을 확인할 수 있습니다.

## 워크플로우에 연락처를 일괄 추가하는 방법

### 1단계: 연락처 필터링

팁: 다른 연락처 필드(이메일, 도시, 연락처 소스, 기회 단계 등)로도 필터링할 수 있으며, 여러 필터와 중첩 필터를 추가하여 등록하려는 연락처를 정확히 좁힐 수 있습니다.

- **Contacts(연락처)** > **Smart Lists(스마트 리스트)**로 이동합니다.

![연락처 스마트 리스트](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155068065636/original/-HJOc2qy7BANkUe-fSLsMPw9GG__fdZUFg.jpeg?1774884143)

- **Advanced Filters(고급 필터)**를 클릭하여 필터 패널을 엽니다.

![고급 필터 패널](https://jumpshare.com/share/3ZPF44Y9Td1JFyQ0vA7D+/Screen+Shot+2026-03-30+at+20.53.26.png)

- **태그**로 필터링하려면 필터 검색창에서 "**Tag(태그)**"를 검색하고 선택합니다.

- **연산자**(예: "Is")를 설정하고 필터링할 태그 **값**(예: "follow-up")을 선택합니다. **Apply(적용)**를 클릭하여 연락처 목록을 필터링합니다. 이제 페이지에 필터 조건과 일치하는 연락처만 표시됩니다.

![태그 필터 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155068066391/original/U4_Jikxh6MB4X4MMC42l_pMGfPjal99vqg.png?1774884407)

### 2단계: 자동화에 추가

- **워크플로우에 추가할 연락처를 선택**: **체크박스**를 사용하여 **개별 연락처**를 선택하거나, 상단의 **Select all(모두 선택)**을 클릭하여 필터링된 목록의 모든 연락처를 선택할 수 있습니다. 최소 한 명의 연락처가 선택되면 일괄 작업 도구 모음이 나타납니다.

- 일괄 작업 도구 모음에서 **Trigger automation(자동화 트리거)**를 클릭합니다. 자동화 추가 대화 상자가 열리며, 상단에 선택된 연락처가 표시되고 아래에 구성 필드가 나타납니다.

![자동화 트리거](https://jumpshare.com/share/KXkEwWAw3OPPq7NcEzVY+/GIF+Recording+2026-03-30+at+21.00.58.gif)

### 3단계: 자동화 구성

- 다음 필드를 입력합니다:

**Action Name(작업 이름):** 이 일괄 작업의 이름으로, 일괄 작업(Bulk Actions) 페이지의 추적 보고서에 표시됩니다. 예: "워크플로우 자동화 일괄 가져오기"

**Automation Type(자동화 유형):** 연락처를 워크플로우에 등록하려면 "**Workflow(워크플로우)**"를 선택합니다.

**Workflow(워크플로우):** 연락처를 추가할 워크플로우를 선택합니다. 드롭다운에는 계정의 발행된 모든 워크플로우가 나열됩니다.

**Mode(모드):** 연락처를 언제, 어떻게 등록할지 선택합니다. 자세한 내용은 아래 예약 모드 섹션을 참조하세요.

- **Add to Automation(자동화에 추가)**를 클릭합니다. 처리되는 연락처 수를 보여주는 성공 확인 메시지가 나타납니다(예: "10명 연락처 자동화 추가가 진행 중/예약됨").

![자동화 구성](https://jumpshare.com/share/QKYX8WMvfTjw98iOGft1+/GIF+Recording+2026-03-30+at+23.01.41.gif)

- **Check Progress(진행 상황 확인)**를 클릭하여 일괄 작업 페이지로 이동하거나, Dismiss(닫기)를 클릭하여 연락처 목록으로 돌아갑니다.

![진행 상황 확인](https://jumpshare.com/share/eBZl9GB92Tm27rhOGW11+/Screen+Shot+2026-03-30+at+23.02.22.png)

## 예약 모드

모드(Mode) 설정은 선택된 연락처가 워크플로우에 등록되는 시점을 제어합니다. 세 가지 옵션이 있습니다:

### 한 번에 모두 전송(Send All at Once)

선택된 모든 연락처가 즉시 워크플로우에 추가됩니다. 기본 모드이며 타이밍이 중요하지 않을 때 적합합니다.

**추가 필드:** 없음

![한 번에 모두 전송](https://jumpshare.com/share/SImCkyIrTvzXvaQEBs3T+/Screen+Shot+2026-03-30+at+23.16.29.png)

### 예약된 시간에 전송(Send at Scheduled Time)

선택된 모든 연락처가 선택한 특정 날짜와 시간에 워크플로우에 추가됩니다. 캠페인 시작이나 업무 시간에 맞춰 등록하는 데 유용합니다.

**추가 필드:**

| 필드 | 설명 |
|------|------|
| Pick a Start Date & Time(시작 날짜 및 시간 선택) | 모든 연락처가 등록될 날짜와 시간입니다. 계정의 시간대로 표시됩니다. |

![예약된 시간에 전송](https://jumpshare.com/share/xiN3hTM2hvE2CJBPuC6T+/Screen+Shot+2026-03-30+at+23.22.40.png)

### 드립 모드로 전송(Send in Drip Mode)

연락처를 한 번에 모두가 아닌 일정 기간에 걸쳐 배치로 등록합니다. 워크플로우 과부하를 방지하거나 커뮤니케이션을 분산시키고 싶은 대량 연락처 목록에 이상적입니다.

**추가 필드:**

| 필드 | 설명 |
|------|------|
| Pick a Start Date & Time(시작 날짜 및 시간 선택) | 첫 번째 연락처 배치가 등록될 날짜와 시간입니다. |
| Number of Messages per Batch(배치당 메시지 수) | 각 배치에 등록할 연락처 수입니다. |
| Repeat after(반복 간격) | 배치 간 간격입니다. 숫자와 단위(Days, Hours, Minutes)를 설정합니다. |

![드립 모드로 전송](https://jumpshare.com/share/edOlM7ALY7k6eenAT0J8+/Screen+Shot+2026-03-30+at+23.29.56.png)

## 진행 상황 추적

제출 후 일괄 작업(Bulk Actions) 페이지에서 일괄 작업을 추적할 수 있습니다. **Contacts(연락처)** > **Bulk Actions(일괄 작업)**으로 이동하여 모든 일괄 작업 요청을 확인하세요.

**일괄 작업 페이지 표시 항목:**

- **Action Label(작업 라벨):** 작업 생성 시 제공한 이름
- **Operation(작업 유형):** 자동화 유형(예: 워크플로우)
- **Status(상태):** 작업이 진행 중(In Progress), 예약됨(Scheduled), 완료(Complete) 상태인지 여부
- **User(사용자):** 일괄 작업을 시작한 사람
- **Created / Completed(생성 / 완료):** 작업이 생성되고 완료된 시간(계정 시간대로 표시)
- **Statistics(통계):** 통계 아이콘을 클릭하면 작업 결과의 상세 분석을 볼 수 있습니다

일괄 작업 페이지에서 날짜 범위, 상태, 작업 유형, 사용자별로 필터링하여 특정 요청을 빠르게 찾을 수 있습니다.

![일괄 작업 페이지](https://jumpshare.com/share/QN7nkdxf16ewjLGNWkrc+/Screen+Shot+2026-03-30+at+23.32.56.png)

## 자주 묻는 질문

**태그 외에 다른 필드로도 연락처를 필터링할 수 있나요?**
네. 고급 필터(Advanced Filters) 패널은 모든 연락처 필드(이메일, 도시, 연락처 소스, 생성 날짜, 기회 단계 등)와 커스텀 필드로도 필터링할 수 있습니다. 여러 필터와 중첩 필터를 결합하여 등록하려는 연락처를 정확히 타겟팅할 수 있습니다.

**워크플로우가 발행된 상태여야 하나요?**
네. 워크플로우(Workflow) 드롭다운에는 발행된 워크플로우만 표시됩니다. 이 기능을 사용하기 전에 워크플로우가 발행되었는지 확인하세요.

**예약된 시간에 전송과 드립 모드로 전송의 차이점은 무엇인가요?**
예약된 시간에 전송은 선택한 단일 날짜와 시간에 선택된 모든 연락처를 등록합니다. 드립 모드로 전송은 정기적인 간격으로 연락처를 배치별로 등록하여 며칠 또는 몇 시간에 걸쳐 커뮤니케이션을 분산시키는 데 유용합니다.

**일괄 작업의 진행 상황을 어디서 추적할 수 있나요?**
연락처(Contacts) > 일괄 작업(Bulk Actions)으로 이동하세요. 현재 상태, 타임스탬프, 통계 분석과 함께 모든 일괄 작업 요청을 볼 수 있습니다.

**스마트 리스트 필터를 저장하고 향후 일괄 가져오기에 재사용할 수 있나요?**
네. 고급 필터를 적용한 후 필터링된 보기를 스마트 리스트로 저장할 수 있습니다. 동일한 세그먼트를 다시 등록해야 할 때는 해당 스마트 리스트를 열고 연락처를 선택한 다음 자동화를 트리거하면 됩니다.
