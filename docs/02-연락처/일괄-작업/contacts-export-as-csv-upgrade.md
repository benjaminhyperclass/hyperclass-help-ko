# 연락처 CSV 내보내기 업그레이드

# 연락처 CSV 내보내기 업그레이드란?

연락처 내보내기 업그레이드는 기존 브라우저 기반 내보내기를 확장 가능한 백엔드 서비스로 대체합니다. CSV가 포그라운드에서 다운로드되기를 기다리는 대신, 내보내기 작업이 Hyperclass 서버에서 대기열에 추가되고, 병렬로 처리되며, 일괄 작업(Bulk Actions) 대시보드에서 실시간으로 추적됩니다. 완료되면 다운로드 준비가 된 파일이 일괄 작업에 나타나며 30일간 이용할 수 있습니다.

## 연락처 내보내기 업그레이드의 주요 장점

- 작업 중단 없음: 내보내기가 비동기적으로 실행되어 UI가 차단되지 않습니다.
- 대용량 데이터 확장성: 수백만 행 데이터셋에서 실패가 훨씬 적습니다.
- 실시간 가시성: 일괄 작업 페이지에서 진행률을 추적할 수 있습니다.
- 자동 알림: 파일이 준비되면 알림을 받아 수동 확인이 불필요합니다.
- 내장 보관: 30일간 클라우드 저장으로 우발적 손실을 방지합니다.
- 접근 제어: 관리자만 다운로드할 수 있어 보안과 규정 준수를 지원합니다.

## 연락처 내보내기 이유

**연락처 내보내기(Export Contacts)** 기능을 사용하면 Hyperclass 연락처를 CSV 파일로 다운로드할 수 있습니다. 내보낸 파일에는 이름, 이메일 주소, 전화번호, 태그, 커스텀 필드(선택 시) 등 중요한 세부 정보가 포함됩니다. 주요 사용 사례는 다음과 같습니다:

- CRM 마이그레이션 – 연락처를 다른 CRM 플랫폼으로 이동
- 마케팅 및 아웃리치 – 이메일 또는 SMS 캠페인용 목록 생성
- 데이터 백업 및 보안 – 연락처의 오프라인 사본 저장
- 팀 협업 – 동료나 비즈니스 파트너와 목록 공유
- 분석 및 리포팅 – 데이터를 사용한 인사이트 생성

## 사전 요구사항 및 제한사항

- 파일은 **CSV 형식**이며 **Excel, Google Sheets 또는 다른 스프레드시트 소프트웨어**에서 열 수 있습니다.
- **255자를 초과하는 노트** 등 일부 데이터는 잘릴 수 있습니다.
- 내보내기에는 **자동화 기록**(예: 과거 이메일, SMS 로그)이 **포함되지 않습니다**.
- 관리자만 연락처를 내보낼 수 있습니다. 내보내기 옵션이 보이지 않으면 사용자 역할 권한을 확인하거나 관리자에게 문의하세요.

## 연락처를 CSV로 내보내는 방법

### **1단계**: 연락처 페이지로 이동

- 하위 계정에서 **연락처(Contacts)**로 이동합니다.
- **스마트 리스트(Smart Lists)**를 클릭합니다.

![스마트 리스트 선택](https://jumpshare.com/share/1mr76Pjvq41YC62X9d9e+/Screen+Shot+2025-10-14+at+11.34.56+PM.png)

### **2단계:** 내보낼 연락처 선택

- 고급 필터를 사용해 선택을 세밀화합니다.

![고급 필터 사용](https://jumpshare.com/share/ccdRtfqt7fDs0vehmNIg+/Screen+Shot+2025-10-14+at+11.41.32+PM.png)

- 내보내려는 연락처 옆의 **체크박스**를 선택합니다.
- 모든 연락처를 내보내려면 필터 필드 옆의 체크박스(왼쪽)를 클릭하고 전체 연락처 선택을 클릭합니다.

![전체 연락처 선택](https://jumpshare.com/share/Vqe73k6mjY8tZM1OvVRd+/Screen+Shot+2025-10-14+at+11.49.03+PM.png)

### 3단계: 필드 커스터마이징 (선택사항)

참고: CSV 파일에는 현재 연락처 테이블에 표시되는 열만 포함됩니다. 내보낸 데이터에서 필드가 누락된 경우, 다시 내보내기 전에 열 설정을 확인하세요.

- **필드 관리(Manage Fields)** 버튼을 클릭합니다.
- 체크박스를 선택하거나 **해제**하여 내보내기에서 특정 열을 포함하거나 제외합니다.
- 원하는 열을 선택하면 **내보낸 CSV 파일**에 반영됩니다.

![필드 관리](https://jumpshare.com/share/O2M5Xzrwz5DHeEd8aIoB+/Screen+Shot+2025-10-14+at+11.56.10+PM.png)

### 4단계: 연락처 내보내기

- 연락처를 선택한 후 더보기를 클릭합니다.
- **내보내기(Export)**를 클릭합니다.

![내보내기 클릭](https://jumpshare.com/share/6zQc3Epuz42Miqq7DVRu+/Screen+Shot+2025-10-15+at+12.02.19+AM.png)

- 또는 + 연락처 추가(Add Contact) 버튼 옆의 **점 세 개**를 클릭하고 내보내기를 클릭합니다.

![다른 내보내기 방법](https://jumpshare.com/share/eP9CCZoxSiHY1eSTvI1A+/Screen+Shot+2025-10-15+at+12.12.16+AM.png)

- 내보내기 모달에서 선택을 확인하고 **내보내기(Export)**를 클릭합니다.

![내보내기 확인](https://jumpshare.com/share/GP9NMEGJgCLCd50wQuF6+/Screen+Shot+2025-10-15+at+12.19.41+AM.png)

- 내보내기가 시작되었음을 알리는 성공 메시지가 표시됩니다.
- **진행률 확인(Check Progress)**을 클릭하여 일괄 작업에서 내보내기 상태를 확인하거나 닫기만 할 수 있습니다.
- 내보내기는 백그라운드에서 비동기적으로 실행되므로 계속 작업할 수 있습니다.

![진행률 확인](https://jumpshare.com/share/bspEDo6C9zEIv4dPwKC7+/Screen+Shot+2025-10-15+at+12.21.06+AM.png)

- 내보내기 파일이 다운로드 준비되면 푸시 알림을 받습니다.

![완료 알림](https://jumpshare.com/share/gDd8rmSp2UE2Pt8RPKSN+/Screen+Shot+2025-10-15+at+12.31.07+AM.png)

### 5단계: 일괄 작업에서 다운로드

- **연락처(Contacts)** > 일괄 작업(Bulk Actions)으로 이동합니다.

![일괄 작업 이동](https://jumpshare.com/share/pWdaJHArDfA6JcwH3SCR+/Screen+Shot+2025-10-15+at+12.34.45+AM.png)

- 내보내기 행을 찾으면 다음이 표시됩니다:
  - **작업:** 내보내기
  - **상태:** 완료
  - **파일명:** 예시: Export_Contacts_All_Oct_2025.csv

- 아래쪽 화살표(다운로드) 아이콘을 클릭하여 파일을 시스템에 저장합니다.
- **30일** 후 다운로드 링크가 **비활성화**됩니다.

![다운로드 완료](https://jumpshare.com/share/VzqB5D4os1qDi2UVAXBx+/Screen+Shot+2025-10-15+at+12.38.43+AM.png)

## 자주 묻는 질문

**Q: 파일이 어디에 다운로드되나요?**
파일은 장치의 로컬 저장소(로컬 드라이브)에 저장됩니다.

**Q: 내보낸 .csv 파일에서 각 연락처의 특정 데이터 필드만 내보낼 수 있나요?**
내보내기를 시작하기 전에 특정 데이터 필드를 선택하거나 선택 해제하여 내보낸 .csv 파일의 열을 커스터마이징할 수 있습니다. 이렇게 하면 가장 관련성 높은 정보에 집중할 수 있습니다.

**Q: 연락처의 최신 노트가 255자를 초과하면 어떻게 되나요?**
연락처를 내보낼 때 최대 255자까지의 최신 노트만 포함됩니다. 연락처의 최신 노트가 이 제한보다 길면 255자 제약에 맞게 잘립니다.

**Q: 여러 계정이나 로케이션의 연락처를 동시에 내보낼 수 있나요?**
각 하위 계정이나 로케이션별로 연락처를 별도로 내보내야 합니다. 하지만 스프레드시트 애플리케이션에서 결과 .csv 파일들을 결합하여 여러 소스의 연락처 데이터를 통합할 수 있습니다.

**Q: 연락처 데이터의 자동 내보내기를 예약할 수 있나요?**
이 기능은 현재 이용할 수 없습니다.

**Q: 누가 내보낸 파일을 다운로드할 수 있나요?**
에이전시 관리자와 로케이션 관리자만 일괄 작업에서 완료된 내보내기를 다운로드할 수 있습니다.

**Q: 내보낸 파일은 얼마나 오래 이용할 수 있나요?**
내보내기 파일은 30일간 이용할 수 있습니다. 그 이후에는 다운로드 옵션이 비활성화됩니다.

**Q: 내보낸 .csv 파일을 다른 연락처 관리 플랫폼으로 가져오려면 어떻게 하나요?**
대부분의 연락처 관리 플랫폼은 .csv 파일을 지원하는 가져오기 기능을 제공합니다. 대상 플랫폼의 특정 가져오기 지침을 따라야 하며, 일반적으로 .csv 파일의 열을 새 시스템의 적절한 필드에 매핑하는 과정이 포함됩니다.
