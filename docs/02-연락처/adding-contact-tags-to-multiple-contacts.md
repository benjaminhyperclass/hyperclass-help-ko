# 여러 연락처에 태그 일괄 추가하기

이 가이드에서는 "태그 추가" 기능을 사용해 여러 연락처에 태그를 빠르게 적용하는 방법을 알아보겠습니다. 라이프사이클 단계별로 리드를 정리하거나, 캠페인을 위한 사용자 세그먼트를 나누거나, 자동화를 위한 연락처를 표시할 때, 일괄 태깅은 수시간의 수작업을 절약해줍니다. 태그가 무엇인지, 왜 중요한지, 그리고 몇 번의 클릭만으로 어떻게 사용하는지 차근차근 설명해드리겠습니다.

# **여러 연락처에 태그 추가란 무엇인가**

일괄 태그 추가는 대량의 연락처를 효율적으로 라벨링하고 정리할 수 있게 해줍니다. 태그는 공통 특성, 행동, 또는 고객 여정의 단계에 따라 연락처를 분류하는 데 도움을 줍니다. 이 기능은 대상 세분화, 자동화 트리거, 그리고 대규모 CRM 워크플로우 관리에 필수적입니다.

---

## **여러 연락처에 태그 추가의 주요 장점**

일괄 태깅은 세그먼테이션, 개인화, 자동화 설정을 간소화할 수 있는 강력한 CRM 작업입니다. 주요 장점은 다음과 같습니다:

- 한 번의 작업으로 여러 연락처에 태그를 적용하여 시간 절약
- 더 타겟팅된 커뮤니케이션을 위한 세그먼테이션 향상
- 태그 기반 조건으로 워크플로우 트리거
- 캠페인 참여나 진행 상황을 쉽게 추적
- 깔끔하고 체계적인 연락처 기록 유지
- 관련 연락처를 그룹화하여 리포팅과 분석 기능 향상

---

## **연락처 태그 일괄 추가 설정하기**

이 섹션에서는 연락처(Contacts) 페이지에서 여러 연락처에 태그를 적용하는 정확한 과정을 안내합니다.

### **연락처(Contacts) 탭으로 이동**

대시보드에서 왼쪽 네비게이션 메뉴의 연락처(Contacts) 탭을 클릭합니다.

![연락처 탭 클릭](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044782500/original/1DiM3__iipDM14SB8jQAqo9moaXjpUOvLw.png?1744188268)

### **여러 연락처 선택**

태그를 추가하려는 연락처들을 왼쪽의 체크박스로 선택합니다. 필요한 만큼 여러 레코드를 선택할 수 있습니다.

![연락처 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044782587/original/cjp_SIJp8yZHAeV_d2TddeW2KS2B_71NDA.png?1744188307)

### **"태그 추가" 아이콘 클릭**

연락처를 선택한 후, 화면 상단 액션 바에서 태그 추가(Add Tag) 아이콘을 클릭합니다.

![태그 추가 아이콘 클릭](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044782657/original/Xw3Tl9-sK1YqpzfnlFjTgn9GudhBMTpVzg.png?1744188333)

### **작업명과 태그 추가**

"태그 추가(Add Tags)" 팝업 창에서 작업명(Action Name) 필드에 작업 이름을 입력합니다. 그다음 태그(Tags) 필드의 드롭다운을 클릭하여 목록에서 하나 이상의 태그를 선택합니다.

![작업명과 태그 입력](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044782714/original/EyTQ2l1QVMz6e3P6CHgNo2wVsneuwtR6Vg.png?1744188367)

### **"태그 추가" 클릭하여 적용**

태그를 선택한 후 **태그 추가(Add Tags)** 버튼을 클릭하여 작업을 시작합니다. 확인 메시지가 나타납니다.

![태그 추가 버튼 클릭](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044782753/original/f5n0FwohiZGkjTH6f51sdOWuafp-sRdyQQ.png?1744188396)

### **진행 상황 추적**

이 확인 팝업은 **"태그 추가"**를 클릭한 직후 나타납니다. 선택한 연락처들에 대한 태그 할당이 진행 중임을 알려줍니다. 작업을 모니터링하려면 **"진행 상황 확인(Check Progress)"** 버튼을 클릭하면 일괄 작업(Bulk Actions) 탭으로 이동하여 실시간으로 작업의 성공/실패 상태를 추적할 수 있습니다.

![진행 상황 추적](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044782795/original/b0LPCVUFablfqYO-DK4jnJnwdcgR0LpZ1w.png?1744188426)

### **일괄 작업 상태 추적**

하이라이트된 영역에서 일괄 태그 작업이 성공적으로 완료된 것을 확인할 수 있습니다. 작업명, 실행자, 생성 시각, 그리고 결과를 검토할 수 있는 "통계 보기(Show Stats)" 링크를 볼 수 있습니다. 이를 통해 모든 것이 예상대로 적용되었는지 확인할 수 있습니다.

![일괄 작업 상태](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044782834/original/jjhtg3fv29GRnfnVyMnBWV9C_Uod2LqnwA.png?1744188457)

## **자주 묻는 질문**

**Q: 한 번에 여러 태그를 연락처에 적용할 수 있나요?**

네, 하나의 일괄 작업으로 여러 태그를 동시에 선택하고 적용할 수 있습니다. 작업을 제출하기 전에 태그(Tags) 필드에 원하는 모든 태그를 추가하기만 하면 됩니다.

**Q: 연락처의 기존 태그가 덮어써지나요?**

아니요, 일괄 태그 추가는 선택한 태그를 연락처의 기존 태그 목록에 추가만 합니다. 기존 태그를 제거하지는 않습니다.

**Q: 태그로 자동화를 트리거할 수 있나요?**

물론입니다. 태그는 Hyperclass 자동화 내에서 워크플로우 트리거로 자주 사용되어, 태그가 추가되거나 제거될 때 시퀀스를 시작할 수 있습니다.

**Q: 일괄로 적용한 태그를 취소하거나 제거하려면 어떻게 하나요?**

연락처에서 태그를 제거하려면 "태그 제거(Remove Tag)" 일괄 작업을 사용하세요. 특정 태그를 제거하거나 선택한 연락처에서 모든 태그를 삭제할 수 있습니다. 이는 별도 가이드에서 다룰 예정입니다.

**Q: 어떤 연락처에 태그가 성공적으로 적용되었는지 확인할 수 있나요?**

네. 일괄 작업(Bulk Actions) 탭으로 이동하여 해당 작업 옆의 "통계 보기(Show Stats)"를 클릭하세요. 이 화면에서 각 태그된 레코드의 성공 상태, 연락처 식별자, 타임스탬프를 확인할 수 있습니다.
