# 연락처에 여러 전화번호 추가하기

한 연락처에 여러 개의 전화번호를 저장해야 하나요? Hyperclass를 사용하면 휴대폰, 직장, 집 전화 등 여러 연락처 번호를 쉽게 관리할 수 있어요. 기본 번호를 설정하고 라벨을 지정해서 더 체계적으로 관리할 수도 있습니다.

## **여러 전화번호 기능이란?**

Hyperclass의 여러 전화번호 기능을 사용하면 한 연락처에 최대 11개의 전화번호를 저장할 수 있어요. 각 번호에 라벨(예: 휴대폰, 직장, 집)을 붙일 수 있고, 그중 하나를 기본 번호로 설정할 수 있습니다. 여러 소통 채널을 사용하는 고객, 리드, 협력업체를 관리할 때 매우 유용해요.

## **여러 전화번호 관리의 주요 장점**

한 연락처에 여러 전화번호를 관리하면 소통이 항상 효과적이고 체계적으로 이루어져요.

- 연락처당 최대 11개 번호 저장 가능
- 각 번호에 명확한 라벨 지정으로 구분
- 발신 우선순위를 위한 기본 번호 설정
- 오래된 전화번호 쉽게 업데이트 또는 삭제
- 워크플로우와 자동화의 정확성 향상
- 영업, 고객지원, 협력업체, 개인 연락처에 이상적

## 여러 전화번호 관리하기

Hyperclass 연락처 상세에서 바로 연락처 전화번호를 추가하고, 라벨을 붙이고, 관리할 수 있어요.

### 1단계: Contacts(연락처) 탭으로 이동

Hyperclass 대시보드에서 **Contacts(연락처) 탭**으로 이동한 다음 기존 연락처를 클릭하거나 새 연락처를 만들어 **Phone numbers(전화번호)** 섹션에 접근하세요.

![연락처 전화번호 섹션으로 이동](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155045772493/original/xke_2ijmRgPxFJWFf8yZylGJVCJMksTEmg.gif?1745846151)

### **2단계:** "+ Add Phone Number" 클릭

여기에서 여러 전화번호(최대 10개)를 입력하고 라벨(휴대폰, 집, 직장 등)을 선택할 수 있어요. 기본 번호로 설정하려면 Primary(기본) 박스에 체크하세요.

![전화번호 추가하기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155045773190/original/9f3o5h-Yw7GuexklaSi7WFyY-2ktPJhGoQ.gif?1745846538)

## **전화번호 편집 또는 삭제하기**

필요에 따라 번호를 업데이트하거나 삭제해서 연락처 기록을 깔끔하고 정확하게 유지하세요.

- 번호 옆의 Edit(편집) 아이콘을 클릭해서 번호나 라벨을 변경하세요
- Trash(휴지통) 아이콘을 클릭해서 번호를 완전히 삭제하세요
- Primary(기본) 번호는 한 번에 하나만 설정할 수 있어요

![전화번호 편집 및 삭제](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155045773516/original/V0UvAF3c8FSZD2JstQLf1pGLcH8Tz4DWQQ.gif?1745846722)

## **여러 전화번호 활용 사례**

이 기능은 다양한 실제 비즈니스 및 소통 시나리오에서 특히 유용해요.

- 영업팀이 여러 번호로 연락해서 리드에게 더 효과적으로 도달
- 고객지원팀이 고객을 위한 백업 연락 수단 확보
- 에이전시가 각 연락처의 개인 번호와 업무 번호를 모두 관리
- 협력업체와 파트너는 종종 부서별로 다른 번호 보유(예: 결제 vs 운영)
- 중요 고객이나 서비스를 위한 백업 번호 저장으로 비상 대비

## **자주 묻는 질문**

**Q: Primary(기본) 번호를 설정하지 않으면 어떻게 되나요?**
A: 목록의 첫 번째 번호가 자동으로 발신 전화나 메시지의 기본 번호로 처리돼요.

**Q: 여러 번호에 같은 라벨을 사용할 수 있나요?**
A: 아니요, 각 라벨은 연락처당 한 번만 사용할 수 있어서 명확성을 유지해요.

**Q: 국제 번호를 추가할 수 있나요?**
A: 네, 적절한 국가 코드와 함께 번호를 입력하기만 하면 돼요(예: 영국의 경우 +44).

**Q: 전화번호 순서를 바꿀 수 있나요?**
A: 번호는 입력한 순서대로 표시돼요. 순서를 바꾸려면 삭제한 후 원하는 순서대로 다시 추가하세요.

**Q: 이 번호들을 워크플로우에서 사용할 수 있나요?**
A: 네! 별도로 설정하지 않으면 워크플로우는 항상 Primary(기본) 번호를 사용해요.
