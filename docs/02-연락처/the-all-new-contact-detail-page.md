# 완전히 새로워진 연락처 상세 페이지

완전히 새로워진 Hyperclass 연락처 상세 페이지는 모든 고객 상호작용을 한 곳에서 더 빠르고, 더 깔끔하고, 더 스마트하게 확인할 수 있는 방법을 제공합니다. 이 가이드에서는 무엇이 바뀌었는지, 왜 중요한지, 그리고 자동 저장, 접을 수 있는 패널, 즉석 키보드 탐색 등 생산성을 높여주는 업그레이드를 어떻게 사용하는지 설명합니다.

**중요**: 이 기능은 현재 Labs에서 제공됩니다. 보이지 않으면 관리자에게 Agency Settings(에이전시 설정) > Labs에서 활성화해 달라고 요청하세요. 자세한 내용은 [Labs 기능 - 완전 개요](../19-에이전시-뷰/Agency-Settings/labs-features-complete-overview.md)를 참조하세요.

## **완전히 새로워진 연락처 상세 페이지란?**

처음부터 다시 설계된 연락처 상세 페이지는 연락처 정보를 검토하고, 대화를 추적하고, 활동을 기록하고, 작업을 실행할 수 있는 중앙 워크스페이스입니다. 이제 속도와 사용성을 위해 최적화되었습니다. 이번 재설계는 끝없는 스크롤을 3패널 레이아웃으로 대체하고, 중요한 데이터를 한눈에 보여주며, 개인 보기 환경설정을 기억하여 진정으로 간소화된 CRM 경험을 제공합니다.

---

## **한눈에 보는 주요 개선 사항**

- 현대적인 디자인: 더 빠르고, 일관되고, 시각적으로 간소화된 레이아웃
- 자동 저장: 필드를 벗어날 때 자동으로 변경 사항이 저장됨 — 클릭 불필요
- 스마트 탐색: 키보드 단축키로 연락처 간 이동
- 접을 수 있는 패널: 중요한 것에 집중하고, 시스템이 레이아웃을 기억
- 인라인 연결 관리: 연락처 페이지에서 바로 회사, 기회, 커스텀 객체 보기 및 편집
- 깔끔한 오른쪽 패널: 할 일, 노트, 문서, 결제, 예약을 위한 통합 경험
- 빈 필드 숨기기: 불필요한 요소를 줄이고 채워진 것만 보기
- 필드 내 검색: 데이터나 폴더를 빠르게 찾기

## **자동 저장: 작동 방식**

자동 저장은 가장 강력한 새 기능 중 하나입니다. 활성화하면 필드를 벗어나는 순간 편집 내용이 자동으로 저장됩니다 — 수동으로 "저장"을 클릭할 필요가 없습니다.

**단계**: 연락처 레코드를 열고 편집하면 하단에 저장 바가 나타납니다. 케밥(점 3개) 메뉴에서 "Enable Auto-Save(자동 저장 활성화)"를 클릭합니다.

![자동 저장 활성화](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155056157667/original/v1gsiG7FiFft22yhfQgr2e5GKTXSEVrOAQ.png?1760623346)

메시지가 나타나면 "Enable Auto-Save(자동 저장 활성화)"를 클릭합니다.

![자동 저장 확인](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055793497/original/sd_dilWAx91RpPK6y4HHqZGFe5eMxpzrEw.png?1760264176)

모달에서 확인합니다 — 이 설정이 하위 계정에 적용됩니다. 관리자는 다음 위치에서도 자동 저장을 활성화하거나 비활성화할 수 있습니다: **Settings(설정) → Contacts(연락처) → Details Tab(세부 정보 탭) → Auto-Save toggle(자동 저장 토글)**

![자동 저장 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055793499/original/lBmu2nddaOpGl69KHwW1vRwhs4U7yn-WCg.png?1760264183)

활성화되면 하위 계정의 모든 사용자가 작업하면서 자동으로 편집 내용을 저장합니다.

## **사용성 개선사항**

- 패널과 탭이 이제 마지막 상태를 기억합니다 — 매번 다시 펼치거나 접을 필요가 없습니다
- 전문적이고 깔끔한 모습을 위한 일관된 간격, 타이포그래피, 구성 요소 동작
- 모든 모듈(할 일, 문서, 노트, 결제)에서 통일된 디자인 언어와 빈 상태 표시

## **자주 묻는 질문**

**Q: 자동 저장이 내 계정의 모든 사람에게 적용되나요?**
네. 관리자가 활성화하면 해당 하위 계정의 모든 사용자에게 자동 저장이 활성화됩니다.

**Q: 자동 저장으로 저장된 변경 사항을 되돌릴 수 있나요?**
언제든지 필드를 다시 편집할 수 있습니다 — 자동 저장은 레코드를 즉시 업데이트하며 완전한 활동 로그를 유지합니다.

**Q: "자동 저장 활성화" 버튼이 보이지 않는 이유는 무엇인가요?**
이미 활성화되어 있을 수 있습니다. Settings(설정) → Contacts(연락처) → Details Tab(세부 정보 탭)에서 확인해보세요.
