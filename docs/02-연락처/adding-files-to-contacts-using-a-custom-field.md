## 왜 연락처에 파일을 추가하나요?

GoHighLevel에서 연락처에 파일을 추가하면 계약서, 인보이스, 메모 등 중요한 문서를 연락처 프로필 내에 직접 저장할 수 있습니다. 이 기능을 통해 모든 필수 파일을 체계적으로 정리하고 쉽게 접근할 수 있어 고객 관리가 간소화됩니다.

**HighLevel에서는 연락처에 파일을 업로드하는 두 가지 방법을 제공합니다:**

- 연락처 프로필에서 직접 업로드
- 파일 업로드용 커스텀 필드 사용

---

## 커스텀 필드 생성 전 주요 고려사항

GoHighLevel에서 커스텀 파일 업로드 필드를 설정할 때, 더 나은 정리와 기능을 위해 구성할 수 있는 여러 중요한 옵션이 있습니다. 이러한 옵션들은 필드의 동작 방식, 표시 위치, 처리할 수 있는 파일 유형을 결정합니다.

| 필드 | 기능 | 세부사항 및 예시 |
|------|------|------------------|
| **파일 유형** | 이 커스텀 필드를 통해 사용자가 업로드할 수 있는 파일 유형을 지정합니다. | 지원 형식: **PDF, DOC/DOCX, JPG/JPEG, PNG, GIF, XLS/CSV**. 호환되는 파일만 허용되도록 합니다. |
| **파일 제한** | 업로드 허용되는 최대 파일 크기를 설정합니다. | 파일당 최대 크기: **250MB**. 업로드 중 오류를 방지하기 위해 사용자에게 이 제한을 알려주세요. |
| **다중 파일 허용** | 사용자가 단일 파일 또는 다중 파일을 업로드할 수 있는지 정의합니다. | **단일 파일:** 간편함을 위해 하나의 파일로 제한.<br>**다중 파일:** 최대 파일 수 지정(예: 3개 또는 5개). |
| **객체 유형** | 커스텀 필드가 적용될 카테고리를 결정합니다. | **Contact(연락처):** 개별 연락처 프로필에 파일 첨부.<br>**Opportunity(기회):** 거래나 리드에 파일 연결.<br>**Company(회사):** 회사 프로필과 파일 연결.<br>**Surfboards:** 고유한 비즈니스 요구에 맞게 사용자 정의. |
| **그룹 탭** | 시스템 내의 특정 섹션이나 그룹 내에서 커스텀 필드를 정리하여 가시성과 사용성을 향상시킵니다. | 예시:<br>**Contact Information(연락처 정보):** 개인 문서 같은 핵심 세부사항.<br>**General Info(일반 정보):** 기타 파일.<br>**Additional Info(추가 정보):** 보조 또는 부차적 문서. |
| **필드 저장** | 생성 과정을 완료하여 필드를 사용할 수 있게 만듭니다. | 저장하면 이 필드가 선택한 객체(예: 연락처 프로필)의 **커스텀 필드 섹션**에 나타납니다. |

## 커스텀 필드를 사용하여 연락처에 파일 업로드하기

HighLevel에서 연락처에 파일을 업로드하는 기능은 **파일 업로드** 유형의 커스텀 필드를 생성한 후에만 사용할 수 있습니다. 커스텀 필드를 생성하면 연락처 프로필로 이동하여 지정된 필드 내에서 직접 파일을 업로드, 조회 또는 삭제할 수 있습니다.

### **1단계:** 파일 업로드 커스텀 필드 만들기

기본적으로 제공되지 않으므로 커스텀 필드를 생성해야 합니다. 생성한 후에야 연락처 섹션에서 해당 필드를 보고 사용할 수 있습니다.

왼쪽 메뉴에서 **Settings(설정)**으로 이동하여 Custom Fields(커스텀 필드)를 선택한 다음 **Add Custom Field(커스텀 필드 추가)** 버튼을 클릭합니다. 사용 가능한 옵션에서 **File Upload(파일 업로드)** 필드 유형을 선택합니다. 필드에 설명적인 이름(예: "계약서 업로드" 또는 "고객 문서")을 지정하고 마지막으로 커스텀 필드를 저장합니다.

![파일 업로드 커스텀 필드 생성](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155040330516/original/90asUWAWuQS5sUbEGHaWKB6hobjw_a6R7A.gif?1737630850)

### **2단계:** 연락처에 파일 추가하기

커스텀 필드를 생성한 후에는 다음 단계에 따라 연락처에 파일을 업로드합니다. 여러 커스텀 필드를 생성한 경우 그룹 탭 내에서 커스텀 필드를 검색하세요.

왼쪽 메뉴에서 **Contacts(연락처)** 섹션으로 이동하여 업데이트하려는 연락처를 선택합니다. 연락처 프로필의 **Custom Fields Section(커스텀 필드 섹션)**에서 생성한 파일 업로드 필드를 찾습니다.

![연락처에서 커스텀 필드 찾기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044805788/original/KEINpxAtJtHfZuKEJ18-PigV6KbkvivM3g.gif?1744203309)

### **3단계:** 파일 업로드하기

연락처 프로필에서 커스텀 필드가 보이면 기기에서 파일을 선택하고 확인하여 원하는 파일을 필드에 직접 업로드할 수 있습니다.

연락처 프로필의 **Custom Fields(커스텀 필드)** 섹션으로 스크롤하여 앞서 생성한 파일 업로드 필드를 찾습니다. **Upload File(파일 업로드)**를 클릭하고 기기에서 파일을 선택한 후 업로드를 확인하여 연락처 프로필 내에 파일을 저장합니다.

![파일 업로드하기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044806000/original/1Yg9M9NTLbd5Z2SAwTLeA2RYUtRDc8ymVQ.gif?1744203431)

커스텀 필드에 파일을 업로드한 후에는 연락처 프로필에서 직접 관리할 수 있습니다. 파일을 보려면 커스텀 필드의 파일 링크를 클릭하여 열어보고 검토하면 됩니다. 파일을 삭제해야 하는 경우 필드에서 제거하거나 새 파일을 업로드하여 대체할 수 있습니다. 실수로 삭제한 경우 즉시 실행 취소 옵션이 나타나므로 빠르고 쉽게 파일을 복원할 수 있습니다.

## 자주 묻는 질문

**Q. 어떤 파일 유형을 업로드할 수 있나요? 비디오 업로드는 지원되나요?**

아니요, 비디오는 지원되지 않습니다. PDF, DOCX, PNG, JPG, CSV, PPTX, XLSX 파일을 업로드할 수 있습니다.

**Q. 허용되는 최대 파일 크기는 얼마인가요?**

각 파일은 최대 250MB까지 가능합니다.

**Q. 업로드 후 파일을 삭제하거나 교체할 수 있나요?**

네, 파일을 삭제하거나 교체할 수 있습니다. 커스텀 필드의 경우 새 파일을 업로드하여 기존 파일을 덮어쓰기만 하면 됩니다.

**Q. 한 번에 여러 파일을 업로드할 수 있나요?**

- 직접 업로드: 아니요, 파일은 한 번에 하나씩 업로드해야 합니다.
- 커스텀 필드: 다중 업로드를 위해 여러 개의 커스텀 필드를 생성할 수 있습니다.

**Q. 파일 업로드가 실패하면 어떻게 해야 하나요?**

파일 업로드가 실패하는 경우:

- 파일 유형과 크기 요구사항을 충족하는지 확인하세요.
- 인터넷 연결을 확인하세요.
- 페이지를 새로고침하고 다시 시도해보세요.
