# 스마트 리스트 생성 및 관리 가이드

## 스마트 리스트를 사용하는 이유

- **시간 절약** – 수동으로 목록을 업데이트하는 대신 자동 세분화 활용
- **타겟팅 향상** – 행동, 리드 상태, 참여도에 따른 연락처 그룹핑
- **CRM 조직 개선** – 연락처 목록을 구조적이고 검색 가능하게 유지
- **자동화 강화** – 워크플로우에서 스마트 리스트를 활용한 자동 액션 실행

---

## 스마트 리스트란?

스마트 리스트는 미리 정의된 필터와 조건에 따라 자동으로 업데이트되는 동적 연락처 목록입니다. 수동으로 업데이트하지 않는 한 변경되지 않는 정적 목록과 달리, 스마트 리스트는 새로운 연락처가 지정된 조건을 충족하거나 더 이상 충족하지 않을 때 지속적으로 조정됩니다. 커스텀 필드와 같은 특정 조건에 따라 연락처를 맞춤 설정하고 필터링할 수 있으며, 표시할 컬럼을 선택할 수 있고, 연락처 데이터가 변경될 때마다 실시간으로 업데이트됩니다.

이 기능은 **하위 계정 → 연락처(Contacts) 탭 → 스마트리스트(Smartlists)**에서 찾을 수 있습니다.

## 스마트 리스트 작동 원리

- **실시간 업데이트**: 태그, 커스텀 필드, 참여 활동, 기회 단계 등의 필터를 기반으로 연락처가 자동으로 추가되거나 제거됩니다.
- **자동 세분화**: 수동 업데이트 없이 리드, 활성 고객, 잠재 고객 목록을 쉽게 생성할 수 있습니다.
- **고급 필터링**: 여러 필터를 동시에 적용할 수 있습니다(예: "지난 30일 이내 이메일 열람" + "미결제 인보이스 보유").
- **워크플로우 연동**: 스마트 리스트는 자동 워크플로우를 트리거하여 대규모 개인화된 참여를 보장합니다.

## 스마트 리스트 설정하기

Hyperclass에서 스마트 리스트를 생성하는 것은 특정 기준에 따라 연락처를 자동으로 세분화할 수 있는 간단한 과정입니다. 수동으로 목록을 업데이트하는 대신, 스마트 리스트는 연락처 그룹을 실시간으로 체계적이고 업데이트된 상태로 유지합니다. 다음 단계에 따라 첫 번째 스마트 리스트를 생성해 보세요:

- 연락처(Contacts) > 연락처/스마트 리스트로 이동하여 "더 많은 필터(More Filters)"를 클릭합니다.
- 화면 오른쪽의 드롭다운 메뉴에서 하나 또는 여러 개의 필터를 선택합니다.
- 필터를 선택한 후 필요한 세부 정보를 입력하고 "적용(Apply)"을 클릭하여 필터를 저장합니다.
- 더 많은 필터를 추가하려면 "AND" 옵션을 선택합니다.
- 필터를 목록으로 영구 저장하려면 화면 하단의 "스마트 리스트로 저장(Save as smart list)"을 클릭합니다.

![스마트 리스트 생성 과정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155041733215/original/JYZsQNRq6e2sAIa_XqmGiqLEDRR6dwv7BA.gif?1739800334)

## 스마트 리스트 관리하기

스마트 리스트는 비즈니스 요구 사항에 더 잘 맞도록 수정, 필터링 및 관리할 수 있습니다. 다양한 필터를 적용하거나 조건을 조정하거나 기존 목록을 복제하여 더 세밀한 세분화를 할 수 있습니다.

- 스마트 리스트에 액세스하려면 연락처 섹션으로 이동하여 상단에서 스마트 리스트 이름을 선택합니다.
- 기존 스마트 리스트를 수정하려면 해당 리스트를 열면 오른쪽에서 저장된 필터를 확인할 수 있습니다.
- 쓰레기통 아이콘이나 X를 선택합니다. 필터를 수정하려면 연필 아이콘이나 드롭다운 화살표를 클릭하여 필터를 제거합니다. 저장 아이콘을 사용하여 변경 사항을 저장합니다.
- 연락처 페이지 상단 오른쪽의 "스마트 리스트 관리(Manage Smart Lists)"로 이동하여 스마트 리스트를 관리할 수 있습니다. 이 영역에서 오른쪽 아이콘을 사용하여 스마트 리스트를 **복제, 공유, 이름 변경 또는 삭제**할 수 있습니다.

## 스마트 리스트 생성 예시

지난 90일 동안 연락한 적이 없거나 한 번도 연락하지 않은 모든 연락처 목록을 만들어보겠습니다.

1. 스마트 리스트 > 전체 탭으로 이동
2. 더 많은 필터 클릭
3. '필터 추가'라고 표시된 텍스트 상자에 '마지막 활동'을 입력합니다. '마지막 활동' 필터가 옵션 중 하나로 표시됩니다
4. 마지막 활동 필터를 선택합니다. 추가되면 '이상' 라디오 버튼을 클릭합니다. 90일을 입력합니다
5. 컬럼 드롭다운을 클릭합니다. 마지막 활동 컬럼을 선택합니다
6. + 아이콘을 눌러 이 목록의 이름을 지정하고 저장합니다

![스마트 리스트 생성 예시](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155041734610/original/DYq6diiU1eO5FTlQUuwDWLaIjFopdJ1LaQ.gif?1739801031)

## 스마트 리스트 컬럼 순서 변경

스마트 리스트는 드래그 앤 드롭 컬럼 관리를 지원하므로 리스트 보기를 빠르게 재배치하고 가장 중요한 필드를 앞쪽 중앙에 유지할 수 있습니다.

- 연락처 → 스마트 리스트로 이동한 다음 편집하려는 스마트 리스트를 엽니다.
- 목록 보기에서 컬럼/필드 관리 옵션을 엽니다(일반적으로 테이블 상단 근처에 있으며 표시할 필드를 관리할 수 있습니다).
- 컬럼 관리자에서 컬럼을 원하는 순서로 드래그 앤 드롭합니다.
- 컬럼 관리자를 닫고 스마트 리스트로 돌아가면 새 컬럼 순서가 즉시 반영됩니다.

![컬럼 순서 변경](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155061802130/original/TrhoI1Q3LVsXorsM-ctDqTz1HI7GvJb9kg.gif?1767376414)

**참고:** 스마트 리스트는 가장 최근에 저장된 컬럼 순서와 표시 상태를 저장합니다. 나중에 컬럼을 재정렬하거나 필드를 표시/숨기는 경우 해당 변경 사항을 유지하려면 스마트 리스트를 다시 저장해야 합니다. 그렇지 않으면 마지막에 저장된 레이아웃을 계속 사용합니다.

## 자주 묻는 질문

**Q: 기존 연락처를 기존 스마트 리스트에 수동으로 추가할 수 있나요?**

아니요, 연락처는 스마트 리스트에 수동으로 추가되지 않습니다. 해당 스마트 리스트에 설정된 필터와 조건에 따라 자동으로 추가됩니다. 스마트 리스트의 필터 조건에 맞는 태그를 연락처에 추가하면 해당 스마트 리스트에 자동으로 추가됩니다.

**Q: 사용자가 '설정' 권한 없이도 스마트 리스트를 관리할 수 있나요?**

사용자는 '설정' 권한 없이는 스마트 리스트를 관리할 수 없습니다. 이 문제는 향후 업데이트를 위해 Hyperclass의 로드맵에 추가되었습니다.

**Q: 누가 스마트 리스트를 생성하고 공유했는지 확인할 수 있나요?**

Hyperclass는 누가 스마트 리스트를 생성하고 공유했는지에 대한 정보를 기록하지 않습니다.

**Q: 모든 사용자가 스마트 리스트를 볼 수 있나요, 아니면 생성한 사용자만 볼 수 있나요?**

스마트 리스트는 로케이션 레벨이 아닌 사용자 레벨에서 생성됩니다. 그러나 "글로벌 대 로컬" 스마트 리스트 기능으로 사용자는 자신의 스마트 리스트를 전체 계정에 푸시하여 모든 사용자에게 표시되도록 할 수 있습니다.

**Q: 스마트 리스트를 계정 간에 이전하거나 스냅샷으로 이동할 수 있나요?**

현재로서는 스마트 리스트를 한 계정에서 다른 계정으로 이동하거나 스냅샷으로 가져오는 기능이 없습니다.

**Q: 스마트 리스트로 이메일 캠페인을 예약하면 발송 전에 목록이 업데이트되나요?**

네, 스마트 리스트는 동적이므로 수신자 목록은 캠페인이 예약될 때가 아니라 발송 시점에 생성됩니다. 즉, 예약된 발송 전에 필터 조건을 충족하는 새 연락처는 포함되고, 더 이상 조건을 충족하지 않는 연락처는 제외됩니다.

**Q: 스마트 리스트에서 페이지당 20개 대신 100개의 연락처를 표시하려면 어떻게 하나요?**

우측 하단의 페이지당 연락처 수 선택기를 사용하여 100을 선택하세요(옵션: 20, 50, 100).
