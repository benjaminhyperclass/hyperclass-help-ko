# 연락처 생년월일 필드 (생일)

Hyperclass의 생일 필드를 사용하면 개인화된 소통이 가능하고, 연락처 생일을 기반으로 자동화 캠페인을 구축하며, 스마트 리스트에서 나이별로 연락처를 필터링할 수 있습니다. 이 가이드에서는 생일 필드를 효과적으로 활용하는 방법을 설명합니다.

## **생일 필드란?**

생일 필드는 연락처의 생년월일(월/일/년)을 저장하는 기본 시스템 필드입니다. 다음 방법으로 정보를 입력할 수 있습니다:

- 수동 입력
- 폼(Form) 제출
- CSV 가져오기(Import)
- API 연동(Integration)

이 필드를 사용하면 생일을 기반으로 한 자동화 트리거(Trigger)를 설정하고 나이별로 연락처를 세분화할 수 있습니다.

## **생일 축하 캠페인 만드는 방법**

생일 축하 메시지나 특별 혜택을 자동으로 보내는 생일 워크플로우(Workflow)를 설정할 수 있습니다.

### **1단계:** 생일 트리거 설정

- Workflows(워크플로우) > Create New Workflow(새 워크플로우 만들기)로 이동합니다.
- 트리거를 Birthday(생일)로 설정합니다.
- 트리거 조건을 선택합니다:
  - On the birthday (생일 당일)
  - Before the birthday (생일 전, 예: 7일 전)
  - After the birthday (생일 후, 예: 1일 후)

![생일 트리거 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155047948330/original/A-EJmS-eU8slwJQO8xWd4wXGOcnUm1t4yg.gif?1749477876)

### **2단계:** 액션(Action) 정의

- Send Email(이메일 발송): 개인화된 생일 축하 메시지나 특별 혜택
- Send SMS(SMS 발송): 간단한 생일 축하 메시지나 쿠폰 코드
- Assign Task(할 일 배정): 필요시 팀에게 수동 후속 조치 알림

![워크플로우 액션 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155047948420/original/dY-FvszUwlOsrnpiCsrWVS72Ety5D6TVsw.gif?1749478000)

### **워크플로우 구조 예시**

| 단계 | 액션 |
|------|------|
| 1 | 생일 트리거 |
| 2 | 생일 축하 SMS 발송 |
| 3 | 생일 축하 이메일 발송 |
| 4 | (선택사항) 내부 팀에 알림 |

## **스마트 리스트에서 나이별로 연락처 필터링하는 방법**

생일 필드를 사용하여 나이를 기준으로 연락처를 세분화하고 타겟팅된 커뮤니케이션을 할 수 있습니다.

### **나이별 필터링 단계**

- Contacts(연락처) > Smart Lists(스마트 리스트)로 이동합니다.
- Birthday(생일) 필드의 Age(나이) 조건을 사용하여 필터를 적용합니다.
- 적절한 비교 조건을 선택합니다:

| 필터 조건 | 사용 예시 |
|-----------|-----------|
| Age is greater than X (나이가 X보다 많음) | 50세 이상 시니어 타겟팅 |
| Age is less than X (나이가 X보다 적음) | 25세 미만 젊은 고객 프로모션 |
| Age is between X and Y (나이가 X~Y 사이) | 특정 연령대 타겟팅 (예: 30-40세) |

![나이별 필터링](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155047948100/original/rQKJ6HKR4-6qSRJEDZATqZb2GlEdLXwR6A.gif?1749477612)

시스템은 현재 날짜와 저장된 생일을 기준으로 나이를 자동으로 계산합니다.

## **모범 사례**

- **생일 정보 조기 수집**: 리드 수집 폼에 생일 필드를 포함하여 초기에 정보를 수집하세요.
- **날짜 형식 검증**: 자동화 트리거가 올바르게 작동하도록 MM/DD/YYYY 형식을 사용하세요.
- **자동화 테스트**: 워크플로우를 실제 적용하기 전에 샘플 연락처로 테스트하세요.
- **시간대 고려**: 설정된 경우 워크플로우가 연락처별 시간대를 사용할 수 있음을 고려하세요.

## **일반적인 문제 및 해결 방법**

| 문제 | 해결 방법 |
|------|-----------|
| 생일 자동화가 작동하지 않음 | 생일 필드가 완전하고 올바른 형식의 날짜로 채워져 있는지 확인하세요. |
| 잘못된 나이 계산 | 출생 연도가 올바르게 입력되었는지 확인하세요. 연도가 누락되거나 잘못되면 오류가 발생할 수 있습니다. |
| 생일에 SMS나 이메일이 발송되지 않음 | 워크플로우 타이밍, 트리거 설정, 연락처 시간대 설정을 다시 확인하세요. |
