# 일반 질문

## 연락처의 문서는 공유가 가능한가요?

현재 연락처(Contacts) 하위에 저장된 모든 문서는 공개 URL이 생성되어 누구나 해당 URL에 접근할 수 있으면 문서를 공유하고 볼 수 있습니다. 단, 이러한 URL은 Google이나 Bing 같은 검색엔진에서는 검색되지 않습니다. 

그러나 연락처의 문서 기능에는 민감한 정보를 저장하지 않는 것을 권장합니다.
