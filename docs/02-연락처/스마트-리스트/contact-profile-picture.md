# 연락처 프로필 사진

이 가이드에서는 Hyperclass의 프로필 사진 기능 사용법을 알아보겠습니다. 연락처에 이미지를 업로드하는 방법과 시스템에서 이미지가 표시되는 위치를 확인할 수 있어요. 프로필 사진은 연락처와의 상호작용을 개인화하고, 특히 대화(Conversations)와 연락처 목록에서 시각적 식별을 개선하는 데 도움이 됩니다.

## 프로필 사진 기능이란?

프로필 사진 기능을 사용하면 Hyperclass의 연락처 레코드에 시각적 식별자(사진 또는 로고)를 추가할 수 있어요. 이 이미지는 해당 연락처가 활동하는 앱의 모든 곳에서 표시되며, 주로 연락처 스마트 리스트와 대화 인박스에서 볼 수 있습니다. 프로필 사진은 현재 연락처 생성 시에만 추가할 수 있고, 나중에 변경할 수는 없어요. 하지만 일단 업로드하면 다양한 인터페이스 영역에 나타나서 팀이 누구와 소통하고 있는지 즉시 알아볼 수 있게 도와줍니다.

---

## 프로필 사진의 주요 장점

프로필 이미지는 단순한 장식이 아니라, 사용자 경험과 커뮤니케이션 워크플로우를 개선하는 실용적인 장점들을 제공해요.

- 연락처와 대화 화면에서 연락처를 빠르게 식별할 수 있어요
- 고객과의 상호작용에 개인화된 느낌을 더해줍니다
- 중복 리드를 관리할 때 혼란을 줄여줘요
- 팀에게 전문적이고 시각적인 인터페이스를 제공합니다
- 한눈에 알아볼 수 있게 해서 사용자 네비게이션을 간편하게 만들어줘요
- 더 빠른 컨텍스트 전환으로 팀 생산성을 향상시킵니다

---

## 현재 프로필 사진 기능의 제한사항

도움이 되는 기능이지만, 아직 발전 중이며 다음과 같은 제한사항이 있어요:

- 연락처 생성 후에는 프로필 이미지를 추가, 변경 또는 업데이트할 수 없습니다
- 프로필 사진이 **전체 연락처 프로필 레코드/탭 내부에서는 표시되지 않아요**
- 업로드된 이미지에 대한 드래그 앤 드롭이나 크롭 도구가 제공되지 않습니다
- 아직 API를 통한 지원이 되지 않아요

---

## 프로필 사진 추가하는 방법

프로필 사진은 **새 연락처를 생성할 때만** 추가할 수 있어요. 연락처를 저장한 후에는 아직 프로필 사진을 업데이트할 방법이 없습니다.

**중요**: 기존 연락처에 프로필 사진을 추가해야 한다면, 새 연락처를 만들어서 프로필 사진을 추가하고, 이 새 연락처를 기존 연락처와 합치기(merge)해야 해요.

프로필 사진을 추가하려면 왼쪽 네비게이션의 연락처(Contacts) 탭으로 이동해서 ➕ (플러스) 버튼을 클릭하여 새 연락처를 추가하세요.

- 연락처 생성 창에서 "Personal Logo"라고 표시된 이미지 영역을 클릭하세요
- 이미지 파일을 업로드하세요 (최대 2.5MB, 512x512px 권장)
- 연락처 세부정보를 입력하고 저장을 클릭하세요

![프로필 사진 추가하기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155045078808/original/97GqJ3UH1TxnPmOimUqQPBq33ZuzlzfoHw.gif?1744660212)

저장하면 이제 플랫폼의 여러 부분에서 업로드한 이미지와 함께 연락처가 표시됩니다.

**중요**: 프로필 사진은 나중에 편집할 수 없으므로, 생성 시에 이미지를 업로드해야 해요.

**참고** - 원하는 프로필 사진으로 새 연락처를 만든 후, 두 연락처를 합칠 수 있습니다. 자세한 내용은 [여러 연락처 합치기](../기타/merging-multiple-contacts.md)를 참조하세요.

## 연락처 탭에서 프로필 사진 보기

## 자주 묻는 질문

**Q: 나중에 프로필 사진을 업데이트하거나 변경할 수 있나요?**  
현재로는 불가능해요. 프로필 사진은 새 연락처를 생성할 때만 추가할 수 있습니다.

**Q: 업로드한 이미지는 어디에 표시되나요?**  
연락처 스마트 리스트, 대화 인박스, 그리고 오른쪽 연락처 패널에서 볼 수 있어요.

**Q: 기존 연락처에 사진을 추가할 수 있나요?**  
아니요. 현재로는 이미 생성된 연락처의 프로필 사진을 수정하는 옵션이 없어요.

**Q: 이상적인 이미지 크기와 형식은 무엇인가요?**  
이미지는 512x512 픽셀이고 JPG 또는 PNG 형식으로 2.5MB 미만이어야 해요.

**Q: 향후 릴리스에서 편집이 가능해질까요?**  
Hyperclass는 연락처 기능을 지속적으로 향상시키고 있으며, 향후 업데이트에서 이미지 편집을 지원할 수도 있어요.
