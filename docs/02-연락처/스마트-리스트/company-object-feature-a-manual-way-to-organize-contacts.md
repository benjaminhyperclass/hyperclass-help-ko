# 회사 객체 기능: 연락처를 정리하는 수동 방식

하이퍼클래스의 회사(Company) 객체 기능은 관련된 연락처들을 하나의 공유 회사 프로필 아래 그룹화하는 데 도움을 줍니다. 이를 통해 같은 클라이언트 비즈니스와 작업하는 팀의 관계 관리, 연락처 정리, 활동 추적을 향상시킬 수 있습니다.

## 회사 객체 기능이란?

회사 객체를 사용하면 여러 연락처를 하나의 회사 엔터티와 연결하여 조직 단위에서 고객 관계를 더 쉽게 관리할 수 있습니다. 이는 특히 팀이 같은 비즈니스의 다양한 사람들과 상호작용할 때 유용합니다. 이들을 하나의 레코드 아래 그룹화함으로써 집합적인 참여도를 확인하고, 데이터를 중앙화하며, 모든 접촉 지점에서 커뮤니케이션을 개선할 수 있습니다.

---

## 회사 객체 기능의 주요 장점

회사 객체를 사용하면 연락처 관계 관리에서 실질적인 이점을 얻을 수 있습니다:

- **정리된 연락처 관리**: 여러 연락처를 하나의 회사 아래 쉽게 그룹화
- **통합된 활동 추적**: 회사와 관련된 상호작용 및 활동 모니터링
- **향상된 관계 인사이트**: 팀이 조직과 어떻게 소통하는지에 대한 완전한 시야 확보
- **효율적인 커뮤니케이션**: 회사에 연결된 모든 연락처와 쉽게 소통
- **향상된 데이터 정확성**: 회사와 연결된 연락처 전반에 걸쳐 일관되고 업데이트된 정보 유지

## 회사 객체 생성 방법

하이퍼클래스에서 새 회사를 설정하려면 다음 단계를 따르세요:

### 1단계: 회사 섹션으로 이동

- 왼쪽 메인 네비게이션 바를 사용하여 연락처(Contacts)로 이동
- 페이지 상단의 보조 네비게이션 리본에서 **회사(Companies)** 탭으로 이동

![회사 섹션 이동](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155041969865/original/afPxEFXXuOBxnAqTaV3LFBqNvqJ_8oglww.png?1740065034)

### 2단계: 새 회사 추가

- 우상단의 파란 **+ 새 회사(+ New Company)** 버튼을 클릭합니다.

![새 회사 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155041969989/original/h0xP2Ge9B26gWwWxU6549DHsoSBCwKFHVA.png?1740065114)

### 3단계: 회사 세부정보 입력

화면 오른쪽 박스에 회사 세부정보를 입력합니다. 추가할 수 있는 세부정보는 다음과 같습니다:

- **회사명 (필수)**
- **전화번호**
- **이메일**
- **웹사이트**
- **설명**
- **주소 세부정보** (거리, 도시, 우편번호, 국가)

참고: 회사명만 필수 입력 사항이지만, 더 많은 세부정보를 추가하면 유용성이 향상됩니다.

![회사 세부정보 입력](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155041970367/original/LbLWS8aCQVKZDC4wUpWeqzJFaU4RmCQzJg.gif?1740065352)

## 회사 정보 편집 및 업데이트

- **회사 목록으로 이동**
  메인 네비게이션 바를 사용하여 **연락처(Contacts)**로 이동
- 상단 네비게이션 바에서 회사(Companies) 탭을 선택
- 편집하고 싶은 회사 옆의 **세 점 (...)** 을 클릭하고 회사 편집(Edit Company)을 선택

- **세부정보 업데이트**
  필요한 필드를 수정합니다.

- **변경사항 저장**
  **저장(Save)**을 클릭하여 업데이트를 적용합니다.

---

## 회사 삭제 방법

- **회사 목록으로 이동**
  메인 네비게이션 바를 사용하여 **연락처(Contacts)**로 이동
- 상단 네비게이션 바에서 회사(Companies) 탭을 선택

- **삭제할 회사 선택**
  편집하고 싶은 회사 옆의 **세 점 (...)** 을 클릭하고 회사 삭제(Delete Company)를 선택

- **삭제 확인**
  **예(Yes)**를 클릭하여 확인합니다.

*참고: 회사를 삭제하면 연결된 연락처의 회사 필드가 지워지지만 연락처 자체는 삭제되지 않습니다.*

## 회사에 연락처 추가하기

- **연락처 프로필로 이동**
  메인 네비게이션 바를 사용하여 연락처(Contacts)로 이동
- 회사에 추가하고 싶은 연락처를 찾아 연락처 이름을 클릭

- **회사 할당**:
  페이지 왼쪽 보조 네비게이션 바에서 회사(Company) 탭을 선택
- 회사를 검색하고 선택합니다.

- **변경사항 저장**
  **저장(Save)**을 클릭하여 연락처의 연결을 업데이트합니다.

*일괄 할당의 경우, 여러 연락처를 선택하여 한 번에 회사에 할당할 수 있습니다.*

## 회사에서 연락처 제거하기

- **회사 프로필 접근**
  메인 네비게이션 바를 사용하여 **연락처(Contacts)**로 이동
- 상단 네비게이션 바에서 회사(Companies) 탭을 선택
- 회사를 선택합니다.

- **연결된 연락처 보기**
  **연락처(Contacts)** 탭으로 이동합니다.

- **연락처 제거**
  연락처 옆의 휴지통 아이콘을 클릭합니다.

- **제거 확인**
  **예(Yes)**를 클릭하여 확정합니다.

*참고: 회사에서 연락처를 제거해도 연락처가 삭제되지 않으며, 단지 연결만 제거됩니다.*

**회사 연결 자동화**: 하이퍼클래스는 회사 할당을 간소화하는 자동화 기능을 제공합니다. [회사 객체 자동화](../../11-설정/company-object-automation.md)를 참조해주세요.

## 연락처-회사 연결 자동화

연락처의 "사업체명" 필드를 기반으로 이 프로세스를 자동화할 수 있습니다.

활성화하려면:
- 설정(Settings) > 객체(Objects) > 회사(Company)로 이동
- "연락처 사업체명으로부터 회사 자동 생성 및 연결" 토글을 켜기

자세한 내용은 [회사 객체 자동화](../../11-설정/company-object-automation.md) 아티클에서 확인하세요.

## 자주 묻는 질문

**Q: 하나의 연락처에 여러 회사를 할당할 수 있나요?**
A: 아니요, 연락처는 한 번에 하나의 회사와만 연결될 수 있습니다.

**Q: 회사를 삭제하면 모든 연락처가 삭제되나요?**
A: 아니요, 회사 연결만 제거됩니다. 연락처는 데이터베이스에 남아있습니다.

**Q: 연락처를 회사에 일괄 할당할 수 있나요?**
A: 네, 연락처 목록에서 여러 연락처를 선택하여 한 번에 회사에 할당할 수 있습니다.

**Q: 특정 회사에 연결된 모든 연락처를 어떻게 찾나요?**
A: 연락처(Contacts) > 회사(Companies)로 이동하여 회사를 선택하고 연락처(Contacts) 탭을 확인하세요.

**Q: 수동으로 할당하지 않고도 연락처-회사 연결을 자동화할 수 있나요?**
A: 네, 설정(Settings) > 객체(Objects) > 회사(Company)에서 "연락처 사업체명으로부터 회사 자동 생성 및 연결"을 활성화하세요.
