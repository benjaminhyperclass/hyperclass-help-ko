# Hyperclass에서 중복 연락처 합치는 방법

이 가이드에서는 Hyperclass 하위 계정에서 중복된 연락처를 하나로 합치는 방법을 설명합니다. CRM을 깔끔하게 정리하고, 커뮤니케이션 정확도를 높이며, 연락처 목록을 효율적으로 관리할 수 있습니다.

## 중복 연락처 합치기란?

Hyperclass에서 중복 연락처를 합치는 것은 **같은 개인이나 회사에 대한 여러 연락처 기록을 하나의 통합된 연락처로 병합하는 기능**입니다. 이를 통해 혼란을 줄이고, 중복된 커뮤니케이션을 방지하며, 깔끔한 연락처 데이터베이스를 유지할 수 있습니다.

중복 연락처는 수동 입력, CSV 가져오기, 폼 제출, 또는 Zapier 같은 외부 연동을 통해 발생할 수 있습니다. Hyperclass는 이름, 이메일, 전화번호 같은 일치하는 필드를 기반으로 중복 연락처를 식별합니다.

## 중복 연락처 합치기의 주요 장점

- CRM 데이터 품질을 보장하고 중복 아웃리치를 방지합니다
- 노트, 할 일, 대화, 워크플로우를 하나의 기록으로 통합합니다
- 캠페인 정확도와 리포팅을 개선합니다
- 고객 지원 및 영업 후속 조치 시 혼란을 줄입니다
- 자동화와 연락처 라우팅을 최적화합니다

## 중복 연락처 합치는 방법

### 1단계: 중복 연락처 찾기

- Hyperclass 하위 계정에서 **연락처(Contacts)** 메뉴로 이동합니다.

- **빠른 검색(Quick Search)** 바를 사용하여 **이름**이나 **이메일** 같은 공통 식별자를 입력해 중복 연락처를 찾습니다.

- 연락처 목록에서 **수동으로 검색**할 수도 있습니다.

- 합치려는 중복 **연락처**를 선택합니다. (한 번에 최대 **10개**)

![중복 연락처 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155047142917/original/qr0Rp-jnERMSwvdOgIRTfc-wRglv8JCDWQ.png?1748013462)

### 2단계: 합치기 과정 시작

- 연락처를 선택한 후 **연락처 합치기(Merge Contacts)** 버튼을 클릭합니다.

![연락처 합치기 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155047143084/original/FlA_20riir6tNwmkoN1SiarRH81gFwbNqA.png?1748013673)

### 3단계: 마스터 기록 선택

마스터 기록을 선택하면 합치기 과정에서 해당 기록의 주요 정보가 보존되고, 다른 기록들의 관련 데이터가 이와 결합되어 하나의 완전한 연락처가 생성됩니다.

- 선택한 연락처들의 세부 정보를 검토합니다.

- **마스터 연락처(Master Contact)**를 선택합니다.

- 선택한 연락처 중에서 마스터 연락처의 **기본 이메일/전화번호(Primary Email/Phone)**를 선택할 수도 있습니다.

- 연락처 합치기 버튼을 클릭합니다.

![마스터 연락처 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155047143124/original/kV2ZVsRXshL5tZo4hZDOnjTNoazPIs0iBw.png?1748013762)

**중요**: 합치기 시 마스터 연락처가 모든 핵심 데이터를 유지합니다. 하위 연락처의 정보는 영구적으로 손실되므로, 확인하기 전에 신중히 검토하세요.

### 4단계: 합치기 확인

- **이 작업은 되돌릴 수 없음을 이해합니다(I understand this action cannot be reverted)** 체크박스를 선택합니다.

- **연락처 합치기(Merge Contacts)** 버튼을 클릭합니다.

![합치기 확인](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155047143139/original/u8eUMD8Z-E6YTM6317ywBN1KrUBsHh5R2g.png?1748013804)

## 연락처 중복 제거 설정

소스에서 중복 연락처를 방지하면 나중에 상당한 시간과 혼란을 절약할 수 있습니다. 연락처 중복 제거 설정을 통해 시스템이 새로운 연락처 입력을 처리하는 방식을 제어하여 중복이 발생하기 전에 예방할 수 있습니다.

- `Settings(설정) → Business Profile(비즈니스 프로필) → Contact Preferences(연락처 설정)`로 이동합니다.

- 중복 연락처 허용(Allow Duplicate Contacts)을 활성화하거나 비활성화합니다.

- 기본 및 보조 중복 제거 필드(이메일 또는 전화번호)를 선택합니다.

![중복 제거 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155047143310/original/15-GXf2cSgSzOMgLpHvtD5YfDxjTxBHgJQ.png?1748014030)

## 자주 묻는 질문

**Q: LeadConnector 모바일 앱에서 연락처를 합칠 수 있나요?**

현재 LeadConnector 모바일 앱에서는 연락처 합치기 기능을 사용할 수 없습니다. 이 기능은 추가 일괄 작업과 함께 3분기에 출시될 예정입니다. 현재로서는 Hyperclass 데스크톱 버전을 사용하여 연락처를 합칠 수 있습니다.

**Q: 합치기 중에 필드 데이터는 어떻게 되나요?**

선택된 마스터 기록이 자신의 세부 정보를 유지하며, 다른 기록의 고유한 필드들이 추가됩니다.

**Q: "연락처 합치기" 버튼이 보이지 않는 이유는 무엇인가요?**

"연락처 합치기" 버튼은 2개 이상의 유효한 연락처를 선택했을 때만 나타납니다(최대 10개). 합치려는 연락처가 핵심 필드(이름, 이메일, 전화번호 등)가 누락된 경우, 시스템이 합치기를 허용하지 않을 수 있습니다. 또한 다음을 확인하세요: **1.** 연락처 관리 관리자 권한이 있는지 **2.** 같은 하위 계정의 연락처를 선택하고 있는지 **3.** 두 연락처 모두 목록 보기에서 보이는지(필터 확인)

**Q: 연락처를 일괄로 합칠 수 있나요?**

아니요, 한 번에 최대 10개의 연락처까지만 합칠 수 있습니다.

**Q: Hyperclass에서 중복 연락처가 생기는 원인은 무엇인가요?**

중복은 수동 입력, CSV 가져오기, 폼, 또는 Zapier 같은 외부 도구에서 발생할 수 있습니다.

**Q: 연락처의 커스텀 값이나 필드에 영향이 있나요?**

보조 연락처의 커스텀 필드는 기본 연락처의 해당 필드가 비어있는 경우에만 추가됩니다. 덮어쓰기는 발생하지 않습니다.

**Q: 어떤 연락처를 마스터로 선택해야 하나요?**

가장 정확하고 최신 정보를 가진 기록을 선택하세요. 마스터 연락처는 대화, 최종 활동, 파일 및 기타 데이터를 유지합니다.

**Q: Hyperclass가 모든 중복 연락처를 자동으로 찾아줄 수 있나요?**

아니요. 이름, 전화번호, 이메일로 수동 검색할 수는 있지만, 현재 Hyperclass는 중복에 대한 전체 데이터베이스 스캔 기능을 제공하지 않습니다.

**Q: 폼이나 캘린더 제출로 인한 중복을 어떻게 방지하나요?**

중복 제거 설정에서 "중복 연락처 허용"을 끄세요.

**Q: 중복을 합칠 때 연락처의 워크플로우는 어떻게 되나요?**

합치기 중에 중복 연락처가 삭제되면, Hyperclass는 워크플로우 실행을 마스터 연락처로 이전하여 자동화가 계속됩니다. 마스터가 이미 동일한 워크플로우에 등록되어 있다면, Hyperclass는 추가 실행을 중지하고 이미 실행 중인 것만 유지합니다.
