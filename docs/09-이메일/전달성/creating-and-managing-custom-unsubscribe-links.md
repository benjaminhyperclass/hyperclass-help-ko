# 커스텀 수신거부 링크란?

커스텀 수신거부 링크는 마케팅 이메일에 포함된 개인화된 수신거부 링크로, 수신자가 브랜드에 맞는 방식으로 향후 커뮤니케이션을 거부할 수 있게 해줍니다. 사용자를 일반적인 확인 페이지로 리다이렉트하는 기본 수신거부 링크와 달리, 커스텀 수신거부 링크를 사용하면 사용자 경험을 완전히 제어할 수 있습니다.

## 커스텀 수신거부 링크의 주요 장점

- 사용자 경험 향상: 수신거부 결정을 확인하는 브랜드 맞춤 랜딩 페이지로 사용자를 안내합니다.
- 구독 설정 제공: 모든 커뮤니케이션이 아닌 특정 유형의 이메일만 수신거부할 수 있는 옵션을 제공합니다.
- 규정 준수 유지: CAN-SPAM Act, GDPR 등 이메일 마케팅 규정을 준수할 수 있습니다.
- 전달성 개선: 수신자가 쉽고 명확하게 수신거부할 수 있는 방법을 제공하여 스팸 신고를 줄입니다.

참고: Hyperclass의 커스텀 수신거부 링크는 트리거 링크와 워크플로우를 사용하여 연락처를 자동으로 "연락 금지(Do Not Disturb, DND)"로 표시하고, 추가 이메일을 받지 않도록 합니다.

## 커스텀 수신거부 링크 생성하기

Hyperclass에서 커스텀 수신거부 링크를 설정하려면 다음 단계를 따르세요:

### 1단계: 수신거부 랜딩 페이지 생성

수신거부 작업을 확인하는 랜딩 페이지를 디자인하세요. 이 페이지는 사용자가 성공적으로 수신거부했음을 알리고, 원하는 경우 구독 설정을 관리할 수 있는 옵션을 제공해야 합니다.

- 하위 계정에서 사이트(Sites) → 퍼널(Funnels)로 이동합니다.
- + 새 퍼널(New Funnel)을 클릭하고 "이메일 수신거부"라고 이름을 지정합니다.
- 이 퍼널 안에 "/unsubscribe" 같은 경로로 "이메일 수신거부 페이지"라는 새 단계를 추가합니다.

![수신거부 퍼널 생성 과정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155045847538/original/vOI_BRnT7Sh8Uhga2X-Qd0bqTKKOvFHLfg.gif?1745934146)

수신거부 확인 메시지를 반영하도록 페이지 콘텐츠를 디자인하세요. 참고용으로 데모 페이지를 아래에 추가했습니다. 자세한 내용은 퍼널 스타일링 가이드를 확인해보세요.

![수신거부 페이지 예시](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155043604358/original/xXa1SJ8bT9_EqeMuuG4TvFyq5KyGinchmg.png?1742397563)

### 2단계: 트리거 링크 설정

Hyperclass의 트리거 링크를 사용하면 연락처가 특정 링크를 클릭할 때 특정 작업을 수행할 수 있습니다. 이 경우, 수신거부 링크를 클릭하면 고객의 설정이 업데이트됩니다.

- 하위 계정에서 마케팅(Marketing) → 트리거 링크(TriggerLinks)로 이동합니다.
- 드롭다운에서 링크(Links)를 선택하고 링크 추가(Add Link) 버튼을 클릭합니다.
- 링크 이름을 추가합니다.
- 앞서 생성한 수신거부 랜딩 페이지의 URL을 링크 URL(Link URL) 필드에 붙여넣습니다.
- 저장(Save)을 클릭합니다.

![트리거 링크 설정 과정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155045848397/original/AZ0q9hJiQmrl8OIkTRxZTnztuHaXuiMRww.gif?1745934719)

### 3단계: 수신거부 관리 워크플로우 생성

수신거부 링크를 클릭할 때 연락처의 상태를 자동으로 업데이트하는 워크플로우를 설정하세요.

- 하위 계정에서 자동화(Automation) → 워크플로우(Workflows)로 이동합니다.
- + 워크플로우 생성(Create Workflow)을 클릭하고 + 처음부터 시작(Start from Scratch)을 선택합니다.
- 워크플로우 이름을 지정합니다.

![워크플로우 생성 과정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155045848742/original/Ls0E4Ede8T3Q2-0MbmOOyRk2HBijnI0PXQ.gif?1745934914)

#### 트리거 설정

- 새 트리거 추가(Add New Trigger)를 클릭하고 이벤트에서 트리거 링크 클릭됨(Trigger Link Clicked)을 선택합니다.
- 워크플로우 트리거 이름을 추가합니다.
- 필터 추가(Add Filter)를 클릭하고 드롭다운에서 트리거 링크(Trigger Link)를 선택합니다.
- 드롭다운에서 앞서 저장한 이메일 수신거부 트리거 링크를 선택합니다.
- 트리거 저장(Save Trigger) 버튼을 클릭합니다.

![트리거 설정 과정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155045849525/original/CAMcn27a9ODqVarW-57audU1K3FsJHKteA.gif?1745935497)

#### DND 액션 추가

- + 버튼을 클릭하여 액션을 추가합니다.
- DND 활성화/비활성화(Enable/Disable DND) 액션을 선택합니다.
- 특정 채널에 대해 DND 활성화(Enable DND for Specific Channels)를 선택하고 채널 드롭다운에서 이메일(Email)을 선택합니다.
- 워크플로우를 저장하고 발행합니다.

![DND 액션 추가 과정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155043621730/original/PyVtFQ7iZvk_ukmK_jOYctNSuWPOiZgCCg.gif?1742416222)

### 4단계: 이메일에 수신거부 링크 삽입

- 이메일 템플릿 편집기에서 "수신거부" 또는 "수신거부하기" 같은 텍스트를 추가합니다.
- 이 텍스트를 강조 표시하고 링크 아이콘을 클릭합니다.
- 링크 URL 드롭다운에서 트리거 링크(Trigger Links)를 클릭하고 저장된 수신거부 링크를 선택합니다.
- 템플릿을 저장합니다.

**참고:**

워크플로우를 통해 전송되는 이메일에는 수신거부 링크가 자동으로 포함되지 않습니다. 규정을 준수하려면 워크플로우에서 사용하는 각 이메일 템플릿에 수신거부 트리거 링크를 수동으로 추가해야 합니다.

이를 위해 이메일 템플릿에서 "수신거부" 또는 **"수신거부하려면 여기를 클릭하세요"** 같은 텍스트를 강조 표시하고, 링크 아이콘을 클릭한 다음, 드롭다운에서 커스텀 수신거부 트리거 링크를 선택하세요. 이렇게 하면 이메일이 어디서 발송되든 수신자가 수신거부할 수 있습니다.

![이메일에 수신거부 링크 삽입 과정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155045850376/original/5srUB4Cs32XWlbEL-Z3X3K9R3WzbrG_bMA.gif?1745936008)

## 자주 묻는 질문

Q: 기본 수신거부 링크 대신 커스텀 수신거부 링크를 만들어야 하는 이유는 무엇인가요?

커스텀 수신거부 링크를 사용하면 브랜드 일관성을 유지하고 수신거부 프로세스를 제어하여 더 나은 사용자 경험을 제공할 수 있습니다.

Q: 수신거부 확인 페이지를 커스터마이징할 수 있나요?

네, 수신거부 랜딩 페이지를 브랜딩에 맞게 디자인하고 사용자가 설정을 관리할 수 있는 메시지나 옵션을 포함시킬 수 있습니다.

Q: "연락처 DND 설정" 액션은 어떻게 작동하나요?

"연락처 DND 설정" 액션은 연락처를 연락 금지(Do Not Disturb)로 표시하여 향후 커뮤니케이션을 받지 않도록 합니다.

Q: 모든 이메일에 수신거부 링크를 포함해야 하나요?

네, 모든 마케팅 이메일에 수신거부 링크를 포함하는 것은 많은 국가에서 법적 요구사항이며, 좋은 이메일 전달성을 유지하는 데 도움이 됩니다.

Q: 누가 내 이메일을 수신거부했는지 추적할 수 있나요?

네, Hyperclass에서는 트리거 링크 분석을 통해 수신거부 링크를 클릭한 연락처를 모니터링할 수 있습니다.
