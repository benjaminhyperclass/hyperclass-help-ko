# 이메일 SMTP 서비스를 위한 새로운 에이전시 레벨 제어 기능

## **개요**

모든 하위 계정의 이메일 서비스를 중앙에서 제어할 수 있는 새로운 에이전시 레벨 설정을 도입했습니다. 이 업데이트를 통해 거버넌스가 개선되고, 관리가 간소화되며, 이메일 설정을 한 곳에서 더 쉽게 찾고 관리할 수 있습니다.

## **새로운 기능**

다음 경로에 새로운 토글이 추가되었습니다:

**Agency View(에이전시 보기) → Settings(설정) → Email Services(이메일 서비스) → Advanced Settings(고급 설정)**

이 토글은 하위 계정 내에서 **"Add Email Service(이메일 서비스 추가)"** 버튼이 표시되고 사용 가능한지를 제어합니다. 이제 에이전시는 이 중앙 설정에서 직접 하위 계정의 이메일 서비스 접근을 활성화하거나 비활성화할 수 있습니다 — 계정별로 개별 관리할 필요가 없습니다.

## **작동 방식**

### **신규 에이전시의 경우**

- 이메일 서비스 제어 토글이 **기본적으로 활성화됨**
- 하위 계정에서 **"Add Services(서비스 추가)"** 버튼이 **숨겨짐**
- 하위 계정의 이메일 서비스 접근이 완전히 에이전시 레벨 설정에 의해 관리됨

### **기존 에이전시의 경우**

- 이메일 서비스 제어 토글이 **기본적으로 비활성화됨**
- 하위 계정에서 **"Add Services(서비스 추가)"** 버튼이 **계속 표시됨**, 현재 워크플로우 유지
- 이미 이메일 서비스를 사용 중인 하위 계정에는 영향 없음

이미 이메일 서비스로 설정된 기존 하위 계정에는 **별도 조치가 필요 없습니다**. 기존 설정은 예상대로 계속 작동합니다.

## **특정 하위 계정의 이메일 서비스 관리**

전체적으로 설정을 적용하는 대신 개별 하위 계정에 대해 "이메일 서비스 추가" 버튼을 활성화하거나 비활성화하고 싶다면, 하위 계정 레벨에서 설정할 수 있습니다:

- **Agency View(에이전시 보기) → Sub-Accounts(하위 계정)**로 이동
- 설정하려는 하위 계정 선택
- **Advanced Settings(고급 설정) → Disable the "Add Email Service" button in the sub-account Email Service Settings(하위 계정 이메일 서비스 설정에서 "이메일 서비스 추가" 버튼 비활성화)**로 이동
- 해당 하위 계정에 대한 선호도에 따라 설정 토글 켜기/끄기

![하위 계정별 이메일 서비스 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155068523675/original/rpCTjGSQFOHGCKZAiIP6q93kcOtVotiJuA.png?1775476100)

이를 통해 필요한 경우 전체 에이전시 레벨 설정을 재정의하여 특정 하위 계정의 접근을 세밀하게 조정할 수 있습니다.

## **주요 장점**

**중앙 집중식 거버넌스**: 계정별로 개별 처리하는 대신 단일 에이전시 레벨 토글로 모든 하위 계정의 이메일 서비스 가용성을 관리할 수 있습니다.

**향상된 검색 가능성**: 모든 이메일 관련 제어 기능이 이제 에이전시 설정의 이메일 서비스 하위에 정리되어, 한 곳에서 설정을 더 쉽게 찾고 관리할 수 있습니다.

**유연한 재정의**: 하위 계정은 여전히 고유한 요구사항을 수용하기 위해 에이전시 레벨 설정을 존중하고, 해당하는 경우 이를 재정의할 수 있습니다.

## **자주 묻는 질문**

**이미 이메일 서비스를 사용 중인 기존 하위 계정에 영향이 있나요?**
아니요. 이미 이메일 서비스가 설정된 기존 하위 계정에는 영향이 없습니다. 이 변경사항은 앞으로 "서비스 추가" 버튼의 표시 여부만을 관리합니다.

**저는 기존 에이전시입니다. 별도로 해야 할 일이 있나요?**
별도 조치가 필요 없습니다. 기존 에이전시의 경우 토글이 기본적으로 비활성화되어 있어 현재 설정이 그대로 유지됩니다.

**하위 계정에서 여전히 자체 이메일 서비스 설정을 관리할 수 있나요?**
네. 하위 계정은 여전히 해당하는 경우 에이전시 레벨 설정을 존중하고 재정의할 수 있습니다.

**이 새로운 설정은 어디서 찾을 수 있나요?**
**Agency View(에이전시 보기) → Settings(설정) → Email Services(이메일 서비스) → Advanced Settings(고급 설정)**로 이동하면 새로운 토글을 찾을 수 있습니다.
