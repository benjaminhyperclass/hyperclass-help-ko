# 하위 계정/로케이션 레벨

## 신규 예약 요청

"신규 예약 요청" 알림은 미배정 캘린더와 그룹 캘린더에서 발송됩니다.

![신규 예약 요청](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48252656817/original/HaUAl_kuzcg5njAp7EfZN7bRcJJFQtvlAA.png?1663872783)

### **그룹 캘린더에서 배정된 사용자의 "신규 예약 요청" 비활성화 방법**

Settings(설정) → Calendar Settings(캘린더 설정) → 캘린더 열기 → 3. Confirmation(확인) 탭 클릭 → 저장

![그룹 캘린더 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48252827089/original/4raiJ9DfsyCZUproWQAFBU8qdbUXaC_Rsg.png?1663941035)

### **미배정 캘린더에서 "신규 예약 요청" 비활성화 방법**

Settings(설정) → Calendar Settings(캘린더 설정) → 캘린더 열기 → 3. Confirmation(확인) 탭 클릭 → 이메일 제거 → 저장

![미배정 캘린더 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48252827085/original/kfcgFMaRBedVPmRfd5hoeEZnQrEu377X3g.png?1663941034)

![미배정 캘린더 설정 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48253316570/original/75I9IfgNAJbDy9tDhOm2aqsOA__d6jBTkA.png?1664264542)

참고사항:

자동 확인 및 예약 알림 이메일 발송이 활성화된 캘린더 설정의 경우, 예약 담당자가 예약 요청 이메일을 받게 됩니다. 개발팀에서 다음 캘린더 업데이트에서 수정 일정을 배포하기 위해 작업 중입니다.
