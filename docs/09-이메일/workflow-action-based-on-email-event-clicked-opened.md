# 이메일 이벤트(클릭/열람) 기반 워크플로우 액션

---

워크플로우에 이메일 이벤트 트리거(클릭/열람)를 추가하는 방법:

**1단계: 워크플로우 내에서 초기 트리거를 선택하고 'email'을 검색하세요**

![워크플로우 트리거 검색](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155029828492/original/bGpvJt4_kQ8_6QVOYFsGdEfwx6kXgkEOLQ.jpg?1721841822)

**2단계: 드롭다운에서 Email Events(이메일 이벤트)를 선택하세요**

![이메일 이벤트 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155029828552/original/N-2L_Wb1zwfqKZdN2qGYmNwgvQkC3hN1Tg.jpg?1721841891)

**3단계: Add filters(필터 추가) 버튼을 클릭하세요**

![필터 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155029828580/original/EpAbf5v6gNamruGMSYT8aGfCODT3AslZeQ.jpg?1721841969)

**4단계: 필터 옵션에서 Email Events(이메일 이벤트)를 선택한 후, Event(이벤트)를 선택하세요**

![이메일 이벤트 및 이벤트 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155029828646/original/t3i6XuxuPNGMGhJSuCpb_-Yuzwmrw0MLZg.jpg?1721842096)

**5단계: 적절한 액션을 선택하세요 - Clicked(클릭됨) 또는 Opened(열람됨)**

![클릭됨 또는 열람됨 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155029828702/original/66704TyO48ntssF31iLO2kcgyVrktX6CWg.jpg?1721842180)

새로운 트리거가 워크플로우에 추가되었습니다!

---

**참고:** 정확한 추적과 강력한 발신자 평판 유지를 위해 발송 도메인에서 Open(열람)과 Click(클릭) 이벤트가 활성화되어 있는지 확인하세요. 이러한 이벤트는 메일박스 제공업체가 전달률을 결정하는 데 사용하는 중요한 참여 데이터를 제공합니다.
