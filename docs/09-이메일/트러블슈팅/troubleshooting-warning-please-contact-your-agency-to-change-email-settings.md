# 경고 해결: 이메일 설정 변경을 위해 에이전시에 문의하세요

![이메일 설정 경고 메시지](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48180856684/original/hnm15p3rph-EYFQ09-2Ix_UBaxPdERt3RQ.png?1642631121)

SMTP 제공업체를 추가하려고 할 때 이 오류 메시지가 나타난다면, 에이전시 SaaS 구성 페이지에서 해당 로케이션의 이메일 리셀링 설정이 활성화되어 있는지 확인해주세요.

# 이메일 재청구를 비활성화하는 방법

1. [https://app.gohighlevel.com/sub-accounts/](https://app.gohighlevel.com/sub-accounts/)로 이동하세요

2. 에이전시 보기에 들어가면, Sub-Accounts(하위 계정) > Search by sub-account(하위 계정으로 검색)를 클릭하세요

![하위 계정 검색 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48280512342/original/R86QkuQrkReIza2aG45_43Rzv42ujoCqJw.png?1675884865)

3. 해당 하위 계정을 찾으면

하위 계정 이름을 클릭하거나 우측 하단의 점 세 개 버튼을 클릭하여 Manage Client(클라이언트 관리)를 선택하세요

![하위 계정 관리 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48280512511/original/yKwzD5ifdBAzF08vPBIyCgt0wdU7aEv3qA.png?1675884922)

4. 페이지 맨 아래로 스크롤하여 Email Resell Settings(이메일 리셀 설정)을 찾고, 강조 표시된 버튼을 토글하여 이메일 재청구를 비활성화하세요.

![이메일 리셀 설정 토글](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48280512843/original/-QsFDNwIZh47kMExMGYCprVoBDWbaDWhjQ.png?1675885077)
