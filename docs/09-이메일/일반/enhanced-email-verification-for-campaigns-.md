# 캠페인 이메일 인증 강화 기능!

기존에는 이메일 캠페인 발송 시 이메일 주소를 인증하지 않았습니다. 이로 인해 신원 사칭이나 평판 손상 문제가 발생할 수 있었습니다.

보안을 강화하고 평판을 보호하기 위해, 이제 이메일 캠페인을 발송하기 전에 발신 이메일 주소 인증이 필수가 되었습니다. 이 조치를 통해 검증된 이메일 주소만 사용할 수 있도록 하여, 누구도 귀하의 이메일 주소를 사칭할 수 없게 됩니다.

다음의 경우에는 이메일 인증이 필요하지 않습니다:

- LC 이메일에서 제공하는 커스텀 도메인
- 설정에 등록된 로케이션 관리자 이메일

## **이메일 인증 방법**

- 이메일 마케팅 섹션으로 이동하여 캠페인을 생성합니다.
- 발신자 이메일 필드에 인증하고자 하는 이메일 주소를 입력합니다.
- "지금 인증하기(Verify Now)" 옵션을 클릭합니다.

![이메일 인증 버튼 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155031199950/original/kaNwiSY0fmopj5MlBf7bYD_wdR0NYuUzDA.png?1723866095)

- 이메일로 받은 OTP(일회용 비밀번호)를 확인하고 6자리 OTP를 입력합니다.

![OTP 입력 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155031199951/original/ckUZs8FxIzpfRLvjOS8JZmn-TUUZvrIAsQ.png?1723866095)

- OTP를 입력하면 이메일 주소가 인증되어 향후 캠페인에 사용할 수 있습니다.
- 또한 설정(Settings) 페이지에서 "이메일 인증하기(Verify an email)"를 클릭하여 인증할 수도 있습니다.

![설정 페이지 이메일 인증 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155031199952/original/8plDBApPogd_CPPjESiBP3Bmm06Kyd2WRQ.jpeg?1723866095)
