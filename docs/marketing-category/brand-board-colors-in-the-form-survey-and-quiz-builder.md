# 폼, 설문, 퀴즈 빌더에서 브랜드 보드 색상 사용하기

폼(Form), 설문(Survey), 퀴즈(Quiz) 빌더에서 저장된 브랜드 보드 색상을 바로 사용할 수 있습니다. 이 가이드는 색상 선택기에서 브랜드 색상을 어떻게 활용하여 색상 코드를 수동으로 복사하지 않고도 모든 자산을 브랜드에 맞게 유지하는 방법을 설명합니다.

## 폼, 설문, 퀴즈 빌더의 브랜드 보드 색상이란?

브랜드 보드 색상을 사용하면 빌더 요소(Element)의 스타일을 설정할 때 색상 선택기에서 저장된 브랜드 색상을 바로 사용할 수 있습니다. 이를 통해 HEX 코드를 수동으로 복사 붙여넣기 하지 않고도 버튼, 입력 필드, 라벨 및 기타 디자인 요소를 브랜드와 일치하게 유지할 수 있습니다.

브랜드 색상은 저장된 브랜드 보드 설정에 따라 표시됩니다. 색상이 보이지 않는다면 다음을 확인해 주세요:

- 브랜드 보드에 색상이 설정되어 있는지 확인
- 올바른 기본 브랜드 보드가 설정되어 있는지 확인

---

## 브랜드 보드 색상의 주요 장점

- **빠른 스타일링**: 매번 HEX 코드를 수동으로 입력하지 않고도 저장된 브랜드 색상을 바로 적용할 수 있습니다.

- **더 나은 브랜드 일관성**: 버튼, 필드, 라벨 및 기타 시각적 요소를 승인된 팔레트와 일치하게 유지할 수 있습니다.

- **간편한 협업**: 폼, 설문, 퀴즈 전반에서 재사용 가능한 색상 세트를 공유할 수 있습니다.

- **쉬운 색상 관리**: 선택기에서 브랜드 설정(Brand Settings)을 열어 재사용 가능한 커스텀 색상을 추가하거나 관리할 수 있습니다.

---

## 빌더에서 브랜드 보드 색상 작동 방식

이 기능은 저장된 브랜드 팔레트를 빌더에 연결하여 작업 흐름을 방해하지 않고도 승인된 색상을 적용할 수 있게 해줍니다. 또한 폼과 다른 자산 간에 색상을 수동으로 맞춰야 하는 필요성을 줄여줍니다.

빌더에서 색상 필드를 열면:

- 색상 선택기에 브랜드 색상(Brand Colors), 전역 커스텀 색상(Global Custom Colors), 기본 색상(Default Colors)을 포함한 사용 가능한 색상 섹션이 표시됩니다.

- 저장된 브랜드 색상을 선택하면 해당 색상이 선택한 요소에 적용되고, 빌더가 자동으로 미리보기를 업데이트합니다.

- 새로운 커스텀 브랜드 색상을 추가하려면 색상 선택기에서 관리(Manage)를 클릭하여 브랜드 설정을 여세요.

참고: 기본 브랜드 보드를 변경해도 기존 폼, 설문, 퀴즈의 스타일이 자동으로 변경되지는 않습니다. 기존 자산은 수동으로 업데이트할 때까지 현재 스타일을 유지합니다.

## 자주 묻는 질문

**Q: 색상 선택기에 브랜드 색상이 표시되지 않는 이유는 무엇인가요?**
브랜드 보드에 저장된 색상이 있는지, 올바른 기본 브랜드 보드가 설정되어 있는지 확인해 주세요.

**Q: 브랜드 색상과 전역 커스텀 색상의 차이점은 무엇인가요?**
브랜드 색상은 브랜드 보드의 승인된 팔레트에서 가져옵니다. 전역 커스텀 색상은 브랜드 설정에서 관리할 수 있고 지원되는 색상 선택기 전반에서 사용할 수 있는 재사용 가능한 저장된 색상입니다.

**Q: 기본 브랜드 보드를 변경하면 기존 폼이 자동으로 업데이트되나요?**
아니요. 기존 폼, 설문, 퀴즈는 수동으로 업데이트할 때까지 현재 스타일을 유지합니다.
