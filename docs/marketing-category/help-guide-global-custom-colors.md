# 글로벌 커스텀 컬러 완벽 가이드

모든 디자인에서 브랜드 일관성을 유지하는 것은 시간이 많이 걸리는 작업입니다. **글로벌 커스텀 컬러(Global Custom Colors)**를 사용하면 브랜드 색상을 한 번만 정의하고 어디서든 사용할 수 있습니다. 색상을 업데이트하면 모든 자료에 즉시 반영되어 시간과 노력을 절약할 수 있어요.

## 글로벌 커스텀 컬러란?

글로벌 커스텀 컬러는 **글로벌 설정(Global Settings)**의 **브랜드 보드(Brand Boards)** 아래에 저장되는 브랜드 색상입니다.

- **양방향 동기화**가 지원됩니다. 색상을 업데이트하면 해당 색상이 사용된 모든 곳(이메일, 퍼널, 폼, 페이지 등)에서 자동으로 업데이트됩니다.

- 기본적으로 플랫폼 전체의 모든 **색상 선택기(Color Picker)**에 표시됩니다.

## 디자인 킷 컬러와 어떻게 다른가요?

- **디자인 킷 컬러**: 디자인 빌더에서 빠르게 접근할 수 있는 색상입니다. **동기화되지 않으므로** 업데이트해도 기존 디자인은 변경되지 않습니다.

- **글로벌 커스텀 컬러**: 브랜드 레벨에서 저장되며 **양방향 동기화**가 됩니다. 한 번 업데이트하면 해당 색상이 사용된 모든 곳에 변경사항이 적용됩니다.

**요약**: 디자인 킷 = 정적인 빠른 접근. 글로벌 커스텀 컬러 = 동적이고 동기화되는 브랜드 일관성.

## 글로벌 커스텀 컬러의 장점

- **자료 전반의 일관성** – 모든 디자인에서 동일한 룩앤필을 유지합니다.

- **시간 절약** – 색상을 한 번 업데이트하면 모든 곳에 반영됩니다.

- **쉬운 리브랜딩** – 에이전시는 퍼널이나 이메일을 복사/공유하고, 글로벌 색상만 업데이트하면 새 로케이션을 빠르게 리브랜딩할 수 있습니다.

- **언제든 접근 가능** – 글로벌 색상은 모든 색상 선택기에 표시됩니다.

- **빠른 가져오기** – 기존 커스텀 값을 가져와서 더 빠르게 설정할 수 있습니다.

## 글로벌 커스텀 컬러 사용법

### 1단계: 글로벌 설정 열기

- `Marketing(마케팅) → Brand Boards(브랜드 보드) → Global Settings(글로벌 설정)`로 이동합니다.

![글로벌 설정 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055017307/original/NXta2BG8hoxRjGHcP3Jfh4ZBZTb0RAxc0g.png?1759329109)

![브랜드 보드 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055017317/original/9Rq7vRFfIuHRkIFC1LvDMTimpMJFyrV8QA.png?1759329123)

### 2단계: 색상 추가 또는 관리

- **새 색상 추가(Add New Color)** → 이름과 hex 코드를 입력합니다.

![색상 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055017349/original/BhA0w3Kh7DT3C1PrwipPppYX7PA-NkeQzg.png?1759329137)

- **편집(Edit)** → 언제든지 색상 세부사항을 업데이트할 수 있습니다.

![색상 편집](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055017450/original/e8KZj4eYKM7BALBMEBYXDgJ_fODWX-ee-Q.png?1759329206)

- **삭제(Delete)** → 사용하지 않는 색상을 제거합니다.

### 3단계: 색상 가져오기

- New를 클릭한 후 **Import(가져오기)**를 선택합니다.

- 가져올 색상을 선택하고 진행합니다.

![색상 가져오기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055017366/original/3cnCwkqRRFbMRrVqki6pnW92G2r1LiwqJA.png?1759329149)

### 4단계: 빌더에서 사용하기

- 이메일, 퍼널, 폼, 또는 페이지에서 **색상 선택기(Color Picker)**를 엽니다.

- 글로벌 커스텀 컬러가 자동으로 표시됩니다.

- 선택기 안의 **관리(Manage)**를 사용하여 즉석에서 색상을 추가/편집할 수 있습니다.

![색상 선택기 사용](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055017432/original/scU-dOhADw-QxVKZg0H-nrMmcdu50lULhA.png?1759329188)

## 자주 묻는 질문

**Q: 글로벌 컬러를 변경하면 기존 자료도 업데이트되나요?**
A: 네, 그렇습니다. 글로벌 커스텀 컬러는 양방향 동기화됩니다. 변경사항이 해당 색상이 적용된 모든 디자인에 즉시 반영됩니다.

**Q: 색상 선택기에서 글로벌 컬러가 보이지 않는데 왜 그런가요?**
A: 글로벌 설정에 최소 하나의 색상을 추가해야 해당 섹션이 나타납니다.

**Q: 기존의 커스텀 값 기반 색상을 가져올 수 있나요?**
A: 네, 가능합니다. **커스텀 값(Custom Values)** 섹션에서 글로벌 설정으로 가져올 수 있습니다. 이는 복사본을 만드는 것이며, 커스텀 값 기반 색상을 완전히 이동하는 것은 아닙니다.

**Q: 디자인 킷 컬러와 어떻게 다른가요?**
A: 디자인 킷 컬러는 빠른 접근용으로만 사용되며 동기화되지 않습니다. 글로벌 커스텀 컬러는 모든 자료에 걸쳐 동기화됩니다.

**Q: 에이전시에서 클라이언트 온보딩에 활용할 수 있나요?**
A: 물론입니다! 에이전시는 퍼널/이메일을 복사하고 글로벌 컬러만 업데이트하면 새 클라이언트를 위해 즉시 리브랜딩할 수 있습니다.
