# 수료증 만료 알림 이메일 빈도 설정하기

**개요:** 최신 업데이트를 통해 이제 만료일 며칠 전에 알림 이메일을 받을지 직접 설정할 수 있습니다. 이 기능은 기존의 만료 2일 전, 7일 전 기본 간격을 대체하여, 중요한 알림을 언제 받을지 더 세밀하게 조절할 수 있게 해줍니다.

**주요 장점:**

- 개인 취향에 맞게 알림 간격을 맞춤 설정할 수 있습니다.

- 일정에 맞는 적시 알림을 받을 수 있습니다.

- 만료 알림을 직접 제어하여 더 나은 사용자 경험을 제공합니다.

**맞춤 만료 알림 간격 설정 방법:**

- **계정 설정 접근:**

계정에 로그인합니다.

- 멤버십 아래 수료증(Certificates) 섹션으로 이동합니다.

- **수료증 템플릿 알림 설정 편집하기**

템플릿에 만료일을 추가하면 해당 수료증을 받는 모든 사람이 동일한 날짜에 만료됩니다.

- 만료 옵션을 추가하려는 수료증 템플릿을 찾습니다.

- 점 3개 버튼 > Set Expiry(만료 설정)를 클릭하고 관련 날짜를 선택하여 설정을 변경합니다.

- 선택한 날짜에 대해 2개의 알림 이메일을 보낼지 1개를 보낼지 선택하고 켜기/끄기를 토글할 수 있습니다.

- 각 알림 이메일에 해당하는 모든 관련 일수를 추가합니다. 모든 이메일은 추가된 일수 후에 발송됩니다.

- 개별 수준 맞춤 설정을 위해 발급된 수료증(Issued certificate) 탭 아래의 모든 수료증에도 동일하게 적용할 수 있습니다.

참고: "즉시"를 선택하면 만료일 당일에만 수료증 만료 이메일이 발송됩니다.

- 이전에 예약된 모든 알림 이메일은 사용자가 수동으로 조정하지 않는 한 2일 전, 7일 전의 기본 간격을 계속 따릅니다.
