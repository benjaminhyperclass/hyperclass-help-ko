# **수료증 내보내기란?**

수료증 내보내기는 발급된 수료증 기록을 **CSV 파일**로 대량 다운로드할 수 있는 기능이에요. 수료자 이름, 이메일, 발급 날짜, 수료증 ID 등의 메타데이터가 모두 포함됩니다. 교육 이수 관리, 강의 수료증 관리, 감사 업무를 담당하는 팀에게 이상적이에요.

## **발급된 수료증 내보내기의 핵심 장점**

수료증 데이터를 CSV 형식으로 내보내서 쉽게 접근하고 관리할 수 있어요. 이런 장점들이 있습니다:

- 최대 500개 수료증을 앱에서 즉시 다운로드
- 500개 초과 시 자동으로 이메일로 CSV 수신
- 커스텀 날짜 범위로 필터링하여 결과 범위 좁히기
- 컴플라이언스, 분석, 연동을 위한 내보내기 활용
- 범용적으로 사용되고 조작하기 쉬운 CSV 형식
- 보고서 작성 간소화 및 외부 LMS 도구와 연동

---

## **수료증을 CSV 파일로 내보내는 방법**

수료증 데이터 내보내기는 간단하고 유연해요. 방법은 다음과 같습니다:

- 수료증(Certificates) 앱의 **"발급된 수료증(Issued Certificates)"** 섹션으로 이동하세요.
- 우측 상단에 있는 **CSV 내보내기(Export CSV)** 버튼을 클릭하세요.
- **날짜 범위 필터**가 나타나면 — 내보낼 수료증의 원하는 기간을 선택하세요.
- **내보내기 확인(Confirm Export)**을 클릭하여 시작하세요.

- 내보내기에 **500개 미만의 수료증**이 포함된 경우, CSV 파일이 즉시 기기로 다운로드됩니다.
- 내보내기에 **500개 초과의 수료증**이 포함된 경우, 파일이 준비되면 **로그인 이메일 주소**로 전송된다는 메시지가 표시됩니다.

## **내보내기 로직 및 전달 규칙**

수료증 내보내기 작동 방식을 이해하면 혼란을 피하고 대용량 데이터 세트를 다룰 때 원활한 운영을 보장할 수 있어요.

- **500개 미만 수료증**: CSV가 브라우저에서 즉시 다운로드됩니다.
- **500개 초과 수료증**: CSV가 로그인한 사용자에게 이메일로 전송됩니다.
- **이메일 제목줄**: "수료증 내보내기가 준비되었습니다(Your Certificate Export is Ready)"
- **링크 만료**: 이메일의 다운로드 링크는 **24시간**만 유효합니다.
- **날짜 범위 필수**: 내보내기 범위를 정의하기 위해 사용자는 내보내기 전에 반드시 날짜 범위를 선택해야 합니다.

## **자주 묻는 질문**

**Q: 날짜로 필터링하지 않고 수료증을 내보낼 수 있나요?**

아니요. 데이터셋을 제한하고 성능을 최적화하기 위해 날짜 범위는 필수입니다.

**Q: CSV 파일은 어디로 전송되나요?**

내보내기가 500개를 초과하면, CSV가 귀하의 Hyperclass 계정과 연결된 이메일 주소로 전송됩니다.

**Q: CSV 다운로드 링크는 얼마나 오래 유효한가요?**

이메일로 발송된 다운로드 링크는 **24시간** 동안 유효합니다.

**Q: CSV 내보내기에는 어떤 데이터가 포함되나요?**

내보내기에는 다음이 포함됩니다: 전체 이름, 이메일 주소, 수료증 ID, 발급 날짜, 그리고 선택적으로 강의명이나 태그(설정된 경우).

**Q: 이전 내보내기를 다시 다운로드할 수 있나요?**

아니요. 원본 링크가 만료되면 같은 날짜 범위를 사용하여 다시 내보내기해야 합니다.
