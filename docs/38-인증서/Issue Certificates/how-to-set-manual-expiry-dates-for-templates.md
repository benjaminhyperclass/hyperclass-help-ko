# 템플릿 수동 만료일 설정하는 방법

**새로운 업데이트로 이제 템플릿 만료일을 직접 선택할 수 있습니다. 다음 간단한 단계를 따라하세요:**

### ⌛️ 특정 만료일 설정하기:

- 대시보드에서 Certificates(인증서) 메뉴로 이동하세요.

- 편집하고자 하는 템플릿을 선택하세요.

- 세 개의 점 아이콘을 클릭하고 Set a Date(날짜 설정)를 선택하세요.

- 캘린더에서 원하는 만료일을 선택하세요. 미리보기에서 선택된 만료일을 확인할 수 있습니다.

- Set Expiry(만료일 설정)를 클릭해서 확인하세요.

![특정 만료일을 설정하는 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155045964926/original/FJTu06rYNz6rAsc0QGyjIhhrnBrpjs3-1w.png?1746098298)

### ∞ 템플릿을 만료되지 않게 설정하기 또는 만료일 제거하기:

- 대시보드에서 Certificates(인증서) 메뉴로 이동하세요.

- 편집하고자 하는 템플릿을 선택하세요.

- 세 개의 점 아이콘을 클릭하고 Never(만료 안함)를 선택하세요.

- 미리보기 메시지에서 템플릿이 만료되지 않을 것이라는 확인을 볼 수 있습니다.

- Set Expiry(만료일 설정)를 클릭해서 변경사항을 적용하세요.
