# 로케이션 간 수료증 템플릿 복제하기

## 개요

이제 한 로케이션(Location)에서 다른 로케이션으로 수료증(Certificate) 템플릿을 복제할 수 있어서, 디자인을 더 쉽게 재활용하고 재사용할 수 있습니다. 이 기능을 사용하면 최대 3개 로케이션까지 동시에 복제할 수 있으며, 관련된 모든 요소가 완벽하게 전송됩니다.

### 주요 특징:

✔ 여러 로케이션으로 수료증 템플릿 복제 가능 (한 번에 최대 3개)  
✔ 템플릿과 관련된 모든 미디어 파일과 이미지 자동 이전  
✔ 모든 템플릿 설정과 버전 히스토리 유지  
✔ 동일한 로케이션 내에서도 복제 가능  

## 수료증 템플릿 복제하는 방법

- **메뉴 이동:**  
  `Memberships(멤버십) → Certificates(수료증)`으로 이동하세요.

- **복제 시작:**  
  복제하려는 템플릿 옆의 점 세 개(⋮) 버튼을 클릭하세요.  
  메뉴에서 `Clone(복제)`를 선택하세요.

- **복제 설정:**  
  필요한 경우 템플릿 이름을 수정하세요.  
  템플릿을 복제할 로케이션을 선택하세요(최대 3개).

- **복제 완료:**  
  선택사항을 확인하면, 시스템이 모든 미디어 파일, 설정, 버전 히스토리와 함께 템플릿을 복제합니다.

**참고:**  
필요한 경우 동일한 로케이션 내에서도 수료증 템플릿을 복제할 수 있습니다.
