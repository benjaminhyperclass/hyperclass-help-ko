# 워크플로우에서 배지 자동 발급하기 ("수료증 발급" 액션 활용)

## ⭐️ 개요

배지는 성취, 진도, 참여를 시각적으로 인정해주는 가벼운 방식입니다. 전용 "배지 발급" 액션을 개발 중이지만, 기존의 "수료증 발급(Issue Certificate)" 워크플로우 액션을 사용하여 지금 당장 배지를 발급할 수 있습니다.

그렇습니다 — 배지는 수료증과 동일한 액션으로 발급됩니다!

## ⚙️ 작동 원리

지원되는 워크플로우 트리거와 기존 "수료증 발급" 액션을 결합하여 고객에게 배지를 자동으로 발송할 수 있습니다.

✅ "수료증 발급" 액션 사용 시 수료증 대신 배지 템플릿을 선택하기만 하면 됩니다.

## 주요 트리거 + 활용 사례
| 워크플로우 트리거 | 배지 활용 사례 (예시) |
|------------------|---------------------|
| 레슨 완료 | "1강 완료" 배지 |
| 카테고리 완료 | "모듈 마스터" 배지 |
| 상품 완료 | "공인 프로" 배지 |
| 퀴즈 제출 | "퀴즈 에이스" 배지 |
| 설문 제출 | "최고 응답자" 배지 |
| 폼 제출 | "빠른 행동가" 배지 |
| 할 일 완료 | "마일스톤 달성" 배지 |

이러한 트리거는 워크플로우 트리거 메뉴의 "강의(Courses)", "연락처(Contact)", "폼 활동(Form Activities)" 섹션에서 찾을 수 있습니다.

## 설정 방법

1. **자동화(Automation)** > **워크플로우(Workflows)**로 이동

2. **처음부터 만들기(Create from Scratch)** 또는 기존 워크플로우 열기

3. 위 목록에서 트리거 추가

4. 액션 추가 → **수료증 발급(Issue Certificate)** 선택

5. 수료증 템플릿 대신 **배지 템플릿** 선택

6. 저장 & 발행 ✅

참고: 배지는 완전히 시각적이고 가벼워서 작은 인정에 완벽합니다. 다운로드 가능한 이미지로 고객에게 전송됩니다.

## ⏳ 곧 출시 예정

더욱 명확하고 사용하기 쉽도록 전용 "배지 발급" 액션을 도입할 예정입니다. 그때까지는 "수료증 발급" 액션이 배지와 수료증을 모두 원활하게 지원합니다.

## 배지 빌더 이동 방법

1. **멤버십(Memberships)**으로 이동
2. **자격증명(Credentials)** 탭 선택
3. 우측 상단의 **만들기(Create)** 클릭
4. 드롭다운에서 **배지 만들기(Create Badge)** 선택
