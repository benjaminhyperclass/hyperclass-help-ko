# 수동 수료증 이메일 템플릿

이 문서에서는 Hyperclass에서 수동으로 수료증을 발송할 때 이메일 템플릿을 선택하고 제목을 맞춤 설정하는 방법을 설명합니다. 이 기능을 통해 수료증 발급 시 더욱 세밀한 제어와 개인화 옵션을 제공할 수 있습니다.

## 수동 수료증 이메일 템플릿 기능이란?

수동 수료증 이메일 템플릿 기능을 사용하면 수료증을 수동으로 발송할 때 기존 이메일 템플릿 중에서 선택할 수 있습니다. 또한 발송 전에 이메일 제목을 맞춤 설정할 수도 있습니다. 이를 통해 이벤트, 대상, 브랜딩 요구사항에 맞는 커뮤니케이션을 보장할 수 있습니다.

이 과정에서 사용자가 템플릿을 선택하지 않으면, 시스템은 클라이언트 포털(Client Portal) 이메일 설정에서 구성된 템플릿을 기본값으로 사용합니다.

## 수동 수료증 이메일 템플릿의 주요 장점

수동 수료증 이메일에 특정 템플릿을 선택하면 명확성, 브랜딩, 참여도가 향상됩니다. 이 섹션에서는 이 기능 사용의 주요 이점을 소개합니다.

- 커스텀 템플릿으로 수료증 발급을 개인화할 수 있습니다.

- 콘텐츠나 이벤트에 맞게 제목을 맞춤 설정할 수 있습니다.

- 모든 커뮤니케이션에서 일관된 브랜딩을 유지할 수 있습니다.

- 템플릿을 재사용하여 시간을 절약하고 수동 편집을 줄일 수 있습니다.

- 템플릿을 선택하지 않았을 때 기본 템플릿으로 실수를 방지할 수 있습니다.

## 선택한 이메일 템플릿으로 수동 수료증 발송하는 방법

수료증을 수동으로 발송할 때 미리 정의된 이메일 템플릿을 선택하면 메시지가 적절하고 전문적으로 포맷됩니다. 다음 단계에 따라 이 기능을 사용하세요:

- Hyperclass 계정 내 수료증(Certificates) 섹션으로 이동합니다.

- 수료증을 발송할 연락처를 선택합니다.

- 수료증 발송(Send Certificate)을 클릭합니다.

- 이메일 모달에서 이메일 템플릿(Email Template) 드롭다운을 찾습니다.

- 목록에서 원하는 이메일 템플릿을 선택합니다.

- 필요에 따라 이메일 제목(Email Subject) 필드를 수정합니다.

- 미리보기를 확인한 후 발송(Send)을 클릭합니다.

## 템플릿 미선택 시 기본 동작

템플릿을 수동으로 선택하지 않았을 때도 적절한 커뮤니케이션이 유지되도록 하는 기본 동작을 이해하는 것이 중요합니다.

수동 수료증 발송 과정에서 템플릿을 선택하지 않은 경우:

- 시스템이 자동으로 기본 이메일 템플릿을 적용합니다.

- 이 기본 템플릿은 클라이언트 포털 설정(Client Portal Settings) > 이메일 설정(Email Settings) 섹션에서 구성됩니다.

## 자주 묻는 질문

**Q: 수료증 이메일 전용 커스텀 템플릿을 만들 수 있나요?**
A: 네. 이메일 템플릿(Email Templates) 설정으로 이동하여 새 템플릿을 만드세요. 이렇게 만든 템플릿은 수동 발송 과정에서 선택할 수 있습니다.

**Q: 선택한 템플릿에 따라 제목이 자동으로 업데이트되나요?**
A: 아니요. 제목은 편집 가능한 상태로 유지되며, 필요시 수동으로 업데이트해야 합니다.

**Q: 수료증을 발송하기 전에 템플릿을 미리보기할 수 있나요?**
A: 네. 템플릿을 선택하면 발송 모달 내에서 미리보기가 표시됩니다.

**Q: 템플릿을 선택하지 않거나 제목을 편집하지 않고 수료증을 발송하면 어떻게 되나요?**
A: 클라이언트 포털 설정의 기본 템플릿과 제목이 사용됩니다.

**Q: 수료증 유형별로 다른 기본 템플릿을 설정할 수 있나요?**
A: 현재는 지원되지 않습니다. 계정당 클라이언트 포털 설정을 통해 하나의 기본 템플릿만 구성할 수 있습니다.
