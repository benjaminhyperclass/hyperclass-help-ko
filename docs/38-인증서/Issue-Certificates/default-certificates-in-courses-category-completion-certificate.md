# 강의의 기본 수료증 및 카테고리 완료 수료증

## 개요

최신 업데이트로 수료증 설정이 더욱 간편해졌습니다. 새로 생성되는 모든 강의에는 자동으로 강의 완료 수료증이 기본 포함되어, 학습자가 강의를 완료하면 자격증명을 받을 수 있습니다. 또한 새로운 카테고리 완료 수료증 기능을 통해 강의의 특정 섹션을 완료했을 때도 수료증을 발급할 수 있습니다.

## 주요 장점

✅ 자동화된 강의 수료증 – 모든 새 강의에 기본 완료 수료증이 포함되어 수동 설정이 줄어듭니다.
✅ 유연한 수료증 관리 – 필요하지 않은 경우 기본 강의 완료 수료증을 삭제하거나 초안으로 변경할 수 있습니다.
✅ 원활한 통합 – 강의 빌더에서 기본 및 커스텀 수료증을 직접 연결할 수 있습니다.
✅ 카테고리 인정 – 강의 내 전체 카테고리를 완료했을 때 수료증을 발급합니다.

## 기본 강의 완료 수료증 사용 방법

### 기본 수료증 액세스

- 계정에 로그인합니다.
- Memberships(멤버십) 아래의 Courses(강의)로 이동하여 새 상품을 생성합니다.
- 기본 강의 완료 수료증이 자동으로 포함됩니다.

### 기본 수료증 관리

- **수료증 템플릿 연결하기:**
  - Course Completion Certificate(강의 완료 수료증) 섹션 아래의 설정 아이콘(점 세 개)을 클릭합니다.
  - Edit Certificate(수료증 편집) 또는 Choose Certificate(수료증 선택)을 선택합니다.
  - 선호하는 템플릿을 강의에 연결합니다.

- **기본 수료증 삭제하거나 초안으로 변경하기:**
  - 설정 아이콘(점 세 개)을 클릭하고 Delete(삭제)를 선택하거나 Published(발행됨) 아이콘을 클릭하고 Draft(초안)를 선택합니다.

## 카테고리 완료 수료증 사용 방법

### 카테고리 완료 수료증 활성화

- 강의로 이동합니다.
- 수료증을 연결할 카테고리를 선택합니다.
- 더하기 아이콘을 클릭하여 카테고리에 수료증을 연결합니다.

### 작동 방식

- 카테고리 완료 수료증은 카테고리 미리보기의 마지막 레슨으로 표시됩니다.
- 카테고리 내 모든 필수 레슨이 완료되면 수료증이 자동으로 발급됩니다.
- 카테고리 수료증은 이동할 수 없으며 항상 카테고리의 끝에 위치합니다.
