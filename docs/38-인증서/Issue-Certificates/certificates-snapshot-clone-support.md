# 수료증: 스냅샷 및 복제 지원

# **스냅샷의 수료증이란?**

스냅샷의 수료증은 스냅샷 안에 수료증 템플릿을 첨부할 수 있는 기능입니다. 이 개선 사항은 수료증 템플릿과 관련된 모든 미디어 파일이 스냅샷 로드 시 함께 이전되도록 보장합니다. 또한 스냅샷에 포함된 수료증에 IP 보호가 적용되어 추가 보안 계층을 제공합니다.

## **스냅샷 수료증의 주요 장점**

수료증 템플릿 이전이 그 어느 때보다 쉬워졌습니다. 이 기능을 사용할 때의 주요 장점은 다음과 같습니다:

- **원활한 이전**: 스냅샷을 사용하여 계정 간에 수료증 템플릿을 손쉽게 이동할 수 있습니다.

- **자동 미디어 파일 전송**: 템플릿과 연결된 모든 미디어 파일이 함께 이전됩니다.

- **시간 효율성**: 데이터 이전 시 수료증을 수동으로 다시 만들 필요가 없습니다.

- **일관된 구성**: 이전 중에 모든 수료증 설정이 그대로 유지됩니다.

- **향상된 보안**: 스냅샷에 포함된 수료증에 IP 보호가 적용됩니다.

## **스냅샷에서 수료증 사용법**

다음 단계를 따라 스냅샷에 수료증 템플릿을 첨부하세요:

- **스냅샷 섹션으로 이동**: Settings(설정) 메뉴로 가서 **Snapshots(스냅샷)**을 선택합니다.

- **새 스냅샷 생성**:

![새 스냅샷 생성](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155043454805/original/7DDtIJQHQKW5sm9OJ7ymMwapDpMLUWCzZQ.png?1742243674)

**Create Snapshot(스냅샷 생성)**을 클릭합니다.

- 스냅샷의 이름과 설명을 입력합니다.

- **수료증 템플릿 첨부**:

![수료증 템플릿 첨부](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155043454823/original/L79flSXtaZ68oiffLVMacdG-dAiF53Tu6g.png?1742243739)

포함하려는 수료증 템플릿을 선택합니다.

- 필요한 모든 미디어 파일이 자동으로 포함되는지 확인합니다.

- **스냅샷 저장 및 내보내기**:

완료되면 스냅샷을 저장합니다.

- 필요에 따라 다른 위치로 내보냅니다.

- **새 위치에 스냅샷 로드**:

원하는 계정이나 위치에 스냅샷을 가져옵니다.

- 모든 수료증과 미디어 파일이 성공적으로 이전되었는지 확인합니다.

## **수료증 템플릿 복제**

### 수료증 템플릿 복제란?

수료증 템플릿 복제를 통해 사용자는 한 위치에서 다른 위치로 수료증 템플릿을 복사할 수 있습니다. 이 기능은 최대 3개 위치에 동시 복제를 지원하여 유연성과 효율성을 보장합니다.

## **수료증 템플릿 복제의 주요 장점**

- **다중 위치 지원**: 최대 3개 위치에 동시에 템플릿을 복제할 수 있습니다.
- **미디어 파일 유지**: 템플릿에 연결된 모든 미디어 파일이 자동으로 전송됩니다.
- **설정 보존**: 수료증의 기존 설정과 구성을 모두 유지합니다.
- **버전 이력 보존**: 복제 후에도 템플릿의 전체 버전 이력이 그대로 유지됩니다.

## **수료증 템플릿 복제 방법**

- **수료증 템플릿 접근**:

대시보드의 Certificates(수료증) 섹션으로 이동합니다.

- **수료증 템플릿 선택**:

복제하려는 템플릿을 선택합니다.

- **복제 시작**:

Clone Template(템플릿 복제)을 클릭합니다.

- 복제할 대상 위치를 최대 3개까지 선택합니다.

- **확인 및 복제**:

복제할 미디어 파일과 구성을 검토합니다.

- Clone(복제)을 클릭하여 프로세스를 완료합니다.

- **복제된 템플릿 확인**:

대상 위치를 확인하여 템플릿이 성공적으로 복제되었는지 확인합니다.

## **자주 묻는 질문**

**Q: 스냅샷에 여러 수료증 템플릿이 포함되어 있으면 어떻게 되나요?**

A: 모든 수료증 템플릿과 해당 미디어 파일이 스냅샷 로드 시 함께 이전됩니다.

**Q: 스냅샷에서 수료증 템플릿을 이전할 때 미디어 파일을 제외할 수 있나요?**

A: 아니요, 완전성을 보장하기 위해 연결된 모든 미디어 파일이 자동으로 이전에 포함됩니다.

**Q: 한 번에 복제할 수 있는 수료증 템플릿 수에 제한이 있나요?**

A: 한 번에 최대 3개 위치에 복제할 수 있습니다. 더 많은 위치에 복제해야 하는 경우 프로세스를 반복하세요.

**Q: 수료증을 복제하거나 이전할 때 IP 보호 설정이 유지되나요?**

A: 예, 스냅샷에 포함된 수료증과 복제된 템플릿에서 IP 보호가 그대로 유지됩니다.
