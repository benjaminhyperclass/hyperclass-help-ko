# 캘린더 위젯 로고 추가/변경/삭제하는 방법

캘린더 위젯 로고를 업데이트하여 고객 회사의 브랜딩을 반영할 수 있어요. 캘린더 예약 위젯을 더욱 개인화하고 맞춤 설정할 수 있는 기능입니다.

## 캘린더 로고를 사용하는 이유와 누구에게 유용한가요?

캘린더 위젯 로고는 고객 회사의 브랜딩을 나타내도록 맞춤 설정할 수 있는 중요한 기능이에요. 기본 로고를 고객의 브랜드 정체성을 반영하는 맞춤 로고로 교체하면, 캘린더 위젯이 더욱 개인화되고 사용자에게 더 전문적인 모습을 제공해요.

캘린더 예약 위젯의 맞춤 설정은 고객의 브랜드 정체성과 이미지를 강화하는 데 도움이 돼요. 고객이 브랜드의 시각적 일관성을 유지할 수 있게 해주어, 사용자가 브랜드를 더 쉽게 인식하고 기억할 수 있게 해줘요.

로고 맞춤 설정 기능은 사용하기 쉽고, 필요한 사양을 충족하는 고해상도 이미지 파일을 업로드하여 설정할 수 있어요. 업로드한 후에는 맞춤 로고가 캘린더 예약 위젯의 기본 로고를 대체하여, 고객의 브랜드를 나타내는 전문적이고 개인화된 모습을 제공해요.

캘린더 예약 위젯 로고 맞춤 설정은 예약이나 미팅을 위해 캘린더 예약 시스템을 사용하는 모든 비즈니스나 개인에게 유용해요. 컨설턴트, 프리랜서, 서비스 제공업체와 같은 개인이나, 의료 서비스 제공자, 미용실, 스파, 법률 사무소 등 다양한 규모와 유형의 서비스 기반 업종의 비즈니스가 포함돼요. 로고를 맞춤 설정하면 비즈니스가 브랜딩 노력을 개선하고 고객에게 더 전문적인 모습을 만들 수 있어요. 또한 경쟁업체와 차별화하고 브랜드 인지도를 높이는 데도 도움이 돼요.

## 캘린더 예약 위젯에 로고를 추가하는 방법

기존 캘린더를 편집하거나 새로 만들어서 로고를 추가할 수 있어요. `Settings(설정) > Calendars(캘린더) > 편집할 캘린더 앞의 세 개 점 > Edit Calendar(캘린더 편집)`로 이동하세요.

![캘린더 편집 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48291041177/original/QrP9t6oFJ20uTa0BIEnhWoMFHa-V-oltyg.png?1680620919)

Team and Event Setup(팀 및 이벤트 설정) 탭에서 아래로 스크롤하여 로고를 추가하세요:

![로고 추가 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48291157004/original/f4KPx8Zqlu-BCfljLn9KQQ01f6KsGc5ecw.gif?1680680846)

위젯에서 로고의 모양을 **Square(정사각형) 또는 Circle(원형)** 중에서 선택할 수 있어요. 또한 이 로고를 어떤 유형의 위젯에 사용할지도 선택할 수 있어요. **Classic 또는 Neo** 중에서 선택하세요.

![로고 설정 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48291158038/original/wKPiwq17ScG3eLuElJXMBxm1S3WqvIk78w.png?1680681174)

참고사항:

로고의 지원 가능한 크기는 180x180px이고, 파일 크기는 2.5MB를 초과하면 안 돼요. jpeg, gif 또는 png 파일만 사용할 수 있어요.

로고가 추가된 후에는 제거하거나 변경할 수 있어요.

![로고 변경/제거 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48291158176/original/CM2ek-w-ILuNJOiPVgayyQ12BPhEcjiIHw.png?1680681231)
