# 이벤트 시작 날짜를 활용한 웨비나 캠페인 구축 방법

이벤트 시작 날짜 기능을 사용하여 효과적인 웨비나 캠페인을 만드는 방법을 알아보세요. 이 기능을 통해 참가자들에게 적절한 타이밍에 알림과 후속 메시지를 자동으로 발송할 수 있습니다.

## 이벤트 시작 날짜란?

이벤트 시작 날짜는 워크플로우(Workflow)에서 특정 날짜와 시간을 기준으로 자동화를 실행할 수 있게 해주는 강력한 기능입니다. 웨비나, 워크샵, 온라인 강의 등의 예정된 이벤트에 맞춰 참가자들에게 맞춤형 커뮤니케이션을 제공할 수 있습니다.

## 웨비나 캠페인 구축 단계

### 1단계: 기본 설정 준비

웨비나 캠페인을 시작하기 전에 다음 요소들을 준비해야 합니다:

- **웨비나 등록 폼(Form)**: 참가자 정보를 수집하는 양식
- **이벤트 날짜 필드**: 연락처(Contacts)에 웨비나 시작 시간을 저장할 커스텀 필드(Custom Field)
- **이메일 템플릿(Email Template)들**: 확인, 알림, 후속 메시지용 템플릿

### 2단계: 워크플로우 생성

1. **Marketing(마케팅) → Workflow(워크플로우)**로 이동합니다
2. **새 워크플로우 만들기**를 클릭합니다
3. 트리거(Trigger)로 **"Form Submission(폼 제출)"**을 선택합니다
4. 웨비나 등록 폼을 선택합니다

### 3단계: 이벤트 시작 날짜 액션 추가

워크플로우에서 이벤트 시작 날짜를 기준으로 한 액션을 추가하는 방법:

1. **액션(Action) 추가**를 클릭합니다
2. **"Wait(대기)"** 액션을 선택합니다
3. 대기 조건에서 **"Event Start Date(이벤트 시작 날짜)"**를 선택합니다
4. 원하는 시간 간격을 설정합니다 (예: 이벤트 1일 전, 1시간 전 등)

### 4단계: 캠페인 메시지 시퀀스 설정

효과적인 웨비나 캠페인을 위한 메시지 시퀀스 예시:

**등록 즉시:**
- 등록 확인 이메일 발송
- 캘린더(Calendar) 초대 발송

**이벤트 3일 전:**
- 웨비나 준비 안내 이메일
- 관련 자료 또는 예습 콘텐츠 제공

**이벤트 1일 전:**
- 리마인더 이메일 발송
- 접속 링크 및 참가 안내

**이벤트 1시간 전:**
- 최종 알림 SMS(문자) 발송
- 마지막 리마인더 이메일

**이벤트 후:**
- 감사 메시지 및 녹화본 제공
- 후속 오퍼(Offer) 또는 다음 이벤트 안내

### 5단계: 조건 분기 활용

참가 여부에 따른 다른 후속 액션을 위해 조건 분기(If/Else)를 추가할 수 있습니다:

1. 웨비나 참가자에게는 감사 메시지와 특별 오퍼
2. 미참가자에게는 녹화본과 다음 기회 안내

## 고급 활용 팁

### 태그(Tag) 기반 세분화
- 참가자: "웨비나-참가" 태그 추가
- 미참가자: "웨비나-미참가" 태그 추가
- 관심 주제별 태그를 통한 맞춤 후속 캠페인

### 개인화된 메시지
- 커스텀 값(Custom Value)을 활용한 개인화
- 참가자 이름, 웨비나 제목, 시간 등 동적 정보 삽입

### 다중 채널 활용
- 이메일과 SMS를 조합한 다중 터치포인트
- 중요한 알림은 여러 채널로 동시 발송

## 주의사항

1. **시간대 설정**: 참가자들의 시간대를 고려하여 설정하세요
2. **테스트 실행**: 실제 캠페인 전에 반드시 테스트해보세요
3. **백업 플랜**: 기술적 문제 발생 시 대응 방안을 준비하세요
4. **법적 준수**: 이메일 마케팅 관련 법규(수신동의 등)를 준수하세요

이벤트 시작 날짜 기능을 활용하면 웨비나 참가자들에게 적시에 적절한 메시지를 전달하여 참가율과 만족도를 크게 높일 수 있습니다. 정기적으로 캠페인 성과를 분석하여 더욱 효과적인 웨비나 마케팅을 구축해보세요.
