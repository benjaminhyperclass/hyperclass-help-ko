# 오퍼 사용 플로우 구축 방법

오퍼(Offer) 사용 플로우를 구축하는 방법에 대해 안내해드립니다.

오퍼 사용 플로우는 고객이 구매한 멤버십이나 강의에 접근할 수 있도록 하는 자동화된 프로세스입니다. 결제 완료 후 고객을 적절한 콘텐츠나 서비스로 안내하는 워크플로우를 설정할 수 있습니다.

## 기본 개념

오퍼 사용 플로우는 다음과 같은 단계로 구성됩니다:

1. **결제 완료**: 고객이 오퍼를 구매합니다
2. **자동 처리**: 시스템이 결제를 확인하고 고객 정보를 업데이트합니다
3. **접근 권한 부여**: 구매한 콘텐츠나 서비스에 대한 접근 권한을 자동으로 부여합니다
4. **안내 및 온보딩**: 고객에게 다음 단계를 안내합니다

## 설정 방법

오퍼 사용 플로우를 설정하려면 워크플로우(Workflow)와 멤버십(Memberships) 기능을 조합하여 사용하는 것이 일반적입니다. 결제 트리거를 활용해 자동화된 고객 여정을 만들 수 있습니다.

자세한 설정 방법은 워크플로우 및 멤버십 관련 가이드를 참고하세요.
