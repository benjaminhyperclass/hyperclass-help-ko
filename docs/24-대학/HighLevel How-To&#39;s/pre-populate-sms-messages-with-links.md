# 링크로 SMS 메시지 미리 입력하기

동영상에서 사용한 예제 코드:

```html
<a href="sms:2155555555&body=Yes, I want Promo Alpha (Claim Code 55)">Link</a>
```

특정 메시지 내용을 감지하는 트리거를 만드는 방법을 알아보려면 [정확히 일치 & 문구 포함 트리거](exact-match-contains-phrase-triggers.md)를 참조하세요.
