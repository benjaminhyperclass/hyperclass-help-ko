# 퍼널과 웹사이트에서 이미지에 커스텀 값 사용하는 방법

퍼널(Funnels)과 웹사이트(Websites)에서 이미지 요소(Element)에 커스텀 값(Custom Values)을 활용하여 동적으로 이미지를 표시할 수 있습니다. 이 기능을 통해 연락처별로 다른 이미지를 보여주거나, 반복 사용되는 이미지를 효율적으로 관리할 수 있어요.

## 커스텀 값으로 이미지 설정하기

1. **페이지 빌더(Page Builder)에서 이미지 요소 선택**: 편집하고 싶은 이미지 요소를 클릭하세요.

2. **이미지 소스 설정**: 이미지 설정 패널에서 이미지 URL 필드에 커스텀 값을 입력합니다.
   - 형식: `{{custom_values.이미지변수명}}`
   - 예시: `{{custom_values.company_logo}}`

3. **커스텀 값 생성**: Settings(설정) → Custom Values(커스텀 값)에서 새로운 이미지 변수를 만들어주세요.

## 활용 사례

### 회사 로고 관리
- 여러 퍼널에서 같은 회사 로고를 사용할 때
- 로고가 변경되면 커스텀 값만 수정하면 모든 페이지에 자동 반영

### 개인화된 이미지
- 연락처별로 다른 프로필 이미지나 맞춤 이미지 표시
- 예: `{{contact.profile_image}}`

### 조건부 이미지 표시
- 워크플로우(Workflow)와 연계하여 상황에 따라 다른 이미지 표시
- 예: VIP 고객에게는 특별 배너 이미지

## 주의사항

- 이미지 URL은 반드시 `https://`로 시작하는 완전한 주소여야 합니다
- 커스텀 값에 이미지 URL이 입력되어 있지 않으면 이미지가 표시되지 않습니다
- 이미지 크기와 형식을 미리 확인하여 레이아웃이 깨지지 않도록 주의하세요

이 방법을 활용하면 더욱 효율적이고 개인화된 퍼널과 웹사이트를 만들 수 있어요.
