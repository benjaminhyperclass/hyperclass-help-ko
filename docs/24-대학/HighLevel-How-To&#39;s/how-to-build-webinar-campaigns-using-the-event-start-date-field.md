# 이벤트 시작 날짜 필드를 활용한 웨비나 캠페인 제작 가이드

이벤트 시작 날짜(Event Start Date) 필드를 활용하면 웨비나나 세미나 등의 이벤트 중심 마케팅 캠페인을 효과적으로 구축할 수 있습니다. 이 필드를 통해 참가자들에게 적절한 타이밍에 리마인더를 발송하고, 이벤트 전후로 체계적인 후속 조치를 자동화할 수 있습니다.

## 이벤트 시작 날짜 필드 설정하기

1. **Contacts(연락처) → Custom Fields(커스텀 필드)** 메뉴로 이동합니다.
2. **Add Field(필드 추가)**를 클릭합니다.
3. 필드 유형을 **Date & Time(날짜 및 시간)**으로 선택합니다.
4. 필드명을 "Event Start Date" 또는 "웨비나 시작일"로 설정합니다.
5. **Save(저장)**를 클릭합니다.

## 웨비나 캠페인 워크플로우 구성

### 1. 등록 확인 워크플로우

**트리거**: 웨비나 등록 폼 제출
- 즉시 확인 이메일 발송
- 이벤트 시작 날짜 필드에 웨비나 일시 자동 입력
- 웨비나 참가자 태그 추가

### 2. 리마인더 워크플로우

**트리거**: 이벤트 시작 날짜 필드 기준
- **7일 전**: 웨비나 준비 안내 이메일
- **1일 전**: 최종 리마인더 이메일 + SMS
- **1시간 전**: 접속 링크가 포함된 긴급 알림

### 3. 이벤트 당일 워크플로우

**트리거**: 이벤트 시작 시간
- 웨비나 시작 15분 전 마지막 알림
- 참석자에게 웰컴 메시지 발송
- 미참석자 추적을 위한 태그 분리

### 4. 후속 조치 워크플로우

**트리거**: 이벤트 종료 후
- **당일**: 감사 이메일 + 녹화 영상 링크
- **3일 후**: 추가 자료 제공 이메일
- **1주 후**: 상담 예약 CTA가 포함된 폴로업

## Smart Lists(스마트 리스트) 활용

이벤트 시작 날짜 필드를 조건으로 활용하여 다음과 같은 스마트 리스트를 만들 수 있습니다:

- **다가오는 웨비나 참가자**: 이벤트 시작 날짜가 오늘 이후인 연락처
- **지난 웨비나 참가자**: 이벤트 시작 날짜가 오늘 이전인 연락처
- **이번 주 웨비나**: 이벤트 시작 날짜가 이번 주 내인 연락처

## 캘린더 연동 팁

웨비나 등록 시 참가자의 개인 캘린더에 일정을 자동 추가하려면:

1. 확인 이메일에 .ics 캘린더 파일 첨부
2. 구글 캘린더, 애플 캘린더 추가 링크 제공
3. 리마인더에 "캘린더에 추가" 버튼 포함

## 주의사항

- 시간대 설정을 정확히 입력해야 리마인더가 올바른 시간에 발송됩니다
- 이벤트 시작 날짜는 참가자가 직접 수정할 수 없도록 읽기 전용으로 설정하세요
- 대량 이벤트의 경우 서버 부하를 고려하여 리마인더 발송 시간을 분산시키세요

이벤트 시작 날짜 필드를 중심으로 한 자동화 워크플로우를 구축하면 웨비나 참석률을 크게 향상시키고, 이벤트 후 전환율도 높일 수 있습니다.
