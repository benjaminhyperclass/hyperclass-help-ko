# Hyperclass에서 예쁜 HTML 이메일 보내는 방법

Hyperclass에서는 시각적으로 매력적인 HTML 이메일을 쉽게 만들고 전송할 수 있습니다. 다음 방법들을 활용하여 전문적인 이메일을 작성해보세요.

## 방법 1: 이메일 템플릿 사용

1. **Marketing(마케팅) → Email Templates(이메일 템플릿)**으로 이동합니다.
2. **Add Template(템플릿 추가)**를 클릭합니다.
3. 미리 제작된 템플릿 중 원하는 디자인을 선택합니다.
4. 드래그 앤 드롭 편집기로 텍스트, 이미지, 버튼을 수정합니다.
5. 완료 후 **Save(저장)**를 클릭합니다.

## 방법 2: 이메일 캠페인으로 HTML 이메일 전송

1. **Marketing(마케팅) → Email Campaigns(이메일 캠페인)**으로 이동합니다.
2. **Create Campaign(캠페인 생성)**을 클릭합니다.
3. 수신자를 선택합니다(스마트 리스트 또는 태그 기반).
4. 이메일 편집기에서 HTML 템플릿을 선택하거나 직접 디자인합니다.
5. **Send(전송)** 또는 **Schedule(예약 전송)**을 선택합니다.

## 방법 3: 대화에서 HTML 이메일 보내기

1. **Conversations(대화)**에서 원하는 연락처를 선택합니다.
2. 이메일 아이콘을 클릭합니다.
3. **Template(템플릿)** 버튼을 클릭하여 저장된 HTML 템플릿을 선택합니다.
4. 필요시 개인화 변수({{first_name}}, {{company}} 등)를 추가합니다.
5. **Send(전송)**을 클릭합니다.

## HTML 이메일 작성 팁

- **반응형 디자인**: 모바일에서도 잘 보이도록 반응형 템플릿을 사용하세요.
- **개인화 변수**: {{first_name}}, {{company}} 등을 활용하여 개인화된 이메일을 보내세요.
- **CTA 버튼**: 명확한 행동 유도 버튼을 포함하세요.
- **이미지 최적화**: 이메일 로딩 속도를 위해 이미지 크기를 최적화하세요.
- **미리보기 테스트**: 전송 전 다양한 기기에서 미리보기를 확인하세요.

HTML 이메일을 통해 고객과의 소통을 한층 더 전문적이고 효과적으로 만들어보세요.
