# 워크플로우 빌더로 자동 예약 후속 설문 만드는 방법

이 가이드에서는 예약이 잡힐 때마다 자동으로 실행되는 4가지 작업을 설정하는 방법을 안내합니다:

1) 예약 확인 이메일/문자 메시지 발송

2) 예약 전 리마인더 이메일/문자 메시지 발송

3) 예약 결과를 확인하는 설문 이메일/문자 메시지 발송

4) 예약이 성공적이었다면 리뷰 요청 이메일/문자 메시지 발송
