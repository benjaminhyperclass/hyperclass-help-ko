# Hyperclass에서 "체크인 루프" 구축하는 방법

체크인 루프는 고객과 지속적인 관계를 유지하고 서비스에 대한 만족도를 확인하는 강력한 자동화 도구입니다. 정기적으로 고객에게 연락하여 현재 상태를 확인하고, 추가 지원이 필요한지 파악할 수 있습니다.

## 체크인 루프란?

체크인 루프는 고객이 서비스를 이용한 후 일정 기간마다 자동으로 연락하여 다음과 같은 목적으로 활용하는 워크플로우입니다:

- 고객 만족도 확인
- 추가 서비스나 제품 제안
- 관계 유지 및 강화
- 피드백 수집
- 재구매 유도

## 체크인 루프 워크플로우 만들기

### 1단계: 새 워크플로우 생성

1. **Marketing(마케팅)** → **Workflows(워크플로우)** 메뉴로 이동합니다
2. **Create Workflow(워크플로우 만들기)** 버튼을 클릭합니다
3. 워크플로우 이름을 "고객 체크인 루프"로 입력합니다

### 2단계: 트리거 설정

적절한 트리거를 선택합니다:

- **Tag Added(태그 추가됨)**: "서비스 완료" 태그가 추가될 때
- **Opportunity Stage Change(기회 단계 변경)**: 파이프라인에서 "완료" 단계로 이동할 때
- **Payment Received(결제 완료)**: 특정 상품의 결제가 완료될 때

### 3단계: 첫 번째 체크인 설정

1. **Wait(대기)** 액션을 추가합니다
   - 서비스 완료 후 적절한 기간을 설정 (예: 7일 후)

2. **Send Email(이메일 발송)** 또는 **Send SMS(SMS 발송)** 액션을 추가합니다
   - 첫 번째 체크인 메시지를 작성합니다

### 4단계: 고객 응답에 따른 분기 설정

**If/Else(조건 분기)** 액션을 사용하여 고객의 응답에 따라 다른 경로로 안내합니다:

#### 만족한 고객의 경우:
- 감사 메시지 발송
- "만족 고객" 태그 추가
- 추천 프로그램 안내
- 리뷰 요청

#### 불만족 고객의 경우:
- 개선 방안 제안
- 직접 상담 일정 안내
- 담당자에게 알림 발송

#### 무응답 고객의 경우:
- 다른 채널로 재연락
- 더 매력적인 메시지로 재시도

### 5단계: 지속적인 체크인 루프 만들기

1. **Wait(대기)** 액션으로 다음 체크인까지의 간격을 설정합니다 (예: 30일)

2. **GoTo(이동)** 액션을 사용하여 체크인 메시지 단계로 다시 돌아갑니다

3. 루프 종료 조건을 설정합니다:
   - 특정 태그가 있는 경우 종료
   - 일정 횟수 반복 후 종료
   - 고객이 구독 해제를 요청한 경우

## 체크인 메시지 작성 팁

### 개인화된 메시지 작성
- 고객 이름과 구매한 서비스를 언급
- 구체적이고 도움이 되는 질문하기
- 친근하면서도 전문적인 톤 유지

### 예시 메시지

**첫 번째 체크인 (7일 후):**
```
안녕하세요 {{contact.first_name}}님,

{{custom_values.company_name}}에서 받으신 [서비스명] 서비스는 어떠셨나요?

혹시 궁금한 점이나 추가로 도움이 필요한 부분이 있으시면 언제든 말씀해 주세요.

감사합니다!
```

**두 번째 체크인 (30일 후):**
```
{{contact.first_name}}님, 안녕하세요!

한 달 전 저희 서비스를 이용해 주셨는데, 그 후 어떻게 지내고 계신가요?

현재 새로운 [관련 서비스]도 준비되어 있어서, 관심 있으시면 안내드릴 수 있습니다.

편한 시간에 연락 주세요!
```

## 고급 체크인 루프 기능

### 스마트 리스트 활용
- 체크인 루프에 참여한 고객들을 별도 스마트 리스트로 관리
- 응답률과 만족도에 따른 세분화

### A/B 테스트
- 다른 메시지 내용이나 발송 시간으로 테스트
- 더 효과적인 체크인 방식 찾기

### 멀티채널 접근
- 이메일 → SMS → 전화 순서로 단계적 접촉
- 고객이 선호하는 채널 파악 후 맞춤 접촉

## 성과 측정 및 개선

### 주요 지표 추적
- 체크인 메시지 개봉률
- 고객 응답률
- 만족도 점수
- 추가 구매 전환율

### 지속적인 개선
- 고객 피드백을 바탕으로 메시지 개선
- 최적의 체크인 주기 찾기
- 효과적인 CTA 개발

체크인 루프를 통해 고객과의 관계를 지속적으로 관리하고, 장기적인 고객 가치를 높일 수 있습니다. 처음에는 간단한 루프로 시작하여 점차 고도화해 나가시기 바랍니다.
