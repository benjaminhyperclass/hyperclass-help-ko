# 포지티브 인텐트(Positive Intent) 트리거를 사용해 완전 자동으로 예약을 생성하는 방법

포지티브 인텐트 트리거는 고객이 예약을 원하는 긍정적인 의도를 보일 때 자동으로 예약 과정을 시작하는 강력한 워크플로우 트리거입니다. 이 기능을 사용하면 사람의 개입 없이도 자동으로 예약을 생성할 수 있어, 24시간 예약 시스템을 구축할 수 있습니다.

## 포지티브 인텐트 트리거란?

포지티브 인텐트 트리거는 고객의 메시지나 행동에서 예약 의도를 자동으로 감지하는 AI 기반 트리거입니다. 예를 들어:

- "상담 예약하고 싶어요"
- "언제 만날 수 있나요?"
- "미팅 잡을 수 있을까요?"

이런 메시지를 받으면 자동으로 워크플로우가 실행되어 예약 과정을 진행합니다.

## 설정 방법

### 1단계: 워크플로우 생성

1. **Marketing(마케팅)** → **Workflow(워크플로우)** 메뉴로 이동합니다
2. **Create Workflow(워크플로우 생성)** 버튼을 클릭합니다
3. 워크플로우 이름을 입력합니다 (예: "자동 예약 생성")

### 2단계: 포지티브 인텐트 트리거 추가

1. 트리거 섹션에서 **Add Trigger(트리거 추가)**를 클릭합니다
2. **Positive Intent(포지티브 인텐트)** 트리거를 선택합니다
3. 다음 설정을 구성합니다:
   - **채널 선택**: SMS, 이메일, 채팅 위젯 등 감지할 채널을 선택
   - **키워드 설정**: 감지할 키워드나 문구를 추가 (선택사항)
   - **민감도 조절**: AI의 감지 민감도를 조정

### 3단계: 자동 액션 설정

포지티브 인텐트가 감지되면 실행할 액션들을 추가합니다:

#### 즉시 응답 메시지
```
안녕하세요! 예약 문의해 주셔서 감사합니다. 
바로 예약 가능한 시간을 안내해 드리겠습니다.
```

#### 캘린더 링크 전송
1. **Send Message(메시지 발송)** 액션을 추가합니다
2. 예약 캘린더 링크를 포함한 메시지를 작성합니다:
```
아래 링크에서 편리한 시간을 선택해서 예약해 주세요:
[예약하기](귀하의 캘린더 링크)

궁금한 점이 있으시면 언제든 문의해 주세요!
```

#### 태그 추가
- **Add Tag(태그 추가)** 액션으로 "예약관심" 태그를 추가합니다
- 이후 추적 및 분석에 활용할 수 있습니다

### 4단계: 후속 워크플로우 설정

예약이 완료되지 않은 경우를 대비한 후속 액션을 설정합니다:

1. **Wait(대기)** 액션으로 30분 대기
2. **If/Else(조건 분기)**로 예약 완료 여부 확인
3. 예약이 안 된 경우 리마인드 메시지 발송:
```
아직 예약을 완료하지 않으셨네요. 
혹시 도움이 필요하시다면 언제든 말씀해 주세요!
```

## 활용 팁

### 1. 다양한 채널에서 활용
- **채팅 위젯**: 웹사이트 방문자의 예약 문의
- **SMS**: 문자로 온 예약 요청
- **이메일**: 이메일로 온 상담 문의
- **소셜 미디어**: 인스타그램, 페이스북 DM

### 2. 업종별 맞춤 설정

**의료/치과**
```
진료 예약 문의해 주셔서 감사합니다.
아래 링크에서 원하는 날짜와 시간을 선택해 주세요.
```

**미용실**
```
예약 문의 감사합니다! 
원하는 서비스와 스타일리스트를 선택해서 예약해 주세요.
```

**컨설팅**
```
상담 문의 감사합니다.
무료 상담 예약은 아래 링크에서 가능합니다.
```

### 3. 성과 측정
- **Reporting(리포팅)**에서 자동 생성된 예약 수 확인
- 포지티브 인텐트 감지 정확도 모니터링
- 예약 완료율 분석

## 주의사항

1. **정확도 확인**: 초기에는 감지 결과를 모니터링하여 정확도를 확인하세요
2. **백업 계획**: 자동화가 실패할 경우를 대비한 수동 대응 방안 준비
3. **개인화**: 가능한 한 고객 이름이나 관심 분야를 포함한 개인화된 메시지 사용

이 설정을 통해 고객이 언제든 예약 의도를 보이면 즉시 자동으로 예약 프로세스가 시작되어, 24시간 예약 시스템을 구축할 수 있습니다.
