# Hyperclass에서 예약 봇 만들기

이 가이드에서는 Hyperclass(Hyperclass)를 사용하여 자동 예약 봇을 구축하는 방법을 안내해드립니다. 예약 봇을 통해 고객이 24시간 언제든지 상담이나 서비스 예약을 할 수 있도록 도울 수 있습니다.

## 예약 봇이란?

예약 봇은 고객과의 대화를 통해 자동으로 예약을 처리하는 AI 챗봇입니다. 고객이 채팅 위젯(Chat Widget)이나 메신저를 통해 문의하면, 봇이 다음과 같은 작업을 자동으로 수행합니다:

- 고객의 예약 의도 파악
- 예약 가능한 시간 안내
- 고객 정보 수집
- 예약 확정 및 확인 메시지 발송

## 예약 봇 설정하기

### 1. 캘린더(Calendars) 준비

먼저 예약을 받을 캘린더가 설정되어 있어야 합니다:

1. `Calendars(캘린더)` 메뉴로 이동
2. 예약 봇에서 사용할 캘린더를 선택하거나 새로 생성
3. 예약 가능 시간(Availability)과 서비스(Services) 설정 확인

### 2. 대화 AI(Conversation AI) 설정

1. `AI` 메뉴에서 `Conversation AI(대화 AI)` 선택
2. **새 AI 직원(AI Employee) 생성** 또는 기존 봇 편집
3. **봇 지시사항(Bot Instructions)**에 예약 관련 안내 추가:

```
당신은 예약 전담 상담원입니다. 고객이 상담 예약을 원하면 다음과 같이 도와주세요:
1. 어떤 서비스를 원하는지 확인
2. 선호하는 날짜와 시간 물어보기
3. 연락처 정보 수집
4. 예약 확정 후 확인 메시지 발송
항상 친절하고 전문적으로 응답하세요.
```

### 3. 워크플로우(Workflow) 연동

예약 완료 후 자동화된 후속 작업을 위해 워크플로우를 설정합니다:

1. `Automation(자동화)` → `Workflows(워크플로우)` 이동
2. **새 워크플로우 생성**
3. **트리거(Trigger)**: "Appointment Booked(예약 완료)"
4. **액션(Actions)** 추가:
   - 확인 SMS 또는 이메일 발송
   - 예약 전 리마인더 설정
   - 담당자에게 알림

### 4. 채팅 위젯(Chat Widget) 설정

웹사이트나 퍼널에 예약 봇을 연결합니다:

1. `Sites(사이트)` → 해당 페이지 편집
2. **채팅 위젯 요소(Chat Widget Element)** 추가
3. 위젯 설정에서 생성한 AI 봇 연결
4. 환영 메시지를 예약 안내로 설정

## 예약 봇 최적화 팁

### 1. 자주 묻는 질문 추가

**커스텀 봇 응답(Custom Bot Responses)**에서 예약 관련 FAQ를 등록하세요:

- "영업시간은 언제인가요?"
- "취소는 어떻게 하나요?"
- "비용은 얼마인가요?"

### 2. 조건 분기 설정

복잡한 예약 프로세스의 경우 If/Else 조건을 사용하여:

- 서비스 유형별로 다른 캘린더 안내
- VIP 고객에게는 우선 예약 시간 제공
- 신규/기존 고객 구분하여 다른 질문 진행

### 3. 수동 모드 활용

중요한 예약이나 복잡한 상황에서는:

- **제안 모드(Suggestive)**로 설정하여 사람이 최종 확인
- **수동 작업(Manual Actions)**으로 담당자에게 알림

## 성과 측정

예약 봇의 효과를 확인하려면:

1. `Reporting(리포팅)` → `Appointment Report(예약 보고서)`에서 예약 건수 확인
2. `Conversations(대화)` 메뉴에서 봇 대화 품질 모니터링
3. 노쇼율과 예약 완료율 비교 분석

예약 봇을 통해 고객 만족도를 높이고 업무 효율성을 크게 개선할 수 있습니다. 초기 설정 후에는 고객 피드백을 바탕으로 지속적으로 봇 응답을 개선해나가세요.
