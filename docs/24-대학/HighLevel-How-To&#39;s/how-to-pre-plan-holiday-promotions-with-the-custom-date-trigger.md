# 커스텀 날짜 트리거로 홀리데이 프로모션 미리 계획하는 방법

커스텀 날짜 트리거(Custom Date Trigger)는 특정 날짜에 자동으로 실행되는 워크플로우를 만들 수 있는 강력한 기능입니다. 홀리데이 시즌이나 특별한 날에 맞춰 프로모션을 미리 계획하고 자동화할 수 있어요.

## 커스텀 날짜 트리거란?

커스텀 날짜 트리거는 정해진 날짜와 시간에 자동으로 워크플로우를 시작하는 기능입니다. 예를 들어:

- 크리스마스 프로모션을 12월 20일에 자동 시작
- 신년 할인 이메일을 1월 1일 오전 9시에 발송
- 밸런타인데이 캠페인을 2월 10일에 시작

## 홀리데이 프로모션 워크플로우 만들기

### 1단계: 새 워크플로우 생성

1. 좌측 메뉴에서 `Automation(자동화)` → `Workflows(워크플로우)`로 이동하세요
2. `Create Workflow(워크플로우 만들기)` 버튼을 클릭하세요
3. 워크플로우 이름을 입력하세요 (예: "크리스마스 프로모션 2024")

### 2단계: 커스텀 날짜 트리거 설정

1. 트리거 섹션에서 `Custom Date(커스텀 날짜)`를 선택하세요
2. 다음 정보를 설정하세요:
   - **실행 날짜**: 프로모션을 시작할 날짜
   - **실행 시간**: 구체적인 시간 (예: 오전 9시)
   - **타임존**: 해당 지역의 시간대

### 3단계: 대상 고객 설정

1. `Who should enter this workflow?(누가 이 워크플로우에 들어갈까요?)` 섹션에서 대상을 설정하세요
2. 옵션은 다음과 같습니다:
   - **All Contacts(전체 연락처)**: 모든 고객
   - **Smart List(스마트 리스트)**: 특정 조건의 고객들
   - **Tagged Contacts(태그된 연락처)**: 특정 태그가 있는 고객들

### 4단계: 프로모션 액션 추가

워크플로우에 다음과 같은 액션들을 추가할 수 있어요:

**이메일 발송**
- `Send Email(이메일 발송)` 액션 추가
- 홀리데이 테마의 이메일 템플릿 선택
- 제목과 내용에 할인 코드나 특별 혜택 포함

**SMS 발송**
- `Send SMS(SMS 발송)` 액션 추가
- 간단하고 임팩트 있는 프로모션 메시지 작성

**태그 추가**
- `Add Tag(태그 추가)` 액션으로 "홀리데이 프로모션 수신자" 태그 추가
- 나중에 프로모션 효과 분석에 활용

### 5단계: 후속 액션 설정

프로모션의 효과를 높이기 위해 추가 액션을 설정하세요:

1. **대기 액션**: `Wait(대기)` 액션을 추가해서 3일 후 후속 이메일 발송
2. **조건 분기**: 이메일을 열어본 고객과 그렇지 않은 고객을 구분해서 다른 메시지 발송
3. **할 일 생성**: 팀원에게 프로모션 결과 확인 태스크 할당

## 활용 예시

### 크리스마스 프로모션 워크플로우

```
트리거: 12월 20일 오전 9시
↓
이메일 발송: "크리스마스 특별 할인 20%"
↓
대기: 3일
↓
조건 분기: 이메일 오픈 여부
├─ 오픈함 → SMS: "마감 임박! 크리스마스 할인"
└─ 미오픈 → 이메일: "놓치신 크리스마스 할인을 확인하세요"
```

### 신년 맞이 캠페인

```
트리거: 1월 1일 오후 12시
↓
SMS 발송: "새해 복 많이 받으세요! 신년 특가 30% 할인"
↓
태그 추가: "2024 신년 캠페인"
↓
대기: 7일
↓
이메일 발송: "신년 다짐을 이루어드릴 특별한 제안"
```

## 주의사항

1. **테스트 필수**: 실제 고객에게 발송하기 전에 반드시 테스트해보세요
2. **시간대 확인**: 타임존 설정을 정확히 해야 의도한 시간에 발송됩니다
3. **컨텐츠 준비**: 홀리데이 테마에 맞는 이메일 템플릿과 메시지를 미리 준비하세요
4. **법적 준수**: 스팸 방지 법규를 준수하고, 수신 거부 링크를 반드시 포함하세요

## 성과 측정

프로모션이 끝나면 다음 지표들을 확인해보세요:

- 이메일 오픈율과 클릭률
- SMS 응답률
- 웹사이트 방문자 증가율
- 실제 매출 증가율
- 신규 고객 전환율

커스텀 날짜 트리거를 활용하면 홀리데이 시즌의 바쁜 시기에도 체계적이고 효과적인 마케팅 캠페인을 자동으로 실행할 수 있습니다. 미리 계획하고 설정해두면, 정확한 타이밍에 고객들에게 매력적인 프로모션을 전달할 수 있어요.
