# 하위 계정에서 왓츠앱 번호 인증 OTP 전송 실패 문제 해결 방법

하위 계정에서 왓츠앱 번호를 인증할 때 OTP(일회용 비밀번호)를 받지 못하는 경우, 이 가이드를 통해 원인을 파악하고 문제를 해결할 수 있습니다.

## 문제 발생 원인

OTP 인증이 실패하는 가장 일반적인 이유는 등록하려는 전화번호가 이미 왓츠앱 비즈니스 앱이나 일반 왓츠앱 메신저에 등록되어 있기 때문입니다. 한 번호는 한 번에 하나의 왓츠앱 플랫폼에서만 활성 상태로 사용할 수 있습니다.

## 번호가 이미 등록되었는지 확인하기

다음 플랫폼에서 해당 전화번호가 현재 활성 상태인지 확인해보세요:

- 왓츠앱 비즈니스 앱
- 일반 왓츠앱 메신저

번호가 이미 등록되어 있다면, 왓츠앱은 온보딩 과정에서 OTP 전송을 차단합니다.

다음은 전화번호가 왓츠앱 비즈니스에 등록되었는지 확인하는 신뢰할 수 있고 실용적인 방법들입니다.

**왓츠앱 공식 Click to Chat 기능 사용**

단계:
- 브라우저를 열어주세요
- 다음 URL을 입력하세요 (국가 코드를 포함한 전체 번호로 교체):

```
https://wa.me/<국가코드><전화번호>
```

예시:
```
https://wa.me/919876543210
```

- Enter를 누르세요

**결과 해석:**
- 왓츠앱이 열리면 → 해당 번호가 왓츠앱에 등록되어 있음 (개인용 또는 비즈니스용)
- "Phone number shared via url is invalid" 메시지가 보이면 → 해당 번호가 왓츠앱에 등록되어 있지 않음

**왓츠앱 비즈니스 앱 내에서 확인**

단계:
- 해당 번호를 휴대폰 연락처에 저장하세요
- 왓츠앱을 열어주세요
- 새 채팅 시작 → 연락처 검색

**확인 지표:**
- 연락처가 나타나면 → 해당 번호가 왓츠앱에 있음
- 연락처가 나타나지 않으면 → 등록되지 않음

**비즈니스 앱 확인 지표:**
- 프로필에 "Business Account" 또는 "Verified Business"가 표시되면, 해당 번호가 왓츠앱 비즈니스에 등록되어 있음
- 비즈니스 계정은 보통 다음을 표시합니다:
  - 비즈니스 이름
  - 비즈니스 카테고리
  - 카탈로그 (선택사항)

## 다음 해결책 중 하나를 선택하세요

### 옵션 1: 다른 전화번호 사용 (권장)

- 왓츠앱에 등록된 적이 없는 새로운 전화번호를 사용하세요.
- 새 번호로 온보딩을 진행하세요.

### 옵션 2: 기존 왓츠앱 계정 삭제

같은 번호를 반드시 사용해야 한다면, 다음 단계를 신중히 따라주세요:

- 같은 번호를 사용하여 모바일 기기에 왓츠앱(비즈니스 또는 일반)을 설치하세요.
- 왓츠앱을 열고 설정(Settings)으로 이동하세요.
- 계정(Account) → 계정 삭제(Delete Account)를 선택하세요.
- 모바일 기기에서 왓츠앱 계정 삭제를 확인하세요.
- 몇 분 기다린 후, 하위 계정에서 온보딩과 OTP 인증을 다시 시도하세요.

⚠️ 왓츠앱 계정을 삭제하면 해당 번호와 연결된 채팅 기록과 데이터가 영구적으로 제거됩니다.

## 여전히 OTP를 받지 못하나요?

전화번호가:
- 왓츠앱이나 왓츠앱 비즈니스에 등록되어 있지 않고
- 여전히 OTP를 받을 수 없는 경우

다음 정보와 함께 지원 티켓을 제출해주세요:

- 전화번호 (국가 코드 포함)
- 하위 계정 ID
- OTP 인증 중 표시된 스크린샷 또는 정확한 오류 메시지
- OTP 시도가 실패한 대략적인 시간
