# **스냅샷을 사용한 하위 계정 생성 개요**

스냅샷을 사용해서 하위 계정을 만들면 에이전시에서 선택한 스냅샷에 포함된 모든 도구, 워크플로우, 설정, 자산이 포함된 새 계정을 쉽게 생성할 수 있습니다. 하위 계정이 생성되면 스냅샷이 자동으로 로드되므로, 생성 후 수동으로 스냅샷을 로드할 필요가 없습니다. 이는 온보딩 프로세스를 간소화하고 새 계정이 일관된 미리 구축된 설정으로 시작할 수 있도록 보장합니다.

---

## 스냅샷을 사용한 하위 계정 생성의 주요 장점

스냅샷을 사용해서 하위 계정을 생성하면 에이전시에서 표준화된 하위 계정을 더 빠르게 설정할 수 있습니다.

- **빠른 온보딩**: 모든 필수 자산과 설정을 미리 로드하여 수동 설정을 줄입니다.
- **계정 간 일관성**: 원하는 계정에 동일한 자산과 설정이 포함되도록 보장합니다.
- **시간 절약**: 생성 후 각 새 하위 계정에 대한 반복적인 구성을 없애줍니다.
- **오류 감소**: 설정 실수나 누락된 자산의 위험을 줄입니다.

---

## 사전 준비사항

설정 요구사항을 이해하면 첫 번째 시도에서 프로세스가 성공할 수 있도록 도와줍니다.

- 에이전시 관리자(Agency Admin) 액세스 권한이 있어야 합니다
- 계정에 스냅샷이 이미 존재해야 합니다
- 스냅샷에는 복사하고자 하는 주요 자산(퍼널, 트리거, 캘린더 등)이 포함되어 있어야 합니다
- 선택사항: 미리 스냅샷을 가져오거나 생성하세요

## 스냅샷을 사용해서 새 하위 계정 만드는 방법

다음 단계에 따라 원하는 스냅샷을 기반으로 새 하위 계정을 생성하세요.

#### 1단계: Account Snapshots(계정 스냅샷)으로 이동

- 에이전시 대시보드에서 왼쪽 메뉴의 **Account Snapshots(계정 스냅샷)**을 클릭하세요

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155049966392/original/uHmwc-IKJ0-JgoaXVruZL5RLaXtK_qnveA.png?1752695024)

#### 2단계: 스냅샷 선택

- 새 하위 계정의 기본으로 사용할 스냅샷을 찾으세요
- 스냅샷 옆에 있는 파란색 Create New Account (+)(새 계정 생성) 버튼을 클릭하세요

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155049966457/original/mHW513SBk2GVrKrvd5IshTz5MDVOXGyJnA.png?1752695169)

#### 3단계: 하위 계정 생성

- 새 창이 열리면서 새 하위 계정 정보를 입력할 수 있습니다
- 모든 정보를 입력한 후 Add Sub-Account(하위 계정 추가)를 클릭하세요

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155049966494/original/NI-WY8pXakXgcO7ipkMbHwE57UlMXkMgIg.png?1752695281)

## 자주 묻는 질문

**Q: 계정이 생성된 후 스냅샷을 삭제할 수 있나요?**
A: 아니요, 한 번 적용된 스냅샷은 "언로드"하거나 "삭제"할 수 없습니다. 특정 요소를 수동으로 삭제하거나 수정할 수는 있습니다.

**Q: 스냅샷을 사용해서 하위 계정을 만들려면 어떤 액세스 권한이 필요한가요?**
A: 에이전시 관리자(Agency Admin) 권한이 있는 사용자만 이 작업을 수행할 수 있습니다.

**Q: 스냅샷에는 어떤 자산이 포함되나요?**
A: 스냅샷에 저장된 내용에 따라 퍼널, 폼, 워크플로우, 캘린더, 설정 등이 포함됩니다. 자세한 정보는 [스냅샷 개요](snapshots-overview.md)를 참조하세요.

**Q: 스냅샷이 새 하위 계정의 기존 내용을 덮어쓰나요?**
A: 아니요. 하위 계정이 새로 생성되기 때문에 덮어쓸 내용이 없습니다. 스냅샷은 계정 생성 중에 적용되므로 그 내용이 초기 설정이 됩니다.

**Q: 새 하위 계정에 두 개 이상의 스냅샷을 적용할 수 있나요?**
A: 생성 과정에서는 불가능합니다. 하위 계정을 생성할 때는 하나의 스냅샷만 선택할 수 있습니다. 하지만 하위 계정이 생성된 후에는 추가 스냅샷을 수동으로 적용할 수 있습니다.

**Q: 스냅샷에서 로드한 후 퍼널과 웹사이트는 어떻게 되나요?**
A: 하위 계정에 스냅샷을 로드하면 포함된 모든 퍼널과 웹사이트가 초안(Draft) 상태로 복사됩니다. **커스텀 도메인을 연결하고 발행할 때까지는 실제로 활성화되지 않습니다.**
