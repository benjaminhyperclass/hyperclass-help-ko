# 스냅샷을 가져오는 방법

이 가이드는 Hyperclass에서 스냅샷(Snapshots)을 가져오는 방법을 설명합니다. 미리 구축된 자산과 설정을 새 계정에 빠르게 적용하여 수동 작업을 줄일 수 있습니다.

## 스냅샷 공유 링크로 가져오기

- 웹 브라우저에서 Hyperclass에 로그인하세요
- 브라우저에서 새 탭을 열어주세요
- 스냅샷 공유 링크를 브라우저 주소창에 붙여넣고 Enter를 누르세요
- **Yes! Import Now** 버튼을 클릭하세요

![스냅샷 공유 링크로 가져오기](https://jumpshare.com/v/8de01yeEoX3J77XiHPoc+/GIF+Recording+2025-06-04+at+1.47.25+AM.gif)

## 이메일로 가져오기

- 웹 브라우저에서 Hyperclass에 로그인하세요
- 이메일에서 **Click here** 링크를 클릭하세요
- **Yes! Import Now** 버튼을 클릭하세요

![이메일로 가져오기](https://jumpshare.com/v/srIshE6SSjPI5Dme35HR+/GIF+Recording+2025-06-04+at+1.54.57+AM.gif)

**참고:** 가져온 스냅샷은 에이전시 레벨에서 `Settings(설정) > Account Snapshot > Imported Snapshots` 탭에 저장됩니다. 하위 계정(Sub-account) 내부에서 자동으로 보이지는 않습니다. 특정 하위 계정에서 가져온 스냅샷을 사용하려면 `Agency View > Sub-accounts > 하위 계정 선택 > Actions > Load Snapshot`을 통해 수동으로 로드해야 합니다.

## 하위 계정에 스냅샷 로드하는 단계

### 1단계: 클라이언트 계정 관리로 이동

- 에이전시 메뉴에서 **Sub-Accounts(하위 계정)** 을 클릭하세요.

![하위 계정 메뉴](https://jumpshare.com/v/I1HO4XLanBPol44LUQKh+/Screen+Shot+2025-06-03+at+8.25.26+PM.png)

- 스냅샷을 로드하려는 하위 계정에서 **Switch to Sub-account** 버튼 옆의 점 세 개를 클릭하세요.
- 팝업에서 **Manage Client** 를 클릭하세요.

![클라이언트 관리](https://jumpshare.com/v/2JbNtHxbjB54KOk40Hyw+/Screen+Shot+2025-06-03+at+8.37.39+PM.png)

### 2단계: 스냅샷 로드

- 클라이언트/하위 계정 관리 페이지에서 **Actions(작업)** 을 클릭하세요.
- 드롭다운에서 **Load Snapshot** 을 클릭하세요.

![스냅샷 로드](https://jumpshare.com/v/tXbPWqmglVgZHmmIXMpG+/Screen+Shot+2025-06-03+at+8.43.52+PM.png)

- 미리 가져온 목록에서 스냅샷을 선택하세요.
- **Proceed** 를 클릭하세요.

![스냅샷 선택](https://jumpshare.com/v/xLIB0TVuCr0zFYMffb0P+/Screen+Shot+2025-06-03+at+8.50.59+PM.png)

- 스냅샷에서 하위 계정으로 푸시하려는 자산을 선택하거나 건너뛰세요.

**참고:** 스냅샷에 웨비나가 포함되어 있다면, 가져오기를 통해 웨비나 페이지가 복제되고 해당 웨비나 세션이 자동으로 생성됩니다. 관련 웨비나 데이터는 가져오기 과정에서 복원됩니다.

- **Proceed** 를 클릭하세요.

![자산 선택](https://jumpshare.com/v/2SW18hbaZRxt04EECGtj+/GIF+Recording+2025-06-03+at+8.59.21+PM.gif)

- + 아이콘을 클릭하여 충돌을 확인하고 **Proceed** 를 클릭하세요.
- 박스에 **Confirm** 을 입력하여 로딩 프로세스를 시작하세요.

![확인 및 로딩 시작](https://jumpshare.com/v/3tlkHuN7JoFOYZPL0GRI+/GIF+Recording+2025-06-03+at+9.03.23+PM.gif)
