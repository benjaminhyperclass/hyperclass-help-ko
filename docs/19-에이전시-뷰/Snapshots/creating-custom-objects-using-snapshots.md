# 스냅샷으로 커스텀 오브젝트 생성하기

이 가이드에서는 스냅샷의 커스텀 오브젝트 지원 기능을 단계별로 설명합니다. 모든 커스텀 필드와 연관 관계를 포함한 커스텀 오브젝트 데이터를 하위 계정 간에 복사할 수 있습니다. 기능의 장점, 사용 방법, 중요한 제한사항을 다루어, 여러 하위 계정에서 커스텀 오브젝트 데이터를 쉽게 복제할 수 있도록 돕습니다.

### 스냅샷에서 커스텀 오브젝트 지원

이제 스냅샷에 커스텀 오브젝트와 연관된 모든 커스텀 필드(기본 필드뿐만 아니라)를 포함할 수 있습니다. 이를 통해 여러 하위 계정에서 커스텀 오브젝트 데이터를 손쉽게 복사할 수 있습니다.

**중요**: 커스텀 오브젝트는 하위 계정 간에서만 복사할 수 있습니다. 커스텀 오브젝트의 직접 내보내기는 지원되지 않습니다.

### 커스텀 오브젝트 연관 관계

커스텀 오브젝트는 스냅샷 과정에서 다른 커스텀 오브젝트나 연락처와 자동으로 연관됩니다. 연관 관계는 기본적으로 포함되며 스냅샷 생성 시 수정하거나 선택 해제할 수 없습니다.

**참고**: 커스텀 오브젝트와 다른 오브젝트(연락처 포함) 간의 연관 관계는 자동으로 생성되거나 업데이트됩니다. 스냅샷 과정에서 이러한 연관 관계를 수동으로 편집할 수는 없습니다.

### 하위 계정 간 커스텀 오브젝트 복사 방법

커스텀 오브젝트 데이터를 복사하려면 다음 단계를 따르세요:

1. `Agency(에이전시) → Account Snapshots(계정 스냅샷) → Create Snapshot(스냅샷 생성)`으로 이동합니다.

2. 스냅샷에 포함하려는 커스텀 오브젝트를 선택합니다.

![커스텀 오브젝트 선택 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044030767/original/n7iRDBYfiTyYb3DoXpfDSxrIyGPW_4o0ZA.jpeg?1743003209)

3. 선택한 커스텀 오브젝트의 필드와 연관 관계(다른 커스텀 오브젝트나 연락처와의)가 자동으로 포함됩니다.

4. 원본 및 대상 하위 계정을 선택하여 스냅샷 전송을 완료합니다.

### 중요한 제한사항

- **스냅샷당 커스텀 오브젝트 제한**: 스냅샷은 하위 계정당 최대 10개의 커스텀 오브젝트를 포함할 수 있습니다. 스냅샷을 가져오는 것이 이 제한을 초과할 경우, 대상 하위 계정에 커스텀 오브젝트가 생성되지 않습니다.
