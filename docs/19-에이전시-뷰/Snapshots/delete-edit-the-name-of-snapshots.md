# 스냅샷 삭제 및 이름 편집

## 스냅샷 삭제

**스냅샷을 삭제하려면 아래 단계를 따라하세요**

- Settings(설정) → Agency view(에이전시 뷰) 클릭
- Account Snapshot(계정 스냅샷) 클릭
- My Snapshots(내 스냅샷), Imported Snapshots(가져온 스냅샷) 또는 Default Snapshots(기본 스냅샷) 탭에서 삭제하려는 스냅샷을 찾으세요
- 삭제하려는 스냅샷 옆의 휴지통 아이콘 클릭
- Ok(확인) 클릭

## 스냅샷 이름 편집

**스냅샷 이름을 편집하려면 아래 단계를 따라하세요**

- Settings(설정) → Agency view(에이전시 뷰) 클릭
- Account Snapshot(계정 스냅샷) 클릭
- My Snapshots(내 스냅샷), Imported Snapshots(가져온 스냅샷) 또는 Default Snapshots(기본 스냅샷) 탭에서 이름을 편집하려는 스냅샷을 찾으세요
- 이름을 편집하려는 스냅샷 옆의 연필 아이콘 클릭
- Snapshot name(스냅샷 이름) 필드에 새 이름을 입력하세요
- Save(저장) 클릭
