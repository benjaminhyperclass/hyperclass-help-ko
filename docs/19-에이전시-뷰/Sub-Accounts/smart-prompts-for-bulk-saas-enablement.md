# 대량 SaaS 활성화를 위한 스마트 프롬프트

아직 SaaS를 사용하지 않는 모든 고객을 식별하고, 안내된 워크플로우를 통해 일괄적으로 활성화하여 하위 계정에서 더 많은 가치를 얻어보세요.

가이드 방식 대량 SaaS 활성화 조건:
- $497/월 Pro 플랜을 사용하는 에이전시
- Labs에서 *SaaS 활성화를 위한 스마트 프롬프트* 기능 활성화
- 10개 이상의 하위 계정 보유
- SaaS를 사용하지 *않는* 하위 계정이 최소 1개 이상

대량 SaaS 활성화 결과:
- 지정된 하위 계정이 자동으로 SaaS로 전환됩니다
- 하위 계정에 결제 카드가 등록되어 있는 경우, 서비스 중단 없이 진행됩니다
- 하위 계정에 결제 카드가 등록되어 있지 않은 경우, 고객이 카드를 추가할 때까지 하위 계정이 잠깁니다

## **가이드 방식 대량 SaaS 활성화 사용법**

### **1단계: 하위 계정으로 이동하여 프롬프트 클릭**

**에이전시 계정** > **하위 계정** 페이지에서 조건이 충족되면 노란색 프롬프트가 표시됩니다. "Click here to enable SaaS(여기를 클릭하여 SaaS 활성화)" 문구를 클릭하세요.

![하위 계정 페이지의 SaaS 활성화 프롬프트](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048944843/original/alXuycsg2DHt0U1gMF9-AB_Z033ZfotAVQ.png?1750958159)

### **2단계: 하위 계정 선택 (전체 또는 여러 개 선택) 후 진행**

"**SaaS를 활성화할 하위 계정 선택**" 모달에서 원하는 하위 계정을 체크하거나 Select All(전체 선택)을 클릭한 다음 Proceed(진행)를 클릭하세요.

![하위 계정 선택 모달](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048944995/original/JLrIZoDW4yY18pfSMw8YzURN1CcE2InwYQ.png?1750958386)

### **3단계: 결제 제공업체 선택 (Stripe)**

"Choose your payment provider(결제 제공업체 선택)" 모달에서 Stripe 아래의 Continue(계속)를 클릭하거나 다른 결제 제공업체를 선택하세요.

![결제 제공업체 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048945147/original/na2CcEH7yiXRSg0UXakkB_FACYhCG77Kqg.png?1750958568)

### **4단계: SaaS 플랜 선택**

계속 진행하기 전에 플랜 세부 정보를 검토하세요. Select Subscription Plan(구독 플랜 선택) 모달에서 Hyperclass은 이제 각 플랜에 대한 더 많은 정보를 표시하므로 활성화하려는 내용을 확인할 수 있습니다.

- Select Plan 버튼 위의 **Show Full Details(전체 세부 정보 표시)** 옵션을 클릭하세요.

![플랜 세부 정보 표시](https://jumpshare.com/share/7v1siL0HSgt6gnu9u2W9+/Screenshot+2025-11-21+at+1.54.11%E2%80%AFPM.png)

- **검토 가능한 내용:** 재청구 요금, 플랜에 포함된 애드온, 플랜에 포함된 마켓플레이스 앱, 플랜에 포함된 커스텀 메뉴 링크.

![플랜 상세 내용](https://jumpshare.com/share/rbz3LnVy8nRbmHFpEwF1+/Screenshot+2025-11-21+at+1.55.15%E2%80%AFPM.png)

### **5단계: 검토 및 확인**

"Confirm Sub-Accounts to Enable SaaS(SaaS를 활성화할 하위 계정 확인)" 모달에서 선택한 SaaS 플랜과 하위 계정을 검토하세요. 만족하시면 Proceed(진행)를 클릭하세요. 이렇게 하면 프로세스가 시작됩니다.

![SaaS 활성화 확인](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048945617/original/bOc-gOfnybb00BIITrZ3pPtUwg6bE-5Xhw.png?1750959305)

### **6단계: 대량 작업 기록 모니터링**

"Bulk Action Initiated Successfully(대량 작업이 성공적으로 시작됨)" 모달에서 "실행 완료까지 몇 분 정도 소요될 수 있습니다. 대량 작업 기록 페이지에서 자세한 진행 상황을 확인하세요"라는 메모를 볼 수 있습니다. View Bulk Action History(대량 작업 기록 보기)를 클릭하여 모니터링하세요.

![대량 작업 시작 확인](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048945733/original/Nd8n_7nDkPR449Gl0uX31U63hYE9QFkddQ.png?1750959453)

### **7단계: 대량 작업 기록으로 돌아가기**

모달이 닫힌 후에는 언제든지 에이전시 계정 > 하위 계정 > Bulk Action History(대량 작업 기록, 시계 아이콘)로 이동할 수 있습니다.

![대량 작업 기록 접근](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048945809/original/mlZTGjMTlnbUkqaopGj6xve3u7ju1WDvWQ.png?1750959659)

![대량 작업 기록 페이지](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048945842/original/3TDLRzHvQIiSfqLW7mMQh6QgaeRbvw432g.png?1750959727)
