## 하위 계정 목록 개편이란?

하위 계정 목록 개편은 HighLevel의 $97 및 $297 플랜 에이전시를 위한 하위 계정 관리 페이지의 대대적인 업데이트입니다. 이 업데이트는 모듈을 HighLevel의 마이크로앱 프레임워크로 마이그레이션하여 시각적으로 새로워진 인터페이스, 향상된 성능, 새로운 관리 도구를 도입합니다. 그 결과 모든 클라이언트 계정을 한 곳에서 관리하는 더욱 효율적이고 확장 가능하며 사용자 친화적인 방법을 제공합니다.

## 하위 계정 목록 개편의 주요 장점

재설계된 하위 계정 목록은 에이전시가 클라이언트 계정을 더욱 효율적으로 관리할 수 있도록 몇 가지 개선사항을 제공합니다:

- **현대적이고 통합된 UI**: 모든 플랜 티어에서 업데이트된 타이포그래피, 간격, 아이콘, 레이아웃으로 시각적으로 일관된 경험을 즐기세요.
- **더 빠른 성능**: 빠른 페이지 로드와 부드러운 상호작용으로 일상적인 관리 작업에 소요되는 시간을 단축합니다.
- **고급 필터링**: 직관적인 필터 컨트롤을 사용하여 특정 하위 계정을 빠르게 찾으세요.
- **일괄 작업**: 여러 하위 계정에 대해 한 번에 작업을 수행하여 반복적인 업무를 간소화합니다.
- **반응형 디자인**: 데스크톱, 태블릿, 모바일 화면에 최적화된 레이아웃으로 모든 기기에서 계정을 원활하게 관리하세요.
- **미래 지향적 인프라**: 하위 계정 태그 지정 및 컬럼 커스터마이징과 같은 향후 기능의 빠른 출시를 가능하게 하는 마이크로앱 아키텍처의 혜택을 누리세요.

## 하위 계정 목록의 고급 필터링

HighLevel의 새로운 하위 계정 목록 UI에는 이제 $97 및 $297 플랜의 에이전시 사용자가 사용할 수 있는 고급 필터 옵션이 포함되어 있습니다. 이 필터는 특정 플랫폼 기능, 구성 또는 상태에 따라 하위 계정을 빠르게 찾는 데 도움이 됩니다.

### 단계 1: 기본 필터 추가

- 하위 계정 목록 상단의 **Filter(필터)** 버튼(깔때기 아이콘)을 클릭하세요.
- 고급 필터 패널이 열리며 4개의 필터 카테고리가 표시됩니다:
  - **General(일반)** (예: Date Added(추가 날짜), Snapshots(스냅샷))
  - **Reselling(리셀링)** (예: Listings(리스팅), WhatsApp(왓츠앱), Email(이메일), AI tools(AI 도구))
  - **Rebilling(재청구)** (예: Email/Phone Verification(이메일/전화 인증), LC Premium(LC 프리미엄))
  - **LeadConnector** (예: LC Email(LC 이메일), LC Phone(LC 전화))
- 위 4개 카테고리 중 하나에서 원하는 검색 필드를 클릭하세요.
- 해당 검색 필드에 대해 필터링할 조건을 선택하세요.
- **+ Add Filter(+ 필터 추가)** 버튼을 클릭하여 OR 로직 필터를 추가하세요.

**OR 로직**: 결과가 표시되려면 최소한 하나의 조건이 참이어야 합니다. 조건 중 하나라도 일치하는 데이터가 포함되도록 필터가 적용됩니다.
예시: (LC Email이 활성화됨 또는 LC Phone이 활성화됨)으로 필터링하면 LC Email 또는 LC Phone 중 하나(또는 둘 다)가 활성화된 하위 계정이 반환됩니다.

![기본 필터 추가 과정](https://jumpshare.com/v/KHp5zkTNQIpoB2hInU9C+/GIF+Recording+2025-07-16+at+9.09.17+PM.gif)

### 단계 2: 중첩 필터 추가 (선택사항)

- **+ Add Nested Filter(+ 중첩 필터 추가)** 버튼을 클릭하세요.
- 이 필터는 AND 로직으로 작동합니다.

**And 로직**: 결과가 표시되려면 모든 조건이 참이어야 합니다. 필터가 결합되어 데이터가 모든 지정된 조건과 일치해야 합니다.
예시: (1월 1일 이후 추가됨 그리고 LC Email이 활성화됨)으로 필터링하면 1월 1일 이후에 추가되고 LC Email도 활성화된 하위 계정만 표시됩니다.

![중첩 필터 추가 과정](https://jumpshare.com/v/yBrNDQAwVcwAGZKCHts6+/GIF+Recording+2025-07-16+at+9.27.42+PM.gif)

### 단계 3: 필터 결과 적용 및 검토

- **Apply(적용)** 버튼을 클릭하여 선택한 필터를 실행하세요.
- 조건과 일치하는 반환된 하위 계정을 검토하세요.
- 일치하는 결과가 없으면 HighLevel에서 로케이션을 찾을 수 없다는 메시지를 표시합니다.
- 필터가 적용된 후 HighLevel은 목록 페이지에서 활성 필터 상태를 계속 표시하므로 사용 중인 조건을 빠르게 확인할 수 있습니다.

![필터 결과 적용](https://jumpshare.com/v/7vhv6D9Z9MaLL00FoGPV+/Screen+Shot+2025-07-16+at+9.32.58+PM.png)

### 단계 4: 적용된 필터 검토 및 관리

표시되는 필터 컨트롤을 통해 특정 하위 계정이 목록에 나타나는 이유를 더 쉽게 이해할 수 있습니다. 이를 통해 매번 전체 필터 워크플로우를 다시 열지 않고도 결과를 더 빠르게 세분화할 수 있습니다.

- HighLevel은 현재 활성화된 필터에 대한 명확한 시각적 표시기를 보여줍니다.
- 적용된 필터 영역에서 목록을 구성하는 필터 기준을 빠르게 미리 볼 수 있습니다.
- 단일 작업으로 필터를 제거하거나 조정하여 결과를 더 효율적으로 세분화할 수 있습니다.

![적용된 필터 관리](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155067709244/original/uRXoHZ4zt9XDHBPFROYMJTeWPBSQ4E4cOg.png?1774444458)

## 하위 계정 관리를 위한 일괄 작업

일괄 작업을 통해 에이전시는 여러 하위 계정을 한 번에 효율적으로 관리하여 반복적인 수동 작업을 줄일 수 있습니다.

- 체크박스를 사용하여 개별 하위 계정을 선택하거나 **Select all(모두 선택)** 체크박스를 클릭하세요.
- 원하는 하위 계정을 선택한 후 **Bulk Actions(일괄 작업)** 드롭다운을 클릭하세요.
- 사용 가능한 두 가지 일괄 작업이 있습니다: **Pause Sub-Account(하위 계정 일시정지)** 및 **Enable Rebilling(재청구 활성화)**.
- **Bulk Action History(일괄 작업 기록)** (시계) 아이콘을 클릭하여 이전 일괄 작업 로그를 확인하세요.

## 자주 묻는 질문

**Q: 새로운 하위 계정 목록이 모든 HighLevel 사용자에게 제공되나요?**

개편된 하위 계정 목록은 $97 및 $297 플랜의 에이전시에서 사용할 수 있으며, $497 Pro 사용자에게는 이미 표준으로 제공됩니다.

**Q: 새로운 UI를 활성화하면 기존 하위 계정이나 데이터에 영향을 미치나요?**

아니요, 업데이트는 인터페이스와 관리 도구만 변경합니다. 하위 계정 데이터는 변경되지 않습니다.

**Q: 새로운 하위 계정 목록이 모바일 친화적인가요?**

네, 업데이트된 UI는 완전히 반응형이며 모든 화면 크기에 최적화되어 있습니다.

**Q: 새로운 하위 계정 목록에 접근하기 위해 추가 비용을 지불해야 하나요?**

아니요, 이 기능은 기존 $97 또는 $297 플랜에 포함되어 있습니다.
