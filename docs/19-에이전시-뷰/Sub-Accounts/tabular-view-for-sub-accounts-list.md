# 하위 계정 목록 표 형태 보기란?

표 형태 보기(Tabular View)는 하위 계정(Sub-Accounts) 페이지를 카드 레이아웃에서 스프레드시트 스타일의 표로 변환합니다. 각 하위 계정은 행으로 표시되고, 열에는 이름, 상태, 생성일, 플랜 등의 세부 정보가 표시됩니다. 에이전시 사용자는 카드와 표 레이아웃을 언제든지 전환할 수 있어, 개괄적인 정보를 스캔하거나 정확성이 중요할 때 정렬 가능한 데이터로 자세히 살펴볼 수 있는 유연성을 제공합니다. 표 형태 보기는 에이전시 소유자가 정렬과 필터링을 사용하여 특정 하위 계정을 빠르게 찾을 수 있게 해주어 소중한 시간을 절약합니다.

---

## 하위 계정 목록 표 형태 보기의 주요 장점

- **중앙 집중식 가시성:** 하나의 스크롤 가능한 그리드에서 모든 하위 계정을 확인할 수 있습니다.

- **운영 효율성:** 정렬과 필터링으로 에이전시 소유자가 특정 하위 계정을 빠르게 찾을 수 있어 소중한 시간을 절약합니다.

- **유연성:** 워크플로우에 따라 시각적 또는 구조화된 레이아웃 중 선택할 수 있습니다.

- **확장 가능한 관리:** 5개든 500개든 로케이션을 관리하더라도 그리드는 빠르게 스캔하고 상호작용할 수 있습니다.

- **데이터 기반 인사이트:** 구조화된 가시성으로 에이전시 소유자는 트렌드를 파악하고 이상 징후를 발견하며 클라이언트 기반 전체에서 더 스마트한 결정을 내릴 수 있습니다.

## 카드 보기와 표 형태 보기 간 전환

Hyperclass는 하위 계정(Sub-Accounts) 페이지에서 두 가지 교체 가능한 레이아웃을 제공합니다:

- 카드 보기(Card View): 시각적 스냅샷(로고, 빠른 통계)에 최적
- 표 형태 보기(Tabular View): 데이터 비교, 일괄 편집, 정렬에 최적

보기를 변경하려면 다음 단계를 따르세요:

#### 1단계: 하위 계정으로 이동

에이전시 뷰 대시보드에서 하위 계정(Sub-Accounts) 페이지로 이동합니다.

![하위 계정 페이지 이동](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055442804/original/oZ7Pfq2Kc8u3g1rKQHshc41UaTXcxFFogQ.png?1759849665)

#### 2단계: 레이아웃 변경

하위 계정 페이지 우측 상단의 레이아웃 버튼(카드/표 보기 전환)을 사용하여 즉시 전환할 수 있습니다. 마지막으로 사용한 보기는 사용자별로 기억되므로 항상 선호하는 워크플로우로 돌아갑니다.

![레이아웃 전환 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055442645/original/idzLmME1nId8N7I-MpqD5I5E_u9Y_sAFdg.png?1759849614)

## 표 형태 보기에서 하위 계정 정렬 및 필터링

정렬 기능은 원클릭 열 순서 정렬을 제공하여 이름을 알파벳순으로 정리하거나 생성일로 순위를 매기는 데 이상적입니다. 계정을 정렬하려면 정렬(Sort) 버튼을 클릭하고 드롭다운에서 기준이 될 카테고리를 선택하세요.

**가나다순/다나가순** 드롭다운을 사용하여 알파벳 순서와 역 알파벳 순서 중에서 선택할 수 있습니다.

## 표시 열 커스터마이징

표 형태 보기에서는 표시되는 열과 표시되지 않는 열을 선택하여 필요에 맞게 보기를 커스터마이징할 수 있습니다!

표시할 열을 선택하려면 열(Columns) 버튼을 클릭하세요. 그런 다음 드롭다운에서 체크박스를 사용하여 표시하고 싶은 열을 선택하세요.

## 자주 묻는 질문

**Q: 표 형태 보기로 전환하면 데이터에 영향을 주나요?**

아니요. 보기는 정보가 표시되는 방식만 변경합니다. 하위 계정 데이터는 그대로 유지됩니다.

**Q: 필터가 세션 간에 기억되나요?**

필터는 페이지를 나가면 리셋됩니다. 하지만 스마트 리스트는 쉬운 접근을 위해 상단 바에 남아있습니다.

**Q: 어떤 열이 나타날지 커스터마이징할 수 있나요?**

예. 열(Columns) 아이콘을 클릭하여 각 데이터 필드의 표시를 토글할 수 있습니다.

**Q: 표 보기에서 리포트를 내보낼 수 있나요?**

예! 표 보기에서 리포트를 내보내려면 우측 상단의 내보내기(Export) 버튼을 클릭하기만 하면 됩니다.
