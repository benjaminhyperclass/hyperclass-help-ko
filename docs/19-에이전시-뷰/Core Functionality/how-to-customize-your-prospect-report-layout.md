# 잠재고객 보고서 레이아웃 커스터마이징이란?

잠재고객 보고서 레이아웃 커스터마이징은 Hyperclass의 잠재고객 발굴(Prospecting) 도구를 통해 생성되는 마케팅 감사 보고서의 구조를 제어할 수 있는 기능입니다. 보고서 섹션을 활성화하거나 비활성화하고 순서를 변경하여 프레젠테이션 흐름을 더 잘 지원할 수 있습니다. 이를 통해 보고서가 에이전시의 영업 프로세스와 클라이언트의 관심 분야에 맞춰지면서도 일회성 조정에 대한 유연성을 유지할 수 있습니다.

## 잠재고객 보고서 레이아웃 커스터마이징의 주요 장점

보고서 레이아웃을 커스터마이징하면 잠재고객 보고서가 피치를 강화하고 클라이언트 우선순위에 맞는 방식으로 인사이트를 제시할 수 있습니다.

- **내러티브 제어:** 영업 대화를 지원하는 논리적 순서로 섹션을 배치하세요.
- **중요한 내용 강조:** 전체 점수(Overall Score)나 SEO 분석 같은 주요 인사이트를 상단으로 이동시키세요.
- **불필요한 섹션 제거:** 잠재고객에게 관련이 없는 기술 스택(Techno Stack) 같은 섹션을 숨기세요.
- **전역 기본값 설정:** 향후 모든 보고서에서 일관성을 위한 표준화된 레이아웃을 만드세요.
- **개별 보고서 커스터마이징:** 기본 구조에 영향을 주지 않고 일회성 조정을 하세요.
- **일관된 PDF 출력:** 섹션 표시 여부와 순서가 웹 보고서와 내보낸 PDF에 모두 적용됩니다.

## 전역 보고서 레이아웃 (모든 보고서의 기본값)

전역 레이아웃 설정은 향후 잠재고객 보고서에서 사용될 기본 구조를 정의합니다. 모든 영업 프레젠테이션에서 일관성을 원하는 에이전시에게 이상적입니다.

### 전역 보고서 레이아웃 설정에 접근하기

전역 레이아웃을 제대로 구성하면 새로운 보고서마다 선호하는 구조가 자동으로 적용됩니다.

- **Marketing(마케팅)**으로 이동하세요
- **Prospecting(잠재고객 발굴)**을 선택하세요
- **Prospect Accounts(잠재고객 계정)**을 클릭하세요
- **Settings(설정)** 탭을 여세요
- **Report Settings(보고서 설정)**을 선택하세요

![잠재고객 보고서 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155066407975/original/GFKSq3tMM8PsV2ie5q-yhUs8rhk3fa_N0A.png?1772837485)

이 화면에는 섹션 표시 여부와 순서를 제어할 수 있는 **Customize Report Layout(보고서 레이아웃 커스터마이징)** 패널이 표시됩니다.

### 섹션 표시 여부 조정하기

섹션 표시 여부를 제어하면 영업 초점에 맞지 않는 콘텐츠를 제거할 수 있습니다.

- 기본적으로 모든 섹션이 활성화되어 있습니다
- 보고서에 포함하지 않을 섹션의 체크를 해제하세요
- 언제든지 섹션을 다시 활성화할 수 있습니다

예시: 에이전시가 SEO와 평판 관리 서비스에만 집중한다면 **Techno Stack(기술 스택)**을 숨기세요.

### 섹션 순서 변경하기

섹션 순서를 변경하면 전략적 순서로 잠재고객을 인사이트를 통해 안내할 수 있습니다.

- 각 섹션 옆의 드래그 앤 드롭 핸들을 사용하세요
- 표시하고 싶은 순서대로 섹션을 배치하세요

예시: **Overall Score(전체 점수)**를 상단으로 이동하여 영향 지표를 즉시 보여주세요.

### 전역 레이아웃 저장하기

저장하면 구조가 표준 레이아웃을 사용하는 보고서의 기본값이 됩니다.

- **Save(저장)**를 클릭하세요
- 업데이트된 구조가 향후 보고서의 기본 레이아웃이 됩니다

**중요사항:**
- **Save = 전역 기본 레이아웃**
- 이전에 커스터마이징된 보고서는 자동으로 업데이트되지 않습니다

언제든지 이 설정으로 돌아와서 섹션을 업데이트, 순서 변경, 포함 또는 제외할 수 있습니다.

## 특정 보고서만 커스터마이징하기

단일 잠재고객을 위해 다른 구조가 필요할 때, 전역 설정을 변경하지 않고 해당 보고서만 조정할 수 있습니다.

### 보고서 내부의 섹션 제어 기능에 접근하기

보고서별 커스터마이징은 표준 템플릿을 보존하면서 독특한 영업 상황에 유연성을 제공합니다.

- 원하는 **Marketing Audit Report(마케팅 감사 보고서)**를 여세요
- 우측 상단의 **Sections(섹션)**을 클릭하세요

![섹션 제어 패널](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155066407982/original/opzFhjGd_zYjgQGss0bxplmTwWWVuCzDPA.jpeg?1772837513)

섹션 패널에서 다음을 할 수 있습니다:

- 섹션을 체크하거나 체크 해제하여 표시 여부를 제어하세요
- 섹션을 드래그하여 순서를 변경하세요
- **Apply(적용)**를 클릭하여 변경사항을 확인하세요

**Apply = 이 보고서에만 변경사항 적용**

전역 레이아웃은 그대로 유지됩니다.

## 잠재고객 보고서 레이아웃 커스터마이징 설정 방법

레이아웃을 올바르게 설정하면 보고서가 에이전시의 포지셔닝과 우선순위를 일관되게 반영할 수 있습니다.

### 전역 기본 레이아웃 설정하기

- **Marketing(마케팅) → Prospecting(잠재고객 발굴) → Prospect Accounts(잠재고객 계정)**으로 이동하세요
- **Settings(설정)**을 클릭하세요
- **Report Settings(보고서 설정)**을 선택하세요
- 필요에 따라 섹션을 활성화하거나 비활성화하세요
- 섹션을 원하는 순서로 드래그하세요
- **Save(저장)**을 클릭하세요

### 개별 보고서 커스터마이징하기

- 특정 **Marketing Audit Report(마케팅 감사 보고서)**를 여세요
- **Sections(섹션)**을 클릭하세요
- 섹션의 순서를 변경하거나 숨기세요
- **Apply(적용)**를 클릭하세요

### 변경사항 확인하기

- 변경사항이 즉시 보이지 않으면 보고서 내에서 새로고침을 클릭하세요
- 보고서 생성 타임스탬프를 확인하세요
- 구조 일관성을 확인하기 위해 Export PDF(PDF 내보내기)를 사용하여 보고서를 내보내세요
- 최신 점수, 감지, 분석 업데이트를 로드하기 위해 주요 감사 개선사항 후 기존 보고서를 새로고침하세요

참고: 일부 보고서 개선사항은 백엔드 처리를 통해 적용됩니다. 기존 보고서에서 업데이트된 결과가 나타나려면 수동 새로고침이 필요할 수 있습니다.

섹션 순서와 표시 여부는 다음에 적용됩니다:
- 웹 기반 보고서 보기
- 공유된 보고서 링크
- 내보낸 PDF 버전

## 자주 묻는 질문

**Q: 전역 레이아웃 변경사항이 모든 보고서에 영향을 주나요?**

전역 변경사항은 표준 구조를 사용하는 향후 보고서의 기본 레이아웃이 됩니다.

**Q: 한 보고서에만 섹션을 숨길 수 있나요?**

네. 해당 특정 보고서 내의 **Sections(섹션)** 패널을 사용하고 **Apply(적용)**를 클릭하세요.

**Q: 전역 레이아웃을 변경하면 이전에 커스터마이징된 보고서도 업데이트되나요?**

아니요. 개별적으로 커스터마이징된 보고서는 자체 레이아웃 설정을 유지합니다.

**Q: 섹션 순서가 내보낸 PDF에도 영향을 주나요?**

네. 섹션 순서와 표시 여부는 웹 보고서와 내보낸 PDF 모두에 적용됩니다.

**Q: 실수로 섹션을 제거했다면 어떻게 하나요?**

전역 레이아웃 설정이나 보고서의 섹션 패널로 돌아가서 섹션을 다시 활성화하세요.

**Q: 레이아웃 변경사항이 나타나지 않는 이유는 무엇인가요?**

**Save(저장)**(전역) 또는 **Apply(적용)**(개별 보고서)를 클릭했는지 확인한 후 **Refresh(새로고침)**를 클릭하세요. 문제가 계속되면 브라우저 캐시를 삭제해보세요.
