# 잠재 고객 삭제하는 방법 (에이전시 뷰)

잠재 고객 발굴 도구에서 잠재 고객을 삭제하려면, 'Agency View(에이전시 뷰)'로 이동 > 'Prospecting(잠재고객 발굴)'을 클릭 > 삭제하고 싶은 잠재 고객을 검색 > 확인 메시지 완료 (이 단계는 되돌릴 수 없습니다)
