# 감사 보고서 섹션 커스터마이징 단계별 가이드

보고서를 작성하거나 커스터마이징할 때는 어떤 정보를 어떤 순서로 제시할지 통제하는 것이 중요합니다. 이 가이드는 보고서의 특정 섹션을 선택하고 원하는 순서대로 재배열하는 방법을 알려드립니다.

목차

- [1. 보고서 커스터마이징 접근](#1-보고서-커스터마이징-접근)
- [2. 보고서 섹션 선택](#2-보고서-섹션-선택)
- [3. 보고서 섹션 순서 변경](#3-보고서-섹션-순서-변경)
- [4. 보고서 저장 및 생성](#4-보고서-저장-및-생성)
- [보고서 커스터마이징 팁](#보고서-커스터마이징-팁)

보고서 섹션의 순서는 외부 보고서와 PDF 모두에 적용됩니다.

### 1. 보고서 커스터마이징 접근

- 보고서를 생성하거나 관리하는 대시보드로 이동합니다.
- 커스터마이징하고자 하는 보고서를 선택합니다.
- 보고서 레이아웃 편집을 위해 "Sections" 버튼을 클릭합니다.

### 2. 보고서 섹션 선택

- 보고서에 포함하고 싶은 섹션 옆의 체크박스를 선택합니다.
- 특정 보고서에 필요한 만큼 섹션을 많거나 적게 선택할 수 있습니다.

![보고서 섹션 선택 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155035142278/original/ect_vUzfPw9TUPUPeiPUrQs4tfekrxWmiw.png?1729574592)

### 3. 보고서 섹션 순서 변경

- 섹션을 선택한 후, 원하는 레이아웃에 맞게 순서를 재배열할 수 있습니다.
- 섹션 이름에 마우스를 올리면 나타나는 드래그 핸들을 클릭하고 누릅니다.
- 섹션을 위아래로 끌어서 순서를 재배열합니다.
- 원하는 위치에 도달하면 섹션을 놓습니다.

### 4. 보고서 저장 및 생성

- 모든 설정이 완료되면 "Apply"를 클릭하여 섹션 선택과 순서를 확정합니다.
- 이제 평소와 같이 보고서를 생성하면 커스터마이징한 레이아웃이 반영됩니다.

![보고서 적용 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155035142340/original/TePCxG70pK5LUTEaOA647FhO3EgvGCeC3A.png?1729574684)

## 보고서 커스터마이징 팁:

- **중요한 정보 우선순위 부여**: 가장 관련성이 높거나 임팩트 있는 섹션을 상단에 배치하여 주목도를 높이세요.
- **논리적 흐름 사용**: 개괄적인 지표부터 세부적인 인사이트까지 일관된 스토리를 전달하는 방식으로 섹션을 구성하세요.
- **간결한 보고서 유지**: 가치를 더하는 섹션만 포함하여 독자가 너무 많은 정보에 압도되지 않도록 하세요.

이러한 단계를 따르면 특정 요구사항에 맞게 보고서를 쉽게 맞춤화할 수 있으며, 유익하고 잘 구조화된 보고서를 만들 수 있습니다.
