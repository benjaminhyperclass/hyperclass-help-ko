# 에이전시 대시보드: 요약 탭

에이전시 대시보드 요약 탭이 HighLevel 기반 소스만을 사용하여 수익과 MRR을 계산하는 방법과, 이것이 요약 탭의 KPI 카드와 차트에 미치는 영향을 알아보세요.

에이전시 대시보드 요약 탭은 HighLevel 기반 수익 소스만을 포함하는 정교한 계산을 사용하여 수익과 MRR을 보여줍니다. 이를 통해 요약 지표의 정확성과 일관성이 향상됩니다.

HighLevel > Agency > Agency Dashboard(에이전시 대시보드)로 이동하세요. **요약 탭**은 다음 소스의 수익과 MRR을 포함합니다:

- SaaS
- 리셀링
- 리빌링

참고: HighLevel이 이제 외부 또는 자격을 갖추지 못한 소스를 필터링하므로, 이전 화면과 값이 다를 수 있습니다.

![에이전시 대시보드 요약 탭](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155064031978/original/vokx9qc6gbrZ2PDP0Hgyt8JX7yFF90PAqQ.png?1770070237)

**KPI 카드**

이 카드들은 이제 HighLevel 기반의 적격 수익만을 반영합니다:

- 총 수익 (지난 달)
- 월 경상수익(MRR)
- 신규 고객
- 총 고객

**성장률 차트**

성장률 차트는 업데이트된 수익 및 MRR 계산과 일치합니다.

**수익 분포 (지난 달)**

수익 분포(지난 달) 차트는 지난 달 수익을 다음과 같이 분류합니다:

- SaaS
- 리셀링
- 리빌링

---

**자주 묻는 질문**

**Q 수익이나 MRR 수치가 이전과 다르게 보이는 이유는 무엇인가요?**

A 요약 탭은 이제 HighLevel 기반 소스(SaaS, 리셀링, 리빌링)만 포함합니다. 이를 통해 외부 노이즈가 제거되고 요약 지표 전반에서 일관성이 향상됩니다.
