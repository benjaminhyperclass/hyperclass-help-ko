# 프로스펙팅 도구: 웹사이트용 프로스펙팅 위젯 만들기

**중요:** 이 기능은 Hyperclass 에이전시 계정에서 프로스펙팅 도구의 유료 버전으로 업그레이드한 후에만 사용할 수 있습니다.

프로스펙팅 위젯은 웹사이트에 임베드할 수 있는 완전히 커스터마이징 가능한 리드 수집 폼입니다. 방문자가 즉석에서 마케팅 감사 리포트를 생성할 수 있게 해줍니다. 이제 여러 개의 위젯을 만들고, 필드 동작을 제어하며, 커스텀 브랜딩을 적용하고, 리드 후속 작업을 자동화할 수 있습니다 — 모두 프로스펙팅 대시보드 안에서 가능합니다.

방문자가 폼을 작성하면, 그들의 세부 정보가 자동으로 CRM에 추가되고, 분류를 위해 태그가 지정되며, 환영 메시지 발송, 파이프라인 배정, 육성 시퀀스 등록을 위한 워크플로우를 트리거할 수 있습니다 — 모든 폼 제출을 전환 준비된 기회로 바꿔줍니다(하위 계정에서 프로스펙팅을 수행하는 경우에만).

## 위젯 만들기 및 커스터마이징

"에이전시 뷰(Agency View)"의 "프로스펙팅(Prospecting)" 섹션으로 이동하거나 하위 계정의 마케팅(Marketing) 탭 아래에서 새 위젯을 만들 수 있습니다. ([하위 계정에서 프로스펙팅 활성화에 대해 자세히 알아보기](../Prospecting-Tool/prospecting-at-sub-accounts-complete-guide-benefits-setup.md))

위젯 커스터마이징: 폼 제목, 색상, 배경, 그리고 위젯의 테마(밝음 또는 어두움)까지 웹사이트에 맞게 커스터마이징할 수 있습니다.

새로운 기능으로 강화:

- 여러 웹사이트나 사용 사례를 위해 최대 9개의 서로 다른 위젯 생성
- 브랜드에 맞게 제출 버튼 색상 커스터마이징
- 어떤 입력 필드가 필수인지, 선택인지, 숨김인지 선택
- 리드가 비즈니스 정보를 직접 입력할 수 있도록 허용 (Google 목록 불필요)
- 폰트 선택, 레이아웃 조정, 배경 또는 헤더 이미지 업로드

## 웹사이트에 위젯 임베드하기

- 위젯 디자인과 설정에 만족하면 설정 탭에서 임베드 코드를 생성합니다.

- 웹사이트의 백엔드나 CMS에 로그인합니다. 이는 WordPress, Joomla, Drupal 또는 웹사이트가 구축된 다른 플랫폼일 수 있습니다.

- 프로스펙팅 도구 위젯을 임베드하고 싶은 페이지나 섹션으로 이동합니다. 전용 리드 생성 페이지, 사이드바 또는 다른 적절한 위치일 수 있습니다.

- 코드를 임베드한 후, 웹사이트의 페이지를 방문하여 프로스펙팅 도구 위젯이 제대로 표시되고 작동하는지 확인하세요. 다음 사항을 점검하세요:
  - 위젯이 오류 없이 로드되는지
  - 위젯의 크기가 페이지 레이아웃에 잘 맞는지
  - 위젯의 모든 인터랙티브 요소가 예상대로 작동하는지

## 리드와 자동화

위젯이 자동으로 작업하도록 하기 — 프로스펙트가 위젯 폼을 작성하면 (프리미엄 프로스펙팅이 활성화된 하위 계정 내에서):

- 연락처가 자동으로 연락처(Contacts)에 저장됩니다 (워크플로우 설정이 필요하지 않음)
- 리드에 태그가 지정됩니다 (예: "신규 프로스펙트 수신됨")
- 워크플로우를 트리거할 수 있습니다:
  - 환영 이메일 또는 SMS 발송
  - 기회 관리(Opportunities) 또는 파이프라인(Pipelines)에 추가
  - 장기 육성 시퀀스 시작

## 프로스펙트 대시보드에서 리드 모니터링

- 프로스펙팅 도구 대시보드에 로그인: 프로스펙팅 도구 계정에 접속합니다.
- 리드 세부 정보 보기: 프로스펙트 대시보드로 이동하여 위젯 폼을 작성한 리드들의 세부 정보를 확인합니다. 리드들은 이름 옆에 "Forms tag"가 표시됩니다.
- 리드와 연락: 리드가 제공한 연락처 정보와 비즈니스 세부 정보를 사용하여 연락을 취하고 성공적인 고객으로 전환시킵니다.

## 에이전시를 위한 혜택

- **향상된 리드 생성**: 브랜딩된 인터랙티브 폼을 사용하여 웹사이트에서 직접 고품질 리드를 획득 — 일반 방문자를 자격 있는 프로스펙트로 전환하는 데 도움을 줍니다.

- **간소화된 고객 획득**: 잠재 고객의 식별, 획득, 육성을 자동화하여 수동 작업을 줄이고 영업 사이클을 가속화합니다.

- **개선된 고객 참여**: 방문자가 에이전시와 상호작용하고 서비스를 더 탐색하도록 격려하는 개인화된, 가치 중심의 경험을 제공합니다.

- **데이터 기반 프로스펙팅**: 실시간 분석과 리포팅을 사용하여 리드 생성 전략을 최적화하고 더 스마트한 고객 획득 결정을 내립니다.

- **확장 가능한 성장**: 고객이나 캠페인 전반에 걸쳐 여러 위젯과 워크플로우를 쉽게 관리하여 에이전시 확장에 따라 프로스펙팅 프로세스를 적응 가능하게 만듭니다.

## SaaS 판매자를 위한 혜택

- **타겟팅된 아웃리치**: SaaS 제품이 제공하는 솔루션을 적극적으로 찾고 있는 잠재 고객을 식별하고 참여시킵니다.
- **높은 전환율**: 참여도와 관심 수준에 따라 리드의 우선순위를 정하여 가장 유망한 프로스펙트에 집중합니다.
- **간소화된 영업 프로세스**: 최초 연락부터 후속 작업까지 영업 프로세스의 핵심 부분을 자동화하여 영업팀이 거래 성사에 집중할 수 있게 합니다.
- **통찰력 있는 분석**: 프로스펙팅 노력에 대한 가치 있는 인사이트를 얻어 접근 방식을 개선하고 전반적인 효과를 향상시킬 수 있습니다.
- **향상된 고객 경험**: 잠재 고객에게 매끄럽고 개인화된 여정을 제공하여 SaaS 솔루션을 선택할 가능성을 높입니다.

프로스펙팅 위젯을 통해 단순히 리드를 수집하는 것이 아니라 — 경험을 창조하고, 신뢰를 구축하며, 성장을 자동화하는 것입니다.

오늘 위젯을 임베드하여 시작하고, 더 적은 마찰과 더 많은 가치로 트래픽을 전환 준비된 기회로 바꿔보세요.
