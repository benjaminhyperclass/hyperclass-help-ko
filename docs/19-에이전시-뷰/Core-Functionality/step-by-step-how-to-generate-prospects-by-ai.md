# AI로 잠재 고객 발굴하는 방법 (단계별 가이드)

AI 잠재 고객 발굴(Prospect by AI)은 선택한 니치와 타겟 지역을 기반으로 매주 2~3개의 고품질 잠재 고객을 자동으로 추천받는 스마트한 기능입니다. 지능형 점수 매기기와 데이터 신호를 활용하여, 에이전시와 로케이션이 수동 작업 없이도 더 나은 리드를 더 빠르게 찾을 수 있도록 도와줍니다.

활성화하면 AI가 설정한 니치와 위치 선호도를 분석하여 가장 적합한 비즈니스를 찾아내고, 아웃리치가 가능한 상태로 대시보드에 직접 전달합니다.

## 사용 가능 대상

- 모든 에이전시 – Labs(랩스)를 통해
- 모든 로케이션 – Labs(랩스)를 통해
→ Premium Prospecting(프리미엄 잠재 고객 발굴)을 구독한 경우 AI를 통해 프리미엄 리포트를 받을 수 있고, 그렇지 않으면 기본적으로 기본 리포트를 받습니다.

# 활성화 방법

## 에이전시에서

- 에이전시 내 Labs(랩스)로 이동
- "Prospect by AI" 토글을 ON으로 설정
- Prospecting(잠재 고객 발굴) → Settings(설정) → AI Report Configuration(AI 리포트 설정)으로 이동
- AI Recommendations(AI 추천) 토글을 ON으로 설정하고 저장
- 무료 플랜을 사용하는 경우: 이 AI 기능을 완전히 사용할 수 있어서 2~3개의 AI 생성 잠재 고객을 받을 수 있지만, 생성되는 리포트는 기본 리포트만 제공됩니다(프리미엄 리포트 아님)

## 하위 계정에서

- 하위 계정 내 Labs(랩스)로 이동
- "Prospect by AI" 토글을 ON으로 설정
- Marketing(마케팅) 탭 → Prospecting(잠재 고객 발굴) → Settings(설정) → AI Report Configuration(AI 리포트 설정)으로 이동 (하위 계정에서 Prospecting에 접근하려면 먼저 에이전시의 리셀링 탭에서 활성화해야 함을 기억하세요)
- 토글을 ON으로 설정

# 설정 방법

- 잠재 고객을 생성할 니치를 최대 3개까지 선택
- 국가, 도시, 지역, 우편번호를 설정하여 AI가 적절한 잠재 고객을 찾을 수 있도록 함
- 기본적으로 기본 리포트는 모든 사용자에게 제공되지만, 프리미엄 리포트를 생성하려면 먼저 프리미엄 잠재 고객 발굴에 구독해야 합니다

## 생성된 잠재 고객 확인 위치

- 새로운 잠재 고객은 Prospecting Dashboard(잠재 고객 발굴 대시보드)에 직접 나타납니다
- AI Recommended(AI 추천) 카드를 확인하세요 — 2~3일마다 새로 고침됩니다
- 새로운 추천이 준비되면 잠재 고객 대시보드를 열 때 알림 배지로 즉시 알려줍니다
