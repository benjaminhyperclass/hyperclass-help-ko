# 내 에이전시 플랜은 어떻게 확인하나요?

에이전시 플랜을 확인하려면 에이전시 레벨 뷰 → Settings(설정)로 이동하세요.

![에이전시 레벨 설정 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48157171439/original/an6ZhUhLiOcvkhej5iK4aAzc3i8TysOEyg.png?1636476058)

왼쪽 메뉴에서 Billing(결제/청구) 옵션을 클릭하면 다음과 같이 에이전시 플랜이 표시됩니다.

![에이전시 플랜 확인 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48157171985/original/XPyJ9RTQHGarnTwPx6keI1thTETnl8AyBQ.png?1636476173)

예를 들어, 위 이미지의 경우 해당 에이전시는 497 Pro 플랜을 사용하고 있습니다.
