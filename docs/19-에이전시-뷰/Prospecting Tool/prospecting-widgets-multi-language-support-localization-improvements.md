# 프로스펙팅 위젯: 다국어 지원 및 현지화 개선

프로스펙팅 위젯(Prospecting Widgets)이 이제 다국어 기능을 완전히 지원합니다. 필드 라벨, 검증 메시지, 제출 버튼, 이메일 커뮤니케이션, PDF 보고서 링크를 포함한 모든 시스템 생성 텍스트가 사용자의 플랫폼 언어 설정에 자동으로 적응합니다. 이를 통해 임베디드 위젯, 이메일, 내보낸 보고서 전반에서 일관된 현지화 경험을 제공합니다.

## 개요

프로스펙팅 툴에서 현지화(i18n) 지원을 강화했습니다.

이전에는 일부 백엔드 생성 문자열이 사용자의 언어 기본 설정에 동적으로 적응하지 않았습니다. 이번 업데이트로:

- 프로스펙팅 위젯이 방문자의 플랫폼 언어 설정에 맞게 표시됩니다
- 이메일 커뮤니케이션이 사용자 언어 설정을 존중합니다
- PDF 보고서 링크와 내보낸 콘텐츠가 플랫폼 언어 기본 설정을 따릅니다
- 추가 설정이 필요하지 않습니다

이 개선으로 고객 대면 구성 요소 전반에서 일관된 다국어 동작이 보장됩니다.

## 프로스펙팅 위젯 접근하기

다음 경로로 이동하세요:

Agency View(에이전시 뷰) → Prospecting(프로스펙팅) → Widgets(위젯)

여기서 위젯을 생성하고 관리할 수 있습니다.

![프로스펙팅 위젯 접근](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155064814687/original/PawKpUsQ2loETU0h17F4Q_TaAexiEgrm2A.jpeg?1770920759)

## 업그레이드 요구사항

계정이 업그레이드되지 않은 경우 업그레이드 안내가 표시됩니다.

## 다국어 지원

프로스펙팅 위젯이 이제 다국어를 완전히 지원합니다.

모든 시스템 생성 텍스트가 사용자의 플랫폼 언어 설정을 자동으로 존중합니다.

여기에는 다음이 포함됩니다:

- 필드 라벨
- 검증 메시지
- 시스템 프롬프트
- 제출 버튼 텍스트
- 오류 메시지
- 확인 메시지

수동 번역 설정은 필요하지 않습니다.

## 위젯 임베딩

- 위젯 내의 Settings(설정) 탭으로 이동
- 임베드 코드 복사
- 웹사이트 백엔드에 붙여넣기

![위젯 임베딩](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155064814684/original/quLJyQfDfNRN9i5ycN2uIPqyBmNDLxr4EA.jpeg?1770920758)

임베딩 후 다음을 확인하세요:

- 위젯이 올바르게 로드되는지
- 레이아웃이 제대로 표시되는지
- 모든 필드가 예상대로 작동하는지

## 워크플로우와 자동화

잠재 고객이 위젯을 제출하면:

- 연락처가 자동으로 저장됩니다
- 리드에 태그가 지정됩니다
- 워크플로우가 트리거될 수 있습니다

![워크플로우와 자동화](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155064814690/original/ppCD79tKN9ZXaa3QZ43ORN4Bs55lNyaPrA.jpeg?1770920759)

이를 통해 다음을 자동화할 수 있습니다:

- 환영 이메일
- 파이프라인 배정
- 육성 시퀀스

## PDF 보고서 언어 동작

내보낸 PDF 보고서가 이제 사용자의 플랫폼 언어 설정을 반영합니다.

다음에 적용됩니다:

- 보고서 버튼
- 링크 라벨
- 시스템 생성 텍스트
- 내보내기 메시지

![PDF 보고서 언어 동작](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155064814689/original/cygW_XGzUqgQOfn_f_y6tgYaojQ1vwv5vw.jpeg?1770920759)

## 이메일 전반의 언어 지원

프로스펙팅 위젯과 관련된 시스템 생성 이메일이 이제 수신자의 플랫폼 언어에 자동으로 적응합니다.

여기에는 다음이 포함됩니다:

- 제목 라인
- 행동 유도 버튼
- 시스템 지시사항

추가 설정은 필요하지 않습니다.

## 자주 묻는 질문

**Q: 위젯에 대한 다국어를 수동으로 활성화해야 하나요?**
아니요. 위젯이 플랫폼 언어에 자동으로 적응합니다.

**Q: 제가 수동으로 입력한 커스텀 텍스트도 변경되나요?**
아니요. 시스템 생성 문자열만 현지화됩니다.

**Q: 이메일 알림도 이제 다국어인가요?**
네. 시스템 생성 이메일이 사용자의 플랫폼 언어 설정을 따릅니다.

**Q: 내보낸 PDF 보고서도 자동으로 언어가 변경되나요?**
네. PDF 보고서 링크와 시스템 텍스트가 플랫폼 언어 설정을 반영합니다.

**Q: 프로스펙팅 툴을 업그레이드해야 하나요?**
네. 위젯은 유료 버전이 필요합니다.
