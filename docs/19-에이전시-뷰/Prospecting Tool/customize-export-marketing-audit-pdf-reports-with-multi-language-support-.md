# 마케팅 감사 PDF 보고서 커스터마이징 및 내보내기 (다국어 지원)

마케팅 감사 PDF에도 실시간 웹 보고서와 동일한 세련되고 브랜드화된 디자인을 적용하세요. Hyperclass 프로스펙팅 도구(Prospecting Tool)의 PDF 커스터마이징 기능으로 색상, 테두리, 제목, 연락처 정보를 제어하여 공유하는 모든 형식이 에이전시의 정체성을 강화할 수 있습니다.

## 마케팅 감사 보고서의 PDF 커스터마이징이란?

PDF 커스터마이징을 사용하면 프로스펙팅 보고서 빌더(Prospecting Report Builder)에서 설정한 브랜드 테마, 헤더, 푸터 설정을 PDF로 내보낸 감사 보고서에 바로 적용할 수 있습니다. 모든 PDF가 자동으로 웹 보고서의 브랜딩을 그대로 반영하므로 추가 디자인 작업이 필요 없습니다. 이를 통해 일관된 잠재 고객 경험을 만들고 설정 시간도 절약할 수 있습니다.

참고: 이 기능은 프리미엄 프로스펙팅 구독이 필요합니다. 미리보기 모드는 에이전시와 로케이션 모두에게 무료로 제공됩니다.

## PDF 커스터마이징의 주요 장점

- **일관된 브랜딩:** 웹 링크와 PDF가 항상 일치하여 통합되고 브랜드화된 경험을 제공
- **고객 발표 준비:** 잠재 고객과의 신뢰를 구축하는 전문적인 프레젠테이션
- **한 번 설정, 모든 곳에 동기화:** 보고서 빌더에서 설정하면 모든 보고서에 변경사항이 적용됨
- **브랜드 맞춤 스타일링:** 테두리와 제목 색상을 어떤 팔레트에든 맞게 커스터마이징
- **명확한 연락처 정보:** 표지와 푸터에 에이전시 정보를 표시하여 잠재 고객이 빠르게 연락할 수 있음

## 프로스펙팅 도구의 보고서 빌더에 접근하기

보고서 빌더(Report Builder)는 **프리미엄 프로스펙팅 구독**을 가진 사용자에게 **에이전시와 하위 계정 레벨 모두**에서 사용할 수 있습니다. 베이직 프로스펙팅 구독을 사용 중이라면 보고서 빌더를 탐색할 수는 있지만 변경사항 저장은 제한됩니다.

### 에이전시 레벨에서

- **에이전시** 계정에 로그인합니다.
- 메인 내비게이션에서 **Prospecting(프로스펙팅)** 탭을 클릭합니다.
- Report Builder 탭을 선택하여 보고서 커스터마이징을 시작합니다.

![Hyperclass 프로스펙팅 에이전시 접근](https://jumpshare.com/v/8pGNdQIymy3P4VACl5iU+/Screen+Shot+2025-07-07+at+9.42.04+PM.png)

### 하위 계정 레벨에서

- **하위 계정**에 로그인합니다.
- Marketing(마케팅) 탭을 클릭합니다.
- **Prospecting(프로스펙팅)**을 클릭합니다.
- **Report Builder** 탭을 클릭하여 보고서 구성을 시작합니다.

![Hyperclass 프로스펙팅 하위계정 접근](https://jumpshare.com/v/q2r18oyQVX3NUdA3iAiB+/Screen+Shot+2025-07-07+at+9.46.07+PM.png)

## PDF 커스터마이징 설정 및 사용 방법

적절한 설정을 통해 모든 보고서가 완벽하게 보이도록 하고 수동 편집을 하지 않아도 됩니다.

참고: 웹 전용 헤더/푸터 블록은 PDF에 나타나지 않습니다.

### 1단계: 브랜딩 및 테마 설정

- **브랜드 이름 및 연락처 정보** - 보고서 빌더에 에이전시 이름, 전화번호, 이메일, 웹사이트를 추가합니다.

![브랜드 정보 설정](https://jumpshare.com/share/DsQNQ3uFjWlSWeHCkniT+/Screenshot+2025-10-02+at+6.05.06%E2%80%AFPM+%281%29.png)

- 테두리 및 색상 - 브랜드에 맞게 테두리 색상을 커스터마이징합니다.
- 제목 색상 - 섹션 제목(Local Rank Tracker, Tech Stack Analysis 등)을 브랜드 색상에 맞춥니다.
- 보고서 커스터마이징에 대한 자세한 가이드는 [프로스펙팅 도구 - 완전히 커스터마이징 가능한 잠재 고객 보고서 빌더](prospecting-tool-fully-customizable-prospect-report-builder.md) 문서를 확인하세요.

![색상 및 테두리 설정](https://jumpshare.com/share/2WaHW6naTKrLmio6qq5k+/Screenshot+2025-10-02+at+6.04.06%E2%80%AFPM.png)

### 2단계: 보고서 미리보기 및 저장

- **Preview(미리보기)**를 클릭하여 보고서 디자인의 실시간 미리보기를 생성합니다.
- 테두리, 제목, 헤더/푸터 세부사항에 브랜드 색상이 적용되었는지 확인합니다.
- **Save(저장)**를 클릭하여 빌더에 선택사항을 적용하고 **향후 PDF 또는 웹 보고서**에 추가 단계 없이 적용되도록 합니다.

![보고서 미리보기 및 저장](https://jumpshare.com/share/QuqoMeQj55SuPUp13qG8+/GIF+Recording+2025-10-09+at+9.39.47+PM.gif)

### 3단계: PDF 보고서 내보내기

- **Prospecting(프로스펙팅)** → **All Accounts(전체 계정)** (잠재 고객 목록)으로 이동합니다.
- 원하는 잠재 고객에 대해 **View Report(보고서 보기)**를 클릭한 다음 **Export PDF**를 클릭합니다.

![보고서 보기 및 PDF 내보내기](https://jumpshare.com/share/WvjWQ0ghnPtZSrfIMN7a+/GIF+Recording+2025-10-09+at+9.50.06+PM.gif)

- 또는 목록에서 바로 Export PDF를 클릭하고 PDF 언어를 선택합니다. 내보낸 PDF 내의 모든 시스템 생성 텍스트(섹션 제목, 버튼 라벨, 보고서 메시지 포함)가 선택한 언어를 자동으로 반영합니다.
- **Export PDF**를 클릭합니다.

![다국어 PDF 내보내기](https://jumpshare.com/share/0SgZyl51esAFtS7qgnzQ+/GIF+Recording+2025-10-09+at+9.54.03+PM.gif)

- 준비가 완료되면 이전에 설정한 모든 브랜드 커스터마이징이 적용된 PDF 보고서가 새 탭에서 열립니다.

![완성된 PDF 보고서](https://jumpshare.com/share/NA3YCMEoKveas0EBBHuS+/GIF+Recording+2025-10-09+at+9.58.42+PM.gif)

참고: 더 많은 콘텐츠와 섹션이 포함된 보고서를 포함하여 마케팅 감사 보고서의 PDF 내보내기 안정성이 개선되었습니다. 오래된 보고서가 최신 감사 결과를 반영하지 않는다면 PDF를 내보내기 전에 보고서를 새로고침하세요.

## 자주 묻는 질문

**다국어 PDF 내보내기를 지원하나요?**
네. PDF를 내보낼 때 선호하는 언어를 선택할 수 있습니다. 보고서 내의 모든 시스템 생성 텍스트(제목, 프롬프트, 라벨 포함)가 선택한 언어에 자동으로 맞춰집니다.

**에이전시 로고가 PDF 헤더에서 올바르게 크기 조정되나요?**
네, 헤더 커스터마이징의 로고 필드는 최적의 인쇄 품질을 위해 최대 300px 폭까지 이미지를 자동 크기 조정합니다.

**이 기능은 프리미엄 프로스펙팅이 필요한가요?**
네.

**실시간으로 변경사항을 볼 수 있나요?**
네, 빌더는 실시간 미리보기를 제공하므로 게시하기 전에 조정 사항을 즉시 확인할 수 있습니다.
