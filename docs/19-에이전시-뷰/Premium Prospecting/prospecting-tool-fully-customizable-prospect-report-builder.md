# 프로스펙팅 도구 - 완전 맞춤형 리드 보고서 빌더

프로스펙팅 보고서(Prospecting Reports) 경험의 대대적인 업그레이드를 소개합니다: 완전히 새롭게 디자인된 웹 보고서 UI와 보고서의 모양과 느낌을 완전히 제어할 수 있는 강력한 보고서 빌더입니다. 이번 업데이트는 고객이 자신의 브랜드 스토리를 전달하고, 더 나은 첫인상을 만들며, 커스텀 브랜딩이 적용된 고전환 보고서로 더 많은 거래를 성사시키는 데 도움을 주는 것입니다.

참고: 이 기능은 프리미엄 프로스펙팅(Premium Prospecting) 구독이 필요합니다. 미리보기 모드는 에이전시와 하위 계정 모두에게 무료로 제공됩니다.

## 완전 맞춤형 리드 보고서 빌더란 무엇인가요?

이 프리미엄 기능은 사용자가 리드 보고서의 외관과 내용을 완전히 제어할 수 있도록 설계되었습니다. 에이전시와 하위 계정이 보고서의 브랜드 테마, 헤더, 소개 메시지, 푸터 섹션을 맞춤 설정하여 모든 보고서가 고유하고 전문적인 정체성을 반영할 수 있게 합니다. 사용하기 쉬운 인터페이스와 실시간 미리보기 기능으로, 멋진 외관은 물론 참여도를 높이는 보고서를 구축하고 개선할 수 있습니다.

UI 개선사항:

- 간소화된 헤더: 너무 많은 공간을 차지했던 기존 고정 헤더가 수정되었습니다.
- 스크롤 시 헤더가 더 이상 콘텐츠를 가리지 않습니다.
- 최적화된 레이아웃: 양쪽의 빈 여백을 제거했습니다.
- 보고서가 더 깔끔하고 컴팩트하며 탐색하기 쉬워졌고, 콘텐츠 가시성과 집중도가 향상되었습니다.

참고: 이 새로운 UI는 현재 웹 보고서에서만 사용할 수 있습니다.

## 완전 맞춤형 리드 보고서 빌더의 주요 장점

리드 보고서 빌더는 고객 보고서의 미적 품질과 기능적 품질을 모두 향상시키는 주요 장점들을 제공합니다.

- 브랜딩된 전문적인 보고서: 고객과의 신뢰를 강화하는 일관성 있는 브랜드 경험을 유지합니다.
- 전환에 최적화된 레이아웃: 최대한의 임팩트를 위해 설계된 레이아웃에서 중요한 CTA와 연락처 정보를 강조합니다.
- 개인화된 메시지: 타겟 청중에게 공감을 불러일으키는 틈새 집중적인 정보로 콘텐츠를 맞춤화합니다.
- 실시간 미리보기: 변경 사항을 즉시 확인하고 방해받지 않고 자신 있게 발행합니다.

## 프로스펙팅 도구에서 보고서 빌더 접근하기

보고서 빌더는 프리미엄 프로스펙팅 구독이 있는 사용자를 위해 에이전시와 하위 계정 레벨에서 모두 사용할 수 있습니다. 기본 프로스펙팅을 사용하는 경우, 보고서 빌더를 탐색할 수는 있지만 변경 사항 저장이 제한됩니다.

### 에이전시 레벨에서

- 에이전시 계정에 로그인합니다.
- 메인 네비게이션에서 Prospecting(프로스펙팅) 탭을 클릭합니다.
- Report Builder(보고서 빌더) 탭을 선택하여 보고서 맞춤 설정을 시작합니다.

![프로스펙팅 탭에서 보고서 빌더 접근](https://jumpshare.com/v/8pGNdQIymy3P4VACl5iU+/Screen+Shot+2025-07-07+at+9.42.04+PM.png)

### 하위 계정 레벨에서

- 하위 계정에 로그인합니다.
- Marketing(마케팅) 탭을 클릭합니다.
- Prospecting(프로스펙팅)을 클릭합니다.
- Report Builder(보고서 빌더) 탭을 클릭하여 보고서 작성을 시작합니다.

![하위 계정에서 보고서 빌더 접근](https://jumpshare.com/v/q2r18oyQVX3NUdA3iAiB+/Screen+Shot+2025-07-07+at+9.46.07+PM.png)

## 리드 보고서 빌더를 사용하여 보고서 맞춤 설정하기

보고서 빌더는 4개의 탭으로 나뉘어 있습니다. 탭을 클릭하고 편집을 시작하세요. 오른쪽에서 실시간 미리보기를 볼 수 있습니다.

### 브랜드 테마

이 섹션을 사용하면 보고서 전반에서 브랜드의 시각적 정체성을 정의할 수 있습니다. 이를 사용하여 보고서를 브랜드의 웹사이트, 로고 또는 마케팅 자산과 일치시키세요.

맞춤 설정할 수 있는 항목:

- Primary Color(기본 색상): 보고서 전반에 사용되는 메인 배경 또는 기본 색상(예: 섹션의 배경)
- Secondary Color(보조 색상): 기본 색상을 보완하여 미묘하게 사용되는 지원 액센트 색상
- Report Text Color(보고서 텍스트 색상): 보고서 콘텐츠 내 메인 텍스트 색상을 제어
- Header Link Color(헤더 링크 색상): 헤더의 클릭 가능한 링크(웹사이트 URL 등) 색상 설정
- CTA Background Color(CTA 배경 색상): "지금 연결" 또는 "지금 문의" 같은 CTA 버튼의 색상
- CTA Text Color(CTA 텍스트 색상): CTA 버튼 안에 표시되는 텍스트의 색상

![브랜드 테마 설정](https://jumpshare.com/v/9iQKsrm0nQSlAV5aJiRT+/Screen+Shot+2025-07-07+at+10.53.16+PM.png)

### 헤더 맞춤 설정

이 섹션에서는 보고서 헤더에 표시되는 요소를 제어할 수 있습니다.

#### 1. 헤더 표시 옵션

헤더에서 어떤 요소가 보이는지 제어합니다.

- Show Header Logo(헤더 로고 표시): 비즈니스 로고를 표시/숨기기 전환
- Show Header Website(헤더 웹사이트 표시): 비즈니스 웹사이트 링크 표시/숨기기
- Show Header Description(헤더 설명 표시): 태그라인이나 지원 텍스트 같은 짧은 문구(예: "감사에 도움이 필요하신가요?")
- Show Header Email / Contact(헤더 이메일/연락처 표시): 연락처 이메일과 전화번호 표시 여부 선택
- Show Header CTA Button(헤더 CTA 버튼 표시): "지금 문의" 같은 메인 액션 버튼 활성화/숨기기

![헤더 표시 옵션 설정](https://jumpshare.com/v/4EgCMVTaFvbagAkrIOMn+/GIF+Recording+2025-07-07+at+11.16.57+PM.gif)

#### 2. 헤더 디자인 옵션

- Font Family(글꼴 패밀리): 글꼴 유형 선택(예: Arial, Roboto)
- Font Weight(글꼴 굵기): 글꼴 두께 설정(보통, 중간, 굵게)
- Font Style(글꼴 스타일): 일반 또는 기울임체

![헤더 디자인 옵션 설정](https://jumpshare.com/v/FAwJnuZp9bRgowJ51l4X+/GIF+Recording+2025-07-07+at+11.38.30+PM.gif)

- Upload Logo(로고 업로드): 권장 크기 80x80px
- Header Title(헤더 제목): 로고 옆에 표시되는 제목 텍스트
- Header Description(헤더 설명): 제목을 위한 지원 문구
- Header Website(헤더 웹사이트): 클릭 가능한 링크로 표시되는 URL
- Header Email(헤더 이메일): 연락처 이메일 주소
- Header Phone Number(헤더 전화번호): 비즈니스 연락처 번호
- CTA Text(CTA 텍스트): 버튼에 표시될 텍스트(예: "지금 예약")
- CTA Action(CTA 액션): 누군가 클릭할 때 발생하는 동작 선택:
  - URL: 리드를 예약 페이지나 웹사이트로 이동
  - Email(이메일): 연락처로 이메일 초안 열기
  - Call(전화): 전화번호로 전화 걸기(모바일 기기)

![헤더 콘텐츠 설정](https://jumpshare.com/v/SI7C3Ji7xUlXvjBLzvcj+/GIF+Recording+2025-07-07+at+11.42.43+PM.gif)

#### 3. 헤더 스타일

세 가지 사용 가능한 옵션에서 헤더 스타일을 선택합니다.

- Left Top Wave(좌측 상단 웨이브)
- Full Background(전체 배경)
- Double Wave Left(더블 웨이브 좌측)

![헤더 스타일 선택](https://jumpshare.com/v/zfneT7kAeyFgSrzdFIzP+/GIF+Recording+2025-07-07+at+11.54.32+PM.gif)

### 소개 맞춤 설정

이 섹션은 헤더 아래에 표시되는 소개 콘텐츠를 구성하는 데 도움이 됩니다. 이 공간을 사용하여 감사의 가치와 식별된 격차를 수정하는 데 어떻게 도움이 될 수 있는지 설명하세요.

- Show Introduction Description(소개 설명 표시): 전체 소개 블록을 표시/숨기기 전환
- Introduction Font Family, Weight, and Style(소개 글꼴 패밀리, 굵기, 스타일): 소개 단락 텍스트의 모양 맞춤 설정
- Introduction Title(소개 제목): 이 섹션의 내용을 설명하는 짧은 헤드라인
- Introduction Description(소개 설명): 맥락을 제공하는 텍스트, 보고서가 다루는 내용과 중요한 이유를 리드에게 교육하는 데 유용

#### 1. 푸터 표시 옵션

푸터에서 어떤 요소가 보이는지 제어합니다.

- Show Footer Description(푸터 설명 표시): 푸터 소개 블록을 표시/숨기기 전환
- Show Footer CTA Button(푸터 CTA 버튼 표시): 푸터 CTA 버튼을 표시/숨기기 전환

### 미리보기 및 저장

- Preview(미리보기)를 클릭하면 새 탭에서 가장 최근의 저장되지 않은 버전의 전체 보고서 보기가 열립니다.
- Save(저장)를 클릭하면 과거 및 향후 모든 보고서에 변경 사항이 적용됩니다.
- Reset to Default(기본값으로 재설정)를 사용하여 맞춤 설정을 되돌릴 수도 있습니다.

## 자주 묻는 질문

**Q: 누가 리드 보고서 빌더에 접근할 수 있나요?**

이 기능은 프리미엄 액세스를 통해 모든 에이전시와 하위 계정에서 사용할 수 있으며, 미리보기 모드는 둘 다 무료입니다.

**Q: 프리미엄 구독에는 무엇이 포함되나요?**

프리미엄 구독은 브랜드 테마, 헤더, 소개, 푸터 설정을 포함한 완전한 맞춤 설정 옵션과 외부 보고서 UI 개편을 제공합니다.

**Q: 변경 사항을 실시간으로 볼 수 있나요?**

네, 빌더는 실시간 미리보기를 제공하여 발행 전에 즉시 조정 사항을 확인할 수 있습니다.

**Q: 외부 UI 개편으로 어떤 개선 사항이 있었나요?**

개편으로 일관된 아이콘과 개선된 텍스트 스타일링으로 더 깔끔하고 현대적인 레이아웃을 제공하여 보고서를 더 시각적으로 매력적으로 만들었습니다.
