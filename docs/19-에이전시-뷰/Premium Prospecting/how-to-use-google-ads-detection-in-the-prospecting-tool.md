# 프로스펙팅 도구에서 구글 광고 탐지 기능 사용하는 방법

프로스펙팅 도구(Prospecting Tool)에 새로운 기능을 추가했습니다! 이제 잠재 고객이 구글 광고(Google Ads)를 운영하고 있는지 확인할 수 있는 새로운 기능을 제공합니다.

이 정보는 이제 감사 보고서에 표시되어, 에이전시가 잠재 고객의 광고 활동에 대한 유용한 인사이트를 얻을 수 있습니다.

주요 혜택:

- **향상된 인사이트**: 잠재 고객이 구글 광고를 적극적으로 운영하고 있는지 파악하여, 그들의 광고 투자에 대한 더 명확한 그림을 제공합니다.
- **새로운 영업 기회**: 구글 광고 사용을 탐지하면 타겟팅된 광고 솔루션으로 잠재 고객에게 접근할 수 있는 새로운 방법이 열리며, 잠재적 전환율을 높이고 에이전시의 가치 제안을 증가시킵니다.
