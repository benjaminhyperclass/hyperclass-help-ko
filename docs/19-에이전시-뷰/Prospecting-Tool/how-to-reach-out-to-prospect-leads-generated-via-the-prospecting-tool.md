# 프로스펙팅 도구로 생성된 잠재 고객 리드에게 연락하는 방법

GoHighLevel의 프로스펙팅 도구를 통해 리드를 생성했다면 — AI, 수동 검색, 프로스펙팅 위젯을 통해서든 — 해당 잠재 고객들이 하위 계정(Sub-Account) CRM의 연락처 목록에 자동으로 추가됩니다. 이를 통해 이메일, SMS, 워크플로우를 통한 아웃리치 활동을 원활하게 시작할 수 있어요.

이 가이드는 이러한 리드를 찾고, 참여를 유도하며, GoHighLevel의 CRM을 활용해 고객으로 전환하는 방법을 안내합니다.

## 프로스펙팅은 어디서 사용할 수 있나요?

프로스펙팅은 두 가지 영역에서 사용할 수 있어요:

- **에이전시 뷰** — 리드 발견 및 기본 프로스펙팅용
- **하위 계정 뷰** — 고급 참여 및 자동화용

이메일, SMS 발송 및 워크플로우 실행은 하위 계정에서만 가능합니다. 따라서 프로스펙팅 작업은 하위 계정 내에서 수행하거나 마이그레이션할 때 가장 강력해요.

### 하위 계정에서 프로스펙팅을 활성화하는 방법

- **에이전시에서 이미 프로스펙팅 구독이 활성화되어 있는 경우**
  - [하위 계정을 연결](how-to-link-a-sub-account-with-your-agency-prospecting-premium-subscription.md)하여 액세스 권한을 얻을 수 있어요
  - 그런 다음 [잠재 고객을 해당 하위 계정으로 마이그레이션](how-to-migrate-prospects-from-agency-to-sub-account.md)하세요 (자동으로 연락처에 복사되고 워크플로우가 있으면 트리거됩니다)
  - 해당 하위 계정으로 이동하여 마케팅 탭에서 프로스펙팅을 찾으세요
  - **여러 하위 계정에서 사용하려면** 에이전시 뷰의 [재판매 탭]([how-to-resell-and-use-prospecting-tool-for-sub-accounts](how-to-resell-and-use-prospecting-tool-for-sub-accounts.md))으로 이동
  - 프로스펙팅 탭을 ON으로 전환하세요
- **이미 하위 계정이 있는 경우**
  - 단순히 에이전시에서 하위 계정으로 [잠재 고객을 마이그레이션](how-to-migrate-prospects-from-agency-to-sub-account.md)할 수 있어요 (자동으로 연락처에 추가되고 워크플로우가 있으면 트리거됩니다)

## 잠재 고객이 연락처 목록에 추가되는 방법

하위 계정 수준에서 프로스펙팅을 수행할 때 가능합니다.

수동 설정이 필요 없어요 — 다음을 통해 생성된 잠재 고객은 하위 계정의 연락처(Contacts)에 자동으로 추가됩니다:

- [AI 생성 프로스펙팅]([step-by-step-how-to-generate-prospects-by-ai](step-by-step-how-to-generate-prospects-by-ai.md))
- 수동 Google 비즈니스 검색
- 웹사이트에 삽입된 프로스펙팅 위젯

이는 다음을 즉시 할 수 있다는 의미예요:

- 이메일/SMS 아웃리치 시작
- CRM 파이프라인에 추가
- 자동 육성을 위한 워크플로우 트리거
- 에이전시에서 하위 계정으로 마이그레이션(복사)

새 잠재 고객을 생성할 때 알림도 받게 됩니다
![새 잠재 고객 알림](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155047915821/original/bIf7Nb3BbWW35G35dBF3lMSChwqzubMiRg.png?1749408585)

## 연락처 목록에서 잠재 고객을 찾는 단계

1. **하위 계정으로 이동**
   에이전시 뷰에서 프로스펙팅을 수행한 하위 계정으로 들어가세요.

2. **"연락처(Contacts)" 탭으로 이동**
   하위 계정 대시보드 내 좌측 내비게이션 사이드바에서 찾으세요.

3. **잠재 고객 리드 필터링**
   연락처 페이지 우측 상단의 "추가 필터(More Filters)" 버튼을 클릭하세요.

4. **필터 드롭다운에서 다음을 선택:**
   - 필드: 태그(Tag)
   - 조건: is
   - 값: New Prospect Received
   ![필터 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155047915809/original/VUVVU1FuBeAGnXyTSuFRIpVkHwgDT5d5iQ.png?1749408463)

5. **필터 적용**
   이제 프로스펙팅 도구를 통해 생성된 모든 연락처 목록을 볼 수 있어요.

![필터링된 연락처 목록](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155047915810/original/VEA1SXXwtgx8pssE9b-yBBBIjCHejLTvfw.jpeg?1749408477)

## 잠재 고객과 참여하는 방법

- **프로스펙트 도구 내에서**: 프로스펙트 보고서를 열고 공유 옵션을 클릭하여 SMS 또는 이메일로 공유하세요
![공유 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155047915739/original/oN-5XoXyAvmXBFehllng8kRXZnbvEW3kpg.png?1749407725)

- 연락처 목록에서 잠재 고객을 찾은 후 다음을 할 수 있어요:

- **연락처 카드 클릭**
  전체 연락처 세부 정보를 보고 수동으로 이메일이나 SMS를 보내세요. 연결된 마케팅 감사 보고서도 볼 수 있어요.

- **연락처의 점 세 개(⋮) 클릭**
  빠른 작업에는 다음이 포함됩니다:
  - 기회 관리/파이프라인에 추가
  - 예약 일정 잡기

## **프로스펙트 카드에서 참여하기 (프로스펙팅 대시보드)**

하위 계정 프로스펙팅에서는 프로스펙트 카드 리드 액션 아이콘을 사용하여:

- 연락처 세부 정보 보기
- 동일한 화면에서 최근 대화(SMS, 이메일 또는 전화) 열기

에이전시 프로스펙팅에서는 이러한 액션이 리드와 계속 상호 작용하기 전에 연결된 하위 계정으로 전환하거나 새로 만들도록 안내할 수 있어요.

## 워크플로우를 사용한 잠재 고객 자동 후속 조치

후속 조치 프로세스를 자동화하고 싶으신가요? 프로스펙팅 워크플로우 트리거를 사용하여 리드가 CRM에 추가되는 순간 자동으로 참여할 수 있어요.

자동화를 통해 다음을 할 수 있습니다:

- 마케팅 감사 보고서와 함께 환영 이메일 또는 SMS 즉시 발송
- 영업팀을 위한 할 일 또는 내부 알림 생성
- 리드를 파이프라인 또는 육성 시퀀스에 추가
- 데이터 보강 또는 태그 자동 할당

### [프로스펙팅 워크플로우 트리거 완전 가이드 보기 →]([workflow-trigger-for-new-prospects-in-the-prospecting-tool](workflow-trigger-for-new-prospects-in-the-prospecting-tool.md))

한 번 설정하면 — 모든 새 잠재 고객에 대해 시스템이 작업을 수행합니다.

## 잠재 고객이 연락처에 나타나지 않는 경우는 어떻게 하나요?

드문 경우지만, 일시적인 동기화 지연이나 중단으로 인해 잠재 고객이 연락처에 자동으로 나타나지 않을 수 있어요.

### 해결 방법:

- 프로스펙팅 도구의 세부 정보를 사용하여 수동으로 연락처 생성
- 일관성을 위해 "New Prospect Received" 태그 추가
- 필요에 따라 워크플로우 또는 파이프라인에 추가

💡 **팁**: 하위 계정에서 프로스펙팅이 활성화되고 활성 상태인지 항상 확인하세요. 그래야만 잠재 고객이 CRM에 제대로 동기화됩니다.

## 프로 팁

- 새 잠재 고객이 추가될 때 트리거되는 자동화된 워크플로우를 생성하세요 ("New Prospect Received" 태그 사용).
- 연락처의 스마트 리스트를 사용하여 잠재 고객 소스별로 성과를 세분화하고 추적하세요.
- 더 빠른 후속 조치를 위해 SMS/이메일 아웃리치에 AI 메시지 제안을 직접 포함하세요. (곧 출시 예정)
