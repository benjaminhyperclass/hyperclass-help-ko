# 프로스펙팅 도구: 잠재고객 카테고리 분류

프로스펙팅 도구(Prospecting Tool)의 강력한 새 기능인 **잠재고객 카테고리**를 소개합니다. 이번 업데이트는 사용자가 늘어나는 잠재고객 목록을 더 효과적으로 정리하고 관리할 수 있도록 설계되었습니다. 커스텀 카테고리를 생성하고, 개별 잠재고객에게 할당하며, 카테고리를 기준으로 쉽게 필터링할 수 있습니다. 유사한 잠재고객을 그룹핑하거나, 우선순위가 높은 리드(Lead)에 태그를 붙이거나, 일괄 업데이트를 간소화하려는 경우, 이 새로운 분류 시스템은 잠재고객 관리를 더욱 효율적이고 확장 가능하게 만들어줍니다. 특히 대량의 리드를 처리하는 에이전시에게 유용합니다.

## 새로운 기능:

프로스펙팅 도구에 잠재고객 카테고리를 추가하고 관리할 수 있는 새로운 기능을 추가했음을 기쁘게 알려드립니다. 이 새로운 옵션은 사용자가 새 카테고리를 생성하고, 잠재고객에게 할당하며, 동일한 카테고리로 잠재고객을 필터링할 수 있는 선택권을 제공하여 사용자 경험을 향상시킵니다.

## **주요 기능:**

- 사용자는 프로스펙팅 도구 내에서 새로운 커스텀 카테고리를 생성할 수 있습니다.
- 이러한 카테고리를 잠재고객에게 할당하여 해당 잠재고객을 고유하게 식별하는 데 도움이 됩니다.
- 특정 카테고리를 가진 잠재고객을 검색할 수 있는 필터(Filter)가 추가되었습니다.
- 기존 고객들도 쉽게 사용할 수 있도록 일괄 카테고리 분류도 가능합니다.

## **에이전시를 위한 장점:**

- 잠재고객 수가 증가하면 효과적으로 관리하기 어려워집니다.
- 카테고리를 기반으로 여러 잠재고객을 그룹화하거나 고유한 잠재고객을 식별하는 데 도움이 됩니다.
- 우선순위부터 틈새시장, 사용자 유형부터 업종 유형까지 다양한 활용 사례가 가능합니다.
