## 잠재 고객 이동이란?

잠재 고객 이동 기능을 통해 에이전시는 에이전시 레벨의 프로스펙팅 대시보드에서 선택한 잠재 고객을 특정 하위 계정으로 전송할 수 있습니다. 이는 리드를 관리하는 더 개인화되고 효율적인 방법을 제공하며, 하위 계정이 자체 워크플로우와 도구를 사용하여 잠재 고객과 소통할 수 있게 해줍니다.

잠재 고객을 하위 계정으로 이동시킴으로써, 에이전시는 개별 팀이나 클라이언트에게 후속 작업을 위임하면서도 에이전시 레벨에서의 가시성과 소유권을 유지할 수 있습니다. 이는 더 집중된 마케팅 노력과 더 나은 리드 전환으로 이어집니다.

## 사전 요구사항

- 대상 하위 계정에 프리미엄 프로스펙팅이 활성화되어 있어야 합니다.
- 에이전시 레벨에서 사용 가능한 기존 잠재 고객이 있어야 합니다.

## 잠재 고객 이동의 주요 혜택

이 기능은 운영 유연성을 개선하고 에이전시가 프로스펙팅 노력을 효율적으로 확장하는 데 도움이 됩니다:

- **고급 육성**: 하위 계정별 자동화, 파이프라인, 커뮤니케이션을 활용합니다.
- **운영 위임**: 후속 조치를 위해 적절한 하위 계정에 잠재 고객을 배정합니다.
- **향상된 잠재 고객 추적**: 모든 잠재 고객 상호작용을 계정별로 중앙화하고 체계화합니다.
- **에이전시 통제**: 이동된 잠재 고객은 향후 사용을 위해 에이전시 레벨에서 계속 접근 가능합니다.

## 잠재 고객 이동 기능 사용 방법

잠재 고객을 하위 계정으로 이동시키는 것은 빠르고 원활합니다. 성공적인 이동을 수행하기 위해 다음 단계를 따라하세요:

### 1단계: 프로스펙팅 대시보드 접속

- Hyperclass 에이전시 계정에 로그인합니다.
- 왼쪽 네비게이션에서 Prospecting(프로스펙팅) 옵션을 클릭합니다.

![](https://jumpshare.com/v/aLIT8F4APsdh1jBLczxc+/Screen+Shot+2025-06-23+at+8.18.34+PM.png)

### 2단계: 이동 탭 열기

- 프로스펙팅 섹션 내에서 상단의 Settings(설정) 탭을 클릭합니다.
- 설정 영역 내에서 Migrate(이동) 하위 탭을 선택하여 이동 인터페이스에 접근합니다.
- Migrate Prospects(잠재 고객 이동) 버튼을 클릭합니다.

![](https://jumpshare.com/v/Ixb5j4rjx15W92zqBs7T+/Screen+Shot+2025-06-23+at+8.21.10+PM.png)

### 3단계: 하위 계정 선택

- 팝업이나 드롭다운에서 제공된 목록에서 하위 계정을 선택합니다.

참고: 활성 프로스펙팅 프리미엄 구독이 있는 하위 계정만 목록에 표시됩니다.

![](https://jumpshare.com/v/37RJbMb6WD6zjTQ3k1W2+/Screen+Shot+2025-06-23+at+9.51.27+PM.png)

### 4단계: 이동할 잠재 고객 선택

- Select Prospects(잠재 고객 선택) 버튼을 클릭합니다.
- 드롭다운 목록에서 다음을 할 수 있습니다:
  - 잠재 고객 하나를 선택합니다.
  - 여러 잠재 고객을 선택합니다.
  - Select All(모두 선택) 옵션을 사용하여 사용 가능한 모든 잠재 고객을 포함시킵니다.

![](https://jumpshare.com/v/fJJwAxCpWenpWbV5HTYs+/GIF+Recording+2025-06-23+at+9.53.14+PM.gif)

### 5단계: 이동 실행

- 선택이 완료되면 Migrate(이동) 버튼을 클릭하여 전송을 시작합니다.

![](https://jumpshare.com/v/cS9oCsLOtv9w1Mp5qM0m+/Screen+Shot+2025-06-23+at+9.58.21+PM.png)

## 이동 완료 후 어떻게 되나요?

이동이 완료되면, 선택된 잠재 고객은:

- 하위 계정의 연락처에 추가됩니다.
- 전체 데이터, 메모, 기록을 유지합니다.
- 향후 이동이나 작업을 위해 에이전시 계정에서 계속 사용 가능합니다.

이러한 이중 가용성은 에이전시가 완전한 가시성과 통제를 유지할 수 있도록 보장합니다.

## 잠재 고객의 유연한 재사용

- 시간에 따라 동일한 잠재 고객을 여러 하위 계정으로 이동시킬 수 있습니다.
- 잠재 고객 데이터는 모든 로케이션에서 일관되고 최신 상태로 유지됩니다.

## 자주 묻는 질문

**Q: 이동 후 잠재 고객이 에이전시에서 삭제되나요?**
A: 아니요, 잠재 고객은 하위 계정으로 복사되지만 에이전시 레벨에서도 계속 유지됩니다.

**Q: 동일한 잠재 고객을 여러 하위 계정으로 이동시킬 수 있나요?**
A: 네, 필요에 따라 잠재 고객을 다른 하위 계정으로 이동시킬 수 있습니다.

**Q: 이동 후에도 잠재 고객이 기록, 메모, 데이터를 유지하나요?**
A: 네, 이동 중에 모든 관련 데이터가 그대로 유지됩니다.

**Q: 일괄 이동을 할 수 있나요?**
A: 네, 여러 잠재 고객을 선택하여 한 번에 이동시킬 수 있습니다.

**Q: 이동 기능이 구독에 포함되나요?**
A: 네, 이 기능은 추가 비용 없이 프로스펙팅 프리미엄 구독에 포함됩니다.

**Q: 이동을 취소할 수 있나요?**
A: 잠재 고객이 에이전시에서 유지되는 동안, 하위 계정에서 만들어진 변경사항은 수동으로 관리해야 합니다.
