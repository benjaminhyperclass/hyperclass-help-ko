# 하이퍼클래스 대량 프로스펙팅

대량 프로스펙팅(Bulk Prospecting)은 프로스펙팅 도구에서 구글 지도의 로컬 리드를 더 빠르게 찾고, 평가하고, 저장할 수 있게 해줍니다. 키워드, 위치("내 근처" 포함), 반경 설정으로 검색당 최대 60개의 리드를 찾을 수 있어요. 중요한 기준으로 정렬하고, 중복을 자동으로 감지하며, 한 번에 최대 15개의 잠재고객을 저장할 수 있습니다. 이 가이드에서는 스마트 검색 실행, 레이아웃 커스터마이징, 자신감 있는 아웃리치 시작 방법을 설명해요.

## **대량 프로스펙팅이란?**

대량 프로스펙팅은 하이퍼클래스 프로스펙팅 도구의 향상된 검색-저장 워크플로우로, 구글 지도에서 리드 발견과 캡처를 간소화합니다. 더 많은 검색 결과, 다중 선택 저장, 중복 감지, 커스터마이징 가능한 결과 보기를 제공하여 에이전시가 스프레드시트나 수동 복사-붙여넣기 없이도 프로스펙팅을 확장할 수 있게 해줘요.

---

## **대량 프로스펙팅의 주요 장점**

핵심 장점을 이해하면 팀이 대량 프로스펙팅을 고객 획득 워크플로우 어디에 활용할지 결정하고, 일상적인 아웃리치에서 속도, 정확성, 확장성을 어떻게 실현할지 알 수 있어요.

- **스마트 검색 제어:** 키워드로 업종이나 브랜드를 검색하고, 위치(주소 또는 "내 근처")와 결합하여 반경(최대 약 30마일/49km)을 설정해 결과를 집중시킬 수 있습니다.

- **더 많은 검색 결과:** 검색당 최대 60개의 리드를 반환(구글 제한)하여 기존 20개 리드 제한을 넘어 커버리지를 확장합니다.

- **규모 있는 배치 저장:** 배치당 최대 15개의 리드를 선택하고 추가할 수 있으며, 선택 사항이 페이지 간에 유지되어 더 적은 클릭으로 더 많이 캡처할 수 있어요.

- **내장 중복 감지:** 상태(Status) 컬럼을 통해 잠재고객이 이미 저장되었는지 확인하여 중복 작업을 방지합니다.

- **우선순위 도구:** 잠재고객 점수, 구글 평점, 웹사이트/GMB 지표별로 정렬하여 고가치 리드에 먼저 집중할 수 있어요.

- **커스터마이징 가능한 레이아웃:** 필요한 컬럼을 선택하고 페이지당 행 수를 설정하여 더 빠른 스캔과 처리가 가능합니다.

---

## **스마트 검색**

검색 설정은 결과에 무엇이 나타날지 제어합니다. 키워드, 위치, 반경을 정확히 설정하면 가장 관련성 높은 비즈니스를 보고 훑어보는 시간을 줄일 수 있어요.

- **키워드 / 비즈니스명:** 업종(예: "치과", "지붕공사업체") 또는 특정 브랜드(예: "KFC")를 입력하세요.

- **위치:** 전체 주소/도시/우편번호를 입력하거나 **내 근처**를 입력하여 IP 기반 위치 정보를 사용하세요(VPN이나 공유 사무실 IP에서는 정확도가 다를 수 있음).

- **반경:** 검색 영역을 최대 30마일(49km)로 제한하세요. 반경이 넓을수록 더 많은 결과가 나오지만 지역적 관련성은 떨어질 수 있어요.

## **확장된 검색 결과**

확장된 검색 결과는 검색당 발견을 늘려 시간을 절약하고 같은 쿼리를 반복 실행하지 않고도 파이프라인을 넓혀줍니다.

- **검색당 최대 60개의 리드.** 더 필요하다면 키워드/위치를 조정하여 추가 검색을 실행하세요.

- **지도 + 표 보기**로 빠른 공간적 맥락과 목록 스캔이 가능해요.

- **상태 힌트**가 결과가 저장되거나 처리 중일 때 나타납니다.

## **대량 저장 및 선택**

다중 선택 저장은 클릭을 줄이고 목록 작성 속도를 높여줍니다. 제한과 지속성을 이해하면 워크플로우를 효율적으로 계획할 수 있어요.

- **배치당 최대 15개 행 선택**하고 **여러 잠재고객 추가(Add Multiple Prospects)**를 클릭하여 한 번에 저장하세요.

- **선택 사항이 페이지 간에 유지됨**—결과를 탐색하고, 행을 선택하고, 함께 저장할 수 있어요.

- 현재 페이지의 **전체 선택 / 전체 선택 해제** 제어를 사용하여 실수로 전역 선택 해제하지 않으면서 빠르게 이동하세요.

## **정렬 및 필터링**

정렬과 필터는 고가치 잠재고객의 우선순위를 정하고 아웃리치에 가장 적합한 기회를 빠르게 찾는 데 도움을 줍니다.

- **잠재고객 점수**, **온라인 존재감/GMB**, **웹사이트(있음/없음)**, **구글 평점/리뷰** 기준으로 정렬하세요.

- **필터 아이콘**(컬럼 관리 옆)을 사용하여 보기를 세분화하고 기준을 충족하는 잠재고객에 집중하세요.

## **커스터마이징 가능한 검색 레이아웃**

컬럼과 페이지 밀도를 조정하면 워크플로우를 깔끔하고 빠르게 유지할 수 있어요. 특히 한 세션에서 많은 리드를 감사할 때 유용합니다.

- **컬럼 관리(Manage Columns)**를 클릭하여 **잠재고객 점수**, **온라인 존재감**, **웹사이트**, **상태** 같은 필드를 표시/숨김하고 순서를 바꾸세요.

- **페이지당 행 수**를 5, 10, 15 중에서 선택하여 더 빠른 처리와 쉬운 스캔이 가능해요.

## **중복 감지 및 상태**

상태는 혼란과 중복 작업을 줄여줍니다. 각 상태의 의미를 알면 중복을 방지하고 저장 후 진행 상황을 이해할 수 있어요.

- **추가되지 않음(Not Added):** 잠재고객이 아직 저장되지 않았습니다.

- **추가됨(Added):** 잠재고객이 이전에 저장되었거나 방금 추가되었습니다.

- **잠재고객 생성 중(Generating Prospect) (xx%):** 시스템이 레코드와 마케팅 감사를 준비 중이며, 진행률 퍼센트가 처리 상태를 반영합니다.

- **상태** 컬럼과 **보기(View)** 액션은 비즈니스가 이미 존재하는지 확인하여 중복 추가를 방지하는 데 도움을 줍니다.

## **저장된 잠재고객이 가는 곳**

레코드가 어디에 저장되는지 알면 팀이 데이터를 찾지 않고도 아웃리치와 추적을 시작할 수 있어요.

- 저장된 레코드는 잠재고객 계정(에이전시 뷰)에 나타나거나, 하위 계정에 있다면 마케팅(Marketing) → 프로스펙팅(Prospecting)에 나타나며, 프로스펙팅 도구에서 액세스할 수 있습니다.

- **프로스펙팅 프리미엄(Prospecting Premium)**이 활성화되고 하위 계정에 연결된 경우, 자동화 및 아웃리치를 위해 해당 하위 계정에 매칭 **연락처(Contacts)**가 생성됩니다.

- 잠재고객의 행(또는 잠재고객 프로필)에서 **마케팅 감사 보고서(Marketing Audit Report)**를 열어 웹사이트/GMB 성과를 검토하고 다음 단계를 계획하세요.

![저장된 잠재고객이 가는 곳](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055132230/original/VTeYdnp2LN-Ej0ZGO43TCiOvM-FY0RaqnQ.png?1759437819)

**팁**: 연락처가 생성되면 하위 계정에서 육성 캠페인을 시작하세요. **[프로스펙팅 도구를 통해 생성된 잠재고객에게 연락하는 방법](프로스펙팅 도구를 통해 생성된 잠재고객에게 연락하는 방법.md)([how-to-reach-out-to-prospect-leads-generated-via-the-prospecting-tool](how-to-reach-out-to-prospect-leads-generated-via-the-prospecting-tool.md)** 참조

## 대량 프로스펙팅 설정 방법

이 가이드 플로우에 따라 첫 검색을 실행하고, 결과를 평가하고, 여러 잠재고객을 효율적으로 저장해보세요.

### **1단계:** 하위 계정에서 마케팅(Marketing) → 프로스펙팅(Prospecting)으로 가서 잠재고객 추가(Add Prospect)를 클릭하세요.

![1단계](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055132370/original/9N0kz28YEIFa9Aqm_7Vyne3pfHrdSCrpkw.png?1759438288)

### **2단계:** **검색 키워드 / 비즈니스명(Search Keyword / Business Name)**에 업종(예: 도미노) 또는 브랜드를 입력하세요. **위치(Location)**에 도시/우편번호를 입력하고, **반경(Radius)**(최대 약 30마일/49km)을 설정한 후 **검색(Search)**을 클릭하세요.

![2단계](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055132415/original/PT0gz_iY6BaycJ405NPgVG14RFFeJ7gviA.png?1759438385)

### **3단계:** **결과 목록 + 지도**를 검토하세요. 컬럼 헤더를 사용하여 잠재고객 점수, 웹사이트, 평점으로 **정렬**하세요.

![3단계](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055132467/original/curO4Ef9Q63gN5QPgc6pi7_k29sHrvn8xg.png?1759438495)

**팁**: **컬럼 관리(Manage Columns)**를 클릭하여 필드를 표시/숨김할 수 있고, 더 빠른 배치를 위해 **페이지당 행 수**를 15로 설정하세요.

### 4단계: 최대 15개 행 선택(선택 사항이 페이지 간에 유지됨)

![4단계](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055132545/original/kdX0XmGnC8zlyHzz43bld6uh7UAJmlqC1A.png?1759438688)

### **5단계:** **여러 잠재고객 추가(Add Multiple Prospects)** 클릭

![5단계](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055132568/original/eneWFZnqLxnmnZFPl2nZ9s8FOZz_NFNJ5g.png?1759438749)

**중요**: **상태(Status)** 컬럼에서 **잠재고객 생성 중(Generating Prospect) (xx%)**에서 **추가됨(Added)**으로 바뀌는 것을 확인하세요. **보기(View)**를 클릭하여 레코드와 **마케팅 감사 보고서(Marketing Audit Report)**를 여세요. 또한 하위 계정에 연결된 프로스펙팅 프리미엄을 사용하는 경우 **연락처(Contacts)**를 열어 동기화를 확인하고 **아웃리치 워크플로우**를 시작하세요.

팁:

1. 60개 리드 제한에 도달하면 반경을 넓히거나, 키워드를 조정하거나(예: "지붕공사업체" vs "루핑 계약업체"), 중심 위치를 이동하여 새 지역을 검색하세요.

2. 내 근처가 이상하게 나오면 VPN/위치 설정을 확인하거나 위치를 명시적으로 입력하세요.

## **자주 묻는 질문**

**Q: 검색당 60개 결과만 나오는 이유는?**

구글이 제한을 두기 때문입니다. 다른 키워드/위치로 추가 검색을 실행하여 커버리지를 확장하세요.

**Q: 한 번에 몇 명의 잠재고객을 저장할 수 있나요?**

배치당 최대 15명입니다. 선택 사항은 추가할 때까지 페이지 간에 유지됩니다.

**Q: "잠재고객 생성 중(Generating Prospect) (xx%)"은 무슨 의미인가요?**

하이퍼클래스가 잠재고객 레코드를 생성하고 마케팅 감사를 준비하고 있다는 뜻입니다. 완료되면 **추가됨(Added)**으로 바뀝니다.

**Q: 중복 감지는 어떻게 작동하나요?**

비즈니스가 이미 계정에 저장된 경우 **상태(Status)**에 **추가됨(Added)**이 표시됩니다. 다시 추가하는 대신 **보기(View)**를 사용하여 기존 레코드를 여세요.

**Q: 웹사이트가 없는 비즈니스만 필터링할 수 있나요?**

네. **웹사이트(Website)** 컬럼으로 정렬/필터하고 **없음(No)**으로 표시된 비즈니스에 아웃리치를 집중하세요.

**Q: 보는 컬럼을 어디서 변경할 수 있나요?**
