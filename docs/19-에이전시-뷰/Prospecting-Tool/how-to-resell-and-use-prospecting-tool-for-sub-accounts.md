# 하위 계정용 프로스펙팅 도구를 재판매하고 사용하는 방법

이 가이드에서는 프로스펙팅 도구(Prospecting Tool)를 클라이언트에게 재판매하는 방법을 설명해드려요.

**하위 계정용 프로스펙팅 도구**는 에이전시가 하위 계정에 새로운 비즈니스 잠재고객 검색, 마케팅 감사 보고서 생성, 그리고 틈새 시장, 지역 등의 선호도에 따른 프로스펙팅 작업 커스터마이징 기능을 제공할 수 있게 해줍니다. 이 도구는 다음과 같은 목적을 가진 에이전시에게 특히 유용해요:

- 클라이언트에게 **프로스펙팅 서비스를 재판매**하고자 하는 경우
- **메인 에이전시 대시보드 접근을 제한**하면서도 하위 계정이 강력한 프로스펙팅 도구를 활용할 수 있게 하고자 하는 경우

이 가이드에서는 **하위 계정용 프로스펙팅 도구**의 주요 기능과 효과적인 사용법을 안내해드릴게요.

# 사전 준비사항

## 에이전시 Stripe 계정 연결하기

에이전시 Stripe 계정은 클라이언트로부터 프로스펙팅 도구 구독 결제를 수집하는데 사용됩니다. 다음 도움말 가이드를 따라 에이전시 Stripe 계정을 연결하실 수 있어요 - 에이전시 대시보드에 Stripe 연결하는 방법

## 재판매 탭에서 프로스펙팅 도구 활성화

프로스펙팅 도구를 활성화하려면 에이전시 대시보드의 **재판매 탭(Reselling Tab)**으로 이동하세요. 여기서 기능을 토글해서 켜면 하위 계정에 접근 권한이 부여됩니다.

![프로스펙팅 도구 활성화](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155039768111/original/7Gsl1htQevORk8hupYjES5Txh17IStm1OQ.png?1736824902)

# 요금제 선택하기

에이전시 좌측 메뉴 → 재판매(Reselling) 화면에서 요금제를 선택하실 수 있어요.

현재 모든 사용자를 위한 표준 요금제가 있습니다:

- 사용한 만큼 지불하는 플랜으로, 월 $29 / 로케이션의 구독료가 청구되며, 원하는 금액으로 재판매할 수 있습니다

![요금제 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155037735410/original/A8fZ5dQNZNdocdXhhU55t7rlU0OVBwzfRA.png?1733314380)

# 판매 가격 설정하기

에이전시 좌측 메뉴 → 재판매(Reselling) 탭에서 설정을 통해 에이전시 전체 오퍼를 설정할 수 있어요.

![판매 가격 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155037735427/original/5S5oCK5OU2tf6d4aAS8VA_CGZ6tF0-U6_g.jpeg?1733314400)

## 특정 클라이언트를 위한 맞춤 오퍼

특정 클라이언트에게 다른 요금을 제공하고 싶다면, 계정 목록(Account List) → 클라이언트 검색 → 세부 정보 보기로 이동해서 설정하실 수 있어요.

![클라이언트 검색](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155037735330/original/VUIGdaPw282IiSbikTs9zRbP3GW5AkxKoQ.png?1733314308)

![맞춤 오퍼 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155037735579/original/z896qXwaKZoHtNnGsZy78ye-BQDjyZZmAQ.png?1733314476)

## 특정 클라이언트의 오퍼 비활성화

특정 클라이언트에게 프로스펙팅 도구를 제공하지 않으려면, 에이전시 좌측 메뉴 → 계정 목록 → 로케이션 검색 → 세부 정보 보기에서 프로스펙팅 오퍼(Offer Prospecting) 옵션을 토글해서 끄실 수 있어요.

![오퍼 비활성화](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155037735606/original/416S8IXl8eH5xza5ZJtQCsdBkkIInszy1A.png?1733314511)

주의사항:

이 오퍼는 판매가 발생하기 전에만 비활성화할 수 있습니다. 로케이션이 이미 프로스펙팅 도구를 구매한 후에는 여기서 비활성화할 수 없어요.

## 클라이언트가 구매하는 방법

클라이언트가 기능을 확인한 후, 지금 구독하기(Subscribe Now) 버튼을 클릭해서 프로스펙팅 도구를 구매할 수 있어요.

![구독 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155037735957/original/jV4xA1i9KU9skGjE9q5Jy3VHn7qdeMcy1g.png?1733314748)

![구매 진행](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155037735338/original/w9G4jGsIE8NRe36OwTLgw3g2dkFGM97CLQ.png?1733314310)

클라이언트가 계속하기(Continue) 버튼을 클릭하면 시스템이 구독을 생성하고 결제를 처리한 후, 프로스펙팅 도구 대시보드로 이동시켜줍니다.

# 청구는 어떻게 작동하나요?

클라이언트가 프로스펙팅 도구를 구매하면 시스템에 2개의 구독이 생성됩니다:

- 클라이언트의 카드와 당신의 Stripe 계정 사이의 구독 (클라이언트로부터 결제를 수집하기 위해)
- HighLevel과 당신의 카드 사이의 구독 (HighLevel에 월 $29를 지불하기 위해)
