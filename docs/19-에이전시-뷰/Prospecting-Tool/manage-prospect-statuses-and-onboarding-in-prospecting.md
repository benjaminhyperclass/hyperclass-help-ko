# 프로스펙팅에서 잠재고객 상태 및 온보딩 관리

프로스펙팅에서 이제 더 명확한 리드 라이프사이클을 지원합니다. 잠재고객을 Open(진행 중), Closed Won(성사), Closed Lost(실패)로 표시하고, 상태별로 필터링하며, 성사된 잠재고객을 더 빠르게 온보딩으로 이동시킬 수 있습니다. Agency Prospecting(에이전시 프로스펙팅)에서 잠재고객을 Closed Won으로 표시하면 Create Sub-Account(하위 계정 생성) 기능이 활성화됩니다. 이를 통해 에이전시는 스프레드시트나 외부 도구에 의존하지 않고도 결과를 추적할 수 있습니다.

## 프로스펙팅의 잠재고객 상태 관리란?

잠재고객 상태 관리는 리드 생성 후 어떤 일이 일어나는지 추적하는 데 도움이 되는 프로스펙팅 기능입니다. 모든 잠재고객을 구분 없이 하나의 목록에 보관하는 대신, 이제 각 잠재고객의 결과를 업데이트하고 라이프사이클 단계별로 필터링할 수 있습니다.

이 기능에는 다음이 포함됩니다:

- 잠재고객 상태: Open(진행 중), Closed Won(성사), Closed Lost(실패)
- 상태 기반 필터링
- Agency Prospecting에서 성사된 잠재고객에 대한 Create Sub-Account(하위 계정 생성)
- 새로운 프로스펙팅 사용자를 위한 Getting Started(시작하기) 온보딩 카드

이는 리드 생성보다는 아웃리치 후 추적에 중점을 두는 기존 프로스펙팅 기능의 공백을 메워줍니다.

## 프로스펙팅 잠재고객 상태 관리의 주요 장점

- 프로스펙팅 내에서 직접 잠재고객 결과 추적
- 활성 리드와 종료된 리드 분리
- 아직 온보딩이 필요한 성사 건 식별
- 스프레드시트에서 수동 추적 줄이기
- 가이드 온보딩으로 신규 사용자가 더 빨리 시작하도록 지원
- 에이전시 뷰에서 프로스펙팅 활동을 하위 계정 생성과 연결

## 프로스펙팅에서 잠재고객 상태 관리 사용 방법

### 잠재고객 상태 업데이트

- Prospecting(프로스펙팅)을 엽니다.
- 업데이트하려는 레코드의 Prospect Card(잠재고객 카드)를 엽니다.
- 상태를 Open(진행 중), Closed Won(성사), 또는 Closed Lost(실패)로 변경합니다.

![잠재고객 상태 업데이트](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155066809540/original/7uAPMET4zAxAeDcQ4NJuDvRTw8wx5qv1sw.png?1773328693)

### 상태별 잠재고객 필터링

- 프로스펙팅의 저장된 잠재고객 목록으로 이동합니다.
- Status(상태) 필터를 엽니다.
- 다음 옵션 중 하나 이상을 선택합니다:
  - Open(진행 중)
  - Closed Won(성사)
  - Closed Lost(실패)

![상태별 잠재고객 필터링](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155066809602/original/oB-9lvPdhMS-AEV8vz5QmyXt3otkA7gD6A.jpeg?1773328727)

### Getting Started 카드 사용

- 신규 사용자로 프로스펙팅을 엽니다.
- Getting Started(시작하기) 카드를 검토합니다.
- 다음과 같은 주요 작업을 완료합니다:
  - 첫 번째 잠재고객 생성
  - 보고서 생성 및 공유
  - 잠재고객을 CRM으로 전송
  - 리드 캡처
  - AI 발견 활성화

## 자주 묻는 질문

**Q. 잠재고객에게 어떤 상태를 할당할 수 있나요?**
잠재고객을 Open(진행 중), Closed Won(성사), 또는 Closed Lost(실패)로 표시할 수 있습니다.

**Q. 상태별로 잠재고객을 필터링할 수 있나요?**
네. Open(진행 중), Closed Won(성사), Closed Lost(실패), Sub-Account Created(하위 계정 생성됨), Sub-Account Not Created(하위 계정 미생성)로 필터링할 수 있습니다.

**Q. 잠재고객을 Closed Won으로 표시하면 어떻게 되나요?**
Agency Prospecting에서 잠재고객을 Closed Won으로 표시하면 Create Sub-Account(하위 계정 생성) 기능을 사용할 수 있게 됩니다.

**Q. 이 기능이 카테고리를 대체하나요?**
아니요. 카테고리와 상태는 서로 다른 목적을 가지고 있습니다. 카테고리는 잠재고객을 조직화하고, 상태는 잠재고객 결과를 추적합니다. 카테고리 관련 아티클은 별도로 있으며, 이러한 구분에 대한 간단한 상호 참조가 필요할 것입니다.

**Q. Getting Started 카드는 누가 볼 수 있나요?**
프로스펙팅을 처음 사용하는 사용자를 위한 것입니다. 사용자가 활성화되거나 충분히 닫으면 자동으로 숨겨집니다.
