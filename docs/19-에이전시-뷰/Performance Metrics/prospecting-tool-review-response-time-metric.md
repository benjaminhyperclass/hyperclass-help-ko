# 프로스펙팅 도구: 리뷰 응답 시간 지표

**목차**

- [새로운 기능:](#새로운-기능)
- [핵심 기능:](#핵심-기능)
- [사용자 혜택:](#사용자-혜택)

## **새로운 기능:**
프로스펙팅 도구(Prospecting Tool)에 새로운 기능이 추가되었습니다! 답변이 있는 리뷰의 응답 시간을 계산하고, 이 정보를 감사 보고서에 표시하는 기능을 추가했습니다.

## 핵심 기능:

프로스펙팅 도구에서 이제 잠재 고객의 구글(Google)과 페이스북(Facebook)에서 받은 리뷰에 대한 평균 리뷰 응답 시간 지표가 감사 보고서에 포함됩니다. 이 중요한 정보는 마케팅 감사 보고서에 자연스럽게 통합되어, 잠재 고객의 리뷰 관리에 대한 모든 중요한 인사이트를 제공합니다.

## 사용자 혜택:

- 에이전시로서, 잠재 고객이 이미 리뷰 응답 자동화 시스템을 구축했는지 파악할 수 있습니다.
- 에이전시로서, 이 데이터를 마케팅/영업에서 강력한 판매 포인트로 활용하여 내 솔루션/서비스를 제안할 수 있습니다.

참고사항:
- 기존 잠재 고객의 경우, 보고서를 새로고침하면 이 정보에 접근할 수 있습니다.
- 마케팅 감사 보고서에 더 많은 가치를 추가했습니다.
