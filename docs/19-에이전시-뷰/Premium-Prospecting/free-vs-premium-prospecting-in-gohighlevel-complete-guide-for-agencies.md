# GoHighLevel 무료 vs 프리미엄 프로스펙팅 – 에이전시를 위한 완벽 가이드

## 프로스펙팅이란?

프로스펙팅(Prospecting)은 당신의 서비스로부터 혜택을 받을 가능성이 높은 잠재 고객을 찾아내고 평가하는 과정입니다. GoHighLevel의 프로스펙팅은 고품질 리드를 발견하고, 마케팅 준비도를 분석하며, 서비스 제안 기회를 찾는 지능형 리드 생성 도구로, 모든 작업을 대시보드 내에서 할 수 있게 해줍니다.

## GoHighLevel 프로스펙팅이란?

GoHighLevel의 프로스펙팅 도구는 리드 생성 작업을 자동화하고 단순화하도록 설계되었습니다. 이제 비즈니스를 수동으로 검색하고 감사(audit)를 실행하는 대신 다음과 같은 작업이 가능합니다:

- 비즈니스 데이터와 AI를 사용하여 리드를 찾고 평가
- 각 리드의 마케팅 현황 격차를 보여주는 감사 보고서 생성
- 이 데이터를 활용해 서비스 업셀, 가치 입증, 파이프라인 구축 가속화

프로스펙팅은 두 가지 방식으로 접근할 수 있습니다:

- 기본(무료 버전)
- 프리미엄(유료 구독)

기본 무료 프로스펙팅 – 업그레이드 전에 먼저 경험해보기

- 처음 시작하거나 도구를 테스트해보고 싶다면, 무료 버전에서 필수 기능들을 이용할 수 있습니다.
- 에이전시에서 접근하는 방법: 에이전시 대시보드로 이동
- 좌측 사이드바의 "Prospecting(프로스펙팅)" 탭 클릭
- "Continue with Free(무료로 계속하기)" 클릭
- 하위 계정에서 접근하는 방법: 하위 계정의 무료 또는 기본 프로스펙팅은 'Agency Sub Account(에이전시 하위 계정)'로 표시된 하위 계정에서만 이용 가능하며, 현재 $497 플랜에서만 제공됩니다.
- 포함 내용: 월 5개 프리미엄 보고서
- 일일 50개 기본 보고서
- 기본 보고서 내용: 비즈니스 리스팅 감사
- 기본 구글 비즈니스 프로필 분석
- 온라인 평판 스냅샷 (리뷰, 평점 등)

⚠️ 무료 프로스펙팅은 에이전시 레벨에서만 이용 가능합니다(하위 계정에서는 불가).

## **프리미엄 프로스펙팅으로 무엇을 얻을 수 있나요?**

프리미엄이 정말 강력한 이유는 다음과 같습니다:

### 월 무제한 보고서

더 이상 제한이 없습니다 — 제한 없이 프로스펙팅하고 보고서를 생성하세요.

### 무제한 보고서 새로고침

원할 때마다 보고서를 업데이트하고 재생성할 수 있습니다!

### AI 기반 프로스펙팅

- AI가 타겟 지역과 틈새시장을 기반으로 자격을 갖춘 리드를 자동으로 발견합니다.
- AI가 며칠마다 바로 사용 가능한 프리미엄 보고서를 대시보드로 직접 전달합니다.

### 웹사이트 성능 감사

- 잠재 고객 웹사이트의 성능(속도, SEO, 모바일 친화성 등)을 확인합니다.
- 격차를 파악하고 제안의 일환으로 해결책을 제시하세요.

### 로컬 SEO 히트맵 & 랭킹

- 잠재 고객 키워드의 지리적 가시성을 확인합니다.
- 각 핀 위치에서 최상위 랭킹 경쟁사를 지도에서 확인하세요.

### 리드 생성 위젯

- 감사를 리드 마그넷으로 웹사이트에 삽입합니다.
- 브랜드 스타일에 맞게 완전히 커스터마이즈 가능합니다.

### CRM & 아웃리치 연동 (하위 계정에서만 이용 가능)

- 잠재 고객을 GHL CRM, 기회 관리 파이프라인, 워크플로우로 전송합니다.
- CRM에서 직접 이메일과 SMS로 연락합니다.
- 빠른 후속 조치를 위한 AI 추천 메시지 템플릿을 받습니다.

### 자동화 지원 (하위 계정에서만 이용 가능)

새로운 잠재 고객이 생성되는 즉시 워크플로우를 트리거합니다. 다음이 가능합니다:

- 환영/후속 메시지 자동 발송
- 잠재 고객 연락처 정보 저장
- 육성 시퀀스에 등록

## 프리미엄 프로스펙팅 활성화 방법

### 에이전시 레벨에서

- 좌측 네비게이션 메뉴에서 프로스펙팅 탭으로 이동
- 우상단의 "Unlock Premium(프리미엄 잠금해제)"를 클릭하거나 "Upgrade Now(지금 업그레이드)"를 선택
- 결제 및 설정 단계를 따릅니다.

### 하나 이상의 하위 계정에서

- 에이전시 뷰에서 Reselling(리셀링) 탭으로 이동
- 하위 계정들에 대해 프로스펙팅을 활성화
- 이 기능을 재판매하고 싶다면 가격을 설정
- 하위 계정에서 이제 자체 대시보드에서 프리미엄을 활성화할 수 있습니다.

## 빠른 정리: 무료 vs 프리미엄

| 기능 | 무료 (에이전시만) | 프리미엄 (에이전시 + 하위 계정) |
|------|------------------|--------------------------------|
| 보고서 제한 | 월 5개 프리미엄/일 50개 기본 | 무제한 |
| 보고서 유형 | 기본 감사 (리스팅, GBP, 리뷰) | 완전 감사 + 웹사이트, SEO 히트맵 |
| AI 프로스펙팅 | ❌ 포함 안됨 | ✅ 포함됨 |
| CRM 연동 | ❌ | ✅ |
| 자동화 | ❌ | ✅ |
| 리드 생성 위젯 | ❌ | ✅ |
| 이메일/SMS 아웃리치 | ❌ | ✅ |
| 하위 계정 접근 | ❌ | ✅ (연결을 통해) |
