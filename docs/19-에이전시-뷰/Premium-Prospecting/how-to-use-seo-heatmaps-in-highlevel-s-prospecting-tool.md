# Hyperclass 프로스펙팅 도구의 SEO 히트맵 사용 방법

프로스펙팅 도구(Prospecting Tool)의 SEO 히트맵 기능은 잠재 고객의 검색엔진 순위를 시각적으로 보여줍니다. 에이전시와 비즈니스가 특정 키워드와 지역에서 웹사이트가 검색 결과에서 어느 위치에 있는지 빠르게 파악할 수 있어, 가시성을 개선하고 SEO 노력을 최적화하는 데 도움이 됩니다.

참고: 이 기능을 사용하려면 에이전시와 로케이션 모두 Premium Prospecting 구독이 필요합니다.

# SEO 히트맵이란?

SEO 히트맵은 지도의 9개 핵심 지점에 검색 순위를 표시하여 웹사이트가 다양한 위치에서 얼마나 잘 순위를 매기는지 지리적 관점을 제공하는 도구입니다. 히트맵의 각 지점은 잠재 고객 비즈니스 주변의 특정 위치에 해당하며, 비즈니스가 강한 가시성을 보이는 영역과 개선이 필요한 영역을 식별하는 데 도움이 됩니다.

## SEO 히트맵 도구의 주요 이점

- **지리적 인사이트**: 다양한 지역에서의 순위 성과를 시각화하여 SEO 노력이 성공하고 있는 곳과 개선이 필요한 곳을 명확히 보여줍니다.
- **타겟 최적화**: 히트맵을 사용해 가시성이 부족한 특정 지리적 영역을 식별하고, 해당 위치에서 순위를 높이기 위해 SEO 전략을 최적화하세요.
- **향상된 로컬 검색 전략**: 로컬 검색에 의존하는 비즈니스의 경우, 이 히트맵은 인근 지역에서 얼마나 잘 성과를 내고 있는지 보여줘 로컬 SEO 전략을 적절히 조정하는 데 도움이 됩니다.

## SEO 히트맵 도구 작동 원리

SEO 히트맵은 여러 지점의 순위를 기준으로 검색 가시성을 평가합니다:

- **중심점 (20% 가중치)**: 비즈니스의 중심 위치입니다. 이 지점은 전체 점수에서 가장 높은 가중치를 받으며, 비즈니스 인근 지역에서의 가시성을 반영합니다.
- **외곽점 (각 10% 가중치)**: 인근 지역을 나타내는 8개의 주변 지점입니다. 이 지점들은 비즈니스가 인접한 위치에서 얼마나 잘 순위를 매기는지에 대한 인사이트를 제공합니다.

각 지점은 선택된 키워드 또는 키워드 세트에 대한 검색 결과에서 비즈니스의 위치를 기준으로 순위가 할당됩니다. 지점의 순위가 높을수록 해당 위치에서의 가시성이 좋고, 전체 SEO 성과가 강하다는 의미입니다.

## SEO 히트맵 도구 사용 방법

참고:
SEO 히트맵 카테고리 데이터가 이제 잠재 고객의 구글 비즈니스 프로필 카테고리와 더욱 밀접하게 일치합니다. 이를 통해 히트맵이 Google에 등록된 비즈니스를 더 정확하게 반영할 수 있습니다.

이전 보고서를 검토하는 경우, 최신 히트맵 데이터를 불러오기 위해 먼저 새로고침하세요.

### 에이전시 계정용

#### 1단계: 프로스펙팅 도구 열기

- Hyperclass 에이전시 계정에 로그인합니다.
- 좌측 메뉴에서 Prospecting(프로스펙팅) 탭을 클릭합니다.

![](https://jumpshare.com/v/D937XgjrDjqUgounid1F+/Screen+Shot+2025-07-14+at+9.52.11+PM.png)

#### 2단계: 잠재 고객 보고서 열기

- Add Prospect(잠재 고객 추가)를 클릭해 새로 만들거나 기존 잠재 고객을 선택합니다.
- View Report(보고서 보기) 버튼을 클릭합니다.

![](https://jumpshare.com/v/R8oNO3OXorB0reTmke87+/Screen+Shot+2025-07-14+at+9.53.53+PM.png)

#### 3단계: SEO 히트맵 보기

- 보고서가 로드되면 좌측 사이드바에서 SEO Analysis(SEO 분석) 탭을 클릭합니다.
- 스크롤하여 키워드 성과를 표시하는 히트맵을 확인합니다.
- 지도의 각 점은 인근의 다른 위치에서의 로컬 검색 결과를 나타냅니다.

- 중심점은 비즈니스의 인근 지역을 반영합니다.
- 정의된 키워드(예: "asian restaurant near me")에 대한 순위가 표시됩니다.
- 데이터는 또한 상위 3개 로컬 경쟁업체를 강조 표시합니다.

![](https://jumpshare.com/v/R8rJJidTaLQ1y7VOvY9j+/Screen+Shot+2025-07-14+at+10.01.37+PM.png)

## 모범 사례
