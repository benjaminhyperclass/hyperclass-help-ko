# 가상 비즈니스의 비즈니스 맵 URL을 찾아 정확한 보고서 생성하기

물리적인 위치가 없는 비즈니스라도 Google 지도에서의 존재감을 갖는 것이 중요합니다. 이 가이드는 가상 비즈니스의 비즈니스 맵 URL을 찾는 방법을 안내합니다.

## 1단계: Google에 비즈니스가 등록되어 있는지 확인

- [Google My Business](https://www.google.com/business/)로 이동합니다.
- Google 계정으로 로그인합니다.
- 아직 등록하지 않았다면, 비즈니스를 추가하고 서비스 지역 비즈니스로 인증받으세요.

## 2단계: Google 지도에서 비즈니스 검색

- [Google 지도](https://www.google.com/maps)를 엽니다.
- 검색창에 비즈니스 이름을 입력합니다.
- 검색 결과에 비즈니스가 나타나면 클릭합니다.

## 3단계: 지도 URL 획득

### 방법 A: 주소 표시줄에서

- 비즈니스 목록이 열리면 브라우저의 주소 표시줄을 확인합니다.
- URL은 다음과 같은 형태입니다:

```
https://www.google.com/maps/place/Your+Business+Name/@XX.XXXXXX,-YY.YYYYYY,17z/data=...
```

- 이것이 비즈니스 맵 URL입니다.

### 방법 B: "공유" 기능 사용

- 비즈니스 목록에서 "공유" 버튼을 클릭합니다.
- 팝업 창에서 "링크 복사" 옵션을 확인할 수 있습니다.
- "링크 복사"를 클릭하여 비즈니스 맵 URL을 얻습니다.

## 5단계: 프로스펙팅 도구에서 URL 사용

- Hyperclass 계정에서 프로스펙팅 도구(Prospecting Tool)를 엽니다.
- "Search for Virtual Businesses"로 이동합니다.
- 폼에서 "Business Maps URL" 섹션을 찾습니다.
- 3단계에서 얻은 가상 GBP 지도 URL을 이 섹션에 붙여넣습니다.

![비즈니스 맵 URL 입력](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155034307941/original/JmUXYoQclVhV3KXuW8rG-KHe_amg41skVw.png?1728387757)

## 6단계: 더 정확한 결과 생성

- URL을 입력한 후, 나머지 폼을 평소대로 작성합니다.
- 폼을 제출하여 프로스펙팅 보고서를 생성합니다.
- 이제 프로스펙팅 도구가 이 URL을 사용하여 가상 또는 서비스 지역 비즈니스에 대한 더 정확한 결과를 제공합니다.

프로스펙팅 도구에서 올바른 가상 GBP 지도 URL을 사용하는 것은 가상 또는 서비스 지역 비즈니스에 대한 정확한 결과를 생성하는 데 매우 중요합니다.
