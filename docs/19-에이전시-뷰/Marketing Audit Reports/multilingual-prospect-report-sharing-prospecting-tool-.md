# 다국어 잠재 고객 리포트 공유 (Prospecting Tool)

마케팅 감사 리포트를 잠재 고객이 선호하는 언어로 공유하여 신뢰를 구축하세요. 이 글에서는 마케팅 감사 리포트를 다양한 언어로 공유하는 방법과, 전송 후 잠재 고객이 선호하는 언어로 변경하는 방법을 안내합니다.

---

## 다국어 잠재 고객 리포트 공유란?

다국어 잠재 고객 리포트 공유는 Prospecting Tool의 마케팅 감사 리포트 웹 버전을 현지화하는 기능입니다. 에이전시와 하위 계정에서 공유 링크를 생성할 때(또는 이메일/SMS로 전송할 때) 언어를 선택할 수 있으며, 잠재 고객도 리포트 페이지의 드롭다운에서 언어를 변경할 수 있습니다.

이를 통해 영어가 아닌 언어를 사용하는 대상에게 더 명확한 정보를 제공하고 참여도를 높일 수 있습니다. 다국어 잠재 고객 리포트 공유는 에이전시 레벨과 하위 계정 레벨 모두에서 작동합니다!

참고: 다국어 잠재 고객 리포트 공유는 웹 리포트에서만 사용 가능합니다(PDF 내보내기는 지원하지 않음).

---

## 다국어 잠재 고객 리포트 공유의 주요 이점

이러한 장점을 이해하면 잠재 고객 개발 프로세스에서 다국어 리포트를 어디에 활용할지 결정하는 데 도움이 됩니다!

- **빠른 이해**: 잠재 고객이 자신의 언어로 핵심 인사이트를 읽을 수 있어 혼란이 줄어들고 후속 작업 시간이 단축됩니다.

- **높은 전환율**: 현지화된 리포트가 신뢰성을 구축하고 응답률을 높입니다.

- **유연한 공유**: 공유 시 언어를 선택하되, 잠재 고객이 필요에 따라 페이지에서 언어를 변경할 수 있습니다.

- **옴니채널 친화적**: 이메일/SMS로 공유한 링크가 선택한 언어로 열립니다.

- **지원 요청 감소**: 영어가 아닌 언어를 사용하는 잠재 고객의 설명 요청이 줄어들어 영업 및 지원팀의 시간을 절약합니다.

## 지원 언어

어떤 언어를 사용할 수 있는지 알면 특정 시장으로의 아웃리치를 계획하고 고객 및 영업팀과 기대치를 설정하는 데 도움이 됩니다. 지원 언어는 다음과 같습니다:

- English(영어)
- German(독일어)
- Spanish(스페인어)
- French(프랑스어)
- French (Canadian)(캐나다 프랑스어)
- Italian(이탈리아어)
- Dutch(네덜란드어)
- Danish(덴마크어)
- Norwegian(노르웨이어)
- Finnish(핀란드어)
- Swedish(스웨덴어)
- Portuguese (Brazil)(브라질 포르투갈어)
- Portuguese (Portugal)(포르투갈 포르투갈어)

## 공유 시 언어 선택하는 방법

공유 전에 언어를 선택하면 수신자가 올바른 언어로 웹 리포트에 도착하면서도, 필요할 때 나중에 언어를 변경할 수 있습니다.

1. **마케팅(Marketing)** → Prospecting Tool로 이동하여 원하는 잠재 고객의 마케팅 감사 리포트를 찾거나 생성합니다.

2. **리포트 공유(Share Report)**를 클릭합니다.

3. 채널을 선택합니다: **이메일(Email)**, **SMS** 또는 **링크 공유(Share Link)**.

4. **리포트 언어 선택(Select Report Language)**에서 수신자가 처음에 볼 언어를 설정합니다.

5. 전송 세부사항을 완료한 후 리포트를 전송하거나 링크를 복사해서 공유합니다.

## 잠재 고객 측 언어 변경

잠재 고객이 웹 리포트에서 직접 언어를 변경할 수 있도록 하여, 다른 언어를 선호하는 팀이 편안하게 인사이트를 볼 수 있게 합니다.

1. 공유된 웹 리포트 링크를 엽니다.

2. 페이지 상단 우측에서 드롭다운에서 언어를 선택합니다.

3. 리포트가 선택한 언어로 즉시 업데이트됩니다.

## 자주 묻는 질문

**Q: PDF 내보내기에서도 작동하나요?**
A: 아직은 지원하지 않습니다. 다국어 지원은 현재 웹 리포트에만 적용됩니다. PDF 다국어 지원은 계획 중입니다.

**Q: 링크를 이미 보낸 후에 언어를 변경할 수 있나요?**
A: 수신자가 새 링크 없이도 공개 리포트에서 언어 드롭다운을 사용하여 언어를 변경할 수 있습니다.

**Q: 리포트의 어느 부분이 번역되나요?**
A: 시스템 라벨과 UI 텍스트가 현지화됩니다. **직접 입력하는 비즈니스 데이터**(예: 이름, 주소, 커스텀 메모)는 입력한 그대로 유지됩니다.

**Q: 이메일/SMS 메시지도 자동으로 번역되나요?**
A: 아닙니다. 리포트만 현지화되며, 이메일/SMS 내용은 번역되지 않습니다. 수신자가 선호하는 언어로 메시지를 작성하세요.

**Q: 에이전시와 하위 계정 레벨 모두에서 사용할 수 있나요?**
A: 네. 두 레벨 모두에서 공유할 때 리포트 언어를 선택할 수 있습니다.

**Q: 수신자의 언어를 확실히 모를 때는 어떻게 하나요?**
A: 공유 시에는 가장 가능성이 높은 언어를 선택하고, 뷰어가 리포트 페이지에서 언어를 변경할 수 있다는 점을 참고하세요.
