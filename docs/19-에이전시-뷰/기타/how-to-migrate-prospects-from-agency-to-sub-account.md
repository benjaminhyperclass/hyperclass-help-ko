# 프로스펙트 이전이란?

프로스펙트 이전은 에이전시가 에이전시 레벨 프로스펙팅 대시보드에서 선택된 프로스펙트를 특정 하위 계정으로 이동시킬 수 있게 해주는 기능입니다. 이는 리드를 관리하는 더 개인화되고 효율적인 방법을 제공하여, 하위 계정이 자체 워크플로우와 도구를 사용해 고객과 소통할 수 있게 합니다.

프로스펙트를 하위 계정으로 이전함으로써, 에이전시는 에이전시 레벨에서 가시성과 소유권을 유지하면서 개별 팀이나 클라이언트에게 후속 작업을 위임할 수 있습니다. 이를 통해 더 집중적인 마케팅 활동과 더 나은 리드 전환이 가능합니다.

## 사전 요구사항

- 대상 하위 계정에서 Premium Prospecting이 활성화되어 있어야 합니다.
- 에이전시 레벨에서 사용 가능한 프로스펙트가 있어야 합니다.

## 프로스펙트 이전의 주요 장점

이 기능은 운영 유연성을 향상시키고 에이전시가 프로스펙팅 활동을 효율적으로 확장할 수 있도록 도움을 줍니다:

- 고급 육성: 하위 계정 전용 자동화, 파이프라인, 커뮤니케이션을 활용할 수 있습니다.
- 운영 위임: 후속 조치를 위해 적절한 하위 계정에 프로스펙트를 배정할 수 있습니다.
- 개선된 프로스펙트 추적: 모든 프로스펙트 상호작용을 계정별로 중앙화하고 체계적으로 관리할 수 있습니다.
- 에이전시 제어: 이전된 프로스펙트는 향후 사용을 위해 에이전시 레벨에서 계속 접근할 수 있습니다.

## 프로스펙트 이전 기능 사용 방법

프로스펙트를 하위 계정으로 이동시키는 것은 빠르고 간단합니다. 성공적인 이전을 위해 다음 단계를 따라하세요:

### 1단계: 프로스펙팅 대시보드 접근

- Hyperclass 에이전시 계정에 로그인합니다.
- 왼쪽 네비게이션에서 Prospecting 옵션을 클릭합니다.

![프로스펙팅 메뉴 접근](https://jumpshare.com/v/aLIT8F4APsdh1jBLczxc+/Screen+Shot+2025-06-23+at+8.18.34+PM.png)

### 2단계: 이전 탭 열기

- Prospecting 섹션 내에서 상단의 Settings(설정) 탭을 클릭합니다.
- Settings(설정) 영역에서 Migrate 하위 탭을 선택하여 이전 인터페이스에 접근합니다.
- Migrate Prospects 버튼을 클릭합니다.

![이전 탭 열기](https://jumpshare.com/v/Ixb5j4rjx15W92zqBs7T+/Screen+Shot+2025-06-23+at+8.21.10+PM.png)

### 3단계: 하위 계정 선택

- 팝업 또는 드롭다운에서 제공된 목록에서 하위 계정을 선택합니다.

참고: 활성 Prospecting Premium 구독을 가진 하위 계정만 목록에 표시됩니다.

![하위 계정 선택](https://jumpshare.com/v/37RJbMb6WD6zjTQ3k1W2+/Screen+Shot+2025-06-23+at+9.51.27+PM.png)

### 4단계: 이전할 프로스펙트 선택

- Select Prospects 버튼을 클릭합니다.
- 드롭다운 목록에서 다음 중 하나를 선택할 수 있습니다:
  - 프로스펙트 하나 선택
  - 여러 프로스펙트 선택
  - 사용 가능한 모든 프로스펙트를 포함하려면 Select All 옵션 사용

![프로스펙트 선택](https://jumpshare.com/v/fJJwAxCpWenpWbV5HTYs+/GIF+Recording+2025-06-23+at+9.53.14+PM.gif)

### 5단계: 이전 실행

- 선택이 완료되면 Migrate 버튼을 클릭하여 이전을 시작합니다.

![이전 실행](https://jumpshare.com/v/cS9oCsLOtv9w1Mp5qM0m+/Screen+Shot+2025-06-23+at+9.58.21+PM.png)

## 이전 완료 후 어떻게 되나요?

이전이 완료되면 선택된 프로스펙트는:

- 하위 계정의 연락처(Contacts)에 추가됩니다.
- 전체 데이터, 노트, 히스토리를 그대로 유지합니다.
- 향후 이전이나 작업을 위해 에이전시 계정에서 계속 사용 가능한 상태로 남아 있습니다.

이러한 이중 가용성은 에이전시가 완전한 가시성과 제어를 유지할 수 있도록 보장합니다.

## 프로스펙트의 유연한 재사용

- 시간이 지나면서 동일한 프로스펙트를 여러 하위 계정으로 이전할 수 있습니다.
- 프로스펙트 데이터는 모든 로케이션에서 일관되고 최신 상태로 유지됩니다.

## 자주 묻는 질문

Q: 이전 후 에이전시에서 프로스펙트가 삭제되나요?
아니요, 프로스펙트는 하위 계정으로 복사되지만 에이전시 레벨에서는 그대로 유지됩니다.

Q: 동일한 프로스펙트를 여러 하위 계정으로 이전할 수 있나요?
네, 필요에 따라 프로스펙트를 다른 하위 계정으로 이전할 수 있습니다.

Q: 이전 후에도 프로스펙트의 히스토리, 노트, 데이터가 유지되나요?
네, 모든 관련 데이터는 이전 중에 그대로 유지됩니다.

Q: 대량으로 이전할 수 있나요?
네, 여러 프로스펙트를 선택해서 한 번에 이전할 수 있습니다.

Q: 이전 기능이 구독에 포함되나요?
네, 이 기능은 추가 비용 없이 Prospecting Premium 구독에 포함됩니다.

Q: 이전을 취소할 수 있나요?
프로스펙트는 에이전시에서 그대로 유지되지만, 하위 계정에서 수행된 변경사항은 수동으로 관리해야 합니다.
