# 하이퍼클래스 무료 vs 프리미엄 프로스펙팅 완전 가이드 – 에이전시를 위한

## 프로스펙팅이란?

프로스펙팅(Prospecting)은 귀하의 서비스로부터 혜택을 받을 가능성이 높은 잠재 고객을 찾고 평가하는 과정입니다. 하이퍼클래스에서 프로스펙팅은 고품질 리드를 발견하고, 마케팅 준비 상태를 분석하며, 서비스 제안 기회를 찾아내는 지능적인 리드 생성 도구입니다. 모든 작업이 대시보드 안에서 이루어집니다.

## 하이퍼클래스 프로스펙팅이란?

하이퍼클래스의 프로스펙팅 도구는 리드 생성 작업을 자동화하고 간소화하도록 설계되었습니다. 수동으로 비즈니스를 검색하고 감사를 실행하는 대신, 이제 다음을 할 수 있습니다:

- 비즈니스 데이터와 AI를 사용하여 리드를 찾고 평가
- 각 리드에 대한 감사 보고서를 생성하여 마케팅 존재감의 공백을 보여줌
- 이 데이터를 사용하여 서비스를 상향 판매하고, 가치를 입증하며, 파이프라인을 더 빠르게 구축

프로스펙팅은 두 가지 방법으로 이용할 수 있습니다:

- 기본(무료 티어)
- 프리미엄(유료 구독)

### 무료 프로스펙팅 – 업그레이드 전에 둘러보기

막 시작하거나 도구를 테스트해보고 싶다면, 무료 티어에서 필수 기능에 접근할 수 있습니다.

**에이전시에서 접근하는 방법:**
- 에이전시 대시보드로 이동
- 좌측 사이드바에서 "Prospecting(프로스펙팅)" 탭 클릭
- "Continue with Free(무료로 계속)" 클릭

**하위 계정에서 접근하는 방법:**
하위 계정의 무료 또는 기본 프로스펙팅은 'Agency Sub Account(에이전시 하위 계정)'로 표시된 하위 계정에서만 사용 가능하며, 현재 $497 플랜에서 제공됩니다.

**포함 내용:**
- 월 5개 프리미엄 보고서
- 일 50개 기본 보고서

**기본 보고서에 포함된 내용:**
- 비즈니스 리스팅 감사
- 기본 Google 비즈니스 프로필 분석
- 온라인 평판 스냅샷(리뷰, 평점 등)

⚠️ 무료 프로스펙팅은 에이전시 레벨에서만 사용 가능합니다(하위 계정에서는 불가).

### 프리미엄 프로스펙팅 – 월 $29로 모든 기능 활용

확장 준비가 된 에이전시를 위해 설계된 프로스펙팅 프리미엄으로 리드 생성을 다음 단계로 끌어올리세요.

**어디서 활성화할 수 있나요?**

다음 중 하나를 선택할 수 있습니다:
- 에이전시 레벨에서 프리미엄 프로스펙팅 구독(하나의 하위 계정을 무료로 연결!)
- 또는 하위 계정에서 직접 프리미엄 프로스펙팅 구매

## 프리미엄 프로스펙팅으로 얻을 수 있는 것들

### 월 무제한 보고서

더 이상 제한이 없습니다 — 제한 없이 프로스펙팅하고 보고서를 생성하세요.

### 무제한 보고서 새로고침

원할 때마다 보고서를 업데이트하고 재생성할 수 있습니다!

### AI 기반 프로스펙팅

- AI가 대상 지역과 틈새시장을 기반으로 자격을 갖춘 리드를 자동으로 발견합니다
- AI가 며칠마다 바로 사용할 수 있는 프리미엄 보고서를 대시보드에 직접 제공합니다

### 웹사이트 성능 감사

- 잠재 고객의 웹사이트 성능(속도, SEO, 모바일 친화성 등)을 확인하세요
- 공백을 파악하고 제안의 일부로 수정 방안을 제시하세요

![웹사이트 감사](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155047852020/original/jaD4XfOD8Ef8uZiS9bj63MxriZ-dotkKow.jpeg?1749196272)

### 로컬 SEO 히트맵 & 순위

- 잠재 고객 키워드의 지리적 가시성을 확인하세요
- 각 핀 위치에서 지도상 상위 순위 경쟁업체를 확인하세요

[프로스펙팅 도구의 SEO 히트맵: 지원 가이드](seo-heatmaps-in-the-prospecting-tool-a-support-guide.md)

![SEO 히트맵](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155047852540/original/MVlJxv5_UJK9SrAtTa0981hTkhlA9_2tTA.jpeg?1749196795)

### 리드 생성 위젯

- 감사를 리드 자석으로 웹사이트에 삽입하세요
- 브랜드 스타일에 맞게 완전히 커스터마이징 가능합니다

### CRM & 아웃리치 연동 (하위 계정에서만 사용 가능)

- 잠재 고객을 GHL CRM, 기회 관리 파이프라인, 워크플로우로 전송하세요
- CRM에서 직접 이메일과 SMS를 통해 연락하세요
- 빠른 후속 조치를 위한 AI 추천 메시지 템플릿을 받으세요

### 자동화 지원 (하위 계정에서만 사용 가능)

새로운 잠재 고객이 생성되자마자 워크플로우를 트리거합니다. 다음을 할 수 있습니다:

- 환영/후속 조치 메시지 자동 발송
- 잠재 고객 연락처 정보 저장
- 육성 시퀀스에 등록

[프로스펙팅 도구의 새로운 잠재 고객을 위한 워크플로우 트리거](../Premium-Prospecting/workflow-trigger-for-new-prospects-in-the-prospecting-tool.md)

## 프리미엄 프로스펙팅 활성화 방법

### 에이전시 레벨에서

- 좌측 네비게이션 메뉴에서 프로스펙팅 탭으로 이동
- 우상단의 "Unlock Premium(프리미엄 잠금해제)" 클릭하거나 "Upgrade Now(지금 업그레이드)" 선택
- 결제 및 설정 단계를 따라하세요

### 하나 이상의 하위 계정에서

- 에이전시 뷰에서 Reselling(재판매) 탭으로 이동
- 하위 계정에 대해 프로스펙팅을 활성화하세요
- 기능을 재판매하려면 가격을 설정하세요
- 이제 하위 계정에서 자체 대시보드에서 프리미엄을 활성화할 수 있습니다

## 빠른 요약: 무료 vs 프리미엄

| 기능 | 무료 (에이전시만) | 프리미엄 (에이전시 + 하위 계정) |
|------|------------------|---------------------------|
| 보고서 제한 | 월 5개 프리미엄/일 50개 기본 | 무제한 |
| 보고서 유형 | 기본 감사 (리스팅, GBP, 리뷰) | 전체 감사 + 웹사이트, SEO 히트맵 |
| AI 프로스펙팅 | ❌ 포함되지 않음 | ✅ 포함됨 |
| CRM 연동 | ❌ | ✅ |
| 자동화 | ❌ | ✅ |
| 리드 생성 위젯 | ❌ | ✅ |
| 이메일/SMS 아웃리치 | ❌ | ✅ |
| 하위 계정 접근 | ❌ | ✅ (연결을 통해) |
