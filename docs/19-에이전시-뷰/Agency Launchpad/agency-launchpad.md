# 에이전시 런치패드

Hyperclass의 에이전시 계정용 런치패드(LaunchPad)에 대한 개요를 안내해 드립니다! 이 기능은 에이전시 사용자에게 가이드형 비디오 기반 온보딩 경험을 제공하여, 빠르게 시작하고 가치를 실현할 수 있도록 도와드립니다.

**중요**: 이 기능은 현재 신규 가입자에게만 지원됩니다.

---

## 에이전시 런치패드란?

에이전시 계정용 런치패드는 Hyperclass를 사용하는 마케팅 에이전시와 SaaS 창업자를 위해 특별히 설계된 큐레이션된 온보딩 도구입니다. 새로운 에이전시 계정에 처음 접속했을 때 무엇부터 해야 할지 막막한 대신, 이 런치패드가 새로운 관리자에게 짧은 튜토리얼 비디오와 실행 가능한 프롬프트를 통해 필수 설정 단계를 안내합니다. 이러한 접근 방식은 새로운 사용자가 다음에 무엇을 해야 할지 이해하도록 도와 혼란을 줄이고 성공으로 가는 길을 가속화합니다.

![에이전시 런치패드 메인 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048034191/original/f7lZPRFM5fGXv5TtXt8Urne9HZAGQwQUQA.png?1749581307)

에이전시 계정용 런치패드는 에이전시가 더 빠르고 효과적으로 온보딩할 수 있도록 도와주는 여러 가지 장점을 제공합니다:

- **가속화된 온보딩**: 사용자가 주요 설정 작업을 안내받아 학습 곡선과 가치 실현 시간을 단축할 수 있습니다.

- **비디오 기반 가이드**: 각 작업에는 짧고 집중된 비디오 튜토리얼이 포함되어 복잡한 단계도 쉽게 따라할 수 있습니다.

- **즉각적인 명확성 향상**: 명확하고 실행 가능한 가이드를 제공하여 에이전시가 다음에 무엇을 해야 할지 정확히 알 가능성이 높아집니다.

## 에이전시 런치패드 사용하기

런치패드 설정과 사용은 간단합니다. 자격을 갖춘 에이전시 계정에 자동으로 활성화되기 때문입니다. 시작하고 최대한 활용하는 방법은 다음과 같습니다:

- **로그인**: Hyperclass 에이전시 계정에 로그인하세요.

- **런치패드로 이동**: (보이지 않는다면 롤아웃을 기다리거나 지원팀에 활성화를 요청하세요).

- 관리자는 역할 권한으로 런치패드를 숨길 수 있습니다. 보이지 않는다면 역할 권한을 확인하거나 에이전시 관리자에게 문의하세요.

- **가이드된 작업 검토**: 런치패드는 일련의 주요 온보딩 작업을 제시합니다. 가장 관련 있는 것부터 시작하세요.

- **클릭, 시청, 사용**: 작업을 클릭하면 Hyperclass의 해당 부분으로 직접 이동하고, 비디오를 시청하여 무엇을 해야 하는지 배운 다음, 플랫폼을 사용하여 작업을 수행하세요.

- **런치패드로 돌아가기**: 런치패드로 다시 이동하세요.

- **진행 상황 추적**: 작업을 완료하면 런치패드가 진행 상황을 반영하여 업데이트됩니다.

## 자주 묻는 질문

**Q: 에이전시 계정용 런치패드는 누가 사용할 수 있나요?**

$97, $297, $497 플랜의 모든 에이전시 계정이 대상입니다. 현재는 신규 계정 기준으로 롤아웃되고 있어 런치(2025년 6월 5일) 이후에 생성된 새 계정만 새로운 런치패드 버전에 접근할 수 있습니다.

**Q: 로그인했는데 런치패드가 보이지 않으면 어떻게 하나요?**

새로운 런치패드는 현재 런치 이후에 생성된 계정에만 롤아웃됩니다. 따라서 2025년 6월 5일 이전에 계정을 생성했다면 새로운 에이전시 런치패드에 접근할 수 없습니다.

**Q: 런치패드 밖에서도 튜토리얼 비디오를 볼 수 있나요?**

현재 비디오는 런치패드 경험에 통합되어 있으며 독립적인 라이브러리로는 제공되지 않습니다.

**Q: 하위 계정의 온보딩에는 어떤 영향을 주나요?**

하위 계정에는 에이전시 런치패드와 유사한 자체 런치패드 경험이 있습니다.

**Q: 우리 에이전시에서 일부 사용자나 계정에 대해 런치패드를 숨길 수 있나요?**

네. 관리자는 역할 레벨에서 런치패드 가시성을 제어할 수 있습니다. 또한 해당되는 경우 하위 계정별로, SaaS 플랜별로 런치패드를 켜거나 끌 수 있습니다.
