# SaaS: 다중 결제 공급자/UI 개선사항

## 개요

SaaS(Software as a Service) 하위 계정의 결제 페이지가 개선되어, Stripe 외에 다른 결제 옵션을 사용하는 SaaS 하위 계정에서 더 쉽게 이용할 수 있게 되었습니다. 새로운 개선사항을 확인해보세요:

## SaaS 활성화 프로세스 업데이트

에이전시에서 하위 계정의 SaaS를 수동으로 활성화할 때, 이제 구독 및 결제 작업을 처리할 계정 유형을 선택할 수 있습니다. Stripe 계정 또는 에이전시 계정 중에서 선택하면, 결제 시스템이 해당 계정의 기본 결제 공급자를 자동으로 사용합니다.

![SaaS 활성화 프로세스](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036471118/original/V18__2_FesOeHVYo8IYE2E8q0EJJjCOk7Q.png?1731489081)

## 유연한 결제 옵션

에이전시에서 이제 각 SaaS 하위 계정에 특정 결제 공급자를 설정할 수 있습니다. 하위 계정 관리자가 결제 수단을 추가하면, 에이전시가 해당 하위 계정에 대해 선택한 공급자로 저장됩니다.

![유연한 결제 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036471050/original/weJwSfEwdaOOwi8dY4O4e7aglbKSC7ZPLA.jpeg?1731489056)

## 요금 및 인보이스

하위 계정에서 이제 결제 페이지에서 바로 요금 내역과 인보이스에 쉽게 접근할 수 있습니다. SaaS 서비스와 관련된 요금을 포함하여 해당 계정과 연결된 모든 요금을 확인할 수 있습니다.

![요금 및 인보이스](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036471003/original/RkBDJiydKhr3qdc5ZmldDYmUwJe7JqRbVg.jpeg?1731489035)

## 청구 주소

하위 계정 관리자가 청구 주소를 추가하면, 계정 정보와 함께 자동으로 저장되어 필요할 때 쉽게 관리하고 불러올 수 있습니다.
