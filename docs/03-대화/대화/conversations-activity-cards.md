# 대화 - 활동 카드

사용자가 플랫폼과 상호작용하는 방식을 재정의하는 혁신적인 기능인 활동 카드(Activity Cards)를 소개합니다. 이 혁신적인 기능은 필수 업데이트를 통합된 뷰로 보여주는 창으로, 사용자 경험을 간소화하고 실시간 인사이트를 제공합니다. 중요한 변경사항을 실시간으로 파악할 수 있도록 도와줍니다. 더 이상 지연이나 다른 모듈로 이동할 필요가 없습니다. 의사결정 능력을 향상시키는 실시간 인식 효과를 제공합니다.

이 카드들은 액션을 사용자에게 가져다줍니다. 패널을 이동하는 번거로움 없이 기회, 예약, 연락처 설정, 인보이스에 대해 응답하고, 편집하고, 관리할 수 있습니다. 결과적으로 더욱 간소화되고 효율적이며 손쉬운 사용자 경험을 제공합니다.

## 1. 기회 관리

새로운 기회가 생성, 수정 또는 삭제되는 즉시 활동 카드가 표시됩니다. 잠재 리드의 시작부터 인식할 수 있어 항상 연결 상태를 유지할 수 있습니다. 카드 내 '기회 보기(View Opportunity)' 버튼을 클릭하면 바로 들어가서 현재 위치에서 실시간 변경사항을 적용할 수 있습니다.

![기회 관리 활동 카드](https://jumpshare.com/share/KHwNOA80up1drCv1sRXU+/Screen+Shot+2026-02-04+at+7.22.36+PM.png)

## 2. 예약

예약이 생성, 일정 변경 또는 취소될 때마다 대화 패널에 활동 카드가 나타납니다. 이 기능은 여러 화면을 오가지 않고도 지속적으로 업데이트를 받을 수 있도록 설계되었습니다. 카드 내 '예약 보기(View Appointment)'를 클릭하면 대화 패널에 머물면서도 예약 세부사항을 원활하게 편집할 수 있습니다.

![예약 활동 카드](https://jumpshare.com/share/7uo0nwrg1XjbH8zeK3bZ+/Screen+Shot+2026-02-04+at+7.27.58+PM.png)

## 3. 수신 거부 (DND)

연락처가 "수신 거부"로 표시되거나 DND 목록에서 제거되는 순간 활동 카드가 알려줍니다. 이 카드들은 명확성을 제공하고 현재 연락처의 선호도에 대해 정보를 유지해줍니다.

### 활동 카드에서 DND 출처 표시

Hyperclass는 대화(Conversations)에서 DND 상태 변경을 누가 발생시켰는지 보여줍니다:

- **DND 활성화:** 워크플로우/사용자/연락처가 특정 채널 또는 모든 채널에 대해 활성화
- **DND 비활성화:** 워크플로우/사용자/연락처가 특정 채널 또는 모든 채널에 대해 비활성화

"출처" 값을 사용하여 변경 근원을 파악하세요:

- **워크플로우:** 워크플로우가 DND를 활성화하거나 비활성화함
- **사용자:** 팀 멤버가 DND를 활성화하거나 비활성화함
- **연락처:** 연락받은 사람이 DND를 활성화하거나 비활성화함

이는 모든 채널에 적용되며 별도 설정이 필요하지 않습니다.

## 4. 인보이스

연락처에게 인보이스가 전송되거나 결제가 접수될 때마다 관련 활동 카드가 나타납니다. '인보이스 보기(View Invoice)'를 간단히 클릭하면 자세한 인보이스를 탐색할 수 있는 새 탭으로 원활하게 리디렉션됩니다.
