# 페이스북 & 인스타그램 빠른 답변 가이드

## **개요**

빠른 답변 기능을 사용하면 페이스북과 인스타그램 메시지에 대한 미리 정의된 응답 프롬프트를 만들 수 있어서 더 빠르고 효율적인 소통이 가능합니다. 이 기능은 문의가 자주 들어오는 비즈니스에 특히 유용하며, 신속한 응답으로 고객 참여도를 유지할 수 있습니다.

## 빠른 답변 설정하기:

- 대시보드의 자동화 섹션으로 이동해서 상단의 **"Workflows(워크플로우)"**를 선택하고 **"Create Workflow(워크플로우 생성)"**을 클릭하거나 기존 워크플로우를 편집하세요.

![워크플로우 생성 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036408900/original/lkPJOf5JrrO40XXLlS019ID_9ofuz2dQWw.png?1731410097)

- 페이스북이나 인스타그램 게시물에 고객이 댓글을 남기는 등의 트리거를 설정하고, 필요한 경우 특정 게시물을 선택하세요.

![트리거 설정 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036409697/original/Fecr-YY7jrWWnlmRWHjA3oe-5ZdeIMoiUg.png?1731410629)

- 페이스북의 인터랙티브 메시징 옵션에서 "Reply to comment via DM(DM으로 댓글 답변)" 또는 "Reply via DM(DM으로 답변)"을 선택하세요. 마케팅 섹션에서 템플릿이나 "스니펫"을 아직 설정하지 않았다면 먼저 설정해주세요.

![DM 답변 설정 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036738833/original/fEukGTost3oTHosq5lmvzyVcE4_cMzQXhg.jpeg?1731932607)

- 설정 영역에서 미리 정의된 옵션의 버튼을 추가할 수 있습니다. 아래 예시에서는 "질문이 있어요"라는 옵션으로 버튼을 추가했고, 이를 클릭하면 전화번호로 연결되도록 설정했습니다.

![버튼 옵션 설정 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036741552/original/A49x9sr3OoDBVOAwxSccGnuEp6f8r3THhw.png?1731934628)

- 빠른 답변은 워크플로우 액션의 Facebook Interactive Messenger(페이스북 인터랙티브 메신저)와 Instagram Interactive Messenger(인스타그램 인터랙티브 메신저)를 사용해서 페이스북이나 인스타그램에 각각 설정할 수 있습니다. 최대 13개의 빠른 답변을 설정할 수 있으며, 각각 최대 20자까지 입력 가능합니다.

![빠른 답변 설정 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036745039/original/H2sel0kmpP7Fy82oxI34C6flRrornQ8NzA.png?1731936733)

![빠른 답변 옵션 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155035120341/original/LPxNSf31EvECrdzoH-cJ8vzNOBQTagXCLw.png?1729526980)

- 빠른 답변은 고객의 인터페이스에 미리 정의된 옵션으로 표시되어서 한 번의 탭으로 쉽게 선택하고 응답을 보낼 수 있습니다.

![고객 화면의 빠른 답변 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155035120368/original/ZuMrhNRz0zCZI41CDnyYrEZ3BMNpmUrStg.png?1729527006)

- 대화(Conversations)에서 사용자는 메시지 아래에 모든 빠른 답변을 볼 수 있습니다.

![대화 화면의 빠른 답변](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155035120429/original/CcpSxKVIu4_4BhnWMpPYuvPsuljuGw1zyA.png?1729527063)

- 선택된 빠른 답변은 수신 메시지로 전송됩니다. 대화 페이지에서는 파란색으로 표시됩니다.

![선택된 빠른 답변 표시](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155035120461/original/_zxKvWe_9LDGIB2xfAD2Pop3phgf0kCp0Q.png?1729527105)

**주의사항:**

- 빠른 답변은 가장 최근 메시지의 일부로만 표시되며, 한 번의 탭으로 응답하는 기능입니다.
- 빠른 답변이 선택되면 고객은 선택을 변경할 수 없으므로 집중된 상호작용이 보장됩니다.

**채널별 조건:**

- 페이스북: 빠른 답변을 버튼 및 이미지와 함께 추가할 수 있어서 고객 상호작용에 더 많은 옵션을 제공합니다.
- 인스타그램: 메시지에 버튼 또는 빠른 답변 중 하나만 추가할 수 있습니다. 빠른 답변은 동일한 메시지에 첨부파일이 포함되지 않은 경우에만 추가할 수 있습니다.
