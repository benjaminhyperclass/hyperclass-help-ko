# 대화에서 쉽게 사용하는 첨부파일

첨부파일은 중요한 파일, 문서, 이미지, 동영상을 공유할 수 있게 해주어 커뮤니케이션을 향상시키는 핵심적인 역할을 합니다. 계약서, 제품 브로셔, 교육 동영상 등 무엇을 보내든 첨부파일은 정보를 효율적으로 전달하는 데 도움이 됩니다. Hyperclass에서는 이메일, SMS, 미디어 라이브러리에 첨부파일을 포함시킬 수 있어 다양한 커뮤니케이션 채널에서 원활한 파일 공유가 가능합니다.

## 첨부파일이란?

첨부파일을 통해 사용자는 대화(Conversations) 내에서 파일을 주고받을 수 있어, 고객과 중요한 문서, 이미지, 동영상을 쉽게 공유할 수 있습니다. Hyperclass에서는 기기에서 직접 업로드하거나 미디어 라이브러리에서 선택할 수 있습니다. SMS 전송 시 너무 큰 파일이거나 이메일 20MB 제한을 초과하는 파일은 자동으로 미디어 라이브러리에 업로드되어 링크로 공유됩니다. 이를 통해 파일 크기 제한 없이 원활한 전달이 보장됩니다.

## 첨부파일의 주요 장점

- **향상된 커뮤니케이션**: 계약서, 이미지, 동영상, PDF를 쉽게 공유하여 메시지를 더 유익하고 효과적으로 만들 수 있습니다.

- **유연한 업로드 옵션**: 기기에서 첨부파일을 업로드하거나 미디어 라이브러리에 저장된 파일을 선택할 수 있어 파일 공유가 더욱 편리합니다.

- **자동 미디어 라이브러리 업로드**: 파일이 이메일 크기 제한(20MB)을 초과하거나 SMS로 전송될 때 자동으로 미디어 라이브러리에 업로드되어 링크로 공유됩니다.

- **매력적인 이메일 썸네일**: 미디어 라이브러리를 통해 업로드된 동영상은 자동으로 GIF 썸네일을 생성하여 이메일을 더욱 인터랙티브하고 시각적으로 매력적으로 만듭니다.

## 첨부파일 사용 방법

대화(Conversations)에서 첨부파일은 몇 가지 간단한 단계를 통해 빠르게 추가할 수 있습니다. 이미지, PDF, 동영상 등 무엇을 보내든 기기에서 업로드하거나 미디어 라이브러리에서 파일을 선택할 수 있습니다. 아래는 첨부파일을 추가하고 관리하는 단계별 가이드입니다.

### 대화 섹션 접근하기

Hyperclass 내의 대화(Conversations) 탭으로 이동한 후, 기존 대화를 선택하거나 연락처와 새 메시지를 시작하세요.

![대화 섹션 접근하기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042894436/original/Eo-zE2F7q_BeGzYCBFb5wu1sS1YCHUQb5A.png?1741433091)

### 메시지에 파일 첨부하기

SMS나 이메일 작성창의 클립 아이콘을 클릭하여 첨부파일 옵션을 엽니다. 기기에서 파일을 선택하거나 이전에 업로드한 경우 미디어 라이브러리에서 파일을 선택하세요. 선택하면 파일이 메시지에 첨부되어 전송할 준비가 완료됩니다.

![메시지에 파일 첨부하기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042894575/original/T9w5u-URZK_4kvOJvydLp9z_5QGFikyFfw.png?1741433485)

### 클립보드에서 첨부파일 붙여넣기

메시지 작성창에 직접 붙여넣어 파일을 첨부할 수도 있습니다.

1. 컴퓨터에서 지원되는 파일이나 이미지를 복사합니다. 지원되는 채널(SMS, 이메일, 왓츠앱 등)의 메시지 작성창 내부를 클릭합니다.

2. **Ctrl + V**(Windows) 또는 **Cmd + V**(Mac)를 누릅니다.

3. 붙여넣은 파일이 자동으로 첨부파일로 나타납니다.

참고: 클립보드 첨부파일도 대화(Conversations)의 다른 첨부파일과 동일한 파일 유형 및 크기 제한을 따릅니다.

![클립보드에서 첨부파일 붙여넣기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155060704344/original/Ltu48pPvRWl-n9IUNWLbMdpIHZOSGi35cg.gif?1765879814)

### 업로드 방법 선택하기

**기기에서 업로드**: 로컬 저장소에서 파일을 선택합니다(예: 이미지, PDF, 동영상).

**미디어 라이브러리 사용**: 이전에 업로드한 파일이 있다면 미디어 저장소에서 선택하여 같은 파일을 다시 업로드할 필요가 없습니다.

![업로드 방법 선택하기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042894577/original/e_8kewNhfTc5r1UWWN1-fPka2kFmBr2Jsg.png?1741433516)

### 자동 미디어 라이브러리 업로드

SMS 첨부파일이나 20MB를 초과하는 이메일 첨부파일이 업로드되면, 시스템이 자동으로 해당 파일을 미디어 라이브러리로 이동시킵니다.

파일을 직접 첨부하는 대신 미디어 라이브러리 링크가 메시지에 삽입되어 크기 제한 없이 원활한 전달을 보장합니다.

![자동 미디어 라이브러리 업로드](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042894610/original/Ujfupsvnie7VKN6SqT8XXCoS-B8U48sXJQ.png?1741433629)

### 인터랙티브 동영상 썸네일

미디어 라이브러리를 통해 업로드된 동영상은 자동으로 GIF 썸네일 미리보기를 생성합니다.

이러한 애니메이션 미리보기는 이메일을 더욱 매력적으로 만들고 수신자가 콘텐츠와 상호작용하도록 유도합니다.

![인터랙티브 동영상 썸네일](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042894654/original/ibYj_4GhpuFyVAAukhWh1Jh7qoWxx09jWg.png?1741433694)

## 자주 묻는 질문

**Q: 첨부파일로 어떤 유형의 파일을 업로드할 수 있나요?**

이미지, 문서, PDF, 동영상을 첨부파일로 업로드할 수 있습니다. 단, 이메일 첨부파일은 20MB로 제한되며, 더 큰 파일은 자동으로 미디어 라이브러리로 이동되어 링크로 공유됩니다.

**Q: 미디어 라이브러리에서 업로드한 파일에 어떻게 접근하나요?**

모든 업로드된 파일은 자동으로 미디어 라이브러리에 저장되며, 대화(Conversations) 인터페이스에서 접근할 수 있습니다. 이를 통해 사용자는 수동으로 다시 첨부할 필요 없이 이전에 업로드한 파일을 재사용할 수 있습니다.

**Q: 이메일 크기 제한을 초과하는 첨부파일을 보내려고 하면 어떻게 되나요?**

첨부파일이 20MB를 초과하면 자동으로 미디어 라이브러리에 업로드되고 파일 링크가 이메일에 삽입됩니다. 이를 통해 큰 파일도 전송 문제 없이 공유할 수 있습니다.

**Q: SMS로도 첨부파일을 보낼 수 있나요?**

네, SMS로도 첨부파일을 보낼 수 있지만 SMS에는 크기 제한이 있어서 파일이 자동으로 미디어 라이브러리 링크로 변환되어 원활한 전송이 가능합니다.

**Q: 인터랙티브 동영상 썸네일은 어떻게 작동하나요?**

동영상이 미디어 라이브러리를 통해 업로드되면 Hyperclass가 자동으로 GIF 썸네일 미리보기를 생성합니다. 이는 이메일을 시각적으로 더 매력적으로 만들고 수신자가 동영상 콘텐츠와 상호작용하도록 유도하여 참여도를 높입니다.
