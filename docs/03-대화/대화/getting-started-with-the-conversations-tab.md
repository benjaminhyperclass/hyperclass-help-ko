# 대화(Conversations) 탭 시작하기

하이퍼클래스의 새롭게 디자인된 대화(Conversations) 기능은 더 빠르고 명확한 인박스를 제공합니다. 4개의 유연한 패널, 업그레이드된 필터링(태그 + AND/OR 조건 포함), 그리고 현대적인 이메일 작성기가 포함되어 있어요. 팀은 스레드를 떠나지 않고도 분류, 협업, 작업을 수행할 수 있습니다. 이 가이드에서는 새로운 기능과 그것이 어떤 도움이 되는지 설명드릴게요.

## 새로운 대화 기능이란?

대화(Conversations) 모듈이 더 빠르고 직관적이며 사용자 친화적인 경험을 제공하기 위해 완전히 재설계되었습니다. 인터페이스의 모든 요소가 명확성과 반응성을 위해 개선되어, 팀이 메시지 관리, 협업, 컨텍스트 접근을 원활하게 할 수 있어요. 새로운 디자인은 상당한 성능 개선, 고급 필터링 옵션, 그리고 더 부드럽고 효율적인 커뮤니케이션 경험을 위한 새로운 이메일 작성기를 도입합니다.

재설계된 기능에는 4패널 레이아웃, 연락처 컨텍스트를 위한 통합 우측 패널, 고급 필터링(태그 + AND/OR), 그리고 전체 화면 작성 또는 인라인 답장을 지원하는 업그레이드된 이메일 작성기가 포함되어 있어요.

## 새로운 대화 기능의 주요 이점

이러한 이점을 이해하면 관리자와 에이전트가 언제 새 UI를 활성화할지, 그리고 일상적으로 어떻게 사용해서 더 빠르고 정확하게 작업할지 결정하는 데 도움이 됩니다.

- **성능 개선**: 모듈 전반에서 측정 가능한 향상
- **더 나은 반응성**: 작업 반응 속도가 ~70% 빨라져 Google의 "좋음" 웹 바이탈 벤치마크 충족
- **더 빠른 네비게이션**: 페이지 이동이 ~35% 빨라짐
- **메모리 사용량 감소**: 메모리 소비가 ~18% 줄어들어 모든 기기에서 성능 향상
- **단일 화면 컨텍스트**: 연락처 세부정보, 문서, 결제, 활동이 스레드와 함께 표시되어 클릭 횟수 감소
- **정밀한 필터링**: 태그와 AND/OR 연산자로 집중된 큐를 만들고 뷰로 저장하여 원클릭 접근 가능
- **유연한 작성**: 긴 초안을 위한 전체 화면 이메일, 빠른 응답을 위한 인라인 답장

측정된 메모리 개선: 전체 메모리 사용량이 ~1.7GB에서 ~0.45GB로 감소. ~50개 연락처 간 전환 후에도 메모리가 ~450MB 이하 유지.

타임라인의 일관된 날짜 형식: 날짜가 이제 **월 이름**을 포함한 명확하고 일관된 형식(예: **1월 5일**)을 사용합니다. 이는 숫자 형식(mm/dd vs dd/mm)으로 인한 혼동을 방지하고 대화 타임라인을 더 쉽게 스캔할 수 있게 해줘요.

- **내장 협업**: 내부 댓글과 @멘션으로 팀 토론을 비공개로 정리

## 네비게이션과 레이아웃

재설계된 대화(Conversations) 모듈은 향상된 구성과 명확성을 위해 4패널 레이아웃을 도입했습니다. 이 패널들은 접을 수 있어서 선호도에 따라 작업 공간을 조정할 수 있어요:

- 인박스 패널을 접거나 펼치려면 왼쪽 패널 중앙에 있는 접기 아이콘을 클릭하세요.
- 우측 패널을 접거나 펼치려면 화면 오른쪽 가장자리의 탭 아이콘을 클릭하세요.

이러한 유연성으로 필요에 따라 더 많은 읽기 또는 답장 공간을 만들 수 있어요.

![새로운 대화 인터페이스 레이아웃](https://jumpshare.com/share/zo1pFU2Xulj4iVAX2B2s+/Screen+Shot+2026-02-03+at+6.01.20+PM.png)

## 1. 인박스 패널

인박스 패널은 보고 있는 대화 세트(개인, 팀, 내부 채팅)를 제어해서 페이지를 변경하지 않고도 빠르게 컨텍스트를 전환할 수 있어요.

- **내 인박스**: 나에게 배정되거나 내가 팔로우하는 대화
- **팀 인박스**: 계정 내 모든 대화(전체 데이터 접근 권한 필요)
- **내부 채팅**: 고객 스레드와 별개인 팀 전용 비공개 토론

접근 참고사항:
- **배정된 데이터 접근** 권한을 가진 계정은 **내 인박스**와 **내부 채팅**만 볼 수 있어요(팀 인박스 제외).
- **전체 접근 권한**을 가진 사용자는 세 가지 모두 보기/관리 가능

![인박스 패널 스크린샷](https://jumpshare.com/share/6lQiGVs2kxLSpAisk2Gr+/Screen+Shot+2026-02-03+at+6.04.19+PM.png)

## 2. 채팅 목록 패널

채팅 목록 패널은 대화 레이아웃의 두 번째 컬럼입니다. 인박스 패널에서 선택한 인박스, 폴더, 또는 필터를 기반으로 모든 대화의 목록을 표시해요. 채팅 목록 패널은 현재 인박스, 폴더, 또는 저장된 뷰와 일치하는 스레드를 보여줘서 빠르게 스캔하고 선택하며 일괄 작업을 수행할 수 있어요.

- 일괄 작업을 활성화하려면 상단의 **선택 박스**를 체크하세요: **읽음/읽지 않음 표시, 즐겨찾기 추가/해제, 삭제**
- 최신/오래된 활동순으로 정렬
- **필터** 아이콘을 사용해서 채널, 태그, 소유자, 날짜 필터 설정—**AND/OR** 조합 지원

![채팅 목록 패널](https://jumpshare.com/share/APFZFvGUSwra7DzTBKwq+/Screen+Shot+2026-02-03+at+6.06.54+PM.png)

## 3. 메시지 기록 패널

메시지 기록 패널은 주요 읽기 및 답장 영역입니다. 멀티 채널 타임라인, 빠른 답장, 메시지 유형 필터링을 지원해서 컨텍스트를 빠르게 찾을 수 있어요.

- **이메일, SMS, WhatsApp, Facebook/Instagram 메시지**, **내부 댓글**에 걸친 전체 기록 읽기
- 특정 항목에 집중하기 위해 표시되는 타임라인 필터링(다음 섹션 참조)

![메시지 기록 패널](https://jumpshare.com/share/uVrxwXvnC310lV8YGVsH+/Screen+Shot+2026-02-03+at+6.10.15+PM.png)

### 채널 간 전환

스레드를 떠나지 않고도 각 답장에 가장 적합한 채널을 선택할 수 있어요. SMS에서 이메일로 대화를 옮기거나 내부 노트를 추가하는 데 이상적이에요.

- 사용 가능한 채널 간 전환을 위해 작성기에서 채널 선택기를 여세요.
- **채널 선택은 접기/펼치기 시 유지됨**: 같은 대화 내에서 작성기를 접었다가 다시 열면, 대화(Conversations)가 마지막으로 선택한 채널을 기억해서 중단한 곳에서 계속할 수 있어요.

![채널 선택기](https://jumpshare.com/share/xIlZ5seZv71JVeIDDEBN+/OGzVi-Ns_g9vPzAJNLWHueH3gmLwZeCqpQ.png)

- **내부 댓글**을 선택해서 비공개 노트 추가(고객에게 알림 전송되지 않음)
- 내부 댓글 **초안**은 채널 전환, 작성기 접기/펼치기, 대화 변경에 걸쳐 **유지됨**

![내부 댓글 작성](https://jumpshare.com/share/pbcdoBpR2LH43DuuUaxe+/Screen+Shot+2026-02-03+at+6.19.13+PM.png)

### 메시지 작성기

작성기는 명확하게 작성하고, 파일을 첨부하고, 올바른 채널을 통해 전송하는 도구를 제공해요. 모두 대화 하단에서 가능합니다.

- 텍스트 서식 지정과 인라인 답장 작성기가 활성 메시지 바로 아래에서 열려서 답장하는 동안 컨텍스트가 보이도록 유지
- 긴 답장을 위한 전체 화면 모드
- 파일 첨부와 링크
- 클립보드에서 파일이나 이미지를 작성기에 직접 붙여넣기해서 빠르게 첨부(Windows에서 Ctrl + V, Mac에서 Cmd + V)
- 오른쪽 전송 아이콘을 사용한 빠른 전송
- 작성기에서 **선택한 채널**을 통한 답장 전송

### 대화 내 메시지 필터링

타임라인 필터는 긴 스레드의 특정 부분(이메일만, 내부 노트만, 또는 활동만)을 분리해서 컨텍스트를 빠르게 찾을 수 있게 도와줘요.

타임라인 상단 우측의 **메시지 필터**를 사용해서 다음을 표시하세요:

- 모든 메시지
- 대화(연락처와의 직접 교환)
- 활동(예약, 업데이트 등)
- 이메일 / 내부 댓글 / 연락처 / 기회 관리 / 결제 / 인보이스 / AI 액션 로그

필터 메뉴 내 **검색 바**를 사용해서 메시지나 활동 유형을 더 빠르게 찾으세요.

![메시지 필터](https://jumpshare.com/share/doS0DQfBXd1MOXH74P3f+/Screen+Shot+2026-02-03+at+6.20.25+PM.png)

## 4. 우측 패널

우측 패널은 연락처 컨텍스트를 중앙집중화해서 대화를 떠나지 않고도 세부 정보, 파일, 결제, 관련 기록을 보고 편집할 수 있어요. 연락처(Contacts) 페이지에서 사용할 수 있었던 모든 것(커스텀 필드, 폴더, 작업 포함)이 이제 이 패널에 통합되어 고객과 채팅하는 동안 완전한 제어와 가시성을 제공합니다.

우측 패널에서 할 수 있는 것:

- **연락처 필드**, **커스텀 필드**, **폴더**, **태그**, 소유자 보기 및 편집
- **기회 관리**, **예약**, **인보이스**, 결제 생성/보기
- **연락처**, **활동**, **연관 관계**, **문서**, 결제 등의 탭 전환

![우측 패널](https://jumpshare.com/share/6IIstUZnLc1sQXmDkqze+/Screen+Shot+2026-02-03+at+6.22.53+PM.png)

## 태그 필터

이제 태그를 기반으로 대화를 필터링할 수 있어서 카테고리나 워크플로우별로 정밀한 세분화가 가능해요.

예시:
- 높은 우선순위로 태그된 모든 대화 보기
- 후속 조치 필요로 표시된 모든 리드 보기
- 여러 태그를 동시에 적용해서 세밀한 제어

![태그 필터](https://jumpshare.com/share/X1qdN8qxGpEd1YLaYmzg+/Screen+Shot+2026-02-03+at+6.24.30+PM.png)

## AND / OR 연산자

새로운 필터링 시스템은 조건부 연산자를 지원해서 여러 기준을 조합하는 유연성을 제공해요:

- **AND**를 사용해서 선택한 모든 조건을 충족하는 대화 표시
  예시: 높은 우선순위로 태그되고 AND 나에게 배정된 대화

- **OR**를 사용해서 선택한 조건 중 하나라도 충족하는 대화 표시
  예시: 이메일 OR WhatsApp 채널의 대화

이를 통해 긴급 지원 요청을 분리하는 것부터 멀티 채널 캠페인을 검토하는 것까지, 특정 워크플로우와 일치하는 복잡한 필터를 더 쉽게 구축할 수 있어요.

![AND/OR 연산자](https://jumpshare.com/share/86qKDLV6jL0BeEdMYoHq+/Screen+Shot+2026-02-03+at+6.26.43+PM.png)

## 이메일 작성기 개선사항

재설계된 대화(Conversations) 모듈의 이메일 작성기는 더 명확하고 부드러우며 유연한 작성 경험을 위해 업그레이드되었습니다.

### 전체 보기 작성기

이제 작성기를 전체 보기 레이아웃으로 확장해서 더 길거나 상세한 이메일을 편안하게 작성할 수 있어요. 이는 서식 지정, 첨부 파일 추가, 전송 전 메시지 검토를 위한 더 많은 공간을 제공합니다.

![전체 보기 작성기](https://jumpshare.com/share/QIxlZv5dnIXd5rvitXDW+/GIF+Recording+2026-02-03+at+6.30.51+PM.gif)

### 인라인 답장 경험

기존 이메일 스레드에 답장할 때 작성기가 자동으로 활성 이메일 바로 아래에서 열려요. 이 레이아웃은 입력하는 동안 컨텍스트가 보이도록 유지해서 대화 기록에 집중할 수 있게 도와줘요.

답장 작성기를 닫거나 종료하려면 작성기 도구 모음 오른쪽의 삭제 아이콘(×)을 클릭하세요.

![인라인 답장](https://jumpshare.com/share/fiRqErvPd3TPAh5e1b2O+/Screen+Shot+2026-02-03+at+6.34.16+PM.png)

## 키보드 단축키

키보드 단축키는 네비게이션과 답장 속도를 높여줘요. 언제든지 단축키 패널을 열어서 사용 가능한 작업을 확인하고, 지원되는 경우 매핑을 사용자 지정할 수 있어요.

- **⌘/Ctrl + K** — 전역 검색
- **⌘/Ctrl + Enter** — 메시지 전송
- **⌘/Ctrl + Shift + E** — 전체 화면 작성기 토글
- 대화(Conversations)에서 **?**를 눌러서 전체 단축키 목록 열기

## 자주 묻는 질문

**Q: 저장된 답장, 템플릿, 서명이 그대로 유지되나요?**
A: 네. 기존 저장된 답장, 템플릿, 서명이 새 작성기에서 계속 사용 가능해요.

**Q: 키보드 단축키를 사용자 지정할 수 있나요?**
A: **?**를 눌러서 단축키 목록을 열 수 있지만 사용자 지정은 불가능해요.

**Q: 모든 채널에서 필터를 사용할 수 있고, 채널별로 필터링할 수 있나요?**
A: 네. 필터 아이콘을 사용해서 채널(이메일, SMS, WhatsApp 등)을 포함/제외할 수 있어요. AND/OR를 사용해서 태그 및 소유자와 조합하세요.

**Q: 팀이 필터로 만든 저장된 뷰를 공유할 수 있나요?**
