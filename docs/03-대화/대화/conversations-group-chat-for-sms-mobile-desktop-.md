# 대화(Conversations): SMS 그룹 채팅 (모바일 & 데스크톱)

SMS 그룹 채팅을 사용하면 팀이 데스크톱과 모바일 모두에서 하나의 대화 스레드로 소규모 연락처 그룹에게 메시지를 보낼 수 있습니다. 이 가이드에서는 그룹 채팅 기능을 사용하여 여러 고객, 팀원 또는 이해관계자와 대화를 시작하고, 더 빠르고 명확하며 효과적으로 소통하는 방법을 알려드립니다.

# **SMS 그룹 채팅이란?**

SMS 그룹 채팅 기능을 사용하면 그룹을 만들어 하나의 대화에서 여러 연락처에게 SMS 메시지를 보낼 수 있습니다. 여러 개의 일대일 대화를 따로따로 관리하는 대신, 이제 여러 연락처(가족, 내부 팀, 전체 부서 등)와 하나의 통합된 스레드에서 협력할 수 있습니다. 이를 통해 커뮤니케이션이 간소화되고 응답 지연이 줄어들며 모든 참여자 간의 투명성이 향상됩니다.

참고: 그룹 채팅은 **첨부 파일 송수신을 지원**합니다. 이미지는 인라인으로 표시되고 파일은 다운로드 가능한 칩 형태로 표시됩니다. 자세한 내용은 [SMS 첨부 파일 크기 제한](file-size-limits-for-attachments-in-sms-and-email.md)을 참고하세요.

---

## SMS 그룹 채팅의 주요 장점

- **협업 대화:** 여러 연락처를 하나의 스레드로 모아 간소화된 토론을 진행할 수 있습니다.

- **간소화된 커뮤니케이션:** 여러 채팅에서 메시지를 반복하지 않고도 모든 사람이 같은 내용을 볼 수 있습니다.

- **향상된 고객 참여:** 관련 연락처를 그룹화하여 상호작용을 맞춤화할 수 있습니다(예: 가족, 부서 또는 팀).

- **모바일 편의성:** 언제 어디서나 그룹 대화를 시작하고 관리하며 이어갈 수 있습니다.

- **향상된 응답성:** 모든 사람이 실시간으로 메시지를 보고 답변할 수 있습니다.

---

## 그룹 채팅 사용 요구사항

SMS 그룹 채팅 기능을 사용하려면 다음이 필요합니다:

- **전화번호:** SMS/MMS가 활성화된 **LeadConnector** 번호를 사용해야 합니다.

- **지역:** 참여자는 **미국/캐나다** 전화번호를 가져야 합니다.

- **접근 권한:** **본인 데이터만 보기** 권한이 있는 사용자는 자신에게 배정된 연락처만 포함하는 그룹을 만들고 볼 수 있습니다. 사용자는 그룹 채팅을 만들 때 사용한 전화번호에 접근 권한이 있는 경우에만 그룹에서 메시지를 보낼 수 있습니다.

---

## 그룹 채팅 동작 방식 및 제한사항

그룹 채팅을 만들기 전에 다음 동작 방식과 제한사항을 검토하세요.

- **최대 그룹 크기:** 그룹 채팅은 **2-9명의 참여자**를 지원합니다.

- **수신 거부(DND):** **수신 거부(Do Not Disturb, DND)** 상태인 연락처는 그룹에 추가하거나 메시지를 보낼 수 없습니다. 생성 후 참여자가 DND를 활성화하면 해당 그룹 스레드를 더 이상 사용할 수 없습니다. 그룹 메시징을 계속하려면 DND로 표시된 참여자를 제외하고 새 그룹을 만들어야 합니다.

- **변경 불가:** 생성 후에는 참여자를 추가/제거하거나 **발신** 번호를 변경할 수 **없습니다**. 멤버를 변경해야 할 때는 새 그룹을 만드세요.

- **참여자 연락처 업데이트 시:** 연락처가 나중에 삭제되거나 번호가 편집되어도 메시지는 그룹 생성 시 추가된 원래 번호로 계속 전달됩니다. 영향을 받는 연락처 옆에 업데이트됨/삭제됨 태그가 표시됩니다.

## 그룹 채팅 만들기 (모바일)

- iOS 또는 Android 기기에서 HighLevel 또는 LeadConnector 모바일 앱을 열고 화면 하단의 **대화(Conversations)** 탭을 누릅니다.

- 대화 유형으로 **그룹 채팅(Group Chat)**을 선택합니다.

- 전화번호를 선택하고 목록에서 여러 연락처를 선택하여 그룹 채팅에 추가합니다. 최소 2명, 최대 9명의 참여자를 추가해야 합니다.

- 선택사항으로 그룹 채팅에 이름을 지정하여 나중에 쉽게 식별할 수 있습니다.

- **채팅 시작(Start Chat)**을 눌러 즉시 대화를 시작합니다.

## 그룹 채팅 관리

그룹 채팅이 활성화되면 일대일 채팅과 동일하게 소통할 수 있습니다. 다음을 할 수 있습니다:

- 메시지 송수신
- 이미지나 파일 공유
- 채팅의 모든 참여자 보기

그룹 채팅 창 상단에서 추가된 모든 연락처를 볼 수 있습니다. 채팅 참여자(Chat Participants) 아이콘을 클릭하여 이름이나 전화번호로 검색할 수 있습니다.

## **자주 묻는 질문**

**Q: 그룹 생성 후 참여자나 비즈니스 번호를 편집할 수 있나요?**

아니요. 업데이트된 수신자 세트로 새 그룹을 만드세요.

**Q: 그룹 채팅에서 첨부 파일이 작동하나요?**

네, 하지만 통신사 파일 크기 제한을 준수해야 합니다. 전송이 실패하면 파일을 압축해 보세요. SMS 파일 제한에 대한 자세한 정보는 [SMS 첨부 파일 크기 제한](file-size-limits-for-attachments-in-sms-and-email.md)을 참고하세요.

**Q: 그룹 생성 후 참여자가 DND를 켜면 어떻게 되나요?**

해당 스레드의 그룹 메시징이 차단됩니다. DND 연락처를 제외한 새 그룹을 만들거나, 해당 연락처가 다시 수신 동의하도록 하여 스레드를 계속 사용하세요.

**Q: 모바일 앱에서 그룹 채팅을 관리할 수 있나요?**

네. HighLevel과 LeadConnector 모바일 앱에서 그룹 채팅을 만들고, 보고, 답장할 수 있습니다.

**Q: 그룹 채팅에서 알림은 어떻게 작동하나요?**

알림은 일대일 채팅과 동일하게 전달됩니다(그룹의 새 메시지마다 모바일 푸시 알림).

**Q: 모든 그룹 채팅 참여자가 과거 메시지를 볼 수 있나요?**

네. 그룹의 모든 멤버가 완전한 메시지 기록을 볼 수 있습니다.
