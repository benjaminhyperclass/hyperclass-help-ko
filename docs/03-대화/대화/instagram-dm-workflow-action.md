# 인스타그램 DM - 워크플로우 액션

### 인스타그램 DM - 액션:

워크플로우 액션 인스타그램 DM은 연락처에게 인스타그램 DM을 보낼 수 있게 해줍니다. (메시지가 성공적으로 전달되려면 연락처가 이 이벤트에 도달하기 전 24시간 이내에 연결된 인스타그램 페이지에 미리 메시지를 보낸 적이 있어야 합니다)

![인스타그램 DM 워크플로우 액션 설정 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155032276721/original/itNs6HpAjaQ_jnPlOZL05oYjlzskKv1teA.jpeg?1725507865)

기존 템플릿에서 선택하거나 직접 메시지를 작성할 수 있습니다. 파일 첨부도 가능합니다. "첨부파일 추가"를 클릭하고 드라이브에서 첨부파일을 선택하거나 URL을 입력해서 파일을 추가하세요.

이 액션은 트리거(Trigger) - "고객이 답장함"과 답장 채널이 "인스타그램 DM"일 때 함께 사용하세요.

**주의사항:**

- 이 액션은 연락처가 지난 24시간 이내에 DM을 보낸 경우에만 DM을 발송하는 용도로 사용해야 합니다.
- 이 액션을 "인스타그램 게시물 댓글" 워크플로우 트리거와 함께 사용하면, 인스타그램 게시물에 최근 댓글이 있을 경우 DM이 댓글 답변으로 전송됩니다.
- 최근 댓글이 없다면, 연락처가 지난 24시간 이내에 비즈니스에 DM을 보낸 적이 있는 경우에 한해 직접 DM으로 전송됩니다.

**권장사항:**

연락처에게 인스타그램 DM을 보낼 때는 "인스타그램 인터랙티브 메신저(Instagram Interactive Messenger)" 액션을 사용하세요.

- 댓글에 답변하는 경우, 답변 유형을 "DM으로 댓글에 답변"으로 설정하세요
- 직접 DM을 보내는 경우, 답변 유형을 "DM에 답변"으로 설정하세요

"인스타그램 인터랙티브 메신저" 액션에 대한 자세한 내용은 [여기](https://help.leadconnectorhq.com/support/solutions/articles/155000002188-facebook-instagram-interactive-messenger)에서 확인하세요.

![인스타그램 인터랙티브 메신저 액션 설정 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155032276722/original/weJlt1qkG6Oza04fP2ovn8Lyk-RykwpNpA.jpeg?1725507865)

**참고:**

2024년 8월 31일 이전에는 "인스타그램 DM" 워크플로우 액션을 통해 인스타그램 게시물에 댓글을 남긴 모든 연락처에게 지난 24시간 이내에 비즈니스에 DM을 보냈는지 여부와 관계없이 직접 메시지를 보낼 수 있었습니다. 이는 Meta 측의 버그 때문이었으며, 현재는 해결되었습니다.
