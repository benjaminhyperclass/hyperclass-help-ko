# 대화 AI(Conversation AI): 스냅샷 지원 - 완전히 구성된 대화 AI 봇을 스냅샷으로 생성하는 방법

### **대화 AI 봇 스냅샷 기능이란?**

대화 AI 봇 스냅샷 기능을 사용하면 프롬프트, 액션, 구성을 포함한 전체 봇을 재사용 가능한 스냅샷으로 복제할 수 있습니다. 이를 통해 하위 계정마다 봇을 수동으로 다시 만들 필요가 없어지며, 고객 상호작용, 학습 데이터, 봇 로직의 일관성을 보장할 수 있습니다.

### **대화 AI 봇 스냅샷의 주요 장점**

봇의 스냅샷을 생성하면 계정 간 설정을 간소화하고 사람의 실수를 줄일 수 있습니다.

- 설정, 프롬프트, 액션을 포함한 전체 봇 구성을 캡처합니다.
- 모든 하위 계정에서 반복적인 봇 설정 작업을 없앱니다.
- 모든 고객 계정에서 톤과 기능의 일관성을 유지합니다.
- 가져온 봇을 자동으로 기본 봇으로 설정합니다(기존 봇이 없는 경우).
- 모든 봇 학습 데이터(웹 크롤러 URL 및 FAQ)를 포함합니다.
- 대상에 기존 봇이 있을 때 덮어쓰기 또는 건너뛰기 방식을 선택할 수 있습니다.

### **봇이 포함된 스냅샷 생성하는 방법**

하위 계정에 봇을 복제하려면 다음 단계에 따라 대화 AI 구성이 포함된 스냅샷을 생성하세요.

- 왼쪽 네비게이션 메뉴에서 **Account Snapshots(계정 스냅샷)** 탭으로 이동합니다.
- **Create New Snapshot(새 스냅샷 생성)**을 클릭합니다.
- 스냅샷으로 만들려는 봇이 있는 **원본 하위 계정**을 선택합니다.
- 스냅샷의 **이름**을 설명적으로 입력합니다.
- **Conversation AI(대화 AI)** 섹션으로 스크롤합니다.
- 나열된 봇을 검토합니다. 이 섹션에는 다음이 포함됩니다:
  - 봇 이름
  - 봇 학습 웹 크롤러 URL
  - 커스텀 봇 응답(FAQ)
- 검토가 완료되면 **Create Snapshot(스냅샷 생성)**을 클릭합니다.

### **대화 AI 스냅샷 옵션 및 작동 방식**

무엇이 복사되고 가져오기 규칙이 어떻게 작동하는지 알면 봇 마이그레이션 과정이 더 원활해집니다.

**기본 봇 자동 할당**

대상 하위 계정에 기본 봇이 없는 경우, 스냅샷에서 가져온 봇이 자동으로 기본 봇으로 설정됩니다.

**충돌 처리**

대상 하위 계정에 동일한 이름의 봇이 이미 있는 경우, 다음 두 가지 옵션 중 하나를 선택할 수 있습니다:

- **Overwrite(덮어쓰기)** – 기존 봇을 스냅샷의 봇으로 교체합니다.
- **Skip(건너뛰기)** – 기존 봇을 유지하고 스냅샷의 봇은 무시합니다.

**스냅샷에 포함되는 내용**

- 완전한 봇 구조: 모든 액션과 프롬프트-응답 매핑
- 봇 설정 및 구성
- 봇 학습 웹 크롤러 URL
- Bot Training(봇 학습) 탭에서 생성된 커스텀 FAQ 데이터

### **자주 묻는 질문**

**Q: 대상 하위 계정에 동일한 이름의 봇이 이미 있으면 어떻게 되나요?**

가져오기 중에 기존 봇을 덮어쓸지 새 봇을 건너뛸지 선택하라는 메시지가 표시됩니다.

**Q: 프롬프트와 액션 로직이 보존되나요?**

네, 스냅샷은 구성된 그대로 모든 플로우, 액션, 응답을 정확히 보존합니다.

**Q: 스냅샷 가져오기 후 기본 봇을 변경할 수 있나요?**

네, 스냅샷 봇이 기본 봇이 되지 않은 경우 대화 AI 설정에서 수동으로 기본 봇으로 할당할 수 있습니다.

**Q: FAQ와 웹 크롤러 링크 같은 학습 데이터가 포함되나요?**

네, URL과 FAQ를 포함한 모든 학습 데이터가 스냅샷에 포함됩니다.

**Q: 나중에 스냅샷을 편집할 수 있나요?**

아니요, 스냅샷은 변경할 수 없습니다. 원본 봇에 변경사항이 있으면 새 스냅샷을 생성해야 합니다.

### **다음 단계**

- 스냅샷을 가져온 후, 새 하위 계정에서 봇이 제대로 작동하는지 확인하세요.
- 자동으로 선택되지 않은 경우 수동으로 기본 봇으로 할당하세요.
- 정확한 응답을 위해 봇의 학습 데이터를 테스트하세요.
- **Bot Training(봇 학습)** 및 **Bot Flow Builder(봇 플로우 빌더)** 도구를 사용하여 특정 계정 요구사항에 맞게 봇을 계속 구축하거나 수정하세요.
