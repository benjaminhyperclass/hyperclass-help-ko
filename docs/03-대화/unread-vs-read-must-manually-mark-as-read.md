# 읽지 않음 VS 읽음 - 수동으로 읽음 표시해야 함

# 대화를 읽음으로 표시하는 방법:

1. **읽음으로 표시(Mark as Read)** 버튼을 클릭하세요

![읽음으로 표시 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48195475159/original/R3kHK8I5vZi9pKQA5U-nYvfk0sqa5J0u5w.png?1646077825)

2. 대화에 직접 답장하세요

![대화 답장하기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48195475496/original/8o4yUNkMhN0RtIPUmqJH4zfP-MoMzJGyPg.png?1646077862)

3. 워크플로우 액션에서 **대화 편집(edit conversation)** 기능을 사용하세요

![워크플로우 대화 편집](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48195475560/original/v9_7nbAEABCMO0V1G54P3bOeAtLhpZpqVw.png?1646077884)

4. 워크플로우 설정에서 자동 읽음 표시(Auto Mark as Read)를 활성화하세요

![자동 읽음 표시 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48195480380/original/P-I5zkbqBwNrxmmEipkfEOojRCw37kGtPQ.png?1646079322)
