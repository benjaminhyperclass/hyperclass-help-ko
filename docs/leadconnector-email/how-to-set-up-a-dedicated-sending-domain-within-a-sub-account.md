# 전용 발송 도메인 설정 방법

이메일 마케팅은 고객에게 도달하고 매출을 늘리는 훌륭한 방법입니다. 하지만 주의하지 않으면 문제가 생길 수 있습니다. 전용 발송 도메인을 사용하면 이메일이 내 비즈니스에서 보내는 것처럼 보이도록 제어할 수 있습니다. 즉, 스팸 필터나 원치 않는 메일 서버에 메시지가 걸리는 다른 문제들을 피할 수 있습니다.

이메일 마케팅 노력이 헛되지 않도록 하세요! 전용 발송 도메인으로 지금 바로 시작해보세요.

#### 참고 사항:
전용 발송 도메인은 [LC - 이메일 시스템](https://help.leadconnectorhq.com/en/support/solutions/articles/48001225442-what-is-leadconnector-email-) 사용자에게만 적용됩니다.

# 전용 발송 도메인이란?

전용 발송 도메인을 사용하면 내 브랜드에서 보내는 것처럼 보이는 이메일을 발송할 수 있으며, 이는 이메일 서비스에서 더 나은 평판을 유지하는 데 도움이 됩니다. 모든 회사가 전용 발송 도메인을 만들 수 있으며, 빠르고 쉽게 설정할 수 있습니다.

기본적으로 Hyperclass 플랫폼에서 보내는 모든 이메일은 이메일 헤더의 "sent on behalf of" 또는 "sent via" 필드에 당사 이메일 발송 서버의 이름이 표시됩니다:

![기본 이메일 헤더 예시](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48265938968/original/spw7PyvCIf6DlxRXiukwE1KRAbxsW5nK9w.png?1669638686)

자체 커스텀 메일 서버 도메인을 설정하면 다른 "mailed-by", "sent on behalf", 또는 "sent via" 필드를 보게 됩니다.
