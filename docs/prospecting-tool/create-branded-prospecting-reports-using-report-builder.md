# 리포트 빌더를 사용한 브랜드 영업개발 보고서 만들기

전문적이고 브랜드화된 보고서를 제작하여 차별화되고 더 많은 잠재고객을 전환하세요. 리포트 빌더(Report Builder)를 사용하면 보고서의 외관, 느낌, 가치 전달 방식을 완전히 커스터마이징할 수 있어 브랜딩, 메시지, 행동 유도(CTA)를 완전히 제어할 수 있습니다.

---

## 리포트 빌더란 무엇인가요?

리포트 빌더는 영업개발 보고서를 브랜드 아이덴티티에 맞게 디자인할 수 있는 강력한 커스터마이징 도구입니다. 색상과 폰트부터 메시지와 CTA까지 모든 요소를 맞춤 설정하여 잠재고객에게 세련되고 높은 전환율을 제공하는 경험을 만들 수 있습니다.

이 기능은 업데이트된 웹 보고서 UI와 함께 작동하여 가독성과 레이아웃을 개선하고 더 원활한 조회 경험을 제공합니다.

---

## 웹 보고서 경험의 새로운 기능

업데이트된 보고서 인터페이스는 보고서가 제시되고 소비되는 방식을 개선하여 더 매력적이고 탐색하기 쉽게 만듭니다.

### 주요 개선사항

- **더 깔끔한 레이아웃:** 콘텐츠에 더 집중할 수 있도록 불필요한 여백 제거
- **향상된 가독성:** 빠른 스캔을 위한 최적화된 구조
- **더 나은 사용자 경험:** 보고서가 더 현대적이고 전문적으로 느껴짐

**참고:** 이러한 개선사항은 현재 웹 기반 보고서에 적용됩니다. PDF 개선사항은 곧 제공될 예정입니다.

---

## 리포트 빌더 액세스 방법

리포트 빌더는 계정 및 관리자 레벨에서 모두 사용할 수 있으며 프리미엄 구독이 필요합니다.

### 액세스 방법:

- **Prospecting(영업개발)**으로 이동
- **Report Builder(리포트 빌더)** 탭 열기
- 보고서 커스터마이징 시작

---

## 보고서 커스터마이징 방법

리포트 빌더는 네 개의 주요 섹션으로 구성됩니다. 각 섹션은 보고서 경험의 다른 부분을 제어합니다.

### 브랜드 테마 커스터마이징

브랜드 색상은 보고서의 전체적인 시각적 아이덴티티를 정의합니다. 일관된 브랜딩은 잠재고객과의 신뢰와 인지도 구축에 도움이 됩니다.

### 커스터마이징 가능한 항목:

- 기본 및 보조 색상
- 보고서 전반의 텍스트 색상
- 링크 색상
- CTA 버튼 색상 및 텍스트

**화면 설명:** 브랜딩 요소의 색상 컨트롤

**해야 할 일:** 브랜드 아이덴티티에 맞는 색상을 선택하고 실시간으로 변경사항을 미리 보기

## 헤더 커스터마이징

헤더는 잠재고객이 가장 먼저 보는 부분입니다. 톤을 설정하고 주요 연락처 정보를 제공합니다.

### 표시 옵션

다음 항목의 표시 여부를 제어할 수 있습니다:

- 로고
- 웹사이트
- 설명
- 이메일 및 전화번호
- CTA 버튼

![헤더 요소의 토글 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155067443240/original/KczCHRd-3Co35kaeCpilT8kosQhIrL9yNg.jpeg?1774212776)

**화면 설명:** 헤더 요소의 토글 옵션

**해야 할 일:** 잠재고객에게 보여주고 싶은 내용에 따라 요소를 활성화하거나 비활성화하세요

### 헤더 콘텐츠 설정

여기에서 헤더에 표시될 실제 콘텐츠를 정의합니다.

### 추가할 수 있는 항목:

- 로고 (권장 크기: 80x80px)
- 제목 및 설명
- 웹사이트 URL
- 이메일 및 전화번호

### CTA 구성

- CTA 텍스트 (예: "지금 연락하기")
- CTA 동작:
  - URL 열기
  - 이메일 보내기
  - 전화 걸기

## 이것이 중요한 이유

보고서를 커스터마이징하면 다음과 같은 이점을 얻을 수 있습니다:

- 브랜드화된 전문적인 프레젠테이션으로 신뢰 구축
- 명확한 메시지와 디자인으로 참여도 향상
- 전략적 CTA로 전환율 증가
- 모든 보고서에 변경사항을 적용하여 시간 절약

---

## 자주 묻는 질문

**Q: 변경사항이 기존 보고서에도 적용되나요?**

네. 저장된 변경사항은 과거와 미래의 보고서 모두에 적용됩니다.

**Q: 게시하기 전에 변경사항을 미리 볼 수 있나요?**

네. 저장하기 전에 미리보기 옵션을 사용하여 보고서를 검토할 수 있습니다.

**Q: 프리미엄 플랜 없이 보고서를 커스터마이징할 수 있나요?**

빌더는 탐색할 수 있지만, 변경사항 저장에는 프리미엄 구독이 필요합니다.

**Q: 헤더나 푸터 같은 섹션을 제거할 수 있나요?**

네. 대부분의 섹션에 대해 표시 여부를 토글할 수 있습니다.

**Q: 헤더와 푸터에서 서로 다른 CTA를 사용할 수 있나요?**

네. 각 섹션마다 고유한 CTA와 동작을 설정할 수 있습니다.
