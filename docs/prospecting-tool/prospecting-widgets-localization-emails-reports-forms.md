# 잠재고객 발굴 위젯의 다국어 지원 - 이메일, 보고서, 폼

잠재고객 발굴 위젯(Prospecting Widgets)이 이제 다국어 기능을 완벽하게 지원합니다. 필드 라벨, 검증 메시지, 제출 버튼, 이메일 커뮤니케이션, PDF 보고서 링크 등 모든 시스템 생성 텍스트가 사용자의 선호 플랫폼 언어에 자동으로 적용됩니다. 이를 통해 임베디드 위젯, 이메일, 내보낸 보고서에서 일관된 현지화 경험을 제공합니다.

## 개요

잠재고객 발굴 도구(Prospecting Tool)의 현지화(i18n) 지원을 개선했습니다.

이전에는 일부 백엔드 생성 문자열이 사용자의 언어 설정에 동적으로 적용되지 않았습니다. 이번 업데이트로:

- 잠재고객 발굴 위젯이 방문자의 선호 플랫폼 언어로 표시됩니다
- 이메일 커뮤니케이션이 사용자 언어 설정을 따릅니다
- PDF 보고서 링크와 내보낸 콘텐츠가 플랫폼 언어 설정을 따릅니다
- 추가 설정이 필요하지 않습니다

이 개선사항은 고객 대면 구성 요소에서 일관된 다국어 동작을 보장합니다.

## 잠재고객 발굴 위젯 접근하기

다음 경로로 이동하세요:

View(보기) → Prospecting(잠재고객 발굴) → Widgets(위젯)

여기서 위젯을 생성하고 관리할 수 있습니다.

![잠재고객 발굴 위젯 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155064814687/original/PawKpUsQ2loETU0h17F4Q_TaAexiEgrm2A.jpeg?1770920759)

## 업그레이드 요구사항

잠재고객 발굴 위젯은 잠재고객 발굴 도구의 유료 버전이 필요합니다.

계정이 업그레이드되지 않은 경우 업그레이드 안내가 표시됩니다.

![업그레이드 안내 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155064814688/original/VY2h0HRG2zmSQjdHE4nexMOrd3lL9bcC2g.jpeg?1770920759)

## 위젯 생성 및 맞춤 설정

업그레이드 후:

- Create Widget(위젯 생성)을 클릭하세요
- 필드를 설정하세요
- 스타일링과 브랜딩을 조정하세요
- 위젯을 저장하세요

![위젯 생성 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155064814685/original/qhAeagPKuWEGbgWhK2Hs9twkJqXSRShYDA.jpeg?1770920758)

## 테마 맞춤 설정

위젯은 Light(라이트)와 Dark(다크) 테마를 지원합니다.

![테마 설정 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155064814686/original/TkgZ9TQfE_onIqI7xDSsqQBO7VRaXKF3pw.jpeg?1770920759)

## 다국어 지원

잠재고객 발굴 위젯이 이제 다국어를 완벽하게 지원합니다.

모든 시스템 생성 텍스트가 사용자의 선호 플랫폼 언어를 자동으로 따릅니다.

여기에는 다음이 포함됩니다:

- 필드 라벨
- 검증 메시지
- 시스템 프롬프트
- 제출 버튼 텍스트
- 오류 메시지
- 확인 메시징

수동 번역 설정이 필요하지 않습니다.

## 위젯 임베딩

- 위젯 안의 Settings(설정) 탭으로 이동하세요
- 임베드 코드를 복사하세요
- 웹사이트 백엔드에 붙여넣으세요

![위젯 임베드 코드](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155064814684/original/quLJyQfDfNRN9i5ycN2uIPqyBmNDLxr4EA.jpeg?1770920758)

임베딩 후 다음을 확인하세요:

- 위젯이 정확히 로드되는지
- 레이아웃이 제대로 표시되는지
- 모든 필드가 예상대로 작동하는지

## 워크플로우 및 자동화

잠재고객이 위젯을 제출하면:

- 연락처가 자동으로 저장됩니다
- 리드에 태그가 붙습니다
- 워크플로우가 트리거될 수 있습니다

![워크플로우 연결](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155064814690/original/ppCD79tKN9ZXaa3QZ43ORN4Bs55lNyaPrA.jpeg?1770920759)

이를 통해 다음을 자동화할 수 있습니다:

- 환영 이메일
- 파이프라인 배정
- 넘처링 시퀀스

## PDF 보고서 언어 동작

내보낸 PDF 보고서가 이제 사용자의 선호 플랫폼 언어를 반영합니다.

여기에는 다음이 적용됩니다:

- 보고서 버튼
- 링크 라벨
- 시스템 생성 텍스트
- 내보내기 메시징

![PDF 보고서 예시](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155064814689/original/cygW_XGzUqgQOfn_f_y6tgYaojQ1vwv5vw.jpeg?1770920759)

## 이메일의 언어 지원

잠재고객 발굴 위젯과 관련된 시스템 생성 이메일이 이제 수신자의 플랫폼 언어에 자동으로 적응합니다.

여기에는 다음이 포함됩니다:

- 제목 줄
- 행동 유도(CTA) 버튼
- 시스템 지시사항

추가 설정이 필요하지 않습니다.

## 자주 묻는 질문

**Q: 위젯에 다국어를 수동으로 활성화해야 하나요?**
아니요. 위젯은 플랫폼 언어에 자동으로 적응합니다.

**Q: 수동으로 입력한 커스텀 텍스트도 변경되나요?**
아니요. 시스템 생성 문자열만 현지화됩니다.

**Q: 이메일 알림도 이제 다국어로 지원되나요?**
네. 시스템 생성 이메일은 사용자의 선호 플랫폼 언어를 따릅니다.

**Q: 내보낸 PDF 보고서도 언어가 자동으로 변경되나요?**
네. PDF 보고서 링크와 시스템 텍스트가 플랫폼 언어 설정을 반영합니다.

**Q: 잠재고객 발굴 도구를 업그레이드해야 하나요?**
네. 위젯은 유료 버전이 필요합니다.

## 추가 도움이 필요한 경우

위젯이 예상 언어로 표시되지 않는 경우, 먼저 계정 언어 설정을 확인하고 위젯이 임베딩된 페이지를 새로고침하세요. 이메일이나 보고서 링크가 예상 언어로 표시되지 않는 경우, 수신자의 언어 설정을 확인하고 메시지를 다시 보내세요.

문제가 계속되면 도움말 센터를 열고 "Prospecting Widgets", "Widget embed", "multi-language"로 검색하세요. 또는 다음 정보를 포함해서 지원팀에 문의하세요:

- 위젯 이름
- 임베딩된 페이지 URL
- 예상했던 언어 vs 실제 표시된 언어
- 문제 상황의 스크린샷
