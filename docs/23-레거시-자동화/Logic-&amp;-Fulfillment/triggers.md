# 트리거(Triggers)

트리거(Triggers)는 워크플로우가 시작되는 조건을 설정하는 기능입니다. 특정 이벤트가 발생했을 때 자동으로 후속 작업들이 실행되도록 하는 시작점 역할을 합니다.

트리거를 통해 다음과 같은 상황에서 자동화를 시작할 수 있습니다:

- 새로운 연락처가 추가되었을 때
- 특정 태그가 연락처에 추가되었을 때  
- 폼이 제출되었을 때
- 예약이 완료되었을 때
- 이메일이 열렸을 때
- 링크가 클릭되었을 때

각 워크플로우는 하나 또는 여러 개의 트리거를 가질 수 있으며, 설정된 조건이 충족되면 즉시 워크플로우의 다음 단계들이 순차적으로 실행됩니다.
