# 커스텀 필드란 무엇인가요?

커스텀 필드는 기본 필드(이름, 전화번호, 이메일 등) 외에 연락처나 기회에 대한 추가 정보를 저장할 수 있는 기능입니다. 폼, 설문, 자동화 워크플로우, 파이프라인에서 사용하여 비즈니스에 특화된 핵심 데이터를 수집하고 참조할 수 있어요.

---

## 커스텀 필드의 주요 장점

커스텀 필드는 데이터 수집, 세분화, 자동화에서 유연성을 제공합니다:

- **맞춤 데이터 수집**: 리드 유형, 예산 범위, 선호 연락 방법 등 비즈니스별 정보를 수집
- **전체 도구에서 활용**: 폼, 설문, 워크플로우, 파이프라인에서 커스텀 필드를 활용하여 완전한 커버리지 제공
- **개인화된 자동화**: 필드 값을 문자, 이메일, 내부 알림에 병합하여 맞춤형 메시지 구성
- **간편한 연락처 세분화**: 커스텀 필드 데이터를 기반으로 스마트 리스트 생성이나 워크플로우 필터링
- **체계적인 정리**: 관련 필드를 폴더로 그룹화하여 연락처와 기회 레코드를 깔끔하게 유지
- **중요 정보 표시**: 기회 카드나 알림에서 핵심 커스텀 필드 값을 직접 표시

---

## 연락처 vs 기회 커스텀 필드

올바른 유형의 커스텀 필드를 선택하는 것은 데이터를 체계적이고 기능적으로 유지하는 데 필수적입니다. Hyperclass에서는 두 가지 유형의 커스텀 필드를 만들 수 있습니다: 연락처 커스텀 필드와 기회 커스텀 필드. 비슷해 보일 수 있지만, 서로 다른 목적으로 사용되며 플랫폼의 다른 부분에서 활용됩니다.

**연락처 커스텀 필드**는 개별 고객에게 연결됩니다. 리드 유형(예: 구매자 또는 판매자), 생일, 선호 연락 방법, 추천 출처 등의 개인 세부 정보를 저장하는 데 사용하세요. 이 필드들은 연락처 레코드에 표시되며 폼, 설문, 연락처 워크플로우, 스마트 리스트, 그리고 병합 필드를 사용한 메시지 개인화에 활용할 수 있습니다.

**기회 커스텀 필드**는 특정 거래나 영업 기회에 연결됩니다. 예산 범위, 긴급도, 클로징 일정, 부동산 유형 등 거래별 정보를 수집하는 데 사용하세요. 이 필드들은 기회 레코드 내에서만 표시되며 파이프라인 뷰, 기회 워크플로우, 거래 수준의 리포팅이나 자동화에 사용됩니다.

**참고:** 필드가 연락처 또는 기회 필드로 생성된 후에는 유형을 변경할 수 없습니다.

## 폴더로 커스텀 필드 정리하기

필드를 정리하면 연락처와 기회 내에서 데이터를 깔끔하게 유지하고 사용하기 쉽게 만들 수 있습니다.

- 폴더(그룹이라고도 함)는 연락처/기회 레코드에서 섹션으로 나타납니다
- 일괄 작업을 사용하여 폴더 간에 필드를 이동할 수 있습니다
- Settings(설정) > Custom Fields(커스텀 필드) > + Add Folder(폴더 추가)에서 그룹을 만들 수 있습니다

## 커스텀 필드 설정 방법

하위 계정에서 커스텀 필드를 만들고 정리하는 단계별 방법은 다음과 같습니다:

#### 1단계: Settings(설정) → Custom Fields(커스텀 필드)로 이동

하위 계정에서 왼쪽 메뉴의 Settings(설정)을 클릭한 후 아래로 스크롤하여 Custom Fields(커스텀 필드)를 클릭하세요. 여기서 기존 필드를 확인하고 연락처 또는 기회용 새 필드를 만들 수 있습니다.

![설정 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155049899570/original/Yh9Qi_nPEsGdQ6sTnTPWqfsfYOoqiPMDKw.jpeg?1752607385)

#### 2단계: Add Field(필드 추가) 클릭

커스텀 필드 화면의 우상단에서 Add Field(필드 추가) 버튼을 클릭하여 새 커스텀 필드 생성을 시작하세요.

![필드 추가 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046564181/original/bs8f5d6nQk7p-aCnTyM49WoSARCdvdnulA.png?1747151424)

#### 3단계: 입력 유형과 객체 선택

생성하고 싶은 필드 유형(텍스트, 드롭다운, 라디오, 날짜 등)을 선택하세요. 그다음 Object(객체) 드롭다운에서 필드가 연락처 프로필에 표시되어야 하면 Contact(연락처)를, 기회에서만 표시되어야 하면 Opportunity(기회)를 선택하세요.

![필드 유형 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046564255/original/bm3NVQawQK_E_Xi60KtqS9lbM5NJ1hgZPA.gif?1747151495)

#### 4단계: 커스텀 필드 세부 정보 입력

Field Name(필드 이름)을 입력하고, Group(그룹)을 선택(또는 새로 생성)하며, 해당하는 경우 Option Names(옵션 이름)과 Values(값)를 추가하세요(예: 드롭다운이나 라디오 버튼).

커스텀 필드의 Prefill Value(기본값) 옵션을 사용하면 해당 필드가 폼과 설문에서 사용될 때마다 수동으로 변경하지 않는 한 자동으로 나타날 기본값을 정의할 수 있습니다.

**예시:** Country(국가)와 같은 커스텀 필드의 기본값을 "대한민국"으로 설정하면, 해당 필드가 사용되는 모든 곳에서 기본값으로 표시되지만 필요 시 사용자가 변경할 수 있습니다.

![필드 세부 정보 입력](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046564390/original/GdktWZxqMNvB1W3LjvCRvVPG_4jufa4m3w.png?1747151642)

#### 5단계: 저장 및 배치 확인

Save(저장)를 클릭하여 커스텀 필드를 생성하세요. 연락처 커스텀 필드의 경우 연락처 레코드로 가서 새 필드를 확인하고, 기회 커스텀 필드의 경우 기회 카드의 "Opportunity Details(기회 세부 정보)" 아래에서 필드를 확인하세요.

## 자주 묻는 질문

**Q: 연락처 필드를 기회 필드로 변환할 수 있나요(또는 그 반대)?**
아니요. 올바른 객체 유형으로 필드를 다시 만들어야 합니다. 필드 객체는 생성 후에는 변경할 수 없습니다.

**Q: 어떤 유형의 필드를 만들 수 있나요?**
짧은 텍스트, 긴 텍스트, 드롭다운, 날짜 선택기, 전화번호, 이메일, 라디오 선택, 체크박스 그룹, URL 등을 만들 수 있습니다.

**Q: 내 필드가 폼이나 워크플로우에서 표시되지 않는 이유는 무엇인가요?**
객체 유형이 사용 사례와 일치하는지 확인하세요. 연락처 필드만 폼과 연락처 워크플로우에 표시됩니다.

**Q: 드롭다운(다중 선택) 커스텀 필드는 몇 개의 옵션을 가질 수 있나요?**
드롭다운(다중 선택) 필드는 많은 옵션(50개 이상)을 가질 수 있으므로 실질적인 제한이 없습니다.
