## 커스텀 값이란?

커스텀 값(Custom Values)은 시스템이나 애플리케이션 내에서 특정 정보를 저장하고 관리하는 사용자 정의 데이터 플레이스홀더입니다. 사용자가 개인화되거나 표준화된 데이터를 효율적으로 정리하고 검색할 수 있게 해주는 기능입니다. 이러한 값들은 고객 관계 관리(CRM) 시스템, 콘텐츠 관리 시스템(CMS), 마케팅 자동화 플랫폼 등 다양한 상황에서 활용됩니다.

Hyperclass에서 커스텀 값은 연락처 정보, 선호사항, 특정 식별자 등 개인화된 정보를 저장합니다. 커스텀 값을 활용하면 기업은 메시지, 이메일, 웹사이트나 퍼널 페이지를 효율적으로 채우고 개인화하여 고객 상호작용을 향상시키고 참여도를 높일 수 있습니다.

## 커스텀 값의 주요 기능

- **개인화**: 커스텀 값을 통해 고객, 사용자, 리드 등 각 개체에 대한 개별화된 정보를 저장할 수 있습니다. 이름, 연락처 정보, 선호사항부터 계정 번호나 예약 참조 번호와 같은 고유 식별자까지 다양한 정보를 포함할 수 있습니다.

- **효율성**: 커스텀 값에 데이터를 저장하면 시스템의 여러 부분에서 이 정보를 쉽게 액세스하고 조작할 수 있습니다. 여러 곳에서 동일한 정보를 반복 입력할 필요가 없어 수작업을 줄이고 오류 가능성을 낮춥니다.

- **자동화**: 커스텀 값은 자동화를 지원합니다. 필드를 자동으로 채우고, 개인화된 이메일을 생성하고, 메시지를 맞춤화하고, 워크플로우를 간소화할 수 있습니다. 예를 들어, 이메일 마케팅에서 수신자의 이름을 위한 커스텀 값을 삽입하여 개인적으로 호칭할 수 있습니다.

- **일관성**: 커스텀 값은 특정 개체 그룹에 대해 일정하게 유지되는 데이터의 일관성을 보장하는 데 사용할 수 있습니다. 회사 주소, 지원 연락처 정보, 브랜딩 요소 등의 세부 정보에 유용합니다.

- **유연성**: 사용자는 특정 요구사항에 따라 커스텀 값을 정의할 수 있습니다. 이러한 유연성으로 커스텀 값은 다양한 시나리오, 산업, 조직 구조에 적응할 수 있습니다.

- **업데이트**: 커스텀 값의 데이터를 수정할 때 사용된 모든 인스턴스에서 변경할 필요가 없습니다. 소스에서 값을 변경하면 사용된 모든 곳에서 자동으로 업데이트됩니다.

- **데이터 무결성**: 커스텀 값은 데이터 무결성을 유지하는 데 도움이 됩니다. 예를 들어, 예약 시간에 커스텀 값을 사용하면 일관성을 보장하고 불일치 위험을 줄입니다.

- **세분화 및 필터링**: 세분화를 지원하는 시스템에서 커스텀 값은 데이터 세트를 나누고 분류하는 기준으로 사용할 수 있습니다. 이는 타겟팅된 커뮤니케이션과 리포팅에 도움이 됩니다.

- **중앙집중식 관리**: 커스텀 값은 종종 중앙 관리 인터페이스를 가지고 있습니다. 사용자는 필요에 따라 커스텀 값을 추가, 수정, 삭제할 수 있어 정보 정리를 감독하는 간소화된 방법을 제공합니다.

- **연동**: 많은 시스템이 데이터베이스나 API와 같은 외부 소스와의 연동을 허용합니다. 커스텀 값을 이러한 소스에 연결하여 시스템의 데이터 기능을 풍부하게 할 수 있습니다.

커스텀 값을 사용하면 개인화 향상, 프로세스 자동화, 정확성 보장, 보다 효과적인 의사결정을 통해 데이터를 최대한 활용할 수 있습니다. 데이터 기반 워크플로우를 최적화하고 사용자와 고객 모두에게 맞춤형 경험을 제공하는 데 중요한 역할을 합니다.

## 커스텀 값 생성, 편집, 삭제 방법

커스텀 값(Custom Values) 관리는 퍼널, 폼, 이메일, 자동화 전반에 걸쳐 일관되고 재사용 가능한 콘텐츠를 유지하는 데 필수적입니다. 커스텀 값을 생성, 업데이트 또는 제거하는 방법은 다음과 같습니다.

[**여기를 클릭**](../../11-설정/서브어카운트-설정/custom-values-settings.md)하여 커스텀 값 관리, 편집 및 정리에 대한 추가 정보를 확인하세요.

#### 커스텀 값 메뉴로 이동하기

- 하위 계정 대시보드에서 왼쪽 네비게이션 메뉴의 Settings(설정)을 클릭합니다.

- 설정 옵션 목록에서 Custom Values(커스텀 값)를 선택합니다.

![커스텀 값 메뉴 접근](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048494809/original/as8y-7P2JUKxW7x8cszYEMHUkRaVcIGkHw.png?1750264418)

#### 새 커스텀 값 생성하기

- 파란색 + Add Custom Value(커스텀 값 추가) 버튼을 클릭합니다.

- 이름과 표시할 값을 입력합니다. 그런 다음 커스텀 값을 저장할 폴더를 선택합니다.

- Save(저장)를 클릭합니다.

![커스텀 값 생성](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048494975/original/X5wrsvC7NkVYscJBEQLJcGI9zwi_AoU-7w.png?1750264576)

#### 커스텀 값 편집 또는 삭제하기

- 목록에서 업데이트하려는 값을 찾습니다.

- 항목 옆의 3점 아이콘(⋮)을 클릭하고 **Edit Custom Value(커스텀 값 편집)** 또는 Delete Custom Value(커스텀 값 삭제)를 선택합니다.

![커스텀 값 편집 삭제](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048496180/original/O0ivZnrMli4mWGhLBO8Kim11nG1r3w_ttQ.png?1750265699)

## 실제 사용 사례

### 예시 #1:

피트니스 클럽의 디지털 마케팅 캠페인을 관리한다고 상상해보세요. 클럽의 각 회원은 서로 다른 피트니스 목표, 선호하는 운동 유형, 커뮤니케이션 선호도를 가지고 있습니다. Hyperclass의 커스텀 값을 사용하면 이러한 개인화된 정보를 저장하고 마케팅 노력을 맞춤화하는 데 사용할 수 있습니다. 이러한 커스텀 값에는 회원의 이름, 선호하는 운동 유형, 멤버십 레벨, 마지막 방문 날짜와 같은 데이터 포인트가 포함될 수 있습니다.

Hyperclass 커스텀 값을 사용하면 다음과 같은 것들이 가능합니다:

- **이메일 개인화**: 각 회원의 이름, 선호하는 운동 유형 및 기타 관련 세부 정보로 이메일 템플릿을 자동으로 채웁니다.

- **메시지 맞춤화**: 커스텀 값을 사용하여 각 회원의 특정 요구사항과 선호도를 다루는 타겟팅된 메시지를 생성합니다.

- **웹 페이지 향상**: 저장된 커스텀 값을 기반으로 웹사이트나 퍼널 페이지에 개인화된 콘텐츠를 동적으로 표시합니다.

### 예시 #2:

피트니스 클럽 회원들을 위한 재참여 캠페인을 진행한다면, 커스텀 값을 사용하여 커뮤니케이션을 더욱 관련성 있고 매력적으로 만들 수 있습니다. 이를 세분화해보겠습니다:

1. **데이터 수집**:
   - 회원 A: 이름: 김철수, 선호 운동: 요가, 멤버십 레벨: 골드, 마지막 방문: 2024년 6월 1일
   - 회원 B: 이름: 박영희, 선호 운동: 유산소, 멤버십 레벨: 실버, 마지막 방문: 2024년 5월 25일

2. **데이터 저장**:
   - Hyperclass에서 이름, 선호 운동, 멤버십 레벨, 마지막 방문 날짜를 포함한 모든 유형의 데이터에 대한 커스텀 값 필드를 생성합니다.
   - 각 회원의 프로필에 이러한 필드에 특정 정보를 저장합니다.

3. **데이터 활용**:
   - **개인화된 이메일**: 김철수에게 "안녕하세요 철수님,"으로 시작하고 다가오는 요가 클래스와 골드 회원 전용 혜택에 대한 내용을 포함하는 이메일을 보냅니다.
   - **타겟팅된 메시지**: 박영희에게 "안녕하세요 영희님, 저희가 보고 싶어요! 이번 주 고강도 유산소 세션에 참여해보세요."라는 문자 메시지를 보냅니다.
   - **맞춤형 웹 페이지**: 김철수가 회원 포털에 로그인하면 "다시 오신 것을 환영합니다, 철수님! 다음 요가 세션 준비되셨나요?"와 같은 환영 메시지를 봅니다.

4. **고객 경험 향상**:
   - 커스텀 값을 사용하여 일반적인 커뮤니케이션을 개인화된 상호작용으로 변환하여 참여 및 재참여 가능성을 높입니다.

Hyperclass 커스텀 값을 활용하면 일반적인 커뮤니케이션을 개인화된 상호작용으로 변환할 수 있습니다. 이는 고객 경험을 향상시킬 뿐만 아니라 더 높은 참여도와 충성도를 이끌어냅니다.

따라서 새로운 프로모션 캠페인을 시작할 때 커스텀 값을 사용하여 메시지, 이메일, 웹 페이지에 데이터를 채워 넣어 각 접점이 수신자의 고유한 선호도와 요구사항에 맞춤화되도록 할 수 있습니다. 이러한 접근 방식은 청중과 더 효과적으로 연결하고 마케팅 노력에서 더 나은 결과를 얻는 데 도움이 됩니다.

## Hyperclass 하위 계정에서 커스텀 값 사용 방법

커스텀 값에 액세스하고 사용하는 것은 빠르고 쉬우며, Hyperclass는 사용자와 고객이 커스텀 값을 사용하기 시작할 뿐만 아니라 이유와 방법을 이해하는 것을 매우 직관적으로 만들어줍니다. 커스텀 값에 액세스하고 사용할 수 있는 Hyperclass 하위 계정 내의 다양한 영역이 있습니다:

### 1. 대화(Conversations)

- 대화(Conversations) 섹션에 있을 때 잠재 고객과 고객과 메시지를 주고받으면서 커스텀 값을 사용할 수 있습니다!

- SMS, 이메일, 페이스북 메신저, 인스타그램 메시징, 왓츠앱 등 Hyperclass에서 지원하는 모든 유형의 커뮤니케이션 채널에서 커스텀 값을 사용할 수 있습니다!

### 3. 웹사이트 & 퍼널

- 웹사이트와 퍼널 페이지 내부에 직접 커스텀 값을 추가하여 웹사이트와 퍼널에 개인화의 멋을 더하고 잠재 고객과 고객에게 직접 이야기할 수 있습니다.

- 웹사이트와 퍼널 내에서 커스텀 값을 활용하려면 페이지를 방문하는 잠재 고객이나 고객이 이전에 웹사이트나 퍼널에서 폼을 제출하거나 예약을 했던 데이터를 저장하고 있어야 합니다.

- 트리거 링크(Trigger Links)를 사용하여 URL 주소창에 커스텀 값 데이터를 추가할 수도 있습니다(**트리거 링크에 대해 자세히 알아보려면 여기를 클릭**하세요).
