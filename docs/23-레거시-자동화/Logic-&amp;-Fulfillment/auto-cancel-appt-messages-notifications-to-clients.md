# 고객에게 예약 자동 취소 메시지 알림 보내기

이 기능을 통해 예약이 자동으로 취소될 때 고객에게 알림 메시지를 자동으로 보낼 수 있습니다.

예약 자동 취소 알림을 설정하려면:

1. **Calendars(캘린더)** 메뉴로 이동합니다
2. 해당 캘린더의 설정을 엽니다
3. **Notifications(알림)** 섹션에서 자동 취소 메시지 옵션을 찾습니다
4. 고객에게 보낼 메시지 내용을 작성합니다
5. 설정을 저장합니다

이렇게 설정하면 시스템에서 예약을 자동으로 취소할 때마다 고객에게 사전에 작성한 메시지가 자동으로 전송됩니다.
