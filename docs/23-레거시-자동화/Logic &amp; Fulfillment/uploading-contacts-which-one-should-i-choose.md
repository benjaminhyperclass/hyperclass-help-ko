# 연락처 업로드 시 어떤 옵션을 선택해야 할까요?

연락처(Contacts)를 가져올 때 중복 처리 옵션을 선택해야 합니다. 각 옵션의 의미를 알아보겠습니다.

## 업로드 옵션 설명

### Skip duplicates (중복 건너뛰기)
- 이미 존재하는 연락처는 무시하고 새로운 연락처만 추가합니다
- 기존 연락처 정보는 전혀 변경되지 않습니다
- 가장 안전한 옵션입니다

### Update existing contacts (기존 연락처 업데이트)
- 중복되는 연락처가 있으면 새로운 정보로 덮어씁니다
- 기존 연락처의 정보가 업데이트됩니다
- **주의**: 기존 태그(Tag)나 커스텀 필드(Custom Field) 정보가 사라질 수 있습니다

### Force update (강제 업데이트)
- 모든 연락처를 강제로 업데이트합니다
- 기존 데이터를 완전히 새로운 데이터로 교체합니다
- **경고**: 기존의 모든 태그, 노트, 커스텀 필드가 삭제될 수 있습니다

## 권장사항

- **첫 업로드**: Skip duplicates 선택
- **정보 업데이트**: Update existing contacts 선택 (단, 백업 필수)
- **완전 교체**: Force update 선택 (매우 신중히 사용)

**중요**: 대량 업로드 전에는 반드시 기존 연락처를 백업하세요!
