# 오래된 기회 트리거 (Stale Opportunity Trigger)

오래된 기회 트리거(Stale Opportunity Trigger)는 특정 기간 동안 활동이 없었던 기회 관리(Opportunities)를 자동으로 감지하는 워크플로우 트리거입니다.

이 트리거는 영업 기회가 정체되어 있을 때 자동으로 후속 조치를 취할 수 있도록 도와줍니다. 예를 들어, 일정 기간 동안 진행이 없는 기회에 대해 자동으로 알림을 보내거나 팀원에게 할 일을 배정할 수 있습니다.

## 주요 활용 사례

- 오랫동안 방치된 영업 기회에 대한 자동 알림
- 정체된 기회에 대한 후속 조치 자동화  
- 팀원에게 기회 관리 업무 자동 배정
- 오래된 기회 정리 및 관리 자동화
