# 예약된 일괄 요청 실행하기

*본 문서는 레거시(구버전) 자동화 시스템에 관한 내용입니다. 현재는 새로운 워크플로우(Workflow) 시스템을 사용하는 것을 권장합니다.*

예약된 일괄 요청은 지정된 시간에 여러 연락처에게 동시에 작업을 실행하는 기능입니다. 이를 통해 특정 시점에 대량의 이메일 발송, SMS 전송, 태그 추가 등의 작업을 자동으로 처리할 수 있습니다.

## 일괄 요청 실행 방법

1. **연락처(Contacts) 메뉴**로 이동합니다
2. 원하는 연락처들을 선택합니다
3. **일괄 작업(Bulk Actions)** 버튼을 클릭합니다
4. 실행하고 싶은 작업을 선택합니다
5. **예약 실행** 옵션을 선택합니다
6. 실행할 날짜와 시간을 설정합니다
7. **확인(Confirm)** 버튼을 클릭하여 예약을 완료합니다

## 예약된 요청 관리

예약된 일괄 요청은 설정 완료 후 지정된 시간에 자동으로 실행됩니다. 필요에 따라 실행 전에 예약을 취소하거나 수정할 수 있습니다.
