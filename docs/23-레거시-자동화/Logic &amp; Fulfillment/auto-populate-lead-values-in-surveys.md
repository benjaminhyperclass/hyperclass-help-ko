# 설문에서 리드 값 자동 채우기

설문(Survey)에서 연락처(Contacts)의 기존 정보를 자동으로 채워넣어 더 나은 사용자 경험을 제공할 수 있습니다. 이 기능을 사용하면 고객이 이미 제공한 정보를 다시 입력할 필요가 없어 편리합니다.

## 설문에서 리드 값 자동 채우기 설정하기

1. **Sites(사이트)** 메뉴로 이동합니다
2. 편집하고자 하는 설문을 선택합니다
3. 설문 편집기에서 자동 채우기를 원하는 입력 필드를 클릭합니다
4. 우측 설정 패널에서 **Auto Populate(자동 채우기)** 옵션을 찾습니다
5. 드롭다운 메뉴에서 해당 필드에 자동으로 채워넣을 연락처 정보를 선택합니다:
   - 이름
   - 이메일
   - 전화번호
   - 커스텀 필드(Custom Field) 값
   - 기타 연락처 속성

## 작동 원리

- 고객이 설문 링크에 접속할 때, 시스템은 해당 연락처의 기존 정보를 확인합니다
- 설정한 필드에 해당하는 정보가 있으면 자동으로 채워집니다
- 고객은 미리 채워진 정보를 확인하고 필요시 수정할 수 있습니다
- 정보가 없는 필드는 빈 상태로 표시되어 고객이 직접 입력할 수 있습니다

## 활용 팁

- **개인화된 경험**: 고객 이름이 이미 채워져 있어 더 개인적인 느낌을 줍니다
- **완성도 향상**: 기본 정보가 미리 입력되어 설문 완료율이 높아집니다
- **데이터 일관성**: 기존 CRM 정보와 일치하는 데이터를 유지할 수 있습니다

이 기능은 특히 기존 고객을 대상으로 하는 만족도 조사, 추가 정보 수집, 후속 설문 등에서 매우 유용합니다.
