# 이벤트 시작일 캠페인

이벤트 시작일 캠페인은 레거시 자동화 시스템의 일부입니다. 현재는 더 강력하고 유연한 워크플로우(Workflow) 기능을 사용하시기를 권장합니다.

이벤트 시작일 캠페인은 특정 날짜나 예약된 이벤트의 시작일을 기준으로 자동화 시퀀스를 실행할 수 있게 해주는 기능입니다. 예를 들어, 세미나 시작 3일 전에 리마인드 이메일을 보내거나, 고객의 생일에 축하 메시지를 자동으로 발송할 수 있습니다.

이 기능은 현재 레거시(Legacy) 시스템으로 분류되어 있으며, 새로운 워크플로우 시스템으로 마이그레이션하시는 것을 권장합니다. 워크플로우에서는 더 다양한 트리거(Trigger)와 액션(Action)을 사용하여 동일한 기능을 구현할 수 있습니다.
