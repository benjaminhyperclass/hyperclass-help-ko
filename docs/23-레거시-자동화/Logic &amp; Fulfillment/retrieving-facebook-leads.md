# 페이스북 리드 가져오기

Facebook 리드 광고를 통해 수집한 리드 정보를 Hyperclass로 자동으로 가져오는 방법을 안내합니다.

Facebook 리드 광고는 고객이 Facebook을 떠나지 않고도 관심을 표현하고 연락처 정보를 남길 수 있는 효과적인 마케팅 도구입니다. Hyperclass와 연동하면 이러한 리드를 자동으로 연락처(Contacts)에 추가하고 후속 마케팅을 진행할 수 있습니다.

## 설정 방법

1. **Facebook 연동 설정**
   - Settings(설정) → Integration(연동) 메뉴로 이동합니다
   - Facebook 연동을 활성화하고 필요한 권한을 부여합니다

2. **워크플로우 생성**
   - Workflow(워크플로우) 메뉴에서 새 워크플로우를 만듭니다
   - 트리거(Trigger)로 "Facebook Lead"를 선택합니다
   - 해당 Facebook 페이지와 리드 폼을 연결합니다

3. **자동화 액션 설정**
   - 리드가 수집되면 자동으로 연락처에 추가되도록 설정합니다
   - 필요에 따라 환영 이메일이나 SMS를 자동 발송하도록 구성합니다
   - 적절한 태그(Tag)를 자동으로 부여하여 리드 소스를 구분합니다

## 활용 팁

- **즉시 응답**: 리드가 들어오면 즉시 자동 응답을 보내 관심을 유지하세요
- **태그 활용**: Facebook 광고별로 다른 태그를 부여하여 성과를 추적하세요
- **후속 마케팅**: 워크플로우를 통해 단계별 후속 메시지를 자동화하세요

Facebook 리드 연동을 통해 광고 투자 대비 효율을 극대화하고 잠재 고객과의 소통을 자동화할 수 있습니다.
