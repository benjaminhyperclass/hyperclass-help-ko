# SMTP 제공업체의 이메일 비용 계산하는 방법

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48120436123/original/Tw23HUiCJ8NkoaGoLo4mF2wupkmzxD99JQ.png?1626186619)

## 이메일 비용을 계산하려면:

- SMTP 제공업체에 로그인한 후, Plan(요금제) 또는 Billing(결제) 섹션으로 이동하여 요금제에 포함된 이메일 수량을 확인하세요.

- 요금제에 포함된 이메일 수량을 확인했다면, 요금제 비용을 이메일 수량으로 나누면 개별 이메일 비용이 나옵니다.

예를 들어, Mailgun의 월 $35 요금제는 50,000통의 이메일이 포함되어 있으므로 35/50000= 이메일당 $0.0007의 비용이 됩니다.

## 사용해야 할 이메일 제공업체

적합한 제공업체를 선택하는 것은 "즉시 사용 가능한" 설정을 원하는지, 아니면 기존 SMTP 도구를 사용하는지에 따라 달라집니다.

- **LC Email**: Mailgun 설정을 피하고 활성화 후 하이퍼클래스 내에서 이메일 발송/수신이 작동하길 원한다면 최고의 선택입니다.

- **Mailgun**: 하이퍼클래스에서 대량 이메일 발송에 흔히 사용되는 선택지로, 특히 여러 로케이션/도메인을 관리하는 에이전시에 적합합니다.

- **SMTP(기타 제공업체)**: 작동은 하지만, 하이퍼클래스는 더 나은 이메일 통계 추적(열람/클릭)을 위해 Mailgun 또는 LC Email을 권장합니다.

**LC Email이 활성화되어 있는 경우:** Mailgun이 필요하지 않습니다—LC Email은 Mailgun이나 다른 서드파티 이메일 제공업체에 가입할 필요성을 없애도록 설계되었습니다.

**LC Email이 활성화되어 있지 않은 경우:** Mailgun(대량 발송에 흔히 사용) 또는 지원되는 다른 SMTP 제공업체 등의 이메일 제공업체를 연결해야 합니다.
