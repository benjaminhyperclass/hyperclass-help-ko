# 평판 관리의 경쟁사 분석으로 경쟁 우위 확보하기

경쟁사 분석(Competitor Analysis) 기능을 통해 에이전시와 비즈니스는 온라인 평판을 최대 3개의 경쟁사와 비교할 수 있습니다. Google, Facebook, Yelp, Foursquare, Trustpilot 등에서 실시간 리뷰 데이터를 활용하여 경쟁사 대비 강점, 약점, 기회를 빠르게 파악할 수 있습니다.

## 🔎 평판 관리의 경쟁사 분석이란?

경쟁사 분석은 평판 관리(Reputation Management) 도구 내 강력한 기능으로, 에이전시가 온라인 평판 지표를 최대 3개의 경쟁사와 비교할 수 있게 해줍니다.

Google, Facebook, Yelp, Foursquare, Trustpilot 등 여러 플랫폼의 리뷰 데이터를 자동으로 수집하고 분석하여 다음을 제공합니다:

- **평판 점수(Reputation Score) (0-100점)** – 리뷰, 개수, 응답률을 가중 평가한 점수
- **감정 히트맵(Sentiment Heatmap)** – 경쟁사별 긍정/중립/부정 리뷰의 시각적 분포
- **키워드 및 테마 분석** – 주요 긍정 요소와 문제점을 트렌드로 분석하며, 클릭하면 실제 리뷰를 볼 수 있음
- **경쟁사 포지셔닝 그리드** – 강점 대비 약점을 사분면으로 매핑하여 고객이 좋아하거나 싫어하는 요소를 시각화
- **소스별 평점** – 플랫폼별 세부 정보(평균 평점, 총 리뷰 수, 응답 시간, 응답률)

## 🌟 경쟁사 분석의 주요 장점

- **실시간 벤치마킹** – 별점, 리뷰 개수, 응답 행태를 즉시 비교
- **다중 플랫폼 인사이트** – 여러 리뷰 사이트 데이터로 전체적인 관점 제공
- **실행 가능한 시각화** – 감정 히트맵과 포지셔닝 그리드로 성과 격차 강조
- **인터랙티브 드릴다운** – 테마 단어나 감정 블록을 클릭하면 해당 인사이트의 실제 리뷰 확인 가능
- **간편한 설정** – 경쟁사만 추가하면 CRM이 자동으로 데이터 수집
- **향상된 보고** – 클라이언트 프레젠테이션에 경쟁사 인사이트 활용
- **시간 절약** – 수동 조사를 없애고 평판 추적 간소화
- **전략적 의사결정** – 데이터 기반 인사이트로 전략 개선 및 시장 점유율 향상

## ⚙️ 경쟁사 분석 설정 방법

### 1단계: 경쟁사 분석으로 이동

- 대시보드에서 평판 관리(Reputation)를 클릭합니다.
- 경쟁사 분석(Competitor Analysis) 탭을 선택합니다.

### 2단계: 내 비즈니스 추가

- + 내 비즈니스 추가(Add Your Business)를 클릭합니다.
- 위치 권한을 허용하고 내 비즈니스를 검색합니다.
- 올바른 목록을 선택하고 + 내 비즈니스 추가를 다시 클릭하여 확인합니다.

### 3단계: 경쟁사 추가

- + 경쟁사 추가(Add Competitor)를 클릭합니다(최대 3개).
- 경쟁사 비즈니스를 검색하고 올바른 것을 선택합니다.
- + 경쟁사 추가를 클릭하여 확인합니다.

### 4단계: 소스별 평점 확인

- 소스별 평점(Ratings by Source) 섹션으로 스크롤합니다(Google, Facebook, Yelp, Trustpilot 등).
- 각 플랫폼에서 다음을 확인합니다:
  - **평균 평점** (예: 4.9 ⭐️)
  - **총 리뷰 수** (예: 295개)
  - **평균 응답 시간** (예: N/A 또는 2일)
  - **응답률** (응답한 리뷰 비율)

## 📊 추가 경쟁사 분석 기능

### 실시간 벤치마킹

- 별점과 리뷰 개수를 나란히 비교합니다.
- 평균 응답 시간과 응답성을 모니터링합니다.

### 다중 플랫폼 인사이트

- Google을 넘어 다른 주요 리뷰 사이트에서의 경쟁사 성과를 확인합니다.
- 경쟁사가 뛰어나거나 부족한 플랫폼을 식별합니다.

### 인터랙티브 시각화: 감정 히트맵 및 포지셔닝 그리드

- **감정 히트맵** – 긍정, 중립, 부정 리뷰 분포를 확인합니다. 어떤 블록이든 클릭하면 실제 리뷰를 볼 수 있습니다.
- **포지셔닝 그리드** – 감정 태그된 키워드를 통해 강점과 약점을 탐색합니다. 테마를 클릭하면 각 강점이나 약점과 연결된 리뷰를 확인할 수 있습니다.

### 간편한 설정 및 자동화

- 경쟁사 데이터는 자동으로 수집되고 정기적으로 새로고침됩니다.
- 수동 입력이 필요 없으며, 한 번 설정하면 시스템이 지속적으로 업데이트합니다.

## ❓ 자주 묻는 질문

**Q: 한 번에 몇 개의 경쟁사를 비교할 수 있나요?**
A: 한 번에 최대 3개의 경쟁사와 비교할 수 있습니다.

**Q: 어떤 리뷰 플랫폼이 지원되나요?**
A: Google, Facebook, Yelp, Foursquare, Trustpilot 및 기타 주요 플랫폼이 지원됩니다.

**Q: 데이터는 얼마나 자주 업데이트되나요?**
A: 최신 인사이트를 제공하기 위해 데이터가 자동으로 수집되고 정기적으로 새로고침됩니다.

**Q: 수동으로 데이터를 입력해야 하나요?**
A: 아니요, 모든 경쟁사 데이터는 자동으로 수집됩니다.

**Q: 설정 후 경쟁사를 변경할 수 있나요?**
A: 네, 언제든지 경쟁사 선택을 편집하거나 업데이트할 수 있습니다.
