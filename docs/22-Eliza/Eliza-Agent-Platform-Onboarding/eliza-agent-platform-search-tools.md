## 연락처 검색

연락처는 Search(검색) > Contacts(연락처)를 통해 검색할 수 있습니다.

![연락처 검색 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48290427971/original/M5YV-pUoDrS0Jp6FRFYQduTM13OP2t2G5g.png?1680263410)

이 화면에서 연락처를 검색할 로케이션을 선택하면, EAP가 해당 로케이션의 모든 연락처를 나열합니다.

로케이션 내에서 이름, 성, 태그, 전화번호, 주소로 연락처를 검색할 수 있습니다.

연락처를 선택하고 Add to Conversations(대화에 추가)를 클릭하여 특정 에이전트의 인박스에 추가할 수 있습니다.

![연락처를 대화에 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48290416644/original/qz5ZDeB73Ne2wQXYKGddnUO22dPzozjavQ.png?1680260295)

### 연락처 추가

Eliza 사용자는 Search(검색) → Contacts(연락처)로 이동하여 Eliza를 통해 하위 계정에 연락처를 추가할 수 있습니다. 이러한 연락처는 연락처 소스 필드가 'Eliza_User'로 표시되어 해당 Hyperclass 하위 계정에 동기화됩니다. Eliza를 통해 하위 계정에 연락처가 추가되면, Eliza 에이전트는 수동으로 자신이나 다른 에이전트에게 배정해야 합니다. 하위 계정의 감사 로그에는 추가된 연락처에 대한 항목이 표시됩니다.

![연락처 추가 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48290416793/original/P2rrcFdQGt7Jh5TJgD0muXgADKd5kCQD4A.png?1680260325)

![연락처 추가 양식](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48290411412/original/R_f7ThWHHh2-e1aMQTA5-1_PKQ6mC5kNHw.jpeg?1680259102)

### 대화에 추가

리드와 소통하고 싶다면, 연락처를 선택하고 드롭다운에서 에이전트에게 배정하세요. 선택된 연락처가 해당 에이전트의 인박스에 추가됩니다.

### 감사 로그

연락처를 선택하면 모든 대화의 감사 추적을 볼 수 있습니다.

감사 추적에는 Eliza 에이전트와 연락처 간의 모든 대화, 배정된 처리 상태, 날짜에 대한 세부 정보가 포함됩니다.

![감사 로그 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48290417377/original/HLMf_GahdUYxNsMyDUJuiwRKRoS00tExJA.png?1680260534)

### 대화 검색

대화 검색은 Eliza 에이전트 플랫폼에서 지원하는 모든 채널을 통한 에이전트와 연락처 간의 대화를 읽기 전용으로 볼 수 있는 기능입니다.

**참고:**
이 검색은 Eliza 에이전트 플랫폼 내에서 관리된 연락처만 찾을 수 있습니다.
