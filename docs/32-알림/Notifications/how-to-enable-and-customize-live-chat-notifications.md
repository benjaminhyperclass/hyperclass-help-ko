# 실시간 채팅 알림 활성화 및 커스터마이징 방법

실시간 채팅 알림은 새 메시지가 올 때 알림을 받을 수 있게 해주어, 빠르게 응답하고 원활한 커뮤니케이션을 유지할 수 있도록 도와줍니다. 이 가이드에서는 실시간 채팅 알림이 무엇인지, 왜 유용한지, 그리고 이를 활성화하고 커스터마이징하는 방법을 알아보겠습니다. 알림 유형 관리와 일반적인 문제 해결 팁도 함께 제공합니다.

# **실시간 채팅 알림 지원이란?**

실시간 채팅 알림 지원은 Hyperclass에서 새로운 실시간 채팅 메시지가 수신될 때마다 알림을 받을 수 있는 기능입니다. 알림은 프로필 설정에서 선택한 채널을 통해 표시되며, 앱 내 알림, 이메일 또는 모바일 푸시 알림 등이 있습니다. 모든 알림을 한 곳에서 관리할 수 있어 체계적이고, 중요한 메시지를 놓치지 않을 수 있습니다.

이 기능은 기존 대화(Conversations) 알림과 함께 작동하므로 실시간 채팅 알림을 별도로 관리할 필요가 없습니다. 사무실에서 업무 중이든, 외부에 있든, 모바일 앱을 사용 중이든, 실시간 채팅 알림을 통해 빠른 응답 시간을 유지하고 고객 참여를 개선하며 대화를 지연 없이 진행할 수 있습니다. 여러 문의를 처리하는 팀에게는 모든 구성원이 실시간으로 정보를 받을 수 있어 특히 유용합니다.

---

## 실시간 채팅 알림의 주요 장점

- **정보 파악:** 모든 새 실시간 채팅 메시지에 대한 알림을 받아 시기적절하게 응답할 수 있습니다.

- **별도 설정 불필요:** 실시간 채팅 알림은 다른 대화 유형과 동일한 설정을 사용합니다.

- **한 곳에서 관리:** 실시간 채팅을 포함한 모든 대화 알림이 프로필 설정에서 관리됩니다.

- **팀 공유:** 대화 알림을 활성화한 팀원들도 실시간 채팅 알림을 받게 됩니다.

- **유연한 설정:** 본인에게 가장 적합한 알림 유형과 빈도를 선택할 수 있습니다.

## **실시간 채팅 알림 설정 및 커스터마이징 방법**

실시간 채팅 알림은 Hyperclass의 대화 알림 설정을 통해 관리됩니다. 대화 알림이 활성화되면 실시간 채팅 알림이 자동으로 포함됩니다. 다음 단계에 따라 설정하고 필요에 맞게 커스터마이징하세요:

### **설정(Settings) 열기**

Hyperclass 대시보드에서 왼쪽 메뉴의 **Settings(설정)** 옵션을 클릭하세요. 여기서 알림 설정을 포함한 모든 계정 및 프로필 설정에 접근할 수 있습니다.

![설정 메뉴 접근](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051513662/original/Wt7BrXA2hZb2EFk_I30zRld9BusX1bP7HQ.png?1755081359)

### **My Profile(내 프로필) 접근**

설정 메뉴에서 **My Profile(내 프로필)**을 선택하세요. 이 섹션에는 실시간 채팅 메시지 알림 수신 방법을 제어하는 알림 설정을 포함한 개인 계정 설정이 포함되어 있습니다.

![내 프로필 접근](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051513735/original/MPlPORnkV4JVbh_AKvmzjWwwSC8EZoAf3Q.png?1755081414)

### **Notifications(알림) 탭 열기**

페이지 상단의 **Notifications(알림)** 탭을 클릭하세요. 실시간 채팅을 포함한 모든 대화 알림을 관리하고 알림 받을 방법을 선택할 수 있습니다.

![알림 탭 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051513868/original/aSdhoy2p2LfAw_IbiLd9srvOm4Gc667Udw.png?1755081473)

### **대화 알림 옵션**

이 설정들은 Hyperclass에서 메시지와 대화에 대한 알림을 언제 어떻게 받을지 제어합니다. 각 옵션의 의미는 다음과 같습니다:

- 계정으로 시작된 모든 새 대화 및 메시지 알림: 누구든지 계정과 새 대화를 시작할 때마다 알림을 받습니다. 모든 수신 커뮤니케이션을 파악하는 데 도움이 됩니다.

- 나에게 대화가 배정될 때 알림: 특정 대화가 본인에게 배정될 때 알림을 받습니다. 본인의 책임 사항을 정확히 알 수 있습니다.

- 배정된 대화에서 새 메시지 알림: 배정된 대화에 새 메시지가 추가될 때마다 알림을 받아 놓치지 않고 따라갈 수 있습니다.

- 내부 채팅에 추가될 때 알림 (앱 내: 웹 전용): 팀 내 개인 채팅에 추가될 때 알림을 보냅니다. 웹 앱에서만 표시됩니다.

- 참여 중인 내부 채팅의 모든 새 메시지 알림 (앱 내: 웹 전용): 이미 참여 중인 내부 채팅에서 새 메시지가 전송될 때마다 알림을 받습니다. 위와 같이 웹 앱 전용입니다.

- 대화에서 언급될 때 알림: 메시지에서 직접 태그되거나 언급될 때 알림을 받습니다. 배정되지 않은 대화에서도 관심이 필요한 내용을 파악하는 데 유용합니다.

- 언급된 대화의 모든 새 메시지 알림: 한 번 언급된 대화에서는 계속해서 모든 새 메시지에 대한 알림을 받아 후속 조치를 쉽게 할 수 있습니다.

![대화 알림 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051513998/original/JvdRajGq5BhuZWbL8tloBUzxcqyxaczZXQ.png?1755081589)

### **알림 채널**

이 옵션들은 선택된 알림 이벤트가 발생할 때 *어떻게* 알림을 받을지 제어합니다. 본인에게 가장 적합한 방식에 따라 하나 또는 여러 채널을 활성화할 수 있습니다:

- 앱 내 알림: 로그인되어 있는 동안 Hyperclass 대시보드 내에서 표시됩니다. 플랫폼에서 직접 업무하는 시간이 많고 즉시 업데이트를 보고 싶을 때 유용합니다.

- 이메일 알림: 등록된 이메일 주소로 알림을 보냅니다. 대시보드에서 떨어져 있는 경우가 많지만 새로운 활동에 대해 계속 알고 싶을 때 도움이 됩니다.

- SMS 알림: 문자 메시지를 통해 휴대폰으로 알림을 보냅니다. 긴급한 업데이트나 이메일을 확인하거나 로그인하지 않고도 즉시 알림을 받고 싶을 때 가장 좋습니다.

![알림 채널 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051514177/original/ZkwyYHulFHLLcqoluERRmMPngm4ktivrZw.png?1755081665)

### **변경사항 저장**

선호하는 알림 옵션과 채널을 설정한 후, 페이지 하단의 **Save(저장)** 버튼을 클릭하세요. 이렇게 해야 모든 업데이트가 저장되고 설정한 대로 알림 설정이 작동합니다. 이 단계를 건너뛰면 변경사항이 적용되지 않습니다.

![변경사항 저장](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051514282/original/dI0m6l9D6dg6sKLzE3UwEiaHg0DTh6gyyw.png?1755081697)

## **자주 묻는 질문**

**Q: "계정으로 시작된 모든 새 대화 및 메시지 알림"을 비활성화하면, 나에게 배정된 실시간 채팅 대화에 대한 알림은 여전히 받을 수 있나요?**

네. 이 옵션을 비활성화하면 배정되지 않은 대화에 대한 알림만 중단됩니다. "나에게 대화가 배정될 때 알림" 또는 "배정된 대화에서 새 메시지 알림"을 활성화해 두시면, 직접 책임지고 있는 실시간 채팅 대화에 대한 알림은 계속 받을 수 있습니다.

**Q: 앱 내 또는 이메일 알림을 활성화하지 않고 모바일 푸시나 SMS로만 실시간 채팅 알림을 받을 수 있나요?**

네. 알림 채널(앱 내, 이메일, SMS)은 서로 독립적입니다. 각 대화 알림 유형에 대해 채널의 어떤 조합이든 활성화하거나 비활성화할 수 있습니다. 예를 들어, 긴급한 업데이트에는 SMS만 선택하고 앱 내와 이메일은 체크하지 않을 수 있습니다.

**Q: "대화에서 언급될 때 알림"과 "언급된 대화의 모든 새 메시지 알림"은 어떻게 함께 작동하나요?**

첫 번째 옵션은 대화에서 언급되는 순간에만 알림을 보냅니다. 두 번째 옵션은 언급 후 해당 대화의 모든 새 메시지에 대해 직접 태그되지 않더라도 계속 알림을 보냅니다. 언급된 후 대화를 계속 팔로우하려면 두 옵션 모두 활성화해야 합니다.

**Q: 알림 설정을 변경하면 같은 하위 계정의 다른 사용자에게 영향을 미치나요?**

아니요. 알림 설정은 본인의 사용자 프로필에 특정되며 하위 계정의 다른 사용자에게는 영향을 미치지 않습니다. 각 사용자는 대화 유형, 알림 이벤트, 채널에 대한 자신만의 설정을 할 수 있습니다.
