# 서비스 메뉴

## 서비스 메뉴란 무엇인가요?

서비스 메뉴는 모든 서비스를 하나의 통합된 예약 링크로 나열하는 중앙 집중식 페이지입니다. 깔끔하게 분류된 서비스로 탐색이 매우 쉬워집니다. 더욱이 고객은 이 메뉴를 통해 자신이나 게스트를 위해 여러 서비스를 하나의 예약 위젯을 통해 손쉽게 예약할 수 있습니다.

서비스 메뉴는 선택된 [서비스 캘린더](https://help.leadconnectorhq.com/en/support/solutions/articles/155000001375)들을 표시합니다. 서비스 캘린더는 서비스 기반 비즈니스를 위해 특별히 제작된 캘린더 유형입니다. 이 캘린더 유형에서 사용자는 다양한 서비스를 생성하고, 직원을 배정하고, 카테고리별로 그룹화하여 모든 것을 하나의 예약 링크인 서비스 메뉴에 표시할 수 있습니다.

예를 들어, 미용실에서는 '헤어컷, 헤어 스파, 헤어 컬러링'과 같은 헤어 서비스를 '헤어' 카테고리 아래에, '페이셜, 클린업, 블리칭'과 같은 스킨 서비스를 '스킨' 카테고리 아래에 둘 수 있습니다. 고객은 원하는 서비스를 선택하고 원활하게 예약을 잡을 수 있습니다.

![서비스 메뉴 예시](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155011801350/original/vynnIgfmoaPWC-3iXBsN8P91F1FNDB6EMQ.jpeg?1698989209)

## 서비스 메뉴로 무엇을 할 수 있나요?

서비스 메뉴를 통해 서비스 기반 비즈니스를 이전과는 다른 차원으로 최적화할 수 있습니다. 다음과 같은 것들을 달성할 수 있습니다:

1. **각 서비스별 캘린더 생성:**
   - 제공하는 각 서비스에 대한 전용 캘린더를 쉽게 설정
   - 이러한 서비스별 캘린더에 직원 배정

2. **캘린더를 카테고리별로 정리:**
   - 캘린더를 카테고리로 그룹화하여 탐색 간소화
   - 예약 페이지에 표시할 그룹/서비스 선택
   - 모든 서비스를 위한 중앙 집중식 예약 위젯 생성. 더 이상 여러 캘린더를 탐색하며 혼란스러워할 필요 없음

3. **고객 예약 간소화:**
   - 고객이 하나의 예약에서 여러 서비스를 쉽게 예약하거나 여러 명을 위한 예약 생성 가능
   - 예약 시 선호하는 직원을 선택할 수 있음
   - 고객이 각 예약에 대한 소요 시간, 비용, 배정된 직원에 대한 중요한 정보를 받을 수 있음

## 누가 사용할 수 있나요?

서비스 메뉴는 서비스를 제공하는 다양한 비즈니스에 완벽합니다. 다음을 포함하되 이에 국한되지 않습니다:

- 미용실 및 헤어샵
- 스파 및 웰니스 센터
- 마사지 및 치료 클리닉
- 피트니스 스튜디오 및 트레이너
- 의료 서비스 제공업체
- 이벤트 기획자
- 펫샵 미용사
- 그리고 더 많은 업종들!

서비스를 제공하는 비즈니스를 운영하며 고객의 예약 경험을 향상시키고 싶다면, 서비스 메뉴가 이상적인 솔루션입니다.

## 시작하기: 사전 요구사항

1. **먼저 그룹 생성:** 그룹을 서비스의 카테고리로 생각하세요. 그룹에 추가된 캘린더는 예약 위젯에서 해당 카테고리 탭 아래에 나타납니다. 새 그룹을 만들려면 캘린더 설정(Calendar Settings) > 그룹 생성(Create Group)으로 이동하세요.

   예시: 헤어컷, 헤어 스파, 헤어 컬러링 등의 헤어 관련 서비스를 위해 "헤어 케어" 그룹을 만들 수 있습니다.

![그룹 생성](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155011801343/original/3A_uoeQDHVVTuQDyAN3cJs95hp4R9yr7bA.jpeg?1698989207)

2. **그 다음 서비스 캘린더 생성:** 제공하는 각 서비스에 대한 전용 캘린더를 디자인하고 해당 그룹에 캘린더를 배정하세요. 서비스 캘린더 생성에 대한 자세한 내용은 [여기](https://help.leadconnectorhq.com/en/support/solutions/articles/155000001375)를 클릭하세요.

   예시: 헤어 케어 서비스를 위해 다양한 캘린더를 만들 수 있습니다. 여기서는 각 서비스마다 하나씩, 총 네 개의 서비스 캘린더가 있을 것입니다 - 헤어컷(남성), 헤어컷(여성), 헤어 스파, 헤어 컬러링.

![서비스 캘린더](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155011801348/original/3n2kc8i4lB62nJH9Zpzqa1BUHTGYE92LFg.jpeg?1698989208)

3. **마지막으로 서비스 페이지 메뉴 생성:** 예약 위젯에 표시할 서비스 캘린더와 그룹을 결정하고 고객에게 예약할 수 있는 단일 장소를 제공하세요.

   예시: 서비스 메뉴는 "헤어 케어"와 "스킨 케어" 두 카테고리를 가질 수 있으며, 각각 아래에 다양한 서비스가 나열됩니다.

![서비스 메뉴 예시](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155011801345/original/tSOLUMDQSGMbsu79Rq6-vgCu4PR5cMa8sQ.jpeg?1698989208)

위의 단계를 따라하면 예약을 효율적으로 관리할 수 있습니다.

## 서비스 메뉴 생성 방법

서비스 메뉴 생성 가이드

### 1단계: 서비스 메뉴 활성화

- 캘린더 설정(Calendar Settings)으로 이동합니다.
- "기본 설정(Preferences)"을 클릭합니다.
- "인앱 기본 설정(In-App Preferences)" 아래에서 "서비스 메뉴(Service Menu)" 옵션을 토글하여 켭니다.
- 이제 "캘린더 설정(Calendar Settings)" > "캘린더(Calendars)" 아래에 "서비스 메뉴(Service Menu)" 탭이 표시됩니다.

![서비스 메뉴 활성화](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155011801344/original/hWFbYCyewa8w5XaUt_3SUfiPASteuKQc4A.jpeg?1698989208)

### 2단계: 서비스 캘린더 준비

서비스 메뉴를 생성하기 전에 [서비스 캘린더](https://help.leadconnectorhq.com/en/support/solutions/articles/155000001375)를 만들고 그룹에 추가했는지 확인하세요. 방법은 다음과 같습니다:

- "캘린더 설정(Calendar Settings)"에서 서비스 캘린더를 생성합니다.
- 이러한 캘린더를 관련 그룹에 추가합니다.

### 3단계: 서비스 메뉴 생성

- "서비스 메뉴(Service Menu)" 탭으로 이동합니다.
- "서비스 메뉴 생성(Create Service Menu)"을 클릭합니다.
- 메뉴에 대한 다음 세부 정보를 입력합니다:
  - **이름:** 서비스 메뉴의 이름을 제공합니다.
  - **설명:** 서비스에 대한 간단한 설명을 추가합니다.
  - **URL/슬러그:** 서비스 메뉴의 URL을 사용자 정의합니다.
- 다음 옵션도 설정할 수 있습니다:
  - 동의 체크박스 토글 켜기/끄기
  - 동의 메시지 사용자 정의
  - 예약 후 감사 메시지 표시 또는 외부 URL로 리다이렉트 선택
  - 예약자가 추가 게스트를 추가할 수 있도록 활성화/비활성화, 예약자가 여러 서비스 대신 하나의 서비스만 예약하도록 제한, 예약자가 선호하는 직원을 선택할 수 있는지 결정
  - 서비스 메뉴의 모양과 느낌을 사용자 정의하기 위한 커스텀 코드 삽입
- 세부 정보를 입력한 후 다음 단계인 "서비스 선택(Select Services)"으로 이동합니다.

![서비스 메뉴 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155011801346/original/2Ca1gcjfbM8pR-DUM9PzvfmhXArGtiTu4A.jpeg?1698989208)

![서비스 메뉴 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155011801340/original/YVDAIxntDHvR4EqOwGWgy9gg1YRN9rdF6w.jpeg?1698989207)

### 4단계: 서비스 선택 및 배열

1. 서비스 선택 탭에서 메뉴에 표시하려는 그룹 또는 개별 서비스를 선택합니다.

   참고: 서비스 캘린더를 생성하지 않았거나 그룹에 추가하지 않았다면, 이 목록에 나타나기 전에 먼저 해야 합니다.

![서비스 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155011801347/original/bbvswFoNAf1Rqb2-a2mJcOOipmNGhyT85g.jpeg?1698989208)

2. 서비스를 선택한 후, 서비스 메뉴에 나타나기 원하는 순서로 배열합니다.

3. 여기서 그룹이 표시되는 순서를 결정할 수 있습니다.

4. 배열에 만족하면 "저장(Save)"을 클릭합니다.

그룹 내 서비스 순서를 변경하려면:

- 그룹 탭(Groups Tab)으로 이동합니다.
- 선택한 그룹 옆의 점 세 개를 클릭합니다.
- "캘린더 재배열(Rearrange Calendars)"을 선택합니다.
- 순서를 조정하고 "저장(Save)"을 클릭합니다.

![캘린더 재배열](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155012083094/original/w1GOKprtxOjrkGjLEov_Td99jn_5bg-6-w.png?1699355039)

완성입니다! 이제 서비스 메뉴를 사용할 준비가 되었습니다. 제공된 예약 링크를 사용하여 고객을 서비스 메뉴로 안내할 수 있습니다.

## 서비스 메뉴에 커스텀 코드 추가하기

서비스 메뉴의 모양과 느낌을 사용자 정의하기 위해 커스텀 코드를 추가할 수 있습니다. 서비스 메뉴 생성 시 커스텀 코드 섹션에 HTML, CSS, JavaScript 코드를 삽입하여 브랜딩과 기능을 맞춤화하세요.

## 서비스 메뉴를 비활성화하는 방법은?

서비스 메뉴를 비활성화하려면 다음 단계를 따르세요:

- 캘린더 설정(Calendar Settings)으로 이동합니다.
- "기본 설정(Preferences)"을 클릭합니다.
- "인앱 기본 설정(In-App Preferences)" 아래에서 "서비스 메뉴(Service Menu)" 옵션을 토글하여 끕니다.
- "캘린더 설정(Calendar Settings)" > "캘린더(Calendars)" 아래의 "서비스 메뉴(Service Menu)" 탭에 액세스할 수 없게 됩니다.

![서비스 메뉴 비활성화](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155019189045/original/IvOMCpFv0NjofduuWirBe3DPPVLJL3kf9Q.png?1706694667)
