# 커스텀 오브젝트 목록 보기 - 정렬 및 데이터 커스터마이징

커스텀 오브젝트 목록 보기는 깔끔한 표 형태의 레이아웃을 제공하여 레코드를 더 효율적으로 정리하고 스캔하며 관리할 수 있도록 도와줍니다. 데이터를 정렬하고 표시할 필드를 제어할 수 있어서 가장 중요한 것에 집중하고 일상 업무 흐름을 간소화할 수 있습니다.

## **커스텀 오브젝트 목록 보기란?**

커스텀 오브젝트 목록 보기는 레코드를 스프레드시트와 유사한 행과 열의 구조화된 표 형식으로 표시합니다. 이 레이아웃은 가시성을 향상시키고 여러 화면을 전환하지 않고도 대량의 데이터를 더 쉽게 관리할 수 있게 해줍니다.

개별 레코드를 열지 않고도 중앙 집중화된 하나의 보기에서 정보를 빠르게 검토하고, 비교하고, 업데이트할 수 있습니다.

---

## **커스텀 오브젝트 목록 보기의 주요 장점**

- **명확한 데이터 가시성:** 모든 레코드를 구조화된 표로 보여 빠른 스캔 가능
- **유연한 정렬:** 중요한 필드를 기준으로 레코드 정리
- **커스터마이징 가능한 레이아웃:** 필요한 필드만 표시
- **향상된 정리:** 복잡한 데이터를 깔끔하고 탐색하기 쉽게 유지
- **효율성 개선:** 정보 검색에 소요되는 시간 단축

---

## **목록 보기에서 커스텀 오브젝트**

표 레이아웃은 레코드를 행으로, 필드를 열로 표시하여 데이터를 한눈에 이해하기 쉽게 만들어줍니다.

![커스텀 오브젝트 목록 보기 예시](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155069745569/original/VcEmj7Hrt6VQJDw0ZNGowl1Qn3u_hcz4lg.png?1776892617)

이 이미지는 여러 레코드와 필드가 표 형식으로 표시된 전체 목록 보기를 보여줍니다.

이 보기를 사용하여 모든 레코드에서 데이터를 빠르게 스캔하고, 비교하고, 관리하세요.

## **필드별 정렬**

정렬 기능을 사용하면 생성일이나 마지막 업데이트 시간 같은 특정 필드를 기준으로 레코드를 정리할 수 있습니다.

![정렬 옵션 드롭다운](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155069745610/original/jUdxv1Csp7o2MGiXj4BM8lWlRUztGGmZ8Q.png?1776892718)

이 이미지는 "생성일"과 "업데이트일" 같은 옵션이 있는 정렬 드롭다운 메뉴를 보여줍니다.

정렬 옵션을 클릭하고 필드를 선택한 다음 오름차순 또는 내림차순을 선택하여 데이터를 즉시 재정리할 수 있습니다.

목록 보기에서 어떤 필드가 나타날지 제어하여 혼란을 줄이고 중요한 정보를 강조할 수 있습니다.

![필드 선택 패널](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155069745583/original/-YvWgqd87ZmZWq_O4jDlZL7huYU5FJnm4Q.png?1776892651)

이 패널을 사용하여 필요에 따라 필드를 추가하고, 제거하고, 순서를 바꾸거나 관련 필드를 포함시킬 수 있습니다.

---

## **워크플로우 개선 효과**

커스텀 오브젝트 목록 보기는 다음을 통해 워크플로우에 맞춰 적응합니다:

- 관련 데이터에만 집중
- 시각적 혼란 감소
- 레코드 빠른 찾기 및 업데이트
- 다양한 팀이나 프로세스에 맞는 보기 커스터마이징

이러한 유연성은 생산성을 향상시키고 작업 공간을 체계적으로 유지하는 데 도움이 됩니다.

---

## **중요 참고 사항**

- 커스텀 오브젝트는 특정 구독 플랜에서만 사용 가능할 수 있습니다
- 필드 커스터마이징은 현재 보기에만 적용됩니다
- 정렬은 데이터 표시 방식만 변경하며 실제 데이터는 변경하지 않습니다

---

## **자주 묻는 질문**

**Q: 목록 보기에서 어떤 필드가 나타날지 커스터마이징할 수 있나요?**

네, 필드 관리(Manage Fields) 옵션을 사용하여 필드를 선택하고, 순서를 바꾸고, 제거할 수 있습니다.

**Q: 어떤 필드로든 레코드를 정렬할 수 있나요?**

네, 생성일(Created At)과 업데이트일(Updated At) 같은 지원되는 필드에서 정렬이 가능합니다.

**Q: 제가 변경한 내용이 다른 사용자에게도 영향을 주나요?**

아니요, 보기 커스터마이징은 사용자별로 적용되며 다른 사용자에게는 영향을 주지 않습니다.

**Q: 목록 보기에 관련 레코드를 포함시킬 수 있나요?**

네, 필드 커스터마이징 패널을 통해 관련 필드를 추가할 수 있습니다.

**Q: 정렬하면 데이터가 영구적으로 변경되나요?**

아니요, 정렬은 레코드가 표시되는 방식만 변경합니다.

목록 보기 커스터마이징이나 데이터 관리에 도움이 필요하시면 지원팀이나 시스템 관리자에게 문의하세요.
