# 커스텀 객체 접근 가이드: 기능, 제한사항, 업그레이드

커스텀 객체(Custom Object) 접근 권한이 더 많은 요금제에서 사용할 수 있게 되었습니다. 미리 만들어진 객체를 사용하면서도 고급 기능에 대한 제어된 접근을 유지할 수 있습니다. 이번 업데이트는 제한적 사용 권한, 업그레이드 안내, 그리고 곧 출시될 스냅샷(Snapshot) 지원을 도입하여 전체 설정 접근 권한 없이도 데이터를 효과적으로 관리하고 정리할 수 있게 도와드립니다.

## **커스텀 객체 접근 레벨이란?**

커스텀 객체 접근 레벨은 현재 요금제에 따라 수행할 수 있는 작업을 결정합니다. 일부 요금제는 완전한 생성 및 설정이 가능한 반면, 다른 요금제는 기존 객체를 특정 제한사항과 함께 사용할 수 있는 접근만 제공합니다.

이러한 구조는 고급 기능에 대한 시스템 일관성과 제어를 유지하면서도 레코드, 관계, 데이터를 효율적으로 관리할 수 있게 해줍니다.

---

## **주요 장점**

접근 방식을 이해하면 향후 업그레이드를 계획하면서 커스텀 객체를 최대한 활용할 수 있습니다.

- **제어된 접근:** 의도하지 않은 구조적 변경의 위험 없이 커스텀 객체 사용
- **유연한 사용:** 기존 객체 내에서 레코드, 필드, 연결 관리
- **가이드형 업그레이드:** 고급 기능이 필요할 때 명확한 업그레이드 안내 제공
- **확장 가능한 성장:** 사용 접근으로 시작해서 필요에 따라 기능 확장

---

## **커스텀 객체 접근 방법**

커스텀 객체에 접근하는 것은 간단하며 권한과 요금제 레벨에 따라 달라집니다. 특정 기능이 제한되어 있다면 업그레이드 안내가 가이드해드립니다.

- 대시보드에서 **설정(Settings)**을 열어주세요
- **Custom Objects(커스텀 객체)**로 이동하세요
- 객체를 선택하여 사용 가능한 레코드와 세부사항을 확인하세요

![커스텀 객체 업그레이드 배너](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155068479818/original/-W8d-GUHGNBQAkm17W-xowbyPkNwRtHPjw.png?1775360124)

이 화면은 고급 기능이 제한되어 있을 때 업그레이드 배너가 나타나는 커스텀 객체 섹션을 보여줍니다.

**화면 설명:** 현재 요금제에서는 객체 구조 생성이나 편집이 불가능하다는 배너입니다.

**해야 할 일:** 메시지를 확인하고 커스텀 객체를 생성하거나 수정할 전체 접근이 필요하다면 **지금 업그레이드(Upgrade Now)**를 클릭하세요.

접근이 가능해지면:

- 기존 커스텀 객체가 계정에 나타납니다
- 레코드, 필드, 연결 작업을 시작할 수 있습니다
- 향후 스냅샷 기능으로 미리 만들어진 객체를 가져올 수 있게 됩니다

---

## **할 수 있는 것과 없는 것**

접근 레벨에 따라 사용 가능한 작업과 업그레이드가 필요한 작업이 정의됩니다.

### **허용되는 작업**

- 기존 커스텀 객체 보기 및 사용
- 레코드 추가 및 편집
- 객체 내에서 필드 생성 및 관리
- 레코드 간 연결 관리

### **제한되는 작업**

- 새로운 커스텀 객체 생성
- 객체 구조 편집(표시명이나 아이콘 등)
- 시스템 레벨 설정 수정

모든 연결은 **다대다(many-to-many)** 관계를 유지하여 레코드 간 유연한 관계를 허용합니다.

---

## **업그레이드 옵션**

업그레이드하면 생성 및 구조적 커스터마이징을 포함한 커스텀 객체에 대한 전체 제어 권한을 얻을 수 있습니다.

업그레이드 방법:

- 커스텀 객체 섹션 내의 **업그레이드 배너**를 클릭하세요
- 제공된 업그레이드 단계를 따라주세요
- 도움이 필요하시면 지원팀에 문의하세요

업그레이드 후 다음을 얻게 됩니다:

- 완전한 객체 생성 기능
- 구조적 편집 접근
- 고급 커스터마이징 옵션

## **자주 묻는 질문**

**Q: 현재 요금제에서 커스텀 객체를 생성할 수 있나요?**

아니요. 새로운 커스텀 객체 생성은 상위 요금제 접근이 필요합니다.

**Q: 업그레이드 없이도 커스텀 객체를 사용할 수 있나요?**

네. 기존 객체를 사용하고 레코드, 필드, 연결을 관리할 수 있습니다.

**Q: 커스텀 객체의 구조를 편집할 수 있나요?**

아니요. 표시명이나 아이콘 같은 구조적 편집은 업그레이드 없이는 제한됩니다.

**Q: 기능이 제한될 때 안내를 받을 수 있나요?**

네. 전체 접근 권한을 해제하는 방법을 안내하는 업그레이드 배너가 나타납니다.

**Q: 스냅샷을 사용해서 커스텀 객체를 가져올 수 있나요?**

네. 스냅샷 지원이 도입되어 계정으로 커스텀 객체를 로드할 수 있게 됩니다.

---

## **추가 도움이 필요하시나요**

커스텀 객체 접근, 기능 제한사항 이해, 또는 요금제 업그레이드에 대한 도움이 필요하시면 Help Center를 열거나 지원팀에 문의하세요.
