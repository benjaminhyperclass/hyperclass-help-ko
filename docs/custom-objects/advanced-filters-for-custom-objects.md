# 커스텀 오브젝트를 위한 고급 필터

고급 필터(Advanced Filters)를 사용하면 커스텀 오브젝트 내에서 특정 레코드를 빠르게 찾을 수 있습니다. 여러 필드에 걸쳐 조건을 적용하여 데이터 뷰를 세분화하고 가장 중요한 내용에만 집중할 수 있습니다.

# 고급 필터란 무엇인가요?

고급 필터는 여러 조건을 사용하여 커스텀 오브젝트 데이터를 검색하고 분류할 수 있게 해주는 기능입니다. 수동으로 레코드를 스캔하는 대신, 소유자, 날짜, 커스텀 속성 등의 필드에 필터를 적용하여 원하는 데이터를 즉시 찾을 수 있습니다.

이 기능은 대용량 데이터셋이나 복잡한 레코드로 작업할 때 특히 유용합니다.

---

## 고급 필터의 주요 장점

**정확한 데이터 검색:** 여러 조건을 사용하여 특정 레코드를 빠르게 찾을 수 있습니다

**유연한 로직:** AND/OR 로직을 결합하여 더 깊이 있는 필터링이 가능합니다

**커스텀 필드 지원:** 기본 필드와 커스텀 오브젝트 필드를 모두 사용하여 필터를 적용할 수 있습니다

**효율성 향상:** 결과를 즉시 좁혀서 시간을 절약할 수 있습니다

**데이터 제어 개선:** 불필요한 내용 없이 관련 레코드에만 집중할 수 있습니다

---

## 고급 필터 사용 방법

고급 필터는 간단하면서도 강력하게 설계되었습니다. 아래 흐름에 따라 효과적으로 필터를 적용하세요.

### 고급 필터 패널에 접근하기

![고급 필터 접근](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155069745235/original/IgS1OzoBUDN0rV-Y9DPa84Ocq6HI7fPcpA.png?1776891703)

커스텀 오브젝트 목록 뷰 상단에서 **"Advanced Filters(고급 필터)"**를 클릭하여 필터 패널을 엽니다.

### 필터 필드 선택 및 추가

![필터 필드 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155069745282/original/ZCHYcdDS2FSM8Wwvs7TdD88gZ8KzYKneFQ.png?1776891825)

검색창을 사용하거나 카테고리를 탐색하여 필터링할 필드를 선택합니다. 다음 항목 중에서 선택할 수 있습니다:

- 기본 필드 (예: Owner(소유자), Created Date(생성일))
- 커스텀 필드 (예: Car Info(차량 정보), 커스텀 속성)

필드를 클릭하여 필터 조건으로 추가합니다.

### 필터 조건 적용

![필터 조건 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155069745272/original/fWdd5EPNtM7ZVr134U6xi4mn1OBbkBtVkA.png?1776891792)

선택한 각 필드에 대해 다음과 같은 조건을 설정합니다:

- Equals (같음)
- Is any of (~중 하나)
- Contains (포함)

다음을 사용하여 여러 필터를 조합할 수 있습니다:

- **AND** → 모든 조건이 일치해야 함
- **OR** → 조건 중 하나만 일치하면 됨

**Apply(적용)**을 클릭하여 데이터를 즉시 필터링합니다.

---

## 작동 원리

필터가 적용되면 목록 뷰가 즉시 업데이트됩니다. 설정한 조건에 맞는 레코드만 표시됩니다.

다음과 같은 작업이 가능합니다:

- 더 세분화된 분류를 위해 여러 필터 추가
- 명확성을 위한 필터 순서 조정
- 언제든지 필터 제거

---

## 중요 참고사항

- 필터는 현재 뷰에만 적용되며 자동으로 저장되지 않습니다
- 언제든지 필터를 수정하거나 재설정할 수 있습니다
- 필터 저장 기능은 향후 업데이트에서 도입될 예정입니다
- 커스텀 오브젝트 사용 가능 여부는 구독 플랜에 따라 달라질 수 있습니다

---

## 사용 사례

지난주에 생성되고 특정 소유자에게 할당된 모든 레코드를 찾고 싶다면:

- **Created Date(생성일)**에 대한 필터 추가
- **Owner(소유자)**에 대한 필터 추가
- **AND 로직**을 사용하여 조합

이렇게 하면 관련 레코드로만 결과가 즉시 좁혀집니다.

---

## 자주 묻는 질문

**Q: 커스텀 필드를 사용하여 필터링할 수 있나요?**

네, 고급 필터는 기본 필드와 커스텀 오브젝트 필드를 모두 지원합니다.

**Q: 여러 필터를 조합할 수 있나요?**

네, AND/OR 로직을 사용하여 복잡한 필터링 조건을 만들 수 있습니다.

**Q: 필터가 자동으로 저장되나요?**

아니요, 필터는 현재 뷰에만 적용되며 저장되지 않습니다.

**Q: 필터를 쉽게 제거할 수 있나요?**

네, 개별 필터를 삭제하거나 패널에서 모든 필터를 한 번에 지울 수 있습니다.

**Q: 고급 필터가 보이지 않는 이유는 무엇인가요?**

이 기능은 구독 플랜에 따라 다를 수 있습니다.
