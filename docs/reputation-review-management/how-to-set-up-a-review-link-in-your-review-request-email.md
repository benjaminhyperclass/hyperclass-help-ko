# 리뷰 요청 이메일에 리뷰 링크 설정하는 방법

리뷰 요청 이메일을 보내는 것은 고객 피드백을 수집하고 비즈니스 평판을 개선하는 데 필수적입니다. 이메일에 리뷰 링크를 포함하면 고객이 쉽게 피드백을 남길 수 있지만, 올바르게 설정하는 것이 중요합니다. 이 가이드에서는 리뷰 요청 이메일에 리뷰 링크를 효과적으로 설정하는 단계를 안내해드립니다.

## 리뷰 링크 설정 가이드

- [1단계: Reputation(평판 관리) 탭에서 Settings(설정) 접근](#step-1-reputation평판-관리-탭에서-settings설정-접근)
- [2단계: 이메일/SMS 발송용 리뷰 링크 설정](#step-2-이메일sms-발송용-리뷰-링크-설정)
- [3단계: 이메일 빌더에서 리뷰 링크 설정](#step-3-이메일-빌더에서-리뷰-링크-설정)

### 1단계: Reputation(평판 관리) 탭에서 Settings(설정) 접근

- 하위 계정에서 Reputation(평판 관리) 탭으로 이동한 후 Settings(설정)를 클릭하세요. 리뷰 링크 설정을 위한 섹션이 표시됩니다.

### 2단계: 이메일/SMS 발송용 리뷰 링크 설정

- 사용자로부터 리뷰를 받고 싶은 플랫폼에 따라 리뷰 링크를 설정할 수 있습니다 - Google, Facebook, 또는 Custom Link(커스텀 링크)

- 이 링크는 이메일/SMS로 발송되는 모든 리뷰 요청에서 기본으로 사용되는 링크입니다.

![리뷰 링크 설정 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155022685692/original/jVpvzsIDyIpTs-Fq-95-Y5-XTAAaUHQO-A.png?1710226015)

### 3단계: 이메일 빌더에서 리뷰 링크 설정

- 또는 Settings(설정) 탭의 Email Review Requests(이메일 리뷰 요청) 섹션으로 이동할 수도 있습니다. Email Builder(이메일 빌더)를 열고 → Add a Button(버튼 추가) → Link URL을 {{reputation.review_link}}로 설정하세요.

![이메일 빌더 설정 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155022686141/original/4FFz2rQtJty6KTI8r9TPej26dGZ3RgXNdw.png?1710226602)

![리뷰 링크 변수 설정 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155022686251/original/HkqyTXgLmXA_kQliWUWAMtscnXRX_Cw7zA.png?1710226756)

**주의사항**: 리뷰 링크를 설정하지 않으면 리뷰 요청을 보낼 수 없습니다.

이러한 단계를 따라하면 리뷰 요청 이메일에 리뷰 링크를 원활하게 설정할 수 있어, 고객으로부터 가치 있는 피드백을 수집하는 과정을 간소화할 수 있습니다.
