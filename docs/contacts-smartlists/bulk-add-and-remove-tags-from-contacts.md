# 연락처에서 태그 일괄 추가 및 제거하기

이 기능들은 태그에 대한 일괄 작업을 향상된 방식으로 제어할 수 있게 해주어, 연락처 관리를 더욱 효율적으로 만들어줍니다. 선택한 연락처에 태그를 쉽게 추가하거나, 선택한 태그 또는 모든 태그를 제거하고, 필요할 때 복원할 수 있습니다.

이 기능은 현재 Labs를 통해 사용할 수 있습니다.

이 문서의 내용:
- [기능 1 - 태그 일괄 추가](#기능-1---태그-일괄-추가)
- [기능 2 - 태그 일괄 제거](#기능-2---태그-일괄-제거)
- [기능 3 - 제거된 태그 복원](#기능-3---제거된-태그-복원)

## 기능 1 - 선택한 연락처에 태그 일괄 추가

여러 연락처를 선택하여 일괄 작업으로 태그를 추가할 수 있습니다.

방법:

**1단계:** 여러 연락처 선택

**2단계:** "Add Tag(태그 추가)" 일괄 작업(Bulk Actions) 클릭

**3단계:** 작업 이름을 입력하고 이 연락처들에 추가할 태그를 선택한 후 "Add Tags(태그 추가)" 클릭

**4단계:** 성공률 퍼센트를 클릭하여 진행 상황 확인

## 기능 2 - 선택한 연락처에서 태그 일괄 제거

여러 연락처를 선택하여 일괄 작업으로 태그를 제거할 수 있습니다.

방법:

**1단계:** 여러 연락처 선택

**2단계:** "Remove Tag(태그 제거)" 일괄 작업(Bulk Actions) 클릭

**3단계:** 작업 이름을 입력하고 이 연락처들에서 제거할 태그를 선택하거나, 모든 태그를 제거하는 옵션 선택

**4단계:** "Remove Tags(태그 제거)" 클릭

**5단계:** 성공률 퍼센트를 클릭하여 진행 상황 확인
