# CSV로 연락처와 기회 가져오기

"연락처와 기회 가져오기" 기능을 사용하면 CSV 파일에서 연락처와 기회를 모두 효율적으로 가져올 수 있습니다. 필드 매핑, 레코드 생성 또는 업데이트, 가져오기 프로세스 모니터링을 원활하게 수행하여 정확한 데이터 입력과 대량 작업의 시간 절약을 보장합니다.

연락처와 기회를 모두 가져오기 전에 이 문서를 확인해보세요: [기회 가져오기](https://help.leadconnectorhq.com/en/support/solutions/articles/155000003119-import-opportunities)

목차

- [1단계: 가져오기 옵션 접근](#step-1-access-import-option)
- [2단계: 가져올 객체 선택](#step-2-select-objects-to-import)

### 1단계: 가져오기 옵션 접근

- 기회 관리(Opportunities) 섹션으로 이동하여 점 3개 버튼(옵션 메뉴)을 클릭합니다.
- 가져오기(Import) 옵션을 선택하여 프로세스를 시작합니다.

![가져오기 옵션 접근](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155034983487/original/mrFhTvXDzXBoTbuukoK89ACy4RQPmU3ebQ.jpeg?1729262604)

### 2단계: 가져올 객체 선택

- 가져오기 화면에서 연락처(Contacts)와 기회(Opportunities)를 모두 가져올 객체로 선택하고 "다음"을 클릭합니다.

![가져올 객체 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155034983490/original/g9mLTO5Z_JluZxxM6Xdpc92qaDn8cBxv0g.jpeg?1729262605)

### 3단계: CSV 파일 업로드

- 연락처와 기회 세부 정보가 모두 포함된 CSV 파일을 업로드합니다.
- 각 객체에 대해 가져오기 중에 레코드를 생성할지, 업데이트할지, 아니면 생성 및 업데이트를 모두 할지 결정합니다.
- CSV에 기회 ID가 제공된 경우 기회가 업데이트됩니다. 제공되지 않은 경우 새 기회가 생성됩니다.
- "다음"을 클릭합니다.

연락처 중복 제거 설정은 전역 설정에서 자동으로 가져오며 가져오기 프로세스 중에는 업데이트할 수 없습니다.

![CSV 파일 업로드](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155034983500/original/ysqtJJt71J3SxnSZQEZfwYfem6l0B5X77w.jpeg?1729262606)

### 4단계: 필드 매핑

- CSV의 필드를 올바른 객체 필드에 매핑합니다.
- 오류를 방지하기 위해 각 필드를 매핑할 때 올바른 객체(연락처 또는 기회)를 선택하고 있는지 확인하세요.
- 이 기능은 여러 이메일 주소와 전화번호를 가진 연락처 생성도 지원합니다. 이러한 추가 정보는 가져오기 프로세스 중에 자동으로 추가됩니다.
- 연락처와 기회가 CSV의 같은 줄에 있는 경우 서로 자동으로 매핑됩니다.

![필드 매핑](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155034983521/original/FYbitXHh9rZK13xLxiPCUe7yq-ALNnHREg.jpeg?1729262606)

### 5단계: 확인 및 가져오기

- 매핑을 검토한 후 확인하고 가져오기를 진행합니다.
- 스마트 리스트 생성, 워크플로우에 연락처 추가 또는 가져온 연락처에 새 태그 추가를 선택할 수도 있습니다.

![확인 및 가져오기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155034983516/original/V88KmKIIaPHJmcMx7b08gZIae380BIzcPQ.jpeg?1729262606)

### 6단계: 가져오기 상태 모니터링

- 연락처와 기회 일괄 작업 페이지를 통해 가져오기의 진행 상황과 상태를 모니터링할 수 있습니다.
- 연락처와 기회 가져오기는 모두 일괄 작업 아래 같은 곳에 나타납니다.
- "통계 보기" 버튼을 클릭하여 가져오기 통계를 확인하세요. 오류 로그를 다운로드하고 "오류" 탭에서 오류에 대한 자세한 내용과 해결 방법을 볼 수도 있습니다.
