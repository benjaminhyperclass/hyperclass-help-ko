# 커스텀 라벨을 이용한 연락처 연결 (출시 예정)

이 기능을 사용하면 커스텀 라벨을 통해 여러 연락처를 서로 연결할 수 있습니다. 가족 구성원이나 비즈니스 파트너 등 연락처 간의 관계를 관리해야 하는 비즈니스에 특히 유용합니다. 연결된 연락처들의 커뮤니케이션과 데이터 정리를 간소화하고 싶으시다면, 아래 정보를 통해 핵심 기능, 장점, 그리고 효과적인 활용법을 알아보세요.

목차

- [기능 1 - 연결 관계 만들기](https://hyperclass.gitbook.io/hyperclass-docs)
- [기능 2 - 연락처 연결 및 연결된 연락처 보기](https://hyperclass.gitbook.io/hyperclass-docs)

## 기능 1 - 연결 관계 만들기

커스텀 라벨을 사용하여 두 명 이상의 연락처를 연결할 수 있습니다. 이 기능은 연락처 상세 페이지에서 "연락처 연결(Associate Contact)" 옵션을 선택하여 이용할 수 있습니다. "부모", "배우자", "비즈니스 파트너" 등의 관계 라벨을 지원합니다.

연결 라벨 생성 및 업데이트는 관리자(Admin) 역할에서만 사용할 수 있습니다.

- **연결 관계를 만드는 방법:**
  - `Settings(설정) → Objects(객체) → Contacts(연락처)`로 이동하세요.
  - "+ Create Association(+ 연결 관계 생성)"을 클릭하세요.

![연결 관계 생성 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155034995727/original/EKs7g3Ivnst59SM8jkR3igktKLbrZLiOlw.jpeg?1729277178)

- 커스텀 단일 라벨(예: 파트너, 배우자) 또는 쌍 라벨(예: 남편 - 아내, 매니저 - 직원)을 만들고 저장을 클릭하세요.

![라벨 생성 양식](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155034995734/original/O92K9GRD58PVlgXPU_Zz40L4Y_vExpLJtw.jpeg?1729277179)

- 이제 연결 라벨이 생성되었습니다.

![생성된 연결 라벨](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155034995732/original/8qUqRZIFswXHFQD-UB3sF02y-2o7xOxKFg.jpeg?1729277179)

- 우측의 케밥 메뉴(점 3개)를 클릭하여 연결 라벨을 편집하거나 삭제할 수 있습니다.

![라벨 편집/삭제 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155034995723/original/1Ps4Y1dpCfnV1aRutiDfBWisNn1CEVY4uQ.jpeg?1729277176)

연결 라벨 삭제는 영구적이며 되돌릴 수 없는 변경입니다. 연결 라벨을 삭제하려면 해당 라벨을 사용하여 생성된 모든 연결을 먼저 삭제해야 합니다.

두 연락처 간에는 최대 10개의 연결 라벨을 설정할 수 있습니다. 
현재 모든 연결 라벨은 다대다(Many to Many) 관계입니다. 다른 제한사항들은 곧 업데이트될 예정입니다.

## 기능 2 - 연락처 연결 및 연결된 연락처 보기

연락처가 연결되면 연결된 연락처와 사용된 라벨을 확인할 수 있습니다. 연결된 연락처는 연락처 상세 페이지의 전용 섹션에 표시됩니다.

- **연결된 연락처를 보는 방법:**
  - 연락처의 상세보기를 열고 우측 패널에서 "Associations(연결 관계)" 옵션으로 이동하세요.

![연결 관계 패널](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155034995733/original/l1KUPePnJhdOWbXJOPcP1hVOCqpc41qetg.jpeg?1729277179)

- Contacts(연락처) 근처의 "Add(추가)"를 클릭하여 새 연락처 연결을 시작하세요.

![연락처 추가 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155034995724/original/nS5dzo-GVSMBGgRcO5uqWVQuQabHja-oXQ.jpeg?1729277176)

- 연락처를 선택하고 두 연락처를 연결할 라벨을 추가하세요. 하나 또는 여러 개의 라벨을 선택할 수 있습니다.

![연락처 선택 및 라벨 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155034995725/original/ukOt8S5U4wxsH88MNPJMH8n26xssPW5AvA.jpeg?1729277177)

- 연결된 연락처가 우측 패널에 표시되기 시작합니다.

![연결된 연락처 표시](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155034995722/original/eAuWFvn_zM5uqHg2U9d3g3I9HEr4f4b8FQ.png?1729277176)

- 상대방 연락처에도 연결 관계가 반영됩니다.

![양방향 연결 반영](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155034995728/original/-c1GRJ2gG2e5KrmS8h8VEOa2TL1HY29Y8g.jpeg?1729277178)

하나의 연결 라벨에 최대 1,000개의 연락처를 추가할 수 있습니다.
