# 퍼널/웹사이트 이벤트 추적을 위한 픽셀 ID 이해하기

웹사이트에서 이벤트 추적을 효과적으로 관리하려면 다음 단계에 따라 픽셀 ID를 찾고 생성하세요:

## 새로운 Facebook 픽셀 만들기

- **Facebook 광고 관리자 접속:**
  Facebook 광고 관리자를 열어주세요.
  상단 메뉴를 클릭하고 "픽셀(Pixels)" 링크를 선택하세요.

- **픽셀 생성:**
  현재 계정에 연결된 픽셀이 없다면 픽셀 생성 안내가 표시됩니다.
  녹색 "픽셀 만들기(Create a Pixel)" 버튼을 클릭하세요.

- **픽셀 이름 지정:**
  새 픽셀의 이름을 입력하세요.
  "만들기(Create)"를 클릭하여 과정을 완료하세요.

이제 새 픽셀이 준비되었으며 퍼널과 웹사이트에 설치할 수 있습니다.
