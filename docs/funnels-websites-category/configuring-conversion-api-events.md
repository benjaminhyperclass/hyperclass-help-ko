# Conversion API 이벤트 설정하기

브라우저 제한과 광고 차단기를 우회하여 자동으로 트리거되는 표준 이벤트를 Facebook 서버로 직접 전송해서 이벤트 추적을 강화하세요.

Conversion API라고 알려진 이 강력한 기능은 원활한 이벤트 추적을 가능하게 하며, 브라우저와 Conversion API 모두를 사용하여 모든 이벤트를 전송할 수 있습니다.

## 액세스 토큰 생성 단계

- **이벤트 관리자 열기**: 상단 메뉴에 접속하여 클릭한 후 "Events Manager"를 선택하세요.
- 또는 직접 [Facebook Events Manager](https://business.facebook.com/events_manager/)로 이동할 수 있습니다.
- **픽셀 설정 찾기**: 픽셀 ID를 클릭하세요.
- 왼쪽 사이드 메뉴에서 "Settings(설정)"을 클릭하세요.
- **액세스 토큰 생성**: "Server-Side API for Web" 섹션까지 아래로 스크롤하세요.
- "Create Access Token(액세스 토큰 생성)"을 클릭하고 안내를 따라하세요.
- 픽셀을 선택하고, 앱을 생성하거나 선택하며, 사용자를 생성하거나 선택하라는 메시지가 표시됩니다.
- 토큰이 생성됩니다. 토큰을 클릭하여 복사하세요.

액세스 토큰을 사용하여 웹사이트에서 견고하고 안정적인 이벤트 추적을 위한 Conversion API 이벤트를 설정할 수 있습니다.
