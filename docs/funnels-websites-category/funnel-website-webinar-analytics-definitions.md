# 퍼널·웹사이트·웨비나 분석 지표 정의

이 가이드는 퍼널(Funnel), 웹사이트(Website), QR 코드 및 웨비나(Webinar) 분석을 볼 때 나타나는 각 지표의 의미를 설명합니다.

이러한 정의를 활용하여 성과를 이해하고, 참여도를 개선하며, 전환율을 최적화하세요.

## 페이지 조회수

퍼널이나 웹사이트 페이지가 방문된 횟수를 추적합니다.

| **유형** | **설명** |
|----------|----------|
| **전체** | 전체 페이지 조회수 — 재방문을 포함한 모든 개별 방문을 포함합니다. |
| **고유** | 페이지를 최소 한 번 방문한 고유 방문자수입니다. |

예시:
한 사람이 같은 페이지를 두 번 방문 → 전체 = 2, 고유 = 1

---

## 옵트인

제안에 관심을 표현한 사용자를 추적합니다.

| **유형** | **설명** |
|----------|----------|
| **전체** | 상품에 옵트인하거나, 상품을 구매하거나, 폼/설문을 제출하거나, 캘린더를 통해 예약을 완료한 사용자 수입니다. |
| **전환율** | "전체 옵트인"을 고유 페이지 조회수로 나눈 값입니다. (옵트인 ÷ 고유 페이지 조회수 × 100) |

예시:
고유 페이지 조회수 200건에서 옵트인 50건 → 전환율 = 25%

---

## QR 스캔

퍼널이나 랜딩 페이지의 QR 코드 참여도를 측정합니다.

| **유형** | **설명** |
|----------|----------|
| **QR 스캔** | QR 스캔 총 횟수입니다. |
| **고유 QR 스캔** | QR 코드를 스캔한 고유 방문자수입니다. |

예시:
3명이 각각 같은 코드를 두 번씩 스캔 → QR 스캔 = 6, 고유 QR 스캔 = 3

## 매출 – 주문

퍼널에서 구매가 발생한 횟수를 계산합니다.

| **레벨** | **설명** |
|----------|----------|
| **퍼널 단계 레벨** | 발생한 판매 주문 수 — 주문이 몇 번 이루어졌는지 (상품 개수나 수량은 포함하지 않음)입니다. |
| **개별 상품 레벨** | 특정 상품이 구매된 횟수입니다. |
| **범프 상품** | 범프 상품이 구매된 횟수입니다. |

예시:
한 단계에서 총 30개 주문이 발생; 18개는 상품 A를 포함, 5개는 범프 오퍼를 포함 →
퍼널 단계 레벨 = 30, 개별 상품 레벨 = 18, 범프 상품 = 5

참고:
범프 상품 데이터 분류는 2023년 4월 27일 오후 12:00 GMT부터 시작되었습니다.
그 이전 날짜의 범프 구매는 개별 상품 레벨 통계에 포함되어 나타납니다.

---

## 매출 – 전환율

페이지 조회수에서 구매까지의 전환을 보여줍니다.

| **레벨** | **설명** |
|----------|----------|
| **퍼널 단계 레벨** | 모든 상품 구매 횟수를 고유 페이지 조회수로 나눈 값입니다. |
| **개별 상품 레벨** | 특정 상품 구매 횟수를 고유 페이지 조회수로 나눈 값입니다. |
| **범프 상품** | 범프 상품 구매 횟수를 해당 단계의 총 주문 수로 나눈 값입니다. |

예시:
총 100개 주문 중 범프 구매 20건 → 전환율 = 20%

참고:
범프 상품 전환율 분류도 2023년 4월 27일 오후 12:00 GMT부터 시작되었습니다.
그 이전에는 범프 통계가 개별 상품 레벨 데이터에 포함되어 계산되었습니다.

## 매출 – 수량

판매된 총 상품 수를 보여줍니다.

| **레벨** | **설명** |
|----------|----------|
| **퍼널 단계 레벨** | 이 퍼널 단계에서 구매된 모든 상품의 총 수량입니다. |
| **상품 레벨** | 해당 퍼널 단계에서 특정 상품의 총 수량입니다. |

예시:
각각 3개 상품이 포함된 10개 주문 → 수량 = 30

## 매출 – 금액

발생한 총 매출액을 추적합니다.

| **레벨** | **설명** |
|----------|----------|
| **퍼널 단계 레벨** | 해당 퍼널 단계에서 구매된 모든 상품의 총 가치입니다. |
| **상품 레벨** | 해당 퍼널 단계에서 특정 상품의 총 매출 가치입니다. |

예시:
상품 A가 각각 $50에 10번 판매 → 금액 = $500

## 평균 장바구니 금액

주문당 평균 매출을 보여줍니다.

| **레벨** | **설명** |
|----------|----------|
| **퍼널 단계 레벨** | 총 금액 ÷ 총 주문 수입니다. |
| **상품 레벨** | 해당 없음입니다. |

예시:
20개 주문에서 총 $1000 매출 → 평균 장바구니 금액 = $50

## 사이트 평균 체류 시간

선택한 날짜 범위 동안 사용자가 사이트에서 보내는 평균 시간을 추적합니다.

| **유형** | **설명** |
|----------|----------|
| **전체** | 선택한 기간(일별, 주별 또는 사용자 지정 범위) 동안 사용자가 사이트에서 보내는 평균 시간입니다. |

**계산 방법:**
모든 사용자의 총 체류 시간 ÷ 총 세션 수

**지표 의미:**
시간에 따른 사용자의 사이트 참여도를 나타냅니다.

**예시:**
사용자들이 400개 세션에서 총 2,000분을 보냄 → **사이트 평균 체류 시간 = 5분**

---

## 평균 세션 시간

선택한 날짜 범위에서 개별 세션의 평균 길이를 추적합니다.

| **유형** | **설명** |
|----------|----------|
| **전체** | 선택한 날짜 범위의 모든 세션에서 사용자가 세션당 보내는 평균 시간입니다. |

**계산 방법:**
모든 세션 시간의 합 ÷ 총 세션 수

**지표 의미:**
세션 기준으로 본 사이트 전반의 사용자 참여도를 나타냅니다.

**예시:**
300개 세션에서 총 세션 시간이 1,500분 → **평균 세션 시간 = 5분**

---

## 이탈률 (30초 미만)

사용자가 사이트 도착 후 곧바로 떠나는 세션 수를 추적합니다.

| **유형** | **설명** |
|----------|----------|
| **전환율** | 사용자가 도착 후 첫 30초 이내에 사이트를 떠나는 세션의 비율입니다. |

**계산 방법:**
(30초 이내에 끝난 세션 ÷ 총 세션 수) × 100

**지표 의미:**
짧은 시간 동안만 머무르거나 참여도가 낮은 세션을 나타내며, 주로 느린 로딩 시간, 관련 없는 콘텐츠 또는 사용자 기대치 불일치로 인해 발생합니다.

예시:
총 300개 세션 중 45개 세션이 30초 이내에 종료 → **이탈률 = 15%**

## 웨비나 통계

사용자가 웨비나에 등록하고 참석하는 방법을 추적합니다.

| **지표** | **설명** |
|----------|----------|
| **등록** | 웨비나에 등록한 고유 사용자 수입니다. |
| **참석자** | 웨비나에 참석한 고유 사용자 수 — 즉, 브로드캐스트 페이지를 통해 라이브 웨비나 링크로 리디렉션된 사용자입니다. |
| **반복** | 반복 웨비나의 모든 세션에 대한 총 등록 수입니다. |
| **재참석자** | 두 번 이상의 웨비나 세션에 참석한 고유 사용자 수입니다. |
| **평균 참석자** | 세션당 평균 참석자 수 (총 참석자 ÷ 총 세션)입니다. |
| **전환율 %** | 웨비나 세션의 총 등록 대비 참석자 비율 (참석자 ÷ 등록 × 100)입니다. |
| **평균 전환율 %** | 모든 세션의 평균 전환율입니다. |
| **총 세션** | 선택한 시간대 내에 진행된 총 세션 수로, 반복 변경으로 인해 취소된 세션도 포함합니다. |

예시:
500명이 등록하고 300명이 참석 → 전환율 % = (300 ÷ 500) × 100 = 60%

## 호스팅 영상 통계

호스팅 웨비나의 영상 상호작용과 참여도를 측정합니다.

| **지표** | **설명** |
|----------|----------|
| **영상 재생** | 웨비나 영상을 재생한 고유 사용자 수입니다. |
| **영상 일시정지** | 영상을 최소 한 번 일시정지한 고유 사용자 수입니다. |
| **평균 시간** | 사용자당 평균 영상 진행률입니다. 사용자가 여러 번 시청한 경우, 가장 높은 진행률 %가 사용됩니다. |
| **이탈 급증** | 가장 일반적인 이탈 지점에서 시청을 중단한 시청자의 비율입니다. |
| **완료율 %** | 총 재생 수 대비 영상을 완료한 사용자의 비율입니다. |

예시:
200명이 영상을 재생하고 120명이 시청 완료 → 완료율 % = (120 ÷ 200) × 100 = 60%

## 영상 참여 그래프

웨비나에서 참석자가 언제 시청을 중단하는지 시각화합니다.

| **지표** | **설명** |
|----------|----------|
| **영상 진행률 %** | 10% 간격(0%, 10%, 20%, ... 100%)으로 측정된 사용자 영상 진행률입니다. |
| **사용자 수** | 각 간격에서 시청을 중단한 사용자 수입니다. |

예시:
대부분의 사용자가 40% 진행률에서 이탈 → 웨비나의 해당 구간 참여도 개선에 집중하세요.

## 성과 지표

#### 목표로 해야 할 지표

- 고유 방문자와 등록 수 증가
- 높은 웨비나 참석률과 재참여율
- 강한 평균 시청 시간과 완료율
- 일관되거나 개선되는 전환율
- 건강한 평균 장바구니 금액과 높은 조회당 매출

#### 성과 경고 신호

- 고유 트래픽 또는 등록 수 감소
- 낮은 옵트인 또는 매출 전환율
- 조기 영상 이탈 또는 낮은 완료율
- 작은 평균 주문 금액 또는 저조한 업셀 성과
- 빈번한 재생 또는 기기별 문제
