# 대화 관리 빠른 시작 가이드

**대화(Conversations)**는 모든 고객 커뮤니케이션을 한 곳에서 관리할 수 있는 통합 메시지 센터입니다. SMS 문자, 이메일, 전화, 채팅 위젯, WhatsApp, Instagram DM, Facebook Messenger 등 모든 채널의 메시지가 여기에 모입니다.

## 대화 섹션 둘러보기

![대화 메인 화면](https://d33v4339jhl8k0.cloudfront.net/docs/assets/632b87c68d9f883fd696a4c3/images/63f8eca1e1dd8d48e2aabf47/file-jkl3nEsZCQ.png)

왼쪽 사이드바에서 **Conversations(대화)**를 클릭하면 다음과 같은 메뉴를 볼 수 있습니다:

### 인박스(Inbox) 메뉴
- **Unread(읽지 않음)**: 아직 확인하지 않은 새 메시지들
- **Starred(즐겨찾기)**: 별표를 눌러 중요 표시한 대화들
- **Recents(최근 대화)**: 가장 최근에 메시지가 오간 대화 목록
- **Manual Actions(수동 작업)**: 워크플로우에서 사람이 직접 처리해야 하는 대기 중인 작업들

### 채널별 대화
각 커뮤니케이션 채널별로 대화를 필터링해서 볼 수 있습니다:
- **Chat Widget(채팅 위젯)**: 웹사이트 실시간 채팅
- **SMS**: 문자 메시지
- **Email(이메일)**: 이메일 대화
- **Calls(전화)**: 통화 기록
- **WhatsApp(왓츠앱)**: WhatsApp 메신저
- **Instagram DM(인스타그램 DM)**: 인스타그램 다이렉트 메시지
- **Facebook Messenger(페이스북 메신저)**: 페이스북 메신저
- **Google My Business(구글 마이 비즈니스)**: GMB를 통한 고객 메시지

## 대화 화면 구성

![대화 상세 화면](https://d33v4339jhl8k0.cloudfront.net/docs/assets/632b87c68d9f883fd696a4c3/images/63f8ece7b69f783cf1afbdce/file-9mOyM7dGks.png)

대화 화면은 크게 3개 영역으로 나뉩니다:

### 1. 대화 목록 (왼쪽)
- 현재 선택한 필터에 해당하는 대화들이 목록으로 표시됩니다
- 각 대화에는 고객 이름, 마지막 메시지, 시간이 표시됩니다
- 읽지 않은 메시지는 굵은 글씨로 강조됩니다

### 2. 메시지 창 (가운데)
- 선택한 고객과의 전체 메시지 기록이 시간순으로 표시됩니다
- 여러 채널(SMS, 이메일, 채팅 등)의 메시지가 하나의 스레드로 통합됩니다
- 하단에서 새 메시지를 작성하고 전송할 수 있습니다

### 3. 연락처 정보 (오른쪽)
- 선택한 고객의 세부 정보가 표시됩니다
- 태그, 커스텀 필드, 예약 기록, 기회 관리 등에 빠르게 접근할 수 있습니다

## 주요 기능 활용법

### 메시지 보내기
1. 대화 목록에서 고객을 선택하거나, 새 대화를 시작하려면 우측 상단의 **+** 버튼을 클릭하세요
2. 하단 메시지 입력창에서 채널을 선택합니다 (SMS, 이메일, 등)
3. 메시지를 작성하고 **전송** 버튼을 클릭합니다

### 대화에 별표 표시하기
중요한 대화는 별표를 눌러 즐겨찾기에 추가할 수 있습니다. 대화 목록에서 각 대화 옆의 ☆ 아이콘을 클릭하세요.

### 수동 작업 처리하기
워크플로우에서 사람이 직접 처리해야 하는 작업들이 **Manual Actions(수동 작업)**에 표시됩니다. 각 작업을 클릭하여 완료하거나 건너뛸 수 있습니다.

### 팀 협업
여러 팀원이 함께 고객 대화를 관리할 때는 다음 기능들을 활용하세요:
- **내부 메모**: 고객에게 보이지 않는 팀 내부 커뮤니케이션
- **배정**: 특정 대화를 특정 팀원에게 할당
- **상태 표시**: 대화가 처리 중인지, 완료되었는지 등의 상태 관리

## 다음 단계

대화 관리에 익숙해지셨다면 다음 기능들도 살펴보세요:
- **워크플로우**: 반복적인 메시지 응답을 자동화하세요
- **템플릿**: 자주 사용하는 메시지를 템플릿으로 저장하세요
- **AI 챗봇**: Conversation AI로 고객 문의에 24시간 자동 응답하세요

대화 관리를 통해 고객과의 모든 커뮤니케이션을 효율적으로 관리하고, 더 나은 고객 경험을 제공하실 수 있습니다.
