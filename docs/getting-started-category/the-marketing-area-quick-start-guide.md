# 마케팅 영역 빠른 시작 가이드

마케팅(Marketing) 영역은 이메일 캠페인, 소셜 미디어 예약, 트리거 링크 등 다양한 마케팅 도구를 제공합니다. 이 가이드를 통해 마케팅 기능을 빠르게 시작해보세요.

## 마케팅 영역 구성 요소

### 이메일 캠페인(Email Campaigns)
- **대량 이메일 발송**: 여러 연락처에게 한 번에 뉴스레터나 프로모션을 전송
- **템플릿 활용**: 미리 제작된 이메일 템플릿으로 빠른 캠페인 제작
- **성과 추적**: 열람률, 클릭률 등 캠페인 성과 분석

### 소셜 플래너(Social Planner)
- **게시물 예약**: 인스타그램, 페이스북 등에 게시물을 미리 예약
- **콘텐츠 관리**: 여러 SNS 계정을 한 곳에서 통합 관리
- **최적 시간 게시**: 타겟 고객이 가장 활발한 시간에 자동 게시

### 트리거 링크(Trigger Links)
- **자동화 연결**: 고객이 링크를 클릭하면 특정 워크플로우가 자동 실행
- **행동 추적**: 고객의 관심사와 참여도를 세밀하게 파악
- **맞춤형 경험**: 클릭 기반으로 개인화된 마케팅 제공

## 마케팅 기능 시작하기

1. **좌측 메뉴에서 Marketing(마케팅) 클릭**
2. **원하는 기능 선택**: Email Campaigns, Social Planner, Trigger Links 중 선택
3. **첫 번째 캠페인/게시물/링크 생성**
4. **타겟 설정**: 적절한 연락처 그룹이나 태그 선택
5. **성과 모니터링**: 리포팅(Reporting) 섹션에서 결과 확인

마케팅 도구들은 모두 연락처(Contacts) 데이터와 연동되어 더욱 정확한 타겟팅과 개인화가 가능합니다.
