# LeadConnector 모바일 앱 빠른 시작 가이드

LeadConnector 모바일 앱으로 언제 어디서나 고객과 소통하고 비즈니스를 관리하세요.

## 앱 다운로드

**iOS:** App Store에서 "LeadConnector"를 검색하여 다운로드하세요.

**Android:** Google Play Store에서 "LeadConnector"를 검색하여 다운로드하세요.

## 로그인

1. 앱을 실행하고 Hyperclass 계정으로 로그인하세요.
2. 이메일과 비밀번호를 입력하세요.
3. 여러 하위 계정(Sub-account)이 있다면 작업할 계정을 선택하세요.

## 주요 기능

### 대화(Conversations)
- 모든 고객 메시지를 한 곳에서 관리
- SMS, 이메일, 채팅 위젯, WhatsApp 등 지원
- 실시간 알림 수신

### 연락처(Contacts)
- 고객 정보 조회 및 편집
- 새 연락처 추가
- 태그 관리

### 캘린더(Calendar)
- 예약 일정 확인
- 새 예약 추가
- 예약 변경 및 취소

### 기회 관리(Opportunities)
- 영업 파이프라인 확인
- 기회 카드 이동
- 진행 상황 업데이트

### 전화(Calls)
- 앱 내에서 바로 전화 걸기
- 통화 기록 확인
- 자동 통화 로깅

## 빠른 팁

- **알림 설정**: 설정에서 푸시 알림을 활성화하여 중요한 메시지를 놓치지 마세요.
- **오프라인 동기화**: 인터넷 연결이 다시 되면 자동으로 데이터가 동기화됩니다.
- **생체 인증**: 보안을 위해 지문이나 얼굴 인식으로 앱 잠금을 설정할 수 있습니다.

모바일 앱으로 이동 중에도 고객과의 소통을 놓치지 않고 효율적으로 비즈니스를 관리하세요.
