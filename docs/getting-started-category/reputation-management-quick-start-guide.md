# 평판 관리 빠른 시작 가이드

평판 관리(Reputation Management)는 고객 리뷰를 자동으로 요청하고, 수집하며, 온라인 평판을 개선하는 데 도움을 주는 강력한 도구입니다. 이 가이드를 통해 빠르게 설정하고 시작해보세요.

## 평판 관리란?

평판 관리는 다음과 같은 기능을 제공합니다:

- **자동 리뷰 요청**: 서비스 완료 후 고객에게 자동으로 리뷰를 요청
- **리뷰 수집**: Google, Facebook 등 주요 플랫폼의 리뷰를 한 곳에서 관리
- **긍정적 평판 구축**: 만족한 고객의 리뷰를 공개 플랫폼으로 유도
- **부정적 피드백 관리**: 불만족 고객의 의견을 비공개로 수집하여 개선에 활용

## 1단계: 평판 관리 설정

### 리뷰 플랫폼 연결

1. **Reputation(평판)** 메뉴로 이동합니다
2. **Settings(설정)** 탭을 클릭합니다
3. 리뷰를 받고 싶은 플랫폼을 추가합니다:
   - Google My Business
   - Facebook
   - 기타 리뷰 사이트

### 리뷰 요청 설정

1. **리뷰 요청 메시지** 작성:
   - 친근하고 정중한 톤으로 작성
   - 고객이 왜 리뷰를 남겨야 하는지 명확히 설명
   - 간단하고 이해하기 쉬운 문구 사용

2. **발송 타이밍** 설정:
   - 서비스 완료 후 적절한 시간 간격 설정
   - 너무 즉시 보내면 부담스러울 수 있음
   - 너무 늦게 보내면 기억이 희미해질 수 있음

## 2단계: 자동화 워크플로우 만들기

### 기본 워크플로우 설정

1. **Workflows(워크플로우)** 메뉴로 이동합니다
2. **Create Workflow(워크플로우 만들기)**를 클릭합니다
3. 트리거를 설정합니다:
   - 예약 완료
   - 특정 태그 추가
   - 프로젝트 완료

### 리뷰 요청 액션 추가

1. 워크플로우에 **Send Review Request(리뷰 요청 보내기)** 액션을 추가합니다
2. 발송 방법을 선택합니다:
   - 이메일
   - SMS
   - 둘 다

3. 메시지 내용을 커스터마이징합니다

## 3단계: 리뷰 관리 프로세스

### 긍정적 리뷰 처리

- **5점 리뷰**: 공개 플랫폼(Google, Facebook)으로 유도
- **감사 메시지**: 자동으로 감사 인사 발송
- **공유 독려**: 추가 홍보를 위한 공유 요청

### 부정적 리뷰 처리

- **3점 이하 리뷰**: 내부 피드백으로 처리
- **즉시 대응**: 문제 해결을 위한 연락
- **개선 기회**: 서비스 품질 향상의 기회로 활용

## 4단계: 리뷰 모니터링

### 대시보드 활용

1. **Reputation(평판)** 대시보드에서 다음을 확인할 수 있습니다:
   - 총 리뷰 수
   - 평균 별점
   - 최근 리뷰 활동
   - 플랫폼별 성과

### 정기적인 분석

- **주간 리뷰**: 새로운 리뷰와 트렌드 확인
- **월간 분석**: 전반적인 평판 변화 추적
- **개선 계획**: 부정적 피드백을 바탕으로 서비스 개선

## 5단계: 리뷰 위젯 활용

### 웹사이트에 리뷰 표시

1. **Review Widget(리뷰 위젯)**을 생성합니다
2. 웹사이트에 삽입하여 긍정적인 리뷰를 방문자에게 보여줍니다
3. 정기적으로 표시되는 리뷰를 업데이트합니다

## 성공을 위한 팁

### 리뷰 요청 모범 사례

- **적절한 타이밍**: 고객이 만족감을 느끼는 순간에 요청
- **개인화**: 고객 이름과 서비스 내용을 포함
- **간단함**: 복잡한 과정 없이 쉽게 리뷰를 남길 수 있도록

### 응답 관리

- **신속한 응답**: 모든 리뷰에 24시간 내 응답
- **진정성**: 템플릿보다는 개인적인 메시지
- **해결 의지**: 부정적 리뷰에는 구체적인 해결 방안 제시

## 다음 단계

평판 관리 기본 설정이 완료되면:

1. **고급 자동화** 설정으로 더욱 정교한 리뷰 수집 시스템 구축
2. **AI Reviews(리뷰 AI)** 활용으로 자동 응답 시스템 도입
3. **다른 마케팅 도구**와 연동하여 종합적인 고객 경험 관리

평판 관리는 꾸준함이 핵심입니다. 정기적으로 모니터링하고 개선하여 온라인 평판을 지속적으로 향상시키세요.
