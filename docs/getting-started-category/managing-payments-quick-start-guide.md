# 결제 관리 빠른 시작 가이드

결제(Payments) 기능을 사용하면 고객으로부터 직접 결제를 받고 관리할 수 있습니다. 이 가이드에서는 결제 시스템을 설정하고 첫 번째 상품을 만드는 방법을 안내해드립니다.

## 1단계: 결제 처리업체 연결

먼저 Stripe 계정을 연결해야 합니다:

1. **Settings(설정) → Integrations(연동)**으로 이동
2. **Stripe** 섹션을 찾아 **Connect(연결)** 클릭
3. Stripe 계정에 로그인하거나 새로 생성
4. 연결 권한 승인

## 2단계: 첫 번째 상품 만들기

1. **Payments(결제) → Products(상품)**으로 이동
2. **Add Product(상품 추가)** 버튼 클릭
3. 상품 정보 입력:
   - **상품명**: 고객에게 표시될 이름
   - **가격**: 금액 설정
   - **설명**: 상품에 대한 간단한 설명
4. **Save(저장)** 클릭

## 3단계: 결제 링크 생성

상품을 만든 후 고객에게 보낼 결제 링크를 생성할 수 있습니다:

1. 상품 목록에서 원하는 상품 선택
2. **Create Payment Link(결제 링크 생성)** 클릭
3. 링크를 복사해서 고객에게 전송

## 4단계: 주문 확인

결제가 완료되면:

1. **Payments(결제) → Orders(주문)**에서 주문 내역 확인
2. 고객에게 자동으로 결제 확인 이메일 발송
3. 필요시 인보이스(Invoice) 생성 및 전송

## 추가 기능

### 구독 상품
정기 결제가 필요한 서비스는 **Subscription(구독)** 타입으로 설정할 수 있습니다.

### 쿠폰 할인
**Coupons(쿠폰)** 메뉴에서 할인 코드를 만들어 고객에게 제공할 수 있습니다.

### 결제 분석
**Reporting(리포팅)**에서 매출 현황과 결제 통계를 확인할 수 있습니다.

결제 기능을 활용하면 복잡한 전자상거래 시스템 없이도 간편하게 온라인 결제를 받을 수 있습니다. 추가 질문이 있으시면 언제든 문의해주세요.
