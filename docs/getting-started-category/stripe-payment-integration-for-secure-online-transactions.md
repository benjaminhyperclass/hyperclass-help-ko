# Stripe 결제 연동으로 안전한 온라인 거래하기

온라인 결제는 간단하고 안전하며 신뢰할 수 있어야 합니다. 결제 제공업체를 연동하면 거래를 처리하고 구독을 관리하며 고객 결제를 원활하게 처리할 수 있습니다.

이 가이드는 Stripe를 연결하여 빠르고 효율적으로 결제를 받기 시작하는 방법을 안내합니다.

---

# **Stripe 연동이란?**

Stripe 연동을 통해 시스템 내에서 직접 신용카드 결제를 안전하게 받고, 구독을 관리하며, 거래를 처리할 수 있습니다. Stripe 계정을 연결하면 고객에게 원활한 결제 경험을 제공하면서 청구 및 리포팅에 대한 완전한 제어권을 유지할 수 있습니다.

이 연결을 통해 모든 거래가 안전하게 처리되고 결제 데이터가 추적 및 관리를 위해 시스템으로 올바르게 전달됩니다.

---

## **Stripe 연동의 주요 장점**

Stripe 연결은 고객 경험과 운영 효율성을 모두 개선하는 여러 가지 장점을 제공합니다:

- **안전한 결제:** 신뢰할 수 있는 결제 인프라로 거래를 안전하게 처리
- **더 빠른 결제:** 간소화된 결제 흐름으로 마찰 감소
- **구독 관리:** 정기 청구 및 플랜을 쉽게 처리
- **중앙 집중식 추적:** 모든 결제 활동을 한 곳에서 관리
- **확장 가능한 설정:** 유연한 결제 기능으로 성장 지원

---

## **Stripe 연결 방법**

적절한 설정으로 결제가 문제없이 처리되도록 합니다. 아래 단계에 따라 Stripe 계정을 연결하세요.

### 1단계: 결제 연동 메뉴로 이동

Payments(결제) 섹션을 열고 결제 제공업체가 관리되는 연동 영역에 접근합니다.

![Payments 섹션 위치 및 Integrations 탭 열기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155069332733/original/QWc6ZqroAXhfgIwtY4s7XUS6_hHbIEMDrg.png?1776367119)

이 이미지는 Payments 섹션을 찾고 Integrations 탭을 여는 방법을 보여줍니다.

- 좌측 네비게이션에서 **Payments(결제)** 클릭
- 상단 메뉴에서 **Integrations(연동)** 선택

### **2단계:** Stripe 선택 및 연결 시작

사용 가능한 제공업체 목록에서 Stripe를 찾고 연결 프로세스를 시작합니다.

![Stripe 옵션 및 Connect 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155069332852/original/yqGyYKAVW6J_EHVYt4hnukNEo8pICSRjIQ.png?1776367283)

이 이미지는 Stripe 옵션과 Connect 버튼을 강조합니다.

- 사용 가능한 제공업체에서 **Stripe** 찾기
- **Connect(연결)** 클릭하여 진행

### **3단계:** 계정 로그인 또는 생성

Stripe 계정을 인증하기 위한 보안 로그인 페이지로 리디렉션됩니다.

- Stripe 자격 증명 입력
- 필요한 경우 새 계정 생성

### **4단계:** 연결 승인

연동을 완료하기 위해 필요한 권한을 부여합니다.

- 액세스 권한 검토
- **Authorize(승인)** 클릭하여 연결 완료

완료되면 계정이 성공적으로 연결됩니다.

### **5단계:** 연동 확인 및 관리

연결 후 연동을 관리할 수 있는 옵션이 표시됩니다.

- 연결 상태 확인
- **Manage(관리)** 클릭하여 설정 업데이트 또는 검토

## 결제 설정 모범 사례

다음 권장 사항을 따르면 원활한 결제 처리를 보장할 수 있습니다:

- 연결하기 전에 인증된 Stripe 계정 사용
- 설정 중 계정 권한 재확인
- 연동 후 결제 테스트하여 기능 확인
- 정확한 리포팅을 위해 청구 및 상품 데이터를 체계적으로 관리

## **자주 묻는 질문**

**Q: Stripe를 연결한 후에는 어떻게 되나요?**
즉시 결제, 구독 및 거래를 받기 시작할 수 있습니다.

**Q: 나중에 Stripe 연결을 해제할 수 있나요?**
네, 언제든지 연동 페이지에서 연동을 관리하거나 연결을 해제할 수 있습니다.

**Q: 연결하기 전에 Stripe 계정이 필요한가요?**
네, 설정 과정에서 로그인하거나 계정을 생성해야 합니다.

**Q: 결제 정보가 안전하게 저장되나요?**
네, 모든 결제 데이터는 결제 제공업체를 통해 안전하게 처리됩니다.

**Q: 여러 결제 제공업체를 사용할 수 있나요?**
네, 필요에 따라 추가 제공업체를 연결할 수 있습니다.
