# 상품을 만들고 퍼널로 판매하는 방법

상품, 퍼널, 체크아웃이 제대로 연결되면 온라인 판매가 간단해집니다. 이 가이드에서는 상품을 만들고, 퍼널에서 판매할 수 있도록 설정하고, 원활한 구매를 위한 주문 양식을 설정하는 과정을 안내합니다. 이 단계들을 따라하면 고객이 쉽게 상품을 보고, 선택하고, 주문을 완료할 수 있습니다.

# **상품 + 퍼널 체크아웃 설정이란?**

상품과 퍼널 체크아웃 설정은 판매하는 것과 고객이 구매하는 곳을 연결합니다. 상품에는 가격과 상세 정보가 저장되고, 퍼널은 고객을 체계적인 구매 여정으로 안내합니다. 주문 양식과 결합하면 안전하고 부드러운 거래가 가능합니다.

---

## **이 설정의 주요 장점**

구조화된 판매 흐름을 만들면 전환율과 사용자 경험이 모두 향상됩니다.

- **간소한 판매**: 상품을 체크아웃 페이지에 직접 연결
- **빠른 설정**: 판매 가능한 상품을 빠르게 만들고 게시
- **향상된 전환율**: 명확한 구매 여정으로 고객을 안내
- **유연한 판매**: 실물, 디지털, 또는 서비스 기반 상품 지원
- **원활한 체크아웃**: 부드럽고 안전한 구매 경험 제공

---

## **상품 만들기**

상품을 추가하는 것이 구매 가능한 상품을 만드는 첫 번째 단계입니다.

### **![상품 메뉴 접근](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155069412622/original/p4g3U43UDLt65h2tgGX9B1y0nXOURDUKlw.png?1776460906)**

결제(Payments) 섹션을 클릭하고 **상품(Products)**을 선택하여 상품 목록에 접근합니다.

### **![상품 만들기 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155069412634/original/ZR-NSUOakFrdShaodFNiQ_FvoTdFCii2Vw.png?1776460920)**

**상품 만들기(Create Product)**를 클릭하여 새 상품을 추가합니다.

**해야 할 일:**

- 상품명 입력
- 가격 세부 정보 추가
- 설명과 이미지 포함 (선택 사항)
- **저장** 클릭

---

## **퍼널에서 상품 사용 가능하게 만들기**

상품을 만든 후에는 고객이 접근할 수 있도록 퍼널에 연결해야 합니다.

### **![퍼널 섹션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155069412662/original/7b7gxuH3swNDrgfGt-Fy8IW5gEvLYiJq2A.png?1776461001)**

**사이트(Sites) 섹션**으로 이동하고 **퍼널(Funnels)**을 클릭하여 상품을 판매할 곳을 선택합니다.

### **![상품 탭](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155069412603/original/idbJCFsD5Tzqljf1SBpt-JyCfyPcqv3RXA.png?1776460893)**

퍼널 스텝을 선택하고 **상품(Products) 탭**을 열어서:

- 상품을 선택
- 가격 설정 선택
- **저장** 클릭

---

## **퍼널 스텝 추가하기 (필요한 경우)**

체크아웃 페이지가 없다면 주문 양식을 추가하기 전에 페이지를 만들어야 합니다.

### **![새 스텝 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155069412599/original/occamCoedDbn4kd-LEKAi-2YVwcIuS_8Ng.png?1776460880)**

**새 스텝 추가(Add New Step)**를 클릭하고, 페이지 이름을 입력한 다음 **퍼널 스텝 만들기(Create Funnel Step)**를 클릭하여 확인합니다.

---

## **체크아웃 페이지에 주문 요소 추가하기**

주문 양식을 통해 고객이 퍼널 내에서 직접 구매를 완료할 수 있습니다.

### **![주문 요소 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155069412595/original/0MPOIPr7CAvHpih8I0XAu7Q-oAzOiUw7KQ.png?1776460867)**

페이지 빌더 내에서:

- **요소(Elements)** 클릭
- "order"를 검색
- **1단계 주문(1 Step Order)** 선택 (원한다면 2단계도 가능)

---

## **주문 양식에 상품 연결하기**

고객이 구매할 수 있도록 주문 양식이 올바른 상품에 연결되어 있는지 확인합니다.

### **![상품 연결 확인](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155069412544/original/P-Dr8XInRyqzXTgtbs4rA1CXgZZY8_BxvA.jpeg?1776460767)**

설정을 완료하기 전에 상품이 주문 요약에 표시되고 선택 가능한지 확인합니다.

**해야 할 일:**

- 상품이 목록에 있는지 확인
- 가격 정확성 확인
- 페이지 저장 및 발행

---

## **퍼널에서 상품 판매 설정하는 방법**

모든 것을 올바르게 설정하면 고객이 성공적으로 구매를 완료할 수 있습니다.

- 가격과 상세 정보가 있는 상품 만들기
- 퍼널을 열고 스텝 선택
- 상품(Products) 탭에서 상품 추가
- 필요한 경우 체크아웃 페이지 만들기
- 주문 양식 요소 추가
- 상품을 주문 양식에 연결
- 저장 및 발행

---

## **자주 묻는 질문**

**Q: 퍼널에서 상품이 표시되지 않는 이유는 무엇인가요?**

퍼널 스텝의 상품(Products) 탭에서 상품이 추가되고 저장되었는지 확인하세요.

**Q: 하나의 퍼널에서 여러 상품을 판매할 수 있나요?**

네, 여러 상품을 추가하고 다양한 가격 옵션을 제공할 수 있습니다.

**Q: 1단계와 2단계 주문 양식의 차이점은 무엇인가요?**

1단계는 모든 정보를 한 번에 수집하고, 2단계는 연락처와 결제 정보를 분리합니다.

**Q: 주문 양식을 추가하기 전에 퍼널 스텝이 필요한가요?**

네, 주문 양식은 퍼널 페이지 안에 배치되어야 합니다.

**Q: 상품을 만든 후 편집할 수 있나요?**

네, 상품은 언제든지 업데이트할 수 있으며, 변경 사항은 연결된 퍼널에 반영됩니다.

---

## **더 많은 도움이 필요하신가요?**

상품, 퍼널, 또는 체크아웃 페이지 설정에 도움이 필요하시면 추가 도움말 리소스를 살펴보거나 지원팀에 문의하세요. 올바른 구성을 통해 부드럽고 성공적인 판매 경험을 보장할 수 있습니다.
