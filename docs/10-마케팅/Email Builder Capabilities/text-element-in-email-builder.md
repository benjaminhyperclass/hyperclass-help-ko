# 이메일 빌더의 텍스트 요소

이메일 빌더의 텍스트 요소(Text Element)를 사용하면 제목, 문단 등의 텍스트 콘텐츠를 쉽게 추가하고 스타일링할 수 있습니다. 유연한 서식 옵션을 제공하여 브랜드의 톤과 메시지에 맞는 시각적으로 매력적이고 읽기 쉬운 이메일 디자인을 만들 수 있습니다.

## 하이레벨 이메일 빌더의 텍스트 요소란?

텍스트 요소는 이메일 빌더에서 드래그 앤 드롭으로 사용할 수 있는 블록으로, 이메일 레이아웃의 어느 곳에나 텍스트 콘텐츠를 추가할 수 있습니다. 폰트 사용자 지정, 색상 선택, 정렬, 하이퍼링크, 글머리 기호 등 다양한 서식 기능을 지원합니다.

눈길을 끄는 헤드라인을 만들거나 본문 텍스트를 추가할 때 사용할 수 있습니다. 텍스트 요소는 유연하고 모바일 반응형으로 설계되어 모든 기기에서 메시지가 멋지게 보이도록 합니다.

참고: 이메일을 보내기 전에 항상 미리보기를 확인하여 다양한 이메일 클라이언트에서 텍스트가 예상대로 표시되는지 확인하세요.

## 텍스트 요소의 주요 장점

이메일 디자인에 텍스트 요소를 추가하면 다음과 같은 주요 장점이 있습니다:

- 제목, 문단 등 모든 유형의 텍스트 콘텐츠를 쉽게 추가하고 스타일링할 수 있습니다.
- 브랜드 일관성을 위해 타이포그래피, 색상, 서식을 완전히 제어할 수 있습니다.
- 클릭 가능한 CTA를 위해 텍스트에 직접 링크를 추가할 수 있습니다.
- 시각적 균형을 개선하기 위해 간격과 정렬을 사용자 지정할 수 있습니다.
- 데스크톱과 모바일 기기 모두에 자동으로 최적화됩니다.

## 텍스트 요소 설정 방법

이메일 템플릿에 텍스트 요소를 추가하고 설정하려면 다음 단계를 따르세요:

- 하위 계정 대시보드에서 `Marketing(마케팅) → Emails(이메일) → Templates(템플릿)`으로 이동합니다.

![마케팅 메뉴 이동](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155057287040/original/43B9eCV3aospbAtnUHY0B-yy_ftVb4KuKA.png?1761903242)

- 기존 이메일 템플릿을 열거나 `+ New` 버튼을 클릭하여 새로 만듭니다.

![이메일 템플릿 생성](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155057287266/original/DfagherlOOXP4wKdJ4DIyxPgUYle7fBu9A.jpeg?1761903319)

- 이메일 빌더에서 좌측 사이드바의 `Add Elements(+)` 아이콘을 클릭합니다.

![요소 추가 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155057287484/original/S6qzvG1P3orkHi3yyeB5keO3WZtYJPTJew.png?1761903417)

- 텍스트 요소(Text Element)를 찾아서 원하는 위치로 드래그합니다.

- 텍스트 요소를 클릭하여 편집(Edit) 탭을 엽니다. 여기서 다음과 같은 작업을 할 수 있습니다:

텍스트 영역에 직접 콘텐츠를 입력하거나 붙여넣기합니다.

- 도구 모음을 사용하여 텍스트 서식을 지정합니다(굵게, 기울임, 밑줄, 목록, 정렬 등).
- 텍스트에 제목 스타일(H1–H4)을 적용하고, 인라인 도구 모음의 **굵게(B)** 버튼을 사용하여 제목 수준을 변경하지 않고 굵게 표시를 켜거나 끌 수 있습니다.
- 가독성 향상을 위해 폰트 크기, 색상, 간격을 조정합니다.
- 네비게이션이나 CTA를 위해 문구에 하이퍼링크를 추가합니다.

![텍스트 요소 편집](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155057287678/original/NJz8gMo9eaeasNS6Flqqrk8AYuYojydFIw.png?1761903523)

![텍스트 서식 도구](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155061312064/original/vokoZXiVkQsmyAEp_zYdbtBnDjSvpS0T0w.png?1766583105)

- 패딩(Padding) 옵션을 사용하여 텍스트 주변의 간격을 조절합니다.

![패딩 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155057287753/original/27BSz2FZ8gB1kXOyi1ACqfdvIX2mWbAd2Q.png?1761903571)

- 가시성(Visibility) 탭을 클릭하여 기기별 표시 설정(Device-Based Visibility)이나 조건부 발송(Conditional Sending) 옵션을 조정합니다.

![가시성 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155057287888/original/rIiBwlYAkGxoIUoHPszxEVEAg1EMGZmC7A.png?1761903618)

- 데스크톱과 모바일 모드 모두에서 이메일을 미리보기하여 서식을 확인합니다.

## 자주 묻는 질문

**Q: 텍스트의 일부분에 다른 폰트나 크기를 사용할 수 있나요?**

네, 특정 텍스트 부분을 선택한 후 인라인 텍스트 도구 모음을 사용하여 폰트, 크기, 색상을 개별적으로 조정할 수 있습니다.

**Q: 텍스트에 하이퍼링크를 추가하는 방법은?**

링크를 걸고 싶은 텍스트를 선택하고, 도구 모음의 링크 아이콘을 클릭한 후 원하는 URL을 입력하면 됩니다.

**Q: 텍스트 요소에서 글머리 기호와 번호 목록을 사용할 수 있나요?**

네, 서식 도구 모음에서 직접 글머리 기호 목록과 번호 목록을 만들 수 있습니다.

**Q: 텍스트 요소가 모바일 반응형인가요?**

물론입니다. 텍스트 요소는 작은 화면에 자동으로 맞춰져 모든 기기에서 가독성을 보장합니다.

**Q: 레이아웃에서 텍스트 요소를 제거하는 방법은?**

텍스트 요소를 클릭하고 휴지통/삭제 아이콘을 선택하면 이메일에서 제거됩니다.

**Q: 텍스트 요소에서 제목 텍스트(H1–H4)의 굵게 표시를 제거할 수 있나요?**

네. 텍스트 요소 안의 제목 텍스트(H1–H4)를 선택한 후 인라인 도구 모음의 **B** 버튼을 클릭하여 굵게 표시를 켜거나 끌 수 있습니다.
