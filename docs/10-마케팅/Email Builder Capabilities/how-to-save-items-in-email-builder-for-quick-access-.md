# 이메일 빌더에서 빠른 접근을 위해 항목을 저장하는 방법

전문적이고 대칭적인 이메일을 만드는 것은 어렵고 시간이 많이 소요될 수 있습니다. 이 가이드에서는 디자인 요소와 레이아웃을 재사용을 위해 저장하는 방법을 보여드리며, 이메일의 대칭성과 일관성을 유지하는 데 도움을 드립니다.

이 팁들을 따르면 이메일 작성 과정을 간소화하고 이메일이 항상 세련되고 잘 구조화된 모습을 유지할 수 있습니다. 시작해보세요!

**이 글에서는 다음을 안내합니다:**

1. 요소/섹션 또는 레이아웃 저장하기
2. 더 나은 관리를 위해 이러한 섹션 이름 바꾸기
3. 빠른 접근을 위한 항목 검색

**요소 또는 레이아웃 저장 단계:**

**1단계 |** Marketing(마케팅) → Email Marketing(이메일 마케팅)으로 이동하여 캠페인/템플릿을 엽니다

**2단계 |** 저장하려는 레이아웃 또는 요소를 만듭니다

**3단계 |** 요소 또는 레이아웃 위에 마우스를 올리고 "Save(저장)" 아이콘을 클릭합니다

![이메일 빌더에서 저장 아이콘 위치](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155032312118/original/MqkEQYaMWDxGUAPQ7URePNtSnnWJ0j5Sbg.png?1725536278)

![섹션 저장 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155032312147/original/KxDL2IyARz8wvNfoFyO6WIZRjxkgB37KuA.png?1725536300)

**4단계 |** 우측 상단의 Saved Items(저장된 항목) 섹션에서 요소에 접근합니다

*참고: 이는 전체적으로 저장되며 해당 로케이션의 모든 기존 또는 새 이메일 템플릿/캠페인 빌더에서 빠른 접근을 위해 사용할 수 있습니다*

![저장된 항목 섹션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155032312229/original/rXjjIxJou6lBUlVUvOWaNZuFBImS4ES74w.png?1725536327)

**5단계 |** 이제 사용자는 세 점을 클릭하고 "Rename(이름 바꾸기)"을 선택하여 빠른 검색을 위해 요소의 이름을 바꿀 수도 있습니다

![이름 바꾸기 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155032312333/original/aT3KEuQkg4qha0tk-qhqPjl_NQqlZJMrAg.png?1725536369)

Rename(이름 바꾸기) 버튼을 클릭하여 텍스트를 원하는 이름으로 편집합니다. 이 예제에서는 "Weekly Update Body Content"로 이름을 바꿨습니다.

![이름 편집 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155032313495/original/IDwwWt6HAJLcUxds3dDyOV7zdiNwGOz8Zg.png?1725536770)

**6단계 |** 검색 바를 사용하여 필요한 항목에 빠르게 접근합니다.

![검색 기능 사용](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155032314110/original/dISTEEBiriDHRbbJio_NqZFTkKIo_QO6DQ.png?1725536987)
