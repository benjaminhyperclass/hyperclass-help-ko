# 템플릿 라이브러리를 활용한 브랜드 리뷰 요청 이메일 만들기

이메일 리뷰 요청(Email Review Requests)의 템플릿 라이브러리 연동 기능으로 개인화된 리뷰 요청 이메일을 손쉽게 제작하는 방법을 알아보세요. 완전히 커스터마이징 가능한 빈 캔버스와 시간 절약형 미리 만들어진 템플릿 중 선택하여 Hyperclass의 평판 관리(Reputation Management) 프로세스를 간소화하는 방법을 설명합니다.

---

## 템플릿 라이브러리 연동이란?

이메일 리뷰 요청의 템플릿 라이브러리 연동은 두 가지 접근 방식을 제공하여 이메일 제작 프로세스를 단순화하는 기능입니다. 완전한 창작의 자유를 누리며 처음부터 이메일을 디자인하거나, 미리 디자인된 레이아웃을 제공하는 엄선된 템플릿 라이브러리에서 선택할 수 있습니다.

이러한 이중 모드 기능을 통해 완전히 커스텀한 비전을 표현하고 싶거나 더 빠른 설정을 선호하는 경우 모두에 대응하여, 브랜드만의 고유한 목소리를 유지하면서 효율적으로 리뷰를 요청할 수 있는 이상적인 옵션을 제공합니다.

---

## 이메일 리뷰 요청에서 템플릿 라이브러리 연동의 주요 이점

이 기능은 생산성과 일관성을 높이도록 설계되어, 전문적인 리뷰 요청 이메일을 쉽게 배포할 수 있게 해줍니다. 주요 이점은 다음과 같습니다:

- **처음부터 만들기**: 빈 이메일 캔버스로 모든 디자인 요소에 대한 완전한 제어권을 누리세요.

- **템플릿 라이브러리**: 세련되고 바로 사용 가능한 템플릿 컬렉션에 즉시 액세스하여 브랜드 아이덴티티에 맞게 빠르게 맞춤화할 수 있습니다.

- **더 빠른 캠페인**: 미리 구축된 레이아웃을 활용하여 이메일 디자인에 소요되는 시간을 줄이므로, 고객과의 소통과 온라인 평판 개선에 집중할 수 있습니다.

---

## 처음부터 만들기

이 옵션은 완전히 빈 캔버스를 제공하여 리뷰 요청 이메일의 모든 측면을 정밀하게 디자인할 수 있게 해줍니다. 완전한 창작 제어권을 통해 브랜드의 개성을 진정으로 반영하는 커스텀 텍스트, 이미지, 스타일링 요소를 추가할 수 있습니다.

- 브랜드 가이드라인과의 일관성을 보장하기 위해 레이아웃, 색상, 폰트를 커스터마이징하세요.
- 실험하고 처음부터 고유한 캠페인을 구축할 수 있는 유연성을 누리세요.

## 템플릿 라이브러리

템플릿 라이브러리 옵션은 캠페인을 위한 빠른 시작 대안으로 전문적으로 디자인된 이메일 템플릿 컬렉션을 제공합니다. 각 템플릿은 쉽게 커스터마이징할 수 있도록 제작되어, 미리 구축된 디자인도 브랜드의 모양과 느낌을 반영하도록 세밀하게 조정할 수 있습니다.

- 리뷰 요청 이메일에 맞춤화된 다양한 레이아웃에 액세스하세요.
- 미리 디자인된 템플릿을 여러분의 콘텐츠와 브랜딩 요소로 손쉽게 조정하세요.
- 일관된 브랜드 프레젠테이션을 유지하면서 캠페인 제작 시간을 상당히 절약하세요.

## 이메일 리뷰 요청에서 템플릿 라이브러리 연동 설정 방법

적절한 설정을 통해 비즈니스 요구사항을 충족하기 위해 이 기능을 효율적으로 활용할 수 있습니다. 시작하기 위한 간단한 단계들을 따라해보세요:

### 이메일 요청 접근하기

메인 메뉴에서 **평판 관리(Reputation)** 섹션으로 이동하세요. 평판 관리 드롭다운 옵션에서 **설정(Settings)**을 선택하세요. **이메일 요청(Email Requests)**을 클릭하세요.

![평판 관리 설정 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054822493/original/R1LPi-bjtVrfyRyA3id_dfKCoVq1xzScdA.png?1759150030)

### 이메일 요청 활성화

여기서 토글 스위치를 볼 수 있습니다. 이메일 요청을 활성화하려면 **ON**으로 켜세요. 또한 다음을 결정할 수 있습니다:

- 첫 번째 이메일이 언제 발송될지 (예: 체크인 후 1시간)
- 연락처가 클릭할 때까지 얼마나 자주 재전송할지
- 최대 재시도 횟수

이를 통해 고객에게 부담을 주지 않으면서 일관된 리뷰 요청을 보장할 수 있습니다.

![이메일 요청 설정 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054825605/original/XecLmNL3FYhlv25UJidNe4Ym7-89RE6enQ.png?1759151200)

### 새 이메일 시작하기

**새로 만들기(Create New)** 버튼을 클릭하세요. 두 가지 옵션이 제공됩니다:

- 새 템플릿 만들기(Create New Template) – 완전히 빈 디자인으로 시작
- 템플릿 라이브러리에서 가져오기(Import from Template Library) – 미리 구축된 레이아웃을 탐색하고 스타일에 맞는 것을 선택

![새 이메일 만들기 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054826432/original/svodne8ma1FErhR-yontNkvakinQr3iR9w.png?1759151468)

### 처음부터 템플릿 구축하기

빈 캔버스(새 템플릿 만들기 옵션)를 선택하면 드래그 앤 드롭 빌더를 사용하게 됩니다.

![템플릿 빌더 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054826758/original/UyNwpP58oPp9YrCcDwchpYldpkg5DO4kPQ.jpeg?1759151606)

### 이메일 빌더

여기서 다음을 추가할 수 있습니다:

- 메시지 구조를 위한 텍스트, 이미지, 버튼
- 연락처가 쉽게 피드백을 남길 수 있는 **리뷰 링크(Review Link)**
- 이메일을 세련되게 만드는 로고, 소셜 아이콘, 구분선

디자인에 만족하면 **저장(Save)**을 클릭하세요.

![이메일 빌더 편집 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054827467/original/7TwmeGsMsxWFbkNsIWN6IQG2-0gG037wgg.png?1759151812)

### 템플릿 라이브러리 사용하기

더 빠른 방법을 선호한다면 **템플릿 라이브러리(Template Library)**를 선택하세요. 비즈니스 코칭, 금융, 크리에이티브 등의 카테고리별로 정리된 전문적으로 디자인된 템플릿 컬렉션을 보게 됩니다.

마음에 드는 디자인을 선택하고 텍스트와 브랜딩을 교체하면 바로 사용할 수 있습니다.

![템플릿 라이브러리 선택 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054826515/original/ZS9wMI1CUSPX_IZ8Bkn8ZLT6w1AN18B1sg.jpeg?1759151529)

### 템플릿 관리하기

템플릿이 저장되면 이메일 요청 대시보드에 목록으로 표시됩니다. 각 템플릿 옆의 **세 점(⋮)** 을 클릭하여 관리 옵션을 여세요:

- **미리보기 및 세부사항 편집(Preview & Edit Details)** – 발송 전 이메일이 어떻게 보이는지 확인하고 제목이나 발신자 이름 등 주요 세부사항을 조정하세요.

- **템플릿 편집(Edit Template)**: 전체 드래그 앤 드롭 편집기를 열어 디자인을 변경하세요.

- **복제(Clone)**: 처음부터 시작하지 않고 변형을 만들고 싶다면 기존 템플릿을 복제하세요.

- **복원(Restore)**: 마음이 바뀌면 이전에 삭제한 템플릿을 되돌리세요.

- **삭제(Delete)**: 더 이상 필요하지 않은 템플릿을 영구적으로 제거하세요.

이를 통해 라이브러리를 체계적으로 정리하고 세밀하게 조정할 수 있습니다.

![템플릿 관리 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054823856/original/aemVab2LroeAv506fHfw7UJ3FYe1HZYLgA.png?1759150462)

## 자주 묻는 질문

**Q: 이 기능에서 템플릿 라이브러리는 무엇을 제공하나요?**

브랜드에 맞게 추가로 커스터마이징할 수 있는 전문적으로 디자인된 템플릿 선택에 즉시 액세스할 수 있습니다.

**Q: 이 기능은 어디에서 사용할 수 있나요?**

이 도구는 Hyperclass 계정의 평판 관리(Reputation) → 설정(Settings) → 이메일 요청(Email Requests) → 이메일 만들기(Create Email)에서 사용할 수 있습니다.

**Q: 템플릿을 선택해도 여전히 커스텀 이메일을 디자인할 수 있나요?**

네, 템플릿을 선택한 후에도 텍스트, 이미지, 색상, 레이아웃 요소를 수정하여 브랜드에 완벽히 맞출 수 있는 능력을 유지합니다.

**Q: "처음부터 만들기" 모드가 템플릿 라이브러리와 함께 사용 가능한가요?**

물론입니다. 완전한 창작 제어를 위한 빈 캔버스와 빠른 설정을 위한 미리 구축된 템플릿 중에서 선택할 수 있습니다.

**Q: 이 기능을 사용하는데 기술적 전문 지식이 필요한가요?**

기술적 스킬은 필요하지 않습니다. 인터페이스는 Hyperclass의 모든 스킬 레벨에서 사용하기 쉽도록 설계되었습니다.

**Q: 이 연동을 통해 누가 가장 많은 이익을 얻을 수 있나요?**

에이전시와 고객 모두 캠페인 제작 프로세스를 간소화하여 리뷰 요청 이메일의 일관성을 보장하면서 시간을 절약할 수 있습니다.
