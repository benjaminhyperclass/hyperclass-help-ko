# 이메일 빌더의 소셜 요소

소셜 요소(Social Element)를 사용하면 이메일 템플릿에 클릭 가능한 링크가 포함된 소셜 미디어 아이콘을 쉽게 추가할 수 있습니다. 고객이 다양한 플랫폼에서 브랜드와 연결되도록 도와주어 참여도를 높이고 채널 간 가시성을 향상시키면서도 이메일 디자인을 시각적으로 균형 있고 전문적으로 유지합니다.

## **이메일 빌더의 소셜 요소란 무엇인가요?**

소셜 요소는 Facebook, Instagram, LinkedIn, X (Twitter), YouTube 등의 인식하기 쉬운 소셜 미디어 아이콘을 포함할 수 있는 드래그 앤 드롭 블록입니다. 각 아이콘은 해당 프로필에 연결할 수 있어 구독자가 소셜 플랫폼에서 브랜드를 팔로우하고, 좋아요를 누르거나 상호작용하도록 유도합니다.

아이콘 스타일, 색상, 크기, 정렬을 사용자 정의하여 소셜 섹션이 이메일의 전체 디자인과 브랜드 아이덴티티에 맞도록 할 수 있습니다.

참고: 소셜 요소는 이메일에 최적화된 아이콘을 사용하여 모든 주요 이메일 클라이언트에서 명확하고 일관되게 표시됩니다.

## **소셜 요소의 주요 장점**

이메일 디자인에 소셜 요소를 포함하면 여러 가지 주요 이점이 있습니다:

- 소셜 플랫폼에 직접 연결하여 고객 참여도를 높입니다.

- 채널 간 일관된 존재감을 통해 브랜드 신뢰성을 강화합니다.

- 완전 사용자 정의 가능 - 아이콘 색상, 크기, 정렬을 이메일 스타일에 맞게 조정할 수 있습니다.

- 소셜 아이콘을 빠르고 쉽게 업데이트하거나 재정렬할 수 있습니다.

- 데스크톱과 모바일 보기 모두에 자동으로 최적화됩니다.

## **소셜 요소 설정 방법**

이메일 템플릿에 소셜 요소를 추가하고 구성하려면 다음 단계를 따르세요:

- 하위 계정 대시보드에서 Marketing(마케팅) → Emails(이메일) → Templates(템플릿)으로 이동합니다.

![마케팅 메뉴에서 이메일 템플릿으로 이동](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155057302127/original/_gQ6TIvSvc62MIdJIIUsFaaGJQuVt9gblg.png?1761910310)

- 기존 템플릿을 열거나 + New 버튼을 클릭하여 새 템플릿을 생성합니다.

![새 이메일 템플릿 생성](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155057302164/original/xQT7mCb31B_gjmVSm_j1MYVyHj39fEjjZw.jpeg?1761910321)

- 이메일 빌더에서 왼쪽 사이드바의 Add Elements(+) 아이콘을 클릭합니다.

![이메일 빌더에서 요소 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155057302380/original/x1Qm1zfQrOYv1hI1sW0IcyUu5BfbTbaHDQ.png?1761910490)

- Elements 섹션에서 Social Element를 찾습니다.

- 소셜 요소를 원하는 위치(일반적으로 이메일 레이아웃의 헤더나 푸터)로 드래그 앤 드롭합니다.

- 소셜 요소를 클릭하여 왼쪽 패널의 Edit 탭을 엽니다. 여기서 다음을 할 수 있습니다:

  아이콘 추가 또는 제거: 표시할 소셜 플랫폼을 선택합니다.

- 링크 삽입: 각 소셜 미디어 프로필의 URL을 붙여넣습니다.

- 모양 사용자 정의: 아이콘 크기, 모양, 간격, 색상(풀컬러 또는 단색)을 조정합니다.

- 아이콘 정렬: 왼쪽, 가운데 또는 오른쪽 정렬을 선택합니다.

- 패딩 추가: 레이아웃 균형을 위해 소셜 아이콘 위아래 간격을 조절합니다.

![소셜 요소 편집 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155057302614/original/tUGsDM2xSYy-aE1MHb3VfceSp5I5v_fcmw.png?1761910576)

- (선택사항) Visibility 탭을 사용하여 기기별 표시 설정이나 조건부 전송 규칙을 조정합니다.

- 데스크톱과 모바일 모두에서 이메일을 미리 보기하여 아이콘이 명확하게 표시되고 제대로 연결되는지 확인합니다.

## 자주 묻는 질문

**Q: 어떤 소셜 미디어 플랫폼이 지원되나요?**

Facebook, Instagram, LinkedIn, X (Twitter), YouTube, TikTok, Pinterest 등 주요 플랫폼의 아이콘을 추가할 수 있습니다.

**Q: 목록에 없는 새로운 소셜 아이콘을 추가하려면 어떻게 하나요?**

기존 아이콘을 복제하고 해당 이미지와 URL을 커스텀 플랫폼의 로고와 링크로 교체할 수 있습니다.

**Q: 소셜 아이콘이 모바일 반응형인가요?**

네, 작은 화면에서 자동으로 크기와 간격이 조정되어 깔끔하고 사용자 친화적인 레이아웃을 보장합니다.
