# 이메일 빌더에서 커스텀 필드를 미리보기하는 방법

[https://youtu.be/a20mMxuOgtE](https://youtu.be/a20mMxuOgtE)

[https://youtu.be/6vAR0K1BBH4](https://youtu.be/6vAR0K1BBH4)

[https://youtu.be/jPoE5YZsfdA](https://youtu.be/jPoE5YZsfdA)

이 새로운 기능을 통해 사용자는 이메일을 발송하기 전에 커스텀 필드가 각 고객에게 어떻게 표시될지 정확히 미리볼 수 있습니다. 이 기능은 테스트 과정을 단순화하고 가짜 테스트 케이스를 만들 필요를 없애서, 이메일 캠페인 관리를 더욱 효율적이고 비용 효과적으로 만들어줍니다.

커스텀 필드 미리보기 단계:

커스텀 필드 미리보기는 머지 필드(예: {{first_name}})가 발송 전에 올바르게 적용되는지 확인해줍니다. 이러한 값을 검증하면 깨진 개인화를 방지하고, 재작업을 줄이며, 전달률 성과를 개선할 수 있습니다.

- 이메일 빌더에서 이메일을 엽니다.

![이메일 빌더 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063988124/original/wr6eqNIz69_t5VjFPzQOtSxZQJ3fBskBDQ.jpeg?1770036745)

- 편집기 상단의 메뉴 바를 찾습니다.

![상단 메뉴 바](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063988165/original/cdzw6lCEJFLOIsEdbokYBXvEaQq0pq_Ulw.jpeg?1770036768)

- 저장 버튼 옆에 위치한 Preview(미리보기) 버튼을 클릭하여 즉시 미리보기를 엽니다.

![미리보기 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063988245/original/fLpBbJKIfhVLDMm-w0akx9J-QhXQeCROUQ.jpeg?1770036810)

- 커스텀 필드를 검토하고 예상한 값이 표시되는지 확인합니다.

![커스텀 필드 검토](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063988325/original/WnsxwN9u6FErCjUmIXn-FdJDrRJFGSE1oQ.jpeg?1770036836)

- 주소 머지 필드는 미리보기에서 완전히 렌더링됩니다. 이메일에서 전체 주소를 사용하는 경우 `{{contact.full_address}}`로 테스트하여 미리보기가 연락처 레코드와 일치하는지 확인하세요.

- 문제가 있다면 미리보기를 닫고 콘텐츠나 테스트 데이터를 수정한 후, 다시 미리보기를 클릭합니다.

![미리보기 수정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063988368/original/wflbIXQFrMNJK-xcW6z41Vwbr2epD9gX6w.jpeg?1770036862)

- 편집을 계속하거나, 만족스럽다면 발송 또는 예약을 진행합니다.

**팁:** 미리보기는 이제 상단 메뉴 바에서 원클릭으로 접근할 수 있습니다. 더 이상 점 3개(케밥) 메뉴를 사용할 필요가 없습니다.
