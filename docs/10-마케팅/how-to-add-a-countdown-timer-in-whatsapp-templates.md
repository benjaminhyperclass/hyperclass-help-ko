# **왓츠앱 템플릿의 카운트다운 타이머란?**

왓츠앱 메시지의 카운트다운 타이머를 사용하면 이메일, 퍼널, 웹사이트에서 이미 사용 중인 Hyperclass 타이머를 왓츠앱 템플릿 버튼 안에 배치할 수 있습니다. 수신자가 버튼을 탭하면 타이머의 실시간 상태에 따라 "활성" 또는 "만료" 링크로 이동하게 되어, 모든 메시지에 내장된 마감일이 클릭률과 전환율을 높입니다.

이 기능은 고정 날짜 타이머, 반복 타이머, 그리고 왓츠앱이 전송되거나 전달되는 순간 각 연락처에게 개별적으로 시작되는 완전히 개인화된 동적 타이머를 지원합니다.

---

## **왓츠앱 카운트다운 타이머의 주요 장점**

- **긴급성 & 전환율**: 실시간 카운트다운이 즉시 행동과 구매를 유도합니다.

- **다양한 타이머 유형**: **고정**, **반복**, 또는 **동적** 타이머를 선택하여 프로모션 모델에 맞춰 사용할 수 있습니다.

- **활성/만료 라우팅**: 만료 전후에 자동으로 방문자를 올바른 페이지로 전송합니다.

- **워크플로우 분기**: 타이머 버튼 클릭을 기반으로 자동화를 분기할 수 있습니다.

---

## **지원되는 카운트다운 타이머 유형**

적절한 타이머 유형을 선택하면 시간대와 전송 시간에 관계없이 메시지, 페이지, 자동화가 동기화된 상태를 유지할 수 있습니다.

- **고정**: 특정 날짜/시간에 종료(블랙 프라이데이 같은 이벤트용). 하드 데드라인이 있는 일회성 캠페인에 최적입니다.

- **반복**: 일정한 주기로 반복(일간/주간). 상시 운영되는 오퍼나 "주간 특가" 제안에 이상적입니다.

- **동적**: 트리거에 따라 시작되는 연락처별 개인 지속 시간(예: 메시지 **전송** 또는 **전달**). "이 메시지를 받은 후 12시간 이내에 사용하세요"와 같은 개인화된 한정 시간 오퍼에 적합합니다.

---

## **왓츠앱 템플릿에 타이머 추가하는 방법**

다음 단계에 따라 템플릿에 타이머를 연결하고, 동적 동작을 구성하며, 만료 전후의 적절한 라우팅을 보장하세요.

#### **1단계: 왓츠앱 템플릿으로 이동**

- Settings(설정) → WhatsApp으로 이동합니다

- 보조 내비게이션 리본에서 Templates 탭을 선택합니다

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155058115832/original/jIwBOvYmAxDtI2W5U-kvaMal1OwA1mAWow.png?1762876545)

#### **2단계: 템플릿 열기**

- 파란색 + Create Template 버튼을 클릭하여 새 템플릿을 만들거나 기존 템플릿을 선택하여 편집합니다

- 이름, 카테고리, 언어, 헤더, 본문, 푸터를 포함한 **템플릿 세부 정보**를 입력합니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155058115892/original/kmcJWIs2IlcWjDi5gRumBIEBIb_cgOKzwQ.png?1762876582)

#### **3단계: 카운트다운 타이머 버튼 추가**

Button 섹션에서 **+** Add Button 버튼을 클릭한 다음 드롭다운에서 Countdown Timer를 선택합니다

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155058116027/original/-RdeqY7zjcSgSr5s6fEOhahaSzdxmKrjsw.png?1762876685)

#### **4단계: 타이머 선택 또는 생성**

**Timer** 드롭다운을 열고 기존 타이머를 선택하거나 **+ Create New**를 클릭합니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155058116198/original/3pAS4ZDGZ057u-dcQGSBoib5eiRy_bY71A.png?1762876772)

#### **5단계: 타이머 구성**

타이머를 선택한 후 Timer URL 하단의 Edit Timer를 클릭합니다. 타이머 설정이 열리면 다음을 조정할 수 있습니다:

- **타이머 유형**: 고정, 반복, 또는 동적

- 카운트다운 지속 시간: 지속 시간 또는 특정 종료 시간을 설정

- 타이머 시작 시점: 동적 타이머의 경우 다음과 같은 트리거를 선택:
  - 왓츠앱 메시지가 전송될 때
  - 왓츠앱 메시지가 전달될 때

- 활성 링크: 만료 **전에** 사용되는 실시간 오퍼/결제/예약 페이지

- 만료 링크: 만료 **후에** 표시되는 대체 페이지(예: 대기자 명단 또는 대안 오퍼)

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155058116445/original/zo2vZ14tOHPQFd9BqIsVukBE5CsiWygyXA.png?1762876922)

#### **6단계: 저장 및 미리보기**

- 메시지 미리보기를 검토하여 템플릿의 미리보기를 확인합니다

- 파란색 Create 버튼을 클릭하여 왓츠앱 템플릿을 완성합니다

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155058116816/original/QbkiJ9cwRh1XV5Ued9gfvpHRIC3Td3A_Eg.png?1762877091)

## **자주 묻는 질문**

**Q: 하나의 왓츠앱 메시지에 여러 개의 타이머를 사용할 수 있나요?**
카운트다운 타이머 버튼을 최대 2개까지 추가할 수 있으며, "Visit Website" 또는 "Trackable Website"와 조합하여 사용할 수 있습니다.

**Q: 타이머가 만료되면 어떻게 되나요?**
만료되면 사용자는 만료된 오퍼 페이지로 리디렉션됩니다(타이머 설정에 따라).

**Q: 타이머를 만든 후 편집할 수 있나요?**
네 — Edit Timer를 클릭하여 언제든지 지속 시간, 스타일 또는 트리거를 변경할 수 있습니다.

**Q: 동적 타이머는 사용자마다 다른 카운트다운을 보여주나요?**
네 — 각 연락처는 트리거 이벤트(예: 왓츠앱 메시지가 전송된 시점)에 따라 고유한 카운트다운을 받게 됩니다.

**Q: Timer URL 아래에 "No link found"가 표시되는 이유는 무엇인가요?**
타이머에 아직 리디렉션 링크가 없다는 의미입니다. Edit Timer를 클릭하고 동적 링크 또는 URL을 추가하세요.

**Q: 활성 링크와 만료 링크란 무엇인가요?**
활성 링크는 오퍼 페이지(할인, 제안, 무료 등록 등이 있는 페이지)의 링크이고, 만료 페이지는 오퍼가 만료되었을 때 사용자가 이동할 페이지입니다. 오퍼 페이지에도 동일한 타이머를 설정하여 타이머와 함께 자동으로 만료되고 만료 페이지로 리디렉션되도록 해야 합니다.

**Q: 카운트다운 타이머 버튼 클릭을 기반으로 워크플로우를 분기할 수 있나요?**
네. Timer URL로 트리거 링크를 사용하면 클릭이 추적되고 워크플로우의 분기를 실행할 수 있습니다.
