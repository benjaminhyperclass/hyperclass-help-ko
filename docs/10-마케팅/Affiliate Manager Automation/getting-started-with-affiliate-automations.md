# 제휴 자동화 시작하기

제휴 관리(Affiliate Manager) 자동화를 활용하면 워크플로우를 통해 작업을 자동화하고 제휴 관리를 효율적으로 처리할 수 있습니다. 쉽게 설정할 수 있는 4가지 필수 자동화 기능을 소개합니다:

**목차**

- [1. 제휴사 생성 워크플로우 트리거](#1-제휴사-생성-워크플로우-트리거)
- [2. 제휴 관리에 추가](#2-제휴-관리에-추가)
- [3. 제휴사 업데이트](#3-제휴사-업데이트)
- [4. 제휴 캠페인에 추가](#4-제휴-캠페인에-추가)
- [5. 제휴 캠페인에서 제거](#5-제휴-캠페인에서-제거)

---

## 1. **제휴사 생성 워크플로우 트리거**

### **트리거 설정**:

- **Add New Workflow Trigger(새 워크플로우 트리거 추가)** > **Affiliate Created(제휴사 생성)** 검색 > 선택하세요.
- **Save Trigger(트리거 저장)**를 클릭하세요.

### **작동 방식**:

- 하위 계정에 새로운 제휴사가 생성될 때마다 이 워크플로우가 트리거됩니다.

### **예시**:

- 마케팅 캠페인을 진행하여 여러 제휴사가 가입할 때, 각 제휴사가 추가되는 순간 이 트리거가 자동으로 워크플로우에 포함시켜 수동 작업 없이 처리할 수 있습니다.

---

## 2. **제휴 관리에 추가**

### **자동화 설정**:

- 연락처를 제휴 관리의 제휴사 목록에 자동으로 추가하고 상태를 "활성"으로 설정합니다.

### **작동 방식**:

- 워크플로우가 트리거되는 즉시 연락처가 시스템에서 활성 제휴사로 추가됩니다.

### **예시**:

- 잠재적인 파트너가 가입하면 즉시 활성 제휴사 목록에 추가되어 온보딩 과정에서 지연이 발생하지 않습니다.

## **3. 제휴사 업데이트**

### **자동화 설정**:

- 제휴사 상태를 "활성" 또는 "비활성"으로 업데이트할 수 있습니다.
- **지원 옵션**:
  - 활성으로 표시
  - 비활성으로 표시

### **예시**:

- 제휴사가 설정된 기간 동안 활동 요구사항을 충족하지 못하면, 이 자동화가 해당 제휴사의 상태를 자동으로 비활성으로 업데이트할 수 있습니다.

![제휴사 업데이트 자동화](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033844790/original/cYmDOShWFpLExXBmjLj0hwNCzbEPL3bKNw.jpeg?1727763735)

## 4. **제휴 캠페인에 추가**

### **자동화 설정**:

- 제휴 관리의 캠페인(Campaigns) 탭에서 특정 캠페인에 제휴사를 자동으로 추가합니다. 드롭다운 메뉴에서 캠페인을 선택하세요.

### **예시**:

- 새로운 제품을 출시하고 모든 활성 제휴사를 이 캠페인에 수동으로 추가하지 않고 자동으로 포함시키고 싶을 때, 이 자동화가 즉시 처리해줍니다.

![제휴 캠페인에 추가 자동화](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033844789/original/_QzifFSd0AmdXr5SY0oUdrja-FQRSRXe7Q.jpeg?1727763735)

---

## **5. 제휴 캠페인에서 제거**

### **자동화 설정**:

- 캠페인에서 제휴사를 제거하는 옵션입니다. 드롭다운 메뉴를 사용하여 제거할 캠페인을 선택하세요.

### **예시**:

- 제휴사가 더 이상 특정 캠페인 자격을 충족하지 않을 때, 시스템이 자동으로 해당 제휴사를 제거하여 관리자의 업무 부담을 줄입니다.

![제휴 캠페인에서 제거 자동화](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033844787/original/rcKHKfi6A2GBbJb_IyK54EMUaZht3my6iA.jpeg?1727763735)
