# 워크플로우 트리거 - 제휴 캠페인 등록

이 글에서는 하이레벨의 **"제휴 캠페인 등록(Affiliate Enrolled In Campaign)"** 워크플로우 트리거를 사용하여 제휴 파트너의 온보딩과 커뮤니케이션을 자동화하는 방법을 설명합니다. 제휴 파트너가 캠페인에 참여하는 순간 주요 데이터를 수집하여 개인화된 메시지를 보내고, 연락처에 태그를 추가하며, 팀에 알림을 보낼 수 있습니다—모든 과정이 수동 작업 없이 자동으로 진행됩니다. 이 트리거를 통해 제휴 파트너 참여를 간소화하고 성과를 향상시키며 원활한 온보딩 경험을 제공할 수 있습니다.

# 제휴 캠페인 등록 워크플로우 트리거란?

**제휴 캠페인 등록** 워크플로우 트리거는 하이레벨 내에서 제휴 파트너가 특정 캠페인에 추가될 때 자동으로 액션을 실행하는 기능입니다. 이는 수동 또는 자동으로 발생할 수 있으며, 제휴 파트너 온보딩과 내부 팀 커뮤니케이션을 간소화하는 데 특히 유용합니다. 활성화되면 제휴 파트너의 이름, 이메일, 전화번호, 캠페인명, 총 수익, 커미션 수익, 추천 링크, 포털 링크, 매직 로그인 링크 등 상세한 제휴 데이터에 접근할 수 있습니다. 이러한 커스텀 값들은 이메일, SMS 메시지, 내부 알림, CRM 업데이트 등에 사용할 수 있어 개인화된 소통, 제휴 파트너 성과 추적, 전반적인 온보딩 경험 개선을 쉽게 만들어줍니다. 이 트리거는 제휴 관리(Affiliate Manager)가 활성화되어 있고 하위 계정에 적어도 하나의 제휴 캠페인이 존재할 때만 나타납니다.

---

## 제휴 캠페인 등록 워크플로우 트리거의 주요 장점

이 트리거는 제휴 파트너 온보딩, 내부 추적, 캠페인 참여를 향상시키는 자동화 기회를 제공합니다. 다음은 이 워크플로우 트리거를 효과적으로 사용하는 5가지 핵심 장점입니다.

- **원활한 제휴 파트너 온보딩:** 로그인 링크, 캠페인 세부사항, 기대사항이 포함된 개인화된 이메일이나 SMS로 새로운 제휴 파트너를 자동으로 환영합니다—수동 설정과 혼란을 제거합니다.

- **동적 제휴 데이터 접근:** 제휴 파트너 이름, 연락처 정보, 수익, 포털 링크 등의 커스텀 값에 즉시 접근할 수 있습니다. 이러한 동적 필드는 자동화된 메시지에 직접 삽입할 수 있습니다.

- **내부 커뮤니케이션 개선:** 새로운 제휴 파트너가 캠페인에 참여할 때마다 팀에 내부 알림을 사용하여 알려줍니다. 이를 통해 직원들이 지체 없이 후속 조치나 새로운 파트너 지원을 할 수 있습니다.

- **캠페인 레벨 세분화:** 제휴 파트너가 등록한 캠페인을 기준으로 태그를 적용합니다. 이를 통해 더 타겟화된 메시징, 더 나은 리포팅, 캠페인별 맞춤 후속 시퀀스가 가능합니다.

- **확장 가능한 제휴 프로그램 관리:** 이러한 온보딩과 알림 단계를 자동화하면 품질이나 개인화를 희생하지 않으면서도 제휴 파트너 증가를 대규모로 관리하기 쉬워집니다.

## 제휴 캠페인 등록 트리거 설정 방법

### 워크플로우 빌더 열기

하이레벨 대시보드의 왼쪽 메뉴에서 **Automation(자동화) > Workflows(워크플로우)**로 이동합니다. 워크플로우 목록에서 오른쪽 상단의 파란색 **+ Create Workflow(+ 워크플로우 만들기)** 버튼을 클릭하고 드롭다운 메뉴에서 **Start from Scratch(처음부터 시작)**를 선택하여 새로운 커스텀 워크플로우를 만들기 시작합니다.

![워크플로우 빌더 열기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050122714/original/rQylRHJ1b_2k3EjHY9H4MZyLYWjYDbN83g.png?1753031219)

### 제휴 캠페인 트리거 추가

워크플로우 빌더 내에서 **+ Add New Trigger(+ 새 트리거 추가)**를 클릭합니다. 트리거 목록에서 **Affiliate(제휴)** 섹션으로 스크롤하여 **Affiliate Enrolled In Campaign(제휴 캠페인 등록)**을 선택합니다. 이 트리거는 제휴 파트너가 특정 캠페인에 참여할 때 워크플로우를 활성화합니다.

![제휴 캠페인 트리거 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050122729/original/Bcrr0WKwL0f-ntuscOCGNv9FyJSqaSfsiQ.png?1753031272)

### 트리거 이름 지정

"제휴 캠페인 등록"과 같이 명확하고 관련성 있는 이름을 트리거에 부여합니다. 이름 지정은 워크플로우를 체계적으로 유지하고 나중에 목적을 쉽게 식별할 수 있게 도와줍니다.

![트리거 이름 지정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050122735/original/cnIk8HirSZv7z9ENKkDfzoBKc6iriofvjw.png?1753031319)

### 필터 추가

**"Add filters(필터 추가)"**를 클릭하여 이 트리거가 언제 활성화되어야 하는지 범위를 좁힙니다. 필터를 사용하면 캠페인명이나 커스텀 필드와 같은 특정 조건을 적용하여 관련성 있는 제휴 등록만 워크플로우를 시작하도록 할 수 있습니다.

![필터 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050122742/original/NxHd82ysrhOxv8ZGp054G_PcDHcKxMDQng.png?1753031365)

### 필터 유형 선택

드롭다운을 사용하여 필터 유형을 선택합니다. **Standard Fields(표준 필드)** 중에서 캠페인이나 커스텀 속성 같은 것을 선택하여 트리거가 활성화되어야 하는 시점을 정의할 수 있습니다. 이를 통해 워크플로우가 관심 있는 특정 제휴 등록에만 반응하도록 보장합니다.

![필터 유형 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050122748/original/AHNmD0VFigF0s7K-9sHG3WaxygbGgKdVQQ.png?1753031407)

### 트리거 설정 저장

캠페인을 선택하고 필터 조건을 정의한 후 **Save Trigger(트리거 저장)**를 클릭하여 구성을 활성화합니다. 이로써 설정이 완료되고 선택한 특정 캠페인에 제휴 파트너가 등록될 때만 워크플로우가 트리거되도록 보장됩니다.

![트리거 설정 저장](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155050122756/original/LDqqXEWe1PoF_jINqEmyxIOYJThICi6SDg.png?1753031463)

## 제휴 캠페인 등록 트리거에서 커스텀 값 사용 방법

커스텀 값을 사용하면 제휴 파트너별 데이터를 활용하여 커뮤니케이션을 개인화하고 내부 프로세스를 간소화할 수 있습니다. 워크플로우 트리거가 구성되면 주요 데이터 필드를 자동으로 가져올 수 있어 제휴 파트너와 팀 모두에게 온보딩 프로세스를 더 동적이고 정보가 풍부하게 만들 수 있습니다.

**제휴 캠페인 등록** 트리거를 설정하고 필요한 필터(예: 캠페인명)를 적용한 후, 이메일, SMS, 내부 알림 등의 워크플로우 단계에서 **커스텀 값**을 사용할 수 있습니다. 이러한 값에는 제휴 파트너의 이름, 연락처 정보, 수익, 커미션, 로그인 링크 등의 세부사항이 포함되어 맥락이 풍부하고 개인화된 콘텐츠를 제공할 수 있습니다.

- **내부 알림 단계 추가:** 제휴 파트너가 특정 캠페인에 참여할 때 팀에 알리는 내부 알림을 만듭니다. 제휴 파트너 이름, 제휴 파트너 이메일, 제휴 파트너 전화번호, 캠페인명과 같은 커스텀 값을 사용하여 등록 이벤트에 대한 전체 컨텍스트를 제공합니다.

- **제휴 파트너에게 개인화된 SMS 발송:** 제휴 파트너의 이름과 캠페인 세부사항을 사용하여 환영 SMS를 보내는 단계를 추가합니다. 매직 로그인 링크를 포함하여 제휴 파트너가 포털에 즉시 접근할 수 있도록 합니다. 이를 통해 지체 없이 오퍼를 홍보할 준비가 됩니다.

- **환영 이메일 발송:** 제휴 추천 링크, 총 수익, 커미션 수익과 같은 커스텀 값이 포함된 환영 이메일을 작성합니다. 클라이언트 포털 링크에 하이퍼링크를 걸고 명확성을 위해 이용약관이나 프로그램 가이드라인을 첨부할 수도 있습니다.

- **캠페인 세분화를 위한 태그 사용:** 캠페인명을 사용하여 워크플로우 중에 고유한 태그(예: "러기지 캠페인 제휴 파트너")를 적용합니다. 이를 통해 속한 캠페인을 기준으로 향후 워크플로우나 프로모션 콘텐츠를 위해 제휴 파트너를 세분화할 수 있습니다.

## 자주 묻는 질문

**Q: 워크플로우 옵션에서 "제휴 캠페인 등록" 트리거가 보이지 않는 이유는 무엇인가요?**

제휴 관리(Affiliate Manager)가 활성화되어 있고 하위 계정에 적어도 하나의 제휴 캠페인이 생성된 경우에만 트리거가 나타납니다. 활성화하려면 Settings(설정) > Affiliate Manager(제휴 관리)로 이동하여 토글을 활성화하세요. 그 다음 제휴 관리 탭에서 캠페인을 만드세요. 두 가지가 모두 활성화되면 워크플로우 빌더의 "Affiliate(제휴)" 카테고리에서 트리거를 볼 수 있습니다.

**Q: 제휴 파트너가 참여한 캠페인에 따라 서로 다르게 온보딩하는 데 이 트리거를 사용할 수 있나요?**

네! 트리거 설정 시 **캠페인별 필터**를 적용할 수 있습니다. 이를 통해 제휴 파트너가 참여한 캠페인에 따라 다른 워크플로우나 로직 분기를 실행할 수 있습니다. 예를 들어, "강의 제휴 프로그램"에는 한 가지 환영 이메일을, "제품 런칭 캠페인"에는 다른 환영 이메일을 보낼 수 있습니다. 조건부 단계를 추가하거나 세그먼트 기반 액션을 위해 태그를 사용할 수도 있습니다.

**Q: 제휴 커스텀 값(예: 추천 링크나 매직 로그인 링크)이 이메일/SMS 단계에서 비어서 나타나는 이유는 무엇인가요?**

추천 링크, 포털 링크, 매직 로그인 링크와 같은 제휴 커스텀 값은 워크플로우에 "제휴 캠페인 등록" 트리거가 포함되어 있고 실제 제휴 등록 이벤트에 의해 트리거될 때만 채워집니다. 이러한 값들은 동적이며 특별히 트리거 이벤트와 연결되어 있어, 실제 제휴 파트너가 캠페인에 참여하여 워크플로우가 활성화되지 않으면 나타나거나 제대로 작동하지 않습니다. 빈 값이 보인다면 캠페인에 추가된 유효한 제휴 기록으로 워크플로우가 테스트되고 있는지, 트리거가 올바르게 구성되고 저장되었는지 확인하세요. 그러면 이메일이나 SMS 단계에 변수를 삽입할 때 "제휴 캠페인 등록" 섹션에서 커스텀 값을 사용할 수 있습니다.

**Q: 제휴 파트너를 수동으로 등록해도 트리거가 작동하나요?**

물론입니다. 제휴 파트너가 **자동으로**(폼, 퍼널, 가입 링크를 통해) 등록되든 **수동으로**(팀이 제휴 관리에서 직접 추가) 등록되든 트리거가 활성화됩니다. 등록이 발생하고 필터 기준(설정된 경우)과 일치하는 한 워크플로우가 트리거되어 구성한 단계들이 실행됩니다.
