# 이메일과 퍼널의 카운트다운 타이머 동기화

카운트다운 타이머가 진짜 효과를 내려면, 이메일과 고객이 실제 행동을 취하는 활성 페이지에서 타이머가 일치해야 해요.

**예시**: 12월 25일 자정까지 유효한 10% 할인 카운트다운 타이머를 설정하는 경우

### 언제 동기화가 필요한가요?

- **세일 카운트다운**: 웹사이트에서 카운트다운 타이머와 함께 세일을 진행 중이에요. 이메일에도 같은 카운트다운 타이머가 있고 세일 페이지로 링크돼요. 두 타이머가 일치하므로 고객은 어디를 클릭하든 남은 시간을 정확히 볼 수 있어요.

- **신상품 출시**: 새 제품을 런칭하면서 랜딩 페이지에 카운트다운 타이머를 넣었어요. 홍보 이메일에도 같은 타이머가 있고 동일한 랜딩 페이지로 연결돼요. 타이머가 끝나면 페이지가 자동으로 제품 구매 페이지로 이동해요.

- **이벤트 등록**: 얼리버드 티켓을 판매하며 카운트다운 타이머로 남은 시간을 보여줘요. 이벤트 이메일에도 같은 타이머가 있고 등록 페이지로 링크돼요. 타이머가 끝나면 등록 페이지가 정가 티켓 가격으로 업데이트돼요.

- **웨비나 등록**: 웨비나를 홍보하며 등록 마감까지의 카운트다운 타이머를 보여줘요. 이메일 초대장에도 같은 타이머가 있고 등록 페이지로 연결돼요. 타이머가 만료되면 페이지가 대기자 명단이나 추가 정보 페이지로 리디렉트돼요.

### **이메일과 퍼널의 카운트다운 타이머 동기화 단계**

- **리디렉트 링크 설정**: 퍼널에서 타이머용 활성 링크와 만료 링크 페이지를 설정하세요 (예: 할인 페이지와 만료 페이지)

- **타이머 생성**: `Marketing(마케팅) → Countdown Timer(카운트다운 타이머)`로 가서 설정한 링크로 새 타이머를 만드세요.
![카운트다운 타이머 생성](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033072656/original/jJz3xEyGsUNBAS-79cQEWL3SpOUJakbVrg.png?1726652337)

- **이메일에 타이머 추가**: 이메일 캠페인을 만들고 카운트다운 타이머 요소를 삽입한 후 생성한 타이머를 선택하세요.
![이메일 카운트다운 타이머 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033073767/original/_03a1A9AcMDdk0NQDm65G6dH37lNc63gng.png?1726652960)

- **스타일 설정**: 설정 섹션에서 타이머의 모양을 조정하세요.
![타이머 스타일 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033074056/original/e34TwS5vZ2o9piwgt5AXrMCe3OqBGh_WOg.png?1726653104)

- **액션 버튼 추가**: 이메일에 버튼을 넣고 활성 타이머 페이지로 링크한 후 설정에서 타이머의 링크를 사용하세요.
![이메일 액션 버튼 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033074400/original/00x9jxNir0qBssRWlTfL3LI4dyQcgf7Vxg.png?1726653218)

- **퍼널 페이지에 타이머 추가**: 퍼널의 활성 오퍼 페이지에 동일한 카운트다운 타이머 요소를 삽입하세요.
![퍼널 타이머 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033074975/original/Srkc2rgkxAZJ9KlrbHXR-eqhWb2AEFOcyg.png?1726653491)

- **타이머 정렬 및 스타일링**: 퍼널 페이지에서 타이머의 위치와 스타일을 이메일 타이머와 일치하도록 조정하세요.
