# 이메일에서 카운트다운 타이머 사용 방법

이메일의 카운트다운 타이머는 특정 마감일까지 시간을 세는 동적 요소로, 긴급감을 조성하는 역할을 합니다. 독점성을 전달하고 고객이 빠르게 행동하도록 유도하는 데 효과적이에요.

## 카운트다운 타이머를 사용하는 이유

- **시각적 매력 향상**: 이메일에 시각적으로 매력적이고 참여도 높은 요소를 추가합니다.
- **긴급감 조성**: 고객이 더 빠르게 결정을 내리고 행동을 지연시키지 않도록 격려합니다.

## 이메일에서 카운트다운 타이머는 어떻게 작동하나요?

이메일의 카운트다운 타이머는 일반적으로 GIF로 구현됩니다. 작동 방식은 다음과 같아요:

- **동적 업데이트**: 타이머는 사용자가 이메일을 열었을 때의 시간을 기준으로 업데이트됩니다. 현재 상태를 확인하고 해당 시점부터 최대 60초간 카운트다운을 계속합니다.

- **예시**: 타이머가 12월 25일 자정에 종료되도록 설정되어 있고 사용자가 12월 24일에 이메일을 열었다면, 타이머는 "24시간 남음"을 보여줍니다. 그 후 "23시간 59분 남음"으로 카운트다운하며, 이메일이 열려있는 동안에는 업데이트가 멈춥니다. 이메일을 닫았다가 다시 열면 타이머가 최신 타임스탬프로 새로고침되어 다음 60초를 카운트다운합니다.

**이런 방식을 사용하는 이유**: 이 방법은 이메일 크기의 균형을 맞추고 사용자가 타이머가 카운트다운을 완료하기 전에 이메일을 닫는 경우가 많다는 점을 고려한 것입니다.

### Apple Mail의 예외 상황

- Apple Mail은 GIF를 캐싱하므로 카운트다운 타이머 업데이트에 영향을 줍니다. 이메일을 열 때마다 타이머가 업데이트되는 다른 이메일 클라이언트와 달리, Apple Mail은 새로고침 없이 동일한 GIF를 반복적으로 표시할 수 있습니다.

- **결과**: 따라서 사용자가 Apple Mail에서 같은 이메일을 여러 번 열면, 카운트다운 타이머가 첫 번째 열람 후 멈춘 것처럼 보일 수 있습니다. 이는 Apple Mail이 GIF를 캐싱하기 때문에 후속 열람에서 카운트다운이 업데이트되지 않기 때문입니다.

## 이메일에서 카운트다운 타이머 사용 방법

- **카운트다운 타이머 생성**: `Marketing(마케팅) → Countdown Timer(카운트다운 타이머)`로 이동하여 타이머를 설정하세요.

- **이메일 템플릿 열기**: `Email Marketing(이메일 마케팅)`으로 이동하여 캠페인 템플릿을 열거나 새로 만드세요.

- **이메일에 타이머 추가**: 카운트다운 타이머 요소를 이메일에 삽입하세요.
![카운트다운 타이머 요소 삽입](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033075446/original/MO7ZtNeg8vztbVn6XkKXG0TnmdQuRIE9Qw.png?1726653765)

- **타이머 선택**: 디자인과 세부사항을 확인하여 원하는 타이머를 선택하세요.
![타이머 선택 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033075704/original/G-KWc4HkUlZLDReZjUvD6AJAG-qDSrPrZQ.png?1726653858)

- **타이머 설정 조정**: Settings(설정)를 클릭하여 이메일 내에서 타이머 정렬을 조정하세요.
![타이머 설정 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033075763/original/3AyK53mWGaolSv961A_Vhb1F0OHeBF3l_w.png?1726653886)

- **외관 커스터마이즈**: 배경색을 설정하고 필요에 따라 패딩을 조정하세요.

- **리다이렉션 설정**: 리다이렉션 링크를 복사하여 이메일 버튼이나 CTA에 사용하세요.
