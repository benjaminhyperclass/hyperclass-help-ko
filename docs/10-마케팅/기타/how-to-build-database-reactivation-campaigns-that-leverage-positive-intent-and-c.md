# 긍정적 의도와 챗봇을 활용한 데이터베이스 재활성화 캠페인 구축 가이드

데이터베이스에 잠들어 있는 기존 연락처들을 다시 활성화시키는 것은 새로운 고객을 찾는 것보다 훨씬 효율적인 마케팅 전략입니다. Hyperclass에서는 긍정적 의도 신호와 AI 챗봇을 결합하여 효과적인 재활성화 캠페인을 만들 수 있습니다.

## 재활성화 캠페인이란?

데이터베이스 재활성화 캠페인은 오랫동안 비활성 상태였던 기존 연락처들에게 다시 접근하여 관심을 불러일으키고 재참여를 유도하는 마케팅 활동입니다. 이미 브랜드를 알고 있는 고객들이기 때문에 완전히 새로운 고객보다 전환율이 높은 편입니다.

## 긍정적 의도 활용하기

### 1. 의도 신호 파악하기

다음과 같은 행동들은 고객의 긍정적 의도를 나타냅니다:
- 이메일 열어보기
- 링크 클릭하기
- 웹사이트 재방문
- 소셜 미디어 참여
- 과거 구매 이력

### 2. 스마트 리스트(Smart List) 만들기

**Contacts(연락처) → Smart List(스마트 리스트)**에서 다음 조건으로 세분화하세요:

**재활성화 타겟 리스트:**
- 마지막 활동: 30일 이상 전
- 과거 구매 이력: 있음
- 이메일 상태: 구독 중
- 태그(Tag): "고객" 또는 "관심 고객"

**긍정적 반응 리스트:**
- 최근 7일 내 이메일 열어봄
- 웹사이트 방문
- 링크 클릭 이력

## 워크플로우(Workflow) 설정하기

### 1. 초기 접촉 워크플로우

**Workflow(워크플로우) → Create New(새로 만들기)**

**트리거(Trigger):**
- 스마트 리스트에 추가됨

**액션(Action) 순서:**
1. **대기(Wait)**: 1일
2. **이메일 발송**: 개인화된 재연결 메시지
3. **대기(Wait)**: 3일
4. **조건 분기(If/Else)**: 이메일 열어봤는지 확인
   - Yes: 다음 단계로
   - No: SMS 발송 후 대기

### 2. 챗봇 연결 워크플로우

긍정적 반응을 보인 연락처들에게:

1. **이메일 발송**: "궁금한 점이 있으시면 채팅으로 문의해 주세요"
2. **트리거 링크(Trigger Link)** 포함
3. 링크 클릭 시 → 챗봇 대화 시작

## 챗봇 설정하기

### 1. 대화 AI(Conversation AI) 설정

**Settings(설정) → AI → Conversation AI**

**봇 지시사항(Bot Instructions):**
```
당신은 [회사명]의 친근한 고객 서비스 담당자입니다. 
오랜만에 연락하는 기존 고객들을 환영하고, 
현재 진행 중인 서비스나 특별 혜택을 안내해 주세요.
항상 정중하고 도움이 되는 톤으로 응답하세요.
```

### 2. 커스텀 봇 응답(Custom Bot Responses) 추가

자주 묻는 질문들을 미리 등록하세요:
- "어떤 서비스가 있나요?"
- "할인 혜택이 있나요?"
- "상담 예약하고 싶어요"
- "이전에 구매한 서비스는 어떻게 되나요?"

### 3. 챗봇과 워크플로우 연결

챗봇 대화에서 특정 키워드 감지 시:
- "상담" 감지 → 캘린더(Calendar) 링크 자동 발송
- "할인" 감지 → 특별 오퍼(Offer) 안내
- "구매" 감지 → 세일즈 팀 즉시 알림

## 개인화 전략

### 1. 커스텀 필드(Custom Field) 활용

- 마지막 구매 상품
- 선호 연락 방식
- 관심 카테고리
- 고객 등급

### 2. 다이나믹 콘텐츠

이메일과 SMS에 개인화 요소 포함:
```
안녕하세요 {{first_name}}님,

{{last_purchase_date}}에 구매해 주신 {{last_product}} 
서비스는 만족스러우셨나요?

이번 달 {{customer_tier}} 고객님만을 위한 
특별 혜택을 준비했습니다.
```

## 멀티 채널 접근법

### 1. 채널별 메시지 차별화

**이메일:** 상세한 정보와 비주얼
**SMS:** 간단하고 즉시성 있는 메시지
**채팅:** 실시간 상호작용과 개인 맞춤 상담

### 2. 채널 간 연결

- 이메일 → 채팅봇 유도
- SMS → 랜딩 페이지(Landing Page) 연결
- 채팅 → 전화 상담 예약

## 성과 측정 지표

### 1. 핵심 KPI

- **재활성화율**: 응답/참여한 연락처 비율
- **전환율**: 실제 구매/예약으로 이어진 비율
- **채팅 참여율**: 챗봇 대화 시작 비율
- **ROI**: 캠페인 비용 대비 매출

### 2. 리포팅(Reporting) 활용

**Reporting(리포팅) → Email Campaign(이메일 캠페인)**에서:
- 오픈율, 클릭률 추적
- 링크별 성과 분석
- 시간대별 참여도 확인

## 자동화 최적화 팁

### 1. A/B 테스트

- 제목 줄 테스트
- 발송 시간 테스트
- 채널 조합 테스트

### 2. 점진적 강도 증가

1차: 부드러운 재연결 메시지
2차: 가치 제안 중심
3차: 긴급성과 한정성 추가

### 3. 이탈 방지

부정적 반응 시:
- 즉시 후속 메시지 중단
- 태그(Tag) 변경: "재활성화 불가"
- 6개월 후 재시도 스케줄링

## 법적 고려사항

- **개인정보보호법** 준수
- **이메일 수신동의** 확인
- **SMS 발송 시간** 제한 (오전 8시~오후 9시)
- **수신거부** 옵션 필수 포함

데이터베이스 재활성화 캠페인은 꾸준한 최적화와 개인화를 통해 높은 성과를 얻을 수 있습니다. Hyperclass의 AI 기능과 자동화 도구를 적극 활용하여 효율적인 고객 재참여 전략을 구축해 보세요.
