# 소셜 플래너에서 페이스북 릴 게시하기

짧은 형식의 동영상은 소셜 미디어 마케팅 세계에서 지속적으로 성장하는 트렌드입니다. 이에 맞춰 소셜 플래너에서 페이스북 릴(Facebook Reels) 기능을 출시합니다.

**페이스북 릴은 어떻게 만드나요?**

1. Marketing(마케팅) > Social Planner(소셜 플래너)로 이동하세요

2. Settings(설정) > Integrations(연동)으로 가서 페이스북 계정을 연결하세요. 페이스북 계정에 필요한 권한을 허용해 주세요.

3. Create New Post(새 게시물 만들기)로 이동하세요

4. 새 게시물을 만들 때, 릴 게시를 위해 페이스북 페이지를 선택하세요.

참고 - 현재 릴은 페이지에서만 사용 가능합니다. 그룹을 선택하면 피드 게시물로 공유됩니다.

![페이스북 페이지 선택 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155022056598/original/tsPbo21y3WF69WxeHpJBPDipwAztgqnMjA.png?1709303675)

5. Reels(릴) 탭으로 이동하세요

6. 캡션을 추가하고 동영상을 하나 첨부하세요

![릴 작성 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155022056626/original/qbG4PeRUeY8xRn3BNfjzi6vIXj1TGCsveQ.png?1709303699)

7. 릴을 예약하세요

![릴 예약 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155022056606/original/BC4_DPD3yzPfe2dw20w7NOOj1rwl709xrQ.png?1709303681)

**페이스북 릴 사양:**

1. 동영상 크기: 1GB 이하
2. 재생 시간: 3초에서 90초(1분 30초)
3. 동영상 형식: .MP4, .MOV, .webm
4. 해상도: 1080 × 1920 픽셀 (권장)
   최소 540 × 960 픽셀
5. 화면 비율: 9:16
6. 페이스북 릴에서는 썸네일 지원이 제공되지 않습니다.
