# 이메일 캠페인 배치 예약 방법

배치 예약(드립 모드)을 사용하면 마케팅 이메일을 한 번에 모두 발송하지 않고 시간 간격을 두고 배치별로 전달할 수 있습니다. 이는 웹사이트나 퍼널로의 트래픽을 관리하고 발송 인프라의 부하를 줄이는 데 도움이 됩니다. 이 가이드를 사용하여 Email Builder(이메일 빌더)의 Batch Schedule(배치 예약) 옵션으로 배치 일정을 설정하는 방법을 알아보세요.

## **이메일 캠페인 배치 예약이란?**

배치 예약(배치 전달 또는 드립 모드라고도 함)은 정의된 간격에 걸쳐 전달을 분산시킵니다. 전체 대상에게 동시에 발송하는 대신, 배치 수량, 타이밍, 활성 요일을 제어하여 전달이 점진적이고 예측 가능하게 이루어지도록 합니다.

## 이메일 캠페인 배치 예약의 주요 장점

- **트래픽 관리**: 웹사이트나 퍼널로의 갑작스러운 급증을 피하기 위해 시간 간격을 두고 배치로 발송합니다.

- **서버 부하 제어**: 발송 속도를 조절하여 서버에 가해질 수 있는 부하를 줄입니다.

- **유연한 타이밍**: 배치 수량, 반복 빈도, 요일, 시작/종료 시간을 선택할 수 있습니다.

## 발송 전 체크리스트

이 과정을 시작하기 전에 알아두어야 할 사항들입니다.

- 이메일을 발송하려면 [캠페인 발송용 이메일 템플릿을 생성](../../../01-시작하기/기본-설정/getting-started-launch-an-email-campaign.md)했는지 확인하세요

- 이메일을 발송하려면 이메일 발송용 도메인이 인증되었는지 확인하세요.

## **배치 이메일 캠페인 설정 방법**

Email Builder에서 다음 단계를 따라 캠페인을 생성하고 배치 전달(드립 모드)을 설정하세요.

### 1단계: 이메일 캠페인 생성

- Marketing(마케팅) > Email(이메일) > Campaigns(캠페인)으로 이동하세요.
![이메일 캠페인 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063151520/original/ZfEYawuF5RMeLtgLcZPxRYP09tVq6PiBQA.jpeg?1768997742)

- New Campaign(새 캠페인) 생성 또는 기존 캠페인을 편집하세요.

![새 캠페인 생성](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063151521/original/MP4pnQs-seNMDb6zgeBWNV4wLg5J9tuBhQ.jpeg?1768997742)

### 2단계: 이메일 디자인

참고: Email Builder와 인라인 에디터 사용 방법에 대한 자세한 내용은 이 [가이드](../../Email-Builder-Capabilities/how-to-use-the-email-builder-and-its-in-line-editor.md)를 참고하세요.

- 템플릿을 구축하거나 선택하세요.

- 콘텐츠, 이미지, 링크를 추가하세요.

- Preview(미리보기)와 Test Email(테스트 이메일)을 사용하여 데스크톱과 모바일에서의 레이아웃을 확인하세요.
![이메일 디자인](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063151522/original/Z3mVbgpb7EtTlR84G9ion3OMCK5N5OlO2g.jpeg?1768997743)

### 3단계: 전달 옵션 선택 (배치 예약)

- Send(발송) 또는 Schedule(예약) 버튼을 클릭하세요.
![발송 또는 예약 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155063151529/original/umIMMvfAC0puh_z5iDvDAUJtmwFDPnc49w.jpeg?1768997744)

- **Sending Options(발송 옵션)**에서 **Batch Schedule(배치 예약)**을 선택하고 아래 단계를 완료하세요:

캠페인의 시작 날짜와 시간을 추가하세요.

- **Batch Quantity(배치 수량)**을 추가하세요.

**참고:** 배치 수량은 배치 개수가 아닌 배치당 발송되는 이메일 개수를 의미합니다. 예를 들어, 수신자가 10,000명이고 배치 수량을 1,000으로 설정하면, 시스템은 각각 1,000개의 이메일로 구성된 10개의 배치로 발송합니다.

- Repeat after(반복 간격)에 빈도에 따른 숫자를 추가하고 일, 시간, 분, 초 단위로 빈도를 설정하세요.

*이전에 저장된 초안이 초 단위를 사용했다면, 예약 오류를 방지하기 위해 해당 필드가 자동으로 초기화됩니다.*

- 배치 전달 이메일 캠페인을 발송하고 싶지 않은 요일의 체크를 해제하세요.

- **시작 및 종료 시간**을 구체적으로 추가하세요.

참고: 검증 (배치 예약)

종료 시간이 시작 시간보다 빠른 경우, 종료 시간 아래에 인라인 오류가 나타납니다: 종료 시간은 시작 시간보다 앞설 수 없습니다.

- 종료 시간이 시작 시간보다 앞서 선택된 경우, 시작 시간 아래에 인라인 메시지가 나타납니다: 유효한 시작 시간을 입력하세요.

- 오류를 수정할 때까지 Send(발송)와 Schedule(예약)이 비활성화됩니다.

- 시작 시간 편집은 단일 OK 버튼으로 확인됩니다.

- 자정(12:00 AM) 경계는 잘못된 오류 없이 처리됩니다.

![배치 예약 설정](https://jumpshare.com/share/nZlsGLmQmgQ0E5XV5aOq+/Screen+Shot+2026-01-21+at+5.58.09+PM.png)

- **Sender Name(발신자 이름)** 섹션에 이름을 추가하세요 (선택사항).

- **Sender Domain(발신자 도메인)**을 선택하거나 설정하세요.

- Sender Email(발신자 이메일) 섹션에 이메일 주소를 추가하세요.

- Subject Line(제목)을 추가하세요.

- **Preview(미리보기)** 섹션에 미리보기 텍스트를 추가하세요. (선택사항)

- **To(받는 사람)** 섹션에서 이메일 또는 스마트 리스트를 선택하여 수신자를 추가하세요.

![발신자 정보 및 수신자 설정](https://jumpshare.com/share/lYpiAnM4jeiK0HjH8DLW+/Screen+Shot+2026-01-21+at+6.02.32+PM.png)

참고: 예약하기 전에 실행 날짜, 시간, 시간대 세부사항을 확인하세요!
이메일 캠페인을 삭제하거나 [일정을 변경](how-to-reschedule-an-email-campaign-email-builder-.md)하는 옵션이 있습니다. 캠페인 일정을 변경하고 싶다면 reschedule(일정 변경)을 클릭하여 캠페인의 날짜-시간을 설정할 수 있습니다.

일정 변경은 이메일 캠페인 발송 1시간 전까지만 허용됩니다. 예를 들어 예약된 날짜가 오전 11시 EST라면, 오전 10시 EST까지 일정을 변경할 수 있습니다.

## **자주 묻는 질문**

**Q: 배치 캠페인의 내용을 편집하고 싶은데 가능한가요? 통계에 영향을 주고 싶지 않습니다.**

네, **Update Email Content(이메일 콘텐츠 업데이트)**를 클릭하여 진행 중인 배치 예약 캠페인의 내용을 편집할 수 있습니다. 실행 날짜 전에 내용을 편집한 후 **save edits(편집 저장)**을 클릭하세요.

참고: 내용이 업데이트된다고 보장할 수는 없지만 서버에서 실행 대기열이 시작되기 전에 업데이트하려고 시도합니다.

**Q: RSS 이메일 캠페인의 일정을 어떻게 변경하나요?**

이메일 캠페인 일정 변경은 실행 날짜 1시간(60분) 전까지 **Simple Schedule(단순 예약), Batch Schedule(배치 예약), RSS Schedule(RSS 예약)**에 대해 가능합니다.

**Marketing(마케팅) > Email(이메일)**에 방문해야 합니다. **Email Campaign(이메일 캠페인)** 탭을 방문하면 실행 날짜/시간 1시간 전까지 일정 변경이 가능한 Simple Schedule, RSS Schedule, Batch Schedule로 구성된 이메일 캠페인 목록이 있습니다.

액션 버튼에 **Reschedule Campaign(캠페인 일정 변경)** 옵션이 있을 것입니다. 사용자는 예약 세부사항을 변경하고 전달 모델을 바꿀 수 있습니다. 변경이 완료되면 설정을 저장하기 위해 일정이 변경된 캠페인으로 표시해야 합니다.

**Q: 시스템에서 발송하는 속도 제한은 어떻게 되나요?**

드립 모드에서 추가:

- **30초 - 1분 빈도:** 분당 1,000개 메시지

- **5분 빈도:** 분당 4,999개 메시지

- **6분 - 10분 빈도:** 분당 5,000개 메시지

- **10분 초과 빈도:** 분당 10,000개 메시지

한 번에 모두 추가 및 예약된 시간에 모두 추가:

- 10k 미만 - 시간당 15k 연락처 처리

- 10k 초과 50k 미만 - 시간당 6k 연락처 처리

- 50k 초과 70k 미만 - 시간당 3k 연락처 처리

- 70k 초과 - 시간당 1.5k 연락처 처리

### Repeat after(반복 간격) 드롭다운에는 초가 포함되지 않습니다. 이전에 초 단위 간격을 사용했다면 1분 이상을 선택하세요.
