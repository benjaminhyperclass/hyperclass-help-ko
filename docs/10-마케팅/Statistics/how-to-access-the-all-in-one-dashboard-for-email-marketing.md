# 이메일 마케팅 통합 대시보드 사용법

이 가이드에서는 Hyperclass의 이메일 마케팅 통합 대시보드 사용법을 알려드립니다. 이 강력한 대시보드를 통해 이메일 캠페인(Campaigns), 워크플로우(Workflows), 일괄 작업(Bulk Actions)으로 발송된 모든 이메일 활동을 한눈에 볼 수 있어, 성과 모니터링과 데이터 기반 의사결정이 더욱 쉬워집니다.

## 이메일 마케팅 통합 대시보드란?

통합 대시보드는 이메일 캠페인(Campaigns), 워크플로우(Workflows), 일괄 작업(Bulk Actions)을 통해 발송된 모든 이메일의 성과 지표를 한곳에서 볼 수 있는 중앙 허브입니다. 여러 도구를 오가며 확인할 필요 없이, 이메일 성과 추적과 계획 수립을 간소화하는 통합 뷰를 제공합니다.

---

## 통합 대시보드의 주요 장점

중앙 집중식 대시보드는 모든 이메일 지표를 한곳에 모아 모니터링과 계획 수립을 간소화합니다.

- 캠페인, 워크플로우, 일괄 작업 이메일의 성과를 한 화면에서 확인
- 유형별 최고 성과 이메일에서 인사이트 획득
- 채널별 데이터 필터링 기능
- 스크롤하여 오픈율/클릭률 등 상세 통계 탐색
- 모든 관련 데이터를 손끝에서 확인하여 의사결정 개선

## 통합 대시보드 사용법

대시보드를 탐색하고 이메일 마케팅 활동을 효과적으로 모니터링하는 방법을 알아보세요.

- `Marketing(마케팅) → Emails(이메일) → Statistics(통계)`로 이동하세요

![이메일 통계 대시보드 접근](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048491545/original/X6uhv8X-9lMLNfKHNNNpV-3cJlJHShaCUg.png?1750261421)

- 기본으로 표시되는 통합 통계를 확인하세요
- 상단 왼쪽 드롭다운을 사용해 이메일 채널별로 필터링하세요. (연락처 → 일괄 작업을 통해 발송된 이메일은 대시보드 필터에서 'Bulk Action Campaigns'로 그룹화됩니다)
- 아래로 스크롤하여 최고 성과 이메일을 포함한 더 자세한 성과 지표를 탐색하세요

**팁**: 연락처(Contacts) → 일괄 작업(Bulk Actions) → 통계 보기(Show Statistics)에서 시작했다면, 'Load More Stats'를 사용하여 일괄 작업 보고서가 포함된 이 이메일 통계 대시보드를 열 수 있습니다.

![이메일 통계 사용 예시](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155064551067/original/tTC8Sy-JS2jUnxplRrreeSSRh9VwxlTNqg.gif?1770712074)

## 주요 기능: 최고 성과 이메일

이 섹션은 가장 높은 참여율을 이끌어낸 이메일 콘텐츠를 유형별로 확인하는 데 도움이 됩니다.

최고 성과 이메일은 가장 효과적인 개별 이메일을 보여줍니다:

- **캠페인(Campaigns)**과 **일괄 작업(Bulk Actions)**: 전체 이메일 발송을 추적
- **워크플로우(Workflows)**: 전체 워크플로우가 아닌 **개별 이메일 액션** 수준에서 성과를 강조

## 자주 묻는 질문

**Q: 대시보드에서 데이터를 내보낼 수 있나요?**

현재 대시보드에서 직접 내보내기 기능은 지원되지 않습니다.

**Q: 대시보드에 SMS나 기타 메시징 통계가 포함되나요?**

아니요, 현재 대시보드는 이메일 활동만을 다룹니다.

**Q: 다양한 이메일 유형 간 성과를 비교할 수 있나요?**

네, 드롭다운을 사용해 채널별로 필터링하고 성과 지표를 시각적으로 비교할 수 있습니다.

**Q: 트리거로 발송되는 자동 이메일도 포함되나요?**

네, 워크플로우 트리거를 통해 발송된 이메일도 포함되어 추적됩니다.
