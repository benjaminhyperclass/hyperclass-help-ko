# 캠페인 통계에서 연락처에 태그 추가하기

연락처 세분화가 더욱 쉬워졌습니다! 이제 캠페인 통계 페이지에서 바로 열람, 클릭 등의 상호작용을 기반으로 태그를 추가하고 자동화할 수 있습니다. 미리 계획하거나 번거로운 수동 작업은 더 이상 필요하지 않습니다.

## 개요
이제 캠페인 통계 페이지에서 바로 캠페인과의 상호작용(열람, 클릭 등)을 기반으로 연락처에 태그를 추가할 수 있습니다. 이를 통해 사전에 태그를 설정하지 않고도 즉시 세분화하고 맞춤형 후속 조치를 취할 수 있습니다. 또한 자동 태그 기능으로 캠페인과 상호작용하는 새로운 연락처들이 실시간으로 자동 태그되어 세그먼트가 손쉽게 최신 상태로 유지됩니다.

## 장점
이전에는 미리 계획된 워크플로우나 수동 일괄 작업을 통해서만 태그를 추가할 수 있었습니다. 이번 업데이트로 다음이 더욱 쉬워졌습니다:

- **즉시 세분화**: 미리 계획할 필요 없이 캠페인 후에 세분화
- **자동 태그 추가**: 새로운 상호작용이 수동 작업 없이 자동으로 태그됨
- **추적 가능한 로그**: 태그 작업의 깔끔한 이력 관리
- **다중 태그**: 더욱 세밀한 세분화를 위한 여러 태그 적용

## 새로운 기능

- **상호작용별 태그**: 통계 페이지에서 바로 열람, 클릭 또는 상호작용한 연락처에 태그 적용
- **미래 상호작용 자동 태그**: 캠페인과 상호작용하는 새로운 연락처를 자동으로 태그

## 단계별 가이드

### 1. 상호작용별 연락처 태그하기

- Marketing(마케팅) → Email Campaigns(이메일 캠페인)으로 이동하세요.
- 발송된 캠페인을 찾고 3점 메뉴 → Statistics(통계)를 클릭하세요.
- Load More를 클릭하여 상호작용 데이터(열람, 클릭 등)를 확인하세요.
- 상호작용 카테고리(예: 열람)를 선택하세요.
- 태그를 추가할 연락처를 선택하세요.
- Add Tags를 클릭하고 원하는 태그를 입력한 후 적용하세요.

### 2. 미래 상호작용 자동 태그

- 상호작용 카테고리(예: 열람)에서 All(전체)을 선택하세요.
- Add Tags를 클릭하고 태그를 입력하세요.
- Auto-Tag Future Interactions(미래 상호작용 자동 태그) 토글을 활성화하세요.

### 3. 태그 및 로그 추적

- 모든 수동 태그 추가는 Tags(태그) 및 Bulk Actions(일괄 작업)에서 확인할 수 있습니다.
- 자동 태그된 연락처는 Bulk Actions 로그에는 나타나지 않지만 상호작용 시 자동으로 태그됩니다.

## 제한사항

- 자동 태그는 다음에서 사용할 수 없습니다:
  - 잘못된 연락처
  - 주문 세부 정보 목록

- 미래 자동 태그를 제외한 모든 작업은 Bulk Actions 탭에 기록됩니다.

## 자주 묻는 질문

**Q1: 이 기능으로 태그를 제거할 수 있나요?**
A: 아니요. 이 도구는 현재 태그 추가만 지원합니다. 태그는 Contacts(연락처) 또는 Tags(태그) 설정에서 수동으로 제거할 수 있습니다.

**Q2: 동일한 태그를 다시 적용하면 어떻게 되나요?**
A: 중복 태그는 무시됩니다. 연락처가 같은 태그로 여러 번 태그되지는 않습니다.

**Q3: A/B 테스트 캠페인에서도 사용할 수 있나요?**
A: 네, A/B 테스트 캠페인에서도 태그 기능이 작동합니다.

**Q4: 한 번에 몇 개의 태그를 적용할 수 있나요?**
A: Contacts 섹션에서와 마찬가지로 여러 태그를 적용할 수 있습니다.

**Q5: 로그 데이터는 어디서 볼 수 있나요?**
A: Statistics 인터페이스 내에서 로그 데이터를 확인할 수 있습니다.
