# 워크플로우 이메일 액션 통계 포함

워크플로우나 이메일 캠페인을 통해 발송된 모든 이메일 캠페인 통계를 동일한 방식으로 볼 수 있도록 개선하여 더욱 효율적인 분석이 가능합니다.

**목차**

- [개선된 내용](#개선된-내용)
- [사용 방법](#사용-방법)  
- [참고사항](#참고사항)

## **개선된 내용:**

- 워크플로우 이메일 통계가 이제 이메일 마케팅(Email Marketing) 하위에 표시됩니다.

- 워크플로우의 전체 및 개별 이메일 액션 통계를 확인할 수 있습니다.

- 같은 화면에서 워크플로우를 바로 편집할 수 있습니다.

## **사용 방법:**

- 이메일 마케팅(Email Marketing) → 캠페인(Campaigns) → 워크플로우 캠페인(Workflow Campaigns)으로 이동합니다.

- Edit Workflow(워크플로우 편집)를 클릭하여 빌더에 접근합니다.

- Statistics(통계)를 클릭하여 워크플로우 통계를 확인합니다.

- 드롭다운을 사용하여 개별 이메일 액션의 통계를 확인할 수 있습니다.

## **참고사항:**

- 이메일 액션 생성 및 템플릿과의 동기화/새 템플릿 생성 기능은 기존과 동일하게 유지됩니다.
