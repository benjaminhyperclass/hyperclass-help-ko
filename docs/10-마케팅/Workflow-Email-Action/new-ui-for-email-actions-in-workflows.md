# 워크플로우 이메일 액션의 새로운 UI

워크플로우의 이메일 액션(Email Action)이 더욱 간단하고 일관된 사용성을 위해 새롭게 디자인되었습니다. 이제 하나의 현대적인 인터페이스에서 이메일을 작성하고, 커스터마이징하고, 분석할 수 있습니다.

## **개요**

워크플로우의 이메일 액션이 통합되고 직관적인 UI를 제공하여, 캠페인(Campaigns), 일괄 작업(Bulk Actions), 워크플로우(Workflows) 전반에서 일관되고 효율적인 이메일 제작 과정을 제공합니다. 이제 하나의 인터페이스에서 UTM 추적, 클릭 분석, 템플릿 동기화 등의 고급 옵션과 함께 이메일을 시작하고, 편집하고, 추적할 수 있습니다.

## **새로운 기능**

#### **통합된 이메일 작성 플로우**

워크플로우, 일괄 작업, 마케팅 캠페인 세 가지 영역 모두 동일한 빌더 경험을 공유합니다. 다음 중에서 시작할 수 있습니다:

- 빠른 작성(Quick Compose) → 빠른 이메일을 위한 가벼운 편집기

![빠른 작성 편집기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055811301/original/TyiBgF_MAuA1O1qwuVi2PIvdrvIF0cNDJA.png?1760327945)

- 스마트 빌더(Smart Builder) → 디자인, 코드, 또는 플레인 텍스트 모드를 제공하는 완전한 기능의 편집기

![스마트 빌더](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055811304/original/FqyPY7dJI9KtEXuG8lznarDuE3E-3WD0Vw.png?1760327974)

- 템플릿 선택(Select Template) → 기존 템플릿을 사용하거나 템플릿 라이브러리에서 선택

![템플릿 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055811308/original/eaHUmmxGwwKjgkJXvfAIWjm-5H8-Xz3qrw.png?1760327994)

#### **고급 설정**

각 이메일 액션에 확장 가능한 고급 설정이 포함되어 캠페인을 세밀하게 조정할 수 있습니다:

- 링크 추적(Link Tracking) → 모든 링크의 클릭 데이터와 CTR 확인
- UTM 파라미터(UTM Parameters) → 캠페인 추적 태그 자동 추가
- 태그 추가(Add Tags) → 열람 또는 클릭에 따라 연락처에 자동으로 태그 부여

![고급 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055811325/original/oY-r-4PLURNUoSjN6TuoOhAEHxKl728pKA.jpeg?1760328065)

#### **템플릿 동기화**

템플릿을 선택할 때 새로운 "템플릿과 편집 동기화(Sync Edits with Template)" 체크박스를 볼 수 있습니다:

- 활성화하면 원본 템플릿의 업데이트가 지속적으로 자동 동기화됩니다
- 언제든 비활성화하여 이메일을 독립적으로 만들 수 있습니다

![템플릿 동기화](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155000006550-new-ui-for-email-actions-in-workflows)

템플릿 동기화에 대한 자세한 단계는 전용 가이드 이메일 액션을 위한 동기화 편집 체크박스를 확인하세요.

## **사용 방법**

1. `Marketing(마케팅) → Email Builder(이메일 빌더)`로 이동합니다
2. `Campaigns(캠페인) → Workflow Campaigns(워크플로우 캠페인)`을 클릭합니다
3. 이메일 액션을 추가하거나 편집합니다
4. 이메일을 시작할 방법을 선택합니다:
   - 빠른 작성(기본값) → 빠른 메시징을 위한 간단한 편집기
   - 스마트 빌더 → 디자인, 코드, 또는 플레인 텍스트 편집기 중 선택
   - 템플릿 선택 → 저장된 템플릿 또는 템플릿 라이브러리에 접근
5. 고급 설정을 확장하여:
   - 링크 추적 활성화
   - UTM 파라미터 추가
   - 태그 설정
6. 저장(Save)을 클릭하여 워크플로우 이메일 설정을 완료합니다

## **자주 묻는 질문**

**Q: "편집 동기화"는 무엇을 하나요?**
A: 이메일을 소스 템플릿과 연결된 상태로 유지합니다. 원본 템플릿에 업데이트가 있으면 여기에도 자동으로 동기화됩니다. 이메일을 독립적으로 만들려면 끄면 됩니다.

**Q: 링크 클릭을 추적하려면 어떻게 하나요?**
A: 고급 설정에서 링크 추적을 활성화하면 각 링크의 클릭과 CTR을 볼 수 있습니다.

**Q: UTM 파라미터는 무엇을 위한 것인가요?**
A: 이메일 트래픽이 어디서 오는지 추적하는 데 도움이 되며, 분석과 캠페인 인사이트에 유용합니다.
