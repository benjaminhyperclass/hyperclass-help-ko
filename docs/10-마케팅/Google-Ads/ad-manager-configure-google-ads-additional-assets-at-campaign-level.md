## **캠페인 수준 Google Ads 추가 에셋이란?**

캠페인 수준 에셋은 Google Ads를 향상시키는 애드온(예: 사이트링크, 통화 버튼, 로고, 이미지)입니다. Hyperclass에서 캠페인 수준으로 설정하면 해당 캠페인의 모든 광고 그룹과 광고에 적용되어 광고 그룹별로 반복 설정할 필요 없이 가시성, 참여도, CTR을 개선하는 데 도움이 됩니다. 이러한 에셋은 Google Ads에 더 풍부한 형식을 제공하고 검색(Search), 디스플레이(Display), YouTube 게재위치에서 성과를 개선합니다.

Hyperclass 광고 관리자에서는 Google Ads의 다음과 같은 캠페인 수준 에셋을 추가할 수 있습니다:

- **비즈니스 로고**
- **이미지 에셋**
- **통화 확장(Call Extensions)**
- **사이트링크(Sitelinks)**

## 추가 에셋 제한사항

| 에셋 유형 | 지원 형식 | 가로세로 비율 | 캠페인당 제한 | 참고사항 |
|---------|----------|-------------|-------------|---------|
| 비즈니스 로고 | JPG, PNG | 1:1 (정사각형) | 로고 1개 | 검색, 디스플레이, YouTube에서 사용 |
| 이미지 에셋 | JPG, PNG | 1:1 (정사각형), 1.91:1 (가로형) | 최대 20개 이미지 | 이미지당 최대 파일 크기: 5MB. 권장: 비율당 2-3개 |
| 통화 확장 | 전화번호 | 해당 없음 | 여러 개 | Google 승인 필수 |
| 사이트링크 | 텍스트 + URL | 해당 없음 | 2-20개 (권장 4개) | 설명 지원 (선택사항) |

## 캠페인 수준 추가 에셋의 주요 이점

- **일관성**: 캠페인 내 모든 광고가 동일한 브랜딩과 CTA를 공유합니다.
- **관련성**: 캠페인의 제안이나 퍼널 단계에 맞게 사이트링크/이미지를 조정합니다.
- **속도**: 중앙 집중식 설정으로 광고 그룹별 반복 작업을 피할 수 있습니다.
- **제어**: 계정 기본값을 재정의하여 캠페인별로 다양한 변형을 테스트합니다.
- **가시성**: 해당 게재위치에서 광고의 눈에 띄는 정도와 CTR을 개선할 수 있습니다.

## 캠페인 수준에서 Google Ads 추가 에셋 설정하기

### 1단계: 이미지 에셋 추가하기

참고: **저품질** 시각 자료, GIF, 오버레이, 오해의 소지가 있는 콘텐츠는 피하세요.
지원하지 않는 비율(예: **4:5**)을 업로드하면 자르기/크기 조정을 안내하는 경고 메시지가 표시됩니다.
일부 이미지가 실패해도 캠페인은 유효한 텍스트 광고와 기타 에셋으로 계속 게시됩니다.

- 추가 에셋(Additional Assets) 섹션에서 **이미지 업로드(Upload Images)**를 클릭하세요.
- **미디어 라이브러리**에서 **하나** 또는 여러 이미지를 선택하거나 새 파일을 업로드하세요.
- **캠페인**당 최대 20개의 이미지를 추가할 수 있습니다.
- 지원되는 가이드라인:
  - 형식: JPG, PNG
  - **가로세로 비율**: 1:1 (정사각형) 및 1.91:1 (가로형)
  - **파일 크기**: 이미지당 최대 5MB
  - 권장: 더 나은 커버리지를 위해 가로세로 비율당 최소 2-3개 이미지를 추가하세요.

![](https://jumpshare.com/share/ZVUSm3ITHwc1GqAWpxor+/Screen+Shot+2026-02-02+at+8.19.35+PM.png)

### 2단계: 비즈니스 로고 추가하기

참고: 1:1 또는 정사각형 이미지만 적합합니다.

- **+ 로고 업로드(+ Upload Logo)**를 클릭하세요.
- 기존 이미지를 선택하거나 미디어 **라이브러리**에서 **새 이미지를 업로드**하세요.
- 업로드한 후 **로고**를 클릭하여 자르기와 같은 편집 **옵션**을 여세요.
- 이 로고는 모든 게재위치에 나타납니다.

![](https://jumpshare.com/share/oPHDB9B2PrfcjjcttDBu+/Screen+Shot+2026-02-02+at+8.22.50+PM.png)

### 3단계: 통화 확장 설정하기

**참고:** 통화 확장은 **모바일 친화적**이며 **즉시 전화걸기** 기능을 제공하여 **CTR을 개선**합니다. '광고에서 전화걸기' 전환 액션을 생성하는 방법에 대해 [**여기서**](https://support.google.com/google-ads/answer/6095882#) **자세히 알아보세요**.

- **통화 확장(Call Extensions)** 드롭다운을 클릭하세요.
- 이전에 **추가한** **전화번호** 중 **하나** 또는 여러 개를 선택하세요.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155064006664/original/5SMkbqOFVqP_Bq3f8YSHtCJ0EypacMeB8A.png?1770044828)

- 또는 선택 드롭다운에서 **+ 새 통화 생성(+ Create a New Call)** 버튼을 클릭하여 새 번호를 추가하세요.

![](https://jumpshare.com/share/y5RNFWZMOX1cP9Iqs2Uo+/Screenshot+2026-02-02+at+8.41.42%E2%80%AFPM.png)

- **국가**를 선택하고 **유효한 번호**를 추가한 다음 생성(Create)을 클릭하세요.

![](https://jumpshare.com/share/wTrVjht24NID1ou8xLbV+/Screen+Shot+2026-02-02+at+8.43.48+PM.png)

### 4단계: 사이트링크 설정하기

참고: **최적의 전송**을 위해 **최소 2개**, **최대 20개**, **권장 4개**의 사이트링크 에셋을 추가해야 합니다.

- **사이트링크(Sitelinks)** 드롭다운을 클릭하세요.
- 기존 사이트링크 중 **하나** 또는 여러 개를 선택하세요.

![](https://jumpshare.com/share/H2x3lvrVQfBjwpAFNAsS+/Screenshot+2026-02-02+at+8.48.46%E2%80%AFPM.png)

- 새 사이트링크를 추가하려면 **+ 새 사이트링크 생성(+ Create a new sitelink)**을 클릭하고 다음을 입력하세요:
  - **사이트링크 텍스트** (최대 **25**자)
  - **선택사항: 설명 라인 1 & 2**
  - **최종 URL**
- **생성(Create)**을 클릭하세요.

![](https://jumpshare.com/share/54aCETQUB5QtaudEdww1+/GIF+Recording+2026-02-02+at+8.53.17+PM.gif)

### 이미지 에셋 추가하기

이미지를 추가하면 Google 서비스 전반에서 검색 광고가 더 눈에 띄게 만들어집니다. 브랜드에 안전한 시각 자료를 한 번 업로드하면 Google이 가장 좋은 성과를 내는 곳에 배치합니다.

**사양**
- 수량: 캠페인당 최대 20개 이미지
- 형식: JPG, PNG
- 가로세로 비율: 정사각형(1:1) 및 가로형(1.91:1)만 가능
- 최대 크기: 이미지당 5MB

**단계**
1. 캠페인 빌더에서 이미지 에셋(Image Assets)을 여세요.
2. 업로드(Upload)를 클릭하고 이미지를 선택하세요.
3. 각 파일이 비율과 크기 요구사항을 충족하는지 확인하세요.
4. 저장(Save)을 클릭하세요.

**동작**: Google은 검색, 디스플레이, YouTube의 게재위치에서 관련 이미지를 자동으로 선택합니다. 일부 이미지가 검사에 실패해도 텍스트 광고는 계속 게시됩니다.

### **검증 및 오류**

- 빌더는 정책, 가로세로 비율 또는 파일 크기 문제에 대한 명확한 메시지를 표시합니다.
- 실패한 에셋은 건너뛰고 유효한 에셋은 계속 적합합니다.

**모범 사례**
- 최대 커버리지를 위해 비율당 2-3개 이미지를 추가하세요.
- 텍스트 오버레이, 워터마크, GIF, 오해의 소지가 있는 콘텐츠를 피하세요.
- 헤드라인/설명을 보완하는 선명하고 정돈된 이미지를 사용하세요.

## 대체 로직

캠페인 수준에서 에셋이 추가되지 않은 경우:

- 시스템은 **계정 수준 에셋을 기본값으로 사용**합니다(사용 가능한 경우).
- 둘 다 없으면 광고가 이러한 향상 기능 없이 실행될 수 있습니다.

**참고:** 계정 수준 에셋에 대해 자세히 알아보기: [Google Ads 지원](https://support.google.com/google-ads/answer/7175034)

## 자주 묻는 질문

**Q: 정사각형이 아닌 로고를 업로드하면 어떻게 되나요?**
1:1(정사각형) 이미지만 적합합니다. 정사각형이 아닌 로고는 폐기됩니다. 내장된 자르기 도구를 사용하여 조정하세요.

**Q: 통화 확장으로 여러 전화번호를 추가할 수 있나요?**
네. 승인된 여러 번호를 할당할 수 있습니다. Google이 위치, 기기, 가용성에 따라 번호를 교체합니다.

**Q: 통화 확장이 "승인 거부됨"으로 표시되는 이유는 무엇인가요?**
통화 확장은 Google Ads 정책을 충족해야 합니다. 승인 거부 이유에는 유효하지 않거나 확인되지 않은 번호, 국가 코드 불일치, 형식 문제가 포함됩니다.

**Q: 사이트링크는 몇 개까지 추가할 수 있나요?**
**2-20개**까지 추가할 수 있지만 최적의 성과를 위해서는 **4개**를 권장합니다.

**Q: 기존 사이트링크나 전화번호를 편집할 수 있나요?**
네. 항목 옆의 연필/편집 아이콘을 사용하여 업데이트할 수 있습니다.

**Q: 캠페인 수준 에셋의 이점은 무엇인가요?**
- 모든 광고 그룹과 광고에 에셋을 적용하여 시간을 절약합니다.
- 브랜드 일관성을 유지합니다.
- 풍부한 광고 형식(로고, 이미지, 사이트링크, 통화)이 항상 포함되도록 보장합니다.

**Q: 이 설정이 계정 수준 에셋을 재정의하나요?**
네. 캠페인 수준 선택이 우선합니다. 없으면 계정 수준 에셋이 사용됩니다.

**Q: 지원하지 않는 비율(예: 4:5)로 이미지를 업로드하면 어떻게 되나요?**
지원하지 않는 비율은 **경고 메시지**를 표시합니다. 시스템이 이미지를 **1:1** 또는 **1.91:1**로 자르거나 크기를 조정하도록 안내합니다. 거부된 이미지는 게시되지 않습니다.

**Q: 이미지 업로드 실패가 캠페인 게시를 중단시키나요?**
아니요. 개별 이미지가 실패해도 캠페인은 유효한 텍스트 광고와 기타 승인된 에셋으로 계속 게시됩니다.
