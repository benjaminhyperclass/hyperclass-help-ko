# 소셜 게시물에 위치 태그 추가하기

소셜 미디어 게시물에 위치를 태그할 수 있는 완전히 새로운 기능을 소개하게 되어 매우 기쁩니다. 위치 태그를 추가하여 참여도 향상, 가시성 증대, 로컬 SEO 강화, 타겟 마케팅을 경험해보세요. 예를 들어, 인플루언서가 위치를 추가하면 사용자들이 해당 위치를 발견하고 트래픽을 유도할 수 있습니다.

**왜 중요한가요:**

- 소셜 미디어 참여도 측면에서, 위치 태그는 가시성을 개선하고 팔로워, 좋아요, 댓글과 같은 더 나은 인사이트를 얻을 가능성을 높여줍니다.

**어떻게 작동하나요?**

- Marketing(마케팅) > Social Planner(소셜 플래너)로 이동하세요
- Create New Post(새 게시물 만들기)를 클릭하세요

![새 게시물 만들기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155031519737/original/afB1M208RGPOSGmDrQ9DUoTPYjBG554Tjg.png?1724342341)

- 첫 번째 단계는 콘텐츠를 게시하고 싶은 소셜 미디어를 선택하는 것입니다

![소셜 미디어 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155027610850/original/HgXpyvL8lfTV-JLqvlU5TmgOU2SbcIJdSw.png?1718287206)

- 콘텐츠와 기타 설정을 추가하세요
- 위치 태그 옵션이 있으며, 이를 통해 장소나 위치를 추가할 수 있습니다. 페이스북과 인스타그램 게시물에 추가할 수 있습니다.
