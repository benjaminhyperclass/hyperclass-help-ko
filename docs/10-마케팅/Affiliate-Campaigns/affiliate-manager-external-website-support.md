# 제휴 관리(Affiliate Manager) 외부 웹사이트 지원

빅 뉴스! 이제 Hyperclass 웹사이트와 퍼널뿐만 아니라 어떤 외부 웹사이트에서도 제휴 캠페인을 실행할 수 있습니다. 이 획기적인 업데이트를 통해 제휴 파트너들이 웹상의 어디서든 당신의 상품을 홍보하고 판매할 수 있어, 더 많은 유연성과 도달 범위를 제공합니다.

## **새로운 기능**

- **외부 웹사이트 지원**: 외부 웹사이트 URL만 추가하면 제휴 파트너들이 몇 분 안에 운영을 시작할 수 있습니다!

- **범용 추적**: 하나의 제휴 링크로 서브도메인을 포함한 전체 웹사이트의 판매를 추적할 수 있습니다.

- **상품별 커미션**: 특정 상품에 대해 커스텀 커미션을 설정하거나, 연결된 Stripe 계정을 통해 판매되는 모든 항목에 기본 설정을 사용할 수 있습니다.

- **반복 구독**: 업그레이드, 다운그레이드, 반복 결제를 자동으로 추적하여 제휴 파트너가 모든 적격 거래에 대해 커미션을 받을 수 있도록 보장합니다.

## **작동 원리:**

- **제휴 관리(Affiliate Manager)로 이동하여 캠페인(Campaigns) 탭을 클릭하세요.**

- +추가(+Add)를 클릭하여 새 캠페인을 만드세요.
- 소스로 "외부 웹사이트(External Website)"를 선택하세요.
- 웹사이트 URL을 입력하여 제휴 추적 링크를 생성하세요.
- 기본 또는 상품별 커미션과 필요시 티어 레벨을 설정하세요.
- 외부 웹사이트에 추적 스크립트를 추가하세요.
- 캠페인을 발행하고 제휴 파트너들이 판매를 시작하도록 하세요!

## **왜 중요한가:**

제휴 파트너들이 어떤 웹사이트에서든 당신의 상품을 판매할 수 있어 마케팅 도달 범위가 확장됩니다. 도메인 전체 추적과 범용 링크를 통해 제휴 캠페인을 간소화하고, 더 많은 판매를 유도하며, 제휴 파트너들에게 손쉽게 보상을 제공할 수 있습니다.

## **중요 사항:**

- 현재는 Stripe를 통해 판매되는 상품만 지원됩니다.

- 일회성 상품은 항상 기본 커미션 설정을 따릅니다.

- 반복 상품은 커스터마이즈된 커미션 요율을 설정할 수 있습니다. 상품별 커미션을 구성하려면 첫 번째 단계에서 상품을 추가하기만 하면 됩니다.

- 상품을 커미션에서 제외하려면, 해당 상품을 추가하고 커미션을 0%로 설정하면 됩니다.
