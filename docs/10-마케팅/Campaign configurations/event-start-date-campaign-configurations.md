# 이벤트 시작 날짜 - 캠페인 설정

참고: 모든 계정에서 워크플로우(Workflows)가 출시되었으므로, 이제 트리거와 캠페인이 하던 모든 기능(그리고 그 이상!)을 하나의 빌더에서 처리할 수 있습니다! 워크플로우에 대해 자세히 알아보기


## 캠페인에서 이벤트 시작 날짜를 언제, 어떻게 사용할까요?

이벤트 시작 날짜 필드를 사용하면 특정 날짜와 시간을 기준으로 자동화를 구축할 수 있습니다. 이벤트 시작 날짜를 설정하면, 해당 캠페인의 모든 항목이 그 날짜/시간을 기준으로 실행됩니다.

예를 들어, 1월 20일에 웨비나를 진행한다면 이벤트 시작 날짜를 1월 20일 오후 2시로 설정하고, 캠페인에 이메일과 문자 알림을 추가할 수 있습니다. 이때 각 항목의 타이밍을 "1일 전", "1시간 전" 등으로 설정하면 전날과 1시간 전에 자동으로 알림이 발송됩니다.
