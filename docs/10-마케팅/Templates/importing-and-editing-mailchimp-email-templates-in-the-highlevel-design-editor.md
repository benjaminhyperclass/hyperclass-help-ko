# **메일침프 이메일 템플릿 가져오기란**

메일침프 이메일 템플릿을 가져오면 기존에 디자인했던 메일침프 레이아웃을 하이퍼클래스의 디자인 에디터로 옮겨올 수 있습니다. 처음부터 새로 만들 필요 없이 기존 메일침프 디자인을 재사용하고, 하이퍼클래스의 드래그 앤 드롭 인터페이스에서 시각적으로 편집할 수 있습니다.

템플릿을 가져오면 하이퍼클래스가 자동으로 메일침프 디자인의 구조(헤더, 이미지, 텍스트 블록 등)를 인식하여 편집 가능한 구성 요소로 변환합니다. 이를 통해 코드를 작성하거나 편집할 필요 없이 콘텐츠를 쉽게 커스터마이징하고, 레이아웃을 조정하며, 브랜드 일관성을 유지할 수 있습니다.

---

## 메일침프 이메일 템플릿 가져오기의 주요 장점

- **원활한 템플릿 재사용:** 처음부터 다시 만들 필요 없이 기존 메일침프 이메일 디자인을 하이퍼클래스로 손쉽게 옮겨올 수 있습니다.

- **시각적 노코드 편집:** 코딩 없이 하이퍼클래스의 직관적인 드래그 앤 드롭 디자인 에디터를 사용해 이메일을 수정하고 개인화할 수 있습니다.

- **플랫폼 간 일관된 브랜딩:** 메일침프 템플릿에서 사용했던 친숙한 레이아웃, 폰트, 컬러 스킴을 재사용하여 브랜드의 룩앤필을 유지할 수 있습니다.

- **빠른 캠페인 제작:** 미리 디자인된 템플릿을 편집 가능한 하이퍼클래스 버전으로 변환하여 빠른 커스터마이징과 런칭을 위한 준비 시간을 절약할 수 있습니다.

## 디자인 에디터에서 메일침프 이메일 템플릿 가져오고 커스터마이징하기

### 이메일 템플릿 섹션 접근하기

좌측 네비게이션 메뉴에서 **Marketing(마케팅) → Emails(이메일) → Templates(템플릿)**으로 이동하세요. 이 섹션에서는 기존 이메일 템플릿 목록을 확인하고 다양한 방법으로 새 템플릿을 만들 수 있습니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155056894975/original/OYe3DEtySJVLcWGjC7Gu9wK-Msk5VizXuQ.png?1761563083)

### 새 가져오기 시작하기

우측 상단의 **New(새로 만들기)** 버튼을 클릭하고 드롭다운 메뉴에서 **Import Email(이메일 가져오기)**을 선택하세요. 이 옵션을 통해 메일침프 같은 외부 서비스의 이메일 템플릿을 가져올 수 있습니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155056895037/original/A2k50IEVFH5Y2q-sc0cJnfolgtI1Un3B7g.png?1761563100)

### 메일침프를 가져오기 소스로 선택하기

Import Email Templates 팝업에서 **Import from(가져올 소스)** 드롭다운을 열고 **Mailchimp**를 선택하세요. 이렇게 하면 하이퍼클래스가 메일침프 계정에서 캠페인 레이아웃을 가져올 준비를 합니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155056895099/original/b_cJmap4szvRsDbQchnUGKjj-8QtICsIYA.png?1761563119)

### 메일침프 캠페인 URL 추가하기

메일침프 계정에서 캠페인 URL을 복사하여 **URL** 필드에 붙여넣으세요. 이 URL을 통해 하이퍼클래스가 메일침프 캠페인의 구조를 가져와서 편집 가능하도록 재구성합니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155056895207/original/Gt4XvZflpbNcACydm3BdiF4zxfFg_BZSWg.png?1761563140)

### 템플릿 이름 입력하기

**Template Name(템플릿 이름)** 필드에 가져온 템플릿의 명확하고 알아보기 쉬운 이름을 입력하세요. 나중에 템플릿 목록에서 쉽게 찾고 편집할 수 있습니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155056895266/original/mpD3jeZh7NBr7hb1ozeXqTWXcoH8Bsb26Q.png?1761563170)

### 디자인 에디터에서 열기 및 템플릿 생성하기

**Open in Design Editor(디자인 에디터에서 열기)**를 선택한 후 **Create Template(템플릿 생성)**을 클릭하세요. 메일침프 템플릿이 하이퍼클래스의 시각적 디자인 에디터에서 열리며, 드래그 앤 드롭 커스터마이징과 콘텐츠 업데이트가 가능합니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155056895332/original/BirT1-2OtP4blMKq4t3nw5Hn8dBz-7dI5g.png?1761563193)

## **자주 묻는 질문**

**Q: 가져온 메일침프 템플릿이 디자인 에디터에서 올바르게 표시되지 않으면 어떻게 하나요?**

가져온 템플릿이 정렬이 맞지 않거나 섹션이 누락된 경우, 보통 메일침프 고유의 HTML 구조가 디자인 에디터의 블록 기반 레이아웃과 완전히 호환되지 않기 때문입니다. 이런 경우 **Code Editor(코드 에디터)**로 전환하여 HTML을 검토하고 수동으로 조정할 수 있습니다. 최상의 결과를 위해서는 가져오기 전에 메일침프 캠페인이 표준 HTML과 인라인 CSS를 사용하는지 확인하세요.

**Q: 동적 콘텐츠, 머지 태그, 조건부 로직을 사용하는 메일침프 템플릿도 가져올 수 있나요?**

메일침프 고유의 머지 태그와 동적 콘텐츠 규칙은 가져오기 과정에서 하이퍼클래스의 해당 기능으로 자동 변환되지 않습니다. 구조는 가져와지지만, 동적 로직은 템플릿이 디자인 에디터에 들어온 후 하이퍼클래스의 **Custom Values(커스텀 값)** 또는 **Dynamic Fields(동적 필드)**를 사용해서 수동으로 다시 설정해야 합니다.

**Q: 가져오기 도구가 제 메일침프 계정 자격 증명을 저장하거나 접근하나요?**

아니요. 가져오기 기능은 하이퍼클래스에 붙여넣는 **공개 캠페인 URL**만을 사용합니다. 메일침프 자격 증명이나 API 키를 요구하거나 저장하지 않습니다. 시스템은 제공된 URL의 HTML 레이아웃만을 읽어서 디자인 에디터 내에서 이메일 디자인을 재구성합니다.

**Q: 메일침프 "캠페인" 가져오기와 "템플릿" 가져오기의 차이점은 무엇인가요?**

메일침프 캠페인은 캠페인 URL을 사용해서 직접 가져올 수 있습니다. 하지만 저장된 메일침프 **템플릿**을 가져오려면 먼저 메일침프에서 해당 템플릿을 사용해 캠페인을 만든 다음 그 캠페인 URL을 복사해야 합니다. 하이퍼클래스 가져오기 도구가 처리할 수 있는 공유 가능한 URL 구조는 캠페인에만 있습니다.
