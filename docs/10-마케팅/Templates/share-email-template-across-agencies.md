# 에이전시 간 이메일 템플릿 공유하기

이 기능을 사용하면 에이전시 관리자가 간단한 링크를 통해 이메일 템플릿을 쉽게 공유하고 관리할 수 있습니다. 아래 가이드를 따라 템플릿을 공유하고 성과 좋은 이메일 생태계를 구축하는 방법을 알아보세요.

## 주요 장점

- **모범 사례 전파**: 에이전시 전체에서 템플릿을 쉽게 공유하고 표준화할 수 있어요.
- **디자인 편집기 직접 접근**: HTML 파일을 다운로드할 필요 없이 편집 가능한 템플릿으로 바로 작업할 수 있어요.

## 사용 방법

1. **Email Marketing(이메일 마케팅)** 하위의 **Templates(템플릿)**으로 이동하세요.

2. 템플릿 옆의 점 3개 버튼을 클릭하고 **Share(공유)**를 선택하세요.

![템플릿 공유 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155030228688/original/WdX4YiKt3jlfQLT9RSfvZRN2spU9wRVOtg.jpeg?1722428458)

3. 접근 권한을 에이전시 내 누구나 또는 링크를 가진 사람으로 설정하세요.

![접근 권한 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155030228686/original/uMI3LAgMH_0IEy2WQP3tVuGFN7THWxFBWg.jpeg?1722428458)

4. 이메일이나 다른 채널을 통해 링크를 공유하세요.

5. 브라우저에 링크를 붙여넣고 **Add Template(템플릿 추가)**를 클릭하여 공유된 템플릿의 복사본을 저장하세요.

![템플릿 추가 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155030228687/original/ZSiRGuZ8cRua8bbQwVsEmC1gRtHpSMJQEw.jpeg?1722428458)

6. 템플릿 이름을 편집하고, 하위 계정을 선택한 후 **Clone template(템플릿 복제)**를 클릭하세요.

![템플릿 복제 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155030228689/original/o6sPNmNdhBFdEs9erikBKNw2aZpPoHby4Q.jpeg?1722428458)

최대 3개의 로케이션을 선택할 수 있습니다.

![로케이션 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155030228690/original/KVfkySufccYkcnqNYSO8niO4c20P6Usmbg.jpeg?1722428458)

7. 첫 번째로 선택한 로케이션의 템플릿 섹션으로 리디렉션됩니다. 공유된 템플릿이 이제 목록에 포함되어 있을 거예요.

**참고**: 에이전시 관리자/사용자만 템플릿을 공유할 수 있습니다. 에이전시 관리자, 에이전시 사용자, 계정...
