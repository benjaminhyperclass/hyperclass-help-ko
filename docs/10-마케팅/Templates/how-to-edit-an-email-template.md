# 이메일 템플릿 편집하는 방법

**이메일 템플릿**은 이메일을 보낼 때 여러 번 재사용할 수 있는 이메일 레이아웃이나 디자인입니다.

매번 이메일을 처음부터 만드는 대신, 이미 다음과 같은 요소들이 포함된 템플릿을 사용할 수 있어요:

- 미리 디자인된 레이아웃과 구조
- 공통 콘텐츠 (예: 헤더, 푸터, 브랜딩)
- 색상, 폰트, 서식 등의 스타일링

템플릿 라이브러리에서 템플릿을 선택하여 필요에 맞게 커스터마이징할 수도 있습니다.

**목차**

- [유용한 팁](#유용한-팁)

## 템플릿 편집 단계

1. **이메일 마케팅으로 이동**

- 왼쪽 메뉴에서 **Marketing(마케팅)**으로 이동해요
- Email Marketing을 클릭합니다

![이메일 마케팅 메뉴 접근](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155068520334/original/McLrM7QGUwWWYFiV14vbJyGgpJ9or9q6DQ.png?1775472219)

2. **Templates(템플릿) 열기**

- **Templates(템플릿)**를 클릭해요

3. **템플릿 선택**

- 편집하고 싶은 템플릿을 찾아요
- 템플릿을 더블클릭하거나

4. **변경사항 적용**

- 필요에 따라 콘텐츠, 레이아웃, 디자인을 업데이트해요

5. **변경사항 저장**

- 완료되면 업데이트된 템플릿을 반드시 **저장**하거나 자동저장이 활성화되어 있는지 확인해요

## 유용한 팁

- 저장하기 전에 템플릿을 검토하여 서식이 올바르게 보이는지 확인해요

- 캠페인에 사용하기 전에 필요하면 이메일을 테스트해보세요

- 커스텀 값(Custom Value)을 올바르게 사용하고 있는지 확인해요
