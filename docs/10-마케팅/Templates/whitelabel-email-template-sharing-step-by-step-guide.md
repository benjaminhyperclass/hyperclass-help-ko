# 화이트라벨 이메일 템플릿 공유: 단계별 가이드

하위 계정 사용자들이 템플릿을 자신의 계정으로 직접 가져올 수 있는 템플릿 공유 링크를 만들 수 있습니다. 간단한 조정을 통해 이 링크들을 화이트라벨 도메인으로 포맷하여 에이전시 브랜드를 완전히 유지할 수 있어요.

## 템플릿 공유 링크 생성 단계

### **1. 템플릿 공유 링크 생성하기**

- `Marketing(마케팅) > Emails(이메일) > Templates(템플릿)`으로 이동하세요.
- 공유하려는 템플릿 옆의 세 개 점을 클릭하세요.
- `Share(공유)`를 선택하세요.
- 생성된 공유 링크를 복사하세요. 다음과 같이 표시됩니다:

```
https://affiliates.gohighlevel.com/?fp_ref=xxxx&email_template_share=6438b5b145411d0ce4332f1b
```

### **2. 공유 ID 복사하기**

링크를 붙여넣고 `template_share=` 뒤에 있는 부분을 찾으세요.

**참고**: 공유 기능은 계정/로케이션 사용자 역할에서는 작동하지 않습니다. 에이전시 소유자/관리자, 에이전시 사용자, 계정 관리자만 접근할 수 있어요.

예시:
```
email_template_share=6438b5b145411d0ce4332f1b
```

여기서 공유 ID는:
```
6438b5b145411d0ce4332f1b
```

### **3. 템플릿에 공유 ID 삽입하기**

커스텀 공유 링크를 만들려면 이 템플릿을 사용하세요:

```
https://app.YOURDOMAIN.com/emails/share/SHAREID
```

- `YOURDOMAIN.com`을 화이트라벨 도메인으로 바꿔주세요.
- `SHAREID`를 복사한 공유 ID로 바꿔주세요.

**예시:**
```
https://app.whitelabel.com/emails/share/6438b5b145411d0ce4332f1b
```

### **4. 커스텀 링크 공유하기**

이제 이 링크를 하위 계정 사용자들과 공유할 수 있어요. 사용자들은 화이트라벨 도메인을 통해 템플릿을 자신의 계정으로 직접 가져올 수 있습니다.
