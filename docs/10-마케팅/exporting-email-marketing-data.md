# 이메일 마케팅 데이터 내보내기

Hyperclass의 이메일 마케팅 데이터 내보내기는 이메일 통계(Email Statistics) 대시보드에서 이메일 성과 지표를 CSV 파일로 다운로드할 수 있게 해줍니다. 캠페인 성과 분석, 관계자와의 결과 공유, 플랫폼 외부에서의 보고서 기록 유지가 더욱 쉬워집니다. 캠페인, 워크플로우, 일괄 이메일 작업의 통계를 내보낼 수 있으며, 다운로드 링크는 이메일로 안전하게 전달됩니다.

# 이메일 마케팅 데이터 내보내기란?

이메일 마케팅 데이터 내보내기를 통해 Hyperclass의 이메일 통계(Email Statistics) 대시보드에서 이메일 성과 통계를 CSV 형식으로 다운로드할 수 있습니다. 팀이 캠페인 성과 지표를 추출해서 스프레드시트 도구나 외부 보고 시스템에서 분석할 수 있어 보고서 작성이 간소화됩니다.

## 이메일 마케팅 데이터 내보내기의 주요 장점

이메일 통계 내보내기를 통해 팀이 성과를 더 효율적으로 분석하고 마케팅, 운영, 고객 보고 워크플로우 전반에서 인사이트를 공유할 수 있습니다.

- **중앙화된 보고**: 여러 캠페인 뷰에서 수동으로 지표를 수집하는 대신 통합된 이메일 통계(Email Statistics) 대시보드에서 바로 이메일 성과 데이터를 다운로드할 수 있습니다.

- **유연한 내보내기 옵션**: 모든 이메일 활동을 내보내거나 캠페인, 워크플로우, 일괄 발송 등 특정 소스에만 집중할 수 있습니다.

- **간편한 성과 분석**: CSV 내보내기는 Excel이나 Google Sheets 같은 스프레드시트 도구에서 열어 심층 분석과 커스텀 보고서 작성이 가능합니다.

- **쉬운 공유와 협업**: 이메일로 전달되는 다운로드 링크로 팀원이나 고객에게 보고서를 간단히 배포할 수 있습니다.

- **보안 파일 전달**: 내보내기 다운로드 링크는 30일간 유효하며, 보고 데이터에 안전하고 시간 제한된 접근을 제공합니다.

- **내보내기 가시성**: 최근 다운로드 영역에서 내보내기 활동을 추적하고 파일이 언제 생성되었는지 확인할 수 있습니다.

## 이메일 성과 내보내기 유형

Hyperclass는 여러 내보내기 범위를 지원하여 보고나 분석에 필요한 정확한 이메일 통계 세트를 다운로드할 수 있습니다. 각 내보내기 유형은 이메일 통계(Email Statistics) 대시보드에서 추적되는 이메일 활동에 해당합니다.

| **내보내기 유형** | **설명** |
|---|---|
| **전체 캠페인** | 캠페인, 워크플로우, 일괄 발송을 포함하여 이메일 통계에서 볼 수 있는 모든 이메일 활동의 통계를 내보냅니다. |
| **이메일 마케팅 캠페인만** | **마케팅(Marketing) → 이메일(Emails) → 캠페인(Campaigns)**에서 생성된 이메일 캠페인의 성과 지표를 내보냅니다. |
| **워크플로우만 (개별 이메일 액션 수준)** | 워크플로우 내 개별 이메일 액션의 성과 데이터를 내보냅니다. |
| **일괄 작업만** | **연락처(Contacts) → 일괄 작업(Bulk Actions)**을 통해 발송된 이메일의 통계를 내보냅니다. |

## 최근 내보내기

최근 내보내기 영역은 이메일 통계(Email Statistics) 대시보드에서 생성된 내보내기 파일을 추적하는 중앙화된 공간을 제공합니다. 팀이 내보내기가 성공적으로 생성되었는지 확인하고 사용자가 해당 로케이션 내 보고 활동을 검토할 수 있게 해줍니다.

최근 내보내기는 최대 30일간 내보내기 기록을 저장하여 최근 보고 요청에 대한 가시성을 제공하고 팀이 보고 작업을 더 쉽게 조율할 수 있게 합니다.

최근 내보내기에 일반적으로 표시되는 정보:

- 내보내기 파일 이름 또는 설명
- 내보내기 유형 (캠페인, 워크플로우, 일괄 작업 등)
- 내보내기를 요청한 사용자
- 내보내기가 생성된 날짜와 시간
- 내보내기 상태 (예: 완료)

## 이메일 마케팅 데이터 내보내기 설정 방법

해당 로케이션에 이메일 통계(Email Statistics)가 사용 가능하면 이메일 성과 데이터 내보내기에는 추가 구성이 필요하지 않습니다. 사용자는 이메일 통계 대시보드에서 바로 내보내기를 생성하고 이메일로 다운로드 링크를 받을 수 있습니다.

- **마케팅(Marketing) → 이메일(Emails) → 통계(Statistics)**로 이동하세요. 캠페인 유형을 선택하고 내보내기(Export)를 클릭하세요.

- 캠페인 유형을 선택하고 내보내기(Export)를 클릭하세요.

![캠페인 유형 선택 및 내보내기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155066758796/original/IToHWbUmLmdyRg5H69j52mHL3VVVCOG1bg.png?1773305701)

- 파일을 받을 이메일 주소를 추가하고 진행하세요.

![이메일 주소 입력](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155066764170/original/nBLIqySj1e_-YSaHaPCgDjqcMFGGnF_hFw.png?1773308249)

- 내보내기된 CSV 파일의 다운로드 링크가 포함된 메시지를 이메일 인박스에서 확인하세요.

## 자주 묻는 질문

**Q: 내보내기에 어떤 지표들이 포함되나요?**
A: 내보내기에는 배송, 열람, 클릭 등 이메일 통계(Email Statistics)에서 제공하는 핵심 이메일 성과 지표가 포함됩니다.

**Q: 내보내기된 파일을 어떻게 받나요?**
A: Hyperclass가 내보내기 요청 생성 시 제공한 이메일 주소로 CSV 다운로드 링크를 보냅니다.

**Q: 내보내기 다운로드 링크는 얼마나 오래 유효한가요?**
A: 다운로드 링크는 내보내기 생성 후 30일간 유효합니다.

**Q: 워크플로우 이메일 성과도 내보낼 수 있나요?**
A: 네. 워크플로우 내보내기는 워크플로우 내 개별 이메일 액션 수준의 성과 데이터를 제공합니다.

**Q: 과거 내보내기는 어디서 볼 수 있나요?**
A: 내보내기 히스토리는 **설정(Settings) → 최근 내보내기(Recent Exports)**에서 확인할 수 있으며, 지난 30일간 생성된 내보내기를 보여줍니다.

**Q: 이메일 데이터 내보내기가 캠페인이나 연락처에 영향을 주나요?**
A: 아니요. 데이터 내보내기는 보고 작업일 뿐이며 캠페인, 워크플로우, 연락처, 통계를 수정하지 않습니다.
