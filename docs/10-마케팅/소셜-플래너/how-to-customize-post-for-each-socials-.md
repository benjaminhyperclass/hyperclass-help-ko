# 각 소셜 미디어별 게시물 맞춤 설정 방법

새로운 '각 게시물별 맞춤 설정' 기능으로 소셜 미디어 마케팅을 혁신하세요. 플랫폼별로 콘텐츠를 맞춤화하여 글자수 제한과 이미지/동영상 형식 장벽을 쉽게 극복하고, 다양한 캠페인을 한 곳에서 관리하세요. 간단한 토글 하나로 각 소셜 네트워크에 맞는 고유한 메시지를 만드는 뛰어난 유연성을 경험해보세요.

## **각 게시물별 맞춤 설정 기능이란?**

'각 게시물별 맞춤 설정' 기능은 마케터, 비즈니스, 개인이 서로 다른 소셜 미디어 채널에 맞춤형 콘텐츠를 만들고 관리할 수 있게 해주는 혁신적인 도구입니다. 이 기능이 제공하는 모든 기능을 자세히 알아보세요:

**멀티 플랫폼 호환성:** 이 기능을 사용하면 여러 소셜 미디어 플랫폼(Facebook, LinkedIn 등)을 선택하고 각 플랫폼의 특정 글자수 제한과 요구 사항에 맞게 콘텐츠를 맞춤화할 수 있습니다.

**콘텐츠 맞춤화:** 다양한 소셜 미디어 채널 간의 글자수 제한 차이를 인식하여, 이 기능은 개별화된 콘텐츠 작성을 가능하게 합니다. LinkedIn용 짧은 메시지든 Facebook용 긴 게시물이든, 동일한 인터페이스 내에서 모두 만들 수 있습니다.

**토글 스위치:** 간단한 토글 스위치를 통해 사용자는 서로 다른 채널에 대한 맞춤 콘텐츠와 선택된 모든 플랫폼에 적용되는 표준 콘텐츠 간을 전환할 수 있습니다. 이는 콘텐츠 제작과 전달에서 유연성을 보장합니다.

**AI 통합:** 콘텐츠 AI(Content AI)와의 통합을 통해 사용자는 인공지능을 활용하여 최대 900단어의 콘텐츠를 생성하고, 사용하는 소셜 미디어 유형에 따라 다양한 콘텐츠 크기(소, 중, 대)에 맞게 조정할 수 있습니다.

**예약 기능:** 맞춤 게시물을 만들 수 있을 뿐만 아니라 나중에 게시하도록 예약할 수도 있어 소셜 미디어 관리 프로세스를 자동화할 수 있습니다.

**비용 효율성과 시간 절약:** 이 기능은 서로 다른 플랫폼에 대해 별도의 게시물을 만드는 노력을 최소화하여 시간과 리소스를 절약하고 일관성 있으면서도 맞춤화된 마케팅 접근 방식을 보장합니다.

## **이 기능을 사용하는 방법은?**

**마케팅 섹션으로 이동:**

- 사이드바 메뉴에서 **"마케팅(Marketing)"** 탭을 선택합니다.
- **"소셜 플래너(Social Planner)"** 옵션을 클릭합니다.
- 새 게시물 만들기를 클릭합니다:
![새 게시물 만들기 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155005176260/original/DyIootGhs_Fe0VM2ODyR-LvWk8jgZx6ktQ.png?1692034961)

**"새 게시물 만들기(Create New Post)"** 버튼을 클릭합니다.

![새 게시물 만들기 인터페이스](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155005176564/original/1yhYsF-j8LZBLpCRIiC3DFx8gY6ZUJXcUg.png?1692035255)

**각 채널별 맞춤 설정 선택:**

맞춤 설정 옵션이 나타나게 하려면 Facebook, Instagram, LinkedIn, GMB(구글 마이 비즈니스) 등과 같은 여러 소셜 미디어 채널을 선택해야 합니다.
![소셜 미디어 채널 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155005176611/original/uD1WdXkMBcdAAioCchZ9KI4iXPrdpzvs_g.png?1692035350)

게시할 소셜 미디어 채널을 선택하면 각 채널에 대한 맞춤 설정을 "토글 켜기" 할 수 있는 옵션이 표시됩니다.

![각 채널별 맞춤 설정 토글](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048869856/original/ptDSqDDA0-lugx9llLeqsJPma_8UINGDRw.png?1750863431)

각 채널별 맞춤 설정

이 토글을 켜면 각 플랫폼의 개별 글자수 제한과 이미지/동영상과 같은 미디어 형식을 고려하여 각 플랫폼에 고유한 콘텐츠를 만들 수 있습니다.

![채널별 맞춤 콘텐츠 작성](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155064535966/original/1Wy6TXP3j0O6X3fe3gZXfd-e_5_oMXRVhA.png?1770691284)

**맞춤 콘텐츠 작성:**

각 플랫폼의 특정 글자수 제한(예: Facebook 5,000자, GMB 1,500자 등)을 준수하면서 각 채널에 대한 콘텐츠를 작성합니다.

### **각 소셜 미디어 채널의 글자수 제한**

다음은 '각 게시물별 맞춤 설정' 기능을 사용할 때 중요한 요소인 다양한 소셜 미디어 채널의 글자수 제한을 보여주는 표입니다:

| 채널 | 글자수 제한 |
|------|-------------|
| Facebook 페이지 및 그룹 | 5,000자 |
| Instagram 프로필 | 2,200자 |
| LinkedIn 페이지 및 프로필 | 3,000자 |
| GMB (구글 마이 비즈니스) | 1,500자 |

각 게시물이 플랫폼의 글자수 제한과 오디언스 선호도에 맞도록 서로 다른 채널의 콘텐츠를 별도로 보고 편집할 수 있습니다.

![채널별 콘텐츠 편집 화면1](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155005176906/original/Hc6I1UBbhapeFzgazVj1Cfl0zRTe5p6WFA.png?1692035608)

![채널별 콘텐츠 편집 화면2](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155005176960/original/5IbS881B_zhWOcA6jL9yt2K-K-aHE71Sxw.png?1692035717)

미디어 추가(이미지/동영상): [파일 크기 가이드라인 참조](posting-content-image-video-and-text-guidelines.md)

각 소셜 미디어에 적용 가능한 경우 이미지, 동영상 또는 기타 멀티미디어 요소를 추가하여 게시물을 향상시킬 수 있습니다.

![미디어 업로드 화면1](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048870075/original/_y71MZip2uZ8eWJwiiyGt4iy3jv8be7M4g.png?1750863597)

![미디어 업로드 화면2](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048870095/original/6mW5SrJoH7NcON2qG_e4_d7NXVMXkPuPVw.png?1750863612)

참고사항:

- 미디어가 여전히 업로드 중인 경우 소셜 계정을 전환하면 진행 중인 업로드가 취소될 수 있습니다.

- 미디어가 여전히 업로드 중인 경우 '각 게시물별 맞춤 설정' 토글을 전환하면 진행 중인 업로드가 제거될 수 있습니다.

- 동영상 썸네일이 처리 중인 경우 소셜 계정이나 '각 게시물별 맞춤 설정' 토글을 전환하면 썸네일 처리가 중단될 수 있습니다. 썸네일을 다시 적용해야 할 수 있습니다.

**콘텐츠 AI 활용 (선택사항):**

콘텐츠 제작에 AI를 활용하고 싶다면 이 단계에서 새로운 텍스트를 생성하거나 소셜 미디어 유형에 따라 콘텐츠 크기를 조정할 수 있습니다.

![콘텐츠 AI 사용 화면1](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155005177049/original/4YnRYABTyjSflTUNTFwCTIhQ7l7fOgp7GA.png?1692035835)

![콘텐츠 AI 사용 화면2](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155005177080/original/Ap_qV0C47UudDnSGpOCti216VeTFlQtsBA.png?1692035892)

**바로 게시하거나 예약하기:**

각 채널에 대한 콘텐츠를 작성했으면 즉시 게시하거나 나중에 게시하도록 예약할 수 있습니다.

플랫폼 전반에 걸친 정확성과 일관성을 보장하기 위해 모든 세부 사항을 확인하고 검토하세요.

**참고사항:**

맞춤 설정 토글을 끄면 활성 소셜 콘텐츠가 선택된 모든 소셜 채널에 적용되며, 맞춤화된 콘텐츠가 손실될 것이라는 경고가 표시됩니다.
![토글 끄기 경고](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155005177120/original/-2ntwlxNMYpzQZYt2ekAmKg8YdNiz6tViQ.png?1692035967)

이 단계를 따르면 '각 게시물별 맞춤 설정' 기능을 활용하여 서로 다른 소셜 미디어 채널에 개인화되고 플랫폼별 특성에 맞는 콘텐츠를 만들어 도달률과 참여도를 극대화할 수 있습니다. 이는 다양한 캠페인을 관리하고 각 플랫폼의 고유한 오디언스에 공감할 수 있는 메시지를 맞춤화하는 효율적인 방법입니다.

## **자주 묻는 질문**

### **Q: 특정 플랫폼의 글자수 제한을 초과하면 어떻게 되나요?**

A: 각 플랫폼의 글자수 제한에 맞게 콘텐츠를 수정해야 합니다. 이 기능은 일반적으로 콘텐츠가 허용된 글자수를 초과하는 경우 경고나 표시기를 제공합니다.
