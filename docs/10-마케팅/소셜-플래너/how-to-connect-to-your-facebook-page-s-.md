# 페이스북 페이지 연결 방법

**소셜 플래너(Social Planner)**를 사용하면 모든 팬 페이지를 추가하여 게시물을 발행하고 페이스북 페이지를 통해 페이스북 오디언스와 소통할 수 있습니다.

---

# **페이스북 페이지 게시 설정 가이드**

아래 단계를 따라하시면 페이스북 페이지를 플랫폼에 연결할 수 있습니다.

1. 이전에 어떤 프로필에서든 페이스북 페이지를 연결한 적이 있고 새로운 페이지를 추가하려는 경우, 페이스북 계정의 Settings > Business Integration 섹션에서 애플리케이션을 '제거(Remove)'한 후 아래 단계를 처음부터 다시 따라하셔야 합니다.

2. 모든 페이스북 페이지를 추가하려면 **관리자 권한**이 있어야 합니다. 해당 페이지들의 관리자 권한을 가진 동일한 프로필로 로그인해야 합니다.

페이스북의 최근 API 변경으로 인해 더 이상 페이스북 개인 프로필에 게시할 수 없지만, 소셜 플래너에서는 다른 프로필의 페이지들을 추가할 수 있습니다.

### 1단계. 소셜 플래너 > 설정(Setting)으로 이동

시작하려면 소셜 플래너(Social Planner)로 이동하세요. 소셜 플래너 우측 상단에 설정(Setting) 옵션(톱니바퀴 아이콘)이 있습니다. 이를 클릭하여 설정으로 이동하세요.

![소셜 플래너 설정 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48194300133/original/2DCAFJ87Mo4PnLGFM7pXBldH73B69XvWPg.png?1645733452)

## 2단계. 새 페이지 연결, 페이스북 페이지 클릭

다음 단계는 아래 그림과 같이 표에서 **새 페이스북 페이지 연결(Connect a new Facebook page)**을 클릭하여 여러 페이스북 페이지를 관리하는 것입니다.

![페이스북 페이지 연결 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155025796453/original/8Ve762oPwJEuCatw0aVVNqL48-ubhHfJMg.png?1715280065)

## 3단계. 페이스북 페이지 가져오기 권한 허용

프로필을 처음 연결하는 경우, 모든 권한을 허용해야 합니다. 페이스북 페이지 가져오기 승인 과정을 완료하려면 Continue를 클릭하세요.

![페이스북 권한 허용 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48194300294/original/M5aicg2aO0Jt7HmivVYhOCtO7ICrCZ87jQ.png?1645733508)

## 4단계. 연결할 페이스북 페이지 선택

이 프로필의 모든 페이스북 페이지가 나열됩니다. 소셜 플래너에 연결하고 싶은 페이지들을 선택하세요.

![페이스북 페이지 선택 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48194300386/original/idLMJiroiLrnkf5XRoC2wWENqdOKYQrWmg.png?1645733550)

## 5단계. 연결 확인

마지막으로, 이전 단계에서 선택한 모든 페이지가 나열되며 연결 성공을 확인할 수 있습니다.

이 지시사항을 따라하시면 동일한 페이스북 프로필이나 다른 프로필에서 원하는 만큼 많은 페이지를 추가할 수 있습니다.
