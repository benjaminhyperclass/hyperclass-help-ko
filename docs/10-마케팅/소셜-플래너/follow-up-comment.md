# 후속 댓글

대부분의 소셜 플랫폼은 게시물의 댓글, 특히 게시물이 처음 발행된 직후에 달리는 댓글에 최적화된 인게이지먼트 알고리즘을 가지고 있습니다.

"후속 댓글(Follow up Comment)" 기능은 다음 소셜 플랫폼에서 작동합니다:

- 페이스북 페이지
- 페이스북 그룹 (댓글은 소셜 플래너(Social Planner) 설정에서 선택한 페이스북 페이지로 표시됩니다)
- 인스타그램 비즈니스 계정
- 링크드인 프로필
- 링크드인 페이지
- X (구 트위터) 계정

*아래 화면에 접근하려면, "Marketing(마케팅)" > "Social Planner(소셜 플래너)" > "Add Post(게시물 추가)" > "Create New Post(새 게시물 만들기)"로 이동하세요*
