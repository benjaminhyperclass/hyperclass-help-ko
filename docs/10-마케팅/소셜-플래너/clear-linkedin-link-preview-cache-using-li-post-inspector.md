# LinkedIn Post Inspector로 링크 미리보기 캐시 초기화하기

LinkedIn에서 웹사이트 링크를 공유했는데 링크 미리보기의 제목, 설명 또는 이미지가 잘못 표시되나요? 웹사이트를 완벽한 미리보기 콘텐츠와 이미지로 업데이트한 후에도 링크를 공유하면 여전히 이전 미리보기 콘텐츠와 이미지가 표시될 수 있습니다. 이는 LinkedIn이 링크 미리보기 콘텐츠를 7일간 캐시하기 때문입니다.

**해결 방법:**

LinkedIn에서 Post Inspector를 출시했습니다. LinkedIn 미리보기 캐시를 초기화하려면 다음 단계를 따라하세요:

**1단계:** [https://www.linkedin.com/post-inspector/](https://www.linkedin.com/post-inspector/)에 접속하세요

**2단계:** URL을 입력하고 Inspect를 클릭하세요. 업데이트된 미리보기 이미지를 확인할 수 있습니다

**3단계:** 이제 LinkedIn에서 URL을 공유해보세요

도움이 되었기를 바랍니다!
