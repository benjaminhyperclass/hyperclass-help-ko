# 소셜 예약을 위한 이미지 자동 최적화 방법

개요:

소셜 플래너(Social Planner) 경험을 향상시키는 새로운 기능인 미디어 최적화를 출시하게 되어 기쁩니다. 이 기능은 여러분의 이미지가 항상 원활하고 오류 없는 콘텐츠 예약에 준비되도록 보장합니다. 번거로운 수동 조정은 이제 안녕, 간편한 최적화를 만나보세요!

주요 장점

- **자동 이미지 크기 조정**: 이미지가 플랫폼별 요구사항에 맞게 즉시 조정됩니다. 수동 크기 조정이 필요 없어요.

- **향상된 품질**: 품질 저하 없이 이미지의 시각적 매력을 유지합니다.

- **향상된 호환성**: 자동으로 포맷된 이미지는 다양한 소셜 미디어 플랫폼의 기술적 요구사항을 충족합니다.

- **빠른 업로드**: 최적화된 이미지는 파일 크기가 가벼워져 업로드 시간이 단축되어 원활한 예약 경험을 제공합니다.

- **오류 방지**: 잘못된 치수, 큰 파일 크기, 지원하지 않는 포맷 등의 일반적인 문제가 자동으로 수정됩니다.

## 최적화 항목

다음을 기반으로 플랫폼별 문제를 해결합니다:

- 포맷
- 크기
- 가로세로 비율
- 치수

**이 기능으로 수정되는 플랫폼별 오류:**

**페이스북 게시물:**
- 페이스북은 JPEG, PNG, GIF 이미지 포맷을 지원합니다. 현재 이미지 포맷은 {format}입니다.
- 페이스북은 최대 10MB까지 이미지를 지원합니다. 현재 이미지는 {size} MB입니다.

**페이스북 스토리:**
- 페이스북 스토리는 JPEG, PNG, GIF 이미지 포맷을 지원합니다. 현재 이미지 포맷은 {format}입니다.
- 페이스북 스토리는 최대 4MB까지 이미지를 지원합니다. 현재 이미지는 {size} MB입니다.

**인스타그램 게시물:**
- 인스타그램은 JPEG, PNG, GIF 이미지 포맷을 지원합니다. 현재 이미지 포맷은 {format}입니다.
- 인스타그램은 최대 8MB까지 이미지를 지원합니다. 현재 이미지는 {size} MB입니다.
- 인스타그램 프로페셔널은 4:5와 1.91:1 사이의 가로세로 비율을 지원합니다. 현재 이미지는 {aspectRatio}입니다.

**인스타그램 스토리:**
- 인스타그램 스토리는 JPEG 이미지 포맷을 지원합니다. 현재 이미지 포맷은 {format}입니다.
- 인스타그램 스토리는 최대 8MB까지 이미지를 지원합니다. 현재 이미지는 {size} MB입니다.

**구글 마이 비즈니스(Google My Business):**
- 구글 마이 비즈니스는 JPEG, PNG 이미지 포맷을 지원합니다. 현재 이미지 포맷은 {format}입니다.

- 구글 마이 비즈니스는 최소 720X720, 최대 3000X3000 치수를 지원합니다. 현재 이미지 치수는 {width}x{height}입니다.

## 이것이 중요한 이유

소셜 콘텐츠를 계획할 때 이미지 관련 오류가 얼마나 짜증나는지 알고 있습니다. 이미지 최적화 기능으로 여러분은 매력적인 게시물 제작에 집중하고, 기술적인 세부사항은 저희가 처리해드려요.
