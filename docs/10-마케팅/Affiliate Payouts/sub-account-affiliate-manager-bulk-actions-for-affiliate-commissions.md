# **제휴 수수료 일괄 작업이란?**

제휴 수수료 일괄 작업을 사용하면 제휴사 프로필이나 메인 수수료 대시보드에서 직접 여러 수수료 기록을 한 번에 관리할 수 있습니다. 이를 통해 반복적인 개별 승인이나 삭제 작업을 할 필요가 없어지고, 소중한 시간을 절약할 수 있습니다.

---

## 제휴 수수료 일괄 작업의 주요 장점

새로운 일괄 작업 도구를 사용하여 더 적은 노력으로 수수료를 더 빠르게 관리하세요.

- 여러 수수료를 한 번에 승인, 거부 또는 삭제
- 제휴사 프로필과 메인 대시보드 모두에서 일괄 작업 접근
- 대량 수수료 검토나 지급 시 시간 절약
- 상태(승인됨, 대기 중, 거부됨) 또는 캠페인별 수수료 필터링
- 수동 관리 업무와 반복적인 클릭 작업 감소
- 팀과 에이전시의 제휴 관리 효율성 향상

---

## **제휴사 프로필에서 일괄 작업 사용하기**

일괄 작업은 수수료 관리 프로세스를 간소화하는 데 도움이 됩니다. 이 섹션에서는 정확한 단계를 안내합니다.

- 마케팅(Marketing) 섹션 내의 제휴 관리(Affiliate Manager)로 이동하세요.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048282254/original/y7jK-LFaCQjrM0JMz3kzCWCHbBESODU8hw.jpeg?1750051301)

- 드롭다운 메뉴에서 "Affiliate" 옵션을 찾아 모든 제휴사를 확인하고, 제휴사 중 하나를 클릭하세요.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048282479/original/lefMd1l8BaxsNIxEmyfOljQ7cc0gLozFyw.jpeg?1750051687)

- 제휴사 프로필에 들어가면 **수수료(Commissions)** 탭을 클릭하세요.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048282558/original/-tJ2_YKnljzI1Liq5yERcfYTdmty-m63_A.jpeg?1750051828)

- 체크박스를 사용하여 하나 이상의 수수료를 선택하세요.

- **일괄 작업(Bulk Actions)** 버튼을 클릭하세요.

- **승인(Approve)**, **거부(Deny)** 또는 **삭제(Delete)** 중에서 선택하세요.

---

## 대시보드에서 일괄 작업 사용하기

전체 수수료 대시보드에서도 동일한 일괄 작업을 수행할 수 있습니다. 모든 제휴사의 최신 활동을 검토하기에 이상적입니다.

- **제휴 관리 대시보드(Affiliate Manager Dashboard)**로 이동하세요.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048282706/original/OBsDIovl82cp-1nmMq5wZlFzWKWOPRSfqQ.png?1750052130)

- **최근 수수료(Latest Commissions)** 섹션으로 스크롤하세요.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048282715/original/_fMPs33uYQSXy-Cn9wustnduJ_eXhBQZJw.png?1750052162)

- 체크박스를 사용하여 여러 수수료를 선택하세요.

- **일괄 작업(Bulk Actions)** 버튼을 클릭하고 원하는 작업을 선택하세요.

## **상태 및 캠페인별 수수료 필터링**

필터를 사용하면 수수료를 빠르게 세분화하여 더 빠른 작업을 수행할 수 있습니다. 대규모 제휴 프로그램을 관리할 때 특히 유용합니다.

- **상태 필터(Status filter)**를 사용하여 대기 중, 승인됨 또는 거부된 수수료만 표시하세요.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048280219/original/Hp-r148sUSBLVSII31cw6p6rllbsG83row.png?1750046696)

- **캠페인 필터(Campaign filter)**를 사용하여 제휴 캠페인별로 수수료를 분리하세요.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048280270/original/MWF-Q5J04IApwdedLmpotODMHKmVxOFMtg.png?1750046864)

- 더욱 정확한 검색을 위해 필터를 결합하세요.

## **자주 묻는 질문**

**Q: 일괄 작업을 취소할 수 있나요?**

아니요. 일괄 작업이 제출되면 최종 결정됩니다. 확인하기 전에 선택 사항을 반드시 다시 한 번 확인하세요.

**Q: 모든 사용자 역할이 일괄 작업을 사용할 수 있나요?**

제휴 수수료를 관리할 권한이 있는 사용자만 일괄 작업을 수행할 수 있습니다.

**Q: 서로 다른 제휴사의 수수료에 일괄 작업을 한 번에 적용할 수 있나요?**

네, 대시보드 보기를 사용할 때 여러 제휴사의 수수료를 선택할 수 있습니다.

**Q: 이러한 일괄 작업이 제휴사에게 이메일 알림을 발송하나요?**

현재 일괄 작업은 제휴사 알림을 자동으로 발송하지 않습니다.

**Q: 한 번에 일괄 승인할 수 있는 수수료 개수에 제한이 있나요?**

고정된 제한은 없지만, 대량의 경우 처리하는 데 몇 초 정도 걸릴 수 있습니다.
