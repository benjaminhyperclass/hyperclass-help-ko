# SMS 옵트인 캠페인 만드는 방법

SMS 옵트인 캠페인(또는 "텍스트인 캠페인")은 광고판을 임대하거나 엽서를 발송하여, 고객이 특정 "키워드"를 비즈니스 번호로 문자 전송해서 혜택을 받을 수 있도록 하려는 클라이언트에게 매우 유용합니다.

[워크플로우에 대해 자세히 알아보기](https://help.gohlighlevel.com/support/solutions/articles/48001179678-workflow-builder-overview)

SMS 옵트인 캠페인의 핵심은 "고객 답장" 워크플로우 트리거(Customer Replied Workflow Trigger)입니다. 설정은 다음과 같아야 합니다:
