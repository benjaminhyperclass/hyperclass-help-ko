# 광고 관리자의 템플릿 라이브러리

광고 캠페인 제작이 더욱 쉬워졌습니다! 광고 관리자(Ads Manager) 템플릿 라이브러리(Template Library)에서 미리 만들어진 템플릿을 사용하여 시간을 절약하고 고성과 광고를 빠르게 런칭하세요.

## ✅ 왜 템플릿을 사용해야 하나요?

광고 관리자의 템플릿을 사용하면 캠페인 제작이 더 빠르고 효과적이 됩니다. 템플릿이 좋은 선택인 이유는 다음과 같습니다:

- **시간 절약**: 미리 만들어진 레이아웃으로 몇 분 만에 광고를 런칭할 수 있습니다.
- **성과 개선**: 검증된 광고 전략으로 설계되어 성과를 높입니다.
- **브랜드 일관성 유지**: 모든 캠페인에서 메시지와 디자인의 일관성을 유지합니다.
- **쉬운 커스터마이징**: 구조는 유지하면서 텍스트, 이미지, 타겟팅을 조정할 수 있습니다.
- **팀 협업 개선**: 여러 브랜드를 관리하는 팀과 에이전시의 프로세스를 간소화합니다.

## 왜 중요한가요?

템플릿을 사용하면 광고 제작 시간을 대폭 단축하면서도 전문적인 품질의 캠페인을 만들 수 있습니다. 특히 여러 고객을 관리하는 에이전시나 반복적인 광고를 운영하는 비즈니스에게 매우 유용합니다.

## 광고 관리자 템플릿 사용 방법

### 1단계: 광고 관리자로 이동

- 계정에서 **마케팅(Marketing)** 메뉴를 클릭합니다.
- **광고 관리자(Ad Manager)** 탭을 클릭합니다.

![광고 관리자 접근](https://jumpshare.com/v/UMX7xWJS4edmpepgwCdk+/Screen+Shot+2025-07-02+at+5.34.20+PM.png)

### 2단계: 캠페인 생성

- **+ 캠페인 생성(Create Campaign)** 버튼을 클릭합니다.
- 광고 캠페인 플랫폼으로 **Google** 또는 **Meta** 중 하나를 선택합니다.
- **다음(Next)**을 클릭합니다.

![캠페인 생성 과정](https://jumpshare.com/v/3JTOOgVzFMRvE4oDPD5V+/GIF+Recording+2025-07-02+at+5.39.57+PM.gif)

### 3단계: 템플릿 선택, 미리보기 및 적용

- 캠페인 생성 방법 선택 화면에서 **광고 관리자 템플릿(Ad Manager Templates)**을 선택합니다.
- **다음(Next)**을 클릭합니다.
- 캠페인에 맞는 광고 템플릿을 찾아서 선택합니다. 다음 조건으로 템플릿을 필터링할 수 있습니다:
  - **카테고리**: 자동차, 법률, 마케팅 에이전시, 금융 등
  - **목적**: 참여(Engagement), 리드 생성(Lead Generation), 판매(Sales), 웹 트래픽(Web Traffic)
  - **태그**
- 특정 템플릿의 **눈 아이콘**을 클릭하면 다양한 배치에서 어떻게 보이는지 미리볼 수 있습니다.
- 템플릿을 클릭하면 미리보기와 세부 정보를 확인할 수 있습니다.
- **계속(Continue)**을 클릭하여 템플릿을 추가합니다.
- 필요에 따라 광고 카피, 이미지, URL, 설정을 편집하고 광고를 발행합니다.

![템플릿 선택 과정](https://jumpshare.com/v/HdwDQdjswxZbUX99dkrd+/GIF+Recording+2025-07-02+at+6.05.53+PM.gif)

## 자주 묻는 질문

**Q: 페이스북과 구글 광고에 동일한 템플릿을 사용할 수 있나요?**
A: 라이브러리의 템플릿은 플랫폼별로 연결되어 있습니다.

**Q: 템플릿을 즐겨찾기에 추가할 수 있나요?**
A: 네, 사용자는 빠른 접근을 위해 템플릿을 즐겨찾기로 표시할 수 있습니다.
