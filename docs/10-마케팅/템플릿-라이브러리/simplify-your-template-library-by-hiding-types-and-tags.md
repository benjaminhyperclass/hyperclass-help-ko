# 템플릿 라이브러리의 유형과 태그란?

템플릿 라이브러리에는 **유형**과 **태그**라는 필터가 있어 템플릿을 용도와 속성별로 정리합니다.

관리자는 이러한 필터의 표시 여부를 제어할 수 있어 관련 없거나 내부용 옵션은 숨기고 필수적인 것만 활성화할 수 있습니다.

필터를 숨기면 관리자 화면에는 여전히 보이지만 하위 계정 인터페이스에서는 제거됩니다.

- 숨긴 항목은 하위 계정의 필터 목록에 나타나지 않습니다.

- 숨긴 필터와 연결된 템플릿은 다른 필터나 이름으로 여전히 접근할 수 있습니다.

- 관리자는 참고나 감사 목적으로 표시된 숨긴 항목을 여전히 볼 수 있습니다.

표시 여부를 제어함으로써 에이전시는 고객에게 간소화된 템플릿 라이브러리를 제공하고, 불필요한 요소를 줄이며, 하위 계정 전반에 걸쳐 일관된 브랜딩을 유지할 수 있습니다.

---

## 숨기기/보이기 옵션의 주요 장점

유형과 태그를 숨기면 템플릿 라이브러리를 정리하고, 불필요한 요소를 줄이며, 고객에게 전문적이고 화이트라벨된 경험을 제공할 수 있습니다.

- **맞춤형 필터:** 각 하위 계정에 가장 관련성 있는 유형과 태그만 보여주어 라이브러리를 집중적이고 관리하기 쉽게 만듭니다.

- **간소화된 검색:** 사용할 수 있는 필터 수를 제한하여 사용자의 템플릿 발견을 더 빠르고 쉽게 만듭니다.

- **화이트라벨 지원:** Hyperclass 브랜드나 내부용 필터를 제거하여 고객에게 매끄럽고 에이전시 브랜드의 경험을 제공합니다.

- **되돌릴 수 있는 설정:** 필요할 때 언제든지 숨긴 필터를 빠르게 복원할 수 있으며, 데이터 손실이나 재태깅이 필요하지 않습니다.

- **범용 접근:** 모든 요금제에서 템플릿 라이브러리 권한이 있는 모든 관리자가 사용할 수 있습니다.

## 필터 표시 관리하기

관리자는 하위 계정의 템플릿 라이브러리에 표시되는 유형과 태그를 맞춤 설정할 수 있습니다. 아래 단계를 따라 템플릿 라이브러리 설정에 접근하고 더 깔끔하고 개인화된 라이브러리 보기를 위해 필터 표시를 관리하세요.

### 에이전시 뷰로 전환하기

계정 전환기를 사용하여 하위 계정에서 **에이전시 뷰(Agency View)**로 변경합니다. 이렇게 하면 모든 하위 계정을 관리하기 위한 관리자 수준 설정에 접근할 수 있습니다.

![에이전시 뷰 전환](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155056571057/original/8Zsr2WWI6w_yihbldEswzGSTEbDsA572ug.png?1761132057)

### 사이드바에서 템플릿 라이브러리 열기

**에이전시 대시보드**에서 왼쪽 메뉴로 이동하여 **템플릿 라이브러리**를 선택합니다.

이렇게 하면 전체 템플릿 컬렉션이 열리고 관리 및 맞춤 설정을 위한 템플릿 라이브러리 설정에 접근할 수 있습니다.

![템플릿 라이브러리 열기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155056571122/original/rzTylLfnm7kgjNi1dy6yCxH9UX7vCNhemg.png?1761132089)

### 템플릿 라이브러리 설정 접근하기

템플릿 라이브러리 안에서 화면 우측 상단의 **톱니바퀴 아이콘**을 클릭합니다.

이렇게 하면 설정 패널이 열리며, 여기서 카테고리, 태그, 유형과 함께 하위 계정의 표시 기본 설정을 관리할 수 있습니다.

![설정 패널 접근](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155056571268/original/QaH2HQZvWHGpHVYM6GW354qGu2wjrzNabA.png?1761132119)

### 관리할 태그 또는 유형 선택하기

설정 패널에서 사이드바를 사용하여 편집하려는 것에 따라 **태그** 또는 **유형**을 선택합니다. 태그는 설명적 라벨을 나타내고, 유형은 용도나 형식별로 템플릿을 정리합니다. 이 섹션에는 생성 날짜와 표시 제어를 포함하여 모든 기존 필터가 나열됩니다.

![태그 또는 유형 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155056571351/original/RT1FCTwsCeV3Zw9u2h_dZiE54x6XlXlwKA.png?1761132148)

### 표시 여부 맞춤 설정하기

목록 오른쪽의 **숨기기** 열을 찾으세요. 각 태그나 유형 옆의 표시 제어(토글 또는 스위치)를 사용하여 표시 여부를 관리합니다:

- **끄기**로 전환하면 모든 하위 계정 뷰에서 필터가 숨겨집니다.

- **켜기**로 전환하면 필터가 다시 보이게 됩니다. 변경 사항은 즉시 적용되며 연결된 모든 하위 계정에 반영됩니다.

### 라이브러리 정리 팁

하위 계정 경험을 단순화하기 위해 내부용, Hyperclass 브랜드 또는 자주 사용하지 않는 필터를 먼저 숨기는 것부터 시작하세요.

이 페이지로 돌아와서 표시 여부를 다시 켜면 언제든지 숨긴 필터를 복원할 수 있습니다.

![표시 여부 맞춤 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155056571519/original/BNzB4qvDgQl5QnQoLgPukbNLG9H5fGo99g.png?1761132191)

### 참고사항

표시 제어의 라벨이나 아이콘은 작업 공간 구성에 따라 다를 수 있지만, 목록의 각 태그나 유형 옆에 항상 나타납니다.

## 자주 묻는 질문

**Q: 누가 필터 표시를 관리할 수 있나요?**

관리자 또는 템플릿 라이브러리 관리 권한이 있는 모든 사용자입니다.

**Q: 숨기기를 하면 템플릿에서 필터가 제거되나요?**

아니요. 숨기기는 필터 패널에서의 표시 여부만 변경하며, 템플릿 연결은 그대로 유지됩니다.

**Q: 필터를 나중에 복원할 수 있나요?**

네. 언제든지 필터를 다시 보이게 만들 수 있습니다.

**Q: 이것이 검색 결과에 영향을 주나요?**

아니요. 템플릿은 이름과 카테고리로 계속 검색할 수 있습니다.

**Q: 모든 요금제에서 사용할 수 있나요?**

네, 표시 제어는 템플릿 라이브러리 기능을 제공하는 모든 요금제에 포함되어 있습니다.
