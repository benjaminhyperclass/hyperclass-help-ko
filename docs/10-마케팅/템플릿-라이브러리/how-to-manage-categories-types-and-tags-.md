# 카테고리, 유형, 태그 관리하기

카테고리, 유형, 태그를 더 쉽고 효율적으로 관리할 수 있도록 관리자 기능을 개선했습니다.

### 지원되는 기능:

- **카테고리 생성, 편집, 삭제**  
더 나은 구성을 위해 **카테고리를 생성, 편집, 삭제**할 수 있습니다. 삭제하기 전에 필요하다면 템플릿을 다른 카테고리로 이동하세요.

- **유형 및 태그 관리**  
템플릿을 일일이 찾지 않고 한 화면에서 **유형과 태그를 쉽게 편집하고 삭제**할 수 있습니다.

### 사용 방법

- **템플릿 라이브러리 관리 화면 접근:**  
하위 계정 뷰에서 **Agency View(에이전시 뷰)**로 전환하고 **"Template Library(템플릿 라이브러리)"**를 찾으세요. 여기서 기존 템플릿과 해당 카테고리, 상태, 태그 등을 접근하고 관리할 수 있습니다.

![템플릿 라이브러리 접근](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048212778/original/S8xH5LSEQlVlwT4DIsqomD0OgHI9QyyN5g.gif?1749814744)

- **카테고리 관리:**  
**Settings(설정)**를 클릭하세요.
- **생성**: Create Categories(카테고리 생성)를 클릭하고 세부 사항을 지정한 후 유형을 카테고리로 선택합니다

![카테고리 생성 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155031429115/original/HHfSae5Qlnu6kyamiBIIlkO8-Vf_gFvA_Q.png?1724247718)

![카테고리 생성 폼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155031429194/original/5p8w8_qSHAmIR9qzb_XCM3NCzi2m0LnRbQ.png?1724247768)

- 제품을 선택하고 **편집**하거나 **삭제**하려는 카테고리를 찾으세요.
- **편집**: 편집 버튼을 클릭하여 카테고리의 이름이나 설명을 업데이트합니다.

![카테고리 편집](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155031429240/original/EOdh427wJu_rbewL5th_9mda64xdILajfw.png?1724247785)

- **삭제**: 삭제 버튼을 클릭하세요. 삭제를 확인하라는 메시지가 표시됩니다. 템플릿을 보존하려면 삭제를 확인하기 전에 다른 카테고리로 이동하세요.

![카테고리 삭제](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155031429311/original/vqxPFihl7kdJ0ZnENGh17w-8Fl93O30jUA.png?1724247802)

- **유형 관리:**  
Types(유형) 섹션으로 이동하세요.
- 생성: Create Categories and Types(카테고리 및 유형 생성)를 클릭하고 유형을 "Type"으로 설정하여 세부 사항을 업데이트합니다

![유형 생성](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155031429423/original/JPP7ncuCl7KmV2n2jf13oCoJJ_PZ_f4t7Q.png?1724247867)

- **편집**: 편집 버튼을 클릭하여 유형의 이름이나 설명을 변경합니다.

![유형 편집](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155031429431/original/rckGEyzSjmhsvt1dfFC5VlBanEliiblRuw.png?1724247881)

- **삭제**: 삭제 버튼을 클릭하고 삭제를 확인합니다.

![유형 삭제](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155031429440/original/P8K33O9BbPozqkLhdYmPesdHsNdZsxzNeA.png?1724247894)

- **태그 관리:**  
Tags(태그) 섹션으로 이동하세요.
- **편집**: 편집 버튼을 클릭하여 태그의 이름을 변경합니다.

![태그 편집](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155031429447/original/lh3f6OAdelcB31j_IXQfUEdkbCG1KJ4bTg.png?1724247910)

- **삭제**: 삭제 버튼을 클릭하고 삭제를 확인합니다.

![태그 삭제1](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155031429472/original/WQaLQ18eFDbJXWYCDhJnHJztZniOCmXA3A.png?1724247921)

## 자주 묻는 질문

**1. 이 모든 기능에 누가 접근할 수 있나요?**  
에이전시 관리자만 관리자 및 설정 페이지에 접근할 수 있습니다.

**2. Hyperclass 팀에서 추가한 카테고리, 태그, 유형을 편집할 수 있나요?**  
Hyperclass 팀에서 추가한 카테고리는 숨기기만 가능합니다.

**3. 카테고리를 삭제하면 템플릿도 삭제되나요?**  
삭제 위험이 있는 템플릿 목록을 경고해 드립니다. 이런 템플릿들은 숨겨진 카테고리로 이동하는 것을 권장합니다.
