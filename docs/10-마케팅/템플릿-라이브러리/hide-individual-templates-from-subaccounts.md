# 하위 계정에서 개별 템플릿 숨기기

에이전시 템플릿 라이브러리(Template Library)에는 하위 계정의 템플릿 라이브러리와 템플릿을 관리하는 여러 설정이 있습니다. 개별 템플릿을 직접 숨겨서 에이전시에서만 볼 수 있게 하는 방법은 없지만, 개별 템플릿을 숨겨진 카테고리에 넣는 방법은 있습니다.

---

## 개별 템플릿을 숨기는 방법

- `Agency(에이전시) → Template Library(템플릿 라이브러리)`로 이동하세요.

![에이전시 템플릿 라이브러리](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048644956/original/SE3Fv8FR8TMvkVZ3OLU17Un7vDR9R8gE-A.png?1750455536)

- **톱니바퀴 아이콘**(설정)을 클릭하세요.

![설정 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048644960/original/HPtLB7Zii9BA_aO8LW4mNk9eR84qmvPAxw.png?1750455605)

- 템플릿 라이브러리 관리자 설정에서 **제품** 중 하나(예: Funnels)를 선택하고, `Categories(카테고리) → Create Category(카테고리 만들기)`를 클릭하세요. 카테고리에 명확한 **이름**(예: HIDDEN Funnels)과 **설명**(예: 하위 계정에 표시되지 않는 퍼널 템플릿)을 입력한 다음 **Create(만들기)**를 클릭하세요.

![카테고리 만들기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048645013/original/NmstDl_U24Er-KxkJzMiU9JLTvkIiN7BRg.png?1750455735)

- 새 카테고리가 만들어진 후, 목록에서 찾아서 **숨기기 토글**(파란색으로 변함)을 클릭하세요. 모든 하위 계정에서 이 카테고리를 숨기겠다는 확인 모달이 나타날 수 있습니다.

![카테고리 숨기기 토글](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048645043/original/SbqW5st5mKYXwKBYzNG8vjt_0yygk006Dg.png?1750455969)

- 에이전시 템플릿 라이브러리로 돌아가서 숨겨진 카테고리와 그 안의 템플릿을 볼 수 있는지 확인하세요.

![숨겨진 카테고리 확인](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048645063/original/i-nOjxm_hkekEglLV0_i6c30E460-_SG0Q.png?1750456142)

- `하위 계정 → [제품]`(예: 숨긴 카테고리가 Funnels에 있었다면 Funnels) → `New [product](새 [제품]) → from Templates(템플릿에서)`로 이동하여 해당 제품의 템플릿 라이브러리를 여세요. 하위 계정에서 볼 수 있는 카테고리 목록에서 숨겨진 카테고리를 볼 수 없는지 확인하세요. 템플릿 이름으로 검색해서 전체 템플릿에서 숨겨져 있는지 확인하세요.

![하위 계정에서 확인](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048645087/original/kuTVF398Wg4o3OuF2Cjf-DndOAZsHxxIEQ.png?1750456317)

- **참고**: 하위 계정에서 숨기려는 템플릿은 숨기지 않은 카테고리에 있으면 안 됩니다. 템플릿의 미리보기 이미지에 "This template is hidden for all your sub-accounts(이 템플릿은 모든 하위 계정에서 숨겨져 있습니다)"라는 회색 박스가 있는지 확인하세요. 템플릿이 숨겨져 있으면 숨겨져 있다고 표시됩니다. 템플릿이 숨겨져 있지 않으면 보인다고 별도로 표시되지 않습니다.

![템플릿 숨김 상태 확인](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048645186/original/ivaR_2Fcfz-vJX__Z-WYNcynfoFM8cM9uQ.png?1750456549)
