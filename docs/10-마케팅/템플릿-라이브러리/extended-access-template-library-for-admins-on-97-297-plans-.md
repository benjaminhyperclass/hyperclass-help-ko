# 확장된 템플릿 라이브러리 관리자 접근 권한이란?

기존에는 템플릿 라이브러리 관리자 접근 권한이 $497 플랜 사용자에게만 제한되어 있었습니다. 이제 $97 "Starter" 플랜과 $297 "Unlimited" 플랜의 관리자도 Agency View(에이전시 보기) → Template Library(템플릿 라이브러리)에 로그인하여 미리 제작된 자료를 살펴보고, 개별 템플릿이나 전체 카테고리를 숨기며, 프리미엄 제작 도구를 미리보기(업그레이드 안내와 함께)할 수 있습니다.

---

## **확장된 템플릿 라이브러리 접근 권한의 주요 이점**

- **템플릿 탐색** - $97 및 $297 플랜 관리자가 이제 템플릿 라이브러리를 탐색할 수 있습니다.
- **템플릿 숨기기** - 특정 템플릿이 로케이션(하위 계정)의 템플릿 라이브러리에 나타나지 않도록 방지할 수 있습니다.
- **카테고리 숨기기** - 전체 카테고리가 템플릿 라이브러리에 나타나지 않도록 방지할 수 있습니다.

---

## 개별 템플릿 숨기기

특정 템플릿을 숨기면 모든 하위 계정에서 해당 템플릿이 나타나지 않아, 에이전시의 브랜드나 틈새 시장 집중도를 보호할 수 있습니다. Agency(에이전시) → Template Library(템플릿 라이브러리)로 이동하세요. 템플릿 위에 마우스를 올리고 Hide(숨기기)를 클릭합니다. 회색 "Hidden(숨겨짐)" 배지가 변경사항을 확인해줍니다.

숨겨진 템플릿은 향후 관리를 위해 에이전시 관리자에게는 계속 표시됩니다.

## 전체 카테고리 숨기기

카테고리 레벨 숨기기는 폴더 내의 템플릿이 귀하의 서비스와 맞지 않을 때 완벽한 기능입니다. 템플릿 라이브러리에서 톱니바퀴 아이콘(Settings(설정))을 클릭합니다. 제품을 선택하고(예: Funnels(퍼널)) → Categories(카테고리)를 선택합니다. 원하는 카테고리 옆의 Hide(숨기기)를 토글하세요. 내부의 모든 템플릿이 모든 로케이션에서 보이지 않게 됩니다.

## **자주 묻는 질문**

**Q: $97 & $297 관리자가 새 템플릿을 만들 수 있나요?**

A: 아직은 불가능합니다. 템플릿 생성은 여전히 $497 Pro 플랜이 필요하지만, 지금 당장 콘텐츠를 탐색하고 숨기는 것은 가능합니다.

**Q: 숨겨진 템플릿이 기존 로케이션에서도 사라지나요?**

A: 네. 템플릿이나 카테고리를 숨기면 현재 및 향후 모든 하위 계정에서 자동으로 제거됩니다.

**Q: 카테고리를 숨기면 제가 커스터마이징한 템플릿도 함께 숨겨지나요?**

A: 커스터마이징한 템플릿이 해당 숨겨진 카테고리에만 속해 있다면 숨겨집니다. 여러 개의 보이는 카테고리에 속한 템플릿은 계속 표시됩니다.

**Q: 나중에 숨김을 해제할 수 있나요?**

A: 물론입니다. Settings(설정) → Categories(카테고리)로 돌아가거나 템플릿 위에 마우스를 올리고 Unhide(숨김 해제)를 클릭하세요. 변경사항은 즉시 적용됩니다.

**Q: 숨기기가 에이전시의 템플릿 사용 능력에 영향을 주나요?**

A: 아니요. 숨겨진 항목은 에이전시 관리자에게는 계속 표시되므로, 내부적으로 복제하거나 편집할 수 있습니다.

**Q: 일괄 숨기기 옵션이 있나요?**

A: 카테고리 숨기기가 가장 빠른 일괄 방법입니다. 카테고리 내에서 여러 템플릿을 다중 선택하고 Hide(숨기기)를 토글할 수도 있습니다.

**Q: 클라이언트가 어딘가에서 Hyperclass 브랜딩을 보게 되나요?**

A: 숨겨진 템플릿은 클라이언트의 화면에서 Hyperclass 브랜드 자료를 제거하여 화이트라벨 존재감을 보존합니다.

**Q: 에이전시 사용자(비관리자)가 라이브러리에 접근할 수 있나요?**

A: $97 & $297 플랜에서는 에이전시 관리자만 라이브러리 권한을 갖습니다. 일반 사용자와 로케이션 역할은 템플릿을 보거나 편집할 수 없습니다.
