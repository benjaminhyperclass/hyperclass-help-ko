# 웨비나용 템플릿 라이브러리

이 문서에서는 Hyperclass의 **웨비나 템플릿**을 사용해서 라이브 또는 온디맨드 웨비나를 빠르게 설정하는 방법을 알려드립니다! 웨비나 템플릿은 선택한 동영상과 폼을 미리 디자인된 웨비나 레이아웃에 자동으로 삽입해서 브랜드 일관성을 유지하고 설정 시간을 단축해줍니다.

## 웨비나 템플릿이란?

**웨비나 템플릿**을 사용하면 재사용 가능한 웨비나 레이아웃을 미리 디자인할 수 있습니다. 웨비나를 만들 때 선택한 동영상과 폼이 선택된 템플릿의 플레이스홀더를 자동으로 교체합니다. 이렇게 하면 라이브든 온디맨드든 상관없이 웨비나 설정이 빨라지고 브랜드 일관성을 유지할 수 있습니다.

---

## 웨비나 템플릿 사용의 주요 이점

웨비나 템플릿은 웨비나 설정 속도를 높이고 여러 웨비나에 걸쳐 일관된 룩앤필을 보장합니다. 소중한 시간을 절약하면서 청중에게 전문적인 경험을 제공할 수 있습니다.

- **빠른 웨비나 설정**: 선택한 동영상과 폼으로 템플릿을 자동 채워서 몇 분 안에 웨비나를 런칭하세요.

- **일관된 브랜딩**: 재사용 가능하고 커스터마이징 가능한 템플릿으로 통일된 브랜드 경험을 유지하세요.

- **유연한 템플릿 관리**: 에이전시 관리자는 다양한 하위 계정에 대해 템플릿을 업로드하고 관리하며 가시성을 제어할 수 있습니다.

- **쉬운 커스터마이징**: 템플릿이나 빈 슬레이트부터 시작해서 필요에 따라 콘텐츠를 개인화하세요.

- **수동 작업 감소**: 플레이스홀더가 자동으로 교체되어 반복적인 작업을 최소화합니다.

## **템플릿으로 웨비나 만드는 방법**

템플릿을 사용해서 웨비나를 만드는 것은 간단합니다. 수동 서식 작업 없이 일관된 디자인을 빠르게 적용할 수 있어서 콘텐츠와 청중 참여에 집중할 수 있습니다.

- **Webinars(웨비나)**로 이동해서 **Create Webinar(웨비나 만들기)**를 클릭하세요 (Live 또는 On-Demand 중 선택).

![웨비나 만들기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048215598/original/5hMpCrderWWYfBa9nANU_rtse6qHHnOV2Q.gif?1749816780)

- **웨비나 세부사항 입력**: 웨비나에 이름을 지어주세요.

![웨비나 세부사항](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048216064/original/5OL8okzFwpUOgDWfswgsgYxne_-AoqYwFA.png?1749817115)

- **폼 선택**: 웨비나 등록용 폼을 선택하세요.

![폼 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048216100/original/UnHpQnPb86H-4JJsPcAjZfxsPuyROgUSYA.png?1749817174)

- **동영상 선택**: 미리 녹화한 웨비나를 선택하세요.

![동영상 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048216171/original/0FjTZGZrVLKsTYV-ijGLY31Uh6wkBbM_Aw.png?1749817244)

- 목록에서 **웨비나 템플릿**을 선택하거나 빈 템플릿부터 시작하세요. **Proceed(계속)**를 클릭하면 선택한 동영상과 폼이 템플릿의 플레이스홀더를 자동으로 교체합니다.

![템플릿 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048216815/original/42CViOvIz-iLyIBOnH0w60f61fuxXxieqw.gif?1749817580)

## **웨비나 템플릿 관리하기 (에이전시 관리자용)**

에이전시 관리자는 웨비나 템플릿을 관리하는 추가적인 제어 기능이 있습니다. 이를 통해 하위 계정이 자신의 브랜딩과 요구사항에 맞는 템플릿만 볼 수 있도록 보장할 수 있습니다.

- **Template Library(템플릿 라이브러리)** → **Webinar** 카테고리를 선택하세요.

![템플릿 라이브러리](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048217050/original/x6qNAX_PsYIG1guX64_byUlT8jiLUF9_KA.gif?1749817727)

- 하위 계정에 대해 템플릿을 숨기거나 표시하려면 각 템플릿 옆의 **가시성 토글**을 클릭하세요.

![가시성 토글](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048217645/original/GInrAHGdHxZwp2IwVl6Gx--N6RxEK6vIxg.png?1749818174)

- 새 템플릿을 업로드하려면 **Create(만들기)** → 새 웨비나 템플릿 업로드를 클릭하세요. 경험을 커스터마이즈하려면 각 하위 계정이 접근하고 사용할 수 있는 템플릿을 관리하세요.

![템플릿 업로드](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048217355/original/zT0IuT_SwwzJZZc_OTSRNnwJQxc_y5lmgg.gif?1749818026)

참고: 기본/빈 템플릿은 숨길 수 없습니다.

## 웨비나 템플릿 사용의 추가 이점

웨비나 템플릿이 어떻게 작동하는지 이해하면 그 기능을 최대한 활용할 수 있습니다. 이러한 기능들은 시간을 절약하고 비즈니스 전반에 걸쳐 고품질의 웨비나를 보장하도록 설계되었습니다.

- **자동 플레이스홀더 교체**: 선택한 동영상과 폼이 선택된 템플릿의 플레이스홀더를 교체합니다.

- **재사용 가능한 디자인**: 템플릿은 여러 웨비나에 걸쳐 사용되어 일관된 브랜딩과 레이아웃을 유지할 수 있습니다.

- **가시성 제어**: 에이전시 관리자는 하위 계정이 접근하고 사용할 수 있는 템플릿을 관리할 수 있습니다.

- **템플릿 라이브러리 연동**: 웨비나 템플릿은 중앙 집중식 템플릿 라이브러리에 있어서 쉽게 찾고 관리할 수 있습니다.

## **자주 묻는 질문**

**Q: 나만의 웨비나 템플릿을 만들 수 있나요?**

네! 에이전시 관리자는 템플릿 라이브러리를 통해 새 웨비나 템플릿을 업로드할 수 있습니다.

**Q: 템플릿을 적용한 후에 동영상이나 폼을 바꿀 수 있나요?**

네, 템플릿을 적용한 후에도 웨비나 설정에서 동영상이나 폼을 편집할 수 있습니다.

**Q: 기본/빈 템플릿을 숨길 수 있나요?**

아니요. 기본/빈 템플릿은 항상 표시됩니다.

**Q: 템플릿이 커스텀 스타일링을 지원하나요?**

네, 커스텀 템플릿 디자인은 보존됩니다. 동영상과 폼 플레이스홀더만 자동 교체됩니다.

**Q: 웨비나 템플릿을 하위 계정에서 기본적으로 사용할 수 있나요?**

네, 에이전시 관리자가 하위 계정에서 특정 템플릿을 숨기지 않는 한 사용할 수 있습니다.

**Q: 사용하기 전에 템플릿을 미리 볼 수 있나요?**

네, 템플릿을 선택하기 전에 템플릿 라이브러리에서 미리 볼 수 있습니다.
