# 워크플로우 템플릿 라이브러리

새로운 워크플로우 템플릿 라이브러리를 통해 Hyperclass에서 자동화를 빠르게 설정할 수 있습니다. 이 기능을 사용하면 미리 만들어진 템플릿으로 워크플로우를 실행할 수 있어 시간을 절약하고 모범 사례를 보장합니다.

## 템플릿 라이브러리란?

워크플로우 템플릿 라이브러리는 전문적으로 설계된 워크플로우 템플릿을 사용하여 자동화 구축을 시작할 수 있게 해주는 고급 기능입니다. 워크플로우를 처음부터 만드는 대신, 필요에 맞는 템플릿을 선택하고 구조를 검토한 후 필요에 따라 커스터마이징할 수 있습니다. 이를 통해 사용자는 더 빠르게 시작할 수 있고, 성공이 입증된 레시피를 바탕으로 워크플로우를 구축할 수 있습니다.

---

## 워크플로우 템플릿 사용의 주요 장점

워크플로우 템플릿 라이브러리를 활용하면 자동화 경험을 향상시키는 여러 가지 장점이 있습니다:

- **빠른 워크플로우 생성**: 미리 만들어진 템플릿을 활용하여 몇 분 만에 새로운 자동화를 실행할 수 있습니다.

- **검증된 모범 사례**: 전문가가 설계한 템플릿을 사용하여 오류와 비효율성 위험을 줄입니다.

- **지속적인 업데이트**: 새롭고 관련성 높은 템플릿이 정기적으로 추가되는 지속적으로 성장하는 라이브러리에 액세스할 수 있습니다.

- **커스터마이징 유연성**: 고유한 비즈니스 프로세스에 맞게 템플릿을 편집하고 조정할 수 있습니다.

- **학습 곡선 단축**: 워크플로우 생성을 표준화하려는 신규 사용자나 팀에게 이상적입니다.

- **시간 절약**: 반복적인 설정 작업을 제거하고 더 가치 있는 활동에 집중할 수 있습니다.

## 워크플로우 템플릿 라이브러리 설정 방법

워크플로우 템플릿 라이브러리를 활성화하고 사용하는 것은 간단하지만, 올바른 단계를 따르면 이점을 극대화하고 일반적인 설정 문제를 피할 수 있습니다. 다음 단계에 따라 워크플로우 템플릿 라이브러리를 활성화하고 사용하기 시작하세요.

### 1단계: 템플릿 라이브러리 접근

템플릿 라이브러리 모달을 열어 미리보기하고 필요에 따라 커스터마이징할 수 있는 바로 사용 가능한 워크플로우 레시피의 큐레이션된 컬렉션을 찾으세요.

- 왼쪽 사이드바에서 Automation(자동화)을 클릭하세요.

- Workflows(워크플로우) 탭으로 전환하세요.

- Create Workflow(워크플로우 생성)를 클릭하세요.

- Select from Template(템플릿에서 선택)을 선택하세요.

![워크플로우 템플릿 라이브러리 접근 방법](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048998068/original/kHL7BwDSKtkObJ3-wevhRnvnnmfgO3EMOA.gif?1751034241)

### 2단계: 템플릿 탐색 및 필터링

카테고리를 탐색하고, 필터를 적용하거나, 키워드를 검색하여 올바른 워크플로우를 빠르게 찾으세요. 필터(Filters) 패널을 사용하여 카테고리별로 좁혀보세요(예: 리드 육성, 온보딩). 키워드로 검색하거나 인기도별로 정렬하세요. 템플릿 카드 위에 마우스를 올리면 간단한 설명을 볼 수 있습니다.

![템플릿 탐색 및 필터링](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048998356/original/7BpKXc4Dg3lmt8YFjtaYNFHsipVJQCLPfA.gif?1751034449)

### 3단계: 템플릿 미리보기 및 선택

워크플로우를 로드하기 전에 세부 사항을 자세히 살펴보세요. 설명, 전제 조건, 전체 캔버스를 검토하여 필요에 맞는지 확인하세요.

- 눈 모양 아이콘이나 템플릿 카드를 클릭하여 미리보기를 여세요.

- "이 템플릿 정보" 설명을 읽고 전제 조건을 확인하세요.

- 로직 흐름을 이해하기 위해 완전한 트리거-액션 캔버스를 검토하세요.

- 준비가 되면 Continue(계속)를 클릭하여 워크플로우 편집기에 로드하세요.

![템플릿 미리보기 및 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048998600/original/YUZWWgRWeY7UfmSlFZj7wgEUVnBzcd0ipQ.gif?1751034683)

**참고**: 워크플로우 편집기에서 프로세스에 맞게 트리거, 액션, 대기, 조건을 커스터마이징하세요. 샘플 연락처나 태그로 테스트를 실행하여 기능을 확인한 후 워크플로우를 저장하고 활성화하세요.

## 고급 팁 및 모범 사례

1. **템플릿 관리 및 업데이트**

- **최신 상태 유지**: 새롭거나 개선된 템플릿을 위해 정기적으로 라이브러리를 확인하세요.

- **피드백 루프**: 지원 포털을 통해 아이디어나 개선 사항을 제출하세요.

2. **비즈니스에 맞게 커스터마이징**

- **로직 조정**: 단계를 추가하거나 제거하고, 메시지를 변경하며, 타이밍을 조정하세요.

- **철저한 테스트**: 전체 활성화 전에 작은 하위 집합으로 "초안" 모드에서 실행하세요.

3. **팀과의 협업**

- **템플릿 공유**: 즐겨찾기를 북마크하거나 링크를 공유하여 팀원들이 표준화된 워크플로우를 재사용할 수 있게 하세요.

- **교육 도구**: 템플릿을 사용하여 자동화 모범 사례에 대해 새 직원을 온보딩하세요.

## 자주 묻는 질문

**Q: 나만의 템플릿을 만들 수 있나요?**

현재로서는 이 기능을 사용할 수 없습니다.

**Q: 템플릿을 선택한 후 커스터마이징할 수 있나요?**

네, 워크플로우 편집기에 로드된 후에는 모든 것을 편집할 수 있습니다.

**Q: 새로운 템플릿이 추가되나요?**

물론입니다. 사용자 피드백과 업계 모범 사례를 바탕으로 지속적으로 라이브러리를 확장합니다.

**Q: 활성화가 기존 워크플로우에 영향을 미치나요?**

아니요, 현재의 자동화는 그대로 유지됩니다.

**Q: 변경 사항을 되돌릴 수 있나요?**

새로 시작하려면 원래 템플릿을 다시 선택하면 됩니다.
