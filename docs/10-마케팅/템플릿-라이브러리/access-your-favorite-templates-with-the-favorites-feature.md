# 즐겨찾기 기능으로 자주 쓰는 템플릿에 빠르게 접근하기

이 가이드에서는 Hyperclass의 즐겨찾기(Favorites) 기능을 사용하여 가장 자주 사용하는 템플릿에 빠르게 접근하는 방법을 알아봅니다. 템플릿을 즐겨찾기로 표시하면 작업 흐름을 간소화하고 캠페인 전반에 걸쳐 일관성을 유지할 수 있습니다.

# **즐겨찾기 기능이란?**

**더 나은 워크플로우를 위한 템플릿 접근 간소화**

Hyperclass 템플릿 라이브러리의 즐겨찾기(Favorites) 기능을 사용하면 이메일, 퍼널, 웹사이트, 소셜 플래너 템플릿 등 자주 사용하는 템플릿에 별표를 표시할 수 있습니다. 이를 통해 매번 전체 라이브러리를 검색하지 않고도 템플릿을 쉽게 찾아서 빠르게 재사용할 수 있습니다.

## **즐겨찾기 기능의 주요 장점**

**효율성 향상 및 체계적인 관리**

- **즉시 접근** – 가장 자주 사용하는 템플릿을 몇 번의 클릭으로 불러올 수 있습니다.
- **간편한 네비게이션** – 캠페인을 만들 때마다 방대한 라이브러리를 탐색할 필요가 없습니다.
- **일관성** – 효과가 검증된 템플릿을 재사용하여 메시지의 일관성을 유지할 수 있습니다.
- **시간 절약** – 검색 시간을 줄여 워크플로우의 속도를 높입니다.
- **맞춤 워크플로우** – 자주 쓰는 항목을 선별하여 개인만의 템플릿 도구함을 만들 수 있습니다.
- **사용자별 즐겨찾기** – 즐겨찾기는 사용자 프로필에 연결되어 개인화된 접근이 가능합니다.

## **템플릿을 즐겨찾기로 표시하는 방법**

**나중에 쉽게 접근할 수 있도록 템플릿에 별표 표시하기**

- **템플릿 라이브러리로 이동**  
  즐겨찾기로 등록할 템플릿 유형(예: 이메일, 퍼널, 웹사이트, 소셜 플래너 템플릿)으로 이동합니다.

- **템플릿 찾기**  
  스크롤하거나 검색하여 즐겨찾기로 등록할 특정 템플릿을 찾습니다.

- **별표 아이콘 클릭**  
  템플릿 위에 마우스를 올리고 우측 상단의 별표 아이콘을 클릭합니다. 별표가 채워지면서 즐겨찾기에 추가되었다는 것을 나타냅니다.

- **완료!**  
  이제 해당 템플릿이 즐겨찾기 목록에 저장되어 빠른 검색이 가능합니다.

## **즐겨찾기 템플릿에 접근하기**

**즐겨찾기 필터로 별표 표시된 템플릿만 보기**

- **템플릿 라이브러리로 돌아가기**  
  해당하는 템플릿 섹션으로 돌아갑니다.

- **필터 메뉴 열기**  
  보통 템플릿 목록 상단에 있는 필터 옵션을 찾습니다.

- **"즐겨찾기" 선택**  
  **Favorites(즐겨찾기)** 필터를 클릭하면 즐겨찾기로 등록한 템플릿만 즉시 표시됩니다.

- **즐겨찾기 사용 또는 편집**  
  원하는 즐겨찾기 템플릿을 클릭하여 편집하거나 배포를 시작합니다.

## **즐겨찾기 활용 모범 사례**

**이러한 팁으로 생산성 극대화하기**

- **즐겨찾기를 최신 상태로 유지** – 가장 자주 사용하는 템플릿을 반영하도록 정기적으로 목록을 검토하고 업데이트하세요.
- **즐겨찾기와 태그 조합** – 태그와 카테고리를 사용하여 템플릿을 더욱 체계적으로 정리하세요.
- **빠른 접근을 위해 즐겨찾기 제한** – 즐겨찾기 목록을 너무 많이 만들지 마세요. 간결하게 유지해야 더 빠르게 접근할 수 있습니다.
- **오래된 템플릿 제거** – 더 이상 유용하지 않은 즐겨찾기는 별표를 해제하여 목록을 신선하게 유지하세요.

## **사용 방법**

- 템플릿 라이브러리에서 새로 만들기를 선택합니다
- 템플릿 위에 마우스를 올리고 별표 아이콘을 클릭하여 즐겨찾기로 표시합니다.

## **자주 묻는 질문**

**Q: 모든 카테고리의 템플릿을 즐겨찾기에 추가할 수 있나요?**  
네. 즐겨찾기 기능은 이메일, 퍼널, 웹사이트, 소셜 플래너 템플릿을 포함한 여러 템플릿 유형에서 작동합니다.

**Q: 템플릿을 즐겨찾기에서 제거하려면 어떻게 하나요?**  
템플릿의 채워진 별표 아이콘을 클릭하여 표시를 해제하세요. 별표가 빈 상태로 돌아가면서 즐겨찾기 목록에서 제거됩니다.

**Q: 팀원들이 내 즐겨찾기를 볼 수 있나요?**  
아니요. 즐겨찾기는 각 사용자 프로필에 고유합니다. 팀이나 하위 계정(Sub-account) 간에 공유되지 않습니다.

**Q: 즐겨찾기로 등록할 수 있는 템플릿 수에 제한이 있나요?**  
엄격한 제한은 없지만, 효율성을 위해 즐겨찾기 목록을 관리 가능한 수준으로 유지하는 것을 권장합니다.

**Q: 템플릿 유형별로 즐겨찾기를 필터링할 수 있나요?**  
네. 각 템플릿 섹션에서 유형(예: 퍼널 템플릿)을 선택한 후 즐겨찾기 필터를 적용하면 해당 카테고리의 즐겨찾기만 별도로 볼 수 있습니다.
