# 관리자 화면에서 템플릿 생성 및 관리하는 방법

에이전시 관리자를 위한 템플릿 라이브러리(Template Library)의 향상된 템플릿 관리 기능을 소개합니다.

**문제 상황**: 이전에는 관리자가 필요한 템플릿을 가져오기 위해 여러 하위 계정을 일일이 탐색해야 했고, 이는 매우 시간이 많이 소요되는 작업이었습니다.

🚀 **개선 사항**:

- **직접 생성**: 이제 템플릿을 관리자 화면에서 바로 가져오거나 생성할 수 있습니다.
- **간소화된 프로세스**: 하위 계정 선택, 템플릿 미리보기, 이름 변경을 원활하게 진행할 수 있습니다.

⭐ **장점**:

- **효율성**: 템플릿 관리를 하나의 화면으로 통합하여 시간을 절약합니다.
- **간소화된 워크플로우**: 여러 화면과 하위 계정을 오가는 복잡함을 제거합니다.

💡 **사용 방법**:

- Template Library(템플릿 라이브러리) - Admin으로 이동합니다.
- 우측 상단의 "Create(생성)" 버튼을 클릭합니다.

![관리자 템플릿 생성 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155030317569/original/cl39u1YDs1XNvVqRCHgUzyGjrC-3p-gWuA.jpeg?1722539011)

- 하위 계정, 상품, 템플릿 이름을 선택합니다.

![하위 계정 및 상품 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155030317573/original/cbK-2fEL55cWRy2i37gxmYgUlTRbgWhCYg.jpeg?1722539011)

- "Continue(계속)" 버튼을 클릭하여 템플릿 썸네일을 미리 볼 수 있습니다. 썸네일을 클릭하면 별도 화면에서 전체 미리보기를 확인할 수 있습니다.

![템플릿 미리보기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155030317571/original/f_VZCjc8kiolD4eGLbc278z4V0xo2iKVxQ.jpeg?1722539011)

- 필요한 경우 템플릿 이름을 변경합니다.

![템플릿 이름 변경](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155030317572/original/L_z9YVzXBkSin-Fs7zAowrMnDtTMNfcw6Q.jpeg?1722539011)

- "Create(생성)" 버튼을 클릭하여 완료합니다.
- 설명과 카테고리 같은 세부 정보를 업데이트합니다.

![템플릿 세부 정보 업데이트](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155030317570/original/zFh8rKDhWlM-ZjPYUiVfFgFjbPg9Q0hECQ.jpeg?1722539011)

- Publish(발행) 또는 Save as draft(초안으로 저장)를 선택합니다.
