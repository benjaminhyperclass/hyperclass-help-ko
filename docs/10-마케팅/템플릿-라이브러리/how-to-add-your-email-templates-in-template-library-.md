# 템플릿 라이브러리에 이메일 템플릿을 추가하는 방법

이메일 템플릿을 템플릿 라이브러리에 업로드하면 모든 것을 체계적으로 정리하고 쉽게 접근할 수 있습니다. 이 가이드에서는 템플릿을 업로드하는 방법을 알려드려, 필요할 때마다 빠르게 찾아서 사용할 수 있도록 도와드리겠습니다.

**왜 유용한가요?**

- **쉬운 접근**: 파일을 뒤지지 않고도 템플릿을 빠르게 찾아 사용할 수 있습니다.
- **일관성**: 모든 커뮤니케이션에서 일관된 룩앤필을 유지합니다.
- **효율성**: 템플릿을 쉽게 업데이트하고 조정하여 시간을 절약합니다.
- **팀 협업**: 팀원들과 템플릿을 원활하게 공유합니다.
- **성과 추적**: 내장된 분석 도구로 이메일 효과를 모니터링하고 개선합니다.

## 템플릿 라이브러리에 템플릿을 추가하는 방법

1. Email Marketing(이메일 마케팅)으로 이동합니다.

2. 목록 보기에서 해당 자산으로 이동합니다.

3. 점 세 개를 클릭하면 Template Library(템플릿 라이브러리)에 추가하는 옵션이 나타납니다.

![템플릿 라이브러리 추가 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155010753289/original/RcVfED_VIaoSb3uxhCpGfgV5JvK-UwHhiw.jpeg?1697999882)

4. 템플릿 이름을 추가하고 카테고리를 선택합니다.

![템플릿 이름 및 카테고리 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155010753279/original/IUeqq3ZTXd3BloUdV-3R08IJw-PTkjCDnA.png?1697999882)

5. Template Library Admin(템플릿 라이브러리 관리자) 모드로 리디렉션됩니다.

- 커버 사진을 추가할 수 있습니다.
- 태그 추가, 이름 편집, 카테고리 선택이 가능합니다.
- 템플릿에 대한 간단한 설명을 작성합니다.
- 퍼널에서 원스텝 주문 폼을 사용하기 위한 Stripe 연동 등 사전 요구사항을 추가합니다.
- 템플릿을 독특하게 만들기 위해 Featured Tags(추천 태그)를 추가합니다.

![템플릿 상세 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155010753295/original/42jwNkVXaT4fM1J0hztNXMqOU6-T8XKcRQ.jpeg?1697999882)

6. 템플릿이 준비되면 public(공개)으로 설정하세요 (하위 계정에서 고객이 사용할 수 있도록 템플릿을 표시). draft(초안)으로 설정하면 템플릿 라이브러리 관리자 패널에서만 표시되어 향후 편집이 가능합니다.

![템플릿 공개 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155010753280/original/YI2TGUEyAZFMBh1Wo2Mnz4tUqJ-7ypt3Hw.jpeg?1697999882)

## 중요 사항

**1. 템플릿 라이브러리에서 템플릿을 추가하거나 편집할 수 있는 사람은 누구인가요?**

에이전시 관리자가 모든 하위 계정에서 템플릿을 추가하고 관리할 수 있습니다.

**2. 모든 요금제에서 템플릿 추가가 가능한가요?**

템플릿 라이브러리에 자체 템플릿을 추가하는 기능은 $497 요금제 이상에서 사용할 수 있습니다.

**3. 템플릿 라이브러리에 추가된 템플릿에 업데이트가 있을 때 어떻게 처리되나요?**

- 템플릿이 초안 단계일 때는 업데이트된 모든 요소의 기록을 유지하고 최신 상태를 유지합니다.
- 템플릿이 공개 상태일 때는 새로운 요소로 템플릿을 업데이트하지 않습니다. 향후 릴리스에서 업데이트 기능을 지원할 예정입니다.

**4. 템플릿 라이브러리에 이미 있는 템플릿을 숨기거나 편집할 수 있나요?**

템플릿 라이브러리에 이미 추가된 템플릿을 숨기거나 편집하는 기능은 지원하지 않습니다. 에이전시 관리자가 생성한 템플릿 기록을 언제든지 복제하고 편집할 수 있습니다. 관리자는 이미 있는 태그와 카테고리를 숨기거나 편집할 수 없습니다.
