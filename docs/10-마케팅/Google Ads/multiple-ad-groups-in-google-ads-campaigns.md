# **Google Ads 캠페인의 다중 광고 그룹이란?**

Google Ads 캠페인의 다중 광고 그룹 기능을 사용하면 최대 10개의 개별 광고 그룹으로 캠페인을 구성할 수 있습니다. 각 광고 그룹은 특정 키워드 세트나 오디언스 세그먼트를 타겟팅하며, 최대 3개의 광고를 포함할 수 있습니다. 이를 통해 더욱 유연하고 세분화된 타겟팅과 향상된 성과 가시성을 얻을 수 있으며, 모든 관리를 Hyperclass의 광고 관리자에서 직접 수행할 수 있습니다.

# **다중 광고 그룹의 주요 이점**

**캠페인을 위한 더 스마트한 구조와 향상된 인사이트**

다중 광고 그룹을 사용하면 타겟팅에서 유연성을 얻고 더 통제된 실험이 가능해집니다. 주요 이점은 다음과 같습니다:

- 서로 다른 제품, 서비스 또는 오디언스를 타겟팅하기 위해 캠페인당 최대 **10개의 광고 그룹**을 생성할 수 있습니다.

- 구조화된 A/B 테스트를 위해 광고 그룹당 최대 **3개의 광고**를 추가할 수 있습니다.

- 광고 그룹별로 **기본 URL**을 설정하여 일관성을 유지하면서도 고유한 표시 경로를 허용합니다.

- 정확한 타겟팅을 위해 **키워드 제안**을 생성하고 매칭 유형을 선택할 수 있습니다.

- **다중 레벨에서 성과 추적**: 캠페인, 광고 그룹, 광고, 키워드 단위로 성과를 추적할 수 있습니다.

- 줄임표(⋮) 메뉴를 사용하여 광고 그룹과 광고를 쉽게 **관리**(복제, 삭제, 추가)할 수 있습니다.

- **검토 화면**을 사용하여 게시하기 전 전체 캠페인 설정을 미리 볼 수 있습니다.

# **Google Ads 캠페인에서 다중 광고 그룹 사용법**

**타겟팅과 테스트 기회를 극대화하기 위한 다중 광고 그룹 설정 및 관리 방법 알아보기**

다중 광고 그룹이 포함된 캠페인을 생성하려면 다음 단계를 따르세요:

- **캠페인 시작**

**광고 관리자(Ad Manager)**로 이동하여 새 캠페인을 만들거나 미리 만들어진 템플릿을 선택합니다.

- **캠페인 레벨 설정**

**입찰 전략**, **일일 예산**, **오디언스/네트워크 기본 설정**을 정의합니다.

- **광고 그룹 구성**

각 광고 그룹에 대해:

키워드 제안을 생성하기 위해 **웹사이트 기본 URL**을 입력합니다.

- 원하는 **매칭 유형**(구문, 완전 일치, 광범위)으로 키워드를 선택하거나 수동으로 입력합니다.

- 광고 그룹당 최대 **3개의 광고**를 추가합니다.

- **줄임표 메뉴**를 사용하여 광고 그룹을 복제, 삭제 또는 추가하며 관리합니다.

- **광고 구성**

각 광고에 대해:

최대 **15개의 헤드라인**을 추가합니다.

- 최대 **4개의 설명**을 추가합니다.

- **표시 URL**을 설정합니다.

- **검토 및 게시**

**검토(Review)** 화면을 사용하여 모든 광고 그룹과 광고에 걸친 전체 캠페인 구조를 확인합니다.

- 준비가 되면 **게시(Publish)**를 클릭합니다.

- **성과 분석**

**광고 관리자 통계(Ad Manager Statistics)** 화면으로 이동합니다.

- 캠페인 성과를 확인합니다.

- 캠페인 이름을 클릭하여 드릴다운하고 **광고 그룹**, **광고**, **키워드**별 통계를 확인합니다.

# **캠페인 구조 개요**

**캠페인 레벨이 어떻게 구성되어 있으며, 이 구조가 명확성과 성과 추적 개선에 어떻게 도움이 되는지 이해하기**

캠페인 플로우는 이제 3개의 직관적인 레벨로 명확하게 구분됩니다:

- **캠페인**

예산, 입찰 전략, 타겟 오디언스를 설정합니다.

- **광고 그룹**

키워드 타겟팅과 기본 URL을 정의합니다.

- 그룹 내 광고를 관리합니다.

- **광고**

헤드라인, 설명, 표시 경로와 같은 크리에이티브 요소를 구성합니다.

이러한 모듈식 접근 방식을 통해 더 나은 **타겟팅**, 더 쉬운 **테스트**, 더 실행 가능한 **인사이트**를 얻을 수 있습니다. 특히 여러 고객을 관리하거나 복잡한 광고 전략을 운영하는 에이전시에게 유용합니다.

# **자주 묻는 질문**

**Q: 단일 캠페인에서 몇 개의 광고 그룹을 만들 수 있나요?**

캠페인당 최대 **10개의 광고 그룹**을 추가할 수 있습니다.

**Q: 각 광고 그룹에서 몇 개의 광고를 만들 수 있나요?**

각 광고 그룹은 최대 **3개의 광고**를 가질 수 있습니다.

**Q: 각 광고마다 다른 기본 URL을 설정할 수 있나요?**

아니요. 광고 그룹 내의 모든 광고는 광고 그룹 레벨에서 설정되는 공통 기본 URL을 공유해야 합니다.

**Q: 전체 광고 그룹을 복제할 수 있나요?**

네, **줄임표(⋮) 메뉴**를 사용하여 더 빠른 설정을 위해 광고 그룹을 복제할 수 있습니다.

**Q: 광고 그룹이나 키워드별 성과를 볼 수 있나요?**

네. 통계 화면에서 이제 **캠페인, 광고 그룹, 광고, 키워드별 성과 분석**을 지원합니다.

**Q: 기존 캠페인은 이 기능과 호환되나요?**

네. 기존 캠페인 플로우, 템플릿, 스냅샷은 계속 지원되며 호환됩니다.

**Q: 같은 광고 그룹 내에서 다른 매칭 유형을 사용할 수 있나요?**

네. 키워드별로 다른 매칭 유형(구문, 완전 일치, 광범위)을 선택할 수 있습니다.

**Q: 광고 그룹을 하나만 만들면 어떻게 되나요?**

캠페인은 여전히 작동합니다. **유효한 캠페인을 위해서는 최소 1개의 광고 그룹과 1개의 광고가 필요**합니다.
