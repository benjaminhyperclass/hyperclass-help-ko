# 제휴 관리: 개선된 미디어 라이브러리

고객 여러분의 피드백을 듣고 제휴 관리(Affiliate Manager)의 미디어 라이브러리를 대폭 개선했습니다! 이제 미디어 파일을 업로드하고, 다운로드하고, 관리하는 것이 그 어느 때보다 쉬워졌습니다. 게다가 미디어를 특정 캠페인에 연결하거나 모든 캠페인에서 사용할 수 있도록 설정할 수 있어, 여러분과 제휴 파트너 모두에게 더욱 효율적인 경험을 제공합니다.

**목차**

- [새로운 기능](#새로운-기능)
- [사용 방법](#사용-방법)
- [왜 중요한가요](#왜-중요한가요)

## **새로운 기능**

- **개선된 사용자 인터페이스**: 미디어 라이브러리가 최신 디자인으로 새롭게 개편되어 더욱 깔끔하고 직관적인 사용 경험을 제공합니다.

- **캠페인별 미디어 관리**: 이제 미디어를 개별 캠페인에 할당하거나 모든 캠페인에서 접근 가능하도록 설정할 수 있어, 제휴 파트너들이 자신의 작업과 관련된 자료만 볼 수 있습니다.

- **빠른 복사 및 다운로드**: 제휴 파트너들이 제휴 포털(Affiliate Portal)에서 미디어 링크를 쉽게 복사하거나 자료를 직접 다운로드할 수 있어 시간과 노력을 절약할 수 있습니다.

## **사용 방법:**

- 제휴 관리(Affiliate Manager)로 이동해서 "Media(미디어)"를 선택합니다
- "Add File(파일 추가)"를 클릭해서 새 미디어 파일을 업로드합니다
- 미디어를 특정 캠페인에 연결할지 또는 모든 캠페인에서 사용 가능하게 할지 선택합니다
- 변경사항을 저장하면 제휴 파트너들이 미디어를 사용할 수 있습니다

## **왜 중요한가요:**

- **효율성 향상**: 미디어 자료 관리가 더욱 빠르고 간단해져서, 파일 정리에 소요되는 시간이 줄어들고 제휴 파트너들이 필요한 자료를 빠르게 찾을 수 있습니다.

- **맞춤형 미디어 접근**: 미디어를 특정 캠페인에 연결함으로써 제휴 파트너들이 자신의 캠페인과 관련된 자료만 보게 되어 혼란을 방지하고 생산성을 높일 수 있습니다.

- **향상된 제휴 파트너 경험**: 쉬운 다운로드와 복사 기능으로 제휴 파트너들이 미디어에 손쉽게 접근하고 공유할 수 있어 작업 흐름이 개선됩니다.
