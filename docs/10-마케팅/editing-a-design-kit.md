# 디자인 키트 편집하기

Hyperclass에서 구축하는 모든 곳에서 브랜드 일관성을 유지하세요. 이 가이드에서는 디자인 키트(브랜드 보드 내)를 편집하는 방법을 설명합니다. 색상, 로고, 폰트를 업데이트하여 브랜딩이 항상 최신 상태를 유지하도록 하는 방법을 알아보세요!

## 디자인 키트란 무엇인가요?

디자인 키트는 Hyperclass의 브랜드 보드에 저장된 브랜드 자산(색상, 로고, 폰트) 세트입니다. 웹사이트, 퍼널, 이메일 전반의 편집기와 선택기가 기본 브랜드 보드의 디자인 키트를 참조하므로, 매번 스타일을 다시 만들지 않고도 더 빠르게 작업하며 브랜드 일관성을 유지할 수 있습니다.

하지만 브랜드의 시각적 요소 변경을 반영하기 위해 브랜드 보드를 업데이트해야 할 때가 있습니다. 디자인 키트를 편집하면 색상을 새로 고치고, 로고를 교체하고, 타이포그래피를 조정할 수 있어 향후 페이지와 이메일에서 최신 브랜드 자산을 사용할 수 있습니다. 기존 콘텐츠는 변경되지 않습니다.

---

## 언제 디자인 키트를 편집해야 하나요?

- **브랜드 로고 변경:** 비즈니스가 발전하면서 로고를 새로 디자인할 수 있습니다. 브랜드 보드를 업데이트하면 새 로고가 모든 플랫폼에서 일관되게 사용됩니다.

- **새로운 색상이나 폰트:** 새로운 색상 팔레트나 폰트를 도입하는 것은 리브랜딩의 일부이거나 단순히 룩을 새로 고치기 위한 업데이트일 수 있습니다. 브랜드 보드를 편집하면 이러한 변경사항이 모든 마케팅 자료에 정확하게 반영됩니다.

- **브랜드 요소의 일관성:** 폰트나 색상의 사소한 변경사항도 시각적 일관성을 유지하고 브랜드 인지도를 강화하기 위해 브랜드 보드에서 업데이트해야 합니다.

## 디자인 키트 편집 방법

#### **1단계:** 브랜드 보드로 이동

**Marketing(마케팅)** → Brand Boards(브랜드 보드) (또는 Settings(설정) → Brand Boards(브랜드 보드))로 이동하세요.

![브랜드 보드 메뉴 이동](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054179753/original/fv_yRBsfl20ohGd8xfSrod38K0Ie1wpitw.png?1758311286)

#### **2단계:** 브랜드 보드 열기

편집하려는 브랜드 보드를 클릭하세요.

![브랜드 보드 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054179777/original/n6uAK6RUGKqDLl6NLFzUItQ_VFbWkG3YQw.png?1758311401)

#### **3단계:** 로고 업데이트

- 편집기 상단 왼쪽의 **Logos(로고)** 탭을 클릭하세요
- **Logo 1** 또는 **Logo 2** 옆의 **Edit(편집)** (연필) 아이콘을 클릭하여 로고를 교체하거나 업로드하세요

참고: 미디어 라이브러리에서 이전 로고를 삭제하려면 미디어 라이브러리에서 직접 삭제해야 합니다.

![로고 편집](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054179854/original/PURuHCYb8Q9ElOEUdcCez_75QvaAJqgx_w.png?1758311686)

#### **4단계:** 색상 업데이트

- 편집기 상단 왼쪽의 **Colors(색상)** 탭을 클릭하세요
- 새 색상을 추가하려면 **+ Add(추가)**를 클릭하세요
- 색상을 변경하려면 색상 이름 왼쪽의 색상 사각형을 클릭한 다음 새 색상을 선택하세요
- 색상 이름 필드를 클릭하여 색상 이름을 변경하세요
- **휴지통** 아이콘을 클릭하여 색상을 삭제하세요

![색상 편집](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054179900/original/uCddbb7iPCUvXUp5r9Dq8knlbnZYTobfjQ.png?1758311798)

#### **5단계:** 폰트 업데이트

- 편집기 상단 왼쪽의 **Typography(타이포그래피)** 탭을 클릭하세요
- **+ Add(추가)**를 클릭하여 폰트를 추가하세요 (최대 5개). 최소 1개 폰트는 필수입니다
- 드롭다운을 사용하여 선택한 폰트를 변경하세요
- **휴지통** 아이콘을 클릭하여 폰트를 제거하세요

![폰트 편집](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054179956/original/jRHSUURbX7wnmJGk8NzdfjKVAV4q4A45cw.png?1758311992)

#### **6단계:** 변경사항 저장

조정을 완료한 후 오른쪽 상단의 파란색 **Save Board(보드 저장)** 버튼을 클릭하여 변경사항을 저장하세요.

![변경사항 저장](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054180021/original/20ZSCDJsn9t27M6cReQ16IAXzoqlvxK7qQ.png?1758312160)

## 자주 묻는 질문

**Q: 디자인 키트 편집이 기존 페이지나 이메일을 자동으로 업데이트하나요?**
아니요. 편집사항은 향후 편집을 위한 선택기에 표시되는 내용에만 영향을 줍니다. 기존 자산은 자동으로 변경되지 않습니다.

**Q: 내 색상이 페이지나 이메일 색상 선택기에 나타나지 않는 이유는 무엇인가요?**
기본 브랜드 보드만 선택기에 표시됩니다. 올바른 보드를 기본값으로 설정하고 색상이 저장되었는지 확인하세요. [기본 브랜드 보드 설정 방법 알아보기](how-to-change-default-brand-boards-.md)

**Q: 어떤 색상 형식을 입력할 수 있나요?**
브랜드 색상은 HEX와 RGB를 지원합니다. 색상 선택기 왼쪽 하단의 "HEX" 또는 "RGBA" 라벨을 클릭하여 두 형식 간에 전환할 수 있습니다.

**Q: 브랜드 보드를 업데이트하면 결제 링크가 변경되나요?**
결제 링크가 브랜드 보드 테마를 사용하는 경우, 색상 업데이트가 링크 전반에 반영될 수 있습니다.

**Q: 빌더에 표시되는 보드를 어떻게 전환하나요?**
원하는 보드를 기본값으로 설정하세요. 기본 보드가 선택기에 나타나는 디자인 키트 자산을 제어합니다.
