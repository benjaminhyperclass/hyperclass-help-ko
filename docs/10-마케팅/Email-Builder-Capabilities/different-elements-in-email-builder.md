# 이메일 빌더의 다양한 요소들

이메일 빌더의 드래그 앤 드롭 요소(Elements)로 이메일을 어떻게 디자인하고 커스터마이징할지 완전히 제어해 보세요. 텍스트와 버튼부터 이미지, 영상, 카운트다운 타이머까지, 요소들을 활용하면 코딩 없이도 시각적으로 매력적이고 기능적인 이메일을 제작할 수 있어요.

## 요소란 무엇인가요?

요소는 코드를 작성하지 않고도 이메일 레이아웃과 콘텐츠를 쉽게 커스터마이징할 수 있게 해주는 미리 만들어진 콘텐츠 구성 요소예요. 각 요소는 텍스트, 이미지, 소셜 링크, 동적 콘텐츠 추가 등 특정 목적을 위해 사용돼요.

---

## 요소 목록과 활용법

| **요소** | **설명** | **주요 활용 사례** |
|----------|----------|-------------------|
| **Text** | 이메일에 텍스트 콘텐츠를 추가해요. 폰트, 색상, 정렬로 스타일링할 수 있어요. | 제목, 설명, CTA, 본문 콘텐츠 추가 |
| **Image** | 이미지나 배너를 삽입해서 이메일을 시각적으로 매력적으로 만들어요. | 제품, 로고, 공지사항 보여주기 |
| **Button** | 클릭할 수 있는 행동 유도(CTA) 버튼을 추가해요. | '지금 예약', '지금 구매', '더 알아보기' 등의 행동 유도 |
| **Logo** | 브랜드 로고를 삽입해요. | 이메일 상단이나 하단에 브랜드 정체성 강화 |
| **Divider** | 섹션 간 시각적 구분선을 만들어요. | 가독성과 구조 개선 |
| **Social** | 소셜 미디어 프로필로 연결되는 아이콘을 추가해요. | 사용자를 소셜 채널로 유도 |
| **Footer** | 연락처 정보, 구독 해지 링크, 법적 고지사항을 추가해요. | 필수 준수 정보 제공 |
| **Code** | 커스텀 HTML이나 코드 스니펫을 삽입해요. | 추적 픽셀, 동적 콘텐츠, 커스텀 디자인 추가 |
| **Video** | 재생 버튼이 있는 YouTube나 Vimeo 영상 미리보기를 삽입해요. | 튜토리얼, 공지사항, 후기 공유 |
| **Shopping Cart** | 연결된 이커머스 스토어의 상품을 보여줘요. | 고객이 장바구니에 남겨둔 상품 표시 |
| **RSS Header / RSS Items** | RSS 피드의 최신 게시물을 자동으로 표시해요. | 뉴스레터나 블로그 업데이트에 유용 |
| **FAQ** | 접을 수 있는 질문-답변 섹션을 추가해요. | 도움말 콘텐츠나 자주 묻는 질문 공유 |
| **Products** | 이미지, 가격, 구매 버튼이 있는 상품 목록을 표시해요. | 추천 상품이나 신상품 홍보 |
| **Image Slider** | 여러 이미지를 캐러셀로 만들어요. | 여러 제품이나 이벤트 강조 |
| **Preview URL** | 웹사이트 스니펫을 보여주는 링크 미리보기 상자를 추가해요. | 블로그 게시물이나 외부 콘텐츠 홍보 |
| **Countdown Timer** | 실시간 타이머를 추가해서 긴급감을 조성해요. | 한정 시간 제안이나 이벤트에 완벽 |
| **Spacer** | 요소 간 빈 공간을 추가해요. | 이메일 레이아웃 간격 세밀 조정 |
| **Review Link** | 수신자에게 피드백이나 리뷰를 요청하는 링크를 삽입해요. | 후기나 만족도 데이터 수집 |
| **Forms** | 이메일에 폼을 직접 삽입해요 (클라이언트 지원 시). | 설문 답변이나 등록 등 사용자 입력 받기 |

## 요소 추가하는 방법

#### **1단계:** 이메일로 이동

- 대시보드에서 좌측 내비게이션 바의 Marketing(마케팅)을 선택하세요

- 상단 내비게이션 리본에서 **Emails(이메일)**을 클릭하세요

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055816469/original/XmDxoPvoq8tTprWM9h9fmOneek1GUunxnQ.jpeg?1760337642)

#### **2단계:** 새 이메일 만들기

- 새 이메일 생성을 위한 시작 타입을 선택하세요.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055816736/original/owoiLxXMaWjljXCSNvnxAWZdjI4Dl3ImZA.png?1760337847)

#### 3단계: 요소 추가

- Elements(요소) 탭에서 원하는 요소를 이메일 캔버스로 드래그 앤 드롭하세요.

- 요소를 클릭해서 콘텐츠, 스타일, 설정을 커스터마이징하세요.

## 자주 묻는 질문

**Q: 요소를 추가한 후 재배열할 수 있나요?**

네, 이메일 레이아웃에서 요소를 어디든 드래그 앤 드롭으로 이동시킬 수 있어요.

**Q: 모든 요소가 모바일에서 반응형인가요?**

네, 모든 요소는 기본적으로 반응형이에요. Mobile View(모바일 뷰) 탭에서 모바일 간격이나 정렬을 미리보고 조정할 수 있어요.

**Q: "Text"와 "Code" 요소의 차이점이 뭔가요?**

Text 요소는 시각적으로 콘텐츠를 추가하고 포맷할 수 있게 해주고, Code 요소는 고급 디자인이나 연동을 위한 커스텀 HTML을 삽입할 수 있게 해줘요.

**Q: "Spacer"와 "Divider"의 차이점이 뭔가요?**

Spacer는 빈 수직 공간을 추가하고, Divider는 섹션 간 시각적 구분을 위한 보이는 선을 추가해요.
