# 이메일 빌더의 로고 요소

로고 요소(Logo Element)를 사용하면 이메일 템플릿에 브랜드 로고를 쉽게 삽입할 수 있어, 모든 커뮤니케이션에서 일관된 브랜딩과 시각적 아이덴티티를 유지할 수 있습니다. 로고를 업로드하고 크기를 조정하며 정렬할 수 있고, 데스크톱과 모바일 기기 모두에서 선명하게 표시되도록 할 수 있습니다.

## **이메일 빌더의 로고 요소란 무엇인가요?**

로고 요소(Logo Element)는 드래그 앤 드롭으로 회사나 브랜드 로고를 이메일 레이아웃에 추가할 수 있는 블록입니다. 즉각적인 브랜드 인식을 만들어내고 메시지에 전문적인 분위기를 조성하는 데 도움이 됩니다.

로고 이미지를 직접 업로드하거나 호스팅된 이미지 링크를 사용할 수 있습니다. 이 요소에는 크기 조정, 정렬, 링크 설정 등의 커스터마이징 옵션이 포함되어 있어 디자인 내에서 로고가 어떻게 표시될지 제어할 수 있습니다.

참고: 최상의 결과를 위해 투명한 PNG 파일이나 고품질 JPG 파일을 사용하세요.

## **로고 요소의 주요 장점**

이메일 디자인에 로고 요소를 추가하면 다음과 같은 여러 가지 장점이 있습니다:

- 브랜드 인식과 일관성을 강화합니다
- 쉬운 드래그 앤 드롭 기능으로 로고 배치를 간소화합니다
- 유연한 레이아웃 옵션을 위한 크기 조정, 정렬, 링크 설정을 지원합니다
- 모든 기기와 화면 크기에서 선명하고 깨끗한 표시를 보장합니다
- 로고를 홈페이지나 랜딩 페이지에 링크하여 쉬운 네비게이션을 제공합니다

## **로고 요소 설정 방법**

이메일 템플릿에 로고 요소를 추가하고 설정하려면 다음 단계를 따르세요:

- 하위 계정 대시보드에서 `Marketing(마케팅) → Emails(이메일) → Templates(템플릿)`로 이동합니다.

![이메일 템플릿 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155057292426/original/nY0mLhEspoXmlh8ONd6mp1_qOkY6f_LksQ.png?1761905787)

- 기존 이메일 템플릿을 열거나 `+ New` 버튼을 클릭해서 새로 만듭니다.

![새 이메일 템플릿 만들기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155057292443/original/le0YDia2NBkLckdHWNxoaTclVpmJ6r7PFg.jpeg?1761905801)

- 이메일 빌더에서 좌측 사이드바의 `Add Elements(+)` 아이콘을 클릭합니다.

![요소 추가 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155057292646/original/lnAEEIGI4tn3rtF4xB6TEkTSlEpQWX3KdA.png?1761905897)

- Elements 섹션에서 로고 요소(Logo Element)를 찾습니다.

- 로고 요소를 원하는 위치(보통 레이아웃 상단)로 드래그 앤 드롭합니다.

- 로고 요소를 클릭하면 좌측 패널에서 Edit 탭이 열립니다. 여기서 다음을 설정할 수 있습니다:

로고 업로드: 기기에서 이미지를 선택합니다.

- 이미지 URL 사용: 호스팅된 로고 이미지의 직접 링크를 붙여넣습니다.
- 크기 조정: 적절한 스케일링을 위해 커스텀 너비와 높이를 설정합니다.
- 정렬 설정: 로고를 이메일 레이아웃의 왼쪽, 가운데, 오른쪽으로 정렬합니다.
- 링크 추가: 로고를 웹사이트나 관련 페이지에 링크합니다.
- 패딩 제어: 균형 잡힌 레이아웃을 위해 로고 주변 여백을 추가합니다.

![로고 요소 설정 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155057292733/original/JrDkfi5fHuPTT0lGwAL3gmFeBKwQRHjL2w.png?1761905940)

- 필요한 경우 Visibility 탭을 클릭하여 기기별 노출 설정(Device-Based Visibility)이나 조건부 발송 설정(Conditional Sending)을 구성합니다.

![로고 노출 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155057293818/original/IEWCCbhwhzO2_R3DwcIiwTcW4i-pJyIa-w.png?1761906349)

- 데스크톱과 모바일 모두에서 이메일을 미리보기하여 로고가 완벽하게 표시되는지 확인합니다.

## 자주 묻는 질문

**Q: 로고에 지원되는 파일 형식은 무엇인가요?**

PNG, JPG, GIF 파일을 사용할 수 있습니다. 투명한 배경을 위해서는 PNG를 권장합니다.

**Q: 로고를 내 웹사이트에 링크할 수 있나요?**

네, 설정에서 하이퍼링크를 추가하여 로고를 클릭할 수 있게 만들 수 있습니다.

**Q: 모바일에서 로고가 자동으로 크기가 조정되나요?**

네, 로고 요소는 반응형이며 작은 화면에 맞게 자동으로 조정됩니다.
