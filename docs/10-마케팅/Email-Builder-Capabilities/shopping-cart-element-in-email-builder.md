# 이메일 빌더의 쇼핑카트 요소

쇼핑카트 요소(Shopping Cart Element)를 사용하면 이메일 내에서 직접 상품 정보, 가격, 결제 링크를 표시할 수 있습니다. 상품을 홍보하고, 장바구니 이탈을 방지하며, 고객의 구매 완료를 유도하는 동시에 이메일을 시각적으로 매력적이고 전환에 최적화된 상태로 유지하도록 설계되었습니다.

## **이메일 빌더의 쇼핑카트 요소란?**

쇼핑카트 요소는 이메일 내부에 상품 상세 정보와 결제 링크를 추가할 수 있는 드래그 앤 드롭 블록입니다. 주로 장바구니 이탈 알림, 상품 추천, 또는 한정 시간 프로모션에 사용됩니다.

각 상품의 이미지, 이름, 설명, 가격, 버튼 링크를 커스터마이징할 수 있어 이메일 레이아웃에서 상품이 어떻게 표시될지 완전히 제어할 수 있습니다.

참고: 쇼핑카트 요소는 직접 결제를 처리하지 않습니다. 대신 사용자를 웹사이트나 스토어의 결제 페이지로 연결하여 구매를 완료하도록 합니다.

## **쇼핑카트 요소의 주요 장점**

이메일 디자인에서 쇼핑카트 요소를 사용하면 다음과 같은 장점이 있습니다:

- 깔끔하고 구조화된 레이아웃으로 상품 상세 정보와 가격을 표시
- 직접 결제 링크가 포함된 "지금 구매" 또는 "장바구니로 돌아가기" 버튼 추가
- 시각적으로 최적화된 장바구니 알림을 통한 전환율 증대
- 브랜드에 맞는 색상, 간격, 글꼴을 쉽게 커스터마이징
- 데스크톱과 모바일 모두에서 아름답게 표시되는 완전 반응형 디스플레이

## **쇼핑카트 요소 설정 방법**

이메일 템플릿에 쇼핑카트 요소를 추가하고 구성하려면 다음 단계를 따르세요:

- 하위 계정 대시보드에서 `Marketing(마케팅) → Emails(이메일) → Templates(템플릿)`로 이동합니다.

![하위 계정 대시보드에서 마케팅 > 이메일 > 템플릿 메뉴 경로](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155057309221/original/C3iYBnaXq8WzbZXbxqj452ol322L6jWakQ.png?1761914253)

- 기존 템플릿을 열거나 `+ New(+ 새로 만들기)` 버튼을 클릭해서 새 템플릿을 만듭니다.

![템플릿 목록에서 새 템플릿 생성 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155057309292/original/JAFAOxuYJhdNVg1lttZC40a0V-iS-O9HLQ.jpeg?1761914282)

- 이메일 빌더에서 좌측 사이드바의 `Add Elements(+ 요소 추가)` 아이콘을 클릭합니다.

![이메일 빌더의 요소 추가 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155057309433/original/k1veiJZ-qY3a6MRAVjo24RKZpSnMV0TxBA.png?1761914350)

- Elements(요소) 섹션에서 Shopping Cart Element(쇼핑카트 요소)를 찾습니다.

- 쇼핑카트 요소를 이메일 레이아웃의 원하는 섹션으로 드래그 앤 드롭합니다.

- 쇼핑카트 요소를 클릭하여 좌측 패널의 Edit(편집) 탭을 엽니다. 여기서 다음을 할 수 있습니다:

레이아웃과 간격 조정: 패딩, 마진, 정렬을 제어하여 깔끔한 상품 그리드를 만듭니다.

![쇼핑카트 요소 편집 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155057312576/original/B_L-d7WgRd-ozrqfxFwywFGL481w9Xd_OA.png?1761915887)

- (선택사항) Visibility(표시) 탭을 사용하여 디바이스별 표시 설정이나 조건부 발송 규칙을 구성합니다.

- 데스크톱과 모바일에서 이메일을 미리 보기하여 상품이 명확하게 표시되고 링크가 정상적으로 작동하는지 확인합니다.
