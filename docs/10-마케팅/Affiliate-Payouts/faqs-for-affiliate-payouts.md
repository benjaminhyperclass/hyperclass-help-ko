## 자동 지급 기능을 사용하려면 어떤 단계가 필수인가요?

이 기능을 활성화하고 사용하려면 두 가지 필수 요건이 있습니다:

- PayPal 비즈니스 계정 연동 (가이드)
- 지급 방식(Payout Methods)에 제휴사의 PayPal 추가 ([가이드](how-to-add-link-a-new-payout-method-for-affiliates.md)

## 제휴 수수료 지급을 위해 PayPal을 연동한 후 퍼널/웹사이트 주문 양식에서 PayPal 버튼을 숨기려면 어떻게 하나요?

주문 양식에서는 PayPal 버튼을 숨기면서 제휴 수수료 지급에는 PayPal을 계속 사용하려면 다음 단계를 따르세요:

- PayPal 버튼을 숨기고 싶은 퍼널(Funnel) 또는 웹사이트(Website)를 여세요.
- **설정(Settings)** 탭으로 이동하세요.
- **Head Tracking Code** 섹션에 다음 코드를 붙여넣으세요:

```css
<style>.paypal-content {
  display: none;
}
.order-form-v2 > div > .divider {
  display: none;
}
</style>
```

- **저장(Save)** 버튼을 클릭하세요.

이렇게 하면 제휴 수수료 지급을 위한 PayPal 연동은 유지하면서 주문 양식에서 PayPal 옵션만 숨길 수 있습니다.

## PayPal 자동 지급을 사용해서 제휴사에게 수수료를 지급할 수 있는 권한은 누가 가지고 있나요?

**에이전시 관리자(Agency Admins)**와 **계정 관리자(Account Admins)**만 PayPal 자동 지급 방식을 사용해서 제휴사에게 수수료를 지급할 권한을 가지고 있습니다.

## PayPal을 사용할 때 구독 상품의 수수료 지급은 어떻게 처리되나요?

구독 상품의 수수료는 자동으로 생성되지만, PayPal을 통한 결제를 완료하려면 여전히 **"지금 지급(Pay Now)"** 버튼을 수동으로 클릭해야 합니다. 보안 및 인증 문제로 인해 Hyperclass에서는 수수료를 자동으로 처리하지 않습니다.

## 일부 제휴사에 대해 PayPal로 수수료를 지급할 수 없는 이유는 무엇인가요?

지급 방식으로 PayPal 계정을 추가한 제휴사에게만 수수료를 지급할 수 있습니다. 제휴사가 PayPal 계정을 연결하지 않은 경우 오류 아이콘이 표시되고 "계정 추가(Add Account)" 옵션을 볼 수 있습니다.
