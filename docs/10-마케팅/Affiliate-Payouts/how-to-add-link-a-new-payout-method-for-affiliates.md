**중요한 안내**: 제휴 파트너 지급 방법 설정
이제 제휴 파트너에게 제공할 지급 방법(PayPal, 은행 송금)을 직접 제어할 수 있습니다.

**설정 방법:**
1. Affiliate Manager(제휴 관리) → Settings(설정)으로 이동
2. Affiliate Portal Settings(제휴 포털 설정) → Configure Payout Methods(지급 방법 구성) 클릭
3. 활성화할 지급 방법 선택
4. 변경사항 저장

제휴 파트너는 관리자가 활성화한 옵션만 볼 수 있어, 더 원활하고 체계적인 지급 프로세스를 제공합니다.

## 관리자용

### 1단계: 제휴 파트너 프로필 페이지로 이동

- Affiliate Manager(제휴 관리)에서 해당 제휴 파트너의 프로필 페이지에 접속하세요.

### 2단계: 지급 방법 탭으로 이동

- 아래로 스크롤하여 "Payout Methods(지급 방법)" 탭으로 전환하세요.

### 3단계: 지급 방법 추가

- "Add Payout Method(지급 방법 추가)"를 클릭하세요.
![지급 방법 추가 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155025232191/original/3RowXIU6AnU3FrHbZeCakFkUTLz3YQ70FA.png?1714391840)

### 4단계: 지급 방법을 선택하고 필수 항목 입력

- PayPal의 경우: 제휴 파트너의 유효한 PayPal 이메일 주소를 입력하세요.
![PayPal 이메일 입력](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155025232235/original/E_I9HKVWL-7IYo9_Gwo3zi2HW1Cn1CPIGw.png?1714391851)

- 은행 계좌의 경우: 국가를 선택하고 필수 정보를 입력하세요.

### 5단계: 완료 클릭

- 정보를 모두 입력한 후 "Done(완료)"를 클릭하여 지급 방법을 저장하세요.
![완료 버튼 클릭](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155025232272/original/sIM3G6KiYqr8k9gsPeSTaH4IafASAow9Cw.png?1714391880)

- 한 제휴 파트너에게 여러 지급 방법을 추가하고, 그 중 하나를 기본 지급 방법으로 선택할 수 있습니다.
![여러 지급 방법 관리](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044056952/original/1M-5szwAbsiGGhN10kOjfbF3pruZQQZXeA.png?1743051536)

## 제휴 파트너용

### 1단계: 제휴 포털 접속

- 제공받은 매직 링크를 사용해 제휴 포털에 로그인하세요.

### 2단계: 수수료 탭으로 이동

- 포털 내에서 "Payout Methods(지급 방법)" 탭으로 전환하세요.
![지급 방법 탭](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044057041/original/NM0Ruzh1JcYkhfqgODg4ORyQBUnqHuYlGA.png?1743051690)

### 3단계: 지급 방법 추가

- "Add Payout Method(지급 방법 추가)"를 클릭하세요.

### 4단계: 지급 방법을 선택하고 필수 항목 입력

- PayPal의 경우: 유효한 PayPal 이메일 주소를 입력하세요.
![PayPal 이메일 입력](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155025232378/original/0KYFgdyfAWlaNP3f3iY03_ImJq56d34JSw.png?1714391935)

- 은행 계좌의 경우: 국가를 선택하고 필수 정보를 입력하세요.

### 5단계: 완료 클릭

- 정보를 모두 입력한 후 "Done(완료)"를 클릭하여 지급 방법을 저장하세요.

**참고**: 현재 PayPal API는 계정 유효성 검사를 지원하지 않으므로, 제휴 파트너의 유효한 PayPal 이메일 주소를 추가할 때 주의 깊게 확인해 주세요.
