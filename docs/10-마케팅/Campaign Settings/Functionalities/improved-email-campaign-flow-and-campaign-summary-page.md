# 개선된 이메일 캠페인 플로우와 캠페인 요약 페이지

직접 생성, 명확한 경고, 향상된 미리보기를 제공하는 간소화된 이메일 캠페인을 소개합니다.

## 템플릿 라이브러리에서 바로 캠페인 시작

누락되거나 잘못된 필드에 대한 체크리스트를 제공하여 발송 전 문제를 빠르게 해결할 수 있습니다.

## 발송/예약 화면 UI 개선

제목과 미리보기 텍스트를 작성하는 동안 이메일 디자인을 확인할 수 있어 더욱 원활한 워크플로우를 제공합니다.

## 사용 방법:

핵심 기능이 어디에 있는지 알면 캠페인 제작 과정이 빨라집니다. 편집 중 미리보기(Preview) 버튼을 사용하면 형식과 개인화 문제를 조기에 발견할 수 있고, 첨부파일 개수 배지는 발송 전 즉시 명확한 확인을 제공합니다.

- 이메일 마케팅(Email Marketing) → 캠페인(Campaigns)으로 이동합니다.

- 새로 만들기(New) → 이메일 마케팅 템플릿(Email Marketing Templates)을 클릭합니다.

- 템플릿을 선택하고 계속(Continue)을 클릭합니다.

- 캠페인 빌더에서 이메일을 엽니다.

- 편집 중 언제든지 상단 메뉴바의 미리보기(Preview) 버튼을 클릭하여 이메일을 검토합니다(저장(Save) 버튼 옆에 있음). 메뉴 없이 한 번만 클릭하면 됩니다.

- 준비가 되면 발송(Send) 또는 예약(Schedule)으로 진행합니다.

- 발송/예약 화면에서 파일 첨부(Attach Files) 옆에 첨부된 파일 개수를 보여주는 숫자 배지가 표시됩니다.

- 발송 전 빠른 한눈 확인용으로 활용하세요.

- 필수 필드(Required Fields)의 경고를 확인한 후 확인 및 발송(Confirm & Send)을 클릭합니다.

## 참고사항:

현재 체크리스트는 필수 필드와 커스텀 필드의 오류에 대한 정보만 제공합니다. 향후 이메일 발송 한도 경고, 콘텐츠 문제 등으로 기능을 확장할 예정입니다.
