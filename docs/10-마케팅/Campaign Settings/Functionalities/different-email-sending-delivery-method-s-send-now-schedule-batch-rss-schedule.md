# 이메일 발송/전달 방식 - 지금 보내기, 예약, 배치, RSS 예약

Hyperclass는 이메일을 보내거나 예약하는 여러 옵션을 제공하여, 캠페인 전달 방식을 유연하게 선택할 수 있습니다. 즉시 발송, 시간 예약, 배치 처리, RSS 피드를 통한 자동 업데이트 등 캠페인 목표에 맞는 최적의 방식을 선택하세요.

## **이메일 발송/전달 방식이란?**

이메일 발송 방식은 캠페인이 언제, 어떻게 고객에게 도달하는지를 결정합니다. Hyperclass는 **지금 보내기, 예약, 배치 예약, RSS 예약** 등 네 가지 전달 옵션을 제공합니다. 각 방식은 긴급 공지부터 블로그 자동 업데이트까지 다양한 마케팅 요구사항을 충족합니다.

## **이메일 발송 방식의 주요 장점**

- 지금 보내기: 시급한 캠페인을 위한 즉시 전달
- 예약: 특정 날짜/시간에 맞춰 캠페인 사전 계획
- 배치 예약: 이메일을 소그룹으로 나누어 전송하여 전달률 개선
- RSS 예약: 블로그, 피드, 콘텐츠 스트림의 업데이트 자동 발송

## **지금 보내기**

지금 보내기 옵션은 선택한 수신자들의 위치 시간대에 맞춰 캠페인을 즉시 발송합니다. 플래시 세일이나 중요한 공지 등 긴급한 커뮤니케이션에 가장 적합합니다.

지금 보내기를 사용할 때:

- 긴급 뉴스나 특가 정보
- 막판 캠페인 필요
- 즉시 제품 업데이트

설정 단계:

- 전달 옵션에서 **지금 보내기**를 선택
- **발신자 이메일**과 선택사항인 **발신자 이름** 입력
- **제목**과 선택사항인 **미리보기 텍스트** 작성
- 수신자 선택(스마트 리스트, 태그, 세그먼트, 연락처)
- **보내기**를 클릭하여 즉시 전달

## **예약**

**예약** 옵션은 특정 날짜와 시간을 선택하여 전달할 수 있게 합니다. 고객이 가장 참여할 가능성이 높은 시점에 캠페인이 도달하도록 보장합니다.

예약을 사용할 때:

- 이벤트 리마인더
- 시즌 프로모션
- 계획된 마케팅 캠페인

설정 단계:

- **예약** 선택
- **날짜와 시간** 선택(시스템은 위치 시간대 사용)
- 발신자 정보, 제목, 미리보기 텍스트 구성
- 수신자 선택
- 예약 확인

## **배치 예약**

**배치 예약** 옵션은 대규모 캠페인을 더 작은 그룹(배치)으로 나누어 발송할 수 있게 합니다. 이는 전달률 문제의 위험을 줄이고 선택한 시간 프레임에 걸쳐 발송을 분산시킵니다.

참고: 배치 예약 유효성 검사

1. 종료 시간이 시작 시간보다 빠른 경우, 종료 시간 아래에 인라인 오류가 나타남: 종료 시간은 시작 시간보다 빠를 수 없습니다.

2. 시작 시간 전에 종료 시간을 선택하면, 시작 시간 아래에 인라인 프롬프트가 나타남: 유효한 시작 시간을 입력하세요.

3. 오류를 수정할 때까지 보내기와 예약이 비활성화됨.

4. 배치 수량과 반복 간격을 설정.

5. 배치 반복을 위한 지원되는 간격을 선택.

**참고: 초 옵션은 사용할 수 없습니다.

초안 동작: 이전에 저장된 초안이 초를 선택했다면, 예약 오류를 방지하기 위해 해당 값이 자동으로 지워집니다.**

6. 시작 시간 편집은 단일 확인 버튼으로 확인.

7. 자정(12:00 AM) 경계는 잘못된 오류 없이 처리됨.

배치 예약을 사용할 때:

- 대규모 수신자 목록(수천 명의 연락처)
- 서버 스로틀링 방지
- 일관된 전달률 보장

예시: 1000명의 연락처가 있고 배치 수량을 100으로 설정하면, 이메일이 10라운드로 나누어 전달됩니다.

설정 단계:

- **배치 예약** 선택
- **시작 날짜/시간** 정의
- **배치 수량**과 **반복 간격**(시간 또는 일) 설정
- 발신자 정보, 제목, 수신자 구성
- 저장 및 확인

## **RSS 예약**

**RSS 예약** 옵션은 RSS 피드에 연결된 자동 캠페인을 위해 설계되었습니다. 피드가 업데이트될 때마다 이메일이 생성되어 설정한 일정에 따라 발송됩니다.

RSS 예약을 사용할 때:

- 블로그 업데이트
- 뉴스레터
- 자동 콘텐츠 전달

설정 단계:

- **RSS 예약** 선택
- **RSS 피드 URL** 입력
- 발송 빈도(매일, 매주 등)와 시간 선택
- 발신자 정보, 제목, 수신자 구성
- 저장 및 확인

## **이메일 캠페인 설정 및 관리 방법**

올바른 설정과 관리는 성공적인 전달을 위한 핵심입니다. 초기 예약 외에도 Hyperclass는 캠페인을 편집, 재예약, 취소할 수 있는 유연성을 제공합니다.

주요 관리 기능:

- 재예약: 런칭 1시간 전까지 캠페인 시간/날짜 변경
- 콘텐츠 편집: 실행 전 예약, 배치, RSS 캠페인의 이메일 콘텐츠 업데이트
- 일시정지 또는 취소: 발송 전 캠페인 중단(예약, 배치, RSS에 적용)
- 테스트 이메일/미리보기: 최종 전달 전 본인에게 테스트 발송

## **자주 묻는 질문**

**Q: 예약 후 캠페인을 편집할 수 있나요?**
네, 발송 1시간 전까지 시간/날짜를 편집할 수 있습니다. 배치와 RSS 캠페인의 경우 예약 중에도 이메일 콘텐츠를 편집할 수 있습니다.

**Q: RSS 피드에 새 콘텐츠가 없으면 어떻게 되나요?**
피드에 새 콘텐츠가 나타날 때까지 이메일이 발송되지 않습니다.

**Q: 전달에 어떤 시간대가 사용되나요?**
시스템은 계정에 설정된 위치 시간대를 사용합니다.

**Q: 예약된 캠페인을 취소할 수 있나요?**
네, 발송 전까지 예약, 배치, RSS 캠페인을 일시정지하거나 취소할 수 있습니다.

**Q: 예약과 배치 예약의 차이점은 무엇인가요?**
예약은 전체 목록에 한 번에 발송하고, 배치 예약은 목록을 시간을 두고 더 작은 그룹으로 나누어 전달합니다.
