# 이메일 캠페인 상태 이해하기

## 개요

이메일 마케팅에서 캠페인 상태를 통해 이메일 발송의 진행 상황과 성과를 추적하세요. 이메일 캠페인, 워크플로우 캠페인, 일괄 작업 캠페인 등 각 캠페인 유형에는 캠페인이 초안, 활성, 발송됨, 완료됨 상태인지 빠르게 파악할 수 있는 특정 상태 표시가 포함되어 있습니다.

## 이메일 캠페인 상태

이메일 캠페인(Email Campaigns)은 수동으로 생성하고 관리할 수 있는 일회성 또는 반복 이메일 발송입니다. 각 캠페인에는 생성 또는 전송 과정에서 현재 위치를 나타내는 상태(Status)가 표시됩니다.

| 상태 | 설명 |
|------|------|
| **Draft(초안)** | 캠페인이 생성 중이거나 편집 중이며 아직 발송되지 않은 상태입니다. 예약하기 전에 콘텐츠나 설정을 계속 수정할 수 있습니다. |
| **Sent(발송됨)** | 캠페인이 수신자에게 성공적으로 전달된 상태입니다. Statistics(통계) 하에서 성과 지표를 확인할 수 있습니다. |
| **Failed(실패)** | 오류로 인해 캠페인을 발송할 수 없는 상태입니다(예: 발신자 이메일 무효, 설정 문제). 재시도하기 전에 캠페인 설정을 검토하세요. |
| **Archived(보관됨)** | 삭제하지 않고 활성 보기에서 제거하기 위해 수동으로 보관된 캠페인입니다. 언제든지 복원할 수 있습니다. |
| **All(전체)** | 현재 상태와 관계없이 모든 캠페인을 표시합니다. |

## 워크플로우 캠페인 상태

워크플로우 캠페인(Workflow Campaigns)은 워크플로우 액션에 의해 트리거되는 자동화된 이메일 시퀀스입니다. 이러한 캠페인은 자동화를 통해 동적으로 관리되므로 더 간단한 상태를 보여줍니다.

| 상태 | 설명 |
|------|------|
| **Draft(초안)** | 워크플로우 캠페인이 생성되었지만 아직 활성화되지 않은 상태입니다. 트리거, 이메일 또는 액션을 여전히 변경할 수 있습니다. |
| **Published(발행됨)** | 워크플로우 캠페인이 설정한 트리거를 기반으로 실행 중인 상태입니다. 워크플로우에 진입하는 연락처는 자동으로 이메일을 받게 됩니다. |
| **All(전체)** | 초안과 발행된 모든 워크플로우 캠페인을 표시합니다. |

## 일괄 작업 캠페인 상태

일괄 작업 캠페인(Bulk Action Campaigns)은 연락처 목록에 즉시 또는 예약된 배치로 대량 이메일을 발송합니다. 이들의 상태는 발송 진행률과 완료에 대한 자세한 통찰을 제공합니다.

| 상태 | 설명 |
|------|------|
| **Draft(초안)** | 캠페인 설정이 불완전하거나 예약 대기 중인 상태입니다. |
| **Queued(대기 중)** | 캠페인이 발송 처리를 위해 대기열에 있는 상태입니다. |
| **Processing(처리 중)** | 캠페인이 수신자에게 적극적으로 발송되고 있는 상태입니다. |
| **Paused(일시정지)** | 캠페인이 일시적으로 중단된 상태입니다. 언제든지 재개할 수 있습니다. |
| **Complete(완료)** | 캠페인이 모든 수신자에게 성공적으로 발송된 상태입니다. |
| **Cancelled(취소됨)** | 캠페인이 완료 전에 중단되었으며 나머지 연락처에게는 발송되지 않습니다. |
| **All(전체)** | 상태와 관계없이 모든 일괄 작업 캠페인을 표시합니다. |

![일괄 작업 캠페인 상태 예시](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055818329/original/og2UZkF6K0vAoRIol2AvsDNwP64xSoLOpw.png?1760339396)

## 자주 묻는 질문

**Q: 완료(Complete) 또는 발송됨(Sent)으로 표시된 캠페인을 다시 발송할 수 있나요?**
네. 캠페인을 복제하여 새로운 세그먼트나 목록에 다시 발송할 수 있습니다.

**Q: 일시정지(Paused)와 취소됨(Cancelled)의 차이점은 무엇인가요?**
일시정지된 캠페인은 언제든지 재개할 수 있습니다. 취소된 캠페인은 영구적으로 중단되며 재개할 수 없습니다.

**Q: 상태를 수동으로 변경할 수 있나요?**
캠페인을 보관하거나 삭제할 수 있지만, 다른 상태는 시스템 활동에 따라 자동으로 업데이트됩니다.

**Q: "보관됨(Archived)" 캠페인을 복원할 수 있나요?**
네. 이메일 캠페인(Email Campaigns) → 필터(Filters) → 보관됨(Archived)으로 이동하여 캠페인을 선택하고 보관 해제(Unarchive)를 클릭하면 활성 보기로 다시 이동됩니다.
