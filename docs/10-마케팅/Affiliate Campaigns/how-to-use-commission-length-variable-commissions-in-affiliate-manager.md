# 제휴 관리에서 커미션 기간 및 변동 커미션 사용 방법

## 개요

새로운 커미션 기간과 시간별 변동 커미션 기능을 사용하면 정기 상품 판매에 대해 제휴사에게 어떻게, 얼마 동안 보상을 제공할지 세부적으로 설정할 수 있습니다.

- **커미션 기간:** 무기한 대신 제한된 횟수의 거래에 대해서만 제휴사에게 수수료를 지급합니다.
- **변동 커미션:** 특정 결제 횟수에 따라 다른 커미션율을 제공합니다.

---

### 1단계: 제휴 캠페인 생성 또는 편집

- 제휴 캠페인을 처음 만드는 경우 [캠페인 생성 가이드를 참고하세요.](how-to-create-an-affiliate-campaign.md)

### 2단계: 기본 커미션 설정

- 캠페인 설정 2단계에서 제휴사가 기본적으로 받을 기본 커미션 값을 추가합니다.

### 3단계: 커미션 기간 설정

- 커미션 기간(Commission Length)을 클릭합니다.

- 기본적으로 고객이 취소할 때까지 커미션이 지급됩니다.
![커미션 기간 기본 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155038550542/original/coBmKW38xPbyR3OYFUahlZjEm6Bd0GcabA.png?1734524683)

- "By Count"를 선택하여 결제 횟수 제한을 설정합니다.
![결제 횟수로 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155038550567/original/n42CUg0zPRRto8uFVNh791gAB_Gmme9Hxg.png?1734524698)

- **예시:** 제한을 3회로 설정하면 제휴사는 첫 3번의 결제에 대해서만 커미션을 받습니다.
![커미션 3회 제한 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155038550601/original/kIdwb6ZK6zIDkqGxZQssvfpwlJcisBurdA.png?1734524719)

- **참고:** 제한은 시간 간격이 아닌 결제 횟수에 적용됩니다. 구독이 월간이든 연간이든 상관없이 커미션은 성공한 결제 횟수를 기준으로 합니다.

### 4단계: 시간별 변동 커미션 설정

- 변동 커미션(Variable Commissions)을 토글하여 이 기능을 활성화합니다.

- 특정 결제 횟수에 대해 다른 요율을 정의하려면 Two Levels를 선택합니다.
![변동 커미션 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155038550631/original/jXSHTfCxN8EIEZ5udCJNAcZ9r4iOvw14UQ.png?1734524783)

- **예시:** "처음 **3**회*[이 숫자는 설정 가능]* 결제에 대해서는 커미션이 **33%**이고*[이 숫자는 설정 가능]*, 그 이후에는 **23%**입니다*[이는 기본 커미션]*."
![변동 커미션 예시](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155038550677/original/tjAOqL0hHnt7WHIfKeJU4OAOfNzT00he_A.png?1734524807)

- 필요한 경우 더 세밀한 제어를 위해 추가 단계를 설정할 수 있습니다.

### 5단계: 캠페인 발행

- 캠페인을 저장하면 업데이트된 커미션 설정이 적용됩니다.

---

참고: 
커미션 우선순위
1. 상품 기반 커미션
2. 변동 커미션
3. 기본 커미션

이렇게 하여 특정 커미션 설정이 일반 설정보다 우선 적용되도록 합니다.
