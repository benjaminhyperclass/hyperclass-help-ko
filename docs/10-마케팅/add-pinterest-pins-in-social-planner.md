# 소셜 플래너에서 Pinterest 핀 추가하기

**새로운 기능 소개**
소셜 플래너(Social Planner)의 새로운 기능을 소개합니다. 이제 여러 Pinterest 계정을 연결하고 무제한으로 핀을 생성할 수 있습니다.

**어떻게 작동하나요?**

1. Marketing(마케팅) > Social Planner(소셜 플래너)로 이동합니다.

![소셜 플래너 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155028731059/original/Vqc3zSYmZxpJm9rO2usqljMFoGg_B3nBmg.png?1720107300)

2. Pinterest 계정을 연결합니다 (해당 계정의 관리자 권한이 있어야 합니다).

![Pinterest 계정 연결](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155028731076/original/b8U2ZAWjYEdlCbLkgwhPf9xsqpQ6zkNylw.png?1720107340)

3. 소셜 플래너 게시물 작성 도구를 사용해서 새 게시물을 만듭니다.

4. Pinterest 계정을 다른 소셜 미디어 계정과 함께 선택합니다.

![계정 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155028731344/original/RAXTf7K72wnMEzy6BXh2FRh0MPnl3Q5Kjw.png?1720107686)

5. Pinterest 보드를 선택하라는 메시지가 표시됩니다.

6. 보드가 없다면 보드 선택 드롭다운에서 "보드 추가(Add board)" 옵션을 사용해서 새로 만들 수 있습니다.
- 잠금 아이콘은 비공개 보드입니다
- 공개 보드에는 잠금 아이콘이 없습니다  
- 사용자는 새 보드를 생성할 수 있습니다

![보드 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155028731372/original/FxmrTcXVwnz8pZ0-p5IIlErmSwsDornpLA.png?1720107718)

**Pinterest 요구사항:**

- Pinterest에서 핀을 생성하려면 이미지 1개 또는 동영상 1개가 필요합니다.
- 이미지: 10MB 이하, PNG 및 JPG 형식 지원
- 동영상: 1GB 이하, MOV, MP4, WEBP 형식 지원. 최소 4초에서 최대 5분 길이 필요. 권장 가로세로 비율은 9:16 (전체 화면 동영상), 1:2, 2:3, 3:4, 4:5 또는 1:1. 동영상 인코딩: H.264 또는 H.265
