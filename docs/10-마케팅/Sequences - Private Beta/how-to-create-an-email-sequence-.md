### **이메일 시퀀스란?**

**이메일 시퀀스**는 순서대로 자동 전송되는 이메일 세트로, 연락처와 **단일 스레드 대화**를 형성합니다.

각 시퀀스는 다음을 포함할 수 있습니다:

- 여러 이메일 단계
- 이메일 간 시간 지연
- 조건 (예: 이메일이 열렸을 때만 발송)
- 연락처가 답장할 때 자동 중단

이를 통해 수동 후속 조치 없이도 개인화되고 행동 기반의 커뮤니케이션을 구축할 수 있습니다.

### **언제 이메일 시퀀스를 사용해야 하나요?**

이메일 시퀀스는 다음 용도에 이상적입니다:

- **후속 조치:** 응답하지 않은 리드에게 자동으로 후속 메시지 전송
- **환영 시리즈:** 신규 사용자나 고객 온보딩
- **알림:** 이벤트, 데모, 액션에 대한 예약된 알림 발송
- **육성 캠페인:** 점진적으로 고객과 소통하며 교육

수동 작업 없이 **일관되고 시의적절한 커뮤니케이션**을 유지하고 싶을 때 시퀀스를 사용하세요.

### 이메일 시퀀스를 만드는 방법

1. 시퀀스 메뉴로 이동

- **Marketing(마케팅) → Email(이메일) → Campaigns(캠페인)**으로 이동합니다

2. 새 시퀀스 생성

- **Create New Sequence(새 시퀀스 생성)** 버튼을 클릭합니다
![새 시퀀스 생성](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155068523294/original/s6f5O7-u0Mu7e5nXYX4MzUjlhQoTLc6Z9g.png?1775475711)

3. 트리거 설정

- 시퀀스가 언제 시작될지 선택합니다:
  - 즉시
  - 예약된 시간

![트리거 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155068523301/original/RCdH9Q7ML0KYwIUooEj6LumLBn7meZR8kg.png?1775475721)

- 시퀀스 유형에 따라 "Stop on Reply(답장 시 중단)" 옵션을 설정합니다

![답장 시 중단 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155068523510/original/OFvh2_OfGwsIFnHBgLSxtYCW6TAENFflsw.png?1775475915)

4. 대상 선택

- 다음 중에서 선택합니다:
  - 리스트
  - 세그먼트
  - 개별 연락처
- 필요한 경우 **Stop on Reply(답장 시 중단)** 옵션을 활성화합니다

5. 이메일 단계 추가

- 시퀀스에 여러 단계를 추가합니다
- 각 단계 간 **지연 시간**을 설정합니다
![이메일 단계 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155068523432/original/ucY4BLunqvdeboYEflyvF_shOiqJZSJNFQ.png?1775475817)

6. 각 단계 설정

각 이메일에 대해:

- 템플릿을 선택하거나 디자인합니다 (또는 일반 텍스트 사용)
- 조건을 설정합니다 (예: 이전 이메일이 열렸거나 전달된 경우에만 발송)
- 필요에 따라 추적 기능을 활성화합니다
![각 단계 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155068523389/original/su_64NEaFPxrdiIjvuD336IMq6-Rf9wP6Q.png?1775475780)

7. 검토 및 실행

- 시퀀스 흐름을 재확인합니다
- 시퀀스를 활성화합니다
![시퀀스 활성화](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155068523444/original/4PNJrDgGfVXIUa14KcKPSyilaejL3mrHJA.png?1775475840)

- 모든 이메일이 대화(Conversations) 섹션에서 하나의 스레드로 연결되어 표시됩니다
![대화 스레드](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155068523466/original/obwy-SRVY92mi0UmpLS4p4a5S9-hhvMYDg.png?1775475866)

8. 목록 보기에서 점 3개 메뉴를 클릭하여 모든 통계에 액세스할 수 있습니다 - 개별 이메일 단계별 통계와 전체 이메일 액션 통계를 확인할 수 있습니다.

![통계 보기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155068523657/original/FwCWCB3rKgQkApDtZaNZciGhcGhMVlOwwg.png?1775476084)

9. 모든 시퀀스 통계를 보려면 이메일 마케팅의 통계 대시보드에서도 확인할 수 있습니다.
![통계 대시보드](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155068523604/original/rqtpAaaOWXSmxF37vmx-Aukrj7e97uKBZg.png?1775476036)

**자주 묻는 질문**

1. 연락처가 답장하면 어떻게 되나요?

**Stop on Reply(답장 시 중단)** 옵션이 활성화되어 있으면, 해당 연락처에 대한 시퀀스가 자동으로 중단됩니다.

2. 디자인된 이메일과 일반 텍스트 이메일을 모두 보낼 수 있나요?

네, 시퀀스는 **디자인된 템플릿**과 **일반 텍스트 이메일**을 모두 지원합니다.

3. 각 이메일을 받는 사람을 제어할 수 있나요?

네, 각 단계에 **조건**을 적용할 수 있습니다 (예: 열람 또는 전달 상태 기반).

4. 성과는 어떻게 추적하나요?

다음 수준에서 성과 지표를 볼 수 있습니다:

- **시퀀스 레벨**
- **개별 단계 레벨**

**점 3개 메뉴 → Statistics(통계)**에서 통계에 액세스할 수 있습니다.

5. 이메일이 별도 메시지로 표시되나요?

아니요, 이메일은 **단일 스레드 대화**의 일부로 전송되어 맥락을 유지하는데 도움이 됩니다.
