# 카운트다운 타이머의 시간대 자동 적응

시간대 자동 적응(Timezone Adaptation) 기능으로 이메일, 퍼널, 웹사이트의 카운트다운 타이머 종료 시간 계산을 완벽하게 제어하세요. 이 스마트 시스템은 각 연락처나 방문자가 자신의 현지 시간에 맞춰 정확한 타이머 종료 시간을 볼 수 있도록 하여, 더욱 개인화되고 정확한 경험을 제공합니다.

## 시간대 자동 적응이란?

시간대 자동 적응을 사용하면 카운트다운 타이머의 종료 시간이 보는 사람의 시간대에 따라 자동으로 조정됩니다. 사용자가 인도, 미국, 호주 어디에 있든 현지 시간에 맞춰 정확한 타이머 종료 시점을 볼 수 있습니다.

**예시:** 크리스마스 카운트다운을 12월 25일에 끝나도록 설정하면, 인도 사용자는 12월 25일 IST에 끝나는 것을 보고, 뉴욕 사용자는 12월 25일 EST에 끝나는 것을 봅니다 — 별도 설정 없이 자동으로 말이죠.

이를 통해 다음을 보장합니다:

- 프로모션의 정확한 타이밍
- 마감 기한 기반 캠페인의 높은 신뢰성과 명확성
- 전 세계 분산된 고객에 대한 글로벌 일관성

## 시간대 로직 작동 방식

시스템은 타이머가 이메일에서 사용되는지 퍼널/웹사이트에서 사용되는지에 따라 명확한 우선순위를 따릅니다.

## 시간대 적응 순서

### 1. 이메일

이메일은 연락처 수준의 데이터에 의존합니다.

| 상황 | 결과 |
|------|------|
| 연락처의 시간대가 설정됨 | 종료 시간이 연락처의 시간대로 변환됩니다. |
| 연락처의 시간대가 설정되지 않음 | 타이머 생성 시 선택한 원래 시간대를 유지합니다(대체값). |

이를 통해 각 연락처에게 가장 정확하고 개인화된 카운트다운을 제공합니다.

### 2. 퍼널 및 웹사이트

웹 경험의 경우 실시간 방문자 맥락이 사용됩니다.

| 상황 | 결과 |
|------|------|
| 방문자가 페이지에 도착함 | 타이머가 IP로 감지된 시간대를 사용해 자동 적응됩니다. |

IP 감지가 실패하면 타이머는 생성자가 선택한 시간대를 사용합니다.

**참고**: 퍼널에서 사용되는 동적 타이머의 경우, 퍼널이 이메일을 통해 리다이렉트되면 시간대 적응에 이메일 로직이 사용됩니다.

## 작동 방식

카운트다운 타이머를 만들 때:

- 사용자가 종료 날짜 및 시간을 선택합니다
- 시간대 드롭다운 아래에 새 설정이 나타납니다: ☑ 연락처 시간대에 적응
- 체크하면:
  - 시스템이 자동으로 적절한 시간대 로직을 적용합니다
  - 시간대가 없는 연락처의 경우, 대체값 = 드롭다운에서 선택한 시간대
  - 웹사이트/퍼널의 경우, IP 기반 시간대 감지가 자동으로 적용됩니다

**참고**: 반복 카운트다운 타이머의 경우, 시작 시간이 각 방문자의 현지 시간대에 적응하고, 종료 시간은 타이머의 반복 조건에 따라 자동 계산되어 모든 사용자에게 정확하고 현지화된 주기를 보장합니다.

## 자주 묻는 질문

**Q: 연락처에 시간대가 없으면 어떻게 되나요?**

타이머는 생성 설정에서 선택한 시간대를 사용합니다(대체값).

**Q: 반복 타이머는 시간대 적응과 어떻게 작동하나요?**

시작 시간이 각 방문자의 시간대에 적응하고, 종료 시간은 반복 규칙에 따라 자동 계산됩니다.

**Q: 적응 기능을 활성화하면 시간대 선택이 제거되나요?**

아니요. 선택 항목은 여전히 표시되어 대체 로직의 명확성을 유지합니다.
