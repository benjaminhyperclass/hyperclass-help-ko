# 빠른 캠페인 작업으로 제휴사 관리하기

## 빠른 캠페인 작업이란?

빠른 캠페인 작업은 제휴사 목록 페이지에서 바로 제휴사 관리에 빠르게 접근할 수 있는 기능입니다. 각 제휴사의 프로필을 열 필요 없이 점 3개 메뉴를 클릭하고 작업을 수행할 수 있습니다. 캠페인 배정과 조정을 빠르고 명확하게 간소화합니다.

**중요**: 모든 빠른 캠페인 작업은 동일한 첫 단계를 따릅니다. Marketing(마케팅) → Affiliate Manager(제휴 관리) 페이지로 이동하여 관리할 제휴사를 찾고, 이름 옆의 점 3개 메뉴를 클릭하세요. 그다음 Manage Affiliate(제휴사 관리)를 선택하면 Suspend(일시정지), Unsuspend(일시정지 해제), Add to Campaign(캠페인에 추가), Remove from Campaign(캠페인에서 제거), Move to Campaign(캠페인으로 이동) 등의 사용 가능한 작업이 나타납니다.

---

## **빠른 캠페인 작업의 주요 장점**

이 기능이 제휴사 관리를 어떻게 간소화하여 시간을 절약하고 수동 단계를 줄이는지 명확하게 설명합니다.

- 몇 초 만에 캠페인에서 제휴사를 추가하거나 제거
- 최소한의 클릭으로 제휴사를 일시정지 또는 재개
- 고객 처리 옵션과 함께 제휴사를 캠페인 간에 이동
- 대규모 프로그램 구성 개선

- 일시정지 해제 후: 제휴사의 기존 링크가 자동으로 다시 작동하기 시작합니다.

## 제휴사를 다른 캠페인으로 이동하기

### 1단계: Campaign Actions(캠페인 작업) 메뉴에서 "Move to Campaign(캠페인으로 이동)" 선택

![캠페인 이동 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155047047001/original/ItKmoImS1G8BqczNbTgTUEEw3vP9vuA5Iw.png?1747900459)

### 2단계: 제휴사를 이동할 대상 캠페인 선택

![대상 캠페인 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051224090/original/rSwDtKGjLG5BQha29SJwtPJL-U1JFA598g.png?1754660610)

### 3단계: 기존 고객 처리 방법 결정

![기존 고객 처리 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051224178/original/RwQ7UOpY2uKFuBqu3YhfYaVbpTmK-uVWtw.png?1754660676)

**중요**: 두 가지 옵션이 있습니다:
1. 기존 캠페인에 그대로 두기: 기존 고객과 그들의 정기 수수료는 현재 캠페인에 그대로 남습니다.
2. 새 캠페인으로 이동: 기존 고객으로부터의 향후 수수료는 새 캠페인의 규칙을 따르게 됩니다.

### 4단계: 환영 이메일 발송 여부 선택

대상 캠페인에서 환영 이메일이 비활성화되어 있다면, 여기서 켜도 이메일이 발송되지 않습니다. 캠페인에서 환영 이메일이 활성화되어 있다면, 여기서 비활성화하여 이 제휴사에게는 발송하지 않을 수 있습니다. 선택을 완료한 후 Move Affiliate(제휴사 이동)를 클릭하여 작업을 완료하세요.

## **자주 묻는 질문**

**Q: "캠페인으로 이동" 작업을 되돌릴 수 있나요?**
A: 아니요. 하지만 다시 이동시켜서 수동으로 되돌릴 수 있습니다.

**Q: 제휴사가 이동하거나 일시정지될 때 이메일 알림을 받나요?**
A: 환영 이메일은 캠페인 설정과 이동 과정에서의 토글 설정에 따라 달라집니다.

**Q: 빠른 캠페인 작업을 일괄적으로 관리할 수 있나요?**
A: 빠른 작업으로는 불가능합니다. 대신 워크플로우나 수동 일괄 처리를 사용하세요.

**Q: 일시정지가 추천 링크에 영향을 주나요?**
A: 네, 일시정지 중에는 링크가 추적되지 않습니다. 일시정지 해제 후에는 링크가 자동으로 다시 활성화됩니다.
