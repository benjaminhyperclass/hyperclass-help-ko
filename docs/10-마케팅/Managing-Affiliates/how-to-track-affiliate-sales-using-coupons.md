# 쿠폰을 사용한 제휴 판매 추적 방법

Hyperclass(Hyperclass)를 통해 추천 링크와 쿠폰 코드를 활용하여 제휴사와 인플루언서를 효과적으로 추적할 수 있습니다. 쿠폰 코드는 인스타그램 같은 플랫폼에서 추천 링크를 사용하기 어려운 상황에서 특히 유용합니다.

- [중요 사항](#중요-사항)

---

## 쿠폰 코드 추적의 장점

- **할인 유연성:** 제휴사의 팔로워에게 할인이나 특별 프로모션을 제공할 수 있습니다.
- **맞춤형 할인:** 0%부터 100%까지 쿠폰 코드를 설정하여 정가 판매와 무료 서비스 모두 가능합니다.

---

## 쿠폰 코드 설정 및 사용 방법

- **제휴 관리(Affiliate Manager) 접근:** `Marketing(마케팅) > Affiliate Manager(제휴 관리)`로 이동합니다.
- **제휴사 선택:** 제휴사/프로모터 모듈에서 쿠폰 코드를 설정할 특정 제휴사를 찾아 선택합니다.
- **캠페인 커스터마이징:** "활성 추천 링크(Active Referral Links)" 섹션으로 스크롤합니다. 여기에서 캠페인 목록과 제휴사의 URL을 확인할 수 있습니다.
- 원하는 캠페인에서 **Customize(커스터마이즈)**를 클릭합니다.

![쿠폰 코드 설정 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033859110/original/CyUb1QDoCKo-vtVW0R2ewt6Isfs3HeGLyg.png?1727774580)

- **쿠폰 설정:** Coupon(쿠폰) 탭으로 전환합니다.
- 쿠폰 유형을 선택합니다: Percentage(백분율) 또는 Fixed(고정액).
- 사용자 친화적인 쿠폰 이름을 입력합니다.
- **Generate Code(코드 생성)**를 클릭하여 쿠폰 코드를 생성합니다.

![쿠폰 설정 상세 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033859199/original/oxSMxDkMOY0RrDIe5xZH7hAKLJSl1JvI1Q.png?1727774643)

- **할인 설정:** 
  - 무료 서비스의 경우 할인을 100%로 설정합니다.
  - 특정 할인의 경우 0%부터 100%까지 백분율을 설정합니다. (참고: 0% 할인은 추적 목적으로만 사용됩니다.)
- **저장 및 관리:** **Save(저장)**를 클릭하여 쿠폰 코드를 완료합니다.
- 같은 팝업 창에서 기존 쿠폰 코드를 편집하거나 관리할 수도 있습니다.

---

## 중요 사항

- 쿠폰 코드가 사용자 친화적이고 제휴사가 고객에게 공유하기 쉽도록 만들어야 합니다.
- 정기적으로 쿠폰 코드를 검토하고 업데이트하여 프로모션 전략에 맞도록 조정하세요.

---

이 단계들을 따라하면 쿠폰 코드를 사용하여 제휴 활동을 효과적으로 추적하고 관리할 수 있으며, 프로모션 효과를 향상시키고 제휴사 성과를 최적화할 수 있습니다.
