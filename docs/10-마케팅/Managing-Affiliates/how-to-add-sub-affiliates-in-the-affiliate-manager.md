# 제휴 관리에서 하위 제휴사를 추가하는 방법

하위 제휴사(Sub-Affiliates)는 상위 제휴사 아래에서 활동하는 제휴사입니다. 자신의 네트워크를 통해 상품이나 서비스를 홍보하며, 상위 제휴사는 하위 제휴사가 만든 매출에서 수수료를 받습니다. 이 모델은 제휴사가 네트워크를 확장하고 수익을 늘리는 데 도움이 됩니다.

하이퍼클래스 플랫폼은 최대 7단계 제휴 캠페인을 지원하여, 제휴사 아래에 여러 하위 제휴사를 둘 수 있습니다.

**참고사항:**
하위 제휴사를 추가하려는 캠페인은 멀티티어(2단계 이상)여야 합니다.

## 하위 제휴사의 장점

하위 제휴사는 계층형 수수료 구조를 가능하게 합니다. 예를 들어, 제휴사 A가 제휴사 B를 추천하고, B가 매출을 만들면 A와 B 모두 수수료를 받습니다. 이는 제휴사들이 팀을 확장하고 더 많은 리드를 생성하도록 장려합니다.

## 하위 제휴사를 추가하는 3가지 방법

### 1. 하위 제휴사 수동 추가

수동으로 하위 제휴사를 추가하면 캠페인에 누가 참여할지 더 세밀하게 제어할 수 있습니다.

#### 단계:

- **제휴 페이지로 이동**: Marketing(마케팅) 섹션에서 Affiliate Manager(제휴 관리)로 이동하세요.
![제휴 관리 페이지](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033798091/original/P6JQCEkwGUGnTzg1-pTv80pU8CLuzbW1yw.jpeg?1727699842)

- **제휴사 선택**: 하위 제휴사를 추가할 제휴사를 클릭하세요.
- **하위 제휴사 탭으로 전환**: 아래로 스크롤하여 "Sub-Affiliate" 탭을 선택하세요.
![하위 제휴사 탭](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033798094/original/_k_S09sc4saLSWEeaY_ZyWeBf0BmbGFyVQ.jpeg?1727699842)

- **하위 제휴사 추가**: "Add Sub-Affiliate" 버튼을 클릭하세요.
- **캠페인 선택**: 드롭다운에서 원하는 캠페인을 선택하세요.
![캠페인 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033798092/original/HNw5za-KF4qHM52GnX72aG10lqg3PediCg.jpeg?1727699842)

- **추가할 하위 제휴사 선택**: 추가하려는 제휴사를 선택하세요.
![하위 제휴사 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033798093/original/LbgQ1dkvCy0xDTZbm_tv1e8PbTQXg6sIGQ.jpeg?1727699842)

- **기존 하위 제휴사 제거**: 필요한 경우 원하지 않는 하위 제휴사를 제거하세요.
- **새 제휴사 프로필 추가 (선택사항)**: 원하는 제휴사가 없다면 새 하위 제휴사를 생성하여 추가하세요.
- **저장**: 변경 사항을 저장하세요.

### 2. 신규 고객을 하위 제휴사로 자동 등록

이 옵션을 사용하면 매출이 성사된 후 해당 고객을 추천한 제휴사 아래에 신규 고객을 하위 제휴사로 자동 등록할 수 있습니다.

#### 단계:

- **하위 계정 제휴 관리**로 이동하세요.
- 기존 캠페인을 열거나 새 캠페인을 만드세요.
- 마지막 단계에서 **Additional Settings(추가 설정)**을 펼치세요.
- **Auto Enroll New Customers as Sub-Affiliates(신규 고객을 하위 제휴사로 자동 등록)** 토글을 켜세요.
![자동 등록 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033798126/original/RrdMYSQoaTVG0nkGU1p8UKC-EyMjy0elhw.jpeg?1727699855)

- 변경 사항을 저장하고 발행하세요.

**Global Settings(전역 설정)** 섹션에서도 이 설정을 전체적으로 활성화하거나 비활성화할 수 있습니다. 이렇게 하면 향후 모든 캠페인에 적용됩니다.

### 3. 제휴사가 직접 하위 제휴사를 모집하도록 허용

제휴사가 가입 링크를 공유하여 직접 하위 제휴사를 모집할 수 있습니다. 이 방식은 온보딩을 간편하게 만들고 제휴사에게 더 많은 자율성을 부여합니다.

#### 단계:

- 캠페인을 생성하거나 편집할 때 마지막 단계에서 **Additional Settings(추가 설정)**을 클릭하세요.
- **Sub-Affiliate Sign-Up Page(하위 제휴사 가입 페이지)** 토글을 켜세요.
![가입 페이지 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033798172/original/jnSh6ig1IfcvWWxPcBUfpM8-XpiRPuHmGQ.jpeg?1727699871)

- 하위 제휴사 가입에 사용할 폼을 선택하세요.
![폼 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033798170/original/Pz0M5TPsN9szKvlSRcLArh8__Vlsd8SC-Q.jpeg?1727699871)

- **Finish** 버튼을 클릭하여 변경 사항을 저장하세요.

**Global Settings(전역 설정)**에서도 이 옵션을 전체적으로 구성할 수 있습니다. 제휴사들은 자신의 제휴 포털에서 가입 링크를 볼 수 있고, 이를 공유하여 하위 제휴사를 모집할 수 있습니다.

![제휴 포털 가입 링크](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033798173/original/xUle4cnCF8zwW0Jyut4oqqrFp5jq5rATQw.jpeg?1727699871)

제휴 포털에서는 각 제휴사 아래의 모든 하위 제휴사 목록도 보여줍니다.

![하위 제휴사 목록](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033798171/original/TVKFriFUpbViGOpo0a9Iu_uxa2UIBjnDtA.jpeg?1727699871)

**참고사항:**
하위 제휴사 가입에 사용되는 폼은 반드시 이름(First Name), 성(Last Name), 이메일(Email) 필드를 포함해야 합니다. 필수 필드가 누락되면 오류가 발생합니다.

하위 제휴사 가입 기능은 멀티티어 캠페인(2단계 이상)에서만 작동합니다.

전역 설정의 변경사항은 향후 캠페인에만 적용됩니다.

이러한 방법들을 활용하면 제휴 네트워크를 확장하고 하위 제휴사를 효과적으로 관리하여 전체적인 매출과 수수료를 증대시킬 수 있습니다.
