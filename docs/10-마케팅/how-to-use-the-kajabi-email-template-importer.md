# Kajabi 이메일 템플릿 가져오기 사용법

사용자는 Kajabi에서 자동 생성된 이메일 주소로 이메일을 보내는 간단한 몇 단계만으로 Kajabi 템플릿을 원활하게 가져올 수 있습니다.

## 왜 이 기능을 만들었나요?

**간편함**: 기술적 지식이 있는 사용자든 없는 사용자든 누구나 쉽게 따라할 수 있는 직관적인 가져오기 과정을 제공하고자 했습니다.

**재사용성**: 이미 Kajabi에서 만든 템플릿을 재사용할 수 있게 하여 효율성을 높이고 소중한 시간을 절약합니다.

## 가져오기 방법

- 계정에서 Email Marketing(이메일 마케팅) → Templates(템플릿)로 이동하세요.
- New(새로 만들기) → Import Templates(템플릿 가져오기)를 클릭하세요.

![Kajabi 템플릿 가져오기 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155026857982/original/fkxidEBubNI6ZDzdPALNQ-tauOiPdlf2Hw.png?1717082020)

- 드롭다운 메뉴에서 "Kajabi"를 선택하세요.

![Kajabi 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155026857981/original/bmgE0a0CFQxdQJWvGO2TijVzjN6CdWv_JA.png?1717082020)

- 자동 생성된 이메일 주소를 복사하세요.

![이메일 주소 복사](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155026857980/original/nIqbbT_bw3RrjZpf-0DTuDEY8j880ZIKZA.png?1717082020)

- Kajabi 계정에서 원하는 템플릿을 테스트 이메일이나 캠페인으로 이 이메일 주소에 보내세요.
- "Continue(계속)"를 클릭해 가져오기를 시작하세요.
- 이 템플릿의 원하는 이름을 입력하고 "Create Template(템플릿 만들기)"를 클릭하세요.

![템플릿 이름 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155026857979/original/-VaReygOdYH4B63qiUEhR7uXhsV97F4M2Q.png?1717082020)
