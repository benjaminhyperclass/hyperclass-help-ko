# 메시징 앱을 전환 위치로 하는 참여용 메타 광고 만들기

메타 광고를 통해 기업은 Facebook 메신저, Instagram 다이렉트, WhatsApp 같은 메시징 앱으로 참여를 유도하고 잠재 고객과 대화를 시작할 수 있습니다. 이 가이드에서는 다음과 같은 설정으로 광고 캠페인을 만드는 방법을 안내합니다:

- 광고 목적: 참여(Engagement)
- 전환 위치: 메시징 앱
- 성과 목표: 대화 수 최대화

또한 이 가이드에서는 광고나 대화 폼이 특정 메시징 앱에서는 작동하지만 다른 앱에서는 작동하지 않는 예외 상황도 다룹니다.

## 1단계: Ad Manager 접속

- 온보딩이 완료되었다면 하위 계정 > Marketing(마케팅) > Ad Manager로 이동합니다. (Ad Manager 활성화 및 재판매 방법은 [이 링크]([how-to-enable-and-resell-ad-manager](how-to-enable-and-resell-ad-manager.md))를 참조하세요)
- + Create Campaign(캠페인 만들기)을 클릭해 새 캠페인을 시작합니다.

![새 캠페인 만들기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042575129/original/tr7HhwDkrf5-B-rVSW1A41FuZA9DTiz-fQ.png?1741042486)

- Facebook Campaign을 선택하고 Start from scratch(처음부터 시작)를 선택합니다.

## 2단계: Facebook 페이지와 Instagram 계정 선택

- 광고에 연결하여 비즈니스를 대표할 Facebook 페이지를 선택합니다.
- 선택한 Facebook 페이지에 연결된 Instagram 계정을 선택하거나, Facebook 페이지 자체를 사용하여 Instagram에 광고를 게재할 수 있습니다. Instagram 계정을 Facebook 페이지와 연결하는 방법과 기타 중요한 정보는 다음 [도움말 문서]([instagram-ads-overview](instagram-ads-overview.md))에서 확인하세요.

![Facebook 페이지와 Instagram 계정 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042575823/original/JGYXvETaEhBJ8kg41MufLTSNLA8QR5mwqg.png?1741045322)

**참고**: 선택한 페이지에 연결된 Instagram 계정이 없으면 Instagram 메시징 앱은 클릭 투 다이렉트 광고에 사용할 수 없습니다.

## 3단계: 광고 목적 선택

- 목적으로 Engagement(참여)를 선택합니다.

![광고 목적 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042578513/original/0MFGWoypNrtw2avl9ugF_aR4lpuJxzzt2A.png?1741055254)

- 전환 위치(Conversion Location)는 기본적으로 'Messaging Apps(메시징 앱)'로 설정됩니다.
- 성과 목표(Performance goal)는 기본적으로 'Maximise number of conversations(대화 수 최대화)'로 설정됩니다. 이는 Meta가 사용자가 비즈니스와 채팅을 시작하도록 광고 송출을 최적화하는 것을 의미합니다.

## 4단계: 대화용 메시징 앱 선택

메시징 앱 섹션에서 메시지를 받고 싶은 플랫폼을 하나 이상 선택합니다:

- Facebook Messenger
- Instagram Direct - 선택한 Facebook 페이지에 Instagram 계정이 연결된 경우에만 활성화됩니다.
- WhatsApp - 광고 캠페인용으로 선택한 Facebook 페이지에 WhatsApp 번호가 추가된 경우에만 CTX 광고의 메시징 앱으로 사용할 수 있습니다.

![메시징 앱 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042578706/original/KRJtQd6ODWP9vQnsNjEeqacbXwREdwBGew.png?1741055680)

### 메시징 앱을 선택하지 않으면 어떻게 되나요?

| 상황 | 영향 |
|------|------|
| 세 앱 모두 사용 가능 (메신저, Instagram, WhatsApp) | 광고가 모든 플랫폼에 표시되며, Meta 알고리즘이 사용자에게 도달하기 가장 적합한 앱을 결정합니다. |
| 메신저와 Instagram 다이렉트만 사용 가능 | WhatsApp은 선택지가 아닙니다. 사용자는 메신저나 Instagram 다이렉트를 통해 메시지를 보낼 수 있습니다. |
| WhatsApp만 사용 가능 | 사용자는 대화를 위해 WhatsApp으로만 연결됩니다. 광고는 메신저나 Instagram에서 실행되지 않습니다. |
| 하나의 앱만 사용 가능 | 광고 캠페인은 모든 플랫폼에 표시되지만, 사용자는 해당 메시징 플랫폼을 통해서만 연락할 수 있습니다. |
| 메시징 앱을 사용할 수 없음 | 이 목적으로는 광고를 만들 수 없습니다. 다른 목적(예: 트래픽 또는 리드)을 선택해야 할 수 있습니다. |

## 5단계: 대화 폼 생성 또는 기존 폼 사용

대화 폼을 만들거나 이미 만든 대화 폼을 사용하는 방법에 대한 자세한 내용은 이 [도움말 문서]([how-to-create-a-conversation-form-in-ad-manager](how-to-create-a-conversation-form-in-ad-manager.md))에서 확인하세요.

## 6단계: 게재 위치 선택

- 자동 게재 위치(권장): Meta가 대화를 생성할 가능성이 가장 높은 곳에 광고를 배치합니다.
- 수동 게재 위치: Facebook 피드, Instagram 피드, 스토리 등 특정 게재 위치를 직접 선택할 수 있습니다.

## 7단계: 광고 만들기

- 광고 형식 선택:
  - 단일 이미지/비디오 – 하나의 비주얼을 사용하는 단순한 광고
  - 캐러셀 – 사용자가 스크롤할 수 있는 여러 이미지/비디오
- 매력적인 광고 문구 작성:
  - 짧고 매력적으로 작성하세요.
  - "지금 메시지 보내기!"와 같은 강력한 행동 유도(CTA)를 강조하세요.
- 행동 유도(CTA) 설정:
  - "Send Message(메시지 보내기)"를 선택합니다.
  - 여러 메시징 앱을 선택한 경우, Meta가 당신이 응답할 가능성이 가장 높은 플랫폼으로 메시지를 보내도록 최적화합니다.

## 8단계: 예산 및 일정

- 일일 예산 또는 총 예산을 설정합니다.
- 필요한 경우 시작 및 종료 날짜를 선택합니다.
- 예산이 캠페인 목표에 맞는지 확인합니다.

## 9단계: 오디언스 설정

- 위치, 연령, 성별, 세부 타겟팅을 기반으로 오디언스를 선택합니다.
- 기존 고객을 타겟팅하려면 커스텀 오디언스를 사용하고, 새로운 사용자를 찾으려면 유사 오디언스를 사용합니다.

## 10단계: 검토 후 발행

- 모든 설정을 다시 확인합니다.
- Publish(발행)를 클릭해 광고를 시작합니다.
- Ad Manager Statistics에서 성과를 모니터링합니다.

# 추적 및 최적화

- Meta Pixel을 사용하여 사용자 상호작용을 추적하세요.
- CAPI 액션을 만들어 대화의 효과적인 추적을 위해 세부 정보를 Meta로 다시 보내세요.
- 다양한 크리에이티브와 오디언스 세그먼트를 테스트하여 최적화하세요.
- 성과 지표를 기반으로 예산과 게재 위치를 조정하세요.
