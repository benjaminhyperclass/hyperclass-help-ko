# 고객 목록 커스텀 오디언스 만드는 방법

이 문서에서는 Ad Manager(광고 관리자)에서 고객 목록 커스텀 오디언스를 만드는 단계와 캠페인 생성 시 활용하는 방법을 설명합니다. 스마트 리스트(Smart List)를 사용할 때 오디언스를 정적(Static) 또는 동적(Dynamic)으로 설정하는 옵션도 포함되어 있습니다.

## 커스텀 오디언스 생성 옵션을 찾는 위치

Ad Manager에서 커스텀 오디언스는 두 가지 방법으로 만들 수 있습니다:

### 캠페인 생성 중

광고 세트 레벨 > Audience Controls(오디언스 컨트롤) 하위의 Intended Audience(목표 오디언스) 섹션에서 찾을 수 있습니다.
![목표 오디언스 섹션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051625719/original/IUicvaGBvMEv3h3e6zj9sIt_57Rv-WHk-Q.png?1755176224)

Retargeting(리타겟팅) 탭에서 "Create new or select retargeting audience" 드롭다운을 클릭하고 Create a Custom Audience(커스텀 오디언스 만들기)를 선택하세요.
![리타겟팅 오디언스 드롭다운](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051625765/original/hBRo2x5sMOvbXvLmIE0_DGvt4JHZ9sVCZA.png?1755176271)

### Ad Manager 설정에서

향후 캠페인을 위해 미리 오디언스를 만들려면 Ad Manager Settings(Ad Manager 설정) > Meta Audiences(메타 오디언스) 탭으로 이동하세요.

![Ad Manager 설정의 메타 오디언스 탭](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051625452/original/rk_mZ2mxJiSjhbmFvIqeyrhZe28BKH9jkw.png?1755176127)

## 고객 목록 커스텀 오디언스란?

### 개요

메타 고객 목록 커스텀 오디언스를 사용하면 Ad Manager에서 자신의 고객 데이터(이메일, 전화번호 등)를 활용하여 페이스북과 인스타그램에서 특정 개인에게 도달할 수 있습니다. 데이터는 사용자 프로필과 매칭되어 타겟 광고를 전달할 수 있게 됩니다.

### 주요 기능

- **오디언스 생성**: 웹사이트 가입, CRM 기록, 구매 기록 등 자신의 고객 데이터 소스에서 목록을 구축합니다.
- **데이터 매칭**: 메타가 귀하의 데이터를 자체 사용자 프로필과 매칭합니다.
- **광고 타겟팅**: 매칭된 고객에게 관련성 높은 광고를 전달합니다.
- **개인정보보호 및 규정 준수**: GDPR 등 개인정보보호 규정을 완전히 준수합니다.
- **오디언스 세분화**: 최근 구매자나 고가치 고객 등의 그룹을 타겟팅합니다.
- **유사 오디언스**: 기존 고객과 유사한 새로운 사람들에게 도달합니다.
- **고객 목록 형식**: 유효한 식별자가 포함된 CSV 형식이 필요합니다.

### 장점

- 기존 고객을 정확하게 타겟팅합니다.
- 광고 관련성과 캠페인 성과를 개선합니다.
- 과거 고객을 재참여시키고 재구매를 유도합니다.

## 고객 목록 커스텀 오디언스 만드는 방법

고객 목록 커스텀 오디언스 만들기는 2단계 과정입니다:

- Create a Custom Audience(커스텀 오디언스 만들기)를 클릭합니다(캠페인 플로우 또는 Audiences 탭에서).

![커스텀 오디언스 만들기 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051625884/original/OFCWvUoFLruMb-ReaqPJHeaiBWAdMW8HWA.png?1755176333)

- Customer List(고객 목록)를 소스로 선택하고 진행합니다.
![고객 목록 소스 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051626090/original/S4eMJsvFxtciLQKmELJhKaFM4rO6bUD1mg.png?1755176398)

![고객 목록 설정 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051626205/original/A8CkfGOI5yy5c327iKkrkechHoKDGvcBjw.png?1755176453)

오디언스에 고객을 추가하는 방법은 두 가지입니다:

### 1. CSV 업로드

- 고객 데이터가 포함된 CSV 파일을 업로드합니다.
- [메타의 형식 가이드라인](https://www.facebook.com/business/help/2082575038703844?id=2469097953376494)을 따르세요.
- 제공된 샘플 파일을 사용하여 데이터를 올바르게 준비하세요.

### 2. 스마트 리스트 사용 (정적 vs 동적)

스마트 리스트에서 고객 목록 커스텀 오디언스를 만들 때, 오디언스가 생성 시점에 고정되어야 하는지(정적) 아니면 스마트 리스트가 변경될 때 자동으로 업데이트되어야 하는지(동적) 선택할 수 있습니다.

![스마트 리스트 선택 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051626211/original/soHNVi7588NVXXLt9fwCdsI7CwKxOjOd-Q.png?1755176461)

![정적/동적 옵션 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051626256/original/mTIm2EVpc8finmjHt8ulHsONRnvbZyTFRA.png?1755176497)

### 정적 오디언스

- 오디언스 생성 시점에 선택된 스마트 리스트에 있는 연락처만 포함됩니다.
- 나중에 연락처가 추가되거나 삭제되어도 업데이트되지 않습니다.
- 통제된 타겟팅이나 고정된 오디언스가 필요한 시나리오에 이상적입니다.

### 동적 오디언스

- 스마트 리스트의 변경 사항을 자동으로 반영합니다.
- 스마트 리스트에 추가된 새 연락처는 포함되고, 삭제된 연락처는 제외됩니다.
- 오디언스를 지속적으로 최신 상태로 유지하는 데 이상적입니다.

단계:

- Use Smart List(스마트 리스트 사용)를 선택합니다.
- 드롭다운에서 하나 이상의 스마트 리스트를 선택합니다(연락처(Contacts) 섹션에서 미리 만든 목록만 사용 가능).
- Advanced Conditions(고급 조건)에서 다음 중 선택합니다:
  - No, only include the contacts currently on the Smart List(아니요, 현재 스마트 리스트에 있는 연락처만 포함) (정적)
  - Yes, include the dynamic updates(예, 동적 업데이트 포함) (동적)
- 총 연락처 수를 검토합니다.
- 오디언스 이름과 선택적 설명을 추가합니다.
- Create Audience(오디언스 만들기)를 클릭하면 요약에서 오디언스가 정적인지 동적인지 표시됩니다.

## 중요 사항

- 캠페인 사용 시 최소 오디언스 크기 요구사항이 적용됩니다.
- CSV 업로드의 경우, 잘못된 형식은 업로드 오류를 발생시킵니다.

![CSV 업로드 오류 예시](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051626346/original/UUZ96IsdNPkk90mDb9SDY-2KKG3VzfdkEQ.png?1755176560)

- 스마트 리스트 오디언스의 경우, 정적/동적 선택이 생성 후 오디언스 동작을 결정합니다.
- 오디언스를 만들기 전에 메타의 [고객 목록 커스텀 오디언스 서비스 약관](https://www.facebook.com/legal/terms/customaudience)을 수락해야 합니다.
- 생성되어 준비되면 오디언스가 리타겟팅 오디언스(Retargeting Audience) 드롭다운과 Audiences(오디언스) 탭에 나타납니다.
