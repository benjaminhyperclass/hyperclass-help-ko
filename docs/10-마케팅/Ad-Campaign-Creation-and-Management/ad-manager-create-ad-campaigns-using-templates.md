# 광고 관리자: 템플릿을 활용한 광고 캠페인 만들기

1. 템플릿 라이브러리의 템플릿을 활용한 캠페인 만들기
2. 기존 캠페인을 템플릿으로 저장하여 템플릿 라이브러리에 업로드하기

## **템플릿 라이브러리에서 광고 관리자 찾기**

에이전시 화면의 템플릿 라이브러리(Template Library) 섹션에 광고 관리자 탭이 추가되어 광고 관리자 전용 템플릿을 모두 볼 수 있습니다. 사용자는 리드 생성(Lead Generation), 웹사이트 트래픽(Website Traffic) 등 광고 캠페인 목표에 따라 템플릿을 필터링할 수 있습니다. 향후 캠페인 생성 과정에 새로운 목표가 추가되면, 템플릿 라이브러리에도 해당 필터가 함께 표시됩니다.

![템플릿 라이브러리 광고 관리자 탭](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155035198034/original/xFkeiS_fhHxzhdyzf0t1B3xgtMshTbdbYg.png?1729608996)

### 광고 관리자에서 템플릿 사용의 장점

템플릿을 활용해 광고 캠페인을 만들면 여러 장점이 있습니다:

- **시간과 노력 절약**: 템플릿 라이브러리의 미리 디자인된 광고 템플릿을 활용하면 광고 캠페인 제작 시간과 노력을 크게 줄일 수 있습니다. 처음부터 시작할 필요가 없어집니다.
- **고품질 디자인**: 라이브러리의 템플릿은 고품질 스톡 이미지와 콘텐츠를 포함하고 있어 자유롭게 사용할 수 있습니다. 전문적이고 일관된 브랜드 이미지를 만드는 데 도움이 됩니다.
- **사용자 친화적**: 템플릿 라이브러리 기능은 사용하기 쉽고 직관적입니다. 비즈니스 필요에 따라 템플릿을 필터링하고 선택하기 전에 미리보기할 수 있습니다.
- **커스터마이징 가능**: 템플릿은 캠페인 제작의 좋은 기반을 제공하지만, 필요에 따라 언제든 수정할 수 있습니다. 이미지, 색상, 콘텐츠를 변경하여 브랜드에 맞는 고유한 디자인을 만들 수 있습니다.
- **비용 효율적이고 접근성 좋음**: 커스텀 디자인을 만들거나 디자이너를 고용하는 것은 비용이 많이 듭니다. 템플릿 라이브러리는 고품질 광고 캠페인을 만드는 비용 효율적인 대안을 제공합니다. 디자인 예산이 충분하지 않은 소규모 사업자, 프리랜서, 마케터에게 특히 유용합니다.

## **템플릿을 활용한 캠페인 만들기**

템플릿 라이브러리에 있는 광고 관리자 템플릿을 활용해 캠페인을 만드는 단계는 다음과 같습니다.

- 광고 관리자 홈 화면에서 기존의 "캠페인 만들기(Create Campaign)" 버튼이 드롭다운으로 변경되어 '처음부터 만들기(Start from Scratch)'와 '광고 관리자 템플릿(Ad Manager Templates)' 두 가지 옵션을 제공합니다.

![캠페인 만들기 드롭다운 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033403706/original/45oYpU9rqCZN4r_I1SVj8Xm6lsd40KXu3w.png?1727120532)

- '처음부터 만들기'를 클릭하면 기존 광고 캠페인 생성 과정이 시작됩니다.
- '광고 관리자 템플릿'을 클릭하면 모든 템플릿을 보여주는 화면이 팝업으로 열립니다.

![템플릿 선택 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033403727/original/gJSt4hxJtApLJ0IgM--L3zEAAzk9-Le9Iw.png?1727120613)

- 사용자가 템플릿 중 하나를 선택하면 미리보기를 볼 수 있고, '계속하기(Continue)'를 클릭하면 템플릿 추가 과정이 시작됩니다.

![템플릿 미리보기 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033403812/original/0zqfUZ2Fm15S5_kxb0Q7YtVL2ElEAHhokg.png?1727120756)

![템플릿 적용 과정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033403863/original/03qGyK7b9S2oqVluV3vc-kUY5G6z8dPBZw.png?1727120850)

- 모든 세부 정보가 미리 입력된 광고 캠페인이 생성되어, 사용자는 세부 정보를 확인하고 목표가 리드 생성인 경우 Facebook 폼을 캠페인에 할당한 후 광고 캠페인을 게시할 수 있습니다.

![완성된 캠페인 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033403881/original/HYaWHbaQaortkNCcOAgjNj3vql7xI2NVAg.png?1727120880)

**참고사항**: 템플릿을 사용해 캠페인을 만들 때마다 다음 사항들은 캠페인에 연결되지 않으므로, 캠페인을 게시하기 전에 사용자가 수동으로 입력해야 합니다:
- 새로 생성된 캠페인에는 연결된 페이지가 없습니다. 사용자가 캠페인 편집을 클릭하면 기본 페이지로 설정된 페이지가 이 캠페인에 할당되며, 원하는 경우 사용자가 변경할 수 있습니다.
- 캠페인에 지정된 페이지가 없기 때문에, '리드 생성' 목표에 사용할 폼을 사용자가 수동으로 선택해야 합니다.

## **기존 캠페인으로 템플릿 만들기**

뛰어난 성과를 거둔 캠페인이 있어서 향후 캠페인 생성 과정에서 템플릿으로 저장하고 싶거나, 다른 하위 계정 사용자들이 이 캠페인 형식을 사용할 수 있도록 하고 싶은 경우가 있습니다. 이를 위해 캠페인으로 템플릿을 만드는 단계는 다음과 같습니다:

- 기존 캠페인 목록에서 사용자는 템플릿으로 사용하고 싶은 캠페인을 선택하여 해당 캠페인의 줄임표를 클릭하고 '템플릿 라이브러리에 업로드(Upload to Template Library)'를 선택할 수 있습니다.

![캠페인 템플릿 업로드 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033404707/original/0HH9mr8LFFvBtCGSc6O6IjwlIpWgmgFwbQ.png?1727122599)

- 대화상자가 열리면 템플릿 이름과 이 템플릿이 속할 카테고리를 입력합니다. 카테고리는 템플릿을 구분하는 중요한 단계이므로, 원하는 카테고리가 없다면 템플릿 라이브러리 설정에서 새로 만들 수 있습니다.

![템플릿 정보 입력 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033404749/original/jVF-wgZopFCtmNJH9C045Y-ZktDkzbd_TA.png?1727122710)

- '업로드(Upload)'를 클릭하면 캠페인이 초안 템플릿으로 저장됩니다.
- '업로드'를 클릭하면 초안 템플릿을 보여주는 화면이 열리며, 필요한 경우 수정하여 초안으로 저장하거나 템플릿으로 게시할 수 있습니다.

![초안 템플릿 편집 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033404765/original/eqMR3T2bKpP8_bCEdaapLKQxJVWPDNnjng.png?1727122770)
