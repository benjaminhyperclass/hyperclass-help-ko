# 도움말 가이드: 글로벌 커스텀 컬러

모든 디자인에서 브랜드 일관성을 유지하는 것은 시간이 많이 걸릴 수 있습니다. 글로벌 커스텀 컬러를 사용하면 브랜드 색상을 한 번만 정의하고 모든 곳에서 사용할 수 있습니다. 이 색상을 업데이트하면 모든 자산에 즉시 반영되어 시간과 노력을 절약할 수 있습니다.

## 글로벌 커스텀 컬러란 무엇인가요?

글로벌 커스텀 컬러는 브랜드 보드 아래 글로벌 설정(Global Settings)에 저장되는 브랜드 색상입니다.

- 양방향 동기화가 지원됩니다. 즉, 색상을 업데이트하면 해당 색상이 사용된 모든 곳(이메일, 퍼널, 폼, 페이지 등)에서 자동으로 업데이트됩니다.

- 기본적으로 플랫폼의 모든 색상 선택기에 표시됩니다.

## 디자인 킷 색상과 어떻게 다른가요?

- **디자인 킷 색상**: 디자인 빌더에서 빠르게 접근할 수 있는 색상입니다. 동기화되지 않으므로 업데이트해도 기존 디자인은 변경되지 않습니다.

- **글로벌 커스텀 컬러**: 브랜드 레벨에 저장되며 양방향 동기화됩니다. 한 번 업데이트하면 해당 색상이 사용된 모든 곳에 변경사항이 적용됩니다.

간단히 말해서: 디자인 킷 = 정적인 빠른 접근, 글로벌 커스텀 컬러 = 동적이고 동기화되는 브랜드 일관성입니다.

## 글로벌 커스텀 컬러의 장점

- **자산 간 일관성** – 모든 디자인에서 동일한 모양과 느낌을 유지합니다.

- **시간 절약** – 색상을 한 번만 업데이트하면 모든 곳에 반영됩니다.

- **쉬운 리브랜딩** – 에이전시에서 퍼널과 이메일을 복제/공유하고, 글로벌 색상만 업데이트하여 새로운 로케이션을 빠르게 리브랜딩할 수 있습니다.

- **항상 접근 가능** – 글로벌 색상은 모든 색상 선택기에 표시됩니다.

- **빠른 가져오기** – 기존 커스텀 값을 가져와서 더 빠른 설정이 가능합니다.

## 글로벌 커스텀 컬러 사용 방법

### 1단계: 글로벌 설정 열기

- Marketing(마케팅) → Brand Boards(브랜드 보드) → Global Settings(글로벌 설정)로 이동합니다.

### 2단계: 색상 추가 또는 관리

- **새 색상 추가** → 이름과 헥스 코드를 입력합니다.

![새 색상 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055821243/original/1Ogy8jwZJvPGxTUD2VXOJ699T1Qohutwjw.jpeg?1760341226)

- **편집** → 언제든지 색상 세부 정보를 업데이트합니다.

![색상 편집](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055821241/original/wahMx1uOrGjxPhGCWZsLJRnWUkM-mnkhKA.jpeg?1760341226)

- **삭제** → 사용하지 않는 색상을 제거합니다.

### 3단계: 색상 가져오기

- New(새로 만들기)를 클릭한 다음 Import(가져오기)를 선택합니다.

- 가져올 색상을 선택하고 진행합니다.

![색상 가져오기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055821249/original/NddzdvgG86xnUBbCI9_7y9BvoPhMcy9cbA.jpeg?1760341226)

### 4단계: 빌더에서 사용

- 이메일, 퍼널, 폼 또는 페이지에서 색상 선택기를 엽니다.

- 글로벌 커스텀 컬러가 자동으로 나타납니다.

- 선택기 안의 Manage(관리)를 사용하여 즉석에서 색상을 추가/편집할 수 있습니다.

![색상 선택기 사용](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055821244/original/3lKw7nGwmrJgczkizZ9LELsytTffanC0lA.jpeg?1760341226)

## 자주 묻는 질문

**Q: 글로벌 컬러를 변경하면 기존 자산도 업데이트되나요?**
A: 네. 글로벌 커스텀 컬러는 양방향으로 동기화됩니다. 모든 변경사항이 해당 색상이 적용된 모든 디자인에 즉시 반영됩니다.

**Q: 색상 선택기에 글로벌 컬러가 보이지 않아요.**
A: 이 섹션은 글로벌 설정에서 최소한 하나의 색상을 추가했을 때만 나타납니다.

**Q: 기존 커스텀 값 기반 색상을 가져올 수 있나요?**
A: 네. 커스텀 값(Custom Values) 섹션에서 글로벌 설정으로 가져올 수 있습니다. 이는 복사본만 생성하며 커스텀 값 기반 색상을 완전히 이전하지는 않습니다.

**Q: 디자인 킷 색상과 어떻게 다른가요?**
A: 디자인 킷 색상은 빠른 접근만을 위한 것이며 동기화되지 않습니다. 글로벌 커스텀 컬러는 모든 자산에 걸쳐 동기화됩니다.

**Q: 에이전시에서 고객 온보딩에 사용할 수 있나요?**
A: 물론입니다! 에이전시는 퍼널/이메일을 복제하고 글로벌 컬러만 업데이트하여 새 고객을 위해 즉시 리브랜딩할 수 있습니다.
