# **카운트다운 타이머 라벨이란?**

커스텀 카운트다운 타이머 라벨을 사용하면 각 타이머 구간 아래 기본으로 표시되는 "Days / Hours / Minutes / Seconds" 텍스트를 원하는 문구로 교체할 수 있습니다. 이 기능은 사용자 계정의 로케일을 자동으로 감지하여 해당 언어로 라벨을 미리 채워주므로 설정 시간을 절약하고 문화적으로 적절한 타이머를 보장합니다.

---

## **카운트다운 타이머 라벨의 주요 장점**

- **즉시 현지화:** 기본 텍스트가 하위 계정의 언어 설정에 맞춰져 진정한 글로벌 경험을 제공합니다.

- **브랜드 보이스 맞춤:** "Hours"를 "Hrs"나 "시간"으로 변경하여 브랜드 카피를 일관성 있게 유지하세요.

- **명확성 향상:** "쇼핑 마감일까지"와 같은 캠페인 특화 문구를 사용하세요.

- **디자인 작업 감소:** 각 언어별로 별도 그래픽을 만들 필요가 없습니다.

- **전환율 증대:** 타이머가 친숙하고 명확하게 느껴질 때 독자들의 참여도가 높아집니다.

## 카운트다운 타이머 라벨 변경 방법

언제든지 자동 생성된 텍스트를 변경할 수 있습니다:

- **Marketing(마케팅) → Countdown Timer(카운트다운 타이머)**로 이동하세요.

![마케팅 메뉴에서 카운트다운 타이머 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054946989/original/R1W119Y7lHgX3dakzp8m_Gjx5lZDibrN1A.png?1759245904)

- 타이머 이름을 클릭하여 기존 타이머를 열거나 + New를 클릭하여 새 타이머를 만드세요.

![타이머 목록에서 기존 타이머 선택 또는 새 타이머 생성](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054947190/original/YIDtgDmw0Fh3drZG8eHC_Bkr2m3FC-EwCw.png?1759245950)

- 라벨 섹션에서 드롭다운을 사용하여 각 라벨을 "Visible(표시)" 또는 "Hidden(숨김)"으로 설정하세요.

## 자주 묻는 질문

**Q: 라벨 변경이 소급 적용되나요?**

네. 라벨 텍스트를 업데이트하면 해당 타이머의 모든 라이브 인스턴스가 새로고침됩니다.

**Q: 이메일과 퍼널에서 다른 라벨을 사용할 수 있나요?**

각 개별 타이머는 하나의 라벨 세트를 가집니다. 다른 문구가 필요한 경우 타이머를 복제하여 생성하세요.

**Q: 자동 번역이 모든 언어에서 작동하나요?**

시스템은 하위 계정 언어를 기반으로 기본값을 설정하지만, 방언의 미묘한 차이를 위해서는 수동 편집이 여전히 필요할 수 있습니다.

**Q: 라벨 변경이 다이나믹 링크나 만료 액션에 영향을 주나요?**

아니요. 라벨 편집은 화면상의 텍스트만 업데이트하며, 모든 리다이렉트와 로직은 그대로 유지됩니다.
