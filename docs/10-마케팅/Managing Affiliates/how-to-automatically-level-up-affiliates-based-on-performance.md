# 성과 기반 제휴 마케터 자동 승급 설정 방법

이 가이드는 Hyperclass에서 성과 목표 달성 시 제휴 마케터를 새로운 캠페인으로 자동 승급시키는 방법을 설명합니다. 제휴 마케터 승급 기능의 작동 원리, 장점, 단계별 설정 방법을 배우게 됩니다. 이 가이드를 마치면 최소한의 수동 작업으로 성과 기반 제휴 구조를 구축할 수 있습니다.

# 제휴 마케터 승급 기능이란?

Hyperclass의 제휴 마케터 승급 기능은 성과를 바탕으로 제휴 마케터를 상위 티어 캠페인으로 자동 승격시키는 프로세스를 자동화합니다. 특정 고객 수 달성이나 목표 매출 달성과 같은 명확한 마일스톤을 설정하여, 수동 추적이나 개입 없이 우수한 성과를 거둔 제휴 마케터에게 보상을 제공할 수 있습니다. 이 기능은 제휴 관리를 효율화하고 파트너들의 동기를 부여하기 위해 설계되었습니다.

---

# 제휴 마케터 승급 기능의 주요 장점

자동화된 제휴 마케터 승급은 비즈니스와 제휴 마케터 모두에게 여러 가지 이점을 제공합니다:

- **자동 티어 승급:** 제휴 마케터가 설정된 목표를 달성하면 자동으로 상위 티어 캠페인으로 이동하여 수동 작업이 줄어듭니다.

- **커스터마이징 가능한 마일스톤:** 고객 수나 매출과 같은 성과 기반 트리거를 설정할 수 있습니다.

- **유연한 캠페인 선택:** 제휴 마케터가 승급할 캠페인을 선택하여 다양한 프로그램 구조를 지원합니다.

- **원활한 고객 처리:** 기존 고객을 현재 캠페인에 유지할지 제휴 마케터와 함께 이동시킬지 결정할 수 있습니다.

- **동기 부여 향상:** 제휴 마케터가 더 높은 목표에 도달하도록 인센티브를 제공하여 참여도와 성과를 높입니다.

- **다양한 캠페인 유형 지원:** 웹사이트, 스토어, 퍼널 캠페인 간 승급을 지원하여 최대한의 유연성을 제공합니다.

# 제휴 마케터 자동 승급 설정 방법

자동 제휴 마케터 승급을 설정하면 프로그램이 공정하고 동기부여가 되도록 유지할 수 있습니다. 다음 단계에 따라 기능을 구성하세요:

### 제휴 캠페인에 접근하기

왼쪽 사이드바에서 마케팅(Marketing)으로 이동하고 상단 네비게이션의 제휴 관리(Affiliate Manager) 드롭다운을 클릭합니다. 그다음 목록에서 캠페인(Campaign)을 선택하여 제휴 캠페인 대시보드를 엽니다.

![제휴 캠페인 접근](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155049089800/original/phMlZxO7wD3gMaBBmI2TNoMU78cv4xXLoQ.png?1751290319)

### 캠페인 생성 또는 찾기

우측 상단의 + 추가(Add) 버튼을 클릭하여 새 제휴 캠페인을 생성합니다. 기존 캠페인을 업데이트하는 경우 이름으로 검색(Search by Name) 바를 사용하여 목록에서 빠르게 찾으세요.

![캠페인 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155049089812/original/32ADABMDqMMaUGPK4LxNW7AGplsmf2GgjQ.png?1751290333)

### 캠페인 상세 정보 입력 (새 캠페인의 경우)

새 캠페인을 생성하는 경우 트래픽 소스를 선택하고 캠페인을 연결하여 캠페인 상세 정보 단계를 완료하세요.

![캠페인 상세 정보](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155049089818/original/mq1iuNr-NEo-tEcDrRAzlvIaeT_sfYN-Lg.png?1751290347)

### 승급 옵션 활성화

커미션(Commissions) 단계에서 다른 캠페인으로 승급(Level Up to another Campaign) 섹션을 확장하여 성과 마일스톤을 바탕으로 제휴 마케터를 상위 티어 캠페인으로 승급시키는 방법과 시기를 커스터마이징하세요.

![승급 옵션 활성화](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155049089857/original/73rd86RGDZhEZr4qfna2tlEZSTUPwWx9rg.png?1751290361)

### 목적지 캠페인 선택

드롭다운을 사용하여 제휴 마케터가 승급할 캠페인을 선택합니다. 여기가 우수한 성과를 거둔 제휴 마케터가 정의한 기준을 충족하면 자동으로 이동할 곳입니다.

![목적지 캠페인 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155049089875/original/AQdnGzYQdan_uyM8QDHn9RYT8TuGYzCyYQ.png?1751290376)

### 승급 기준 설정

고객 수(Customers) 또는 매출(Revenue)을 선택하고 해당하는 숫자나 금액을 입력하여 승급 조건을 정의합니다. 이는 제휴 마케터가 다음 캠페인으로 승급할 자격을 갖추는 시점을 결정합니다.

![승급 기준 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155049089886/original/VhOk7H8y2tACKurKKYSA5_tXW71_PLN0bg.png?1751290389)

### 고객 처리 방식 선택

기존 고객의 커미션을 어떻게 처리할지 결정하세요. 원래 캠페인에 유지하거나 제휴 마케터와 함께 이동시킬 수 있습니다. 선택에 따라 향후 커미션이 어디서 추적될지 결정됩니다.

![고객 처리 방식](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155049089906/original/o0I03QY4i_r2u5Mq0zgYKmlt7uGfYvWOjA.png?1751290403)

### 완료 및 발행

커미션 구조, 승급 로직, 고객 처리 규칙을 포함한 모든 캠페인 설정을 검토합니다. 모든 것이 올바르게 보이면 **다음(Next)**을 클릭하고 캠페인을 발행하여 성과 기반 제휴 흐름을 활성화하세요.

# 자주 묻는 질문

**Q: 승급 자동화를 일시적으로 멈출 수 있나요?**

기본적인 "일시정지" 버튼은 없지만, 드롭다운에서 선택된 캠페인을 다시 "승급 캠페인 없음"으로 전환하여 일시적으로 승급을 비활성화할 수 있습니다. 나중에 재개할 때는 수동으로 다시 구성해야 합니다.

**Q: 제휴 마케터를 다른 하위 계정의 캠페인으로 승급시킬 수 있나요?**

아니요, 승급 프로세스는 단일 하위 계정 내에서만 작동합니다. 여러 하위 계정을 관리하는 경우, 각 계정마다 제휴 마케터와 캠페인 구성을 수동으로 복제하고 전환을 수동으로 관리해야 합니다.

**Q: 제휴 마케터가 이미 승급한 후 원본 캠페인에서 상품이 제거되면 어떻게 되나요?**

원래 캠페인에서 상품을 제거해도 이미 승급한 제휴 마케터에게는 영향을 주지 않지만, 원래 캠페인에서 해당 상품에 대한 향후 커미션 추적은 중단됩니다. 상품이 목적지 캠페인에도 존재하는 경우, 선택한 승급 규칙과 고객 처리 옵션에 따라 향후 구매가 해당 캠페인에서 추적됩니다.

**Q: 태그, 연락처 속성, 또는 제휴 마케터별 필터를 사용하여 승급 자격을 제어할 수 있나요?**

현재 승급은 정량적 성과(고객 수 또는 매출)에 의해서만 트리거됩니다. 태그, 속성, 또는 수동 자격 논리를 기반으로 한 필터는 지원되지 않지만, 더 광범위한 자동화 전략의 일부로 승급 후 태그를 지정하는 워크플로우를 사용할 수 있습니다.
