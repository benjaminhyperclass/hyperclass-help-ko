# 제휴사별 고객 관리

제휴사별 고객을 효과적으로 관리하는 것은 정확한 수수료 추적을 유지하고 제휴 프로그램이 원활히 운영되도록 하는 핵심입니다. 비활성 고객을 제거하거나, 다른 제휴사로 재배정하거나, 수수료 지급 대상을 제어해야 할 때, 적절한 도구가 있으면 프로세스가 매끄러워집니다.

## **제휴사에서 고객 삭제**

### 1단계: 제휴사 프로필로 이동

- 제휴 관리(Affiliate Manager) 페이지로 이동하세요
- 고객을 제거하려는 제휴사를 찾습니다
- 제휴사 이름을 클릭하여 상세 정보를 엽니다

### 2단계: 고객 찾기

- 고객(Customers) 탭에서 삭제하려는 고객을 찾습니다
- 검색창이나 필터를 사용하면 빠르게 찾을 수 있습니다

![고객 검색 및 삭제](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155041694830/original/Rs28ZeQLTK7EYNKOTdjNimkI6ixmeeaq3w.png?1739776643)

### 3단계: 고객 삭제

- 고객 이름 옆의 삭제(Delete) 버튼을 클릭합니다
- 메시지가 나타나면 작업을 확인합니다

![고객 삭제 확인](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155041694841/original/ze_KOIAsP6a9KNKaSC6XlRXbR8IRcmulDw.png?1739776653)

**고객 삭제 후 어떻게 되나요?**
- 해당 고객은 더 이상 제휴사와 연결되지 않습니다
- 이 고객으로부터 발생하는 향후 수수료는 더 이상 생성되지 않습니다

## 고객 비활성화/활성화

### 1단계: 제휴사 프로필로 이동

- 제휴 관리(Affiliate Manager) 페이지로 이동하세요
- 고객을 제거하려는 제휴사를 찾습니다
- 제휴사 이름을 클릭하여 상세 정보를 엽니다

### 2단계: 고객 찾기

- 고객(Customers) 탭에서 비활성화/활성화하려는 고객을 찾습니다
- 검색창이나 필터를 사용하면 빠르게 찾을 수 있습니다

### 3단계: 고객 비활성화

- 점 3개(⋮) 버튼을 클릭하고 고객 비활성화(Disable Customer)를 선택합니다
- 작업을 확인합니다
- 상태 태그가 "비활성화됨(Disabled)"으로 변경된 것을 확인할 수 있습니다
- 활성화하려면 같은 과정을 따르고 고객 활성화(Enable Customer)를 클릭하세요. 비활성화된 고객만 다시 활성화할 수 있습니다

![고객 비활성화](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042193140/original/ENSjul7U4c4ZIhsSsXwqQGIA0wdanpeK4Q.png?1740479229)

**언제 고객을 비활성화해야 하나요?**
고객을 "비활성화"로 설정하면 더 이상 매출이 추적되지 않고, 수수료도 발생하지 않습니다. 이는 허위 가입을 처리하거나 특정 상황에서 수수료 배정을 방지하는 데 유용합니다.

## 수수료 삭제

### 1단계: 제휴사 프로필로 이동

- 제휴 관리(Affiliate Manager) 페이지로 이동하세요

### 2단계: 수수료 찾기

- 수수료(Commissions) 탭에서 삭제하려는 수수료를 찾습니다

## 제휴사 간 고객 이동

**중요 사항:**
- 같은 캠페인 내에서만 고객을 이동할 수 있습니다
- 수수료를 함께 이동하기로 선택하면, 미지급 수수료만 이전됩니다. 이미 지급된 수수료는 원래 제휴사에게 그대로 남습니다

### 1단계: 제휴 관리로 이동

- 현재 고객이 배정된 제휴사 프로필로 이동합니다

### 2단계: "고객" 탭으로 스크롤

- 제휴사 프로필 내에서 고객(Customers) 섹션을 찾아 스크롤합니다

### 3단계: 점 3개 메뉴(⋮) 클릭

- 이동하려는 고객 옆의 점 3개 버튼을 클릭하여 작업 메뉴를 엽니다

### 4단계: "고객 이동" 선택

- 드롭다운에서 "고객 이동(Move Customer)" 옵션을 선택합니다

![고객 이동 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044773814/original/-Kx6P7L6GA8Wf6CanKVN7wqjtiCCX_ikAw.png?1744181966)

### 5단계: 대상 제휴사 선택

- 팝업에서 고객을 이동할 제휴사를 선택합니다
- 미지급 수수료를 함께 이전하려면 "이 고객의 모든 관련 수수료를 대상 제휴사로 함께 이동(Also move all associated commissions of this Customer to destination Affiliate)" 옵션을 켜세요
- 이동(Move)을 클릭하여 이동을 완료합니다

![고객 이동 확인](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044773868/original/zitPHCIS7LlmlG0TdOQCm55QuXWZ2PLjCA.png?1744182029)
