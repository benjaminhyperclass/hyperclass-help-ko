# 제휴 관리에서 어필리에이트 추가하는 방법

어필리에이트란 당신의 제품이나 서비스를 홍보하고 그들이 생성한 매출이나 리드를 기반으로 수수료를 받는 개인이나 기업을 말합니다. 고유한 제휴 링크를 사용하여 홍보 활동을 추적하며, 일반적으로 "어필리에이트 마케터" 또는 "퍼블리셔"라고 불립니다.

제휴 관리(Affiliate Manager)에서 어필리에이트를 추가하고 관리하는 4가지 간단한 방법을 소개합니다:

- [1. 수동으로 새 어필리에이트 추가](#1-수동으로-새-어필리에이트-추가)
- [2. 어필리에이트 일괄 업로드](#2-어필리에이트-일괄-업로드)
- [3. 연락처에서 어필리에이트 가져오기](#3-연락처에서-어필리에이트-가져오기)
- [4. 워크플로우를 사용한 어필리에이트 관리 자동화](#4-워크플로우를-사용한-어필리에이트-관리-자동화)

### 1. **수동으로 새 어필리에이트 추가**

- 제휴 관리 페이지로 이동하여 **"Add"** 버튼을 클릭하세요.
![새 어필리에이트 추가 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033793072/original/7WqyX4a-9AKlhlDodikpn26lci7qqEZnvw.jpeg?1727697097)

- 옵션에서 **"New Affiliate"**를 선택하세요.
![새 어필리에이트 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033793073/original/Pq_NWDhk7ErHR67u-Kf1EXir0VGQKoGexg.jpeg?1727697097)

- 어필리에이트의 세부 정보를 입력하고 캠페인에 배정하세요.
![어필리에이트 정보 입력](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033793075/original/H8tE3OOG2uihj4EmcjvAX6gog_ZaOHMwXA.jpeg?1727697097)

- **"Save"**를 클릭하여 추가를 완료하세요.

### 2. **어필리에이트 일괄 업로드**

- 제휴 관리 페이지로 이동하여 **"Add"**를 클릭하세요.
- **"Upload Affiliate"**를 선택하세요.
![어필리에이트 업로드 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033793078/original/vnAEJi9fYRYVY9Da9lj6e8X1mHBU9W72Lw.jpeg?1727697097)

- 샘플 CSV 파일을 다운로드하고 어필리에이트 데이터를 입력하세요.
![CSV 파일 다운로드](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033793074/original/cm8YGH0SwHASPxi77crSynaeRlWVTEJsLQ.jpeg?1727697097)

- 완성된 파일을 업로드하고 **"Save"**를 클릭하여 일괄 추가하세요.

### 3. **연락처에서 어필리에이트 가져오기**

- 제휴 관리 페이지에서 **"Add"**를 클릭하고 **"Import from Contacts"**를 선택하세요.
![연락처에서 가져오기 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033793077/original/gYok_d1vvEnu8WDudM7ngNzs66UW_0BcVg.jpeg?1727697097)

- 어필리에이트로 전환하고 싶은 연락처를 선택하세요.
![연락처 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033793076/original/FPEVF2ZY8R4_Gyk0ZiA11GQ-aOsjfwNn2w.jpeg?1727697097)

- **"Import"**를 클릭하여 제휴 프로그램에 추가하세요.

### 4. **워크플로우를 사용한 어필리에이트 관리 자동화**

워크플로우 액션을 사용하여 어필리에이트를 자동으로 추가, 제거, 업데이트할 수 있습니다. 다음 액션들을 사용할 수 있습니다:

- **제휴 관리에 추가(Add to Affiliate Manager)**: 연락처를 활성 상태로 제휴 관리에 자동 추가합니다.
- **어필리에이트 업데이트(Update Affiliate)**: 어필리에이트의 상태를 활성 또는 비활성으로 변경합니다. 
  옵션: **활성으로 표시(Mark as Active)** / **비활성으로 표시(Mark as Inactive)**
- **제휴 캠페인에 추가(Add to Affiliate Campaign)**: 캠페인 탭의 특정 캠페인에 어필리에이트를 추가합니다.
- **제휴 캠페인에서 제거(Remove from Affiliate Campaign)**: 기존 캠페인에서 어필리에이트를 제거합니다.

[자세한 내용은 여기에서 확인하세요.]([getting-started-with-affiliate-automations](getting-started-with-affiliate-automations.md))
