# 텍스트 스니펫 (템플릿)

미리 저장한 텍스트 스니펫을 이메일 편집기에 삽입하여 이메일 제작 과정을 쉽게 간소화하세요. 템플릿을 만들거나 캠페인을 시작할 때 모두 사용 가능합니다. 이 기능으로 효율성을 높이고, 일관성을 보장하며, 캠페인 제작을 그 어느 때보다 빠르게 만들 수 있습니다.

# 텍스트 스니펫(템플릿)이란?

텍스트 스니펫을 사용하면 재사용 가능한 텍스트 블록(그리고 캠페인에서는 첨부파일)을 이메일에 빠르게 삽입할 수 있습니다. 자주 사용하는 콘텐츠를 스니펫으로 저장하여 브랜드 일관성을 유지하고, 반복 작업을 줄이며, 템플릿과 캠페인 전반에서 이메일 제작 과정을 가속화할 수 있습니다.

---

## 텍스트 스니펫의 주요 장점

이메일 워크플로우에서 텍스트 스니펫을 사용하면 다음과 같은 여러 가지 장점이 있습니다:

- 콘텐츠 재사용으로 시간 절약: 자주 사용하는 섹션을 다시 작성하거나 복사-붙여넣기할 필요가 없습니다.

- 이메일 간 일관성 보장: 메시징, 면책조항 또는 프로모션 섹션을 표준화합니다.

- 캠페인 제작 간소화: 즉시 사용 가능한 섹션으로 빠르게 이메일을 구성합니다.

- 캠페인에서 자동 첨부파일 포함: 스니펫에 저장된 첨부파일이 캠페인에서 사용될 때 자동으로 추가됩니다.

- 오류 감소: 미리 승인된 콘텐츠를 사용하여 수동 입력 실수를 최소화합니다.

---

## 텍스트 스니펫 설정 방법

텍스트 스니펫을 설정하고 사용하는 것은 간단하며, 이메일을 더 효율적으로 만들 수 있도록 도와줍니다. 적절한 설정으로 팀이 필요할 때마다 올바른 콘텐츠에 접근하고 삽입할 수 있습니다.

### **스니펫 섹션 접근하기**

좌측 네비게이션 메뉴를 열고 Marketing(마케팅) > Snippets(스니펫)을 클릭하여 계정에서 텍스트 스니펫을 생성하거나 관리하기 시작하세요. 여기서 저장된 모든 스니펫을 보관하고 재사용할 수 있습니다.

![스니펫 섹션 접근](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048076510/original/kwx7hpFdLDvnMqAJXVEPjz_kXOzE1IXwsA.png?1749642540)

### **새 텍스트 스니펫 만들기**

Snippets(스니펫) 패널에서 **+ Add Snippet(+ 스니펫 추가)**를 클릭하여 스니펫 생성 창을 실행하세요. 이름, 본문을 입력하고 선택적으로 파일을 첨부하거나 전화로 메시지를 테스트할 수 있습니다. 오른쪽의 미리보기 창에서 모바일 기기에서 메시지가 어떻게 표시될지 확인할 수 있습니다.

![새 텍스트 스니펫 만들기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048076866/original/hWRVXk36tckfj8Yt8jYKKvTj3ly4VMSa-A.png?1749642747)

### **스니펫 이름 지정 및 미리보기**

원하는 텍스트를 입력하고, 해당되는 경우 첨부파일을 추가하세요(첨부파일은 캠페인에서만 포함됩니다).

향후 사용을 위해 스니펫을 저장하세요. 스니펫에 대한 설명적인 제목을 제공하되, 가급적 다루는 질문이나 사용 사례를 기반으로 하세요(예: "어떤 서비스를 제공하시나요?"). 모바일 미리보기는 저장하기 전에 텍스트가 명확하고 개인화되어 보이는지 확인하는 데 도움이 됩니다.

![스니펫 이름 지정 및 미리보기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155047320213/original/BHmkOZ33Xuvei6KFL-TROM2ANyLpukTZ7g.png?1748378760)

### **동적 콘텐츠로 스니펫 맞춤화하기**

커스텀 값(Custom Values)과 트리거 링크(Trigger Links)를 사용하여 메시지를 개인화하세요. 실시간 미리보기는 연락처에게 메시지가 어떻게 표시될지 정확히 보여줍니다. 이 보기를 사용하여 톤, 명확성, 서식을 다듬으세요.

![동적 콘텐츠로 스니펫 맞춤화](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048095065/original/YKGJp47d3YgbzwPmdZZH9BqK5jbq49YCQA.png?1749654257)

### **테스트 및 저장**

휴대폰 번호를 입력하고 Send Test(테스트 발송)를 클릭하여 스니펫을 테스트할 수도 있습니다. 스니펫에 만족하면 Save(저장)를 클릭하세요.

**참고:** SMS 메시지로 발송될 때 스니펫의 문자 수, 세그먼트 수, 대략적인 비용을 주의 깊게 확인하세요.

![테스트 및 저장](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048095144/original/FIDnHHQkOXcGUkUlx1qr5DanGqn3drjYA.png?1749654323)

### 대화에서 스니펫 삽입하기

Conversations(대화) 탭에서 연락처에게 응답할 때, Snippets(스니펫) 아이콘을 클릭하고 관련 스니펫을 검색한 후 클릭하여 메시지에 직접 삽입하세요. 시간을 절약하고 일관성을 보장합니다.

![대화에서 스니펫 삽입](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048078800/original/fWdS3ioVT8ZauMSl7AjRRpKr4JpbeeHPkg.png?1749644053)

### 커스텀 스니펫을 사용한 SMS 미리보기

저장된 스니펫을 사용하여 완전히 작성된 SMS 메시지의 예시입니다. 동적 플레이스홀더가 실제 연락처 정보로 대체되어 캠페인이나 수동 아웃리치를 통해 발송되는 각 메시지를 개인화하는 방법을 확인할 수 있습니다.

![커스텀 스니펫 SMS 미리보기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155048078966/original/2gKpHLCrhxoiIwNANwn5e1xx6Tieq_qyvA.png?1749644148)

### 저장된 스니펫 관리하기

효율적인 이메일 작성을 위해 스니펫 라이브러리를 체계적으로 유지하세요. 계정의 Snippets(스니펫) 섹션으로 이동하여 기존 스니펫을 쉽게 **편집**, **삭제** 또는 **이름 변경**할 수 있습니다. 콘텐츠를 최신 상태로 유지하면 팀이 항상 최신 승인된 메시징을 사용할 수 있습니다.

## **자동화(워크플로우)에서 스니펫 삽입하기**

스니펫은 워크플로우 내의 자동 메시지에도 삽입할 수 있습니다. 이를 통해 자주 사용되는 텍스트 블록을 자동화를 구축할 때마다 다시 입력할 필요가 없어 시간을 절약하면서 일관된 메시징을 유지할 수 있습니다.

### **Automation(자동화) > Workflows(워크플로우)로 이동**

기존 워크플로우를 열거나 새로 만드세요.

![워크플로우 이동](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054023090/original/BiHSxTR3GJ0FibdgEyt3JziIfRDpmUnAJw.png?1758186181)

### 액션 추가

Send SMS(SMS 발송) 또는 Send Email(이메일 발송)과 같은 메시징 액션을 추가하세요. 메시지 작성기에서 Templates(템플릿) 옵션을 클릭하여 모든 스니펫의 드롭다운을 확인하세요.

![액션 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054024535/original/CrC7ZfU2-KCpi3hNdfuPUmKu_olJGbSDQA.gif?1758186871)

### **스니펫 선택**

삽입하려는 스니펫을 선택하세요. 텍스트가 메시지 본문에 직접 나타납니다. 워크플로우 액션을 저장하세요.

![스니펫 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054024874/original/BY36tW24-b0C0-qORBHswulKJJzXhybQnQ.png?1758187033)

## 스니펫의 첨부파일 작동 방식

스니펫과 함께 첨부파일이 언제 포함되는지 이해하면 이메일 제작 중 혼란을 피할 수 있습니다.

- **이메일 템플릿:** 스니펫에 연결된 첨부파일은 템플릿에 삽입될 때 포함되지 **않습니다**. 텍스트 콘텐츠만 나타납니다.

- **캠페인 이메일:** 캠페인 이메일에 스니펫을 삽입할 때 첨부파일이 자동으로 **포함됩니다**. 이를 통해 PDF, 브로슈어 또는 제품 가이드와 같은 리소스를 일관되게 쉽게 제공할 수 있습니다.

- **워크플로우:** 스니펫에 포함된 첨부파일은 워크플로우 이메일이나 SMS에서 사용될 때 자동으로 추가되지 않습니다. 자동 이메일에 파일을 포함하려면 워크플로우 단계 내에서 직접 첨부해야 합니다.

## 자주 묻는 질문

**Q: 이메일 템플릿과 캠페인 모두에서 텍스트 스니펫을 사용할 수 있나요?**

네, 텍스트 스니펫은 이메일 템플릿과 캠페인 모두에 삽입할 수 있습니다.

**Q: 스니펫을 폴더나 카테고리로 정리할 수 있나요?**

네, 스니펫을 폴더로 정리할 수 있습니다.

**Q: 저장할 수 있는 스니펫 개수에 제한이 있나요?**

현재 공개된 제한은 없지만, 사용 편의성을 위해 스니펫 라이브러리를 체계적으로 유지하는 것이 모범 사례입니다.

**Q: 내 계정에서 누가 스니펫을 만들거나 편집할 수 있나요?**

스니펫 생성 및 편집 권한은 계정의 사용자 역할(User Role)과 설정에 따라 다릅니다.

**Q: 스니펫에 이미지나 서식이 있는 콘텐츠를 추가할 수 있나요?**

네, 스니펫은 서식이 있는 텍스트와 이미지를 포함할 수 있어 다양한 콘텐츠 요구사항에 다용도로 사용할 수 있습니다.

**Q: 스니펫 사용이 이메일 전달률에 영향을 주나요?**

아니요, 스니펫 사용은 전달률에 영향을 주지 않지만, 콘텐츠가 이메일 컴플라이언스 모범 사례를 따르는지 확인하세요.

**Q: 모든 워크플로우 액션에서 스니펫을 사용할 수 있나요?**

스니펫은 Send SMS(SMS 발송)나 Send Email(이메일 발송)과 같이 메시지 편집기가 포함된 모든 워크플로우 액션에 삽입할 수 있습니다. Wait(대기) 단계나 If/Else(조건 분기) 조건과 같은 메시징이 아닌 액션에서는 사용할 수 없습니다.

**Q: 첨부파일이 있는 스니펫이 워크플로우에서 작동하나요?**

아니요. 스니펫에 포함된 첨부파일은 워크플로우 메시지에서 사용될 때 자동으로 추가되지 않습니다. 파일을 포함하려면 워크플로우 이메일 단계에서 직접 업로드해야 합니다.
