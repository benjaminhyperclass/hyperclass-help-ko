# 기본 브랜드 보드를 변경하는 방법

기본 브랜드 보드는 특히 이메일과 퍼널에서 브랜드의 시각적 정체성을 관리하는 핵심 요소입니다. 색상 및 글꼴 선택기에 어떤 색상과 글꼴 세트가 표시될지를 결정하여 브랜딩 요소에 쉽게 접근할 수 있게 해줍니다. 기본 브랜드 보드를 이해하고 변경하는 방법에 대한 간단한 가이드를 제공합니다.

# 기본 브랜드 보드란?

브랜드 보드는 Hyperclass 전체에서 일관된 스타일을 적용할 수 있도록 핵심 시각적 정체성(색상, 글꼴, 로고)을 저장합니다. 한 보드를 기본으로 설정하면 편집기와 선택기에서 어떤 브랜딩 가이드라인을 표시할지 알려주어 디자인 속도를 높이고 실수를 줄입니다. 기본적으로 처음 만든 브랜드 보드가 기본값으로 설정됩니다.

아래는 사이트 편집기 내에서 표시되는 기본 브랜드 보드 색상의 예시로, 사이트 디자인을 브랜드 가이드라인에 맞게 쉽게 유지할 수 있습니다.

## **기본 브랜드 보드 설정의 주요 장점**

기본값을 선택하면 일상적인 디자인 속도와 표준화가 개선됩니다.

- **일관성**: 편집기 전체에서 색상, 글꼴, 로고를 일치시킵니다
- **속도**: 선택기에서 올바른 색상 견본과 서체를 먼저 표시합니다
- **단순성**: 시즌 프로모션 중에 여러 브랜드 가이드라인 간을 쉽게 전환할 수 있습니다
- **온보딩**: 새로운 팀원이 올바른 스타일을 빠르게 적용할 수 있습니다
- **재사용성**: 브랜드 요소의 단일 신뢰 소스를 장려합니다

## 언제 기본 브랜드 보드를 변경해야 할까요?

- **가이드라인 업데이트**: 새로운 색상, 로고 또는 기타 핵심 요소와 같이 브랜드 가이드라인에 중요한 업데이트가 있을 때 기본 보드를 변경하세요. 이를 통해 모든 자료와 커뮤니케이션이 최신 브랜딩에 맞춰집니다.

- **특별한 경우**: 크리스마스를 위한 빨간색 테마와 같이 시즌별 또는 이벤트별 브랜딩을 반영하고자 할 때 특별한 경우나 테마에 맞춰 기본 보드를 변경하세요. 하지만 기본 보드를 변경하는 것보다는 이러한 경우를 위해 별도의 테마나 템플릿을 만드는 것이 더 효과적일 수 있으며, 필요에 따라 기본 보드와 테마 보드 간을 쉽게 전환할 수 있습니다.

## 기본 브랜드 보드 변경 방법

기본 브랜드 보드 변경은 간단합니다:

#### **1단계: **브랜드 보드로 이동

- 메인 네비게이션 바에서 **Marketing(마케팅)**으로 이동
- 상단 네비게이션 리본에서 **Brand Boards(브랜드 보드)** 탭을 선택
- 또는 Settings(설정) > Brand Boards(브랜드 보드)를 통해 접근

![브랜드 보드 접근 경로](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155031778789/original/5LtHXudHOE-29MDzOYAYw0xfob-suYvIhg.png?1724821543)

#### **2단계: **보드를 기본값으로 설정

- 기본값으로 설정하고 싶은 브랜드 보드를 찾습니다
- 원하는 브랜드 보드 옆의 **점 세 개** 아이콘을 클릭
- **Default(기본값)**으로 설정을 선택

![브랜드 보드를 기본값으로 설정하는 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155031778901/original/PwCjPdd5F8O1fCo_8Sggmdf0XVxODclW6A.png?1724821674)

#### **3단계: **편집 모드

브랜드 보드를 편집하면서 기본값으로 만들 수도 있습니다:

- 원하는 브랜드 보드를 클릭하여 편집기를 엽니다
- 우측 상단에서 Set as Default(기본값으로 설정) 버튼을 클릭

![편집기에서 기본값으로 설정하는 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155031779794/original/as0Wtkew_62Vg8xEbrCI1rQQDXI_oQ8yUA.png?1724823372)

#### 4단계: 기본 보드 확인

- 기본값으로 설정된 브랜드 보드에는 상태로 **Default(기본값)**이 표시됩니다
- 편집기를 열어서 선택한 브랜드 색상이 나타나는지 확인할 수도 있습니다

![기본 브랜드 보드 상태 표시](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155053030180/original/LtcdBbmOvxbrp0bFlkK8y5H7gGy3NW9Fjw.jpeg?1756913959)

## **자주 묻는 질문**

**Q: 나중에 기본값을 다시 바꿀 수 있나요?**

네. 다른 브랜드 보드에서 같은 단계를 반복하여 해당 보드를 기본값으로 설정하면 됩니다. 한 번에 하나의 브랜드 보드만 기본값으로 설정할 수 있습니다.

**Q: 아직 브랜드 보드가 없다면 어떻게 하나요?**

먼저 하나를 만들면 자동으로 기본 브랜드 보드로 설정됩니다. 브랜드 보드 만드는 방법을 알아보려면: [브랜드 보드 만드는 방법](how-to-create-a-brand-board.md)을 참고하세요.

**Q: 기본값을 변경하면 기존 이메일과 페이지가 자동으로 업데이트되나요?**

아니요. 새로운 편집이나 향후 편집 시 선택기에 먼저 나타나는 것을 업데이트합니다. 기존 자산은 수동으로 편집할 때까지 현재 스타일을 유지합니다. 기본 브랜드 보드를 변경하는 것은 기존 자산에 변경 사항을 적용하지 않습니다.
