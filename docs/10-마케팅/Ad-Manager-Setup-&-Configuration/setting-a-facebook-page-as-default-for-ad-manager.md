# Ad Manager 기본 페이스북 페이지 설정하기

이 가이드에서는 연결된 페이스북 페이지 중에서 하나를 Ad Manager의 기본 페이지로 설정하는 방법을 설명해드립니다. 기본 페이지로 설정된 페이지는 새로운 광고 캠페인을 게시할 때 기본적으로 선택되는 페이지가 됩니다.

**참고**: 캠페인을 만들 때는 언제든 원하는 페이지를 직접 선택할 수 있습니다. 기본 페이지 설정은 캠페인 빌더에서 미리 선택되어 있는 페이지가 있어야 하며, 필요에 따라 언제든 변경할 수 있도록 하는 것이 목적입니다.

## **연결된 페이스북 페이지를 기본 페이지로 설정하는 방법**

Ad Manager에 연결된 모든 페이스북 페이지는 Ad Manager 설정(Settings)의 Pages 탭에서 확인할 수 있습니다. Ad Manager 홈 페이지 우측 상단의 톱니바퀴 아이콘이나 설정(Settings) 버튼을 클릭하여 접근할 수 있습니다.

![Ad Manager 설정 접근하기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155031780362/original/2T5jgX_6ZyoIKK2_28qaQhsSkInEsbFgrA.png?1724824290)

![Pages 탭에서 연결된 페이지 목록 보기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155031780463/original/xxu2GeoPT0r9vN6HPSKPMIdVDe0f0wx8Aw.png?1724824438)

연결된 페이스북 페이지를 기본 페이지로 설정하려면 각 페이지 행의 오른쪽 끝에 있는 점 세 개(⋯) 버튼을 클릭하세요. 메뉴가 열리면 '기본값으로 설정(Set as default)' 버튼이 나타납니다.

![기본 페이지 설정 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155031780643/original/NZNWMjjFdKnHMS_hUaArAx9mOwo7QgqdJA.png?1724824702)

'기본값으로 설정(Set as default)' 버튼을 클릭하면 해당 페이지가 기본 페이지로 설정되고, 이전에 기본값으로 설정되어 있던 페이지는 더 이상 기본 선택 페이지가 아니게 됩니다.

기본 페이지 설정 시 알아두실 점들:

- 처음 설정할 때: Ad Manager에 첫 번째 페이스북 페이지를 연결하면 해당 페이지가 자동으로 기본 페이지로 설정됩니다.
- 기본 페이지 삭제 시: 기본값으로 설정된 페이지를 삭제하면, 다음 페이지가 자동으로 기본 페이지가 됩니다.
- 기본 페이지는 오직 하나만: 연결된 모든 페이스북 페이지 중에서 기본 페이지는 단 하나만 설정할 수 있습니다.

**참고**: 캠페인을 만들 때는 언제든 원하는 페이지를 직접 선택할 수 있습니다. 기본 페이지 설정은 캠페인 빌더에서 미리 선택되어 있는 페이지가 있어야 하며, 필요에 따라 언제든 변경할 수 있도록 하는 것이 목적입니다.
