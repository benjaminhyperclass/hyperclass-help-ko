# 오디언스 탭 개요 - Ad Manager 설정

이 글에서는 기존의 커스텀 및 유사 오디언스를 찾는 방법과 새 오디언스 생성, 기존 오디언스의 이름과 설명 편집, 기존 오디언스 삭제 등 수행할 수 있는 작업들에 대해 알아보겠습니다.

## Ad Manager에서 오디언스 목록은 어디에서 확인할 수 있나요?

Ad Manager 홈페이지에서 설정 아이콘(아래 스크린샷에 강조 표시됨)을 클릭하여 Ad Manager 설정으로 이동할 수 있습니다.

![Ad Manager 설정 아이콘](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155035978259/original/CAPm8iRNDuRFQLIovIC7dQ7dZkaF0zdpgQ.jpg?1730753002)

![Ad Manager 설정 페이지](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155035978306/original/KOe2_FWDcRwPjuWtxA9gRF99T2XYRaEo5w.jpg?1730753038)

Ad Manager 설정에는 'Integrations(연동)', 'Pages(페이지)', 'Conversions(전환)', 'Audiences(오디언스)'라는 네 개의 탭이 있으며, 각각 고유한 기능과 정보를 담고 있습니다. 이 글에서는 "Audiences(오디언스)" 탭과 그 구성 요소에 중점을 두겠습니다.

사용자가 Audiences(오디언스) 탭을 클릭하면, Ad Manager 내에서 생성된 모든 오디언스 목록과 Meta 플랫폼에서 이미 생성되어 연결된 광고 계정에서 가져온 오디언스를 볼 수 있습니다.

![오디언스 탭 목록](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155035978330/original/J-O8nEzQ6liiE_ZdV1gCX6qvWKNLqhfKxg.jpg?1730753084)

## 새 오디언스는 어떻게 생성하나요?

새 오디언스(유사 오디언스와 커스텀 오디언스 모두)를 생성하는 방법은 두 가지가 있습니다:

- 캠페인 생성 과정에서 커스텀 및 유사 오디언스를 모두 생성할 수 있으며, 자세한 단계는 각각의 관련 글에서 확인할 수 있습니다: [Ad Manager에서 유사 오디언스 생성하기](../Ad-Campaign-Creation-and-Management/create-lookalike-audiences-in-ad-manager.md)와 [Ad Manager에서 커스텀 오디언스 생성하기](../Ad-Campaign-Creation-and-Management/create-custom-audiences-in-ad-manager.md). 해당 오디언스 생성에 대한 자세한 단계는 위 글들을 참조하시기 바랍니다.
- Ad Manager 설정(Settings) > 오디언스(Audiences) 탭에서 '새 오디언스 생성(Create new audience)' 버튼을 클릭하여 새 오디언스를 생성할 수 있습니다. 이 버튼을 누르면 유사 오디언스(Lookalike)와 커스텀 오디언스(Custom Audience)라는 두 가지 옵션이 나타납니다. 둘 중 하나를 클릭하면 해당 오디언스를 생성하는 절차가 시작되고, 성공적으로 오디언스를 생성하면 아래 오디언스 목록에 표시됩니다. 자세한 오디언스 생성 단계는 1번에서 언급한 도움말 글들을 참조하세요.

![새 오디언스 생성 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155032740430/original/wyrtDpcE8JWcQZx8QvkpsHBdf01ZMtVdJA.png?1726135595)

## 기존 오디언스에 대해 수행할 수 있는 다른 작업은 무엇인가요?

새 오디언스 생성 옵션 외에도, 기존 오디언스에 대해 수행할 수 있는 작업 목록은 다음과 같습니다:

- '오디언스 검색(search for audience)' 박스를 클릭하고 오디언스 이름을 입력하여 목록에 있는 특정 오디언스를 검색할 수 있습니다.
- 검색 박스 옆에는 '전체(All)', '커스텀(Custom)', '유사 오디언스(Lookalike)' 오디언스를 표시하는 추가 선택 박스가 있습니다. 목록에서 각각을 클릭하면 다음과 같이 목록 동작이 변경됩니다:
  - 전체(All) - Ad Manager 내에 있는 모든 유형의 오디언스를 목록에 표시합니다.
  - 커스텀 오디언스(Custom Audience) - Ad Manager 내에 있는 커스텀 오디언스만 목록에 표시합니다.
  - 유사 오디언스(Lookalike Audience) - Ad Manager 내에 있는 유사 오디언스만 목록에 표시합니다.

![오디언스 필터 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155032784474/original/VMTytMoat9ZGxkzFP6Rq9ObWNX3aKRCadw.png?1726163181)

- 각 오디언스와 같은 줄에 있는 더보기(ellipses) 버튼을 클릭하고 편집(Edit)을 클릭하여 기존 오디언스의 이름과 설명을 편집할 수 있습니다.

![오디언스 편집 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155032784630/original/T2QxEyAGsj0NJGvXkeFiF1XIkKgfi2-eSQ.png?1726163467)

4. 대화 상자가 나타나며 선택한 오디언스의 이름과 설명을 편집할 수 있습니다.

![오디언스 편집 대화상자](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155032784692/original/AmpWfxONa0fJrBpzibLKZM1gO2TawRr5fA.png?1726163565)

## 오디언스는 어떻게 삭제하나요?

각 오디언스와 같은 줄에 삭제(Delete) 옵션이 있는 더보기 버튼이 있습니다. 해당 오디언스가 활성 캠페인에 연결되어 있지 않을 때만 삭제 버튼을 클릭하여 모든 초안(draft) 및 일시정지(paused) 캠페인에서 해당 오디언스를 제거할 수 있습니다.

![오디언스 삭제 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155032785180/original/-lM5NaElLyE2SbTvnizthA6h2vAcCgEATg.png?1726164316)

1. 사용자가 하나 이상의 활성 캠페인에 연결된 오디언스를 삭제하려고 할 때, 해당 오디언스가 활성 캠페인의 일부이므로 삭제할 수 없다는 검증 메시지가 나타납니다.

![활성 캠페인 연결 오디언스 삭제 오류](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155032785790/original/Qcbt7HgUZ5a4l5mXrajXzGISTOV6pIuiVA.png?1726165337)

**참고:** 오디언스를 삭제하려면 해당 오디언스와 연결된 모든 활성 캠페인을 먼저 일시정지해야 합니다.

2. 유사 오디언스를 생성하는 데 사용된 커스텀 오디언스를 삭제하려고 하면, 연결된 유사 오디언스를 먼저 제거하지 않으면 삭제할 수 없다는 오류 메시지가 표시됩니다.

![유사 오디언스와 연결된 커스텀 오디언스 삭제 오류](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155032806025/original/f0NXTcOFTImTgPb8HLulRjiSRgJlSwGgQw.png?1726214415)

# 자주 묻는 질문

- **생성할 수 있는 오디언스 수에 제한이 있나요?**
  구체적인 제한은 명시되어 있지 않지만, 효과적인 관리와 활용을 위해 오디언스 목록을 체계적으로 정리하는 것이 좋습니다. 과도한 오디언스는 캠페인 전략을 복잡하게 만들 수 있습니다.

- **오디언스 조건을 얼마나 자주 업데이트할 수 있나요?**
  오디언스 조건은 필요할 때마다 업데이트할 수 있습니다. 하지만 너무 자주 변경하면 오디언스 타겟팅의 성과와 효과에 영향을 줄 수 있습니다.

- **오디언스를 삭제하면 어떻게 되나요?**
  오디언스가 삭제되면 복구할 수 없습니다. 해당 오디언스를 사용하는 모든 캠페인이 영향을 받으므로, 삭제하기 전에 신중하게 확인하는 것이 중요합니다.

- **하나의 광고 캠페인에서 여러 오디언스를 사용할 수 있나요?**
  네, 하나의 광고 캠페인에서 여러 오디언스를 사용할 수 있어 더 타겟팅된 광고 전략이 가능합니다. 하지만 결과를 최적화하기 위해 이러한 오디언스들의 성과를 모니터링해야 합니다.
