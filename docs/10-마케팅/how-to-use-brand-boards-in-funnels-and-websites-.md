# 퍼널과 웹사이트에서 브랜드 보드 사용 개요

브랜드 보드는 회사의 로고, 색상 팔레트, 타이포그래피를 묶은 재사용 가능한 디자인 키트입니다. 브랜드 보드가 생성되면 그 에셋들이 퍼널과 웹사이트 빌더 내에 나타납니다. 이를 통해 원클릭으로 올바른 스타일을 적용할 수 있어서, 모든 웹사이트와 퍼널에서 일관된 브랜딩을 쉽게 유지할 수 있습니다.

---

## 퍼널과 웹사이트에서 브랜드 보드 사용의 주요 장점

브랜드 보드는 페이지 제작을 가속화하고 브랜드 일관성을 보장합니다:

- **브랜드 일관성**: 브랜드 보드 옵션을 사용하면 브랜드에 맞는 사이트를 쉽게 만들거나 업데이트할 수 있습니다

- **시간 절약**: 올바른 색상을 찾거나 HEX 코드를 기억할 필요가 없습니다. 브랜드 보드가 작업 흐름을 간소화합니다

- **즉시 브랜드 팔레트 접근**: 승인된 브랜드 색상과 폰트가 사이트 빌더 내에 표시되어 HEX 코드를 복사하거나 외부 브랜딩 가이드를 참조할 필요가 없습니다

- **중앙화된 관리**: 브랜드 보드를 한 번만 새로고침하면 업데이트된 색상, 폰트, 로고가 하위 계정 내 빌더의 모든 피커에 적용됩니다

## 웹사이트와 퍼널에서 브랜드 보드 사용하기

퍼널이나 웹사이트 빌더에서 사용하려면 미리 생성된 브랜드 보드가 있어야 합니다. 브랜드 보드 생성 방법은 [브랜드 보드 만들기](how-to-create-a-brand-board.md)를 참고하세요.

#### **1단계: 퍼널 또는 웹사이트 빌더로 이동**

- 메인 네비게이션 바에서 사이트(Sites)를 선택합니다
- 상단 네비게이션 리본에서 퍼널(Funnels) 또는 웹사이트(Websites) 탭을 클릭합니다
- 편집할 사이트를 선택하거나 파란색 + 새 퍼널(+ New Funnel) / + 새 웹사이트(+ New Website) 버튼을 클릭하여 새 사이트를 만듭니다

![퍼널과 웹사이트 네비게이션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051653594/original/I2lSEp8s6rNNUuRU6fUc2WCdF3KYe0GTYg.png?1755193970)

#### **2단계: 색상 팔레트 접근하기**

- 스타일을 적용할 요소(Element)를 클릭합니다
- 오른쪽 사이드바에서 색상 견본(Color Swatch)을 클릭하여 색상 피커를 엽니다
- 타이포그래피 색상의 경우 인라인 툴바를 사용하여 색상 피커를 열 수도 있습니다

![색상 피커 접근](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051653880/original/jJituTG-0f-3Y0r-fgrpKvqX0vIsElivoA.png?1755194558)

#### **3단계: 색상 피커 사용하기**

- 색상 피커의 맨 위 행에 브랜드 보드의 색상이 나타납니다
- 브랜드 보드에서 색상을 클릭하여 적용합니다

"기본값(Default)"으로 설정된 브랜드 보드만 색상 피커에 나타납니다. 기본 브랜드 보드 변경 방법은 [기본 브랜드 보드를 변경하는 방법](how-to-change-default-brand-boards-.md)을 참고하세요.

![브랜드 보드 색상 적용](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051653940/original/O-ItumJSa-5wsyqdJ6_0ApazZRpjj3Bjaw.png?1755194693)

## 자주 묻는 질문

**Q: 브랜드 보드를 업데이트하면 기존 퍼널이나 웹사이트의 색상이 자동으로 변경되나요?**
A: 아니요. 브랜드 보드를 업데이트해도 이전에 생성된 퍼널이나 웹사이트 페이지에는 소급 적용되지 않습니다.

**Q: 새 프로젝트의 기본 브랜드 보드는 어떻게 설정하나요?**
A: **마케팅(Marketing) → 브랜드 보드(Brand Boards)**로 이동하여 사용할 보드 옆의 **점 세 개** 메뉴를 클릭하고 **기본값으로 설정(Set as Default)**을 선택합니다. 이렇게 하면 선택한 브랜드 보드가 새 퍼널과 웹사이트에 기본적으로 적용됩니다. [이 주제에 대한 자세한 내용은 여기를 클릭하세요.](how-to-change-default-brand-boards-.md)

**Q: 색상 피커에 브랜드 보드 색상이 표시되지 않는 이유는 무엇인가요?**
A: 브랜드 보드 색상은 브랜드 보드가 활성화된 경우에만 퍼널이나 웹사이트 빌더에 나타납니다. 디자인(Design) 탭으로 이동하여 브랜드 보드가 적용되어 있는지 확인하세요.

**Q: 하나의 하위 계정에서 여러 브랜드 보드를 사용할 수 있나요?**
A: 네. 원하는 만큼 만들 수 있지만, 한 번에 하나만 *기본값*으로 설정할 수 있습니다.

**Q: 브랜드 보드를 삭제하면 어떻게 되나요?**
A: 페이지에서 이미 사용된 스타일이나 색상은 그대로 남아있지만, 보드가 피커에서 사라집니다.
