# FirstPromoter에서 제휴 관리로 전환하는 방법

이 가이드에서는 FirstPromoter에서 제휴 업체 목록을 내보내고 제휴 관리(Affiliate Manager)로 가져오는 단계를 안내해드립니다.

**목차**

- [1단계: FirstPromoter에서 제휴 업체 목록 내보내기](#1단계-firstpromoter에서-제휴-업체-목록-내보내기)
- [2단계: 내보낸 파일 형식 맞추기](#2단계-내보낸-파일-형식-맞추기)
- [3단계: CSV 파일을 제휴 관리로 가져오기](#3단계-csv-파일을-제휴-관리로-가져오기)
- [다음 단계](#다음-단계)

---

### **1단계: FirstPromoter에서 제휴 업체 목록 내보내기**

- **FirstPromoter 대시보드에 로그인**: FirstPromoter 계정의 대시보드로 이동합니다.
- **'Promoters' 섹션으로 이동**: 메뉴에서 'Promoters' 탭을 클릭합니다.
- **'Export' 클릭**: 'Export' 버튼을 찾아서 클릭합니다(빨간색으로 표시됨).

![FirstPromoter Export 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033778116/original/gQZNh6e4aUX1DfXT7XHC8dZE9QOiaGWDpg.jpeg?1727688873)

### **2단계: 내보낸 파일 형식 맞추기**

- **내보낸 파일 다운로드**: 내보내기가 완료되면 CSV 형식으로 파일을 다운로드합니다.
- **파일 형식 맞추기**: CSV 파일이 지원되는 가져오기 형식과 일치하는지 확인합니다.

![CSV 파일 형식 예시](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155033778117/original/eDST5HHCtzIAlFoHaBlpKZXY4HybDqKB4Q.jpeg?1727688873)

### **3단계: CSV 파일을 제휴 관리로 가져오기**

- **제휴 관리 접근**: 제휴 관리(Affiliate Manager) 계정에 로그인합니다.
- **CSV 업로드**: 제휴 업체(Affiliate) 섹션으로 이동하여 "추가(Add)"를 클릭하고 "제휴 업체 업로드(Upload Affiliate)"를 선택합니다. 파일 형식이 샘플 파일([샘플 파일](https://storage.googleapis.com/affiliate-manager-app/static-assets/Affiliate-Sample-Upload-Sheet.csv))과 일치하는지 확인하세요.
- 파일을 준비했으면 업로드하고 "다음(Next)"을 클릭합니다.

### **다음 단계**

제휴 업체가 CRM에 성공적으로 가져와지면 새로운 제휴 캠페인을 만들거나 기존 캠페인을 편집하여 관리를 시작할 수 있습니다. "추가(Add)" 버튼을 사용하여 새 캠페인을 설정하거나 현재 캠페인을 필요에 맞게 수정하세요.
