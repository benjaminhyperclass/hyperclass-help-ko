## 새 연락처 만들기

첫 번째 연락처를 추가하는 것부터 시작해보세요. 고객 네트워크를 구축하는 첫 번째 단계입니다:

- 메인 메뉴에서 **연락처(Contacts)**로 이동합니다.

- **연락처 추가(Add Contact)** 버튼을 클릭합니다.

- 연락처 정보(이름, 이메일, 전화번호 등)를 입력합니다. 더 많은 정보를 입력할수록 개인화된 소통이 쉬워집니다. 필요한 연락처 필드가 보이지 않는다면, 새 커스텀 필드 만들기 체크리스트를 활용하세요.

- **저장(Save)**을 클릭합니다.

![새 연락처 추가하기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046118741/original/_SFy8qWDDQkit27oUmNKBSqs-QeWROSsVg.gif?1746448228)

**알고 계셨나요?** 새 리드가 폼이나 설문을 작성하면 새 연락처가 자동으로 생성됩니다!

## 기존 연락처 편집하기

연락처 정보를 최신 상태로 정확하게 유지하세요. 연락처 편집은 빠르고 간단하여 데이터를 항상 최신으로 관리할 수 있습니다:

- **연락처(Contacts)**로 이동합니다.

- 편집하려는 연락처 행을 클릭하여 **연락처 상세(contact card)**를 엽니다.

![연락처 상세 열기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046207640/original/_S8wQrXl3nonHSKb0Cfu33hS5avjsqv6hg.png?1746554134)

3. 왼쪽에서 필요한 정보를 업데이트합니다. 필요한 연락처 필드가 보이지 않는다면, 새 커스텀 필드 만들기 체크리스트를 활용하세요. **저장(Save)**을 클릭하여 변경사항을 적용합니다.

![연락처 정보 편집하기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046207878/original/B-6fYha__KvBC40PM7r8FrKvLMXYyamySg.png?1746554563)

## 기존 연락처 삭제하기

연락처를 삭제하면 관련된 대화, 노트, 기회, 할 일, 예약, 수동 작업, 커뮤니티 그룹 소유자 정보도 함께 제거됩니다. 또한 해당 연락처에 대한 모든 활성 캠페인과 워크플로우도 중단됩니다. 삭제된 연락처는 2개월 내에 복구할 수 있습니다.

연락처를 삭제하려면 다음 단계를 따르세요:

- 메인 메뉴에서 **연락처(Contacts)**로 이동합니다.

![연락처 메뉴로 이동](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054627003/original/3bDRO698_ItgDeyvBGHb2P5ZpWyNpG0BDg.png?1758836091)

- 삭제하려는 연락처의 이름 왼쪽에 있는 체크박스를 선택합니다. 그다음 삭제(Delete) 버튼을 클릭합니다.

![연락처 선택 및 삭제](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054627011/original/2jkrrgU2OAaiNwjyhhrh42GygH2ub0CsXg.png?1758836133)

- 팝업 창에서 DELETE를 입력하여 작업을 확인합니다. 그다음 삭제(Delete) 버튼을 클릭합니다.

![삭제 확인](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054627033/original/gqOv_ErvMssDK2fvvFfAe_wmaxItEjZVng.png?1758836223)

## 새 커스텀 필드 만들기

커스텀 필드를 사용하면 특정 비즈니스 요구사항에 맞게 연락처 데이터베이스를 맞춤 설정할 수 있습니다. 귀하와 팀에게 가장 중요한 데이터를 쉽게 추적해보세요:

- **설정(Settings) > 커스텀 필드(Custom Fields)**로 이동합니다.

![커스텀 필드 메뉴로 이동](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046123056/original/IOVUY0XtJBkQa31zmiTGHcaHjyc1SYbt4w.gif?1746451754)

2. **필드 추가(Add Field)**를 클릭합니다.

![필드 추가 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046123240/original/DQ6uNjs8OxiCrO_e4-BAQ25Drfj5BknF6A.png?1746451919)

3. 만들고자 하는 필드의 **유형(type)**을 선택합니다(예: 텍스트, 드롭다운, 날짜).

![필드 유형 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046123940/original/L77XuTevZyk5uNwnDmikDqSxHGmklOdG6Q.png?1746452389)

4. 필드 세부사항(이름, 그룹, 플레이스홀더 등)을 완성합니다. 이는 팀이 어떤 정보가 필요한지 정확히 이해할 수 있도록 도와줍니다. **저장(Save)**을 클릭합니다.

![필드 세부사항 입력](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046124319/original/vhSyrtZLdctnrJNFlWfz9vmnqy5V_itGtQ.png?1746452662)
