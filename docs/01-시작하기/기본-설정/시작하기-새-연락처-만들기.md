# 시작하기 — 새 연락처 만들기

첫 번째 연락처(Contact)를 간편하게 추가하고, 바로 고객과의 소통을 시작하세요.

## 새 연락처 만들기

첫 번째 연락처를 추가해봐요. 네트워크 구축은 이 간단한 단계에서 시작돼요.

1. 메인 메뉴에서 **Contacts**(연락처)로 이동하세요.
2. **Add Contact**(연락처 추가) 버튼을 클릭하세요.
3. 연락처 정보(이름, 이메일, 전화번호 등)를 입력하세요. 정보를 많이 입력할수록 나중에 개인화된 소통이 훨씬 쉬워져요. 필요한 입력 필드가 보이지 않으면 아래의 **커스텀 필드 만들기**를 참고하세요.
4. **Save**(저장)를 클릭하세요.

![연락처 추가 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046118741/original/_SFy8qWDDQkit27oUmNKBSqs-QeWROSsVg.gif)

> 💡 **팁:** 폼(Form)이나 설문(Survey)을 새 리드가 작성하면, 새 연락처가 자동으로 생성돼요!

---

## 기존 연락처 수정하기

연락처 정보를 최신 상태로 유지하세요. 수정은 빠르고 간단해요.

1. **Contacts**(연락처)로 이동하세요.
2. 수정하려는 연락처의 행을 클릭해서 **연락처 카드**를 여세요.

![연락처 목록에서 카드 열기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046207640/original/_S8wQrXl3nonHSKb0Cfu33hS5avjsqv6hg.png)

3. 왼쪽에서 필요한 정보를 수정하세요. 필요한 입력 필드가 보이지 않으면 아래의 **커스텀 필드 만들기**를 참고하세요. **Save**(저장)를 클릭하면 변경 사항이 적용돼요.

![연락처 카드 편집 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046207878/original/B-6fYha__KvBC40PM7r8FrKvLMXYyamySg.png)

## 연락처 삭제하기

연락처를 삭제하면 관련된 대화(Conversations), 노트, 기회(Opportunities), 태스크, 예약(Appointments), 수동 작업(Manual Actions), 커뮤니티 그룹 소유자 정보도 함께 삭제돼요. 또한 해당 연락처에 대해 실행 중이던 캠페인과 워크플로우(Workflow)도 중단돼요. 삭제된 연락처는 **2개월 이내에 복구**할 수 있어요.

연락처를 삭제하려면 다음 단계를 따르세요:

1. 메인 메뉴에서 **Contacts**(연락처)로 이동하세요.

![연락처 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054627003/original/3bDRO698_ItgDeyvBGHb2P5ZpWyNpG0BDg.png)

2. 삭제할 연락처의 이름 왼쪽 체크박스를 선택한 다음, **Delete**(삭제) 버튼을 클릭하세요.

![체크박스 선택 후 삭제 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054627011/original/2jkrrgU2OAaiNwjyhhrh42GygH2ub0CsXg.png)

3. 팝업 창에 **DELETE**를 입력해서 삭제를 확인하고, **Delete**(삭제) 버튼을 클릭하세요.

![삭제 확인 팝업](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155054627033/original/gqOv_ErvMssDK2fvvFfAe_wmaxItEjZVng.png)

## 커스텀 필드 만들기

커스텀 필드(Custom Field)를 사용하면 비즈니스에 맞게 연락처 데이터를 맞춤 설정할 수 있어요. 팀에게 가장 중요한 데이터를 쉽게 추적하세요.

1. `Settings(설정) → Custom Fields(커스텀 필드)`로 이동하세요.

![설정 > 커스텀 필드 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046123056/original/IOVUY0XtJBkQa31zmiTGHcaHjyc1SYbt4w.gif)

2. **Add Field**(필드 추가)를 클릭하세요.

![필드 추가 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046123240/original/DQ6uNjs8OxiCrO_e4-BAQ25Drfj5BknF6A.png)

3. 만들고 싶은 필드의 **유형**(type)을 선택하세요. 예: 텍스트(text), 드롭다운(dropdown), 날짜(date) 등.

![필드 유형 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046123940/original/L77XuTevZyk5uNwnDmikDqSxHGmklOdG6Q.png)

4. 필드 상세 정보(이름, 그룹, 플레이스홀더 등)를 입력하세요. 팀원들이 어떤 정보를 입력해야 하는지 정확히 이해할 수 있도록 해주세요. **Save**(저장)를 클릭하세요.

![필드 상세 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046124319/original/vhSyrtZLdctnrJNFlWfz9vmnqy5V_itGtQ.png)
