# 시작하기 — 기존 연락처 가져오기

고객 데이터는 비즈니스의 핵심이에요. 다양한 플랫폼에 흩어져 있는 기존 연락처를 한 곳으로 빠르게 가져와서, 바로 소통을 시작하세요. 모든 연락처가 한곳에 모이면 번거로움은 줄고, 효과적인 마케팅과 더 깊은 고객 관계를 바로 시작할 수 있어요.

## 연락처 가져오기

효과적인 마케팅의 시작은 연락처 가져오기예요. 아래 단계를 따라 간편하게 진행하세요.

1. **Contacts**(연락처)로 이동하세요.
2. **Import**(가져오기) 버튼을 클릭하세요. 아래쪽 화살표 모양 아이콘이에요.

![가져오기 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155058441185/original/tr8zV_KRpz8yFGutf5QmFlmQFML_aGP5TA.png)

3. 가져오기 화면에서 **Contacts**(연락처)를 선택하고 **Next**(다음)를 클릭하세요.

![가져오기 유형 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046215598/original/lUtMFGKtXA-f_dz_aDClWP0xhSZlEMwwGw.png)

4. 파일 형식을 확인하세요.
5. 파일을 업로드하세요 (CSV 파일, 최대 50MB). 기본 연락처 필드가 어떤 형식인지 보려면 **Download Sample File**(샘플 파일 다운로드)를 클릭하세요.

![파일 업로드](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046216140/original/1aVLfm5EKJlqs4Ti13u6lR9h2bGK2PUxrA.png)

6. 파일의 열(column)을 해당하는 연락처 필드에 매핑하세요.
   - 기본 필드에 없는 데이터가 있으면 커스텀 필드를 새로 만들 수 있어요.
   - 또는 **"Don't import data in unmatched columns"**(매칭되지 않은 열은 가져오지 않기)를 선택하세요.

![필드 매핑](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046216235/original/TMntSMbquKeGYMSwCwF5YEXqtOaRDhH5CA.png)

7. 가져오기 상세 정보(이름, 리스트 생성, 동의 설정 등)를 입력하고 **Bulk Import**(일괄 가져오기)를 클릭하세요.

![일괄 가져오기 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046216248/original/C1fc4vamtJlCsNjarrszZW6rQMsO0tBvKg.png)

8. 연락처 가져오기가 완료될 때까지 기다리세요.

> 💡 **더 알아보기:**
> - [CSV 파일로 연락처 가져오기](../../02-연락처/importing-contacts-using-a-csv-file.md)

## 커스텀 필드 만들기

비즈니스에 꼭 맞는 연락처 데이터를 수집하세요. 커스텀 필드(Custom Field)를 만들면 필요한 정보를 정확하게 수집할 수 있어서, 타겟 마케팅과 깊이 있는 분석이 가능해져요.

1. `Settings(설정) → Custom Fields(커스텀 필드)`로 이동하세요.
2. **Add Field**(필드 추가)를 클릭하세요.

![필드 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046216564/original/_KY3LSJEDIHXp2GRjNL4KZNEGo7jnICWYw.png)

3. 필요한 필드 유형을 선택하세요.

![필드 유형 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046216609/original/TWy7U03FDfm8qPkWApA9G0ujmmbWPux32g.png)

4. 필드 상세 정보(이름, 그룹, 플레이스홀더 등)를 입력하세요.

![필드 상세 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046216614/original/5lrHi73tFyW2sHhFdawu9KNMu5hOzKIbTw.png)

5. **Save**(저장)를 클릭하세요.

> 💡 **더 알아보기:** [커스텀 필드 사용법](../../23-레거시-자동화/Logic-&amp;-Fulfillment/how-to-use-custom-fields.md)

## 일괄 작업 (Bulk Actions)

대량의 연락처를 한 번에 관리하세요. 태그 추가, 이메일 발송, 자동화 실행 등 일괄 작업으로 시간을 절약하고 효율을 높일 수 있어요.

1. **Contacts**(연락처)로 이동하세요.

![연락처 목록](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046216682/original/g4P9dPnIE-3D9M_mIn12is7Hnp49C2DWCg.png)

2. 필터를 적용해서 대상을 좁히세요.

![필터 적용](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046216730/original/ErMW3Cyblh2BwxCwkzlcV8cL3a_9JoVSFw.png)

3. 원하는 연락처 옆의 체크박스를 선택하세요.

![체크박스 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046216740/original/nXzOqDX0FttIxf0kUVsRK4FvjXg8O8S0FQ.png)

4. 일괄 작업 버튼(예: 자동화 추가, SMS 보내기, 이메일 보내기)을 클릭하세요.

![일괄 작업 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046216756/original/GyoqnvFpzPOPeKHiLCIAwROySc-WnikCaw.png)

> 💡 **더 알아보기:** [여러 연락처에 태그 일괄 추가하기](../../02-연락처/adding-contact-tags-to-multiple-contacts.md)

## 가져오기에 사용 가능한 기본 연락처 필드

연락처를 가져올 때 사용할 수 있는 기본 필드 목록이에요:

Contact ID, Name, First Name, Last Name, Date of Birth, Contact Owner, Contact Source, Contact Type, Business Name, Phone, Email, Address, State, City, Postal Code, Country, DND, Time Zone, Website, Additional Email, Additional Phones, Notes
