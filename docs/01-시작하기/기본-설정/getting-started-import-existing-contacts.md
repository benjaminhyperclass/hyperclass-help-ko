# 시작하기 - 기존 연락처 가져오기

고객은 사업의 심장과 같습니다. 다양한 플랫폼에 있는 기존 연락처를 하나의 편리한 공간으로 빠르게 가져와서 즉시 소통을 시작하세요. 모든 연락처를 한 곳에 모으면 번거로움이 줄고, 더 효과적인 아웃리치와 더 강한 고객 관계를 바로 구축할 수 있습니다.

## 연락처 가져오는 방법

효과적인 아웃리치는 연락처 가져오기부터 시작됩니다. 다음 간단한 단계를 따라 원활한 가져오기를 진행하세요:

- **Contacts(연락처)** 메뉴로 이동합니다.

- **Import(가져오기)** 버튼을 클릭합니다 (아래쪽 화살표가 박스 안으로 들어가는 아이콘).

![연락처 가져오기 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155058441185/original/tr8zV_KRpz8yFGutf5QmFlmQFML_aGP5TA.png?1763144072)

- Imports-1에서 **Contacts(연락처)**를 선택하고 **Next(다음)**를 클릭합니다.

![연락처 선택 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046215598/original/lUtMFGKtXA-f_dz_aDClWP0xhSZlEMwwGw.png?1746573893)

- 파일 형식을 확인합니다.

- 파일을 업로드합니다 (CSV 파일 최대 50MB). 표준 연락처 필드를 보여주는 샘플 파일을 다운로드하려면 **Download Sample File(샘플 파일 다운로드)**을 클릭하세요.

[표준 연락처 필드 목록 보기](#가져오기-시-표준-연락처-필드)

![파일 업로드 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046216140/original/1aVLfm5EKJlqs4Ti13u6lR9h2bGK2PUxrA.png?1746575591)

- 파일의 열을 해당하는 연락처 필드와 매핑합니다.

표준이 아닌 데이터가 있다면 새로운 커스텀 필드를 생성하세요.

- 또는 "Don't import data in unmatched columns(매칭되지 않는 열의 데이터는 가져오지 않음)"을 선택하세요.

![필드 매핑 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046216235/original/TMntSMbquKeGYMSwCwF5YEXqtOaRDhH5CA.png?1746576028)

- 가져오기 세부 정보를 입력하고 (가져오기 이름 지정, 목록 생성, 동의 설정 등) **Bulk Import(일괄 가져오기)**를 클릭합니다.

![가져오기 설정 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046216248/original/C1fc4vamtJlCsNjarrszZW6rQMsO0tBvKg.png?1746576108)

- 연락처 가져오기가 완료될 때까지 기다립니다.

다음 단계를 준비하셨나요? 더 자세히 알아보기:

- [CSV 파일을 사용한 연락처 가져오기]([importing-contacts-using-a-csv-file](importing-contacts-using-a-csv-file.md))

- [CSV를 통한 연락처 및 기회 가져오기]([importing-contacts-and-opportunities-via-csv](importing-contacts-and-opportunities-via-csv.md))

- [중복 연락처 허용 (연락처 중복 제거 설정)]([allow-duplicate-contacts-contact-deduplication-preferences-](allow-duplicate-contacts-contact-deduplication-preferences-.md))

이제 타겟팅된 참여를 위해 고객을 세분화하기 시작할 수 있습니다!

## 새 커스텀 필드 만드는 방법

고유한 사업 요구 사항에 맞게 연락처 데이터를 개인화하세요. 커스텀 필드를 만들면 정확히 원하는 정보를 수집할 수 있어 정밀한 타겟팅과 더 풍부한 인사이트가 가능합니다.

커스텀 필드를 설정하는 방법:

- **Settings(설정) > Custom Fields(커스텀 필드)**로 이동합니다.

- **Add Field(필드 추가)**를 클릭합니다.

![커스텀 필드 추가 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046216564/original/_KY3LSJEDIHXp2GRjNL4KZNEGo7jnICWYw.png?1746577051)

- 필요한 필드 유형을 선택합니다.

![필드 유형 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046216609/original/TWy7U03FDfm8qPkWApA9G0ujmmbWPux32g.png?1746577171)

- 필드 세부 정보를 입력합니다 (이름, 그룹, 플레이스홀더 등).

![필드 세부 정보 입력](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046216614/original/5lrHi73tFyW2sHhFdawu9KNMu5hOzKIbTw.png?1746577217)

- **Save(저장)**를 클릭합니다.

더 자세히 알아보기:

- [커스텀 필드 사용 방법]([how-to-use-custom-fields](how-to-use-custom-fields.md))

이제 커스텀 필드를 활용해서 캠페인과 메시징을 향상시켜보세요!

## 연락처 일괄 작업

강력한 일괄 작업으로 많은 연락처 그룹을 손쉽게 관리하세요. 태그 추가, 이메일 발송, 자동화 시작 등 무엇을 하든 일괄 작업은 시간을 절약하고 효율성을 높여줍니다.

방법은 다음과 같습니다:

- **Contacts(연락처)** 메뉴로 이동합니다.

![연락처 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046216682/original/g4P9dPnIE-3D9M_mIn12is7Hnp49C2DWCg.png?1746577568)

- 필터를 적용해서 대상을 좁힙니다.

![필터 적용](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046216730/original/ErMW3Cyblh2BwxCwkzlcV8cL3a_9JoVSFw.png?1746577637)

- 선택할 연락처 옆의 체크박스를 선택합니다.

![연락처 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046216740/original/nXzOqDX0FttIxf0kUVsRK4FvjXg8O8S0FQ.png?1746577775)

- 일괄 작업 버튼을 클릭합니다 (예: Add to Automation(자동화에 추가), Send SMS(SMS 발송), Send Email(이메일 발송)).

![일괄 작업 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046216756/original/GyoqnvFpzPOPeKHiLCIAwROySc-WnikCaw.png?1746577843)

더 자세히 알아보기:

- [일괄 작업: 여러 연락처에 연락처 태그 추가하기]([adding-contact-tags-to-multiple-contacts](adding-contact-tags-to-multiple-contacts.md))

## 가져오기 시 표준 연락처 필드

연락처를 가져올 때 사용할 수 있는 표준 연락처 필드 목록입니다:

- Contact ID (연락처 ID)
- Name (이름)
- First Name (이름)
- Last Name (성)
- Date of Birth (생년월일)
- Contact Owner (연락처 담당자)
- Contact Source (연락처 출처)
- Contact Type (연락처 유형)
- Business Name (회사명)
- Phone (전화번호)
- Email (이메일)
- Business Name (회사명)
- Address (주소)
- State (시/도)
- City (시/구)
- Postal Code (우편번호)
- Country (국가)
- DND (수신거부)
- Time Zone (시간대)
- Website (웹사이트)
- Additional Email (추가 이메일)
- Additional Phones (추가 전화번호)
- Notes (메모)
