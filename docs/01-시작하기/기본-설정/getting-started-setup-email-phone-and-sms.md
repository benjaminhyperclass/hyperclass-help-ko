# 이메일 설정

전용 이메일 도메인을 설정하면 더 나은 받은편지함 도달률, 높은 전달률, 강력한 발신자 평판을 확보할 수 있습니다.

### **단계**

- Settings(설정) → Email Services(이메일 서비스)로 이동합니다.

*
![이메일 서비스 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155058341148/original/p1uuASqVD_gUorwClGdjJsDbkvOGbDv4Cg.png?1763065965)
*

- Dedicated Domain and IP(전용 도메인과 IP)를 클릭합니다.

- Add Domain(도메인 추가)을 클릭합니다.

*
![도메인 추가 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155058341567/original/-4I43XNEJPeJp7W6ONjNQwHUrOWzOxkItA.png?1763066983)
*

- 서브도메인(예: lc.mydomain.com)을 입력하고 Add & Verify(추가 및 확인)을 클릭합니다.

- 도메인 등록업체에서 제공하는 화면의 DNS 레코드 지침을 따릅니다.

- 업데이트가 완료되면 Verify Records(레코드 확인)를 클릭합니다.

- 점 3개 메뉴(3-dot menu) → Set Headers(헤더 설정)를 클릭합니다.

*
![헤더 설정 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155058341607/original/EjWmtRa3meoFv9y5e5I1i0Gj2kasFgP_rA.png?1763067103)
*

- From Name(발신자 이름)과 From Email(발신자 이메일)을 입력한 후 Save(저장)를 클릭합니다.

이제 이메일 설정이 완료되어 인증된 발송이 가능합니다.

## 전화 설정

전화번호를 추가하면 Hyperclass 안에서 직접 전화를 걸고 받을 수 있습니다.

### **단계**

- Settings(설정) → Phone System(전화 시스템)으로 이동하여 Add Number(번호 추가) → Add Phone Number(전화번호 추가)를 클릭합니다.

![전화번호 추가 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155067473684/original/zXamnRL6zAN3RZJqdA2I4K1RQlfUTxvrQQ.png?1774260664)

- Country(국가) 또는 Area Code(지역번호)로 번호를 필터링합니다.

- 원하는 번호를 선택합니다.

- Proceed to Buy(구매 진행)를 클릭합니다.

추가된 번호는 Manage Numbers(번호 관리)에 표시됩니다.

## 전화 착신 전환

착신 전환을 설정하면 들어오는 전화가 선택한 번호로 자동으로 전달됩니다.

### **단계**

- Settings(설정) → Phone Numbers(전화번호)로 이동합니다.

- 점 3개 메뉴(3-dot menu) → Edit Configuration(설정 편집)을 클릭합니다.

![전화번호 설정 편집](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155058341810/original/SOrLz4UVwEnUwJrWr5PyaRFZxaKiDyGktA.png?1763067317)

- 전화가 전환될 번호를 입력합니다.

*
![착신 전환 번호 입력](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155058341921/original/DJSa9bnLPYzqTqIT6HAGJTMveo-XgaJhCQ.png?1763067474)
*

- Save(저장)를 클릭합니다.

착신 전환 설정이 활성화되었습니다.

## 개별 연락처에게 연락하기

연락처 프로필에서 SMS, 이메일, 전화를 사용하여 쉽게 개별 연락처에게 연락할 수 있습니다.

### **단계**

- Contacts(연락처)로 이동합니다.

- 연락하고자 하는 연락처를 선택합니다.

- 메시지 작성 탭에서 SMS, Email(이메일), Phone(전화)을 선택합니다.

**화면:** SMS, WhatsApp, Email 탭이 표시된 연락처 상세 화면

## 다수 연락처에게 일괄 연락하기

여러 연락처에게 동시에 메시지를 보내거나 메시지 발송, 목록 업데이트 등의 일괄 작업을 수행할 수 있습니다.

### **단계**

- Contacts(연락처) → Smart Lists(스마트 리스트)로 이동합니다.

- 필터를 적용하여 대상을 선정합니다.

**화면:** Business Name, Company Name, Tag 등의 필터 옵션이 표시된 필터 패널

![필터 옵션 패널](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155058342133/original/FvGqWx-WJtxH6crdO0HJksIJOHxI5ubPSQ.png?1763068050)

- 왼쪽 체크박스를 사용하여 연락처를 선택합니다.

- 상단 도구 모음에서 Email(이메일) 또는 SMS 발송과 같은 일괄 작업을 수행합니다.

**화면:** 연락처 목록 위에 표시된 일괄 작업(Bulk Actions) 도구 모음

![일괄 작업 도구 모음](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155058342260/original/oA3aLJaqmuX_mlhfJgxrX9GJpR4VS-G38g.png?1763068394)

대량 SMS 기능을 사용하면 몇 번의 클릭으로 여러 연락처에게 개인화된 문자 메시지를 보낼 수 있습니다. 프로모션부터 알림까지, 연락처 탭 안에서 대규모 SMS 캠페인을 작성, 예약, 추적할 수 있습니다.

## 대량 SMS란?

대량 SMS는 하나의 SMS 메시지를 여러 연락처에게 동시에 보내는 기능입니다. 마케팅 캠페인, 서비스 알림, 긴급 업데이트에 이상적입니다. 각 연락처에게 개별적으로 메시지를 보낼 필요 없이 일관된 소통을 보장합니다.

## 대량 SMS 발송의 주요 장점

대량 SMS는 개인화와 발송 제어를 유지하면서 커뮤니케이션을 효율화합니다.

- 시간 절약: 하나의 작업으로 수백, 수천 명의 연락처에게 메시지를 보낼 수 있습니다.
- 일관된 소통: 모든 수신자가 동일한 정보를 받도록 보장합니다.
- 개인화된 콘텐츠: 커스텀 필드와 템플릿을 사용하여 맞춤형 메시지를 작성합니다.
- 유연한 발송 옵션: 즉시 발송, 예약 발송, 또는 배치로 나누어 드립 발송을 선택할 수 있습니다.
- 결과 추적: 일괄 작업(Bulk Actions) 대시보드를 통해 발송 및 성과를 모니터링합니다.

## 연락처에서 대량 SMS 보내는 방법

연락처 탭에서 직접 SMS 캠페인을 만들고 발송하는 단계별 방법입니다.

#### 1단계: 연락처 선택

연락처로 이동하여 왼쪽 체크박스를 사용해 메시지 수신자를 선택합니다.

![연락처 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155058342922/original/lDvptq2pRwRDPNvkV6yceTz8XMKCDqYabA.jpeg?1763069378)

#### 2단계: 대량 SMS 창 열기

상단 메뉴 바의 SMS 아이콘을 클릭하여 대량 SMS 설정 창을 엽니다.

![대량 SMS 창 열기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155058342921/original/1ZYiKri7cJEzz8qKVYHgcwPDeKhf1lptdA.jpeg?1763069378)

#### 3단계: 발송 전화번호 선택

From Phone Number(발신 전화번호) 드롭다운에서 발송할 번호를 선택합니다.

빈 칸으로 두면 시스템이 연락처별 마지막 사용 번호 또는 계정 기본 번호를 사용합니다.

#### 4단계: 메시지 작성 또는 템플릿 선택

사용자 정의 메시지를 입력하거나 SMS Templates(SMS 템플릿)에서 저장된 템플릿을 선택합니다.

커스텀 필드를 사용하여 메시지를 개인화하고 필요시 이모지나 첨부파일을 추가할 수 있습니다.

![메시지 작성](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155058342923/original/ihr8RdkQ8Dx_ieTlrQQlVtXcRcoi6SIPJg.jpeg?1763069378)

#### 5단계: 발송 방법 선택

#### 모두 한 번에 발송 (Send All at Once)

선택된 모든 연락처에게 즉시 메시지를 발송합니다.

#### 예약 발송 (Send at Scheduled Time)

특정 날짜와 시간을 설정하여 발송합니다.

#### 드립 모드 발송 (Send in Drip Mode)

발송 속도와 전달률 향상을 위해 작은 배치로 나누어 발송합니다.

드립 모드 설정 항목:

- Pick a Start Date & Time(시작 날짜 및 시간): 발송 시작 시점
- Number of Messages per Batch(배치당 메시지 수): 한 번에 발송할 연락처 수
- Repeat After(반복 간격): 각 배치 간의 시간 간격(분, 시간, 일)
- Send On(발송 요일): 발송할 특정 요일 선택
- Time Window(시간 창): 계정 시간대 기준 허용 발송 시간
- End At(종료 시간): 하루 중 메시지 발송 가능한 최대 시간; 이 시간 이후 메시지는 다음 날 재개

![드립 모드 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155058342920/original/W84znlfIZeBiArRa4ORKsRyyvb0e-ocM_Q.png?1763069378)

#### 6단계: SMS 발송

Send SMS(SMS 발송)를 클릭하여 캠페인을 제출합니다. 작업이 대기열에 추가되고 자동으로 처리됩니다.

![SMS 발송 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155058342918/original/dIqeYOGfnqOGWLpEVA2bzdzfJxIjm3NiFg.png?1763069378)

#### 7단계: 일괄 작업에서 결과 추적

Contacts(연락처) → Bulk Actions(일괄 작업)로 이동하여 캠페인 상태와 발송됨, 전달됨, 실패, 건너뜀 메시지 등의 성과 지표를 확인합니다.

Show Stats(통계 보기)를 클릭하면 통신사 응답과 실패 원인을 포함한 상세 발송 보고서를 볼 수 있습니다.

![일괄 작업 통계](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155058342919/original/DZOA0I8ec-_6dqwR-Exgzi9G7uJTvp6VoA.jpeg?1763069378)

## 자주 묻는 질문

Q: "발신" 전화번호를 선택하지 않으면 어떻게 되나요?

시스템이 해당 연락처에 마지막으로 사용된 번호를 사용하거나 계정의 기본 번호를 사용합니다.

**Q: 예약된 대량 SMS를 취소할 수 있나요?**

발송된 메시지는 취소할 수 없습니다. 예약되었거나 드립 발송 중인 메시지는 즉시 고객지원팀에 문의하세요.

**Q: 드립 모드는 시간대를 어떻게 처리하나요?**

드립 모드는 계정의 로컬 시간대를 사용합니다.

**Q: 수신 거부한 연락처도 메시지를 받나요?**

아니요. Hyperclass는 컴플라이언스 유지를 위해 수신 거부 또는 DND 연락처를 자동으로 제외합니다.

**Q: 실패한 메시지를 추적할 수 있나요?**

예. Bulk Actions(일괄 작업) → Show Stats(통계 보기)에서 상세 오류 보고서를 확인할 수 있습니다.

**Q: 통화와 SMS에 같은 번호를 사용할 수 있나요?**

예, 해당 번호가 두 기능 모두에 활성화되어 있다면 가능합니다.

**Q: 이메일 도메인을 인증해야 하는 이유는 무엇인가요?**

인증된 도메인은 더 나은 받은편지함 도달률을 보장하고 스팸 문제를 방지합니다.

**Q: 메시징 한도를 늘리려면 어떻게 하나요?**

일관되고 규정을 준수하는 발송 활동을 보여준 후 고객지원팀에 증량을 요청하세요.
