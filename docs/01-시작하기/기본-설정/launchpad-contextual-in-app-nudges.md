# 런치패드 상황별 인앱 가이드

Hyperclass의 새로운 런치패드 상황별 인앱 가이드(Contextual In-App Nudges)를 통해 더욱 매끄러운 온보딩을 경험해보세요. 이 혁신적인 기능은 사용자가 막히는 지점에서 정확한 타이밍에 동적 링크와 함께 적시 지원을 제공하여 작업 완료를 가속화하고 무료 체험에서 유료 전환률을 높이는 데 도움을 줍니다.

# **상황별 인앱 가이드란?**

런치패드용 상황별 인앱 가이드는 사용자가 온보딩이나 활성화 과정에서 막힐 때를 감지하도록 설계된 동적 기능입니다. 도움말 아티클, 튜토리얼 영상, 킥오프 상담 예약, 지원 AI와의 채팅 등 유용한 링크가 포함된 타겟팅된 가이드를 자동으로 표시하여 사용자가 중요한 작업을 수월하게 진행할 수 있도록 실시간 지원을 제공합니다.

---

## 상황별 인앱 가이드의 주요 이점

이 기능은 마찰을 줄이고 적절한 순간에 도움을 제공함으로써 사용자 경험을 향상시키도록 설계되었습니다. 주요 이점은 다음과 같습니다:

- 즉각적이고 상황에 맞는 지원을 제공하여 온보딩 마찰을 줄입니다.

- 런치패드에서 더 높은 작업 완료율을 달성하여 무료 체험에서 유료 전환을 개선합니다.

- 사용자가 막힌 부분을 직접적으로 해결하는 동적 링크를 통해 개인화된 지원을 제공합니다.

- 최소한의 방해로 온보딩 과정을 간소화하여 전반적인 사용자 만족도를 향상시킵니다.

---

## 동적 상황별 링크 옵션

동적 상황별 링크 옵션은 여러 지원 리소스를 하나의 매끄러운 가이드로 통합하여 사용자가 워크플로우를 벗어나지 않고도 필요한 정확한 도움을 찾을 수 있도록 보장합니다. 이 접근 방식은 사용자를 관련 지원과 직접 연결하여 셀프 서비스를 최적화합니다.

- 지원 아티클 – 심화 가이드를 위한 자세한 문서를 직접 엽니다.

- 튜토리얼 영상 – 더 직관적인 학습 과정을 위한 시각적 단계별 지침을 제공합니다.

- 킥오프/구현 상담 링크 – 개인화된 구현이나 전략 세션 예약을 간소화합니다.

- 지원 AI와 채팅 – 문제를 신속하게 해결하기 위한 실시간 대화형 지원을 제공합니다.

## **자동 가이드 숨김 기능**

자동 가이드 숨김 기능은 중복된 프롬프트를 제거하여 사용자 경험을 개선합니다. 하위 계정 내 어떤 사용자든 특정 작업을 성공적으로 완료하면, 해당 작업에 대한 모든 상황별 가이드가 해당 하위 계정의 모든 사용자에게 자동으로 숨겨집니다. 이 스마트 기능은 방해 요소를 최소화하고 정말 필요할 때만 가이드가 나타나도록 보장합니다.

---

## 상황별 인앱 가이드 예시

이 기능은 모든 플랜에 자동으로 적용되므로 상황별 인앱 가이드 시작은 간단합니다. Hyperclass 계정에서 이 기능을 경험하고 활용하려면 다음 단계를 따르세요:

- Hyperclass 계정에 로그인하고 런치패드 섹션으로 이동합니다.

- 가이드 투어 시작, 튜토리얼 시청, 플로우 참여 등 온보딩 작업을 시작합니다.

- 작업 중 일시정지하거나, 중단하거나, 머뭇거릴 때 하나 이상의 동적 링크가 포함된 상황별 가이드가 나타납니다.

필요한 지원에 즉시 접근하려면 사용가능한 링크(지원 아티클, 튜토리얼 영상, 킥오프/구현 상담, 지원 AI와 채팅) 중 아무거나 클릭하세요.

## **자주 묻는 질문**

**Q: 런치패드에서 상황별 가이드는 언제 나타나나요?**

가이드는 시스템이 사용자가 막히고 있다고 감지할 때 나타납니다. 예를 들어 투어를 완료하지 않고 나가거나, 진행 중인 프로세스를 중단하거나, 튜토리얼에서 추가 시간을 보낼 때입니다.

**Q: 상황별 가이드에서는 어떤 동적 링크가 제공되나요?**

가이드에는 지원 아티클, 튜토리얼 영상, 킥오프/구현 상담 링크, 지원 AI와의 채팅에 대한 동적 링크가 포함됩니다.

**Q: 자동 가이드 숨김 기능은 어떻게 작동하나요?**

하위 계정의 어떤 사용자든 작업을 성공적으로 완료하면, 해당 작업과 관련된 모든 가이드가 해당 하위 계정의 모든 사용자에게 자동으로 숨겨집니다.

**Q: 이 가이드를 수동으로 활성화해야 하나요?**

수동 설정은 필요하지 않습니다. 가이드는 기본적으로 사용할 수 있지만, 사용자가 런치패드에 접근할 수 있을 때만 나타납니다. 플랜, 하위 계정 또는 역할에 대해 런치패드가 숨겨진 경우 사용자는 런치패드 기반 가이드를 볼 수 없습니다.

**Q: 상황별 가이드는 에이전시와 하위 계정 런치패드 작업 모두에서 사용할 수 있나요?**

네, 이 기능은 에이전시 및 하위 계정 사용자 모두가 사용할 수 있어 다양한 계정 유형에서 일관된 온보딩 경험을 보장합니다.
