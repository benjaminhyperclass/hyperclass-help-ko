# 개인 예약 캘린더 설정

개인 예약 캘린더는 예약 관리에서 가장 좋은 친구입니다. 한번 설정하면 어떻게 이 없이 관리했었나 싶을 것입니다.

Google 캘린더로 예약을 관리하려면 먼저 Hyperclass 개인 예약 캘린더(Personal Booking Calendar)를 만드는 것이 필수입니다. 이 예약 캘린더는 스케줄링 규칙의 기반이 되어 예약 가능 시간, 버퍼 시간, 미팅 시간, 알림 등을 구성할 수 있게 해줍니다. 평소 Google 캘린더를 사용하더라도 Hyperclass에서 예약이 어떻게 접수되고 표시될지 관리하려면 이 캘린더가 필요합니다.

- 캘린더(Calendar) > 캘린더 설정(Calendar Settings)으로 이동하세요.

![캘린더 설정 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155066592799/original/z9nVrbIH5ziNObPoQJWmABjrzbrlyxe0RA.png?1773144800)

- + 새 캘린더(New Calendar)를 클릭하세요.

![새 캘린더 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155066221388/original/i0KgyPsA8ygTbGvKczimClmFQQgzZSvr_w.png?1772653939)

- 캘린더 유형을 선택하세요 (1:1 미팅의 경우 **개인 예약(Personal Booking)**을 선택).

![캘린더 유형 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155066221401/original/iIRSFaAMr2iQg1XPih0fKwe6weLO4QDa6Q.png?1772653976)

- 캘린더 세부사항을 입력하세요 (친근한 이름 지정, 자신을 호스트로 선택, 커스텀 URL 확인, 예약 시간 지정, 예약 가능 시간 설정). 그다음 **확인(Confirm)**을 클릭하세요.

![캘린더 세부 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046581684/original/w_jfRs4qEjTzZY8tjdID9d54ZYKM1U83vw.png?1747182590)

캘린더 설정이 이제 빠른 작업을 위해 섹션별로 그룹화되었습니다: 기본 정보, 직원 & 위치, 예약 가능 시간, 예약 규칙, 고급 설정.

![캘린더 설정 섹션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155066592686/original/d1Y3-9gB28e1z4WOabePsJ4H3epFc9g_lg.png?1773144729)

- 캘린더 **링크** 또는 임베드 코드를 복사하세요.

![캘린더 링크 복사](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155066221479/original/xzDvgh3Mo4DKQoLy_zldlRZSBw2ebsCwAw.png?1772654105)

- 캘린더 링크를 **공유**해서 예약을 받기 시작하세요!

**다음으로**, 브랜딩에 맞게 캘린더 디자인을 커스터마이징해서 더욱 매력적으로 만들 수 있습니다.

## 자신을 팀원으로 추가

혼자 일하는 경우에도 자신을 팀원으로 추가하면 모든 것이 원활하게 작동합니다. 자신의 스케줄링 시스템에 VIP 접근 권한을 주는 것으로 생각하세요.

- 설정(Settings) > 내 직원(My Staff)으로 이동하세요.

- 사용자 추가(Add User)를 클릭하세요.

- 자신을 추가하고 세부사항을 저장하세요.

**다음으로**, 자신이나 추가 팀원에게 특정 권한을 설정해서 워크플로우를 간소화할 수 있습니다.

## 자동화 알림 설정

자동화 알림은 놓친 예약을 줄이고 고객을 더 만족시킵니다. 캘린더가 고객의 어깨를 가볍게 두드려서 곧 만날 예정임을 알려주는 것과 같습니다.

- 자동화(Automation)로 이동하세요.

- 워크플로우 생성(Create Workflow)을 클릭하고 레시피(Recipe)를 선택하세요.

![워크플로우 생성](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046582042/original/X0KXph5Qggv6ls_gtEKqdD6gJd_GTgIOrg.png?1747184480)

- **예약 확인 + 알림(Appointment Confirmation + Reminder)** 레시피를 선택하세요.

![예약 확인 알림 레시피](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046582055/original/Y6Y4q8Sd5uM3TAT6Pr2Ohu-XfrZJhRt7tQ.png?1747184652)

- **확인 이메일(Confirmation Email)** 액션을 편집하세요.

- 미팅 링크(Zoom, Google Meet 등)를 업데이트하고 액션 저장(Save Action)을 클릭하세요.

![미팅 링크 업데이트](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046582067/original/zwK15IKOBl0WlNnqnTP_tlBBcjUYKdk8Yg.png?1747184728)

- 모든 액션을 검토하세요.

- 발행(Publish)을 클릭하세요.

- 저장(Save)을 클릭하세요.

![워크플로우 저장](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046582073/original/oqAx0Yzr78U2Z1i-kUPeDx5B1v1HCCWp4Q.png?1747184767)

**다음으로**, 개인화된 메시지로 알림을 커스터마이징해서 더 강한 고객 관계를 구축하는 것을 고려해보세요.

## 개인 캘린더 연동

예약 시스템을 개인 캘린더와 연결하면 모든 것이 동기화됩니다. 더 이상 중복 예약이나 놓친 미팅 없이 손쉬운 일정 관리가 가능합니다.

- 캘린더(Calendars) > 캘린더 설정(Calendar Settings)으로 이동하세요.

![캘린더 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046582081/original/DosSWHTt2THvSPzLG8neIvdu_7_Y1QY67A.png?1747184874)

- 연결(Connections) 탭을 선택하고 **새로 추가(Add New)**를 클릭해서 개인 캘린더를 연결하세요.

![캘린더 연결](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046582097/original/OL5bIDvosxda9GtMpgvjvYhP-rNFQZT6ww.png?1747184942)

- 그다음 **화상 회의(Video Conferencing)** 탭을 선택하고 **새로 추가(Add New)**를 클릭해서 선호하는 화상회의 도구를 연결하세요.

![화상회의 연결](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046582104/original/KE9SnwYzFVB8Eb0-C2mJqKhvd8hvPHwiEw.png?1747185004)

**다음으로**, 연동이 제대로 동기화되는지 다시 한번 확인해서 원활한 스케줄링을 즐기세요!

## 캘린더 헤더에서 공유

편집 중인 캘린더에서 직접 공유하면 시간이 절약되고 복사-붙여넣기 오류를 방지할 수 있습니다.

**공유 버튼 사용하기**

설정(Settings) → 캘린더(Calendars) → [내 캘린더]를 열고 **우상단 헤더**의 **공유(Share)**를 클릭하면 다음을 할 수 있습니다:

- 예약 링크 복사

- **일회용 링크(One-Time Link)** 생성

- **임베드 코드(Embed Code)** 가져오기

- 방문자가 보는 모습으로 캘린더 미리보기

![캘린더 헤더 공유](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155064116937/original/VW3GXgSdWHyuZ2vVZsSVXQKadZ6ZV-u-Fg.png?1770138871)

## 자주 묻는 질문

**Q: 예약 링크는 어디서 찾을 수 있나요?**
설정(Settings) → 캘린더(Calendars) → [내 캘린더]를 열고 우상단 헤더의 **공유(Share)**를 클릭하세요. 링크 복사(Copy Link)를 클릭하면 됩니다.

**Q: 일회용 링크는 어떻게 생성하나요?**
헤더의 **공유(Share)**를 클릭하고 **일회용 링크(One-Time Link)**를 선택한 다음, 안내에 따라 생성하고 복사하세요.

**Q: 웹사이트용 임베드 코드는 어디에 있나요?**
헤더의 **공유(Share)**를 클릭하고 **임베드 코드(Embed Code)**를 선택해서 HTML 코드를 복사하세요.

**Q: 방문자 관점에서 캘린더를 미리볼 수 있나요?**
네. **공유(Share)**를 클릭하고 **미리보기(Preview)**를 선택하거나 복사한 링크를 시크릿 창에서 여세요.

**Q: "캘린더 문제 해결(Troubleshoot Calendar)"은 무엇을 하나요?**
설정 및 예약 가능 시간 문제를 확인하고 해결 방법을 안내하는 진단을 실행합니다.

**Q: 이 버튼들이 기존 링크를 변경하나요?**
아니요. 빠른 작업을 표시하는 기능이며, 새 일회용 링크를 생성하지 않는 한 기존 예약 링크는 그대로 유지됩니다.
