# 시작하기 - 워드프레스 사이트 이주하기

워드프레스 사이트 이주가 어렵게 느껴지실 필요 없어요. 새 집으로 이사하는 것과 같다고 생각해보세요. 소중한 콘텐츠와 기능들이 모두 매끄럽게 함께 옮겨질 거예요. 단계별로 차근차근 함께 진행해서 아무것도 놓치지 않도록 해보겠습니다!

**목차**

- [LC Migrator - 원클릭 이주](#lc-migrator-원클릭-이주)
- [All-In-One Migration 플러그인](#all-in-one-migration-플러그인)

## **LC Migrator - 원클릭 이주**

이전에는 워드프레스 사이트를 이주하려면 All-in-One Migrator 같은 서드파티 플러그인을 설치하고, 백업을 만들고, 거대한 파일들을 다운로드한 후 다시 HighLevel에 업로드해야 했습니다. 큰 사이트의 경우 몇 시간이 걸릴 수도 있어서 정말 지치는 일이었죠! 😴

새로운 LC Migrator 플러그인으로 이주 과정을 완전히 간소화했습니다. 로그인하고, 사이트를 선택하고, 몇 번 클릭하면 끝! 몇 시간이 걸리던 작업이 이제는 몇 분이면 완료됩니다!

**주요 기능**

- **원클릭 이주:** LeadConnector 플러그인에서 바로 이주를 시작할 수 있습니다.

- **다운타임 없음:** 백그라운드에서 처리하는 동안 기존 사이트는 계속 운영됩니다.

- **명확한 진행 상황 업데이트:** 백업부터 확인까지 모든 단계를 실시간으로 확인할 수 있습니다.

### **1단계: 사이트 설정으로 이동**

먼저 **Sites(사이트) > WordPress**를 클릭하세요. 기존 사이트를 위한 새로운 공간을 만들기 위한 첫 번째 단계입니다.

### **2단계: WordPress 활성화**

Activate WordPress(WordPress 활성화)를 찾아서 클릭하세요. 새 집의 전등을 켜는 것처럼, 여러분의 도착을 준비하는 과정입니다.

![WordPress 활성화](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155047320508/original/7CrG8ZYicNN2JMiyRGD0fqbkznnGAsAsug.png?1748379551)

### 3단계: 사이트 설정

WordPress 사이트 카드를 클릭하고 From blank(빈 페이지부터)를 선택한 후 템플릿을 선택하세요. 다시 From blank를 선택하고 Create(생성)을 클릭한 후, 사이트 세부 정보를 입력하고 Create WordPress site(WordPress 사이트 생성)를 클릭하세요. 간단하죠?

### 4단계: 기존 사이트 가져오기

WordPress 사이트 > Info(정보) > **Import Your Existing Website(기존 웹사이트 가져오기)**에서 One-click migration using **LeadConnector Migrator Plugin**(LeadConnector Migrator 플러그인을 사용한 원클릭 이주)을 선택하세요. zip 파일이 다운로드되었는지 확인하고, 이제 마법이 시작됩니다!

![기존 웹사이트 가져오기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155047323207/original/OAbjjgHV7WN2oHIP6U8kAshgda544LUGTg.png?1748391487)

### **5단계: 플러그인 추가**

기존 워드프레스 Admin(관리자) > Plugins(플러그인) > Add Plugin(플러그인 추가)로 가서 zip 파일을 추가하세요.

![플러그인 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155047323227/original/SSikmONy00shcgsjzfl_5tmdZs6U4nSYqg.png?1748391610)

### 6단계: 계정 연결

워드프레스 대시보드에서 **LC Migrator**를 클릭하세요. LeadConnector/HighLevel 계정으로 로그인하세요.

![계정 연결](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155047323516/original/A066gfBS1-UdNfuLEUSZpTWnw_HhlG5BoQ.png?1748393257)

### 7단계: 웹사이트 선택

제공되는 드롭다운 목록에서 이주하려는 웹사이트를 선택하세요.

![웹사이트 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155047323526/original/-1Fih9nLBXb3KwwxsgpgwZ5pE78f-Jk5AQ.png?1748393280)

### 8단계: 이주 시작

**Start Migration(이주 시작)** 버튼을 클릭하세요. 명확한 진행 상황 업데이트를 확인할 수 있습니다. 백그라운드에서 이주가 진행되는 동안 편안히 기다리세요.

![이주 시작](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155047323554/original/2Jv_Yykqul43iRWf0ZDZQpJiWaILMYUlIQ.png?1748393299)

### 9단계: 이주 완료

이주가 완료되면 **View Your New Website(새 웹사이트 보기)**를 클릭해서 새로 이주된 사이트를 확인하세요. 선택적으로 경험을 평가해 주시면 서비스 개선에 도움이 됩니다!

![이주 완료](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155047323567/original/E8m5b7MsJoo0zliTlTnkTvdzYxUVa1wCFg.png?1748393315)

## All-In-One Migration 플러그인

### 1단계: 사이트 설정으로 이동

먼저 **Sites(사이트) > WordPress**를 클릭하세요. 기존 사이트를 위한 새로운 공간을 만들기 위한 첫 번째 단계입니다.

### **2단계: WordPress 활성화**

Activate WordPress(WordPress 활성화)를 찾아서 클릭하세요. 새 집의 전등을 켜는 것처럼, 여러분의 도착을 준비하는 과정입니다.

![WordPress 활성화](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155047320508/original/7CrG8ZYicNN2JMiyRGD0fqbkznnGAsAsug.png?1748379551)

### 3단계: 사이트 설정

WordPress 사이트 카드를 클릭하고 From blank(빈 페이지부터)를 선택한 후 템플릿을 선택하세요. 다시 From blank를 선택하고 Create(생성)을 클릭한 후, 사이트 세부 정보를 입력하고 Create WordPress site(WordPress 사이트 생성)를 클릭하세요. 간단하죠?

### **4단계: 기존 사이트 가져오기**

Import Your Existing Website(기존 웹사이트 가져오기)에서 All-in-One WP Migration 플러그인을 선택하세요.

![기존 사이트 가져오기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155047323167/original/ZcZNlK6uYAefSTTQm9E8YzuynQqVEpD47Q.png?1748391155)

### **5단계: 플러그인 받기**

Get Plugin(플러그인 받기)을 클릭한 후 Download(다운로드)를 클릭하세요. 또는 이미 워드프레스 대시보드에 있다면 Plugins(플러그인)으로 이동해서 "all in one wp migration"을 검색하고 선택하세요. 어느 쪽이든 편한 방법으로 하시면 됩니다!

### **6단계: 사이트 내보내기**

플러그인 내에서 Export Site(사이트 내보내기)를 클릭한 후 Export to(내보내기 대상)를 선택하고 File(파일)을 선택하세요. 이제 해당 파일을 다운로드해서 안전하게 보관하세요. 웹사이트의 여행 가방을 싸서 준비하는 것이라고 생각하시면 됩니다!

### **7단계: HighLevel에 업로드**

HighLevel WordPress 사이트 대시보드에서 Click to upload or drag and drop(클릭해서 업로드하거나 드래그 앤 드롭) 버튼을 클릭하세요. 업로드가 완료될 때까지 편안히 기다리시면 됩니다.

### **8단계: 이주 요청**

거의 다 끝났습니다! Request for Migration(이주 요청)을 클릭하세요. 이주 과정이 완료되는 동안 커피 한 잔 드시며 기다리세요. 무거운 작업은 저희가 대신 처리해드릴게요.

## HighLevel 하위 계정 간 WordPress 사이트 이동 (복제)

복제는 원본 사이트를 오프라인으로 만들지 않고 기존 HighLevel WordPress 사이트를 다른 하위 계정으로 복사하는 기능입니다. 한 하위 계정에서 다른 하위 계정으로 사이트를 "내보내는" 가장 빠르고 안정적인 방법입니다.

단계:

- 에이전시 뷰에서 Sites(사이트) → WordPress로 이동하세요.

- Clone(복제)을 선택하세요(또는 원본 사이트를 열고 Clone을 선택하세요).

![사이트 복제](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155061935643/original/kc2WPBJUR-soXxKZOmOl_U1lKGSRDu58Sw.png?1767628547)

- Destination Sub-Account(대상 하위 계정)를 선택하세요.

- Source Site(원본 사이트)를 선택하고(아직 선택되지 않은 경우) 복제본에 Name(이름)을 지정하세요.

![대상 하위 계정 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155061935751/original/-zqw5PCHVaZhJA9gFb3U7ZtCdX-XZURUyg.png?1767628615)

- Clone(복제)을 클릭해서 복제를 시작하세요.

- 상태가 Completed(완료)로 표시될 때까지 진행 상황을 모니터링하세요.
