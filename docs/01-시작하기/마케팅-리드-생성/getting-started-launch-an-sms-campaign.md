## **대량 SMS 캠페인 실행하기**

- Contacts(연락처) 메뉴로 이동
- 연락처 필터를 사용해서 체크박스로 선택 (또는 전체 선택)
- Send SMS(SMS 보내기) 버튼 클릭 (말풍선 아이콘)
- Ok, proceed(확인, 계속) 클릭
- From Phone Number(발신 전화번호) 선택
- 메시지 본문 작성 (메시지 스니펫, 텍스트, 커스텀 값 등)
- Drip Mode(드립 모드) 선택
- 드립 설정 입력 (배치 크기, 날짜 등)
- Send SMS(SMS 보내기) 클릭

## **SMS 템플릿 만들기**

- Marketing(마케팅) > Snippets(스니펫) 메뉴로 이동
- Add Snippet(스니펫 추가) > Add Text Snippet(텍스트 스니펫 추가) 클릭
- 스니펫 세부 정보 입력 (이름, 메시지 등)
- Save(저장) 클릭

**자세히 알아보기**

## **SMS 캠페인 확인하거나 편집하기**

- Contacts(연락처) > Smart Lists(스마트 리스트) > Bulk Actions(일괄 작업) 메뉴로 이동
- 캠페인 상태 확인
- 3점 액션 메뉴를 사용해서 편집, 일시정지, 또는 취소
---

## **대량 SMS 워크플로우 실행하기**

- Automation(자동화) 메뉴로 이동
- Create Workflow(워크플로우 만들기) > Create From Scratch(처음부터 만들기) 클릭
- Form Submitted(폼 제출) 트리거 추가
- Send SMS(SMS 보내기) 액션 추가
- Publish(발행) 클릭
- Save(저장) 클릭
