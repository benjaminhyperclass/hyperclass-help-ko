# Gmail 또는 Outlook 연동

이메일 관리를 간소화할 준비가 되셨나요? 인박스를 빠르게 연동하는 방법은 다음과 같습니다:

- `Settings(설정) → My Profile(내 프로필) → Email (2-way sync)`로 이동합니다.
- Gmail 또는 Outlook을 선택한 후 Connect(연결)를 클릭합니다.

![Gmail 또는 Outlook 연동 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046886503/original/wlIzt4ohhqao3OMU4ruXMtXaEn8TD4dRXg.png?1747694876)

- 권한 승인을 위한 안내에 따라 진행하세요. 안전하고 보안이 보장됩니다!

![권한 승인 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046886508/original/QeM2ih-tIpDggV2SEmCbDnF2TXxV7IVVlA.png?1747694925)

## 연동된 이메일 사용하기

인박스를 연동했으므로 이제 Hyperclass에서 직접 이메일을 관리할 수 있습니다. 이메일 보내기를 시작하는 방법은 다음과 같이 간단합니다:

- Conversations(대화)로 이동합니다.
- Create New Message(새 메시지 작성) 버튼(종이 위의 연필 아이콘)을 클릭합니다.
- Direct Message(직접 메시지)를 선택합니다.

![새 메시지 작성 옵션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046886699/original/2RmZnJEROfqY_iXzlGMbjIQTtVQXtu-zaQ.png?1747695680)

- 연락처를 선택하고 Continue(계속)를 클릭합니다.

![연락처 선택 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046886724/original/skyZC4rN3t9yD9qwALmBqjA6UVIh0R5GcQ.png?1747695777)

- 연동된 이메일 주소가 "From(발신자)" 필드에 자동으로 채워집니다.

![이메일 작성 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046886742/original/LACGHPQyZi4BUtuW-86xpEQdzqOK_Y6xtw.png?1747695895)

이메일은 양방향으로 완벽하게 동기화됩니다. Hyperclass에서 개인 인박스로, 그리고 다시 돌아오는 방식으로 동작하므로 항상 동기화 상태를 유지할 수 있습니다.

이제 자동 응답과 워크플로우를 설정하여 고객 커뮤니케이션을 더욱 쉽게 만드는 방법을 살펴보세요!
