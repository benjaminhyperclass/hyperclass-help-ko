# 대시보드 템플릿 만드는 방법

대시보드 템플릿을 사용하면 여러 하위 계정에 동일한 대시보드를 쉽게 복제할 수 있어 시간과 노력을 절약할 수 있습니다. 이 가이드에서는 대시보드 템플릿을 만들고 이를 활용하여 새로운 대시보드를 생성하는 단계별 방법을 안내합니다. 간단한 단계로 대시보드를 효율적으로 관리하고 재사용하는 방법을 알아보세요.

## 대시보드 템플릿이란?

대시보드 템플릿은 여러 하위 계정에서 동일한 대시보드를 복제하기 위해 설계된 재사용 가능한 프레임워크입니다. 기존 대시보드를 템플릿으로 저장하면 계정 간 일관성을 유지하면서도 설정 시간을 크게 단축할 수 있습니다. 템플릿을 사용하면 리포팅을 표준화하고 모든 계정이 필수 지표와 시각화 자료에 접근할 수 있도록 보장할 수 있습니다.

## 대시보드 템플릿을 사용해야 하는 이유

대시보드 템플릿을 사용하면 여러 계정의 대시보드를 관리할 때 발생하는 일반적인 문제를 해결할 수 있습니다:

• **시간 절약:** 각 하위 계정마다 대시보드를 수동으로 다시 만들 필요가 없습니다.
• **일관성:** 계정 간 데이터 표현의 통일성을 보장합니다.
• **확장성:** 여러 고객을 관리하는 에이전시의 프로세스를 간소화합니다.
• **효율성:** 대시보드 템플릿을 중앙에서 관리하여 업데이트와 수정을 간편하게 만듭니다.

## 대시보드 템플릿은 어디서 찾을 수 있나요?

대시보드 템플릿에 접근하려면:

**1. 소프트웨어에서 Agency View(에이전시 뷰)로 전환합니다.**

![Agency View로 전환](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036893465/original/zC-uteTZSrL6caX1j_IF7fksDNArS2Xgvg.jpeg?1732105688)

**2. Template Library(템플릿 라이브러리) 섹션으로 이동하여 대시보드 템플릿을 찾아보고, 생성하거나 관리할 수 있습니다.**

![템플릿 라이브러리 위치](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036893664/original/aSh44oIlB68E80eX5VpAVoPa09NR1gV55w.jpeg?1732105785)

## 대시보드 템플릿 만드는 방법

다음 단계를 따라 대시보드 템플릿을 생성하세요:

**1단계: Agency View(에이전시 뷰)로 전환**

소프트웨어가 Agency View 모드인지 확인하세요.

**2단계: Template Library(템플릿 라이브러리)로 이동**

Agency View에서 네비게이션 메뉴의 Template Library를 찾아 클릭하세요.

**3단계: 새 템플릿 만들기**

Create a Template(템플릿 만들기)을 클릭하여 대시보드 템플릿 생성 프로세스를 시작하세요.

![템플릿 만들기 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036897041/original/B7pJ-TN9r0WNmxSZ028tB5lcK_DdlSDW0w.png?1732107773)

**4단계: 소스 하위 계정 선택**

복제하려는 소스 대시보드가 있는 하위 계정을 선택하세요.

![소스 하위 계정 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036897716/original/9PRJ6YnqGpdzWyLw9FdVSh1pPj-3QtZsfA.png?1732108216)

**5단계: 제품 유형 선택**

드롭다운 메뉴에서 Dashboards(대시보드)를 제품 유형으로 선택하세요.

![제품 유형 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036897834/original/pUuTYcRGcrXL1QBmLjMMzfqC6LJ2KC5KFg.png?1732108261)

**6단계: 대시보드 선택**

템플릿으로 만들고 싶은 대시보드를 선택한 다음 **"Continue(계속)"**를 클릭하세요.

![대시보드 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036897857/original/FTNHAM7f2ygt_IufNCIuxSa5cnx9mtHbiQ.png?1732108278)

**7단계: 세부 정보 추가**

다음과 같은 추가 정보를 입력하세요:

• **정리를 위한 관련 카테고리**

![카테고리 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036899452/original/PlF760Xvbgz9C4vcgLis6EExy83K77P5sg.png?1732108902)

• **검색을 쉽게 하기 위한 태그**

![태그 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036899471/original/JVRVbn1Oko7qh4nJ1d8VTQS90_53VCQbtA.png?1732108917)

• **식별을 위한 설명과 커버 이미지**

![설명과 커버 이미지 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036899485/original/b3lh7NEebELBRlHMDriPLVirt42ZK8hCHA.png?1732108931)

**8단계: 템플릿 발행**

설정을 검토한 후 Publish Template(템플릿 발행)을 클릭하세요. 새로운 템플릿이 템플릿 목록에 표시됩니다.

![템플릿 발행](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036899821/original/hLNDvYlWcHjYW9MOcYx6Il8QS-k-TPtGQw.png?1732109092)

**빠른 팁:** 대시보드를 템플릿으로 빠르게 업로드하려면 대시보드 옆의 점 세 개를 클릭하고 Upload as Template(템플릿으로 업로드)을 선택하세요.

![빠른 템플릿 업로드](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036900022/original/1hykCR5ZeYgTfSchJOLQCkR3vZFMsvgeXA.png?1732109241)

## 템플릿으로 새 대시보드 만드는 방법

기존 템플릿으로 새 대시보드를 만들려면 다음 단계를 따르세요:

**1단계: 대시보드 페이지로 이동**

계정의 Dashboard(대시보드) 페이지로 이동하세요.

![대시보드 페이지](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036911788/original/wuDnSQSmSOf1SSmhL-74-OqBwXimugW1qg.jpeg?1732116459)

**2단계: Add Dashboard(대시보드 추가) 클릭**

파란색 대시보드 아이콘의 드롭다운을 클릭하고 Add Dashboard를 선택하세요.

![대시보드 추가 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036911822/original/KYi6Tj02T44UotbPFUOHlfuTZYYZIZVhuw.jpeg?1732116485)

**3단계: From Template Library(템플릿 라이브러리에서) 선택**

제공된 옵션에서 From Template Library를 선택하고 하단의 "Select(선택)" 탭을 클릭하세요.

![템플릿 라이브러리에서 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036911837/original/BSvIJyNxIXChq5MasxFtT8g-TMMD-TXrmQ.png?1732116513)

**4단계: 템플릿 선택**

라이브러리를 둘러보고 사용하고 싶은 템플릿을 선택하세요. 다음 페이지에서 오른쪽 상단의 **"Continue(계속)"** 버튼을 클릭하세요.

![템플릿 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036911959/original/7rjOdESaBxQOMwGvnMeWyuZpkuFy3hXGiw.jpeg?1732116557)

![계속 버튼 클릭](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036934283/original/kPaqe0LuAhFx4p8dWi9ql77xuKSFrzKlxw.jpeg?1732164073)

**5단계: 완료!**

템플릿이 대시보드에 추가될 때까지 잠시 기다리세요. 완료되면 "edit(편집)" 옵션으로 이동하여 대시보드 레이아웃을 사용자 정의할 수 있습니다.

![대시보드 완성](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036934046/original/rUghA-9aglhaJkKB2lymqKwFpqiZWGvNRw.jpeg?1732163012)
