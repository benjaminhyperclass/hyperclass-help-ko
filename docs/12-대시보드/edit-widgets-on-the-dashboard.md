# 대시보드 위젯 편집하기

"편집 모드"로 전환한 다음, 수정하고 싶은 위젯의 점 세 개 "..." 메뉴를 선택하세요.

**목차**

- [1단계: 편집 모드로 전환하기](#1단계-편집-모드로-전환하기)
- [2단계: 편집하려는 위젯 선택하기](#2단계-편집하려는-위젯-선택하기)
- [3단계: 원하는 대로 커스터마이징하기](#3단계-원하는-대로-커스터마이징하기)

---

### **1단계: 편집 모드로 전환하기**

- 대시보드에서 "EDIT" 버튼을 클릭하세요.

![편집 모드로 전환하기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155010311482/original/zMFVquYS38cwkN7WsuhIqbhWiI5VjUnO1Q.jpeg?1697544206)

### **2단계: 편집하려는 위젯 선택하기**

- 위젯을 편집하려면, 편집하고 싶은 위젯의 오른쪽 상단에 있는 점 세 개 메뉴를 찾아 클릭하세요. 나타나는 드롭다운 메뉴에서 "Edit"을 선택하세요.

![위젯 편집 메뉴 선택하기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155010311481/original/8RsMtYM5XcgaamCxbXXr5Ijy2k3BVMOV6A.jpeg?1697544206)

- 오른쪽에 편집 메뉴가 나타납니다.

### **3단계: 원하는 대로 커스터마이징하기**

- 편집 메뉴에서 원하는 커스터마이징과 변경사항을 적용한 다음, "Save"를 클릭하고 "Save Changes"를 클릭하세요.

![위젯 편집 설정 적용하기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155010311480/original/y_Nm2DAhnuTvXnxuxklipZLTC9mwSdGZ3w.jpeg?1697544206)

**참고:** 일반 위젯은 편집할 수 없습니다.

---

## 참고사항

커스터마이징 가능한 대시보드를 사용하려면 어떤 요금제가 필요한지 궁금하신가요?

현재 $497 요금제 이상에서만 독점적으로 제공됩니다. $97 또는 $297 요금제를 사용 중이시라면, 이 기능을 잠금 해제하기 위해 업그레이드하시기를 권장합니다.

$497+ 요금제의 커스텀 대시보드는 **에이전시 관리자**와 **에이전시 사용자**가 접근할 수 있습니다.

대시보드 권한 관리에 대한 자세한 내용은 여기를 참고하세요
