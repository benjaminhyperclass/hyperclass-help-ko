# 하위 계정 대시보드란?

대시보드는 하위 계정의 상태를 실시간으로 보여주는 고도로 사용자 정의 가능한 보고서입니다. 먼저 페이지의 몇 가지 기본 상태를 알아보겠습니다.

## 대시보드 모드와 상태란?

### 대시보드 상태:

- 위젯 있음(1개 이상) - 대시보드의 일반적인 상태로, 여러 위젯과 요소를 포함합니다.

![위젯이 있는 대시보드 상태](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155043962435/original/42rt1_NUSVoEgFDUC7qMwwUvaj0gCkKGvw.png?1742938777)

- 위젯 없음(0개) - 대시보드에 위젯이 없을 때는 빈 화면 대신 다양한 빠른 시작 옵션을 보여줍니다. 위젯을 추가/생성해서 시작하세요.

![위젯이 없는 대시보드 상태](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155043962426/original/9Mrfu22FVBOevZWDfvO2EFTXXZiO55j-YA.jpeg?1742938753)

### 대시보드 모드:

- 보기 모드 - 필터가 있는 위젯 목록 형태입니다. 날짜와 필터를 변경하여 하위 계정의 활동을 확인하는 대시보드의 일반적인 상태입니다.

![보기 모드 대시보드](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155043962462/original/0n2XUYlt926yHzEMkK_U9aRMpO0TPFEGDQ.png?1742938880)

- 편집 모드 - 편집 옵션이 있는 위젯 목록 형태입니다. 편집 모드로 임시 전환하면 위젯을 드래그 앤 드롭으로 재배치하고, 위젯과 요소를 추가/삭제하거나, 테마나 제목을 변경할 수 있습니다.

![편집 모드 대시보드](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155043962477/original/ZXzWuAE06vDPAqSuRzOMFO4x_RXh_xeuxQ.png?1742938954)

## 대시보드 구성 요소 개요

- 대시보드 선택 메뉴 - 패널 아이콘과 드롭다운 화살표가 있는 파란색 버튼입니다. 클릭하면 대시보드 선택 메뉴가 펼쳐집니다.

- 날짜 선택기 - 두 개의 날짜 또는 날짜 범위가 있는 캘린더입니다. 클릭하여 다른 날짜 범위를 선택할 수 있습니다.

- 편집 버튼 - 연필 아이콘입니다. 클릭하면 보기 모드에서 편집 모드로 전환됩니다.

- 액션 메뉴 - 세로 점 3개 아이콘입니다. 클릭하면 액션 메뉴가 열립니다.

- 빠른 필터 버튼 - 파란색 연필 아이콘과 "빠른 필터" 텍스트입니다. 클릭하면 빠른 필터 사이드 패널이 열립니다.

- 필터 알약 - 필터 이름과 드롭다운 화살표가 있는 흰색 알약 형태입니다. 클릭하면 필터 조정 모달이 열립니다.

- 위젯 - 대시보드에 있는 위젯 목록입니다.

- 빠른 시작 옵션 - 대시보드에 위젯이 없을 때 표시되는 것입니다.

### 대시보드 선택(드롭다운 메뉴)

- 검색 필드 - "대시보드 검색"입니다. 입력하면 대시보드가 필터링됩니다.

- 대시보드 추가 버튼 - 파란색 "+ 대시보드 추가" 버튼입니다. 클릭하면 대시보드 만들기 페이지가 열립니다.

- 내 대시보드 목록 - "내 대시보드" 제목과 함께 내가 만든 대시보드 목록입니다.

- 공유받은 대시보드 버튼 - "공유받은 대시보드" 제목과 함께 다른 사람이 공유한 대시보드 목록입니다.

- 대시보드 열기 버튼 - 아이콘과 대시보드 이름이 있는 버튼입니다. 클릭하면 보기 모드에서 대시보드가 열립니다.

- 대시보드 고정 버튼 - 압정 아이콘입니다. 클릭하면 대시보드를 목록 맨 위에 고정하거나 해제할 수 있습니다(일반적으로는 알파벳순으로 정렬).

![대시보드 선택 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155043696399/original/g6ikXcxk5KcX69vrk47Q-r0YlC2KGp_lmw.png?1742501210)

## 대시보드 추가/생성

- 뒤로 버튼 - "< 뒤로" 버튼입니다. 클릭하면 대시보드로 돌아갑니다.

- 빈 대시보드 만들기 버튼 - "빈 대시보드 만들기. 깨끗한 상태에서 새로 시작합니다. 선택"이 있는 카드입니다. 클릭하면 새로운 빈 대시보드 만들기가 시작됩니다.

- 템플릿 라이브러리에서 버튼 - "템플릿 라이브러리에서. 라이브러리의 기성 템플릿이나 내 템플릿을 사용합니다. 선택"이 있는 카드입니다. 클릭하면 템플릿으로 새 대시보드 만들기가 시작됩니다.

- 기존 대시보드 복제 버튼 - "기존 대시보드 복제. 기존 대시보드를 복제합니다. 선택"이 있는 카드입니다. 클릭하면 기존 대시보드 복제가 시작됩니다.

![대시보드 추가/생성](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155043958511/original/3UyApDwKSBdtKqDhPx71RW2EPMfVINC9qw.png?1742930300)

### 새 대시보드 만들기

- 대시보드 이름 필드 - "(예) 기회 개요"라는 예시가 있는 텍스트 입력 필드입니다. 새 대시보드 이름을 입력하세요.

- 비공개 대시보드 토글 - "이 대시보드를 나만 볼 수 있도록 설정합니다. 다른 팀원들에게는 표시되지 않습니다"라는 라벨이 있는 파란색/회색 토글입니다. 토글을 켜면(파란색 오른쪽) 대시보드가 비공개가 되고, 끄면(회색 왼쪽) 접근 설정을 표시합니다.

- 에이전시 사용자 접근 드롭다운 - "에이전시 사용자"라는 이름의 드롭다운입니다. 에이전시 사용자 접근 옵션을 선택하세요.

- 계정 관리자 접근 드롭다운 - "계정 관리자"라는 이름의 드롭다운입니다. 계정 관리자 접근 옵션을 선택하세요.

- 계정 사용자 접근 드롭다운 - "계정 사용자"라는 이름의 드롭다운입니다. 계정 사용자 접근 옵션을 선택하세요.

접근 옵션:
- 전체 옵션 - "전체"로 표시됩니다. 전체 접근을 위해 선택하세요.
- 편집 옵션 - "편집"으로 표시됩니다. 편집만 가능하도록 선택하세요.
- 보기 옵션 - "보기"로 표시됩니다. 보기만 가능하도록 선택하세요.
- 접근 없음 옵션 - "접근 없음"으로 표시됩니다. 접근을 허용하지 않으려면 선택하세요.

- 취소 버튼 - "취소"라고 표시된 흰색 버튼입니다. 클릭하면 새 대시보드 모달이 닫히고 설정이 취소됩니다.

- 확인 버튼 - "확인"이라고 표시된 파란색 버튼입니다. 클릭하면 새 대시보드가 생성되고 열립니다.

![새 대시보드 만들기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155043958849/original/eqKzv7TGekrNGrzWc35VVE5NuLTTTLCyOg.png?1742930902)

### 템플릿 라이브러리에서 대시보드 만들기

- 필터 버튼 - "필터 ="로 표시됩니다. 클릭하면 필터 사이드바를 보이거나 숨깁니다.

- 필터 지우기 버튼 - "X 필터 지우기"로 표시됩니다. 클릭하면 모든 필터가 초기화됩니다.

- 검색 필드 - "템플릿 검색"으로 표시됩니다. 키워드를 입력하여 템플릿을 필터링합니다.

- 정렬 드롭다운 - 흰색 드롭다운입니다. 템플릿을 특정 순서로 정렬하는 옵션을 선택하세요.
  - 인기순 옵션 - "인기순"으로 표시됩니다. 인기도별로 정렬하려면 선택하세요.
  - 최신순 옵션 - "최신순"으로 표시됩니다. 최신순으로 정렬하려면 선택하세요.

- 필터 목록 패널 - 많은 필터가 있는 왼쪽 패널입니다. 필터를 활성화하거나 비활성화하여 결과를 세분화합니다.
  - 전체 템플릿 버튼 - "전체 템플릿"으로 표시됩니다. 클릭하면 모든 필터가 초기화됩니다.
  - 내 템플릿 버튼 - "내 템플릿"으로 표시됩니다. 클릭하면 내가 만든 템플릿만 표시됩니다.
  - 공유받은 템플릿 버튼 - "공유받은 템플릿"으로 표시됩니다. 클릭하면 가져온 템플릿만 표시됩니다.
  - 즐겨찾기 버튼 - "즐겨찾기"로 표시됩니다. 클릭하면 즐겨찾기한 템플릿만 표시됩니다.
  - 카테고리 탐색 버튼 - "카테고리 탐색"으로 표시됩니다. 클릭하면 카테고리 체크박스를 보이거나 숨깁니다.
  - 태그 버튼 - "태그"로 표시됩니다. 클릭하면 태그 체크박스를 보이거나 숨깁니다.

- 템플릿 목록 - 스크롤 가능한 템플릿 카드 목록입니다. 스크롤하여 사용 가능한 템플릿을 둘러보세요.

- 템플릿 카드 - 색상, 아이콘, 이름을 표시하거나 빈 상태일 수 있습니다. 마우스를 올리면 즐겨찾기 및 열기 버튼이 표시됩니다.

- 즐겨찾기 버튼 - 별 아이콘입니다. 클릭하면 템플릿을 즐겨찾기에 추가합니다.

- 열기 버튼 - 눈 아이콘입니다. 클릭하면 템플릿 상세 정보가 열립니다.

#### 템플릿 상세 정보

- 뒤로 버튼 - "< 뒤로"로 표시됩니다. 클릭하면 템플릿 목록으로 돌아갑니다.

- 즐겨찾기 버튼 - "별 즐겨찾기"로 표시됩니다. 클릭하면 즐겨찾기 상태가 토글됩니다.

- 계속 버튼 - "계속 >"으로 표시됩니다. 클릭하면 선택한 대시보드가 생성되고 열립니다.

- 카테고리 알약 - 카테고리 이름으로 표시됩니다. 그룹화를 이해하기 위해 확인하세요.

- 미리보기 이미지 - 템플릿의 스크린샷으로 표시됩니다. 모양을 미리 보기 위해 확인하세요.

- 미리보기 확장 버튼 - 확장 아이콘으로 표시됩니다. 클릭하면 미리보기 이미지를 전체 화면으로 볼 수 있습니다.

- 태그 알약 - 태그 이름으로 표시됩니다. 템플릿 분류를 이해하기 위해 확인하세요.

- 설명 텍스트 - 템플릿에 대한 긴 설명입니다. 더 자세히 알아보기 위해 읽어보세요.

- 필수 조건 텍스트 - 필요한 것에 대한 짧은 설명입니다. 하위 계정이 필요한 설정을 갖추고 있는지 확인하기 위해 읽어보세요.

### 기존 대시보드 복제

- 대시보드 선택 드롭다운 - "대시보드 선택"으로 표시됩니다. 기존 대시보드를 선택하는 데 사용하세요.

- 대시보드 이름 필드(새) - 선택한 대시보드 이름과 " - 복제"가 표시된 텍스트 필드입니다. 새 대시보드 이름을 설정하기 위해 편집하세요.

- 비공개 대시보드 토글 - 위와 동일합니다. 표시 설정을 토글하세요.

- 에이전시 사용자 접근 드롭다운 - "에이전시 사용자"라는 이름의 드롭다운입니다. 접근 수준을 선택하세요.

- 계정 관리자 접근 드롭다운 - "계정 관리자"라는 이름의 드롭다운입니다. 접근 수준을 선택하세요.

- 계정 사용자 접근 드롭다운 - "계정 사용자"라는 이름의 드롭다운입니다. 접근 수준을 선택하세요.

접근 옵션:
- 전체 옵션 - "전체"로 표시됩니다. 전체 접근을 위해 선택하세요.
- 편집 옵션 - "편집"으로 표시됩니다. 편집만 가능하도록 선택하세요.
- 보기 옵션 - "보기"로 표시됩니다. 보기만 가능하도록 선택하세요.

- 복제 버튼 - "복제"라고 표시된 파란색 버튼입니다. 클릭하면 복제된 대시보드가 생성되고 열립니다.

### 위젯 추가(또는 편집) 패널

- 위젯 탭 - 밑줄이 그어진 "위젯"입니다. 클릭하면 위젯 컨트롤로 전환됩니다.
- 요소 탭 (하위 계정 대시보드 - 요소 추가 또는 편집 아티클 참조) - 밑줄이 그어진 "요소"입니다. 클릭하면 요소 컨트롤로 전환됩니다.
- 테마 탭 (하위 계정 대시보드 - 테마 편집 아티클 참조) - 밑줄이 그어진 "테마"입니다. 클릭하면 테마 컨트롤로 전환됩니다.
- 위젯 검색 - "위젯 검색"입니다. 키워드를 입력하여 위젯을 필터링하세요(목록과 각 카테고리 옆의 녹색 위젯 개수가 변하는 것을 확인할 수 있습니다).

차트 유형별 표시:
- 모든 차트 버튼 - "모두"로 표시됩니다. 클릭하면 위젯 목록 필터가 제거됩니다.
- 숫자 차트 버튼 - "#" 아이콘으로 표시됩니다. 클릭하면 숫자만 표시하는 위젯으로 필터링됩니다.
- 도넛 차트 버튼 - 도넛 아이콘으로 표시됩니다. 클릭하면 도넛(원형) 차트를 표시하는 위젯으로 필터링됩니다.
- 선 차트 버튼 - 선 아이콘으로 표시됩니다. 클릭하면 선 그래프를 표시하는 위젯으로 필터링됩니다.
- 막대 차트 버튼 - 막대 아이콘으로 표시됩니다. 클릭하면 세로 막대 그래프를 표시하는 위젯으로 필터링됩니다.
- 가로 막대 차트 - 가로 막대 아이콘으로 표시됩니다. 클릭하면 가로 막대 그래프를 표시하는 위젯으로 필터링됩니다.
- 테이블 차트 버튼 - 데이터 테이블 아이콘으로 표시됩니다. 클릭하면 테이블에 데이터를 표시하는 위젯으로 필터링됩니다.
