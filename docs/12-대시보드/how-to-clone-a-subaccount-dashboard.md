# 하위 계정 대시보드 복제하는 방법

# **대시보드 복제**

## **1단계: 대시보드 열기**

`Subaccount(하위 계정) → Dashboard(대시보드)`로 이동해서 대시보드 선택 메뉴를 클릭하세요. 복제하려는 대시보드를 검색하거나 찾아보세요. 선택하면 보기 및 편집을 위해 대시보드가 열립니다.

![대시보드 선택 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044573427/original/SabBltMQD22qnDtAa81ooJp0HJUWRRLghQ.png?1743804171)

## **2단계: 대시보드 복제하기**

원본 대시보드에서 작업 메뉴(점 3개)를 클릭하고 복제 버튼을 클릭하세요.

![대시보드 복제 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044573391/original/7QxGqD4lg8IgpiqZmBFyvs4K7HMBNSF2ew.png?1743803929)

## 3단계: 대시보드 복제본 설정 및 저장

대시보드 복제 모달에서 복제된 대시보드의 제목을 작성하고, 대시보드의 개인정보 보호/접근 권한을 설정한 다음 복제 버튼을 클릭하세요. 그러면 적절한 설정으로 새로운 복제 대시보드가 생성됩니다.

![대시보드 복제 설정 모달](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044573401/original/lzYJqJReqAwa75pXQ5EGJ9wnffGvqI7J1g.png?1743804027)
