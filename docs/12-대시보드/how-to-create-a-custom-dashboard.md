# 커스텀 대시보드 만드는 방법

여러 대시보드를 생성하고 관리하여 고유한 요구사항과 인사이트에 맞춰 보고서 경험을 손쉽게 맞춤화하세요. 관련 위젯들을 통합 뷰로 구성하여 데이터 시각화를 효율적으로 정리할 수 있습니다.

커스텀 위젯과 보고서는 **모든 플랜에서 사용 가능**합니다(무제한 커스텀 대시보드).

---

## 커스텀 대시보드란?

커스텀 대시보드는 각 하위 계정에서 연락처, 예약, 기회, 통화, 매출 등의 KPI를 추적할 수 있는 구성 가능한 분석 공간입니다. 레이아웃, 추가할 위젯, 위젯이 사용할 지표, 심지어 대시보드의 모양과 자동 보고서 전송까지 모든 것을 직접 제어할 수 있습니다.

## 커스텀 대시보드의 주요 장점

- **여러 대시보드:** 영업, 마케팅, 재무 또는 기타 팀별로 전용 대시보드를 생성할 수 있습니다. 기본 대시보드로 설정하는 것도 가능합니다.

- **커스텀 위젯 & 지표:** 가장 중요한 인사이트를 표면화하기 위해 무제한 유연성으로 지표를 추가, 편집, 업데이트할 수 있습니다.

- **예약 보고서:** 원하는 빈도로 보고서를 예약하고 팀에게 자동으로 전송할 수 있습니다.

- **완전한 커스터마이징:** 브랜드에 맞춰 테마, 색상, 배경으로 대시보드를 개인화할 수 있습니다.

- **심층 인사이트:** 테이블을 포함한 다양한 형식으로 데이터를 보고 분석할 수 있습니다.

- **범용 위젯:** 범용 위젯을 추가하거나 제거하여 에이전시의 워크플로우에 맞는 대시보드 경험을 만들 수 있습니다.

---

## 커스텀 대시보드 만드는 방법

필요에 맞는 커스텀 대시보드를 만들기 위해 다음 단계를 따라하세요!

#### 1단계: 대시보드 영역으로 이동

하위 계정의 대시보드 섹션으로 이동합니다.

![대시보드 섹션으로 이동](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044225708/original/c2DzFwVfMi2tICWSVmbrqzYVnAZgoFaiHA.png?1743298746)

#### 2단계: 새 대시보드 추가

**Dashboard Selection Menu(대시보드 선택 메뉴)**를 클릭하고 드롭다운에서 "+ Add Dashboard" 버튼을 클릭합니다.

참고: 이 버튼이 보이지 않는다면 대시보드 뷰 모드에 있는지 확인하세요.

![새 대시보드 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044353813/original/kR7TkjtsM2b7qeuqmuypFcq9mVp_3AlIPQ.png?1743545895)

#### 3단계: 대시보드 이름 입력

Dashboard Name(대시보드 이름) 필드에 이름을 입력합니다.

![대시보드 이름 입력](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044353863/original/Zil6-u30DIU_lRTQhSZDJhWf5sEaZqZzKw.png?1743546127)

#### 4단계: 대시보드 권한 설정

Private Dashboard(비공개 대시보드) 토글을 사용하여 표시 여부를 제어합니다. **ON**으로 설정하면 비공개가 되고, **OFF**로 설정하면 팀 액세스가 허용됩니다.

Private Dashboard가 OFF로 설정된 경우, Agency User(에이전시 사용자), Account Admin(계정 관리자), Account User(계정 사용자)에 대한 **접근 권한**을 설정하세요.

참고: 이 토글이 보이지 않는다면 "New Dashboard" 페이지에 있는지 확인하세요.

#### 5단계: 대시보드 생성 확인

"Confirm" 버튼을 클릭하여 생성을 완료합니다. 새로운 빈 대시보드가 열립니다.

![대시보드 생성 확인](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044353921/original/mO8decxxk166CBFXYkMVHqq7IYTTik0zMw.png?1743546348)

#### 6단계: 위젯 추가 방법 선택

대시보드가 생성되면 Quick Start Widgets(빠른 시작 위젯) 페이지가 나타납니다. 이는 대시보드가 비어있을 때(위젯이 없을 때) 항상 표시되는 기본 페이지입니다. 여기서 다음 옵션 중 하나를 선택하세요:

**옵션 A: 제안된 위젯 사용**

"**What is your goal?(목표가 무엇인가요?)**" 또는 "**Top used widgets(자주 사용하는 위젯)**" 빠른 시작 옵션을 사용하여 하나 이상의 인기 위젯을 선택합니다. 이는 대시보드가 비어있을 때마다 표시됩니다.

"Confirm & Add"를 클릭하여 자주 사용되는 위젯을 빠르게 삽입합니다.

9단계로 이동합니다.

![제안된 위젯 사용](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044353940/original/93v7TUuBYpKB9b74ZV8QZu0Gy9dBl0qicw.png?1743546469)

**옵션 B: 위젯 수동 추가**

"+ Add a widget manually"를 클릭하여 Add New Widget(새 위젯 추가) 패널을 엽니다.

7단계로 이동합니다.

![위젯 수동 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044353962/original/fGrCtpdkC53BdmRMkUQphyEXqE_1wcKKJQ.png?1743546541)

**옵션 C: 편집 모드에서 위젯 추가**

Edit(편집) 버튼(연필 아이콘)을 클릭한 후 편집 모드에서 "+ Add Widget"을 클릭합니다.

참고: 이 버튼이 보이지 않는다면 편집 모드에 진입했는지 확인하세요.

7단계로 이동합니다.

![편집 모드에서 위젯 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044353966/original/YjAZUQzh-9Siut7pCQGsuS2yy3cRkxfIHw.png?1743546583)

#### 7단계: 새 위젯 구성

Add New Widget 패널을 사용하여 위젯 설정을 정의합니다.

자세한 내용은 [하위 계정 대시보드 페이지 - 위젯 추가 또는 편집](subaccount-dashboard-page-view-or-edit.md) 아티클을 참고하세요.

![새 위젯 구성](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044354006/original/NTqv3HbxTGH8pQ-8JNPnmknTN3zl42bjDw.png?1743546678)

#### 8단계: 위젯 저장

위젯 패널 내에서 "Save" 버튼을 클릭합니다.

![위젯 저장](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044227004/original/qrQfZeuE1i03OEhJ78hoxVK1LP1a6I1LeA.png?1743309668)

#### 9단계: 대시보드 편집 내용 저장

"✓ Save Changes" 버튼을 클릭하여 완료합니다. 이렇게 하면 레이아웃이 저장되고 편집 모드에서 나갑니다.

![대시보드 편집 내용 저장](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044227016/original/Cuoneqt7gq0pL1dRLM9BCG7pKjJR9pIlOg.png?1743309746)

## 자주 묻는 질문

**Q: 어떤 플랜에 커스텀 대시보드와 커스텀 보고서가 포함되나요?**
모든 Hyperclass 플랜($97, $297, $497)에 여러 대시보드와 커스텀 보고서가 포함됩니다.

**Q: 어떤 차트 유형을 사용할 수 있나요?**
위젯은 스코어카드, 시계열, 막대/세로막대, 도넛/파이, 테이블 등 여러 차트 유형을 지원합니다. 가용성은 위젯에 따라 달라집니다.

**Q: 비사용자(예: 클라이언트)에게 보고서를 보낼 수 있나요?**
네. **Recipients(수신자)**에 유효한 이메일 주소를 추가하면 됩니다.

**Q: 대시보드를 복제할 수 있나요?**
네. 대시보드 작업을 사용하여 복제한 다음 위젯, 필터, 테마를 조정하면 됩니다.
