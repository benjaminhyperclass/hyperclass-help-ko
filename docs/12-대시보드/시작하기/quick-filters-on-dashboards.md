# 대시보드 빠른 필터

빠른 필터(Quick Filters)는 사용자가 편집 권한 없이도 대시보드 위젯을 분석할 수 있게 해주는 기능입니다. 이 기능을 통해 사용자는 개별적으로 필터를 적용할 수 있어 수동 조정의 필요성을 줄이고 대시보드의 무결성을 보호할 수 있습니다. 빠른 필터는 사용자별로 저장되어 각 사용자가 개인화된 필터링 경험을 할 수 있습니다.

**중요:** 빠른 필터는 현재 커스텀 위젯(직접 생성한 위젯)에서만 작동합니다. 기본 일반 위젯에는 적용되지 않습니다.

# **대시보드에 빠른 필터 추가하기**

### 1단계: 대시보드로 이동

- 빠른 필터를 추가하려는 대시보드를 엽니다.

### 2단계: 빠른 필터 메뉴 접근

- 대시보드 상단의 "Quick Filters(빠른 필터)" 버튼을 클릭합니다.
![빠른 필터 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155040114997/original/T0suHQk2dCQpgAc8UIf4qC9T34ZpayF4hw.png?1737371785)

### 3단계: 필터 추가

- "+ Add Quick Filters(빠른 필터 추가)" 버튼을 클릭합니다.
![빠른 필터 추가 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155040115064/original/tDwpX6ApzJjmkYzNb2ZoWSoAN8iNukRJ0w.png?1737371824)

- 데이터 소스별로 분류된 사용 가능한 필터가 포함된 사이드바가 나타납니다.

- 검색창을 사용해 특정 필터를 찾거나 카테고리를 탐색합니다.

### 4단계: 필터 선택

- 추가하려는 필터 옆의 체크박스를 선택합니다.

- "Add(추가)"를 클릭하여 진행합니다.
![필터 선택 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155040115095/original/YcUMzjaWVnlaVI82og2WERYk9c5_dk9R3Q.png?1737371849)

### 5단계: 검토 및 적용

- 선택한 필터를 검토하고 필요시 설명을 추가합니다.
![필터 검토 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155040115115/original/oI-ZetIP6_0wzfe1a5x7FiUDx4T6kmX5pA.png?1737371866)

- 설명은 모든 사용자에게 표시되며 각 필터의 목적을 이해하는 데 도움이 됩니다.
![필터 설명 표시](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155040115130/original/yWVGZqpNCy3h_jVkv14yrpd_-ZpmVtYhmw.png?1737371884)

- "Apply(적용)"를 클릭하여 설정을 완료합니다.

**중요사항:**

1. 빠른 필터는 각 대시보드별로 설정됩니다. 모든 대시보드에 대해 별도로 구성해야 합니다.
2. 대시보드에 대한 완전 액세스 또는 편집 권한이 있는 사용자만 빠른 필터를 추가할 수 있습니다.
3. 보기 권한만 있는 사용자는 필터를 사용할 수 있지만 수정할 수는 없습니다.

# **빠른 필터 사용하기**

### 1단계: 대시보드로 이동

- 빠른 필터가 구성된 대시보드를 엽니다.
![필터가 구성된 대시보드](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155040115168/original/LkPIK_jpc9dtbo9uMk3KzGw4GmiPQJveQA.png?1737371919)

### 2단계: 필터 적용

- 빠른 필터 바를 사용하여 필터를 선택하고 적용합니다.

- 필터 속성을 조정하여 보기를 세분화합니다.
![필터 적용 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155040115200/original/iRGkULrbFMewxJyMi2JdJCFb1JlH7TjavA.png?1737371939)

### 3단계: 필터링된 결과 보기

- 적용된 필터에 따라 위젯이 업데이트됩니다.

**중요사항:**

필터는 해당 필터 속성과 동일한 데이터 소스를 사용하는 위젯에만 적용됩니다.

예시: 연락처 데이터 소스의 태그 속성에 대한 필터는 연락처 데이터 소스를 사용하는 위젯에만 영향을 미칩니다.

**참고:**

필터는 설정한 사용자에게만 적용되며 다른 사용자에게는 영향을 주지 않습니다.

필터는 현재 브라우저 세션 내에 저장됩니다. 같은 세션에서 대시보드를 다시 방문하면 마지막으로 선택한 필터가 그대로 적용되어 있습니다.

# **빠른 필터와 기존 위젯 레벨 필터의 작동 방식**

- 빠른 필터와 위젯 필터는 AND 논리로 함께 작동합니다. 즉, 빠른 필터가 위젯 레벨 필터를 재정의하지 않고 추가됩니다.
- 잘못되거나 누락된 데이터를 방지하려면, 위젯별 요구사항에는 위젯 필터를, 더 광범위한 팀 차원의 필터링에는 빠른 필터를 사용하세요.

예시:
다음과 같이 설정한 경우:
- 위젯 필터: 예약 상태 = 확정됨
- 빠른 필터: 예약 상태 = 취소됨

시스템은 상태가 확정됨 AND 취소됨인 예약을 찾으려 하는데, 이는 불가능하므로 데이터가 표시되지 않습니다.

✅ **권장사항**: 팀 전체의 유연한 필터링에는 빠른 필터를 사용하고, 위젯 레벨에서 유사한 필터를 중복 설정하지 마세요.
