# 대시보드를 편집하는 방법

대시보드 소유자는 대시보드를 편집하여 보고 기능을 향상시킬 수 있습니다. 대시보드 이름 수정, 권한 업데이트, 새로운 커스텀 위젯 추가, 기존 위젯 편집이 가능합니다. 다음 단계를 따라 대시보드 편집 기능을 최대한 활용해보세요.

---

## 참고사항

여러 개의 커스터마이즈 가능한 대시보드는 어떤 플랜에서 사용할 수 있는지 궁금하신가요?

여러 개의 커스텀 대시보드, 위젯, 커스텀 리포트는 $297, $497+ 플랜에서 이용 가능합니다.

---

### 1단계: 대시보드로 이동

편집하려는 특정 대시보드를 찾아 이동하세요. 참고: 대시보드 소유자만 대시보드를 편집할 수 있습니다.

![대시보드 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155014394504/original/cQPxedtzIl5veBjtZ0NZ0jIi0jjBi4TuuA.png?1701773142)

### 2단계: 편집 모드 접근

우측 상단의 연필 편집 아이콘을 클릭하세요. 이 작업으로 편집 모드로 전환됩니다.

![편집 아이콘](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155014394681/original/bv-tzE3LS8lEAE8DiOkETyfSECw70HWQ1w.png?1701773220)

### 3단계: 대시보드 이름 수정

필요하다면 보고 요구사항에 더 적합하도록 대시보드 이름을 변경하세요.

![대시보드 이름 수정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155014394750/original/9_9umKUhFSLapo_GjA65IU4TqVxX8iRO1w.png?1701773251)

### 4단계: 새 위젯 추가

"위젯 추가(Add Widgets)"를 클릭하여 대시보드에 새로운 커스텀 위젯을 추가하세요. 커스텀 위젯 추가에 대한 자세한 가이드는 [여기](../how-to-create-add-dashboard-widgets.md)를 참고하세요.

![위젯 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155014394784/original/ZhoKJ6P6bREAc0e4j2FBI5pl8IBrNbkdVg.png?1701773273)

### 5단계: 변경사항 저장

필요한 편집을 완료했다면 "변경사항 저장(Save Changes)"을 클릭하여 수정 사항을 적용하고 완료하세요.

![변경사항 저장](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155014394842/original/btZlr2oGsHubw3rjLfN-mVLqpLlfDE1gvw.png?1701773303)
