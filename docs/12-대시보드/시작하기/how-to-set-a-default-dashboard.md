# 기본 대시보드 설정하는 방법

기본 대시보드는 팀 협업을 원활하게 하고 팀원들이 항상 가장 중요한 인사이트에 집중할 수 있도록 도와주는 핵심 도구입니다. 기본 대시보드로 설정하면 팀원들이 대시보드 섹션으로 이동했을 때 가장 먼저 보게 되는 화면이 되어, 필수적인 인사이트와 지표에 즉시 접근할 수 있습니다.

## 중요 사항

- **필요한 권한:** **계정 관리자(Account Admin)** 또는 대시보드에 대한 **전체(FULL)** 접근 권한을 가진 **에이전시(Agency)** 역할의 사용자만 기본 대시보드를 설정할 수 있습니다. 대시보드 권한에 대해 더 자세히 알아보려면 [여기를 클릭하세요](how-to-manage-permissions-for-a-dashboard.md)!
- **보기 권한:** 모든 팀원이 해당 대시보드에 대해 최소한 **보기(VIEW)** 권한을 가지고 있어야 기본 대시보드로 설정할 수 있습니다.
- **기본 대시보드는 삭제 불가:** 기본 대시보드로 설정되면, 다른 대시보드를 기본으로 지정하기 전까지는 삭제할 수 없습니다.
- **기본 대시보드와 즐겨찾기 대시보드의 차이:** 기본 대시보드는 모든 사용자에게 공통으로 적용되는 기본 화면이고, 즐겨찾기 대시보드는 개별 사용자가 빠르게 접근하기 위해 개인적으로 설정하는 것입니다.

**중요**

이번 업데이트 후 기본 대시보드가 변경되어 문제가 발생한다면, 위의 안내에 따라 다시 원하는 대시보드로 설정하실 수 있습니다. 기존 대시보드와 위젯은 영향받지 않습니다. 업데이트 후 발견된 버그가 있었지만, 이를 해결하는 패치가 이미 배포되었습니다. 여전히 문제가 발생한다면 고객지원팀에 문의해 주세요.
