## 중요 참고사항

- 선택한 시간대는 보는 모든 대시보드에 적용됩니다.

- 시간대 변경은 내부 데이터 지표에만 영향을 주며, 외부 서비스 종속 위젯에는 영향을 주지 않습니다.

- 커스텀 대시보드에 대해 더 자세히 알고 싶으시면 여기를 클릭하세요!

## 대시보드 시간대 변경하는 방법

로그인한 사용자의 대시보드에 표시되는 시간대를 변경하려면 아래 단계를 따라하세요!

#### **1단계:** 대시보드 열기

개인 대시보드든 공유 대시보드든, 왼쪽 네비게이션 바에서 **Dashboard(대시보드)**를 선택하여 커스터마이징하고 싶은 대시보드로 이동하세요.

![대시보드 메뉴 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155061627072/original/0rr6MBwvKjxpqsvZ5naROHsGBSlBqcqj0Q.png?1767115891)

#### **2단계:** 시간대 설정 접근

우측 상단에 있는 **점 3개** 메뉴 아이콘을 클릭하고, 드롭다운에서 **Manage Dashboard Timezone(대시보드 시간대 관리)**를 선택하세요.

![시간대 설정 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155061627188/original/f3RDaBSAtTAd_vqHbGR7j2uu2yYU-SWiAw.png?1767115984)

#### 3단계: 시간대 선택

드롭다운 메뉴에서 원하는 시간대를 선택한 후 변경사항을 저장하세요.

![시간대 선택 및 저장](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155061627277/original/3NueyrIVMlAxD0OAbN6IWncXc9rRtofXUQ.png?1767116057)
