# "본인 데이터만 보기" 설정을 대시보드에서 활용하는 방법

역할 및 권한의 "본인 데이터만 보기(Only Assigned Data)" 설정은 데이터 보안을 강화하고 사용자 경험을 개선하는 강력한 도구입니다. 이 설정을 통해 팀원들은 자신에게 할당된 데이터만 볼 수 있어 혼란을 최소화하고 데이터 무결성을 유지할 수 있습니다.

### 대시보드에서 "본인 데이터만 보기"를 사용하는 이유와 방법

#### 사용 사례와 장점:

- **보안 강화:** 데이터 가시성을 제한하여 민감한 정보를 무단 액세스로부터 보호하고, 관련 팀원만 특정 데이터를 볼 수 있도록 보장합니다.

- **효율성 향상:** 이 기능은 수동 필터링 없이도 사용자와 관련된 데이터만 표시하도록 대시보드 위젯을 자동으로 필터링합니다. 설정 시간을 단축하고 사용자가 자신의 업무에 집중할 수 있게 해줍니다.

- **워크플로우 간소화:** 시스템이 자동으로 위젯 데이터를 제한하여 사용자가 자신의 역할과 무관한 정보를 보지 못하게 차단합니다. 이를 통해 더 체계적이고 사용자 친화적인 대시보드 환경을 구현할 수 있습니다.

#### 사용 예시:

- **영업팀:** 영업 관리자는 영업 담당자가 자신의 리드와 기회만 볼 수 있도록 하여 데이터 중복을 방지하고 집중도를 높일 수 있습니다.

- **고객 성공팀:** 팀원들은 자신에게 할당된 고객, 통화, 예약만 볼 수 있도록 제한하여 더 빠른 응답 시간을 달성할 수 있습니다.

---

### 1단계: 설정으로 이동

- Settings(설정) → My Staff(내 직원)를 클릭합니다.

- 데이터 가시성을 설정하고 싶은 특정 팀원을 검색합니다. 해당 팀원 이름 옆의 Edit(편집) 버튼을 클릭합니다.

### 2단계: "본인 데이터만 보기" 활성화

- 프로필의 Roles and Permissions(역할 및 권한) 섹션에서 Data Visibility Scope(데이터 가시성 범위) 옵션을 찾습니다.

- 드롭다운 메뉴에서 Only Assigned Data(본인 데이터만 보기)를 선택합니다.
