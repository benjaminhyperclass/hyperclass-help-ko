# 대시보드 위젯 커스터마이징

위젯은 데이터를 시각화하는 강력한 도구이며, 커스터마이징을 통해 필요한 인사이트를 정확히 얻을 수 있습니다. 이 가이드에서는 위젯 설정 수정, 필터 추가, 차트 타입 실험, 그리고 목표에 맞는 표시 방식 세부 조정 방법을 설명합니다. 이 간단한 단계들로 대시보드를 효과적이고 시각적으로 매력적인 데이터 센터로 변환해보세요.

## 대시보드 위젯이란?

대시보드 위젯은 차트, 그래프, 표와 같은 시각적 형태로 핵심 데이터 포인트를 보여주는 소형 인터랙티브 도구입니다. 이러한 위젯은 복잡한 정보를 읽기 쉬운 형태로 요약하는 데 필수적입니다. 위젯을 커스터마이징하면 외관과 기능을 맞춤화하여 리포팅 요구사항에 완벽하게 맞출 수 있습니다.

## 대시보드 위젯을 커스터마이징해야 하는 이유

위젯 커스터마이징은 다음과 같은 장점을 제공합니다:

• **개인화된 인사이트**: 비즈니스 목표와 가장 관련성이 높은 지표를 강조표시합니다.
• **향상된 명확성**: 제목, 필터, 레이아웃을 조정하여 데이터 해석을 간단하게 만듭니다.
• **개선된 비교**: 필터와 조건을 추가하여 다양한 지표나 기간을 쉽게 비교할 수 있습니다.
• **효율성**: 관련 없는 정보를 탐색할 필요 없이 필요한 데이터에 빠르게 접근합니다.

## 위젯 커스터마이징 옵션 찾기

1. 플랫폼 내 대시보드로 이동합니다.
2. 목록에서 수정하려는 위젯을 선택합니다.
3. 위젯을 클릭하여 구성 및 커스터마이징 옵션에 접근합니다.

## 대시보드 위젯 커스터마이징 방법

### 1단계: 위젯 선택

위젯 목록에서 수정하려는 위젯을 선택합니다. 기본적으로 각 위젯은 "ALL" 필터 하에 미리 선택된 차트 타입으로 제공됩니다. 차트 타입을 변경하려면 차트 아이콘을 클릭하고 데이터에 가장 적합한 것을 선택하세요.

![위젯 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036973584/original/-sbUU1m-Iag_gKL5qr_6w-rKjrpdWbc5yg.png?1732194078)

### 2단계: 위젯 설정 구성

"Configure(구성)" 탭으로 이동하여 위젯을 조정합니다:

**Title(제목)**: 위젯의 목적이 명확하도록 이름을 변경합니다.

![제목 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036971813/original/q47ytBo1FNk_2lYiOY8NXjr2d_ylcfADuQ.png?1732193192)

**Metrics(지표)**: 위젯에 표시되는 데이터 포인트를 업데이트합니다. 라인, 바, 수평 바 차트에는 여러 지표를 추가할 수 있습니다.

![지표 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155041049991/original/UKczdpHmNcfdoCiQvuBbme_d917ysh220A.png?1738753060)

**Group(그룹)**: 도넛 차트와 같은 차트의 그룹핑 옵션을 정의합니다.

![그룹 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036971934/original/1OqFcV4ReLAS_i46TNZxVPqGfpy7v5ZwtA.jpeg?1732193250)

**View By(보기 기준)**: 라인이나 바 차트와 같은 시각화를 위한 보조 차원을 선택합니다.

![보기 기준 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036971986/original/hamS3uRkGn5V9-pvAANAWfAV9HTl1KT-3Q.jpeg?1732193278)

**Breakdown(세분화)**: 시간 주기 빈도를 조정합니다.

**Date Property(날짜 속성)**: 결과를 가져올 날짜 타입(예: Created Date, Updated Date)을 선택합니다.

![날짜 속성 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036972033/original/eT0TdHBymUtrOe0dXxjYKlACBRj8UsFJ8Q.jpeg?1732193312)

**Date Range Override(날짜 범위 재정의)**: 이 위젯에 대해 대시보드의 전역 날짜 범위를 재정의합니다.

![날짜 범위 재정의](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036972116/original/2xC-KBovFoHDFfRoHENRqqZX2tSay5WWNQ.jpeg?1732193350)

**Order(정렬)**: 결과를 오름차순 또는 내림차순으로 정렬합니다.

**Limit(제한)**: 표시되는 최대 결과 수를 설정합니다.

**Comparison Date Property(비교 날짜 속성)**: 더 나은 맥락을 위해 비교 퍼센티지를 추가합니다.

고급 설정은 추가 개인화를 위해 사용할 수 있지만 선택사항입니다.

### **위젯 설정의 차트 타입 선택기**

적절한 시각화를 선택하면 가독성과 의사결정이 향상됩니다. 워크플로우를 벗어나지 않고도 위젯 설정에서 직접 **차트 타입**을 선택할 수 있습니다.

차트 선택 방법

1. **Settings(설정)** → Configure(구성)을 엽니다.
2. 원하는 **Chart Type(차트 타입)**(예: 바, 라인, 파이)을 선택하고 실시간 미리보기를 확인합니다.
3. **Conditions(조건)**으로 계속 진행하거나 위젯을 저장합니다.

![차트 타입 선택기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155064044294/original/1nOQf-WnqgcRL_P1MbmUz2DaOWy8DeDAcQ.png?1770103110)

### 3단계: 조건 추가

### **중첩 필터 그룹 (OR 논리)**

필터 그룹핑을 통해 실제 세그먼트와 일치하는 고급 논리를 구현할 수 있습니다. 여러 그룹을 사용하여 OR 조건을 평가하고, 각 그룹 내의 AND로 결과를 정확히 좁힙니다.

그룹 만드는 방법

- 위젯을 열고 → **Conditions(조건)**을 클릭합니다.
- Add Filter(필터 추가)를 클릭하여 그룹 1을 만듭니다.
- 같은 그룹 내에 더 많은 필터를 추가하려면 +AND를 사용합니다.
- +Add Filter(필터 추가)를 다시 클릭하여 그룹 2를 시작합니다(**OR** 조건 추가).
- 필요에 따라 반복하고 저장합니다.

![중첩 필터 그룹](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155064046099/original/X5zNRmy7_jo_vKJEoncmM3hyZFPwV02TwA.gif?1770104178)

참고: 방문자 데이터 위젯은 **하나의** 필터 그룹만 지원합니다.

---

### **그룹을 통한 중복 필터**

때로는 같은 필드를 두 번 사용해야 할 때가 있습니다(예: Source = google OR Source = facebook). 이제 별도의 그룹을 통해 중복이 지원되어, 우회 방법 없이도 더 넓은 매칭이 가능합니다.

예시

- **그룹 1**: Source = google
- **그룹 2**: Source = facebook

![중복 필터](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155064046556/original/mXTCbGDLdExJSiN4VA0nmpjmmwxhWr8fpw.png?1770104518)

### 연결된 필터 경고

특정 필터들은 서로 의존적입니다(예: **Pipeline**과 Stage). 조합이 충돌할 때 정확한 데이터를 위해 조건을 조정할 수 있도록 명확한 경고가 표시됩니다.

**동작 방식**

- 연결된 필드를 강조하기 위해 **Conditions(조건)** 내에 인라인으로 경고가 나타납니다.
- 메시지를 해결하기 위해 필터 중 하나를 조정하고 저장합니다.

1. "Conditions(조건)" 탭에서 Add Condition(조건 추가)을 클릭합니다.

![조건 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036977930/original/USh6mlxHkJysmVvjp6zLSh5cW2Ufqoe-AA.png?1732196531)

2. 필터, 연산자, 매개변수를 선택합니다.

![필터 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036978025/original/Ri6GHKn2rVzbIVVl3rAOn3uLr2LzS3W2yA.png?1732196593)

3. AND/OR 조건을 사용하여 여러 필터를 추가하거나 불필요한 필터를 삭제합니다.

![필터 관리](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036978054/original/gfiSlqWNmM6YRjnQt0DprzmBFsHWQGB49w.png?1732196603)

참고: 필터와 일치하는 데이터가 없으면 필터를 조정하거나 제거하지 않는 한 위젯을 저장할 수 없습니다.

### 4단계: 변경사항 저장

모든 변경이 완료되면:

1. Save(저장)를 클릭하여 위젯을 업데이트합니다.

![저장](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036982621/original/adkGcYPXFyL5cPLE7ePgI3-PezATv-jJgg.png?1732199201)

2. 필요에 따라 대시보드에서 위젯의 크기나 위치를 조정합니다.

![위젯 크기 조정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036982751/original/5LcLFoqfEqONlVqoeDiyB3oDHBY1nGaR_A.gif?1732199269)

3. 대시보드 레이아웃을 저장하여 변경사항을 완료합니다.

![레이아웃 저장](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155036982786/original/4S5WaP-sd44FUXWWicvQ3ucnyt6ZJtfMKw.png?1732199304)
