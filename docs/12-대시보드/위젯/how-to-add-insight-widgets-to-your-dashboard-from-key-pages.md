# 주요 페이지에서 대시보드에 인사이트 위젯 추가하기

이제 주요 페이지에서 대시보드에 관련 인사이트 위젯을 바로 추가할 수 있습니다. 이렇게 하면 화면을 전환하지 않고도 일상 업무 흐름에 분석 데이터를 쉽게 가져올 수 있어요.

✨ **언제 사용하나요?**
일상 업무 워크플로우에 **대시보드** 분석 데이터를 가져와서 주요 지표를 빠르게 확인하고 추적할 때 사용합니다.
