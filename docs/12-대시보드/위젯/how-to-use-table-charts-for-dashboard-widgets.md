# 대시보드 위젯에서 테이블 차트 사용하는 방법

### 1단계: 위젯 추가

- 대시보드로 이동하여 편집 아이콘을 클릭한 다음 Add widget(위젯 추가)를 선택하세요

### 2단계: 테이블 차트 선택

- 추가하려는 위젯까지 스크롤을 내린 후 차트 유형을 table(테이블)로 변경하세요

![테이블 차트 선택 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155022885087/original/Tv-DNpkI8qeG5vXbj_anl0A3YGPmNnTQKg.png?1710421262)

### 3단계: 위젯 설정

- Select Columns(컬럼 선택)를 클릭하여 위젯에 표시할 컬럼을 관리하세요

![컬럼 선택 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155022885097/original/EyMRUGmsd3nGgTZ5OvmnuAA5nQVsch5a9w.png?1710421277)

- Advance Settings(고급 설정)에서 테이블을 정렬할 기준 컬럼을 선택할 수 있습니다

![고급 설정 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155022885142/original/8rZzu6fO66Tq1o4iEUxrllJh7L3--ovZlQ.png?1710421311)

**참고 (이메일 위젯): Source Action Name**
이메일 위젯 테이블 차트에서 Source Action Name은 해당 이메일이 무엇에 의해 실행되었는지 식별하는 데 도움이 됩니다. 이메일이 워크플로우와 연결되어 있는 경우, Source Action Name에는 워크플로우 단계 이름이 표시되어 대시보드와 리포트에서 추적성을 향상시킵니다.

### 4단계: 변경사항 저장

- Save(저장)를 클릭하여 대시보드에 위젯을 추가하세요

![위젯 저장 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155022886687/original/cRu3tWKj6h2y424JHndURAxemS2w7hd57Q.png?1710422439)
