# 위젯 크기 조정하기

모서리를 드래그하여 위젯을 확대하거나 축소해서 크기를 빠르게 조정할 수 있습니다.

**목차**

- [1단계: 편집 모드로 전환하기](#1단계-편집-모드로-전환하기)
- [2단계: 위젯 모서리의 "화살표" 아이콘을 클릭하고 드래그하기](#2단계-위젯-모서리의-화살표-아이콘을-클릭하고-드래그하기)
- [3단계: 변경사항 저장하기](#3단계-변경사항-저장하기)

---

### **1단계: 편집 모드로 전환하기**

- 대시보드에서 "편집(EDIT)" 버튼을 클릭하세요.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155010312014/original/wuCiEW339qP1nJXuhe8nIIkWPExQpqPbEQ.jpeg?1697544454)

### **2단계: 위젯 모서리의 "화살표" 아이콘을 클릭하고 드래그하기**

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155010312017/original/v-5n2Zkjlzfe7hpijadi1piM-0g9qBPrOg.png?1697544455)

### **3단계: 변경사항 저장하기**

- "변경사항 저장(SAVE CHANGES)" 버튼을 클릭하는 것을 잊지 마세요.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155010312013/original/gOpwSLsBUP1OG_6CohEgt0kNP8LUCTphdQ.jpeg?1697544454)

- 만약 변경사항을 되돌리고 싶다면 "취소(Cancel)"를 클릭하면 위젯이 이전 상태로 복원됩니다.

## 참고사항:

커스터마이징 가능한 대시보드를 사용할 수 있는 플랜이 궁금하신가요?

현재 $497 플랜 이상에서만 사용 가능합니다. $97 또는 $297 플랜을 사용 중이시라면, 이 기능을 사용하려면 플랜을 업그레이드하시기 바랍니다.

$497+ 플랜의 커스텀 대시보드는 **에이전시 관리자**와 **에이전시 사용자**가 접근할 수 있습니다.
