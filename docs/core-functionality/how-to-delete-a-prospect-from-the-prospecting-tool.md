# 프로스펙팅 도구에서 리드 삭제하는 방법

이 가이드는 프로스펙팅 대시보드에서 리드를 영구적으로 삭제하는 방법을 설명합니다. 올바른 워크스페이스로 전환하고, 리드를 찾아서, 삭제를 확인하는 과정을 안내합니다. 리드 삭제는 되돌릴 수 없으며, 리드 항목과 해당 항목에 연결된 모든 데이터가 제거됩니다.

# **리드 삭제가 하는 일**

리드를 삭제하면 리드 목록에서 해당 리드가 영구적으로 제거되고, 그 리드 항목에 연결된 모든 관련 데이터가 삭제됩니다. 이 작업은 되돌릴 수 없습니다.

---

## **삭제하기 전에**

올바른 리드를 삭제하는지 확인하세요. 아웃리치는 중단하고 싶지만 참고용으로 정보를 보관하고 싶다면, 삭제 대신 비파괴적인 정리 방법(내부 상태 추적이나 목록 정리 등)을 사용하세요.

---

## **리드 삭제하기**

### **올바른 워크스페이스에 있는지 확인**

![프로스펙팅 워크스페이스 전환](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155066438266/original/f-szSQ2iMh-np-ZMZvcBKF3JcNNK6TrZLQ.png?1772940425)

- 계정/워크스페이스 전환기(왼쪽 상단 영역)를 클릭하세요.

- 리드가 있는 워크스페이스를 선택해서 변경 작업 전에 올바른 목록에서 작업하도록 하세요.

### **삭제할 리드 찾기**

![리드 찾기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155066438268/original/A9ZObEg8cXHwIrXsZK4oGCTJSpn8KbIDWQ.png?1772940451)

- 왼쪽 네비게이션에서 **Prospecting(프로스펙팅)**을 열어주세요.

- 검색창과/또는 필터를 사용해서 삭제할 리드를 찾으세요.

- 올바른 레코드를 선택하고 있는지 확인하기 위해 리드 세부정보(이름, 위치 등)를 확인하세요.

### 삭제 확인 및 완료

![리드 삭제 확인](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155066438296/original/A42kIKN9_79bgoQPg8ULWgqinYQn4mQaUA.png?1772940542)

- 삭제할 리드의 **Delete(삭제)** 버튼을 클릭하세요.

- 확인 대화상자의 경고 메시지를 검토하세요.

- 요구되는 경우 확인 체크박스(예: "이해했습니다")를 선택하세요.

- 최종 **Delete(삭제)** 버튼을 클릭해서 리드를 영구적으로 삭제하세요.

---

## **중요 참고사항**

- 리드 삭제는 되돌릴 수 없습니다.

- 삭제 후에는 리드와 관련 데이터가 리드 목록에서 더 이상 사용할 수 없습니다.

- 단순히 아웃리치에서 누군가를 제외하고 싶다면, 삭제 대신 비파괴적인 정리 옵션을 사용하세요.

---

## **자주 묻는 질문**

**Q: 리드 삭제를 되돌릴 수 있나요?**

아니요. 삭제는 영구적이며 되돌릴 수 없습니다.

**Q: 리드를 삭제하면 어떤 정보가 제거되나요?**

리드 항목이 리드 목록에서 제거되고, 그 리드 항목에 연결된 관련 데이터가 삭제됩니다.

**Q: Delete(삭제) 옵션이 보이지 않는 이유는 무엇인가요?**

리드를 삭제할 권한이 없거나, 잘못된 워크스페이스/계정에 있을 수 있습니다. 올바른 워크스페이스로 전환한 후 다시 시도해보세요.

**Q: 잘못된 리드를 삭제했는데 어떻게 해야 하나요?**

삭제는 되돌릴 수 없으므로, (가능하다면) 리드를 새 항목으로 다시 추가하고 필요한 세부정보를 다시 만들어야 합니다.

**Q: 삭제 확인 창이 나타나지만 확인 후에도 아무 일이 일어나지 않는다면 어떻게 해야 하나요?**

페이지를 새로고침하고 다시 시도해보세요. 문제가 지속되면 다른 브라우저를 사용하거나 대화상자나 동작을 차단할 수 있는 브라우저 확장 프로그램을 일시적으로 비활성화해보세요.

---

## 도움이 필요하신가요?

리드를 삭제할 수 없거나 옵션이 나타나지 않는 경우:

- 리드를 검색하기 전에 올바른 워크스페이스/계정에 있는지 확인하세요.

- 페이지를 새로고침하고 다시 시도해보세요.

- 사용자 권한이 리드 삭제를 허용하는지 확인하세요.

- 확인 대화상자가 삭제를 완료하지 않는 경우, 다른 브라우저를 사용하거나 팝업이나 페이지 동작을 방해할 수 있는 브라우저 확장 프로그램을 비활성화해보세요.
