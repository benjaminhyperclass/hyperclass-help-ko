# 템플릿, 폼, 커스텀 도메인으로 퍼널 만들기

퍼널은 방문자를 단계별로 안내하여 연락처 정보를 수집하고 다음 행동으로 유도하는 집중된 페이지 세트입니다. 이 가이드에서는 템플릿으로 퍼널을 만들고, 폼을 추가하거나 선택하고, 커스텀 도메인을 연결하여 퍼널을 공개하는 방법을 설명합니다.

## 시작하기 전에

다음 사항을 확인해주세요:

- 계정의 **Sites(사이트)** 영역에 접근할 수 있어야 합니다
- 커스텀 도메인을 연결할 계획이라면 **도메인 등록업체**(도메인의 DNS 레코드를 관리하는 웹사이트)에 로그인할 수 있어야 합니다

## 퍼널 템플릿 선택하기

템플릿을 사용하면 바로 편집할 수 있는 준비된 레이아웃으로 시작할 수 있습니다.

![퍼널 템플릿 선택 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155067651511/original/yZyMtGrz2zkFI2kZfn3cMOpj1O0JnJis6A.png?1774392856)

**필터(Filters)**, **검색창**, **정렬 드롭다운**을 사용하여 원하는 목적에 맞는 템플릿을 찾으세요.

## 새 퍼널 시작하기

새 퍼널을 만들고 구축 옵션을 선택합니다.

![새 퍼널 버튼 위치](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155067651533/original/FaGeKY5d9jE1PhEgguVsFrVxgci82RemPw.png?1774392935)

퍼널 페이지에서 우측 상단의 **New Funnel(새 퍼널)** 버튼을 클릭하세요.

![퍼널 생성 옵션 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155067651619/original/E2cRuht-kxrUYfxDSIq2BoKwx-cD3_xsCA.png?1774393384)

"Create New Funnel(새 퍼널 만들기)" 창에서 **From templates(템플릿에서)** 을 선택한 후 **Continue(계속)** 버튼을 클릭하세요.

## 계정에 템플릿 추가하기

템플릿을 선택한 후 선택 내용을 확인하고 편집기에 로드합니다.

![템플릿 미리보기 확인](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155067651642/original/iqEIYk9v9uG1YePv_cormxV6uEj4K5OtgQ.png?1774393485)

템플릿 미리보기 페이지를 검토한 후 우측 상단의 **Continue(계속)** 버튼을 클릭하세요.

![템플릿 로딩 중](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155067651659/original/S5NvJ02j2DK6-3f9lxrry9rT7cgUBs96fA.png?1774393569)

템플릿이 로딩되는 동안 창을 열어둔 상태로 기다리세요. 로딩이 완료되면 퍼널 편집기가 열리며 커스터마이징하고 저장할 수 있습니다.

## 새 폼 만들기

폼은 방문자의 정보(이름, 이메일, 전화번호 등)를 수집합니다. 폼 영역에서 새 폼을 만들 수 있습니다.

![폼 추가 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155067651809/original/Q3Ugea6fQAxrSq7_SYK8vut4z0L7xw9SEw.png?1774393853)

폼 페이지에서 우측 상단의 **Add Form(폼 추가)** 버튼을 클릭하여 폼 만들기를 시작하세요.

폼을 만든 후 필요에 따라 필드와 스타일을 조정한 다음 폼을 **저장**하세요.

![기존 폼 목록](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155067652035/original/q5GrTunOg99CjZGKMjfLqFWZOpOQzxa9qg.png?1774394214)

## 커스텀 도메인 연결하기

커스텀 도메인을 사용하면 직접 관리하는 URL로 퍼널을 발행할 수 있습니다.

![도메인 연결 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155067652200/original/rNP2Oap6vELGtTIyWJWiWW1hUhLcO7kAHw.png?1774394460)

도메인 페이지에서 우측 상단의 **Connect a domain(도메인 연결)** 버튼을 클릭하여 시작하세요.

![도메인 입력](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155067652214/original/33Uan1O9Wnz2VxVOvsq-H-gCwLRAQVPuEQ.png?1774394521)

도메인 필드에 도메인을 입력한 후 **Continue(계속)** 버튼을 클릭하세요.

![DNS 레코드 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155067652329/original/C5t-5Z4Wwjpxlf1zsVWsNeEuvzfG4fFHHA.png?1774394776)

화면에 표시된 각 DNS 레코드를 도메인 등록업체의 DNS 설정에 복사하여 입력하세요. DNS 변경사항을 저장한 후 이 화면으로 돌아와서 **Verify records(레코드 확인)** 버튼을 클릭하세요.

확인이 성공하면 도메인 연결 과정을 완료하고, 퍼널을 발행할 때 연결된 도메인을 선택하세요.

## FAQ

**Q: 퍼널은 무엇에 사용되나요?**

퍼널은 방문자를 집중된 흐름으로 안내하여 연락처 정보를 수집하거나 특정 다음 행동으로 유도하는 데 사용됩니다.

**Q: 반드시 템플릿을 사용해야 하나요?**

아니요. 템플릿은 선택사항입니다. 하지만 바로 편집할 수 있는 레이아웃을 제공하므로 더 빠르게 시작할 수 있습니다.

**Q: 새로 만드는 대신 기존 폼을 사용할 수 있나요?**

네. 폼 목록에서 기존 폼을 열어 사용하기 전에 검토하거나 업데이트할 수 있습니다.

**Q: 왜 커스텀 도메인을 연결해야 하나요?**

커스텀 도메인을 사용하면 직접 관리하는 URL로 퍼널을 발행할 수 있어 링크의 일관성을 유지하고 인지도를 높일 수 있습니다.

**Q: 도메인 확인이 실패하면 어떻게 하나요?**

각 DNS 레코드가 정확히 일치하는지 다시 확인하세요. 모든 것이 올바른 것 같다면 DNS 변경사항이 적용될 시간을 기다린 후 다시 확인해보세요.

## 도움이 필요하신가요?

퍼널 설정 중 문제가 발생하면 다음 사항을 확인해보세요:

- **표시된 메뉴 옵션을 찾을 수 없어요:** 해당 영역에 대한 접근 권한이 없을 수 있습니다. 계정 관리자에게 권한을 검토해달라고 요청하세요.

- **페이지가 로드되지 않거나 버튼이 작동하지 않아요:** 페이지를 새로고침하거나, 시크릿/개인 브라우징 창을 사용하거나, 브라우저 캐시를 삭제해보세요.

- **템플릿 로딩이 멈춘 것 같아요:** 조금 더 기다린 후 새로고침해서 다시 시도해보세요. 계속 실패하면 다른 브라우저를 사용해보세요.

- **도메인 확인이 실패해요:** DNS 레코드 유형, 호스트/이름, 값이 정확히 일치하는지 확인하세요. 또한 DNS 변경사항을 저장했는지 확인하고 DNS 업데이트가 적용될 시간을 기다려주세요.
