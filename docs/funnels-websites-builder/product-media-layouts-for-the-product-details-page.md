# 상품 상세 페이지의 상품 미디어 레이아웃

새로운 미디어 레이아웃 옵션으로 상품 페이지를 더욱 세련되고 유연하게 만들어보세요. 상품 상세 페이지에서 상품 이미지가 표시되는 방식을 선택하여 브랜드에 더 잘 맞도록 하고, 상품 세부사항을 강조하며, 쇼핑 경험을 개선할 수 있습니다. 이러한 레이아웃 옵션은 데스크톱과 모바일 기기 모두에서 더 부드러운 탐색 경험을 제공합니다.

## 상품 상세 페이지의 상품 미디어 레이아웃이란?

상품 미디어 레이아웃은 상품 상세 페이지에서 상품 이미지가 표시되는 방식을 제어합니다. 이를 통해 상점 운영자는 쇼핑객이 상품 사진을 보고, 상품 세부사항을 비교하며, 상품 갤러리를 탐색하는 방식을 더욱 유연하게 조정할 수 있습니다.

적절한 레이아웃을 선택하면 상품 페이지를 더 쉽게 탐색할 수 있고 시각적으로 더 매력적으로 만들 수 있으며, 특히 여러 이미지가 있는 상품에 효과적입니다.

상품 상세 페이지는 상품 미디어를 위한 다양한 레이아웃 스타일을 지원합니다. 이러한 레이아웃은 판매자가 상점 디자인에 맞는 방식으로 상품 이미지를 표시하는 동시에 고객이 상품 이미지와 상호작용하는 방식을 개선하는 데 도움이 됩니다.

## 상품 미디어 레이아웃의 주요 장점

적절한 상품 미디어 레이아웃을 선택하면 상품이 표시되는 방식을 개선하고 쇼핑객이 구매하기 전에 이미지를 더 쉽게 탐색할 수 있습니다.

- **더 유연한 상품 표시:** 다양한 레이아웃 스타일 중에서 선택하여 상점 디자인과 상품 유형에 맞출 수 있습니다.

- **더 나은 쇼핑 경험:** 개선된 레이아웃과 네비게이션 옵션으로 고객이 상품 이미지를 더 쉽게 탐색할 수 있도록 도와줍니다.

- **향상된 비주얼 머천다이징:** 다양한 이미지 보기 선호도를 지원하는 레이아웃으로 상품 세부사항을 더 효과적으로 보여줍니다.

- **최적화된 모바일 보기:** 모바일 기기에서 자동으로 캐러셀(Carousel)이 적용되어 더 부드러운 모바일 경험을 보장합니다.

- **하위 호환성:** 레이아웃을 수동으로 변경하지 않는 한 기존 상점의 시각적 일관성을 유지합니다.

## 모바일 레이아웃 동작

모바일 쇼핑객에게는 탐색하기 쉽고, 터치 친화적이며, 작은 화면에 최적화된 레이아웃이 필요합니다. 플랫폼은 자동으로 모바일 친화적인 미디어 환경을 적용하여 기기 전반에서 상품 이미지를 쉽게 볼 수 있도록 보장합니다.

모바일 기기에서 플랫폼은 다음 항목에 대해 자동으로 캐러셀 레이아웃을 사용합니다:

- 상품 상세 페이지
- 추천 상품 요소(Element)

주의할 중요한 동작:

- 모바일에서는 2단 레이아웃을 사용하지 않습니다.
- 데스크톱에서 선택한 미디어 레이아웃과 관계없이 모바일에서는 캐러셀이 적용됩니다.
- 이는 작은 화면에서 더 나은 보기 경험을 유지하는 데 도움이 됩니다.

- 기존 상점은 하위 호환성을 위해 기본적으로 갤러리를 계속 사용합니다.

## 자주 묻는 질문

**Q: 기존 상점이 자동으로 새로운 미디어 레이아웃으로 변경되나요?**
A: 아니요. 기존 상점은 수동으로 변경하지 않는 한 현재 레이아웃을 유지합니다.

**Q: 기존 상점은 기본적으로 어떤 레이아웃을 사용하나요?**
A: 기존 상점은 하위 호환성을 위해 계속해서 갤러리 레이아웃을 사용합니다.

**Q: 데스크톱에서 2단을 선택했는데 모바일 상품 페이지에서 캐러셀을 사용하는 이유는 무엇인가요?**
A: 플랫폼이 작은 화면에서 더 나은 보기 경험을 제공하기 위해 모바일에서는 자동으로 캐러셀을 적용하기 때문입니다.

**Q: 추천 상품 요소에서 2단을 사용할 수 있나요?**
A: 아니요. 추천 상품 요소는 갤러리와 캐러셀만 지원합니다.

**Q: 2단 레이아웃에서 네비게이션 화살표가 없는 이유는 무엇인가요?**
A: 2단 레이아웃에서는 네비게이션 화살표가 지원되지 않습니다.

**Q: 갤러리와 캐러셀 모두 네비게이션 화살표를 지원하나요?**
A: 네. 갤러리와 캐러셀 레이아웃 모두에서 네비게이션 화살표를 사용할 수 있습니다.

**Q: 새로운 레이아웃을 사용하려면 각 상품을 개별적으로 업데이트해야 하나요?**
A: 아니요. 레이아웃은 상품 상세 페이지의 상품 요소 설정에서 구성됩니다.

**Q: 모바일 기기에서 2단을 사용할 수 있나요?**
A: 아니요. 2단은 데스크톱 전용 레이아웃입니다.
