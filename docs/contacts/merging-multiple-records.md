# 여러 연락처 합치기

연락처 합치기는 데이터베이스를 정리하고, 중복 레코드를 제거하며, 중요한 정보를 하나의 마스터 레코드로 통합하는 강력한 방법입니다.

합치기가 도움이 되는 부분:

- 중복 항목 제거
- 데이터를 하나의 정확한 레코드로 통합
- 커뮤니케이션과 추적 간소화

### 
연락처 합치기는 **되돌릴 수 없는** 작업입니다. 관리자만 해당 로케이션에서 연락처를 합칠 수 있습니다.

### 1단계: 합칠 연락처 선택

- CRM의 **연락처(Contacts)** 섹션으로 이동하세요.
- 이름 옆의 체크박스를 선택하여 합치고 싶은 연락처들을 선택하세요.
- 상단 바의 **합치기(Merge)** 아이콘을 클릭하세요. 한 번에 최대 10개의 연락처까지만 합칠 수 있습니다.

![연락처 선택 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155038689655/original/msrC0ziRBWrwOgDlFUy-wpa50OzGcxwn7Q.png?1734694202)

### 2단계: 마스터 레코드 선택

- 합치기 미리보기 화면에서 **마스터 레코드(Master Record)**를 선택하세요. 마스터 레코드는 통합된 데이터가 저장될 기본 레코드입니다.
- 다른 레코드의 충돌하지 않는 모든 필드는 기본적으로 마스터 레코드의 값으로 설정됩니다.

![마스터 레코드 선택 화면](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155038676361/original/uGLPN-Upvogy9ALR5qvchypXQqliCVGCvw.png?1734686848)

### 3단계: 필드 검토 및 매핑

- 필수 필드: 연락처에 추가해야 할 이메일 주소와 전화번호가 있는 경우 마스터 레코드의 기본 이메일과 전화번호를 선택했는지 확인하세요. 기본 이메일이 없는 경우, 추가 이메일을 제거하고 하나를 기본으로 지정해야 진행할 수 있습니다.
- 충돌하는 필드(예: 전화번호, 주소)의 경우 유지하고 싶은 데이터를 선택하세요.
- **강의(Courses), 커뮤니티 그룹(Community Groups), 상품(Products)** 등의 정보는 합쳐지는 모든 레코드에서 자동으로 결합됩니다. **활동(Activities), 결제(Payments), 주문(Orders), 인보이스(Invoices), 예약(Appointments)** 등의 기타 정보는 자동으로 마스터 레코드의 활동으로 기본 설정됩니다.

### 4단계: 합치기 확인

- 합치기 미리보기에서 선택 사항을 다시 한번 확인하세요.
- **합치기(Merge)**를 클릭하여 선택한 연락처들을 하나의 레코드로 통합하세요.
- 합치기가 완료되면 마스터가 아닌 레코드의 데이터가 마스터 레코드로 통합되고, 중복된 항목은 제거됩니다.
