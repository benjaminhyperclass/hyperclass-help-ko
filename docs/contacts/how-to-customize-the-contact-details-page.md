# **연락처 상세 페이지 커스터마이징이란?**

연락처 상세 페이지 커스터마이징을 통해 관리자는 플랫폼 내에서 연락처 레코드가 어떻게 표시되는지 맞춤 설정할 수 있습니다. 커스터마이징 가능한 뷰를 통해 관리자는 패널을 구성하고, 표시할 모듈을 선택하며, 필드 표시를 제어하고, 사용자에게 특정 뷰를 배정할 수 있습니다. 이를 통해 역할별 경험(예: 영업, 지원, 관리자)을 쉽게 만들 수 있습니다.

**참고:** **관리자**만 뷰를 **생성**하고 **관리**할 수 있습니다.

## **연락처 상세 페이지 커스터마이징의 주요 장점**

- 역할별 뷰: 다른 사용자를 위해 다른 연락처 레이아웃을 생성할 수 있습니다.
- 더 나은 구성: 워크플로우 우선순위에 따라 모듈, 폴더, 필드를 배치할 수 있습니다.
- 정리된 화면: 불필요한 필드와 모듈을 숨겨서 페이지를 단순화할 수 있습니다.
- 효율성 향상: 자주 사용하는 도구를 중앙 패널이나 동적 탭에 배치할 수 있습니다.
- 더 많은 제어: 삭제 접근을 제한하고 적절한 사용자에게 적절한 뷰를 배정할 수 있습니다.

## **뷰와 접근 관리**

뷰를 통해 관리를 중앙화하면서 다른 사용자를 위한 맞춤형 레이아웃을 만들 수 있습니다. 관리자는 어떤 사용자가 어떤 레이아웃을 볼지 결정할 수 있고, 계정의 기본 경험을 설정할 수 있습니다.

- 계정당 최대 5개의 뷰를 생성할 수 있습니다.
- 관리자만 뷰를 생성하고 관리할 수 있습니다.
- 기본 뷰는 미리 설정되어 있으며 변경할 수 없습니다.
- 뷰는 사용자 관리를 통해 사용자에게 배정됩니다.
- 각 뷰에 매핑된 사용자를 확인하고 필요에 따라 재배정할 수 있습니다.

![뷰 관리 화면](https://jumpshare.com/share/Qp3mqpaUkSIctHNUHu6a+/Screen+Shot+2026-04-09+at+19.04.56.png)

## **연락처 상세 페이지 레이아웃 구조**

연락처 상세 페이지는 섹션으로 나누어져 있어서 관리자가 사용자의 작업 방식에 따라 정보를 구성할 수 있습니다. 각 영역을 이해하면 탐색과 유지 관리가 더 쉬운 레이아웃을 구축할 수 있습니다.

**중요:** **대화(Conversations) 모듈**은 **고정**되어 있으며 항상 **중앙 패널**에 **표시**됩니다.

각 뷰는 네 개의 주요 섹션으로 구성됩니다:

- 연락처 카드: 주요 연락처 세부 정보를 표시하는 상단 섹션
- 왼쪽 패널: 커스텀 필드, 표준 필드, 폴더에 사용
- 중앙 패널: 주요 모듈을 위한 기본 작업 공간
- 오른쪽 패널: 추가 모듈에 사용

![레이아웃 구조](https://jumpshare.com/share/D6u5iMTZ2zzB36gtkjsr+/2.png)

## **연락처 상세 페이지 커스터마이징 방법**

신중한 설정으로 연락처 상세 페이지를 사용하기 쉽게 만들고 각 사용자가 적절한 위치에서 올바른 정보를 볼 수 있도록 도와줍니다. 다음 단계에 따라 커스터마이징된 뷰를 생성하고 배정하세요.

#### 1단계: 연락처 상세 뷰 커스터마이징 탭으로 이동

- **연락처(Contacts)** 탭을 클릭합니다.
- 스마트 리스트(Smart Lists)로 이동합니다.
- 점 세 개를 클릭합니다.
- **설정(Settings)**을 클릭합니다.

![설정 접근](https://jumpshare.com/share/9Zq2TRzX1TaIQT9ojjMP+/3.png)

- **연락처 상세 뷰 커스터마이징(Customize Contact detail view)** 탭을 클릭합니다.

![커스터마이징 탭](https://jumpshare.com/share/vzGeX6HXTVu90Iwf1xtC+/4.png)

#### 2단계: 뷰 생성 및 관리

필요한 경우 새 뷰를 생성하고 지원할 사용자나 워크플로우에 따라 이름을 지정합니다.

참고: **계정당 최대 5개의 뷰**를 생성할 수 있으며 **관리자**만 뷰를 **생성하고 관리**할 수 있습니다.

- **+ 뷰 추가(+ Add View)** 버튼을 클릭합니다.

![뷰 추가](https://jumpshare.com/share/uYg5YsfTO2JG8yhJhUHn+/5.png)

- **뷰 이름(View Name)**을 추가합니다.
- 뷰에 사용자를 배정합니다. (**사용자**는 하나의 **뷰**에만 있을 수 있습니다. 제거되면 기본 **뷰**로 **돌아갑니다**.)
- **생성(Create)**을 클릭합니다.

![뷰 생성](https://jumpshare.com/share/TQf5kCbnzQFfKeoXYbBp+/6.png)

- 뷰가 생성되면 뷰 오른쪽 상단의 **점 세 개**를 클릭하여 다음을 수행할 수 있습니다:
  - 뷰 이름 변경
  - 사용자 관리
  - 뷰 복제
  - 뷰 삭제

![뷰 옵션](https://jumpshare.com/share/zuo9mc8AOqdoUKe2e6bq+/7.png)

- **사용자 관리(Manage Users)**에서 적절한 사용자에게 뷰를 배정합니다.
- **모든 사용자 관리(Manage all users)**를 클릭하여 사용자 배정의 전체 보기를 확인합니다.
- 각 뷰에 매핑된 사용자를 검토하고 필요에 따라 재배정합니다.

![사용자 관리](https://jumpshare.com/share/rtGRtj1hKzPg981eW4gc+/8.png)

#### 3단계: 세부 정보 페이지 커스터마이징

레이아웃 편집기를 사용하여 연락처 상세 페이지를 구성합니다.

1. 연락처 카드 커스터마이징

연락처 카드는 레코드 상단에 중요한 연락처 정보를 계속 표시되도록 합니다.

다음 항목을 추가하고 배열할 수 있습니다:

- 전화번호
- 이메일 주소
- 연락처 주소
- 담당자 및 팔로워
- 태그

중요한 세부 사항:

- **기본 전화번호**만 표시할 수 있습니다.
- **기본 이메일 주소**만 표시할 수 있습니다.
- 연락처 카드 내에서 사용 가능한 항목을 재정렬할 수 있습니다.
- 필요하지 않은 필드를 숨길 수 있습니다.
- 실수로 연락처를 삭제하는 것을 방지하기 위해 **삭제(Delete)** 버튼을 비활성화할 수 있습니다.

![연락처 카드 커스터마이징](https://jumpshare.com/share/6s12zKkKwvQmHWuYhCKU+/GIF+Recording+2026-04-09+at+19.37.45.gif)

2. 왼쪽 패널 커스터마이징

왼쪽 패널은 사용자가 연락처 정보를 빠르게 검토하고 업데이트할 수 있도록 필드 데이터를 정리하는 데 도움이 됩니다.

왼쪽 패널을 사용하여 다음을 수행할 수 있습니다:

- 커스텀 폴더와 필드 재배열
- 명확성을 위해 폴더 이름 변경
- 관련이 없는 필드 숨기기
- **추가(Add)** 버튼을 사용하여 제거된 필드 다시 추가
- 현재 표시된 필드 수 확인

![왼쪽 패널 커스터마이징](https://jumpshare.com/share/yuRl1nOaELN0wRKe1FJR+/GIF+Recording+2026-04-09+at+19.40.15.gif)

3. 동적 탭 기능 사용

동적 탭(Dynamic Tab)은 더 빠른 접근을 위해 하나의 우선순위 모듈을 전용 탭에 표시하는 데 도움이 됩니다.

- 동적 탭은 **모든 필드(All Fields)**와 **작업(Actions)** 사이에 나타날 수 있습니다.
- 지원되는 모듈을 이 탭에 배정할 수 있습니다.
- 모듈이 동적 탭에 배치되면 이전 탭 위치에서 제거됩니다.
- 빠른 접근이 필요할 때 **기회 관리(Opportunities)**나 **클라이언트 포털(Client Portal)** 같은 항목에 유용합니다.

![동적 탭](https://jumpshare.com/share/2aNbDS5007FyxK6mfNyL+/GIF+Recording+2026-04-09+at+19.42.26.gif)

4. 중앙 패널 커스터마이징

작업 탭이나 오른쪽 패널에서 모듈을 드래그 앤 드롭하여 레이아웃을 커스터마이징합니다. 대화 모듈은 중앙에 항상 고정되어 있으며 제거할 수 없습니다.

참고:
- 작업 모듈은 중앙 패널로 이동하거나 왼쪽 패널에 유지할 수 있습니다.
- 작업 모듈은 오른쪽 패널로 이동할 수 없습니다.
- 오른쪽 패널 모듈은 **오른쪽 패널에 유지하거나 중앙 패널로 이동**할 수 있습니다.

표시 옵션:

- 탭 보기(Tabs View): 스마트 리스트와 유사하게 모듈이 탭으로 표시됩니다.
- 드롭다운 보기(Dropdown View): 시각적 혼란을 줄이기 위해 모듈이 드롭다운 메뉴 아래 그룹화됩니다.

필요에 가장 적합한 표시 스타일을 선택하세요.

![중앙 패널 커스터마이징](https://jumpshare.com/share/f5H20LlNrguJ9o8hEW2f+/GIF+Recording+2026-04-09+at+19.47.22.gif)

5. 오른쪽 패널 커스터마이징

빠른 접근을 위해 모듈을 재정렬합니다.

![오른쪽 패널 커스터마이징](https://jumpshare.com/share/rONq9kvPATi7W0Zb2AiU+/GIF+Recording+2026-04-09+at+19.50.44.gif)

6. 패널 레이아웃 선택

한 번에 표시해야 하는 정보량에 가장 적합한 레이아웃 스타일을 선택합니다.

사용 가능한 레이아웃 옵션:

- 3패널 보기(Three-Panel View) (기본값): 왼쪽, 중앙, 오른쪽 패널을 표시합니다.
- 2패널 보기(Two-Panel View): 더 간결한 경험을 위해 오른쪽 패널 모듈을 중앙 패널로 병합합니다.

중요한 세부 사항:

- 2패널 보기에서는 오른쪽 패널 항목이 중앙 패널로 병합됩니다.
- 다시 전환할 때 레이아웃이 이전 배열을 복원합니다.

![패널 레이아웃](https://jumpshare.com/share/rYlfjBBQu81LnoObC0yA+/GIF+Recording+2026-04-09+at+19.54.20.gif)

#### 4단계: 레이아웃 검토 및 저장

실시간 미리보기를 검토하여 레이아웃을 확인한 다음 뷰를 **저장**합니다.

![레이아웃 저장](https://jumpshare.com/share/X1M3q7wDm9ecm6IhoWdX+/9.png)

## **연락처 상세 페이지 커스터마이징 모범 사례**

- 다른 팀과 책임에 대해 별도의 뷰를 생성하세요(예: 마케팅, 영업, 지원, 관리자).
- 우선순위가 높은 모듈을 중앙 패널에 유지하세요.
- 사용하지 않는 필드를 숨겨서 혼란을 줄이세요.
- 가장 자주 사용하는 작업이나 모듈에 동적 탭을 사용하세요.
- 연락처 데이터 보호가 우선순위라면 삭제 버튼을 비활성화하세요.

## **자주 묻는 질문**

**Q: 연락처 상세 페이지 뷰를 누가 생성하거나 관리할 수 있나요?**
관리자만 뷰를 생성, 편집, 관리할 수 있습니다.

**Q: 계정당 몇 개의 뷰를 생성할 수 있나요?**
계정당 최대 5개의 뷰를 생성할 수 있습니다.

**Q: 여러 개의 기본 뷰를 설정할 수 있나요?**
아니요. 하나의 기본 뷰만 한 번에 활성화할 수 있습니다.

**Q: 사용자에게 다른 뷰를 배정할 수 있나요?**
예. 관리자는 사용자 관리를 통해 뷰를 배정하거나 재배정할 수 있습니다.

**Q: 대화 모듈을 다른 패널로 이동할 수 있나요?**
아니요. 대화 모듈은 중앙 패널에 고정되어 있습니다.

**Q: 작업을 오른쪽 패널로 이동할 수 있나요?**
아니요. 작업은 왼쪽 패널에 유지하거나 중앙 패널로 이동할 수 있지만 오른쪽 패널로는 이동할 수 없습니다.

**Q: 2패널과 3패널 레이아웃 사이를 전환할 수 있나요?**
예. 관리자는 필요에 따라 두 레이아웃 옵션 사이를 전환할 수 있습니다.

**Q: 숨겨진 모듈이나 필드를 나중에 복원할 수 있나요?**
예. 제거된 모듈과 숨겨진 필드는 추가 버튼을 사용하여 다시 추가할 수 있습니다.
