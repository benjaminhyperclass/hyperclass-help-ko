# 연락처를 워크플로우에 일괄 추가란 무엇인가요?

자동화 추가(Add to Automation) 기능을 사용하면 연락처(Contacts) 페이지에서 연락처를 선택하고 워크플로우에 일괄로 등록할 수 있어요. 연락처가 트리거를 통해 워크플로우에 들어오기를 기다리는 대신, 발행된 워크플로우에 여러 연락처를 한 번에 수동으로 밀어 넣을 수 있습니다 - 즉시, 예약된 시간에, 또는 드립 패턴으로요.

가장 일반적인 사용 사례는 특정 속성(예: 태그)으로 연락처 목록을 필터링하고, 필터된 결과를 선택한 후, 타겟 아웃리치, 재참여 유도, 일괄 처리를 위해 워크플로우에 추가하는 것입니다.

## 연락처를 워크플로우에 일괄 추가하는 주요 장점

- **연락처를 더 빠르게 등록:** 개별적으로 등록하는 대신 한 번의 일괄 작업으로 여러 연락처를 워크플로우에 추가할 수 있어요.

- **적절한 대상을 타겟팅:** 고급 필터(태그, 필드, 기회 데이터 등)를 사용해서 필요한 연락처만 등록할 수 있어요.

- **연락처가 들어오는 시점 조절:** 즉시, 예약된 날짜/시간에, 또는 드립 모드 배치로 등록할 시점을 선택할 수 있어요.

- **세그먼트를 쉽게 재사용:** 필터된 뷰를 스마트 리스트로 저장하여 언제든지 동일한 일괄 등록을 반복할 수 있어요.

- **진행 상황과 결과 추적:** 일괄 작업(Bulk Actions) 페이지에서 상태, 타임스탬프, 통계를 모니터링하여 처리 내용을 확인할 수 있어요.

## 연락처를 워크플로우에 일괄 추가하는 방법

### **1단계:** 연락처 필터링

팁: 다른 연락처 필드(이메일, 도시, 연락처 소스, 기회 단계 등)로도 필터링할 수 있고, 여러 필터와 중첩 필터를 추가해서 등록하고 싶은 연락처를 정확히 좁혀낼 수 있어요.

- `Contacts(연락처) → Smart Lists(스마트 리스트)`로 이동하세요.

![스마트 리스트 메뉴](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155068065636/original/-HJOc2qy7BANkUe-fSLsMPw9GG__fdZUFg.jpeg?1774884143)

- **Advanced Filters(고급 필터)**를 클릭하여 필터 패널을 열어주세요.

![고급 필터 버튼](https://jumpshare.com/share/3ZPF44Y9Td1JFyQ0vA7D+/Screen+Shot+2026-03-30+at+20.53.26.png)

- **태그(Tag)**로 필터링하려면 필터 검색창에서 "**Tag**"를 검색하고 선택하세요.

- **연산자**(예: "Is")를 설정하고 필터링하고 싶은 태그 **값**(예: "follow-up")을 선택하세요. **Apply(적용)**을 클릭해서 연락처 목록을 필터링하세요. 페이지에 필터 조건에 일치하는 연락처만 표시됩니다.

![태그 필터 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155068066391/original/U4_Jikxh6MB4X4MMC42l_pMGfPjal99vqg.png?1774884407)

### 2단계: 자동화에 추가

- **워크플로우에 추가하고 싶은 연락처를 선택하세요:** **체크박스**를 사용해서 **개별 연락처**를 선택하거나, 상단의 **Select all(모두 선택)**을 클릭해서 필터된 목록의 모든 연락처를 선택할 수 있어요. 적어도 하나의 연락처가 선택되면 일괄 작업 툴바가 나타납니다.

- 일괄 작업 툴바에서 **Trigger automation(자동화 트리거)**을 클릭하세요. 자동화 추가 대화상자가 열리고, 상단에는 선택된 연락처가, 하단에는 설정 필드가 표시됩니다.

![연락처 선택 및 자동화 트리거](https://jumpshare.com/share/KXkEwWAw3OPPq7NcEzVY+/GIF+Recording+2026-03-30+at+21.00.58.gif)

### **3단계:** 자동화 설정

- 다음 필드들을 채워주세요:

**Action Name(작업 이름):** 이 일괄 작업의 이름으로, 일괄 작업 페이지의 추적 보고서에 표시됩니다. 예: "워크플로우 자동화 일괄 가져오기"

**Automation Type(자동화 유형):** 연락처를 워크플로우에 등록하려면 "**Workflow(워크플로우)**"를 선택하세요.

**Workflow(워크플로우):** 연락처를 추가하고 싶은 워크플로우를 선택하세요. 드롭다운에는 계정의 모든 발행된 워크플로우가 표시됩니다.

**Mode(모드):** 연락처가 언제, 어떻게 등록될지 선택하세요. 자세한 내용은 아래 예약 모드 섹션을 참조하세요.

- **Add to Automation(자동화에 추가)**을 클릭하세요. 처리되는 연락처 수를 보여주는 성공 확인 메시지가 나타납니다(예: "10개 연락처 자동화 추가가 진행 중입니다/예약되었습니다").

![자동화 설정 및 추가](https://jumpshare.com/share/QKYX8WMvfTjw98iOGft1+/GIF+Recording+2026-03-30+at+23.01.41.gif)

- **Check Progress(진행 상황 확인)**을 클릭해서 일괄 작업 페이지로 이동하거나, Dismiss를 클릭해서 연락처 목록으로 돌아가세요.

![진행 상황 확인](https://jumpshare.com/share/eBZl9GB92Tm27rhOGW11+/Screen+Shot+2026-03-30+at+23.02.22.png)

## 예약 모드

모드(Mode) 설정은 선택된 연락처가 언제 워크플로우에 등록될지 조절합니다. 세 가지 옵션이 있어요:

### **Send All at Once(모두 한 번에 전송)**

선택된 모든 연락처가 즉시 워크플로우에 추가됩니다. 기본 모드이며, 타이밍이 중요하지 않을 때 적합해요.

**추가 필드:** 없음.

![모두 한 번에 전송](https://jumpshare.com/share/SImCkyIrTvzXvaQEBs3T+/Screen+Shot+2026-03-30+at+23.16.29.png)

### **Send at Scheduled Time(예약 시간에 전송)**

선택된 모든 연락처가 지정한 특정 날짜와 시간에 워크플로우에 추가됩니다. 캠페인 출시나 업무 시간에 맞춰 등록을 조율할 때 유용해요.

**추가 필드:**

| 필드 | 설명 |
|------|------|
| Pick a Start Date & Time(시작 날짜 & 시간 선택) | 모든 연락처가 등록될 날짜와 시간. 계정의 시간대로 표시됩니다. |

![예약 시간에 전송](https://jumpshare.com/share/xiN3hTM2hvE2CJBPuC6T+/Screen+Shot+2026-03-30+at+23.22.40.png)

### **Send in Drip Mode(드립 모드로 전송)**

연락처가 한 번에 모두가 아니라 일정 기간에 걸쳐 배치별로 등록됩니다. 워크플로우에 과부하를 주지 않거나 커뮤니케이션을 분산시키고 싶을 때, 대용량 연락처 목록에 등록을 단계적으로 진행하기에 이상적이에요.

**추가 필드:**

| 필드 | 설명 |
|------|------|
| Pick a Start Date & Time(시작 날짜 & 시간 선택) | 첫 번째 배치의 연락처가 등록될 날짜와 시간 |
| Number of Messages per Batch(배치당 메시지 수) | 각 배치에 등록할 연락처 수 |
| Repeat after(반복 간격) | 배치 간의 간격. 숫자와 단위(일, 시간, 분)를 설정하세요 |

![드립 모드로 전송](https://jumpshare.com/share/edOlM7ALY7k6eenAT0J8+/Screen+Shot+2026-03-30+at+23.29.56.png)

## 진행 상황 추적

제출 후 일괄 작업 페이지에서 일괄 작업을 추적할 수 있어요. `Contacts(연락처) → Bulk Actions(일괄 작업)`로 이동해서 모든 일괄 작업 요청을 확인하세요.

**일괄 작업 페이지에는 다음 정보가 표시됩니다:**

- **Action Label(작업 라벨):** 작업을 만들 때 제공한 이름
- **Operation(작업):** 자동화 유형(예: Workflow)
- **Status(상태):** 작업이 진행 중인지, 예약됨인지, 완료됨인지
- **User(사용자):** 일괄 작업을 시작한 사람
- **Created / Completed(생성됨 / 완료됨):** 작업이 생성되고 완료된 타임스탬프. 계정 시간대로 표시
- **Statistics(통계):** 통계 아이콘을 클릭하면 작업 결과의 상세 분석을 볼 수 있어요

일괄 작업 페이지에서 날짜 범위, 상태, 작업 유형, 사용자별로 필터링해서 특정 요청을 빠르게 찾을 수 있어요.

![일괄 작업 페이지](https://jumpshare.com/share/QN7nkdxf16ewjLGNWkrc+/Screen+Shot+2026-03-30+at+23.32.56.png)

## 자주 묻는 질문

**Q: 태그 외에 다른 필드로도 연락처를 필터링할 수 있나요?**

네. 고급 필터 패널은 모든 연락처 필드(이메일, 도시, 연락처 소스, 생성 날짜, 기회 단계 등)와 커스텀 필드로 필터링을 지원합니다. 여러 필터와 중첩 필터를 결합해서 등록하고 싶은 연락처를 정확히 타겟할 수 있어요.

**Q: 워크플로우가 발행되어야 하나요?**

네. 발행된 워크플로우만 워크플로우 드롭다운에 나타납니다. 이 기능을 사용하기 전에 워크플로우가 발행되었는지 확인하세요.

**Q: 예약 시간에 전송과 드립 모드로 전송의 차이점은 무엇인가요?**

예약 시간에 전송은 선택된 모든 연락처를 선택한 하나의 날짜와 시간에 등록합니다. 드립 모드로 전송은 정기적인 간격으로 연락처를 배치별로 보내서 등록을 단계적으로 진행합니다 - 며칠 또는 몇 시간에 걸쳐 커뮤니케이션을 분산시킬 때 유용해요.

**Q: 일괄 작업의 진행 상황을 어디서 추적할 수 있나요?**

연락처 → 일괄 작업으로 이동하세요. 현재 상태, 타임스탬프, 통계 분석과 함께 모든 일괄 작업 요청을 볼 수 있어요.

**Q: 스마트 리스트 필터를 저장해서 향후 일괄 가져오기에 재사용할 수 있나요?**

네. 고급 필터를 적용한 후, 필터된 뷰를 스마트 리스트로 저장할 수 있어요. 다음에 동일한 세그먼트를 등록해야 할 때는 해당 스마트 리스트를 열고, 연락처를 선택한 후, 자동화를 트리거하기만 하면 돼요.
