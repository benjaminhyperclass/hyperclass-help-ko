## 데스크톱 앱 설치하기 (macOS, Windows & Linux)

아래에서 사용하는 운영 체제를 선택하세요. 각 과정은 짧고 직관적이며, 4단계를 순서대로 따라하시면 됩니다.

### **macOS**

- **설치 파일 다운로드** - 위의 macOS **(.dmg)** 링크를 사용하세요.

![macOS 다운로드](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155065819711/original/4bLNJXdlLOiVdCBuwACiaN2PxXrFgBoo3w.png?1772133049)

- **설치** - .dmg 파일을 열고 LeadConnector를 응용 프로그램(Applications) 폴더로 드래그하세요.

![macOS 설치](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155065819705/original/KizKCpox9u7Qhz7aY7HVSLLinBHUOO4j5Q.png?1772133020)

- **LeadConnector 열기** - **응용 프로그램**, **런치패드**, 또는 **Spotlight**에서 실행하세요.

- **로그인** - 데스크톱 앱에서 **Sign In** 버튼을 클릭하세요. 기본 브라우저가 열립니다. 해당 브라우저에 이미 웹앱에 로그인되어 있다면 자동으로 데스크톱 앱으로 돌아옵니다. 그렇지 않다면 브라우저에서 로그인을 완료하면 앱으로 돌아갑니다.

### **Windows**

- **설치 파일 다운로드** — 위의 Windows **(.exe)** 링크를 사용하세요.

![Windows 다운로드](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155065819763/original/luLp7xjyLjzJNEiHMKN3xFB5iRjoNKwBxA.png?1772133132)

- **설치** - .exe 파일을 열고 설치 마법사를 완료하세요.

- **LeadConnector 열기** — **시작 메뉴**에서 실행하세요.

- **로그인** - 데스크톱 앱에서 **Sign In** 버튼을 클릭하세요. 기본 브라우저가 열립니다. 해당 브라우저에 이미 웹앱에 로그인되어 있다면 자동으로 데스크톱 앱으로 돌아옵니다. 그렇지 않다면 브라우저에서 로그인 정보를 입력한 후 앱으로 돌아갑니다.

### **Linux**

- 위 링크에서 사용하는 배포판과 아키텍처에 맞는 Linux 설치 파일을 다운로드하세요.

- 시스템의 표준 설치 과정을 사용하여 설치하세요.

- 애플리케이션 메뉴에서 LeadConnector를 실행하세요.

![Linux 실행](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155068382385/original/uosg1WQnTdjwFYxvzHJKUQOctaJjZAr16A.png?1775148088)

- Sign In을 클릭하여 기본 브라우저에서 로그인을 완료한 후 데스크톱 앱으로 돌아오세요.

![Linux 로그인](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155068382391/original/QxUr8-LZ1kbqd4LBCdS5xdNHtz4LsMOjWg.png?1775148101)

## 자주 묻는 질문

**Q: 데스크톱 앱이 웹앱과 다른가요?**
사용 경험과 데이터는 동일합니다. 데스크톱 앱은 웹앱을 전용 최적화 창에 패키징하여 네이티브 알림을 제공합니다.

**Q: 알림이 보이지 않으면 어떻게 하나요?**
LeadConnector에 대한 OS 레벨 알림을 활성화하고 앱 내 설정을 확인하세요.
