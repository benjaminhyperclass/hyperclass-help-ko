# 개요

### 이제 하위 계정에서 생성한 상품을 사용하여 고객에게 인보이스(청구서)를 발송할 수 있습니다.

# 새 인보이스 작성 방법

### 앱 왼쪽 상단의 세 줄 메뉴를 클릭하고 Invoicing(인보이스) 탭을 선택합니다

### "+" 버튼을 클릭해서 새 인보이스 작성을 시작합니다.

참고:

오른쪽 상단의 기어(톱니바퀴) 아이콘을 클릭하여 전역 서비스 약관/메모를 추가할 수 있습니다. 또는 세 번째 "Terms(약관)" 단계에서 개별 인보이스에 대해서도 설정할 수 있습니다:

# 인보이스 정보 편집 방법

### 1단계: 인보이스 작성을 시작할 때 먼저 고객을 추가하고 발행 날짜와 만료 날짜를 편집합니다.

![고객 정보 입력](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48232200103/original/XlcXgc_vp-qQcd0pmhVRpb3FA9kf4iSiuA.png?1655148440)

![날짜 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48232200124/original/aHwA0MLsGu5wG3t306vYyq-DyKhdSj7K7g.png?1655148448)

참고:

모바일에서는 현재 "발신자 정보", 이미지, 인보이스 번호를 편집할 수 없습니다.

### 2단계: "+Add Item(+항목 추가)"를 클릭하여 상품 탭에서 생성한 상품을 추가합니다. 여기서 새 상품을 생성할 수도 있습니다.

![항목 추가 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48232200217/original/nP8MX6o2Q86ezHVtYuuoOmHJ9_kMQOsYzw.png?1655148475)

# 인보이스 발송 방법

### 1단계: 인보이스를 미리 보기한 후 하단의 버튼을 사용하여 고객에게 발송할 수 있습니다. 왼쪽의 테스트 인보이스 또는 오른쪽의 실제 인보이스 발송을 선택합니다.

![인보이스 발송 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48232211917/original/eUFAHi8MzTBwQ9oltRYgz18nwoVz2YnSGg.png?1655153601)

### 2단계: 원하는 방식으로 이메일 및/또는 문자를 통해 인보이스를 발송할 수 있습니다.

![발송 방식 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48232200528/original/2r2lf_8g4n2oYjT3am6jWao4Axcien-ZyA.png?1655148619)

# 인보이스 상태 확인 방법

### 1단계: Invoicing(인보이스) 탭으로 이동하면 인보이스 목록을 볼 수 있으며, 초안(Draft), 만료(Due), 연체(Overdue) 상태에 따라 색상으로 구분되어 표시됩니다.

![인보이스 목록](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48232212366/original/5LszXt993j8EZNm_G1j0N9_WOVGcNzR_tg.png?1655153735)

### 2단계: 인보이스를 검색하거나 상태 및 날짜 범위로 필터링할 수도 있습니다.

![검색 및 필터](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48232214441/original/IgSeBIXUo3_-_PjRDw_Kque9IXUpJ1H2tA.png?1655154919)

참고:

모바일에는 거래(Transaction) 탭이 없습니다.
