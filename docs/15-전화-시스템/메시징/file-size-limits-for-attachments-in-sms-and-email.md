# SMS 및 이메일 첨부파일 용량 제한

**개요**

이 글에서는 Hyperclass를 통해 발송되는 SMS와 이메일의 첨부파일 용량 제한에 대해 설명합니다. 또한 대화(Conversations)에서 미디어 라이브러리 링크를 통해 대용량 파일을 쉽게 전송하고 비디오 썸네일을 자동 생성하여 참여도를 높이는 강력한 기능인 '대화에서 첨부파일 쉽게 사용하기'도 소개합니다.

## **파일 용량 제한 이해의 주요 이점**

첨부파일 제한을 알아두면 전송 실패를 방지하고 이메일과 메시지의 전달률과 참여도를 최적화할 수 있습니다.

- 대용량 파일로 인한 전송 문제 방지
- MMS 및 이메일 서비스 제공업체 규칙 준수
- 내장 미디어 라이브러리를 활용한 파일 처리 간소화
- 인터랙티브 비디오 썸네일로 열람률과 클릭률 향상
- 수작업 및 외부 파일 호스팅 서비스 의존도 감소

---

## **사용자에게 미치는 영향**

- 파일 압축이나 외부 호스팅 같은 수작업 해결 방법 감소
- 기기에서 직접 업로드로 첨부 과정 단축
- 이메일을 시각적으로 매력적이고 인터랙티브하게 만들어 참여도 향상
- 제한 사항을 자동으로 관리하여 SMS와 이메일의 원활한 전송 보장

## **SMS 첨부파일 용량 제한**

SMS 메시지는 텍스트만 포함할 수 있습니다. 이미지, 비디오, 오디오 파일 등의 첨부파일을 포함하려면 MMS를 사용해야 하며, 이는 전화번호 유형과 통신사에 따라 지원이 다릅니다. SMS에 파일을 첨부하면 자동으로 MMS로 변환됩니다. MMS에서 지원되는 파일 형식은 다음과 같습니다:

- JPEG
- PNG
- GIF

**참고:** MP3, MP4, PDF 등의 파일 형식은 통신사와 기기에 따라 작동할 수 있습니다.

Twilio에서 지원하는 미디어 유형 목록도 확인해보세요: [Twilio 지원 파일 형식](https://help.twilio.com/articles/223181188)

### 1. MMS 통신사별 파일 용량 제한

권장 모범 사례는 모든 통신사에서 성공적인 전송을 보장하기 위해 MMS 첨부파일을 500KB 미만으로 유지하는 것입니다.

| 통신사 | Long Code MMS 첨부파일 크기 | Toll-Free MMS 첨부파일 크기 | Short Code MMS 첨부파일 크기 |
|--------|---------------------------|--------------------------|----------------------------|
| AT&T   | 1.0 MB                    | 0.6 MB                   | 0.6 MB                     |
| T-Mobile | 1.5 MB                  | 0.6 MB                   | 1.0 MB                     |
| Verizon | 1.0 MB                   | 0.6 MB                   | 1.2 MB                     |

### **2. Short Code, Toll-Free, Long Code의 MMS 지원**

2022년 10월부터 Toll-Free 번호가 MMS를 지원합니다. 그 이전에는 SMS만 지원했습니다.

| 메시지 유형 | MMS 지원? | 권장 최대 크기 |
|-------------|-----------|----------------|
| Long Code (10DLC) | ✅ 예 | ~500KB – 1MB |
| Short Code | ✅ 예 | 최대 5MB |
| Toll-Free | ✅ 예 (2022년 10월부터) | 최대 5MB |

## **이메일 첨부파일 용량 제한**

이메일의 첨부파일 제한은 사용하는 이메일 서비스 제공업체에 따라 다릅니다. 이러한 제한을 초과하면 이메일이 반송되거나 전달되지 않을 수 있습니다.

### 서비스 제공업체별 파일 크기 제한

Hyperclass 이메일 작성기는 최대 20MB의 첨부파일을 허용합니다. 이 크기를 초과하는 파일은 자동으로 미디어 라이브러리에 업로드되어 클릭 가능한 링크로 전송됩니다.

| 이메일 서비스 제공업체 | 최대 첨부파일 크기 |
|----------------------|-------------------|
| Gmail | 25MB |
| Yahoo Mail | 25MB |
| Outlook.com | 10MB |
| Mailgun | 25MB |

## **대화에서 첨부파일 업로드하는 방법**

대화 작성기에서 파일을 직접 업로드하는 것이 새로운 첨부파일 워크플로우로 더욱 쉽고 유연해졌습니다. SMS나 이메일을 보낼 때 기기에서 파일을 업로드하거나 미디어 라이브러리에서 선택할 수 있습니다. 모든 작업이 메시지 작성기 내에서 이루어집니다. 이 과정으로 파일을 수동으로 압축하거나 외부 호스팅 플랫폼을 사용할 필요가 없어져 워크플로우가 간소화되고 고객 경험이 향상됩니다. 다음은 파일을 업로드하거나 첨부하는 단계별 과정입니다.

#### 1. 첨부파일 옵션 접근

먼저 대화 탭에서 대화를 열고 메시지 작성기에서 점 3개 아이콘(•••)을 클릭합니다.

![첨부파일 옵션 접근](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044663694/original/VghkbqBAuqr7F9TfukAwR8BHZYkE5mrw0w.png?1744036309)

#### 2. 첨부파일 소스 선택

점 3개 아이콘을 클릭한 후 "Attach Files(파일 첨부)"를 선택합니다. 두 가지 옵션이 표시됩니다:

![첨부파일 소스 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155044664243/original/e_BCXDXK9OyFh8A9HY6SbLrdGNs4rJ6OUg.png?1744036710)

- Upload from System(시스템에서 업로드) – 로컬 기기에서 파일 선택
- Choose from Media Library(미디어 라이브러리에서 선택) – 이전에 업로드한 파일 선택

#### 3. 대용량 파일 자동 처리

- 파일이 20MB(이메일) 또는 SMS의 경우 모든 크기를 초과하면 Hyperclass가 자동으로 미디어 라이브러리에 업로드하고 메시지에 클릭 가능한 링크를 삽입합니다.
- 이렇게 하면 크기 제한에 걸리지 않고 메시지가 전송됩니다.

**중요**: 비디오 업로드 시 자동 생성 썸네일 미디어 라이브러리를 통해 비디오 파일을 업로드하면 Hyperclass가 자동으로 GIF 썸네일을 생성하여 이메일에 표시함으로써 메시지를 더욱 매력적이고 클릭하기 쉽게 만듭니다.

## **클립보드에서 첨부파일 붙여넣기**

첨부파일 메뉴를 열지 않고도 파일을 붙여넣어 첨부할 수 있습니다.

1. 컴퓨터에서 지원되는 파일이나 이미지를 복사합니다.
2. 대화의 메시지 작성기 안을 클릭합니다.
3. **Ctrl + V** (Windows) 또는 **Cmd + V** (Mac)을 누르거나 우클릭하여 **붙여넣기**를 선택합니다.

붙여넣은 파일이 현재 메시지에 즉시 첨부되며 위에서 설명한 동일한 크기 제한 및 미디어 처리 규칙이 적용됩니다.

![클립보드에서 첨부파일 붙여넣기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155060706026/original/GL2S766DxhqHpZ2qVnlPEZTuSqpCUN0luA.gif?1765880533)

## **대용량 파일을 전송하는 방법**

- 이메일/SMS 작성기를 통해 직접 업로드합니다.
- 20MB를 초과하는 파일은 미디어 라이브러리를 통해 자동 호스팅되고 링크로 삽입됩니다.
- Hyperclass 외부에서 필요한 경우에만 클라우드 호스팅을 사용합니다.
- 인터랙티브 썸네일을 위해 미디어 라이브러리를 통해 비디오를 추가합니다.

## **자주 묻는 질문**

**Q: 20MB를 초과하는 파일은 어떻게 되나요?**
A: 미디어 라이브러리에 자동 업로드되어 클릭 가능한 링크로 전송됩니다.

**Q: 기기에서 직접 첨부파일을 업로드할 수 있나요?**
A: 네! 시스템에서 업로드하거나 미디어 라이브러리에서 찾아보기 중에서 선택할 수 있습니다.

**Q: SMS에 비디오를 첨부할 수 있나요?**
A: 네, 비디오는 전송 문제를 피하기 위해 임베드된 파일이 아닌 호스팅된 링크로 전송됩니다.

**Q: Toll-Free 번호가 MMS를 지원하나요?**
A: 네! 2022년 10월부터 Toll-Free 번호가 MMS와 첨부파일을 지원합니다.

**Q: 비디오 첨부파일이 이메일에서 미리보기를 생성하나요?**
A: 네 — 미디어 라이브러리를 통해 업로드된 비디오 파일에 대해 GIF 썸네일이 자동 생성됩니다.
