# **SMS 캠페인에서 미디어란 무엇인가요?**

SMS 캠페인의 미디어는 메시지에 이미지, 동영상 또는 파일을 첨부하는 것을 의미해요. 미디어를 포함하면 메시지는 자동으로 MMS(멀티미디어 메시징 서비스)가 되어 풍부한 콘텐츠를 보낼 수 있습니다. 일반 텍스트 SMS와 달리 MMS는 더 긴 메시지와 시각적 요소를 포함할 수 있어요. 하지만 미디어가 어떻게 표시되고 성공적으로 전송되는지는 파일 형식, 메시지 길이, 수신자의 기기와 통신사에 따라 달라집니다.

## **SMS에서 미디어 사용의 주요 장점**

메시지에 미디어를 추가하면 참여도를 크게 높이고, 브랜딩을 강화하며, 캠페인 효과를 향상시킬 수 있어요. 미디어가 어떻게 작동하는지 이해하면 매력적이면서도 제대로 전송되는 콘텐츠를 만들 수 있습니다.

- 시각적 콘텐츠로 참여도 증대
- 상품 이미지나 동영상으로 프로모션 지원
- SMS를 MMS로 전환하여 더 풍부한 메시지 전송
- SMS의 160자 제한 우회
- 다운로드 가능한 파일을 링크로 호스팅 및 공유
- 클릭률과 전환율 향상

## **MMS vs SMS: 미디어를 추가하면 무엇이 달라질까요?**

SMS와 MMS의 차이점을 이해하면 형식 문제와 전송 실패를 피할 수 있어요. 미디어를 첨부하면 메시지는 MMS가 되어 일반 SMS와는 다른 규칙을 따르게 됩니다.

미디어가 메시지에 첨부되면 Hyperclass는 자동으로 SMS를 MMS로 전환해요. 이 전환으로 SMS의 160자 제한이 없어지고 시각적 요소를 포함할 수 있지만, 파일 형식, 크기, 통신사 호환성 등의 새로운 제약이 생겨요.

## **iPhone과 Android에서 미디어가 표시되는 방식**

기기와 통신사마다 미디어 표시 방식이 다를 수 있어요. 무엇을 기대해야 하는지 알면 기기에 관계없이 깔끔하고 전문적으로 보이는 메시지를 디자인할 수 있습니다.

- **Android**: 미디어를 텍스트와 함께 인라인으로 표시. 미디어와 메시지를 하나의 콘텐츠 블록으로 보여줌
- **iPhone**: 미디어를 텍스트 위나 아래에 별도 메시지로 표시할 수 있음. iPhone은 이미지와 메시지를 별도의 말풍선으로 분할할 수 있음

이러한 차이점은 기기별 특성이므로 발신자가 제어할 수 없어요.

## **글자 수, 미디어, 메시지 분할**

글자 수와 미디어 형식은 메시지 전송과 형식에 영향을 줍니다. 전송 문제를 피하기 위해 알아야 할 내용이에요. 미디어가 없는 메시지는 SMS로 전송되며 160자로 제한돼요. 이를 초과하면 메시지가 여러 세그먼트로 분할됩니다.

- 미디어 포함(MMS): 긴 메시지가 허용되지만 미디어 크기 제한 내에서 유지해야 함
- 미디어 없음(SMS): 긴 메시지는 160자씩 여러 세그먼트로 분할됨

중요: 메시지에 미디어와 너무 많은 텍스트가 포함되면, 특히 통신사나 기기가 해당 메시지 형식을 지원하지 않는 경우 전송에 실패할 수 있어요.

## **일반적인 문제와 전송 위험**

미디어 메시지는 파일 크기, 기기 유형, 통신사 정책에 따라 예측하기 어렵게 작동할 수 있어요. 주의해야 할 가장 일반적인 문제들이에요:

- 큰 파일은 제거되거나 메시지 전송 실패를 일으킬 수 있음
- iPhone과 Android에서 메시지 렌더링이 다를 수 있음
- 일부 통신사는 플래그된 콘텐츠나 크기 제한을 초과한 미디어가 있으면 MMS를 차단할 수 있음
- 직접 전송할 수 없는 파일은 호스팅되어 클릭 가능한 링크로 공유됨

팁: 목록에 발송하기 전에 항상 미리보기를 하거나 본인의 휴대폰으로 테스트 발송해보세요.

## **자주 묻는 질문**

**Q: 메시지가 미디어가 포함된 하나의 메시지가 아닌 여러 개의 문자로 전송되는 이유는 무엇인가요?**
메시지에 지원되는 미디어가 포함되지 않거나 제한을 초과하면 하나의 MMS가 아닌 여러 개의 SMS 메시지로 전송돼요.

**Q: 문자 메시지에 파일을 첨부할 수 있나요?**
직접적으로는 안 돼요. 파일은 미디어 라이브러리(Media Library)에 업로드되어 클릭 가능한 링크로 공유되며, 메시지 자체에 삽입되지는 않아요.

**Q: 메시지가 어떻게 보일지 확인하는 가장 좋은 방법은 무엇인가요?**
"미리보기(Preview)" 기능을 사용하거나 Android와 iPhone 기기 모두에 테스트 메시지를 보내서 메시지가 어떻게 렌더링되는지 확인해보세요.

**Q: 모든 휴대폰에서 메시지가 똑같이 보이나요?**
아니에요. 메시지 형식과 이미지 배치는 기본 메시징 앱과 통신사 설정에 따라 Android와 iPhone 간에 다를 수 있어요.
