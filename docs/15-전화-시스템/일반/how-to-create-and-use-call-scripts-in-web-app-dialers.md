# 웹 & 앱 다이얼러에서 통화 스크립트 만들고 사용하는 방법

이 가이드는 Hyperclass의 웹 다이얼러, 앱 다이얼러, Lead Connector 앱에서 통화 스크립트를 만들고 사용하는 방법을 보여드립니다. 통화 스크립트는 상담사가 구조화된 스크립트를 따라 일관된 메시지를 전달하고 생산성을 높일 수 있도록 도와주며, 탭을 전환하거나 메모를 찾을 필요 없이 통화에 집중할 수 있게 해줍니다.

참고: 이 기능은 Hyperclass 웹 다이얼러, Hyperclass 앱 다이얼러, Lead Connector 앱에서 사용할 수 있습니다.

# 통화 스크립트란?

통화 스크립트는 Hyperclass 웹 다이얼러, 앱 다이얼러, Lead Connector 앱 다이얼러에서 상담사에게 실시간 대화 가이드를 제공하는 새로운 기능입니다. 이 도구는 모든 통화에서 팀에게 체계적인 정보를 제공하여 중요한 자격 요건을 놓치지 않고, 이의 제기를 원활하게 처리하며, 컴플라이언스 요구 사항을 일관되게 충족할 수 있도록 지원합니다.

## 통화 스크립트의 주요 이점

통화 스크립트는 효과적인 커뮤니케이션을 지원하고 교육 시간을 단축하는 명확하고 반복 가능한 대화 가이드를 제공하여 팀의 성과를 향상시킵니다.

- 상담사에게 고객 발굴과 이의 처리를 위한 신뢰할 수 있는 프레임워크를 제공합니다.
- 신입 직원을 위한 일관된 가이드라인을 제공하여 교육을 가속화합니다.
- 수동 메모 작성이나 리소스 검색을 없애 시간을 절약합니다.
- 향상된 자격 심사와 구조화된 대화 흐름을 통해 전환율을 높입니다.
- 반복 가능하고 고품질의 상호 작용으로 컴플라이언스를 향상시키고 고객 신뢰를 구축합니다.

## 통화 스크립트 만드는 방법

통화 스크립트 만들기는 기존 워크플로우에 가이드 통화 지원을 통합하도록 설계된 간단한 과정입니다. 상담사가 적절한 시기에 올바른 스크립트에 액세스할 수 있도록 다음 단계를 따라해주세요:

- 하위 계정에 로그인하고 설정(Settings)을 클릭합니다.
![](https://jumpshare.com/share/rCL4O7JdW3kUkHJbD0QY+/Screen+Shot+2025-07-31+at+9.06.14+PM.png)

- Phone Systems(전화 시스템)으로 이동하고 Voice(음성)를 클릭합니다.
![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155062377171/original/S3hbiAE9QBep9UgxIogosdpjCvCVVzl7XQ.png?1768060143)

- Call Scripts(통화 스크립트)를 클릭하고 Add Scripts(스크립트 추가) 버튼을 클릭합니다.
![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155062377191/original/WGeW8B1xLQEkKvgN2S4VtmujknPdjIEUoA.png?1768060204)

- 원하는 내용을 입력하여 새로운 통화 스크립트를 만들고 저장(Save)을 클릭합니다.

## 웹 & 앱 다이얼러에서 통화 스크립트 사용하는 방법

상담사는 다이얼러 내에서 직접 실시간 통화 중에 통화 스크립트에 액세스할 수 있어서 전체 워크플로우를 한 곳에서 유지할 수 있습니다. 이를 통해 새 탭을 열거나 외부 문서를 참조할 필요 없이 대화에 집중하고 일관성을 유지할 수 있습니다.

통화 스크립트 사용 방법:

- 지원되는 다이얼러 중 하나를 사용하여 전화를 걸거나 받습니다.
- 다이얼러 인터페이스에서 Scripts(스크립트) 버튼을 클릭합니다.

## 자주 묻는 질문

**Q: 어떤 앱 버전에서 통화 스크립트를 지원하나요?**
통화 스크립트는 Hyperclass v3.103.4 이상 및 LeadConnector v3.103.4 이상에서 사용할 수 있습니다.

**Q: 통화 스크립트를 만든 후에도 편집할 수 있나요?**
네, 통화 스크립트는 웹 인터페이스에서 언제든지 업데이트하여 대화 가이드를 개선할 수 있습니다.

**Q: 화이트라벨 앱에서도 통화 스크립트 기능을 사용할 수 있나요?**
네, 다만 화이트라벨 앱의 업데이트는 별도로 요청해야 합니다.

**Q: 로케이션의 모든 사용자가 통화 스크립트를 볼 수 있나요?**
네, 로케이션의 모든 사용자는 통화 중에 사용 가능한 모든 스크립트 목록에 액세스할 수 있습니다.

**Q: 통화 스크립트에 텍스트 서식을 지정하거나 링크를 추가할 수 있나요?**
현재 통화 스크립트는 일반 텍스트만 지원합니다. 서식 지정과 하이퍼링크는 지원되지 않습니다.

**Q: 스크립트 업데이트가 즉시 반영되나요?**
네, 스크립트에 대한 모든 업데이트는 다음에 웹 다이얼러를 열 때 모든 사용자에게 자동으로 적용됩니다.
