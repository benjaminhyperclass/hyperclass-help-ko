# **커스텀 음성사서함이란**

시스템에는 미리 녹음된 메시지가 있고 텍스트 음성 변환 기능도 있습니다. 커스텀 음성사서함은 사용자가 직접 오디오를 녹음해서 파일을 업로드하는 것입니다. 이 파일은 시스템 파일 대신 음성사서함 메시지로 재생되거나, 다른 사람의 음성사서함(보통 고객)에 메시지를 남기는 음성사서함 드롭에 사용될 수 있습니다.

## **온라인 오디오 품질 조정**

모든 통신사에서 작동하도록 오디오 품질을 낮춰주세요. "online audio converter"를 검색하거나 다음 웹사이트 중 하나를 사용해보세요:

- [https://online-audio-converter.com/](https://online-audio-converter.com/)
- [https://onlineaudioconverter.com/](https://onlineaudioconverter.com/)

가장 작은 크기 / 이코노미 / 64 kbps 옵션을 선택하세요.

# 데스크톱 오디오 애플리케이션

오디오를 녹음하고 변환하는 데스크톱 애플리케이션을 사용할 수 있습니다.

- [https://www.audacityteam.org/](https://www.audacityteam.org/)
- [https://lame.sourceforge.io/](https://lame.sourceforge.io/)
- [https://obsproject.com/](https://obsproject.com/)

### **Audacity 사용법**

1. 오디오 파일을 Audacity로 드래그 앤 드롭하세요

2. 좌측 하단에서 프로젝트 레이트를 8000으로 낮추세요

3. Tracks → Resample... → 8000 → OK

4. File → Export Audio... → Save As에서 파일명 입력 → Format: Other uncompressed files → Options → Header: WAV → Encoding: U-Law → OK → Save → OK
