# 모바일 앱에서 수신 전화란?

모바일 앱의 수신 전화 기능을 사용하면 Hyperclass 전화번호로 걸려온 비즈니스 전화를 Lead Connector 앱을 통해 모바일 기기에서 직접 받을 수 있습니다. 전화가 성공적으로 전달되려면 플랫폼 설정(웹 앱 내부)과 기기 설정(모바일 앱 내부)을 모두 올바르게 구성해야 합니다.

**참고:** 수신 전화가 작동하려면 하위 계정 내에서 전화번호를 구매하고 사용자에게 할당해야 합니다.

## 모바일 앱 수신 전화의 주요 장점

모바일 앱을 통해 전화를 받으면 고객에게 접근 가능하고 반응성 있게 응답할 수 있는 강력한 장점들을 제공합니다.

- 데스크톱에서 떨어져 있을 때도 중요한 고객 전화를 놓치지 않습니다.
- 모바일 기기에서 직접 수신 전화를 처리할 수 있습니다.
- 원격 근무와 이동 중 커뮤니케이션을 지원합니다.
- 여러 위치에 있는 직원들의 통화 관리를 단순화합니다.
- 전용 비즈니스 라인을 사용해서 비즈니스의 전문적인 이미지를 향상시킵니다.
- 부재중 전화 자동 문자 발송 같은 자동화 기능과 통합하여 고객 후속 관리를 개선합니다.

## 모바일 앱에서 수신 전화를 활성화하는 방법

수신 전화를 설정하려면 웹 앱에서 팀원(Staff Member) 설정을 구성하고 모바일 기기의 권한을 업데이트해야 합니다. 팀원들은 전화를 모바일 앱으로 전달하도록 프로필이 구성되어 있어야 합니다.

- 메인 메뉴에서 Settings(설정)으로 이동합니다.
- My Staff(내 팀원)를 선택합니다.
- 활성화하려는 특정 팀원을 선택하고 이름 옆의 **Edit(편집)** 버튼을 클릭합니다.
- Call and Voicemail Settings(전화 및 음성메일 설정) 탭을 클릭합니다.
- Inbound Number(수신 번호)를 할당합니다. 사용자에게 전용 직통 번호가 필요한 경우 드롭다운 목록에서 하나를 선택합니다. 그렇지 않으면 전체 통화 분배 설정에 따라 전화가 라우팅됩니다.
- **Forward Calls to(전화 전달 대상)**라는 섹션을 찾고 Mobile app(모바일 앱) 박스에 체크합니다.
- 변경사항을 적용하려면 우측 하단의 파란색 Save(저장) 버튼을 클릭합니다.

# 일반적인 문제 해결 팁

수신 전화가 모바일 앱에 도달하지 않는 경우 다음 영역을 확인해보세요:

- 위에 표시된 대로 팀원 설정에서 "Send calls to mobile app(모바일 앱으로 전화 전송)"이 활성화되고 저장되었는지 확인합니다.
- Hyperclass 모바일 앱이 기기에서 필요한 모든 권한을 가지고 있는지 확인합니다.
- 앱에 대한 배터리 최적화가 비활성화되어 있는지 확인합니다(Android만 해당).
- 알림이 활성화되어 있고 앱이 백그라운드에서 실행될 수 있도록 허용되어 있는지 확인합니다.

# 자주 묻는 질문

**Q: "Send calls to mobile app(모바일 앱으로 전화 전송)"을 활성화하지 않으면 어떻게 되나요?**
사용자는 모바일 앱에서 수신 전화를 받을 수 없습니다. 전화가 모바일 앱으로 전달되려면 이 박스에 체크가 되어 있어야 합니다.

**Q: 전화를 받으려면 앱을 계속 열어두어야 하나요?**
아니요. 하지만 앱이 백그라운드에서 실행될 수 있도록 허용되고 알림이 활성화되어 있어야 합니다.

**Q: 전화를 놓치면 자동으로 문자 메시지를 보낼 수 있나요?**
네, 부재중 전화 자동 문자 발송(Missed Call Text Back) 기능을 사용하여 리드에게 SMS로 자동 후속 관리할 수 있습니다.

**Q: 설정했는데도 전화를 받을 수 없는 이유는 무엇인가요?**
팀원 설정에서 "Save(저장)" 버튼을 클릭했는지, 모바일 권한이 올바르게 활성화되어 있는지 다시 확인해보세요.

**Q: 여러 기기에 로그인하면 모든 기기에서 전화를 받을 수 있나요?**
같은 사용자 계정에 로그인한 최대 10대의 기기에서 동시에 수신 전화를 받을 수 있습니다. 어느 한 기기에서 전화를 받으면 다른 모든 기기에서 자동으로 통화가 끊어집니다. 이 기능은 동일한 사용자 계정이 여러 기기에 로그인한 경우에만 적용됩니다. 여러 사용자가 로그인을 공유하는 것은 지원하지 않습니다.

**Q: 여러 하위 계정의 구성원이고 수신 전화가 활성화되어 있으면 어떻게 되나요?**
마지막으로 전환한 하위 계정의 수신 전화만 받을 수 있습니다. 모바일 앱이 완전히 종료된 후 다시 열리면 마지막 하위 계정을 기억하고 해당 계정의 전화를 계속 라우팅합니다. 수신 전화가 활성화되지 않은 하위 계정으로 전환하면 어떤 하위 계정의 전화도 더 이상 받을 수 없습니다.

**Q: 두 통의 전화가 오고 하나를 받으면 어떻게 되나요?**
여러 기기에서 수신 전화가 울리는 경우, 어느 한 기기에서 전화를 받으면 다른 모든 기기에서 자동으로 통화 시도가 끊어집니다.
