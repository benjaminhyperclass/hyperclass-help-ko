# Twilio 전화번호 삭제/리셋하기

## 개요
이 문서에서는 계정에서 LC Phone 전화번호를 삭제하는 과정을 설명합니다. 

**중요:** 이 단계는 자체 Twilio 계정을 사용하는 하위 계정에만 적용됩니다. 이 방법으로 LC Phone 전화번호를 삭제하면 해당 번호가 영구적으로 해제되어 다른 사람이 구매할 수 있게 됩니다. 이 작업의 영향을 완전히 이해할 수 있도록 진행하기 전에 안내사항을 주의 깊게 읽어주세요.

---

## LC Phone 전화번호 삭제 단계:

### 1. 하위 계정 접근하기
- 하위 계정으로 이동한 후, 페이지 왼쪽 하단에 있는 Settings(설정)를 클릭하세요.

![하위 계정 설정 접근](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48253459059/original/Acko3r7gU1EXhaGVlciv1f1vaEuvREsFnw.png?1664293797)

### 2. Phone Numbers(전화번호) 선택하기
- Phone Numbers(전화번호)를 클릭하여 하위 계정에 연결된 모든 전화번호를 확인하세요.

![전화번호 메뉴 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48253459271/original/jvanFTE46EjlAIWU9ko4xiqzlAQqQCquyA.png?1664293843)

### 3. 전화번호 제거하기
- 삭제하려는 번호를 찾아서 옆에 있는 휴지통 아이콘을 클릭하여 번호를 제거하세요.

![전화번호 삭제 아이콘](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48253459678/original/U9UrwbbmZBjoq9Azkkw7-1W_12xcTqVfXQ.png?1664293940)

### 4. 삭제 확인하기
- 삭제를 확인하는 프롬프트가 나타납니다. 입력 필드에 `Delete`를 입력한 후 `Confirm Delete`를 클릭하여 진행하세요.

![삭제 확인 대화상자](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48253459269/original/tFFuLThjDV14VAJMdE1MUxnzhknhEefziA.png?1664293843)

### 5. 웹훅 유지 또는 리셋 결정하기
- **Twilio 번호의 웹훅을 유지하고 Twilio 하위 계정에 보관하려면**: `NO`를 클릭하세요. 이렇게 하면 번호가 해제되지 않습니다.
- **번호를 해제하려면**: 입력 필드에 `Release`를 입력한 후 `Confirm Release`를 클릭하세요.

![웹훅 유지/해제 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48253459925/original/FC7mICyxO7QBH4d0nXCRY1eFPT7G1h_uGQ.png?1664294007)

### 6. 해제된 번호 복원하기
- 번호를 해제하지 않았지만 Twilio 웹훅을 리셋하려는 경우, `Add`를 클릭하여 Twilio 번호를 계정에 다시 할당하세요.

![번호 복원하기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48253460342/original/FEqAA2F3CilifjU1Y_KY9pX0A0C-Lg-7Mw.png?1664294097)
