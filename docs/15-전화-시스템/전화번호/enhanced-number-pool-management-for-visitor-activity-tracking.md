# 향상된 번호 풀 관리란 무엇인가요?

향상된 번호 풀 관리는 하이퍼클래스 방문자 추적 시스템 내에서 전화번호를 관리하는 새로운 도구와 제어 기능 세트를 의미합니다. 이러한 개선 사항을 통해 번호 풀과 일반 전화번호 목록 간에 전화번호를 이동, 제거, 삭제, 할당할 수 있어 캠페인 관리의 유연성과 효율성이 크게 향상됩니다.

참고: 이러한 변경 사항은 현재 방문자 활동 추적을 위해 생성된 번호 풀(일반적으로 4~8개 번호)에만 적용됩니다.

## 향상된 번호 풀 관리의 주요 장점

사용자 중심의 개선 사항으로 간소화된 전화번호 관리와 향상된 캠페인 유연성을 경험하세요:

- 추가 절차나 지원 요청 없이 번호 풀 간 번호 이동
- 전체 풀을 삭제하지 않고 풀에서 번호를 제거하여 일반 목록으로 반환
- 풀의 나머지 부분을 그대로 유지하면서 풀에서 개별 번호 삭제
- 일반 목록에서 추적 풀로 직접 번호 추가
- 모든 작업에 대한 완전한 감사 로깅으로 투명성과 추적성 보장
- 일상적인 번호 관리 작업에 대한 지원 의존성 감소
- 동적 방문자 추적 캠페인을 더 효율적으로 관리

## 번호 풀 관리 기능에 접근하는 방법

새로운 기능에 접근하려면 다음 단계를 따라하세요:

### 1단계: 번호 풀로 이동

- 계정에서 Settings(설정)으로 이동하세요.

- Phone Numbers(전화번호) 탭을 클릭하세요.
![전화번호 탭 클릭](https://jumpshare.com/v/xbooWyaTGHXUqGJVBbRP+/Screen+Shot+2025-06-24+at+11.16.38+PM.png)

- Manage Numbers(번호 관리) 아래의 Number Pools(번호 풀) 옵션을 선택하세요.
![번호 풀 옵션 선택](https://jumpshare.com/v/AOrsHtBLEZCIUIpxaetz+/Screen+Shot+2025-06-24+at+11.18.31+PM.png)

### 2단계: 번호 풀 정보에 접근

- 관리하려는 번호 풀 옆의 점 세 개를 클릭하세요.
- Number Pool Info(번호 풀 정보)를 클릭하세요.
![번호 풀 정보 접근](https://jumpshare.com/v/VwtbxhdgqpsHUZG0tS0m+/Screen+Shot+2025-06-24+at+11.23.22+PM.png)

참고 (+1 번호에 대한 신원 인증): 풀 구매/보충에 미국/캐나다/푸에르토리코(+1) 번호가 포함되어 있고 하위 계정이 Persona 인증을 완료하지 않은 경우, 구매를 완료하기 전에 인증하라는 메시지가 표시될 수 있습니다. 인증 후에는 향후 +1 풀 구매/보충이 Persona를 반복하지 않고 진행될 수 있습니다.

## 새로운 번호 풀 관리 기능들

Number Pool Info(번호 풀 정보) 모달을 열면 다음과 같은 새로운 관리 옵션을 사용할 수 있습니다:

### 1. 다른 번호 풀로 이전

- 다른 풀로 이동하려는 번호를 찾으세요.
- Transfer to a Different Number Pool(다른 번호 풀로 이전) 버튼을 클릭하세요.
![다른 번호 풀로 이전](https://jumpshare.com/v/xuJ9PxKIhH7xyc7F9GdS+/Screen+Shot+2025-06-24+at+11.37.31+PM.png)

- Select Destination(대상 선택) 드롭다운에서 번호를 이전하려는 풀을 선택하세요.
- Move(이동)를 클릭하세요.
![대상 풀 선택 및 이동](https://jumpshare.com/v/lnfFvfGmDq8jGYLK3wfQ+/Screen+Shot+2025-06-24+at+11.40.04+PM.png)

### 2. 전화번호로 이동

- 원하는 풀에서 번호를 찾으세요.
- Move to Phone Numbers(전화번호로 이동) 버튼을 클릭하세요.
- 번호가 삭제되지 않고 일반 번호 목록으로 반환됩니다.
![전화번호 목록으로 이동](https://jumpshare.com/v/hY5nQ3IP9Crlb4Jx82Ug+/Screen+Shot+2025-06-24+at+11.50.31+PM.png)

### 3. 번호 제거 및 삭제

- 삭제하려는 번호를 선택하세요.
- Remove and Delete Number(번호 제거 및 삭제) 버튼을 클릭하세요.
- 선택된 번호만 풀에서 제거되고 삭제됩니다.
![번호 제거 및 삭제](https://jumpshare.com/v/Odxh1maQuqelyd80sbqg+/Screen+Shot+2025-06-25+at+12.16.10+AM.png)

### 4. 일반 목록에서 풀에 번호 추가

- Manage Numbers(번호 관리) 아래의 Phone Numbers(전화번호)로 이동하세요.
![전화번호 메뉴로 이동](https://jumpshare.com/v/zTHFkASZ0LCCwd6A7kXM+/Screen+Shot+2025-06-25+at+12.20.39+AM.png)

- 풀에 추가하려는 번호 옆의 점 세 개를 클릭하세요.
- Add to Existing Number Pool(기존 번호 풀에 추가) 버튼을 클릭하세요.
![기존 번호 풀에 추가](https://jumpshare.com/v/aU9I6sg0FAAGKZEdv4nd+/Screen+Shot+2025-06-25+at+12.31.40+AM.png)

- Select Destination(대상 선택) 드롭다운에서 원하는 풀을 선택하세요.
- Add to Number Pool(번호 풀에 추가) 버튼을 클릭하세요.

## 감사 로깅

감사 로깅은 번호 풀 내에서 수행된 모든 작업을 기록하여 컴플라이언스와 문제 해결을 위한 투명한 기록을 제공합니다.

- 누가 번호를 이동, 제거, 또는 삭제했는지 추적
- 각 작업의 타임스탬프와 세부 정보 검토
- 분쟁 해결이나 문제 조사에 로그 활용

## 번호 풀 정리 모범 사례

번호 풀을 효과적으로 정리하면 정확한 방문자 추적과 캠페인 분석이 가능합니다.

- 명확성을 위해 캠페인, 지역 또는 사용 사례별로 번호를 그룹화
- 효율성을 유지하기 위해 사용하지 않는 번호를 정기적으로 검토하고 정리
- 쉬운 식별을 위해 설명적인 풀 이름 사용

## 자주 묻는 질문

**Q: 삭제된 번호를 풀로 복원할 수 있나요?**
A: 번호가 삭제되면 복원할 수 없습니다. 제거된(삭제되지 않은) 번호만 재할당할 수 있습니다.

**Q: 풀 간에 번호를 이동할 수 있는 횟수에 제한이 있나요?**
A: 아니요, 필요한 만큼 자주 풀 간에 번호를 이동할 수 있습니다.

**Q: 풀에서 번호를 제거하면 추적 데이터에 영향을 미치나요?**
A: 번호 제거는 풀 할당만 변경하며, 기존 추적 데이터는 삭제되지 않습니다.

**Q: 번호 풀을 누가 변경했는지 어떻게 알 수 있나요?**
A: 모든 작업은 사용자, 작업, 타임스탬프를 포함하여 감사 로그에 기록됩니다.

**Q: 풀을 삭제하면 어떻게 되나요?**
A: 풀을 삭제하면 그 안의 모든 번호가 제거됩니다. 번호를 유지하려면 풀을 삭제하기 전에 번호를 이동하거나 제거하세요.
