# 설정 수정 방법

사용자는 운영 요구사항에 따라 언제든지 이러한 설정을 조정할 수 있습니다. 변경하려면:

- 에이전시 계정에 로그인하고 설정(Settings)을 클릭합니다.
- 전화 연동(Phone Integration)을 클릭합니다.
- 계정 생성(Account Creation) 탭을 클릭합니다.
- 필요에 따라 설정을 업데이트합니다.

## 기본 설정 이해하기

### 1. 전화 시스템 자동 연결

✅ 기본적으로 활성화됨

이 설정은 신규 하위 계정이 전화 시스템에 자동으로 연결되도록 합니다. 메시징과 음성 서비스에 즉시 접근할 수 있도록 하여 수동 설정의 필요성을 제거합니다.

### 2. 재청구가 비활성화된 경우 하위 계정 사용자의 A2P 등록 허용

기본적으로 하위 계정은 재청구가 활성화된 경우에만 A2P(Application-to-Person) 등록을 제출할 수 있습니다. 이 옵션을 활성화하면 재청구가 비활성화된 경우에도 사용자가 A2P 등록을 제출할 수 있습니다. (선택 설정)

### 3. 번호 인텔리전스 자동 활성화

이 기능은 다음과 같은 향상된 전화번호 인텔리전스를 제공합니다:

- 첫 번째 SMS 발송 전 번호 유효성 검사
- 수신 스팸 전화 감지
- 전화 응답 전 알 수 없는 발신자 이름 검색

이 설정을 활성화하면 전화 커뮤니케이션의 효율성과 신뢰성이 향상됩니다. (선택 설정)

### 4. 메시징 제한

메시징 제한은 새로 생성되는 모든 하위 계정에 적용됩니다. 선택된 제한은 신규 하위 계정의 메시징 용량을 결정하여 사용 정책 준수를 보장합니다.

- 현재 제한: 메시징 램프(기본 설정)
- 사용자는 필요에 따라 이 설정을 수정할 수 있습니다.
