# 부재중 전화 왓츠앱 회신 — 개요 및 설정

부재중 전화 왓츠앱 회신(Missed Call WhatsApp Back)은 전화를 받지 못했을 때 발신자에게 왓츠앱으로 자동 메시지를 보내서 팀이 더 빠르게 재연결할 수 있도록 도와줍니다. 워크플로우 구축 없이도 작동합니다. 전화 시스템(Phone System)에서 활성화하고 비즈니스 이름이나 예약 링크 등의 변수를 포함한 친근하고 브랜드에 맞는 후속 메시지를 맞춤 설정하세요. 왓츠앱 응답률이 SMS보다 높은 지역에 이상적입니다.

## **부재중 전화 왓츠앱 회신이란?**

부재중 전화 왓츠앱 회신은 Hyperclass의 전화 시스템 기능으로, 전화가 "부재중"으로 분류된 후 연락처에게 왓츠앱 메시지를 보냅니다. 전화 후 공백을 줄여서 리드가 빠른 확인과 다음 단계를 받을 수 있도록 하여 전환율과 고객 경험을 개선합니다.

---

## **부재중 전화 왓츠앱 회신의 주요 이점**

이점을 이해하면 왓츠앱과 SMS 중 언제 사용할지, 이 기능이 리드 대응 속도와 고객 만족도에 어떤 영향을 주는지 명확해집니다.

- **즉시 참여**: 연락처가 부재중 전화 직후 적시에 왓츠앱 메시지를 받습니다.

- **워크플로우 불필요**: 빠른 설정으로 몇 분 내에 활성화 — 자동화가 필요하지 않습니다.

- **높은 가독성**: 왓츠앱은 종종 SMS보다 더 높은 열람/응답률을 보입니다.

- **브랜드 일관성**: 브랜드 톤에 맞게 문구를 맞춤 설정하고 유용한 링크를 포함할 수 있습니다.

- **개인화**: 비즈니스 이름이나 예약 링크 같은 동적 세부사항을 삽입합니다.

- **운영 효율성**: 수동 작업 감소로 놓친 콜백이 줄어들고 연락처에게 명확한 다음 단계를 제공합니다.

---

## **사전 요구사항**

사전 요구사항을 확인하면 일반적인 설정 실패와 전송 문제를 방지할 수 있습니다.

- 하위 계정에 연결된 왓츠앱 지원 전화번호 (왓츠앱이 활성화되어 있고 정상 상태여야 함)

- 기능을 설정하는 사용자는 Settings(설정) → Phone System(전화 시스템) 접근 권한이 있어야 함

- 연락처가 왓츠앱으로 연락 가능해야 함 (그렇지 않으면 왓츠앱으로 메시지 전송 불가)

- 비즈니스가 왓츠앱으로 연락처에게 메시지를 보낼 적절한 동의를 받았는지 확인

## **부재중 전화 왓츠앱 회신 설정 방법**

정확한 경로를 아는 것으로 유사한 이름의 SMS 기능과의 혼동을 줄입니다.

명확한 단계별 설정으로 기능이 올바르게 활성화되고 메시지가 브랜드에 맞게 설정된 후 실행되도록 합니다.

- Settings(설정) → Phone System(전화 시스템) → Voice(음성) → Voicemail & Missed Call TextBack(보이스메일 및 부재중 전화 문자 회신) → Missed Call Textback(부재중 전화 문자 회신)으로 이동하세요.

- Missed Call WhatsApp Back(부재중 전화 왓츠앱 회신)을 On으로 토글하세요.

- 변경사항을 저장하세요.

- 테스트 전화를 걸어서 (아래 테스트 및 검증 참조) 왓츠앱 메시지가 전송되고 예상대로 표시되는지 확인하세요.

## 기본 왓츠앱 메시지 및 개인화 변수

미리 입력된 메시지에서 시작하면 출시 속도를 높이고, 변수를 사용하면 관련성 있고 사람답게 들리는 응답을 보장합니다.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155062282164/original/iful2fnHrP6SY77_TDWTy20Jex3B409gdw.png?1767944369)

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155062282253/original/j2R_3vncouAigYf5_EdXDyMgtngdqdVQzw.png?1767944399)

- **미리 입력된 메시지:**
"안녕하세요, {{location.name}}입니다. 방금 전화를 놓친 것 같습니다. 무엇을 도와드릴까요?"

- **맞춤 설정:** 메시지 본문은 Missed Call Textback(부재중 전화 문자 회신) 영역에서 직접 편집할 수 있습니다 (WhatsApp(왓츠앱) → Templates(템플릿)에서 편집하지 않음).

- **지원되는 변수 (예시):**
  - **{agent_name}** – 사용 가능한 경우 배정된 상담원의 이름을 삽입합니다.
  - **{booking_link}** – 로케이션에 대해 설정한 예약 링크를 삽입합니다.
  - **{business_hours}** – 운영 시간을 사람이 읽기 쉬운 요약으로 삽입합니다.
  - **{location_name}** 또는 **{{location.name}}** – 현재 로케이션의 이름을 삽입합니다.

## 문제 해결

빠른 진단으로 다운타임을 줄이고 팀이 지원 티켓 없이 일반적인 오류를 해결할 수 있도록 도와줍니다.

- **메시지가 전송되지 않음:** 발신 번호에 왓츠앱이 연결되어 있고 토글이 활성화되어 있는지 확인하세요.

- **변수가 해석되지 않음:** 변수 구문과 값이 존재하는지 확인하세요 (예: 예약 링크가 설정되어 있는지).

- **중복 메시지:** SMS Textback(SMS 문자 회신)이나 부재중 전화 시 메시지를 보내는 중복된 워크플로우가 있는지 확인하세요.

- **연락처가 받지 못함:** 연락처가 왓츠앱을 사용하고 수신 거부하지 않았는지 확인하고, 번호 형식(국가 코드 포함)을 확인하세요.

- **운영시간 외 메시징 우려사항:** 운영시간 외에 메시지를 보내고 싶지 않다면 해당 시간에 비활성화하거나 전화 라우팅 및 일정으로 관리하세요.

## **자주 묻는 질문**

**Q: SMS 부재중 전화 문자 회신을 활성화하지 않고 왓츠앱 회신만 활성화할 수 있나요?**
A: 네. 왓츠앱 회신을 단독으로 활성화하거나 SMS와 함께 활성화할 수 있습니다. 단일 주 채널을 선택하여 중복을 피하세요.

**Q: 메시지 본문은 어디서 편집하나요?**
A: Settings(설정) → Phone System(전화 시스템) → Voice(음성) → Missed Call Textback(부재중 전화 문자 회신) 안에서 편집합니다. WhatsApp(왓츠앱) → Templates(템플릿)에서는 이 메시지를 편집하지 마세요.

**Q: 운영시간 외에도 전송되나요?**
A: 네, 전화가 부재중으로 표시되면 전송됩니다. 연락처에게 알리려면 {business_hours}를 추가하거나, 다른 동작을 원하면 전화 라우팅을 조정하세요.

**Q: 연락처가 왓츠앱을 사용하지 않으면 어떻게 되나요?**
A: 왓츠앱 메시지가 전송되지 않습니다. 원한다면 SMS 부재중 전화 문자 회신이나 대안 워크플로우 활성화를 고려하세요.
