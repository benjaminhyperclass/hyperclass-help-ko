# Twilio 오류 21610 "블랙리스트 규칙 위반" SMS 발송 오류 해결

## **개요**

Twilio(또는 연동된 플랫폼)를 통해 SMS(문자)/MMS를 발송할 때, 수신자가 수신 거부를 했거나 방해 금지(DND) 설정을 활성화한 경우 메시지가 전달되지 않을 수 있습니다. 이는 사용자 선호도를 존중하고 메시징 규정을 준수하기 위한 조치입니다.

## **상세 설명**

메시지를 보내려는 상대방이 다음 중 하나에서 수신 거부를 설정한 상태입니다:

- 귀하의 전화번호
- 귀하의 채널 발송자
- 귀하의 메시징 서비스

또는 해당 연락처의 프로필에서 DND(방해 금지) 상태가 활성화되어 있을 수 있습니다.

수신자가 이전에 귀하의 메시지에 "STOP"으로 답장한 경우, 그들이 "START"로 응답할 때까지 다시 메시지를 보낼 수 없습니다.

## **가능한 원인**

- 수신자의 휴대폰이 STOP 또는 다른 수신 거부 키워드로 응답했습니다.
- 전화번호가 수신 동의를 하지 않은 새로운 사용자에게 재할당되었을 수 있습니다.
- 연락처가 SMS(문자), 전화 또는 모든 커뮤니케이션 유형에 대해 DND를 활성화했습니다.
- 플랫폼이 규정 및 컴플라이언스 요구사항을 준수하기 위해 발신 메시지를 차단했습니다.

## **해결 방법**

- 구독 취소한 번호를 연락처 목록에서 제거하세요.
- 수신자에게 귀하의 번호로 "START"(또는 다른 수신 동의 키워드)를 문자로 보내 재구독하도록 요청하세요.
- 발송 전 동의를 확인하세요 — 수신자가 메시지 수신에 명시적으로 동의했는지 확인하세요.

## **Twilio 수신 거부 동작 방식**

수신 거부한 수신자에게 메시지를 보내려고 시도하면, Twilio API가 오류를 반환하며 메시지는 발송되지 않습니다. 이 시도에 대해서는 요금이 청구되지 않습니다.

### 수신 거부 키워드 (구독 취소)

수신자가 다음 키워드 중 하나로 답장하면 메시지가 차단됩니다:

- STOP
- STOPALL
- UNSUBSCRIBE
- CANCEL
- END
- QUIT

### 수신 동의 키워드 (재구독)

메시지 수신을 재개하려면 수신자가 다음 키워드로 답장해야 합니다:

- START
- YES

## **중요 안내사항**

- 사용자가 메시징 서비스 번호에서 수신 거부하면, 해당 서비스에서 발송되는 모든 메시지에 수신 거부가 적용됩니다.
- 이는 구독자가 원치 않는 커뮤니케이션으로부터 완전히 보호받도록 보장합니다.

**핵심 포인트:** 항상 적절한 수신 동의를 확보하고, 수신 거부 요청을 존중하세요. 누군가가 수신 거부한 경우, 다시 메시지를 보내기 전에 반드시 수동으로 재동의를 받아야 합니다.
