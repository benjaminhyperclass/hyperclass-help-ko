# 왜 Hyperclass로 착신전환을 해야 하나요?

착신전환을 통해 개인 또는 회사 전화번호를 Hyperclass의 음성 AI 시스템과 매끄럽게 연결할 수 있습니다. Hyperclass 전화번호로 착신전환을 설정하면:

- 음성 AI 활용: AI 기반 통화 처리 및 음성 텍스트 변환으로 고객 커뮤니케이션을 향상시킬 수 있습니다.
- 기존 번호 유지: 마케팅 자료나 디렉토리를 업데이트할 필요 없이 현재 전화번호를 고객에게 그대로 노출할 수 있습니다.
- 업무 효율화: 모든 수신 전화를 Hyperclass 계정에서 관리하여 더 나은 추적과 관리가 가능합니다.

# 착신전환 작동 방식

착신전환이 활성화되면, 개인 번호(휴대폰 또는 유선전화)로 걸려온 전화가 자동으로 Hyperclass 전화번호로 리다이렉트됩니다. 그러면 Hyperclass가 고급 음성 AI 기능을 사용해 전화를 처리합니다.

# 주요 통신사별 착신전환 설정

통신사에 따라 아래 단계를 따라하세요. 해당하는 통신사가 없다면 해당 통신사의 지원 페이지에서 자세한 안내를 확인하세요.

### Verizon Wireless

Verizon Wireless 전화에서 Hyperclass 전화번호로 착신전환하는 방법은 여러 가지가 있습니다:

- 모든 통화 전환: *72를 누른 후 Hyperclass 전화번호를 입력합니다 (예: *72 555-555-5555).
- 부재중 통화 전환 (응답하지 않거나 통화 중 신호를 받는 통화): *71을 누른 후 Hyperclass 전화번호를 입력합니다 (예: *71 555-555-5555).
- 통화 버튼을 누릅니다.
- 확인음이나 메시지를 기다립니다.
- 착신전환을 해제하려면 *73을 누릅니다.

[Verizon Wireless 지원 페이지에서 자세히 보기](https://www.verizon.com/support/call-forwarding-faqs/)

### AT&T (Mobile)

- *21을 누른 후 Hyperclass 전화번호를 입력합니다 (예: *21 555-555-5555).
- 통화 또는 보내기 버튼을 누릅니다.
- 확인음을 기다립니다.
- 해제하려면 #21#을 누릅니다.

[AT&T 무선 지원 페이지에서 자세히 보기](https://www.att.com/device-support/article/wireless/KM1501781/ATT/WirelessHomePhoneWF721/)

### AT&T (Business)

- 유선전화에서 *72를 누르고 발신음을 들으세요.
- Hyperclass 전화번호를 입력합니다 (예: 555-555-5555).
- 확인음을 기다립니다. 시스템이 전환될 번호로 전화를 걸어 확인합니다. 전환 대상이 전화를 받으면 기능이 활성화됩니다.
- 착신전환을 해제하려면 *73을 누릅니다.

[AT&T 비즈니스 지원 페이지에서 자세히 보기](https://www.att.com/support/smallbusiness/article/smb-local-long-distance/KM1194925/)

### T-Mobile

T-Mobile 전화에서 Hyperclass 전화번호로 착신전환하는 방법은 여러 가지가 있습니다:

- 모든 통화 전환: **21*1 + Hyperclass 전화번호 + #을 누릅니다 (예: **21*1555-555-5555#)
- 부재중 통화 전환 (응답하지 않거나 통화 중 신호를 받는 통화): **61*1 + Hyperclass 전화번호 + #을 누릅니다 (예: **61*1555-555-5555#).
- 통화 버튼을 누릅니다.
- 착신전환을 해제하려면 각각 ##61# 또는 ##21#을 누릅니다.

[T-Mobile 지원 페이지에서 자세히 보기](https://www.t-mobile.com/support/plans-features/self-service-short-codes#secondheading)

# 자주 묻는 질문

**Q: 여러 번호에서 Hyperclass로 착신전환을 설정할 수 있나요?**
네, 각 번호가 착신전환을 지원한다면 여러 번호를 Hyperclass 전화번호로 전환할 수 있습니다.

**Q: 전화를 거는 사람이 착신전환되는 것을 알 수 있나요?**
아니요, 착신전환은 매끄럽게 이루어지며 발신자에게 리다이렉트 알림이 가지 않습니다.

**Q: 착신전환 설정 후에도 개인 번호로 전화를 걸 수 있나요?**
네, 착신전환은 수신 전화에만 영향을 미칩니다. 개인 번호에서 거는 발신 통화는 영향받지 않습니다.

**Q: 착신전환이 국제적으로도 작동하나요?**
통신사에 따라 다릅니다. 국제 착신전환이 지원되는지 통신사에 문의하여 확인하세요.
