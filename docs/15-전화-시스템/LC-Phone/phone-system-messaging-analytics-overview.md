## 메시징 분석이란?

메시징 분석은 LC Phone에 내장된 리포팅 도구로, 발신 메시징 활동에 대한 가시성을 제공합니다. 주요 지표, 상세한 메시지 상태, 필터링 기능에 대한 접근을 제공하여 사용자가 전송 성능을 이해하고 트렌드를 식별할 수 있도록 돕습니다.

## 메시징 분석의 주요 이점

- **성능 모니터링**: 전송, 전달, 미전달, 오류율을 실시간으로 추적합니다.
- **타겟팅 필터링**: 캠페인, 워크플로우, 날짜, 대량 발송 유형별로 데이터를 분석합니다.
- **실행 가능한 리포팅**: 메시지 수준의 세부 정보를 검토하고 대화로 클릭하여 맥락을 파악할 수 있습니다.
- **전송률 개선 인사이트**: 전송 패턴을 기반으로 문제를 식별하고 교정 조치를 취할 수 있습니다.

## 메시징 분석 옵션은 어디서 찾나요?

- 하위 계정 Settings(설정)으로 이동합니다.

- Messaging Analytics(메시징 분석) 탭을 클릭합니다.

## 메시징 분석 지표 설명

이러한 지표들은 참여 성능을 평가하고, 전송률 문제를 감지하며, 컴플라이언스 우려사항을 식별하는 데 도움을 줍니다.

- **Sent(전송)**: CRM 내에서 실패한 메시지를 제외하고 텔레포니 제공업체로 전송된 총 메시지 수입니다.
- **Delivered(전달)**: 수신자에게 성공적으로 전달된 총 메시지 수입니다.
- **Failed(실패)**: 번호 오류, 제공업체 또는 통신사 차단으로 인해 수신자에게 도달하지 못한 메시지입니다.
- **Error [30007](오류 [30007])**: 전화 통신사가 스팸, 피싱 또는 사기 콘텐츠로 메시지에 플래그를 지정했음을 나타냅니다.
- **Received(수신)**: 시스템에서 성공적으로 수신한 인바운드 메시지 수입니다.
- **Opted-Out(수신 거부)**: 추가 메시지 수신을 구독 해지하거나 거부한 총 수신자 수입니다.

## 고급 필터링 옵션

다양한 매개변수에 걸쳐 메시징 성능을 세그먼트할 수 있는 동적 필터를 적용하여 분석 뷰를 맞춤화하세요.

### 날짜 필터

사용자가 사용자 정의 기간에 걸쳐 결과를 비교하여 성능의 트렌드, 급증 또는 하락을 발견할 수 있게 해줍니다. 사용자는 다음을 기준으로 날짜 범위를 필터링할 수 있습니다:

- 현재 날짜
- 최근 7일
- 현재 월
- 현재 연도
- 원하는 사용자 정의 날짜 범위

![날짜 필터 사용법](https://jumpshare.com/v/JOuick7YSHCllPF1G75L+/GIF+Recording+2025-04-25+at+1.03.23+AM.gif)

### 메시징 필터

메시징 필터는 캠페인, 워크플로우, 대량 요청과 같은 특정 소스를 기반으로 메시징 데이터를 세그먼트하는 데 도움을 줍니다.

- **Campaign(캠페인)**: 할당된 캠페인 이름별로 메시지를 필터링합니다.
- **Workflow(워크플로우)**: 특정 자동화 내에서 메시지 성공률을 분석합니다.
- **Bulk Request(대량 요청)**: 대량 메시지 발송 결과를 확인합니다.

## 상세 메시지 리포팅

각 상태에 대한 상세한 메시지 보고서를 제공하며, 전체 맥락을 위해 대화 페이지로 직접 이동할 수 있는 기능을 제공합니다.

- 전송, 전달, 미전달 메시지 목록을 자세히 살펴보세요
- 각 항목을 클릭하여 전체 대화 뷰에 접근하세요
- 맥락 안에서 메시징 경로와 결과를 추적하세요

## 자주 묻는 질문

**Q: 분석 데이터를 다운로드하거나 내보낼 수 있나요?**
현재 메시징 분석 UI에서는 내보내기가 불가능합니다. 스크린샷을 찍거나 대화 탭의 상세 로그를 참조할 수 있습니다.

**Q: 캠페인이나 워크플로우 아래에 메시지가 표시되지 않는 이유는 무엇인가요?**
생성 시 메시지가 캠페인 또는 워크플로우 ID로 적절히 태그되었는지 확인하세요. 또한 선택한 날짜 범위에 예상 활동이 포함되는지 확인하세요.

**Q: 메시징 분석의 데이터는 얼마나 자주 업데이트되나요?**
분석 대시보드는 거의 실시간으로 업데이트되지만, 메시지 볼륨과 통신사 피드백에 따라 지연이 발생할 수 있습니다.

**Q: 전송률이 갑자기 떨어지면 어떻게 해야 하나요?**
최근 콘텐츠 변경 사항을 확인하고, 오류 코드를 검토하며, 발송 관행이 LC 메시징 정책을 준수하는지 확인하세요.

**Q: 전화번호나 연락처별로 필터링할 수 있나요?**
현재는 불가능합니다. 메시징 분석 필터는 캠페인, 워크플로우, 대량 요청, 날짜 수준에서만 작동합니다.

**Q: 'Failed(실패)'와 'Error(오류)' 상태의 차이점은 무엇인가요?**
'Failed'는 수신자에게 도달하지 못한 전송 시도(예: 유효하지 않은 번호)를 의미하고, 'Error'는 일반적으로 통신사별 거부 또는 시스템 문제를 반영합니다.

**Q: 수신 거부 추적이 중요한 이유는 무엇인가요?**
구독을 해지한 수신자를 식별하여 메시징 전략을 조정하고 컴플라이언스를 보장할 수 있게 도와줍니다.
