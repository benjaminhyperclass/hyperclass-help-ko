## **캠페인 사용 사례 유형이란?**

캠페인 사용 사례 유형은 SMS 또는 MMS 메시지의 구체적인 목적을 나타냅니다. 캠페인을 등록할 때는 고객 관리, 이중 인증, 마케팅 등 커뮤니케이션의 성격을 정확하게 설명하는 사용 사례를 선택해야 합니다. 올바른 캠페인 유형을 선택하는 것은 업계 규정 준수 기준을 충족하고, 필터링이나 차단을 방지하며, 통신사와 수신자의 신뢰를 유지하는 데 필수적입니다.

[A2P 10DLC와 캠페인 등록이 필요한 이유 자세히 알아보기](what-is-a2p-10-dlc-brand-and-campaign-registration.md)

---

## **올바른 사용 사례 선택의 주요 이점**

올바른 캠페인 사용 사례를 선택하면 메시지가 안정적으로 전달되고 규정을 준수하며 승인 과정에서 지연을 최소화할 수 있습니다.

- TCR(캠페인 등록소)의 캠페인 거부 위험 감소
- 통신사가 메시지 의도를 이해하고 스팸 필터링을 시행하는 데 도움
- 메시징 관행을 업계 및 법적 기준에 맞춤
- 더 빠른 승인과 규정 준수 문제 감소 지원
- 장기적인 전달률과 평판 개선

## **승인된 캠페인 사용 사례 유형 목록**

다음은 The Campaign Registry에서 정의한 일반적으로 허용되는 캠페인 사용 사례 유형 표입니다. Hyperclass에서 캠페인을 등록할 때 메시징 의도에 가장 적합한 것을 선택하세요.

| **사용 사례 유형** | **설명** | **예시** |
|---|---|---|
| 2FA | 로그인 또는 신원 확인을 위한 이중 인증 메시지 | "인증 코드는 654321입니다." |
| Account Notifications(계정 알림) | 계정 상태, 변경 사항 또는 활동에 대한 업데이트 | "청구 정보가 업데이트되었습니다." |
| Customer Care(고객 관리) | 고객이 시작한 지원 또는 서비스 메시지 | "요청을 받았습니다. 업데이트 내용은 다음과 같습니다..." |
| Delivery Notifications(배송 알림) | 상품이나 서비스의 배송 또는 이행 알림 | "패키지가 배송되었습니다." |
| Fraud Alert Messaging(사기 알림) | 의심스럽거나 무단 계정 활동에 대한 알림 | "500달러 출금을 시도하셨나요?" |
| Higher Education(고등교육) | 대학교에서 발송하는 커뮤니케이션 | "수강 등록이 시작되었습니다." |
| Marketing(마케팅) | 홍보 콘텐츠 및 판매 제안 | "SPRING25 코드로 25% 절약하세요!" |
| Mixed(혼합) | 마케팅 + 알림처럼 여러 사용 사례를 결합 | 프로모션 + 예약 알림 메시지 |
| Polling and Voting(설문 및 투표) | 비정치적 설문 조사 또는 투표 메시지 | "다음에 추가했으면 하는 기능은 무엇인가요?" |
| Public Service Announcement(공공 서비스 공지) | 비상업적, 인식 개선 기반 메시징 | "독감 예방접종을 클리닉에서 받으실 수 있습니다." |
| Security Alerts(보안 알림) | 잠재적인 시스템 또는 물리적 보안 위험에 대한 알림 | "무단 로그인 시도가 감지되었습니다." |

## **특별 캠페인 사용 사례**

특정 캠페인은 통신사에서 "특별"로 분류하며 추가 문서화, 검증 또는 사전 자격 요건이 필요합니다. 이는 종종 민감하거나 규제 대상인 커뮤니케이션과 관련됩니다. 특별 사용 사례는 다른 수수료, 검증 요구 사항, 통신사별 제한사항이 적용될 수 있습니다.

- **Charity / 501(c)(3) Nonprofit(자선단체/비영리)**: 등록된 비영리 조직의 자선 활동을 위함. IRS 비영리 검증 자료 제공 필요
- **Emergency Services(비상 서비스)**: 정부 기관의 긴급 안전 또는 비상 메시지 전달용. 엄격히 제한되고 규제됨
- **Political(정치)**: 선거 캠페인, 정당, PAC을 포함한 정치적 메시징. Campaign Verify를 통한 검증 필요

## **소량 발송 사용 사례**

표준 "Mixed(혼합)" 또는 "Marketing(마케팅)" 캠페인 유형 외에도 Low Volume Mixed(소량 혼합)라는 전용 사용 사례가 있습니다. 이 옵션은 SMS 발송량이 적은 비즈니스를 위한 것입니다. 표준 혼합 캠페인과 동일한 범위의 메시지 유형(마케팅, 고객 관리, 알림)을 지원하지만 월 사용료가 더 저렴합니다. 다른 캠페인 유형과 달리 소량 혼합은 브랜드의 Trust Score에 관계없이 가장 낮은 처리량 등급으로 고정됩니다. 이는 대용량 메시징이 필요하지 않은 소규모 비즈니스나 조직에 이상적이고 비용 효율적인 선택입니다.

## **금지된 사용 사례**

| **사용 사례 유형** | **예시** | **세부 사항** |
|---|---|---|
| 고위험 금융 서비스 | 급전 대출, 단기 고금리 대출, 제3자 자동차 대출, 제3자 주택담보대출, 학자금 대출, 암호화폐 | "제3자"는 대출을 서비스할 당사자가 아닌 다른 모든 당사자를 의미합니다. |
| 제3자 리드 생성 서비스 | 소비자 정보를 구매, 판매 또는 공유하는 회사 |  |
| 채무 추심 또는 탕감 | 제3자 채무 추심, 채무 통합, 채무 감축, 신용 수리 프로그램 | "제3자"는 채무를 빌려준 당사자가 아닌 다른 모든 당사자를 의미합니다. 예를 들어, 병원은 환자가 메시징 수신에 동의한 경우 환자의 청구서에 대한 메시지를 보낼 수 있습니다. |
| "쉽게 돈 버는" 계획 | 재택근무 프로그램, 위험 투자 기회, 피라미드 사기 | 이는 규정을 준수한 옵트인 관행으로 인한 고용 관련 홍보, 중개업체가 회원에게 보내는 메시지, 투자 뉴스 알림 또는 기타 투자 관련 메시지와는 다릅니다. |
| 불법 물질 | 대마초(미국), CBD(미국), 처방약 | 대마초는 미국에서 연방적으로 불법입니다. 대마초 사업체는 메시지 내용에 관계없이 미국에서 SMS/MMS 메시징을 사용할 수 없습니다.<br><br>CBD는 연방적으로는 합법이지만 일부 미국 주에서는 불법이므로, 미국 통신사는 CBD 관련 메시징을 허용하지 않습니다.<br><br>미국/캐나다에서 일반의약품으로 판매될 수 없는 약물 제안은 금지됩니다. |
| 도박 | 카지노 앱, 도박 웹사이트 | 도박 트래픽은 미국과 캐나다에서 모든 번호 유형(무료 전화, 단축 번호, 긴 번호)에서 금지됩니다. |
| "S.H.A.F.T." 사용 사례 | 성인, 혐오, 주류, 총기, 담배 | 담배 트래픽은 무료 전화에서는 금지되지만, 적절한 연령 확인 절차가 있는 경우 단축 또는 긴 번호에서는 허용됩니다. 주류 트래픽은 적절한 연령 확인 절차가 있는 경우 무료 전화(미국만), 단축 번호, 긴 번호에서 허용됩니다. 총기 및 전자담배 관련 트래픽은 금지됩니다. |

## **자주 묻는 질문**

**Q: 잘못된 캠페인 사용 사례를 선택하면 어떻게 되나요?**
잘못된 사용 사례를 선택하면 캠페인 거부, 전달 필터링 또는 규정 준수 문제가 발생할 수 있습니다. 메시지의 주요 의도에 가장 잘 맞는 사용 사례를 항상 선택하세요.

**Q: 둘 이상의 캠페인 사용 사례를 등록할 수 있나요?**
네. 비즈니스에서 다른 유형의 메시지(예: 마케팅 및 고객 지원)를 보내는 경우 각각 적절한 사용 사례로 여러 캠페인을 등록해야 할 수 있습니다.

**Q: 캠페인이 승인된 후 사용 사례를 변경할 수 있나요?**
아니요. 사용 사례를 변경하려면 업데이트된 세부 사항과 샘플 메시지로 새 캠페인을 등록해야 합니다.
