웹사이트 방문자와 쉽게 소통하고 고객으로 전환하는 간편한 방법을 찾고 계신가요? Squarespace용 LeadConnector 웹채팅이 완벽한 솔루션입니다! 이 가이드에서는 Squarespace용 LeadConnector 웹채팅을 시작하는 단계별 방법을 안내해드립니다.

- **방법 1:** 코드 인젝션을 사용하여 커스텀 채팅 위젯 추가

# 웹사이트용 웹 채팅 임베드 코드 얻는 방법:

Sites(사이트) > Chat Widget(채팅 위젯) > Get Code(코드 가져오기)에서 임베드 코드를 복사하세요.

![채팅 위젯 임베드 코드 복사하기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48297593113/original/-2HR2Sq7Z1k6RMe8Up24qHI6Wrph22F4jA.gif?1684166690)

## Squarespace 웹사이트에 커스텀 채팅 위젯을 추가하는 다양한 방법의 상세 단계:

### 방법 1: 코드 인젝션을 사용하여 커스텀 채팅 위젯 추가

- 위에서 설명한 대로 채팅 위젯 코드를 얻습니다.
- Squarespace 계정에 로그인하고 채팅 위젯을 추가할 웹사이트로 이동합니다.
- 메인 메뉴에서 "Settings"를 클릭합니다.

![Settings 메뉴 클릭](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48297298463/original/7osDmJaDbDDGyz3eoae8jWLZgNntqPfhXQ.jpeg?1683912499)

- "Website" 섹션에서 "Developer Tools"를 선택합니다.

![Developer Tools 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48297298462/original/lEupdS6onJfVFtT3B7N2vJtchbGiLmOkcg.jpeg?1683912499)

- "Code Injection"을 클릭합니다.

![Code Injection 클릭](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48297298464/original/avj99lSZILiLAh4n0vlBtaJvR05DoOeg-w.jpeg?1683912499)

- "Header" 또는 "Footer" 필드에 채팅 위젯 코드를 붙여넣습니다.

![헤더나 푸터에 코드 붙여넣기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48297298461/original/PuvfzPL-G4QWwZZBSituwfgcKuI0inTvbQ.jpeg?1683912498)

- "Save"를 클릭하여 변경사항을 저장합니다.
- 웹사이트를 미리보기하여 채팅 위젯이 올바르게 표시되는지 확인합니다.

#### 방법 2: 임베드 블록을 사용하여 커스텀 채팅 위젯 추가

- Squarespace 계정에 로그인하고 채팅 위젯을 추가할 페이지로 이동합니다.
- 페이지 편집기를 클릭하고 채팅 위젯을 추가할 섹션을 선택합니다.
- 더하기 아이콘을 클릭하여 새 콘텐츠 블록을 추가하고, 드롭다운 메뉴에서 "Embed"를 선택합니다.
- "Code Snippet" 필드에 채팅 위젯 코드를 붙여넣습니다.
- "Apply"를 클릭하여 변경사항을 적용합니다.
- 웹사이트를 미리보기하여 채팅 위젯이 올바르게 표시되는지 확인합니다.
