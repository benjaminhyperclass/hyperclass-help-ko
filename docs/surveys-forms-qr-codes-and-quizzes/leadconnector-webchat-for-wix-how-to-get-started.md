# 윅스(Wix)용 LeadConnector 웹채팅 시작 가이드

웹사이트 방문자와 쉽게 소통하고 고객으로 전환시킬 수 있는 방법을 찾고 계신가요? LeadConnector의 웹채팅이 완벽한 솔루션입니다! 이 가이드에서는 윅스(Wix) 웹사이트에 LeadConnector 웹채팅을 시작하는 방법을 단계별로 안내해드리겠습니다.

# 웹사이트용 채팅 위젯 임베드 코드 얻는 방법 {#how-to-obtain-embed-code}

`Sites(사이트) > Chat Widget(채팅 위젯) > Get Code(코드 얻기)`에서 임베드 코드를 복사하세요.

![윅스용 채팅 위젯 코드 복사](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48297592882/original/jTgT3BrxGK8oc5KmblvC2gSRcbGNHz6d6A.gif?1684166612)

## 윅스 웹사이트에 커스텀 채팅 위젯을 추가하는 상세 방법 {#detailed-steps-for-adding-widget}

### 방법 1: 커스텀 코드 사용 {#method-1}

- 윅스 계정에 로그인한 후 웹사이트 대시보드로 이동하세요.
- 왼쪽 메뉴에서 "Settings(설정)"을 클릭한 다음, "Advanced(고급)" 탭 아래의 "Custom Code(커스텀 코드)"를 선택하세요.

![윅스 설정에서 커스텀 코드 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48297317762/original/afvuactXKgGh8MrZioPfgp44yPZvZ9dVMQ.jpeg?1683921093)

- 해당 섹션에서 "Add Custom Code(커스텀 코드 추가)"를 클릭하세요.
- 팝업 창의 "Code Settings(코드 설정)"에 코드 스니펫을 붙여넣으세요.
- 다음 옵션을 설정하세요:
  - Name(이름): 코드의 이름을 입력하세요
  - Add Code to Pages(코드를 추가할 페이지): "All Pages(모든 페이지)"를 선택하고 "Load code once(코드 한 번만 로드)"를 선택하세요.
  - Place Code In(코드 위치): "Head"를 선택하세요
- "Apply(적용)"을 클릭하여 변경사항을 저장하세요.

![윅스 커스텀 코드 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48297317756/original/w4V_CFuVKnEFUt7QitNud1aLOySbUz5emQ.png?1683921092)

- 이제 위젯이 윅스 사이트의 모든 페이지에 표시됩니다.

### 방법 2: HTML 임베드 사용 {#method-2}

- 윅스 계정에 로그인한 후 윅스 에디터를 실행하세요. 대시보드에서 "Edit Site(사이트 편집)"을 클릭하세요.

![윅스 에디터에서 사이트 편집](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48297317759/original/9B-EBPNVDe9gBv25cEYvyoVTLUMwuVXzNQ.jpeg?1683921093)

- 왼쪽 메뉴에서 플러스(+) 아이콘을 클릭하여 "Add Elements(요소 추가)"에 접근하세요.
- 왼쪽 사이드바 메뉴에서 "Embed(임베드)"를 선택하세요.
- "Popular Embeds(인기 임베드)" 아래의 "Embed HTML(HTML 임베드)"를 선택하세요.

![윅스에서 HTML 임베드 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48297317761/original/If6mts5CUAfyzOaTOstmGhmjsYsATMm4MA.jpeg?1683921093)

- 나타나는 새 HTML 박스에서 "Code(코드)"를 클릭하세요.
- 위젯 코드를 텍스트 필드에 붙여넣고 "Update(업데이트)"를 누르세요.

![윅스 HTML 코드 붙여넣기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48297317760/original/KLsgOAnNolAWPwLltKgtg67R88vUyXUHug.jpeg?1683921093)

- "HTML Settings(HTML 설정)" 박스를 닫으세요.
- 이제 위젯이 페이지의 HTML 박스에 표시됩니다.
