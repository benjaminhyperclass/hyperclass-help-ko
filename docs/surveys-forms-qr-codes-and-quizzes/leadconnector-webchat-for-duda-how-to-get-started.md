웹사이트 방문자와 쉽게 소통하고 고객으로 전환시킬 수 있는 방법을 찾고 계신가요? Duda용 LeadConnector 웹채팅이 완벽한 해결책입니다!

## 웹사이트용 웹채팅 임베드 코드 받는 방법

Sites(사이트) > Chat Widget(채팅 위젯) > Get Code(코드 받기)에서 임베드 코드를 복사하세요.

![웹채팅 임베드 코드 복사하기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48297592777/original/6rdSZ7JqMTshr8uKd5s-GywNe4t5E2A5MQ.gif?1684166586)

## Duda 웹사이트에 커스텀 채팅 위젯을 추가하는 자세한 방법들

### 방법 1:

- Duda 계정에 로그인하세요: Duda 대시보드에 로그인하고 웹사이트의 편집 모드를 여세요.
- 사이트 편집: 채팅 위젯을 추가하고 싶은 사이트로 이동한 후 "Edit Site(사이트 편집)" 버튼을 클릭하세요.

![Duda 사이트 편집 버튼](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48297323152/original/IVR_6V44Ah7b44fAlmFne9rKqT0eVapOBA.jpeg?1683924088)

- 왼쪽 컬럼에서 Settings(설정)을 클릭하세요.
- 그 다음 Head HTML을 클릭하세요.
- 임베드 코드를 붙여넣으세요: 채팅 위젯의 임베드 코드를 편집기에 붙여넣으세요.

![Duda Head HTML 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48297323153/original/L6onCa0dc86TLDlBnuCndLMfBTdHuPS3MA.jpeg?1683924088)

- 변경사항을 저장하세요: "Save(저장)" 버튼을 클릭하여 변경사항을 저장하세요.
- 채팅 위젯을 미리보기하세요: 사이트를 미리보기하여 위젯이 올바른 위치에 나타나고 예상대로 작동하는지 확인하세요.

### 방법 2:

편집기에서 HTML 요소를 사용하여 채팅 위젯을 추가할 수도 있습니다.

- HTML 요소 추가: 편집기에서 채팅 위젯을 표시하고 싶은 위치에 HTML 요소를 드래그 앤 드롭하세요.
- HTML 편집기 열기: 방금 추가한 HTML 요소를 더블클릭하여 HTML 편집기를 여세요.
- 임베드 코드 붙여넣기: 채팅 위젯의 임베드 코드를 HTML 편집기에 붙여넣으세요.
- 변경사항 저장: "Save(저장)" 버튼을 클릭하여 변경사항을 저장하세요.
