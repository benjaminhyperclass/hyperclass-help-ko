# Shopify용 LeadConnector 웹챗 시작하기

웹사이트 방문자와 쉽게 소통하고 고객으로 전환할 수 있는 방법을 찾고 계신가요? Shopify용 LeadConnector 웹챗이 완벽한 솔루션입니다! 이 가이드에서는 Shopify용 LeadConnector 웹챗을 시작하는 단계를 안내해드립니다.

# 

---

# 웹사이트용 웹챗 임베드 코드 얻기 방법:

사이트(Sites) > 채팅 위젯(Chat Widget) > 코드 가져오기(Get Code)에서 임베드 코드를 복사하세요.

![웹챗 임베드 코드 복사하기](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48297596658/original/PSsUcBbLJWJKhtf5HfaJApA-SnuxjakuYw.gif?1684167749)

## 다양한 방법으로 Shopify 웹사이트에 커스텀 채팅 위젯을 추가하는 자세한 단계:

- Shopify 계정에 로그인: Shopify 계정으로 이동해서 로그인하세요.
- "온라인 스토어(Online Store)" 섹션으로 이동: Shopify 대시보드에서 "온라인 스토어(Online Store)" 섹션으로 이동하세요.

![Shopify 온라인 스토어 섹션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48297327387/original/Xo0wZGJkc3cetfK5pkEHnbLEqKKVYkMMbQ.jpeg?1683926759)

- 웹사이트 테마 편집: 편집할 테마를 선택하고 "작업(Actions)" 버튼을 클릭하세요. 드롭다운 메뉴에서 "코드 편집(Edit code)"을 선택하세요.

![테마 선택 및 코드 편집](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48297327386/original/Na1JE2ESDFLGPh9FlOa3HgnYSXpg23FAkA.jpeg?1683926759)

![코드 편집 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48297327380/original/HBRae8Gcy0FfwAMHQGIXtLY7973APBmdtg.png?1683926759)

- 테마의 theme.liquid 파일 열기: 나타나는 파일 목록에서 "**Layout**" 폴더를 선택한 다음 **"theme.liquid"** 파일을 선택하세요.

![Layout 폴더에서 theme.liquid 파일 선택](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48297327383/original/EapE6bR1rjU8x9KAx6BKgkx_JsYvceQbpw.jpeg?1683926759)

- 임베드 코드 붙여넣기: theme.liquid 파일 내에서 </body> 태그를 찾은 다음 그 위에 임베드 코드를 추가하세요.

![body 태그 위에 임베드 코드 추가](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48297327382/original/HNLFBinrelhZHO3a_FDjxSHnogvs7bXbbA.jpeg?1683926759)

- 변경사항 저장: 임베드 코드를 추가한 후 저장 버튼을 클릭하세요.

![변경사항 저장](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/48297327381/original/BPtt2-YEGJ4uatoEbuusWWpgAi4rYOruqg.jpeg?1683926759)

- 채팅 위젯 미리보기: 웹사이트로 이동해서 페이지를 새로고침하세요. 이제 채팅 위젯이 웹사이트에 표시됩니다.
- 정상적으로 작동하는지 테스트해보세요.
