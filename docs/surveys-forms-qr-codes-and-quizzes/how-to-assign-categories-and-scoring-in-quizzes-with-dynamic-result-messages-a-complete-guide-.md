# 퀴즈 카테고리 분류와 점수 설정 방법 - 동적 결과 메시지 완벽 가이드

퀴즈에서 답변에 점수를 부여하고 카테고리로 분류하는 기능입니다. 응답에 따라 평가, 추천, 세분화 등 유용한 퀴즈 결과를 생성할 수 있고, 점수 구간별로 개인화된 결과 메시지를 표시할 수 있습니다. 평가 도구, 성격 테스트, 온보딩 설문 등에 완벽합니다.

# 퀴즈에 점수 추가하는 방법

## 1단계: 퀴즈 생성 또는 편집

- 계정에 로그인합니다.
- Sites(사이트) > Surveys(설문)으로 이동합니다.
- 기존 퀴즈를 선택하거나 + New Survey(새 설문)를 클릭해서 새로 만듭니다.

## 2단계: 퀴즈에 질문 추가

- Add Question(질문 추가)을 클릭하고 질문 유형을 선택합니다(단일 선택, 다중 선택, 드롭다운 등).
- 질문과 가능한 답변 선택지를 입력합니다.
- 각 답변 선택지에 가중치에 따라 점수를 부여합니다.

## 3단계: 카테고리에 점수 할당

- 점수 카테고리를 생성합니다(예: 카테고리 A, 카테고리 B 등).
- 각 답변에 점수를 할당합니다:
  - 단일 선택/드롭다운 질문: 답변당 고정 점수를 할당합니다.
  - 다중 선택/다중 드롭다운: 여러 선택지를 선택할 경우 누적되는 옵션별 점수를 할당합니다.
- 각 답변의 점수를 해당 카테고리에 할당합니다.
- 질문의 답변이 여러 카테고리에 적용되는 경우, 해당하는 모든 카테고리를 선택합니다.

## 4단계: 저장 및 미리보기

- 모든 질문과 점수 설정을 완료했으면 Save Quiz(퀴즈 저장)를 클릭합니다.
- Preview(미리보기) 옵션을 사용해서 퀴즈를 테스트하고 점수가 올바르게 계산되는지 확인합니다.

# 퀴즈 점수 계산 방식

## 1단계: 각 질문의 최대 점수 설정

모든 질문에는 유형에 따라 최대 점수가 있습니다:

- **단일 선택/드롭다운**: 하나의 답변만 선택할 수 있으므로 최대 점수는 고정입니다.
- **다중 선택/다중 드롭다운**: 여러 답변을 선택할 수 있으므로 최대 점수가 더 높습니다. 최대 점수는 고정이며 드롭다운/다중 선택의 모든 옵션 점수의 합입니다.

#### 예시:
- 단일 선택 질문의 최대 점수는 10점일 수 있습니다.
- 다중 선택 질문은 최대 50점까지 허용할 수 있습니다.

## 2단계: 답변에 점수 할당

퀴즈 응답자가 답변을 선택하면 다음에 기여합니다:

- **전체 점수만(Overall Score Only)**: 전체 퀴즈 점수에만 계산되고 특정 카테고리에는 포함되지 않습니다.
- **카테고리별 점수**: 특정 카테고리와 전체 점수 모두에 계산됩니다.
- **여러 카테고리**: 동시에 두 개 이상의 카테고리에 영향을 줍니다.

예시:
- 퀴즈에 카테고리 A, B, C가 있고 답변이 카테고리 A에 할당된 경우, 점수는 카테고리 A와 전체 점수에 계산됩니다.

## 3단계: 최종 점수 계산

모든 답변이 선택되면 다음 공식으로 최종 점수가 계산됩니다:

**최종 점수 = (선택된 점수 / 최대 점수) × 100**

이는 전체 점수와 각 카테고리에 대해 별도로 계산됩니다.

#### 계산 예시

| 카테고리 | 선택된 점수 | 최대 점수 | 최종 % | 최종 점수 공식 |
|---------|------------|----------|--------|---------------|
| 전체 | 90 | 130 | 69.23% | 전체 선택 점수 / 전체 최대 가능 점수 |
| 카테고리 A | 20 | 50 | 40.00% | 카테고리 A 선택 점수 / 카테고리 A 최대 가능 점수 |
| 카테고리 B | 20 | 50 | 40.00% | 카테고리 B 선택 점수 / 카테고리 B 최대 가능 점수 |
| 카테고리 C | 10 | 50 | 20.00% | 카테고리 C 선택 점수 / 카테고리 C 최대 가능 점수 |

이 예시에서:
- 퀴즈 응답자는 전체 69.23%를 득점했습니다.
- 카테고리 A에서는 50점 중 20점으로 40%를 득점했습니다.

## "전체 점수만"에 대한 핵심 사항

- **"전체 점수만"은 통합 점수입니다**
  - 답변이 "전체 점수만"에만 할당되면 어떤 카테고리에도 계산되지 않습니다.
  - 답변이 카테고리와 "전체 점수만" 모두에 할당되면 둘 다에 계산됩니다.

- **카테고리는 독립적이지만 전체 점수에 기여합니다**
  - 각 카테고리의 점수는 별도로 계산되지만, 전체 점수는 모든 카테고리 점수의 합입니다.

- **공식 분석**
  - 최종 점수 공식은 다음과 같이 계산됩니다: 각 카테고리에서 X의 최대 선택 점수 / 최대 가능 점수
  - 이는 전체, 카테고리 A, 카테고리 B 등에 적용되어 백분율 점수를 도출합니다.

# 점수 구간별 동적 결과 메시지 추가

최종 점수 백분율에 따라 맞춤형 결과 메시지를 표시할 수 있습니다(예: 0–40% = 낮음, 41–70% = 보통, 71–100% = 높음).

#### 1. 결과 페이지로 이동

- 퀴즈 빌더 내에서 "Result Page(결과 페이지)"라고 표시된 마지막 섹션으로 스크롤합니다.

![결과 페이지 섹션](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046729123/original/CgN0gGBMJjl4Qh88m9ro2sSUHJkFnXdsXg.png?1747377244)

#### 2. 점수 구간 정의

- 결과 페이지 아래에서 "Score Range Settings(점수 구간 설정)"을 찾습니다.
- 임계값을 정의합니다:
  - 낮음: 0% ~ 40%
  - 보통: 41% ~ 70%
  - 높음: 71% ~ 100%
- 퀴즈 요구사항에 따라 이러한 구간을 맞춤 설정할 수 있습니다.

![점수 구간 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046729134/original/5fZthUNBRAMblaMiV3xFQV4dg4r-iioN6Q.png?1747377244)

#### 3. 각 구간에 맞춤 메시지 작성

- 각 구간(낮음/보통/높음)에 대해 맞춤 메시지를 추가합니다.
- 예시:
  - 낮음: "개선의 여지가 있어 보입니다. 저희가 어떻게 도움을 드릴 수 있는지 알아보세요."
  - 보통: "올바른 방향으로 가고 있습니다! 더욱 최적화할 수 있는 방법을 알아보세요."
  - 높음: "정말 훌륭합니다! 완벽하게 해내셨네요. 다음 단계에 대해 이야기해보세요."

![맞춤 메시지 설정](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046729132/original/rMdRpDnvfg2xi7Wl2RIkUhBMAT2a6coWGg.png?1747377244)

#### 4. CTA(행동 유도) 추가 (선택사항)

- 다음과 같은 CTA를 포함할 수 있습니다:
  - 구독
  - 상담 예약
  - 퍼널로 리디렉션
- 결과 페이지의 "CTA" 박스에서 맞춤 설정할 수 있습니다.

# 권장 사항 및 문제 해결

- 퀴즈 결과를 세분화하기 위해 카테고리를 효과적으로 사용하세요.
- 답변 옵션이 해당 카테고리에 올바르게 매핑되었는지 확인하세요.
- 최종 점수가 잘못된 경우? 선택된 점수와 카테고리 할당이 올바른지 확인하세요.
- 카테고리 점수가 누락된 경우? 답변 옵션이 카테고리에 할당되었는지 확인하세요.
- 총점이 맞지 않는 경우? "전체 점수만"은 개별 카테고리에 계산되지 않는다는 점을 기억하세요.

이 단계들을 따르면 더 나은 평가와 인사이트를 위해 퀴즈에서 계산 필드를 쉽게 설정하고 관리할 수 있습니다!
